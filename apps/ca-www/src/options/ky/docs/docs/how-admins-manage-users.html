Админдер колдонуучуларды кантип башкарат

Администраторлор колдонмолорго жана кейстерге кире ала турган адамдарды түзөт, өзгөртөт же өчүрөт. Колдонуучу платформага биринчи жолу киргенде, профилдик иш процессин бүтүрүшү керек.

  • Жаңы колдонуучу профилинин иштөө процесси жөнүндө көбүрөөк билүү үчүн бул жерди басыңыз.

Колдонуучуну башкаруу

Төмөнкү таблицада колдонуучуну башкаруунун беш жолу көрсөтүлгөн. Кантип үйрөнүү үчүн шилтемени басыңыз.

Метод Type Аракеттер Толук маалымат
UI колдонуу UI кошуу жана өзгөртүү Администратор баскычы > Колдонуучулар
Ишти түзүп жатканда жана колдонуучуну ишке дайындаңыз UI кошуу Администратор баскычы > Иштер > Жаңы баскыч > Иш барагын түзүү
Коомдук катталуу шилтемеси аркылуу колдонуучуну автоматтык түрдө түзүү Колдонуучунун аракети кошуу Администратор баскычы > Орнотуулар > Системаны орнотуу барагы
Импорт куралы UI кошуу жана өзгөртүү Администратор баскычы > Интеграциялар > Импорттор
API колдонуу (жакында) API кошуу жана өзгөртүү Жок

Эскертмелер

Платформа колдонуучуларга колдонмого кошулганда эскертме жөнөтпөйт. Бирок, администраторлор ишти дайындоо үчүн эскертмелерди иштете алышат.
