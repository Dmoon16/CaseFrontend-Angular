Импорт куралы менен кейстерди кантип түзүү же өзгөртүү керек

Бул макалада администраторлорго Импорт куралы менен учурдагы учурларды кантип түзүү же өзгөртүү керектиги көрсөтүлөт.

Импорт куралын ачуу

  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Администратор панелинен Интеграция өтмөгүн, андан кийин субменюдагы ИМПОРТ баскычын чыкылдатыңыз.
  4. ЖАҢЫ баскычын басыңыз. Импортту кошуу диалог кутусу жаңы серепчинин өтмөгүндө көрсөтүлөт.

Иштерди түзүү

Импортту кошуу диалог кутусу ачылгандан кийин:

  1. Тип талаасын чыкылдатып, параметрди тандаңыз (төмөндө караңыз). Шилтеме көрсөтүлөт.
  2. Үлгү жүктөп алуу шилтемесин басыңыз. CSV файлы түзмөгүңүзгө жүктөлөт.

Андан кийин, файлды ачып, иштерди кошуңуз.

Type

Тип талаасын басканда, ылдый түшүүчү меню пайда болот. Жаппай импорттоо үчүн эки варианттын бирин тандаңыз.

  • Эскертүү Ашылма менюда көптөгөн параметрлер бар, бирок экөө гана ишти башкарууга байланыштуу. Ар бир параметрде башка үлгү CSV файлы бар.

Ишти кошуу

  • Иштерди жөнөкөй жапырт жүктөө үчүн бул параметрди тандаңыз.
  • Иштерди түзүү импорт түрү үчүн тилке аталыштары менен CSV файлын ачуу үчүн Жүктөп алуу үлгүсүн басыңыз.

Иштерди өзгөртүү

  • Учурдагы иштерди жаңыртуу үчүн бул параметрди тандаңыз.
  • Жүктөө үлгүсүн басыңыз шилтемени чыкылдатыңыз CSV файлын ачуу үчүн графанын аталыштары менен Иштерди импорттоо түрүн өзгөртүү.

Файл жүктөлүүдө

Сиз файлды дароо жүктөй аласыз же кийинчерээк Импортту кошуу диалог кутусуна кайтып келип, бул кадамдарды аткара аласыз.

  1. Импортту кошуу диалог кутусун ачыңыз.
  2. CSV файлы үчүн уникалдуу атын киргизиңиз.
  3. Файлды түзмөгүңүздөн жүктөө үчүн плюс (+) белгисин басыңыз.
  4. Тип талаасын чыкылдатып, параметрди тандаңыз. (жогоруда кара)
  5. SAVE басыңыз.

Форматтоо

Бул талап кылынган талаалар жана биринчи талаптар. Форматтоо чоо-жайын карап чыгуу үчүн бул жерди басыңыз.

tag_id (же case_ID) уникалдуу идентификатор (алфавиттик-сандык белгилерди колдонуңуз, дефис кабыл алынат)

case_status 0 = жабык 1 = активдүү

дата_ачылган ай/күн/жыл мм/кд/жжж

иштин уникалдуу сүрөттөлүшү жөнүндө (алфавиттик-сандык белгилерди колдонуу, дефис кабыл алынат)

Иштетүү

Файлдагы жана тутумдук жүктөгү жазуулардын санына жараша, платформа импортту иштетүү үчүн бир нече мүнөт талап кылынышы мүмкүн.

img

Импорттоо аяктаганда, платформа администратордун Иштер тактасында иш жазууларын тууралайт (б.а. кошот же өзгөртөт).

Ийгиликсиздик

Импорттоо операциясы ишке ашпай калса, платформа Failed билдирүүсүн кайтарат. Көпчүлүк учурларда, ката себеби маалыматтарды форматтоо болуп саналат.

img

Мүчүлүштүктүн себебин көрүү үчүн:

  1. АРАКЕТ башынын астындагы Көрүү дегенди басыңыз.
  2. Кийинки бетте чычканды Failed сөлөкөтүнө алып барыңыз. Курал кеңеши катаны көрсөтөт.
  3. CVS файлына өтүп, катаны тууралап, оңдоңуз.
  4. Файлды кайра жүктөңүз.
