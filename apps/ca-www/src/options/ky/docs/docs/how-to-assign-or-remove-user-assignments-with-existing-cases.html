Колдонуучунун дайындоолорун учурдагы учурлар менен кантип ыйгаруу же алып салуу керек

Бул макалада администраторлорго ишти колдонуучуларды кантип дайындоо жана алып салуу керектиги көрсөтүлөт. Биз ошондой эле колдонуучу дайындарын кантип өзгөртүү керектигин көрсөтөбүз.

Учурдагы ишти ачуу

  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Cases меню өтмөктү чыкылдатыңыз.
  4. Субменюда ACTIVE тандалганын текшериңиз.
  5. АРАКЕТ тилкесине чейин ылдый сыдырыңыз.

Дайындоо же алып салуу

Колдонуучуну ишке дайындоо же колдонуучуну иштен алып салуу (бирок платформадан эмес) үчүн "Колдонуучуларды дайындоо" баскычын басыңыз. Кийинки бетте: КОШУУ баскычын чыкылдатып, иш процессин бүтүрүңүз. Колдонуучуну жок кылуу үчүн өчүрүү чыкылдатыңыз.

Өзгөртүү

Cases бетинде колдонуучунун маалыматтарын түзөтүү же жаңыртуу мүмкүн эмес. Анын ордуна, бул кадамдарды аткарыңыз:

  1. Колдонуучулар меню өтмөктү чыкылдатыңыз.
  2. Колдонуучунун атын табыңыз жана чыкылдатыңыз Түзөтүү .
  3. Өзгөртүүлөрдү киргизип, САКТОО баскычын басыңыз.
