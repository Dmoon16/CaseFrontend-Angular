UI менюсу аркылуу иштерди кантип түзүү же өзгөртүү керек

Бул макалада администраторлор платформанын колдонуучу интерфейси менен ишти кантип түзүүнү көрсөтөт. Администраторлор колдонуучуларды жаңы ишке кошуп, учурдагы иштерди жаңырта алышат.

Иш түзүү

  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Администратор панелинен Cases менюсу өтмөктү чыкылдатыңыз.
  4. ЖАҢЫ баскычын басыңыз. Жаңы терезе пайда болот.

Керектүү талааларды толтуруп, САКТОО баскычын басыңыз.

img

Колдонуучуларды кошуу

Кааласаңыз, администраторлор бул баракчадагы ишке колдонуучуларды кошо алышат. Төмөн сыдырып, КОЛДОНУУЧУЛАРДЫ ТАПШЫРУУ баскычын басыңыз.

Колдонуучуну дайындоо модальында талааларды толтуруңуз. Колдонуучуга электрондук почта билдирүүсүн жөнөтүү үчүн кутучаны белгилеңиз. (милдеттүү эмес)

img

Ишти өзгөртүү

Аз сандагы иштерди жаңыртуу үчүн, Түзөтүү куралын колдонуп, ар бир жеке ишке өзгөртүүлөрдү киргизиңиз.

  • Эскертүү Көп сандагы иштерди бир отурушта жаңыртуу үчүн, Импорт куралын колдонуңуз
  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Администратор панелинен Cases менюсу өтмөктү чыкылдатыңыз.
  4. Ишти таап, Түзөтүүнү чыкылдатыңыз.
  5. Өзгөртүүлөрдү киргизип, SAVE баскычын басыңыз.
