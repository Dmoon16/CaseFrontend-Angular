Коомдук Каттоо Шилтемеси менен Автоматтык Иштерди кантип түзүү керек

Бул макалада администраторлорго бир катталуу шилтемесин жөнөтүү менен кантип кейс түзүү (жана ошол эле учурда колдонуучуну кошуу) көрсөтүлөт.

Шилтеменин жайгашкан жери

  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Администратор панелинен Орнотуулар меню өтмөктү чыкылдатыңыз.
  4. СИСТЕМАНЫ ОРНОТУУ субменюсунун өтмөгү тандалганын текшериңиз.

Орнотуулар конфигурацияланууда

Колдонуучунун КАТТАЛУУ рубрикасына ылдый сыдырыңыз.

Катталууга уруксат берүү талаасы Күйүк деп коюлганын текшериңиз. (талап кылынат) Бул функция иштетилгенде, Case Active уникалдуу шилтемени жаратат.

  1. Автоматтык ишти түзүү талаасын чыкылдатып, On дегенди тандаңыз. Case User Role талаасы көрсөтүлөт.
  2. Рол тандаңыз. (төмөндө кара)

Case колдонуучу ролдору

Колдонуучунун үч ролу бар.

  • кардарлар (сунушталат)
  • демейки
  • кызматкерлери

Шилтемени жөнөтүү

Шилтемени бир же бир нече адам менен бөлүшүңүз. Шилтемени чыкылдатып, катталууну аяктаган ар бир колдонуучу иштин бетинде жаңы ишти түзүү үчүн платформаны ишке киргизет жана учурда колдонуучуну автоматтык түрдө дайындайт.
