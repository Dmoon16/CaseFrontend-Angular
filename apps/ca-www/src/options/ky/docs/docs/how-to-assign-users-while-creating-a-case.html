Ишти түзүү учурунда колдонуучуларды кантип дайындоо керек

Бул макалада администраторлор жаңы ишти түзүүдө колдонуучуну кантип дайындоо керектигин көрсөтөт. Эгер колдонуучу буга чейин платформада катталбаган болсо, сиз жаңы колдонуучуну кошуп, жаңы ишти түзө аласыз.

Иш түзүү

  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Cases меню өтмөктү чыкылдатыңыз.
  4. ЖАҢЫ баскычын басыңыз. Жаңы терезе пайда болот.

  • Ишти түзүү үчүн талааларды толтуруңуз, андан кийин Дайындалган колдонуучулар рубрикасына ылдый сыдырыңыз.

Колдонуучуну дайындоо

Учурдагы колдонуучу

Учурдагы колдонуучуну жаңы ишке дайындоо үчүн бул кадамдарды аткарыңыз.

КОЛДОНУУЧУЛАРДЫ ТАПШЫРУУ баскычын чыкылдатып, талааларды толтуруңуз.

  1. Колдонуучу ачылуучу ылдый жагын басып, идентификаторду тандаңыз (мисалы, электрондук почта).
  2. Бош талаага маани киргизиңиз (мисалы, электрондук почта дареги).
  3. Эгер колдонуучунун жазуусу платформада бар болсо, талаа автоматтык түрдө ат сунуштайт.
  4. Туура колдонуучуну тандаңыз.
  5. Колдонуучунун ролун тандаңыз (кардарлар, демейки же кызматкерлер).
  6. Кошумча, колдонуучуга электрондук почта билдирүүсүн жөнөтүү үчүн кутучаны белгилеңиз.
  7. SAVE басыңыз.

Case Active колдонуучуну ишке кошот. Эми администраторлор терезени жабуу, жаңы колдонуучуну кошуу же колдонуучуну алып салуу үчүн Сактоо баскычын чыкылдата алышат.

Жаңы колдонуучуну кошуу

Эгер платформа учурдагы жазууну таба албаса, жаңы колдонуучуну кошуу үчүн баскычты басыңыз.

Талааларды толтуруп, андан кийин колдонуучуну платформага кошуу үчүн SAVE баскычын чыкылдатыңыз. Андан кийин, колдонуучунун ролун коюп, колдонуучуну ишке дайындоо үчүн SAVE баскычын кайра басыңыз. Акыр-аягы, ишти сактоо үчүн дагы бир жолу SAVE баскычын басыңыз.
