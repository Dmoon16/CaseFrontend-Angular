Formatting User Case Assignments Data with the Import tool

Infra catalogum notitiarum agrorum in fasciculis CSV. Optio importa singula diversam complexionem agrorum habet.

case_ID

  • Intrant causam User vel Tag ID.
  • Vade ad paginas > Edit > Tag ID OR Ire ad paginas > Edit > Case ID in url.

user_ID

  • Intrant usoris ID vel Tag ID.
  • Adi paginam User > Edit > Tag ID OR Ire ad paginam User > Edit > User ID in url.

case_role_order

  • Relinquere cellam blank.

case_role_ID

  • Intra: clientes, baculum, vel defaltam.

case_notify

  • 0 = no
  • 1 = Ita, notificationem inscriptionem usoris mitte.
