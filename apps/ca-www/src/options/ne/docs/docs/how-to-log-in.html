कसरी लग इन गर्ने

तपाइँको अवस्थित केस सक्रिय खातामा कसरी लग इन गर्ने जान्नुहोस्।

लग इन गर्दै

  1. लगइन पृष्ठमा जानुहोस्।
  2. लगइन बटन क्लिक गर्नुहोस्।
  3. आफ्नो इमेल प्रविष्ट गर्नुहोस्।
  4. अर्को क्लिक गर्नुहोस्।

खाता प्रमाणीकरण गर्दै

  1. नयाँ ब्राउजर ट्याब खोल्नुहोस् र आफ्नो इमेल खातामा लग इन गर्नुहोस्।
  2. छ अंकको कोड प्रतिलिपि गर्नुहोस्।
  3. केस सक्रिय ट्याबमा फर्कनुहोस्।
  4. क्षेत्रहरूमा कोड टाँस्नुहोस्। एउटा नयाँ सञ्झ्याल प्रदर्शित हुन्छ।

केस एक्टिभले एप्स पृष्ठ देखाउँछ।
