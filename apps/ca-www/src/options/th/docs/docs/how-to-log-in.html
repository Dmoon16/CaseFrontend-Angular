วิธีเข้าสู่ระบบ

เรียนรู้วิธีลงชื่อเข้าใช้บัญชี Case Active ที่คุณมีอยู่

เข้าสู่ระบบ

  1. ไปที่ หน้าเข้าสู่ระบบ
  2. คลิกปุ่มเข้าสู่ระบบ
  3. กรอกอีเมล์ของคุณ.
  4. คลิกถัดไป

การยืนยันบัญชี

  1. เปิดแท็บเบราว์เซอร์ใหม่และลงชื่อเข้าใช้บัญชีอีเมลของคุณ
  2. คัดลอกรหัสหกหลัก
  3. กลับไปที่แท็บ Case Active
  4. วางรหัสลงในช่อง หน้าต่างใหม่จะปรากฏขึ้น

Case Active แสดงหน้าแอพ

  • หมายเหตุ แพลตฟอร์มจะแสดงปุ่ม ADMIN ถัดจากปุ่ม APP สำหรับผู้ใช้ที่มีสิทธิ์ของผู้ดูแลระบบ
