ผู้ดูแลระบบจัดการกรณีและปัญหาอย่างไร

มีสี่วิธีในการจัดการกรณีและปัญหา คลิกลิงก์เพื่อเรียนรู้วิธีการ

วิธี พิมพ์ การกระทำ รายละเอียด
การใช้ UI UI เพิ่มและแก้ไข ปุ่มผู้ดูแลระบบ > เคส
สร้างกรณีอัตโนมัติโดยใช้ลิงก์ลงทะเบียนสาธารณะ การกระทำของผู้ใช้ เพิ่ม ปุ่มผู้ดูแลระบบ > การตั้งค่า > หน้าการตั้งค่าระบบ
เครื่องมือนำเข้า UI เพิ่มและแก้ไข ปุ่มผู้ดูแลระบบ > การผสานรวม > นำเข้า
การใช้ API (เร็วๆ นี้) เอพีไอ เพิ่มและแก้ไข ไม่มีข้อมูล
