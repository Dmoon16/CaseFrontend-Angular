วิธีสร้างผู้ใช้ด้วยลิงค์สมัคร

บทความนี้แสดงวิธีสร้างผู้ใช้โดยส่งลิงก์ลงทะเบียนให้ผู้ดูแลระบบ

การกำหนดค่าลิงค์สมัคร

  1. คลิก ลิงก์ส่งการสมัครให้พวกเขา ที่ด้านล่างของกล่องโต้ตอบเพิ่มผู้ใช้ หน้าใหม่ปรากฏขึ้น
  2. กำหนดค่าตัวเลือกการสมัคร (ดูด้านล่าง)

img

ตรวจสอบให้แน่ใจว่าช่อง อนุญาตการลงทะเบียน ถูกตั้งค่าเป็นเปิด (จำเป็น) เมื่อเปิดใช้งาน Case Active จะสร้างลิงค์เฉพาะ

ตรวจสอบตัวเลือกด้านล่าง จากนั้นคลิกไอคอนเพื่อคัดลอกลิงก์ คุณสามารถวางลิงก์บนเว็บไซต์ของคุณหรือส่งลิงก์ไปยังบุคคลอื่นทางอีเมลหรือช่องทางการสื่อสารใดก็ได้

img

การสร้างกรณี

คุณสามารถสร้างกรณีสำหรับผู้ใช้ใหม่แต่ละรายโดยอัตโนมัติและกำหนดบทบาทของผู้ใช้

  1. คลิกช่องสร้างกรณีและปัญหาอัตโนมัติแล้วเลือกเปิด ฟิลด์ Case User Role จะปรากฏขึ้น
  2. คลิกที่ Case User Role และเลือกตัวเลือก (ดูด้านล่าง)

บทบาทของผู้ใช้กรณี

มีบทบาทผู้ใช้สามกรณี

  • ลูกค้า
  • ค่าเริ่มต้น
  • พนักงาน

กำลังส่งลิงค์

แบ่งปันลิงก์กับบุคคลหนึ่งคนขึ้นไป แต่ละคนคลิกลิงก์และทำตามขั้นตอนการทำงานเพื่อเปิดใช้งานบัญชี Case Active ของตน เมื่อผู้ใช้ดำเนินการลงทะเบียนเวิร์กโฟลว์เสร็จสิ้น Case Active จะเพิ่มผู้ใช้ไปยังรายการผู้ใช้

img

ขั้นตอนถัดไป

Case Active จะส่งการแจ้งเตือนทางอีเมลให้กับผู้ใช้พร้อมลิงก์ลงทะเบียน ผู้ใช้คลิกลิงก์และทำตามขั้นตอนการทำงานสำหรับการลงชื่อสมัครใช้ใหม่

  • บทความนี้จะ อธิบายขั้นตอนการลงชื่อสมัครใช้สำหรับผู้ใช้ใหม่
