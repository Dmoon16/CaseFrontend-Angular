วิธีสร้างผู้ใช้ด้วยเครื่องมือนำเข้า

บทความนี้แสดงวิธีสร้างผู้ใช้ด้วยเครื่องมือนำเข้าสำหรับผู้ดูแลระบบ

การเปิดเครื่องมือนำเข้า

ผู้ดูแลระบบสามารถใช้เครื่องมือนำเข้าเพื่ออัปโหลดผู้ใช้ใหม่จำนวนมากและแก้ไขบันทึกผู้ใช้ที่มีอยู่

  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่มผู้ดูแลระบบ
  3. ในแผงการดูแลระบบ ให้คลิกแท็บการรวม จากนั้นคลิกนำเข้าในเมนูย่อย
  4. คลิกปุ่มใหม่ กล่องโต้ตอบเพิ่มการนำเข้าจะแสดงในแท็บเบราว์เซอร์ใหม่

การสร้างบันทึก

หลังจากกล่องโต้ตอบเพิ่มการนำเข้าเปิดขึ้น:

  1. คลิกช่อง Type แล้วเลือกตัวเลือก (ดูด้านล่าง) ลิงค์จะปรากฏขึ้น
  2. คลิกลิงค์ ดาวน์โหลดตัวอย่าง ไฟล์ CSV จะดาวน์โหลดไปยังอุปกรณ์ของคุณ

จากนั้นเปิดไฟล์และเพิ่มบันทึก คลิก ที่นี่ เพื่อเรียนรู้วิธีใช้รูปแบบข้อมูลในไฟล์ CSV

พิมพ์

เมื่อคุณคลิกฟิลด์ Type เมนูแบบเลื่อนลงจะปรากฏขึ้น เลือกหนึ่งในสองตัวเลือกสำหรับการนำเข้าจำนวนมาก

: อัศเจรีย์: หมายเหตุ เมนูแบบเลื่อนลงมีหลายตัวเลือก แต่มีเพียงสองตัวเลือกเท่านั้นที่เกี่ยวข้องกับเครื่องมือนำเข้า แต่ละตัวเลือกมีไฟล์ CSV ตัวอย่างที่แตกต่างกัน

เพิ่มผู้ใช้

  • เลือกตัวเลือกนี้สำหรับการอัปโหลดบันทึกผู้ใช้จำนวนมากแบบง่ายๆ
  • คลิกลิงก์ดาวน์โหลดตัวอย่างเพื่อเปิดไฟล์ CSV ที่มีส่วนหัวของคอลัมน์สำหรับประเภทการนำเข้าเพิ่มผู้ใช้

แก้ไขผู้ใช้

  • เลือกตัวเลือกนี้เพื่ออัปเดตข้อมูลสำหรับผู้ใช้ปัจจุบัน
  • คลิกลิงก์ดาวน์โหลดตัวอย่างเพื่อเปิดไฟล์ CSV ที่มีส่วนหัวของคอลัมน์สำหรับประเภทนำเข้าแก้ไขผู้ใช้

กำลังอัปโหลดไฟล์

คุณสามารถอัปโหลดไฟล์ได้ทันทีหรือกลับไปที่กล่องโต้ตอบเพิ่มการนำเข้าในภายหลังและดำเนินการตามขั้นตอนเหล่านี้

  1. เปิดกล่องโต้ตอบเพิ่มการนำเข้า
  2. ป้อนชื่อเฉพาะสำหรับไฟล์ CSV
  3. คลิกไอคอนเครื่องหมายบวก (+) เพื่ออัปโหลดไฟล์จากอุปกรณ์ของคุณ
  4. คลิกช่อง Type แล้วเลือกตัวเลือก (ดูด้านบน)
  5. คลิกบันทึก

กำลังประมวลผล

ขึ้นอยู่กับจำนวนของบันทึกในไฟล์และโหลดของระบบ แพลตฟอร์มอาจใช้เวลาหลายนาทีในการประมวลผลการนำเข้า

เมื่อการนำเข้าเสร็จสิ้น แพลตฟอร์มจะปรับบันทึกผู้ใช้ (เช่น เพิ่ม แก้ไข หรือปิดใช้งาน) บนแดชบอร์ด ผู้ใช้ ของผู้ดูแลระบบ

ความล้มเหลว

หากการดำเนินการนำเข้าล้มเหลว แพลตฟอร์มจะส่งคืนข้อความล้มเหลว ในกรณีส่วนใหญ่ สาเหตุของความล้มเหลวคือการจัดรูปแบบข้อมูล

หากต้องการดูสาเหตุของความล้มเหลว:

  1. คลิก ดู ใต้หัวข้อการดำเนินการ
  2. ในหน้าถัดไป ให้เลื่อนเมาส์ไปที่ไอคอนล้มเหลว คำแนะนำเครื่องมือแสดงข้อผิดพลาด
  3. ไปที่ไฟล์ CVS แล้วปรับและแก้ไขข้อผิดพลาด
  4. อัปโหลดไฟล์อีกครั้ง
