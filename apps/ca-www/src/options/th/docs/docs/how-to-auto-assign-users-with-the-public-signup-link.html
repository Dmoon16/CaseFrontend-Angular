วิธีกำหนดผู้ใช้โดยอัตโนมัติด้วยลิงก์ลงทะเบียนสาธารณะ

ผู้ดูแลระบบเรียนรู้วิธีกำหนดการตั้งค่าระบบเพื่อให้ผู้ใช้ใหม่ถูกกำหนดให้กับกรณีและปัญหาที่สร้างขึ้นโดยอัตโนมัติใหม่โดยอัตโนมัติ

กำลังเปิดหน้าการตั้งค่า

ขั้นตอนแรกคือการเปิดหน้าการตั้งค่าระบบ

  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่ม ADMIN
  3. คลิกที่แท็บ ระบบ จากนั้น ตั้งค่าระบบ บนเมนูย่อย
  4. เลื่อนลงไปที่หัวข้อ USER SIGNUP

การกำหนดการตั้งค่าระบบ

  1. ตั้งค่าเมนูแบบเลื่อนลง Allow Signup and Auto Create Case เป็น On
  2. คลิกเมนูบทบาทและเลือกตัวเลือก (แนะนำให้ใช้ค่าเริ่มต้น)
  3. คัดลอกและแชร์ลิงก์ลงทะเบียนผู้ใช้สาธารณะ

กำลังส่งลิงค์

ผู้ใช้แต่ละรายที่คลิกลิงก์และลงทะเบียนเสร็จสมบูรณ์จะทริกเกอร์แพลตฟอร์มเพื่อสร้างกรณีและปัญหาใหม่ในหน้ากรณีและกำหนดผู้ใช้ในกรณีนั้นโดยอัตโนมัติ
