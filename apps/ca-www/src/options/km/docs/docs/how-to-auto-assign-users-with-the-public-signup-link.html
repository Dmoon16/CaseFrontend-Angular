របៀបកំណត់អ្នកប្រើប្រាស់ដោយស្វ័យប្រវត្តិជាមួយនឹងតំណភ្ជាប់ចុះឈ្មោះសាធារណៈ

អ្នកគ្រប់គ្រងរៀនពីរបៀបកំណត់រចនាសម្ព័ន្ធការកំណត់ប្រព័ន្ធ ដូច្នេះអ្នកប្រើប្រាស់ថ្មីត្រូវបានចាត់តាំងដោយស្វ័យប្រវត្តិចំពោះករណីដែលបានបង្កើតដោយស្វ័យប្រវត្តិថ្មី។

ការបើកទំព័រដំឡើង

ជំហានដំបូងគឺត្រូវបើកទំព័រដំឡើងប្រព័ន្ធ។

  1. ចូលទៅគណនីអ្នកគ្រប់គ្រង។
  2. ស្វែងរកទីតាំងកម្មវិធីនៅលើទំព័រកម្មវិធី ហើយចុចប៊ូតុង ADMIN ។
  3. ចុចផ្ទាំង Systems ហើយបន្ទាប់មក SYSTEM SETUP នៅលើម៉ឺនុយរង។
  4. រំកិលចុះក្រោមទៅផ្នែក USER SIGNUP ។

កំណត់រចនាសម្ព័ន្ធការកំណត់ប្រព័ន្ធ

  1. កំណត់ម៉ឺនុយទម្លាក់ចុះអនុញ្ញាតការចុះឈ្មោះ និងបង្កើតករណីដោយស្វ័យប្រវត្តិទៅបើក។
  2. ចុចម៉ឺនុយតួនាទី ហើយជ្រើសរើសជម្រើសមួយ (លំនាំដើមត្រូវបានណែនាំ)។
  3. ចម្លង និងចែករំលែកតំណចុះឈ្មោះអ្នកប្រើប្រាស់សាធារណៈ។

កំពុងផ្ញើតំណ

អ្នកប្រើប្រាស់នីមួយៗដែលចុចលើតំណ និងបញ្ចប់ការចុះឈ្មោះ បង្កឱ្យវេទិកាបង្កើតករណីថ្មីនៅលើទំព័រករណី ហើយប្រគល់ឱ្យអ្នកប្រើប្រាស់ដោយស្វ័យប្រវត្តិនៅក្នុងករណីនោះ។
