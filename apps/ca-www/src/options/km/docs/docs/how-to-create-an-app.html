របៀបបង្កើតកម្មវិធី

អត្ថបទនេះបង្ហាញអ្នកប្រើប្រាស់ថ្មីអំពីរបៀបបង្កើតគណនីសាកល្បងឥតគិតថ្លៃ។ Case Active ផ្តល់សិទ្ធិជាអ្នកគ្រប់គ្រងអ្នកប្រើប្រាស់ថ្មីដោយស្វ័យប្រវត្តិសម្រាប់គណនីនេះ។

ការបង្កើតគណនីមួយ។

  1. ចូលទៅកាន់ ទំព័រចូល ។
  2. ចុចប៊ូតុងចាប់ផ្តើម។
  3. បញ្ចូលអាសយដ្ឋានអ៊ីមែល។
  4. ចុចបន្ទាប់។

ការផ្ទៀងផ្ទាត់គណនីមួយ។

  1. បើកផ្ទាំងកម្មវិធីរុករកថ្មី ហើយចូលគណនីអ៊ីមែលរបស់អ្នក។
  2. ចម្លងលេខកូដប្រាំមួយខ្ទង់។
  3. ត្រឡប់ទៅផ្ទាំង Case Active វិញ។
  4. បិទភ្ជាប់កូដទៅក្នុងវាល។ បង្អួចថ្មីបង្ហាញ។

នៅក្នុងបង្អួចថ្មី៖

  1. បំពេញក្នុងវាលដែលត្រូវការ។
  2. អានលក្ខខណ្ឌនៃសេវាកម្ម ហើយធីកប្រអប់ដើម្បីយល់ព្រម។
  3. ចុច បង្កើតគណនីឥតគិតថ្លៃ។

img

កំពុងបញ្ចប់ការដំឡើងកម្មវិធី

លំហូរការងារបង្ហាញនៅទំព័របន្ទាប់។ បំពេញជំហានដើម្បីបញ្ចប់ការដំឡើងកម្មវិធីរបស់អ្នក។ អ្នកតែងតែអាចត្រឡប់ទៅទំព័រទាំងនេះ ដើម្បីកែសម្រួល ឬធ្វើបច្ចុប្បន្នភាពព័ត៌មាន ឬស្ថានភាពគណនីរបស់អ្នក។

នៅលើទំព័រអំពីកម្មវិធី សូមបំពេញឈ្មោះក្រុមហ៊ុន និងវាលគេហទំព័រ (ទាមទារ) ហើយបន្ទាប់មកចុចបន្ទាប់។

img

នៅលើទំព័រព័ត៌មានអំពីការចេញវិក្កយបត្រ ចុចលើវាល ផែនការទូទាត់ ហើយពិនិត្យមើលជម្រើស។ ការកំណត់លំនាំដើមគឺឥតគិតថ្លៃ។ ចុច SUBMIT ។

បន្ទាប់ពីអ្នកដាក់ស្នើទម្រង់ Case Active បង្ហាញទំព័រកម្មវិធី។ សម្រាប់កម្មវិធីនីមួយៗដែលអ្នកបង្កើត ឬជាកម្មសិទ្ធិ ទំព័របង្ហាញ៖

  1. ឈ្មោះកម្មវិធី
  2. ប្រភេទរបស់អ្នក (តួនាទីអ្នកប្រើប្រាស់)
  3. ស្ថានភាព

img

ទំព័រកម្មវិធីគឺជាចំណុចចាប់ផ្តើមសម្រាប់មុខងារអ្នកគ្រប់គ្រងជាច្រើន៖

  1. ចុចប៊ូតុង APP ដើម្បីទំនាក់ទំនង បំពេញឯកសារ ការជជែកជាវីដេអូ (និងច្រើនទៀត) លើបញ្ហាទាក់ទងនឹងករណី។
  2. ចុចប៊ូតុង ADMIN ដើម្បីបើកផ្ទាំងគ្រប់គ្រង។ នេះគឺជាកន្លែងដែលអ្នកគ្រប់គ្រងគ្រប់គ្រងអ្នកប្រើប្រាស់ ករណី ការរួមបញ្ចូល និងច្រើនទៀត។
  3. ចុចពងក្រពើ ដើម្បីគ្រប់គ្រងការចេញវិក្កយបត្រគណនី និងការកំណត់ការជូនដំណឹង។
  • Note Case Active បង្ហាញប៊ូតុង ADMIN សម្រាប់អ្នកប្រើប្រាស់អ្នកគ្រប់គ្រង។ អ្នកប្រើប្រាស់ដែលមិនមានសិទ្ធិគ្រប់គ្រងមិនឃើញប៊ូតុង ADMIN (ដូចបង្ហាញក្នុងអេក្រង់ខាងក្រោម)។
