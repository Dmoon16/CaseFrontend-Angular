Nigute ushobora gukora abakoresha hamwe nibikoresho byinjira

Iyi ngingo irerekana abadmin uburyo bwo gukora abakoresha nigikoresho cyo Kuzana.

Gufungura ibikoresho byinjira

Abadamu barashobora gukoresha igikoresho cyo Kuzana ibicuruzwa byinshi kugirango bashireho abakoresha bashya kandi bahindure inyandiko zabakoresha.

  1. Injira kuri konte ya admin.
  2. Shakisha porogaramu kurupapuro rwa porogaramu hanyuma ukande buto ya ADMIN.
  3. Kuruhande rwa Admin, kanda ahanditse Integrated hanyuma IMPORTS kuri submenu.
  4. Kanda buto NSHYA. Ongeramo Ibitumizwa mu kiganiro agasanduku kerekana muri tab nshya ya mushakisha.

Gukora inyandiko

Nyuma yo Kongera Ibitumizwa mu kiganiro agasanduku gafungura:

  1. Kanda Ubwoko hanyuma uhitemo amahitamo (reba hano hepfo). Ihuza ryerekana.
  2. Kanda ahanditse Gukuramo icyitegererezo . Idosiye ya CSV ikuramo ibikoresho byawe.

Ibikurikira, fungura dosiye hanyuma wongere inyandiko. Kanda hano wige uburyo bwo gukoresha amakuru yimiterere muri dosiye ya CSV.

Andika

Iyo ukanze Ubwoko bwumurima, ibimanuka byamanutse byerekana. Hitamo bumwe muburyo bubiri bwo gutumiza hanze.

: gutangaza: ICYITONDERWA Ibimanuka bimanuka bifite amahitamo menshi, ariko bibiri gusa bifitanye isano nigikoresho cyo Kuzana. Buri cyiciro gifite dosiye itandukanye ya CSV.

Ongeraho abakoresha

  • Hitamo ubu buryo kubwinshi bworoshye bwo kohereza inyandiko zabakoresha.
  • Kanda ahanditse Gukuramo icyitegererezo kugirango ufungure dosiye ya CSV ifite imitwe yinkingi yo kongeramo abakoresha Kuzana ubwoko.

Hindura abakoresha

  • Hitamo ubu buryo bwo kuvugurura amakuru kubakoresha ubu.
  • Kanda ahanditse Gukuramo icyitegererezo kugirango ufungure dosiye ya CSV ifite imitwe yinkingi kubihindura abakoresha Ubwoko bwinjiza.

Gukuramo dosiye

Urashobora kohereza dosiye ako kanya cyangwa ugasubira kuri Ongera Ibitumizwa mu kiganiro nyuma hanyuma ukarangiza izi ntambwe.

  1. Fungura Ongera Ibitumizwa mu kiganiro.
  2. Injiza izina ryihariye rya dosiye ya CSV.
  3. Kanda ahanditse (+) kugirango ushire dosiye mubikoresho byawe.
  4. Kanda Ubwoko hanyuma uhitemo amahitamo. (reba hejuru)
  5. Kanda SAVE.

Gutunganya

Ukurikije umubare wibyanditswe muri dosiye hamwe nuburemere bwa sisitemu, urubuga rushobora gufata iminota mike yo gutunganya ibicuruzwa.

Iyo kwinjiza birangiye, urubuga ruhindura inyandiko zabakoresha (nukuvuga, guhindura, cyangwa guhagarika) kumwanya wabakoresha ba admin .

Kunanirwa

Niba ibikorwa byo gutumiza mu mahanga binaniwe, urubuga rusubiza ubutumwa bwatsinzwe. Mu bihe byinshi, igitera kunanirwa ni uguhindura amakuru.

Kugirango ubone icyateye kunanirwa:

  1. Kanda Reba munsi yumutwe wigikorwa.
  2. Kurupapuro rukurikira, uzamure imbeba kumashusho Yatsinzwe. Igikoresho cyerekana ikosa.
  3. Jya kuri dosiye ya CVS hanyuma uhindure kandi ukosore amakosa.
  4. Ongera ushyire dosiye.
