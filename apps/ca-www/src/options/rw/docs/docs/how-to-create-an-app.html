Nigute ushobora gukora porogaramu

Iyi ngingo yerekana abakoresha bashya uburyo bwo gukora konti yubusa. Case Active ihita itanga uburenganzira bushya bwumukoresha wa admin kuriyi konti.

Gukora konti

  1. Jya kurupapuro rwinjira .
  2. Kanda buto yo gutangira.
  3. Injiza aderesi imeri.
  4. Kanda ahakurikira.

img

Kugenzura konti

  1. Fungura amashusho mashya hanyuma winjire kuri konte yawe imeri.
  2. Gukoporora kode y'imibare itandatu.
  3. Garuka kumurongo wibikorwa.
  4. Shyira kode mumirima. Idirishya rishya ryerekana.

Mu idirishya rishya:

  1. Uzuza imirima isabwa.
  2. Soma ibikubiye muri serivisi hanyuma urebe agasanduku kugirango wemere.
  3. Kanda SHAKA KONTI YUBUNTU.

img

Kurangiza gushiraho porogaramu

Urupapuro rwakazi rwerekana kurupapuro rukurikira. Uzuza intambwe zo kurangiza gushiraho porogaramu yawe. Urashobora buri gihe gusubira kuriyi page kugirango uhindure cyangwa uvugurure amakuru cyangwa konte yawe.

Kurupapuro rwa About App, uzuza izina ryisosiyete nimirima yurubuga (bisabwa) hanyuma ukande ahakurikira.

Kurupapuro rwamakuru yo kwishura, kanda ahanditse Gahunda yo Kwishura hanyuma usuzume amahitamo. Igenamiterere risanzwe ni Ubuntu. Kanda SUBMIT.

img

Nyuma yo gutanga ifishi, Case Active yerekana urupapuro rwa porogaramu. Kuri buri porogaramu ukora cyangwa urimo, urupapuro rwerekana:

  1. Izina rya porogaramu
  2. Ubwoko bwawe (uruhare rwabakoresha)
  3. Imiterere

Urupapuro rwa porogaramu ni ingingo yo gutangiza ibikorwa byinshi bya admin:

  1. Kanda buto ya APP kugirango ushyikirane, wuzuze doc, kuganira kuri videwo (nibindi byinshi) kubibazo bijyanye nurubanza.
  2. Kanda buto ya ADMIN kugirango ufungure akanama ka admin. Aha niho abayobozi bayobora abakoresha, imanza, kwishyira hamwe, nibindi byinshi.
  3. Kanda kuri ellipsis kugirango ucunge konti yo kwishyuza no kumenyesha.
  • Icyitonderwa Case Active yerekana buto ya ADMIN kubakoresha admin. Abakoresha badafite uburenganzira bwa admin ntibabona buto ya ADMIN (nkuko bigaragara kuri ecran hepfo).
