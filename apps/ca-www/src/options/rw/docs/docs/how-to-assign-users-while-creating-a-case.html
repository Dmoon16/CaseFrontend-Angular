Nigute washyiraho abakoresha mugihe utegura urubanza

Iyi ngingo irerekana abadmin uburyo bwo guha umukoresha mugihe urema urubanza rushya. Niba umukoresha atarangije kwiyandikisha kurubuga, urashobora gukora kongeramo umukoresha mushya mugihe urema urubanza rushya.

Gukora urubanza

  1. Injira kuri konte ya admin.
  2. Shakisha porogaramu kurupapuro rwa porogaramu hanyuma ukande buto ya ADMIN.
  3. Kanda ahanditse Imanza.
  4. Kanda buto NSHYA. Idirishya rishya ryerekana.

img

  • Uzuza mu murima kugirango ukore urubanza hanyuma umanure hasi kubakoresha bashinzwe umutwe.

Kugena umukoresha

Umukoresha

Kurikiza izi ntambwe kugirango ugenera umukoresha kurubu urubanza rushya.

Kanda buto ya ASSIGN USERS hanyuma wuzuze mumirima.

  1. Kanda Umukoresha wamanutse hanyuma uhitemo ikiranga (urugero imeri).
  2. Injiza agaciro mumwanya muto (urugero aderesi imeri).
  3. Niba umukoresha yanditseho murubuga, umurima auto-yerekana izina.
  4. Hitamo umukoresha neza.
  5. Hitamo uruhare rwumukoresha (abakiriya, isanzwe, cyangwa abakozi).
  6. Bitabaye ibyo, reba agasanduku kohereza umukoresha imenyesha rya imeri.
  7. Kanda SAVE.

Case Active yongerera umukoresha murubanza. Noneho abadamu barashobora gukanda Kubika kugirango bafunge idirishya, cyangwa bongereho umukoresha mushya, cyangwa bakureho umukoresha.

Ongeraho umukoresha mushya

Niba urubuga rudashobora kubona inyandiko iriho, kanda buto kugirango wongere umukoresha mushya.

Uzuza imirima hanyuma ukande SAVE kugirango wongere umukoresha kurubuga. Ibikurikira, shiraho umukoresha uruhare hanyuma ukande SAVE kugirango uhe umukoresha murubanza. Hanyuma, kanda SAVE ikindi gihe kugirango ubike urubanza.
