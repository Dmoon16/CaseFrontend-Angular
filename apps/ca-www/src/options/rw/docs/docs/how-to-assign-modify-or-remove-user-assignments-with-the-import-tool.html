Nigute Kugena, Guhindura cyangwa Gukuraho Umukoro Umukoro hamwe nigikoresho cyo Kuzana

Iyi ngingo irerekana abadmin uburyo bwo gucunga abakoresha imanza hamwe nigikoresho cyo Kuzana.

Abayobozi barashobora:

  • shyira abakoresha murubanza
  • hindura abakoresha basanzwe bashinzwe urubanza
  • kuvanaho abakoresha bashinzwe urubanza

Gufungura ibikoresho byinjira

  1. Injira kuri konte ya admin.
  2. Shakisha porogaramu kurupapuro rwa porogaramu hanyuma ukande buto ya ADMIN.
  3. Kanda ahanditse Integrated hanyuma IMPORTS kuri submenu.
  4. Kanda buto NSHYA. Ongeramo Ibitumizwa mu kiganiro agasanduku kerekana.

Gukora inyandiko

Kuramo dosiye imwe cyangwa nyinshi za CSV. Koresha iyi dosiye kugirango wongere, uhindure, cyangwa ukureho abakoresha imanza. Nyuma yo Kongera Ibitumizwa mu kiganiro agasanduku gafungura:

  1. Kanda ahanditse Ubwoko.
  2. Kuzenguruka unyuze kuri menu hanyuma uhitemo amahitamo (reba hano hepfo). Ihuza ryerekana.
  3. Kanda ahanditse Gukuramo icyitegererezo. Idosiye ya CSV ikuramo ibikoresho byawe.

img

Fungura dosiye wongere inyandiko. Kanda hano wige uburyo bwo gukoresha amakuru yimiterere muri dosiye ya CSV.

Amahitamo ya dosiye

Shinga abakoresha kubibazo * Hitamo ubu buryo kugirango wongere abakoresha murubanza. * Kureka urubanza_role_umwanya muto. * Jya kurupapuro rwabakoresha hanyuma wandukure Tag ID kuri uyu muntu. Niba ikirangantego kiranga ubusa, andika ID idasanzwe, kanda Kubika, hanyuma wandukure ID kuri dosiye ya CSV.

Hindura abakoresha bashinzwe kubibazo * Hitamo ubu buryo kugirango uvugurure amakuru kubakoresha ubu bashinzwe urubanza. * Koresha indangamuntu y'urubanza kubibazo_ID inkingi.

Kuraho abakoresha bashinzwe kubibazo * Hitamo ubu buryo kugirango ukureho abakoresha bashinzwe urubanza.

Gukuramo dosiye

Urashobora kohereza dosiye ako kanya cyangwa ugasubira kuri Ongera Ibitumizwa mu kiganiro nyuma hanyuma ukarangiza izi ntambwe.

  1. Fungura Ongera Ibitumizwa mu kiganiro.
  2. Injiza izina ryihariye rya dosiye ya CSV.
  3. Kanda ahanditse (+) kugirango ushire dosiye mubikoresho byawe.
  4. Kanda Ubwoko hanyuma uhitemo amahitamo. (reba hejuru)
  5. Kanda SAVE.

Gutunganya

Ukurikije umubare wibyanditswe muri dosiye hamwe nuburemere bwa sisitemu, urubuga rushobora gufata iminota mike yo gutunganya ibicuruzwa.

img

Iyo kwinjiza birangiye, urubuga ruhindura inyandiko zabakoresha (nukuvuga, guhindura, cyangwa gukuraho) kurupapuro rwimanza.

Kunanirwa

Niba ibikorwa byo gutumiza mu mahanga binaniwe, urubuga rusubiza ubutumwa bwatsinzwe. Mu bihe byinshi, igitera kunanirwa ni uguhindura amakuru.

Kugirango ubone icyateye kunanirwa:

  1. Kanda Reba munsi yumutwe wigikorwa.
  2. Kurupapuro rukurikira, uzamure imbeba kumashusho Yatsinzwe. Igikoresho cyerekana ikosa.
  3. Jya kuri dosiye ya CVS hanyuma uhindure kandi ukosore amakosa.
  4. Ongera ushyire dosiye.
