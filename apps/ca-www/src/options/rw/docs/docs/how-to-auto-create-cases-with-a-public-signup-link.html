Nigute ushobora gukora Auto gukora imanza hamwe na rusange yo kwiyandikisha

Iyi ngingo yerekana abadmin uburyo bwo gukora urubanza (no kongeramo umukoresha icyarimwe) wohereza umurongo umwe wo kwiyandikisha.

Kubona ihuriro

  1. Injira kuri konte ya admin.
  2. Shakisha porogaramu kurupapuro rwa porogaramu hanyuma ukande buto ya ADMIN.
  3. Kuruhande rwa Admin, kanda ahanditse Igenamiterere.
  4. Menya neza ko SYSTEM SETUP submenu tab yatoranijwe.

img

Kugena igenamiterere

Kanda hasi kumutwe wa USER SIGNUP.

Menya neza ko Emera Kwiyandikisha Umwanya washyizweho Kuri. (bisabwa) Iyo iyi mikorere ishoboye, Case Active itanga umurongo wihariye.

  1. Kanda Auto Kurema Urubanza hanyuma uhitemo Kuri. Urubanza Umukoresha Uruhare rwerekana.
  2. Hitamo uruhare. (reba hano hepfo)

Uruhare rwabakoresha

Hariho ibintu bitatu byabakoresha inshingano.

  • abakiriya (basabwe)
  • Mburabuzi
  • abakozi

Kohereza umurongo

Sangira ihuriro numuntu umwe cyangwa benshi. Buri mukoresha ukanze umurongo hanyuma arangiza kwiyandikisha atera urubuga rwo kubyara urubanza rushya kurupapuro rwimanza hanyuma ugaha umukoresha murubanza.
