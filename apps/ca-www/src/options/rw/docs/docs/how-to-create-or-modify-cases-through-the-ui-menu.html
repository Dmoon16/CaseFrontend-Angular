Nigute ushobora gukora cyangwa guhindura imanza ukoresheje UI menu

Iyi ngingo irerekana abadmin uburyo bwo gukora urubanza hamwe nurubuga rwabakoresha. Abadamu barashobora kandi kongera abakoresha murubanza rushya no kuvugurura imanza zisanzwe.

Gukora urubanza

  1. Injira kuri konte ya admin.
  2. Shakisha porogaramu kurupapuro rwa porogaramu hanyuma ukande buto ya ADMIN.
  3. Kuruhande rwa Admin, kanda ahanditse Imanza.
  4. Kanda buto NSHYA. Idirishya rishya ryerekana.

Uzuza imirima isabwa hanyuma ukande SAVE.

Ongeraho abakoresha

Bitabaye ibyo, abadamu barashobora kongeramo abakoresha mururu rupapuro. Kanda hasi hanyuma ukande buto ya ASSIGN USERS .

Kugenera Umukoresha modal, uzuza imirima. Reba agasanduku kohereza umukoresha imenyesha rya imeri. (bidashoboka)

img

Guhindura urubanza

Kuvugurura umubare muto wimanza, koresha igikoresho cyo Guhindura hanyuma uhindure kuri buri rubanza.

  • Icyitonderwa Kugirango uhindure umubare munini wimanza wicaye, tekereza gukoresha igikoresho cyo Kuzana
  1. Injira kuri konte ya admin.
  2. Shakisha porogaramu kurupapuro rwa porogaramu hanyuma ukande buto ya ADMIN.
  3. Kuruhande rwa Admin, kanda ahanditse Imanza.
  4. Shakisha urubanza hanyuma ukande ahanditse.
  5. Kora impinduka hanyuma ukande SAVE.
