Nigute ushobora kugenera abakoresha hamwe na rusange yo kwiyandikisha

Abadmin biga uburyo bwo gushiraho sisitemu igenamigambi kugirango abakoresha bashya bahite bashirwa murubanza rushya rwimodoka.

Gufungura urupapuro

Intambwe yambere ni ugukingura page ya sisitemu.

  1. Injira kuri konte ya admin.
  2. Shakisha porogaramu kurupapuro rwa porogaramu hanyuma ukande buto ya ADMIN.
  3. Kanda ahanditse Sisitemu hanyuma SYSTEM SETUP kuri submenu.
  4. Kanda hasi kumutwe wa USER SIGNUP.

Kugena igenamiterere rya sisitemu

  1. Shiraho Emera Kwiyandikisha na Auto Kurema Urubanza rumanuka kuri Kuri.
  2. Kanda kurutonde rwibikorwa hanyuma uhitemo amahitamo (birasabwa).
  3. Gukoporora no gusangira ihuza rusange ryabakoresha.

Kohereza umurongo

Buri mukoresha ukanze ihuriro akuzuza kwiyandikisha atera urubuga rwo kubyara urubanza rushya kurupapuro rwimanza hanyuma ugaha umukoresha icyo gihe.
