Nigute ushobora gukora cyangwa guhindura imanza hamwe nibikoresho byinjira

Iyi ngingo yerekana abadmin uburyo bwo gukora cyangwa guhindura imanza zisanzwe hamwe nigikoresho cyo Kuzana.

Gufungura ibikoresho byinjira

  1. Injira kuri konte ya admin.
  2. Shakisha porogaramu kurupapuro rwa porogaramu hanyuma ukande buto ya ADMIN.
  3. Kuruhande rwa Admin, kanda ahanditse Integrated hanyuma IMPORTS kuri submenu.
  4. Kanda buto NSHYA. Ongeramo Ibitumizwa mu kiganiro agasanduku kerekana muri tab nshya ya mushakisha.

img

Gukora imanza

Nyuma yo Kongera Ibitumizwa mu kiganiro agasanduku gafungura:

  1. Kanda Ubwoko hanyuma uhitemo amahitamo (reba hano hepfo). Ihuza ryerekana.
  2. Kanda ahanditse Gukuramo icyitegererezo. Idosiye ya CSV ikuramo ibikoresho byawe.

Ibikurikira, fungura dosiye hanyuma wongere imanza.

Andika

Iyo ukanze Ubwoko bwumurima, ibimanuka byamanutse byerekana. Hitamo bumwe muburyo bubiri kubintu byinshi bitumizwa mu mahanga.

  • Icyitonderwa Ibimanuka byamanutse bifite amahitamo menshi, ariko bibiri gusa bifitanye isano no gucunga urubanza. Buri cyiciro gifite dosiye itandukanye ya CSV.

Ongeraho imanza

  • Hitamo ubu buryo kubworoshye bwo kohereza imanza.
  • Kanda ahanditse Gukuramo icyitegererezo kugirango ufungure dosiye ya CSV ifite imitwe yinkingi yo Kurema imanza zitumizwa.

Hindura imanza

  • Hitamo ubu buryo bwo kuvugurura imanza zubu.
  • Kanda ahanditse Gukuramo icyitegererezo kugirango ufungure dosiye ya CSV ifite imitwe yinkingi yo guhindura imanza zitumizwa.

Gukuramo dosiye

Urashobora kohereza dosiye ako kanya cyangwa ugasubira kuri Ongera Ibitumizwa mu kiganiro nyuma hanyuma ukarangiza izi ntambwe.

  1. Fungura Ongera Ibitumizwa mu kiganiro.
  2. Injiza izina ryihariye rya dosiye ya CSV.
  3. Kanda ahanditse (+) kugirango ushire dosiye mubikoresho byawe.
  4. Kanda Ubwoko hanyuma uhitemo amahitamo. (reba hejuru)
  5. Kanda SAVE.

Imiterere

Nibisabwa bikenewe nibisabwa byambere. Kanda hano kugirango usuzume ibisobanuro birambuye.

tag_id (cyangwa urubanza_ID) ikiranga kidasanzwe (koresha inyuguti zinyuguti, inyuguti zemewe)

urubanza_status 0 = ifunze 1 = ikora

itariki_yakinguye ukwezi / umunsi / umwaka mm / dd / yyyy

kubyerekeranye nibisobanuro byihariye byurubanza (koresha inyuguti zinyuguti, inyuguti zemewe)

Gutunganya

Ukurikije umubare wibyanditswe muri dosiye hamwe nuburemere bwa sisitemu, urubuga rushobora gufata iminota mike yo gutunganya ibicuruzwa.

Iyo itumizwa ryarangiye, urubuga ruhindura inyandiko zurubanza (ni ukuvuga wongeyeho cyangwa uhindura) kurubuga rwa admin.

Kunanirwa

Niba ibikorwa byo gutumiza mu mahanga binaniwe, urubuga rusubiza ubutumwa bwatsinzwe. Mu bihe byinshi, igitera kunanirwa ni uguhindura amakuru.

Kugirango ubone icyateye kunanirwa:

  1. Kanda Reba munsi yumutwe wibikorwa.
  2. Kurupapuro rukurikira, uzamure imbeba kumashusho Yatsinzwe. Igikoresho cyerekana ikosa.
  3. Jya kuri dosiye ya CVS hanyuma uhindure kandi ukosore amakosa.
  4. Ongera ushyire dosiye.
