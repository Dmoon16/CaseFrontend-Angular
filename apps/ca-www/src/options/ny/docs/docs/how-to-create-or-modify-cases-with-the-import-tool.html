Momwe Mungapangire Kapena Kusintha Milandu ndi Chida Chothandizira

Nkhaniyi ikuwonetsa ma admins momwe angapangire kapena kusintha milandu yomwe ilipo ndi chida cha Import.

Kutsegula chida cha Import

  1. Lowani ku akaunti ya admin.
  2. Pezani pulogalamu patsamba la Mapulogalamu ndikudina batani la ADMIN.
  3. Pa gulu la Admin, dinani tabu Yophatikiza ndiyeno IMPORTS pa submenu.
  4. Dinani batani la CHATSOPANO. Bokosi la dialog la Add Imports likuwonekera pa tabu yatsopano ya msakatuli.

img

Kupanga milandu

Pambuyo pa bokosi la Add Imports litatsegulidwa:

  1. Dinani Type field ndikusankha njira (onani pansipa). Ulalo ukuwonekera.
  2. Dinani ulalo wa Tsitsani Zitsanzo. Fayilo ya CSV imatsitsidwa ku chipangizo chanu.

Kenako, tsegulani fayilo ndikuwonjezera milandu.

Mtundu

Mukadina Type field, menyu yotsitsa imawonekera. Sankhani imodzi mwa njira ziwiri zogulitsira katundu wambiri.

  • Zindikirani Menyu yotsitsa ili ndi zosankha zambiri, koma ziwiri zokha ndizogwirizana ndi kasamalidwe kamilandu. Njira iliyonse ili ndi fayilo yosiyana ya CSV.

Onjezani milandu

  • Sankhani njira iyi kuti mukweze milandu yambiri.
  • Dinani ulalo wa Tsitsani Zitsanzo kuti mutsegule fayilo ya CSV yokhala ndi mitu yagawo la mtundu wa Pangani milandu.

Sinthani milandu

  • Sankhani izi kuti musinthe zochitika zomwe zikuchitika.
  • Dinani ulalo wa Chitsanzo Chotsitsa kuti mutsegule fayilo ya CSV yokhala ndi mitu yazakusintha kwamtundu wa Kusintha milandu.

Kukweza fayilo

Mutha kukweza fayilo nthawi yomweyo kapena kubwerera ku bokosi la Add Imports pambuyo pake ndikumaliza izi.

  1. Tsegulani bokosi la Add Imports dialog.
  2. Lowetsani dzina lapadera la fayilo ya CSV.
  3. Dinani chizindikiro chowonjezera (+) kuti mukweze fayilo kuchokera pa chipangizo chanu.
  4. Dinani Type kumunda ndikusankha njira. (onani pamwambapa)
  5. Dinani KUSUNGA.

Kukonza

Izi ndizomwe zimafunikira komanso zofunikira zoyambirira. Dinani apa kuti muwone zambiri za masanjidwe.

tag_id (kapena case_ID) chizindikiritso chapadera (gwiritsani ntchito zilembo za alphanumeric, ma hyphens amavomerezedwa)

case_status 0 = chatsekedwa 1 = yogwira

tsiku_lotsegulidwa mwezi/tsiku/chaka mm/dd/yyyy

za kufotokozera kwapadera kwa mlanduwo (gwiritsani ntchito zilembo za alphanumeric, ma hyphens amavomerezedwa)

Kukonza

Kutengera kuchuluka kwa zolemba zomwe zili mufayilo ndi katundu wamakina, nsanja imatha kutenga mphindi zingapo kuti ikonze zomwe zatumizidwa.

img

Kutumizako kukamaliza, nsanjayo imasintha zolemba zamilandu (mwachitsanzo, kuwonjezera kapena kusintha) padashboard yamilandu ya admin.

Kulephera

Ngati ntchito yoitanitsa ikulephera, nsanja imabwezera uthenga Wolephera. Nthawi zambiri, chifukwa cha kulephera ndi masanjidwe deta.

img

Kuti muwone chifukwa cha kulephera:

  1. Dinani Onani pansi pa mutu wa ACTION.
  2. Patsamba lotsatira, ikani mbewa pa chizindikiro Cholephereka. Chida chothandizira chikuwonetsa cholakwika.
  3. Pitani ku fayilo ya CVS ndikusintha ndikukonza zolakwikazo.
  4. Kwezani fayilo kachiwiri.
