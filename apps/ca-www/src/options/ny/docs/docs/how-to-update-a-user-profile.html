Momwe Mungasinthire Mbiri Yogwiritsa Ntchito

Nkhaniyi ikuwonetsa ogwiritsa ntchito momwe angasinthire zambiri patsamba lawo la Mbiri.

  • Ogwiritsa akhoza kusintha mbiri yawo.
  • Ma Admins amatha kusintha Tsamba la Mbiri ya wogwiritsa ntchito aliyense.

Kwa ogwiritsa ntchito

Gawo loyamba ndikutsegula Tsamba la Mbiri. Ogwiritsa ndi ma admin amatsata izi kuti asinthe tsamba lawo la Mbiri.

  1. Lowani muakaunti yanu ya Case Active.
  2. Dinani Profile tabu pamwamba pa tsamba.
  3. Sinthani.
  4. Dinani Save.

Minda yambiri ndi yosankha, ndipo minda ina ndiyofunika. Minda yofunikira ndi:

  • Dzina loyamba
  • Dzina lomaliza
  • Imelo adilesi
  • Chiyankhulo
  • Zone nthawi

Za ma admin

Oyang'anira amathanso kusintha zambiri za mbiri yanu kwa ogwiritsa ntchito ena. Izi zikuphatikiza magawo atatu pansi pamutu wa HOST USER INFO.

  1. Lowani ku akaunti yanu ya Case Active admin.
  2. Pezani pulogalamu ndikudina batani la ADMIN.
  3. Dinani Ogwiritsa ntchito pamwamba menyu.
  4. Pitani ku menyu yaing'ono ndikusankha mndandanda wa ogwiritsa ntchito omwe mukufuna kupeza (ZOCHITIKA kapena WOLEMA). Zokonda zokhazikika ndi Active.
  5. Pezani wogwiritsa ntchito pamndandanda ndikudina Sinthani.
  6. Sinthani ngati pakufunika. (Onani pansipa)
  7. Dinani Save.

Sinthani zambiri za ogwiritsa ntchito

Ma Admins amatha kusintha zambiri za ogwiritsa ntchito.

  1. Khazikitsani Mtundu Wogwiritsa (Wogwiritsa kapena Woyang'anira).
  2. Khazikitsani Mkhalidwe Wogwiritsa Ntchito (Wogwira Ntchito mpaka Woyimitsidwa).
  3. Lowetsani Tag ID yapadera. (posankha)
  4. Dinani Save.
