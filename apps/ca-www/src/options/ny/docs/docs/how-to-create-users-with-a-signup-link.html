Momwe Mungapangire Ogwiritsa ndi Ulalo Wolembetsa

Nkhaniyi ikuwonetsa ma admins momwe angapangire ogwiritsa ntchito potumiza ulalo wolembetsa.

Kukonza ulalo wolembetsa

  1. Dinani Atumizireni ulalo wolembetsa pansi pa Add user dialog box. Tsamba latsopano likuwonekera.
  2. Konzani zolembetsa. (Onani pansipa)

img

Onetsetsani kuti Lolani Kulembetsa kuti muyambitse. (yofunikira) Ikayatsidwa, Case Active imapanga ulalo wapadera.

Onaninso zomwe zili pansipa ndikudina chizindikirochi kuti mukopere ulalo. Mutha kuyika ulalo patsamba lanu kapena kutumiza ulalo kwa munthu m'modzi kapena angapo kudzera pa imelo kapena njira iliyonse yolumikizirana.

img

Kupanga mlandu

Mwachidziwitso, mutha kupanga zokha mlandu kwa wogwiritsa watsopano aliyense ndikumupatsa wogwiritsa ntchito.

  1. Dinani gawo la Auto Pangani Case ndikusankha On. Magawo a Case User Role amawonekera.
  2. Dinani Case User Role ndikusankha njira. (Onani pansipa)

img

Maudindo ogwiritsa ntchito milandu

Pali maudindo atatu ogwiritsira ntchito.

  • makasitomala
  • kusakhulupirika
  • ndodo

Kutumiza ulalo

Gawani ulalo ndi munthu m'modzi kapena angapo. Munthu aliyense amadina ulalo ndikumaliza ntchito kuti atsegule akaunti yake ya Case Active. Wogwiritsa ntchito akamaliza kusaina, Case Active imawonjezera wogwiritsa ntchito pamndandanda wa Ogwiritsa.

Masitepe otsatira

Case Active imatumiza chidziwitso cha imelo ndi ulalo wolembetsa. Wogwiritsa amadina ulalo ndikumaliza mayendedwe a olembetsa atsopano.

  • Nkhaniyi ikufotokoza za kulembetsa kwa ogwiritsa ntchito atsopano.
