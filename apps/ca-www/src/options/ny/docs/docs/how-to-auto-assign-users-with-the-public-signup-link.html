Momwe Mungagawire Ogwiritsa Ntchito ndi Public Signup Link

Oyang'anira amaphunzira momwe angakhazikitsire zosintha zamakina kuti ogwiritsa ntchito atsopano azingoperekedwa kumilandu yomwe yangopangidwa kumene.

Kutsegula tsamba lokonzekera

Gawo loyamba ndikutsegula tsamba la Systems Setup.

  1. Lowani ku akaunti ya admin.
  2. Pezani pulogalamu patsamba la Mapulogalamu ndikudina batani la ADMIN.
  3. Dinani Systems tabu ndiyeno SYSTEM SETUP pa submenu.
  4. Pitani pansi mpaka pamutu wa USER SIGNUP.

Kukonza makonda adongosolo

  1. Khazikitsani mindandanda yotsikira pansi ya Lolani Kusaina ndi Kungopanga Mlandu kuti On.
  2. Dinani mndandanda wa maudindo ndikusankha njira (zosasintha ndizovomerezeka).
  3. Koperani ndikugawana ulalo wa Public User Signup.

Kutumiza ulalo

Wogwiritsa ntchito aliyense amene amadina ulalo ndikumaliza kulembetsa kumayambitsa nsanja kuti ipange nkhani yatsopano patsamba la Milandu ndikugawa wogwiritsa ntchitoyo.
