Momwe Mungapangire Kapena Kusintha Ogwiritsa ndi Chida Chothandizira

Nkhaniyi ikuwonetsa ma admins momwe angapangire ogwiritsa ntchito ndi chida cha Import.

Kutsegula chida cha Import

Oyang'anira atha kugwiritsa ntchito chida cha Import kuti akweze ogwiritsa ntchito ambiri ndikusintha ma rekodi omwe alipo.

  1. Lowani ku akaunti ya admin.
  2. Pezani pulogalamu patsamba la Mapulogalamu ndikudina batani la ADMIN.
  3. Pa gulu la Admin, dinani tabu Yophatikiza ndiyeno IMPORTS pa submenu.
  4. Dinani batani la CHATSOPANO. Bokosi la dialog la Add Imports likuwonekera pa tabu yatsopano ya msakatuli.

img

Kupanga zolemba

Pambuyo pa bokosi la Add Imports litatsegulidwa:

  1. Dinani Type field ndikusankha njira (onani pansipa). Ulalo ukuwonekera.
  2. Dinani ulalo wa Tsitsani Zitsanzo . Fayilo ya CSV imatsitsidwa ku chipangizo chanu.

Kenako, tsegulani fayilo ndikuwonjezera zolemba. Dinani apa kuti mudziwe momwe mungagwiritsire ntchito mafayilo amtundu wa CSV.

img

Mtundu

Mukadina Type field, menyu yotsitsa imawonekera. Sankhani imodzi mwa njira ziwiri zogulitsira kunja zambiri.

:kufuula: ZINDIKIRANI Menyu yotsikira pansi ili ndi zosankha zambiri, koma ziwiri zokha ndizogwirizana ndi chida cha Import. Njira iliyonse ili ndi fayilo yosiyana ya CSV.

Onjezani ogwiritsa ntchito

  • Sankhani njira iyi kuti muyike zolemba zambiri za ogwiritsa ntchito mosavuta.
  • Dinani ulalo wa Chitsanzo Chotsitsa kuti mutsegule fayilo ya CSV yokhala ndi mitu yagawo la mtundu wa Add users Import.

Sinthani ogwiritsa ntchito

  • Sankhani izi kuti musinthe data ya ogwiritsa ntchito pano.
  • Dinani ulalo wa Tsitsani Zitsanzo kuti mutsegule fayilo ya CSV yokhala ndi mitu yazakusintha kwa mtundu wa Sinthani ogwiritsa Ntchito.

Kukweza fayilo

Mutha kukweza fayilo nthawi yomweyo kapena kubwerera ku bokosi la Add Imports pambuyo pake ndikumaliza izi.

  1. Tsegulani bokosi la Add Imports dialog.
  2. Lowetsani dzina lapadera la fayilo ya CSV.
  3. Dinani chizindikiro chowonjezera (+) kuti mukweze fayilo kuchokera pa chipangizo chanu.
  4. Dinani Type kumunda ndikusankha njira. (onani pamwambapa)
  5. Dinani KUSUNGA.

img

Kukonza

Kutengera kuchuluka kwa zolemba zomwe zili mufayilo ndi katundu wamakina, nsanja imatha kutenga mphindi zingapo kuti ikonze zomwe zatumizidwa.

img

Kutumizako kukamaliza, nsanja imasintha zolemba za ogwiritsa ntchito (mwachitsanzo, kuwonjezera, kusintha, kapena kuyimitsa) pa dashboard ya Admin's Users .

Kulephera

Ngati ntchito yoitanitsa ikulephera, nsanja imabwezera uthenga Wolephera. Nthawi zambiri, chifukwa cha kulephera ndi masanjidwe deta.

Kuti muwone chifukwa cha kulephera:

  1. Dinani View pansi pa mutu wa Action.
  2. Patsamba lotsatira, ikani mbewa pa chizindikiro Cholephereka. Chida chothandizira chikuwonetsa cholakwika.
  3. Pitani ku fayilo ya CVS ndikusintha ndikukonza zolakwikazo.
  4. Kwezani fayilo kachiwiri.
