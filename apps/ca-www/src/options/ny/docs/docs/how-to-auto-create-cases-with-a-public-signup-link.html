Momwe Mungapangire Milandu Yokha Ndi Ulalo Wolembetsa Pagulu

Nkhaniyi ikuwonetsa ma admins momwe angapangire mlandu (ndikuwonjezera wogwiritsa ntchito nthawi yomweyo) potumiza ulalo umodzi wolembetsa.

Kupeza ulalo

  1. Lowani ku akaunti ya admin.
  2. Pezani pulogalamu patsamba la Mapulogalamu ndikudina batani la ADMIN.
  3. Pa gulu la Admin, dinani tabu ya Zikhazikiko.
  4. Onetsetsani kuti tabu ya SYSTEM SETUP yasankhidwa.

Kukonza zokonda

Pitani pansi mpaka pamutu wa USER SIGNUP.

Onetsetsani kuti malo a Lolani Kusaina akhazikitsidwa. (chofunikira) Izi zikayatsidwa, Case Active imapanga ulalo wapadera.

  1. Dinani gawo la Auto Pangani Case ndikusankha On. Gawo la Case User Role likuwonetsedwa.
  2. Sankhani gawo. (Onani pansipa)

Maudindo ogwiritsa ntchito milandu

Pali maudindo atatu ogwiritsira ntchito.

  • makasitomala (ovomerezeka)
  • kusakhulupirika
  • ndodo

Kutumiza ulalo

Gawani ulalo ndi munthu m'modzi kapena angapo. Wogwiritsa ntchito aliyense amene amadina ulalo ndikumaliza kulembetsa amayambitsa nsanja kuti ipange nkhani yatsopano patsamba lamilandu ndikugawa wogwiritsa ntchitoyo.
