Momwe Mungagawire, Kusintha kapena Kuchotsa Ntchito Zogwiritsa Ntchito ndi Chida Chothandizira

Nkhaniyi ikuwonetsa ma admins momwe angasamalire ogwiritsa ntchito ndi chida cha Import.

Ma Admin atha:

  • perekani ogwiritsa ntchito mlandu
  • sinthani ogwiritsa ntchito omwe apatsidwa kale mlandu
  • chotsani ogwiritsa ntchito omwe apatsidwa mlandu

Kutsegula chida cha Import

  1. Lowani ku akaunti ya admin.
  2. Pezani pulogalamu patsamba la Mapulogalamu ndikudina batani la ADMIN.
  3. Dinani Integration tabu ndiyeno IMPORTS pa submenu.
  4. Dinani batani la CHATSOPANO. The Add Imports dialog box ikuwonekera.

Kupanga zolemba

Tsitsani fayilo imodzi kapena zingapo za CSV. Gwiritsani ntchito mafayilowa kuwonjezera, kusintha, kapena kuchotsa ogwiritsa ntchito. Pambuyo pa bokosi la Add Imports litatsegulidwa:

  1. Dinani Type kumunda.
  2. Pitani ku menyu yotsitsa ndikusankha njira (onani pansipa). Ulalo ukuwonekera.
  3. Dinani ulalo wa Tsitsani Zitsanzo. Fayilo ya CSV imatsitsidwa ku chipangizo chanu.

img

Tsegulani fayilo ndikuwonjezera zolemba. Dinani apa kuti mudziwe momwe mungagwiritsire ntchito mafayilo amtundu wa CSV.

Zosankha zamafayilo

Perekani ogwiritsa ntchito milandu * Sankhani njira iyi kuti muwonjezere ogwiritsa ntchito pamlandu. * Siyani nkhani_role_order cell yopanda kanthu. * Pitani patsamba la Ogwiritsa ntchito ndikukopera Tag ID ya munthuyu. Ngati gawo la Tag ID lilibe kanthu, lowetsani ID yapadera, dinani Sungani, ndiyeno koperani ID ku fayilo ya CSV.

img

Sinthani ogwiritsa ntchito omwe aperekedwa kumilandu * Sankhani izi kuti musinthe zambiri za ogwiritsa ntchito omwe apatsidwa mlandu. * Gwiritsani ntchito Case Tag ID pagawo la case_ID.

Chotsani ogwiritsa ntchito omwe apatsidwa milandu * Sankhani izi kuti muchotse ogwiritsa ntchito omwe apatsidwa mlandu.

Kukweza fayilo

Mutha kukweza fayilo nthawi yomweyo kapena kubwerera ku bokosi la Add Imports pambuyo pake ndikumaliza izi.

  1. Tsegulani bokosi la Add Imports dialog.
  2. Lowetsani dzina lapadera la fayilo ya CSV.
  3. Dinani chizindikiro chowonjezera (+) kuti mukweze fayilo kuchokera pa chipangizo chanu.
  4. Dinani Type kumunda ndikusankha njira. (onani pamwambapa)
  5. Dinani KUSUNGA.

Kukonza

Kutengera kuchuluka kwa zolemba zomwe zili mufayilo ndi kasamalidwe kadongosolo, nsanja imatha kutenga mphindi zingapo kuti ikonze zolowera.

img

Kutumizako kukamaliza, nsanja imasintha zolemba za ogwiritsa ntchito (mwachitsanzo, kuwonjezera, kusintha, kapena kuchotsa) patsamba la Milandu.

Kulephera

Ngati ntchito yoitanitsa ikulephera, nsanja imabwezera uthenga Wolephera. Nthawi zambiri, chifukwa cha kulephera ndi masanjidwe deta.

img

Kuti muwone chifukwa cha kulephera:

  1. Dinani View pansi pa mutu wa Action.
  2. Patsamba lotsatira, ikani mbewa pa chizindikiro Cholephereka. Chida chothandizira chikuwonetsa cholakwika.
  3. Pitani ku fayilo ya CVS ndikusintha ndikukonza zolakwikazo.
  4. Kwezani fayilo kachiwiri.
