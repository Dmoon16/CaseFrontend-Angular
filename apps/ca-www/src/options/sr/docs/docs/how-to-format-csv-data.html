Како форматирати ЦСВ податке

Испод је листа поља података у ЦСВ датотекама. Свака опција увоза има другачију комбинацију поља.

корисничко име

  • Унесите адресу е-поште корисника.

име

  • Унесите име корисника.

презиме

  • Унесите презиме корисника.

лоцале

  • Ова колона проверава жељени језик корисника.
  • Унесите ен за енглески, сп за шпански.

зонеинфо

  • Ово поставља временску зону корисника. Унесите континент и град без размака (користите доњу црту).
  • Мора се користити опција у падајућем менију Профил > ЈЕЗИК & РЕГИОН > Временска зона.
  • Користите временску зону која вам највише одговара. На пример, Бостон није у падајућем менију, тако да можемо да користимо Њујорк (или било који град у временској зони ЕСТ/ЕДТ).

хост_таг_ид

  • Унесите трајну ознаку.
  • аз, 0-9, -, / су дозвољени.

цасе_статус

  • 0 = затворено
  • 1 = активан
