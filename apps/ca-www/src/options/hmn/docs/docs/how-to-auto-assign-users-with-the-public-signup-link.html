Thawj kauj ruam yog qhib nplooj ntawv Systems Setup.

  3. Nyem rau Systems tab thiab tom qab ntawd SYSTEM SETUP ntawm lub submenu.
  4. Scroll down mus rau USER SIGNUP nqe lus.

Configuring system nqis

  1. Teem lub Allow Signup thiab Auto Create Case drop-down menus rau On.
  2. Nyem rau lub luag hauj lwm ntawv qhia zaub mov thiab xaiv ib qho kev xaiv (default yog pom zoo).
  3. Luam thiab muab faib rau Public User Signup link.
