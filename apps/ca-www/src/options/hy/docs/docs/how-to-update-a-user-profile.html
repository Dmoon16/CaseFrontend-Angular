Ինչպես թարմացնել օգտվողի պրոֆիլը

Այս հոդվածը ցույց է տալիս օգտվողներին, թե ինչպես թարմացնել տեղեկատվությունը իրենց պրոֆիլի էջում:

  • Օգտագործողները կարող են թարմացնել իրենց սեփական պրոֆիլները:
  • Ադմինները կարող են թարմացնել ցանկացած օգտատիրոջ պրոֆիլի էջը։

Օգտագործողների համար

Առաջին քայլը պրոֆիլի էջը բացելն է: Օգտագործողները և ադմինիստրատորները հետևում են այս քայլերին՝ թարմացնելու իրենց սեփական պրոֆիլի էջը:

  1. Մուտք գործեք ձեր Case Active հաշիվ:
  2. Կտտացրեք «Պրոֆիլ» ներդիրը էջի վերևում:
  3. Փոփոխություններ կատարեք։
  4. Սեղմեք Պահպանել:

Ադմինիստրատորների համար

Ադմինները կարող են նաև թարմացնել պրոֆիլի տվյալները այլ օգտվողների համար: Սա ներառում է HOST USER INFO վերնագրի տակ գտնվող երեք դաշտ:

  1. Մուտք գործեք ձեր Case Active ադմինիստրատորի հաշիվ:
  2. Տեղադրեք հավելվածը և սեղմեք ADMIN կոճակը:
  3. Սեղմեք Users վերևի ընտրացանկում:
  4. Գնացեք ենթացանկ և ընտրեք օգտատերերի ցանկը, որին ցանկանում եք մուտք գործել (ԱԿՏԻՎ կամ ԱԿՏԻՎԱԾ): Կանխադրված պարամետրը Ակտիվ է:
  5. Գտեք օգտվողին ցուցակում և սեղմեք Խմբագրել:
  6. Փոփոխություններ կատարեք ըստ պահանջի: (տես ներքեւում)
  7. Սեղմեք Պահպանել:
