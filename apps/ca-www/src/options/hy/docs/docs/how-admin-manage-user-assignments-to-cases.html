Ինչպես են ադմինները կառավարում օգտատերերի հանձնարարությունները գործերին

Օգտատերերին և գործերի առաջադրանքները կառավարելու հինգ եղանակ կա: Սեղմեք հղման վրա՝ իմանալու համար, թե ինչպես:

Մեթոդ Տիպ Գործողություններ Մանրամասներ
Գործ ստեղծելիս UI ավելացնել Ադմինիստրատորի կոճակ > Գործեր > Նոր կոճակ > Նշանակել օգտվողներին կոճակ
Ավտոմատ նշանակել օգտվողին` օգտագործելով հանրային գրանցման հղումը Օգտագործողի գործողություն ավելացնել Ադմինիստրատորի կոճակ > Կարգավորումներ > Համակարգի կարգավորումների էջ
Օգտագործելով UI գոյություն ունեցող գործի վրա UI ավելացնել և հեռացնել Ադմինիստրատորի կոճակ > Գործեր > Նշանակել օգտվողներին
Ներմուծման գործիք UI ավելացնել, փոփոխել և հեռացնել Ադմինիստրատորի կոճակ > Ինտեգրումներ > Ներմուծումներ
