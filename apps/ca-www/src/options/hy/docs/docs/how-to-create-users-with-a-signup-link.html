Ինչպես ստեղծել օգտվողներ գրանցման հղումով

Այս հոդվածը ցույց է տալիս ադմինիստրատորներին, թե ինչպես ստեղծել օգտվողներ՝ ուղարկելով գրանցման հղում:

Գրանցման հղումի կարգավորում

  1. Կտտացրեք « Ուղարկել նրանց գրանցման» հղումը «Ավելացնել օգտվողին» երկխոսության տուփի ներքևում: Ցուցադրվում է նոր էջ:
  2. Կարգավորեք գրանցման ընտրանքները: (տես ներքեւում)

Համոզվեք, որ Թույլատրել գրանցման դաշտը միացված է: (պարտադիր) Երբ միացված է, Case Active-ը ստեղծում է եզակի հղում:

Դիտեք ստորև ներկայացված ընտրանքները և սեղմեք պատկերակը՝ հղումը պատճենելու համար: Դուք կարող եք հղումը տեղադրել ձեր կայքում կամ ուղարկել հղումը մեկ կամ մի քանի մարդկանց էլ.փոստի կամ հաղորդակցման որևէ ալիքի միջոցով:

img

Գործի ստեղծում

Ընտրովի, դուք կարող եք ավտոմատ կերպով ստեղծել գործ յուրաքանչյուր նոր օգտվողի համար և նշանակել օգտվողի դեր:

  1. Սեղմեք Auto Create Case դաշտը և ընտրեք Միացնել: Ցուցադրվում են Case User Role դաշտերը:
  2. Սեղմեք Case User Role-ը և ընտրեք որևէ տարբերակ: (տես ներքեւում)

Գործի օգտագործողի դերերը

Գործի օգտագործողի երեք դեր կա:

  • հաճախորդներ
  • լռելյայն
  • անձնակազմը

Ուղարկում է հղումը

Կիսվեք հղումով մեկ կամ մի քանի մարդկանց հետ: Յուրաքանչյուր մարդ կտտացնում է հղմանը և ավարտում է աշխատանքային հոսքը՝ ակտիվացնելու իր Case Active հաշիվը: Երբ օգտատերը ավարտում է գրանցման աշխատանքային հոսքը, Case Active-ն ավելացնում է օգտվողին օգտվողների ցանկում:

Հաջորդ քայլերը

Case Active-ը օգտատիրոջը էլեկտրոնային փոստով ծանուցում է ուղարկում գրանցման հղումով: Օգտագործողը կտտացնում է հղմանը և ավարտում է աշխատանքի ընթացքը նոր գրանցումների համար:

  • Այս հոդվածը նկարագրում է գրանցման աշխատանքային հոսքը նոր օգտվողների համար:
