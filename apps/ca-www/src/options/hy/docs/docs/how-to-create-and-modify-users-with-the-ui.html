Ինչպես ստեղծել և փոփոխել օգտվողներին միջերեսի միջոցով

Այս հոդվածը ցույց է տալիս ադմինիստրատորներին, թե ինչպես ստեղծել օգտատերեր և թարմացնել գրառումները հարթակի օգտատիրոջ միջերեսով: Ադմինները կարող են նաև օգտատերեր ավելացնել նոր գործին և խմբագրել առկա գործերը:

Օգտագործողի ստեղծում

  1. Մուտք գործեք ադմինիստրատորի հաշիվ:
  2. Տեղադրեք հավելվածը Apps էջում և սեղմեք ADMIN կոճակը:
  3. Ադմինիստրատորի վահանակում կտտացրեք Օգտագործողների ցանկի ներդիրին:
  4. Սեղմեք ՆՈՐ կոճակը: Ցուցադրվում է նոր պատուհան:

Լրացրեք դաշտերը և սեղմեք ՊԱՀՊԱՆԵԼ: Case Active-ն ավելացնում է անունը այս հավելվածի հետ կապված օգտատերերի ցանկում:

Օգտատիրոջ փոփոխում

Օգտատիրոջ փոքր թվով գրառումները թարմացնելու համար օգտագործեք «Խմբագրել» գործիքը և փոփոխություններ կատարեք յուրաքանչյուր առանձին օգտագործողի համար:

  • Նշում Մեկ նիստում օգտագործողների մեծ թվով գրառումները փոփոխելու համար օգտագործեք Ներմուծման գործիքը :
  1. Ադմինիստրատորի վահանակում կտտացրեք Օգտագործողների ցանկի ներդիրին:
  2. Գտեք օգտվողին և սեղմեք Խմբագրել:
  3. Փոփոխություններ կատարեք և սեղմեք ՊԱՀՊԱՆԵԼ:

img

Հաջորդ քայլերը

Case Active-ը օգտատիրոջը էլեկտրոնային ծանուցում չի ուղարկում: Ադմինները պետք է կապվեն օգտատիրոջ հետ և ուղարկեն հղում դեպի Case Active գլխավոր էջի: Այնուհետև օգտագործողը սեղմում է LOGIN կոճակը և ավարտում է նոր գրանցումների աշխատանքային ընթացքը:

  • Այս հոդվածը նկարագրում է գրանցման աշխատանքային հոսքը նոր օգտվողների համար:
