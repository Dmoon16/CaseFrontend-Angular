Ինչպես օգտատերերին նշանակել պատյան ստեղծելիս

Այս հոդվածը ցույց է տալիս ադմինիստրատորներին, թե ինչպես նշանակել օգտվողին նոր գործ ստեղծելիս: Եթե օգտատերը դեռ գրանցված չէ հարթակում, կարող եք ստեղծել ավելացնել նոր օգտվող՝ նոր գործ ստեղծելիս:

Գործի ստեղծում

  3. Սեղմեք Դեպքեր ցանկի ներդիրը:
  4. Սեղմեք ՆՈՐ կոճակը: Ցուցադրվում է նոր պատուհան:

Օգտագործողի նշանակում

Ընթացիկ օգտատեր

Հետևեք այս քայլերին` նոր գործին ներկա օգտվողին նշանակելու համար:

Սեղմեք ASSIGN USERS կոճակը և լրացրեք դաշտերը:

  1. Սեղմեք Օգտվողի բացվող ցանկը և ընտրեք նույնացուցիչ (օրինակ՝ էլ. փոստ):
  2. Մուտքագրեք արժեք դատարկ դաշտում (օրինակ՝ էլփոստի հասցե):
  3. Եթե օգտատիրոջ գրառումը գոյություն ունի հարթակում, դաշտը ավտոմատ կերպով առաջարկում է անուն:
  4. Ընտրեք ճիշտ օգտվողին:
  5. Ընտրեք օգտվողի դերը (հաճախորդներ, լռելյայն կամ անձնակազմ):
  6. Ցանկության դեպքում նշեք վանդակը՝ օգտատիրոջը էլեկտրոնային փոստով ծանուցում ուղարկելու համար:
  7. Սեղմեք ՊԱՀՊԱՆԵԼ:

Case Active-ն ավելացնում է օգտվողին գործին: Այժմ ադմինները կարող են սեղմել «Պահել»՝ պատուհանը փակելու համար, կամ նոր օգտվող ավելացնելու կամ օգտվողին հեռացնելու համար:
