Ինչպես են ադմինները կառավարում գործերը

Գործը կառավարելու չորս եղանակ կա. Սեղմեք հղման վրա՝ իմանալու համար, թե ինչպես:

Մեթոդ Տիպ Գործողություններ Մանրամասներ
Օգտագործելով UI UI ավելացնել և փոփոխել Ադմինիստրատորի կոճակ > Գործեր
Ավտոմատ ստեղծել պատյան՝ օգտագործելով հանրային գրանցման հղումը Օգտագործողի գործողություն ավելացնել Ադմինիստրատորի կոճակ > Կարգավորումներ > Համակարգի կարգավորումների էջ
Ներմուծման գործիք UI ավելացնել և փոփոխել Ադմինիստրատորի կոճակ > Ինտեգրումներ > Ներմուծումներ
