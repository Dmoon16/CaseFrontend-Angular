Ինչպես ստեղծել հավելված

Այս հոդվածը ցույց է տալիս նոր օգտվողներին, թե ինչպես ստեղծել անվճար փորձնական հաշիվ: Case Active-ն ավտոմատ կերպով նոր օգտվողին տալիս է ադմինիստրատորի արտոնություններ այս հաշվի համար:

Հաշվի ստեղծում

  1. Գնացեք մուտքի էջ :
  2. Սեղմեք «Սկսել» կոճակը:
  3. Մուտքագրեք էլփոստի հասցե:
  4. Սեղմեք Հաջորդը:

Հաշիվը ստուգելը

  1. Բացեք դիտարկիչի նոր ներդիր և մուտք գործեք ձեր էլ.փոստի հաշիվ:
  2. Պատճենեք վեցանիշ կոդը:
  3. Վերադարձեք Case Active ներդիրին։
  4. Տեղադրեք կոդը դաշտերում: Ցուցադրվում է նոր պատուհան:

Նոր պատուհանում.

  1. Լրացրեք պահանջվող դաշտերը:
  2. Կարդացեք ծառայության պայմանները և նշեք վանդակը՝ համաձայնելու համար:
  3. Սեղմեք ՍՏԵՂԾԵԼ ԱԶԱՏ ՀԱՇԻՎ:

Հավելվածի կարգավորումն ավարտվում է

Աշխատանքային հոսքը ցուցադրվում է հաջորդ էջում: Կատարեք քայլերը՝ ձեր հավելվածի կարգավորումն ավարտելու համար: Դուք միշտ կարող եք վերադառնալ այս էջեր՝ տեղեկատվությունը կամ ձեր հաշվի կարգավիճակը խմբագրելու կամ թարմացնելու համար:

Հավելվածի մասին էջում լրացրեք ընկերության անվանումը և վեբկայքի դաշտերը (պարտադիր է), այնուհետև սեղմեք Հաջորդը:

img

Վճարման տեղեկատվության էջում սեղմեք Վճարման պլան դաշտը և վերանայեք տարբերակները: Լռելյայն կարգավորումն անվճար է: Սեղմեք SUBMIT:

Ձևը ներկայացնելուց հետո Case Active-ը ցուցադրում է Apps էջը: Ձեր ստեղծած կամ պատկանող յուրաքանչյուր հավելվածի համար էջը ցույց է տալիս.

  1. Հավելվածի անունը
  2. Ձեր տեսակը (օգտագործողի դերը)
  3. Կարգավիճակ

img

Հավելվածների էջը մեկնարկային կետ է ադմինիստրատորի բազմաթիվ գործառույթների համար.

  1. Սեղմեք APP կոճակը՝ գործի հետ կապված խնդիրների վերաբերյալ շփվելու, փաստաթղթերը լրացնելու, տեսազրույցի (և ավելին):
  2. Սեղմեք ADMIN կոճակը՝ ադմինիստրատորի վահանակը բացելու համար: Այստեղ ադմինները կառավարում են օգտատերերին, գործերը, ինտեգրումները և այլն:
  3. Սեղմեք էլիպսը՝ հաշվի վճարման և ծանուցման կարգավորումները կառավարելու համար:
  • Note Case Active-ը ցուցադրում է ADMIN կոճակը ադմինիստրատորի օգտագործողների համար: Օգտատերերը, ովքեր չունեն ադմինիստրատորի արտոնություններ, չեն տեսնում ADMIN կոճակը (ինչպես ցույց է տրված ստորև ներկայացված էկրանին):
