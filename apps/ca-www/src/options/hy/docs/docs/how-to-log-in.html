Ինչպես մուտք գործել

Իմացեք, թե ինչպես մուտք գործել ձեր գոյություն ունեցող Case Active հաշիվ:

Մուտք գործեք

  1. Գնացեք մուտքի էջ :
  2. Սեղմեք LOGIN կոճակը:
  3. Մուտքագրեք ձեր էլ.
  4. Սեղմեք ՀԱՋՈՐԴ:

Հաշիվը ստուգելը

  1. Բացեք դիտարկիչի նոր ներդիր և մուտք գործեք ձեր էլ.փոստի հաշիվ:
  2. Պատճենեք վեցանիշ կոդը:
  3. Վերադարձեք Case Active ներդիրին։
  4. Տեղադրեք կոդը դաշտերում: Ցուցադրվում է նոր պատուհան:
