Bagaimana Admin Membuat Pengguna

Admin membuat, mengubah, atau menonaktifkan siapa yang dapat mengakses aplikasi dan kasus. Saat pengguna masuk ke platform untuk pertama kalinya, mereka harus menyelesaikan alur kerja profil.

  • Klik di sini untuk mempelajari lebih lanjut tentang alur kerja profil pengguna baru.

Membuat pengguna

Tabel di bawah menunjukkan lima cara untuk membuat pengguna. Klik tautan untuk mempelajari caranya.

| Metode | Ketik | Tindakan | Detail | Pengguna baru menerima notifikasi? |
|---------------------------------------|-------------------|------- ------|-----------------------|------------------------- ------------------------|

Menggunakan UI | UI | tambah & ubah | Tombol Admin > Pengguna | Tidak |

Saat membuat kasus dan menetapkan pengguna ke kasus | UI | tambahkan | Tombol Admin > Kasus > Tombol baru > Halaman Buat Kasus | Ya, jika diaktifkan |

Otomatis buat pengguna menggunakan tautan pendaftaran publik | Tindakan pengguna | tambahkan | Tombol Admin > Pengaturan > halaman Pengaturan Sistem | Ya, jika pembuatan kasus otomatis diaktifkan |

Alat impor | UI | tambah & ubah | Tombol Admin > Integrasi > Impor | Tidak |

Menggunakan API (segera hadir) | API | tambah & ubah | na | na |

Notifikasi

Platform tidak mengirimkan pemberitahuan kepada pengguna saat mereka ditambahkan ke aplikasi. Namun, admin dapat mengaktifkan notifikasi untuk penetapan kasus.
