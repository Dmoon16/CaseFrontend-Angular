Cara Membuat atau Memodifikasi Kasus dengan Alat Impor

Artikel ini menunjukkan kepada admin cara membuat atau mengubah kasus yang ada dengan alat Impor.

Membuka alat Impor

  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Pada panel Admin, klik tab Integrasi lalu IMPOR pada submenu.
  4. Klik tombol BARU. Kotak dialog Tambah Impor ditampilkan di tab browser baru.

Membuat kasus

Setelah kotak dialog Tambah Impor terbuka:

  1. Klik bidang Jenis dan pilih opsi (lihat di bawah). Tautan ditampilkan.
  2. Klik tautan Unduh Sampel. File CSV diunduh ke perangkat Anda.

Selanjutnya, buka file dan tambahkan case.

Jenis

Saat Anda mengklik kolom Type , menu drop-down ditampilkan. Pilih salah satu dari dua opsi untuk impor kotak massal.

  • Catatan Menu tarik-turun memiliki banyak opsi, tetapi hanya dua yang terkait dengan pengelolaan kasus. Setiap opsi memiliki contoh file CSV yang berbeda.

Tambahkan kasus

  • Pilih opsi ini untuk mengunggah kasus secara massal.
  • Klik tautan Unduh Sampel untuk membuka file CSV dengan judul kolom untuk jenis impor Buat kasus.

Ubah kasus

  • Pilih opsi ini untuk memperbarui kasus saat ini.
  • Klik tautan Unduh Sampel untuk membuka file CSV dengan judul kolom untuk jenis impor Modifikasi kasus.

Mengunggah file

Anda dapat langsung mengunggah file atau kembali ke kotak dialog Tambahkan Impor di kemudian hari dan selesaikan langkah-langkah ini.

  1. Buka kotak dialog Tambah Impor.
  2. Masukkan nama unik untuk file CSV.
  3. Klik ikon tambah (+) untuk mengunggah file dari perangkat Anda.
  4. Klik bidang Jenis dan pilih opsi. (Lihat di atas)
  5. Klik SIMPAN.

Pemformatan

Ini adalah bidang wajib dan persyaratan pertama. Klik di sini untuk meninjau detail pemformatan.

tag_id (atau case_ID) pengidentifikasi unik (menggunakan karakter alfanumerik, tanda hubung diterima)

case_status 0 = tertutup 1 = aktif

date_opened bulan/hari/tahun mm/dd/yyyy

tentang deskripsi unik kasus (gunakan karakter alfanumerik, tanda hubung diterima)

Pengolahan

Bergantung pada jumlah rekaman dalam file dan beban sistem, platform dapat membutuhkan waktu beberapa menit untuk memproses impor.

Saat impor selesai, platform menyesuaikan catatan kasus (yaitu menambahkan atau memodifikasi) di dasbor Kasus admin.

Kegagalan

Jika operasi impor gagal, platform mengembalikan pesan Gagal. Dalam kebanyakan kasus, penyebab kegagalan adalah pemformatan data.

Untuk melihat penyebab kegagalan:

  1. Klik Lihat di bawah tajuk TINDAKAN.
  2. Di halaman berikutnya, arahkan mouse ke ikon Gagal. Tooltip menampilkan kesalahan.
  3. Buka file CVS dan sesuaikan dan perbaiki kesalahannya.
  4. Unggah file lagi.
