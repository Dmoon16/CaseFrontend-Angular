Cara Menetapkan Pengguna Saat Membuat Kasus

Artikel ini menunjukkan kepada admin cara menetapkan pengguna saat membuat kasus baru. Jika pengguna belum terdaftar di platform, Anda dapat menambahkan pengguna baru sambil membuat kasus baru.

Membuat kasus

  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Klik tab menu Kasus.
  4. Klik tombol BARU. Jendela baru ditampilkan.
  • Isi bidang untuk membuat kasus, lalu gulir ke bawah ke tajuk Pengguna yang Ditugaskan.

Menetapkan pengguna

Pengguna saat ini

Ikuti langkah-langkah ini untuk menetapkan pengguna saat ini ke kasus baru.

Klik tombol TUGASKAN PENGGUNA dan isi kolomnya.

  1. Klik tarik-turun Pengguna dan pilih pengidentifikasi (mis. email).
  2. Masukkan nilai di kolom kosong (misalnya alamat email).
  3. Jika rekaman pengguna ada di platform, bidang akan menyarankan nama secara otomatis.
  4. Pilih pengguna yang tepat.
  5. Pilih peran pengguna (klien, default, atau staf).
  6. Secara opsional, centang kotak untuk mengirimkan pemberitahuan email kepada pengguna.
  7. Klik SIMPAN.

img

Case Active menambahkan pengguna ke kasus. Sekarang admin dapat mengklik Simpan untuk menutup jendela, atau menambahkan pengguna baru, atau menghapus pengguna.

img

Menambahkan pengguna baru

Jika platform tidak dapat menemukan catatan saat ini, klik tombol untuk menambahkan pengguna baru.

img

Isi kolom, lalu klik SIMPAN untuk menambahkan pengguna ke platform. Selanjutnya, tetapkan peran pengguna dan klik SIMPAN lagi untuk menetapkan pengguna ke kasus tersebut. Terakhir, klik SIMPAN sekali lagi untuk menyimpan kasing.
