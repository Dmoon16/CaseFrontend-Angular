Cara Menetapkan atau Menghapus Tugas Pengguna dengan Kasus yang Ada

Artikel ini menunjukkan kepada admin cara menetapkan dan menghapus pengguna kasus. Kami juga menunjukkan cara mengubah data pengguna.

Membuka kasus yang ada

  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Klik tab menu Kasus .
  4. Pastikan AKTIF dipilih di submenu.
  5. Gulir ke bawah ke kolom TINDAKAN.

img

Tetapkan atau hapus

Klik Tetapkan Pengguna untuk menetapkan pengguna ke kasus atau menghapus pengguna dari kasus (namun bukan platform). Di halaman berikutnya: Klik TAMBAHKAN dan selesaikan alur kerja. Klik hapus untuk menghapus pengguna.

img

Memodifikasi

Tidak mungkin mengedit atau memperbarui data pengguna di halaman Kasus. Sebagai gantinya, ikuti langkah-langkah berikut:

  1. Klik tab menu Pengguna .
  2. Cari nama pengguna dan klik Edit .
  3. Lakukan perubahan dan klik SIMPAN.
