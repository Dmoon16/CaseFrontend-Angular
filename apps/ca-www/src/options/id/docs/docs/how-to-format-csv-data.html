Cara Memformat Data CSV

Di bawah ini adalah daftar bidang data dalam file CSV. Setiap opsi impor memiliki kombinasi bidang yang berbeda.

nama belakang

  • Masukkan alamat email pengguna.

nama pemberian

  • Masukkan nama depan pengguna.

nama keluarga

  • Masukkan nama belakang pengguna.

lokal

  • Kolom ini memeriksa bahasa pilihan pengguna.
  • Masukkan en untuk bahasa Inggris, sp untuk bahasa Spanyol.

zoneinfo

  • Ini mengatur zona waktu pengguna. Masukkan benua dan kota tanpa spasi (gunakan garis bawah).
  • Harus menggunakan opsi di menu tarik-turun Profil > BAHASA & WILAYAH > Zona Waktu.
  • Gunakan zona waktu yang paling sesuai. Misalnya, Boston tidak ada di drop-down, jadi kita bisa menggunakan New York (atau kota mana pun di zona waktu EST/EDT).

host_tag_id

  • Masukkan label permanen.
  • az, 0-9, -, / diperbolehkan.

kasus_status

  • 0 = tertutup
  • 1 = aktif
