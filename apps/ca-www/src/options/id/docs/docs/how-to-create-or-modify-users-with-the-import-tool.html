Cara Membuat atau Memodifikasi Pengguna dengan Alat Impor

Artikel ini menunjukkan kepada admin cara membuat pengguna dengan alat Impor.

Membuka alat Impor

Admin dapat menggunakan alat Impor untuk mengunggah pengguna baru secara massal dan mengubah rekaman pengguna yang ada.

  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Pada panel Admin, klik tab Integrasi lalu IMPOR pada submenu.
  4. Klik tombol BARU. Kotak dialog Tambah Impor ditampilkan di tab browser baru.

img

Membuat rekaman

Setelah kotak dialog Tambah Impor terbuka:

  1. Klik bidang Jenis dan pilih opsi (lihat di bawah). Tautan ditampilkan.
  2. Klik tautan Unduh Sampel . File CSV diunduh ke perangkat Anda.

Selanjutnya, buka file dan tambahkan catatan. Klik di sini untuk mempelajari cara menggunakan format data dalam file CSV.

img

Jenis

Saat Anda mengklik kolom Type, menu drop-down ditampilkan. Pilih salah satu dari dua opsi untuk impor massal.

:exclamation: CATATAN Menu drop-down memiliki banyak opsi, tetapi hanya dua yang terkait dengan alat Impor. Setiap opsi memiliki contoh file CSV yang berbeda.

Tambahkan pengguna

  • Pilih opsi ini untuk pengunggahan rekaman pengguna secara massal.
  • Klik tautan Unduh Sampel untuk membuka file CSV dengan judul kolom untuk jenis Tambahkan pengguna Impor.

Ubah pengguna

  • Pilih opsi ini untuk memperbarui data bagi pengguna saat ini.
  • Klik tautan Unduh Sampel untuk membuka file CSV dengan judul kolom untuk jenis Impor pengguna Modifikasi.

Mengunggah file

Anda dapat langsung mengunggah file atau kembali ke kotak dialog Tambahkan Impor di kemudian hari dan selesaikan langkah-langkah ini.

  1. Buka kotak dialog Tambah Impor.
  2. Masukkan nama unik untuk file CSV.
  3. Klik ikon tambah (+) untuk mengunggah file dari perangkat Anda.
  4. Klik bidang Jenis dan pilih opsi. (Lihat di atas)
  5. Klik SIMPAN.

Pengolahan

Bergantung pada jumlah rekaman dalam file dan beban sistem, platform dapat membutuhkan waktu beberapa menit untuk memproses impor.

Saat impor selesai, platform menyesuaikan catatan pengguna (yaitu menambahkan, memodifikasi, atau menonaktifkan) di dasbor Pengguna admin.

Kegagalan

Jika operasi impor gagal, platform mengembalikan pesan Gagal. Dalam kebanyakan kasus, penyebab kegagalan adalah pemformatan data.

img

Untuk melihat penyebab kegagalan:

  1. Klik Lihat di bawah judul Tindakan.
  2. Di halaman berikutnya, arahkan mouse ke ikon Gagal. Tooltip menampilkan kesalahan.
  3. Buka file CVS dan sesuaikan dan perbaiki kesalahannya.
  4. Unggah file lagi.
