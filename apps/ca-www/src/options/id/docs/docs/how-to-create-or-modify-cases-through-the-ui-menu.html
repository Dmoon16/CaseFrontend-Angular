Cara Membuat atau Memodifikasi Kasus Melalui Menu UI

Artikel ini menunjukkan kepada admin cara membuat kasus dengan antarmuka pengguna platform. Admin juga dapat menambahkan pengguna ke kasus baru dan memperbarui kasus yang ada.

Membuat kasus

  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Pada panel Admin, klik tab menu Kasus.
  4. Klik tombol BARU. Jendela baru ditampilkan.

Isi bidang yang wajib diisi dan klik SIMPAN.

img

Menambahkan pengguna

Secara opsional, admin dapat menambahkan pengguna ke kasus di halaman ini. Gulir ke bawah dan klik tombol ASSIGN USERS .

img

Dalam modal Tetapkan Pengguna, isi kolom. Centang kotak untuk mengirimkan pemberitahuan email kepada pengguna. (opsional)

Memodifikasi kasus

Untuk memperbarui sejumlah kecil kasus, gunakan alat Edit dan buat perubahan pada masing-masing kasus.

  • Catatan Untuk memperbarui sejumlah besar kasus sekaligus, pertimbangkan untuk menggunakan alat Impor
  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Pada panel Admin, klik tab menu Kasus.
  4. Temukan kasus dan klik Edit.
  5. Lakukan perubahan dan klik SIMPAN.
