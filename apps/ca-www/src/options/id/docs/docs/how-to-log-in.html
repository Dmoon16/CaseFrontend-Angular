Cara Masuk

Pelajari cara masuk ke akun Case Active Anda yang sudah ada.

Masuk

  1. Pergi ke halaman login .
  2. Klik tombol LOGIN.
  3. Masukkan email Anda.
  4. Klik BERIKUTNYA.

Memverifikasi akun

  1. Buka tab browser baru dan masuk ke akun email Anda.
  2. Salin kode enam digit.
  3. Kembali ke tab Kasus Aktif.
  4. Rekatkan kode ke dalam bidang. Jendela baru ditampilkan.

Case Active menampilkan halaman Aplikasi.

  • Catatan Platform menampilkan tombol ADMIN di sebelah tombol APP untuk pengguna dengan hak istimewa admin.
