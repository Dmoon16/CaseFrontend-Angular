Cara Membuat Kasus Secara Otomatis dengan Tautan Pendaftaran Publik

Artikel ini menunjukkan kepada admin cara membuat kasus (dan menambahkan pengguna pada saat yang sama) dengan mengirimkan satu tautan pendaftaran.

Menemukan tautan

  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Pada panel Admin, klik tab menu Pengaturan.
  4. Pastikan tab submenu PENGATURAN SISTEM dipilih.

Mengonfigurasi pengaturan

Gulir ke bawah ke tajuk USER SIGNUP.

Pastikan bidang Izinkan Pendaftaran disetel ke Aktif. (wajib) Saat fitur ini diaktifkan, Case Active menghasilkan tautan unik.

  1. Klik bidang Buat Kasus Otomatis dan pilih Aktif. Bidang Peran Pengguna Kasus ditampilkan.
  2. Pilih peran. (Lihat di bawah)

Kasus peran pengguna

Ada tiga peran pengguna kasus.

  • klien (disarankan)
  • bawaan
  • staf

Mengirim tautan

Bagikan tautan dengan satu atau beberapa orang. Setiap pengguna yang mengklik tautan dan menyelesaikan pendaftaran memicu platform untuk membuat kasus baru di halaman kasus dan secara otomatis menetapkan pengguna dalam kasus tersebut.
