ବିଦ୍ୟମାନ ମାମଲାଗୁଡ଼ିକ ସହିତ ଉପଭୋକ୍ତା ଆସାଇନମେଣ୍ଟଗୁଡିକ କିପରି ନ୍ୟସ୍ତ କରିବେ କିମ୍ବା ଅପସାରଣ କରିବେ |

କେସ୍ ଉପଭୋକ୍ତାମାନଙ୍କୁ କିପରି ନ୍ୟସ୍ତ କରିବେ ଏବଂ ଅପସାରଣ କରିବେ, ଏହି ଆର୍ଟିକିଲ୍ ଆଡମିନ୍ମାନଙ୍କୁ ଦର୍ଶାଏ | ୟୁଜର୍ ଡାଟା କିପରି ରୂପାନ୍ତର କରିବେ ଆମେ ମଧ୍ୟ ଆପଣଙ୍କୁ ଦେଖାଇବୁ |

ଏକ ବିଦ୍ୟମାନ ମାମଲା ଖୋଲିବା |

  1. ଆଡମିନ୍ ଆକାଉଣ୍ଟକୁ ଲଗ୍ ଇନ୍ କରନ୍ତୁ |
  2. ଆପ୍ ପୃଷ୍ଠାରେ ଏକ ଆପ୍ ସନ୍ଧାନ କରନ୍ତୁ ଏବଂ ADMIN ବଟନ୍ କ୍ଲିକ୍ କରନ୍ତୁ |
  3. କେସ୍ ମେନୁ ଟ୍ୟାବ୍ କ୍ଲିକ୍ କରନ୍ତୁ |
  4. ନିଶ୍ଚିତ କରନ୍ତୁ ଯେ ସବମେନୁରେ ACTIVE ଚୟନ କରାଯାଇଛି |
  5. ACTION ସ୍ତମ୍ଭକୁ ସ୍କ୍ରୋଲ୍ କରନ୍ତୁ |

ନ୍ୟସ୍ତ କରନ୍ତୁ କିମ୍ବା ଅପସାରଣ କରନ୍ତୁ |

ଏକ ଉପଭୋକ୍ତାଙ୍କୁ ଏହି ମାମଲାରେ ନ୍ୟସ୍ତ କରିବାକୁ କିମ୍ବା ଉପଭୋକ୍ତାଙ୍କୁ କେସ୍ ରୁ ଅପସାରଣ କରିବାକୁ କ୍ଲିକ୍ କରନ୍ତୁ (କିନ୍ତୁ ପ୍ଲାଟଫର୍ମ ନୁହେଁ) | ପରବର୍ତ୍ତୀ ପୃଷ୍ଠାରେ: ADD କ୍ଲିକ୍ କରନ୍ତୁ ଏବଂ କାର୍ଯ୍ୟଧାରା ସଂପୂର୍ଣ୍ଣ କରନ୍ତୁ | ଏକ ଉପଭୋକ୍ତା ବିଲୋପ କରିବାକୁ ଅପସାରଣ କ୍ଲିକ୍ କରନ୍ତୁ |

ପରିବର୍ତ୍ତନ କରନ୍ତୁ |

କେସ୍ ପୃଷ୍ଠାରେ ଉପଭୋକ୍ତା ତଥ୍ୟକୁ ଏଡିଟ୍ କିମ୍ବା ଅପଡେଟ୍ କରିବା ସମ୍ଭବ ନୁହେଁ | ଏହା ପରିବର୍ତ୍ତେ, ଏହି ପଦକ୍ଷେପଗୁଡ଼ିକୁ ଅନୁସରଣ କରନ୍ତୁ:

  1. ଉପଯୋଗକର୍ତ୍ତା ମେନୁ ଟ୍ୟାବ୍ କ୍ଲିକ୍ କରନ୍ତୁ |
  2. ଉପଯୋଗକର୍ତ୍ତା ନାମ ଖୋଜ ଏବଂ ସଂପାଦନ କ୍ଲିକ୍ କର |
  3. ପରିବର୍ତ୍ତନ କରନ୍ତୁ ଏବଂ ସଞ୍ଚୟ କ୍ଲିକ୍ କରନ୍ତୁ |
