ସର୍ବସାଧାରଣ ସାଇନ୍ ଅପ୍ ଲିଙ୍କ୍ ସହିତ ଉପଭୋକ୍ତାମାନଙ୍କୁ କିପରି ଅଟୋ ନ୍ୟସ୍ତ କରିବେ |

ସିଷ୍ଟମ ସେଟିଂସମୂହକୁ କିପରି ବିନ୍ୟାସ କରାଯିବ ତାହା ଆଡମିନମାନେ ଶିଖନ୍ତି ଯାହା ଦ୍ new ାରା ନୂତନ ଉପଭୋକ୍ତାମାନେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଏକ ନୂତନ ଅଟୋ ସୃଷ୍ଟି ହୋଇଥିବା ମାମଲାରେ ନ୍ୟସ୍ତ ହୁଅନ୍ତି |

ସେଟଅପ୍ ପେଜ୍ ଖୋଲିବା |

ପ୍ରଥମ ପଦକ୍ଷେପ ହେଉଛି ସିଷ୍ଟମ୍ ସେଟଅପ୍ ପେଜ୍ ଖୋଲିବା |

  1. ଆଡମିନି ଆକାଉଣ୍ଟକୁ ଲଗ୍ ଇନ୍ କରନ୍ତୁ |
  2. ଆପ୍ ପୃଷ୍ଠାରେ ଏକ ଆପ୍ ସନ୍ଧାନ କରନ୍ତୁ ଏବଂ ADMIN ବଟନ୍ କ୍ଲିକ୍ କରନ୍ତୁ |
  3. ସିଷ୍ଟମ୍ ଟ୍ୟାବ୍ କ୍ଲିକ୍ କରନ୍ତୁ ଏବଂ ତାପରେ ସବମେନୁରେ SYSTEM SETUP କ୍ଲିକ୍ କରନ୍ତୁ |
  4. USER SIGNUP ହେଡିଙ୍ଗକୁ ସ୍କ୍ରୋଲ୍ କରନ୍ତୁ |

ସିଷ୍ଟମ୍ ସେଟିଂସମୂହକୁ ବିନ୍ୟାସ କରୁଅଛି |

  1. ସାଇନ୍ ଅପ୍ ଏବଂ ଅଟୋ ସୃଷ୍ଟି କେସ୍ ଡ୍ରପ୍ ଡାଉନ୍ ମେନୁଗୁଡ଼ିକୁ ସେଟ୍ କରନ୍ତୁ |
  2. ଭୂମିକା ମେନୁ କ୍ଲିକ୍ କରନ୍ତୁ ଏବଂ ଏକ ବିକଳ୍ପ ବାଛନ୍ତୁ (ଡିଫଲ୍ଟ ସୁପାରିଶ କରାଯାଏ) |
  3. ପବ୍ଲିକ୍ ୟୁଜର୍ ସାଇନ୍ ଅପ୍ ଲିଙ୍କ୍ କପି ଏବଂ ଅଂଶୀଦାର କରନ୍ତୁ |

ଲିଙ୍କ୍ ପଠାଉଛି |

ପ୍ରତ୍ୟେକ ଉପଭୋକ୍ତା ଯିଏ ଲିଙ୍କ୍ କ୍ଲିକ୍ କରନ୍ତି ଏବଂ ସାଇନ୍ ଅପ୍ ସମାପ୍ତ କରନ୍ତି, କେସ୍ ପୃଷ୍ଠାରେ ଏକ ନୂତନ କେସ୍ ସୃଷ୍ଟି କରିବାକୁ ପ୍ଲାଟଫର୍ମକୁ ଟ୍ରିଗର କରିଥାଏ ଏବଂ ସେହି କ୍ଷେତ୍ରରେ ଉପଭୋକ୍ତାଙ୍କୁ ଅଟୋ-ନ୍ୟସ୍ତ କରିଥାଏ |
