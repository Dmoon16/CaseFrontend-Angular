用户如何创建新帐户

管理员将用户添加到个案后： * 如果用户是个案新手，则用户必须完成工作流程以创建帐户并接受访问个案的邀请。 * 如果用户有现有的案例帐户，则用户必须接受邀请才能访问案例。

本文介绍了用户创建帐户必须完成的步骤。

入门

可以通过两种方式将用户添加到案例应用程序

  • 通过管理员：如果管理员通过平台 UI或导入工具创建用户，则管理员必须向用户发送指向 account.caseactive.com 的链接以继续登录。
  • 按用户：如果用户使用注册链接创建了一个帐户，用户将被自动重定向到 account.caseactive.com 以继续登录。

在登录

当用户首次登录其 CaseActive 帐户时，平台会提示用户：

  • 打开他们的电子邮件帐户，然后复制/粘贴六位数的安全代码
  • 完成帐户资料

图片

用户单击更新后，将显示应用程序页面。该页面显示应用程序名称（公司名称）、用户类型（角色）、状态和邀请按钮。

图片

要访问应用程序，用户必须接受邀请。用户应该：

  1. 单击邀请按钮。
  2. 查看权限。
  3. 单击“通知”选项卡以查看和编辑（可选）条款。
  4. 单击批准访问。

图片

当用户同意后，平台会在用户页面显示一个APP按钮。此按钮允许用户访问分配的案例。

要编辑用户设置，用户单击省略号。

图片

用户可以： * 删除应用访问权限 * 修改通知设置

图片

用户完成工作流程后，用户名会出现在管理员的用户列表中。该列表显示所有用户，无论是否登录。
