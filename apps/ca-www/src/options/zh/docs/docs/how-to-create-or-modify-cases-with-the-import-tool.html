如何使用导入工具创建或修改案例

本文向管理员展示如何使用导入工具创建或修改现有案例。

打开导入工具

  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 在管理面板上，单击集成选项卡，然后单击子菜单上的导入。
  4. 单击新建按钮。添加导入对话框显示在新的浏览器选项卡中。

图片

创建案例

添加导入对话框打开后：

  1. 单击类型字段并选择一个选项（见下文）。显示一个链接。
  2. 单击下载示例链接。 CSV 文件将下载到您的设备。

接下来，打开文件并添加案例。

类型

单击“类型”字段时，将显示一个下拉菜单。选择批量案例导入的两个选项之一。

  • 注意下拉菜单有很多选项，但只有两个与管理案例相关。每个选项都有一个不同的示例 CSV 文件。

添加案例

  • 选择此选项以简单地批量上传个案。
  • 单击下载示例链接打开一个 CSV 文件，其中包含创建案例导入类型的列标题。

修改案例

  • 选择此选项以更新当前案例。
  • 单击下载示例链接打开一个 CSV 文件，其中包含修改案例导入类型的列标题。

上传文件

您可以立即上传文件或稍后返回添加导入对话框并完成这些步骤。

  1. 打开添加导入对话框。
  2. 输入 CSV 文件的唯一名称。
  3. 单击加号 (+) 图标从您的设备上传文件。
  4. 单击类型字段并选择一个选项。 （看上面）
  5. 单击保存。

格式化

这些是必填字段和首要要求。单击此处查看格式详细信息。

tag_id（或 case_ID）唯一标识符（使用字母数字字符，接受连字符）

case_status 0 = 关闭 1 = 活动

date_opened月/日/年 mm/dd/yyyy

关于案例的独特描述（使用字母数字字符，接受连字符）

加工

根据文件中记录的数量和系统负载，平台可能需要几分钟时间来处理导入。

图片

导入完成后，平台会在管理员的案例仪表板上调整案例记录（即添加或修改）。

失败

如果导入操作失败，平台将返回失败消息。在大多数情况下，失败的原因是数据格式化。

图片

查看失败原因：

  1. 单击操作标题下的查看。
  2. 在下一页上，将鼠标悬停在“失败”图标上。工具提示显示错误。
  3. 转到 CVS 文件并调整并修复错误。
  4. 再次上传文件。
