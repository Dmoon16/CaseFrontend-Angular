如何在创建案例时分配用户

本文向管理员展示了如何在创建新案例时分配用户。如果用户尚未在平台上注册，您可以在创建新案例时创建添加新用户。

创建案例

  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 单击个案菜单选项卡。
  4. 单击新建按钮。将显示一个新窗口。

图片

  • 填写字段以创建案例，然后向下滚动到分配的用户标题。

分配用户

当前用户

按照以下步骤将当前用户分配给新案例。

单击分配用户按钮并填写字段。

  1. 单击用户下拉菜单并选择一个标识符（例如电子邮件）。
  2. 在空白字段中输入一个值（例如电子邮件地址）。
  3. 如果用户记录存在于平台中，该字段会自动建议一个名称。
  4. 选择正确的用户。
  5. 选择用户角色（客户、默认或员工）。
  6. 或者，选中该框以向用户发送电子邮件通知。
  7. 单击保存。

图片

Case Active 将用户添加到案例中。现在管理员可以单击“保存”关闭窗口，或添加新用户或删除用户。

添加新用户

如果平台找不到当前记录，点击按钮添加新用户。

图片

填写字段，然后单击保存以将用户添加到平台。接下来，设置用户角色并再次单击“保存”以将用户分配给案例。最后，再次单击 SAVE 以保存案例。
