Felhasználói eset-hozzárendelések adatainak formázása az importáló eszközzel

Az alábbiakban a CSV-fájlok adatmezőinek listája látható. Minden importálási opcióhoz más-más mezőkombináció tartozik.

case_ID

  • Adja meg a felhasználói vagy címkeazonosítót.
  • Lépjen az Esetek oldalra > Szerkesztés > Címkeazonosító VAGY Lépjen az Esetek oldalra > Szerkesztés > Esetazonosító az URL-ben.

Felhasználói azonosító

  • Adja meg a felhasználói azonosítót vagy a címkeazonosítót.
  • Lépjen a Felhasználók oldalra > Szerkesztés > Címkeazonosító VAGY Lépjen a Felhasználók oldalra > Szerkesztés > Felhasználói azonosító az URL-ben.

ügy_szerep_sorrendje

  • Hagyja üresen a cellát.

case_role_ID

  • Írja be: ügyfelek, személyzet vagy alapértelmezett.

case_notify

  • 0 = nem
  • 1 = Igen, e-mail értesítés küldése a felhasználónak.
