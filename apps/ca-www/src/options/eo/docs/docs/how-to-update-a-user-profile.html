Kiel Ĝisdatigi Uzantan Profilon

Ĉi tiu artikolo montras al uzantoj kiel ĝisdatigi informojn sur ilia Profila paĝo.

  • Uzantoj povas ĝisdatigi siajn proprajn profilojn.
  • Administrantoj povas ĝisdatigi la Profilan paĝon de iu ajn uzanto.

Por uzantoj

La unua paŝo estas malfermi la paĝon de Profilo. Uzantoj kaj administrantoj sekvas ĉi tiujn paŝojn por ĝisdatigi sian propran Profilpaĝon.

  1. Ensalutu al via Case Active-konto.
  2. Alklaku la langeton Profilo ĉe la supro de la paĝo.
  3. Faru ŝanĝojn.
  4. Klaku Konservi.

Plej multaj kampoj estas laŭvolaj, kaj kelkaj kampoj estas postulataj. La postulataj kampoj estas:

  • Antaŭnomo
  • Familia nomo
  • Retpoŝta adreso
  • Lingvo
  • Horzono

Por administrantoj

Administrantoj ankaŭ povas ĝisdatigi la profilinformojn por aliaj uzantoj. Ĉi tio inkluzivas tri kampojn sub la rubriko HOST USER INFO.

  1. Ensalutu al via administra konto de Case Active.
  2. Trovu aplikaĵon kaj alklaku la butonon ADMIN.
  3. Alklaku Uzantoj en la supra menuo.
  4. Iru al la submenuo kaj elektu la uzantliston, kiun vi volas aliri (AKTIVA aŭ MALPITA). La defaŭlta agordo estas Aktiva.
  5. Trovu uzanton en la listo kaj alklaku Redakti.
  6. Faru ŝanĝojn laŭbezone. (Vidu sube)
  7. Klaku Konservi.

Ŝanĝi uzantajn informojn

Administrantoj povas ŝanĝi uzantinformojn.

  1. Agordu la Uzantan Tipon (Uzanto aŭ Administranto).
  2. Agordu la Uzantan Statuson (Aktiva al Malŝaltita).
  3. Enigu unikan Etikedan ID. (laŭvola)
  4. Klaku Konservi.
