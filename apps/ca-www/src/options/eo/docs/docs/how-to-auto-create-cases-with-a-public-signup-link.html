Kiel Aŭtomate Krei Kazojn kun Publika Subligo

Ĉi tiu artikolo montras al administrantoj kiel krei kazon (kaj aldoni uzanton samtempe) sendante ununuran subskriban ligon.

Lokante la ligilon

  1. Ensalutu al la administra konto.
  2. Trovu aplikaĵon sur la paĝo de Aplikoj kaj alklaku la butonon ADMIN.
  3. Sur la Admin panelo, alklaku la Agordoj menuo langeto.
  4. Certiĝu, ke la submenua langeto SYSTEM SETUP estas elektita.

Agordante agordojn

Rulumu malsupren al la titolo de UZANTO.

Certigu, ke la kampo Permesi Subskribon estas ŝaltita. (postulata) Kiam ĉi tiu funkcio estas ebligita, Case Active generas unikan ligon.

  1. Alklaku la kampon Aŭtomate Krei Kazon kaj elektu On. La kampo de Kaza Uzanto Rolo montriĝas.
  2. Elektu rolon. (Vidu sube)

Kazaj uzantroloj

Estas tri kazaj uzantroloj.

  • klientoj (rekomenditaj)
  • defaŭlte
  • personaro

Sendante la ligilon

Kunhavigu la ligilon kun unu aŭ pluraj homoj. Ĉiu uzanto, kiu klakas la ligilon kaj kompletigas aliĝon, ekigas la platformon por generi novan kazon sur la paĝo de kazoj kaj aŭtomate atribui la uzanton en la kazo.
