Hoe beheerders gebruikers beheren

Beheerders maken, wijzigen of deactiveren wie toegang heeft tot apps en cases. Wanneer een gebruiker zich voor de eerste keer aanmeldt bij het platform, moet hij een profielworkflow voltooien.

  • Klik hier voor meer informatie over de nieuwe workflow voor gebruikersprofielen.

Een gebruiker beheren

De onderstaande tabel toont vijf manieren om een gebruiker te beheren. Klik op een koppeling voor meer informatie.

Methode Type Acties Details
Gebruikersinterface gebruiken gebruikersinterface toevoegen & wijzigen Beheerknop > Gebruikers
Tijdens het maken van een case en wijs gebruiker toe aan case gebruikersinterface toevoegen Knop Beheerder > Aanvragen > Knop Nieuw > Pagina Aanvraag maken
Maak automatisch een gebruiker aan met behulp van een openbare aanmeldingslink Actie van de gebruiker toevoegen Admin-knop > Instellingen > Systeeminstellingen pagina
Importeer hulpmiddel gebruikersinterface toevoegen & wijzigen Beheerknop > Integraties > Importeren
API gebruiken (binnenkort beschikbaar) API toevoegen & wijzigen NVT

Meldingen

Het platform stuurt gebruikers geen melding wanneer ze aan een app worden toegevoegd. Beheerders kunnen echter meldingen voor casetoewijzingen inschakelen.
