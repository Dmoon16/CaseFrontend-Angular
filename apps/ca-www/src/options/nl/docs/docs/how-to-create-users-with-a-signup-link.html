Gebruikers aanmaken met een aanmeldingslink

Dit artikel laat beheerders zien hoe ze gebruikers kunnen maken door een aanmeldingslink te sturen.

De aanmeldingslink configureren

  1. Klik op de link Stuur ze een aanmelding onder aan het dialoogvenster Gebruiker toevoegen. Er wordt een nieuwe pagina weergegeven.
  2. Configureer de aanmeldingsopties. (zie onder)

img

Zorg ervoor dat het veld Aanmelden toestaan is ingesteld op Aan. (vereist) Indien ingeschakeld, genereert Case Active een unieke link.

Een zaak aanmaken

Optioneel kunt u voor elke nieuwe gebruiker automatisch een case aanmaken en een gebruikersrol toewijzen.

  1. Klik op het veld Auto Create Case en selecteer Aan. De velden Casegebruikersrol worden weergegeven.
  2. Klik op de Casegebruikersrol en selecteer een optie. (zie onder)

img

Case gebruikersrollen

Er zijn drie casusgebruikersrollen.

  • klanten
  • standaard
  • personeel

De link verzenden

Deel de link met een of meer mensen. Elke persoon klikt op de link en voltooit de workflow om zijn Case Active-account te activeren. Zodra de gebruiker de aanmeldingsworkflow heeft voltooid, voegt Case Active de gebruiker toe aan de gebruikerslijst.

img

Volgende stappen

Case Active stuurt de gebruiker een e-mailmelding met de aanmeldingslink. De gebruiker klikt op de link en voltooit de workflow voor nieuwe aanmeldingen.
