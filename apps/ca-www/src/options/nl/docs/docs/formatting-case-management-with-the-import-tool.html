Casemanagementgegevens opmaken met de importtool

Hieronder vindt u een lijst met verplichte velden in CSV-bestanden om cases te beheren met de importtool.

gebruikersnaam

  • Voer het e-mailadres van de gebruiker in.

tag_id (een id waarnaar u later kunt verwijzen)

  • een unieke identificatiecode (gebruik alfanumerieke tekens, koppeltekens zijn toegestaan)

case_status

  • 0 = gesloten
  • 1 = actief

datum_geopend

  • maand dag jaar
  • mm/dd/jjjj

wat betreft

  • een unieke beschrijving van de zaak (gebruik alfanumerieke tekens, koppeltekens zijn toegestaan)
