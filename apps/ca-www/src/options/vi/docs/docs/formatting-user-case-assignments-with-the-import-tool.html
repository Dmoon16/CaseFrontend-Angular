Định dạng Dữ liệu Chỉ định Trường hợp Người dùng bằng Công cụ Nhập

Dưới đây là danh sách các trường dữ liệu trong tệp CSV. Mỗi tùy chọn nhập có một tổ hợp các trường khác nhau.

case_ID

  • Nhập trường hợp Người dùng hoặc ID thẻ.
  • Chuyển đến trang Trường hợp > Chỉnh sửa > ID thẻ HOẶC Chuyển đến trang Trường hợp > Chỉnh sửa > ID trường hợp trong url.

tên người dùng

  • Nhập ID người dùng hoặc ID thẻ.
  • Chuyển đến trang Người dùng > Chỉnh sửa > ID thẻ HOẶC Chuyển đến trang Người dùng > Chỉnh sửa > ID người dùng trong url.

case_role_order

  • Để trống ô.

case_role_ID

  • Nhập: khách hàng, nhân viên hoặc mặc định.

case_notify

  • 0 = không
  • 1 = Có, gửi thông báo qua email cho người dùng.
