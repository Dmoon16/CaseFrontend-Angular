Yöneticiler Vakalara Kullanıcı Atamalarını Nasıl Yönetir?

Kullanıcıları ve servis talebi atamalarını yönetmenin beş yolu vardır. Nasıl yapılacağını öğrenmek için bir bağlantıya tıklayın.

Yöntem Tip Hareketler Detaylar
Bir vaka oluştururken kullanıcı arayüzü Ekle Yönetici düğmesi > Destek Talepleri > Yeni düğmesi > Kullanıcı Ata düğmesi
Herkese açık kayıt bağlantısını kullanarak kullanıcıyı otomatik ata Kullanıcı İşlemi Ekle Yönetici düğmesi > Ayarlar > Sistem Kurulumu sayfası
Mevcut vakada kullanıcı arayüzünü kullanma kullanıcı arayüzü ekle Kaldır Yönetici düğmesi > Servis Talepleri > Kullanıcı Ata
Aracı içe aktar kullanıcı arayüzü ekle, değiştir ve kaldır Yönetici düğmesi > Entegrasyonlar > İçe Aktarımlar
API Kullanma (Çok Yakında) API ekle, değiştir ve kaldır Yok
