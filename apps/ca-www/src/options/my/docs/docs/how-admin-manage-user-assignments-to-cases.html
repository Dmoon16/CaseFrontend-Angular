အက်ဒ်မင်များသည် အမှုကိစ္စများအတွက် အသုံးပြုသူ၏ တာဝန်များကို စီမံခန့်ခွဲပုံ

သုံးစွဲသူများနှင့် အမှုတွဲတာဝန်များကို စီမံခန့်ခွဲရန် နည်းလမ်းငါးခုရှိသည်။ လုပ်နည်းကိုလေ့လာရန် လင့်ခ်ကိုနှိပ်ပါ။

နည်းလမ်း ရိုက်ပါ။ လုပ်ဆောင်ချက်များ အသေးစိတ်
အမှုတစ်ခုဖန်တီးနေစဉ် UI ထည့်ပါ။ စီမံခန့်ခွဲသူ ခလုတ် > Cases > New ခလုတ် > အသုံးပြုသူများကို သတ်မှတ်ပေးသည့် ခလုတ်
အများသူငှာ အကောင့်ဖွင့်ခြင်းလင့်ခ်ကို အသုံးပြု၍ သုံးစွဲသူကို အလိုအလျောက် သတ်မှတ်ပေးသည်။ အသုံးပြုသူလုပ်ဆောင်ချက် ထည့်ပါ။ စီမံခန့်ခွဲသူခလုတ် > ဆက်တင်များ > စနစ်ထည့်သွင်းမှု စာမျက်နှာ
ရှိပြီးသားကိစ္စတွင် UI ကိုအသုံးပြုခြင်း။ UI ထည့် & ဖယ်ရှားပါ။ စီမံခန့်ခွဲသူ ခလုတ် > ဖြစ်ရပ်များ > အသုံးပြုသူများကို ခန့်အပ်ပါ။
သွင်းကုန်တူးလ် UI ထည့်၊ ပြင်ဆင်၊ ဖယ်ရှားပါ။ စီမံခန့်ခွဲရေးခလုတ် > ပေါင်းစပ်မှုများ > တင်သွင်းမှုများ
