UI ဖြင့် အသုံးပြုသူများကို ဖန်တီးပြီး ပြုပြင်နည်း

ဤဆောင်းပါးတွင် သုံးစွဲသူများအား ပလပ်ဖောင်း၏ အသုံးပြုသူမျက်နှာပြင်ဖြင့် မှတ်တမ်းများကို အပ်ဒိတ်လုပ်နည်း စီမံခန့်ခွဲသူများကို ပြသထားသည်။ အက်ဒ်မင်များသည် သုံးစွဲသူများကို အမှုအသစ်တစ်ခုသို့ ပေါင်းထည့်နိုင်ပြီး ရှိပြီးသားကိစ္စများကို တည်းဖြတ်နိုင်သည်။

အသုံးပြုသူဖန်တီးခြင်း။

  1. စီမံခန့်ခွဲသူအကောင့်သို့ ဝင်ရောက်ပါ။
  2. အက်ပ်များ စာမျက်နှာတွင် အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. Admin panel တွင်၊ Users menu tab ကိုနှိပ်ပါ။
  4. NEW ခလုတ်ကိုနှိပ်ပါ။ ဝင်းဒိုးအသစ်တစ်ခုပေါ်လာသည်။

img

ကွက်လပ်များကိုဖြည့်ပြီး SAVE ကိုနှိပ်ပါ။ Case Active သည် ဤအက်ပ်နှင့်ဆက်စပ်နေသော အသုံးပြုသူများစာရင်းသို့ အမည်ကို ပေါင်းထည့်သည်။

အသုံးပြုသူတစ်ဦးကို ပြုပြင်မွမ်းမံခြင်း။

အသုံးပြုသူမှတ်တမ်းအနည်းငယ်ကို အပ်ဒိတ်လုပ်ရန်၊ တည်းဖြတ်ကိရိယာကို အသုံးပြုပြီး သုံးစွဲသူတစ်ဦးစီအတွက် အပြောင်းအလဲများ ပြုလုပ်ပါ။

  • မှတ်ချက် တစ်ထိုင်တည်းတွင် အသုံးပြုသူ မှတ်တမ်းအများအပြားကို မွမ်းမံပြင်ဆင်ရန်၊ သွင်းကုန်တူး လ်ကို အသုံးပြု၍ စဉ်းစားပါ။
  1. Admin panel တွင်၊ Users menu tab ကိုနှိပ်ပါ။
  2. အသုံးပြုသူတစ်ဦးကို ရှာဖွေပြီး တည်းဖြတ်မည်ကို နှိပ်ပါ။
  3. အပြောင်းအလဲများပြုလုပ်ပြီး SAVE ကိုနှိပ်ပါ။

img

နောက်တစ်ဆင့်

Case Active သည် အသုံးပြုသူကို အီးမေးလ်အကြောင်းကြားစာ မပို့ပါ။ အက်ဒ်မင်များသည် အသုံးပြုသူကို ဆက်သွယ်ပြီး Case Active ပင်မစာမျက်နှာသို့ လင့်ခ်တစ်ခု ပေးပို့သင့်သည်။ ထို့နောက် အသုံးပြုသူသည် အကောင့်ဝင်ရန် ခလုတ်ကို နှိပ်ပြီး အကောင့်အသစ်များအတွက် အလုပ်အသွားအလာကို အပြီးသတ်သည်။

  • ဤ ဆောင်းပါး သည် သုံးစွဲသူအသစ်များအတွက် အကောင့်ဖွင့်ခြင်းလုပ်ငန်းစဉ်ကို ဖော်ပြသည်။
