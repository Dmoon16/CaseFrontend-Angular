Conas Úsáideoirí a Chruthú le Nasc Cláraigh

Léiríonn an t-alt seo do riarthóirí conas úsáideoirí a chruthú trí nasc sínithe a sheoladh.

Cumrú an nasc clárúcháin

  1. Cliceáil ar an Seol nasc sínithe chucu ag bun an bhosca dialóige Cuir úsáideoir leis. Taispeánann leathanach nua.
  2. Cumraigh na roghanna clárúcháin. (Féach thíos)

img

Cinntigh go bhfuil an réimse Ceadaigh Cláraigh chun socrú ar siúl. (riachtanach) Nuair atá sé cumasaithe, gineann Case Active nasc uathúil.

Déan athbhreithniú ar na roghanna thíos agus ansin cliceáil ar an deilbhín chun an nasc a chóipeáil. Is féidir leat an nasc a chur ar do shuíomh Gréasáin nó an nasc a sheoladh chuig duine amháin nó níos mó trí ríomhphost nó trí aon chainéal cumarsáide.

img

Cás a chruthú

Roghnach, is féidir leat cás a chruthú go huathoibríoch do gach úsáideoir nua agus ról úsáideora a shannadh.

  1. Cliceáil ar an Auto Cruthaigh Cás réimse agus roghnaigh On. Taispeánann na réimsí Ról Úsáideora Cásanna.
  2. Cliceáil ar an Ról Úsáideora Cás agus roghnaigh rogha. (Féach thíos)

img

Róil úsáideora cásanna

Tá trí ról úsáideora cásanna ann.

  • cliaint
  • réamhshocraithe
  • foirne

Ag seoladh an nasc

Roinn an nasc le duine amháin nó níos mó. Cliceáil ar an nasc agus críochnaíonn gach duine an sreabhadh oibre chun a gcuntas Case Active a ghníomhachtú. Nuair a chríochnaíonn an t-úsáideoir an sreabhadh oibre sínithe, cuireann Case Active an t-úsáideoir leis an liosta Úsáideoirí.

Na chéad chéimeanna eile

Seolann Case Active fógra ríomhphoist chuig an úsáideoir leis an nasc sínithe. Déanann an t-úsáideoir cliceáil ar an nasc agus críochnaíonn sé an sreabhadh oibre le haghaidh sínithe nua.

  • Déanann an t- alt seo cur síos ar an sreabhadh oibre sínithe d'úsáideoirí nua.
