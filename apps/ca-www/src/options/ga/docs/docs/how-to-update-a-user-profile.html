Conas Próifíl Úsáideora a Nuashonrú

Léiríonn an t-alt seo d'úsáideoirí conas faisnéis a nuashonrú ar a leathanach Próifíl.

  • Is féidir le húsáideoirí a bpróifílí féin a nuashonrú.
  • Is féidir le riarthóirí leathanach Próifíl aon úsáideora a nuashonrú.

D'úsáideoirí

Is é an chéad chéim ná an leathanach Próifíl a oscailt. Leanann úsáideoirí agus riarthóirí na céimeanna seo chun a leathanach Próifíl féin a nuashonrú.

  1. Logáil isteach i do chuntas Case Active.
  2. Cliceáil ar an táb Próifíl ag barr an leathanaigh.
  3. Déan athruithe.
  4. Cliceáil Sábháil.

Tá formhór na réimsí roghnach, agus tá gá le roinnt réimsí. Is iad na réimsí riachtanacha:

  • Ainm
  • Sloinne
  • Seoladh ríomhphoist
  • Teanga
  • Crios ama

Do riarthóirí

Is féidir le riarthóirí an fhaisnéis phróifíle d'úsáideoirí eile a nuashonrú freisin. Áirítear leis seo trí réimse faoin gceannteideal FAISNÉIS ÍOSTA ÚSÁIDEORA.

  1. Logáil isteach i do chuntas riaracháin Case Active.
  2. Aimsigh app agus cliceáil ar an gcnaipe RIARACHÁIN.
  3. Cliceáil Úsáideoirí ar an roghchlár barr.
  4. Téigh go dtí an fo-roghchlár agus roghnaigh an liosta úsáideoirí is mian leat a rochtain (GNÍOMHACH nó MÍCHUMAIS). Is é Gníomhach an socrú réamhshocraithe.
  5. Aimsigh úsáideoir ar an liosta agus cliceáil Edit.
  6. Déan athruithe de réir mar is gá. (Féach thíos)
  7. Cliceáil Sábháil.

Athraigh faisnéis úsáideora

Is féidir le riarthóirí faisnéis úsáideora a athrú.

  1. Socraigh an Cineál Úsáideora (Úsáideoir nó Bainisteoir).
  2. Socraigh an Stádas Úsáideora (Gníomhach do Dhaoine faoi Mhíchumas).
  3. Cuir isteach ID Clibe uathúil. (roghnach)
  4. Cliceáil Sábháil.
