Conas Cásanna a Chruthú go Uathoibríoch le Nasc Cláraithe Poiblí

Léiríonn an t-alt seo do riarthóirí conas cás a chruthú (agus úsáideoir a chur leis ag an am céanna) trí nasc sínithe amháin a sheoladh.

An nasc a aimsiú

  1. Logáil isteach sa chuntas riaracháin.
  2. Aimsigh aip ar an leathanach Apps agus cliceáil ar an gcnaipe RIARACHÁIN.
  3. Ar an bpainéal Riaracháin, cliceáil ar an roghchlár Socruithe.
  4. Cinntigh go bhfuil an cluaisín fo-roghchlár SYSTEM SETUP roghnaithe.

img

Socruithe á chumrú

Scrollaigh síos go dtí an ceannteideal USER SIGNUP.

Cinntigh go bhfuil an réimse Ceadaigh Cláraigh socraithe go Ar. (riachtanach) Nuair atá an ghné seo cumasaithe, gineann Case Active nasc uathúil.

  1. Cliceáil ar an Auto Cruthaigh Cás réimse agus roghnaigh On. Taispeánann an réimse Ról Úsáideora Cásanna.
  2. Roghnaigh ról. (Féach thíos)

img

Róil úsáideora cásanna

Tá trí ról úsáideora cásanna ann.

  • cliaint (molta)
  • réamhshocraithe
  • foirne

Ag seoladh an nasc

Roinn an nasc le duine amháin nó níos mó. Spreagann gach úsáideoir a chliceálann ar an nasc agus a chríochnaíonn an síniú an t-ardán chun cás nua a ghiniúint ar leathanach na gcásanna agus sannfaidh sé an t-úsáideoir sa chás go huathoibríoch.
