Hoe om 'n gebruikersprofiel op te dateer

Hierdie artikel wys gebruikers hoe om inligting op hul profielbladsy op te dateer.

  • Gebruikers kan hul eie profiele opdateer.
  • Administrateurs kan die profielbladsy van enige gebruiker opdateer.

Vir gebruikers

Die eerste stap is om die profielbladsy oop te maak. Gebruikers en administrateurs volg hierdie stappe om hul eie profielbladsy op te dateer.

  1. Meld aan by jou Case Active-rekening.
  2. Klik op die Profiel-oortjie bo-aan die bladsy.
  3. Maak veranderinge.
  4. Klik op Stoor.

Die meeste velde is opsioneel, en sommige velde word vereis. Die vereiste velde is:

  • Eerste naam
  • Van
  • E-pos adres
  • Taal
  • Tydsone

Vir admins

Administrateurs kan ook die profielinligting vir ander gebruikers opdateer. Dit sluit drie velde onder die HOST USER INFO-opskrif in.

  1. Meld aan by jou Case Active admin rekening.
  2. Soek 'n toepassing en klik op die ADMIN-knoppie.
  3. Klik op Gebruikers op die boonste kieslys.
  4. Gaan na die subkieslys en kies die gebruikerslys waartoe jy toegang wil hÃª (AKTIEF of GEDEAKTIVEERD). Die verstekinstelling is aktief.
  5. Soek 'n gebruiker op die lys en klik Wysig.
  6. Maak veranderinge soos vereis. (sien onder)
  7. Klik op Stoor.

Verander gebruikerinligting

Administrateurs kan gebruikersinligting verander.

  1. Stel die gebruikertipe (Gebruiker of Bestuurder).
  2. Stel die gebruikerstatus (aktief na gedeaktiveer).
  3. Voer 'n unieke merker-ID in. (opsioneel)
  4. Klik op Stoor.
