Како администраторите управуваат со случаи

Постојат четири начини за управување со случај. Кликнете на врската за да дознаете како.

Метод Тип Акции Детали
Користење на интерфејс UI додадете и измени Администраторско копче > Случаи
Автоматско креирање футрола користејќи јавна врска за регистрација Корисничка акција додадете Администраторско копче > Поставки > Страница за поставување систем
Алатка за увоз UI додадете и измени Администраторско копче > Интеграции > Увоз
Користење на API (наскоро) API додадете и измени N/A
