Форматирање податоци за задачи на кориснички случаи со алатката за увоз

Подолу е листа на полиња со податоци во CSV-датотеките. Секоја опција за увоз има различна комбинација на полиња.

случај_ID

  • Внесете го случајот Корисник или ID на ознака.
  • Одете на страницата „Случаи“ > Уреди > ИД на ознака ИЛИ Одете на страницата „Случаи“ > Уреди > ИД на случај во URL-то.

user_ID

  • Внесете ID на корисникот или ID на ознака.
  • Одете на страницата со корисници > Уреди > ИД на ознака ИЛИ Одете на страницата со корисници > Уреди > Кориснички ID во URL-то.

случај_улога_ред

  • Оставете ја ќелијата празна.

ИД на улогата на случајот

  • Внесете: клиенти, персонал или стандардно.

случај_извести

  • 0 = бр
  • 1 = Да, испратете му е-пошта известување на корисникот.
