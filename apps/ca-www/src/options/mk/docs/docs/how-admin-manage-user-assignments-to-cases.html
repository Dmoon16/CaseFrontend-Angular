Како администраторите управуваат со корисничките задачи до случаите

Постојат пет начини за управување со корисниците и задачите на случаи. Кликнете на врската за да дознаете како.

Метод Тип Акции Детали
Додека креирате случај UI додадете Администраторско копче > Случаи > Ново копче > Копче за доделување корисници
Автоматски додели корисник користејќи јавна врска за регистрација Корисничка акција додадете Администраторско копче > Поставки > Страница за поставување систем
Користење на интерфејс на постоечки случај UI додадете и отстранете Администраторско копче > Случаи > Доделување корисници
Алатка за увоз UI додадете, изменете и отстраните Администраторско копче > Интеграции > Увоз
Користење на API (наскоро) API додадете, изменете и отстраните N/A
