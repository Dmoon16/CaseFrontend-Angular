Форматирање податоци за управување со случаи со алатката за увоз

Подолу е листа на потребни полиња во CSV-датотеките за управување со случаи со алатката Увоз.

корисничко име

  • Внесете ја адресата на е-пошта на корисникот.

tag_id (ид кој може да го упатите подоцна)

  • единствен идентификатор (користете алфанумерички знаци, цртичките се прифаќаат)

случај_статус

  • 0 = затворено
  • 1 = активен

датум_отворен

  • месец/ден/година
  • мм/дд/гггг

за

  • уникатен опис на случајот (користете алфанумерички знаци, цртички се прифаќаат)
