باشقۇرغۇچى خادىملارنىڭ ۋەزىپىلەرنى قانداق باشقۇرىدۇ

ئىشلەتكۈچىلەر ۋە دېلو تاپشۇرۇقلىرىنى باشقۇرۇشنىڭ بەش خىل ئۇسۇلى بار. قانداق قىلىش ئۈچۈن ئۇلىنىشنى چېكىڭ.
