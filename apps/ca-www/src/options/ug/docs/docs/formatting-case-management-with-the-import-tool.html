ئىمپورت قوراللىرى بىلەن دېلو باشقۇرۇش سانلىق مەلۇماتلىرىنى فورماتلاش

تۆۋەندىكىسى CSV ھۆججىتىدىكى ئىمپورت قوراللىرى بىلەن دېلولارنى باشقۇرۇش ئۈچۈن لازىملىق بۆلەكلەرنىڭ تىزىملىكى.

ئىشلەتكۈچى ئىسمى

  • ئىشلەتكۈچىنىڭ ئېلېكترونلۇق خەت ئادرېسىنى كىرگۈزۈڭ.

tag_id (كېيىن پايدىلىنالايدىغان id)

  • ئۆزگىچە پەرقلىگۈچ (ھەرپ-بەلگىلەرنى ئىشلىتىڭ ، سىزىقچىلار قوبۇل قىلىنىدۇ)

case_status

  • 0 = تاقالدى
  • 1 = ئاكتىپ

date_opened

  • ئاي / كۈن / يىل
  • mm / dd / yyyy

ھەققىدە

  • بۇ ئەھۋالنىڭ ئۆزگىچە تەسۋىرى (ھەرپ-بەلگىلەرنى ئىشلىتىڭ ، سىزىقچىلار قوبۇل قىلىنىدۇ)
