ئاممىۋى تىزىملاش ئۇلىنىشى بىلەن دېلولارنى قانداق قۇرۇش كېرەك

بۇ ماقالىدە بىر تىزىمغا كىرىش ئۇلانمىسى ئەۋەتىش ئارقىلىق باشقۇرغۇچىلار قانداق قىلىپ دېلو تۇرغۇزۇش (ۋە بىرلا ۋاقىتتا ئىشلەتكۈچى قوشۇش) كۆرسىتىلدى.

ئۇلىنىشنى تېپىش

  1. باشقۇرغۇچى ھېساباتىغا كىرىڭ.
  2. ئەپلەر بېتىدىن بىر ئەپنى تېپىپ ، ADMIN كۇنۇپكىسىنى بېسىڭ.
  3. باشقۇرۇش تاختىسىدا «تەڭشەك تىزىملىكى» بەتكۈچىنى چېكىڭ.
  4. SYSTEM SETUP تارماق تىزىملىك بەتكۈچىنىڭ تاللانغانلىقىنى جەزملەشتۈرۈڭ.

img

تەڭشەكلەرنى تەڭشەش

USER SIGNUP ماۋزۇسىغا تۆۋەنگە يۆتكىلۈڭ.

تىزىملىتىشقا رۇخسەت قىلىش مەيدانىنىڭ تەڭشەلگەنلىكىنى جەزملەشتۈرۈڭ. (تەلەپ) بۇ ئىقتىدار قوزغىتىلغاندا ، Case Active ئۆزگىچە ئۇلىنىش ھاسىل قىلىدۇ.

  1. «ئاپتوماتىك قۇرۇش دېلوسى» نى چېكىپ «On» نى تاللاڭ. Case ئىشلەتكۈچى رول مەيدانى كۆرسىتىلىدۇ.
  2. رول تاللاڭ. (تۆۋەنگە قاراڭ)

ئىشلەتكۈچىنىڭ رولى

ئىشلەتكۈچىنىڭ ئۈچ خىل رولى بار.

  • خېرىدارلار (تەۋسىيە قىلىنغان)
  • كۆڭۈلدىكى
  • خىزمەتچىلەر

ئۇلانمىنى ئەۋەتىش

ئۇلىنىشنى بىر ياكى بىر قانچە ئادەم بىلەن ئورتاقلىشىڭ. ئۇلىنىشنى چېكىپ تىزىملىتىشنى تاماملىغان ھەر بىر ئىشلەتكۈچى سۇپىنى قوزغىتىپ ، دېلو بېتىدە يېڭى ئەھۋال پەيدا قىلىدۇ ۋە بۇ ئەھۋالدا ئىشلەتكۈچىنى ئاپتوماتىك تەقسىملەيدۇ.
