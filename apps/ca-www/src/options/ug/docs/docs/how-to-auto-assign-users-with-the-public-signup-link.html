ئاممىۋى تىزىملاش ئۇلىنىشى ئارقىلىق ئابونتلارنى ئاپتوماتىك تەقسىملەش ئۇسۇلى

باشقۇرغۇچى سىستېما تەڭشەكلىرىنى قانداق تەڭشەشنى ئۆگىنىدۇ ، بۇنداق بولغاندا يېڭى ئىشلەتكۈچىلەر ئاپتوماتىك ھالدا يېڭىدىن ياسالغان دېلوغا تەقسىملىنىدۇ.

تەڭشەش بېتىنى ئېچىش

بىرىنچى قەدەم: سىستېما تەڭشەش بېتىنى ئېچىش.

  1. باشقۇرغۇچى ھېساباتىغا كىرىڭ.
  2. ئەپلەر بېتىدىن بىر ئەپنى تېپىپ ، ADMIN كۇنۇپكىسىنى بېسىڭ.
  3. سىستېما بەتكۈچىنى چېكىڭ ، ئاندىن تارماق تىزىملىكتىكى SYSTEM SETUP نى بېسىڭ.
  4. USER SIGNUP ماۋزۇسىغا تۆۋەنگە يۆتكىلۈڭ.

سىستېما تەڭشىكىنى تەڭشەش

  1. تىزىملىتىپ كىرىشكە يول قويۇڭ ۋە ئاپتوماتىك قۇرۇش دېلو تىزىملىكىنى قوزغىتىڭ.
  2. رول تىزىملىكىنى چېكىپ بىر تاللاشنى تاللاڭ (سۈكۈتتىكى ھالەتتە تەۋسىيە قىلىنىدۇ).
  3. ئاممىۋى ئىشلەتكۈچى تىزىملاش ئۇلانمىسىنى كۆچۈرۈپ ھەمبەھىرلەڭ.

ئۇلانمىنى ئەۋەتىش

ئۇلىنىشنى چېكىپ تىزىملىتىشنى تاماملىغان ھەر بىر ئىشلەتكۈچى سۇپىنى قوزغىتىپ ، دېلو بېتىدە يېڭى ئەھۋال ھاسىل قىلىدۇ ۋە بۇ ئەھۋالدا ئىشلەتكۈچىنى ئاپتوماتىك تەقسىملەيدۇ.
