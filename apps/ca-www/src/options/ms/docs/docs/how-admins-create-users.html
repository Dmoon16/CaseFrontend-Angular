Cara Pentadbir Mencipta Pengguna

Pentadbir membuat, mengubah suai atau melumpuhkan orang yang boleh mengakses apl dan kes. Apabila pengguna log masuk ke platform buat kali pertama, mereka perlu melengkapkan aliran kerja profil.

  • Klik di sini untuk mengetahui lebih lanjut tentang aliran kerja profil pengguna baharu.

Mencipta pengguna

Jadual di bawah menunjukkan lima cara untuk mencipta pengguna. Klik pautan untuk mengetahui caranya.

| Kaedah | Taip | Tindakan | Butiran | Pengguna baharu menerima pemberitahuan? |
|-------------------------|----------------|------- ---------------------|--------------------|---------------------- ------------------------|

Menggunakan UI | UI | tambah & ubah suai | Butang pentadbir > Pengguna | Tidak |

Semasa mencipta kes dan menetapkan pengguna kepada kes | UI | tambah | Butang pentadbir > Kes > Butang baharu > Cipta halaman Kes | Ya, jika didayakan |

Auto cipta pengguna menggunakan pautan pendaftaran awam | Tindakan pengguna | tambah | Butang pentadbir > Tetapan > Halaman Persediaan Sistem | Ya, jika auto create case didayakan |

Alat import | UI | tambah & ubah suai | Butang pentadbir > Integrasi > Import | Tidak |

Menggunakan API (akan datang) | API | tambah & ubah suai | na | na |

Pemberitahuan

Platform ini tidak menghantar pemberitahuan kepada pengguna apabila mereka ditambahkan pada aplikasi. Walau bagaimanapun, pentadbir boleh mendayakan pemberitahuan untuk tugasan kes.
