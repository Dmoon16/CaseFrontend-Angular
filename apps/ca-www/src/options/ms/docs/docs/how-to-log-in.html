Cara Log Masuk

Ketahui cara untuk log masuk ke akaun Case Active anda yang sedia ada.

Log masuk

  1. Pergi ke halaman log masuk .
  2. Klik butang LOGIN.
  3. Masukkan emel anda.
  4. Klik SETERUSNYA.

Mengesahkan akaun

  1. Buka tab penyemak imbas baharu dan log masuk ke akaun e-mel anda.
  2. Salin kod enam digit.
  3. Kembali ke tab Case Active.
  4. Tampalkan kod ke dalam medan. Tetingkap baharu dipaparkan.

Case Active memaparkan halaman Apl.

  • Nota Platform memaparkan butang ADMIN di sebelah butang APP untuk pengguna yang mempunyai keistimewaan pentadbir.
