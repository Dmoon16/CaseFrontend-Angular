Cara Menugaskan, Mengubah Suai atau Mengalih Keluar Tugasan Pengguna dengan Alat Import

Artikel ini menunjukkan pentadbir cara mengurus pengguna kes menggunakan alat Import.

Admin boleh:

  • menugaskan pengguna kepada kes
  • ubah suai pengguna yang telah diberikan kepada kes
  • alih keluar pengguna yang diberikan kepada kes

Membuka alat Import

  1. Log masuk ke akaun admin.
  2. Cari apl pada halaman Apl dan klik butang ADMIN.
  3. Klik tab Integrasi dan kemudian IMPORT pada submenu.
  4. Klik butang BARU. Kotak dialog Tambah Import dipaparkan.

Mencipta rekod

Muat turun satu atau lebih fail CSV. Gunakan fail ini untuk menambah, mengubah suai atau mengalih keluar pengguna kes. Selepas kotak dialog Tambah Import dibuka:

  1. Klik medan Jenis.
  2. Tatal melalui menu lungsur dan pilih pilihan (lihat di bawah). Satu pautan dipaparkan.
  3. Klik pautan Muat Turun Contoh. Fail CSV dimuat turun ke peranti anda.

Buka fail dan tambah rekod. Klik di sini untuk mengetahui cara menggunakan data format dalam fail CSV.

Pilihan fail

Tetapkan pengguna kepada kes * Pilih pilihan ini untuk menambah pengguna pada kes. * Biarkan sel case_role_order kosong. * Pergi ke halaman Pengguna dan salin ID Teg untuk orang ini. Jika medan ID Tag kosong, masukkan ID unik, klik Simpan, dan kemudian salin ID ke fail CSV.

img

Ubah suai pengguna yang diberikan kepada kes * Pilih pilihan ini untuk mengemas kini data bagi pengguna semasa yang diberikan kepada kes. * Gunakan ID Teg Kes untuk lajur case_ID.

Alih keluar pengguna yang diberikan kepada kes * Pilih pilihan ini untuk mengalih keluar pengguna yang diberikan kepada kes.

Memuat naik fail

Anda boleh memuat naik fail dengan segera atau kembali ke kotak dialog Tambah Import pada tarikh kemudian dan lengkapkan langkah ini.

  1. Buka kotak dialog Tambah Import.
  2. Masukkan nama unik untuk fail CSV.
  3. Klik ikon tambah (+) untuk memuat naik fail daripada peranti anda.
  4. Klik medan Jenis dan pilih pilihan. (lihat di atas)
  5. Klik SIMPAN.

Memproses

Bergantung pada bilangan rekod dalam fail dan beban sistem, platform boleh mengambil masa beberapa minit untuk memproses import.

img

Apabila import selesai, platform melaraskan rekod pengguna (iaitu menambah, mengubah suai atau mengalih keluar) pada halaman Kes.

Kegagalan

Jika operasi import gagal, platform mengembalikan mesej Gagal. Dalam kebanyakan kes, punca kegagalan ialah pemformatan data.

Untuk melihat punca kegagalan:

  1. Klik Lihat di bawah tajuk Tindakan.
  2. Pada halaman seterusnya, tuding tetikus pada ikon Gagal. Petua alat memaparkan ralat.
  3. Pergi ke fail CVS dan laraskan dan betulkan ralat.
  4. Muat naik fail sekali lagi.
