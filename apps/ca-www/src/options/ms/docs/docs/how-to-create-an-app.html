Cara Membuat Apl

Artikel ini menunjukkan kepada pengguna baharu cara membuat akaun percubaan percuma. Case Active secara automatik memberikan keistimewaan pentadbir pengguna baharu untuk akaun ini.

Membuat akaun

  1. Pergi ke halaman log masuk .
  2. Klik butang Bermula.
  3. Masukkan alamat e-mel.
  4. Klik Seterusnya.

Mengesahkan akaun

  1. Buka tab penyemak imbas baharu dan log masuk ke akaun e-mel anda.
  2. Salin kod enam digit.
  3. Kembali ke tab Case Active.
  4. Tampalkan kod ke dalam medan. Tetingkap baharu dipaparkan.

Dalam tetingkap baharu:

  1. Isikan medan yang diperlukan.
  2. Baca syarat perkhidmatan dan tandai kotak untuk bersetuju.
  3. Klik BUAT AKAUN PERCUMA.

Melengkapkan persediaan apl

Aliran kerja dipaparkan pada halaman seterusnya. Lengkapkan langkah untuk menyelesaikan penyediaan apl anda. Anda sentiasa boleh kembali ke halaman ini untuk mengedit atau mengemas kini maklumat atau status akaun anda.

Pada halaman Perihal Apl, isikan nama syarikat dan medan tapak web (diperlukan) dan kemudian klik Seterusnya.

img

Pada halaman Maklumat Pengebilan, klik medan Pelan Pembayaran dan semak pilihan. Tetapan lalai ialah Percuma. Klik SERAH.

Selepas anda menyerahkan borang, Case Active memaparkan halaman Apl. Untuk setiap apl yang anda cipta atau milik, halaman menunjukkan:

  1. Nama aplikasi
  2. Jenis Anda (peranan pengguna)
  3. Status

img

Halaman Apl ialah titik pelancaran untuk banyak fungsi pentadbir:

  1. Klik butang APP untuk berkomunikasi, mengisi dokumen, sembang video (dan banyak lagi) tentang isu berkaitan kes.
  2. Klik butang ADMIN untuk membuka panel admin. Di sinilah Pentadbir mengurus pengguna, kes, penyepaduan dan banyak lagi.
  3. Klik elipsis untuk mengurus pengebilan akaun dan tetapan pemberitahuan.
  • Note Case Active memaparkan butang ADMIN untuk pengguna pentadbir. Pengguna yang tidak mempunyai keistimewaan pentadbir tidak melihat butang ADMIN (seperti yang ditunjukkan dalam skrin di bawah).
