Cara Membuat Pengguna dengan Pautan Pendaftaran

Artikel ini menunjukkan pentadbir cara membuat pengguna dengan menghantar pautan pendaftaran.

Mengkonfigurasi pautan pendaftaran

  1. Klik pautan Hantar kepada mereka pendaftaran di bahagian bawah kotak dialog Tambah pengguna. Halaman baharu dipaparkan.
  2. Konfigurasikan pilihan pendaftaran. (lihat di bawah)

img

Pastikan medan Benarkan Pendaftaran ditetapkan kepada Hidup. (diperlukan) Apabila didayakan, Case Active menjana pautan unik.

Semak pilihan di bawah dan kemudian klik ikon untuk menyalin pautan. Anda boleh meletakkan pautan di tapak web anda atau menghantar pautan kepada satu atau lebih orang melalui e-mel atau mana-mana saluran komunikasi.

img

Mencipta kes

Secara pilihan, anda boleh membuat kes secara automatik untuk setiap pengguna baharu dan menetapkan peranan pengguna.

  1. Klik medan Auto Cipta Kes dan pilih Hidupkan. Medan Peranan Pengguna Kes dipaparkan.
  2. Klik Peranan Pengguna Kes dan pilih pilihan. (lihat di bawah)

img

Peranan pengguna kes

Terdapat tiga peranan pengguna kes.

  • pelanggan
  • lalai
  • kakitangan

Menghantar pautan

Kongsi pautan dengan seorang atau lebih orang. Setiap orang mengklik pautan dan melengkapkan aliran kerja untuk mengaktifkan akaun Case Active mereka. Setelah pengguna melengkapkan aliran kerja pendaftaran, Case Active menambahkan pengguna pada senarai Pengguna.

img

Langkah seterusnya

Case Active menghantar pemberitahuan e-mel kepada pengguna dengan pautan pendaftaran. Pengguna mengklik pautan dan melengkapkan aliran kerja untuk pendaftaran baharu.

  • Artikel ini menerangkan aliran kerja pendaftaran untuk pengguna baharu.
