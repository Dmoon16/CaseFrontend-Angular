So weisen Sie Benutzerzuweisungen mit dem Import-Tool zu, ändern oder entfernen sie

Dieser Artikel zeigt Administratoren, wie sie Fallbenutzer mit dem Import-Tool verwalten.

Administratoren können:

  • Benutzer einem Fall zuweisen
  • bereits einem Fall zugewiesene Benutzer ändern
  • Benutzer entfernen, die einem Fall zugewiesen sind

Öffnen des Import-Tools

  1. Melden Sie sich beim Administratorkonto an.
  2. Suchen Sie eine App auf der Seite Apps und klicken Sie auf die Schaltfläche ADMIN.
  3. Klicken Sie auf die Registerkarte Integration und dann im Untermenü auf IMPORTS.
  4. Klicken Sie auf die Schaltfläche NEU. Das Dialogfeld „Importe hinzufügen“ wird angezeigt.

Aufzeichnungen erstellen

Laden Sie eine oder mehrere CSV-Dateien herunter. Verwenden Sie diese Dateien, um Fallbenutzer hinzuzufügen, zu ändern oder zu entfernen. Nachdem das Dialogfeld Importe hinzufügen geöffnet wurde:

  1. Klicken Sie auf das Feld Typ.
  2. Blättern Sie durch das Dropdown-Menü und wählen Sie eine Option aus (siehe unten). Ein Link wird angezeigt.
  3. Klicken Sie auf den Link Beispiel herunterladen. Eine CSV-Datei wird auf Ihr Gerät heruntergeladen.

Bild

Öffnen Sie die Datei und fügen Sie Datensätze hinzu. Klicken Sie hier, um zu erfahren, wie Sie Formatdaten in der CSV-Datei verwenden.

Dateioptionen

Benutzer zu Fällen zuweisen * Wählen Sie diese Option, um Benutzer zu einem Fall hinzuzufügen. * Lassen Sie die Zelle case_role_order leer. * Gehen Sie zur Seite Benutzer und kopieren Sie die Tag-ID für diese Person. Wenn das Tag-ID-Feld leer ist, geben Sie eine eindeutige ID ein, klicken Sie auf Speichern und kopieren Sie dann die ID in die CSV-Datei.

Bild

Kundenvorgängen zugewiesene Benutzer ändern * Wählen Sie diese Option, um Daten für aktuelle Benutzer zu aktualisieren, die einem Kundenvorgang zugewiesen sind. * Verwenden Sie die Case-Tag-ID für die Spalte case_ID.

Bild

Kundenvorgängen zugewiesene Benutzer entfernen * Wählen Sie diese Option, um Benutzer zu entfernen, die einem Kundenvorgang zugewiesen sind.

Bild

Hochladen der Datei

Sie können eine Datei sofort hochladen oder zu einem späteren Zeitpunkt zum Dialogfeld „Importe hinzufügen“ zurückkehren und diese Schritte ausführen.

  1. Öffnen Sie das Dialogfeld „Importe hinzufügen“.
  2. Geben Sie einen eindeutigen Namen für die CSV-Datei ein.
  3. Klicken Sie auf das Pluszeichen (+), um die Datei von Ihrem Gerät hochzuladen.
  4. Klicken Sie auf das Feld Typ und wählen Sie eine Option aus. (siehe oben)
  5. Klicken Sie auf SPEICHERN.

wird bearbeitet

Abhängig von der Anzahl der Datensätze in der Datei und der Systemlast kann die Plattform mehrere Minuten benötigen, um den Import zu verarbeiten.

Bild

Wenn der Import abgeschlossen ist, passt die Plattform Benutzerdatensätze auf der Seite „Fälle“ an (dh fügt hinzu, ändert oder entfernt sie).

Fehler

Wenn der Importvorgang fehlschlägt, gibt die Plattform eine Fehlernachricht zurück. In den meisten Fällen ist die Fehlerursache die Datenformatierung.

Bild

So sehen Sie die Fehlerursache:

  1. Klicken Sie unter der Überschrift Aktion auf Ansicht.
  2. Bewegen Sie auf der nächsten Seite die Maus über das Symbol „Fehlgeschlagen“. Der Tooltip zeigt den Fehler an.
  3. Gehen Sie zur CVS-Datei und passen Sie den Fehler an und beheben Sie ihn.
  4. Laden Sie die Datei erneut hoch.
