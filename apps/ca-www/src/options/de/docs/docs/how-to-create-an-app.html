So erstellen Sie eine App

Dieser Artikel zeigt neuen Benutzern, wie sie ein kostenloses Testkonto erstellen. Case Active gewährt dem neuen Benutzer automatisch Administratorrechte für dieses Konto.

Ein Profil erstellen

  1. Gehen Sie zur Anmeldeseite .
  2. Klicken Sie auf die Schaltfläche Erste Schritte.
  3. Gib eine E-mail Adresse ein.
  4. Weiter klicken.

Bild

Verifizieren eines Kontos

  1. Öffnen Sie einen neuen Browser-Tab und melden Sie sich bei Ihrem E-Mail-Konto an.
  2. Kopieren Sie den sechsstelligen Code.
  3. Kehren Sie zur Registerkarte „Fall aktiv“ zurück.
  4. Fügen Sie den Code in die Felder ein. Ein neues Fenster wird angezeigt.

Im neuen Fenster:

  1. Füllen Sie die erforderlichen Felder aus.
  2. Lesen Sie die Nutzungsbedingungen und aktivieren Sie das Kontrollkästchen, um zuzustimmen.
  3. Klicken Sie auf EIN KOSTENLOSES KONTO ERSTELLEN.

Bild

App-Setup abschließen

Auf der nächsten Seite wird ein Workflow angezeigt. Führen Sie die Schritte aus, um die Einrichtung Ihrer App abzuschließen. Sie können jederzeit zu diesen Seiten zurückkehren, um Informationen oder Ihren Kontostatus zu bearbeiten oder zu aktualisieren.

Füllen Sie auf der Seite Über die App den Firmennamen und die Website-Felder aus (erforderlich) und klicken Sie dann auf Weiter.

Bild

Klicken Sie auf der Seite Rechnungsinformationen auf das Feld Zahlungsplan und überprüfen Sie die Optionen. Die Standardeinstellung ist Kostenlos. Klicken Sie auf SENDEN.

Bild

Nachdem Sie das Formular gesendet haben, zeigt Case Active die Seite „Apps“ an. Für jede App, die Sie erstellen oder zu der Sie gehören, zeigt die Seite:

  1. Der App-Name
  2. Ihr Typ (Benutzerrolle)
  3. Status

Bild

Die Apps-Seite ist ein Ausgangspunkt für viele Verwaltungsfunktionen:

  1. Klicken Sie auf die APP-Schaltfläche, um über fallbezogene Probleme zu kommunizieren, Dokumente auszufüllen, Video-Chats (und mehr) durchzuführen.
  2. Klicken Sie auf die Schaltfläche ADMIN, um das Admin-Panel zu öffnen. Hier verwalten Administratoren Benutzer, Fälle, Integrationen und mehr.
  3. Klicken Sie auf die Auslassungspunkte, um die Kontoabrechnungs- und Benachrichtigungseinstellungen zu verwalten.
  • Hinweis Case Active zeigt die ADMIN-Schaltfläche für Admin-Benutzer an. Benutzer ohne Administratorrechte sehen die ADMIN-Schaltfläche nicht (wie im folgenden Bildschirm gezeigt).
