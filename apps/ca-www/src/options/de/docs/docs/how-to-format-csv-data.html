So formatieren Sie CSV-Daten

Nachfolgend finden Sie eine Liste von Datenfeldern in CSV-Dateien. Jede Importoption hat eine andere Kombination von Feldern.

Nutzername

  • Geben Sie die E-Mail-Adresse des Benutzers ein.

Vorname

  • Geben Sie den Vornamen des Benutzers ein.

Nachname

  • Geben Sie den Nachnamen des Benutzers ein.

Gebietsschema

  • Diese Spalte sucht nach der bevorzugten Sprache des Benutzers.
  • Geben Sie en für Englisch und sp für Spanisch ein.

Zoneninfo

  • Dies legt die Zeitzone des Benutzers fest. Geben Sie den Kontinent und die Stadt ohne Leerzeichen ein (verwenden Sie einen Unterstrich).
  • Muss eine Option im Dropdown-Menü Profil > SPRACHE & REGION > Zeitzone verwenden.
  • Verwenden Sie die am besten geeignete Zeitzone. Boston befindet sich beispielsweise nicht im Dropdown-Menü, sodass wir New York (oder eine beliebige Stadt in der EST/EDT-Zeitzone) verwenden können.

host_tag_id

  • Geben Sie eine dauerhafte Bezeichnung ein.
  • az, 0-9, -, / sind erlaubt.

Fallstatus

  • 0 = geschlossen
  • 1 = aktiv
