So melden Sie sich an

Erfahren Sie, wie Sie sich bei Ihrem bestehenden Case Active-Konto anmelden.

Einloggen

  1. Gehen Sie zur Anmeldeseite .
  2. Klicken Sie auf die Schaltfläche ANMELDEN.
  3. Geben sie ihre E-Mail Adresse ein.
  4. Weiter klicken.

Verifizieren eines Kontos

  1. Öffnen Sie einen neuen Browser-Tab und melden Sie sich bei Ihrem E-Mail-Konto an.
  2. Kopieren Sie den sechsstelligen Code.
  3. Kehren Sie zur Registerkarte „Fall aktiv“ zurück.
  4. Fügen Sie den Code in die Felder ein. Ein neues Fenster wird angezeigt.

Case Active zeigt die Apps-Seite an.

  • Hinweis Die Plattform zeigt eine ADMIN-Schaltfläche neben der APP-Schaltfläche für Benutzer mit Administratorrechten an.
