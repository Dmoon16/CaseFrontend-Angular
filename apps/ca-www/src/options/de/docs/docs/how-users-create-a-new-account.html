Wie Benutzer ein neues Konto erstellen

Nachdem ein Administrator einen Benutzer zu einem Fall hinzugefügt hat: * Wenn der Benutzer neu bei caseactive ist, muss der Benutzer einen Workflow abschließen, um ein Konto zu erstellen und die Einladung zum Zugriff auf den Fall anzunehmen. * Wenn der Benutzer ein bestehendes caseactive-Konto hat, muss der Benutzer die Einladung annehmen, um auf den Fall zuzugreifen.

Dieser Artikel beschreibt die Schritte, die ein Benutzer ausführen muss, um sein Konto zu erstellen.

Einstieg

Es gibt zwei Möglichkeiten, wie ein Benutzer zu einer caseactive App hinzugefügt werden kann

  • Durch Administrator: Wenn der Administrator einen Benutzer über die Benutzeroberfläche der Plattform oder das Importtool erstellt, muss der Administrator dem Benutzer einen Link zu account.caseactive.com senden, um mit der Anmeldung fortzufahren.
  • Nach Benutzer: Wenn der Benutzer ein Konto mit einem Anmeldelink erstellt hat, wird der Benutzer automatisch zu account.caseactive.com umgeleitet, um mit der Anmeldung fortzufahren.

Einloggen

Wenn sich der Benutzer zum ersten Mal bei seinem CaseActive-Konto anmeldet, fordert die Plattform den Benutzer auf:

  • Öffnen Sie sein E-Mail-Konto und kopieren Sie den sechsstelligen Sicherheitscode und fügen Sie ihn ein
  • Vervollständigen Sie ein Kontoprofil

Bild

Nachdem der Benutzer auf UPDATE geklickt hat, wird die Seite „Apps“ angezeigt. Die Seite zeigt den Namen der App (Firmenname), den Benutzertyp (Rolle), den Status und eine EINLADUNGS-Schaltfläche.

Bild

Um auf die App zuzugreifen, muss der Benutzer die Einladung annehmen. Der Benutzer sollte:

  1. Klicken Sie auf die Schaltfläche EINLADUNG.
  2. Überprüfen Sie die Berechtigungen.
  3. Klicken Sie auf die Registerkarte Benachrichtigungen, um die Bedingungen zu überprüfen und (optional) zu bearbeiten.
  4. Klicken Sie auf ZUGRIFF GENEHMEN.

Bild

Wenn der Benutzer zustimmt, zeigt die Plattform eine APP-Schaltfläche auf der Benutzerseite an. Diese Schaltfläche gibt dem Benutzer Zugriff auf zugewiesene Fälle.

Um Benutzereinstellungen zu bearbeiten, klickt der Benutzer auf die Auslassungspunkte.

Bild

Der Benutzer kann: * App-Zugriff entfernen * Benachrichtigungseinstellungen ändern

Bild

Nachdem der Benutzer den Arbeitsablauf abgeschlossen hat, wird der Benutzername in der Benutzerliste des Administrators angezeigt. Die Liste zeigt alle Benutzer, angemeldet oder nicht.
