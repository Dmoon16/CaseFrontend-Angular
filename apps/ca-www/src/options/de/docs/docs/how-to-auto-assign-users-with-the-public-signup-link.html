So weisen Sie Benutzer automatisch mit dem öffentlichen Anmeldelink zu

Administratoren erfahren, wie sie Systemeinstellungen so konfigurieren, dass neue Benutzer automatisch einem neu automatisch erstellten Fall zugewiesen werden.

Öffnen der Setup-Seite

Der erste Schritt besteht darin, die Systemeinrichtungsseite zu öffnen.

  1. Melden Sie sich beim Administratorkonto an.
  2. Suchen Sie eine App auf der Seite Apps und klicken Sie auf die Schaltfläche ADMIN.
  3. Klicken Sie auf die Registerkarte Systeme und dann im Untermenü auf SYSTEM SETUP.
  4. Scrollen Sie nach unten zur Überschrift USER SIGNUP.

Systemeinstellungen konfigurieren

  1. Setzen Sie die Drop-down-Menüs Anmeldung zulassen und Fall automatisch erstellen auf Ein.
  2. Klicken Sie auf das Rollenmenü und wählen Sie eine Option aus (Standard wird empfohlen).
  3. Kopieren und teilen Sie den Link zur öffentlichen Benutzeranmeldung.

Senden des Links

Jeder Benutzer, der auf den Link klickt und die Anmeldung abschließt, veranlasst die Plattform, einen neuen Fall auf der Seite „Fälle“ zu erstellen und den Benutzer in diesem Fall automatisch zuzuweisen.
