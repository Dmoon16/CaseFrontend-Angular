So weisen Sie Benutzer beim Erstellen eines Falls zu

Dieser Artikel zeigt Administratoren, wie sie beim Erstellen eines neuen Falls einen Benutzer zuweisen. Wenn der Benutzer noch nicht auf der Plattform registriert ist, können Sie beim Erstellen eines neuen Falls einen neuen Benutzer hinzufügen.

Fall erstellen

  1. Melden Sie sich beim Administratorkonto an.
  2. Suchen Sie eine App auf der Seite Apps und klicken Sie auf die Schaltfläche ADMIN.
  3. Klicken Sie auf die Menüregisterkarte Fälle.
  4. Klicken Sie auf die Schaltfläche NEU. Ein neues Fenster wird angezeigt.

Bild

  • Füllen Sie die Felder aus, um einen Fall zu erstellen, und scrollen Sie dann nach unten zur Überschrift „Zugewiesene Benutzer“.

Benutzer zuweisen

Aktueller Benutzer

Befolgen Sie diese Schritte, um dem neuen Fall einen aktuellen Benutzer zuzuweisen.

Klicken Sie auf die Schaltfläche BENUTZER ZUWEISEN und füllen Sie die Felder aus.

  1. Klicken Sie auf das Dropdown-Menü Benutzer und wählen Sie eine Kennung (z. B. E-Mail) aus.
  2. Geben Sie einen Wert in das leere Feld ein (z. B. E-Mail-Adresse).
  3. Wenn der Benutzerdatensatz in der Plattform vorhanden ist, schlägt das Feld automatisch einen Namen vor.
  4. Wählen Sie den richtigen Benutzer aus.
  5. Wählen Sie eine Benutzerrolle aus (Kunden, Standard oder Mitarbeiter).
  6. Aktivieren Sie optional das Kontrollkästchen, um dem Benutzer eine E-Mail-Benachrichtigung zu senden.
  7. Klicken Sie auf SPEICHERN.

Bild

Fall aktiv fügt den Benutzer zum Fall hinzu. Jetzt können Administratoren auf Speichern klicken, um das Fenster zu schließen, oder einen neuen Benutzer hinzufügen oder einen Benutzer entfernen.

Bild

Hinzufügen eines neuen Benutzers

Wenn die Plattform keinen aktuellen Datensatz finden kann, klicken Sie auf die Schaltfläche, um einen neuen Benutzer hinzuzufügen.

Bild

Füllen Sie die Felder aus und klicken Sie dann auf SPEICHERN, um den Benutzer zur Plattform hinzuzufügen. Legen Sie als Nächstes eine Benutzerrolle fest und klicken Sie erneut auf SPEICHERN, um den Benutzer dem Fall zuzuweisen. Klicken Sie abschließend noch einmal auf SPEICHERN, um den Fall zu speichern.
