So erstellen Sie automatisch Fälle mit einem öffentlichen Registrierungslink

Dieser Artikel zeigt Administratoren, wie sie einen Fall erstellen (und gleichzeitig einen Benutzer hinzufügen), indem sie einen einzigen Registrierungslink senden.

Lokalisierung des Links

  1. Melden Sie sich beim Administratorkonto an.
  2. Suchen Sie eine App auf der Seite Apps und klicken Sie auf die Schaltfläche ADMIN.
  3. Klicken Sie im Admin-Bereich auf die Menüregisterkarte Einstellungen.
  4. Stellen Sie sicher, dass das Untermenü SYSTEM SETUP ausgewählt ist.

Bild

Einstellungen konfigurieren

Scrollen Sie nach unten zur Überschrift USER SIGNUP.

Stellen Sie sicher, dass das Feld Registrierung zulassen auf Ein gesetzt ist. (erforderlich) Wenn diese Funktion aktiviert ist, generiert Case Active einen eindeutigen Link.

  1. Klicken Sie auf das Feld Fall automatisch erstellen und wählen Sie Ein. Das Feld Kundenvorgangsbenutzerrolle wird angezeigt.
  2. Wählen Sie eine Rolle aus. (siehe unten)

Bild

Fallbenutzerrollen

Es gibt drei Fallbenutzerrollen.

  • Kunden (empfohlen)
  • Ursprünglich
  • Mitarbeiter

Senden des Links

Teilen Sie den Link mit einer oder mehreren Personen. Jeder Benutzer, der auf den Link klickt und die Anmeldung abschließt, veranlasst die Plattform, einen neuen Fall auf der Fallseite zu generieren und den Benutzer automatisch dem Fall zuzuweisen.
