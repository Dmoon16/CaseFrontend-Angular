Case-Management-Daten mit dem Import-Tool formatieren

Nachfolgend finden Sie eine Liste der erforderlichen Felder in CSV-Dateien, um Fälle mit dem Import-Tool zu verwalten.

Nutzername

  • Geben Sie die E-Mail-Adresse des Benutzers ein.

tag_id (eine ID, auf die Sie später verweisen können)

  • eine eindeutige Kennung (alphanumerische Zeichen verwenden, Bindestriche werden akzeptiert)

Fallstatus

  • 0 = geschlossen
  • 1 = aktiv

datum_eröffnet

  • Monat Tag Jahr
  • mm/tt/jjjj

Über

  • eine eindeutige Beschreibung des Falls (alphanumerische Zeichen verwenden, Bindestriche werden akzeptiert)
