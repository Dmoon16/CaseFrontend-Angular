Como fazer login

Saiba como fazer login em sua conta Case Active existente.

Fazendo login

  1. Vá para a página de login .
  2. Clique no botão LOGIN.
  3. Digite seu e-mail.
  4. Clique em PRÓXIMO.

Verificando uma conta

  1. Abra uma nova guia do navegador e faça login na sua conta de e-mail.
  2. Copie o código de seis dígitos.
  3. Retorne à guia Caso Ativo.
  4. Cole o código nos campos. Uma nova janela é exibida.

Case Active exibe a página Aplicativos.

  • Observação A plataforma exibe um botão ADMIN ao lado do botão APP para usuários com privilégios de administrador.
