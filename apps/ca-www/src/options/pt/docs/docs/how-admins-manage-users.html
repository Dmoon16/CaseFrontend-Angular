Como os administradores gerenciam os usuários

Os administradores criam, modificam ou desativam quem pode acessar aplicativos e casos. Quando um usuário faz login na plataforma pela primeira vez, ele precisa concluir um fluxo de trabalho de perfil.

  • Clique aqui para saber mais sobre o novo fluxo de trabalho de perfil de usuário.

Gerenciando um usuário

A tabela abaixo mostra cinco maneiras de gerenciar um usuário. Clique em um link para saber como.

Método Modelo Ações Detalhes
Usando IU IU adicionar e modificar Botão Admin > Usuários
Ao criar um caso e atribuir usuário ao caso IU adicionar Botão Admin > Casos > botão Novo > página Criar caso
Criar usuário automaticamente usando o link de inscrição público Ação do usuário adicionar Botão Admin > Configurações > página Configuração do sistema
ferramenta de importação IU adicionar e modificar Botão Admin > Integrações > Importações
Usando API (em breve) API adicionar e modificar N / D

Notificações

A plataforma não envia uma notificação aos usuários quando eles são adicionados a um aplicativo. No entanto, os administradores podem habilitar notificações para atribuições de caso.
