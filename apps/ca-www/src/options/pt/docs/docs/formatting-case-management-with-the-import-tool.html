Formatando dados de gerenciamento de casos com a ferramenta de importação

Abaixo está uma lista de campos obrigatórios em arquivos CSV para gerenciar casos com a ferramenta Importar.

nome do usuário

  • Digite o endereço de e-mail do usuário.

tag_id (um id que você pode referenciar mais tarde)

  • um identificador exclusivo (use caracteres alfanuméricos, hífens são aceitos)

caso_status

  • 0 = fechado
  • 1 = ativo

data_aberta

  • mês dia ano
  • mm/dd/aaaa

cerca de

  • uma descrição única do caso (use caracteres alfanuméricos, hífens são aceitos)
