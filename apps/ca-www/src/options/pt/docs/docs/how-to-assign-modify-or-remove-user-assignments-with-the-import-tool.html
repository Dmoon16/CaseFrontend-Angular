Como atribuir, modificar ou remover atribuições de usuário com a ferramenta de importação

Este artigo mostra aos administradores como gerenciar usuários de casos com a ferramenta Importar.

Os administradores podem:

  • atribuir usuários a um caso
  • modificar usuários já atribuídos a um caso
  • remover usuários atribuídos a um caso

Abrindo a ferramenta Importar

  1. Faça login na conta de administrador.
  2. Localize um aplicativo na página Aplicativos e clique no botão ADMIN.
  3. Clique na guia Integração e depois em IMPORTAÇÕES no submenu.
  4. Clique no botão NOVO. A caixa de diálogo Adicionar importações é exibida.

Criando registros

Baixe um ou mais arquivos CSV. Use esses arquivos para adicionar, modificar ou remover usuários do caso. Depois que a caixa de diálogo Adicionar importações for aberta:

  1. Clique no campo Tipo.
  2. Percorra o menu suspenso e selecione uma opção (veja abaixo). Um link é exibido.
  3. Clique no link Baixar amostra. Um arquivo CSV é baixado para o seu dispositivo.

img

Abra o arquivo e adicione registros. Clique aqui para saber como usar dados de formato no arquivo CSV.

Opções de arquivo

Atribuir usuários a casos * Escolha esta opção para adicionar usuários a um caso. * Deixe a célula case_role_order em branco. * Vá para a página Usuários e copie o ID da tag para esta pessoa. Se o campo Tag ID estiver em branco, insira um ID exclusivo, clique em Salvar e copie o ID para o arquivo CSV.

Modificar usuários atribuídos a casos * Escolha esta opção para atualizar os dados dos usuários atuais atribuídos a um caso. * Use o ID da tag Case para a coluna case_ID.

Remover usuários atribuídos a casos * Escolha esta opção para remover usuários atribuídos a um caso.

img

Carregando o arquivo

Você pode carregar um arquivo imediatamente ou retornar à caixa de diálogo Adicionar importações posteriormente e concluir essas etapas.

  1. Abra a caixa de diálogo Adicionar importações.
  2. Insira um nome exclusivo para o arquivo CSV.
  3. Clique no ícone de mais (+) para carregar o arquivo do seu dispositivo.
  4. Clique no campo Tipo e selecione uma opção. (Veja acima)
  5. Clique em SALVAR.

Em processamento

Dependendo da quantidade de registros do arquivo e da carga do sistema, a plataforma pode demorar alguns minutos para processar a importação.

Quando a importação é concluída, a plataforma ajusta os registros do usuário (ou seja, adiciona, modifica ou remove) na página Casos.

Falha

Se a operação de importação falhar, a plataforma retornará uma mensagem de falha. Na maioria dos casos, a causa da falha é a formatação dos dados.

Para ver a causa da falha:

  1. Clique em Exibir sob o cabeçalho Ação.
  2. Na próxima página, passe o mouse sobre o ícone de falha. A dica de ferramenta exibe o erro.
  3. Vá para o arquivo CVS e ajuste e corrija o erro.
  4. Carregue o arquivo novamente.
