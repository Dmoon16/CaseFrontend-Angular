Como criar usuários com um link de inscrição

Este artigo mostra aos administradores como criar usuários enviando um link de inscrição.

Configurando o link de inscrição

  1. Clique no link Enviar uma inscrição para eles na parte inferior da caixa de diálogo Adicionar usuário. Uma nova página é exibida.
  2. Configure as opções de inscrição. (Veja abaixo)

Certifique-se de que o campo Permitir inscrição esteja definido como Ativado. (obrigatório) Quando ativado, Case Active gera um link exclusivo.

Revise as opções abaixo e clique no ícone para copiar o link. Você pode colocar o link em seu site ou enviar o link para uma ou mais pessoas por e-mail ou qualquer canal de comunicação.

img

Criando um caso

Opcionalmente, você pode criar automaticamente um caso para cada novo usuário e atribuir uma função de usuário.

  1. Clique no campo Auto Create Case e selecione On. Os campos Função do usuário do caso são exibidos.
  2. Clique na função de usuário do caso e selecione uma opção. (Veja abaixo)

img

Funções do usuário do caso

Há três funções de usuário de caso.

  • clientes
  • predefinição
  • funcionários

Enviando o link

Compartilhe o link com uma ou mais pessoas. Cada pessoa clica no link e conclui o fluxo de trabalho para ativar sua conta Case Active. Depois que o usuário conclui o fluxo de trabalho de inscrição, Case Active adiciona o usuário à lista de usuários.

Próximos passos

O Case Active envia ao usuário uma notificação por e-mail com o link de inscrição. O usuário clica no link e conclui o fluxo de trabalho para novas inscrições.
