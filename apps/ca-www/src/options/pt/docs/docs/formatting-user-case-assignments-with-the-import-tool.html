Formatando dados de atribuições de caso de usuário com a ferramenta de importação

Abaixo está uma lista de campos de dados em arquivos CSV. Cada opção de importação tem uma combinação diferente de campos.

caso_ID

  • Insira o ID do usuário ou da tag do caso.
  • Vá para a página Casos > Editar > ID da tag OU Vá para a página Casos > Editar > ID do caso no URL.

ID do usuário

  • Insira o ID do usuário ou o ID da tag.
  • Vá para a página Usuários > Editar > ID da tag OU Vá para a página Usuários > Editar > ID do usuário no url.

case_role_order

  • Deixe a célula em branco.

case_role_ID

  • Digite: clientes, equipe ou padrão.

case_notify

  • 0 = não
  • 1 = Sim, envie uma notificação por e-mail ao usuário.
