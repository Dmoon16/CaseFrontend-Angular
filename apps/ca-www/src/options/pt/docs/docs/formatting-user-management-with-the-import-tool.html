Formatando dados de gerenciamento de usuários com a ferramenta de importação

Abaixo está uma lista de campos de dados em arquivos CSV. Cada opção de importação tem uma combinação diferente de campos.

nome do usuário

  • Digite o endereço de e-mail do usuário.

nome dado

  • Insira o primeiro nome do usuário.

nome de família

  • Insira o sobrenome do usuário.

localidade

  • Esta coluna verifica o idioma preferido do usuário.
  • Digite en para inglês, sp para espanhol.

informações de zona

  • Isso define o fuso horário do usuário. Digite o continente e a cidade sem espaços (use um sublinhado).
  • Deve usar uma opção no menu suspenso Perfil > IDIOMA E REGIÃO > Fuso horário.
  • Use o fuso horário mais adequado. Por exemplo, Boston não está no menu suspenso, então podemos usar Nova York (ou qualquer cidade no fuso horário EST/EDT).

host_tag_id

  • Insira um rótulo permanente.
  • az, 0-9, -, / são permitidos.

caso_status

  • 0 = fechado
  • 1 = ativo
