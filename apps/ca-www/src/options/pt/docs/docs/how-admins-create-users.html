Como os administradores criam usuários

Os administradores criam, modificam ou desativam quem pode acessar aplicativos e casos. Quando um usuário faz login na plataforma pela primeira vez, ele precisa concluir um fluxo de trabalho de perfil.

  • Clique aqui para saber mais sobre o novo fluxo de trabalho de perfil de usuário.

Criando um usuário

A tabela abaixo mostra cinco maneiras de criar um usuário. Clique em um link para saber como.

| Método | Tipo | Ações | Detalhes | Novo usuário recebe uma notificação? |
|-------------------------|---------------|------- ------|--------------------|---------------------- ------------------------|

Usando IU | IU | adicionar e modificar | Botão Admin > Usuários | Não |

Ao criar um caso e atribuir usuário ao caso | IU | adicionar | Botão Admin > Casos > botão Novo > página Criar caso | Sim, se ativado |

Criar usuário automaticamente usando o link de inscrição público | Ação do usuário | adicionar | Botão Admin > Configurações > página Configuração do sistema | Sim, se a criação automática de caso estiver habilitada |

Ferramenta de importação | IU | adicionar e modificar | Botão Admin > Integrações > Importações | Não |

Usando API (em breve) | API | adicionar e modificar | na | na |

Notificações

A plataforma não envia uma notificação aos usuários quando eles são adicionados a um aplicativo. No entanto, os administradores podem habilitar notificações para atribuições de caso.
