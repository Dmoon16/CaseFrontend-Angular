Como os usuários criam uma nova conta

Depois que um administrador adiciona um usuário a um caso: * se o usuário for novo no caso ativo, o usuário deverá concluir um fluxo de trabalho para criar uma conta e aceitar o convite para acessar o caso. * se o usuário tiver uma conta caseactive existente, o usuário deve aceitar o convite para acessar o caso.

Este artigo descreve as etapas que um usuário deve concluir para criar sua conta.

Começando

Há duas maneiras de adicionar um usuário a um aplicativo caseativo

  • Por administrador: se o administrador criar um usuário por meio da interface do usuário da plataforma ou da ferramenta de importação , o administrador deverá enviar ao usuário um link para account.caseactive.com para prosseguir com o login.
  • Por usuário: Se o usuário criou uma conta usando um link de inscrição , o usuário será redirecionado automaticamente para account.caseactive.com para prosseguir para o login.

Fazendo login

Quando o usuário faz login em sua conta CaseActive pela primeira vez, a plataforma solicita que o usuário:

  • abra a conta de e-mail e copie/cole o código de segurança de seis dígitos
  • completar um perfil de conta

Depois que o usuário clicar em ATUALIZAR, a página Aplicativos será exibida. A página mostra o nome do aplicativo (nome da empresa), o tipo de usuário (função), status e um botão CONVITE.

Para acessar o aplicativo, o usuário deve aceitar o convite. O usuário deve:

  1. Clique no botão CONVITE.
  2. Revise as permissões.
  3. Clique na guia Notificações para revisar e editar (opcional) os termos.
  4. Clique em APROVAR ACESSO.

Quando o usuário aprova, a plataforma exibe um botão APP na página do usuário. Este botão dá ao usuário acesso aos casos atribuídos.

Para editar as configurações do usuário, o usuário clica nas reticências.
