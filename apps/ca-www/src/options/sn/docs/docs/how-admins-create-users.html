MaAdmins Anogadzira Vashandisi

MaAdmins gadzira, gadzirisa kana kudzima anogona kuwana maapplication nemakesi. Kana mushandisi achipinda papuratifomu kekutanga, ivo vanofanirwa kupedzisa profil workflow.

  • Dzvanya apa kuti udzidze zvakawanda nezve hutsva hwemushandisi chimiro chekushanda.

Kugadzira mushandisi

Tafura iri pazasi inoratidza nzira shanu dzekugadzira mushandisi. Dzvanya link kuti udzidze sei.

| Nzira | Type | Zviito | Details | Mushandisi mutsva anogamuchira chiziviso? |
|------------------------------------------------ ------|------------------------------------------ -------------------------|

Kushandisa UI | UI | wedzera & gadzirisa | Admin bhatani > Vashandisi | Kwete |

Paunenge uchigadzira kesi uye upe mushandisi kune kesi | UI | wedzera | Bhatani rekutonga> Nyaya> Bhatani idzva> Gadzira Nyaya peji | Ehe, kana ikagoneswa |
