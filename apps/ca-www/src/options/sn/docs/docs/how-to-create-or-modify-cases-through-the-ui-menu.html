Maitiro ekugadzira kana kugadzirisa nyaya kuburikidza neUI Menu

Ichi chinyorwa chinoratidza admins maitiro ekugadzira nyaya ine chikuva mushandisi interface. MaAdmins anogonawo kuwedzera vashandisi kune imwe nyaya uye kugadzirisa nyaya dziripo.

Kugadzira nyaya

  1. Pinda kune admin account.
  2. Tsvaga app papeji yeMapurogiramu wobva wadzvanya ADMIN bhatani.
  3. Pane iyo Admin pani, tinya iyo Cases menyu tebhu.
  4. Baya bhatani ITSVA. Hwindo idzva rinoratidza.

Kuwedzera vashandisi

Sarudzo, admins vanogona kuwedzera vashandisi kune kesi iri papeji ino. Skroka pasi wodzvanya bhatani rekuti ASSIGN USERS.

Mune iyo Govera Mushandisi modal, zadza minda. Tarisa bhokisi kuti utumire mushandisi email chiziviso. (sarudzo)

Kugadzirisa nyaya

Kuti uvandudze nhamba diki yekesi, shandisa iyo Edit chishandiso uye chinja kune yega yega kesi.
