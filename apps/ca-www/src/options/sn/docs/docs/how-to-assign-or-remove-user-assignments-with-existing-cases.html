Maitiro Ekupa kana Kubvisa Mushandisi Migove neAripo Nyaya

Ichi chinyorwa chinoratidza admins nzira yekugovera uye kubvisa vashandisi vekesi. Isu tinokuratidza zvakare maitiro ekugadzirisa mushandisi data.

Kuvhura kesi iripo

  1. Pinda kune admin account.
  2. Tsvaga app papeji yeMapurogiramu wobva wadzvanya ADMIN bhatani.
  3. Dzvanya iyo Cases menyu tab.
  4. Ita shuwa ACTIVE yakasarudzwa mune submenu.
  5. Skikira pasi kune ACTION column.

Govera kana kubvisa

Dzvanya Govera Vashandisi kugovera mushandisi kune iyo kesi kana kubvisa mushandisi kubva pane iyo kesi (asi kwete chikuva). Pa peji rinotevera: Dzvanya ADD uye pedzisa kufambiswa kwebasa. Dzvanya bvisa kuti udzime mushandisi.

Modify

Hazvigoneke kugadzirisa kana kugadzirisa data yemushandisi pane peji reMakesi. Pane kudaro, tevera matanho aya:

  1. Dzvanya iyo Users menyu tab.
  2. Tsvaga zita remushandisi wobva wadzvanya Edit .
  3. Ita shanduko uye tinya SAVE.
