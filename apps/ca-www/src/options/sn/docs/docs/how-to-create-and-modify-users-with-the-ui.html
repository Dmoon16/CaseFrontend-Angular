Maitiro ekugadzira uye kugadzirisa vashandisi neUI

Ichi chinyorwa chinoratidza admins maitiro ekugadzira vashandisi uye kugadzirisa marekodhi ane chikuva mushandisi interface. MaAdmins anogonawo kuwedzera vashandisi kune imwe nyaya uye kugadzirisa nyaya dziripo.

Kugadzira mushandisi

  1. Pinda kune admin account.
  2. Tsvaga app papeji yeMapurogiramu wobva wadzvanya ADMIN bhatani.
  3. Pane iyo Admin pani, tinya iyo Vashandisi menyu tebhu.
  4. Baya bhatani ITSVA. Hwindo idzva rinoratidza.

Zadza minda wodzvanya SAVE. Case Active inowedzera zita kune rondedzero yevashandisi vane chekuita neiyi app.

Kugadzirisa mushandisi

Kuti uvandudze nhamba diki yemarekodhi emushandisi, shandisa iyo Rongedza chishandiso uye ita shanduko kune wega wega mushandisi.

  • Note Kuti ugadzirise huwandu hukuru hwemarekodhi emushandisi mune imwechete kugara, funga kushandisa iyo Import tool .
  1. Pane iyo Admin pani, tinya iyo Vashandisi menyu tebhu.
  2. Tsvaga mushandisi uye tinya Edit.
  3. Ita shanduko uye tinya SAVE.

Matanho anotevera

Case Active haitumire mushandisi email chiziviso. MaAdmins anofanirwa kubata mushandisi uye kuvatumira chinongedzo kuCase Active peji repamba. Mushandisi anobva adzvanya bhatani reLOGIN uye anopedzisa mafambiro ebasa rekusaina kutsva.

  • Ichi chinyorwa chinotsanangura kusaina mafambiro evashandisi vatsva.
