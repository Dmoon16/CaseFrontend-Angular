Kutsvaga chinongedzo

  1. Pinda kune admin account.
  2. Tsvaga app papeji yeMapurogiramu wobva wadzvanya ADMIN bhatani.
  3. Pane Admin panel, tinya iyo Settings menyu tab.
  4. Ita shuwa kuti SYSTEM SETUP submenu tab yasarudzwa.

Kugadzirisa zvirongwa

Skira pasi kune USER SIGNUP musoro.

Ita shuwa kuti Bvumira Signup ndima yaiswa kuOn. (inodiwa) Kana chimiro ichi chagoneswa, Case Active inogadzira chinongedzo chakasiyana.

  1. Dzvanya iyo Auto Gadzira Nyaya uye sarudza On. Iyo Case User Role ndima inoratidza.
  2. Sarudza basa. (ona pazasi)
