Admins dzidza kugadzirisa masisitimu kuitira kuti vashandisi vatsva vagopihwa otomatiki kune ichangobva kugadzirwa kesi.

Kuvhura peji yekuseta

Nhanho yekutanga ndeyekuvhura iyo Systems Setup peji.

  1. Pinda kune admin account.
  2. Tsvaga app papeji yeMapurogiramu wobva wadzvanya ADMIN bhatani.
  3. Dzvanya iyo Systems tab uye ipapo SYSTEM SETUP pane submenu.
  4. Bhura pasi kune USER SIGNUP musoro.

Kugadzirisa zvirongwa zvehurongwa

  1. Seta iyo Bvumira Signup uye Auto Gadzira Nyaya yekudonha-pasi menyu kuOn.
  2. Dzvanya iyo menyu yebasa uye sarudza sarudzo (default inokurudzirwa).
  3. Kopa uye ugovane iyo Public User Signup link.

Kutumira chinongedzo
