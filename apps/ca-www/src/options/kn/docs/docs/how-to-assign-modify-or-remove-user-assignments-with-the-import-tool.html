ಆಮದು ಉಪಕರಣದೊಂದಿಗೆ ಬಳಕೆದಾರರ ನಿಯೋಜನೆಗಳನ್ನು ನಿಯೋಜಿಸುವುದು, ಮಾರ್ಪಡಿಸುವುದು ಅಥವಾ ತೆಗೆದುಹಾಕುವುದು ಹೇಗೆ

ಆಮದು ಉಪಕರಣದೊಂದಿಗೆ ಕೇಸ್ ಬಳಕೆದಾರರನ್ನು ಹೇಗೆ ನಿರ್ವಹಿಸುವುದು ಎಂಬುದನ್ನು ಈ ಲೇಖನವು ನಿರ್ವಾಹಕರಿಗೆ ತೋರಿಸುತ್ತದೆ.

ನಿರ್ವಾಹಕರು ಮಾಡಬಹುದು:

  • ಒಂದು ಪ್ರಕರಣಕ್ಕೆ ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಿ
  • ಪ್ರಕರಣಕ್ಕೆ ಈಗಾಗಲೇ ನಿಯೋಜಿಸಲಾದ ಬಳಕೆದಾರರನ್ನು ಮಾರ್ಪಡಿಸಿ
  • ಪ್ರಕರಣಕ್ಕೆ ನಿಯೋಜಿಸಲಾದ ಬಳಕೆದಾರರನ್ನು ತೆಗೆದುಹಾಕಿ

ಆಮದು ಉಪಕರಣವನ್ನು ತೆರೆಯಲಾಗುತ್ತಿದೆ

  1. ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ನಿರ್ವಾಹಕ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ಇಂಟಿಗ್ರೇಶನ್ ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ನಂತರ ಉಪಮೆನುವಿನಲ್ಲಿ ಆಮದು ಮಾಡಿ.
  4. ಹೊಸ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ಆಮದು ಸೇರಿಸಿ ಡೈಲಾಗ್ ಬಾಕ್ಸ್ ಪ್ರದರ್ಶಿಸುತ್ತದೆ.

ದಾಖಲೆಗಳನ್ನು ರಚಿಸುವುದು

ಒಂದು ಅಥವಾ ಹೆಚ್ಚಿನ CSV ಫೈಲ್‌ಗಳನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ. ಕೇಸ್ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಲು, ಮಾರ್ಪಡಿಸಲು ಅಥವಾ ತೆಗೆದುಹಾಕಲು ಈ ಫೈಲ್‌ಗಳನ್ನು ಬಳಸಿ. ಆಮದು ಸೇರಿಸಿ ಸಂವಾದ ಪೆಟ್ಟಿಗೆ ತೆರೆದ ನಂತರ:

  1. ಟೈಪ್ ಫೀಲ್ಡ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  2. ಡ್ರಾಪ್-ಡೌನ್ ಮೆನು ಮೂಲಕ ಸ್ಕ್ರಾಲ್ ಮಾಡಿ ಮತ್ತು ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ (ಕೆಳಗೆ ನೋಡಿ). ಲಿಂಕ್ ಪ್ರದರ್ಶಿಸುತ್ತದೆ.
  3. ಡೌನ್‌ಲೋಡ್ ಮಾದರಿ ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ. ನಿಮ್ಮ ಸಾಧನಕ್ಕೆ CSV ಫೈಲ್ ಡೌನ್‌ಲೋಡ್ ಆಗುತ್ತದೆ.

img

ಫೈಲ್ ತೆರೆಯಿರಿ ಮತ್ತು ದಾಖಲೆಗಳನ್ನು ಸೇರಿಸಿ. CSV ಫೈಲ್‌ನಲ್ಲಿ ಫಾರ್ಮ್ಯಾಟ್ ಡೇಟಾವನ್ನು ಹೇಗೆ ಬಳಸುವುದು ಎಂಬುದನ್ನು ತಿಳಿಯಲು ಇಲ್ಲಿ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಫೈಲ್ ಆಯ್ಕೆಗಳು

ಪ್ರಕರಣಗಳಿಗೆ ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಿ * ಪ್ರಕರಣಕ್ಕೆ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಲು ಈ ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ. * ಕೇಸ್_ರೋಲ್_ಆರ್ಡರ್ ಸೆಲ್ ಅನ್ನು ಖಾಲಿ ಬಿಡಿ. * ಬಳಕೆದಾರರ ಪುಟಕ್ಕೆ ಹೋಗಿ ಮತ್ತು ಈ ವ್ಯಕ್ತಿಯ ಟ್ಯಾಗ್ ಐಡಿಯನ್ನು ನಕಲಿಸಿ. ಟ್ಯಾಗ್ ಐಡಿ ಕ್ಷೇತ್ರವು ಖಾಲಿಯಾಗಿದ್ದರೆ, ಅನನ್ಯ ಐಡಿಯನ್ನು ನಮೂದಿಸಿ, ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ನಂತರ ಐಡಿಯನ್ನು CSV ಫೈಲ್‌ಗೆ ನಕಲಿಸಿ.

ಪ್ರಕರಣಗಳಿಗೆ ನಿಯೋಜಿಸಲಾದ ಬಳಕೆದಾರರನ್ನು ಮಾರ್ಪಡಿಸಿ * ಪ್ರಕರಣಕ್ಕೆ ನಿಯೋಜಿಸಲಾದ ಪ್ರಸ್ತುತ ಬಳಕೆದಾರರಿಗೆ ಡೇಟಾವನ್ನು ನವೀಕರಿಸಲು ಈ ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ. * case_ID ಕಾಲಮ್‌ಗಾಗಿ ಕೇಸ್ ಟ್ಯಾಗ್ ಐಡಿ ಬಳಸಿ.

img

ಪ್ರಕರಣಗಳಿಗೆ ನಿಯೋಜಿಸಲಾದ ಬಳಕೆದಾರರನ್ನು ತೆಗೆದುಹಾಕಿ * ಪ್ರಕರಣಕ್ಕೆ ನಿಯೋಜಿಸಲಾದ ಬಳಕೆದಾರರನ್ನು ತೆಗೆದುಹಾಕಲು ಈ ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ.

ಫೈಲ್ ಅನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಲಾಗುತ್ತಿದೆ

ನೀವು ಈಗಿನಿಂದಲೇ ಫೈಲ್ ಅನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಬಹುದು ಅಥವಾ ನಂತರದ ದಿನಾಂಕದಲ್ಲಿ ಆಮದು ಸೇರಿಸಿ ಸಂವಾದ ಪೆಟ್ಟಿಗೆಗೆ ಹಿಂತಿರುಗಿ ಮತ್ತು ಈ ಹಂತಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿ.

  1. ಆಮದು ಸೇರಿಸಿ ಸಂವಾದ ಪೆಟ್ಟಿಗೆಯನ್ನು ತೆರೆಯಿರಿ.
  2. CSV ಫೈಲ್‌ಗಾಗಿ ಅನನ್ಯ ಹೆಸರನ್ನು ನಮೂದಿಸಿ.
  3. ನಿಮ್ಮ ಸಾಧನದಿಂದ ಫೈಲ್ ಅನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಲು ಪ್ಲಸ್ (+) ಐಕಾನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಟೈಪ್ ಫೀಲ್ಡ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ. (ಮೇಲೆ ನೋಡು)
  5. ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.

ಸಂಸ್ಕರಣೆ

ಫೈಲ್ ಮತ್ತು ಸಿಸ್ಟಮ್ ಲೋಡ್ನಲ್ಲಿನ ದಾಖಲೆಗಳ ಸಂಖ್ಯೆಯನ್ನು ಅವಲಂಬಿಸಿ, ಆಮದು ಪ್ರಕ್ರಿಯೆಗೊಳಿಸಲು ವೇದಿಕೆಯು ಹಲವಾರು ನಿಮಿಷಗಳನ್ನು ತೆಗೆದುಕೊಳ್ಳಬಹುದು.

ಆಮದು ಪೂರ್ಣಗೊಂಡಾಗ, ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಬಳಕೆದಾರರ ದಾಖಲೆಗಳನ್ನು ಸರಿಹೊಂದಿಸುತ್ತದೆ (ಅಂದರೆ ಸೇರಿಸುತ್ತದೆ, ಮಾರ್ಪಡಿಸುತ್ತದೆ ಅಥವಾ ತೆಗೆದುಹಾಕುತ್ತದೆ) ಪ್ರಕರಣಗಳ ಪುಟದಲ್ಲಿ.

ವೈಫಲ್ಯ

ಆಮದು ಕಾರ್ಯಾಚರಣೆ ವಿಫಲವಾದರೆ, ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ವಿಫಲ ಸಂದೇಶವನ್ನು ನೀಡುತ್ತದೆ. ಹೆಚ್ಚಿನ ಸಂದರ್ಭಗಳಲ್ಲಿ, ವೈಫಲ್ಯದ ಕಾರಣ ಡೇಟಾ ಫಾರ್ಮ್ಯಾಟಿಂಗ್ ಆಗಿದೆ.

ವೈಫಲ್ಯದ ಕಾರಣವನ್ನು ನೋಡಲು:

  1. ಕ್ರಿಯೆಯ ಶೀರ್ಷಿಕೆಯ ಅಡಿಯಲ್ಲಿ ವೀಕ್ಷಿಸಿ ಕ್ಲಿಕ್ ಮಾಡಿ.
  2. ಮುಂದಿನ ಪುಟದಲ್ಲಿ, ವಿಫಲ ಐಕಾನ್ ಮೇಲೆ ಮೌಸ್ ಅನ್ನು ಸುಳಿದಾಡಿ. ಟೂಲ್ಟಿಪ್ ದೋಷವನ್ನು ತೋರಿಸುತ್ತದೆ.
  3. CVS ಫೈಲ್‌ಗೆ ಹೋಗಿ ಮತ್ತು ದೋಷವನ್ನು ಸರಿಹೊಂದಿಸಿ ಮತ್ತು ಸರಿಪಡಿಸಿ.
  4. ಫೈಲ್ ಅನ್ನು ಮತ್ತೆ ಅಪ್ಲೋಡ್ ಮಾಡಿ.
