ಲಾಗ್ ಇನ್ ಮಾಡುವುದು ಹೇಗೆ

ನಿಮ್ಮ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಕೇಸ್ ಆಕ್ಟಿವ್ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡುವುದು ಹೇಗೆ ಎಂದು ತಿಳಿಯಿರಿ.

ಲಾಗಿನ್ ಆಗುತ್ತಿದೆ

  1. ಲಾಗಿನ್ ಪುಟಕ್ಕೆ ಹೋಗಿ.
  2. LOGIN ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ನಿಮ್ಮ ಇಮೇಲ್ ನಮೂದಿಸಿ.
  4. ಮುಂದೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಖಾತೆಯನ್ನು ಪರಿಶೀಲಿಸಲಾಗುತ್ತಿದೆ

  1. ಹೊಸ ಬ್ರೌಸರ್ ಟ್ಯಾಬ್ ತೆರೆಯಿರಿ ಮತ್ತು ನಿಮ್ಮ ಇಮೇಲ್ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಆರು-ಅಂಕಿಯ ಕೋಡ್ ಅನ್ನು ನಕಲಿಸಿ.
  3. ಕೇಸ್ ಆಕ್ಟಿವ್ ಟ್ಯಾಬ್‌ಗೆ ಹಿಂತಿರುಗಿ.
  4. ಕ್ಷೇತ್ರಗಳಲ್ಲಿ ಕೋಡ್ ಅನ್ನು ಅಂಟಿಸಿ. ಹೊಸ ವಿಂಡೋ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ.

ಕೇಸ್ ಆಕ್ಟಿವ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟವನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ.

  • ಗಮನಿಸಿ ನಿರ್ವಾಹಕ ಸವಲತ್ತುಗಳನ್ನು ಹೊಂದಿರುವ ಬಳಕೆದಾರರಿಗೆ APP ಬಟನ್‌ನ ಪಕ್ಕದಲ್ಲಿ ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ನಿರ್ವಾಹಕ ಬಟನ್ ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ.
