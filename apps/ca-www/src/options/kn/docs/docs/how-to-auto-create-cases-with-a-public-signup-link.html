ಸಾರ್ವಜನಿಕ ಸೈನ್‌ಅಪ್ ಲಿಂಕ್‌ನೊಂದಿಗೆ ಕೇಸ್‌ಗಳನ್ನು ಸ್ವಯಂ ರಚಿಸುವುದು ಹೇಗೆ

ಒಂದೇ ಸೈನ್‌ಅಪ್ ಲಿಂಕ್ ಅನ್ನು ಕಳುಹಿಸುವ ಮೂಲಕ ಪ್ರಕರಣವನ್ನು ಹೇಗೆ ರಚಿಸುವುದು (ಮತ್ತು ಅದೇ ಸಮಯದಲ್ಲಿ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸುವುದು) ಹೇಗೆ ಎಂಬುದನ್ನು ಈ ಲೇಖನವು ನಿರ್ವಾಹಕರಿಗೆ ತೋರಿಸುತ್ತದೆ.

ಲಿಂಕ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಲಾಗುತ್ತಿದೆ

  1. ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ನಿರ್ವಾಹಕ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ನಿರ್ವಾಹಕ ಫಲಕದಲ್ಲಿ, ಸೆಟ್ಟಿಂಗ್‌ಗಳ ಮೆನು ಟ್ಯಾಬ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಸಿಸ್ಟಮ್ ಸೆಟಪ್ ಉಪಮೆನು ಟ್ಯಾಬ್ ಅನ್ನು ಆಯ್ಕೆ ಮಾಡಲಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.

ಅನುಮತಿಸಿ ಸೈನ್ ಅಪ್ ಕ್ಷೇತ್ರವನ್ನು ಆನ್‌ಗೆ ಹೊಂದಿಸಲಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ. (ಅಗತ್ಯವಿದೆ) ಈ ವೈಶಿಷ್ಟ್ಯವನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ, ಕೇಸ್ ಆಕ್ಟಿವ್ ಅನನ್ಯ ಲಿಂಕ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ.

  1. ಸ್ವಯಂ ರಚಿಸಿ ಕೇಸ್ ಕ್ಷೇತ್ರವನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಆನ್ ಆಯ್ಕೆಮಾಡಿ. ಕೇಸ್ ಬಳಕೆದಾರರ ಪಾತ್ರದ ಕ್ಷೇತ್ರವನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ.
  2. ಪಾತ್ರವನ್ನು ಆಯ್ಕೆಮಾಡಿ. (ಕೆಳಗೆ ನೋಡಿ)

ಲಿಂಕ್ ಕಳುಹಿಸಲಾಗುತ್ತಿದೆ

ಒಬ್ಬರು ಅಥವಾ ಹೆಚ್ಚಿನ ಜನರೊಂದಿಗೆ ಲಿಂಕ್ ಅನ್ನು ಹಂಚಿಕೊಳ್ಳಿ. ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡುವ ಮತ್ತು ಸೈನ್‌ಅಪ್ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ಪ್ರತಿಯೊಬ್ಬ ಬಳಕೆದಾರರು ಪ್ರಕರಣಗಳ ಪುಟದಲ್ಲಿ ಹೊಸ ಪ್ರಕರಣವನ್ನು ರಚಿಸಲು ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಅನ್ನು ಪ್ರಚೋದಿಸುತ್ತಾರೆ ಮತ್ತು ಪ್ರಕರಣದಲ್ಲಿ ಬಳಕೆದಾರರನ್ನು ಸ್ವಯಂ ನಿಯೋಜಿಸುತ್ತಾರೆ.
