ಬಳಕೆದಾರರು ಹೊಸ ಖಾತೆಯನ್ನು ಹೇಗೆ ರಚಿಸುತ್ತಾರೆ

ನಿರ್ವಾಹಕರು ಬಳಕೆದಾರರನ್ನು ಕೇಸ್‌ಗೆ ಸೇರಿಸಿದ ನಂತರ: * ಬಳಕೆದಾರರು ಕೇಸ್‌ಆಕ್ಟಿವ್‌ಗೆ ಹೊಸಬರಾಗಿದ್ದರೆ, ಬಳಕೆದಾರರು ಖಾತೆಯನ್ನು ರಚಿಸಲು ವರ್ಕ್‌ಫ್ಲೋ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸಬೇಕು ಮತ್ತು ಪ್ರಕರಣವನ್ನು ಪ್ರವೇಶಿಸಲು ಆಹ್ವಾನವನ್ನು ಸ್ವೀಕರಿಸಬೇಕು. * ಬಳಕೆದಾರರು ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಕೇಸ್ಯಾಕ್ಟಿವ್ ಅಕೌಂಟ್ ಹೊಂದಿದ್ದರೆ, ಬಳಕೆದಾರರು ಪ್ರಕರಣವನ್ನು ಪ್ರವೇಶಿಸಲು ಆಹ್ವಾನವನ್ನು ಸ್ವೀಕರಿಸಬೇಕು.

ಬಳಕೆದಾರರು ತಮ್ಮ ಖಾತೆಯನ್ನು ರಚಿಸಲು ಪೂರ್ಣಗೊಳಿಸಬೇಕಾದ ಹಂತಗಳನ್ನು ಈ ಲೇಖನವು ವಿವರಿಸುತ್ತದೆ.

ಶುರುವಾಗುತ್ತಿದೆ

ಕ್ಯಾಸೆಕ್ಟಿವ್ ಅಪ್ಲಿಕೇಶನ್‌ಗೆ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಲು ಎರಡು ಮಾರ್ಗಗಳಿವೆ

  • ನಿರ್ವಾಹಕರಿಂದ: ನಿರ್ವಾಹಕರು ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ UI ಅಥವಾ ಆಮದು ಉಪಕರಣದ ಮೂಲಕ ಬಳಕೆದಾರರನ್ನು ರಚಿಸಿದರೆ, ಲಾಗಿನ್ ಮಾಡಲು ಮುಂದುವರಿಯಲು ನಿರ್ವಾಹಕರು ಬಳಕೆದಾರರಿಗೆ account.caseactive.com ಗೆ ಲಿಂಕ್ ಅನ್ನು ಕಳುಹಿಸಬೇಕು.
  • ಬಳಕೆದಾರರಿಂದ: ಸೈನ್ ಅಪ್ ಲಿಂಕ್ ಅನ್ನು ಬಳಸಿಕೊಂಡು ಬಳಕೆದಾರರು ಖಾತೆಯನ್ನು ರಚಿಸಿದರೆ, ಲಾಗಿನ್ ಮಾಡಲು ಮುಂದುವರಿಯಲು ಬಳಕೆದಾರರನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ account.caseactive.com ಗೆ ಮರುನಿರ್ದೇಶಿಸಲಾಗುತ್ತದೆ.

ಲಾಗಿನ್ ಆಗುತ್ತಿದೆ

ಬಳಕೆದಾರರು ತಮ್ಮ CaseActive ಖಾತೆಗೆ ಮೊದಲ ಬಾರಿಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿದಾಗ, ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಬಳಕೆದಾರರನ್ನು ಹೀಗೆ ಕೇಳುತ್ತದೆ:

  • ಅವರ ಇಮೇಲ್ ಖಾತೆಯನ್ನು ತೆರೆಯಿರಿ ಮತ್ತು ನಂತರ ಆರು-ಅಂಕಿಯ ಭದ್ರತಾ ಕೋಡ್ ಅನ್ನು ನಕಲಿಸಿ/ಅಂಟಿಸಿ
  • ಖಾತೆಯ ಪ್ರೊಫೈಲ್ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸಿ

ಬಳಕೆದಾರರು UPDATE ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿದ ನಂತರ, ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟವು ಪ್ರದರ್ಶಿಸುತ್ತದೆ. ಪುಟವು ಅಪ್ಲಿಕೇಶನ್‌ನ ಹೆಸರು (ಕಂಪೆನಿ ಹೆಸರು), ಬಳಕೆದಾರರ ಪ್ರಕಾರ (ಪಾತ್ರ), ಸ್ಥಿತಿ ಮತ್ತು ಆಹ್ವಾನ ಬಟನ್ ಅನ್ನು ತೋರಿಸುತ್ತದೆ.

ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪ್ರವೇಶಿಸಲು, ಬಳಕೆದಾರರು ಆಹ್ವಾನವನ್ನು ಸ್ವೀಕರಿಸಬೇಕು. ಬಳಕೆದಾರರು ಹೀಗೆ ಮಾಡಬೇಕು:

  1. ಆಮಂತ್ರಣ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  2. ಅನುಮತಿಗಳನ್ನು ಪರಿಶೀಲಿಸಿ.
  3. ನಿಯಮಗಳನ್ನು ಪರಿಶೀಲಿಸಲು ಮತ್ತು ಸಂಪಾದಿಸಲು (ಐಚ್ಛಿಕ) ಅಧಿಸೂಚನೆಗಳ ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಪ್ರವೇಶವನ್ನು ಅನುಮೋದಿಸಿ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಬಳಕೆದಾರರು ಅನುಮೋದಿಸಿದಾಗ, ವೇದಿಕೆಯು ಬಳಕೆದಾರರ ಪುಟದಲ್ಲಿ APP ಬಟನ್ ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ. ಈ ಬಟನ್ ಬಳಕೆದಾರರಿಗೆ ನಿಯೋಜಿಸಲಾದ ಪ್ರಕರಣಗಳಿಗೆ ಪ್ರವೇಶವನ್ನು ನೀಡುತ್ತದೆ.
