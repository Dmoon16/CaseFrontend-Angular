ಸಾರ್ವಜನಿಕ ಸೈನ್‌ಅಪ್ ಲಿಂಕ್‌ನೊಂದಿಗೆ ಬಳಕೆದಾರರನ್ನು ಸ್ವಯಂ ನಿಯೋಜಿಸುವುದು ಹೇಗೆ

ನಿರ್ವಾಹಕರು ಸಿಸ್ಟಂ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಹೇಗೆ ಕಾನ್ಫಿಗರ್ ಮಾಡಬೇಕೆಂದು ಕಲಿಯುತ್ತಾರೆ ಇದರಿಂದ ಹೊಸ ಬಳಕೆದಾರರನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೊಸದಾಗಿ ರಚಿಸಲಾದ ಕೇಸ್‌ಗೆ ನಿಯೋಜಿಸಲಾಗುತ್ತದೆ.

ಸೆಟಪ್ ಪುಟವನ್ನು ತೆರೆಯಲಾಗುತ್ತಿದೆ

ಸಿಸ್ಟಮ್ ಸೆಟಪ್ ಪುಟವನ್ನು ತೆರೆಯುವುದು ಮೊದಲ ಹಂತವಾಗಿದೆ.

  1. ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ನಿರ್ವಾಹಕ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ಸಿಸ್ಟಮ್ಸ್ ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಉಪಮೆನುವಿನಲ್ಲಿ ಸಿಸ್ಟಮ್ ಸೆಟಪ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. USER SIGNUP ಶಿರೋನಾಮೆಗೆ ಕೆಳಗೆ ಸ್ಕ್ರಾಲ್ ಮಾಡಿ.

ಸಿಸ್ಟಮ್ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಲಾಗುತ್ತಿದೆ

  1. ಸೈನ್ಅಪ್ ಅನ್ನು ಅನುಮತಿಸಿ ಮತ್ತು ಸ್ವಯಂ ರಚಿಸಿ ಕೇಸ್ ಡ್ರಾಪ್-ಡೌನ್ ಮೆನುಗಳನ್ನು ಆನ್‌ಗೆ ಹೊಂದಿಸಿ.
  2. ಪಾತ್ರ ಮೆನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ (ಡೀಫಾಲ್ಟ್ ಅನ್ನು ಶಿಫಾರಸು ಮಾಡಲಾಗಿದೆ).
  3. ಸಾರ್ವಜನಿಕ ಬಳಕೆದಾರರ ಸೈನ್ ಅಪ್ ಲಿಂಕ್ ಅನ್ನು ನಕಲಿಸಿ ಮತ್ತು ಹಂಚಿಕೊಳ್ಳಿ.

ಲಿಂಕ್ ಕಳುಹಿಸಲಾಗುತ್ತಿದೆ

ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡುವ ಮತ್ತು ಸೈನ್‌ಅಪ್ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ಪ್ರತಿಯೊಬ್ಬ ಬಳಕೆದಾರರು ಕೇಸ್‌ಗಳ ಪುಟದಲ್ಲಿ ಹೊಸ ಪ್ರಕರಣವನ್ನು ರಚಿಸಲು ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಅನ್ನು ಪ್ರಚೋದಿಸುತ್ತಾರೆ ಮತ್ತು ಆ ಸಂದರ್ಭದಲ್ಲಿ ಬಳಕೆದಾರರನ್ನು ಸ್ವಯಂ-ನಿಯೋಜಿಸುತ್ತಾರೆ.
