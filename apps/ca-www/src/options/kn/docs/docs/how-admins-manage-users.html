ನಿರ್ವಾಹಕರು ಬಳಕೆದಾರರನ್ನು ಹೇಗೆ ನಿರ್ವಹಿಸುತ್ತಾರೆ

ಅಪ್ಲಿಕೇಶನ್‌ಗಳು ಮತ್ತು ಪ್ರಕರಣಗಳನ್ನು ಯಾರು ಪ್ರವೇಶಿಸಬಹುದು ಎಂಬುದನ್ನು ನಿರ್ವಾಹಕರು ರಚಿಸುತ್ತಾರೆ, ಮಾರ್ಪಡಿಸುತ್ತಾರೆ ಅಥವಾ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸುತ್ತಾರೆ. ಬಳಕೆದಾರರು ಮೊದಲ ಬಾರಿಗೆ ಪ್ಲಾಟ್‌ಫಾರ್ಮ್‌ಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿದಾಗ, ಅವರು ಪ್ರೊಫೈಲ್ ವರ್ಕ್‌ಫ್ಲೋ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸಬೇಕಾಗುತ್ತದೆ.

  • ಹೊಸ ಬಳಕೆದಾರರ ಪ್ರೊಫೈಲ್ ವರ್ಕ್‌ಫ್ಲೋ ಕುರಿತು ಇನ್ನಷ್ಟು ತಿಳಿದುಕೊಳ್ಳಲು ಇಲ್ಲಿ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಬಳಕೆದಾರರನ್ನು ನಿರ್ವಹಿಸುವುದು

ಕೆಳಗಿನ ಕೋಷ್ಟಕವು ಬಳಕೆದಾರರನ್ನು ನಿರ್ವಹಿಸಲು ಐದು ಮಾರ್ಗಗಳನ್ನು ತೋರಿಸುತ್ತದೆ. ಹೇಗೆ ಎಂದು ತಿಳಿಯಲು ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.

ವಿಧಾನ ಮಾದರಿ ಕ್ರಿಯೆಗಳು ವಿವರಗಳು
UI ಬಳಸುವುದು UI ಸೇರಿಸಿ ಮತ್ತು ಮಾರ್ಪಡಿಸಿ ನಿರ್ವಾಹಕ ಬಟನ್ > ಬಳಕೆದಾರರು
ಪ್ರಕರಣವನ್ನು ರಚಿಸುವಾಗ ಮತ್ತು ಬಳಕೆದಾರರನ್ನು ಕೇಸ್‌ಗೆ ನಿಯೋಜಿಸಿ UI ಸೇರಿಸಿ ನಿರ್ವಾಹಕ ಬಟನ್ > ಪ್ರಕರಣಗಳು > ಹೊಸ ಬಟನ್ > ಕೇಸ್ ಪುಟವನ್ನು ರಚಿಸಿ
ಸಾರ್ವಜನಿಕ ಸೈನ್ ಅಪ್ ಲಿಂಕ್ ಬಳಸಿ ಬಳಕೆದಾರರನ್ನು ಸ್ವಯಂ ರಚಿಸಿ ಬಳಕೆದಾರರ ಕ್ರಿಯೆ ಸೇರಿಸಿ ನಿರ್ವಹಣೆ ಬಟನ್> ಸೆಟ್ಟಿಂಗ್‌ಗಳು> ಸಿಸ್ಟಮ್ ಸೆಟಪ್ ಪುಟ
ಆಮದು ಸಾಧನ UI ಸೇರಿಸಿ ಮತ್ತು ಮಾರ್ಪಡಿಸಿ ನಿರ್ವಹಣೆ ಬಟನ್> ಇಂಟಿಗ್ರೇಷನ್‌ಗಳು> ಆಮದುಗಳು

ಅಧಿಸೂಚನೆಗಳು

ಅಪ್ಲಿಕೇಶನ್‌ಗೆ ಸೇರಿಸಿದಾಗ ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಬಳಕೆದಾರರಿಗೆ ಅಧಿಸೂಚನೆಯನ್ನು ಕಳುಹಿಸುವುದಿಲ್ಲ. ಆದಾಗ್ಯೂ, ನಿರ್ವಾಹಕರು ಕೇಸ್ ಅಸೈನ್‌ಮೆಂಟ್‌ಗಳಿಗೆ ಅಧಿಸೂಚನೆಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಬಹುದು.
