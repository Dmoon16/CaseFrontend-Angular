ಆಮದು ಉಪಕರಣದೊಂದಿಗೆ ಬಳಕೆದಾರ ಕೇಸ್ ನಿಯೋಜನೆಗಳ ಡೇಟಾವನ್ನು ಫಾರ್ಮ್ಯಾಟ್ ಮಾಡುವುದು

CSV ಫೈಲ್‌ಗಳಲ್ಲಿನ ಡೇಟಾ ಕ್ಷೇತ್ರಗಳ ಪಟ್ಟಿಯನ್ನು ಕೆಳಗೆ ನೀಡಲಾಗಿದೆ. ಪ್ರತಿಯೊಂದು ಆಮದು ಆಯ್ಕೆಯು ಕ್ಷೇತ್ರಗಳ ವಿಭಿನ್ನ ಸಂಯೋಜನೆಯನ್ನು ಹೊಂದಿದೆ.

ಕೇಸ್_ಐಡಿ

  • ಕೇಸ್ ಬಳಕೆದಾರ ಅಥವಾ ಟ್ಯಾಗ್ ಐಡಿಯನ್ನು ನಮೂದಿಸಿ.
  • ಪ್ರಕರಣಗಳ ಪುಟಕ್ಕೆ ಹೋಗಿ > ಎಡಿಟ್ > ಟ್ಯಾಗ್ ಐಡಿ ಅಥವಾ ಕೇಸ್ ಪುಟಕ್ಕೆ ಹೋಗಿ > ಎಡಿಟ್ > ಕೇಸ್ ಐಡಿ url ನಲ್ಲಿ.

ಬಳಕೆದಾರರ ಗುರುತು

  • ಬಳಕೆದಾರರ ID ಅಥವಾ ಟ್ಯಾಗ್ ID ಅನ್ನು ನಮೂದಿಸಿ.
  • ಬಳಕೆದಾರರ ಪುಟಕ್ಕೆ ಹೋಗಿ > ಸಂಪಾದಿಸು > ಟ್ಯಾಗ್ ಐಡಿ ಅಥವಾ ಬಳಕೆದಾರರ ಪುಟಕ್ಕೆ ಹೋಗಿ > ಸಂಪಾದಿಸು > URL ನಲ್ಲಿ ಬಳಕೆದಾರ ID.

ಪ್ರಕರಣ_ಪಾತ್ರ_ಆದೇಶ

  • ಕೋಶವನ್ನು ಖಾಲಿ ಬಿಡಿ.

case_role_ID

  • ನಮೂದಿಸಿ: ಗ್ರಾಹಕರು, ಸಿಬ್ಬಂದಿ, ಅಥವಾ ಡೀಫಾಲ್ಟ್.

ಪ್ರಕರಣ_ಅಧಿಸೂಚನೆ

  • 0 = ಇಲ್ಲ
  • 1 = ಹೌದು, ಬಳಕೆದಾರರಿಗೆ ಇಮೇಲ್ ಅಧಿಸೂಚನೆಯನ್ನು ಕಳುಹಿಸಿ.
