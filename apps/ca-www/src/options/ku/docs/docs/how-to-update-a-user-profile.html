Meriv çawa Profîla Bikarhêner nûve dike

Ev gotar nîşanî bikarhêneran dide ka meriv çawa agahdariya li ser rûpelê Profîla xwe nûve dike.

  • Bikarhêner dikarin profîlên xwe nûve bikin.
  • Rêvebir dikarin rûpela Profîla her bikarhênerek nûve bikin.

Ji bo bikarhêneran

Gava yekem ev e ku hûn rûpelê Profîl vekin. Bikarhêner û rêvebir van gavan bişopînin da ku rûpela Profîla xwe nûve bikin.

  1. Têkeve hesabê xweya Case Active.
  2. Li tabloya Profîl li serê rûpelê bikirtînin.
  3. Guhertinan bikin.
  4. Bikirtînin Save.

Pir zevî vebijarkî ne, û hin zevî jî hewce ne. Qadên pêwîst ev in:

  • Nav
  • Paşnav
  • Navnîşana emailê
  • Ziman
  • Qada demê

Ji bo rêveberan

Rêvebir jî dikarin agahdariya profîlê ji bo bikarhênerên din nûve bikin. Ev sê qadan di bin sernavê Agahdariya HOST USER.

  1. Têkeve hesabê xweya rêveberê Case Active.
  2. Serlêdanek bibînin û bişkoka ADMIN bikirtînin.
  3. Bikarhêner li ser menuya jorîn bikirtînin.
  4. Biçe jêr-menuyê û navnîşa bikarhênerê ya ku hûn dixwazin xwe bigihînin hilbijêrin (ÇAKTIV an DISABLED). Mîhenga xwerû Çalak e.
  5. Bikarhênerek di navnîşê de bibînin û bikirtînin Biguherîne.
  6. Guhertinên ku hewce dike çêbikin. (li jêr binêre)
  7. Bikirtînin Save.

Agahdariya bikarhêner biguherînin

Admin dikarin agahdariya bikarhêner biguherînin.

  1. Tîpa Bikarhêner (Bikarhêner an Rêvebir) saz bikin.
  2. Rewşa Bikarhêner Set (Çalak ji bo Neçalak).
  3. Nasnameyek Tagê ya yekta binivîse. (bixwe)
  4. Bikirtînin Save.
