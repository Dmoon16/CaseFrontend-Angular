Meriv Çawa Têkeve

Fêr bibe ka meriv çawa têkeve hesabê xweya Case Active ya heyî.

Têketinê

  1. Biçe rûpela têketinê .
  2. Bişkojka LOGIN bikirtînin.
  3. E-nameya xwe binivîse.
  4. NEXT bikirtînin.

Verastkirina hesabek

  1. Tabloyek gerokek nû vekin û têkevin hesabê e-nameya xwe.
  2. Koda şeş-reqemî kopî bikin.
  3. Vegere tabloya Case Active.
  4. Kodê di nav qadan de bişînin. Pencereyek nû tê xuyang kirin.

Case Active rûpela Apps nîşan dide.

  • Nîşe Platform ji bo bikarhênerên xwedan mafên rêveberiyê bişkojek ADMIN li kêleka bişkoja APP nîşan dide.
