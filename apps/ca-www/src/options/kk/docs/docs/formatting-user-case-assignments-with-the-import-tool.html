Пайдаланушы регистрінің тағайындау деректерін импорттау құралымен пішімдеу

Төменде CSV файлдарындағы деректер өрістерінің тізімі берілген. Әрбір импорттау опциясында өрістердің әртүрлі тіркесімі болады.

case_ID

  • Іс пайдаланушысын немесе тег идентификаторын енгізіңіз.
  • Істер бетіне өтіңіз > Өңдеу > Тег идентификаторы НЕМЕСЕ url мекенжайындағы Істер бетіне > Өңдеу > Іс идентификаторына өтіңіз.

Қолданушының ID

  • Пайдаланушының идентификаторын немесе тег идентификаторын енгізіңіз.
  • Пайдаланушылар бетіне өтіңіз > Өңдеу > Тег идентификаторы НЕМЕСЕ Пайдаланушылар беті > Өңдеу > URL мекенжайындағы пайдаланушы идентификаторы тармағына өтіңіз.

іс_рөлінің_тәртібі

  • Ұяшықты бос қалдырыңыз.

іс_рөлінің_идентификаторы

  • Енгізіңіз: клиенттер, қызметкерлер немесе әдепкі.

case_notify

  • 0 = жоқ
  • 1 = Иә, пайдаланушыға электрондық пошта хабарландыруын жіберіңіз.
