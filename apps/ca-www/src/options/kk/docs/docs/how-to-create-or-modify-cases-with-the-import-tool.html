Импорттау құралымен істерді қалай жасауға немесе өзгертуге болады

Бұл мақала әкімшілерге Импорттау құралымен бар істерді жасау немесе өзгерту жолын көрсетеді.

Импорттау құралын ашу

  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Басқару тақтасында Интеграция қойындысын, содан кейін ішкі мәзірде ИМПОРТТАР түймесін басыңыз.
  4. ЖАҢА түймесін басыңыз. Импорттарды қосу тілқатысу терезесі жаңа шолғыш қойындысында көрсетіледі.

img

Істерді құру

Импорттарды қосу тілқатысу терезесі ашылғаннан кейін:

  1. Түр өрісін басып, опцияны таңдаңыз (төменде қараңыз). Сілтеме көрсетіледі.
  2. Үлгіні жүктеп алу сілтемесін басыңыз. CSV файлы құрылғыңызға жүктеледі.

Содан кейін файлды ашып, істерді қосыңыз.

Түр

Түр өрісін басқан кезде ашылмалы мәзір көрсетіледі. Жаппай импорттау үшін екі опцияның бірін таңдаңыз.

  • Ескертпе Ашылмалы мәзірде көптеген опциялар бар, бірақ тек екеуі істі басқаруға қатысты. Әрбір опцияда басқа үлгі CSV файлы бар.

Істерді қосыңыз

  • Істерді қарапайым жаппай жүктеп салу үшін осы опцияны таңдаңыз.
  • Істерді жасау импорт түріне арналған баған тақырыптары бар CSV файлын ашу үшін Жүктеп алу үлгісі сілтемесін басыңыз.

Істерді өзгерту

  • Ағымдағы істерді жаңарту үшін осы опцияны таңдаңыз.
  • Істерді өзгерту импорт түріне арналған баған тақырыптары бар CSV файлын ашу үшін Жүктеп алу үлгісі сілтемесін басыңыз.

Файлды жүктеп салу

Файлды бірден жүктеп салуыңызға немесе кейінірек импортты қосу тілқатысу терезесіне оралып, осы қадамдарды орындауға болады.

  1. Импорттарды қосу тілқатысу терезесін ашыңыз.
  2. CSV файлы үшін бірегей атау енгізіңіз.
  3. Файлды құрылғыдан жүктеп салу үшін плюс (+) белгішесін басыңыз.
  4. Түр өрісін басып, опцияны таңдаңыз. (жоғарыдан қараңыз)
  5. САҚТАУ түймесін басыңыз.

Пішімдеу

Бұл міндетті өрістер және бірінші талаптар. Пішімдеу мәліметтерін қарау үшін осы жерді басыңыз.

tag_id (немесе case_ID) бірегей идентификатор (әріптік-сандық таңбаларды пайдаланыңыз, сызықшалар қабылданады)

case_status 0 = жабық 1 = белсенді

күні_ашылған ай/күн/жыл мм/кк/жжж

істің бірегей сипаттамасы туралы (әріптік-цифрлық таңбаларды пайдаланыңыз, сызықшалар қабылданады)

Өңдеу

Файлдағы жазбалар санына және жүйе жүктемесіне байланысты, платформаға импортты өңдеуге бірнеше минут кетуі мүмкін.

img

Импорттау аяқталғанда, платформа әкімшінің Істер бақылау тақтасында іс жазбаларын реттейді (яғни қосады немесе өзгертеді).

Сәтсіздік

Импорттау әрекеті сәтсіз болса, платформа сәтсіз хабарды қайтарады. Көп жағдайда сәтсіздіктің себебі деректерді пішімдеу болып табылады.

Ақаулықтың себебін көру үшін:

  1. ӘРЕКЕТ тақырыбының астындағы Көрініс түймесін басыңыз.
  2. Келесі бетте тінтуірді «Сәтсіз» белгішесіне апарыңыз. Құрал кеңесі қатені көрсетеді.
  3. CVS файлына өтіп, қатені реттеңіз және түзетіңіз.
  4. Файлды қайта жүктеп салыңыз.
