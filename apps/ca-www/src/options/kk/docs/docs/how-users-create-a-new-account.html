Пайдаланушылар жаңа тіркелгіні қалай жасайды

Әкімші пайдаланушыны іске қосқаннан кейін: * егер пайдаланушы caseactive үшін жаңа болса, пайдаланушы тіркелгі жасау үшін жұмыс процесін аяқтауы және іске кіруге шақыруды қабылдауы керек. * егер пайдаланушыда бар кейсбелсенді шот болса, пайдаланушы іске кіруге шақыруды қабылдауы керек.

Бұл мақалада пайдаланушы тіркелгісін жасау үшін орындау керек қадамдарды сипаттайды.

Басталу

Пайдаланушыны кейсбелсенді қолданбаға қосудың екі жолы бар

  • Әкімші: Әкімші пайдаланушыны UI платформасы немесе Импорттау құралы арқылы жасаса, жүйеге кіруді жалғастыру үшін әкімші пайдаланушыға account.caseactive.com сілтемесін жіберуі керек.
  • Пайдаланушы: Егер пайдаланушы тіркелу сілтемесін пайдаланып тіркелгіні жасаса, кіруді жалғастыру үшін пайдаланушы автоматты түрде account.caseactive.com сайтына қайта бағытталады.

Жүйеге кіру

Пайдаланушы CaseActive тіркелгісіне бірінші рет кіргенде, платформа пайдаланушыдан:

  • электрондық пошта тіркелгісін ашыңыз, содан кейін алты таңбалы қауіпсіздік кодын көшіріңіз/қойыңыз
  • тіркелгі профилін толтырыңыз

Пайдаланушы UPDATE түймесін басқаннан кейін қолданбалар беті көрсетіледі. Бет қолданбаның атын (компания атауы), пайдаланушы түрін (рөлін), күйді және ШАҚЫРУ түймешігін көрсетеді.

img

Қолданбаға қол жеткізу үшін пайдаланушы шақыруды қабылдауы керек. Пайдаланушы:

  1. ШАҚЫРУ түймесін басыңыз.
  2. Рұқсаттарды қарап шығыңыз.
  3. Шарттарды қарап шығу және өңдеу (міндетті емес) үшін Хабарландырулар қойындысын басыңыз.
  4. ҚОСУДЫ РЕТТЕУ түймесін басыңыз.

Пайдаланушы мақұлдаған кезде, платформа пайдаланушы бетінде APP түймешігін көрсетеді. Бұл түйме пайдаланушыға тағайындалған істерге рұқсат береді.
