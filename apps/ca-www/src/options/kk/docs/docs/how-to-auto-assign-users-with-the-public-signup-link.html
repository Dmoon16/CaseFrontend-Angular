Жалпы тіркелу сілтемесі арқылы пайдаланушыларды автоматты түрде қалай тағайындауға болады

Әкімшілер жүйе параметрлерін конфигурациялауды үйренеді, осылайша жаңа пайдаланушылар автоматты түрде жаңадан жасалған іске автоматты түрде тағайындалады.

Орнату бетін ашу

Бірінші қадам - Жүйелерді орнату бетін ашу.

  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Ішкі мәзірде Жүйелер қойындысын, содан кейін ЖҮЙЕ ОРНАТУ түймесін басыңыз.
  4. ПАЙДАЛАНУШЫНЫҢ ТІРКЕЛУ тақырыбына төмен жылжыңыз.

Жүйе параметрлерін конфигурациялау

  1. Тіркеуге рұқсат беру және Істі автоматты түрде жасау ашылмалы мәзірлерін Қосу күйіне орнатыңыз.
  2. Рөл мәзірін басып, опцияны таңдаңыз (әдепкі ұсынылады).
  3. Жалпы пайдаланушыға тіркелу сілтемесін көшіріп, бөлісіңіз.

Сілтеме жіберу

Сілтемені басқан және тіркелуді аяқтаған әрбір пайдаланушы Істер бетінде жаңа істі жасау үшін платформаны іске қосады және сол жағдайда пайдаланушыны автоматты түрде тағайындайды.
