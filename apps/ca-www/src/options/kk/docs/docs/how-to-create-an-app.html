Қолданбаны қалай жасауға болады

Бұл мақала жаңа пайдаланушыларға тегін сынақ есептік жазбасын қалай жасау керектігін көрсетеді. Case Active жаңа пайдаланушыға осы тіркелгі үшін әкімші артықшылықтарын автоматты түрде береді.

Тіркелгі жасау

  1. Жүйеге кіру бетіне өтіңіз.
  2. Жұмысты бастау түймесін басыңыз.
  3. Электрондық пошта мекенжайын енгізіңіз.
  4. Келесі түймесін басыңыз.

img

Есептік жазбаны растау

  1. Жаңа шолғыш қойындысын ашып, электрондық пошта тіркелгіңізге кіріңіз.
  2. Алты таңбалы кодты көшіріңіз.
  3. Іс белсенді қойындысына оралыңыз.
  4. Кодты өрістерге қойыңыз. Жаңа терезе көрсетіледі.

Жаңа терезеде:

  1. Қажетті өрістерді толтырыңыз.
  2. Қызмет көрсету шарттарын оқып, келісу үшін құсбелгіні қойыңыз.
  3. ТЕГІН ТЕГІН ЖАСАУ түймесін басыңыз.

img

Қолданбаны орнату аяқталуда

Келесі бетте жұмыс процесі көрсетіледі. Қолданбаны орнатуды аяқтау үшін қадамдарды орындаңыз. Ақпаратты немесе тіркелгі күйін өңдеу немесе жаңарту үшін осы беттерге әрқашан оралуға болады.

Қолданба туралы бетінде компания атауы мен веб-сайт өрістерін (міндетті) толтырып, «Келесі» түймесін басыңыз.

img

Төлем туралы ақпарат бетінде Төлем жоспары өрісін басып, опцияларды қарап шығыңыз. Әдепкі параметр тегін. ЖІБЕРУ түймесін басыңыз.

Пішінді жібергеннен кейін Case Active қолданбалар бетін көрсетеді. Сіз жасаған немесе тиесілі әрбір қолданба үшін бетте мыналар көрсетіледі:

  1. Қолданба атауы
  2. Сіздің түріңіз (пайдаланушы рөлі)
  3. Күй

Қолданбалар беті көптеген әкімші функцияларын іске қосу нүктесі болып табылады:

  1. Іске қатысты мәселелер бойынша байланысу, құжаттарды толтыру, бейне чат (және т.б.) үшін APP түймесін басыңыз.
  2. Әкімші тақтасын ашу үшін ADMIN түймесін басыңыз. Бұл жерде әкімшілер пайдаланушыларды, істерді, интеграцияларды және т.б. басқарады.
  3. Тіркелгі төлемін және хабарландыру параметрлерін басқару үшін көп нүктені басыңыз.
  • Ескерту Case Active әкімші пайдаланушылары үшін ADMIN түймешігін көрсетеді. Әкімші артықшылықтары жоқ пайдаланушылар ADMIN түймесін көрмейді (төмендегі экранда көрсетілгендей).
