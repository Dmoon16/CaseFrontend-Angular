Пайдаланушы профилін қалай жаңартуға болады

Бұл мақала пайдаланушыларға профиль бетіндегі ақпаратты жаңарту жолын көрсетеді.

  • Пайдаланушылар өздерінің профильдерін жаңарта алады.
  • Әкімшілер кез келген пайдаланушының профиль бетін жаңарта алады.

Пайдаланушылар үшін

Бірінші қадам - Профиль бетін ашу. Пайдаланушылар мен әкімшілер өздерінің жеке профиль бетін жаңарту үшін осы қадамдарды орындаңыз.

  1. Case Active есептік жазбаңызға кіріңіз.
  2. Беттің жоғарғы жағындағы Профиль қойындысын басыңыз.
  3. Өзгерістер енгізіңіз.
  4. Сақтау түймесін басыңыз.

Көптеген өрістер міндетті емес және кейбір өрістер міндетті болып табылады. Міндетті өрістер:

  • Аты
  • Тек
  • Электрондық пошта
  • Тіл
  • Уақыт белдеуі

Админдер үшін

Әкімшілер сонымен қатар басқа пайдаланушылар үшін профиль ақпаратын жаңарта алады. Бұл HOST ПАЙДАЛАНУШЫ АҚПАРАТ тақырыбының астындағы үш өрісті қамтиды.

  1. Case Active әкімші тіркелгісіне кіріңіз.
  2. Қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Жоғарғы мәзірде «Пайдаланушылар» түймесін басыңыз.
  4. Ішкі мәзірге өтіп, қол жеткізгіңіз келетін пайдаланушылар тізімін таңдаңыз (БЕЛСЕНДІ немесе ӨШІРІЛГЕН). Әдепкі параметр - Белсенді.
  5. Тізімнен пайдаланушыны тауып, Өңдеу түймесін басыңыз.
  6. Қажетті өзгерістерді енгізіңіз. (төменде қараңыз)
  7. Сақтау түймесін басыңыз.

Пайдаланушы ақпаратын өзгерту

Әкімшілер пайдаланушы ақпаратын өзгерте алады.

  1. Пайдаланушы түрін орнатыңыз (пайдаланушы немесе менеджер).
  2. Пайдаланушы күйін орнатыңыз (Белсендіден Өшіруліге дейін).
  3. Бірегей тег идентификаторын енгізіңіз. (міндетті емес)
  4. Сақтау түймесін басыңыз.
