Әкімшілер жағдайларға пайдаланушы тағайындауларын қалай басқарады

Пайдаланушыларды және іс тапсырмаларын басқарудың бес жолы бар. Қалай білу үшін сілтемені басыңыз.

Әдіс Түр Әрекеттер Егжей
Істі құру кезінде UI қосу Әкімші түймесі > Істер > Жаңа түйме > Пайдаланушыларды тағайындау түймесі
Қоғамдық тіркелу сілтемесі арқылы пайдаланушыны автоматты түрде тағайындау Пайдаланушы әрекеті қосу Әкімші түймесі > Параметрлер > Жүйені орнату беті
Бар жағдайда UI пайдалану UI қосу және жою Әкімші түймесі > Істер > Пайдаланушыларды тағайындау
Импорттау құралы UI қосу, өзгерту және жою Әкімші түймесі > Интеграциялар > Импорттар
API пайдалану (Жақында) API қосу, өзгерту және жою Жоқ
