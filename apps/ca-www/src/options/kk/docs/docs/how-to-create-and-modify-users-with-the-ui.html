UI көмегімен пайдаланушыларды қалай жасауға және өзгертуге болады

Бұл мақала әкімшілерге платформаның пайдаланушы интерфейсімен пайдаланушыларды жасау және жазбаларды жаңарту жолын көрсетеді. Сондай-ақ әкімшілер пайдаланушыларды жаңа іске қосып, бар істерді өңдей алады.

Пайдаланушыны құру

  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Әкімші тақтасында Пайдаланушылар мәзірі қойындысын басыңыз.
  4. ЖАҢА түймесін басыңыз. Жаңа терезе көрсетіледі.

img

Өрістерді толтырып, САҚТАУ түймесін басыңыз. Case Active атауын осы қолданбамен байланысты пайдаланушылар тізіміне қосады.

img

Пайдаланушыны өзгерту

Пайдаланушы жазбаларының аз санын жаңарту үшін Өңдеу құралын пайдаланыңыз және әрбір пайдаланушыға өзгертулер енгізіңіз.

  • Ескерту Бір отырыста пайдаланушы жазбаларының көп санын өзгерту үшін Импорттау құралын пайдалануды қарастырыңыз.
  1. Әкімші тақтасында Пайдаланушылар мәзірі қойындысын басыңыз.
  2. Пайдаланушыны тауып, Өңдеу түймесін басыңыз.
  3. Өзгерістерді енгізіп, САҚТАУ түймесін басыңыз.

img

Келесі қадамдар

Case Active пайдаланушыға электрондық пошта хабарландыруын жібермейді. Әкімшілер пайдаланушымен байланысып, оларға Case Active басты бетіне сілтеме жіберуі керек. Содан кейін пайдаланушы КІРУ түймесін басып, жаңа тіркелулер үшін жұмыс процесін аяқтайды.

  • Бұл мақалада жаңа пайдаланушылар үшін тіркелу жұмыс процесі сипатталады.
