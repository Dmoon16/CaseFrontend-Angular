Импорт құралы арқылы пайдаланушыларды қалай жасауға немесе өзгертуге болады

Бұл мақала әкімшілерге Импорт құралымен пайдаланушыларды жасау жолын көрсетеді.

Импорттау құралын ашу

Әкімшілер жаңа пайдаланушыларды жаппай жүктеп салу және бұрыннан бар пайдаланушы жазбаларын өзгерту үшін Импорттау құралын пайдалана алады.

  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Басқару тақтасында Интеграция қойындысын, содан кейін ішкі мәзірде ИМПОРТТАР түймесін басыңыз.
  4. ЖАҢА түймесін басыңыз. Импорттарды қосу тілқатысу терезесі жаңа шолғыш қойындысында көрсетіледі.

img

Жазбаларды құру

Импорттарды қосу тілқатысу терезесі ашылғаннан кейін:

  1. Түр өрісін басып, опцияны таңдаңыз (төменде қараңыз). Сілтеме көрсетіледі.
  2. Үлгіні жүктеп алу сілтемесін басыңыз. CSV файлы құрылғыңызға жүктеледі.

Содан кейін файлды ашып, жазбаларды қосыңыз. CSV файлында пішім деректерін пайдалану жолын білу үшін осы жерді басыңыз.

img

Түр

Түр өрісін басқан кезде ашылмалы мәзір көрсетіледі. Жаппай импорттау үшін екі опцияның бірін таңдаңыз.

:леп белгісі: ЕСКЕРТПЕ Ашылмалы мәзірде көптеген опциялар бар, бірақ тек екеуі ғана Импорттау құралына қатысты. Әрбір опцияда басқа үлгі CSV файлы бар.

Пайдаланушыларды қосыңыз

  • Пайдаланушы жазбаларын қарапайым жаппай жүктеп салу үшін осы опцияны таңдаңыз.
  • Пайдаланушыларды қосу Импорт түрі үшін баған тақырыптары бар CSV файлын ашу үшін Жүктеп алу үлгісі сілтемесін басыңыз.

Пайдаланушыларды өзгерту

  • Ағымдағы пайдаланушылар үшін деректерді жаңарту үшін осы опцияны таңдаңыз.
  • Пайдаланушыларды өзгерту импорт түрі үшін баған тақырыптары бар CSV файлын ашу үшін Жүктеп алу үлгісі сілтемесін басыңыз.

Файлды жүктеп салу

Файлды бірден жүктеп салуыңызға немесе кейінірек импортты қосу тілқатысу терезесіне оралып, осы қадамдарды орындауға болады.

  1. Импорттарды қосу тілқатысу терезесін ашыңыз.
  2. CSV файлы үшін бірегей атау енгізіңіз.
  3. Файлды құрылғыдан жүктеп салу үшін плюс (+) белгішесін басыңыз.
  4. Түр өрісін басып, опцияны таңдаңыз. (жоғарыдан қараңыз)
  5. САҚТАУ түймесін басыңыз.

Өңдеу

Файлдағы және жүйе жүктемесіндегі жазбалар санына байланысты платформаға импортты өңдеуге бірнеше минут кетуі мүмкін.

img

Импорттау аяқталғанда, платформа әкімшінің Пайдаланушылар бақылау тақтасында пайдаланушы жазбаларын реттейді (яғни қосады, өзгертеді немесе өшіреді).

Сәтсіздік

Импорттау әрекеті сәтсіз болса, платформа сәтсіз хабарды қайтарады. Көп жағдайда сәтсіздіктің себебі деректерді пішімдеу болып табылады.

Ақаулықтың себебін көру үшін:

  1. Әрекет тақырыбының астындағы Көрініс түймесін басыңыз.
  2. Келесі бетте тінтуірді «Сәтсіз» белгішесіне апарыңыз. Құрал кеңесі қатені көрсетеді.
  3. CVS файлына өтіп, қатені реттеңіз және түзетіңіз.
  4. Файлды қайта жүктеп салыңыз.
