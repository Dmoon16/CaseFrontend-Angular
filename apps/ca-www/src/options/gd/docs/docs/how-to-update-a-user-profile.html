Mar a bheir thu ùrachadh air pròifil cleachdaiche

Tha an artaigil seo a’ sealltainn do luchd-cleachdaidh mar as urrainn dhaibh fiosrachadh ùrachadh air an duilleag Pròifil aca.

  • Faodaidh luchd-cleachdaidh na pròifilean aca fhèin ùrachadh.
  • Faodaidh luchd-rianachd duilleag Pròifil neach-cleachdaidh sam bith ùrachadh.

Airson luchd-cleachdaidh

Is e a’ chiad cheum an duilleag Pròifil fhosgladh. Bidh luchd-cleachdaidh agus rianairean a’ leantainn nan ceumannan seo gus an duilleag Pròifil aca fhèin ùrachadh.

  1. Log a-steach don chunntas Case Active agad.
  2. Cliog air an tab Profile aig mullach na duilleige.
  3. Dèan atharrachaidhean.
  4. Cliog Save.

Tha a’ mhòr-chuid de raointean roghainneil, agus tha feum air cuid de raointean. Is iad na raointean a tha a dhìth:

  • A’ chiad ainm
  • An t-ainm mu dheireadh
  • Seòladh puist-d
  • Cànain
  • Sòn ùine

Airson admins

Faodaidh luchd-rianachd am fiosrachadh pròifil ùrachadh airson luchd-cleachdaidh eile cuideachd. Tha seo a’ toirt a-steach trì raointean fon cheann HOST USER INFO.

  1. Log a-steach don chunntas rianachd Case Active agad.
  2. Lorg app agus cliog air a’ phutan ADMIN.
  3. Cliog Luchd-cleachdaidh air a 'chlàr gu h-àrd.
  4. Rach don fho-chlàr agus tagh an liosta cleachdaiche a tha thu airson faighinn a-steach (GNÌOMHACH no DISABLED). Is e Active an suidheachadh bunaiteach.
  5. Lorg cleachdaiche air an liosta agus cliog Deasaich.
  6. Dèan atharrachaidhean mar a dh’ fheumar. (faic gu h-ìosal)
  7. Cliog Save.

Atharraich fiosrachadh cleachdaiche

Faodaidh luchd-rianachd fiosrachadh luchd-cleachdaidh atharrachadh.

  1. Suidhich an seòrsa cleachdaiche (cleachdaiche no manaidsear).
  2. Suidhich an Inbhe Cleachdaiche (Gnìomhach gu Ciorramach).
  3. Cuir a-steach ID Tag sònraichte. (roghainneil)
  4. Cliog Save.
