Dar formato a los datos de gestión de usuarios con la herramienta de importación

A continuación se muestra una lista de campos de datos en archivos CSV. Cada opción de importación tiene una combinación diferente de campos.

nombre de usuario

  • Introduzca la dirección de correo electrónico del usuario.

nombre de pila

  • Introduzca el nombre del usuario.

apellido

  • Introduzca el apellido del usuario.

lugar

  • Esta columna verifica el idioma preferido del usuario.
  • Ingrese en para inglés, sp para español.

zonainfo

  • Esto establece la zona horaria del usuario. Introduzca el continente y la ciudad sin espacios (utilice un guión bajo).
  • Debe usar una opción en el menú desplegable Perfil > IDIOMA Y REGIÓN > Zona horaria.
  • Utilice la zona horaria que mejor se ajuste. Por ejemplo, Boston no está en el menú desplegable, por lo que podemos usar Nueva York (o cualquier ciudad en la zona horaria EST/EDT).

host_tag_id

  • Introduzca una etiqueta permanente.
  • az, 0-9, -, / están permitidos.

caso_estado

  • 0 = cerrado
  • 1 = activo
