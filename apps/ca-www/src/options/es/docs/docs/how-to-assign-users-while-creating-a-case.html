Cómo asignar usuarios al crear un caso

Este artículo muestra a los administradores cómo asignar un usuario al crear un nuevo caso. Si el usuario aún no está registrado en la plataforma, puede crear agregar un nuevo usuario mientras crea un nuevo caso.

Creando un caso

  1. Inicie sesión en la cuenta de administrador.
  2. Busque una aplicación en la página Aplicaciones y haga clic en el botón ADMINISTRAR.
  3. Haga clic en la pestaña del menú Casos.
  4. Haga clic en el botón NUEVO. Aparece una nueva ventana.

  • Complete los campos para crear un caso y luego desplácese hacia abajo hasta el encabezado Usuarios asignados.

Asignación de un usuario

Usuario actual

Siga estos pasos para asignar un usuario actual al nuevo caso.

Haga clic en el botón ASIGNAR USUARIOS y complete los campos.

  1. Haga clic en el menú desplegable Usuario y seleccione un identificador (por ejemplo, correo electrónico).
  2. Introduzca un valor en el campo en blanco (por ejemplo, dirección de correo electrónico).
  3. Si el registro de usuario existe en la plataforma, el campo sugiere automáticamente un nombre.
  4. Seleccione el usuario correcto.
  5. Seleccione un rol de usuario (clientes, predeterminado o personal).
  6. Opcionalmente, marque la casilla para enviar al usuario una notificación por correo electrónico.
  7. Clic en Guardar.

Case Active agrega al usuario al caso. Ahora los administradores pueden hacer clic en Guardar para cerrar la ventana, agregar un nuevo usuario o eliminar un usuario.

Agregar un nuevo usuario

Si la plataforma no puede encontrar un registro actual, haga clic en el botón para agregar un nuevo usuario.

imagen

Complete los campos y luego haga clic en GUARDAR para agregar el usuario a la plataforma. A continuación, establezca un rol de usuario y haga clic en GUARDAR nuevamente para asignar el usuario al caso. Finalmente, haga clic en GUARDAR una vez más para guardar el caso.
