Cómo crear casos automáticamente con un enlace de registro público

Este artículo muestra a los administradores cómo crear un caso (y agregar un usuario al mismo tiempo) enviando un solo enlace de registro.

Localizando el enlace

  1. Inicie sesión en la cuenta de administrador.
  2. Busque una aplicación en la página Aplicaciones y haga clic en el botón ADMINISTRAR.
  3. En el panel de administración, haga clic en la pestaña del menú Configuración.
  4. Asegúrese de que la pestaña del submenú CONFIGURACIÓN DEL SISTEMA esté seleccionada.

Configuración de ajustes

Desplácese hacia abajo hasta el encabezado REGISTRO DE USUARIO.

Asegúrese de que el campo Permitir registro esté activado. (requerido) Cuando esta característica está habilitada, Case Active genera un enlace único.

  1. Haga clic en el campo Crear caso automáticamente y seleccione Activado. Se muestra el campo Rol de usuario de caso.
  2. Seleccione una función. (vea abajo)

imagen

Roles de usuario de caso

Hay tres roles de usuario de caso.

  • clientes (recomendado)
  • defecto
  • personal

Enviando el enlace

Comparte el enlace con una o más personas. Cada usuario que hace clic en el enlace y completa el registro activa la plataforma para generar un nuevo caso en la página de casos y asignar automáticamente al usuario en el caso.
