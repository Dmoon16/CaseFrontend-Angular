Cómo gestionan los administradores las asignaciones de usuarios a los casos

Hay cinco formas de administrar usuarios y asignaciones de casos. Haga clic en un enlace para saber cómo.

Método Escribe Comportamiento Detalles
Al crear un caso interfaz de usuario agregar Botón Admin > Casos > botón Nuevo > botón Asignar usuarios
Asignación automática de usuarios usando un enlace de registro público Acción del usuario agregar Botón Admin > Configuración > página Configuración del sistema
Uso de la interfaz de usuario en un caso existente interfaz de usuario agregar eliminar Botón de administración > Casos > Asignar usuarios
herramienta de importación interfaz de usuario añadir, modificar y eliminar Botón de administración > Integraciones > Importaciones
