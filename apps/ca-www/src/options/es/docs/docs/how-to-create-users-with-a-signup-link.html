Cómo crear usuarios con un enlace de registro

Este artículo muestra a los administradores cómo crear usuarios enviando un enlace de registro.

Configuración del enlace de registro

  1. Haga clic en el enlace Enviarles un registro en la parte inferior del cuadro de diálogo Agregar usuario. Aparece una nueva página.
  2. Configura las opciones de registro. (vea abajo)

imagen

Asegúrese de que el campo Permitir registro esté activado. (requerido) Cuando está habilitado, Case Active genera un enlace único.

Revise las opciones a continuación y luego haga clic en el icono para copiar el enlace. Puede colocar el enlace en su sitio web o enviar el enlace a una o más personas por correo electrónico o cualquier canal de comunicación.

imagen

Creando un caso

Opcionalmente, puede crear automáticamente un caso para cada nuevo usuario y asignarle una función de usuario.

  1. Haga clic en el campo Crear caso automáticamente y seleccione Activado. Aparecen los campos Rol de usuario del caso.
  2. Haga clic en Rol de usuario de caso y seleccione una opción. (vea abajo)

imagen

Roles de usuario de caso

Hay tres roles de usuario de caso.

  • clientela
  • defecto
  • personal

Enviando el enlace

Comparte el enlace con una o más personas. Cada persona hace clic en el enlace y completa el flujo de trabajo para activar su cuenta Case Active. Una vez que el usuario completa el flujo de trabajo de registro, Case Active agrega al usuario a la lista de usuarios.

imagen

Próximos pasos

Case Active envía al usuario una notificación por correo electrónico con el enlace de registro. El usuario hace clic en el enlace y completa el flujo de trabajo para nuevos registros.
