Dar formato a los datos de gestión de casos con la herramienta de importación

A continuación se muestra una lista de campos obligatorios en archivos CSV para administrar casos con la herramienta Importar.

nombre de usuario

  • Introduzca la dirección de correo electrónico del usuario.

tag_id (una identificación a la que puede hacer referencia más adelante)

  • un identificador único (use caracteres alfanuméricos, se aceptan guiones)

caso_estado

  • 0 = cerrado
  • 1 = activo

fecha_abierto

  • mes día año
  • mm/dd/aaaa

sobre

  • una descripción única del caso (use caracteres alfanuméricos, se aceptan guiones)

imagen
