امپورٹ ٹول کے ساتھ کیسز کا نظم کرنے کے لیے نیچے CSV فائلوں میں مطلوبہ فیلڈز کی فہرست ہے۔

صارف نام

  • صارف کا ای میل ایڈریس درج کریں۔

tag_id (ایک ID جس کا آپ بعد میں حوالہ دے سکتے ہیں)

  • ایک منفرد شناخت کنندہ (حروف نمبری حروف کا استعمال کریں، ہائفنز قبول کیے جاتے ہیں)

کیس_سٹیٹس

  • 0 = بند
  • 1 = فعال

تاریخ_کھول گئی۔

  • مہینہ دن سال
  • mm/dd/yyyy

  • کیس کی انوکھی تفصیل (حروف عددی حروف کا استعمال کریں، ہائفنز قبول کیے جاتے ہیں)
