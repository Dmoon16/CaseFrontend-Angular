UI مینو کے ذریعے کیسز کیسے بنائیں یا ان میں ترمیم کریں۔

یہ مضمون منتظمین کو دکھاتا ہے کہ پلیٹ فارم کے صارف انٹرفیس کے ساتھ کیس کیسے بنایا جائے۔ ایڈمنز صارفین کو نئے کیس میں شامل کر سکتے ہیں اور موجودہ کیسز کو اپ ڈیٹ کر سکتے ہیں۔

کیس بنانا

  1. ایڈمن اکاؤنٹ میں لاگ ان کریں۔
  2. ایپس پیج پر ایک ایپ تلاش کریں اور ایڈمن بٹن پر کلک کریں۔
  3. ایڈمن پینل پر، کیسز مینو ٹیب پر کلک کریں۔
  4. نیو بٹن پر کلک کریں۔ ایک نئی ونڈو دکھائی دیتی ہے۔

کیس میں ترمیم کرنا

بہت کم کیسز کو اپ ڈیٹ کرنے کے لیے، ایڈیٹ ٹول کا استعمال کریں اور ہر انفرادی کیس میں تبدیلیاں کریں۔

  • نوٹ ایک نشست میں کیسز کی ایک بڑی تعداد کو اپ ڈیٹ کرنے کے لیے، امپورٹ ٹول استعمال کرنے پر غور کریں۔
  1. ایڈمن اکاؤنٹ میں لاگ ان کریں۔
  2. ایپس پیج پر ایک ایپ تلاش کریں اور ایڈمن بٹن پر کلک کریں۔
  3. ایڈمن پینل پر، کیسز مینو ٹیب پر کلک کریں۔
  4. ایک کیس تلاش کریں اور ترمیم پر کلک کریں۔
