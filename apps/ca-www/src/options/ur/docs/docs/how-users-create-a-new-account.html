شروع ہوا چاہتا ہے

کیس ایکٹیو ایپ میں صارف کو شامل کرنے کے دو طریقے ہیں۔

  • بذریعہ منتظم: اگر منتظم پلیٹ فارم UI یا امپورٹ ٹول کے ذریعے صارف تخلیق کرتا ہے، تو منتظم کو لاگ ان کرنے کے لیے صارف کو account.caseactive.com پر ایک لنک بھیجنا چاہیے۔
  • صارف کے ذریعہ: اگر صارف نے سائن اپ لنک کا استعمال کرتے ہوئے ایک اکاؤنٹ بنایا ہے، تو صارف کو لاگ ان کرنے کے لیے خود بخود account.caseactive.com پر بھیج دیا جائے گا۔
