관리자가 사용자를 관리하는 방법

관리자는 앱 및 사례에 액세스할 수 있는 사용자를 생성, 수정 또는 비활성화합니다. 사용자가 처음으로 플랫폼에 로그인하면 프로필 워크플로를 완료해야 합니다.

  • 새 사용자 프로필 작업 과정에 대해 자세히 알아보려면 여기 를 클릭하십시오.

사용자 관리

아래 표는 사용자를 관리하는 5가지 방법을 보여줍니다. 방법을 알아보려면 링크를 클릭하십시오.

방법 유형 행위 세부
UI 사용 UI 추가 및 수정 관리자 버튼 > 사용자
사례를 만들고 사례에 사용자를 할당하는 동안 UI 추가하다 관리자 버튼 > 사례 > 새로 만들기 버튼 > 사례 만들기 페이지
공개 가입 링크를 사용하여 사용자 자동 생성 사용자 작업 추가하다 관리자 버튼 > 설정 > 시스템 설정 페이지
가져오기 도구 UI 추가 및 수정 관리 버튼 > 통합 > 가져오기
API 사용(출시 예정) API 추가 및 수정 해당 없음

알림

플랫폼은 사용자가 앱에 추가될 때 사용자에게 알림을 보내지 않습니다. 그러나 관리자는 사례 할당에 대한 알림을 활성화할 수 있습니다.
