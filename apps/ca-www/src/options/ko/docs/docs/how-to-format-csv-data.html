CSV 데이터 형식 지정 방법

다음은 CSV 파일의 데이터 필드 목록입니다. 각 가져오기 옵션에는 서로 다른 필드 조합이 있습니다.

사용자 이름

  • 사용자의 이메일 주소를 입력합니다.

주어진_이름

  • 사용자의 이름을 입력합니다.

family_name

  • 사용자의 성을 입력합니다.

장소

  • 이 열은 사용자의 기본 언어를 확인합니다.
  • 영어는 en, 스페인어는 sp를 입력합니다.

구역 정보

  • 이것은 사용자 시간대를 설정합니다. 공백 없이 대륙과 도시를 입력하십시오(밑줄 사용).
  • 프로필 > 언어 및 지역 > 시간대 드롭다운 메뉴에서 옵션을 사용해야 합니다.
  • 가장 적합한 시간대를 사용하십시오. 예를 들어 보스턴은 드롭다운에 없으므로 뉴욕(또는 EST/EDT 시간대의 모든 도시)을 사용할 수 있습니다.

host_tag_id

  • 영구 라벨을 입력하세요.
  • az, 0-9, -, /가 허용됩니다.

케이스_상태

  • 0 = 닫힘
  • 1 = 활성
