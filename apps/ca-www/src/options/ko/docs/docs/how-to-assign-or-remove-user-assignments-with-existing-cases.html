기존 사례로 사용자 할당을 할당하거나 제거하는 방법

이 문서에서는 관리자에게 사례 사용자를 할당하고 제거하는 방법을 보여줍니다. 또한 사용자 데이터를 수정하는 방법도 보여줍니다.

기존 사례 열기

  1. 관리자 계정에 로그인합니다.
  2. 앱 페이지에서 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 사례 메뉴 탭을 클릭합니다.
  4. 하위 메뉴에서 ACTIVE가 선택되어 있는지 확인하십시오.
  5. ACTION 열까지 아래로 스크롤합니다.

이미지

할당 또는 제거

사용자 할당을 클릭하여 사례에 사용자를 할당하거나 사례에서 사용자를 제거합니다(플랫폼 제외). 다음 페이지에서 추가를 클릭하고 워크플로를 완료합니다. 제거 를 클릭하여 사용자를 삭제합니다.

이미지

수정하다

사례 페이지에서 사용자 데이터를 편집하거나 업데이트할 수 없습니다. 대신 다음 단계를 따르십시오.

  1. 사용자 메뉴 탭을 클릭합니다.
  2. 사용자 이름을 찾고 편집 을 클릭합니다.
  3. 변경하고 저장을 클릭합니다.
