공개 가입 링크로 사용자를 자동 할당하는 방법

관리자는 새 사용자가 새로 자동 생성된 사례에 자동으로 할당되도록 시스템 설정을 구성하는 방법을 배웁니다.

설정 페이지 열기

첫 번째 단계는 시스템 설정 페이지를 여는 것입니다.

  1. 관리자 계정에 로그인합니다.
  2. 앱 페이지에서 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 시스템 탭을 클릭한 다음 하위 메뉴에서 시스템 설정을 클릭합니다.
  4. USER SIGNUP 제목까지 아래로 스크롤합니다.

시스템 설정 구성

  1. 등록 허용 및 사례 자동 생성 드롭다운 메뉴를 켜짐으로 설정합니다.
  2. 역할 메뉴를 클릭하고 옵션을 선택합니다(기본값 권장).
  3. 공개 사용자 가입 링크를 복사하여 공유합니다.

링크 보내기

링크를 클릭하고 가입을 완료하는 각 사용자는 플랫폼을 트리거하여 사례 페이지에서 새 사례를 생성하고 해당 사례에 사용자를 자동 할당합니다.
