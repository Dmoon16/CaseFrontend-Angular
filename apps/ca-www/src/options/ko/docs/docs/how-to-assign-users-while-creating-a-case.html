사례를 만드는 동안 사용자를 할당하는 방법

이 문서에서는 관리자에게 새 사례를 만드는 동안 사용자를 할당하는 방법을 보여줍니다. 사용자가 플랫폼에 아직 등록되지 않은 경우 새 사례를 만드는 동안 새 사용자를 추가할 수 있습니다.

사례 만들기

  1. 관리자 계정에 로그인합니다.
  2. 앱 페이지에서 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 사례 메뉴 탭을 클릭합니다.
  4. 신규 버튼을 클릭합니다. 새 창이 표시됩니다.

이미지

  • 필드를 채워 사례를 만든 다음 할당된 사용자 제목까지 아래로 스크롤합니다.

사용자 할당

현재 사용자

다음 단계에 따라 현재 사용자를 새 사례에 할당합니다.

사용자 할당 버튼을 클릭하고 필드를 채웁니다.

  1. 사용자 드롭다운을 클릭하고 식별자(예: 이메일)를 선택합니다.
  2. 빈 필드에 값을 입력합니다(예: 이메일 주소).
  3. 사용자 레코드가 플랫폼에 존재하는 경우 필드는 이름을 자동으로 제안합니다.
  4. 올바른 사용자를 선택하십시오.
  5. 사용자 역할(클라이언트, 기본 또는 스태프)을 선택합니다.
  6. 필요에 따라 상자를 선택하여 사용자에게 이메일 알림을 보냅니다.
  7. 저장을 클릭합니다.

이미지

케이스 활성이 케이스에 사용자를 추가합니다. 이제 관리자는 저장을 클릭하여 창을 닫거나 새 사용자를 추가하거나 사용자를 제거할 수 있습니다.

이미지

새 사용자 추가

플랫폼에서 현재 레코드를 찾을 수 없는 경우 버튼을 클릭하여 새 사용자를 추가하십시오.

이미지

필드를 채운 다음 SAVE를 클릭하여 사용자를 플랫폼에 추가합니다. 그런 다음 사용자 역할을 설정하고 SAVE를 다시 클릭하여 케이스에 사용자를 할당합니다. 마지막으로 SAVE를 한 번 더 클릭하여 케이스를 저장합니다.
