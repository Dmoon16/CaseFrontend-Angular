앱을 만드는 방법

이 문서에서는 새 사용자에게 무료 평가판 계정을 만드는 방법을 보여줍니다. Case Active는 이 계정에 대한 새 사용자 관리자 권한을 자동으로 부여합니다.

계정 만들기

  1. 로그인 페이지 로 이동합니다.
  2. 시작하기 버튼을 클릭합니다.
  3. 이메일 주소를 입력하세요.
  4. 다음을 클릭합니다.

이미지

계정 확인

  1. 새 브라우저 탭을 열고 이메일 계정에 로그인합니다.
  2. 6자리 코드를 복사합니다.
  3. Case Active 탭으로 돌아갑니다.
  4. 필드에 코드를 붙여넣습니다. 새 창이 표시됩니다.

새 창에서:

  1. 필수 필드를 채우십시오.
  2. 서비스 약관을 읽고 동의하는 확인란을 선택합니다.
  3. 무료 계정 만들기를 클릭합니다.

이미지

앱 설정 완료

다음 페이지에 워크플로가 표시됩니다. 단계를 완료하여 앱 설정을 완료하세요. 언제든지 이 페이지로 돌아와 정보 또는 계정 상태를 수정하거나 업데이트할 수 있습니다.

앱 정보 페이지에서 회사 이름 및 웹사이트 필드(필수)를 입력하고 다음을 클릭합니다.

이미지

청구 정보 페이지에서 지불 계획 필드를 클릭하고 옵션을 검토하십시오. 기본 설정은 무료입니다. 제출을 클릭합니다.

이미지

양식을 제출하면 Case Active에 앱 페이지가 표시됩니다. 생성하거나 속한 각 앱에 대해 페이지에 다음이 표시됩니다.

  1. 앱 이름
  2. 귀하의 유형(사용자 역할)
  3. 상태

이미지

앱 페이지는 많은 관리 기능의 시작점입니다.

  1. APP 버튼을 클릭하여 케이스 관련 문제에 대해 의사소통하고, 문서를 작성하고, 화상 채팅 등을 할 수 있습니다.
  2. 관리자 버튼을 클릭하여 관리자 패널을 엽니다. 여기서 관리자는 사용자, 사례, 통합 등을 관리합니다.
  3. 계정 청구 및 알림 설정을 관리하려면 줄임표를 클릭하십시오.
  • 참고 Case Active는 관리 사용자를 위한 ADMIN 버튼을 표시합니다. 관리자 권한이 없는 사용자에게는 ADMIN 버튼이 표시되지 않습니다(아래 화면 참조).
