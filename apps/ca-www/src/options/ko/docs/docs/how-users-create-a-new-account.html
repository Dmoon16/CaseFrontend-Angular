사용자가 새 계정을 만드는 방법

관리자가 사례에 사용자를 추가한 후: * 사용자가 caseactive를 처음 사용하는 경우 사용자는 계정을 생성하고 사례에 액세스하기 위한 초대를 수락하는 워크플로를 완료해야 합니다. * 사용자에게 기존 케이스 활성 계정이 있는 경우 케이스에 액세스하려면 초대를 수락해야 합니다.

이 문서에서는 사용자가 계정을 만들기 위해 완료해야 하는 단계를 설명합니다.

시작하기

케이스활성 앱에 사용자를 추가할 수 있는 두 가지 방법이 있습니다.

  • 관리자: 관리자가 플랫폼 UI 또는 가져오기 도구 를 통해 사용자를 생성하는 경우 관리자는 사용자에게 account.caseactive.com에 대한 링크를 보내 로그인을 진행해야 합니다.
  • 사용자: 사용자가 가입 링크 를 사용하여 계정을 만든 경우 사용자는 자동으로 account.caseactive.com으로 리디렉션되어 로그인을 진행합니다.

로그인

사용자가 CaseActive 계정에 처음으로 로그인하면 플랫폼에서 사용자에게 다음과 같은 메시지를 표시합니다.

  • 이메일 계정을 연 다음 6자리 보안 코드를 복사/붙여넣기
  • 계정 프로필 작성

이미지

사용자가 업데이트를 클릭하면 앱 페이지가 표시됩니다. 페이지에는 앱 이름(회사 이름), 사용자 유형(역할), 상태 및 초대 버튼이 표시됩니다.

이미지

앱에 액세스하려면 사용자가 초대를 수락해야 합니다. 사용자는 다음을 수행해야 합니다.

  1. 초대 버튼을 클릭합니다.
  2. 권한을 검토하십시오.
  3. 알림 탭을 클릭하여 조건을 검토하고 편집합니다(선택 사항).
  4. 액세스 승인을 클릭합니다.

이미지

사용자가 승인하면 플랫폼은 사용자 페이지에 APP 버튼을 표시합니다. 이 버튼은 할당된 사례에 대한 사용자 액세스를 제공합니다.

사용자 설정을 편집하려면 사용자가 줄임표를 클릭합니다.

이미지

사용자는 다음을 수행할 수 있습니다. * 앱 액세스 권한 제거 * 알림 설정 수정

이미지

사용자가 워크플로를 완료하면 관리자의 사용자 목록에 사용자 이름이 나타납니다. 목록에는 로그인 여부에 관계없이 모든 사용자가 표시됩니다.
