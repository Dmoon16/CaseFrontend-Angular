가져오기 도구를 사용하여 사용자 사례 할당 데이터 형식 지정

다음은 CSV 파일의 데이터 필드 목록입니다. 각 가져오기 옵션에는 서로 다른 필드 조합이 있습니다.

case_ID

  • 사례 사용자 또는 태그 ID를 입력합니다.
  • 사례 페이지 > 편집 > 태그 ID로 이동하거나 사례 페이지 > 편집 > URL의 사례 ID로 이동합니다.

사용자 ID

  • 사용자의 ID 또는 태그 ID를 입력하세요.
  • 사용자 페이지 > 편집 > 태그 ID로 이동하거나 사용자 페이지 > 편집 > URL의 사용자 ID로 이동합니다.

케이스_역할_순서

  • 셀을 비워 둡니다.

케이스_역할_ID

  • 입력: 클라이언트, 직원 또는 기본.

case_notify

  • 0 = 아니요
  • 1 = 예, 사용자에게 이메일 알림을 보냅니다.
