Як абнавіць профіль карыстальніка

Гэты артыкул паказвае карыстальнікам, як абнаўляць інфармацыю на старонцы свайго профілю.

  • Карыстальнікі могуць абнаўляць свае профілі.
  • Адміністратары могуць абнаўляць старонку профілю любога карыстальніка.

Для карыстальнікаў

Першы крок - адкрыць старонку профілю. Карыстальнікі і адміністратары выконваюць наступныя дзеянні, каб абнавіць уласную старонку профілю.

  1. Увайдзіце ў свой уліковы запіс Case Active.
  2. Націсніце на ўкладку «Профіль» у верхняй частцы старонкі.
  3. Унясіце змены.
  4. Націсніце Захаваць.

Большасць палёў неабавязковыя, а некаторыя абавязковыя. Абавязковыя палі:

  • Імя
  • Прозвішча
  • Адрас электроннай пошты
  • мова
  • Часавы пояс

Для адміністратараў

Адміністратары таксама могуць абнаўляць інфармацыю профіляў для іншых карыстальнікаў. Гэта ўключае ў сябе тры палі пад загалоўкам HOST USER INFO.

  1. Увайдзіце ў свой уліковы запіс адміністратара Case Active.
  2. Знайдзіце праграму і націсніце кнопку АДМІНІСТРАЦЫЯ.
  3. Націсніце Карыстальнікі ў верхнім меню.
  4. Перайдзіце ў падменю і абярыце спіс карыстальнікаў, да якіх вы хочаце атрымаць доступ (АКТЫЎНЫ або ВЫКЛЮЧАНЫ). Параметр па змаўчанні - "Актыўны".
  5. Знайдзіце карыстальніка ў спісе і націсніце «Рэдагаваць».
  6. Унясіце неабходныя змены. (Глядзі ніжэй)
  7. Націсніце Захаваць.

Змяніць інфармацыю пра карыстальніка

Адміністратары могуць змяняць інфармацыю пра карыстальніка.

  1. Усталюйце тып карыстальніка (карыстальнік або менеджэр).
  2. Усталюйце статус карыстальніка (Актыўны да Адключаны).
  3. Увядзіце унікальны ідэнтыфікатар тэга. (неабавязкова)
  4. Націсніце Захаваць.
