Фарматаванне дадзеных кіравання справамі з дапамогай інструмента імпарту

Ніжэй прыведзены спіс абавязковых палёў у файлах CSV для кіравання запытамі з дапамогай інструмента "Імпарт".

імя карыстальніка

  • Увядзіце адрас электроннай пошты карыстальніка.

tag_id (ідэнтыфікатар, на які можна спасылацца пазней)

  • унікальны ідэнтыфікатар (выкарыстоўвайце літары і лічбы, дапускаюцца злучкі)

статус_справы

  • 0 = закрыта
  • 1 = актыўны

дата_адкрыцця

  • месяц/дзень/год
  • дд/мм/гггг

аб

  • унікальнае апісанне справы (выкарыстоўвайце літары і лічбы, дапускаюцца злучкі)

малюнак
