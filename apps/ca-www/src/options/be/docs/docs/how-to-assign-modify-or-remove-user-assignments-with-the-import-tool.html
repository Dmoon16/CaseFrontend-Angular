Як прызначыць, змяніць або выдаліць прызначэнне карыстальнікаў з дапамогай інструмента імпарту

У гэтым артыкуле адміністратарам паказана, як кіраваць карыстальнікамі з дапамогай інструмента "Імпарт".

Адміністратары могуць:

  • прызначыць карыстальнікаў да справы
  • змяняць карыстальнікаў, ужо прызначаных для справы
  • выдаліць карыстальнікаў, прызначаных для справы

Адкрыццё інструмента імпарту

  1. Увайдзіце ва ўліковы запіс адміністратара.
  2. Знайдзіце праграму на старонцы праграм і націсніце кнопку АДМІНІСТРАЦЫЯ.
  3. Пстрыкніце ўкладку «Інтэграцыя», а затым «ІМПАРТ» у падменю.
  4. Націсніце кнопку НОВЫ. Адлюструецца дыялогавае акно "Дадаць імпарт".

Стварэнне запісаў

Спампуйце адзін або некалькі файлаў CSV. Выкарыстоўвайце гэтыя файлы, каб дадаваць, змяняць або выдаляць карыстальнікаў рэгістра. Пасля адкрыцця дыялогавага акна "Дадаць імпарт":

  1. Націсніце поле Тып.
  2. Пракруціце выпадальнае меню і абярыце опцыю (гл. ніжэй). Адлюструецца спасылка.
  3. Націсніце спасылку «Спампаваць узор». Файл CSV спампоўваецца на вашу прыладу.

малюнак

Адкрыйце файл і дадайце запісы. Націсніце тут, каб даведацца, як выкарыстоўваць дадзеныя фармату ў файле CSV.

Загрузка файла

Вы можаце загрузіць файл адразу ці вярнуцца ў дыялогавае акно "Дадаць імпарт" пазней і выканаць гэтыя дзеянні.

  1. Адкрыйце дыялогавае акно «Дадаць імпарт».
  2. Увядзіце унікальнае імя для файла CSV.
  3. Націсніце на значок плюса (+), каб загрузіць файл з прылады.
  4. Націсніце поле "Тып" і абярыце варыянт. (гл. вышэй)
  5. Націсніце ЗАХАВАЦЬ.

Апрацоўка

У залежнасці ад колькасці запісаў у файле і загрузкі сістэмы платформе можа спатрэбіцца некалькі хвілін, каб апрацаваць імпарт.

малюнак

Пасля завяршэння імпарту платформа карэктуе запісы карыстальнікаў (напрыклад, дадае, змяняе або выдаляе) на старонцы "Справы".

Няўдача

Калі аперацыя імпарту не атрымоўваецца, платформа вяртае паведамленне "Збой". У большасці выпадкаў прычынай збою з'яўляецца фарматаванне дадзеных.

малюнак

Каб убачыць прычыну збою:

  1. Націсніце "Прагляд" пад загалоўкам "Дзеянне".
  2. На наступнай старонцы навядзіце курсор мышы на значок "Збой". Падказка адлюстроўвае памылку.
  3. Перайдзіце да файла CVS і выправіце памылку.
  4. Загрузіце файл яшчэ раз.
