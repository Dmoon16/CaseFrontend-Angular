Як адміністратары кіруюць прызначэннямі карыстальнікаў для спраў

Ёсць пяць спосабаў кіравання карыстальнікамі і прызначэннем спраў. Націсніце на спасылку, каб даведацца, як.

Метад Тып Дзеянні Дэталі
Пры стварэнні корпуса карыстацкі інтэрфейс дадаць Кнопка «Адміністратар» > «Выпадкі» > кнопка «Новы» > кнопка «Прызначыць карыстальнікаў».
Аўтаматычнае прызначэнне карыстальніка па публічнай спасылцы для рэгістрацыі Дзеянне карыстальніка дадаць Кнопка адміністратара > Налады > Старонка налад сістэмы
Выкарыстанне карыстацкага інтэрфейсу ў існуючай справе карыстацкі інтэрфейс дадаць і выдаліць Кнопка адміністратара > Справы > Прызначыць карыстальнікаў
