Як ствараць і змяняць карыстальнікаў з дапамогай карыстацкага інтэрфейсу

Гэты артыкул паказвае адміністратарам, як ствараць карыстальнікаў і абнаўляць запісы з дапамогай карыстальніцкага інтэрфейсу платформы. Адміністратары таксама могуць дадаваць карыстальнікаў у новую справу і рэдагаваць існуючыя справы.

Стварэнне карыстальніка

  1. Увайдзіце ва ўліковы запіс адміністратара.
  2. Знайдзіце праграму на старонцы праграм і націсніце кнопку АДМІНІСТРАЦЫЯ.
  3. На панэлі адміністратара націсніце ўкладку меню «Карыстальнікі».
  4. Націсніце кнопку НОВЫ. Адлюструецца новае акно.

малюнак

Запоўніце палі і націсніце ЗАХАВАЦЬ. Case Active дадае імя ў спіс карыстальнікаў, звязаных з гэтай праграмай.

малюнак

Змена карыстальніка

Каб абнавіць невялікую колькасць запісаў карыстальнікаў, выкарыстоўвайце інструмент "Праўка" і ўнясіце змены для кожнага асобнага карыстальніка.

  • Заўвага Каб змяніць вялікую колькасць карыстальніцкіх запісаў за адзін прысест, паспрабуйце выкарыстоўваць інструмент імпарту .
  1. На панэлі адміністратара націсніце ўкладку меню «Карыстальнікі».
  2. Знайдзіце карыстальніка і націсніце «Рэдагаваць».
  3. Унясіце змены і націсніце ЗАХАВАЦЬ.

малюнак

Наступныя крокі

Case Active не адпраўляе карыстальніку апавяшчэнне па электроннай пошце. Адміністратары павінны звязацца з карыстальнікам і адправіць яму спасылку на галоўную старонку Case Active. Затым карыстальнік націскае кнопку УВАХОД і завяршае працоўны працэс для новых рэгістрацый.

  • У гэтым артыкуле апісваецца працоўны працэс рэгістрацыі для новых карыстальнікаў.
