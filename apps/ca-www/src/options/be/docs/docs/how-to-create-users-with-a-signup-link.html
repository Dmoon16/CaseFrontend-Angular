Як стварыць карыстальнікаў па спасылцы для рэгістрацыі

Гэты артыкул паказвае адміністратарам, як ствараць карыстальнікаў, даслаўшы спасылку для рэгістрацыі.

Настройка спасылкі для рэгістрацыі

  1. Пстрыкніце спасылку « Адправіць ім заяўку » ўнізе дыялогавага акна «Дадаць карыстальніка». Адлюструецца новая старонка.
  2. Наладзьце параметры рэгістрацыі. (Глядзі ніжэй)

малюнак

Пераканайцеся, што для поля « Дазволіць рэгістрацыю» ўстаноўлена значэнне «Уключана». (абавязкова) Калі ўключана, Case Active стварае унікальную спасылку.

Праглядзіце параметры ніжэй, а затым націсніце на значок, каб скапіяваць спасылку. Вы можаце размясціць спасылку на сваім сайце або адправіць спасылку аднаму або некалькім людзям па электроннай пошце або па любым канале сувязі.

малюнак

Стварэнне справы

Пры жаданні вы можаце аўтаматычна стварыць справу для кожнага новага карыстальніка і прызначыць ролю карыстальніка.

  1. Націсніце поле "Аўтаматычнае стварэнне справы" і выберыце "Уключана". Адлюструюцца палі Case User Role.
  2. Націсніце Ролю карыстальніка Case і абярыце опцыю. (Глядзі ніжэй)

малюнак

Ролі карыстальнікаў Case

Ёсць тры ролі карыстальніка case.

  • кліентаў
  • па змаўчанні
  • персанал

Адпраўка спасылкі

Падзяліцеся спасылкай з адным або некалькімі людзьмі. Кожны чалавек націскае спасылку і завяршае працоўны працэс, каб актываваць свой уліковы запіс Case Active. Пасля таго, як карыстальнік завершыць працоўны працэс рэгістрацыі, Case Active дадае карыстальніка ў спіс карыстальнікаў.

малюнак

Наступныя крокі

Case Active адпраўляе карыстальніку паведамленне па электроннай пошце са спасылкай для рэгістрацыі. Карыстальнік націскае спасылку і завяршае працоўны працэс для новых рэгістрацый.

  • У гэтым артыкуле апісваецца працоўны працэс рэгістрацыі для новых карыстальнікаў.
