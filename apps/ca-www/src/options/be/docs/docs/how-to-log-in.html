Як увайсці

Даведайцеся, як увайсці ў існуючы ўліковы запіс Case Active.

Уваход у сістэму

  1. Перайдзіце на старонку ўваходу .
  2. Націсніце кнопку УВАХОД.
  3. Увядзіце свой адрас электроннай пошты.
  4. Націсніце ДАЛЕЙ.

Праверка акаўнта

  1. Адкрыйце новую ўкладку браўзера і ўвайдзіце ў свой уліковы запіс электроннай пошты.
  2. Скапіруйце шасцізначны код.
  3. Вярніцеся на ўкладку Case Active.
  4. Устаўце код у палі. Адлюструецца новае акно.

Case Active адлюстроўвае старонку праграм.

  • Заўвага Платформа адлюстроўвае кнопку ADMIN побач з кнопкай APP для карыстальнікаў з правамі адміністратара.
