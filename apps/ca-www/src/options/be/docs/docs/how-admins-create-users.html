Як адміністратары ствараюць карыстальнікаў

Апавяшчэнні

Платформа не адпраўляе карыстальнікам апавяшчэнне, калі яны дадаюцца ў праграму. Аднак адміністратары могуць уключыць апавяшчэнні аб прызначэнні спраў.
