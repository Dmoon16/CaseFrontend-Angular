Як стварыць прыкладанне

Гэты артыкул паказвае новым карыстальнікам, як стварыць бясплатны пробны ўліковы запіс. Case Active аўтаматычна дае новаму карыстальніку прывілеі адміністратара для гэтага ўліковага запісу.

Стварэнне акаўнта

  1. Перайдзіце на старонку ўваходу .
  2. Націсніце кнопку Пачаць.
  3. Увядзіце адрас электроннай пошты.
  4. Націсніце Далей.

малюнак

Праверка акаўнта

  1. Адкрыйце новую ўкладку браўзера і ўвайдзіце ў свой уліковы запіс электроннай пошты.
  2. Скапіруйце шасцізначны код.
  3. Вярніцеся на ўкладку Case Active.
  4. Устаўце код у палі. Адлюструецца новае акно.

У новым акне:

  1. Запоўніце неабходныя палі.
  2. Прачытайце ўмовы абслугоўвання і пастаўце галачку, каб пагадзіцца.
  3. Націсніце СТВАРЫЦЬ БЯСПЛАТНЫ ЎЛІКОВЫ ЗАПІС.

малюнак

Завяршэнне наладкі праграмы

Працоўны працэс адлюстроўваецца на наступнай старонцы. Выканайце крокі, каб завяршыць наладжванне праграмы. Вы заўсёды можаце вярнуцца на гэтыя старонкі, каб адрэдагаваць або абнавіць інфармацыю або стан вашага ўліковага запісу.

На старонцы "Аб праграме" запоўніце палі з назвай кампаніі і вэб-сайтам (абавязкова), а затым націсніце "Далей".

малюнак

На старонцы плацежнай інфармацыі пстрыкніце поле «План аплаты» і праглядзіце параметры. Налада па змаўчанні - бясплатна. Націсніце АДПРАВІЦЬ.

малюнак

Пасля адпраўкі формы Case Active адлюстроўвае старонку праграм. Для кожнай праграмы, якую вы ствараеце або якой належыце, на старонцы паказваецца:

  1. Назва праграмы
  2. Ваш тып (роля карыстальніка)
  3. Статус

малюнак

Старонка праграм з'яўляецца адпраўной кропкай для многіх функцый адміністратара:

  1. Націсніце кнопку APP, каб мець зносіны, запаўняць дакументы, весці відэачат (і многае іншае) па пытаннях, звязаных са справамі.
  2. Націсніце кнопку ADMIN, каб адкрыць панэль адміністратара. Тут адміністратары кіруюць карыстальнікамі, кейсамі, інтэграцыяй і іншым.
  3. Націсніце шматкроп'е, каб кіраваць наладамі выстаўлення рахункаў і апавяшчэнняў.
  • Note Case Active адлюстроўвае кнопку ADMIN для карыстальнікаў-адміністратараў. Карыстальнікі, якія не маюць прывілеяў адміністратара, не бачаць кнопку АДМІНІСТРАЦЫЯ (як паказана на экране ніжэй).
