നിലവിലുള്ള കേസുകളിൽ ഉപയോക്തൃ അസൈൻമെന്റുകൾ എങ്ങനെ അസൈൻ ചെയ്യാം അല്ലെങ്കിൽ നീക്കം ചെയ്യാം

കേസ് ഉപയോക്താക്കളെ എങ്ങനെ അസൈൻ ചെയ്യാമെന്നും നീക്കംചെയ്യാമെന്നും ഈ ലേഖനം അഡ്‌മിനുകളെ കാണിക്കുന്നു. ഉപയോക്തൃ ഡാറ്റ എങ്ങനെ പരിഷ്കരിക്കാമെന്നും ഞങ്ങൾ കാണിച്ചുതരുന്നു.

നിലവിലുള്ള ഒരു കേസ് തുറക്കുന്നു

  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. കേസുകൾ മെനു ടാബിൽ ക്ലിക്ക് ചെയ്യുക.
  4. ഉപമെനുവിൽ ACTIVE തിരഞ്ഞെടുത്തിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക.
  5. ACTION നിരയിലേക്ക് താഴേക്ക് സ്ക്രോൾ ചെയ്യുക.

അസൈൻ ചെയ്യുക അല്ലെങ്കിൽ നീക്കം ചെയ്യുക

കേസിൽ ഒരു ഉപയോക്താവിനെ അസൈൻ ചെയ്യുന്നതിനോ അല്ലെങ്കിൽ കേസിൽ നിന്ന് ഉപയോക്താവിനെ നീക്കം ചെയ്യുന്നതിനോ ഉപയോക്താക്കളെ അസൈൻ ചെയ്യുക ക്ലിക്ക് ചെയ്യുക (എന്നാൽ പ്ലാറ്റ്ഫോം അല്ല). അടുത്ത പേജിൽ: ADD ക്ലിക്ക് ചെയ്ത് വർക്ക്ഫ്ലോ പൂർത്തിയാക്കുക. ഒരു ഉപയോക്താവിനെ ഇല്ലാതാക്കാൻ നീക്കം ചെയ്യുക ക്ലിക്ക് ചെയ്യുക.

പരിഷ്ക്കരിക്കുക

കേസുകൾ പേജിൽ ഉപയോക്തൃ ഡാറ്റ എഡിറ്റ് ചെയ്യാനോ അപ്ഡേറ്റ് ചെയ്യാനോ സാധ്യമല്ല. പകരം, ഈ ഘട്ടങ്ങൾ പാലിക്കുക:

  1. ഉപയോക്താക്കളുടെ മെനു ടാബിൽ ക്ലിക്ക് ചെയ്യുക.
  2. ഉപയോക്തൃനാമം കണ്ടെത്തി എഡിറ്റ് ക്ലിക്ക് ചെയ്യുക.
  3. മാറ്റങ്ങൾ വരുത്തി സേവ് ക്ലിക്ക് ചെയ്യുക.
