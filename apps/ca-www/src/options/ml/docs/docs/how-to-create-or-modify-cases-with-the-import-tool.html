ഇറക്കുമതി ടൂൾ ഉപയോഗിച്ച് കേസുകൾ എങ്ങനെ സൃഷ്ടിക്കാം അല്ലെങ്കിൽ പരിഷ്ക്കരിക്കാം

ഇംപോർട്ട് ടൂൾ ഉപയോഗിച്ച് നിലവിലുള്ള കേസുകൾ എങ്ങനെ സൃഷ്‌ടിക്കാമെന്നും പരിഷ്‌ക്കരിക്കാമെന്നും ഈ ലേഖനം അഡ്‌മിനുകളെ കാണിക്കുന്നു.

ഇറക്കുമതി ഉപകരണം തുറക്കുന്നു

  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. അഡ്മിൻ പാനലിൽ, ഇന്റഗ്രേഷൻ ടാബിൽ ക്ലിക്ക് ചെയ്യുക, തുടർന്ന് ഉപമെനുവിലെ ഇറക്കുമതി ചെയ്യുക.
  4. പുതിയ ബട്ടൺ ക്ലിക്ക് ചെയ്യുക. ഇമ്പോർട്ടുകൾ ചേർക്കുക ഡയലോഗ് ബോക്സ് ഒരു പുതിയ ബ്രൗസർ ടാബിൽ പ്രദർശിപ്പിക്കുന്നു.

കേസുകൾ സൃഷ്ടിക്കുന്നു

ഇറക്കുമതി ചേർക്കുക ഡയലോഗ് ബോക്സ് തുറന്ന ശേഷം:

  1. ടൈപ്പ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്ത് ഒരു ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക (ചുവടെ കാണുക). ഒരു ലിങ്ക് പ്രദർശിപ്പിക്കുന്നു.
  2. ഡൗൺലോഡ് സാമ്പിൾ ലിങ്ക് ക്ലിക്ക് ചെയ്യുക. ഒരു CSV ഫയൽ നിങ്ങളുടെ ഉപകരണത്തിലേക്ക് ഡൗൺലോഡ് ചെയ്യുന്നു.

അടുത്തതായി, ഫയൽ തുറന്ന് കേസുകൾ ചേർക്കുക.

ടൈപ്പ് ചെയ്യുക

നിങ്ങൾ ടൈപ്പ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഒരു ഡ്രോപ്പ്-ഡൗൺ മെനു ദൃശ്യമാകും. ബൾക്ക് കെയ്‌സ് ഇമ്പോർട്ടുകൾക്കായി രണ്ട് ഓപ്ഷനുകളിൽ ഒന്ന് തിരഞ്ഞെടുക്കുക.

  • ശ്രദ്ധിക്കുക ഡ്രോപ്പ്-ഡൗൺ മെനുവിൽ നിരവധി ഓപ്ഷനുകൾ ഉണ്ട്, എന്നാൽ രണ്ടെണ്ണം മാത്രമേ കേസ് കൈകാര്യം ചെയ്യുന്നതുമായി ബന്ധപ്പെട്ടിട്ടുള്ളൂ. ഓരോ ഓപ്‌ഷനും വ്യത്യസ്‌തമായ സാമ്പിൾ CSV ഫയലുണ്ട്.

കേസുകൾ ചേർക്കുക

  • കേസുകളുടെ ലളിതമായ ബൾക്ക് അപ്‌ലോഡിനായി ഈ ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക.
  • ക്രിയേറ്റ് കേസുകൾ ഇംപോർട്ട് തരത്തിനായുള്ള കോളം തലക്കെട്ടുകളുള്ള ഒരു CSV ഫയൽ തുറക്കാൻ ഡൗൺലോഡ് സാമ്പിൾ ലിങ്കിൽ ക്ലിക്ക് ചെയ്യുക.

കേസുകൾ പരിഷ്കരിക്കുക

  • നിലവിലെ കേസുകൾ അപ്ഡേറ്റ് ചെയ്യാൻ ഈ ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക.
  • മോഡിഫൈ കേസുകൾ ഇംപോർട്ട് തരത്തിനായുള്ള കോളം തലക്കെട്ടുകളുള്ള ഒരു CSV ഫയൽ തുറക്കാൻ ഡൗൺലോഡ് സാമ്പിൾ ലിങ്കിൽ ക്ലിക്ക് ചെയ്യുക.

ഫയൽ അപ്‌ലോഡ് ചെയ്യുന്നു

നിങ്ങൾക്ക് ഉടൻ തന്നെ ഒരു ഫയൽ അപ്‌ലോഡ് ചെയ്യാം അല്ലെങ്കിൽ പിന്നീടുള്ള തീയതിയിൽ ചേർക്കുക ഇമ്പോർട്ടുകൾ ഡയലോഗ് ബോക്സിലേക്ക് മടങ്ങുകയും ഈ ഘട്ടങ്ങൾ പൂർത്തിയാക്കുകയും ചെയ്യാം.

  1. ഇറക്കുമതി ചേർക്കുക ഡയലോഗ് ബോക്സ് തുറക്കുക.
  2. CSV ഫയലിനായി ഒരു അദ്വിതീയ നാമം നൽകുക.
  3. നിങ്ങളുടെ ഉപകരണത്തിൽ നിന്ന് ഫയൽ അപ്‌ലോഡ് ചെയ്യാൻ പ്ലസ് (+) ഐക്കണിൽ ക്ലിക്ക് ചെയ്യുക.
  4. ടൈപ്പ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്ത് ഒരു ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക. (മുകളിൽ കാണുന്ന)
  5. സേവ് ക്ലിക്ക് ചെയ്യുക.

പരാജയം

ഇറക്കുമതി പ്രവർത്തനം പരാജയപ്പെടുകയാണെങ്കിൽ, പ്ലാറ്റ്ഫോം ഒരു പരാജയപ്പെട്ട സന്ദേശം നൽകുന്നു. മിക്ക കേസുകളിലും, പരാജയത്തിന്റെ കാരണം ഡാറ്റ ഫോർമാറ്റിംഗ് ആണ്.

പരാജയത്തിന്റെ കാരണം കാണാൻ:

  1. പ്രവർത്തന തലക്കെട്ടിന് കീഴിലുള്ള കാണുക ക്ലിക്ക് ചെയ്യുക.
  2. അടുത്ത പേജിൽ, പരാജയപ്പെട്ട ഐക്കണിൽ മൗസ് ഹോവർ ചെയ്യുക. ടൂൾടിപ്പ് പിശക് കാണിക്കുന്നു.
  3. CVS ഫയലിലേക്ക് പോയി ക്രമീകരിച്ച് പിശക് പരിഹരിക്കുക.
  4. ഫയൽ വീണ്ടും അപ്‌ലോഡ് ചെയ്യുക.
