ഒരു പൊതു സൈൻഅപ്പ് ലിങ്ക് ഉപയോഗിച്ച് കേസുകൾ എങ്ങനെ സ്വയമേവ സൃഷ്ടിക്കാം

ഒരൊറ്റ സൈൻഅപ്പ് ലിങ്ക് അയച്ചുകൊണ്ട് എങ്ങനെ ഒരു കേസ് സൃഷ്ടിക്കാമെന്ന് (ഒരേ സമയം ഒരു ഉപയോക്താവിനെ ചേർക്കുക) ഈ ലേഖനം അഡ്‌മിനുകളെ കാണിക്കുന്നു.

ലിങ്ക് കണ്ടെത്തുന്നു

  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. അഡ്മിൻ പാനലിൽ, ക്രമീകരണ മെനു ടാബിൽ ക്ലിക്ക് ചെയ്യുക.
  4. SYSTEM SETUP ഉപമെനു ടാബ് തിരഞ്ഞെടുത്തിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക.

ക്രമീകരണങ്ങൾ ക്രമീകരിക്കുന്നു

USER SIGNUP തലക്കെട്ടിലേക്ക് താഴേക്ക് സ്ക്രോൾ ചെയ്യുക.

സൈൻഅപ്പ് അനുവദിക്കുക എന്ന ഫീൽഡ് ഓണായി സജ്ജീകരിച്ചിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക. (ആവശ്യമാണ്) ഈ ഫീച്ചർ പ്രവർത്തനക്ഷമമാക്കുമ്പോൾ, Case Active ഒരു അദ്വിതീയ ലിങ്ക് സൃഷ്ടിക്കുന്നു.

  1. ഓട്ടോ ക്രിയേറ്റ് കേസ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്ത് ഓൺ തിരഞ്ഞെടുക്കുക. കേസ് യൂസർ റോൾ ഫീൽഡ് പ്രദർശിപ്പിക്കുന്നു.
  2. ഒരു റോൾ തിരഞ്ഞെടുക്കുക. (താഴെ നോക്കുക)

കേസ് ഉപയോക്തൃ റോളുകൾ

മൂന്ന് കേസ് യൂസർ റോളുകൾ ഉണ്ട്.

  • ഉപഭോക്താക്കൾ (ശുപാർശ ചെയ്യുന്നു)
  • സ്ഥിരസ്ഥിതി
  • സ്റ്റാഫ്

ലിങ്ക് അയക്കുന്നു

ഒന്നോ അതിലധികമോ ആളുകളുമായി ലിങ്ക് പങ്കിടുക. ലിങ്കിൽ ക്ലിക്കുചെയ്‌ത് സൈൻഅപ്പ് പൂർത്തിയാക്കുന്ന ഓരോ ഉപയോക്താവും കേസുകൾ പേജിൽ ഒരു പുതിയ കേസ് സൃഷ്ടിക്കുന്നതിനും കേസിൽ ഉപയോക്താവിനെ സ്വയമേവ അസൈൻ ചെയ്യുന്നതിനും പ്ലാറ്റ്‌ഫോമിനെ ട്രിഗർ ചെയ്യുന്നു.
