പബ്ലിക് സൈനപ്പ് ലിങ്ക് ഉപയോഗിച്ച് ഉപയോക്താക്കളെ എങ്ങനെ സ്വയമേവ അസൈൻ ചെയ്യാം

സിസ്റ്റം ക്രമീകരണങ്ങൾ എങ്ങനെ കോൺഫിഗർ ചെയ്യാമെന്ന് അഡ്മിനുകൾ പഠിക്കുന്നു, അതിലൂടെ പുതിയ ഉപയോക്താക്കളെ പുതിയതായി സ്വയമേവ സൃഷ്ടിച്ച ഒരു കേസിലേക്ക് സ്വയമേവ അസൈൻ ചെയ്യപ്പെടും.

സജ്ജീകരണ പേജ് തുറക്കുന്നു

സിസ്റ്റം സെറ്റപ്പ് പേജ് തുറക്കുക എന്നതാണ് ആദ്യപടി.

  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. ഉപമെനുവിലെ സിസ്റ്റംസ് ടാബിൽ ക്ലിക്ക് ചെയ്ത് SYSTEM SETUP ക്ലിക്ക് ചെയ്യുക.
  4. USER SIGNUP തലക്കെട്ടിലേക്ക് താഴേക്ക് സ്ക്രോൾ ചെയ്യുക.

സിസ്റ്റം ക്രമീകരണങ്ങൾ ക്രമീകരിക്കുന്നു

  1. സൈനപ്പ് അനുവദിക്കുക, സ്വയമേവ സൃഷ്ടിക്കുക കേസ് ഡ്രോപ്പ്-ഡൗൺ മെനുകൾ ഓണാക്കി സജ്ജമാക്കുക.
  2. റോൾ മെനുവിൽ ക്ലിക്ക് ചെയ്ത് ഒരു ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക (സ്ഥിരസ്ഥിതി ശുപാർശ ചെയ്യുന്നു).
  3. പബ്ലിക് യൂസർ സൈനപ്പ് ലിങ്ക് പകർത്തി പങ്കിടുക.

ലിങ്ക് അയക്കുന്നു

ലിങ്കിൽ ക്ലിക്കുചെയ്‌ത് സൈൻഅപ്പ് പൂർത്തിയാക്കുന്ന ഓരോ ഉപയോക്താവും കേസുകൾ പേജിൽ ഒരു പുതിയ കേസ് സൃഷ്ടിക്കാൻ പ്ലാറ്റ്‌ഫോമിനെ ട്രിഗർ ചെയ്യുകയും ആ സാഹചര്യത്തിൽ ഉപയോക്താവിനെ സ്വയമേവ അസൈൻ ചെയ്യുകയും ചെയ്യുന്നു.
