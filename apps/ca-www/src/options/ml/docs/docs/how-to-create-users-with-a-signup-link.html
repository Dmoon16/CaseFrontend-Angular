ഒരു സൈൻഅപ്പ് ലിങ്ക് അയച്ചുകൊണ്ട് ഉപയോക്താക്കളെ എങ്ങനെ സൃഷ്ടിക്കാമെന്ന് ഈ ലേഖനം അഡ്‌മിനുകളെ കാണിക്കുന്നു.

സൈൻഅപ്പ് ലിങ്ക് കോൺഫിഗർ ചെയ്യുന്നു

  1. ആഡ് യൂസർ ഡയലോഗ് ബോക്‌സിന്റെ താഴെയുള്ള Send them a signup ലിങ്ക് ക്ലിക്ക് ചെയ്യുക. ഒരു പുതിയ പേജ് പ്രദർശിപ്പിക്കുന്നു.
  2. സൈൻഅപ്പ് ഓപ്ഷനുകൾ കോൺഫിഗർ ചെയ്യുക. (താഴെ നോക്കുക)

ചുവടെയുള്ള ഓപ്‌ഷനുകൾ അവലോകനം ചെയ്‌ത് ലിങ്ക് പകർത്താൻ ഐക്കണിൽ ക്ലിക്കുചെയ്യുക. നിങ്ങൾക്ക് ലിങ്ക് നിങ്ങളുടെ വെബ്‌സൈറ്റിൽ സ്ഥാപിക്കാം അല്ലെങ്കിൽ ഇമെയിൽ വഴിയോ ഏതെങ്കിലും ആശയവിനിമയ ചാനൽ വഴിയോ ഒന്നോ അതിലധികമോ ആളുകൾക്ക് ലിങ്ക് അയയ്ക്കാം.

  1. ഓട്ടോ ക്രിയേറ്റ് കേസ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്ത് ഓൺ തിരഞ്ഞെടുക്കുക. കേസ് യൂസർ റോൾ ഫീൽഡുകൾ പ്രദർശിപ്പിക്കുന്നു.
  2. കേസ് യൂസർ റോളിൽ ക്ലിക്ക് ചെയ്ത് ഒരു ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക. (താഴെ നോക്കുക)

ലിങ്ക് അയക്കുന്നു

ഒന്നോ അതിലധികമോ ആളുകളുമായി ലിങ്ക് പങ്കിടുക. ഓരോ വ്യക്തിയും അവരുടെ കേസ് ആക്റ്റീവ് അക്കൗണ്ട് സജീവമാക്കുന്നതിന് ലിങ്കിൽ ക്ലിക്ക് ചെയ്യുകയും വർക്ക്ഫ്ലോ പൂർത്തിയാക്കുകയും ചെയ്യുന്നു. ഉപയോക്താവ് സൈൻഅപ്പ് വർക്ക്ഫ്ലോ പൂർത്തിയാക്കിക്കഴിഞ്ഞാൽ, Case Active ഉപയോക്താവിനെ ഉപയോക്താക്കളുടെ പട്ടികയിലേക്ക് ചേർക്കുന്നു.
