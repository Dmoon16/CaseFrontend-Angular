Kā administratori izveido lietotājus

Administratori izveido, pārveido vai atspējo, kas var piekļūt lietotnēm un gadījumiem. Kad lietotājs pirmo reizi piesakās platformā, viņam ir jāpabeidz profila darbplūsma.

  • Noklikšķiniet šeit , lai uzzinātu vairāk par jaunā lietotāja profila darbplūsmu.

Lietotāja izveide

Paziņojumi

Platforma nesūta lietotājiem paziņojumu, kad viņi tiek pievienoti lietotnei. Tomēr administratori var iespējot paziņojumus par lietu piešķiršanu.
