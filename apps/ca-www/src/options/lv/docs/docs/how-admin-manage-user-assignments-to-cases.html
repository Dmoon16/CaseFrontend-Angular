Kā administratori pārvalda lietotāju piešķiršanu gadījumiem

Ir pieci veidi, kā pārvaldīt lietotājus un lietu piešķiršanu. Noklikšķiniet uz saites, lai uzzinātu, kā to izdarīt.

Metode Tips Darbības Sīkāka informācija
Lietas veidošanas laikā UI pievienot Poga Administrēšana > Lietas > Jauna poga > Poga Piešķirt lietotājus
Automātiski piešķirt lietotāju, izmantojot publisko reģistrēšanās saiti Lietotāja darbība pievienot Administratora poga > Iestatījumi > Sistēmas iestatīšanas lapa
UI izmantošana esošajā gadījumā UI pievienot un noņemt Poga Administrēšana > Lietas > Piešķirt lietotājus
Importēšanas rīks UI pievienot, modificēt un noņemt Administratora poga > Integrācijas > Importēšana
API izmantošana (drīzumā) API pievienot, modificēt un noņemt N/A
