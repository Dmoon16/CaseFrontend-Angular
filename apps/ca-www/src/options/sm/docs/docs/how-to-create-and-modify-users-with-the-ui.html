Fa'afefea ona Fausia ma Suia Tagata fa'aoga ile UI

O lenei tusiga o loʻo faʻaalia ai le au pulega pe faʻafefea ona fatuina tagata faʻaoga ma faʻafou faʻamaumauga i le faʻaoga faʻaoga a le platform. E mafai foi e le au pulega ona faaopoopo tagata faaaoga i se mataupu fou ma faasa'o mataupu o iai.

Fausia se tagata fa'aoga

  1. Ulufale i le admin account.
  2. Su'e se app ile itulau Apps ma kiliki le ADMIN button.
  3. I luga o le Admin panel, kiliki le Users menu tab.
  4. Kiliki le faamau FOU. Ua fa'aalia se faamalama fou.

img

Faatumu fanua ma kiliki SAVE. Case Active fa'aopoopo le igoa i le lisi o tagata fa'aoga e feso'ota'i ma lenei app.

img

Suia se tagata fa'aoga

Ina ia faʻafouina se numera itiiti o faʻamaumauga a tagata faʻaoga, faʻaoga le Faʻatonu meafaigaluega ma fai suiga i tagata taʻitoʻatasi.

  • Fa'aaliga Ina ia suia se numera tele o fa'amaumauga a tagata fa'aoga i se fonotaga e tasi, mafaufau e fa'aaoga le mea faigaluega Fa'aulufale mai .
  1. I luga o le Admin panel, kiliki le Users menu tab.
  2. Su'e se tagata fa'aoga ma kiliki Fa'atonu.
  3. Fai suiga ma kiliki SAVE.

img

Laasaga e sosoo ai

Case Active e le tu'uina atu i le tagata fa'aoga se fa'amatalaga imeli. E tatau i le au pulega ona fa'afeso'ota'i le tagata fa'aoga ma lafo ia i latou se so'oga i le itulau autu o Case Active. Ona kiliki lea e le tagata fa'aoga le faamau LOGIN ma fa'amae'a le fa'asologa o galuega mo saini fou.

  • O lenei tusiga o loʻo faʻamatalaina ai le faʻasologa o galuega saini mo tagata fou.
