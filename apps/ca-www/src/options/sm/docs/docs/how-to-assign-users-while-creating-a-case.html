Fa'afefea ona Tofi Tagata Fa'aoga A'o Faia se Fa'ailoga

O lenei tusiga o loʻo faʻaalia ai le au pulega pe faʻapefea ona tofia se tagata faʻaoga aʻo faia se mataupu fou. Afai e leʻi resitalaina le tagata faʻaoga i luga o le faʻavae, e mafai ona e fatuina se faʻaoga fou aʻo e fatuina se mataupu fou.

Fausia se mataupu

  1. Ulufale i le admin account.
  2. Su'e se app ile itulau Apps ma kiliki le ADMIN button.
  3. Kiliki le lisi lisi o mataupu.
  4. Kiliki le faamau FOU. Ua fa'aalia se faamalama fou.
  • Faatumu fanua e fai ai se mataupu ona tusi lea i lalo i le ulutala Tagata Fa'atonu.

Tofia se tagata fa'aoga

Tagata fa'aoga nei

Mulimuli i laasaga nei e tofia ai se tagata o loʻo iai nei i le mataupu fou.

Kiliki le ASSIGN USERS button ma faʻatumu fanua.

  1. Kiliki le Fa'aoga matāua i lalo ma filifili se fa'amatalaga (eg imeli).
  2. Ulufale se tau i le avanoa avanoa (eg tuatusi imeli).
  3. Afai o loʻo iai le faʻamaumauga a le tagata faʻaoga i luga o le faʻavae, o le fanua e faʻaalia e aunoa ma se igoa.
  4. Filifili le tagata fa'aoga sa'o.
  5. Filifili se matafaioi fa'aoga (tagata fa'atau, fa'aletonu, po'o le aufaigaluega).
  6. Ile faitalia, siaki le atigipusa e lafo atu i le tagata faʻaoga se faʻamatalaga imeli.
  7. Kiliki SAVE.

Case Active fa'aopoopo le tagata fa'aoga ile mataupu. Ole taimi nei e mafai e le au pulega ona kiliki Save e tapuni ai le faamalama, pe faaopoopo se tagata fou, pe aveese se tagata faʻaoga.

Fa'aopoopoina se tagata fa'aoga fou

Afai e le mafai e le faʻavae ona maua se faʻamaumauga o loʻo iai nei, kiliki le ki e faʻaopoopo ai se tagata fou.

Faatumu fanua ona kiliki lea SAVE e faʻaopoopo le tagata faʻaoga i luga o le tulaga. O le isi, seti se matafaioi a le tagata faʻaoga ma toe kiliki SAVE e tuʻuina atu le tagata faʻaoga i le mataupu. Mulimuli, kiliki SAVE tasi le taimi e teu ai le mataupu.
