Fa'afefea ona Fausia Tagata fa'aoga i Mea Fa'aulufale mai

O lenei tusiga o loʻo faʻaalia ai le au pulega le auala e fai ai tagata faʻaoga i le mea faigaluega Faʻaulufale mai.

Tatala le meafaigaluega Fa'aulufale mai

E mafai e le au pulega ona fa'aoga le mea faigaluega Fa'aulufaleina e fa'apipi'i tele ai tagata fou ma suia fa'amaumauga o lo'o iai.

  1. Ulufale i le admin account.
  2. Su'e se app ile itulau Apps ma kiliki le ADMIN button.
  3. I luga o le Admin panel, kiliki le Integration tab ona fa'aulufale mai lea ile submenu.
  4. Kiliki le faamau FOU. O le Add Imports dialog box o lo'o fa'aalia i se fa'ailoga fou o su'esu'ega.

Fausia faamaumauga

A uma ona tatala le pusa fa'atalanoaga Add Imports:

  1. Kiliki le Ituaiga fanua ma filifili se filifiliga (silasila i lalo). O lo'o fa'aalia se so'oga.
  2. Kiliki le download Fa'ata'ita'iga so'oga. O se faila CSV e la'u mai i lau masini.

Le isi, tatala le faila ma faʻaopoopo faʻamaumauga. Kiliki iinei e a'oa'o ai pe fa'apefea ona fa'aoga fa'amaumauga fa'asologa i le faila CSV.

Ituaiga

A e kiliki i le Ituaiga fanua, e faʻaalia se lisi faʻalalo. Filifili se tasi o filifiliga e lua mo le tele o oloa mai fafo.

:'u'u: FAAMANATU O le lisi o le pa'ū i lalo e tele filifiliga, ae na o le lua e fesoʻotaʻi ma le mea faigaluega Faʻaulufale mai. O filifiliga ta'itasi e iai se fa'ata'ita'iga faila CSV eseese.

Fa'aopoopo tagata fa'aoga

  • Filifili lenei filifiliga mo se fa'apipi'i fa'atele faigofie o fa'amaumauga a tagata fa'aoga.
  • Kiliki le Download Fa'ata'ita'iga so'otaga e tatala ai se faila CSV ma fa'aulutala koluma mo le Add users Import type.

Suia tagata faaaoga

  • Filifili le filifiliga lea e fa'afou ai fa'amaumauga mo tagata fa'aoga o iai nei.
  • Kiliki le Download Fa'ata'ita'iga so'otaga e tatala ai se faila CSV ma fa'aulutala koluma mo le Suia tagata fa'aulu mai ituaiga.

Tuuina atu o le faila

E mafai ona e tu'u i luga se faila i le taimi lava lena pe toe fo'i i le Add Imports dialog box i se aso mulimuli ane ma fa'auma nei la'asaga.

  1. Tatala le pusa fa'atalanoaga Add Imports.
  2. Ulufale se igoa tulaga ese mo le faila CSV.
  3. Kiliki le fa'aopoopo (+) icon e lafo ai le faila mai lau masini.
  4. Kiliki le Ituaiga fanua ma filifili se filifiliga. (silasila i luga)
  5. Kiliki SAVE.

Fa'agasologa

Faʻalagolago i le numera o faʻamaumauga i le faila ma le utaina o le polokalama, e mafai e le faʻavae ona faʻaalu ni nai minute e faʻagasolo ai le faʻaulufale mai.

A mae'a le fa'aulufale mai, e fetu'una'i e le tulaga fa'amaumauga a tagata fa'aoga (fa'aopoopo, suia, pe fa'amalo) i luga o le lisi o Users a le admin.

To'ilalo

Afai e le manuia le fa'aulufaleina mai, e toe fa'afo'i mai e le tulaga se fe'au Fa'aletonu. I le tele o tulaga, o le mafuaʻaga o le toilalo o le faʻatulagaina o faʻamaumauga.

Ina ia iloa le mafuaaga o le faaletonu:

  1. Kiliki Va'ai i lalo o le Fa'aulutala Fa'atino.
  2. I luga o le itulau e sosoo ai, fa'ae'e le isumu ile fa'ailoga Fai. O le meafaigaluega e faʻaalia ai le mea sese.
  3. Alu i le faila CVS ma fetuutuunai ma faaleleia le mea sese.
  4. Toe lafo le faila.
