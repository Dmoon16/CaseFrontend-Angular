Fa'afefea ona Fausia pe Suia mataupu e ala ile UI Menu

O lenei tusiga o loʻo faʻaalia ai le au pulega pe faʻafefea ona fai se mataupu ma le faʻaoga faʻaoga a le platform. E mafai foi e le au pulega ona faaopoopo tagata faaaoga i se mataupu fou ma faafou mataupu o iai.

Fausia se mataupu

  1. Ulufale i le admin account.
  2. Su'e se app ile itulau Apps ma kiliki le ADMIN button.
  3. I luga o le Admin panel, kiliki le lisi lisi mataupu.
  4. Kiliki le faamau FOU. Ua fa'aalia se faamalama fou.

Faatumu fanua mana'omia ma kiliki SAVE.

img

Fa'aopoopo tagata fa'aoga

Ile filifiliga, e mafai e le au pulega ona fa'aopoopo tagata fa'aoga ile mataupu ile itulau lenei. Tusi i lalo ma kiliki le ASSIGN USERS button.

img

I le Assign User modal, faʻatumu fanua. Siaki le pusa e lafo atu i le tagata faʻaoga se faʻamatalaga imeli. (filifiliga)

Suia se mataupu

Ina ia faʻafouina se numera laʻititi o mataupu, faʻaoga le Edit tool ma fai suiga i mataupu taʻitasi.

  • Fa'aaliga Ina ia fa'afou se numera tele o mataupu i le fonotaga e tasi, mafaufau e fa'aoga le mea faigaluega Fa'aulufale mai
  1. Ulufale i le admin account.
  2. Su'e se app ile itulau Apps ma kiliki le ADMIN button.
  3. I luga o le Admin panel, kiliki le lisi lisi mataupu.
  4. Su'e se mataupu ma kiliki Fa'atonu.
  5. Fai suiga ma kiliki SAVE.
