Formatere saksbehandlingsdata med importverktøyet

Nedenfor er en liste over obligatoriske felt i CSV-filer for å administrere saker med importverktøyet.

brukernavn

  • Skriv inn brukerens e-postadresse.

tag_id (en id du kan referere til senere)

  • en unik identifikator (bruk alfanumeriske tegn, bindestreker godtas)

case_status

  • 0 = lukket
  • 1 = aktiv

date_opened

  • måned dag år
  • mm/dd/åååå

Om

  • en unik beskrivelse av saken (bruk alfanumeriske tegn, bindestreker godtas)
