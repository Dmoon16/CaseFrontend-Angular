Hvordan administratorer administrerer brukertilordninger til saker

Det er fem måter å administrere brukere og saksoppdrag på. Klikk på en lenke for å finne ut hvordan.

Metode Type Handlinger Detaljer
Mens du oppretter en sak UI legge til Admin-knapp > Saker > Ny knapp > Tildel brukere-knapp
Tilordne bruker automatisk ved hjelp av offentlig registreringslenke Brukerhandling legge til Admin-knapp > Innstillinger > Systemoppsett-siden
Bruker UI på eksisterende sak UI Legg til fjern Admin-knapp > Saker > Tildel brukere
Importverktøy UI legge til, endre og fjerne Admin-knapp > Integrasjoner > Importer
Bruker API (kommer snart) API legge til, endre og fjerne N/A
