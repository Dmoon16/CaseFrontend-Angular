መተግበሪያን እንዴት መፍጠር እንደሚቻል

ይህ ጽሑፍ አዲስ ተጠቃሚዎች እንዴት ነፃ የሙከራ መለያ መፍጠር እንደሚችሉ ያሳያል። ኬዝ ንቁ ለዚህ መለያ ለአዲሱ ተጠቃሚ አስተዳዳሪ ልዩ መብቶችን ይሰጣል።

መለያ መፍጠር

  1. ወደ የመግቢያ ገጹ ይሂዱ.
  2. ጀምር የሚለውን ቁልፍ ጠቅ ያድርጉ።
  3. የኢሜል አድራሻ ያስገቡ።
  4. ቀጣይ የሚለውን ጠቅ ያድርጉ።

img

መለያ በማረጋገጥ ላይ

  1. አዲስ የአሳሽ ትር ይክፈቱ እና ወደ ኢሜል መለያዎ ይግቡ።
  2. ባለ ስድስት አሃዝ ኮድ ይቅዱ።
  3. ወደ ኬዝ ገቢር ትር ተመለስ።
  4. ኮዱን ወደ መስኮቹ ይለጥፉ። አዲስ መስኮት ይታያል.

በአዲሱ መስኮት:

  1. አስፈላጊዎቹን መስኮች ይሙሉ.
  2. የአገልግሎት ውሉን ያንብቡ እና ለመስማማት ሳጥኑ ላይ ምልክት ያድርጉ።
  3. ነፃ መለያ ፍጠር የሚለውን ጠቅ ያድርጉ።

የመተግበሪያ ማዋቀርን በማጠናቀቅ ላይ

የስራ ፍሰት በሚቀጥለው ገጽ ላይ ይታያል። መተግበሪያዎን ማዋቀር ለመጨረስ ደረጃዎቹን ያጠናቅቁ። መረጃን ወይም የመለያዎን ሁኔታ ለማርትዕ ወይም ለማዘመን ሁልጊዜ ወደ እነዚህ ገጾች መመለስ ይችላሉ።

ስለ መተግበሪያ ገጽ የኩባንያውን ስም እና የድር ጣቢያ መስኮችን ይሙሉ (አስፈላጊ) እና በመቀጠል ቀጣይን ጠቅ ያድርጉ።

img

በሂሳብ አከፋፈል መረጃ ገጽ ላይ የክፍያ እቅድ መስኩን ጠቅ ያድርጉ እና አማራጮቹን ይገምግሙ። ነባሪው ቅንብር ነፃ ነው። አስገባን ጠቅ ያድርጉ።

img

ቅጹን ካስገቡ በኋላ ኬዝ አክቲቭ የመተግበሪያዎች ገጽን ያሳያል። ለፈጠሩት ወይም ለሆናችሁት እያንዳንዱ መተግበሪያ፣ ገጹ የሚከተለውን ያሳያል፡-

  1. የመተግበሪያው ስም
  2. የእርስዎ ዓይነት (የተጠቃሚ ሚና)
  3. ሁኔታ

የመተግበሪያዎች ገጽ ለብዙ የአስተዳዳሪ ተግባራት መነሻ ነጥብ ነው፡-

  1. ከጉዳይ ጋር በተያያዙ ጉዳዮች ላይ ለመግባባት፣ ሰነዶችን ለመሙላት፣ የቪዲዮ ውይይት (እና ሌሎችም) ለማድረግ የAPP አዝራሩን ጠቅ ያድርጉ።
  2. የአስተዳዳሪ ፓነልን ለመክፈት የ ADMIN ቁልፍን ጠቅ ያድርጉ። አስተዳዳሪዎች ተጠቃሚዎችን፣ ጉዳዮችን፣ ውህደቶችን እና ሌሎችን የሚያስተዳድሩበት ይህ ነው።
  3. የመለያ ክፍያን እና የማሳወቂያ ቅንብሮችን ለማስተዳደር ellipsis ን ጠቅ ያድርጉ።
  • ማስታወሻ መያዣ አክቲቭ የአስተዳዳሪ ተጠቃሚዎች የ ADMIN ቁልፍን ያሳያል። የአስተዳዳሪ ልዩ መብቶች የሌላቸው ተጠቃሚዎች የ ADMIN ቁልፍን አያዩም (ከዚህ በታች ባለው ስክሪን ላይ እንደሚታየው)።
