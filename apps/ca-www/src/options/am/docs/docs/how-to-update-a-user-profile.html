የተጠቃሚ መገለጫን እንዴት ማዘመን እንደሚቻል

ይህ ጽሑፍ ተጠቃሚዎች በመገለጫ ገጻቸው ላይ መረጃን እንዴት ማዘመን እንደሚችሉ ያሳያል።

  • ተጠቃሚዎች የራሳቸውን መገለጫ ማዘመን ይችላሉ።
  • አስተዳዳሪዎች የማንኛውንም ተጠቃሚ የመገለጫ ገጽ ማዘመን ይችላሉ።

ለተጠቃሚዎች

የመጀመሪያው እርምጃ የመገለጫ ገጹን መክፈት ነው. ተጠቃሚዎች እና አስተዳዳሪዎች የራሳቸውን የመገለጫ ገጽ ለማዘመን እነዚህን ደረጃዎች ይከተላሉ።

  1. ወደ የጉዳይ ገቢር መለያዎ ይግቡ።
  2. በገጹ አናት ላይ ያለውን የመገለጫ ትርን ጠቅ ያድርጉ።
  3. ለውጦችን ያድርጉ።
  4. አስቀምጥን ጠቅ ያድርጉ።

አብዛኞቹ መስኮች አማራጭ ናቸው፣ እና አንዳንድ መስኮች ያስፈልጋሉ። የሚፈለጉት መስኮች፡-

  • ቋንቋ

ለአድሚኖች

አስተዳዳሪዎች የመገለጫውን መረጃ ለሌሎች ተጠቃሚዎች ማዘመን ይችላሉ። ይህ በHOST USER መረጃ ርዕስ ስር ሶስት መስኮችን ያካትታል።

  1. ወደ የጉዳይ ገቢር አስተዳዳሪ መለያዎ ይግቡ።
  2. አንድ መተግበሪያ ያግኙ እና ADMIN ቁልፍን ጠቅ ያድርጉ።
  3. በላይኛው ምናሌ ላይ ተጠቃሚዎችን ጠቅ ያድርጉ።
  4. ወደ ንዑስ ምናሌው ይሂዱ እና ሊደርሱበት የሚፈልጉትን የተጠቃሚ ዝርዝር ይምረጡ (አክቲቭ ወይም ተሰናክሏል)። ነባሪው ቅንብር ገባሪ ነው።
  5. በዝርዝሩ ውስጥ ተጠቃሚን ያግኙ እና አርትዕን ጠቅ ያድርጉ።
  6. እንደ አስፈላጊነቱ ለውጦችን ያድርጉ። (ከታች ይመልከቱ)
  7. አስቀምጥን ጠቅ ያድርጉ።
