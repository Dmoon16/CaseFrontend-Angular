በዩአይ እንዴት ተጠቃሚዎችን መፍጠር እና ማሻሻል እንደሚቻል

ይህ መጣጥፍ አስተዳዳሪዎች ተጠቃሚዎችን እንዴት መፍጠር እንደሚችሉ እና መዝገቦችን ከመድረክ የተጠቃሚ በይነገጽ ጋር እንደሚያዘምኑ ያሳያል። አስተዳዳሪዎች ተጠቃሚዎችን ወደ አዲስ ጉዳይ ማከል እና ነባር ጉዳዮችን ማርትዕ ይችላሉ።

  3. በአስተዳዳሪው ፓነል ላይ የተጠቃሚዎች ምናሌን ጠቅ ያድርጉ።
  4. አዲስ ቁልፍን ጠቅ ያድርጉ። አዲስ መስኮት ይታያል.

ተጠቃሚን በማስተካከል ላይ

አነስተኛ ቁጥር ያላቸውን የተጠቃሚ መዝገቦች ለማዘመን፣ የአርትዕ መሳሪያውን ይጠቀሙ እና በእያንዳንዱ ተጠቃሚ ላይ ለውጦችን ያድርጉ።

  • ማስታወሻ በአንድ መቀመጫ ውስጥ ብዙ ቁጥር ያላቸውን የተጠቃሚ መዝገቦችን ለማሻሻል፣ የማስመጣት መሳሪያውን ለመጠቀም ያስቡበት።
  1. በአስተዳዳሪው ፓነል ላይ የተጠቃሚዎች ምናሌን ጠቅ ያድርጉ።
  2. ተጠቃሚን ያግኙ እና አርትዕን ጠቅ ያድርጉ።
  3. ለውጦችን ያድርጉ እና አስቀምጥን ጠቅ ያድርጉ።

ቀጣይ እርምጃዎች

ኬዝ አክቲቭ ለተጠቃሚው የኢሜይል ማሳወቂያ አይልክም። አስተዳዳሪዎች ተጠቃሚውን ማነጋገር እና ወደ ኬዝ ገቢር መነሻ ገጽ አገናኝ መላክ አለባቸው። ከዚያ ተጠቃሚው የመግቢያ አዝራሩን ጠቅ በማድረግ ለአዲስ ምዝገባዎች የስራ ሂደቱን ያጠናቅቃል።

  • ይህ መጣጥፍ ለአዲስ ተጠቃሚዎች የምዝገባ የስራ ሂደትን ይገልጻል።
