በአስመጪ መሳሪያው እንዴት ተጠቃሚዎችን መፍጠር ወይም ማሻሻል እንደሚቻል

ይህ መጣጥፍ አስተዳዳሪዎች በአስመጪ መሳሪያው ተጠቃሚዎችን እንዴት መፍጠር እንደሚችሉ ያሳያል።

የማስመጣት መሣሪያውን በመክፈት ላይ

አስተዳዳሪዎች አዳዲስ ተጠቃሚዎችን በጅምላ ለመስቀል እና ነባር የተጠቃሚ መዝገቦችን ለማስተካከል የማስመጣት መሳሪያውን መጠቀም ይችላሉ።

  1. ወደ የአስተዳዳሪ መለያ ይግቡ።
  2. በመተግበሪያዎች ገጽ ላይ አንድ መተግበሪያ ያግኙ እና የ ADMIN ቁልፍን ጠቅ ያድርጉ።
  3. በአስተዳዳሪው ፓነል ላይ የውህደት ትርን ጠቅ ያድርጉ እና በንዑስ ምናሌው ላይ IMPORTS ን ጠቅ ያድርጉ።
  4. አዲስ ቁልፍን ጠቅ ያድርጉ። የማስመጣት አክል የንግግር ሳጥን በአዲስ አሳሽ ትር ውስጥ ይታያል።

መዝገቦችን መፍጠር

የማስመጣት አክል የንግግር ሳጥን ከተከፈተ በኋላ፡-

  1. ዓይነት መስኩን ጠቅ ያድርጉ እና አንድ አማራጭ ይምረጡ (ከዚህ በታች ይመልከቱ)። አንድ አገናኝ ያሳያል.
  2. የማውረድ ናሙና ማገናኛን ጠቅ ያድርጉ። የCSV ፋይል ወደ መሳሪያዎ ይወርዳል።

በመቀጠል ፋይሉን ይክፈቱ እና መዝገቦችን ያክሉ. በCSV ፋይል ውስጥ የቅርጸት ውሂብን እንዴት መጠቀም እንደሚቻል ለማወቅ እዚህ ጠቅ ያድርጉ።

ዓይነት

የአይነት መስኩን ሲጫኑ ተቆልቋይ ሜኑ ይታያል። ለጅምላ ማስመጣት ከሁለት አማራጮች አንዱን ይምረጡ።

: አጋኖ: ማስታወሻ ተቆልቋይ ሜኑ ብዙ አማራጮች አሉት፣ ግን ሁለቱ ብቻ ከአመጫ መሳሪያ ጋር የተያያዙ ናቸው። እያንዳንዱ አማራጭ የተለየ የCSV ፋይል ናሙና አለው።

ተጠቃሚዎችን ያክሉ

  • ለቀላል የተጠቃሚ መዝገቦች ጭነት ይህን አማራጭ ይምረጡ።
  • የተጠቃሚዎችን አስመጪ አይነት የአምድ አርእስት ያለው የCSV ፋይል ለመክፈት የማውረጃ ናሙና አገናኙን ጠቅ ያድርጉ።

ተጠቃሚዎችን አስተካክል።

  • ለአሁኑ ተጠቃሚዎች ውሂብ ለማዘመን ይህን አማራጭ ይምረጡ።
  • ለተጠቃሚዎች አስመጪ አይነት የአምድ አርእስት ያለው የCSV ፋይል ለመክፈት የማውረጃ ናሙና አገናኙን ጠቅ ያድርጉ።

ፋይሉን በመስቀል ላይ

ፋይሉን ወዲያውኑ መስቀል ወይም ወደ ተጨማሪ ቀን ወደ አስመጪ ማስታወቂያ ሳጥን መመለስ እና እነዚህን ደረጃዎች ማጠናቀቅ ትችላለህ።

  1. ከውጭ የሚመጡትን አክል የንግግር ሳጥን ይክፈቱ።
  2. ለCSV ፋይል ልዩ ስም ያስገቡ።
  3. ፋይሉን ከመሳሪያዎ ለመስቀል የመደመር (+) አዶን ጠቅ ያድርጉ።
  4. ዓይነት መስኩን ጠቅ ያድርጉ እና አንድ አማራጭ ይምረጡ። (ከላይ ይመልከቱ)
  5. አስቀምጥን ጠቅ ያድርጉ።
