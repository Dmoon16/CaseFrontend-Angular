የተጠቃሚ ኬዝ ምደባዎች ውሂብን በአስመጪ መሳሪያው መቅረጽ

ከዚህ በታች በCSV ፋይሎች ውስጥ ያሉ የውሂብ መስኮች ዝርዝር አለ። እያንዳንዱ የማስመጣት አማራጭ የተለየ የመስኮች ጥምረት አለው።

መያዣ_መታወቂያ

  • የጉዳይ ተጠቃሚ ወይም የመለያ መታወቂያ ያስገቡ።
  • ወደ የጉዳይ ገጽ > አርትዕ > መለያ መታወቂያ ወይም ወደ ኬዝ ገጽ > አርትዕ > የጉዳይ መታወቂያ በዩአርኤል ይሂዱ።

የተጠቃሚው መለያ

  • የተጠቃሚውን መታወቂያ ወይም የመለያ መታወቂያ ያስገቡ።
  • ወደ የተጠቃሚዎች ገጽ > አርትዕ > መለያ መታወቂያ ወይም ወደ የተጠቃሚዎች ገጽ > አርትዕ > የተጠቃሚ መታወቂያ በዩአርኤል ይሂዱ።

የጉዳይ_ሚና_ትእዛዝ

  • ሕዋስ ባዶውን ይተውት።

የጉዳይ_ሚና_መታወቂያ

  • አስገባ፡ ደንበኞች፣ ሰራተኞች ወይም ነባሪ።

ጉዳይ_ማሳወቅ

  • 1 = አዎ፣ ለተጠቃሚ የኢሜይል ማሳወቂያ ይላኩ።
