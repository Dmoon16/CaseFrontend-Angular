Formatarea datelor de gestionare a cazurilor cu Instrumentul de import

Mai jos este o listă de câmpuri obligatorii din fișierele CSV pentru a gestiona cazurile cu instrumentul Import.

nume de utilizator

  • Introdu adresa de e-mail a utilizatorului.

tag_id (un id pe care îl puteți referi mai târziu)

  • un identificator unic (utilizați caractere alfanumerice, cratimele sunt acceptate)

stare_caz

  • 0 = închis
  • 1 = activ

data_deschise

  • luna zi an
  • mm/zz/aaaa

despre

  • o descriere unică a cazului (utilizați caractere alfanumerice, se acceptă cratimele)
