Cum să atribuiți, să modificați sau să eliminați asignările de utilizator cu Instrumentul de import

Acest articol arată administratorilor cum să gestioneze utilizatorii de cazuri cu instrumentul Import.

Administratorii pot:

  • atribuiți utilizatori unui caz
  • modifica utilizatorii deja alocați unui caz
  • eliminați utilizatorii alocați unui caz

Deschiderea instrumentului Import

  1. Conectați-vă la contul de administrator.
  2. Găsiți o aplicație pe pagina Aplicații și faceți clic pe butonul ADMIN.
  3. Faceți clic pe fila Integrare și apoi pe IMPORTĂRI din submeniu.
  4. Faceți clic pe butonul NOU. Se afișează caseta de dialog Adăugați importuri.

Crearea înregistrărilor

Descărcați unul sau mai multe fișiere CSV. Utilizați aceste fișiere pentru a adăuga, modifica sau elimina utilizatori de caz. După ce se deschide caseta de dialog Adăugați importuri:

  1. Faceți clic pe câmpul Tip.
  2. Derulați prin meniul derulant și selectați o opțiune (vezi mai jos). Se afișează un link.
  3. Faceți clic pe linkul Descărcare eșantion. Un fișier CSV se descarcă pe dispozitivul dvs.

Deschideți fișierul și adăugați înregistrări. Faceți clic aici pentru a afla cum să utilizați datele de format în fișierul CSV.

Opțiuni pentru fișiere

Atribuiți utilizatori la cazuri * Alegeți această opțiune pentru a adăuga utilizatori la un caz. * Lăsați necompletat celula case_role_order. * Accesați pagina Utilizatori și copiați ID-ul etichetei pentru această persoană. Dacă câmpul ID etichetă este gol, introduceți un ID unic, faceți clic pe Salvare, apoi copiați ID-ul în fișierul CSV.

Modificați utilizatorii alocați cazurilor * Alegeți această opțiune pentru a actualiza datele pentru utilizatorii actuali alocați unui caz. * Utilizați ID-ul etichetei cazului pentru coloana case_ID.

Eliminați utilizatorii alocați cazurilor * Alegeți această opțiune pentru a elimina utilizatorii alocați unui caz.

Încărcarea fișierului

Puteți încărca un fișier imediat sau puteți reveni la caseta de dialog Adăugați importuri la o dată ulterioară și puteți parcurge acești pași.

  1. Deschideți caseta de dialog Adăugați importuri.
  2. Introduceți un nume unic pentru fișierul CSV.
  3. Faceți clic pe pictograma plus (+) pentru a încărca fișierul de pe dispozitiv.
  4. Faceți clic pe câmpul Tip și selectați o opțiune. (Vezi deasupra)
  5. Faceți clic pe SALVARE.

Prelucrare

În funcție de numărul de înregistrări din fișier și de încărcarea sistemului, platforma poate dura câteva minute pentru a procesa importul.

Când se finalizează importul, platforma ajustează înregistrările utilizatorilor (adică adaugă, modifică sau elimină) pe pagina Cazuri.

Eșec

Dacă operațiunea de import eșuează, platforma returnează un mesaj Eșuat. În cele mai multe cazuri, cauza eșecului este formatarea datelor.

Pentru a vedea cauza defecțiunii:

  1. Faceți clic pe Vizualizare sub titlul Acțiune.
  2. Pe pagina următoare, plasați mouse-ul pe pictograma Eșuat. Sfatul explicativ afișează eroarea.
  3. Accesați fișierul CVS și ajustați și remediați eroarea.
  4. Încărcați din nou fișierul.
