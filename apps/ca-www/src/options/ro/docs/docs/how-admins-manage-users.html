Cum gestionează administratorii utilizatorii

Administratorii creează, modifică sau dezactivează cine poate accesa aplicații și cazuri. Când un utilizator se conectează la platformă pentru prima dată, trebuie să finalizeze un flux de lucru de profil.

  • Faceți clic aici pentru a afla mai multe despre fluxul de lucru al noului profil de utilizator.

Gestionarea unui utilizator

Tabelul de mai jos prezintă cinci moduri de a gestiona un utilizator. Faceți clic pe un link pentru a afla cum.

Metodă Tip Acțiuni Detalii
Folosind UI UI adăugați și modificați Buton Administrator > Utilizatori
În timp ce creați un caz și atribuiți utilizator la caz UI adăuga Buton Administrator > Cazuri > Buton Nou > Pagina Creare caz

Notificări

Platforma nu trimite utilizatorilor o notificare atunci când sunt adăugați la o aplicație. Cu toate acestea, administratorii pot activa notificările pentru atribuirea cazurilor.
