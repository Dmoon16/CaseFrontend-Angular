Formatarea datelor de gestionare a utilizatorilor cu Instrumentul de import

Mai jos este o listă de câmpuri de date din fișierele CSV. Fiecare opțiune de import are o combinație diferită de câmpuri.

nume de utilizator

  • Introdu adresa de e-mail a utilizatorului.

prenume

  • Introduceți prenumele utilizatorului.

nume de familie

  • Introduceți numele de familie al utilizatorului.

local

  • Această coloană verifică limba preferată a utilizatorului.
  • Introduceți en pentru engleză, sp pentru spaniolă.

zoneinfo

  • Aceasta setează fusul orar al utilizatorului. Introduceți continentul și orașul fără spații (utilizați un caracter de subliniere).
  • Trebuie să utilizați o opțiune din meniul derulant Profil > LIMBĂ & REGIUNE > Fus orar.
  • Utilizați fusul orar cel mai potrivit. De exemplu, Boston nu este în meniul drop-down, așa că putem folosi New York (sau orice oraș din fusul orar EST/EDT).

host_tag_id

  • Introduceți o etichetă permanentă.
  • az, 0-9, -, / sunt permise.

stare_caz

  • 0 = închis
  • 1 = activ
