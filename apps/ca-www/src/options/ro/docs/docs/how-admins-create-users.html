Cum creează administratorii utilizatori

Administratorii creează, modifică sau dezactivează cine poate accesa aplicații și cazuri. Când un utilizator se conectează la platformă pentru prima dată, trebuie să finalizeze un flux de lucru de profil.

  • Faceți clic aici pentru a afla mai multe despre fluxul de lucru al noului profil de utilizator.

Crearea unui utilizator

Tabelul de mai jos prezintă cinci moduri de a crea un utilizator. Faceți clic pe un link pentru a afla cum.

| Metoda | Tip | Acțiuni | Detalii | Un utilizator nou primește o notificare? |
|--------------------------|----------------|------- ------|---------------------|----------------------- ------------------------|

Folosind UI | UI | adăugați și modificați | Buton Administrator > Utilizatori | Nu |

În timp ce creați un caz și atribuiți utilizatorului cazului | UI | adauga | Buton Administrator > Cazuri > Buton Nou > Pagina Creare caz | Da, dacă este activat |

Creare automată a utilizatorului utilizând linkul de înregistrare public | Acțiunea utilizatorului | adauga | Buton Administrator > Setări > Pagina Configurare sistem | Da, dacă este activată crearea automată a majusculei |

Instrument de import | UI | adăugați și modificați | Buton Administrator > Integrari > Importuri | Nu |

Utilizarea API (în curând) | API | adăugați și modificați | na | na |

Notificări

Platforma nu trimite utilizatorilor o notificare atunci când sunt adăugați la o aplicație. Cu toate acestea, administratorii pot activa notificările pentru atribuirea cazurilor.
