Cum să creați sau să modificați cazuri cu Instrumentul de import

Acest articol arată administratorilor cum să creeze sau să modifice cazurile existente cu instrumentul Import.

Deschiderea instrumentului Import

  1. Conectați-vă la contul de administrator.
  2. Găsiți o aplicație pe pagina Aplicații și faceți clic pe butonul ADMIN.
  3. În panoul de administrare, faceți clic pe fila Integrare și apoi pe IMPORTĂRI din submeniu.
  4. Faceți clic pe butonul NOU. Caseta de dialog Adăugați importuri se afișează într-o filă nouă de browser.

img

Crearea cazurilor

După ce se deschide caseta de dialog Adăugați importuri:

  1. Faceți clic pe câmpul Tip și selectați o opțiune (vezi mai jos). Se afișează un link.
  2. Faceți clic pe linkul Descărcare eșantion. Un fișier CSV se descarcă pe dispozitivul dvs.

Apoi, deschideți fișierul și adăugați cazuri.

Tip

Când faceți clic pe câmpul Tip , se afișează un meniu derulant. Alegeți una dintre cele două opțiuni pentru importurile de cazuri în bloc.

  • Notă Meniul drop-down are multe opțiuni, dar doar două sunt legate de gestionarea cazului. Fiecare opțiune are un fișier CSV exemplu diferit.

Adăugați cazuri

  • Alegeți această opțiune pentru o încărcare simplă în bloc de cazuri.
  • Faceți clic pe linkul Descărcare eșantion pentru a deschide un fișier CSV cu titluri de coloană pentru tipul de import Creare cazuri.

Modificați cazuri

  • Alegeți această opțiune pentru a actualiza cazurile curente.
  • Faceți clic pe linkul Descărcare eșantion pentru a deschide un fișier CSV cu titluri de coloană pentru tipul de import Modificare cazuri.

Încărcarea fișierului

Puteți încărca un fișier imediat sau puteți reveni la caseta de dialog Adăugați importuri la o dată ulterioară și puteți parcurge acești pași.

  1. Deschideți caseta de dialog Adăugați importuri.
  2. Introduceți un nume unic pentru fișierul CSV.
  3. Faceți clic pe pictograma plus (+) pentru a încărca fișierul de pe dispozitiv.
  4. Faceți clic pe câmpul Tip și selectați o opțiune. (Vezi deasupra)
  5. Faceți clic pe SALVARE.

Formatare

Acestea sunt câmpurile obligatorii și primele cerințe. Faceți clic aici pentru a examina detaliile de formatare.

tag_id (sau case_ID) un identificator unic (utilizați caractere alfanumerice, cratimele sunt acceptate)

case_status 0 = închis 1 = activ

data_deschis luna/zi/an mm/zz/aaaa

despre o descriere unică a cazului (utilizați caractere alfanumerice, se acceptă cratime)

img

Prelucrare

În funcție de numărul de înregistrări din fișier și de încărcarea sistemului, platforma poate dura câteva minute pentru a procesa importul.

img

Când se finalizează importul, platforma ajustează înregistrările cazurilor (adică adaugă sau modifică) în tabloul de bord Cazuri al administratorului.

Eșec

Dacă operațiunea de import eșuează, platforma returnează un mesaj Eșuat. În cele mai multe cazuri, cauza eșecului este formatarea datelor.

img

Pentru a vedea cauza defecțiunii:

  1. Faceți clic pe Vizualizare sub titlul ACȚIUNE.
  2. Pe pagina următoare, plasați mouse-ul pe pictograma Eșuat. Sfatul explicativ afișează eroarea.
  3. Accesați fișierul CVS și ajustați și remediați eroarea.
  4. Încărcați din nou fișierul.
