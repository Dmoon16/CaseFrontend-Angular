Cum gestionează administratorii atribuțiile utilizatorilor către cazuri

Există cinci moduri de a gestiona utilizatorii și atribuirile de cazuri. Faceți clic pe un link pentru a afla cum.

Metodă Tip Acțiuni Detalii
În timpul creării unui caz UI adăuga Buton Administrator > Cazuri > Buton Nou > Buton Atribuire utilizatori
Atribuire automată a utilizatorului utilizând linkul de înscriere publică Acțiunea utilizatorului adăuga Buton Administrator > Setări > pagina Configurare sistem
Utilizarea interfeței de utilizare pe cazul existent UI adăugați și eliminați Buton Administrator > Cazuri > Atribuire utilizatori
Instrument de import UI adăugați, modificați și eliminați Buton Administrator > Integrari > Importuri
Utilizarea API (în curând) API adăugați, modificați și eliminați N / A
