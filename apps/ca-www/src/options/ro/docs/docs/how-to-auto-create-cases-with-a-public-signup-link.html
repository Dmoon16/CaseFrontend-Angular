Cum să creați automat cazuri cu un link de înregistrare public

Acest articol arată administratorilor cum să creeze un caz (și să adauge un utilizator în același timp) trimițând un singur link de înscriere.

Localizarea linkului

  1. Conectați-vă la contul de administrator.
  2. Găsiți o aplicație pe pagina Aplicații și faceți clic pe butonul ADMIN.
  3. În panoul de administrare, faceți clic pe fila meniului Setări.
  4. Asigurați-vă că fila submeniului SYSTEM SETUP este selectată.

img

Configurarea setărilor

Derulați în jos la titlul ÎNSCRIERE UTILIZATOR.

Asigurați-vă că câmpul Permitere înscriere este setat la Activat. (obligatoriu) Când această caracteristică este activată, Case Active generează o legătură unică.

  1. Faceți clic pe câmpul Creare automată a cazului și selectați Activat. Se afișează câmpul Rol utilizator al cazului.
  2. Selectați un rol. (vezi mai jos)

Roluri de utilizator de caz

Există trei roluri de utilizator de caz.

  • clienti (recomandat)
  • Mod implicit
  • personal

Se trimite link-ul

Distribuiți linkul cu una sau mai multe persoane. Fiecare utilizator care face clic pe link și finalizează înregistrarea declanșează platforma să genereze un nou caz pe pagina cazurilor și să atribuie automat utilizatorul în caz.
