বিদ্যমান কেসগুলির সাথে ব্যবহারকারীর অ্যাসাইনমেন্টগুলি কীভাবে বরাদ্দ করা যায় বা সরানো যায়

এই নিবন্ধটি প্রশাসকদের দেখায় কিভাবে কেস ব্যবহারকারীদের বরাদ্দ করতে এবং সরাতে হয়। আমরা আপনাকে ব্যবহারকারীর ডেটা কীভাবে পরিবর্তন করতে হয় তাও দেখাই।

একটি বিদ্যমান মামলা খোলা

  1. অ্যাডমিন অ্যাকাউন্টে লগ ইন করুন।
  2. অ্যাপস পৃষ্ঠায় একটি অ্যাপ খুঁজুন এবং অ্যাডমিন বোতামে ক্লিক করুন।
  3. কেস মেনু ট্যাবে ক্লিক করুন।
  4. নিশ্চিত করুন যে সাবমেনুতে ACTIVE নির্বাচন করা হয়েছে।
  5. ACTION কলামে নিচে স্ক্রোল করুন।

বরাদ্দ বা অপসারণ

কেসটিতে একজন ব্যবহারকারীকে বরাদ্দ করতে বা কেস থেকে ব্যবহারকারীকে সরিয়ে দিতে ব্যবহারকারীদের বরাদ্দ করুন ক্লিক করুন (কিন্তু প্ল্যাটফর্ম নয়)। পরবর্তী পৃষ্ঠায়: ADD এ ক্লিক করুন এবং কর্মপ্রবাহটি সম্পূর্ণ করুন। একটি ব্যবহারকারী মুছে ফেলতে ক্লিক করুন.

পরিবর্তন করুন

কেস পৃষ্ঠায় ব্যবহারকারীর ডেটা সম্পাদনা বা আপডেট করা সম্ভব নয়। পরিবর্তে, এই পদক্ষেপগুলি অনুসরণ করুন:

  1. ব্যবহারকারী মেনু ট্যাবে ক্লিক করুন।
  2. ব্যবহারকারীর নাম সনাক্ত করুন এবং সম্পাদনা ক্লিক করুন।
  3. পরিবর্তন করুন এবং সংরক্ষণ করুন ক্লিক করুন.
