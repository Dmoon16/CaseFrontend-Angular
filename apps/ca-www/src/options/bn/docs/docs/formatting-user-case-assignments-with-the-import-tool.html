ইম্পোর্ট টুলের সাথে ইউজার কেস অ্যাসাইনমেন্ট ডেটা ফর্ম্যাট করা

নীচে CSV ফাইলগুলিতে ডেটা ক্ষেত্রগুলির একটি তালিকা রয়েছে৷ প্রতিটি আমদানি বিকল্পের ক্ষেত্রের একটি ভিন্ন সমন্বয় আছে।

case_ID

  • কেস ব্যবহারকারী বা ট্যাগ আইডি লিখুন।
  • কেস পৃষ্ঠাতে যান > সম্পাদনা > ট্যাগ আইডি বা কেস পৃষ্ঠাতে যান > সম্পাদনা > url-এ কেস আইডি।

ব্যবহারকারীর প্রমানপত্র

  • ব্যবহারকারীর আইডি বা ট্যাগ আইডি লিখুন।
  • ইউজার পেজ > এডিট > ট্যাগ আইডিতে যান অথবা ইউজার পেজে যান > এডিট > ইউআরএলে ইউজার আইডি।

case_role_ID

  • 1 = হ্যাঁ, ব্যবহারকারীকে একটি ইমেল বিজ্ঞপ্তি পাঠান।
