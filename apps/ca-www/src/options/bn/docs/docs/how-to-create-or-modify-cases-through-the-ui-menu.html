UI মেনুর মাধ্যমে কীভাবে কেস তৈরি বা পরিবর্তন করবেন

এই নিবন্ধটি প্রশাসকদের দেখায় কিভাবে প্ল্যাটফর্মের ব্যবহারকারী ইন্টারফেসের সাথে একটি কেস তৈরি করতে হয়। প্রশাসকরা একটি নতুন ক্ষেত্রে ব্যবহারকারীদের যোগ করতে এবং বিদ্যমান ক্ষেত্রে আপডেট করতে পারেন।

মামলা তৈরি করছে

  1. অ্যাডমিন অ্যাকাউন্টে লগ ইন করুন।
  2. অ্যাপস পৃষ্ঠায় একটি অ্যাপ খুঁজুন এবং অ্যাডমিন বোতামে ক্লিক করুন।
  3. অ্যাডমিন প্যানেলে, কেস মেনু ট্যাবে ক্লিক করুন।
  4. নতুন বোতামে ক্লিক করুন। একটি নতুন উইন্ডো প্রদর্শিত হবে।

প্রয়োজনীয় ক্ষেত্রগুলি পূরণ করুন এবং সংরক্ষণ করুন ক্লিক করুন।

ব্যবহারকারীদের যোগ করা হচ্ছে

ঐচ্ছিকভাবে, অ্যাডমিনরা এই পৃষ্ঠার ক্ষেত্রে ব্যবহারকারীদের যোগ করতে পারেন। নীচে স্ক্রোল করুন এবং ব্যবহারকারীদের নিয়োগ বোতামে ক্লিক করুন।

img

অ্যাসাইন ইউজার মোডেলে, ক্ষেত্রগুলি পূরণ করুন। ব্যবহারকারীকে একটি ইমেল বিজ্ঞপ্তি পাঠাতে বাক্সটি চেক করুন৷ (ঐচ্ছিক)

একটি মামলা পরিবর্তন

অল্প সংখ্যক ক্ষেত্রে আপডেট করতে, সম্পাদনা টুল ব্যবহার করুন এবং প্রতিটি পৃথক ক্ষেত্রে পরিবর্তন করুন।

  • দ্রষ্টব্য এক বৈঠকে বিপুল সংখ্যক কেস আপডেট করতে, আমদানি টুল ব্যবহার করার কথা বিবেচনা করুন
  1. অ্যাডমিন অ্যাকাউন্টে লগ ইন করুন।
  2. অ্যাপস পৃষ্ঠায় একটি অ্যাপ খুঁজুন এবং অ্যাডমিন বোতামে ক্লিক করুন।
  3. অ্যাডমিন প্যানেলে, কেস মেনু ট্যাবে ক্লিক করুন।
  4. একটি কেস সনাক্ত করুন এবং সম্পাদনা ক্লিক করুন।
  5. পরিবর্তন করুন এবং সংরক্ষণ করুন ক্লিক করুন.
