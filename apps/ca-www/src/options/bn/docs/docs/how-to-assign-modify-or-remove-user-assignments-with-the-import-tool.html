ইম্পোর্ট টুলের সাহায্যে কিভাবে ইউজার অ্যাসাইনমেন্ট বরাদ্দ, পরিবর্তন বা অপসারণ করবেন

এই নিবন্ধটি প্রশাসকদের দেখায় কিভাবে কেস ব্যবহারকারীদের ইম্পোর্ট টুল দিয়ে পরিচালনা করতে হয়।

অ্যাডমিনরা করতে পারেন:

  • একটি ক্ষেত্রে ব্যবহারকারীদের বরাদ্দ করুন
  • ইতিমধ্যে একটি ক্ষেত্রে বরাদ্দ করা ব্যবহারকারীদের সংশোধন করুন
  • একটি ক্ষেত্রে বরাদ্দ করা ব্যবহারকারীদের সরান

আমদানি টুল খোলা হচ্ছে

  1. অ্যাডমিন অ্যাকাউন্টে লগ ইন করুন।
  2. অ্যাপস পৃষ্ঠায় একটি অ্যাপ খুঁজুন এবং অ্যাডমিন বোতামে ক্লিক করুন।
  3. ইন্টিগ্রেশন ট্যাবে ক্লিক করুন এবং তারপর সাবমেনুতে আমদানি করুন।
  4. নতুন বোতামে ক্লিক করুন। আমদানি যোগ করুন ডায়ালগ বক্স প্রদর্শন করে।

রেকর্ড তৈরি করা

এক বা একাধিক CSV ফাইল ডাউনলোড করুন। কেস ব্যবহারকারীদের যোগ করতে, সংশোধন করতে বা অপসারণ করতে এই ফাইলগুলি ব্যবহার করুন৷ আমদানি যোগ করুন ডায়ালগ বক্স খোলার পরে:

  1. Type ক্ষেত্রে ক্লিক করুন।
  2. ড্রপ-ডাউন মেনু দিয়ে স্ক্রোল করুন এবং একটি বিকল্প নির্বাচন করুন (নীচে দেখুন)। একটি লিঙ্ক প্রদর্শন করে।
  3. ডাউনলোড নমুনা লিঙ্কে ক্লিক করুন. একটি CSV ফাইল আপনার ডিভাইসে ডাউনলোড হয়।

ফাইল খুলুন এবং রেকর্ড যোগ করুন. CSV ফাইলে ফর্ম্যাট ডেটা কীভাবে ব্যবহার করবেন তা শিখতে এখানে ক্লিক করুন।

ফাইল অপশন

ক্ষেত্রে ব্যবহারকারীদের বরাদ্দ করুন * একটি ক্ষেত্রে ব্যবহারকারীদের যুক্ত করতে এই বিকল্পটি চয়ন করুন৷ * case_role_order ঘরটি ফাঁকা রাখুন। * ব্যবহারকারীদের পৃষ্ঠায় যান এবং এই ব্যক্তির জন্য ট্যাগ আইডি কপি করুন। ট্যাগ আইডি ক্ষেত্রটি ফাঁকা থাকলে, একটি অনন্য আইডি লিখুন, সংরক্ষণ করুন ক্লিক করুন এবং তারপর CSV ফাইলে আইডিটি অনুলিপি করুন৷

ক্ষেত্রে বরাদ্দকৃত ব্যবহারকারীদের পরিবর্তন করুন * একটি ক্ষেত্রে বরাদ্দ করা বর্তমান ব্যবহারকারীদের জন্য ডেটা আপডেট করতে এই বিকল্পটি বেছে নিন। * case_ID কলামের জন্য কেস ট্যাগ আইডি ব্যবহার করুন।

ক্ষেত্রে বরাদ্দ করা ব্যবহারকারীদের সরান * একটি ক্ষেত্রে বরাদ্দ করা ব্যবহারকারীদের সরাতে এই বিকল্পটি বেছে নিন।

ফাইল আপলোড করা হচ্ছে

আপনি এখনই একটি ফাইল আপলোড করতে পারেন বা পরবর্তী তারিখে আমদানি যোগ করুন ডায়ালগ বক্সে ফিরে আসতে পারেন এবং এই পদক্ষেপগুলি সম্পূর্ণ করুন৷

  1. আমদানি যোগ করুন ডায়ালগ বক্স খুলুন।
  2. CSV ফাইলের জন্য একটি অনন্য নাম লিখুন।
  3. আপনার ডিভাইস থেকে ফাইল আপলোড করতে প্লাস (+) আইকনে ক্লিক করুন।
  4. টাইপ ক্ষেত্রে ক্লিক করুন এবং একটি বিকল্প নির্বাচন করুন। (উপরে দেখুন)
  5. SAVE এ ক্লিক করুন।

প্রক্রিয়াকরণ

ফাইল এবং সিস্টেম লোডের রেকর্ডের সংখ্যার উপর নির্ভর করে, প্ল্যাটফর্মটি আমদানি প্রক্রিয়া করতে কয়েক মিনিট সময় নিতে পারে।

যখন আমদানি সম্পূর্ণ হয়, প্ল্যাটফর্মটি কেস পৃষ্ঠায় ব্যবহারকারীর রেকর্ডগুলিকে সামঞ্জস্য করে (যেমন যোগ করে, সংশোধন করে বা অপসারণ করে)।

ব্যর্থতা

যদি আমদানি ক্রিয়াকলাপ ব্যর্থ হয়, প্ল্যাটফর্মটি একটি ব্যর্থ বার্তা প্রদান করে। বেশিরভাগ ক্ষেত্রে, ব্যর্থতার কারণ ডেটা ফরম্যাটিং।

ব্যর্থতার কারণ দেখতে:

  1. অ্যাকশন শিরোনামের অধীনে দেখুন ক্লিক করুন।
  2. পরের পৃষ্ঠায়, ব্যর্থ আইকনে মাউস নিয়ে যান। টুলটিপ ত্রুটি প্রদর্শন করে।
  3. CVS ফাইলে যান এবং সামঞ্জস্য করুন এবং ত্রুটিটি ঠিক করুন।
  4. ফাইলটি আবার আপলোড করুন।
