কিভাবে লগ ইন করবেন

আপনার বিদ্যমান কেস অ্যাক্টিভ অ্যাকাউন্টে কীভাবে লগ ইন করবেন তা শিখুন।

লগ ইন করছি

  1. লগইন পৃষ্ঠায় যান।
  2. লগইন বোতামে ক্লিক করুন।
  3. তুমার ইমেইল প্রবেশ করাও.
  4. পরবর্তী ক্লিক করুন.

একটি অ্যাকাউন্ট যাচাই করা হচ্ছে

  1. একটি নতুন ব্রাউজার ট্যাব খুলুন এবং আপনার ইমেল অ্যাকাউন্টে লগ ইন করুন।
  2. ছয় সংখ্যার কোড কপি করুন।
  3. কেস অ্যাক্টিভ ট্যাবে ফিরে যান।
  4. ক্ষেত্রগুলিতে কোডটি আটকান। একটি নতুন উইন্ডো প্রদর্শিত হবে।

কেস অ্যাক্টিভ অ্যাপস পৃষ্ঠা প্রদর্শন করে।
