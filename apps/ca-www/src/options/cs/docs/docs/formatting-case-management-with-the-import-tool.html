Formátování dat správy případů pomocí nástroje pro import

Níže je uveden seznam povinných polí v souborech CSV pro správu případů pomocí nástroje Import.

uživatelské jméno

  • Zadejte e-mailovou adresu uživatele.

tag_id (ID, na které se můžete odkázat později)

  • jedinečný identifikátor (použijte alfanumerické znaky, pomlčky jsou akceptovány)

case_status

  • 0 = uzavřeno
  • 1 = aktivní

datum_otevřeno

  • měsíc den rok
  • mm/dd/rrrr

o

  • jedinečný popis případu (použijte alfanumerické znaky, pomlčky jsou akceptovány)
