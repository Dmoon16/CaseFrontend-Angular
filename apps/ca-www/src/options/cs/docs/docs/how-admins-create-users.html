Jak administrátoři vytvářejí uživatele

Správci vytvářejí, upravují nebo zakazují, kdo má přístup k aplikacím a případům. Když se uživatel přihlásí k platformě poprvé, musí dokončit pracovní postup profilu.

  • Kliknutím sem získáte další informace o pracovním postupu nového uživatelského profilu.

Vytvoření uživatele

Níže uvedená tabulka ukazuje pět způsobů, jak vytvořit uživatele. Kliknutím na odkaz zjistíte, jak na to.

| Metoda | Typ | Akce | Podrobnosti | Dostane nový uživatel oznámení? |
|-------------------------|---------------|-------- ------|--------------------|---------------------- -------------------------|

Použití uživatelského rozhraní | UI | přidat a upravit | Tlačítko Admin > Uživatelé | Ne |

Při vytváření případu a přiřazení uživatele k případu | UI | přidat | Tlačítko Správce > Případy > tlačítko Nový > stránka Vytvořit případ | Ano, pokud je povoleno |

Automaticky vytvořit uživatele pomocí veřejného registračního odkazu | Akce uživatele | přidat | Tlačítko Admin > Nastavení > stránka Nastavení systému | Ano, pokud je povoleno automatické vytváření případu |

Nástroj pro import | UI | přidat a upravit | Tlačítko Správce > Integrace > Importy | Ne |

Používání API (již brzy) | API | přidat a upravit | na | na |

Oznámení

Platforma neposílá uživatelům oznámení, když jsou přidáni do aplikace. Správci však mohou povolit upozornění na přiřazení případů.
