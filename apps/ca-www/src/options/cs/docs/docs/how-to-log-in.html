Jak se přihlásit

Přečtěte si, jak se přihlásit ke svému stávajícímu účtu Case Active.

Přihlašování

  1. Přejděte na přihlašovací stránku .
  2. Klikněte na tlačítko PŘIHLÁSIT.
  3. Vložte svůj e-mail.
  4. Klikněte na NEXT.

Ověření účtu

  1. Otevřete novou kartu prohlížeče a přihlaste se ke svému e-mailovému účtu.
  2. Zkopírujte šestimístný kód.
  3. Vraťte se na kartu Case Active.
  4. Vložte kód do polí. Zobrazí se nové okno.

Case Active zobrazí stránku Aplikace.

  • Poznámka Pro uživatele s oprávněními správce se na platformě vedle tlačítka APP zobrazí tlačítko ADMIN.
