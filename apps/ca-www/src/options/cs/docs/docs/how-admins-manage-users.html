Jak administrátoři spravují uživatele

Správci vytvářejí, upravují nebo zakazují, kdo má přístup k aplikacím a případům. Když se uživatel přihlásí k platformě poprvé, musí dokončit pracovní postup profilu.

  • Kliknutím sem získáte další informace o pracovním postupu nového uživatelského profilu.

Správa uživatele

Níže uvedená tabulka ukazuje pět způsobů správy uživatele. Kliknutím na odkaz zjistíte, jak na to.

Metoda Typ Akce Podrobnosti
Pomocí uživatelského rozhraní UI přidat a upravit tlačítko Správce > Uživatelé
Při vytváření případu a přiřazení uživatele k případu UI přidat Tlačítko Správce > Případy > tlačítko Nový > stránka Vytvořit případ
Automaticky vytvořit uživatele pomocí veřejného registračního odkazu Akce uživatele přidat Tlačítko Admin > Nastavení > Stránka Nastavení systému
Nástroj pro import UI přidat a upravit Tlačítko Správce > Integrace > Importy
Používání API (již brzy) API přidat a upravit N/A

Oznámení

Platforma neposílá uživatelům oznámení, když jsou přidáni do aplikace. Správci však mohou povolit upozornění na přiřazení případů.
