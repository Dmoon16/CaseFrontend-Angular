ວິທີການສ້າງຫຼືແກ້ໄຂກໍລະນີຜ່ານເມນູ UI

ບົດຄວາມນີ້ສະແດງໃຫ້ເຫັນ admins ວິທີການສ້າງກໍລະນີທີ່ມີການໂຕ້ຕອບຜູ້ໃຊ້ຂອງເວທີ. ຜູ້ເບິ່ງແຍງຍັງສາມາດເພີ່ມຜູ້ໃຊ້ເຂົ້າໃນກໍລະນີໃຫມ່ແລະປັບປຸງກໍລະນີທີ່ມີຢູ່ແລ້ວ.

ການສ້າງກໍລະນີ

  1. ເຂົ້າສູ່ລະບົບບັນຊີ admin.
  2. ຊອກຫາແອັບຯໃນຫນ້າ Apps ແລະຄລິກໃສ່ປຸ່ມ ADMIN.
  3. ໃນກະດານ Admin, ໃຫ້ຄລິກໃສ່ແຖບເມນູກໍລະນີ.
  4. ໃຫ້ຄລິກໃສ່ປຸ່ມ NEW. ໜ້າຈໍໃໝ່ສະແດງ.

ເພີ່ມຜູ້ໃຊ້

ທາງເລືອກ, ຜູ້ເບິ່ງແຍງສາມາດເພີ່ມຜູ້ໃຊ້ໃສ່ກໍລະນີໃນຫນ້ານີ້. ເລື່ອນລົງແລະຄລິກໃສ່ປຸ່ມ ASSIGN USERS .

img

ໃນ Modal ກໍານົດຜູ້ໃຊ້, ຕື່ມຂໍ້ມູນໃສ່ໃນຊ່ອງຂໍ້ມູນ. ກວດເບິ່ງກ່ອງເພື່ອສົ່ງອີເມວແຈ້ງເຕືອນຜູ້ໃຊ້. (ທາງເລືອກ)
