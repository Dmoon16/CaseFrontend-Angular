ວິທີການເຂົ້າສູ່ລະບົບ

ຮຽນຮູ້ວິທີການເຂົ້າສູ່ລະບົບບັນຊີ Case Active ທີ່ມີຢູ່ຂອງທ່ານ.

ເຂົ້າສູ່ລະບົບ

  1. ໄປ​ທີ່ ​ຫນ້າ​ເຂົ້າ​ສູ່​ລະ​ບົບ .
  2. ຄລິກທີ່ປຸ່ມເຂົ້າສູ່ລະບົບ.
  3. ໃສ່ອີເມວຂອງເຈົ້າ.
  4. ຄລິກຕໍ່ໄປ.

ຢືນຢັນບັນຊີ

  1. ເປີດແຖບຕົວທ່ອງເວັບໃຫມ່ແລະເຂົ້າສູ່ລະບົບບັນຊີອີເມວຂອງທ່ານ.
  2. ສຳເນົາລະຫັດຫົກຕົວເລກ.
  3. ກັບໄປທີ່ແຖບ Case Active.
  4. ວາງລະຫັດໃສ່ໃນຊ່ອງຂໍ້ມູນ. ໜ້າຈໍໃໝ່ສະແດງ.
