ວິທີການສ້າງແລະແກ້ໄຂຜູ້ໃຊ້ດ້ວຍ UI

ບົດ​ຄວາມ​ນີ້​ສະ​ແດງ​ໃຫ້​ເຫັນ admins ວິ​ທີ​ການ​ສ້າງ​ຜູ້​ໃຊ້​ແລະ​ການ​ປັບ​ປຸງ​ການ​ບັນ​ທຶກ​ທີ່​ມີ​ການ​ໂຕ້​ຕອບ​ຜູ້​ໃຊ້​ຂອງ​ເວ​ທີ​. ຜູ້ເບິ່ງແຍງຍັງສາມາດເພີ່ມຜູ້ໃຊ້ເຂົ້າໃນກໍລະນີໃຫມ່ແລະແກ້ໄຂກໍລະນີທີ່ມີຢູ່ແລ້ວ.

ການສ້າງຜູ້ໃຊ້

  1. ເຂົ້າສູ່ລະບົບບັນຊີ admin.
  2. ຊອກຫາແອັບຯໃນຫນ້າ Apps ແລະຄລິກໃສ່ປຸ່ມ ADMIN.
  3. ໃນກະດານ Admin, ໃຫ້ຄລິກໃສ່ແຖບເມນູຜູ້ໃຊ້.
  4. ໃຫ້ຄລິກໃສ່ປຸ່ມ NEW. ໜ້າຈໍໃໝ່ສະແດງ.

ຕື່ມຂໍ້ມູນໃສ່ໃນຊ່ອງຂໍ້ມູນແລະຄລິກບັນທຶກ. Case Active ເພີ່ມຊື່ໃສ່ລາຍຊື່ຜູ້ໃຊ້ທີ່ກ່ຽວຂ້ອງກັບແອັບນີ້.

ຂັ້ນ​ຕອນ​ຕໍ່​ໄປ
