ວິທີການສ້າງກໍລະນີອັດຕະໂນມັດດ້ວຍລິ້ງລົງທະບຽນສາທາລະນະ

ບົດຄວາມນີ້ສະແດງໃຫ້ເຫັນຜູ້ເບິ່ງແຍງວິທີການສ້າງກໍລະນີ (ແລະເພີ່ມຜູ້ໃຊ້ໃນເວລາດຽວກັນ) ໂດຍການສົ່ງລິ້ງລົງທະບຽນດຽວ.

ການຊອກຫາການເຊື່ອມຕໍ່

  1. ເຂົ້າສູ່ລະບົບບັນຊີ admin.
  2. ຊອກຫາແອັບຯໃນຫນ້າ Apps ແລະຄລິກໃສ່ປຸ່ມ ADMIN.
  3. ໃນກະດານ Admin, ໃຫ້ຄລິກໃສ່ແຖບເມນູ Settings.
  4. ໃຫ້ແນ່ໃຈວ່າແຖບເມນູຍ່ອຍຂອງລະບົບຖືກເລືອກ.

ການຕັ້ງຄ່າການຕັ້ງຄ່າ

ເລື່ອນລົງໄປຫາຫົວຂໍ້ USER SIGNUP.

ໃຫ້ແນ່ໃຈວ່າຊ່ອງໃສ່ອະນຸຍາດໃຫ້ລົງທະບຽນຖືກຕັ້ງເປັນ On. (ຕ້ອງການ) ເມື່ອຄຸນສົມບັດນີ້ຖືກເປີດໃຊ້, Case Active ຈະສ້າງລິ້ງທີ່ເປັນເອກະລັກ.

  1. ໃຫ້ຄລິກໃສ່ Auto Create Case ພາກສະຫນາມແລະເລືອກ On. ຊ່ອງສະແດງບົດບາດຜູ້ໃຊ້ກໍລະນີ.
  2. ເລືອກບົດບາດ. (ເບິ່ງຂ້າງລຸ່ມນີ້)

ກໍລະນີບົດບາດຂອງຜູ້ໃຊ້

ມີສາມກໍລະນີຂອງຜູ້ໃຊ້.

  • ລູກຄ້າ (ແນະນໍາ)
  • ຄ່າເລີ່ມຕົ້ນ
  • ພະນັກງານ

ກຳລັງສົ່ງລິ້ງ

ແບ່ງປັນລິ້ງກັບນຶ່ງຄົນ ຫຼືຫຼາຍກວ່ານັ້ນ. ຜູ້ໃຊ້ແຕ່ລະຄົນທີ່ຄລິກໃສ່ການເຊື່ອມຕໍ່ແລະສໍາເລັດການລົງທະບຽນກະຕຸ້ນເວທີເພື່ອສ້າງກໍລະນີໃຫມ່ໃນຫນ້າກໍລະນີແລະອັດຕະໂນມັດມອບຫມາຍຜູ້ໃຊ້ໃນກໍລະນີ.
