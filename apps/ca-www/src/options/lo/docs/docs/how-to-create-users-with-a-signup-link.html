ວິທີການສ້າງຜູ້ໃຊ້ທີ່ມີການເຊື່ອມຕໍ່ລົງທະບຽນ

ບົດຄວາມນີ້ສະແດງໃຫ້ເຫັນຜູ້ເບິ່ງແຍງວິທີການສ້າງຜູ້ໃຊ້ໂດຍການສົ່ງລິ້ງລົງທະບຽນ.

ການຕັ້ງຄ່າການເຊື່ອມຕໍ່ການລົງທະບຽນ

  1. ຄລິກທີ່ ສົ່ງການລົງທະບຽນໃຫ້ເຂົາເຈົ້າ ຢູ່ທາງລຸ່ມຂອງກ່ອງໂຕ້ຕອບເພີ່ມຜູ້ໃຊ້. ໜ້າໃໝ່ສະແດງ.
  2. ຕັ້ງຄ່າຕົວເລືອກການລົງທະບຽນ. (ເບິ່ງຂ້າງລຸ່ມນີ້)

img

ໃຫ້ແນ່ໃຈວ່າພາກສະຫນາມ ອະນຸຍາດໃຫ້ ລົງທະບຽນເພື່ອຕັ້ງເປັນ On. (ຕ້ອງການ) ເມື່ອເປີດໃຊ້ງານ, Case Active ຈະສ້າງລິ້ງທີ່ເປັນເອກະລັກ.

ກວດເບິ່ງທາງເລືອກຂ້າງລຸ່ມນີ້ແລະຫຼັງຈາກນັ້ນໃຫ້ຄລິກໃສ່ໄອຄອນເພື່ອສໍາເນົາການເຊື່ອມຕໍ່. ທ່ານ​ສາ​ມາດ​ວາງ​ການ​ເຊື່ອມ​ຕໍ່​ໃນ​ເວັບ​ໄຊ​ທ​໌​ຂອງ​ທ່ານ​ຫຼື​ສົ່ງ​ການ​ເຊື່ອມ​ຕໍ່​ກັບ​ຫນຶ່ງ​ຫຼື​ຫຼາຍ​ຄົນ​ໂດຍ​ຜ່ານ​ທາງ​ອີ​ເມລ​໌​ຫຼື​ຊ່ອງ​ທາງ​ການ​ສື່​ສານ​.

ການສ້າງກໍລະນີ

ທາງເລືອກອື່ນ, ທ່ານສາມາດສ້າງກໍລະນີອັດຕະໂນມັດສໍາລັບຜູ້ໃຊ້ໃຫມ່ແຕ່ລະຄົນແລະກໍານົດບົດບາດຂອງຜູ້ໃຊ້.

  1. ໃຫ້ຄລິກໃສ່ Auto Create Case ພາກສະຫນາມແລະເລືອກ On. ຊ່ອງສະແດງບົດບາດຜູ້ໃຊ້ກໍລະນີ.
  2. ຄລິກທີ່ Case User Role ແລະເລືອກທາງເລືອກ. (ເບິ່ງຂ້າງລຸ່ມນີ້)

  • ຄ່າເລີ່ມຕົ້ນ
  • ພະນັກງານ

ກຳລັງສົ່ງລິ້ງ

ແບ່ງປັນລິ້ງກັບນຶ່ງຄົນ ຫຼືຫຼາຍກວ່ານັ້ນ. ແຕ່ລະຄົນຄລິກໃສ່ການເຊື່ອມຕໍ່ແລະສໍາເລັດຂັ້ນຕອນການເຮັດວຽກເພື່ອເປີດໃຊ້ບັນຊີ Case Active ຂອງເຂົາເຈົ້າ. ເມື່ອຜູ້ໃຊ້ສໍາເລັດຂັ້ນຕອນການລົງທະບຽນ, Case Active ຈະເພີ່ມຜູ້ໃຊ້ເຂົ້າໃນລາຍຊື່ຜູ້ໃຊ້.

ຂັ້ນ​ຕອນ​ຕໍ່​ໄປ

Case Active ສົ່ງໃຫ້ຜູ້ໃຊ້ແຈ້ງການອີເມລ໌ທີ່ມີລິ້ງລົງທະບຽນ. ຜູ້ໃຊ້ຄລິກໃສ່ການເຊື່ອມຕໍ່ແລະສໍາເລັດຂັ້ນຕອນການເຮັດວຽກສໍາລັບການລົງທະບຽນໃຫມ່.

  • ບົດຄວາມນີ້ ອະທິບາຍຂັ້ນຕອນການເຮັດວຽກການລົງທະບຽນສໍາລັບຜູ້ໃຊ້ໃຫມ່.
