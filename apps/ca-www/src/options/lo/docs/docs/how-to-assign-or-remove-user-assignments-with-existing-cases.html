ວິທີການມອບຫມາຍຫຼືເອົາການມອບຫມາຍຂອງຜູ້ໃຊ້ກັບກໍລະນີທີ່ມີຢູ່ແລ້ວ

ບົດຄວາມນີ້ສະແດງໃຫ້ເຫັນຜູ້ເບິ່ງແຍງວິທີການມອບຫມາຍແລະລຶບຜູ້ໃຊ້ກໍລະນີ. ພວກເຮົາຍັງສະແດງໃຫ້ທ່ານຮູ້ວິທີການດັດແປງຂໍ້ມູນຜູ້ໃຊ້.

ເປີດກໍລະນີທີ່ມີຢູ່ແລ້ວ

  1. ເຂົ້າສູ່ລະບົບບັນຊີ admin.
  2. ຊອກຫາແອັບຯໃນຫນ້າ Apps ແລະຄລິກໃສ່ປຸ່ມ ADMIN.
  3. ຄລິກແຖບເມນູ ກໍລະນີ .
  4. ໃຫ້ແນ່ໃຈວ່າ ACTIVE ຖືກເລືອກຢູ່ໃນເມນູຍ່ອຍ.
  5. ເລື່ອນລົງໄປຫາຖັນ ACTION.
