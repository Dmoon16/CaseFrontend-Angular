Ho Fometa Lintlha tsa Mosebetsi oa Mosebelisi ka Sesebelisoa sa Thepa

Ka tlase ke lethathamo la likarolo tsa data lifaeleng tsa CSV. Khetho e 'ngoe le e' ngoe ea ho kenya e na le mefuta e fapaneng ea likarolo.

case_ID

  • Kenya nyeoe ea Mosebelisi kapa Tag ID.
  • Eya ho Leqephe la Maemo > Edita > Tag ID KAPA Eya leqepheng la Maemo > Edita > Case ID ho url.

Theneketso

  • Kenya ID ea mosebelisi kapa Tag ID.
  • Eya ho Leqephe la Basebedisi > Edita > Tag ID KAPA Eya ho Leqephe la Basebedisi > Edita > ID ya mosebedisi ho url.

case_role_taelo

  • Tlohela sele e se na letho.

case_role_ID

  • Kena: bareki, basebetsi, kapa kamehla.

case_tsebisa

  • 0 = no
  • 1 = E, romella mosebelisi tsebiso ea lengolo-tsoibila.
