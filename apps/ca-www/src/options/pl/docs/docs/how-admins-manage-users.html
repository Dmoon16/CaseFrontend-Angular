Jak administratorzy zarządzają użytkownikami

Administratorzy tworzą, modyfikują lub wyłączają dostęp do aplikacji i spraw. Kiedy użytkownik loguje się do platformy po raz pierwszy, musi wypełnić obieg profilu.

  • Kliknij tutaj, aby dowiedzieć się więcej o przepływie pracy nowego profilu użytkownika.

Zarządzanie użytkownikiem

Poniższa tabela przedstawia pięć sposobów zarządzania użytkownikiem. Kliknij łącze, aby dowiedzieć się, jak to zrobić.

metoda Rodzaj działania Detale
Korzystanie z interfejsu użytkownika interfejs użytkownika dodawać i modyfikować Przycisk administratora > Użytkownicy
Podczas tworzenia sprawy i przypisania użytkownika do sprawy interfejs użytkownika Dodaj Przycisk Administrator > Sprawy > Przycisk Nowy > Strona Utwórz sprawę
Automatycznie utwórz użytkownika za pomocą publicznego linku rejestracyjnego Akcja użytkownika Dodaj Przycisk administratora > Ustawienia > strona Konfiguracja systemu
Narzędzie importu interfejs użytkownika dodawać i modyfikować Przycisk administratora > Integracje > Importy
Korzystanie z API (wkrótce) API dodawać i modyfikować Nie dotyczy

Powiadomienia

Platforma nie wysyła użytkownikom powiadomienia, gdy zostaną dodani do aplikacji. Administratorzy mogą jednak włączyć powiadomienia dotyczące przypisań spraw.
