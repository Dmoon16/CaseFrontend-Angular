Jak tworzyć użytkowników za pomocą linku rejestracyjnego

Ten artykuł pokazuje administratorom, jak tworzyć użytkowników, wysyłając link do rejestracji.

Konfigurowanie linku rejestracyjnego

  1. Kliknij link Wyślij im rejestrację u dołu okna dialogowego Dodaj użytkownika. Zostanie wyświetlona nowa strona.
  2. Skonfiguruj opcje rejestracji. (patrz poniżej)

img

Upewnij się, że pole Zezwalaj na rejestrację jest ustawione na Wł. (wymagane) Po włączeniu Case Active generuje unikalne łącze.

Przejrzyj poniższe opcje, a następnie kliknij ikonę, aby skopiować link. Możesz umieścić link na swojej stronie internetowej lub wysłać link do jednej lub kilku osób za pośrednictwem poczty elektronicznej lub dowolnego kanału komunikacji.

img

Tworzenie sprawy

Opcjonalnie możesz automatycznie utworzyć sprawę dla każdego nowego użytkownika i przypisać mu rolę.

  1. Kliknij pole Automatyczne tworzenie sprawy i wybierz Włączone. Zostaną wyświetlone pola Rola użytkownika sprawy.
  2. Kliknij Rolę użytkownika sprawy i wybierz opcję. (patrz poniżej)

img

Role użytkownika sprawy

Istnieją trzy role użytkownika sprawy.

  • klienci
  • domyślna
  • personel

Wysyłanie linku

Udostępnij link co najmniej jednej osobie. Każda osoba klika link i kończy przepływ pracy, aby aktywować swoje konto Case Active. Gdy użytkownik ukończy proces rejestracji, Case Active doda go do listy Użytkownicy.

img

Następne kroki

Case Active wysyła użytkownikowi powiadomienie e-mail z linkiem do rejestracji. Użytkownik klika link i kończy przepływ pracy dla nowych rejestracji.
