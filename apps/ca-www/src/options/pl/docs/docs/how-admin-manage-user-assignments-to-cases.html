Jak administratorzy zarządzają przypisaniami użytkowników do spraw

Istnieje pięć sposobów zarządzania użytkownikami i przypisaniami spraw. Kliknij łącze, aby dowiedzieć się, jak to zrobić.

metoda Rodzaj działania Detale
Podczas tworzenia sprawy interfejs użytkownika Dodaj Przycisk Administrator > Sprawy > przycisk Nowy > przycisk Przypisz użytkowników
Automatycznie przypisuj użytkownika za pomocą publicznego linku rejestracyjnego Akcja użytkownika Dodaj Przycisk administratora > Ustawienia > strona Konfiguracja systemu
Używanie interfejsu użytkownika w istniejącej sprawie interfejs użytkownika Dodaj usuń Przycisk Administrator > Sprawy > Przypisz użytkowników
Narzędzie importu interfejs użytkownika dodawać, modyfikować i usuwać Przycisk administratora > Integracje > Importy
Korzystanie z interfejsu API (wkrótce) API dodawać, modyfikować i usuwać Nie dotyczy
