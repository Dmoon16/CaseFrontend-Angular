Formatowanie danych przypisań przypadków użytkownika za pomocą narzędzia do importowania

Poniżej znajduje się lista pól danych w plikach CSV. Każda opcja importu ma inną kombinację pól.

ID sprawy

  • Wprowadź identyfikator użytkownika lub znacznika sprawy.
  • Przejdź do strony Sprawy > Edytuj > Identyfikator tagu LUB Przejdź do strony Sprawy > Edytuj > Identyfikator sprawy w adresie URL.

identyfikator użytkownika

  • Wprowadź identyfikator użytkownika lub identyfikator znacznika.
  • Przejdź do strony Użytkownicy > Edytuj > Identyfikator tagu LUB Przejdź do strony Użytkownicy > Edytuj > Identyfikator użytkownika w adresie URL.

sprawa_rola_kolejność

  • Pozostaw komórkę pustą.

identyfikator_roli_sprawy

  • Wpisz: klienci, personel lub wartość domyślna.

sprawa_notify

  • 0 = nie
  • 1 = Tak, wyślij użytkownikowi powiadomienie e-mail.
