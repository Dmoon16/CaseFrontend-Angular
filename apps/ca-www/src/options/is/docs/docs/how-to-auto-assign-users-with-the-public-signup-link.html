Hvernig á að úthluta notendum sjálfkrafa með opinberum skráningartengli

Stjórnendur læra hvernig á að stilla kerfisstillingar þannig að nýjum notendum sé sjálfkrafa úthlutað nýlega búið til mál.

Að opna uppsetningarsíðuna

Fyrsta skrefið er að opna Systems Setup síðuna.

  1. Skráðu þig inn á admin reikninginn.
  2. Finndu app á Apps síðunni og smelltu á ADMIN hnappinn.
  3. Smelltu á Systems flipann og síðan SYSTEM SETUP á undirvalmyndinni.
  4. Skrunaðu niður að fyrirsögninni NOTANDA SKRÁNING.

Stilla kerfisstillingar

  1. Stilltu fellivalmyndirnar Leyfa skráningu og sjálfkrafa búa til mál á Kveikt.
  2. Smelltu á hlutverkavalmyndina og veldu valkost (sjálfgefið er mælt með því).
  3. Afritaðu og deildu hlekknum til að skrá opinber notanda.
