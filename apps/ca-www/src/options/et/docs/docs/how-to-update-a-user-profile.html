Kuidas värskendada kasutajaprofiili

See artikkel näitab kasutajatele, kuidas oma profiililehel teavet värskendada.

  • Kasutajad saavad ise oma profiile värskendada.
  • Administraatorid saavad värskendada iga kasutaja profiililehte.

Kasutajatele

Esimene samm on profiililehe avamine. Kasutajad ja administraatorid järgivad oma profiililehe värskendamiseks neid samme.

  1. Logige sisse oma Case Active kontole.
  2. Klõpsake lehe ülaosas vahekaarti Profiil.
  3. Muudatusi tegema.
  4. Klõpsake nuppu Salvesta.

Enamik välju on valikulised ja mõned väljad on kohustuslikud. Nõutavad väljad on:

  • Eesnimi
  • Perekonnanimi
  • E-posti aadress
  • Keel
  • Ajavöönd

Administraatoritele

Administraatorid saavad ka teiste kasutajate profiiliteavet värskendada. See sisaldab kolme välja pealkirja HOST KASUTAJA INFO all.

  1. Logige sisse oma Case Active'i administraatorikontole.
  2. Otsige üles rakendus ja klõpsake nuppu ADMIN.
  3. Klõpsake ülemises menüüs nuppu Kasutajad.
  4. Minge alammenüüsse ja valige kasutajaloend, millele soovite juurde pääseda (ACTIVE või DISBLED). Vaikesäte on Aktiivne.
  5. Leidke loendist kasutaja ja klõpsake nuppu Redigeeri.
  6. Tehke muudatused vastavalt vajadusele. (vt allpool)
  7. Klõpsake nuppu Salvesta.

img

Muutke kasutajateavet

Administraatorid saavad kasutajateavet muuta.

  1. Määrake kasutaja tüüp (kasutaja või haldur).
  2. Määrake kasutaja olek (Aktiivne kuni Keelatud).
  3. Sisestage kordumatu sildi ID. (valikuline)
  4. Klõpsake nuppu Salvesta.
