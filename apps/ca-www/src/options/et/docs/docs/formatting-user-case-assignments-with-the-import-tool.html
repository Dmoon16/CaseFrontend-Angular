Kasutajajuhtumite määramise andmete vormindamine imporditööriistaga

Allpool on CSV-failide andmeväljade loend. Igal impordivalikul on erinev väljade kombinatsioon.

case_ID

  • Sisestage suurjuht kasutaja või sildi ID.
  • Avage leht Juhtumid > Muuda > Sildi ID VÕI Avage leht Juhtumid > Muuda > Juhtumi ID URL-is.

kasutaja ID

  • Sisestage kasutaja ID või sildi ID.
  • Avage leht Kasutajad > Muuda > Sildi ID VÕI Avage leht Kasutajad > Muuda > Kasutaja ID URL-is.

juhtumi_rolli_järjestus

  • Jätke lahter tühjaks.

juhtumi_rolli_ID

  • Sisestage: kliendid, töötajad või vaikimisi.

case_notify

  • 0 = ei
  • 1 = Jah, saada kasutajale meiliteade.
