Kuidas administraatorid haldavad juhtumitele kasutajate määramisi

Kasutajate ja juhtumiülesannete haldamiseks on viis võimalust. Kuidas õppida, klõpsake lingil.

meetod Tüüp Tegevused Üksikasjad
Juhtumi loomisel UI lisama Administraatori nupp > Juhtumid > Uus nupp > Nupp Kasutajate määramine
Määra kasutaja automaatselt avaliku registreerumislingi abil Kasutaja tegevus lisama Administraatori nupp > Sätted > Süsteemi seadistuse leht
UI kasutamine olemasoleval juhul UI lisa ja eemalda Administraatori nupp > Juhtumid > Kasutajate määramine
Impordi tööriist UI lisada, muuta ja eemaldada Administraatori nupp > Integratsioonid > Impordid
