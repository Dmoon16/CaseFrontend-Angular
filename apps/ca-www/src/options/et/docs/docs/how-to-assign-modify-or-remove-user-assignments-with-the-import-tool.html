Kuidas määrata, muuta või eemaldada imporditööriista abil kasutajamääranguid

See artikkel näitab administraatoritele, kuidas hallata juhtumi kasutajaid imporditööriistaga.

Administraatorid saavad:

  • määrata juhtumile kasutajad
  • muuta juhtumile juba määratud kasutajaid
  • juhtumile määratud kasutajate eemaldamine

Importimise tööriista avamine

  1. Logige sisse administraatori kontole.
  2. Leidke rakenduste lehelt rakendus ja klõpsake nuppu ADMIN.
  3. Klõpsake vahekaarti Integratsioon ja seejärel alammenüüs IMPORTID.
  4. Klõpsake nuppu UUS. Kuvatakse dialoogiboks Import Imports.

Kirjete loomine

Laadige alla üks või mitu CSV-faili. Kasutage neid faile juhtumi kasutajate lisamiseks, muutmiseks või eemaldamiseks. Pärast dialoogiboksi Import Imports avanemist:

  1. Klõpsake väljal Tüüp.
  2. Kerige rippmenüüd ja tehke valik (vt allpool). Kuvatakse link.
  3. Klõpsake linki Laadi näidis alla. CSV-fail laaditakse teie seadmesse alla.

Avage fail ja lisage kirjed. Klõpsake siin, et saada teavet selle kohta, kuidas kasutada CSV-failis vorminguandmeid.

Faili valikud

Juhtudele kasutajate määramine * Valige see suvand juhtumile kasutajate lisamiseks. * Jätke lahter case_role_order tühjaks. * Minge lehele Kasutajad ja kopeerige selle inimese sildi ID. Kui väli Sildi ID on tühi, sisestage kordumatu ID, klõpsake nuppu Salvesta ja seejärel kopeerige ID CSV-faili.

Juhtumitele määratud kasutajate muutmine * Valige see suvand juhtumile määratud praeguste kasutajate andmete värskendamiseks. * Kasutage juhtumi_ID veeru jaoks Case Tag ID.

img

Eemalda juhtumitele määratud kasutajad * Valige see suvand juhtumile määratud kasutajate eemaldamiseks.

Faili üleslaadimine

Saate faili kohe üles laadida või hiljem naasta dialoogiboksi Importimise lisamine ja need sammud lõpule viia.

  1. Avage dialoogiboks Impordi lisamine.
  2. Sisestage CSV-faili kordumatu nimi.
  3. Faili seadmest üleslaadimiseks klõpsake plussmärgi (+) ikoonil.
  4. Klõpsake väljal Tüüp ja tehke valik. (vt eespool)
  5. Klõpsake SAVE.

Töötlemine

Sõltuvalt failis olevate kirjete arvust ja süsteemi laadimisest võib platvormil impordi töötlemiseks kuluda mitu minutit.

img

Kui importimine on lõppenud, kohandab platvorm lehel Juhtumid kasutajakirjeid (st lisab, muudab või eemaldab).

Ebaõnnestumine

Kui importimine ebaõnnestub, tagastab platvorm Failed teate. Enamasti on rikke põhjuseks andmete vormindamine.

Rikke põhjuse nägemiseks tehke järgmist.

  1. Klõpsake jaotise Toiming all nuppu Vaade.
  2. Järgmisel lehel hõljutage kursorit ebaõnnestunud ikoonil. Tööriistaspikker kuvab vea.
  3. Minge CVS-faili ning korrigeerige ja parandage viga.
  4. Laadige fail uuesti üles.
