Juhtumite automaatne loomine avaliku registreerumislingiga

See artikkel näitab administraatoritele, kuidas luua juhtumit (ja lisada samal ajal kasutaja), saates ühe registreerumislingi.

Lingi leidmine

  1. Logige sisse administraatori kontole.
  2. Leidke rakenduste lehelt rakendus ja klõpsake nuppu ADMIN.
  3. Klõpsake administraatoripaneelil vahekaarti Sätted.
  4. Veenduge, et valitud on alammenüü vahekaart SÜSTEEMI SEADISTUS.

Seadete konfigureerimine

Kerige alla pealkirja USER SIGNUP juurde.

Veenduge, et välja Luba registreerumine on seatud väärtusele Sees. (nõutav) Kui see funktsioon on lubatud, loob Case Active ainulaadse lingi.

  1. Klõpsake väljal Automaatne loo juhtum ja valige Sees. Kuvatakse väli Case User Role.
  2. Valige roll. (vt allpool)

Juhtumi kasutaja rollid

Juhtumikasutaja rolle on kolm.

  • kliendid (soovitatav)
  • vaikimisi
  • töötajad

Lingi saatmine

Jagage linki ühe või mitme inimesega. Iga kasutaja, kes klõpsab lingil ja lõpetab registreerumise, käivitab platvormi, et luua juhtumite lehel uus juhtum ja määrata juhtumis kasutaja automaatselt.
