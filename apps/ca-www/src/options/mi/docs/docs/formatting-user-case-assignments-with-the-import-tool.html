Whakahōputu Raraunga Taumahi Take Kaiwhakamahi me te Utauta Kawemai

Kei raro ko te rarangi o nga mara raraunga kei roto i nga konae CSV. He rereke nga huinga o nga mara kei ia whiringa kawemai.

take_ID

  • Whakauruhia te take Kaiwhakamahi, ID Tag ranei.
  • Haere ki te wharangi Take > Whakatika > Tag ID RĀNEI Haere ki te whārangi Take > Whakatika > Take ID kei te url.

ID_kaiwhakamahi

  • Whakauruhia te ID o te kaiwhakamahi, ID Tag ranei.
  • Haere ki te wharangi Kaiwhakamahi> Whakatika> ID Tag OR Haere ki te wharangi Kaiwhakamahi> Whakatika> ID Kaiwhakamahi i te url.

take_role_order

  • Waiho pūtau kia pātea.

case_role_ID

  • Whakauruhia: kiritaki, kaimahi, taunoa ranei.

case_notify

  • 0 = kahore
  • 1 = Ae, tukuna he panui imeera ki te kaiwhakamahi.
