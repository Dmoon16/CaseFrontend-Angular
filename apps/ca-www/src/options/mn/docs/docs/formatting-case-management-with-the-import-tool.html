Импортын хэрэгслээр кейс менежментийн өгөгдлийг форматлах

Импортын хэрэгслээр хэргийг удирдахын тулд CSV файл дахь шаардлагатай талбаруудын жагсаалтыг доор харуулав.

хэрэглэгчийн нэр

  • Хэрэглэгчийн имэйл хаягийг оруулна уу.

tag_id (та дараа лавлаж болох id)

  • өвөрмөц танигч (үсэг, тоон тэмдэгт ашиглах, зураасыг зөвшөөрнө)

case_status

  • 0 = хаалттай
  • 1 = идэвхтэй

нээсэн_огноо

  • сар өдөр жил
  • мм/дд/жж

тухай

  • хэргийн өвөрмөц тайлбар (үсэг, тоон тэмдэгт ашиглах, зураасыг зөвшөөрнө)
