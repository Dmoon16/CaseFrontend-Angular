Бүртгэлийн холбоосоор хэрэглэгчдийг хэрхэн үүсгэх вэ

Энэ нийтлэл нь админуудад бүртгүүлэх холбоосыг илгээж хэрэглэгчдийг хэрхэн үүсгэхийг харуулж байна.

Бүртгэлийн холбоосыг тохируулж байна

  1. Хэрэглэгч нэмэх харилцах цонхны доод талд байгаа " Тэдэнд бүртгүүлэх илгээх" холбоос дээр дарна уу. Шинэ хуудас гарч ирнэ.
  2. Бүртгэлийн сонголтуудыг тохируулна уу. (доороос үзнэ үү)

img

Бүртгүүлэхийг зөвшөөрөх талбарыг асаалттай болгож байгаа эсэхийг шалгана уу. (шаардлагатай) Идэвхжүүлсэн үед Case Active өвөрмөц холбоос үүсгэдэг.

Доорх сонголтуудыг хянаж үзээд холбоосыг хуулахын тулд дүрс дээр дарна уу. Та линкийг өөрийн вэбсайт дээр байрлуулж эсвэл холбоосыг нэг буюу хэд хэдэн хүнд имэйлээр эсвэл ямар ч харилцааны сувгаар илгээж болно.

Кейс үүсгэх

Сонголтоор та шинэ хэрэглэгч бүрт автоматаар кейс үүсгэж, хэрэглэгчийн үүрэг оноож болно.

  1. Auto Create Case талбарыг товшоод On-г сонгоно уу. Case User Role талбарууд гарч ирнэ.
  2. Case User Role дээр товшоод сонголтоо сонгоно уу. (доороос үзнэ үү)

Холбоосыг илгээж байна

Холбоосыг нэг буюу хэд хэдэн хүнтэй хуваалцаарай. Хүн бүр холбоос дээр дарж Case Active бүртгэлээ идэвхжүүлэхийн тулд ажлын явцыг дуусгана. Хэрэглэгч бүртгүүлэх ажлын явцыг дуусгасны дараа Case Active нь хэрэглэгчийг Хэрэглэгчдийн жагсаалтад нэмнэ.

Дараагийн алхмууд

Case Active нь хэрэглэгчдэд бүртгүүлэх холбоос бүхий имэйл мэдэгдэл илгээдэг. Хэрэглэгч холбоос дээр дарж, шинэ бүртгүүлэх ажлын явцыг дуусгана.

  • Энэ нийтлэлд шинэ хэрэглэгчдэд бүртгүүлэх ажлын явцыг тайлбарласан болно.
