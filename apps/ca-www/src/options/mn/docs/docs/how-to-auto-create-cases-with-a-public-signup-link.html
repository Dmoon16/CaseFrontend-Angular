Нийтийн бүртгэлийн холбоосоор хэрхэн автоматаар хэргийг үүсгэх вэ

Энэ нийтлэл нь админуудад нэг бүртгүүлэх холбоосыг илгээх замаар хэрхэн кейс үүсгэхийг (мөн нэгэн зэрэг хэрэглэгч нэмэх) харуулж байна.

Холбоосыг хайж байна

  1. Админ данс руу нэвтэрнэ үү.
  2. Програмын хуудаснаас програмаа олоод АДМИН товчийг дарна уу.
  3. Админ самбар дээрх Тохиргоо цэсийн таб дээр дарна уу.
  4. СИСТЕМ ТОХИРУУЛАЛТ дэд цэсийн таб сонгогдсон эсэхийг шалгана уу.

Тохиргоог тохируулж байна

ХЭРЭГЛЭГЧИЙН БҮРТГЭЛИЙН гарчиг руу доош гүйлгэнэ үү.

Бүртгүүлэхийг зөвшөөрөх талбарыг асаалттай гэж тохируулсан эсэхийг шалгана уу. (шаардлагатай) Энэ функц идэвхжсэн үед Case Active өвөрмөц холбоос үүсгэдэг.

  1. Auto Create Case талбарыг товшоод On-г сонгоно уу. Case User Role талбар гарч ирнэ.
  2. Дүр сонгох. (доороос үзнэ үү)

Кейс хэрэглэгчийн үүрэг

Гурван тохиолдол хэрэглэгчийн үүрэг байдаг.

  • үйлчлүүлэгчид (санал болгосон)
  • анхдагч
  • ажилтнууд

Холбоосыг илгээж байна

Холбоосыг нэг буюу хэд хэдэн хүнтэй хуваалцаарай. Холбоос дээр дарж, бүртгэлээ дуусгасан хэрэглэгч бүр хэргийн хуудсан дээр шинэ хэрэг үүсгэж, тухайн хэрэглэгчийг автоматаар оноохын тулд платформыг идэвхжүүлдэг.
