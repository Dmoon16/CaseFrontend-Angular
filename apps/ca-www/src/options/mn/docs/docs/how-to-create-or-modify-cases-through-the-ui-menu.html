UI цэсээр дамжуулан хэргийг хэрхэн үүсгэх эсвэл өөрчлөх вэ

Энэ нийтлэл нь админуудад платформын хэрэглэгчийн интерфэйсээр хэрхэн кейс үүсгэхийг харуулж байна. Админууд мөн хэрэглэгчдийг шинэ тохиолдолд нэмж, одоо байгаа тохиолдлуудыг шинэчлэх боломжтой.

Кейс үүсгэх

  1. Админ данс руу нэвтэрнэ үү.
  2. Програмын хуудаснаас програмаа олоод ADMIN товчийг дарна уу.
  3. Админ самбар дээрх Cases цэсийн таб дээр дарна уу.
  4. ШИНЭ товчийг дарна уу. Шинэ цонх гарч ирнэ.

img

Шаардлагатай талбаруудыг бөглөж, SAVE товчийг дарна уу.

Хэрэглэгч нэмж байна

Сонголтоор админууд энэ хуудсан дээрх кейст хэрэглэгч нэмж болно. Доош гүйлгээд, ХЭРЭГЛЭГЧДИЙГ ОЛГОХ товчийг дарна уу.

img

Assign User горимд талбаруудыг бөглөнө үү. Хэрэглэгчид имэйл мэдэгдэл илгээхийн тулд хайрцгийг чагтална уу. (заавал биш)

Кейсийг өөрчилж байна

Цөөн тооны тохиолдлыг шинэчлэхийн тулд Засварлах хэрэгслийг ашиглан тохиолдол бүрт өөрчлөлт оруулна уу.

  • Тайлбар : Нэг удаагийн олон тооны хэргийг шинэчлэхийн тулд Импортын хэрэгслийг ашиглана уу
  1. Админ данс руу нэвтэрнэ үү.
  2. Програмын хуудаснаас програмаа олоод ADMIN товчийг дарна уу.
  3. Админ самбар дээрх Cases цэсийн таб дээр дарна уу.
  4. Кейс олоод Засах гэснийг товшино уу.
  5. Өөрчлөлтүүдийг хийгээд SAVE дээр дарна уу.
