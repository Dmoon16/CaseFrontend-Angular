UI ашиглан хэрэглэгчдийг хэрхэн үүсгэх, өөрчлөх вэ

Энэ нийтлэл нь админуудад платформын хэрэглэгчийн интерфэйсээр хэрэглэгчдийг хэрхэн үүсгэх, бүртгэлийг шинэчлэх талаар харуулж байна. Админууд мөн хэрэглэгчдийг шинэ тохиолдолд нэмж, одоо байгаа тохиолдлуудыг засах боломжтой.

Хэрэглэгч үүсгэх

  1. Админ данс руу нэвтэрнэ үү.
  2. Програмын хуудаснаас програмаа олоод ADMIN товчийг дарна уу.
  3. Админ самбар дээрх Хэрэглэгчид цэсийн таб дээр дарна уу.
  4. ШИНЭ товчийг дарна уу. Шинэ цонх гарч ирнэ.

Талбаруудыг бөглөж, SAVE дээр дарна уу. Case Active нь энэ програмтай холбоотой хэрэглэгчдийн жагсаалтад нэрийг нэмдэг.

Хэрэглэгчийг өөрчилж байна

Цөөн тооны хэрэглэгчийн бүртгэлийг шинэчлэхийн тулд Засварлах хэрэгслийг ашиглан хэрэглэгч бүрт өөрчлөлт оруулна уу.

  • Тэмдэглэл Нэг суултаар олон тооны хэрэглэгчийн бүртгэлийг өөрчлөхийн тулд Импортын хэрэгслийг ашиглана уу.
  1. Админ самбар дээрх Хэрэглэгчид цэсийн таб дээр дарна уу.
  2. Хэрэглэгчийг олоод "Засварлах" дээр дарна уу.
  3. Өөрчлөлтүүдийг хийгээд SAVE дээр дарна уу.

Дараагийн алхмууд

Case Active нь хэрэглэгчдэд имэйл мэдэгдэл илгээдэггүй. Админууд хэрэглэгчтэй холбогдож Case Active нүүр хуудас руу линк илгээнэ үү. Дараа нь хэрэглэгч НЭВТРЭХ товчийг дарж, шинээр бүртгүүлэх ажлын явцыг дуусгана.

  • Энэ нийтлэлд шинэ хэрэглэгчдэд бүртгүүлэх ажлын явцыг тайлбарласан болно.
