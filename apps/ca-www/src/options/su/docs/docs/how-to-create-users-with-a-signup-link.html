Kumaha Jieun Pamaké sareng Link Signup

Tulisan ieu nunjukkeun ka admin kumaha cara nyiptakeun pangguna ku ngirim tautan pendaptaran.

Ngonpigurasikeun link signup

  1. Klik Kirim aranjeunna a signup link di handap kotak dialog Tambah pamaké. A mintonkeun kaca anyar.
  2. Konpigurasikeun pilihan signup. (tingali kahandap)

img

Pastikeun widang Ngidinan Signup disetel ka Hurung. (diperlukeun) Nalika diaktipkeun, Case Active ngahasilkeun tumbu unik.

Pariksa pilihan di handap teras klik ikon pikeun nyalin tautan. Anjeun tiasa nempatkeun tautan dina halaman wéb anjeun atanapi ngirim tautan ka hiji atanapi langkung jalma ngalangkungan email atanapi saluran komunikasi naon waé.

img

Nyiptakeun pasualan

Opsional, anjeun tiasa sacara otomatis nyiptakeun kasus pikeun unggal pangguna énggal sareng napelkeun peran pangguna.

  1. Klik widang Otomatis Jieun Case tur pilih Hurung. Widang Peran Pamaké Kasus dipintonkeun.
  2. Klik Peran Pamaké Kasus tur pilih hiji pilihan. (tingali kahandap)

img

Kasus kalungguhan pamaké

Aya tilu kalungguhan pamaké kasus.

  • klien
  • standar
  • patugas

Ngirimkeun tautan

Bagikeun tautan sareng hiji atanapi langkung jalma. Unggal jalma ngaklik tautan sareng ngalengkepan alur kerja pikeun ngaktipkeun akun Case Active na. Sakali pamaké ngalengkepan workflow signup, Case Active nambahkeun pamaké kana daptar pamaké.

img

Léngkah saterusna

Case Active ngirimkeun bewara email ka pangguna kalayan tautan pendaptaran. Pamaké ngaklik tautan sareng ngalengkepan alur kerja pikeun pendaptaran énggal.

  • Tulisan ieu ngajelaskeun alur kerja pendaptaran pikeun pangguna énggal.
