Ngaformat Data Manajemén Kasus nganggo Alat Impor

Di handap ieu daptar widang anu diperlukeun dina file CSV pikeun ngatur kasus kalawan alat Impor.

ngaran pamaké

  • Lebetkeun alamat surélék pamaké.

tag_id (id anjeun tiasa ngarujuk engké)

  • identifier unik (make karakter alfanumerik, hyphens ditarima)

kasus_status

  • 0 = ditutup
  • 1 = aktip

date_open

  • bulan/dinten/taun
  • mm/dd/yyyy

ngeunaan

  • pedaran unik ngeunaan kasus (nganggo karakter alfanumerik, hyphens ditarima)
