Ngaformat Data Patugas Kasus Pamaké nganggo Alat Impor

Di handap ieu daptar widang data dina file CSV. Unggal pilihan impor boga kombinasi béda tina widang.

kasus_ID

  • Lebetkeun kasus Pamaké atanapi Tag ID.
  • Pindah ka halaman Kasus> Édit> ID Tag ATAWA Buka halaman Kasus> Édit> ID Kasus dina url.

ID Pangguna

  • Lebetkeun ID pangguna atanapi ID Tag.
  • Pindah ka halaman Pamaké> Édit> ID Tag ATAWA Pindah ka halaman Pamaké> Édit> ID Pamaké dina url.

case_role_order

  • Ninggalkeun sél kosong.

case_role_ID

  • Lebetkeun: klien, staf, atanapi standar.

case_notify

  • 0 = henteu
  • 1 = Leres, kirimkeun bewara email ka pangguna.
