Kumaha Nyiptakeun atanapi Ngarobih Kasus nganggo Alat Impor

Tulisan ieu nunjukkeun ka admin kumaha cara nyiptakeun atanapi ngarobih kasus anu tos aya nganggo alat Impor.

Buka alat Impor

  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Dina panel Admin, klik tab Integrasi teras IMPOR dina submenu.
  4. Klik tombol NEW. Kotak dialog Tambah Impor dipintonkeun dina tab browser anyar.

img

Nyiptakeun kasus

Saatos kotak dialog Tambah Impor dibuka:

  1. Klik widang Tipe tur pilih hiji pilihan (tingali di handap). Hiji link mintonkeun.
  2. Pencét tautan Unduh Sampel. A file CSV diundeur ka alat Anjeun.

Teras, buka file sareng tambahkeun kasus.

Tipe

Nalika anjeun ngaklik médan Tipe , ménu turun-handap dipintonkeun. Pilih salah sahiji tina dua pilihan pikeun impor kasus bulk.

  • Catetan Menu turun-handap ngagaduhan seueur pilihan, tapi ngan ukur dua anu aya hubunganana sareng ngatur kasus. Unggal pilihan gaduh sampel file CSV anu béda.

Tambahkeun kasus

  • Milih pilihan ieu pikeun unggah bulk basajan tina kasus.
  • Klik tautan Unduh Sampel pikeun muka file CSV sareng judul kolom pikeun jinis impor Jieun kasus.

Ngaropea kasus

  • Pilih pilihan ieu pikeun ngapdet kasus ayeuna.
  • Klik tautan Unduh Sampel pikeun muka file CSV kalayan judul kolom pikeun jinis impor Modify case.

Ngunggah file

Anjeun tiasa unggah file langsung atanapi uih deui ka kotak dialog Tambah Impor dina waktos engké sareng ngalengkepan léngkah-léngkah ieu.

  1. Buka kotak dialog Tambah Impor.
  2. Lebetkeun nami unik pikeun file CSV.
  3. Pencét ikon tambah (+) pikeun unggah file tina alat anjeun.
  4. Klik widang Tipe tur pilih hiji pilihan. (tingali luhureun)
  5. Pencét SAVE.

pormat

Ieu mangrupikeun widang anu diperyogikeun sareng sarat munggaran. Klik di dieu pikeun marios rinci pormat.

tag_id (atawa case_ID) identifier unik (make karakter alfanumerik, hyphens ditarima)

case_status 0 = ditutup 1 = aktip

tanggal_dibuka bulan/dinten/taun mm/dd/yyyy

ngeunaan pedaran unik ngeunaan kasus (nganggo karakter alfanumerik, hyphens ditarima)

Ngolah

Gumantung kana jumlah rékaman dina file sareng beban sistem, platform tiasa nyandak sababaraha menit pikeun ngolah impor.

img

Nalika impor réngsé, platform nyaluyukeun rékaman kasus (nyaéta nambihan atanapi ngarobih) dina dasbor Kasus admin.

Kagagalan

Lamun operasi impor gagal, platform nu mulih pesen Gagal. Dina kalolobaan kasus, anu ngabalukarkeun gagalna nyaéta pormat data.

Pikeun ningali sabab gagalna:

  1. Klik View handapeun judul ACTION.
  2. Dina kaca salajengna, hover beurit dina ikon Gagal. tooltip mintonkeun kasalahan.
  3. Pindah ka file CVS sareng saluyukeun sareng ngalereskeun kasalahan.
  4. Unggah deui filena.
