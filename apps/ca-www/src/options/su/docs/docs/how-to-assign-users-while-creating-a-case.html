Kumaha Napelkeun Pamaké Nalika Nyiptakeun Kasus

Tulisan ieu nunjukkeun ka admin kumaha napelkeun pangguna nalika nyiptakeun kasus énggal. Upami pangguna henteu acan kadaptar dina platform, anjeun tiasa nyiptakeun tambihan pangguna énggal nalika nyiptakeun kasus énggal.

Nyiptakeun pasualan

  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Klik tab menu Kasus.
  4. Klik tombol NEW. Jandéla anyar némbongan.

  • Eusian widang pikeun nyieun kasus teras gulung ka handap ka judul Pamaké Ditugaskeun.

Napelkeun pamaké

pamaké ayeuna

Turutan léngkah-léngkah ieu pikeun napelkeun pangguna ayeuna kana kasus énggal.

Pencét tombol ASSIGN USERS sareng eusian widang.

  1. Klik leupaskeun-handap Pamaké tur pilih hiji identifier (misalna surelek).
  2. Asupkeun nilai dina widang kosong (misalna alamat surélék).
  3. Upami catetan pangguna aya dina platform, médan otomatis nunjukkeun nami.
  4. Pilih pangguna anu leres.
  5. Pilih peran pamaké (klien, standar, atawa staf).
  6. Opsional, centang kotak pikeun ngirim bewara email ka pangguna.
  7. Pencét SAVE.

Case Active nambihan pangguna kana kasus éta. Ayeuna admin tiasa ngaklik Simpen pikeun nutup jandela, atanapi nambihan pangguna énggal, atanapi ngahapus pangguna.

Nambahkeun pamaké anyar

Lamun platform nu teu bisa manggihan rékaman ayeuna, klik tombol pikeun nambahkeun pamaké anyar.

Eusian widang teras klik SAVE pikeun nambahkeun pamaké kana platform nu. Salajengna, atur peran pangguna teras klik SAVE deui pikeun napelkeun pangguna kana kasus éta. Tungtungna, klik SAVE sakali deui pikeun nyimpen kasus.
