Kumaha Otomatis Nyiptakeun Kasus nganggo Tautan Pendaptaran Umum

Tulisan ieu nunjukkeun ka admin kumaha cara ngadamel kasus (sareng nambihan pangguna dina waktos anu sami) ku ngirim tautan pendaptaran tunggal.

Locating link

  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Dina panel Admin, klik tab menu Setélan.
  4. Pastikeun tab submenu SYSTEM SETUP dipilih.

img

Ngonpigurasikeun setélan

Gulung ka handap ka judul USER SIGNUP.

Pastikeun widang Ngidinan Signup disetel ka Hurung. (diperlukeun) Nalika fitur ieu diaktipkeun, Case Active ngahasilkeun tumbu unik.

  1. Klik widang Otomatis Jieun Case tur pilih Hurung. Widang Peran Pamaké Kasus dipintonkeun.
  2. Pilih peran. (tingali kahandap)

Kasus kalungguhan pamaké

Aya tilu kalungguhan pamaké kasus.

  • klien (disarankeun)
  • standar
  • patugas

Ngirimkeun tautan

Bagikeun tautan sareng hiji atanapi langkung jalma. Masing-masing pangguna anu ngaklik tautan sareng ngalengkepan pendaptaran micu platform pikeun ngahasilkeun kasus anyar dina halaman kasus sareng otomatis ngadaptarkeun pangguna dina kasus éta.
