Di handap ieu daptar widang data dina file CSV. Unggal pilihan impor boga kombinasi béda tina widang.

ngaran pamaké

  • Lebetkeun alamat surélék pamaké.

nami anu dipasihkeun

  • Lebetkeun ngaran hareup pamaké.

nami kulawarga

  • Lebetkeun ngaran tukang pamaké.

lokal

  • Kolom ieu mariksa basa anu dipikaresep ku pangguna.
  • Lebetkeun en pikeun Inggris, sp pikeun Spanyol.

info zona

  • Ieu netepkeun zona waktos pangguna. Lebetkeun buana sareng kota anu henteu aya spasi (nganggo garis handap).
  • Kedah nganggo pilihan dina menu turun-handap Profil > BAHASA & WILAYAH > Timezone.
  • Anggo zona waktos anu paling pas. Contona, Boston henteu dina turun-handap, ku kituna urang tiasa nganggo New York (atanapi kota mana waé di zona waktos EST / EDT).
