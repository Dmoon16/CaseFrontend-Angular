Kumaha Nunjukkeun atanapi Ngahapus Tugas Pamaké sareng Kasus anu Aya

Tulisan ieu nunjukkeun ka admin cara napelkeun sareng ngahapus pangguna kasus. Kami ogé nunjukkeun anjeun kumaha ngarobih data pangguna.

Muka pasualan nu aya

  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Klik tab menu Kasus .
  4. Pastikeun AKTIF dipilih dina submenu.
  5. Gulung ka handap ka kolom ACTION.

Napelkeun atawa cabut

Klik Napelkeun Pamaké pikeun napelkeun pamaké kana pasualan atawa cabut pamaké tina pasualan éta (tapi sanés platformna). Dina kaca salajengna: Klik ADD tur ngalengkepan workflow nu. Pencét hapus pikeun ngahapus pangguna.

img

Ngarobah

Henteu mungkin pikeun ngédit atanapi ngapdet data pangguna dina halaman Kasus. Gantina, tuturkeun léngkah ieu:

  1. Klik tab menu Pamaké .
  2. Panggihan ngaran pamaké teras klik Édit .
  3. Jieun parobahan teras klik SAVE.
