Kumaha Otomatis Napelkeun Pamaké sareng Patalina Pendaptaran Umum

Admins diajar kumaha ngonpigurasikeun setélan sistem supados pangguna énggal ditugaskeun sacara otomatis kana kasus anu nembé diciptakeun sacara otomatis.

Muka kaca setelan

Léngkah munggaran nyaéta muka halaman Setup Sistem.

  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Klik tab Systems lajeng SYSTEM SETUP dina submenu.
  4. Gulung ka handap ka judul USER SIGNUP.

Ngonpigurasikeun setélan sistem

  1. Setel menu turun-handap Ngidinan Signup sareng Jieun Kasus ka Hurung.
  2. Klik menu peran tur pilih hiji pilihan (standar dianjurkeun).
  3. Salin sareng bagikeun tautan Pendaptaran Pamaké Umum.

Ngirimkeun tautan

Unggal pangguna anu ngaklik tautan sareng ngalengkepan pendaptaran micu platform pikeun ngahasilkeun kasus anyar dina halaman Kasus sareng otomatis napelkeun pangguna dina kasus éta.
