Kumaha Nyiptakeun atanapi Ngarobih Kasus Ngaliwatan Menu UI

Tulisan ieu nunjukkeun ka admin kumaha cara ngadamel kasus nganggo antarmuka pangguna platform. Admin ogé tiasa nambihan pangguna kana kasus énggal sareng ngapdet kasus anu tos aya.

Nyiptakeun pasualan

  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Dina panel Admin, klik tab menu Kasus.
  4. Klik tombol NEW. Jandéla anyar némbongan.

img

Eusian widang anu diperyogikeun teras klik SAVE.

img

Nambahan pamaké

Opsional, admin tiasa nambihan pangguna kana kasus dina halaman ieu. Gulung ka handap teras klik tombol ASSIGN USERS .

Dina Assign User modal, eusian widang. Centang kotak pikeun ngirim bewara email ka pangguna. (opsional)

Ngaropéa pasualan

Pikeun ngapdet sajumlah leutik kasus, paké alat Édit sareng jieun perobahan ka unggal kasus individu.

  • Catetan Pikeun ngapdet sajumlah ageung kasus dina hiji waktos, pertimbangkeun nganggo alat Impor
  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Dina panel Admin, klik tab menu Kasus.
  4. Panggihan hiji pasualan teras klik Édit.
  5. Jieun parobahan teras klik SAVE.
