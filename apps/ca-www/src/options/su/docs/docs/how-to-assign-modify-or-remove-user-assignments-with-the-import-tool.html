Kumaha Napelkeun, Ngarobih atanapi Ngahapus Tugas Pamaké nganggo Alat Impor

Tulisan ieu nunjukkeun ka admin kumaha ngatur pangguna kasus nganggo alat Impor.

Admin tiasa:

  • napelkeun pamaké pikeun pasualan
  • ngaropéa pamaké geus ditugaskeun ka kasus
  • miceun pangguna anu ditugaskeun kana hiji pasualan

Buka alat Impor

  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Klik tab Integrasi teras IMPOR dina submenu.
  4. Klik tombol NEW. Kotak dialog Tambah Impor mintonkeun.

Nyieun rékaman

Unduh hiji atanapi langkung file CSV. Anggo file ieu pikeun nambihan, ngarobih, atanapi ngahapus pangguna kasus. Saatos kotak dialog Add Imports dibuka:

  1. Klik widang Tipe.
  2. Gulung ngaliwatan menu turun-handap tur pilih hiji pilihan (tempo di handap). Hiji link mintonkeun.
  3. Pencét tautan Unduh Sampel. A file CSV diundeur ka alat Anjeun.

Buka file sareng tambahkeun rékaman. Klik di dieu pikeun neuleuman kumaha carana make data format dina file CSV.

Pilihan file

Netepkeun pangguna kana kasus * Pilih pilihan ieu pikeun nambihan pangguna kana kasus. * Ninggalkeun sél case_role_order kosong. * Pindah ka halaman Pamaké sareng salin ID Tag pikeun jalma ieu. Upami kolom Tag ID kosong, lebetkeun ID unik, klik Simpen, teras salin ID kana file CSV.

Ngarobah pamaké ditugaskeun ka kasus * Pilih pilihan ieu pikeun ngapdet data pikeun pamaké ayeuna ditugaskeun ka kasus. * Paké Case Tag ID pikeun kolom case_ID.

Hapus pangguna anu ditugaskeun ka kasus * Pilih pilihan ieu pikeun ngahapus pangguna anu ditugaskeun ka kasus.

Ngunggah file

Anjeun tiasa unggah file langsung atanapi uih deui ka kotak dialog Tambah Impor dina waktos engké sareng ngalengkepan léngkah-léngkah ieu.

  1. Buka kotak dialog Tambah Impor.
  2. Lebetkeun nami unik pikeun file CSV.
  3. Pencét ikon tambah (+) pikeun unggah file tina alat anjeun.
  4. Klik widang Tipe tur pilih hiji pilihan. (tingali luhureun)
  5. Pencét SAVE.

Ngolah

Gumantung kana jumlah rékaman dina file sareng beban sistem, platform tiasa nyandak sababaraha menit pikeun ngolah impor.

Nalika impor parantos réngsé, platform nyaluyukeun rékaman pangguna (nyaéta nambihan, ngarobih, atanapi ngahapus) dina halaman Kasus.

Kagagalan

Lamun operasi impor gagal, platform nu mulih pesen Gagal. Dina kalolobaan kasus, anu ngabalukarkeun gagalna nyaéta pormat data.

Pikeun ningali sabab gagalna:

  1. Klik Témbongkeun dina judul Aksi.
  2. Dina kaca salajengna, hover beurit dina ikon Gagal. tooltip mintonkeun kasalahan.
  3. Pindah ka file CVS sareng saluyukeun sareng ngalereskeun kasalahan.
  4. Unggah deui filena.
