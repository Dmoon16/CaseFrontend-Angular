सार्वजनिक साइनअप लिंक के साथ स्वतः मामले कैसे बनाएँ

यह आलेख व्यवस्थापकों को एकल साइनअप लिंक भेजकर मामला बनाने (और उसी समय उपयोगकर्ता जोड़ने) का तरीका दिखाता है.

लिंक का पता लगाना

  1. व्यवस्थापक खाते में लॉग इन करें।
  2. ऐप पेज पर ऐप ढूंढें और एडमिन बटन पर क्लिक करें।
  3. एडमिन पैनल पर, सेटिंग मेन्यू टैब पर क्लिक करें.
  4. सुनिश्चित करें कि सिस्टम सेटअप सबमेनू टैब चयनित है।

सुनिश्चित करें कि साइनअप की अनुमति दें फ़ील्ड ऑन पर सेट है। (आवश्यक) जब यह सुविधा सक्षम होती है, तो केस एक्टिव एक अद्वितीय लिंक उत्पन्न करता है।

  1. स्वतः मामला बनाएँ फ़ील्ड पर क्लिक करें और चालू चुनें। मामला उपयोगकर्ता भूमिका फ़ील्ड प्रदर्शित करता है।
  2. कोई भूमिका चुनें. (नीचे देखें)

लिंक को एक या अधिक लोगों के साथ साझा करें। प्रत्येक उपयोगकर्ता जो लिंक पर क्लिक करता है और साइनअप पूरा करता है, प्लेटफ़ॉर्म को मामला पृष्ठ पर एक नया मामला उत्पन्न करने के लिए ट्रिगर करता है और मामले में उपयोगकर्ता को स्वत: असाइन करता है।
