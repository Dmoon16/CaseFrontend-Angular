मौजूदा मामलों के साथ उपयोगकर्ता असाइनमेंट कैसे असाइन करें या निकालें

यह लेख व्यवस्थापकों को मामला उपयोगकर्ताओं को असाइन करने और निकालने का तरीका दिखाता है. हम आपको यह भी दिखाते हैं कि उपयोगकर्ता डेटा को कैसे संशोधित किया जाए।

एक मौजूदा मामला खोलना

  1. व्यवस्थापक खाते में लॉग इन करें।
  2. ऐप पेज पर ऐप ढूंढें और एडमिन बटन पर क्लिक करें।
  3. मामले मेनू टैब पर क्लिक करें।
  4. सुनिश्चित करें कि सबमेनू में सक्रिय चुना गया है।
  5. कार्रवाई कॉलम तक नीचे स्क्रॉल करें।

आईएमजी

असाइन करें या हटाएं

किसी उपयोगकर्ता को मामले को असाइन करने या मामले से उपयोगकर्ता को निकालने के लिए उपयोगकर्ता असाइन करें पर क्लिक करें (लेकिन प्लेटफ़ॉर्म नहीं). अगले पृष्ठ पर: जोड़ें पर क्लिक करें और कार्यप्रवाह पूर्ण करें। किसी उपयोगकर्ता को हटाने के लिए निकालें पर क्लिक करें।

आईएमजी

संशोधित

मामले पृष्ठ पर उपयोगकर्ता डेटा को संपादित या अद्यतन करना संभव नहीं है। इसके बजाय, इन चरणों का पालन करें:

  1. उपयोगकर्ता मेनू टैब पर क्लिक करें।
  2. उपयोगकर्ता नाम का पता लगाएँ और संपादित करें पर क्लिक करें।
  3. परिवर्तन करें और सहेजें पर क्लिक करें।
