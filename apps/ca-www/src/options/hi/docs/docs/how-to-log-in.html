लॉग इन कैसे करें

अपने मौजूदा केस एक्टिव खाते में लॉग इन करना सीखें।

प्रवेश किया

  1. लॉगिन पेज पर जाएं।
  2. लॉगिन बटन पर क्लिक करें।
  3. अपना ईमेल दर्ज करें।
  4. अगला पर क्लिक करें।

एक खाता सत्यापित करना

  1. एक नया ब्राउज़र टैब खोलें और अपने ईमेल खाते में लॉग इन करें।
  2. छह अंकों का कोड कॉपी करें।
  3. केस एक्टिव टैब पर लौटें।
  4. कोड को फ़ील्ड में पेस्ट करें। एक नई विंडो प्रदर्शित करता है।

केस एक्टिव ऐप पेज प्रदर्शित करता है।

  • नोट प्लेटफ़ॉर्म व्यवस्थापक विशेषाधिकार वाले उपयोगकर्ताओं के लिए APP बटन के आगे एक व्यवस्थापक बटन प्रदर्शित करता है।
