UI मेनू के माध्यम से मामले कैसे बनाएं या संशोधित करें

यह आलेख व्यवस्थापकों को प्लेटफ़ॉर्म के उपयोगकर्ता इंटरफ़ेस के साथ मामला बनाने का तरीका दिखाता है. व्यवस्थापक उपयोगकर्ताओं को एक नए मामले में भी जोड़ सकते हैं और मौजूदा मामलों को अपडेट कर सकते हैं।

केस बनाना

  1. व्यवस्थापक खाते में लॉग इन करें।
  2. ऐप पेज पर ऐप ढूंढें और एडमिन बटन पर क्लिक करें।
  3. एडमिन पैनल पर, केस मेन्यू टैब पर क्लिक करें.
  4. न्यू बटन पर क्लिक करें। एक नई विंडो प्रदर्शित करता है।

किसी मामले में संशोधन करना

कम संख्या में मामलों को अपडेट करने के लिए, संपादन टूल का उपयोग करें और प्रत्येक अलग-अलग मामले में परिवर्तन करें।

  • नोट एक बैठक में बड़ी संख्या में मामलों को अद्यतन करने के लिए, आयात उपकरण का उपयोग करने पर विचार करें
  1. व्यवस्थापक खाते में लॉग इन करें।
  2. ऐप पेज पर ऐप ढूंढें और एडमिन बटन पर क्लिक करें।
  3. एडमिन पैनल पर, केस मेन्यू टैब पर क्लिक करें.
  4. एक मामले का पता लगाएँ और संपादित करें पर क्लिक करें।
  5. परिवर्तन करें और सहेजें पर क्लिक करें।
