आयात उपकरण के साथ केस प्रबंधन डेटा को फ़ॉर्मेट करना

आयात उपकरण के साथ मामलों को प्रबंधित करने के लिए CSV फ़ाइलों में आवश्यक फ़ील्ड की सूची नीचे दी गई है.

उपयोगकर्ता नाम

  • उपयोगकर्ता का ईमेल पता दर्ज करें।

tag_id (एक आईडी जिसे आप बाद में संदर्भित कर सकते हैं)

  • एक अद्वितीय पहचानकर्ता (अल्फ़ान्यूमेरिक वर्णों का उपयोग करें, हाइफ़न स्वीकार किए जाते हैं)

case_status

  • 0 = बंद
  • 1 = सक्रिय

date_open

  • महीना दिन वर्ष
  • मिमी/दिन/वर्ष

के बारे में

  • मामले का एक अनूठा विवरण (अल्फ़ान्यूमेरिक वर्णों का उपयोग करें, हाइफ़न स्वीकार किए जाते हैं)
