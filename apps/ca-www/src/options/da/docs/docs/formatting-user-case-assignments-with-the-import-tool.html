Formatering af brugertildelingsdata med importværktøjet

Nedenfor er en liste over datafelter i CSV-filer. Hver importmulighed har en anden kombination af felter.

case_ID

  • Indtast sagens bruger- eller tag-id.
  • Gå til siden Sager > Rediger > Tag ID ELLER Gå til siden Sager > Rediger > Sags-id i url.

bruger ID

  • Indtast brugerens ID eller Tag ID.
  • Gå til siden Brugere > Rediger > Tag ID ELLER Gå til siden Brugere > Rediger > Bruger ID i url.

case_rolle_order

  • Lad cellen være tom.

sagsrolle-id

  • Indtast: klienter, personale eller standard.

case_notify

  • 0 = nej
  • 1 = Ja, send brugeren en e-mail-meddelelse.
