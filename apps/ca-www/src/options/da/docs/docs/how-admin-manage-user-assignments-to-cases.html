Hvordan administratorer administrerer brugertildelinger til sager

Der er fem måder at administrere brugere og sagsopgaver på. Klik på et link for at lære hvordan.

Metode Type Handlinger detaljer
Mens du opretter en sag UI tilføje Admin-knap > Sager > Ny knap > Knappen Tildel brugere
Tildel automatisk bruger ved hjælp af offentligt tilmeldingslink Brugerhandling tilføje Admin-knap > Indstillinger > Systemopsætningsside
Brug af UI på eksisterende sag UI tilføje og fjerne Admin-knap > Sager > Tildel brugere
Importværktøj UI tilføje, ændre og fjerne Admin-knap > Integrationer > Importer
Brug af API (kommer snart) API tilføje, ændre og fjerne N/A
