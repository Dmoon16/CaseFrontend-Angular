સાર્વજનિક સાઇનઅપ લિંક સાથે ઑટોમૅટિક રીતે કેસ કેવી રીતે બનાવવો

આ લેખ એડમિન્સને એક સાઇનઅપ લિંક મોકલીને કેસ કેવી રીતે બનાવવો (અને તે જ સમયે વપરાશકર્તાને ઉમેરવા) બતાવે છે.

લિંક શોધી રહ્યાં છીએ

  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્સ પેજ પર એક એપ શોધો અને એડમિન બટન પર ક્લિક કરો.
  3. એડમિન પેનલ પર, સેટિંગ્સ મેનુ ટેબ પર ક્લિક કરો.
  4. ખાતરી કરો કે SYSTEM SETUP સબમેનુ ટેબ પસંદ કરેલ છે.

સેટિંગ્સ ગોઠવી રહ્યું છે

USER SIGNUP મથાળા સુધી નીચે સ્ક્રોલ કરો.

ખાતરી કરો કે સાઇનઅપને મંજૂરી આપો ફીલ્ડ ચાલુ પર સેટ કરેલ છે. (જરૂરી) જ્યારે આ સુવિધા સક્ષમ હોય, ત્યારે કેસ એક્ટિવ એક અનન્ય લિંક જનરેટ કરે છે.

  1. ઓટો ક્રિએટ કેસ ફીલ્ડ પર ક્લિક કરો અને ઓન પસંદ કરો. કેસ યુઝર રોલ ફીલ્ડ દર્શાવે છે.
  2. ભૂમિકા પસંદ કરો. (નીચે જુઓ)

કેસ વપરાશકર્તા ભૂમિકાઓ

ત્રણ કેસ વપરાશકર્તા ભૂમિકાઓ છે.

  • ગ્રાહકો (ભલામણ કરેલ)
  • મૂળભૂત
  • સ્ટાફ

લિંક મોકલી રહ્યું છે

એક અથવા વધુ લોકો સાથે લિંક શેર કરો. દરેક વપરાશકર્તા જે લિંક પર ક્લિક કરે છે અને સાઇનઅપ પૂર્ણ કરે છે તે કેસ પેજ પર નવો કેસ જનરેટ કરવા માટે પ્લેટફોર્મને ટ્રિગર કરે છે અને કેસમાં વપરાશકર્તાને સ્વતઃ સોંપી દે છે.
