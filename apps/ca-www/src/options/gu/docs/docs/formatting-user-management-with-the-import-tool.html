આયાત ટૂલ સાથે યુઝર મેનેજમેન્ટ ડેટાનું ફોર્મેટિંગ

નીચે CSV ફાઇલોમાં ડેટા ફીલ્ડ્સની સૂચિ છે. દરેક આયાત વિકલ્પમાં ક્ષેત્રોનું અલગ સંયોજન હોય છે.

વપરાશકર્તા નામ

  • વપરાશકર્તાનું ઇમેઇલ સરનામું દાખલ કરો.

આપેલા નામ

  • વપરાશકર્તાનું પ્રથમ નામ દાખલ કરો.

અટક

  • વપરાશકર્તાનું છેલ્લું નામ દાખલ કરો.

લોકેલ

  • આ કૉલમ વપરાશકર્તાની પસંદગીની ભાષા માટે તપાસ કરે છે.
  • અંગ્રેજી માટે en, સ્પેનિશ માટે sp દાખલ કરો.

ઝોન માહિતી

  • આ વપરાશકર્તાનો સમય ઝોન સેટ કરે છે. ખાલી જગ્યાઓ વિના ખંડ અને શહેર દાખલ કરો (અંડરસ્કોરનો ઉપયોગ કરો).
  • પ્રોફાઇલ > ભાષા અને પ્રદેશ > ટાઇમઝોન ડ્રોપ-ડાઉન મેનૂમાં વિકલ્પનો ઉપયોગ કરવો આવશ્યક છે.
  • શ્રેષ્ઠ-યોગ્ય સમય ઝોનનો ઉપયોગ કરો. ઉદાહરણ તરીકે, બોસ્ટન ડ્રોપ-ડાઉન પર નથી, તેથી અમે ન્યૂ યોર્ક (અથવા EST/EDT ટાઇમ ઝોનમાં કોઈપણ શહેર) નો ઉપયોગ કરી શકીએ છીએ.
