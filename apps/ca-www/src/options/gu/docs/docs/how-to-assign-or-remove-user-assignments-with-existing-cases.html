હાલના કેસો સાથે વપરાશકર્તાની સોંપણીઓ કેવી રીતે સોંપવી અથવા દૂર કરવી

આ લેખ એડમિન્સને બતાવે છે કે કેસ વપરાશકર્તાઓને કેવી રીતે સોંપવું અને દૂર કરવું. અમે તમને યુઝર ડેટાને કેવી રીતે સંશોધિત કરવો તે પણ બતાવીએ છીએ.

હાલનો કેસ ખોલી રહ્યા છીએ

  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્સ પેજ પર એક એપ શોધો અને એડમિન બટન પર ક્લિક કરો.
  3. કેસ મેનુ ટેબ પર ક્લિક કરો.
  4. ખાતરી કરો કે સબમેનુમાં ACTIVE પસંદ કરેલ છે.
  5. ACTION કૉલમ સુધી નીચે સ્ક્રોલ કરો.

સોંપો અથવા દૂર કરો

કેસમાં વપરાશકર્તાને સોંપવા અથવા વપરાશકર્તાને કેસમાંથી દૂર કરવા માટે વપરાશકર્તાઓને સોંપો પર ક્લિક કરો (પરંતુ પ્લેટફોર્મ નહીં). આગલા પૃષ્ઠ પર: ADD પર ક્લિક કરો અને વર્કફ્લો પૂર્ણ કરો. વપરાશકર્તાને કાઢી નાખવા માટે દૂર કરો ક્લિક કરો.

ફેરફાર કરો

કેસ પેજ પર વપરાશકર્તાના ડેટાને સંપાદિત અથવા અપડેટ કરવું શક્ય નથી. તેના બદલે, આ પગલાં અનુસરો:

  1. વપરાશકર્તાઓ મેનૂ ટેબ પર ક્લિક કરો.
  2. વપરાશકર્તા નામ શોધો અને સંપાદિત કરો ક્લિક કરો.
  3. ફેરફારો કરો અને સાચવો પર ક્લિક કરો.
