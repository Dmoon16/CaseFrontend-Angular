આયાત ટૂલ વડે વપરાશકર્તાની સોંપણીઓ કેવી રીતે સોંપવી, સંશોધિત કરવી અથવા દૂર કરવી

આ લેખ આયાત ટૂલ વડે કેસ યુઝર્સને કેવી રીતે મેનેજ કરવું તે એડમિન્સને બતાવે છે.

સંચાલકો આ કરી શકે છે:

  • વપરાશકર્તાઓને કેસ સોંપો
  • પહેલાથી જ કેસ માટે સોંપેલ વપરાશકર્તાઓને સંશોધિત કરો
  • કેસ માટે સોંપેલ વપરાશકર્તાઓને દૂર કરો

આયાત સાધન ખોલી રહ્યું છે

  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્લિકેશન પૃષ્ઠ પર એક એપ્લિકેશન શોધો અને એડમિન બટનને ક્લિક કરો.
  3. એકીકરણ ટેબ પર ક્લિક કરો અને પછી સબમેનુ પર આયાત કરો.
  4. નવું બટન ક્લિક કરો. આયાત ઉમેરો સંવાદ બોક્સ દર્શાવે છે.

રેકોર્ડ બનાવી રહ્યા છે

એક અથવા વધુ CSV ફાઇલો ડાઉનલોડ કરો. કેસ વપરાશકર્તાઓને ઉમેરવા, સંશોધિત કરવા અથવા દૂર કરવા માટે આ ફાઇલોનો ઉપયોગ કરો. આયાત ઉમેરો સંવાદ બોક્સ ખુલે તે પછી:

  1. પ્રકાર ફીલ્ડ પર ક્લિક કરો.
  2. ડ્રોપ-ડાઉન મેનૂ દ્વારા સ્ક્રોલ કરો અને એક વિકલ્પ પસંદ કરો (નીચે જુઓ). એક લિંક પ્રદર્શિત થાય છે.
  3. ડાઉનલોડ નમૂના લિંક પર ક્લિક કરો. તમારા ઉપકરણ પર CSV ફાઇલ ડાઉનલોડ થાય છે.

ફાઇલ ખોલો અને રેકોર્ડ ઉમેરો. CSV ફાઇલમાં ફોર્મેટ ડેટાનો ઉપયોગ કેવી રીતે કરવો તે જાણવા માટે અહીં ક્લિક કરો.

ફાઇલ વિકલ્પો

વપરાશકર્તાઓને કેસમાં સોંપો * વપરાશકર્તાઓને કેસમાં ઉમેરવા માટે આ વિકલ્પ પસંદ કરો. * કેસ_રોલ_ઓર્ડર સેલ ખાલી છોડો. * યુઝર્સ પેજ પર જાઓ અને આ વ્યક્તિ માટે ટેગ આઈડી કોપી કરો. જો ટેગ ID ફીલ્ડ ખાલી હોય, તો અનન્ય ID દાખલ કરો, સાચવો પર ક્લિક કરો અને પછી ID ને CSV ફાઇલમાં કૉપિ કરો.

કેસોને સોંપેલ વપરાશકર્તાઓને સંશોધિત કરો * કેસને સોંપેલ વર્તમાન વપરાશકર્તાઓ માટે ડેટા અપડેટ કરવા માટે આ વિકલ્પ પસંદ કરો. * કેસ_આઈડી કૉલમ માટે કેસ ટૅગ ID નો ઉપયોગ કરો.

img

કેસ માટે સોંપેલ વપરાશકર્તાઓને દૂર કરો * કેસ માટે સોંપેલ વપરાશકર્તાઓને દૂર કરવા માટે આ વિકલ્પ પસંદ કરો.

ફાઈલ અપલોડ કરી રહ્યા છીએ

તમે તરત જ ફાઇલ અપલોડ કરી શકો છો અથવા પછીની તારીખે આયાત ઉમેરો સંવાદ બોક્સ પર પાછા આવી શકો છો અને આ પગલાંઓ પૂર્ણ કરી શકો છો.

  1. આયાત ઉમેરો સંવાદ બોક્સ ખોલો.
  2. CSV ફાઇલ માટે અનન્ય નામ દાખલ કરો.
  3. તમારા ઉપકરણમાંથી ફાઇલ અપલોડ કરવા માટે વત્તા (+) આયકન પર ક્લિક કરો.
  4. પ્રકાર ફીલ્ડ પર ક્લિક કરો અને એક વિકલ્પ પસંદ કરો. (ઉપર જુવો)
  5. સેવ પર ક્લિક કરો.

પ્રક્રિયા

ફાઇલ અને સિસ્ટમ લોડમાં રેકોર્ડ્સની સંખ્યાના આધારે, પ્લેટફોર્મ આયાતની પ્રક્રિયા કરવા માટે થોડી મિનિટો લઈ શકે છે.

જ્યારે આયાત પૂર્ણ થાય છે, ત્યારે પ્લેટફોર્મ કેસ પેજ પર વપરાશકર્તા રેકોર્ડ્સ (એટલે કે ઉમેરે છે, સુધારે છે અથવા દૂર કરે છે) ગોઠવે છે.

નિષ્ફળતા

જો આયાત કામગીરી નિષ્ફળ જાય, તો પ્લેટફોર્મ નિષ્ફળ સંદેશ પરત કરે છે. મોટાભાગના કિસ્સાઓમાં, નિષ્ફળતાનું કારણ ડેટા ફોર્મેટિંગ છે.

નિષ્ફળતાનું કારણ જોવા માટે:

  1. ક્રિયા શીર્ષક હેઠળ જુઓ ક્લિક કરો.
  2. આગલા પૃષ્ઠ પર, નિષ્ફળ ચિહ્ન પર માઉસને હોવર કરો. ટૂલટીપ ભૂલ દર્શાવે છે.
  3. CVS ફાઇલ પર જાઓ અને એડજસ્ટ કરો અને ભૂલને ઠીક કરો.
  4. ફાઇલ ફરીથી અપલોડ કરો.
