કેવી રીતે લોગ ઇન કરવું

તમારા હાલના કેસ એક્ટિવ એકાઉન્ટમાં કેવી રીતે લૉગ ઇન કરવું તે જાણો.

લૉગ ઇન

  1. લોગિન પેજ પર જાઓ.
  2. LOGIN બટન પર ક્લિક કરો.
  3. તમારું ઈમેલ એડ્રેસ લખો.
  4. આગળ ક્લિક કરો.

એકાઉન્ટ ચકાસી રહ્યા છીએ

  1. એક નવું બ્રાઉઝર ટેબ ખોલો અને તમારા ઈમેલ એકાઉન્ટમાં લૉગ ઇન કરો.
  2. છ-અંકનો કોડ કૉપિ કરો.
  3. કેસ એક્ટિવ ટેબ પર પાછા ફરો.
  4. કોડને ક્ષેત્રોમાં પેસ્ટ કરો. એક નવી વિન્ડો દેખાય છે.

કેસ એક્ટિવ એપ્સ પેજ દર્શાવે છે.

  • નોંધ પ્લેટફોર્મ એડમિન વિશેષાધિકારો ધરાવતા વપરાશકર્તાઓ માટે APP બટનની બાજુમાં એડમિન બટન દર્શાવે છે.
