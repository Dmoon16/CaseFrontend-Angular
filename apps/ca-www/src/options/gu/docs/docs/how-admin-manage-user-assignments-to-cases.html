એડમિન્સ કેવી રીતે કેસમાં વપરાશકર્તાની સોંપણીઓનું સંચાલન કરે છે

વપરાશકર્તાઓ અને કેસ સોંપણીઓનું સંચાલન કરવાની પાંચ રીતો છે. કેવી રીતે જાણવા માટે એક લિંક પર ક્લિક કરો.

પદ્ધતિ પ્રકાર ક્રિયાઓ વિગતો
કેસ બનાવતી વખતે UI ઉમેરો એડમિન બટન > કેસ > નવું બટન > વપરાશકર્તાઓને સોંપો બટન
જાહેર સાઇનઅપ લિંકનો ઉપયોગ કરીને વપરાશકર્તાને સ્વતઃ સોંપો વપરાશકર્તા ક્રિયા ઉમેરો એડમિન બટન > સેટિંગ્સ > સિસ્ટમ સેટઅપ પેજ
હાલના કેસ પર UI નો ઉપયોગ કરવો UI ઉમેરો અને દૂર કરો એડમિન બટન > કેસ > વપરાશકર્તાઓને સોંપો
આયાત સાધન UI ઉમેરો, સંશોધિત કરો અને દૂર કરો એડમિન બટન > એકીકરણ > આયાત
