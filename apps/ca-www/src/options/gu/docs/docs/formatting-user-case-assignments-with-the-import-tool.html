આયાત ટૂલ વડે યુઝર કેસ અસાઇનમેન્ટ ડેટાનું ફોર્મેટિંગ

નીચે CSV ફાઇલોમાં ડેટા ફીલ્ડ્સની સૂચિ છે. દરેક આયાત વિકલ્પમાં ક્ષેત્રોનું અલગ સંયોજન હોય છે.

કેસ_આઈડી

  • કેસ વપરાશકર્તા અથવા ટેગ ID દાખલ કરો.
  • કેસ પેજ > એડિટ > ટૅગ આઈડી પર જાઓ અથવા કેસ પેજ > એડિટ > કેસ આઈડી પર જાઓ.

user_ID

  • વપરાશકર્તાનું ID અથવા ટેગ ID દાખલ કરો.
  • યુઝર્સ પેજ > એડિટ > ટેગ આઈડી પર જાઓ અથવા યુઆરએલમાં યુઝર્સ પેજ > એડિટ > યુઝર આઈડી પર જાઓ.

કેસ_રોલ_ઓર્ડર

  • સેલ ખાલી છોડો.

કેસ_રોલ_આઈડી

  • દાખલ કરો: ક્લાયંટ, સ્ટાફ અથવા ડિફોલ્ટ.

કેસ_સૂચના

  • 0 = ના
  • 1 = હા, વપરાશકર્તાને ઈમેલ સૂચના મોકલો.
