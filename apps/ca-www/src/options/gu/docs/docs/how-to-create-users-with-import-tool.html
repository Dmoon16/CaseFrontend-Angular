આયાત સાધન સાથે વપરાશકર્તાઓ કેવી રીતે બનાવવું

આ લેખ આયાત ટૂલ વડે વપરાશકર્તાઓને કેવી રીતે બનાવવું તે સંચાલકોને બતાવે છે.

આયાત સાધન ખોલી રહ્યું છે

નવા વપરાશકર્તાઓને જથ્થાબંધ અપલોડ કરવા અને હાલના વપરાશકર્તા રેકોર્ડ્સને સંશોધિત કરવા માટે એડમિન્સ આયાત સાધનનો ઉપયોગ કરી શકે છે.

  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્લિકેશન પૃષ્ઠ પર એક એપ્લિકેશન શોધો અને એડમિન બટનને ક્લિક કરો.
  3. એડમિન પેનલ પર, એકીકરણ ટેબ પર ક્લિક કરો અને પછી સબમેનુ પર આયાત કરો.
  4. નવું બટન ક્લિક કરો. નવા બ્રાઉઝર ટેબમાં આયાત ઉમેરો સંવાદ બોક્સ પ્રદર્શિત થાય છે.

રેકોર્ડ બનાવી રહ્યા છે

આયાત ઉમેરો સંવાદ બોક્સ ખુલે તે પછી:

  1. પ્રકાર ફીલ્ડ પર ક્લિક કરો અને એક વિકલ્પ પસંદ કરો (નીચે જુઓ). એક લિંક પ્રદર્શિત થાય છે.
  2. ડાઉનલોડ નમૂના લિંક પર ક્લિક કરો. તમારા ઉપકરણ પર CSV ફાઇલ ડાઉનલોડ થાય છે.

આગળ, ફાઇલ ખોલો અને રેકોર્ડ્સ ઉમેરો. CSV ફાઇલમાં ફોર્મેટ ડેટાનો ઉપયોગ કેવી રીતે કરવો તે જાણવા માટે અહીં ક્લિક કરો.

પ્રકાર

જ્યારે તમે પ્રકાર ફીલ્ડ પર ક્લિક કરો છો, ત્યારે ડ્રોપ-ડાઉન મેનૂ પ્રદર્શિત થાય છે. બલ્ક આયાત માટે બેમાંથી એક વિકલ્પ પસંદ કરો.

: ઉદ્ગાર: નોંધ ડ્રોપ-ડાઉન મેનૂમાં ઘણા વિકલ્પો છે, પરંતુ માત્ર બે આયાત સાધન સાથે સંબંધિત છે. દરેક વિકલ્પમાં અલગ નમૂનાની CSV ફાઇલ હોય છે.

વપરાશકર્તાઓ ઉમેરો

  • વપરાશકર્તા રેકોર્ડ્સના સરળ બલ્ક અપલોડ માટે આ વિકલ્પ પસંદ કરો.
  • વપરાશકર્તાઓ ઉમેરો આયાત પ્રકાર માટે કૉલમ હેડિંગ સાથે CSV ફાઇલ ખોલવા માટે નમૂના ડાઉનલોડ કરો લિંકને ક્લિક કરો.

વપરાશકર્તાઓને સંશોધિત કરો

  • વર્તમાન વપરાશકર્તાઓ માટે ડેટા અપડેટ કરવા માટે આ વિકલ્પ પસંદ કરો.
  • મોડિફાઈ યુઝર્સ ઈમ્પોર્ટ ટાઈપ માટે કોલમ હેડિંગ સાથે CSV ફાઈલ ખોલવા માટે સેમ્પલ ડાઉનલોડ કરો લિંક પર ક્લિક કરો.

ફાઈલ અપલોડ કરી રહ્યા છીએ

તમે તરત જ ફાઇલ અપલોડ કરી શકો છો અથવા પછીની તારીખે આયાત ઉમેરો સંવાદ બોક્સ પર પાછા આવી શકો છો અને આ પગલાંઓ પૂર્ણ કરી શકો છો.

  1. આયાત ઉમેરો સંવાદ બોક્સ ખોલો.
  2. CSV ફાઇલ માટે અનન્ય નામ દાખલ કરો.
  3. તમારા ઉપકરણમાંથી ફાઇલ અપલોડ કરવા માટે વત્તા (+) આયકન પર ક્લિક કરો.
  4. પ્રકાર ફીલ્ડ પર ક્લિક કરો અને એક વિકલ્પ પસંદ કરો. (ઉપર જુવો)
  5. સેવ પર ક્લિક કરો.

પ્રક્રિયા

ફાઇલ અને સિસ્ટમ લોડમાં રેકોર્ડ્સની સંખ્યાના આધારે, પ્લેટફોર્મ આયાતની પ્રક્રિયા કરવા માટે થોડી મિનિટો લઈ શકે છે.

જ્યારે આયાત પૂર્ણ થાય છે, ત્યારે પ્લેટફોર્મ એડમિનનાં વપરાશકર્તાઓ ડેશબોર્ડ પર વપરાશકર્તા રેકોર્ડ્સ (એટલે કે ઉમેરે છે, સંશોધિત કરે છે અથવા અક્ષમ કરે છે) ગોઠવે છે.

નિષ્ફળતા

જો આયાત કામગીરી નિષ્ફળ જાય, તો પ્લેટફોર્મ નિષ્ફળ સંદેશ પરત કરે છે. મોટાભાગના કિસ્સાઓમાં, નિષ્ફળતાનું કારણ ડેટા ફોર્મેટિંગ છે.

નિષ્ફળતાનું કારણ જોવા માટે:

  1. ક્રિયા શીર્ષક હેઠળ જુઓ ક્લિક કરો.
  2. આગલા પૃષ્ઠ પર, નિષ્ફળ આઇકોન પર માઉસને હોવર કરો. ટૂલટીપ ભૂલ દર્શાવે છે.
  3. CVS ફાઇલ પર જાઓ અને એડજસ્ટ કરો અને ભૂલને ઠીક કરો.
  4. ફાઇલ ફરીથી અપલોડ કરો.
