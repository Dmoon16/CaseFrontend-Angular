Форматиране на данни за управление на случаи с инструмента за импортиране

По-долу е даден списък със задължителни полета в CSV файлове за управление на случаи с инструмента за импортиране.

потребителско име

  • Въведете имейл адреса на потребителя.

tag_id (идентификатор, който можете да посочите по-късно)

  • уникален идентификатор (използвайте буквено-цифрови знаци, тирета се приемат)

case_status

  • 0 = затворено
  • 1 = активно

дата_отваряне

  • месец ден година
  • дд/мм/гггг

относно

  • уникално описание на случая (използвайте буквено-цифрови знаци, тирета се приемат)
