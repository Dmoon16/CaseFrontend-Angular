பயனர்களையும் வழக்குப் பணிகளையும் நிர்வகிக்க ஐந்து வழிகள் உள்ளன. எப்படி என்பதை அறிய இணைப்பைக் கிளிக் செய்யவும்.

முறை வகை செயல்கள் விவரங்கள்
ஒரு வழக்கை உருவாக்கும் போது UI கூட்டு நிர்வாகம் பொத்தான் > வழக்குகள் > புதிய பொத்தான் > பயனர்களை ஒதுக்கு பொத்தான்
பொது பதிவு இணைப்பைப் பயன்படுத்தி பயனரைத் தானாக ஒதுக்கவும் பயனர் செயல் கூட்டு நிர்வாகம் பொத்தான் > அமைப்புகள் > கணினி அமைவு பக்கம்
ஏற்கனவே உள்ள வழக்கில் UI ஐப் பயன்படுத்துதல் UI சேர்க்க & நீக்க நிர்வாகம் பொத்தான் > வழக்குகள் > பயனர்களை ஒதுக்கவும்
