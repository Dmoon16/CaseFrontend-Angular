பொது பதிவு இணைப்பு மூலம் பயனர்களை தானாக ஒதுக்குவது எப்படி

கணினி அமைப்புகளை எவ்வாறு கட்டமைப்பது என்பதை நிர்வாகிகள் கற்றுக்கொள்கிறார்கள், இதனால் புதிய பயனர்கள் தானாகவே புதிதாக உருவாக்கப்பட்ட கேஸுக்கு ஒதுக்கப்படுவார்கள்.

அமைவுப் பக்கத்தைத் திறக்கிறது

சிஸ்டம்ஸ் அமைவு பக்கத்தைத் திறப்பது முதல் படி.

  1. நிர்வாகி கணக்கில் உள்நுழைக.
  2. ஆப்ஸ் பக்கத்தில் பயன்பாட்டைக் கண்டறிந்து, நிர்வாகி பொத்தானைக் கிளிக் செய்யவும்.
  3. கணினிகள் தாவலைக் கிளிக் செய்து, துணைமெனுவில் SYSTEM SETUP என்பதைக் கிளிக் செய்யவும்.
  4. USER SIGNUP தலைப்புக்கு கீழே உருட்டவும்.

கணினி அமைப்புகளை கட்டமைக்கிறது

  1. அனுமதி பதிவு மற்றும் தானாக உருவாக்கு வழக்கு கீழ்தோன்றும் மெனுக்களை ஆன் ஆக அமைக்கவும்.
  2. பங்கு மெனுவைக் கிளிக் செய்து ஒரு விருப்பத்தைத் தேர்ந்தெடுக்கவும் (இயல்புநிலை பரிந்துரைக்கப்படுகிறது).
  3. பொது பயனர் பதிவு இணைப்பை நகலெடுத்து பகிரவும்.

இணைப்பை அனுப்புகிறது

இணைப்பைக் கிளிக் செய்து, பதிவுசெய்தலை நிறைவு செய்யும் ஒவ்வொரு பயனரும், வழக்குகள் பக்கத்தில் ஒரு புதிய வழக்கை உருவாக்க தளத்தைத் தூண்டுகிறது மற்றும் அந்த வழக்கில் பயனரைத் தானாக ஒதுக்குகிறது.
