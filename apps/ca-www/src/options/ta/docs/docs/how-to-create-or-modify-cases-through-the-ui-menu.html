UI மெனு மூலம் கேஸ்களை உருவாக்குவது அல்லது மாற்றுவது எப்படி

தளத்தின் பயனர் இடைமுகத்துடன் ஒரு வழக்கை எவ்வாறு உருவாக்குவது என்பதை நிர்வாகிகளுக்கு இந்தக் கட்டுரை காட்டுகிறது. நிர்வாகிகள் புதிய வழக்கில் பயனர்களைச் சேர்க்கலாம் மற்றும் ஏற்கனவே உள்ள வழக்குகளைப் புதுப்பிக்கலாம்.

ஒரு வழக்கை உருவாக்குதல்

  1. நிர்வாகி கணக்கில் உள்நுழைக.
  2. ஆப்ஸ் பக்கத்தில் பயன்பாட்டைக் கண்டறிந்து, நிர்வாகி பொத்தானைக் கிளிக் செய்யவும்.
  3. நிர்வாக குழுவில், வழக்குகள் மெனு தாவலைக் கிளிக் செய்யவும்.
  4. புதிய பொத்தானைக் கிளிக் செய்யவும். ஒரு புதிய சாளரம் தோன்றும்.

விருப்பமாக, நிர்வாகிகள் இந்தப் பக்கத்தில் பயனர்களைச் சேர்க்கலாம். கீழே உருட்டி, பயனர்களை ஒதுக்கு பொத்தானைக் கிளிக் செய்யவும்.

ஒரு வழக்கை மாற்றுதல்

சிறிய எண்ணிக்கையிலான வழக்குகளைப் புதுப்பிக்க, திருத்து கருவியைப் பயன்படுத்தி ஒவ்வொரு தனிப்பட்ட வழக்கிலும் மாற்றங்களைச் செய்யுங்கள்.

  • குறிப்பு ஒரே அமர்வில் அதிக எண்ணிக்கையிலான வழக்குகளைப் புதுப்பிக்க, இறக்குமதி கருவியைப் பயன்படுத்தவும்
  1. நிர்வாகி கணக்கில் உள்நுழைக.
  2. ஆப்ஸ் பக்கத்தில் பயன்பாட்டைக் கண்டறிந்து, நிர்வாகி பொத்தானைக் கிளிக் செய்யவும்.
  3. நிர்வாக குழுவில், வழக்குகள் மெனு தாவலைக் கிளிக் செய்யவும்.
  4. ஒரு வழக்கைக் கண்டுபிடித்து, திருத்து என்பதைக் கிளிக் செய்யவும்.
  5. மாற்றங்களைச் செய்து சேமி என்பதைக் கிளிக் செய்யவும்.
