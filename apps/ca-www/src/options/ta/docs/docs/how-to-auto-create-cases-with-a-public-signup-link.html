பொது பதிவு இணைப்பு மூலம் கேஸ்களை தானாக உருவாக்குவது எப்படி

ஒரே ஒரு பதிவு இணைப்பை அனுப்புவதன் மூலம் ஒரு வழக்கை எவ்வாறு உருவாக்குவது (அதே நேரத்தில் ஒரு பயனரைச் சேர்ப்பது) என்பதை நிர்வாகிகளுக்கு இந்தக் கட்டுரை காட்டுகிறது.

இணைப்பைக் கண்டறிதல்

  1. நிர்வாகி கணக்கில் உள்நுழைக.
  2. ஆப்ஸ் பக்கத்தில் பயன்பாட்டைக் கண்டறிந்து, நிர்வாகி பொத்தானைக் கிளிக் செய்யவும்.
  3. நிர்வாக குழுவில், அமைப்புகள் மெனு தாவலைக் கிளிக் செய்யவும்.
  4. SYSTEM SETUP துணைமெனு தாவல் தேர்ந்தெடுக்கப்பட்டிருப்பதை உறுதிசெய்யவும்.

அனுமதி பதிவு புலம் இயக்கத்தில் அமைக்கப்பட்டுள்ளதை உறுதிசெய்யவும். (அவசியம்) இந்த அம்சம் இயக்கப்பட்டால், கேஸ் ஆக்டிவ் ஒரு தனித்துவமான இணைப்பை உருவாக்குகிறது.

  1. தானாக உருவாக்கு வழக்கு புலத்தை கிளிக் செய்து, ஆன் என்பதைத் தேர்ந்தெடுக்கவும். கேஸ் யூசர் ரோல் புலம் காட்டுகிறது.
  2. ஒரு பாத்திரத்தைத் தேர்ந்தெடுக்கவும். (கீழே பார்)

இணைப்பை அனுப்புகிறது

ஒன்று அல்லது அதற்கு மேற்பட்டவர்களுடன் இணைப்பைப் பகிரவும். இணைப்பைக் கிளிக் செய்து, பதிவுசெய்தலை நிறைவு செய்யும் ஒவ்வொரு பயனரும், வழக்குகள் பக்கத்தில் ஒரு புதிய வழக்கை உருவாக்குவதற்கும், வழக்கில் பயனரைத் தானாக ஒதுக்குவதற்கும் இயங்குதளத்தைத் தூண்டுகிறது.
