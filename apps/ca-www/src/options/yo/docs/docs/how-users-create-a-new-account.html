Bawo ni Awọn olumulo Ṣẹda Akọọlẹ Tuntun kan

Lẹhin ti alabojuto kan ṣafikun olumulo kan si ọran kan: * ti olumulo ba jẹ tuntun si ọran, olumulo gbọdọ pari ṣiṣan iṣẹ lati ṣẹda akọọlẹ kan ati gba ifiwepe lati wọle si ọran naa. * ti olumulo ba ni akọọlẹ caseactive ti o wa tẹlẹ, olumulo gbọdọ gba ifiwepe lati wọle si ọran naa.

Nkan yii ṣe apejuwe awọn igbesẹ ti olumulo gbọdọ pari lati ṣẹda akọọlẹ wọn.

Bibẹrẹ

Awọn ọna meji lo wa ti olumulo le ṣe afikun si ohun elo caseactive kan

  • Nipa abojuto: Ti alabojuto ba ṣẹda olumulo nipasẹ UI Syeed tabi irinṣẹ Wọwọle , alabojuto gbọdọ fi ọna asopọ ranṣẹ si olumulo account.caseactive.com lati tẹsiwaju lati buwolu wọle.
  • Nipa olumulo: Ti olumulo ba ṣẹda akọọlẹ kan nipa lilo ọna asopọ iforukọsilẹ , olumulo yoo jẹ darí laifọwọyi si account.caseactive.com lati tẹsiwaju lati buwolu wọle.

Wọle

Nigbati olumulo ba wọle si akọọlẹ CaseActive wọn fun igba akọkọ, pẹpẹ naa ta olumulo lati:

  • ṣii iwe apamọ imeeli wọn lẹhinna daakọ / lẹẹmọ koodu aabo oni-nọmba mẹfa naa
  • pari profaili akọọlẹ kan

img

Lẹhin ti olumulo tẹ imudojuiwọn, oju-iwe Awọn ohun elo han. Oju-iwe naa fihan orukọ app naa (orukọ ile-iṣẹ), iru olumulo (ipa), ipo, ati bọtini IPE.

img

Lati wọle si ohun elo naa, olumulo gbọdọ gba ifiwepe naa. Olumulo gbọdọ:

  1. Tẹ bọtini INVITATION.
  2. Ṣe ayẹwo awọn igbanilaaye.
  3. Tẹ taabu Awọn iwifunni lati ṣe atunyẹwo ati ṣatunkọ (aṣayan) awọn ofin naa.
  4. Tẹ fọwọsi Wiwọle.

Nigbati olumulo ba fọwọsi, pẹpẹ ṣe afihan bọtini APP kan lori oju-iwe olumulo. Bọtini yii n fun olumulo ni iraye si awọn ọran ti a sọtọ.

Lati ṣatunkọ awọn eto olumulo, olumulo tẹ ellipsis.
