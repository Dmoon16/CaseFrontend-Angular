Bii o ṣe le Ṣẹda tabi Ṣatunṣe Awọn olumulo pẹlu Irinṣẹ Wọwọle

Nkan yii fihan awọn admins bi o ṣe le ṣẹda awọn olumulo pẹlu irinṣẹ Wọle.

Nsii ohun elo agbewọle

Awọn alabojuto le lo ohun elo agbewọle lati gbejade awọn olumulo tuntun ati yi awọn igbasilẹ olumulo ti o wa tẹlẹ pada.

  1. Wọle si akọọlẹ abojuto.
  2. Wa ohun elo kan lori oju-iwe Awọn ohun elo ki o tẹ bọtini ADMIN naa.
  3. Lori igbimọ abojuto, tẹ taabu Integration ati lẹhinna gbe wọle lori akojọ aṣayan.
  4. Tẹ bọtini TITUN. Awọn Fikun Awọn agbewọle lati inu apoti ibanisọrọ han ni taabu aṣawakiri tuntun kan.

Ṣiṣe awọn igbasilẹ

Lẹhin apoti ibaraẹnisọrọ Fikun Awọn agbewọle lati ilu okeere ṣii:

  1. Tẹ aaye Iru ki o yan aṣayan kan (wo isalẹ). A ọna asopọ han.
  2. Tẹ ọna asopọ Ayẹwo Gbigba lati ayelujara . Awọn igbasilẹ faili CSV si ẹrọ rẹ.

Nigbamii, ṣii faili naa ki o ṣafikun awọn igbasilẹ. Tẹ ibi lati kọ ẹkọ bi o ṣe le lo data ọna kika ninu faili CSV.

img

Iru

Nigbati o ba tẹ aaye Iru, akojọ aṣayan-silẹ yoo han. Yan ọkan ninu awọn aṣayan meji fun agbewọle olopobobo.

: exclamation: AKIYESI Akojọ aṣayan-isalẹ ni ọpọlọpọ awọn aṣayan, ṣugbọn meji nikan ni o ni ibatan si ohun elo agbewọle. Aṣayan kọọkan ni faili CSV ti o yatọ.

Fi awọn olumulo kun

  • Yan aṣayan yii fun ikojọpọ olopobobo ti o rọrun ti awọn igbasilẹ olumulo.
  • Tẹ ọna asopọ Ayẹwo Gbigba lati ayelujara lati ṣii faili CSV pẹlu awọn akọle iwe fun Fikun iru awọn olumulo gbe wọle.

Ṣatunṣe awọn olumulo

  • Yan aṣayan yii lati ṣe imudojuiwọn data fun awọn olumulo lọwọlọwọ.
  • Tẹ ọna asopọ Ayẹwo Gbigba lati ayelujara lati ṣii faili CSV pẹlu awọn akọle iwe fun Yipada awọn olumulo gbe wọle iru.

Ikojọpọ faili

O le gbe faili kan silẹ lẹsẹkẹsẹ tabi pada si Fikun apoti ibanisọrọ Awọn agbewọle ni ọjọ ti o tẹle ki o pari awọn igbesẹ wọnyi.

  1. Ṣii apoti ibaraẹnisọrọ Fikun Awọn agbewọle lati ilu okeere.
  2. Tẹ orukọ alailẹgbẹ sii fun faili CSV.
  3. Tẹ aami afikun (+) lati gbe faili naa sori ẹrọ rẹ.
  4. Tẹ aaye Iru ki o yan aṣayan kan. (wo loke)
  5. Tẹ Fipamọ.

Ṣiṣẹda

Ti o da lori nọmba awọn igbasilẹ ninu faili ati fifuye eto, pẹpẹ le gba awọn iṣẹju pupọ lati ṣe ilana gbigbe wọle.

img

Nigbati agbewọle ba pari, pẹpẹ n ṣatunṣe awọn igbasilẹ olumulo (ie ṣafikun, ṣe atunṣe, tabi mu ṣiṣẹ) lori Dasibodu Awọn olumulo abojuto.

Ikuna

Ti iṣẹ agbewọle ba kuna, pẹpẹ naa da ifiranṣẹ ti o kuna pada. Ni ọpọlọpọ igba, idi ti ikuna jẹ kika data.

img

Lati wo idi ti ikuna:

  1. Tẹ Wo labẹ akọle Action.
  2. Lori oju-iwe ti o tẹle, gbe asin naa lori aami ti o kuna. Ohun elo irinṣẹ ṣe afihan aṣiṣe naa.
  3. Lọ si faili CVS ki o ṣatunṣe ati ṣatunṣe aṣiṣe naa.
  4. Po si faili lẹẹkansi.
