Bii o ṣe le Ṣẹda Awọn olumulo pẹlu Ọna asopọ Iforukọsilẹ

Nkan yii fihan awọn admins bi o ṣe le ṣẹda awọn olumulo nipa fifiranṣẹ ọna asopọ iforukọsilẹ.

Ṣiṣeto ọna asopọ iforukọsilẹ

  1. Tẹ Firanṣẹ ọna asopọ iforukọsilẹ si wọn ni isalẹ ti Fi apoti ibanisọrọ olumulo kun. Awọn ifihan oju-iwe tuntun kan.
  2. Tunto awọn aṣayan iforukọsilẹ. (wo isalẹ)

Rii daju aaye Gba Iforukọsilẹ laaye lati ṣeto si Tan-an. (beere fun) Nigbati o ba mu ṣiṣẹ, Case Active ṣe agbekalẹ ọna asopọ alailẹgbẹ kan.

Ṣe ayẹwo awọn aṣayan isalẹ ati lẹhinna tẹ aami lati daakọ ọna asopọ naa. O le gbe ọna asopọ si oju opo wẹẹbu rẹ tabi firanṣẹ ọna asopọ si ọkan tabi diẹ sii eniyan nipasẹ imeeli tabi eyikeyi ikanni ibaraẹnisọrọ.

Ṣiṣẹda ọran kan

Ni yiyan, o le ṣẹda ọran laifọwọyi fun olumulo tuntun kọọkan ati fi ipa olumulo kan sọtọ.

  1. Tẹ aaye Aifọwọyi Ṣẹda Case ko si yan Tan-an. Awọn aaye Ipa Olumulo ọran nfihan.
  2. Tẹ Ipa Olumulo ọran ki o yan aṣayan kan. (wo isalẹ)

Fifiranṣẹ ọna asopọ

Pin ọna asopọ pẹlu eniyan kan tabi diẹ sii. Olukuluku eniyan tẹ ọna asopọ naa ki o pari iṣiṣẹ iṣẹ lati mu akọọlẹ Iṣiṣẹ Nṣiṣẹ wọn ṣiṣẹ. Ni kete ti olumulo ba pari ṣiṣiṣẹ iforukosile, Iṣe Active ṣafikun olumulo si atokọ Awọn olumulo.
