Bii o ṣe le Fi, Ṣatunṣe tabi Yọ Awọn iṣẹ iyansilẹ Olumulo kuro pẹlu Irinṣẹ Wọwọle

Nkan yii fihan awọn admins bi o ṣe le ṣakoso awọn olumulo ọran pẹlu ohun elo agbewọle.

Awọn alabojuto le:

  • fi awọn olumulo si a irú
  • yipada awọn olumulo tẹlẹ sọtọ si ọran kan
  • yọ awọn olumulo sọtọ si a irú

Nsii ohun elo agbewọle

  1. Wọle si akọọlẹ abojuto.
  2. Wa ohun elo kan lori oju-iwe Awọn ohun elo ki o tẹ bọtini ADMIN naa.
  3. Tẹ taabu Integration ati lẹhinna IMPORTS lori akojọ aṣayan.
  4. Tẹ bọtini TITUN. Awọn ifihan apoti ajọṣọ Awọn agbewọle lati ilu okeere.

Ṣiṣe awọn igbasilẹ

Ṣe igbasilẹ ọkan tabi diẹ sii awọn faili CSV. Lo awọn faili wọnyi lati ṣafikun, yipada, tabi yọ awọn olumulo ọran kuro. Lẹhin apoti ibaraẹnisọrọ Fikun Awọn agbewọle lati ilu okeere ṣii:

  1. Tẹ aaye Iru.
  2. Yi lọ nipasẹ akojọ aṣayan-silẹ ko si yan aṣayan kan (wo isalẹ). A ọna asopọ han.
  3. Tẹ ọna asopọ Ayẹwo Gbigba lati ayelujara. Awọn igbasilẹ faili CSV si ẹrọ rẹ.

Ṣii faili naa ki o ṣafikun awọn igbasilẹ. Tẹ ibi lati kọ ẹkọ bi o ṣe le lo data ọna kika ninu faili CSV.

Awọn aṣayan faili

Fi awọn olumulo si awọn ọran * Yan aṣayan yii lati ṣafikun awọn olumulo si ọran kan. * Fi case_role_order cell silẹ ofo. * Lọ si oju-iwe Awọn olumulo ati daakọ ID Tag fun eniyan yii. Ti aaye ID Tag ba ṣofo, tẹ ID alailẹgbẹ sii, tẹ Fipamọ, lẹhinna da ID naa si faili CSV.

Ṣe atunṣe awọn olumulo ti a sọtọ si awọn ọran * Yan aṣayan yii lati ṣe imudojuiwọn data fun awọn olumulo lọwọlọwọ ti a yàn si ọran kan. * Lo ID Tag Case fun iwe case_ID.

Yọ awọn olumulo ti a yàn si awọn ọran * Yan aṣayan yii lati yọ awọn olumulo ti a yàn si ọran kan.

Ikojọpọ faili

O le gbe faili kan silẹ lẹsẹkẹsẹ tabi pada si Fikun apoti ibanisọrọ Awọn agbewọle ni ọjọ ti o tẹle ki o pari awọn igbesẹ wọnyi.

  1. Ṣii apoti ibaraẹnisọrọ Fikun Awọn agbewọle lati ilu okeere.
  2. Tẹ orukọ alailẹgbẹ sii fun faili CSV.
  3. Tẹ aami afikun (+) lati gbe faili naa sori ẹrọ rẹ.
  4. Tẹ aaye Iru ki o yan aṣayan kan. (wo loke)
  5. Tẹ Fipamọ.

Ṣiṣẹda

Ti o da lori nọmba awọn igbasilẹ ninu faili ati fifuye eto, pẹpẹ le gba awọn iṣẹju pupọ lati ṣe ilana gbigbe wọle.

Nigbati agbewọle ba pari, pẹpẹ n ṣatunṣe awọn igbasilẹ olumulo (ie ṣafikun, ṣe atunṣe, tabi yọkuro) lori oju-iwe Awọn ọran.

Ikuna

Ti iṣẹ agbewọle ba kuna, pẹpẹ naa da ifiranṣẹ ti o kuna pada. Ni ọpọlọpọ igba, idi ti ikuna jẹ kika data.

Lati wo idi ti ikuna:

  1. Tẹ Wo labẹ akọle Action.
  2. Ni oju-iwe ti o tẹle, gbe eku lori aami ti o kuna. Ohun elo irinṣẹ ṣe afihan aṣiṣe naa.
  3. Lọ si faili CVS ki o ṣatunṣe ati ṣatunṣe aṣiṣe naa.
  4. Po si faili lẹẹkansi.
