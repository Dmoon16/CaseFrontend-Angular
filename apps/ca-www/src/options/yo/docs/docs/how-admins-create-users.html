Bawo ni Admins Ṣẹda Awọn olumulo

Awọn alabojuto ṣẹda, yipada tabi mu ṣiṣẹ tani o le wọle si awọn ohun elo ati awọn ọran. Nigbati olumulo kan ba wọle si pẹpẹ fun igba akọkọ, wọn nilo lati pari iṣan-iṣẹ profaili kan.

  • Tẹ ibi lati ni imọ siwaju sii nipa iṣan-iṣẹ profaili olumulo tuntun.

Awọn iwifunni

Syeed naa ko fi ifitonileti ranṣẹ si awọn olumulo nigbati wọn ṣafikun ohun elo kan. Sibẹsibẹ, awọn alabojuto le mu awọn iwifunni ṣiṣẹ fun awọn iṣẹ iyansilẹ ọran.
