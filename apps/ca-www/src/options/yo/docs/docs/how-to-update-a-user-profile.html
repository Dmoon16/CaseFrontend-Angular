Bi o ṣe le ṣe imudojuiwọn Profaili olumulo kan

Nkan yii fihan awọn olumulo bi o ṣe le ṣe imudojuiwọn alaye lori oju-iwe Profaili wọn.

  • Awọn olumulo le ṣe imudojuiwọn awọn profaili tiwọn.
  • Awọn alabojuto le ṣe imudojuiwọn oju-iwe Profaili ti eyikeyi olumulo.

Fun awọn olumulo

Igbesẹ akọkọ ni lati ṣii oju-iwe Profaili. Awọn olumulo ati awọn alabojuto tẹle awọn igbesẹ wọnyi lati ṣe imudojuiwọn oju-iwe Profaili tiwọn.

  1. Wọle si akọọlẹ Iṣiṣẹ Nṣiṣẹ rẹ.
  2. Tẹ Profaili taabu ni oke ti oju-iwe naa.
  3. Ṣe awọn ayipada.
  4. Tẹ Fipamọ.

Pupọ awọn aaye jẹ iyan, ati diẹ ninu awọn aaye nilo. Awọn aaye ti a beere ni:

  • Orukọ akọkọ
  • Oruko idile
  • Adirẹsi imeeli
  • Ede
  • Aago aago

Fun admins

Awọn alabojuto tun le ṣe imudojuiwọn alaye profaili fun awọn olumulo miiran. Eyi pẹlu awọn aaye mẹta labẹ akọle ALAYE OLUMULO HOST.

  1. Wọle si akọọlẹ alabojuto ọran ti nṣiṣe lọwọ.
  2. Wa ohun elo kan ki o tẹ bọtini ADMIN.
  3. Tẹ Awọn olumulo lori akojọ aṣayan oke.
  4. Lọ si akojọ aṣayan-ipin ko si yan atokọ olumulo ti o fẹ wọle si (IṢẸ tabi alaabo). Eto aiyipada naa nṣiṣẹ.
  5. Wa olumulo kan lori atokọ ki o tẹ Ṣatunkọ.
  6. Ṣe awọn ayipada bi o ṣe nilo. (wo isalẹ)
  7. Tẹ Fipamọ.

Yi alaye olumulo pada

Awọn alabojuto le yi alaye olumulo pada.

  1. Ṣeto Iru Olumulo (Olumulo tabi Alakoso).
  2. Ṣeto Ipo Olumulo (Nṣiṣẹ lati Disabled).
  3. Tẹ ID Tag alailẹgbẹ kan sii. (aṣayan)
  4. Tẹ Fipamọ.
