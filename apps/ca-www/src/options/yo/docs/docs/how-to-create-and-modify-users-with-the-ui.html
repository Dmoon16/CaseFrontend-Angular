Bii o ṣe le Ṣẹda ati Ṣatunṣe Awọn olumulo pẹlu UI

Nkan yii fihan awọn abojuto bi o ṣe le ṣẹda awọn olumulo ati awọn igbasilẹ imudojuiwọn pẹlu wiwo olumulo pẹpẹ. Awọn alabojuto tun le ṣafikun awọn olumulo si ọran tuntun ati ṣatunkọ awọn ọran ti o wa tẹlẹ.

Ṣiṣẹda olumulo kan

  1. Wọle si akọọlẹ abojuto.
  2. Wa ohun elo kan lori oju-iwe Awọn ohun elo ki o tẹ bọtini ADMIN naa.
  3. Lori awọn Abojuto nronu, tẹ awọn olumulo taabu taabu.
  4. Tẹ bọtini TITUN. A titun window han.

Ṣatunṣe olumulo kan

Lati ṣe imudojuiwọn nọmba kekere ti awọn igbasilẹ olumulo, lo ohun elo Ṣatunkọ ati ṣe awọn ayipada si olumulo kọọkan.

  • Akiyesi Lati ṣe atunṣe nọmba nla ti awọn igbasilẹ olumulo ni ijoko kan, ronu nipa lilo ohun elo agbewọle .
  1. Lori awọn Abojuto nronu, tẹ awọn olumulo taabu taabu.
  2. Wa olumulo kan ki o tẹ Ṣatunkọ.
  3. Ṣe awọn ayipada ki o tẹ Fipamọ.
