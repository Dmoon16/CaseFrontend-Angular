Bawo ni lati Ṣẹda ohun App

Nkan yii fihan awọn olumulo tuntun bi o ṣe le ṣẹda akọọlẹ idanwo ọfẹ kan. Case Active laifọwọyi funni ni awọn anfani abojuto olumulo titun fun akọọlẹ yii.

Ṣiṣẹda akọọlẹ kan

  1. Lọ si oju-iwe iwọle .
  2. Tẹ bọtini Bẹrẹ.
  3. Tẹ adirẹsi imeeli sii.
  4. Tẹ Itele.

Ṣiṣayẹwo akọọlẹ kan

  1. Ṣii taabu aṣawakiri tuntun kan ki o wọle si iwe apamọ imeeli rẹ.
  2. Da koodu oni-nọmba mẹfa naa.
  3. Pada si taabu Nṣiṣẹ lọwọ.
  4. Lẹẹmọ koodu naa sinu awọn aaye. A titun window han.

Ninu ferese tuntun:

  1. Fọwọsi awọn aaye ti a beere.
  2. Ka awọn ofin iṣẹ ati ṣayẹwo apoti lati gba.
  3. Tẹ Ṣẹda iroyin ỌFẸ.

Ipari iṣeto app

Awọn ifihan iṣan-iṣẹ ni oju-iwe atẹle. Pari awọn igbesẹ lati pari iṣeto ohun elo rẹ. O le nigbagbogbo pada si awọn oju-iwe wọnyi lati ṣatunkọ tabi imudojuiwọn alaye tabi ipo akọọlẹ rẹ.

Lori oju-iwe About App, fọwọsi orukọ ile-iṣẹ ati awọn aaye oju opo wẹẹbu (ti a beere) ati lẹhinna tẹ Itele.

Lori oju-iwe Alaye Ìdíyelé, tẹ aaye Eto Isanwo ati ṣayẹwo awọn aṣayan. Eto aiyipada jẹ Ọfẹ. Tẹ SUBMIT.

Lẹhin ti o fi fọọmu naa silẹ, Case Active ṣe afihan oju-iwe Awọn ohun elo naa. Fun ohun elo kọọkan ti o ṣẹda tabi jẹ ti, oju-iwe naa fihan:

  1. Orukọ app naa
  2. Iru rẹ (ipa olumulo)
  3. Ipo

img

Oju-iwe Awọn ohun elo jẹ aaye ifilọlẹ fun ọpọlọpọ awọn iṣẹ abojuto:

  1. Tẹ bọtini APP lati baraẹnisọrọ, fọwọsi awọn iwe aṣẹ, iwiregbe fidio (ati diẹ sii) lori awọn ọran ti o jọmọ ọran.
  2. Tẹ bọtini ADMIN lati ṣii nronu abojuto. Eyi ni ibi ti Awọn alabojuto ṣakoso awọn olumulo, awọn ọran, awọn iṣọpọ, ati diẹ sii.
  3. Tẹ ellipsis lati ṣakoso ìdíyelé iroyin ati awọn eto iwifunni.
  • Akiyesi Case Active ṣe afihan bọtini ADMIN fun awọn olumulo alabojuto. Awọn olumulo ti ko ni awọn anfani abojuto ko rii bọtini ADMIN (bii o han ni iboju ni isalẹ).
