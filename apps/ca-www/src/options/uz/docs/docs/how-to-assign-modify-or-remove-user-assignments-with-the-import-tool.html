Import vositasi yordamida foydalanuvchi topshiriqlarini qanday belgilash, o'zgartirish yoki o'chirish

Ushbu maqola administratorlarga Import vositasi yordamida ishi foydalanuvchilarini qanday boshqarishni ko'rsatadi.

Adminlar:

  • foydalanuvchilarni ishga tayinlash
  • ishi uchun allaqachon tayinlangan foydalanuvchilarni o'zgartirish
  • ish uchun tayinlangan foydalanuvchilarni olib tashlash

Import vositasini ochish

  1. Administrator hisobiga kiring.
  2. Ilovalar sahifasida ilovani toping va ADMIN tugmasini bosing.
  3. Integratsiya yorlig'ini, keyin esa pastki menyuda IMPORTS ni bosing.
  4. YANGI tugmasini bosing. Importlarni qo'shish dialog oynasi paydo bo'ladi.

Yozuvlarni yaratish

Bir yoki bir nechta CSV fayllarini yuklab oling. Ish foydalanuvchilarini qo'shish, o'zgartirish yoki olib tashlash uchun ushbu fayllardan foydalaning. Importlarni qo'shish dialog oynasi ochilgandan so'ng:

  1. Tur maydonini bosing.
  2. Ochiladigan menyu bo'ylab aylantiring va variantni tanlang (pastga qarang). Havola ko'rsatiladi.
  3. Namuna yuklab olish havolasini bosing. CSV fayli qurilmangizga yuklanadi.

Faylni oching va yozuvlarni qo'shing. CSV faylida format ma'lumotlaridan qanday foydalanishni o'rganish uchun shu yerni bosing.

Fayl parametrlari

Foydalanuvchilarni holatlarga tayinlash * Ishga foydalanuvchi qo‘shish uchun ushbu parametrni tanlang. * Case_role_order katakchasini bo'sh qoldiring. * Foydalanuvchilar sahifasiga o'ting va ushbu shaxs uchun teg identifikatoridan nusxa oling. Agar teg identifikatori maydoni bo'sh bo'lsa, noyob identifikatorni kiriting, Saqlash tugmasini bosing va keyin IDni CSV fayliga nusxalang.

Fayl yuklanmoqda

Siz darhol faylni yuklashingiz yoki keyinroq Importlarni qo'shish dialog oynasiga qaytishingiz va ushbu amallarni bajarishingiz mumkin.

  1. Importlarni qo'shish dialog oynasini oching.
  2. CSV fayli uchun noyob nom kiriting.
  3. Faylni qurilmangizdan yuklash uchun ortiqcha (+) belgisini bosing.
  4. Tur maydonini bosing va variantni tanlang. (yuqoriga qarang)
  5. SAQLASH tugmasini bosing.

Qayta ishlash

Fayl va tizim yukidagi yozuvlar soniga qarab, platforma importni qayta ishlash uchun bir necha daqiqa vaqt olishi mumkin.

img

Import tugagach, platforma Ishlar sahifasida foydalanuvchi yozuvlarini moslashtiradi (ya'ni, qo'shadi, o'zgartiradi yoki o'chiradi).

Muvaffaqiyatsizlik

Import operatsiyasi bajarilmasa, platforma Muvaffaqiyatsiz xabarni qaytaradi. Ko'pgina hollarda, muvaffaqiyatsizlik sababi ma'lumotlarni formatlashdir.

Muvaffaqiyatsizlik sababini ko'rish uchun:

  1. Harakat sarlavhasi ostidagi Ko'rish-ni bosing.
  2. Keyingi sahifada sichqonchani Muvaffaqiyatsiz belgisiga olib boring. Asboblar maslahati xatoni ko'rsatadi.
  3. CVS fayliga o'ting va xatoni sozlang va tuzating.
  4. Faylni qayta yuklang.
