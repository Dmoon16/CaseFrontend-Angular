Import vositasi yordamida foydalanuvchilarni qanday yaratish yoki o'zgartirish mumkin

Ushbu maqola administratorlarga Import vositasi yordamida foydalanuvchilarni qanday yaratishni ko'rsatadi.

Import vositasini ochish

Administratorlar yangi foydalanuvchilarni ommaviy yuklash va mavjud foydalanuvchi yozuvlarini oʻzgartirish uchun Import vositasidan foydalanishlari mumkin.

  1. Administrator hisobiga kiring.
  2. Ilovalar sahifasida ilovani toping va ADMIN tugmasini bosing.
  3. Administrator panelida Integratsiya yorlig'ini va keyin pastki menyuda IMPORTS ni bosing.
  4. YANGI tugmasini bosing. Importlarni qo'shish dialog oynasi yangi brauzer yorlig'ida ko'rsatiladi.

img

Yozuvlarni yaratish

Importlarni qo'shish dialog oynasi ochilgandan so'ng:

  1. Tur maydonini bosing va variantni tanlang (pastga qarang). Havola ko'rsatiladi.
  2. Namuna yuklab olish havolasini bosing. CSV fayli qurilmangizga yuklanadi.

Keyin faylni oching va yozuvlarni qo'shing. CSV faylida format ma'lumotlaridan qanday foydalanishni o'rganish uchun shu yerni bosing .

img

Turi

Tur maydonini bosganingizda, ochiladigan menyu ko'rsatiladi. Ommaviy import qilish uchun ikkita variantdan birini tanlang.

:undov belgisi: DIQQAT Ochiladigan menyuda ko'p variantlar mavjud, ammo faqat ikkitasi Import vositasi bilan bog'liq. Har bir variantda boshqa CSV fayl namunasi mavjud.

Foydalanuvchilarni qo'shish

  • Foydalanuvchi yozuvlarini oddiy ommaviy yuklash uchun ushbu parametrni tanlang.
  • Foydalanuvchilarni qo‘shish Import turi uchun ustun sarlavhalari bilan CSV faylni ochish uchun Namunani yuklab olish havolasini bosing.

Foydalanuvchilarni o'zgartirish

  • Joriy foydalanuvchilar uchun ma'lumotlarni yangilash uchun ushbu parametrni tanlang.
  • Foydalanuvchilarni import qilish turini o‘zgartirish uchun ustun sarlavhalari bilan CSV faylini ochish uchun Namunani yuklab olish havolasini bosing.

Fayl yuklanmoqda

Siz darhol faylni yuklashingiz yoki keyinroq Importlarni qo'shish dialog oynasiga qaytishingiz va ushbu amallarni bajarishingiz mumkin.

  1. Importlarni qo'shish dialog oynasini oching.
  2. CSV fayli uchun noyob nom kiriting.
  3. Faylni qurilmangizdan yuklash uchun ortiqcha (+) belgisini bosing.
  4. Tur maydonini bosing va variantni tanlang. (yuqoriga qarang)
  5. SAQLASH tugmasini bosing.

img

Qayta ishlash

Fayl va tizim yukidagi yozuvlar soniga qarab, platforma importni qayta ishlash uchun bir necha daqiqa vaqt olishi mumkin.

img

Import tugagach, platforma administratorning " Foydalanuvchilar " boshqaruv panelidagi foydalanuvchi yozuvlarini sozlaydi (ya'ni qo'shadi, o'zgartiradi yoki o'chiradi).

Muvaffaqiyatsizlik

Import operatsiyasi bajarilmasa, platforma Muvaffaqiyatsiz xabarni qaytaradi. Ko'pgina hollarda, muvaffaqiyatsizlik sababi ma'lumotlarni formatlashdir.

img

Muvaffaqiyatsizlik sababini ko'rish uchun:

  1. Harakat sarlavhasi ostidagi Ko'rish -ni bosing.
  2. Keyingi sahifada sichqonchani Muvaffaqiyatsiz belgisiga olib boring. Asboblar maslahati xatoni ko'rsatadi.
  3. CVS fayliga o'ting va xatoni sozlang va tuzating.
  4. Faylni qayta yuklang.
