Foydalanuvchi profilini qanday yangilash kerak

Ushbu maqola foydalanuvchilarga o'zlarining Profil sahifasidagi ma'lumotlarni qanday yangilashlarini ko'rsatadi.

  • Foydalanuvchilar o'z profillarini yangilashlari mumkin.
  • Administratorlar istalgan foydalanuvchining profil sahifasini yangilashi mumkin.

Foydalanuvchilar uchun

Birinchi qadam profil sahifasini ochishdir. Foydalanuvchilar va administratorlar o ªzlarining profil sahifalarini yangilash uchun quyidagi amallarni bajaradilar.

  1. Case Active hisobingizga kiring.
  2. Sahifaning yuqori qismidagi Profil yorlig'ini bosing.
  3. O'zgartirishlar kiriting.
  4. Saqlash tugmasini bosing.

Ko'pgina maydonlar ixtiyoriy va ba'zi maydonlar talab qilinadi. Majburiy maydonlar:

  • Ism
  • Familiya
  • E-pochta manzili
  • Til
  • Vaqt zonasi

Adminlar uchun

Administratorlar boshqa foydalanuvchilar uchun profil ma'lumotlarini ham yangilashlari mumkin. Bunga HOST USER INFO sarlavhasi ostidagi uchta maydon kiradi.

  1. Case Active administrator hisobingizga kiring.
  2. Ilovani toping va ADMIN tugmasini bosing.
  3. Yuqori menyuda "Foydalanuvchilar" ni bosing.
  4. Pastki menyuga o'ting va o'zingiz foydalanmoqchi bo'lgan foydalanuvchilar ro'yxatini tanlang (FAOL yoki NOGIRON). Standart sozlama Faol.
  5. Ro'yxatdagi foydalanuvchini toping va Tahrirlash tugmasini bosing.
  6. Zarur bo'lganda o'zgarishlar qiling. (pastga qarang)
  7. Saqlash tugmasini bosing.

Foydalanuvchi ma'lumotlarini o'zgartirish

Administrator foydalanuvchi ma'lumotlarini o'zgartirishi mumkin.

  1. Foydalanuvchi turini o'rnating (foydalanuvchi yoki menejer).
  2. Foydalanuvchi holatini o'rnating (faoldan o'chirilgan).
  3. Noyob teg identifikatorini kiriting. (ixtiyoriy)
  4. Saqlash tugmasini bosing.
