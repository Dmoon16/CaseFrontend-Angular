Ifformattjar tad-Data tal-Ġestjoni tal-Każijiet bl-Għodda tal-Importazzjoni

Hawn taħt hawn lista ta' oqsma meħtieġa f'fajls CSV biex jiġu mmaniġġjati l-każijiet bl-għodda tal-Importazzjoni.

username

  • Daħħal l-indirizz elettroniku tal-utent.

tag_id (id li tista' tirreferi aktar tard)

  • identifikatur uniku (uża karattri alfanumeriċi, is-singi huma aċċettati)

case_status

  • 0 = magħluq
  • 1 = attiv

data_ftuħ

  • xahar/jum/sena
  • mm/jj/ssss

dwar

  • deskrizzjoni unika tal-każ (uża karattri alfanumeriċi, is-singi huma aċċettati)
