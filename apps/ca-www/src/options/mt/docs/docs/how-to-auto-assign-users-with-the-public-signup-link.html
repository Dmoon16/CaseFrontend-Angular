Kif Tassenja l-Utenti Awtomatikament bil-Pubbliku Signup Link

L-amministraturi jitgħallmu kif jikkonfiguraw is-settings tas-sistema sabiex utenti ġodda jiġu assenjati awtomatikament għal każ li għadu kif inħoloq awtomatikament.

Ftuħ il-paġna tas-setup

L-ewwel pass huwa li tiftaħ il-paġna Systems Setup.

  1. Idħol fil-kont tal-amministratur.
  2. Sib app fuq il-paġna Apps u kklikkja l-buttuna ADMIN.
  3. Ikklikkja t-tab Sistemi u mbagħad SETUP SISTEMA fis-submenu.
  4. Skrollja 'l isfel għall-intestatura SIGNUP UTENTI.

Konfigurazzjoni tas-settings tas-sistema

  1. Issettja l-menus drop-down Allow Signup u Auto Create Case fuq Fuq.
  2. Ikklikkja l-menu tar-rwol u agħżel għażla (default huwa rakkomandat).
  3. Ikkopja u aqsam il-link għall-Iskrizzjoni tal-Utent Pubbliku.

Tibgħat il-link

Kull utent li jikklikkja l-link u jlesti l-iskrizzjoni jqajjem il-pjattaforma biex tiġġenera każ ġdid fuq il-paġna Każijiet u jassenja l-utent awtomatikament f'dak il-każ.
