Como iniciar sesión

Aprende a iniciar sesión na túa conta de Case Active existente.

Iniciando sesión

  1. Vaia á páxina de inicio de sesión.
  2. Fai clic no botón LOGIN.
  3. Introduce o teu correo electrónico.
  4. Fai clic en SEGUINTE.

Verificando unha conta

  1. Abre unha nova pestana do navegador e inicia sesión na túa conta de correo electrónico.
  2. Copia o código de seis díxitos.
  3. Volve á pestana Caso activo.
  4. Pega o código nos campos. Aparece unha nova xanela.

Case Active mostra a páxina de aplicacións.

  • Nota A plataforma mostra un botón ADMINISTRATIVO xunto ao botón APP para os usuarios con privilexios de administrador.
