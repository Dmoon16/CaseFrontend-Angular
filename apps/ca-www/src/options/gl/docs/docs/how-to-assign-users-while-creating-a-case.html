Como asignar usuarios ao crear un caso

Este artigo mostra aos administradores como asignar un usuario ao crear un novo caso. Se o usuario aínda non está rexistrado na plataforma, pode crear engadir un novo usuario mentres crea un novo caso.

Creando un caso

  1. Inicie sesión na conta de administrador.
  2. Localiza unha aplicación na páxina Aplicacións e fai clic no botón ADMINISTRADOR.
  3. Fai clic na pestana do menú Casos.
  4. Fai clic no botón NOVO. Aparece unha nova xanela.
  • Encha os campos para crear un caso e, a continuación, desprácese ata o encabezado Usuarios asignados.

Asignación dun usuario

Usuario actual

Siga estes pasos para asignar un usuario actual ao novo caso.

Fai clic no botón ASIGNAR USUARIOS e enche os campos.

  1. Fai clic no menú despregable Usuario e selecciona un identificador (por exemplo, correo electrónico).
  2. Introduza un valor no campo en branco (por exemplo, enderezo de correo electrónico).
  3. Se o rexistro de usuario existe na plataforma, o campo suxire automaticamente un nome.
  4. Seleccione o usuario correcto.
  5. Seleccione un rol de usuario (clientes, predeterminado ou persoal).
  6. Opcionalmente, marque a caixa para enviar ao usuario unha notificación por correo electrónico.
  7. Fai clic en GARDAR.

Case Active engade o usuario ao caso. Agora os administradores poden facer clic en Gardar para pechar a xanela, engadir un novo usuario ou eliminar un usuario.

Engadindo un novo usuario

Se a plataforma non pode atopar un rexistro actual, fai clic no botón para engadir un novo usuario.

Encha os campos e despois fai clic en GARDAR para engadir o usuario á plataforma. A continuación, establece un rol de usuario e fai clic de novo en GARDAR para asignar o usuario ao caso. Finalmente, fai clic en GARDAR unha vez máis para gardar o caso.
