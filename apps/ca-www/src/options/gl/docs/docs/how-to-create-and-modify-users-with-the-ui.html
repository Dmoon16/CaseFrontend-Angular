Como crear e modificar usuarios coa IU

Este artigo mostra aos administradores como crear usuarios e actualizar rexistros coa interface de usuario da plataforma. Os administradores tamén poden engadir usuarios a un novo caso e editar os existentes.

Creando un usuario

  1. Inicie sesión na conta de administrador.
  2. Localiza unha aplicación na páxina Aplicacións e fai clic no botón ADMINISTRADOR.
  3. No panel de administración, faga clic na pestana do menú Usuarios.
  4. Fai clic no botón NOVO. Aparece unha nova xanela.

Completa os campos e fai clic en GARDAR. Case Active engade o nome á lista de usuarios asociados con esta aplicación.

Modificando un usuario

Para actualizar un pequeno número de rexistros de usuarios, use a ferramenta Editar e realice cambios en cada usuario individual.

  • Nota Para modificar un gran número de rexistros de usuario nunha sesión, considere usar a ferramenta Importar .
  1. No panel de administración, faga clic na pestana do menú Usuarios.
  2. Localiza un usuario e fai clic en Editar.
  3. Fai cambios e fai clic en GARDAR.

Próximos pasos

Case Active non envía ao usuario unha notificación por correo electrónico. Os administradores deben poñerse en contacto co usuario e enviarlle unha ligazón á páxina de inicio de Case Active. A continuación, o usuario fai clic no botón LOGIN e completa o fluxo de traballo para novos rexistros.
