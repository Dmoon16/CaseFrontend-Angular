Como xestionan os usuarios os administradores

Os administradores crean, modifican ou desactivan quen pode acceder a aplicacións e casos. Cando un usuario inicia sesión na plataforma por primeira vez, debe completar un fluxo de traballo de perfil.

  • Fai clic aquí para obter máis información sobre o fluxo de traballo do novo perfil de usuario.

Xestionar un usuario

A seguinte táboa mostra cinco formas de xestionar un usuario. Fai clic nunha ligazón para saber como.

Método Tipo Accións Detalles
Usando UI IU engadir e modificar Botón Administrador > Usuarios
Ao crear un caso e asignar usuario a caso IU engadir Botón de administración > Casos > Botón Novo > páxina Crear caso
Creación automática de usuario mediante a ligazón de rexistro público Acción do usuario engadir Botón de administración > Configuración > páxina Configuración do sistema
Ferramenta de importación IU engadir e modificar Botón de administración > Integracións > Importacións
Usando API (próximamente) API engadir e modificar N / A

Notificacións

A plataforma non envía aos usuarios unha notificación cando se engaden a unha aplicación. Non obstante, os administradores poden activar as notificacións para as asignacións de casos.
