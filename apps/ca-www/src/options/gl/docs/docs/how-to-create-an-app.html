Como crear unha aplicación

Este artigo mostra aos novos usuarios como crear unha conta de proba gratuíta. Case Active concede automaticamente ao novo usuario privilexios de administrador para esta conta.

Creando unha conta

  1. Vaia á páxina de inicio de sesión.
  2. Fai clic no botón Comezar.
  3. Introduza un enderezo de correo electrónico.
  4. Fai clic en Seguinte.

img

Verificando unha conta

  1. Abre unha nova pestana do navegador e inicia sesión na túa conta de correo electrónico.
  2. Copia o código de seis díxitos.
  3. Volve á pestana Caso activo.
  4. Pega o código nos campos. Aparece unha nova xanela.

Na nova xanela:

  1. Encha os campos obrigatorios.
  2. Le as condicións do servizo e marca a caixa para aceptar.
  3. Fai clic en CREAR UNHA CONTA GRATIS.

Completando a configuración da aplicación

Na páxina seguinte móstrase un fluxo de traballo. Completa os pasos para rematar de configurar a túa aplicación. Sempre podes volver a estas páxinas para editar ou actualizar a información ou o estado da túa conta.

Na páxina Acerca da aplicación, encha os campos do nome da empresa e do sitio web (obrigatorio) e, a continuación, fai clic en Seguinte.

img

Na páxina Información de facturación, faga clic no campo Plan de pago e revise as opcións. A configuración predeterminada é Gratis. Fai clic en ENVIAR.

img

Despois de enviar o formulario, Case Active mostra a páxina de aplicacións. Para cada aplicación que crees ou á que pertences, a páxina mostra:

  1. O nome da aplicación
  2. O teu tipo (función de usuario)
  3. Estado

A páxina de aplicacións é un punto de inicio para moitas funcións de administración:

  1. Fai clic no botón APP para comunicarte, encher documentos, chat de vídeo (e moito máis) sobre problemas relacionados cos casos.
  2. Fai clic no botón ADMINISTRADOR para abrir o panel de administración. Aquí é onde os administradores xestionan usuarios, casos, integracións e moito máis.
  3. Fai clic nos puntos suspensivos para xestionar a configuración de notificacións e facturación da conta.
  • Nota Case Active mostra o botón ADMIN para os usuarios administradores. Os usuarios que non teñen privilexios de administrador non ven o botón ADMINISTRADOR (como se mostra na pantalla de abaixo).
