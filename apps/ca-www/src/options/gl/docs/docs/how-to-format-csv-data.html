Como formatar datos CSV

A continuación móstrase unha lista de campos de datos en ficheiros CSV. Cada opción de importación ten unha combinación diferente de campos.

nome de usuario

  • Introduza o enderezo de correo electrónico do usuario.

nome_denominado

  • Introduza o nome do usuario.

apelido

  • Introduza o apelido do usuario.

local

  • Esta columna verifica o idioma preferido do usuario.
  • Introduza en para inglés, sp para español.

zona info

  • Isto establece a zona horaria do usuario. Introduce o continente e a cidade sen espazos (utiliza un guión baixo).
  • Debe utilizar unha opción no menú despregable Perfil > IDIOMA E REXION > Fuso horario.
  • Usa o fuso horario que mellor se axusta. Por exemplo, Boston non está no menú despregable, polo que podemos usar Nova York (ou calquera cidade da zona horaria EST/EDT).

host_tag_id

  • Introduce unha etiqueta permanente.
  • az, 0-9, -, / están permitidos.

caso_estado

  • 0 = pechado
  • 1 = activo
