Como os usuarios crean unha conta nova

Despois de que un administrador engade un usuario a un caso: * se o usuario é novo en caseactive, o usuario debe completar un fluxo de traballo para crear unha conta e aceptar a invitación para acceder ao caso. * se o usuario ten unha conta activa de casos, o usuario debe aceptar a invitación para acceder ao caso.

Este artigo describe os pasos que debe completar un usuario para crear a súa conta.

Comezando

Hai dúas formas de engadir un usuario a unha aplicación caseactiva

  • Por administrador: se o administrador crea un usuario a través da IU da plataforma ou da ferramenta de importación , o administrador debe enviarlle unha ligazón a account.caseactive.com para iniciar sesión.
  • Por usuario: se o usuario creou unha conta mediante unha ligazón de rexistro , o usuario será automaticamente redirixido a account.caseactive.com para iniciar sesión.

Iniciando sesión

Cando o usuario inicia sesión na súa conta CaseActive por primeira vez, a plataforma solicita ao usuario que:

  • abra a súa conta de correo electrónico e despois copie/pegue o código de seguridade de seis díxitos
  • completar un perfil de conta

img

Despois de que o usuario faga clic en ACTUALIZAR, móstrase a páxina Aplicacións. A páxina mostra o nome da aplicación (nome da empresa), o tipo de usuario (función), o estado e un botón de INVITACIÓN.

Para acceder á aplicación, o usuario debe aceptar a invitación. O usuario debe:

  1. Fai clic no botón INVITACIÓN.
  2. Revisa os permisos.
  3. Fai clic na pestana Notificacións para revisar e editar (opcional) os termos.
  4. Fai clic en APROBAR ACCESO.

Cando o usuario aproba, a plataforma mostra un botón APP na páxina do usuario. Este botón dálle ao usuario acceso aos casos asignados.

Para editar a configuración do usuario, o usuario fai clic nos puntos suspensivos.

img

O usuario pode: * eliminar o acceso á aplicación * modificar a configuración de notificacións

img

Despois de que o usuario complete o fluxo de traballo, o nome de usuario aparece na lista de usuarios do administrador. A lista mostra todos os usuarios, iniciados ou non.
