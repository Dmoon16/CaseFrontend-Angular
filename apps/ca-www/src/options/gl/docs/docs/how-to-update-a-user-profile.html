Como actualizar un perfil de usuario

Este artigo mostra aos usuarios como actualizar a información na súa páxina de perfil.

  • Os usuarios poden actualizar os seus propios perfís.
  • Os administradores poden actualizar a páxina de perfil de calquera usuario.

Para usuarios

O primeiro paso é abrir a páxina de perfil. Os usuarios e administradores seguen estes pasos para actualizar a súa propia páxina de perfil.

  1. Inicie sesión na súa conta de Case Active.
  2. Fai clic na pestana Perfil na parte superior da páxina.
  3. Fai cambios.
  4. Fai clic en Gardar.

A maioría dos campos son opcionais e algúns son obrigatorios. Os campos obrigatorios son:

  • Nome
  • Apelido
  • Enderezo electrónico
  • Lingua
  • Franxa horaria

Para administradores

Os administradores tamén poden actualizar a información do perfil doutros usuarios. Isto inclúe tres campos baixo o título INFORMACIÓN DO USUARIO DO HOST.

  1. Inicie sesión na súa conta de administrador de Case Active.
  2. Localiza unha aplicación e fai clic no botón ADMINISTRADOR.
  3. Fai clic en Usuarios no menú superior.
  4. Vaia ao submenú e seleccione a lista de usuarios á que desexa acceder (ACTIVA ou DESACTIVADA). A configuración predeterminada é Activo.
  5. Localiza un usuario na lista e fai clic en Editar.
  6. Fai os cambios necesarios. (Ver abaixo)
  7. Fai clic en Gardar.

Cambiar a información do usuario

Os administradores poden cambiar a información do usuario.

  1. Establece o tipo de usuario (usuario ou xestor).
  2. Establece o estado do usuario (Activo en Desactivado).
  3. Introduza un ID de etiqueta único. (opcional)
  4. Fai clic en Gardar.
