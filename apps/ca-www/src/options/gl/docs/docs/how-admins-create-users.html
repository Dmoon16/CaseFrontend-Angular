Como os administradores crean usuarios

Os administradores crean, modifican ou desactivan quen pode acceder a aplicacións e casos. Cando un usuario inicia sesión na plataforma por primeira vez, debe completar un fluxo de traballo de perfil.

  • Fai clic aquí para obter máis información sobre o fluxo de traballo do novo perfil de usuario.

Creando un usuario

A seguinte táboa mostra cinco formas de crear un usuario. Fai clic nunha ligazón para saber como.

| Método | Tipo | Accións | Detalles | O novo usuario recibe unha notificación? |
|--------------------------|----------------|------- ------|--------------------------------------------- ------------------------|

Usando UI | IU | engadir e modificar | Botón Admin > Usuarios | Non |

Ao crear un caso e asignar un usuario ao caso | IU | engadir | Botón Administrador > Casos > Botón Novo > Crear páxina de caso | Si, se está activado |

Creación automática de usuario mediante a ligazón de rexistro público | Acción do usuario | engadir | Botón Admin > Configuración > Páxina de configuración do sistema | Si, se a creación automática de maiúsculas e minúsculas está activada |

Ferramenta de importación | IU | engadir e modificar | Botón de administración > Integracións > Importacións | Non |

Usando API (próximamente) | API | engadir e modificar | na | na |

Notificacións

A plataforma non envía unha notificación aos usuarios cando se engaden a unha aplicación. Non obstante, os administradores poden activar as notificacións para as asignacións de casos.
