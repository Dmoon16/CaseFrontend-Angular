Como asignar automaticamente usuarios coa ligazón de rexistro público

Os administradores aprenden a configurar os axustes do sistema para que os novos usuarios se asignen automaticamente a un caso recentemente creado.

Abrindo a páxina de configuración

O primeiro paso é abrir a páxina de configuración do sistema.

  1. Inicie sesión na conta de administrador.
  2. Localiza unha aplicación na páxina Aplicacións e fai clic no botón ADMINISTRADOR.
  3. Fai clic na pestana Sistemas e logo CONFIGURACIÓN DO SISTEMA no submenú.
  4. Desprácese cara abaixo ata o título REGISTRO DE USUARIO.

Configurando os axustes do sistema

  1. Establece os menús despregables Permitir rexistro e Creación automática de casos en Activado.
  2. Fai clic no menú de funcións e escolla unha opción (recoméndase a opción predeterminada).
  3. Copia e comparte a ligazón de rexistro de usuario público.

Enviando a ligazón

Cada usuario que fai clic na ligazón e completa o rexistro activa a plataforma para xerar un novo caso na páxina Casos e asigna automaticamente o usuario nese caso.
