Käyttäjien luominen ja muokkaaminen käyttöliittymällä

Tämä artikkeli näyttää järjestelmänvalvojille, kuinka luoda käyttäjiä ja päivittää tietueita alustan käyttöliittymän avulla. Järjestelmänvalvojat voivat myös lisätä käyttäjiä uuteen tapaukseen ja muokata olemassa olevia tapauksia.

Käyttäjän luominen

  1. Kirjaudu sisään järjestelmänvalvojan tilille.
  2. Etsi sovellus Sovellukset-sivulta ja napsauta JÄRJESTELMÄNVALVOJA-painiketta.
  3. Napsauta hallintapaneelissa Käyttäjät-valikkovälilehteä.
  4. Napsauta UUSI-painiketta. Uusi ikkuna tulee näkyviin.

Täytä kentät ja napsauta TALLENNA. Case Active lisää nimen tähän sovellukseen liittyvien käyttäjien luetteloon.

img

Käyttäjän muokkaaminen

Jos haluat päivittää pienen määrän käyttäjätietueita, käytä Muokkaa-työkalua ja tee muutoksia jokaiseen yksittäiseen käyttäjään.

  • Huomautus Jos haluat muokata suurta määrää käyttäjätietueita kerralla, harkitse Tuo-työkalun käyttöä.
  1. Napsauta hallintapaneelissa Käyttäjät-valikkovälilehteä.
  2. Etsi käyttäjä ja napsauta Muokkaa.
  3. Tee muutokset ja napsauta TALLENNA.

img

Seuraavat vaiheet

Case Active ei lähetä käyttäjälle sähköposti-ilmoitusta. Järjestelmänvalvojien tulee ottaa yhteyttä käyttäjään ja lähettää hänelle linkki Case Activen kotisivulle. Tämän jälkeen käyttäjä napsauttaa LOGIN-painiketta ja suorittaa uusien kirjautumisten työnkulun.

  • Tässä artikkelissa kuvataan uusien käyttäjien kirjautumisen työnkulku.
