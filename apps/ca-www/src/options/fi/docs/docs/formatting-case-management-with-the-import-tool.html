Tapausten hallintatietojen muotoilu tuontityökalulla

Alla on luettelo pakollisista kentistä CSV-tiedostoissa tapausten hallintaa varten tuontityökalulla.

käyttäjätunnus

  • Anna käyttäjän sähköpostiosoite.

tag_id (tunnus, johon voit viitata myöhemmin)

  • yksilöllinen tunniste (käytä aakkosnumeerisia merkkejä, väliviivat hyväksytään)

case_status

  • 0 = suljettu
  • 1 = aktiivinen

päivämäärä_avaaminen

  • kuukausi päivä Vuosi
  • kk/pp/vvvv

noin

  • yksilöllinen kuvaus tapauksesta (käytä aakkosnumeerisia merkkejä, väliviivat hyväksytään)
