Kuinka luoda sovellus

Tämä artikkeli näyttää uusille käyttäjille ilmaisen kokeilutilin luomisen. Case Active myöntää automaattisesti uudelle käyttäjälle järjestelmänvalvojan oikeudet tälle tilille.

Tilin luominen

  1. Mene kirjautumissivulle .
  2. Napsauta Aloita-painiketta.
  3. Syötä sähköpostiosoite.
  4. Napsauta Seuraava.

Tilin vahvistaminen

  1. Avaa uusi selaimen välilehti ja kirjaudu sisään sähköpostitilillesi.
  2. Kopioi kuusinumeroinen koodi.
  3. Palaa Case Active -välilehdelle.
  4. Liitä koodi kenttiin. Uusi ikkuna tulee näkyviin.

Uudessa ikkunassa:

  1. Täytä pakolliset kentät.
  2. Lue käyttöehdot ja hyväksy ne valitsemalla valintaruutu.
  3. Napsauta LUO ILMAINEN TILI.

Suoritetaan sovelluksen asennusta

Työnkulku näkyy seuraavalla sivulla. Viimeistele sovelluksesi määritys suorittamalla vaiheet. Voit aina palata näille sivuille muokataksesi tai päivittääksesi tietoja tai tilisi tilaa.

Täytä Tietoja sovelluksesta -sivulla yrityksen nimi ja verkkosivuston kentät (pakollinen) ja napsauta sitten Seuraava.

Napsauta Laskutustiedot-sivulla Maksusuunnitelma-kenttää ja tarkista vaihtoehdot. Oletusasetus on Free. Napsauta LÄHETÄ.

img

Kun olet lähettänyt lomakkeen, Case Active näyttää Sovellukset-sivun. Jokaisen luomasi tai johon kuuluvan sovelluksen kohdalla sivulla näkyy:

  1. Sovelluksen nimi
  2. Tyyppisi (käyttäjärooli)
  3. Tila

img

Sovellukset-sivu on monien järjestelmänvalvojan toimintojen aloituspiste:

  1. Napsauta APP-painiketta viestiäksesi, täyttääksesi asiakirjoja, käydäksesi videokeskustelussa (ja paljon muuta) tapaukseen liittyvissä asioissa.
  2. Napsauta JÄRJESTELMÄNVALVOJA-painiketta avataksesi hallintapaneelin. Täällä järjestelmänvalvojat hallitsevat käyttäjiä, tapauksia, integraatioita ja muuta.
  3. Napsauta ellipsiä hallitaksesi tilin laskutus- ja ilmoitusasetuksia.
  • Huomautus Case Active näyttää PÄÄKÄYTTÄJÄ-painikkeen järjestelmänvalvojakäyttäjille. Käyttäjät, joilla ei ole järjestelmänvalvojan oikeuksia, eivät näe JÄRJESTELMÄNVALVOJA-painiketta (kuten alla olevassa näytössä näkyy).
