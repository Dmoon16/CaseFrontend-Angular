Kuinka järjestelmänvalvojat hallitsevat käyttäjien määrityksiä tapauksiin

Käyttäjiä ja tapausmäärityksiä voi hallita viidellä tavalla. Napsauta linkkiä saadaksesi lisätietoja.

Menetelmä Tyyppi Toiminnot Yksityiskohdat
Tapausta luotaessa UI lisätä Järjestelmänvalvoja-painike > Tapaukset > Uusi painike > Määritä käyttäjät -painike
Määritä käyttäjä automaattisesti julkisen kirjautumislinkin avulla Käyttäjän toiminto lisätä Järjestelmänvalvoja-painike > Asetukset > Järjestelmän asetukset -sivu
Käyttöliittymän käyttö olemassa olevassa tapauksessa UI lisää poista Järjestelmänvalvoja-painike > Tapaukset > Määritä käyttäjät
Tuo työkalu UI lisätä, muokata ja poistaa Järjestelmänvalvoja-painike > Integroinnit > Tuonnit
API:n käyttö (tulossa pian) API lisätä, muokata ja poistaa Ei käytössä
