Käyttäjäprofiilin päivittäminen

Tämä artikkeli näyttää käyttäjille, kuinka he voivat päivittää profiilisivunsa tietoja.

  • Käyttäjät voivat päivittää omia profiilejaan.
  • Järjestelmänvalvojat voivat päivittää minkä tahansa käyttäjän profiilisivun.

Käyttäjille

Ensimmäinen vaihe on profiilisivun avaaminen. Käyttäjät ja järjestelmänvalvojat päivittävät oman profiilisivunsa noudattamalla näitä ohjeita.

  1. Kirjaudu sisään Case Active -tilillesi.
  2. Napsauta sivun yläreunassa olevaa Profiili-välilehteä.
  3. Tehdä muutoksia.
  4. Napsauta Tallenna.

Useimmat kentät ovat valinnaisia, ja jotkut kentät ovat pakollisia. Pakolliset kentät ovat:

  • Etunimi
  • Sukunimi
  • Sähköpostiosoite
  • Kieli
  • Aikavyöhyke

Järjestelmänvalvojille

Järjestelmänvalvojat voivat myös päivittää muiden käyttäjien profiilitietoja. Tämä sisältää kolme kenttää HOST USER INFO -otsikon alla.

  1. Kirjaudu sisään Case Active -järjestelmänvalvojatilillesi.
  2. Etsi sovellus ja napsauta JÄRJESTELMÄNVALVOJA-painiketta.
  3. Napsauta ylävalikosta Käyttäjät.
  4. Siirry alivalikkoon ja valitse käyttäjäluettelo, jota haluat käyttää (AKTIIVINEN tai POIS KÄYTÖSTÄ). Oletusasetus on Aktiivinen.
  5. Etsi käyttäjä luettelosta ja napsauta Muokkaa.
  6. Tee tarvittavat muutokset. (Katso alempaa)
  7. Napsauta Tallenna.

Muuta käyttäjätietoja

Järjestelmänvalvojat voivat muuttaa käyttäjätietoja.

  1. Aseta käyttäjätyyppi (käyttäjä tai johtaja).
  2. Aseta käyttäjän tila (aktiivinen arvoon Ei käytössä).
  3. Anna yksilöllinen tunnistetunnus. (valinnainen)
  4. Napsauta Tallenna.
