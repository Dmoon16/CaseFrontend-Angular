Kuinka järjestelmänvalvojat hallitsevat käyttäjiä

Järjestelmänvalvojat luovat, muokkaavat tai poistavat käytöstä, kuka voi käyttää sovelluksia ja koteloita. Kun käyttäjä kirjautuu alustalle ensimmäistä kertaa, hänen on suoritettava profiilityönkulku.

  • Napsauta tätä saadaksesi lisätietoja uuden käyttäjäprofiilin työnkulusta.

Käyttäjän hallinta

Alla olevassa taulukossa on viisi tapaa hallita käyttäjää. Napsauta linkkiä saadaksesi lisätietoja.

Menetelmä Tyyppi Toiminnot Yksityiskohdat
Käyttöliittymän käyttö UI lisää & muokkaa Järjestelmänvalvoja-painike > Käyttäjät
Kun luot tapausta ja määrität käyttäjän tapaukselle UI lisätä Järjestelmänvalvoja-painike > Tapaukset > Uusi painike > Luo tapaus -sivu
Luo käyttäjä automaattisesti julkisen kirjautumislinkin avulla Käyttäjän toiminta lisätä Järjestelmänvalvoja-painike > Asetukset > Järjestelmän asetukset -sivu
Tuo työkalu UI lisää & muokkaa Järjestelmänvalvoja-painike > Integroinnit > Tuonnit
API:n käyttö (tulossa pian) API lisää & muokkaa Ei käytössä

Ilmoitukset

Alusta ei lähetä käyttäjille ilmoitusta, kun heidät lisätään sovellukseen. Järjestelmänvalvojat voivat kuitenkin ottaa käyttöön ilmoitukset tapausmäärityksistä.
