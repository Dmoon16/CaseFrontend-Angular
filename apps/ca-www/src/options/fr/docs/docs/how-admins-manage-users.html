Comment les administrateurs gèrent les utilisateurs

Les administrateurs créent, modifient ou désactivent qui peut accéder aux applications et aux cas. Lorsqu'un utilisateur se connecte à la plateforme pour la première fois, il doit compléter un workflow de profil.

  • Cliquez ici pour en savoir plus sur le nouveau workflow de profil utilisateur.

Gestion d'un utilisateur

Le tableau ci-dessous montre cinq façons de gérer un utilisateur. Cliquez sur un lien pour savoir comment.

Méthode Taper Actions Détails
Utilisation de l'interface utilisateur interface utilisateur ajouter et modifier Bouton Admin > Utilisateurs
Lors de la création d'un cas et de l'attribution d'un utilisateur au cas interface utilisateur ajouter Bouton Admin > Requêtes > Nouveau bouton > Page Créer une requête
Création automatique d'un utilisateur à l'aide d'un lien d'inscription public Intervention de l'utilisateur ajouter Bouton Admin > Paramètres > page Configuration du système
Outil d'importation interface utilisateur ajouter et modifier Bouton Admin > Intégrations > Importations
Utilisation de l'API (à venir) API ajouter et modifier N / A

Avis

La plateforme n'envoie pas de notification aux utilisateurs lorsqu'ils sont ajoutés à une application. Cependant, les administrateurs peuvent activer les notifications pour les attributions de cas.
