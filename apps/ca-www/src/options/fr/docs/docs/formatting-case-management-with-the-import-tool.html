Formatage des données de gestion de cas avec l'outil d'importation

Vous trouverez ci-dessous une liste des champs obligatoires dans les fichiers CSV pour gérer les cas avec l'outil d'importation.

Nom d'utilisateur

  • Saisissez l'adresse e-mail de l'utilisateur.

tag_id (un identifiant que vous pouvez référencer plus tard)

  • un identifiant unique (utilisez des caractères alphanumériques, les traits d'union sont acceptés)

État de cas

  • 0 = fermé
  • 1 = actif

date_opened

  • année mois jour
  • mm/jj/aaaa

sur

  • une description unique du cas (utiliser des caractères alphanumériques, les traits d'union sont acceptés)

image
