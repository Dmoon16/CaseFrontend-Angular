Comment s'identifier

Découvrez comment vous connecter à votre compte Case Active existant.

Se connecter

  1. Allez sur la page de connexion .
  2. Cliquez sur le bouton CONNEXION.
  3. Entrer votre Email.
  4. Cliquez sur Suivant.

Vérification d'un compte

  1. Ouvrez un nouvel onglet de navigateur et connectez-vous à votre compte de messagerie.
  2. Copiez le code à six chiffres.
  3. Revenez à l'onglet Cas actif.
  4. Collez le code dans les champs. Une nouvelle fenêtre s'affiche.

Case Active affiche la page Applications.

  • Remarque La plateforme affiche un bouton ADMIN à côté du bouton APP pour les utilisateurs disposant de privilèges d'administrateur.
