Comment créer automatiquement des cas avec un lien d'inscription public

Cet article montre aux administrateurs comment créer un cas (et ajouter un utilisateur en même temps) en envoyant un seul lien d'inscription.

Localisation du lien

  1. Connectez-vous au compte administrateur.
  2. Localisez une application sur la page Applications et cliquez sur le bouton ADMIN.
  3. Dans le panneau d'administration, cliquez sur l'onglet du menu Paramètres.
  4. Assurez-vous que l'onglet du sous-menu SYSTEM SETUP est sélectionné.

image

Configuration des paramètres

Faites défiler jusqu'à l'en-tête INSCRIPTION UTILISATEUR.

Assurez-vous que le champ Autoriser l'inscription est défini sur Activé. (obligatoire) Lorsque cette fonctionnalité est activée, Case Active génère un lien unique.

  1. Cliquez sur le champ Créer automatiquement un cas et sélectionnez Activé. Le champ Rôle de l'utilisateur du dossier s'affiche.
  2. Sélectionnez un rôle. (voir ci-dessous)

image

Rôles d'utilisateur de cas

Il existe trois rôles d'utilisateur de dossier.

  • clients (recommandé)
  • défaut
  • Personnel

Envoi du lien

Partagez le lien avec une ou plusieurs personnes. Chaque utilisateur qui clique sur le lien et termine l'inscription déclenche la plate-forme pour générer un nouveau cas sur la page des cas et affecter automatiquement l'utilisateur dans le cas.
