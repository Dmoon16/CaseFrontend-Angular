Comment créer ou modifier des cas avec l'outil d'importation

Cet article montre aux administrateurs comment créer ou modifier des cas existants avec l'outil d'importation.

Ouverture de l'outil d'importation

  1. Connectez-vous au compte administrateur.
  2. Localisez une application sur la page Applications et cliquez sur le bouton ADMIN.
  3. Dans le panneau d'administration, cliquez sur l'onglet Intégration, puis sur IMPORTATIONS dans le sous-menu.
  4. Cliquez sur le bouton NOUVEAU. La boîte de dialogue Ajouter des importations s'affiche dans un nouvel onglet du navigateur.

image

Création de cas

Une fois la boîte de dialogue Ajouter des importations ouverte :

  1. Cliquez sur le champ Type et sélectionnez une option (voir ci-dessous). Un lien s'affiche.
  2. Cliquez sur le lien Télécharger l'échantillon. Un fichier CSV se télécharge sur votre appareil.

Ensuite, ouvrez le fichier et ajoutez des cas.

Taper

Lorsque vous cliquez sur le champ Type , un menu déroulant s'affiche. Choisissez l'une des deux options pour les importations de cas groupés.

  • Remarque Le menu déroulant comporte de nombreuses options, mais seules deux sont liées à la gestion du cas. Chaque option a un exemple de fichier CSV différent.

Ajouter des cas

  • Choisissez cette option pour un téléchargement groupé simple de cas.
  • Cliquez sur le lien Télécharger l'exemple pour ouvrir un fichier CSV avec des en-têtes de colonne pour le type d'importation Créer des cas.

Modifier les cas

  • Choisissez cette option pour mettre à jour les cas actuels.
  • Cliquez sur le lien Télécharger l'exemple pour ouvrir un fichier CSV avec des en-têtes de colonne pour le type d'importation Modifier les cas.

Téléchargement du fichier

Vous pouvez télécharger un fichier immédiatement ou revenir ultérieurement à la boîte de dialogue Ajouter des importations et effectuer ces étapes.

  1. Ouvrez la boîte de dialogue Ajouter des importations.
  2. Entrez un nom unique pour le fichier CSV.
  3. Cliquez sur l'icône plus (+) pour télécharger le fichier depuis votre appareil.
  4. Cliquez sur le champ Type et sélectionnez une option. (voir au dessus)
  5. Cliquez sur ENREGISTRER.

image

Mise en page

Ce sont les champs obligatoires et les premières exigences. Cliquez ici pour revoir les détails de formatage.

tag_id (ou case_ID) un identifiant unique (utilisez des caractères alphanumériques, les traits d'union sont acceptés)

case_status 0 = fermé 1 = actif

date_ouverture mois/jour/année mm/jj/aaaa

sur une description unique du cas (utiliser des caractères alphanumériques, les traits d'union sont acceptés)

image

Traitement

Selon le nombre d'enregistrements dans le fichier et la charge du système, la plateforme peut prendre plusieurs minutes pour traiter l'importation.

image

Une fois l'importation terminée, la plateforme ajuste les enregistrements de cas (c'est-à-dire ajoute ou modifie) sur le tableau de bord des cas de l'administrateur.

Échec

Si l'opération d'importation échoue, la plateforme renvoie un message Failed. Dans la plupart des cas, la cause de l'échec est le formatage des données.

image

Pour voir la cause de l'échec :

  1. Cliquez sur Afficher sous l'en-tête ACTION.
  2. Sur la page suivante, passez la souris sur l'icône Échec. L'info-bulle affiche l'erreur.
  3. Accédez au fichier CVS et ajustez et corrigez l'erreur.
  4. Chargez à nouveau le fichier.
