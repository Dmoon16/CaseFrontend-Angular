Comment mettre à jour un profil utilisateur

Cet article montre aux utilisateurs comment mettre à jour les informations sur leur page de profil.

  • Les utilisateurs peuvent mettre à jour leurs propres profils.
  • Les administrateurs peuvent mettre à jour la page de profil de n'importe quel utilisateur.

Pour les utilisateurs

La première étape consiste à ouvrir la page Profil. Les utilisateurs et les administrateurs suivent ces étapes pour mettre à jour leur propre page de profil.

  1. Connectez-vous à votre compte Case Active.
  2. Cliquez sur l'onglet Profil en haut de la page.
  3. Faire des changements.
  4. Cliquez sur Enregistrer.

La plupart des champs sont facultatifs et certains champs sont obligatoires. Les champs obligatoires sont :

  • Prénom
  • Nom de famille
  • Adresse e-mail
  • Langue
  • Fuseau horaire

Pour les administrateurs

Les administrateurs peuvent également mettre à jour les informations de profil des autres utilisateurs. Cela inclut trois champs sous l'en-tête HOST USER INFO.

  1. Connectez-vous à votre compte administrateur Case Active.
  2. Localisez une application et cliquez sur le bouton ADMIN.
  3. Cliquez sur Utilisateurs dans le menu supérieur.
  4. Allez dans le sous-menu et sélectionnez la liste d'utilisateurs à laquelle vous souhaitez accéder (ACTIF ou DÉSACTIVÉ). Le paramètre par défaut est Actif.
  5. Localisez un utilisateur dans la liste et cliquez sur Modifier.
  6. Apportez les modifications nécessaires. (voir ci-dessous)
  7. Cliquez sur Enregistrer.

Modifier les informations utilisateur

Les administrateurs peuvent modifier les informations de l'utilisateur.

  1. Définissez le type d'utilisateur (utilisateur ou gestionnaire).
  2. Définissez le statut de l'utilisateur (actif à désactivé).
  3. Entrez un ID de balise unique. (optionnel)
  4. Cliquez sur Enregistrer.
