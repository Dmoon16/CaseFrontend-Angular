Formatage des données de gestion des utilisateurs avec l'outil d'importation

Vous trouverez ci-dessous une liste des champs de données dans les fichiers CSV. Chaque option d'importation a une combinaison différente de champs.

Nom d'utilisateur

  • Saisissez l'adresse e-mail de l'utilisateur.

prénom

  • Saisissez le prénom de l'utilisateur.

nom de famille

  • Entrez le nom de famille de l'utilisateur.

lieu

  • Cette colonne vérifie la langue préférée de l'utilisateur.
  • Entrez en pour l'anglais, sp pour l'espagnol.

infozone

  • Cela définit le fuseau horaire de l'utilisateur. Entrez le continent et la ville sans espaces (utilisez un trait de soulignement).
  • Doit utiliser une option dans le menu déroulant Profil > LANGUE ET RÉGION > Fuseau horaire.
  • Utilisez le fuseau horaire le mieux adapté. Par exemple, Boston ne figure pas dans la liste déroulante, nous pouvons donc utiliser New York (ou n'importe quelle ville du fuseau horaire EST/EDT).

host_tag_id

  • Entrez une étiquette permanente.
  • az, 0-9, -, / sont autorisés.

État de cas

  • 0 = fermé
  • 1 = actif
