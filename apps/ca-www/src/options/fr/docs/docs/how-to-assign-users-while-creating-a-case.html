Comment affecter des utilisateurs lors de la création d'un cas

Cet article montre aux administrateurs comment attribuer un utilisateur lors de la création d'un nouveau cas. Si l'utilisateur n'est pas déjà enregistré sur la plateforme, vous pouvez créer ajouter un nouvel utilisateur lors de la création d'un nouveau cas.

Création d'un cas

  1. Connectez-vous au compte administrateur.
  2. Localisez une application sur la page Applications et cliquez sur le bouton ADMIN.
  3. Cliquez sur l'onglet du menu Requêtes.
  4. Cliquez sur le bouton NOUVEAU. Une nouvelle fenêtre s'affiche.

image

  • Remplissez les champs pour créer un cas, puis faites défiler jusqu'à l'en-tête Utilisateurs assignés.

Affectation d'un utilisateur

Utilisateur actuel

Suivez ces étapes pour affecter un utilisateur actuel au nouveau cas.

Cliquez sur le bouton ATTRIBUER UTILISATEURS et remplissez les champs.

  1. Cliquez sur le menu déroulant Utilisateur et sélectionnez un identifiant (par exemple, un e-mail).
  2. Saisissez une valeur dans le champ vide (par exemple, adresse e-mail).
  3. Si la fiche utilisateur existe dans la plateforme, le champ propose automatiquement un nom.
  4. Sélectionnez le bon utilisateur.
  5. Sélectionnez un rôle d'utilisateur (clients, par défaut ou personnel).
  6. Si vous le souhaitez, cochez la case pour envoyer à l'utilisateur une notification par e-mail.
  7. Cliquez sur ENREGISTRER.

image

Case Active ajoute l'utilisateur au cas. Désormais, les administrateurs peuvent cliquer sur Enregistrer pour fermer la fenêtre, ajouter un nouvel utilisateur ou supprimer un utilisateur.

image

Ajout d'un nouvel utilisateur

Si la plateforme ne trouve pas un enregistrement actuel, cliquez sur le bouton pour ajouter un nouvel utilisateur.

image

Remplissez les champs puis cliquez sur ENREGISTRER pour ajouter l'utilisateur à la plateforme. Ensuite, définissez un rôle d'utilisateur et cliquez à nouveau sur ENREGISTRER pour affecter l'utilisateur au cas. Enfin, cliquez sur ENREGISTRER une fois de plus pour enregistrer le cas.
