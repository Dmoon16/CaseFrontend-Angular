Giunsa ang mga Gumagamit Paghimo og Bag-ong Account

Human ang admin makadugang ug user sa usa ka kaso: * kung ang user bag-o sa caseactive, ang user kinahanglang mokompleto sa workflow aron makahimo ug account ug modawat sa imbitasyon sa pag-access sa kaso. * kung ang user adunay kasamtangan nga caseactive account, ang user kinahanglan nga modawat sa imbitasyon sa pag-access sa kaso.

Gihubit niining artikuloha ang mga lakang nga kinahanglang kompletohon sa usa ka user sa paghimo sa ilang account.

Pagsugod

Adunay duha ka paagi nga ang usa ka user mahimong idugang sa usa ka caseactive nga app

  • Pinaagi sa admin: Kung ang admin maghimo ug user pinaagi sa platform UI o Import tool , ang admin kinahanglang magpadala sa user ug link sa account.caseactive.com aron makapadayon sa pag-login.
  • Pinaagi sa user: Kung ang user naghimo ug account gamit ang signup link , ang user awtomatikong i-redirect sa account.caseactive.com aron makapadayon sa pag-login.

Pag-log in

Sa diha nga ang user nag-log in sa ilang CaseActive nga account sa unang higayon, ang plataporma nag-aghat sa tiggamit sa:

  • ablihi ang ilang email account ug dayon kopyaha/idikit ang unom ka digit nga security code
  • kompletoha ang account profile

img

Human ma-klik sa user ang UPDATE, ang pahina sa Apps magpakita. Gipakita sa panid ang ngalan sa app (ngalan sa kompanya), ang tipo sa gumagamit (papel), kahimtang, ug usa ka buton sa INVITATION.

img

Aron ma-access ang app, kinahanglang dawaton sa user ang imbitasyon. Ang tiggamit kinahanglan:

  1. I-klik ang INVITATION nga buton.
  2. Ribyuha ang mga permiso.
  3. I-klik ang Notifications tab para marepaso ug ma-edit (opsyonal) ang mga termino.
  4. I-klik ang APPROVE ACCESS.

Kung giaprobahan sa user, ang plataporma magpakita ug APP button sa user page. Kini nga buton naghatag sa user og access sa gi-assign nga mga kaso.

Aron ma-edit ang mga setting sa gumagamit, gi-klik sa tiggamit ang ellipsis.

Mahimong: * tangtangon ang access sa app * usba ang mga setting sa pagpahibalo

Human makompleto sa user ang workflow, makita ang user name sa lista sa Users sa admin. Gipakita sa lista ang tanan nga tiggamit, naka-log in o wala.
