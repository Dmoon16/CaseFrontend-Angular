Pag-format sa Data sa Mga Assignment sa Kaso sa Gumagamit gamit ang Tool sa Pag-import

Sa ubos usa ka lista sa mga natad sa datos sa mga file sa CSV. Ang matag kapilian sa pag-import adunay lahi nga kombinasyon sa mga natad.

kaso_ID

  • Isulod ang case User o Tag ID.
  • Adto sa Cases page > Edit > Tag ID O Adto sa Cases page > Edit > Case ID sa url.

user_ID

  • Isulod ang user ID o Tag ID.
  • Adto sa Users page > Edit > Tag ID O Adto sa Users page > Edit > User ID sa url.

kaso_role_order

  • Biyai nga blangko ang cell.

case_role_ID

  • Pagsulod: mga kliyente, kawani, o default.

case_notify

  • 0 = dili
  • 1 = Oo, ipadala ang user og email notification.
