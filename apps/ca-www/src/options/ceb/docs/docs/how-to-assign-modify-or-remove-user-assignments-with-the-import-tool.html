Unsaon Pag-assign, Pag-usab o Pagtangtang sa Mga Assignment sa Gumagamit gamit ang Import Tool

Gipakita sa kini nga artikulo ang mga admin kung giunsa pagdumala ang mga tiggamit sa kaso gamit ang tool sa Import.

Ang mga admin mahimong:

  • itudlo ang mga tiggamit sa usa ka kaso
  • usba ang mga tiggamit nga na-assign na sa usa ka kaso
  • kuhaa ang mga tiggamit nga gi-assign sa usa ka kaso

Pag-abli sa Import nga himan

  1. Log in sa admin account.
  2. Pangita ug app sa Apps page ug i-klik ang ADMIN button.
  3. I-klik ang Integration tab ug dayon IMPORTS sa submenu.
  4. I-klik ang BAG-O nga buton. Ang Add Imports dialog box nagpakita.

Paghimo og mga rekord

Pag-download sa usa o daghang mga CSV file. Gamita kini nga mga file sa pagdugang, pag-usab, o pagtangtang sa mga tiggamit sa kaso. Human maablihan ang Add Imports dialog box:

  1. I-klik ang Type field.
  2. Pag-scroll sa drop-down menu ug pagpili og opsyon (tan-awa sa ubos). Nagpakita ang usa ka link.
  3. I-klik ang Download Sample nga link. Usa ka CSV file ang ma-download sa imong device.

Ablihi ang file ug idugang ang mga rekord. I-klik dinhi aron makat-on unsaon paggamit sa format nga datos sa CSV file.

Mga kapilian sa file

I-assign ang mga tiggamit sa mga kaso * Pilia kini nga kapilian aron idugang ang mga tiggamit sa usa ka kaso. * Biyai nga blangko ang case_role_order cell. * Adto sa Users page ug kopyaha ang Tag ID para niining tawhana. Kung blangko ang field sa Tag ID, pagsulod og talagsaon nga ID, i-klik ang Save, ug dayon kopyaha ang ID sa CSV file.

Usba ang mga tiggamit nga gi-assign sa mga kaso * Pilia kini nga opsyon aron ma-update ang datos alang sa kasamtangang mga tiggamit nga gi-assign sa usa ka kaso. * Gamita ang Case Tag ID para sa case_ID column.

Kuhaa ang mga tiggamit nga gi-assign sa mga kaso * Pilia kini nga kapilian aron tangtangon ang mga tiggamit nga gi-assign sa usa ka kaso.

Pag-upload sa file

Mahimo kang mag-upload dayon og file o mobalik sa Add Imports dialog box sa ulahi nga petsa ug kompletoha kini nga mga lakang.

  1. Ablihi ang Add Imports dialog box.
  2. Pagsulod ug talagsaong ngalan para sa CSV file.
  3. I-klik ang plus (+) icon aron ma-upload ang file gikan sa imong device.
  4. I-klik ang Type field ug pagpili og opsyon. (tan-awa sa ibabaw)
  5. I-klik ang SAVE.

Pagproseso

Depende sa gidaghanon sa mga rekord sa file ug system load, ang plataporma mahimong molungtad ug pipila ka minuto aron maproseso ang import.

Kung makompleto na ang import, ang plataporma mag-adjust sa mga rekord sa user (pananglitan, pagdugang, pag-usab, o pagtangtang) sa pahina sa Mga Kaso.

Kapakyasan

Kung mapakyas ang operasyon sa pag-import, ang plataporma magbalik sa usa ka Napakyas nga mensahe. Sa kadaghanan nga mga kaso, ang hinungdan sa pagkapakyas mao ang pag-format sa datos.

Aron makita ang hinungdan sa kapakyasan:

  1. I-klik ang View ubos sa Action heading.
  2. Sa sunod nga panid, i-hover ang mouse sa Failed icon. Gipakita sa tooltip ang sayup.
  3. Adto sa CVS file ug i-adjust ug ayoha ang sayop.
  4. I-upload pag-usab ang file.
