Unsaon Paghimo sa mga Gumagamit gamit ang Import Tool

Kini nga artikulo nagpakita sa mga admin kung giunsa paghimo ang mga tiggamit gamit ang Import tool.

Pag-abli sa Import nga himan

Mahimong gamiton sa mga admin ang tool sa Import aron daghan ang mag-upload sa mga bag-ong tiggamit ug usbon ang mga naa na nga rekord sa gumagamit.

  1. Log in sa admin account.
  2. Pangita ug app sa Apps page ug i-klik ang ADMIN button.
  3. Sa Admin panel, i-klik ang Integration tab ug dayon IMPORTS sa submenu.
  4. I-klik ang BAG-O nga buton. Ang Add Imports dialog box nagpakita sa bag-ong tab sa browser.

Paghimo og mga rekord

Human maablihan ang Add Imports dialog box:

  1. I-klik ang Type field ug pilia ang opsyon (tan-awa sa ubos). Nagpakita ang usa ka link.
  2. I-klik ang Download Sample nga link. Usa ka CSV file ang ma-download sa imong device.

Sunod, ablihi ang file ug idugang ang mga rekord. I-klik dinhi aron makat-on unsaon paggamit sa format nga datos sa CSV file.

Matang

Kung imong i-klik ang Type field, usa ka drop-down menu ang makita. Pagpili og usa sa duha ka kapilian alang sa daghang importasyon.

:exclamation: NOTE Ang drop-down menu adunay daghang mga opsyon, apan duha lang ang may kalabutan sa Import tool. Ang matag kapilian adunay lahi nga sample nga CSV file.

Idugang ang mga tiggamit

  • Pilia kini nga kapilian alang sa usa ka yano nga daghang pag-upload sa mga rekord sa gumagamit.
  • I-klik ang link sa Download Sample aron maablihan ang CSV file nga adunay mga ulohan sa column para sa Add users Import type.

Usba ang mga tiggamit

  • Pilia kini nga opsyon sa pag-update sa datos alang sa kasamtangang tiggamit.
  • I-klik ang link sa Pag-download sa Sample aron maablihan ang usa ka CSV file nga adunay mga ulohan sa kolum alang sa tipo sa Pag-import sa mga tiggamit.

Pag-upload sa file

Mahimo kang mag-upload dayon og file o mobalik sa Add Imports dialog box sa ulahi nga petsa ug kompletoha kini nga mga lakang.

  1. Ablihi ang Add Imports dialog box.
  2. Pagsulod ug talagsaong ngalan para sa CSV file.
  3. I-klik ang plus (+) icon aron ma-upload ang file gikan sa imong device.
  4. I-klik ang Type field ug pagpili og opsyon. (tan-awa sa ibabaw)
  5. I-klik ang SAVE.

Pagproseso

Depende sa gidaghanon sa mga rekord sa file ug system load, ang plataporma mahimong molungtad ug pipila ka minuto aron maproseso ang import.

Kung makompleto ang import, ang plataporma mag-adjust sa mga rekord sa user (pananglitan, pagdugang, pag-usab, o pag-disable) sa dashboard sa Users sa admin.

Kapakyasan

Kung mapakyas ang operasyon sa pag-import, ang plataporma magbalik sa usa ka Napakyas nga mensahe. Sa kadaghanan nga mga kaso, ang hinungdan sa pagkapakyas mao ang pag-format sa datos.

Aron makita ang hinungdan sa kapakyasan:

  1. I-klik ang View ubos sa Action heading.
  2. Sa sunod nga panid, i-hover ang mouse sa Failed icon. Gipakita sa tooltip ang sayup.
  3. Adto sa CVS file ug i-adjust ug ayoha ang sayop.
  4. I-upload pag-usab ang file.
