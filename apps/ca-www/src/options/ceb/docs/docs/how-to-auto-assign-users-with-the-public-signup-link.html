Giunsa ang Awtomatikong Pag-assign sa mga Gumagamit gamit ang Public Signup Link

Nakat-on ang mga admin kung unsaon pag-configure ang mga setting sa sistema aron ang mga bag-ong tiggamit awtomatik nga ma-assign sa usa ka bag-ong awtomatik nga gihimo nga kaso.

Pag-abli sa panid sa pag-setup

Ang unang lakang mao ang pag-abli sa System Setup page.

  1. Log in sa admin account.
  2. Pangita ug app sa Apps page ug i-klik ang ADMIN button.
  3. I-klik ang Systems tab ug dayon SYSTEM SETUP sa submenu.
  4. Pag-scroll paubos sa USER SIGNUP nga ulohan.

Pag-configure sa mga setting sa sistema

  1. Ibutang ang Allow Signup ug Auto Create Case drop-down menus ngadto sa On.
  2. I-klik ang role menu ug pilia ang opsyon (girekomendar ang default).
  3. Kopyaha ug ipaambit ang link sa Public User Signup.

Nagpadala sa link

Ang matag user nga mo-klik sa link ug mokompleto sa signup mag-trigger sa plataporma sa pagmugna og bag-ong kaso sa Cases page ug auto-assign sa user sa maong kaso.
