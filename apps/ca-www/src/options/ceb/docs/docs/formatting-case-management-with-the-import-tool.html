Pag-format sa Data sa Pagdumala sa Kaso gamit ang Import Tool

Sa ubos usa ka lista sa gikinahanglan nga mga natad sa mga CSV file aron madumala ang mga kaso gamit ang Import tool.

username

  • Pagsulod sa email address sa user.

tag_id (usa ka id nga mahimo nimong i-refer sa ulahi)

  • usa ka talagsaon nga identifier (gamit ang alphanumeric nga mga karakter, gidawat ang mga hyphen)

kaso_status

  • 0 = sirado
  • 1 = aktibo

date_open

  • bulan adlaw Tuig
  • mm/dd/yyyy

mahitungod sa

  • usa ka talagsaon nga paghulagway sa kaso (gamit ang alphanumeric nga mga karakter, gidawat ang mga hyphen)
