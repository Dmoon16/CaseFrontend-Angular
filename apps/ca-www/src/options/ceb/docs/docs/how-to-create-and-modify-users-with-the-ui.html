Giunsa Paghimo ug Pag-usab ang mga Gumagamit gamit ang UI

Gipakita sa kini nga artikulo ang mga admin kung giunsa paghimo ang mga tiggamit ug pag-update sa mga rekord gamit ang interface sa gumagamit sa platform. Mahimo usab nga idugang sa mga admin ang mga tiggamit sa usa ka bag-ong kaso ug i-edit ang mga naa na nga kaso.

Paghimo og user

  1. Log in sa admin account.
  2. Pangita ug app sa Apps page ug i-klik ang ADMIN button.
  3. Sa Admin panel, i-klik ang Users menu tab.
  4. I-klik ang BAG-O nga buton. Usa ka bag-ong bintana nagpakita.

img

Pun-a ang mga uma ug i-klik ang SAVE. Gidugang sa Case Active ang ngalan sa lista sa mga tiggamit nga nakig-uban niini nga app.

img

Pag-usab sa usa ka tiggamit

Aron ma-update ang gamay nga gidaghanon sa mga rekord sa tiggamit, gamita ang tool sa Pag-edit ug paghimo og mga pagbag-o sa matag indibidwal nga tiggamit.

  • Mubo nga sulat Aron usbon ang daghang mga rekord sa tiggamit sa usa ka paglingkod, hunahunaa ang paggamit sa tool sa Import .
  1. Sa Admin panel, i-klik ang Users menu tab.
  2. Pangitaa ang usa ka user ug i-klik ang Edit.
  3. Paghimo og mga kausaban ug i-klik ang SAVE.

img

Sunod nga mga lakang

Ang Case Active dili magpadala sa user og email notification. Kinahanglang kontakon sa mga admin ang user ug padalhan sila og link sa Case Active home page. Gi-klik dayon sa user ang LOGIN button ug gikompleto ang workflow para sa mga bag-ong signup.

  • Kini nga artikulo naghulagway sa signup workflow alang sa bag-ong mga tiggamit.
