Giunsa Paghimo ang mga Gumagamit nga adunay Link sa Pag-signup

Gipakita sa kini nga artikulo ang mga admin kung giunsa paghimo ang mga tiggamit pinaagi sa pagpadala usa ka link sa pag-signup.

Pag-configure sa link sa pag-signup

  1. I-klik ang Send them a signup link sa ubos sa Add user dialog box. Usa ka bag-ong panid nagpakita.
  2. I-configure ang mga opsyon sa pag-signup. (Tan-awa sa ubos)

img

Siguroha nga ang Allow Signup field i-set sa On. (gikinahanglan) Kung ma-enable, ang Case Active makamugna og talagsaong link.

Ribyuha ang mga opsyon sa ubos ug dayon i-klik ang icon aron kopyahon ang link. Mahimo nimong ibutang ang link sa imong website o ipadala ang link sa usa o daghang mga tawo pinaagi sa email o bisan unsang channel sa komunikasyon.

img

Paghimo og kaso

Opsyonal, mahimo ka nga awtomatiko nga maghimo usa ka kaso alang sa matag bag-ong tiggamit ug maghatag usa ka tahas sa tiggamit.

  1. I-klik ang Auto Create Case field ug pilia ang On. Gipakita ang mga field sa Case User Role.
  2. I-klik ang Case User Role ug pagpili og opsyon. (Tan-awa sa ubos)

img

Mga papel sa tiggamit sa kaso

Adunay tulo ka mga papel sa tiggamit sa kaso.

  • mga kliyente
  • default
  • kawani

Nagpadala sa link

Ipakigbahin ang link sa usa o daghang mga tawo. Ang matag tawo mag-klik sa link ug mokompleto sa workflow aron ma-activate ang ilang Case Active account. Kung makompleto na sa user ang workflow sa pag-signup, idugang sa Case Active ang user sa lista sa Users.

Sunod nga mga lakang

Ang Case Active nagpadala sa user og email notification uban sa signup link. Gi-klik sa user ang link ug gikompleto ang workflow para sa mga bag-ong signup.

  • Kini nga artikulo naghulagway sa signup workflow alang sa bag-ong mga tiggamit.
