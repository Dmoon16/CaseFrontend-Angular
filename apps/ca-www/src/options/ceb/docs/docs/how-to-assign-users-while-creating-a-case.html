Giunsa Pag-assign ang mga Gumagamit Samtang Naghimo og Kaso

Gipakita sa kini nga artikulo ang mga admin kung giunsa ang pag-assign sa usa ka tiggamit samtang naghimo usa ka bag-ong kaso. Kung wala pa nakarehistro ang tiggamit sa plataporma, mahimo ka maghimo pagdugang usa ka bag-ong tiggamit samtang naghimo usa ka bag-ong kaso.

Paghimo og kaso

  1. Log in sa admin account.
  2. Pangita ug app sa Apps page ug i-klik ang ADMIN button.
  3. I-klik ang tab sa Cases menu.
  4. I-klik ang BAG-O nga buton. Usa ka bag-ong bintana nagpakita.

  • Pun-a ang mga natad aron makahimo usa ka kaso ug dayon i-scroll paubos sa ulohan nga Gi-assign nga mga Gumagamit.

Pag-assign sa usa ka user

Kasamtangang tiggamit

Sunda kini nga mga lakang aron ma-assign ang usa ka karon nga tiggamit sa bag-ong kaso.

I-klik ang ASSIGN USERS button ug pun-a ang mga field.

  1. I-klik ang User drop-down ug pagpili og identifier (eg email).
  2. Pagsulod ug bili sa blangko nga field (eg email address).
  3. Kung ang rekord sa tiggamit anaa sa plataporma, ang field awtomatik nga nagsugyot og usa ka ngalan.
  4. Pilia ang husto nga tiggamit.
  5. Pagpili usa ka tahas sa tiggamit (kliyente, default, o kawani).
  6. Opsyonal, susiha ang kahon aron ipadala ang user og pahibalo sa email.
  7. I-klik ang SAVE.

Gidugang sa Case Active ang user sa case. Karon mahimo nang i-klik sa mga admin ang Save aron isira ang bintana, o makadugang ug bag-ong user, o tangtangon ang usa ka user.

Pagdugang og bag-ong user

Kung ang plataporma dili makit-an ang usa ka karon nga rekord, i-klik ang buton aron makadugang usa ka bag-ong tiggamit.

Pun-a ang mga uma ug dayon i-klik ang SAVE aron idugang ang user sa plataporma. Sunod, ibutang ang papel sa user ug i-klik pag-usab ang SAVE aron i-assign ang user sa kaso. Sa katapusan, i-klik ang SAVE sa makausa pa aron i-save ang kaso.
