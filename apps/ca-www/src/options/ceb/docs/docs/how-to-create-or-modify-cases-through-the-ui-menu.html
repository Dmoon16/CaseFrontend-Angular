Giunsa Paghimo o Pag-usab ang mga Kaso Pinaagi sa Menu sa UI

Gipakita sa kini nga artikulo ang mga admin kung giunsa paghimo ang usa ka kaso nga adunay interface sa gumagamit sa platform. Mahimo usab nga idugang sa mga admin ang mga tiggamit sa usa ka bag-ong kaso ug i-update ang mga naa na nga kaso.

Paghimo og kaso

  1. Log in sa admin account.
  2. Pangita ug app sa Apps page ug i-klik ang ADMIN button.
  3. Sa Admin panel, i-klik ang Cases menu tab.
  4. I-klik ang BAG-O nga buton. Usa ka bag-ong bintana nagpakita.

Pun-a ang gikinahanglan nga mga natad ug i-klik ang SAVE.

img

Pagdugang mga tiggamit

Opsyonal, ang mga admin makadugang sa mga tiggamit sa kaso niini nga panid. I-scroll down ug i-klik ang ASSIGN USERS button.

Sa Assign User modal, pun-a ang mga field. I-tsek ang kahon aron ipadala ang user og pahibalo sa email. (kapilian)

Pag-usab sa usa ka kaso

Aron ma-update ang gamay nga gidaghanon sa mga kaso, gamita ang tool sa Pag-edit ug paghimo og mga pagbag-o sa matag indibidwal nga kaso.

  • Mubo nga sulat Aron ma-update ang daghang mga kaso sa usa ka paglingkod, hunahunaa ang paggamit sa tool sa Import
  1. Log in sa admin account.
  2. Pangita ug app sa Apps page ug i-klik ang ADMIN button.
  3. Sa Admin panel, i-klik ang Cases menu tab.
  4. Pangitaa ang usa ka kaso ug i-klik ang Edit.
  5. Paghimo og mga kausaban ug i-klik ang SAVE.
