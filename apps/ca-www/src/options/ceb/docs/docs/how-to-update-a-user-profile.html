Giunsa ang Pag-update sa usa ka Profile sa Gumagamit

Gipakita sa kini nga artikulo ang mga tiggamit kung giunsa ang pag-update sa kasayuran sa ilang panid sa Profile.

  • Ang mga tiggamit mahimong mag-update sa ilang kaugalingon nga mga profile.
  • Mahimong i-update sa mga admin ang panid sa Profile sa bisan kinsa nga tiggamit.

Alang sa mga tiggamit

Ang unang lakang mao ang pag-abli sa pahina sa Profile. Gisunod sa mga tiggamit ug admin kini nga mga lakang aron ma-update ang ilang kaugalingon nga panid sa Profile.

  1. Log in sa imong Case Active account.
  2. I-klik ang tab nga Profile sa ibabaw sa panid.
  3. Paghimo og mga kausaban.
  4. I-klik ang Save.

Kadaghanan sa mga natad kay opsyonal, ug pipila ka mga natad gikinahanglan. Ang gikinahanglan nga mga field mao ang:

  • Unang ngalan
  • Apelyido
  • Email adres
  • Pinulongan
  • Timezone

Para sa mga admin

Mahimo usab nga i-update sa mga admin ang impormasyon sa profile alang sa ubang mga tiggamit. Naglakip kini sa tulo ka mga natad ubos sa ulohan sa HOST USER INFO.

  1. Log in sa imong Case Active admin account.
  2. Pangitaa ang app ug i-klik ang ADMIN button.
  3. I-klik ang Users sa ibabaw nga menu.
  4. Adto sa sub-menu ug pilia ang lista sa user nga gusto nimo nga ma-access (AKTIBO o DISABLED). Ang default setting mao ang Aktibo.
  5. Pangitaa ang usa ka tiggamit sa lista ug i-klik ang Edit.
  6. Paghimo og mga kausaban kon gikinahanglan. (Tan-awa sa ubos)
  7. I-klik ang Save.

Usba ang impormasyon sa tiggamit

Mahimong usbon sa mga admin ang impormasyon sa tiggamit.

  1. Ibutang ang Uri sa Gumagamit (User o Manager).
  2. I-set ang Status sa Gumagamit (Aktibo hangtod Na-disable).
  3. Pagsulod ug talagsaong Tag ID. (kapilian)
  4. I-klik ang Save.
