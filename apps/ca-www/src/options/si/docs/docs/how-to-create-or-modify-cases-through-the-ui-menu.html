UI මෙනුව හරහා සිද්ධි සාදන්න හෝ වෙනස් කරන්නේ කෙසේද

මෙම ලිපිය පරිපාලකයින්ට වේදිකාවේ පරිශීලක අතුරුමුහුණත සමඟ නඩුවක් සාදන ආකාරය පෙන්වයි. පරිපාලකයින්ට නව නඩුවකට පරිශීලකයින් එක් කිරීමට සහ පවතින අවස්ථා යාවත්කාලීන කිරීමටද හැකිය.

නඩුවක් නිර්මාණය කිරීම

  1. පරිපාලක ගිණුමට ලොග් වන්න.
  2. යෙදුම් පිටුවේ යෙදුමක් සොයාගෙන ADMIN බොත්තම ක්ලික් කරන්න.
  3. පරිපාලක පැනලයේ, සිද්ධි මෙනු ටැබය ක්ලික් කරන්න.
  4. නව බොත්තම ක්ලික් කරන්න. නව කවුළුවක් දිස්වේ.

අවශ්‍ය ක්ෂේත්‍ර පුරවා සුරකින්න ක්ලික් කරන්න.

පරිශීලකයින් එකතු කිරීම

විකල්ප වශයෙන්, පරිපාලකයින්ට මෙම පිටුවෙහි නඩුවට පරිශීලකයින් එක් කළ හැක. පහළට අනුචලනය කර ASSIGN USERS බොත්තම ක්ලික් කරන්න.

නඩුවක් වෙනස් කිරීම

අවස්ථා කුඩා සංඛ්‍යාවක් යාවත්කාලීන කිරීම සඳහා, සංස්කරණ මෙවලම භාවිතා කර එක් එක් අවස්ථා සඳහා වෙනස්කම් කරන්න.

  • සටහන එක් වරකදී නඩු විශාල සංඛ්‍යාවක් යාවත්කාලීන කිරීමට, ආනයන මෙවලම භාවිතා කිරීම සලකා බලන්න
  1. පරිපාලක ගිණුමට ලොග් වන්න.
  2. යෙදුම් පිටුවේ යෙදුමක් සොයාගෙන ADMIN බොත්තම ක්ලික් කරන්න.
  3. පරිපාලක පැනලයේ, සිද්ධි මෙනු ටැබය ක්ලික් කරන්න.
  4. නඩුවක් සොයාගෙන සංස්කරණය කරන්න ක්ලික් කරන්න.
  5. වෙනස්කම් සිදු කර සුරකින්න ක්ලික් කරන්න.
