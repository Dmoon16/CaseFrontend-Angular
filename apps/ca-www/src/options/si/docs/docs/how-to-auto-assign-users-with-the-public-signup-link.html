අලුතින් ස්වයංක්‍රීයව සාදන ලද නඩුවකට නව පරිශීලකයින් ස්වයංක්‍රීයව පවරනු ලබන පරිදි පද්ධති සැකසුම් වින්‍යාස කරන්නේ කෙසේදැයි පරිපාලකයින් ඉගෙන ගනී.

සැකසුම් පිටුව විවෘත කිරීම

  1. පරිපාලක ගිණුමට ලොග් වන්න.
  2. යෙදුම් පිටුවේ යෙදුමක් සොයාගෙන ADMIN බොත්තම ක්ලික් කරන්න.
  3. පද්ධති පටිත්ත ක්ලික් කර උප මෙනුවේ SYSTEM SETUP ක්ලික් කරන්න.
  4. USER SIGNUP ශීර්ෂයට පහළට අනුචලනය කරන්න.

පද්ධති සැකසුම් වින්‍යාස කිරීම

  1. Allow Signup සහ Auto Create Case drop-down menu On වෙත සකසන්න.
  2. භූමිකාව මෙනුව ක්ලික් කර විකල්පයක් තෝරන්න (පෙරනිමිය නිර්දේශ කෙරේ).
  3. පොදු පරිශීලක ලියාපදිංචි වීමේ සබැඳිය පිටපත් කර බෙදා ගන්න.
