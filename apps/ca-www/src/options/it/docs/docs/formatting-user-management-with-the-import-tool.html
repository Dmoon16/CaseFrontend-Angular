Formattazione dei dati di gestione degli utenti con lo strumento di importazione

Di seguito è riportato un elenco di campi dati nei file CSV. Ogni opzione di importazione ha una diversa combinazione di campi.

nome utente

  • Inserisci l'indirizzo email dell'utente.

nome di battesimo

  • Immettere il nome dell'utente.

cognome

  • Immettere il cognome dell'utente.

locale

  • Questa colonna verifica la lingua preferita dell'utente.
  • Digita en per l'inglese, sp per lo spagnolo.

zoneinfo

  • Imposta il fuso orario dell'utente. Inserisci il continente e la città senza spazi (usa un trattino basso).
  • È necessario utilizzare un'opzione nel menu a discesa Profilo > LINGUA E REGIONE > Fuso orario.
  • Usa il fuso orario più adatto. Ad esempio, Boston non è nell'elenco a discesa, quindi possiamo utilizzare New York (o qualsiasi città nel fuso orario EST/EDT).

host_tag_id

  • Inserisci un'etichetta permanente.
  • az, 0-9, -, / sono consentiti.

caso_stato

  • 0 = chiuso
  • 1 = attivo
