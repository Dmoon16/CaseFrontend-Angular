Formattazione dei dati delle assegnazioni dei casi utente con lo strumento di importazione

Di seguito è riportato un elenco di campi dati nei file CSV. Ogni opzione di importazione ha una diversa combinazione di campi.

ID_caso

  • Inserisci il caso Utente o Tag ID.
  • Vai alla pagina Casi > Modifica > ID tag OPPURE Vai alla pagina Casi > Modifica > ID caso nell'URL.

ID utente

  • Inserisci l'ID dell'utente o l'ID tag.
  • Vai alla pagina Utenti > Modifica > ID tag OPPURE Vai alla pagina Utenti > Modifica > ID utente nell'URL.

caso_ruolo_ordine

  • Lascia la cella vuota.

ID_ruolo_caso

  • Immettere: clienti, personale o predefinito.

case_notify

  • 0 = n
  • 1 = Sì, invia all'utente una notifica e-mail.

imm
