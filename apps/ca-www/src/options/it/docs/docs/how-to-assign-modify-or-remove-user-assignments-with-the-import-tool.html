Come assegnare, modificare o rimuovere le assegnazioni utente con lo strumento di importazione

Questo articolo mostra agli amministratori come gestire gli utenti del caso con lo strumento di importazione.

Gli amministratori possono:

  • assegnare gli utenti a un caso
  • modificare gli utenti già assegnati a un caso
  • rimuovere gli utenti assegnati a un caso

Apertura dello strumento di importazione

  1. Accedi all'account amministratore.
  2. Individua un'app nella pagina App e fai clic sul pulsante ADMIN.
  3. Fare clic sulla scheda Integrazione e quindi su IMPORTAZIONI nel sottomenu.
  4. Fare clic sul pulsante NUOVO. Viene visualizzata la finestra di dialogo Aggiungi importazioni.

Creazione di record

Scarica uno o più file CSV. Utilizzare questi file per aggiungere, modificare o rimuovere gli utenti del caso. Dopo l'apertura della finestra di dialogo Aggiungi importazioni:

  1. Fare clic sul campo Tipo.
  2. Scorri il menu a discesa e seleziona un'opzione (vedi sotto). Viene visualizzato un collegamento.
  3. Fare clic sul collegamento Scarica campione. Un file CSV viene scaricato sul tuo dispositivo.

imm

Apri il file e aggiungi i record. Fare clic qui per informazioni su come utilizzare i dati di formato nel file CSV.

Opzioni file

Assegna utenti a casi * Scegli questa opzione per aggiungere utenti a un caso. * Lascia vuota la cella case_role_order. * Vai alla pagina Utenti e copia l'ID tag per questa persona. Se il campo ID tag è vuoto, inserisci un ID univoco, fai clic su Salva, quindi copia l'ID nel file CSV.

imm

Modifica utenti assegnati a casi * Scegliere questa opzione per aggiornare i dati per gli utenti correnti assegnati a un caso. * Utilizzare l'ID tag caso per la colonna case_ID.

imm

Rimuovi gli utenti assegnati ai casi * Scegliere questa opzione per rimuovere gli utenti assegnati a un caso.

imm

Caricamento del file

Puoi caricare subito un file o tornare alla finestra di dialogo Aggiungi importazioni in un secondo momento e completare questi passaggi.

  1. Aprire la finestra di dialogo Aggiungi importazioni.
  2. Immettere un nome univoco per il file CSV.
  3. Fare clic sull'icona più (+) per caricare il file dal dispositivo.
  4. Fare clic sul campo Tipo e selezionare un'opzione. (vedi sopra)
  5. Fare clic su SALVA.

in lavorazione

A seconda del numero di record nel file e del carico del sistema, la piattaforma può impiegare diversi minuti per elaborare l'importazione.

imm

Al termine dell'importazione, la piattaforma regola i record utente (ovvero aggiunge, modifica o rimuove) nella pagina Casi.

Fallimento

Se l'operazione di importazione non riesce, la piattaforma restituisce un messaggio Failed. Nella maggior parte dei casi, la causa dell'errore è la formattazione dei dati.

imm

Per vedere la causa dell'errore:

  1. Fare clic su Visualizza sotto l'intestazione Azione.
  2. Nella pagina successiva, passa il mouse sull'icona Fallito. La descrizione comandi visualizza l'errore.
  3. Vai al file CVS e regola e correggi l'errore.
  4. Carica di nuovo il file.
