Come creare o modificare utenti con lo strumento di importazione

Questo articolo mostra agli amministratori come creare utenti con lo strumento di importazione.

Apertura dello strumento di importazione

Gli amministratori possono utilizzare lo strumento di importazione per caricare in blocco nuovi utenti e modificare i record degli utenti esistenti.

  1. Accedi all'account amministratore.
  2. Individua un'app nella pagina App e fai clic sul pulsante ADMIN.
  3. Nel pannello di amministrazione, fai clic sulla scheda Integrazione e poi su IMPORTAZIONI nel sottomenu.
  4. Fare clic sul pulsante NUOVO. La finestra di dialogo Aggiungi importazioni viene visualizzata in una nuova scheda del browser.

Creazione di record

Dopo l'apertura della finestra di dialogo Aggiungi importazioni:

  1. Fare clic sul campo Tipo e selezionare un'opzione (vedere di seguito). Viene visualizzato un collegamento.
  2. Fare clic sul collegamento Scarica campione . Un file CSV viene scaricato sul tuo dispositivo.

Successivamente, apri il file e aggiungi i record. Fare clic qui per informazioni su come utilizzare i dati di formato nel file CSV.

imm

Tipo

Quando si fa clic sul campo Tipo, viene visualizzato un menu a discesa. Scegli una delle due opzioni per le importazioni all'ingrosso.

:esclamazione: NOTA Il menu a discesa ha molte opzioni, ma solo due sono relative allo strumento Importa. Ogni opzione ha un file CSV di esempio diverso.

Aggiungi utenti

  • Scegli questa opzione per un semplice caricamento collettivo dei record degli utenti.
  • Fare clic sul collegamento Scarica esempio per aprire un file CSV con le intestazioni di colonna per il tipo di importazione Aggiungi utenti.

Modifica utenti

  • Scegli questa opzione per aggiornare i dati per gli utenti attuali.
  • Fare clic sul collegamento Scarica esempio per aprire un file CSV con le intestazioni di colonna per il tipo di importazione Modifica utenti.

Caricamento del file

Puoi caricare subito un file o tornare alla finestra di dialogo Aggiungi importazioni in un secondo momento e completare questi passaggi.

  1. Aprire la finestra di dialogo Aggiungi importazioni.
  2. Immettere un nome univoco per il file CSV.
  3. Fare clic sull'icona più (+) per caricare il file dal dispositivo.
  4. Fare clic sul campo Tipo e selezionare un'opzione. (vedi sopra)
  5. Fare clic su SALVA.

in lavorazione

A seconda del numero di record nel file e del carico del sistema, la piattaforma può impiegare diversi minuti per elaborare l'importazione.

imm

Al termine dell'importazione, la piattaforma regola i record dell'utente (ovvero aggiunge, modifica o disabilita) nella dashboard Utenti dell'amministratore.

Fallimento

Se l'operazione di importazione non riesce, la piattaforma restituisce un messaggio Failed. Nella maggior parte dei casi, la causa dell'errore è la formattazione dei dati.

imm

Per vedere la causa dell'errore:

  1. Fare clic su Visualizza sotto l'intestazione Azione.
  2. Nella pagina successiva, passa il mouse sull'icona Fallito. La descrizione comandi visualizza l'errore.
  3. Vai al file CVS e regola e correggi l'errore.
  4. Carica di nuovo il file.
