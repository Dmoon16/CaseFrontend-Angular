Come creare utenti con un collegamento di registrazione

Questo articolo mostra agli amministratori come creare utenti inviando un collegamento di registrazione.

Configurazione del collegamento di registrazione

  1. Fare clic sul collegamento Invia loro un'iscrizione nella parte inferiore della finestra di dialogo Aggiungi utente. Viene visualizzata una nuova pagina.
  2. Configura le opzioni di registrazione. (vedi sotto)

imm

Assicurati che il campo Consenti iscrizione sia impostato su On. (obbligatorio) Se abilitato, Case Active genera un collegamento univoco.

Esaminare le opzioni di seguito e quindi fare clic sull'icona per copiare il collegamento. Puoi posizionare il collegamento sul tuo sito Web o inviare il collegamento a una o più persone tramite e-mail o qualsiasi canale di comunicazione.

imm

Creazione di un caso

Facoltativamente, puoi creare automaticamente un caso per ogni nuovo utente e assegnare un ruolo utente.

  1. Fare clic sul campo Creazione automatica caso e selezionare On. Vengono visualizzati i campi Ruolo utente caso.
  2. Fare clic sul ruolo utente del caso e selezionare un'opzione. (vedi sotto)

imm

Ruoli utente caso

Esistono tre ruoli utente del caso.

  • clienti
  • predefinito
  • personale

Invio del collegamento

Condividi il link con una o più persone. Ogni persona fa clic sul collegamento e completa il flusso di lavoro per attivare il proprio account Case Active. Una volta che l'utente ha completato il flusso di lavoro di registrazione, Case Active aggiunge l'utente all'elenco Utenti.

imm

Prossimi passi

Case Active invia all'utente una notifica e-mail con il link di registrazione. L'utente fa clic sul collegamento e completa il flusso di lavoro per le nuove iscrizioni.

  • Questo articolo descrive il flusso di lavoro di registrazione per i nuovi utenti.
