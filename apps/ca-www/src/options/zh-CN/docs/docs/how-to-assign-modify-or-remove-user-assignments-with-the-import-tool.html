如何使用导入工具分配、修改或删除用户分配

本文向管理员展示如何使用导入工具管理案例用户。

管理员可以：

  • 将用户分配给案例
  • 修改已分配给案例的用户
  • 删除分配给案例的用户

打开导入工具

  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 单击 Integration 选项卡，然后单击子菜单上的 IMPORTS。
  4. 单击新建按钮。显示添加导入对话框。

创建记录

下载一个或多个 CSV 文件。使用这些文件来添加、修改或删除案例用户。添加导入对话框打开后：

  1. 单击类型字段。
  2. 滚动下拉菜单并选择一个选项（见下文）。显示一个链接。
  3. 单击下载示例链接。 CSV 文件将下载到您的设备。

打开文件并添加记录。单击此处了解如何在 CSV 文件中使用格式数据。

文件选项

将用户分配给个案* 选择此选项可将用户添加到个案。 * 将 case_role_order 单元格留空。 * 转到“用户”页面并复制此人的标签 ID。如果标记 ID 字段为空，请输入唯一 ID，单击保存，然后将 ID 复制到 CSV 文件。

上传文件

您可以立即上传文件或稍后返回添加导入对话框并完成这些步骤。

  1. 打开添加导入对话框。
  2. 输入 CSV 文件的唯一名称。
  3. 单击加号 (+) 图标从您的设备上传文件。
  4. 单击类型字段并选择一个选项。 （看上面）
  5. 单击保存。

加工

根据文件中记录的数量和系统负载，平台可能需要几分钟时间来处理导入。

图片

导入完成后，平台会在案例页面调整用户记录（即添加、修改或删除）。

失败

如果导入操作失败，平台将返回失败消息。在大多数情况下，失败的原因是数据格式化。

图片

查看失败原因：

  1. 单击“操作”标题下的“查看”。
  2. 在下一页上，将鼠标悬停在“失败”图标上。工具提示显示错误。
  3. 转到 CVS 文件并调整并修复错误。
  4. 再次上传文件。
