如何分配或删除现有个案的用户分配

本文向管理员展示如何分配和删除个案用户。我们还向您展示了如何修改用户数据。

打开现有案例

  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 单击个案菜单选项卡。
  4. 确保在子菜单中选择了活动。
  5. 向下滚动到操作列。

图片

分配或删除

单击分配用户将用户分配到个案或从个案（但不是平台）中删除用户。在下一页上：单击添加并完成工作流程。单击移除以删除用户。

图片

调整

无法在案例页面上编辑或更新用户数据。相反，请按照下列步骤操作：

  1. 单击用户菜单选项卡。
  2. 找到用户名并单击编辑。
  3. 进行更改并单击“保存”。
