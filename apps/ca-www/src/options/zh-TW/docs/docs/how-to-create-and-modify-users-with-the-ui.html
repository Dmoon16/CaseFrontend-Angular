如何使用 UI 創建和修改用戶

本文向管理員展示瞭如何使用平台的用戶界面創建用戶和更新記錄。管理員還可以將用戶添加到新案例並編輯現有案例。

創建用戶

  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 在管理面板上，單擊用戶菜單選項卡。
  4. 單擊新建按鈕。將顯示一個新窗口。

圖片

填寫字段並單擊保存。 Case Active 將名稱添加到與此應用關聯的用戶列表中。

修改用戶

要更新少量用戶記錄，請使用“編輯”工具並對每個用戶進行更改。

  • 注意要一次性修改大量用戶記錄，請考慮使用導入工具。
  1. 在管理面板上，單擊用戶菜單選項卡。
  2. 找到一個用戶並單擊編輯。
  3. 進行更改並單擊“保存”。

下一步

Case Active 不會向用戶發送電子郵件通知。管理員應該聯繫用戶並向他們發送一個指向 Case Active 主頁的鏈接。然後用戶單擊“登錄”按鈕並完成新註冊的工作流程。
