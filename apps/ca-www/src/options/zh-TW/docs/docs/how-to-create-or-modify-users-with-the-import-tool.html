如何使用導入工具創建或修改用戶

本文向管理員展示瞭如何使用導入工具創建用戶。

打開導入工具

管理員可以使用導入工具批量上傳新用戶和修改現有用戶記錄。

  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 在管理面板上，單擊集成選項卡，然後單擊子菜單上的導入。
  4. 單擊新建按鈕。添加導入對話框顯示在新的瀏覽器選項卡中。

創建記錄

添加導入對話框打開後：

  1. 單擊類型字段並選擇一個選項（見下文）。顯示一個鏈接。
  2. 單擊下載示例鏈接。 CSV 文件將下載到您的設備。

接下來，打開文件並添加記錄。單擊此處了解如何在 CSV 文件中使用格式數據。

圖片

類型

單擊“類型”字段時，將顯示一個下拉菜單。選擇批量導入的兩個選項之一。

:exclamation:注意下拉菜單有很多選項，但只有兩個與導入工具相關。每個選項都有一個不同的示例 CSV 文件。

添加用戶

  • 選擇此選項可簡單地批量上傳用戶記錄。
  • 單擊“下載示例”鏈接以打開一個 CSV 文件，其中包含“添加用戶”導入類型的列標題。

修改用戶

  • 選擇此選項可為當前用戶更新數據。
  • 單擊下載示例鏈接打開一個 CSV 文件，其中列標題為修改用戶導入類型。

上傳文件

您可以立即上傳文件或稍後返回添加導入對話框並完成這些步驟。

  1. 打開添加導入對話框。
  2. 輸入 CSV 文件的唯一名稱。
  3. 單擊加號 (+) 圖標從您的設備上傳文件。
  4. 單擊類型字段並選擇一個選項。 （看上面）
  5. 單擊保存。

加工

根據文件中記錄的數量和系統負載，平台可能需要幾分鐘時間來處理導入。

圖片

導入完成後，平台會在管理員的用戶儀表板上調整用戶記錄（即添加、修改或禁用）。

失敗

如果導入操作失敗，平台將返回失敗消息。在大多數情況下，失敗的原因是數據格式化。

圖片

查看失敗原因：

  1. 單擊“操作”標題下的“查看”。
  2. 在下一頁上，將鼠標懸停在“失敗”圖標上。工具提示顯示錯誤。
  3. 轉到 CVS 文件並調整併修復錯誤。
  4. 再次上傳文件。
