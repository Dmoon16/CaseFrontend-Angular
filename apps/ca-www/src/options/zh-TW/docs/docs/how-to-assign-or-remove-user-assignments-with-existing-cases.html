如何分配或刪除現有個案的用戶分配

本文向管理員展示如何分配和刪除個案用戶。我們還向您展示瞭如何修改用戶數據。

打開現有案例

  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 單擊個案菜單選項卡。
  4. 確保在子菜單中選擇了活動。
  5. 向下滾動到操作列。

圖片

分配或刪除

單擊分配用戶將用戶分配到個案或從個案（但不是平台）中刪除用戶。在下一頁上：單擊添加並完成工作流程。單擊移除以刪除用戶。

圖片

調整

無法在案例頁面上編輯或更新用戶數據。相反，請按照下列步驟操作：

  1. 單擊用戶菜單選項卡。
  2. 找到用戶名並單擊編輯。
  3. 進行更改並單擊“保存”。
