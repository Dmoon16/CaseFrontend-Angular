如何使用導入工具創建或修改案例

本文向管理員展示如何使用導入工具創建或修改現有案例。

打開導入工具

  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 在管理面板上，單擊集成選項卡，然後單擊子菜單上的導入。
  4. 單擊新建按鈕。添加導入對話框顯示在新的瀏覽器選項卡中。

創建案例

添加導入對話框打開後：

  1. 單擊類型字段並選擇一個選項（見下文）。顯示一個鏈接。
  2. 單擊下載示例鏈接。 CSV 文件將下載到您的設備。

接下來，打開文件並添加案例。

類型

單擊“類型”字段時，將顯示一個下拉菜單。選擇批量案例導入的兩個選項之一。

  • 注意下拉菜單有很多選項，但只有兩個與管理案例相關。每個選項都有一個不同的示例 CSV 文件。

添加案例

  • 選擇此選項以簡單地批量上傳個案。
  • 單擊下載示例鏈接打開一個 CSV 文件，其中包含創建案例導入類型的列標題。

修改案例

  • 選擇此選項以更新當前案例。
  • 單擊下載示例鏈接打開一個 CSV 文件，其中包含修改案例導入類型的列標題。

上傳文件

您可以立即上傳文件或稍後返回添加導入對話框並完成這些步驟。

  1. 打開添加導入對話框。
  2. 輸入 CSV 文件的唯一名稱。
  3. 單擊加號 (+) 圖標從您的設備上傳文件。
  4. 單擊類型字段並選擇一個選項。 （看上面）
  5. 單擊保存。

格式化

這些是必填字段和首要要求。單擊此處查看格式詳細信息。

tag_id（或 case_ID）唯一標識符（使用字母數字字符，接受連字符）

case_status 0 = 關閉 1 = 活動

date_opened月/日/年 mm/dd/yyyy

關於案例的獨特描述（使用字母數字字符，接受連字符）

加工

根據文件中記錄的數量和系統負載，平台可能需要幾分鐘時間來處理導入。

圖片

導入完成後，平台會在管理員的案例儀表板上調整案例記錄（即添加或修改）。

失敗

如果導入操作失敗，平台將返回失敗消息。在大多數情況下，失敗的原因是數據格式化。

圖片

查看失敗原因：

  1. 單擊操作標題下的查看。
  2. 在下一頁上，將鼠標懸停在“失敗”圖標上。工具提示顯示錯誤。
  3. 轉到 CVS 文件並調整併修復錯誤。
  4. 再次上傳文件。
