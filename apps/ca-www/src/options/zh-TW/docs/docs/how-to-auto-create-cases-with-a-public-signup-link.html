如何使用公共註冊鏈接自動創建案例

本文向管理員展示瞭如何通過發送單個註冊鏈接來創建案例（並同時添加用戶）。

定位鏈接

  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 在管理面板上，單擊設置菜單選項卡。
  4. 確保選擇了系統設置子菜單選項卡。

圖片

配置設置

向下滾動到用戶註冊標題。

確保允許註冊字段設置為開。 （必需）啟用此功能後，Case Active 會生成一個唯一鏈接。

  1. 單擊“自動創建案例”字段並選擇“打開”。個案用戶角色字段顯示。
  2. 選擇角色。 （見下文）

圖片

個案用戶角色

共有三種案例用戶角色。

  • 客戶端（推薦）
  • 默認
  • 職員

發送鏈接

與一個或多個人共享鏈接。單擊鏈接並完成註冊的每個用戶都會觸發平台在案例頁面上生成一個新案例並自動分配案例中的用戶。
