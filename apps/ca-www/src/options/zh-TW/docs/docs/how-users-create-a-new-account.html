用戶如何創建新帳戶

管理員將用戶添加到個案後： * 如果用戶是個案新手，則用戶必須完成工作流程以創建帳戶並接受訪問個案的邀請。 * 如果用戶有現有的案例帳戶，則用戶必須接受邀請才能訪問案例。

本文介紹了用戶創建帳戶必須完成的步驟。

入門

可以通過兩種方式將用戶添加到案例應用程序

  • 通過管理員：如果管理員通過平台 UI或導入工具創建用戶，則管理員必須向用戶發送指向 account.caseactive.com 的鏈接以繼續登錄。
  • 按用戶：如果用戶使用註冊鏈接創建了一個帳戶，用戶將被自動重定向到 account.caseactive.com 以繼續登錄。

在登錄

當用戶首次登錄其 CaseActive 帳戶時，平台會提示用戶：

  • 打開他們的電子郵件帳戶，然後復制/粘貼六位數的安全代碼
  • 完成帳戶資料

圖片

用戶單擊更新後，將顯示應用程序頁面。該頁面顯示應用程序名稱（公司名稱）、用戶類型（角色）、狀態和邀請按鈕。

圖片

要訪問應用程序，用戶必須接受邀請。用戶應該：

  1. 單擊邀請按鈕。
  2. 查看權限。
  3. 單擊“通知”選項卡以查看和編輯（可選）條款。
  4. 單擊批准訪問。

圖片

當用戶同意後，平台會在用戶頁面顯示一個APP按鈕。此按鈕允許用戶訪問分配的案例。

要編輯用戶設置，用戶單擊省略號。

圖片

用戶可以： * 刪除應用訪問權限 * 修改通知設置

圖片

用戶完成工作流程後，用戶名會出現在管理員的用戶列表中。該列表顯示所有用戶，無論是否登錄。
