使用導入工具格式化用戶管理數據

以下是 CSV 文件中的數據字段列表。每個導入選項都有不同的字段組合。

用戶名

  • 輸入用戶的電子郵件地址。

給定的名稱

  • 輸入用戶的名字。

姓

  • 輸入用戶的姓氏。

語言環境

  • 此列檢查用戶的首選語言。
  • 輸入 en 表示英語，sp 表示西班牙語。

區域信息

  • 這將設置用戶時區。輸入沒有空格的大陸和城市（使用下劃線）。
  • 必須使用配置文件 > 語言和地區 > 時區下拉菜單中的選項。
  • 使用最合適的時區。例如，波士頓不在下拉列表中，因此我們可以使用紐約（或 EST/EDT 時區中的任何城市）。

host_tag_id

  • 輸入永久標籤。
  • 允許使用 az、0-9、-、/。

案例狀態

  • 0 = 關閉
  • 1 = 活躍
