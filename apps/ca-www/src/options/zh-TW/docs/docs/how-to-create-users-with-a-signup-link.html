如何使用註冊鏈接創建用戶

本文向管理員展示瞭如何通過發送註冊鏈接來創建用戶。

配置註冊鏈接

  1. 單擊添加用戶對話框底部的向他們發送註冊鏈接。顯示一個新頁面。
  2. 配置註冊選項。 （見下文）

圖片

確保允許註冊字段設置為開。 （必需）啟用後，Case Active 會生成一個唯一鏈接。

查看以下選項，然後單擊圖標以復制鏈接。您可以將鏈接放在您的網站上，或通過電子郵件或任何通信渠道將鏈接發送給一個或多個人。

圖片

創建案例

或者，您可以為每個新用戶自動創建案例並分配用戶角色。

  1. 單擊“自動創建案例”字段並選擇“打開”。個案用戶角色字段顯示。
  2. 單擊案例用戶角色並選擇一個選項。 （見下文）

圖片

個案用戶角色

存在三種案例用戶角色。

  • 客戶
  • 默認
  • 職員

發送鏈接

與一個或多個人共享鏈接。每個人都單擊鏈接並完成工作流程以激活他們的 Case Active 帳戶。用戶完成註冊工作流程後，Case Active 會將用戶添加到用戶列表中。

圖片

下一步

Case Active 向用戶發送帶有註冊鏈接的電子郵件通知。用戶單擊鏈接並完成新註冊的工作流程。
