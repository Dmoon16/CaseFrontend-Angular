如何使用公共註冊鏈接自動分配用戶

管理員學習如何配置系統設置，以便將新用戶自動分配給新自動創建的案例。

打開設置頁面

第一步是打開“系統設置”頁面。

  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 單擊系統選項卡，然後單擊子菜單上的系統設置。
  4. 向下滾動到用戶註冊標題。

配置系統設置

  1. 將允許註冊和自動創建案例下拉菜單設置為開。
  2. 單擊角色菜單並選擇一個選項（建議使用默認值）。
  3. 複製並共享公共用戶註冊鏈接。

發送鏈接

單擊鏈接並完成註冊的每個用戶都會觸發平台在案例頁面上生成一個新案例，並在該案例中自動分配用戶。
