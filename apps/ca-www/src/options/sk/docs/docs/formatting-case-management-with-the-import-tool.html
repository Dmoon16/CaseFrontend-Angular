Formátovanie údajov správy prípadov pomocou nástroja na import

Nižšie je uvedený zoznam povinných polí v súboroch CSV na správu prípadov pomocou nástroja Import.

užívateľské meno

  • Zadajte e-mailovú adresu používateľa.

tag_id (identifikátor, na ktorý sa môžete odvolať neskôr)

  • jedinečný identifikátor (použite alfanumerické znaky, pomlčky sú akceptované)

case_status

  • 0 = zatvorené
  • 1 = aktívny

date_opened

  • mesiac deň rok
  • mm/dd/rrrr

o

  • jedinečný popis prípadu (použite alfanumerické znaky, pomlčky sú akceptované)
