Formátovanie údajov priradení prípadu používateľa pomocou nástroja na import

Nižšie je uvedený zoznam dátových polí v súboroch CSV. Každá možnosť importu má inú kombináciu polí.

case_ID

  • Zadajte ID používateľa alebo značky prípadu.
  • Prejdite na stránku Prípady > Upraviť > ID značky ALEBO prejdite na stránku Prípady > Upraviť > ID prípadu v adrese URL.

ID používateľa

  • Zadajte ID používateľa alebo ID značky.
  • Prejdite na stránku Používatelia > Upraviť > ID značky ALEBO prejdite na stránku Používatelia > Upraviť > ID používateľa v adrese URL.

case_role_order

  • Nechajte bunku prázdnu.

case_role_ID

  • Zadajte: klienti, zamestnanci alebo predvolená hodnota.

case_notify

  • 0 = nie
  • 1 = Áno, poslať používateľovi e-mailové upozornenie.
