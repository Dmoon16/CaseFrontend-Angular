قالب‌بندی داده‌های تکالیف پرونده کاربر با ابزار Import

در زیر لیستی از فیلدهای داده در فایل های CSV آمده است. هر گزینه واردات دارای ترکیب متفاوتی از فیلدها است.

case_ID

  • شناسه کاربر یا برچسب موردی را وارد کنید.
  • به صفحه Cases > Edit > Tag ID یا به صفحه Cases > Edit > Case ID در url بروید.

شناسه کاربر

  • شناسه یا شناسه برچسب کاربر را وارد کنید.
  • به صفحه کاربران > ویرایش > شناسه برچسب یا به صفحه کاربران > ویرایش > شناسه کاربر در آدرس اینترنتی بروید.

مورد_نقش_سفارش

  • سلول را خالی بگذارید.

شناسه_نقش_مورد

  • را وارد کنید: مشتریان، کارکنان، یا پیش فرض.

مورد_اطلاع رسانی

  • 0 = خیر
  • 1 = بله، یک اعلان ایمیل برای کاربر ارسال شود.
