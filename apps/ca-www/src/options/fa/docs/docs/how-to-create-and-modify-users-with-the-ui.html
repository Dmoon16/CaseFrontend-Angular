نحوه ایجاد و تغییر کاربران با رابط کاربری

این مقاله به مدیران نحوه ایجاد کاربران و به‌روزرسانی رکوردها با رابط کاربری پلتفرم را نشان می‌دهد. ادمین ها همچنین می توانند کاربران را به یک پرونده جدید اضافه کنند و موارد موجود را ویرایش کنند.

ایجاد کاربر

  1. وارد حساب کاربری مدیریت شوید.
  2. یک برنامه را در صفحه برنامه ها پیدا کنید و روی دکمه ADMIN کلیک کنید.
  3. در پنل مدیریت، روی تب منوی کاربران کلیک کنید.
  4. روی دکمه NEW کلیک کنید. یک پنجره جدید نمایش داده می شود.

img

فیلدها را پر کرده و SAVE را بزنید. Case Active نام را به لیست کاربران مرتبط با این برنامه اضافه می کند.

img

تغییر کاربری

برای به روز رسانی تعداد کمی از سوابق کاربر، از ابزار ویرایش استفاده کنید و تغییراتی را برای هر کاربر ایجاد کنید.

  • توجه برای تغییر تعداد زیادی از رکوردهای کاربر در یک جلسه، از ابزار Import استفاده کنید .
  1. در پنل مدیریت، روی تب منوی کاربران کلیک کنید.
  2. یک کاربر را پیدا کنید و روی ویرایش کلیک کنید.
  3. تغییرات ایجاد کنید و روی ذخیره کلیک کنید.

img

مراحل بعدی

Case Active اعلان ایمیلی برای کاربر ارسال نمی کند. مدیران باید با کاربر تماس بگیرند و پیوندی به صفحه اصلی Case Active برای آنها ارسال کنند. سپس کاربر روی دکمه LOGIN کلیک کرده و گردش کار را برای ثبت نام های جدید تکمیل می کند.

  • این مقاله گردش کار ثبت نام برای کاربران جدید را شرح می دهد.
