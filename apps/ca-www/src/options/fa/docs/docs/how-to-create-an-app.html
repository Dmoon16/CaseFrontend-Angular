نحوه ایجاد اپلیکیشن

این مقاله به کاربران جدید نحوه ایجاد یک حساب آزمایشی رایگان را نشان می دهد. Case Active به طور خودکار به کاربر جدید امتیازات مدیریت این حساب را اعطا می کند.

ایجاد یک حساب کاربری

  1. به صفحه ورود بروید.
  2. روی دکمه Get Started کلیک کنید.
  3. یک آدرس ایمیل وارد کنید.
  4. روی Next کلیک کنید.

img

تایید یک حساب کاربری

  1. یک برگه مرورگر جدید باز کنید و به حساب ایمیل خود وارد شوید.
  2. کد شش رقمی را کپی کنید.
  3. به تب Case Active برگردید.
  4. کد را در فیلدها قرار دهید. یک پنجره جدید نمایش داده می شود.

در پنجره جدید:

  1. فیلدهای مورد نیاز را پر کنید.
  2. شرایط خدمات را بخوانید و برای موافقت، کادر را علامت بزنید.
  3. روی ایجاد یک حساب رایگان کلیک کنید.

img

تکمیل تنظیمات برنامه

یک گردش کار در صفحه بعد نمایش داده می شود. مراحل را برای تکمیل راه‌اندازی برنامه خود کامل کنید. همیشه می توانید برای ویرایش یا به روز رسانی اطلاعات یا وضعیت حساب خود به این صفحات بازگردید.

در صفحه درباره برنامه، فیلدهای نام شرکت و وب سایت را پر کنید (الزامی) و سپس روی Next کلیک کنید.

img

در صفحه اطلاعات صورتحساب، روی قسمت طرح پرداخت کلیک کنید و گزینه ها را بررسی کنید. تنظیمات پیش فرض رایگان است. روی SUBMIT کلیک کنید.

img

پس از ارسال فرم، Case Active صفحه Apps را نمایش می دهد. برای هر برنامه ای که ایجاد می کنید یا به آن تعلق دارید، صفحه نشان می دهد:

  1. نام برنامه
  2. نوع شما (نقش کاربر)
  3. وضعیت

img

صفحه برنامه ها نقطه راه اندازی برای بسیاری از عملکردهای سرپرست است:

  1. برای برقراری ارتباط، پر کردن اسناد، چت ویدیویی (و موارد دیگر) در مورد مسائل مربوط به پرونده، روی دکمه APP کلیک کنید.
  2. روی دکمه ADMIN کلیک کنید تا پنل مدیریت باز شود. اینجا جایی است که ادمین ها کاربران، موارد، ادغام ها و موارد دیگر را مدیریت می کنند.
  3. برای مدیریت تنظیمات صورتحساب و اعلان حساب، روی بیضی کلیک کنید.
  • Note Case Active دکمه ADMIN را برای کاربران ادمین نمایش می دهد. کاربرانی که امتیازات مدیریت ندارند، دکمه ADMIN را نمی بینند (همانطور که در صفحه زیر نشان داده شده است).
