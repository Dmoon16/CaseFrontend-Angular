نحوه ایجاد کاربران با لینک ثبت نام

این مقاله به مدیران نشان می دهد که چگونه با ارسال لینک ثبت نام، کاربر ایجاد کنند.

پیکربندی لینک ثبت نام

  1. روی پیوند Send them a signup در پایین کادر محاوره ای افزودن کاربر کلیک کنید. یک صفحه جدید نمایش داده می شود.
  2. گزینه های ثبت نام را پیکربندی کنید. (پایین را ببینید)

img

مطمئن شوید که قسمت Allow Signup روی On تنظیم شود. (الزامی) وقتی فعال باشد، Case Active یک پیوند منحصر به فرد ایجاد می کند.

گزینه های زیر را بررسی کنید و سپس روی نماد کلیک کنید تا پیوند را کپی کنید. می توانید لینک را در وب سایت خود قرار دهید یا پیوند را از طریق ایمیل یا هر کانال ارتباطی برای یک یا چند نفر ارسال کنید.

img

ایجاد پرونده

به صورت اختیاری، می‌توانید به‌طور خودکار برای هر کاربر جدید یک مورد ایجاد کنید و یک نقش کاربری اختصاص دهید.

  1. روی قسمت Auto Create Case کلیک کنید و On را انتخاب کنید. قسمت های Case User Role نمایش داده می شود.
  2. روی Case User Role کلیک کنید و یک گزینه را انتخاب کنید. (پایین را ببینید)

img

نقش های کاربر موردی

سه نقش کاربر موردی وجود دارد.

  • مشتریان
  • پیش فرض
  • کارکنان

ارسال لینک

پیوند را با یک یا چند نفر به اشتراک بگذارید. هر فرد روی پیوند کلیک می کند و گردش کار را تکمیل می کند تا حساب Case Active خود را فعال کند. هنگامی که کاربر گردش کار ثبت نام را کامل کرد، Case Active کاربر را به لیست کاربران اضافه می کند.

img

مراحل بعدی

Case Active یک اعلان ایمیل با لینک ثبت نام برای کاربر ارسال می کند. کاربر روی پیوند کلیک می کند و گردش کار را برای ثبت نام های جدید تکمیل می کند.

  • این مقاله گردش کار ثبت نام برای کاربران جدید را شرح می دهد.
