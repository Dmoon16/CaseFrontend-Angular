قالب‌بندی داده‌های مدیریت پرونده با ابزار Import

در زیر فهرستی از فیلدهای مورد نیاز در فایل‌های CSV برای مدیریت موارد با ابزار Import آمده است.

نام کاربری

  • آدرس ایمیل کاربر را وارد کنید.

tag_id (شناسه ای که بعدا می توانید به آن ارجاع دهید)

  • یک شناسه منحصر به فرد (از کاراکترهای الفبایی استفاده کنید، خط فاصله پذیرفته می شود)

مورد_وضعیت

  • 0 = بسته
  • 1 = فعال

date_opened

  • ماه روز سال
  • mm/dd/yyyy

در باره

  • شرح منحصر به فرد مورد (استفاده از کاراکترهای الفبایی عددی، خط فاصله پذیرفته می شود)
