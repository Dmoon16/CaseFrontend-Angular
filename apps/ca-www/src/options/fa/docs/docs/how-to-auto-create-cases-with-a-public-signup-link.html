نحوه ایجاد خودکار موارد با پیوند ثبت نام عمومی

این مقاله به مدیران نشان می دهد که چگونه با ارسال یک لینک ثبت نام، یک مورد (و همزمان اضافه کردن یک کاربر) ایجاد کنند.

مکان یابی لینک

  1. وارد حساب کاربری مدیریت شوید.
  2. یک برنامه را در صفحه برنامه ها پیدا کنید و روی دکمه ADMIN کلیک کنید.
  3. در پنل مدیریت، روی تب منوی تنظیمات کلیک کنید.
  4. مطمئن شوید که زبانه زیر منوی SYSTEM SETUP انتخاب شده است.

پیکربندی تنظیمات

به عنوان USER SIGNUP به پایین بروید.

مطمئن شوید که قسمت Allow Signup روی On تنظیم شده است. (الزامی) هنگامی که این ویژگی فعال است، Case Active یک پیوند منحصر به فرد ایجاد می کند.

  1. روی قسمت Auto Create Case کلیک کرده و On را انتخاب کنید. قسمت Case User Role نمایش داده می شود.
  2. یک نقش را انتخاب کنید. (پایین را ببینید)

نقش های کاربر موردی

سه نقش کاربر موردی وجود دارد.

  • مشتریان (توصیه می شود)
  • پیش فرض
  • کارکنان

ارسال لینک

پیوند را با یک یا چند نفر به اشتراک بگذارید. هر کاربری که روی پیوند کلیک می‌کند و ثبت‌نام را تکمیل می‌کند، پلتفرم را راه‌اندازی می‌کند تا یک مورد جدید در صفحه موارد ایجاد کند و کاربر را به صورت خودکار در کیس اختصاص دهد.
