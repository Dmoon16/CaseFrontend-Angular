نحوه ایجاد یا اصلاح موارد از طریق منوی UI

این مقاله نحوه ایجاد کیس با رابط کاربری پلتفرم را به مدیران نشان می دهد. ادمین ها همچنین می توانند کاربران را به یک پرونده جدید اضافه کنند و موارد موجود را به روز کنند.

ایجاد پرونده

  1. وارد حساب کاربری مدیریت شوید.
  2. یک برنامه را در صفحه برنامه ها پیدا کنید و روی دکمه ADMIN کلیک کنید.
  3. در پنل مدیریت، روی تب منو Cases کلیک کنید.
  4. روی دکمه NEW کلیک کنید. یک پنجره جدید نمایش داده می شود.

img

فیلدهای مورد نیاز را پر کرده و SAVE را کلیک کنید.

img

افزودن کاربران

در صورت تمایل، مدیران می توانند کاربران را در این صفحه به پرونده اضافه کنند. به پایین بروید و روی دکمه ASSIGN USERS کلیک کنید.

img

در بخش Assign User، فیلدها را پر کنید. برای ارسال اعلان ایمیل به کاربر، کادر را علامت بزنید. (اختیاری)

img

اصلاح یک مورد

برای به روز رسانی تعداد کمی از موارد، از ابزار ویرایش استفاده کنید و تغییراتی را در هر مورد جداگانه ایجاد کنید.

  • توجه برای به‌روزرسانی تعداد زیادی از موارد در یک جلسه، از ابزار Import استفاده کنید
  1. وارد حساب کاربری مدیریت شوید.
  2. یک برنامه را در صفحه برنامه ها پیدا کنید و روی دکمه ADMIN کلیک کنید.
  3. در پنل مدیریت، روی تب منو Cases کلیک کنید.
  4. یک مورد را پیدا کنید و روی Edit کلیک کنید.
  5. تغییرات ایجاد کنید و روی ذخیره کلیک کنید.
