როგორ შევქმნათ ან შეცვალოთ საქმეები UI მენიუს მეშვეობით

ეს სტატია აჩვენებს ადმინისტრატორებს, თუ როგორ უნდა შექმნან ქეისი პლატფორმის მომხმარებლის ინტერფეისით. ადმინისტრატორებს ასევე შეუძლიათ მომხმარებლების დამატება ახალ საქმეში და განაახლონ არსებული შემთხვევები.

საქმის შექმნა

  1. შედით ადმინისტრატორის ანგარიშზე.
  2. იპოვნეთ აპლიკაცია აპების გვერდზე და დააჭირეთ ღილაკს ADMIN.
  3. ადმინისტრაციულ პანელზე დააწკაპუნეთ ქეისების მენიუს ჩანართზე.
  4. დააჭირეთ ღილაკს NEW. გამოჩნდება ახალი ფანჯარა.

შეავსეთ საჭირო ველები და დააჭირეთ შენახვას.

მომხმარებლების დამატება

სურვილისამებრ, ადმინისტრატორებს შეუძლიათ ამ გვერდზე მომხმარებლების დამატება საქმეში. გადადით ქვემოთ და დააწკაპუნეთ ღილაკზე ASSIGN USERS .

Assign User modal-ში შეავსეთ ველები. მონიშნეთ ყუთი მომხმარებლის ელ.ფოსტის შეტყობინების გასაგზავნად. (სურვილისამებრ)

საქმის შეცვლა

მცირე რაოდენობის შემთხვევების გასაახლებლად გამოიყენეთ რედაქტირების ხელსაწყო და შეიტანეთ ცვლილებები თითოეულ ცალკეულ შემთხვევაზე.

  • შენიშვნა იმისათვის, რომ განაახლოთ შემთხვევების დიდი რაოდენობა ერთ სხდომაზე, განიხილეთ იმპორტის ინსტრუმენტის გამოყენება
  1. შედით ადმინისტრატორის ანგარიშზე.
  2. იპოვნეთ აპლიკაცია აპების გვერდზე და დააჭირეთ ღილაკს ADMIN.
  3. ადმინისტრაციულ პანელზე დააწკაპუნეთ ქეისების მენიუს ჩანართზე.
  4. იპოვნეთ საქმე და დააწკაპუნეთ რედაქტირებაზე.
  5. შეიტანეთ ცვლილებები და დააჭირეთ შენახვას.
