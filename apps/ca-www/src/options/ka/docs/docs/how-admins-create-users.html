როგორ ქმნიან ადმინები მომხმარებლებს

ადმინისტრატორები ქმნიან, ცვლიან ან თიშავენ, ვისაც შეუძლია აპებსა და საქმეებზე წვდომა. როდესაც მომხმარებელი პირველად შედის პლატფორმაზე, მათ უნდა დაასრულონ პროფილის სამუშაო პროცესი.

  • დააწკაპუნეთ აქ , რომ შეიტყოთ მეტი მომხმარებლის ახალი პროფილის სამუშაო პროცესის შესახებ.
