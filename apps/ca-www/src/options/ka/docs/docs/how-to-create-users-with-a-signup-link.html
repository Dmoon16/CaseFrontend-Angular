როგორ შევქმნათ მომხმარებლები რეგისტრაციის ბმულით

ეს სტატია აჩვენებს ადმინისტრატორებს, თუ როგორ შექმნან მომხმარებლები რეგისტრაციის ბმულის გაგზავნით.

მიმდინარეობს რეგისტრაციის ბმულის კონფიგურაცია

  1. დააწკაპუნეთ მათ რეგისტრაციის ბმულზე გაგზავნის ბმულზე მომხმარებლის დამატება დიალოგის ფანჯრის ბოლოში. გამოჩნდება ახალი გვერდი.
  2. რეგისტრაციის პარამეტრების კონფიგურაცია. (იხილეთ ქვემოთ)

დარწმუნდით, რომ ნება დართოთ რეგისტრაციის ველი ჩართული იყოს. (აუცილებელია) როდესაც ჩართულია, Case Active წარმოქმნის უნიკალურ ბმულს.

გადახედეთ ქვემოთ მოცემულ ვარიანტებს და შემდეგ დააწკაპუნეთ ხატულაზე ბმულის დასაკოპირებლად. თქვენ შეგიძლიათ განათავსოთ ბმული თქვენს ვებსაიტზე ან გაუგზავნოთ ბმული ერთ ან მეტ ადამიანს ელექტრონული ფოსტით ან ნებისმიერი საკომუნიკაციო არხის საშუალებით.

საქმის შექმნა

სურვილისამებრ, შეგიძლიათ ავტომატურად შექმნათ საქმე ყოველი ახალი მომხმარებლისთვის და მიანიჭოთ მომხმარებლის როლი.

  1. დააწკაპუნეთ ველზე Auto Create Case და აირჩიეთ ჩართვა. გამოჩნდება Case User Role ველები.
  2. დააწკაპუნეთ საქმის მომხმარებლის როლზე და აირჩიეთ ვარიანტი. (იხილეთ ქვემოთ)

  • ნაგულისხმევი
  • პერსონალი

ლინკის გაგზავნა

გაუზიარეთ ბმული ერთ ან მეტ ადამიანს. თითოეული ადამიანი დააჭერს ბმულს და ასრულებს სამუშაო პროცესს, რათა გაააქტიუროს თავისი Case Active ანგარიში. მას შემდეგ რაც მომხმარებელი დაასრულებს რეგისტრაციის სამუშაო პროცესს, Case Active ამატებს მომხმარებელს მომხმარებლების სიაში.

Შემდეგი ნაბიჯები

Case Active მომხმარებელს ელფოსტის შეტყობინებას უგზავნის რეგისტრაციის ბმულით. მომხმარებელი დააჭერს ბმულს და ასრულებს სამუშაო პროცესს ახალი რეგისტრაციისთვის.

  • ეს სტატია აღწერს რეგისტრაციის სამუშაო პროცესს ახალი მომხმარებლებისთვის.
