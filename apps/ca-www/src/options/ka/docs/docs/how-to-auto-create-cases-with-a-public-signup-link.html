როგორ შევქმნათ ქეისები ავტომატურად საჯარო რეგისტრაციის ბმულით

ეს სტატია გვიჩვენებს ადმინისტრატორებს, თუ როგორ უნდა შექმნან ქეისი (და ამავე დროს დაამატონ მომხმარებელი) ერთი რეგისტრაციის ბმულის გაგზავნით.

ბმულის განთავსება

  1. შედით ადმინისტრატორის ანგარიშზე.
  2. იპოვნეთ აპლიკაცია აპების გვერდზე და დააჭირეთ ღილაკს ADMIN.
  3. ადმინისტრაციულ პანელზე დააწკაპუნეთ მენიუს პარამეტრების ჩანართზე.
  4. დარწმუნდით, რომ არჩეულია SYSTEM SETUP ქვემენიუ ჩანართი.

პარამეტრების კონფიგურაცია

გადადით ქვემოთ USER SIGNUP სათაურზე.

დარწმუნდით, რომ რეგისტრაციის დაშვების ველი დაყენებულია ჩართვაზე. (აუცილებელია) როდესაც ეს ფუნქცია ჩართულია, Case Active წარმოქმნის უნიკალურ ბმულს.

  1. დააწკაპუნეთ ველზე Auto Create Case და აირჩიეთ ჩართვა. გამოჩნდება Case User Role ველი.
  2. აირჩიეთ როლი. (იხილეთ ქვემოთ)

საქმის მომხმარებლის როლები

არსებობს სამი შემთხვევის მომხმარებლის როლი.

  • კლიენტები (რეკომენდებულია)
  • ნაგულისხმევი
  • პერსონალი

ლინკის გაგზავნა

გაუზიარეთ ბმული ერთ ან მეტ ადამიანს. თითოეული მომხმარებელი, რომელიც დააჭერს ბმულს და დაასრულებს რეგისტრაციას, ააქტიურებს პლატფორმას ახალი შემთხვევის გენერირებისთვის საქმეების გვერდზე და ავტომატურად მიაკუთვნებს მომხმარებელს საქმეში.
