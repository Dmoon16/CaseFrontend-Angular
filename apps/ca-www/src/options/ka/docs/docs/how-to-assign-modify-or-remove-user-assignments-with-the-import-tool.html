როგორ მივცეთ, შეცვალოთ ან წაშალოთ მომხმარებლის დავალებები იმპორტის ხელსაწყოთი

ეს სტატია აჩვენებს ადმინისტრატორებს, თუ როგორ უნდა მართონ ქეისის მომხმარებლები იმპორტის ხელსაწყოთი.

ადმინისტრატორებს შეუძლიათ:

  • მომხმარებლების მინიჭება საქმეზე
  • შეცვალეთ საქმისთვის უკვე მინიჭებული მომხმარებლები
  • წაშალეთ საქმეზე მინიჭებული მომხმარებლები

იმპორტის ხელსაწყოს გახსნა

  1. შედით ადმინისტრატორის ანგარიშზე.
  2. იპოვნეთ აპლიკაცია აპების გვერდზე და დააჭირეთ ღილაკს ADMIN.
  3. დააჭირეთ ინტეგრაციის ჩანართს და შემდეგ იმპორტს ქვემენიუში.
  4. დააჭირეთ ღილაკს NEW. გამოჩნდება იმპორტის დამატების დიალოგური ფანჯარა.

ჩანაწერების შექმნა

ჩამოტვირთეთ ერთი ან მეტი CSV ფაილი. გამოიყენეთ ეს ფაილები ქეისის მომხმარებლების დასამატებლად, შესაცვლელად ან წასაშლელად. იმპორტის დამატების დიალოგური ფანჯრის გახსნის შემდეგ:

  1. დააწკაპუნეთ ველზე Type.
  2. გადაახვიეთ ჩამოსაშლელ მენიუში და აირჩიეთ ვარიანტი (იხ. ქვემოთ). გამოჩნდება ბმული.
  3. დააჭირეთ ჩამოტვირთვა Sample ბმულს. CSV ფაილი ჩამოიტვირთება თქვენს მოწყობილობაში.

img

გახსენით ფაილი და დაამატეთ ჩანაწერები. დააწკაპუნეთ აქ, რომ გაიგოთ, როგორ გამოიყენოთ ფორმატის მონაცემები CSV ფაილში.

ფაილის პარამეტრები

მომხმარებლების მინიჭება ქეისებზე * აირჩიეთ ეს ვარიანტი, რათა დაამატოთ მომხმარებლები საქმეში. * დატოვეთ case_role_order უჯრედი ცარიელი. * გადადით მომხმარებლების გვერდზე და დააკოპირეთ ამ პიროვნების Tag ID. თუ Tag ID ველი ცარიელია, შეიყვანეთ უნიკალური ID, დააწკაპუნეთ Save-ზე და შემდეგ დააკოპირეთ ID CSV ფაილში.

შემთხვევისთვის მინიჭებული მომხმარებლების შეცვლა * აირჩიეთ ეს ვარიანტი, რათა განაახლოთ საქმისთვის მინიჭებული ამჟამინდელი მომხმარებლების მონაცემები. * გამოიყენეთ Case Tag ID სვეტისთვის case_ID.

საქმეებზე მინიჭებული მომხმარებლების წაშლა * აირჩიეთ ეს ვარიანტი საქმეზე მინიჭებული მომხმარებლების წასაშლელად.

ფაილის ატვირთვა

შეგიძლიათ დაუყოვნებლივ ატვირთოთ ფაილი ან მოგვიანებით დაუბრუნდეთ იმპორტის დამატების დიალოგურ ფანჯარას და დაასრულოთ ეს ნაბიჯები.

  1. გახსენით იმპორტის დამატების დიალოგური ფანჯარა.
  2. შეიყვანეთ უნიკალური სახელი CSV ფაილისთვის.
  3. დააწკაპუნეთ პლუს (+) ხატულაზე, რომ ატვირთოთ ფაილი თქვენი მოწყობილობიდან.
  4. დააწკაპუნეთ ველზე Type და აირჩიეთ ვარიანტი. (იხილეთ ზემოთ)
  5. დააწკაპუნეთ შენახვა.

დამუშავება

ფაილში ჩანაწერების რაოდენობისა და სისტემის დატვირთვის მიხედვით, პლატფორმას შეიძლება რამდენიმე წუთი დასჭირდეს იმპორტის დამუშავებას.

როდესაც იმპორტი დასრულდება, პლატფორმა არეგულირებს მომხმარებლის ჩანაწერებს (ანუ ამატებს, ცვლის ან შლის) საქმეების გვერდზე.

წარუმატებლობა

თუ იმპორტის ოპერაცია ვერ მოხერხდა, პლატფორმა აბრუნებს წარუმატებელ შეტყობინებას. უმეტეს შემთხვევაში, მარცხის მიზეზი არის მონაცემთა ფორმატირება.

მარცხის მიზეზის სანახავად:

  1. დააწკაპუნეთ ხედვა მოქმედების სათაურის ქვეშ.
  2. შემდეგ გვერდზე გადაიტანეთ მაუსი წარუმატებელი ხატულაზე. ინსტრუმენტის მინიშნება აჩვენებს შეცდომას.
  3. გადადით CVS ფაილში და შეცვალეთ და გამოასწორეთ შეცდომა.
  4. ისევ ატვირთეთ ფაილი.
