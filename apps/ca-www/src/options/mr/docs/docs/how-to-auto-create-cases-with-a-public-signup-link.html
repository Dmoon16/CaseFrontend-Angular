सार्वजनिक साइनअप लिंकसह केसेस स्वयंचलितपणे कसे तयार करावे

हा लेख एकच साइनअप लिंक पाठवून एक केस कसा तयार करायचा (आणि त्याच वेळी वापरकर्ता जोडायचा) प्रशासकांना दाखवतो.

लिंक शोधत आहे

  1. प्रशासक खात्यात लॉग इन करा.
  2. अॅप्स पृष्ठावर अॅप शोधा आणि प्रशासक बटणावर क्लिक करा.
  3. प्रशासन पॅनेलवर, सेटिंग्ज मेनू टॅबवर क्लिक करा.
  4. SYSTEM SETUP सबमेनू टॅब निवडला असल्याची खात्री करा.

साइनअपला परवानगी द्या फील्ड चालू वर सेट केले आहे याची खात्री करा. (आवश्यक) जेव्हा हे वैशिष्ट्य सक्षम केले जाते, तेव्हा केस सक्रिय एक अद्वितीय दुवा व्युत्पन्न करते.

  1. ऑटो क्रिएट केस फील्डवर क्लिक करा आणि चालू निवडा. केस वापरकर्ता भूमिका फील्ड प्रदर्शित करते.
  2. एक भूमिका निवडा. (खाली पहा)

  • डीफॉल्ट

लिंक पाठवत आहे

लिंक एक किंवा अधिक लोकांसह शेअर करा. प्रत्येक वापरकर्ता जो लिंकवर क्लिक करतो आणि साइनअप पूर्ण करतो तो केस पृष्ठावर नवीन केस तयार करण्यासाठी प्लॅटफॉर्मला ट्रिगर करतो आणि केसमध्ये वापरकर्त्याला स्वयं नियुक्त करतो.
