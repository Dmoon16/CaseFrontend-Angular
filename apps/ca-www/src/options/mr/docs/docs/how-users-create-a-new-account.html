वापरकर्ते नवीन खाते कसे तयार करतात

अॅडमिनने वापरकर्त्याला केसमध्ये जोडल्यानंतर: * जर वापरकर्ता केसअॅक्टिव्हसाठी नवीन असेल, तर वापरकर्त्याने खाते तयार करण्यासाठी वर्कफ्लो पूर्ण करणे आणि केसमध्ये प्रवेश करण्यासाठी आमंत्रण स्वीकारणे आवश्यक आहे. * वापरकर्त्याकडे विद्यमान केसअॅक्टिव्ह खाते असल्यास, वापरकर्त्याने केसमध्ये प्रवेश करण्यासाठी आमंत्रण स्वीकारले पाहिजे.

हा लेख वापरकर्त्याने त्यांचे खाते तयार करण्यासाठी पूर्ण केलेल्या चरणांचे वर्णन करतो.

सुरू करणे

केसअॅक्टिव्ह अॅपमध्ये वापरकर्त्याला जोडले जाऊ शकते असे दोन मार्ग आहेत

  • प्रशासकाद्वारे: जर प्रशासकाने प्लॅटफॉर्म UI किंवा आयात साधनाद्वारे वापरकर्ता तयार केला, तर प्रशासकाने लॉगिन करण्यासाठी पुढे जाण्यासाठी वापरकर्त्याला account.caseactive.com वर एक लिंक पाठवणे आवश्यक आहे.
  • वापरकर्त्याद्वारे: वापरकर्त्याने साइनअप लिंक वापरून खाते तयार केल्यास, वापरकर्त्यास लॉगिन करण्यासाठी पुढे जाण्यासाठी account.caseactive.com वर स्वयंचलितपणे पुनर्निर्देशित केले जाईल.

लॉग इन करत आहे

जेव्हा वापरकर्ता प्रथमच त्यांच्या CaseActive खात्यात लॉग इन करतो, तेव्हा प्लॅटफॉर्म वापरकर्त्याला पुढील गोष्टी करण्यास सूचित करतो:

  • त्यांचे ईमेल खाते उघडा आणि नंतर सहा-अंकी सुरक्षा कोड कॉपी/पेस्ट करा
  • खाते प्रोफाइल पूर्ण करा

img

वापरकर्त्याने अद्यतन क्लिक केल्यानंतर, अॅप्स पृष्ठ प्रदर्शित होईल. पृष्ठ अॅपचे नाव (कंपनीचे नाव), वापरकर्ता प्रकार (भूमिका), स्थिती आणि आमंत्रण बटण दर्शवते.

img

अॅपमध्ये प्रवेश करण्यासाठी, वापरकर्त्याने आमंत्रण स्वीकारणे आवश्यक आहे. वापरकर्त्याने हे केले पाहिजे:

  1. आमंत्रण बटणावर क्लिक करा.
  2. परवानग्यांचे पुनरावलोकन करा.
  3. अटींचे पुनरावलोकन आणि संपादन (पर्यायी) करण्यासाठी सूचना टॅबवर क्लिक करा.
  4. प्रवेश मंजूर करा क्लिक करा.

img

जेव्हा वापरकर्ता मंजूर करतो, तेव्हा प्लॅटफॉर्म वापरकर्ता पृष्ठावर APP बटण प्रदर्शित करतो. हे बटण वापरकर्त्याला नियुक्त केलेल्या प्रकरणांमध्ये प्रवेश देते.

वापरकर्ता सेटिंग्ज संपादित करण्यासाठी, वापरकर्ता दीर्घवृत्तावर क्लिक करतो.
