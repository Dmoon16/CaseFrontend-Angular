लॉग इन कसे करावे

तुमच्या विद्यमान केस सक्रिय खात्यात लॉग इन कसे करायचे ते शिका.

लॉग इन करत आहे

  1. लॉगिन पृष्ठावर जा.
  2. लॉगिन बटणावर क्लिक करा.
  3. तुमचा ईमेल टाका.
  4. पुढील क्लिक करा.

खाते सत्यापित करत आहे

  1. एक नवीन ब्राउझर टॅब उघडा आणि तुमच्या ईमेल खात्यात लॉग इन करा.
  2. सहा-अंकी कोड कॉपी करा.
  3. केस सक्रिय टॅबवर परत या.
  4. फील्डमध्ये कोड पेस्ट करा. एक नवीन विंडो दिसेल.

केस सक्रिय अॅप्स पृष्ठ प्रदर्शित करते.

  • टीप प्लॅटफॉर्म प्रशासक विशेषाधिकार असलेल्या वापरकर्त्यांसाठी APP बटणाच्या पुढे प्रशासक बटण प्रदर्शित करते.
