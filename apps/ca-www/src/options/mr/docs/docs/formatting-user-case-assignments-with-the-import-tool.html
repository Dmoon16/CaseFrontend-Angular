आयात साधनासह वापरकर्ता केस असाइनमेंट डेटाचे स्वरूपन करणे

खाली CSV फाइल्समधील डेटा फील्डची सूची आहे. प्रत्येक आयात पर्यायामध्ये फील्डचे भिन्न संयोजन असते.

केस_आयडी

  • केस वापरकर्ता किंवा टॅग आयडी प्रविष्ट करा.
  • केसेस पेज > एडिट > टॅग आयडी वर जा किंवा केसेस पेज > एडिट > केस आयडी वर जा.

user_ID

  • वापरकर्त्याचा आयडी किंवा टॅग आयडी प्रविष्ट करा.
  • युजर पेज > एडिट > टॅग आयडी वर जा किंवा युजर पेज > एडिट > युजर आयडी वर जा.

केस_भूमिका_क्रम

  • सेल रिकामा सोडा.

केस_रोल_आयडी

  • प्रविष्ट करा: क्लायंट, कर्मचारी किंवा डीफॉल्ट.

केस_सूचना

  • 0 = नाही
  • 1 = होय, वापरकर्त्यास ईमेल सूचना पाठवा.
