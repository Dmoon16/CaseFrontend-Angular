आयात साधनासह वापरकर्ता असाइनमेंट कसे नियुक्त करावे, सुधारित करावे किंवा काढावे

हा लेख इंपोर्ट टूलसह केस वापरकर्ते कसे व्यवस्थापित करावे हे प्रशासकांना दाखवतो.

प्रशासक हे करू शकतात:

  • वापरकर्त्यांना केससाठी नियुक्त करा
  • एखाद्या केसला आधीच नियुक्त केलेले वापरकर्ते सुधारित करा
  • केससाठी नियुक्त केलेल्या वापरकर्त्यांना काढून टाका

आयात साधन उघडत आहे

  1. प्रशासक खात्यात लॉग इन करा.
  2. अॅप्स पृष्ठावर अॅप शोधा आणि प्रशासक बटणावर क्लिक करा.
  3. इंटिग्रेशन टॅबवर क्लिक करा आणि नंतर सबमेनूवर आयात करा.
  4. नवीन बटणावर क्लिक करा. जोडा आयात संवाद बॉक्स प्रदर्शित होतो.

रेकॉर्ड तयार करणे

एक किंवा अधिक CSV फाइल डाउनलोड करा. केस वापरकर्ते जोडण्यासाठी, सुधारण्यासाठी किंवा काढण्यासाठी या फायली वापरा. Add Imports डायलॉग बॉक्स उघडल्यानंतर:

  1. Type फील्डवर क्लिक करा.
  2. ड्रॉप-डाउन मेनूमधून स्क्रोल करा आणि एक पर्याय निवडा (खाली पहा). एक लिंक दिसते.
  3. डाउनलोड नमुना लिंक वर क्लिक करा. तुमच्या डिव्हाइसवर CSV फाइल डाउनलोड होते.

img

फाइल उघडा आणि रेकॉर्ड जोडा. CSV फाइलमध्ये फॉरमॅट डेटा कसा वापरायचा हे जाणून घेण्यासाठी येथे क्लिक करा.

फाइल पर्याय

वापरकर्त्यांना प्रकरणांमध्ये नियुक्त करा * वापरकर्त्यांना केसमध्ये जोडण्यासाठी हा पर्याय निवडा. * केस_रोल_ऑर्डर सेल रिक्त सोडा. * वापरकर्ते पृष्ठावर जा आणि या व्यक्तीसाठी टॅग आयडी कॉपी करा. टॅग आयडी फील्ड रिक्त असल्यास, एक अद्वितीय आयडी प्रविष्ट करा, जतन करा क्लिक करा आणि नंतर आयडी CSV फाइलमध्ये कॉपी करा.

प्रकरणांसाठी नियुक्त केलेले वापरकर्ते सुधारित करा * केससाठी नियुक्त केलेल्या वर्तमान वापरकर्त्यांसाठी डेटा अद्यतनित करण्यासाठी हा पर्याय निवडा. * केस_आयडी कॉलमसाठी केस टॅग आयडी वापरा.

img

केसेससाठी नियुक्त केलेले वापरकर्ते काढा * केससाठी नियुक्त केलेल्या वापरकर्त्यांना काढून टाकण्यासाठी हा पर्याय निवडा.

फाइल अपलोड करत आहे

तुम्ही लगेच फाइल अपलोड करू शकता किंवा नंतरच्या तारखेला जोडा आयात डायलॉग बॉक्समध्ये परत येऊ शकता आणि या पायऱ्या पूर्ण करू शकता.

  1. जोडा आयात संवाद बॉक्स उघडा.
  2. CSV फाइलसाठी एक अद्वितीय नाव प्रविष्ट करा.
  3. तुमच्या डिव्हाइसवरून फाइल अपलोड करण्यासाठी प्लस (+) चिन्हावर क्लिक करा.
  4. Type फील्डवर क्लिक करा आणि पर्याय निवडा. (वर पहा)
  5. SAVE वर क्लिक करा.

प्रक्रिया करत आहे

फाइल आणि सिस्टम लोडमधील रेकॉर्डच्या संख्येवर अवलंबून, प्लॅटफॉर्मला आयात प्रक्रिया करण्यासाठी काही मिनिटे लागू शकतात.

जेव्हा आयात पूर्ण होते, तेव्हा प्लॅटफॉर्म केसेस पृष्ठावर वापरकर्ता रेकॉर्ड (म्हणजे जोडते, सुधारित करते किंवा काढून टाकते) समायोजित करते.

अपयश

आयात ऑपरेशन अयशस्वी झाल्यास, प्लॅटफॉर्म एक अयशस्वी संदेश परत करतो. बर्याच बाबतीत, अपयशाचे कारण डेटा स्वरूपन आहे.

img

अपयशाचे कारण पाहण्यासाठी:

  1. क्रिया शीर्षकाखाली पहा वर क्लिक करा.
  2. पुढील पृष्ठावर, अयशस्वी चिन्हावर माउस फिरवा. टूलटिप त्रुटी दाखवते.
  3. CVS फाइलवर जा आणि समायोजित करा आणि त्रुटी दूर करा.
  4. फाइल पुन्हा अपलोड करा.
