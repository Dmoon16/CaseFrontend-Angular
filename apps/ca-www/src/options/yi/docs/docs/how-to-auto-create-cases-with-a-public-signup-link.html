ווי צו אַוטאָ שאַפֿן קאַסעס מיט אַ ציבור סיגנופּ לינק

דער אַרטיקל ווייַזן אַדמיניסטראַטאָרס ווי צו שאַפֿן אַ פאַל (און לייגן אַ באַניצער אין דער זעלביקער צייט) דורך שיקן אַ איין סיגנופּ לינק.

געפֿינען די לינק

  1. קלאָץ אין צו די אַדמיניסטראַטאָר חשבון.
  2. געפֿינען אַן אַפּ אויף די אַפּפּס בלאַט און גיט די ADMIN קנעפּל.
  3. אויף די אַדמין טאַפליע, גיט די סעטטינגס מעניו קוויטל.
  4. מאַכן זיכער אַז די SYSTEM SETUP סובמעניו קוויטל איז אויסגעקליבן.

קאַנפיגיער סעטטינגס

מעגילע אַראָפּ צו די USER SIGNUP כעדינג.

מאַכט זיכער אַז די לאָזן סיגנופּ פעלד איז באַשטימט צו אויף. (פארלאנגט) ווען דעם שטריך איז ענייבאַלד, קאַסע אַקטיווע דזשענערייץ אַ יינציק לינק.

  1. דריקט דעם אַוטאָ שאַפֿן קאַסע פעלד און סעלעקטירן אויף. די פאַל באַניצער ראָלע פעלד דיספּלייז.
  2. אויסקלייַבן אַ ראָלע. (זע אונטן)

שיקט די לינק

ייַנטיילן דעם לינק מיט איינער אָדער מער מענטשן. יעדער באַניצער וואָס גיט די לינק און קאַמפּליץ סיגנופּ טריגערז די פּלאַטפאָרמע צו דזשענערייט אַ נייַע פאַל אויף די קאַסעס בלאַט און אויטאָמאַטיש באַשטימען דעם באַניצער אין דעם פאַל.
