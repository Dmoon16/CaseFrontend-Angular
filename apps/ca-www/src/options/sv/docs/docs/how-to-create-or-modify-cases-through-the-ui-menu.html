Hur man skapar eller ändrar ärenden via UI-menyn

Den här artikeln visar administratörer hur man skapar ett ärende med plattformens användargränssnitt. Administratörer kan också lägga till användare i ett nytt ärende och uppdatera befintliga ärenden.

Skapa ett ärende

  1. Logga in på administratörskontot.
  2. Leta upp en app på appsidan och klicka på ADMIN-knappen.
  3. På adminpanelen klickar du på menyfliken ärenden.
  4. Klicka på knappen NY. Ett nytt fönster visas.

Fyll i de obligatoriska fälten och klicka på SPARA.

img

Lägger till användare

Valfritt kan administratörer lägga till användare i ärendet på den här sidan. Rulla ned och klicka på knappen TILLDELA ANVÄNDARE .

img

Fyll i fälten i modalen Tilldela användare. Markera rutan för att skicka ett e-postmeddelande till användaren. (frivillig)

Ändra ett ärende

För att uppdatera ett litet antal ärenden, använd redigeringsverktyget och gör ändringar i varje enskilt ärende.

  • Obs! För att uppdatera ett stort antal ärenden på en gång, överväg att använda importverktyget
  1. Logga in på administratörskontot.
  2. Leta reda på en app på appsidan och klicka på ADMIN-knappen.
  3. På adminpanelen klickar du på menyfliken ärenden.
  4. Leta upp ett ärende och klicka på Redigera.
  5. Gör ändringar och klicka på SPARA.
