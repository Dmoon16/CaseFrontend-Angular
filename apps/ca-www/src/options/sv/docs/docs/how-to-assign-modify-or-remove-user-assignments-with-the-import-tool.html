Hur man tilldelar, ändrar eller tar bort användartilldelningar med importverktyget

Den här artikeln visar administratörer hur man hanterar ärendeanvändare med importverktyget.

Administratörer kan:

  • tilldela användare till ett ärende
  • ändra användare som redan tilldelats ett ärende
  • ta bort användare som tilldelats ett ärende

Öppnar importverktyget

  1. Logga in på administratörskontot.
  2. Leta upp en app på appsidan och klicka på ADMIN-knappen.
  3. Klicka på fliken Integration och sedan IMPORTER på undermenyn.
  4. Klicka på knappen NY. Dialogrutan Lägg till importer visas.

Skapar poster

Ladda ner en eller flera CSV-filer. Använd dessa filer för att lägga till, ändra eller ta bort ärendeanvändare. När dialogrutan Lägg till importer öppnas:

  1. Klicka på fältet Typ.
  2. Bläddra igenom rullgardinsmenyn och välj ett alternativ (se nedan). En länk visas.
  3. Klicka på länken Ladda ner exempel. En CSV-fil laddas ner till din enhet.

Filalternativ

Tilldela användare till ärenden * Välj det här alternativet för att lägga till användare till ett ärende. * Lämna cellen case_role_order tom. * Gå till sidan Användare och kopiera tag-ID:t för denna person. Om fältet Tag-ID är tomt anger du ett unikt ID, klickar på Spara och kopierar sedan ID:t till CSV-filen.

Ändra användare som tilldelats ärenden * Välj det här alternativet för att uppdatera data för nuvarande användare som tilldelats ett ärende. * Använd Case Tag ID för kolumnen Case_ID.

Ta bort användare som tilldelats ärenden * Välj det här alternativet för att ta bort användare som tilldelats ett ärende.

Laddar upp filen

Du kan ladda upp en fil direkt eller gå tillbaka till dialogrutan Lägg till importer vid ett senare tillfälle och slutföra dessa steg.

  1. Öppna dialogrutan Lägg till importer.
  2. Ange ett unikt namn för CSV-filen.
  3. Klicka på plusikonen (+) för att ladda upp filen från din enhet.
  4. Klicka på fältet Typ och välj ett alternativ. (se ovan)
  5. Klicka på SPARA.

Bearbetning

Beroende på antalet poster i filen och systemladdningen kan plattformen ta flera minuter att bearbeta importen.

När importen är klar justerar plattformen användarposter (dvs. lägger till, modifierar eller tar bort) på sidan Fall.

Fel

Om importen misslyckas returnerar plattformen ett misslyckat meddelande. I de flesta fall är orsaken till felet dataformatering.

Så här ser du orsaken till felet:

  1. Klicka på Visa under rubriken Åtgärd.
  2. På nästa sida håller du musen över ikonen Misslyckades. Verktygstipset visar felet.
  3. Gå till CVS-filen och justera och åtgärda felet.
  4. Ladda upp filen igen.
