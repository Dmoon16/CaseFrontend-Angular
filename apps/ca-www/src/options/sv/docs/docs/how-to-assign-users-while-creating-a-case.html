Hur man tilldelar användare när man skapar ett ärende

Den här artikeln visar administratörer hur man tilldelar en användare när de skapar ett nytt ärende. Om användaren inte redan är registrerad på plattformen kan du skapa lägg till en ny användare samtidigt som du skapar ett nytt ärende.

Skapa ett ärende

  1. Logga in på administratörskontot.
  2. Leta upp en app på appsidan och klicka på ADMIN-knappen.
  3. Klicka på menyfliken Fall.
  4. Klicka på knappen NY. Ett nytt fönster visas.
  • Fyll i fälten för att skapa ett ärende och scrolla sedan ned till rubriken Tilldelade användare.

Tilldela en användare

Nuvarande användaren

Följ dessa steg för att tilldela en aktuell användare till det nya ärendet.

Klicka på knappen TILLDELA ANVÄNDARE och fyll i fälten.

  1. Klicka på rullgardinsmenyn Användare och välj en identifierare (t.ex. e-post).
  2. Ange ett värde i det tomma fältet (t.ex. e-postadress).
  3. Om användarposten finns i plattformen, föreslår fältet automatiskt ett namn.
  4. Välj rätt användare.
  5. Välj en användarroll (klienter, standard eller personal).
  6. Om du vill kan du markera rutan för att skicka ett e-postmeddelande till användaren.
  7. Klicka på SPARA.

Case Active lägger till användaren i ärendet. Nu kan administratörer klicka på Spara för att stänga fönstret, eller lägga till en ny användare eller ta bort en användare.

Lägger till en ny användare

Om plattformen inte kan hitta en aktuell post, klicka på knappen för att lägga till en ny användare.

Fyll i fälten och klicka sedan på SPARA för att lägga till användaren på plattformen. Ange sedan en användarroll och klicka på SPARA igen för att tilldela användaren ärendet. Klicka slutligen på SPARA en gång till för att spara ärendet.
