Hur man uppdaterar en användarprofil

Den här artikeln visar användare hur man uppdaterar information på sin profilsida.

  • Användare kan uppdatera sina egna profiler.
  • Administratörer kan uppdatera profilsidan för vilken användare som helst.

För användare

Det första steget är att öppna profilsidan. Användare och administratörer följer dessa steg för att uppdatera sin egen profilsida.

  1. Logga in på ditt Case Active-konto.
  2. Klicka på fliken Profil högst upp på sidan.
  3. Göra ändringar.
  4. Klicka på Spara.

De flesta fält är valfria och vissa fält är obligatoriska. De obligatoriska fälten är:

  • Förnamn
  • Efternamn
  • E-postadress
  • Språk
  • Tidszon

För administratörer

Administratörer kan också uppdatera profilinformationen för andra användare. Detta inkluderar tre fält under rubriken HOST USER INFO.

  1. Logga in på ditt Case Active-administratörskonto.
  2. Leta upp en app och klicka på ADMIN-knappen.
  3. Klicka på Användare i toppmenyn.
  4. Gå till undermenyn och välj den användarlista du vill komma åt (AKTIV eller AVAKTIVERAD). Standardinställningen är Aktiv.
  5. Leta upp en användare i listan och klicka på Redigera.
  6. Gör ändringar efter behov. (se nedan)
  7. Klicka på Spara.

img

Ändra användarinformation

Administratörer kan ändra användarinformation.

  1. Ställ in användartyp (användare eller chef).
  2. Ställ in användarstatus (aktiv till inaktiverad).
  3. Ange ett unikt tagg-ID. (frivillig)
  4. Klicka på Spara.
