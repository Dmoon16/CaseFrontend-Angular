Hur användare skapar ett nytt konto

Efter att en administratör har lagt till en användare i ett ärende: * om användaren är ny på caseactive måste användaren slutföra ett arbetsflöde för att skapa ett konto och acceptera inbjudan att få tillgång till ärendet. * om användaren har ett befintligt ärendeaktivt konto måste användaren acceptera inbjudan för att få tillgång till ärendet.

Den här artikeln beskriver de steg en användare måste utföra för att skapa sitt konto.

Komma igång

Det finns två sätt som en användare kan läggas till i en caseactive app

  • Av administratör: Om administratören skapar en användare via plattformens användargränssnitt eller importverktyget måste administratören skicka en länk till account.caseactive.com för att fortsätta med inloggningen.
  • Av användare: Om användaren skapade ett konto med en registreringslänk kommer användaren automatiskt att omdirigeras till account.caseactive.com för att fortsätta med inloggningen.

Loggar in

När användaren loggar in på sitt CaseActive-konto för första gången uppmanar plattformen användaren att:

  • öppna sitt e-postkonto och kopiera/klistra sedan in den sexsiffriga säkerhetskoden
  • fyll i en kontoprofil

img

När användaren klickar på UPPDATERA visas appsidan. Sidan visar namnet på appen (företagets namn), användartyp (roll), status och en INBJUDAN-knapp.

img

För att komma åt appen måste användaren acceptera inbjudan. Användaren bör:

  1. Klicka på knappen INBJUDAN.
  2. Granska behörigheterna.
  3. Klicka på fliken Meddelanden för att granska och redigera (valfritt) villkoren.
  4. Klicka på GODKÄNN ÅTKOMST.

img

När användaren godkänner, visar plattformen en APP-knapp på användarsidan. Denna knapp ger användaren åtkomst till tilldelade ärenden.

För att redigera användarinställningar klickar användaren på ellipsen.

Användaren kan: * ta bort appåtkomst * ändra aviseringsinställningar

img

När användaren har slutfört arbetsflödet visas användarnamnet på administratörens användarlista. Listan visar alla användare, inloggade eller inte.
