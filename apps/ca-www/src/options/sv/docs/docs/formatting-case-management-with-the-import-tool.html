Formatera ärendehanteringsdata med importverktyget

Nedan finns en lista över obligatoriska fält i CSV-filer för att hantera ärenden med importverktyget.

Användarnamn

  • Ange användarens e-postadress.

tag_id (ett id som du kan referera till senare)

  • en unik identifierare (använd alfanumeriska tecken, bindestreck accepteras)

case_status

  • 0 = stängd
  • 1 = aktiv

date_opened

  • månad dag år
  • mm/dd/åååå

handla om

  • en unik beskrivning av fallet (använd alfanumeriska tecken, bindestreck accepteras)
