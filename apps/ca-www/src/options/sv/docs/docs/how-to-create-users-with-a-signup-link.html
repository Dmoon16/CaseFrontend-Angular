Hur man skapar användare med en registreringslänk

Den här artikeln visar administratörer hur man skapar användare genom att skicka en registreringslänk.

Konfigurerar registreringslänken

  1. Klicka på länken Skicka dem en registrering längst ned i dialogrutan Lägg till användare. En ny sida visas.
  2. Konfigurera registreringsalternativen. (se nedan)

Se till att fältet Tillåt registrering är inställt på På. (obligatoriskt) När det är aktiverat genererar Case Active en unik länk.

Granska alternativen nedan och klicka sedan på ikonen för att kopiera länken. Du kan placera länken på din webbplats eller skicka länken till en eller flera personer via e-post eller någon kommunikationskanal.

img

Skapa ett ärende

Alternativt kan du automatiskt skapa ett ärende för varje ny användare och tilldela en användarroll.

  1. Klicka på fältet Skapa ärende automatiskt och välj På. Fälten Case User Roll visas.
  2. Klicka på Case User Roll och välj ett alternativ. (se nedan)

img

Case användarroller

Det finns tre fallanvändarroller.

  • kunder
  • standard
  • personal

Skickar länken

Dela länken med en eller flera personer. Varje person klickar på länken och slutför arbetsflödet för att aktivera sitt Case Active-konto. När användaren har slutfört registreringsarbetsflödet lägger Case Active till användaren i användarlistan.

Nästa steg

Case Active skickar ett e-postmeddelande till användaren med registreringslänken. Användaren klickar på länken och slutför arbetsflödet för nya registreringar.

  • Den här artikeln beskriver registreringsarbetsflödet för nya användare.
