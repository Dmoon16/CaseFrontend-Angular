Formatera data för användarfallstilldelningar med importverktyget

Nedan finns en lista över datafält i CSV-filer. Varje importalternativ har olika kombination av fält.

Ärende ID

  • Ange ärendet User eller Tag ID.
  • Gå till ärendesidan > Redigera > Tagg-ID ELLER Gå till ärendesidan > Redigera > Ärende-ID i url.

användar ID

  • Ange användarens ID eller Tag ID.
  • Gå till sidan Användare > Redigera > Tagg-ID ELLER Gå till sidan Användare > Redigera > Användar-ID i url.

case_rolle_order

  • Lämna cellen tom.

case_rolle_ID

  • Ange: klienter, personal eller standard.

case_notify

  • 0 = nej
  • 1 = Ja, skicka ett e-postmeddelande till användaren.
