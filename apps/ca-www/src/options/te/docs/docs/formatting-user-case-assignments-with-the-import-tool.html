దిగుమతి సాధనంతో వినియోగదారు కేస్ అసైన్‌మెంట్‌ల డేటాను ఫార్మాటింగ్ చేయడం

CSV ఫైల్‌లలోని డేటా ఫీల్డ్‌ల జాబితా క్రింద ఉంది. ప్రతి దిగుమతి ఎంపిక ఫీల్డ్‌ల విభిన్న కలయికను కలిగి ఉంటుంది.

కేసు_ID

  • కేసు వినియోగదారు లేదా ట్యాగ్ IDని నమోదు చేయండి.
  • కేసుల పేజీ > సవరించు > ట్యాగ్ IDకి వెళ్లండి లేదా urlలో కేసుల పేజీ > సవరించు > కేస్ IDకి వెళ్లండి.

వినియోగదారుని గుర్తింపు

  • వినియోగదారు ID లేదా ట్యాగ్ IDని నమోదు చేయండి.
  • వినియోగదారుల పేజీ > సవరించు > ట్యాగ్ IDకి వెళ్లండి లేదా urlలో వినియోగదారుల పేజీ > సవరించు > వినియోగదారు IDకి వెళ్లండి.

కేసు_పాత్ర_క్రమం
