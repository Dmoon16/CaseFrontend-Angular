పబ్లిక్ సైన్అప్ లింక్‌తో కేసులను స్వయంచాలకంగా ఎలా సృష్టించాలి

ఒకే సైన్అప్ లింక్‌ని పంపడం ద్వారా కేసును ఎలా సృష్టించాలో (మరియు అదే సమయంలో వినియోగదారుని జోడించడం) ఈ కథనం నిర్వాహకులకు చూపుతుంది.

లింక్‌ను గుర్తించడం

  1. నిర్వాహక ఖాతాకు లాగిన్ చేయండి.
  2. యాప్‌ల పేజీలో యాప్‌ను గుర్తించి, అడ్మిన్ బటన్‌ను క్లిక్ చేయండి.
  3. అడ్మిన్ ప్యానెల్‌లో, సెట్టింగ్‌ల మెను ట్యాబ్‌ను క్లిక్ చేయండి.
  4. సిస్టమ్ సెటప్ సబ్‌మెను ట్యాబ్ ఎంచుకోబడిందని నిర్ధారించుకోండి.

సెట్టింగులను కాన్ఫిగర్ చేస్తోంది

USER SIGNUP శీర్షికకు క్రిందికి స్క్రోల్ చేయండి.

అనుమతించు సైన్అప్ ఫీల్డ్ ఆన్‌కి సెట్ చేయబడిందని నిర్ధారించుకోండి. (అవసరం) ఈ ఫీచర్ ప్రారంభించబడినప్పుడు, కేస్ యాక్టివ్ ఒక ప్రత్యేక లింక్‌ను రూపొందిస్తుంది.

  1. ఆటో క్రియేట్ కేస్ ఫీల్డ్‌ని క్లిక్ చేసి, ఆన్‌ని ఎంచుకోండి. కేస్ యూజర్ రోల్ ఫీల్డ్ డిస్ప్లేలు.
  2. ఒక పాత్రను ఎంచుకోండి. (క్రింద చూడగలరు)

మూడు కేస్ యూజర్ పాత్రలు ఉన్నాయి.

  • క్లయింట్లు (సిఫార్సు చేయబడింది)
  • డిఫాల్ట్
  • సిబ్బంది

లింక్ పంపుతోంది

ఒకరు లేదా అంతకంటే ఎక్కువ మంది వ్యక్తులతో లింక్‌ను భాగస్వామ్యం చేయండి. లింక్‌ను క్లిక్ చేసి, సైన్అప్ పూర్తి చేసిన ప్రతి వినియోగదారు కేసుల పేజీలో కొత్త కేసును రూపొందించడానికి ప్లాట్‌ఫారమ్‌ను ట్రిగ్గర్ చేస్తుంది మరియు కేసులో వినియోగదారుని స్వయంచాలకంగా కేటాయించారు.
