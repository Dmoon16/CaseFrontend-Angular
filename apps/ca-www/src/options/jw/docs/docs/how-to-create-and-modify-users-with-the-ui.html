Cara Nggawe lan Ngowahi Pangguna nganggo UI

Artikel iki nuduhake admin carane nggawe pangguna lan nganyari cathetan nganggo antarmuka pangguna platform. Admin uga bisa nambah pangguna menyang kasus anyar lan ngowahi kasus sing wis ana.

Nggawe pangguna

  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Ing panel Admin, klik tab menu Pangguna.
  4. Klik tombol NEW. Jendhela anyar ditampilake.

Isi kolom banjur klik SAVE. Case Active nambahake jeneng menyang dhaptar pangguna sing ana gandhengane karo aplikasi iki.

img

Ngowahi pangguna

Kanggo nganyari sawetara cathetan pangguna, gunakake alat Sunting lan gawe owah-owahan kanggo saben pangguna.

  • Wigati Kanggo ngowahi nomer akeh cathetan pangguna ing siji lungguh, nimbang nggunakake alat Impor .
  1. Ing panel Admin, klik tab menu Pangguna.
  2. Goleki pangguna banjur klik Sunting.
  3. Gawe owah-owahan banjur klik SAVE.

img

Langkah sabanjure

Case Active ora ngirim pangguna kabar email. Admin kudu ngubungi pangguna lan ngirim link menyang kaca ngarep Case Active. Pangguna banjur ngeklik tombol LOGIN lan ngrampungake alur kerja kanggo ndhaptar anyar.
