Cara Nggawe Pangguna nganggo Link Ndaftar

Artikel iki nuduhake admin carane nggawe pangguna kanthi ngirim link signup.

Konfigurasi link signup

  1. Klik Send them a signup link ing ngisor kotak dialog Tambah pangguna. Kaca anyar ditampilake.
  2. Konfigurasi opsi ndhaptar. (ndeleng ngisor)

img

Priksa manawa kolom Allow Signup disetel menyang On. (dibutuhake) Yen diaktifake, Case Active ngasilake link unik.

Deleng opsi ing ngisor iki banjur klik lambang kanggo nyalin link. Sampeyan bisa nyelehake link kasebut ing situs web sampeyan utawa ngirim link menyang siji utawa luwih wong liwat email utawa saluran komunikasi apa wae.

Nggawe kasus

Opsional, sampeyan bisa nggawe kasus kanthi otomatis kanggo saben pangguna anyar lan nemtokake peran pangguna.

  1. Klik lapangan Nggawe Kasus Otomatis banjur pilih On. Kolom Peran Panganggo Kasus ditampilake.
  2. Klik Peran Panganggo Kasus lan pilih opsi. (ndeleng ngisor)

Ngirim link

Nuduhake link karo siji utawa luwih wong. Saben wong ngeklik link kasebut lan ngrampungake alur kerja kanggo ngaktifake akun Case Active. Sawise pangguna ngrampungake alur kerja pandhaptaran, Case Active nambahake pangguna menyang dhaptar Pangguna.

img

Langkah sabanjure

Case Active ngirim kabar email menyang pangguna kanthi link signup. Pangguna ngeklik link lan ngrampungake alur kerja kanggo ndhaptar anyar.
