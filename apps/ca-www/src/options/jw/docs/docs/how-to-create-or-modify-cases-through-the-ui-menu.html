Cara Nggawe utawa Ngowahi Kasus Liwat Menu UI

Artikel iki nuduhake admin carane nggawe kasus karo antarmuka pangguna platform. Admin uga bisa nambah pangguna menyang kasus anyar lan nganyari kasus sing wis ana.

Nggawe kasus

  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Ing panel Admin, klik tab menu Kasus.
  4. Klik tombol NEW. Jendhela anyar ditampilake.

img

Isi kolom sing dibutuhake banjur klik SAVE.

Nambahake pangguna

Opsional, admin bisa nambah pangguna menyang kasus ing kaca iki. Gulung mudhun lan klik tombol ASSIGN USERS .

Ing modal Assign User, isi kolom. Centhang kothak kanggo ngirim kabar email marang pangguna. (opsional)

Ngowahi kasus

Kanggo nganyari sawetara kasus cilik, gunakake alat Sunting lan gawe owah-owahan ing saben kasus.

  • Wigati Kanggo nganyari akeh kasus ing siji lungguh, nimbang nggunakake alat Impor
  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Ing panel Admin, klik tab menu Kasus.
  4. Goleki kasus lan klik Sunting.
  5. Gawe owah-owahan banjur klik SAVE.
