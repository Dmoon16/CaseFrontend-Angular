Cara Nemtokake utawa Mbusak Tugas Panganggo nganggo Kasus sing Ana

Artikel iki nuduhake admin carane nemtokake lan mbusak pangguna cilik. Kita uga nuduhake sampeyan carane ngowahi data pangguna.

Mbukak kasus sing ana

  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Klik tab menu Cases .
  4. Priksa manawa AKTIF dipilih ing submenu.
  5. Gulung mudhun menyang kolom ACTION.

img

Nemtokake utawa mbusak

Klik Assign Users kanggo nemtokake pangguna menyang kasus kasebut utawa mbusak pangguna saka kasus kasebut (nanging dudu platform). Ing kaca sabanjure: Klik ADD lan ngrampungake alur kerja. Klik mbusak kanggo mbusak pangguna.

img

Ngowahi

Ora bisa ngowahi utawa nganyari data pangguna ing kaca Cases. Nanging, tindakake langkah iki:

  1. Klik tab menu Users .
  2. Goleki jeneng pangguna banjur klik Sunting .
  3. Gawe owah-owahan banjur klik SAVE.
