Cara Nemtokake, Ngowahi utawa Mbusak Tugas Panganggo nganggo Alat Impor

Artikel iki nuduhake admin carane ngatur pangguna kasus nganggo alat Impor.

Admin bisa:

  • nemtokake pangguna menyang kasus
  • ngowahi pangguna sing wis ditugasake kanggo kasus
  • mbusak pangguna sing ditugasake kanggo kasus

Mbukak alat Impor

  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Klik tab Integrasi banjur IMPOR ing submenu.
  4. Klik tombol NEW. Kothak dialog Tambah Impor ditampilake.

Nggawe cathetan

Ngundhuh siji utawa luwih file CSV. Gunakake file kasebut kanggo nambah, ngowahi, utawa mbusak pangguna cilik. Sawise kothak dialog Tambah Impor mbukak:

  1. Klik kolom Tipe.
  2. Gulung menu gulung mudhun lan pilih pilihan (ndeleng ngisor). A link ditampilake.
  3. Klik link Download Sample. File CSV diundhuh menyang piranti sampeyan.

Mbukak file lan nambah cathetan. Klik kene kanggo mangerteni carane nggunakake data format ing file CSV.

Pilihan file

Temtokake pangguna menyang kasus * Pilih pilihan iki kanggo nambah pangguna menyang kasus. * Ninggalake sel case_role_order kosong. * Bukak kaca Pangguna lan salin ID Tag kanggo wong iki. Yen kolom Tag ID kosong, ketik ID unik, klik Simpen, banjur salin ID menyang file CSV.

img

Ngowahi pangguna sing ditugasake kanggo kasus * Pilih pilihan iki kanggo nganyari data kanggo pangguna saiki sing ditugasake kanggo kasus. * Gunakake ID Tag Case kanggo kolom case_ID.

Mbusak pangguna sing ditugasake kanggo kasus * Pilih pilihan iki kanggo mbusak pangguna sing ditugasake kanggo kasus.

Ngunggah file

Sampeyan bisa langsung ngunggah file utawa bali menyang kothak dialog Tambah Impor mengko lan ngrampungake langkah-langkah iki.

  1. Bukak kothak dialog Tambah Impor.
  2. Ketik jeneng unik kanggo file CSV.
  3. Klik lambang plus (+) kanggo ngunggah file saka piranti sampeyan.
  4. Klik kolom Tipe banjur pilih opsi. (ndeleng ndhuwur)
  5. Klik SAVE.

Ngolah

Gumantung saka jumlah rekaman ing file lan beban sistem, platform bisa njupuk sawetara menit kanggo proses impor.

Nalika impor rampung, platform nyetel cathetan pangguna (yaiku nambah, ngowahi, utawa mbusak) ing kaca Kasus.

Gagal

Yen operasi impor gagal, platform ngasilake pesen Gagal. Umume kasus, panyebab kegagalan yaiku format data.

Kanggo ndeleng sabab saka kegagalan:

  1. Klik Ndeleng ing judhul Tindakan.
  2. Ing kaca sabanjure, kisaran mouse ing lambang Gagal. Tooltip nampilake kesalahan.
  3. Pindhah menyang file CVS lan atur lan ndandani kesalahan kasebut.
  4. Upload file maneh.
