Carane Admin Ngatur Pangguna

Admin nggawe, ngowahi utawa mateni sapa sing bisa ngakses aplikasi lan kasus. Nalika pangguna mlebu ing platform kanggo pisanan, dheweke kudu ngrampungake alur kerja profil.

  • Klik kene kanggo sinau luwih lengkap babagan alur kerja profil pangguna anyar.

Ngatur pangguna

Tabel ing ngisor iki nuduhake limang cara kanggo ngatur pangguna. Klik link kanggo mangerteni carane.

Metode Jinis Tumindak Rincian
Nggunakake UI UI nambah & ngowahi Tombol Admin > Pangguna
Nalika nggawe kasus lan nemtokake pangguna menyang kasus UI nambahi Tombol Admin > Kasus > Tombol anyar > Gawe kaca Kasus
Nggawe pangguna kanthi otomatis nggunakake link signup umum Tumindak pangguna nambahi Tombol Admin > Setelan > kaca Persiyapan Sistem
Alat ngimpor UI nambah & ngowahi Tombol Admin > Integrasi > Impor
Nggunakake API (Rawuh) API nambah & ngowahi N/A

Kabar

Platform kasebut ora ngirim kabar marang pangguna nalika ditambahake menyang aplikasi. Nanging, admin bisa ngaktifake kabar kanggo tugas kasus.
