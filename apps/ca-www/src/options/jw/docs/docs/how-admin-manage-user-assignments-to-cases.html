Cara Admin Ngatur Tugas Panganggo Kanggo Kasus

Ana limang cara kanggo ngatur pangguna lan tugas kasus. Klik link kanggo mangerteni carane.

Metode Jinis Tumindak Rincian
Nalika nggawe kasus UI nambahi Tombol Admin > Kasus > Tombol Anyar > Tombol Temtokake Pangguna
Otomatis nemtokake pangguna nggunakake link signup umum Tindakan pangguna nambahi Tombol Admin > Setelan > kaca Persiyapan Sistem
Nggunakake UI ing kasus sing ana UI nambah & mbusak Tombol Admin > Kasus > Temtokake Pangguna
Alat ngimpor UI nambah, ngowahi, & mbusak Tombol Admin > Integrasi > Impor
