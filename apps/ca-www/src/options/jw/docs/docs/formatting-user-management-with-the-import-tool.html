Format Data Manajemen Panganggo nganggo Alat Impor

Ing ngisor iki dhaptar kolom data ing file CSV. Saben opsi impor duwe kombinasi kolom sing beda.

jeneng panganggo

  • Ketik alamat email pangguna.

diwenehi_jeneng

  • Ketik jeneng ngarep pangguna.

jeneng_keluarga

  • Ketik jeneng mburi pangguna.

lokal

  • Kolom iki mriksa basa sing disenengi pangguna.
  • Ketik en kanggo Inggris, sp kanggo Spanyol.

zonainfo

  • Iki nyetel zona wektu pangguna. Ketik bawana lan kutha tanpa spasi (nggunakake garis ngisor).
  • Kudu nggunakake pilihan ing menu gulung mudhun Profil > BAHASA & WILAYAH > Zona wektu.
  • Gunakake zona wektu sing paling pas. Contone, Boston ora ana ing gulung mudhun, supaya kita bisa nggunakake New York (utawa kutha apa wae ing zona wektu EST / EDT).

host_tag_id

  • Ketik label permanen.
  • az, 0-9, -, / diijini.

status_kasus

  • 0 = ditutup
  • 1 = aktif
