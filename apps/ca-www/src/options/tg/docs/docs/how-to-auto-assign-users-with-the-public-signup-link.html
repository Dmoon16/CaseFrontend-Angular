Чӣ тавр ба таври худкор корбаронро бо истиноди бақайдгирии ҷамъиятӣ таъин кардан мумкин аст

Администраторҳо тарзи танзим кардани танзимоти системаро меомӯзанд, то корбарони нав ба таври худкор ба парвандаи навтаъсисшуда таъин карда шаванд.

Кушодани саҳифаи танзимот

Қадами аввал кушодани саҳифаи Setup Systems аст.

  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Ҷадвали Системаҳо ва сипас дар зерменюи СИСТЕМА ро пахш кунед.
  4. Ба сарлавҳаи РАЙЗИ ИСТИФОДАБАР ба поён ҳаракат кунед.

Танзимоти танзимоти система

  1. Менюҳои афтанда-ро "Иҷозат ба сабти ном" ва "Авто эҷод кардани парванда" ба Фаъол танзим кунед.
  2. Менюи нақшро клик кунед ва интихоберо интихоб кунед (пешфарз тавсия дода мешавад).
  3. Истиноди бақайдгирии корбари ҷамъиятиро нусхабардорӣ кунед ва мубодила кунед.

Фиристодани пайванд

Ҳар як корбаре, ки истинодро клик мекунад ва сабти номро анҷом медиҳад, платформаро барои тавлиди парвандаи нав дар саҳифаи парвандаҳо бармеангезад ва корбарро дар ин ҳолат худкор таъин мекунад.
