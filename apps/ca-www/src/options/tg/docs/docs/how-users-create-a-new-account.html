Чӣ тавр корбарон ҳисоби нав эҷод мекунанд

Пас аз он ки администратор корбарро ба парванда илова мекунад: * агар корбар дар парванда нав бошад, корбар бояд ҷараёни корро барои эҷод кардани ҳисоб анҷом диҳад ва даъватро барои дастрасӣ ба парванда қабул кунад. * агар корбар дорои ҳисоби мавҷудаи ҳолати фаъол бошад, корбар бояд даъвати дастрасӣ ба парвандаро қабул кунад.

Ин мақола қадамҳоеро тавсиф мекунад, ки корбар бояд барои эҷод кардани ҳисоби худ анҷом диҳад.

Сар кардани кор

Ду роҳе вуҷуд дорад, ки корбарро ба барномаи caseactive илова кардан мумкин аст

  • Аз ҷониби администратор: Агар администратор тавассути платформаи UI ё Воситаи воридот корбар эҷод кунад, администратор бояд ба корбар пайвандеро ба account.caseactive.com фиристад, то воридшавиро идома диҳад.
  • Аз ҷониби корбар: Агар корбар бо истифода аз истиноди сабти ном ҳисоб эҷод кунад, корбар ба таври худкор ба account.caseactive.com барои идомаи воридшавӣ равона карда мешавад.

Воридшавӣ

Вақте ки корбар бори аввал ба ҳисоби CaseActive худ ворид мешавад, платформа аз корбар хоҳиш мекунад, ки:

  • ҳисоби почтаи электронии худро кушоед ва сипас рамзи амнияти шаш-рақамаро нусхабардорӣ/часбонед
  • профили ҳисобро пур кунед

img

Пас аз пахш кардани UPDATE, саҳифаи Барномаҳо намоиш дода мешавад. Саҳифа номи барнома (номи ширкат), навъи корбар (нақш), вазъ ва тугмаи ДАЪВАТ-ро нишон медиҳад.

img

Барои дастрасӣ ба барнома, корбар бояд даъватро қабул кунад. Истифодабаранда бояд:

  1. Тугмаи ДАЪВАТРО пахш кунед.
  2. Иҷозатҳоро баррасӣ кунед.
  3. Барои баррасӣ ва таҳрир кардани шартҳо (ихтиёрӣ) ҷадвали Огоҳиҳоро клик кунед.
  4. Тасдиқи ДАСТРАСРО клик кунед.

img

Вақте ки корбар тасдиқ мекунад, платформа тугмаи APP-ро дар саҳифаи корбар намоиш медиҳад. Ин тугма ба корбар дастрасӣ ба парвандаҳои таъиншударо медиҳад.

Барои таҳрир кардани танзимоти корбар, корбар эллипсро клик мекунад.

img

Истифодабаранда метавонад: * дастрасии барномаро нест кунад * танзимоти огоҳиро тағир диҳад

img

Пас аз он ки корбар ҷараёни корро анҷом медиҳад, номи корбар дар рӯйхати корбарони администратор пайдо мешавад. Рӯйхат ҳамаи корбаронро нишон медиҳад, ки ба система ворид шудаанд ё не.
