Чӣ тавр таъин ё хориҷ кардани супоришҳои корбар бо парвандаҳои мавҷуда

Ин мақола ба маъмурон нишон медиҳад, ки чӣ гуна корбарони парвандаро таъин ва хориҷ кунанд. Мо инчунин ба шумо чӣ гуна тағир додани маълумоти корбарро нишон медиҳем.

Кушодани парвандаи мавҷуда

  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Ҷадвали менюи Ҳолатҳоро клик кунед.
  4. Боварӣ ҳосил кунед, ки ACTIVE дар зерменю интихоб шудааст.
  5. Ба сутуни АМРИЯ ба поён ҳаракат кунед.

Таъин кунед ё хориҷ кунед

Барои таъин кардани корбар ба парванда ё хориҷ кардани корбар аз парванда (вале на платформа) "Таъин кардани корбарон" -ро клик кунед. Дар саҳифаи навбатӣ: ADD -ро клик кунед ва ҷараёни корро анҷом диҳед. Барои нест кардани корбар клик кунед.

Тағйир додан

Таҳрир ё навсозии маълумоти корбар дар саҳифаи парвандаҳо ғайриимкон аст. Ба ҷои ин, ин қадамҳоро иҷро кунед:

  1. Ҷадвали менюи Истифодабарандагонро клик кунед.
  2. Номи корбарро ҷойгир кунед ва Таҳрир -ро клик кунед.
  3. Тағирот ворид кунед ва ЗАХИРА-ро пахш кунед.
