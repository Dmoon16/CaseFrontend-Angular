Чӣ тавр навсозӣ кардани профили корбар

Ин мақола ба корбарон нишон медиҳад, ки чӣ гуна маълумотро дар саҳифаи профили худ навсозӣ кунанд.

  • Истифодабарандагон метавонанд профилҳои худро навсозӣ кунанд.
  • Администраторҳо метавонанд саҳифаи профили ҳар як корбарро навсозӣ кунанд.

Барои истифодабарандагон

Қадами аввал кушодани саҳифаи профил аст. Истифодабарандагон ва маъмурон ин қадамҳоро иҷро мекунанд, то саҳифаи профили худро навсозӣ кунанд.

  1. Ба ҳисоби Case Active-и худ ворид шавед.
  2. Ҷадвали Профилро дар болои саҳифа клик кунед.
  3. Тағирот ворид кунед.
  4. Захира клик кунед.

Аксари майдонҳо ихтиёрӣ мебошанд ва баъзе майдонҳо талаб карда мешаванд. Майдонҳои ҳатмӣ инҳоянд:

  • Ном
  • Насаб
  • Суроғаи имэйл
  • Забон
  • Минтақаи вақт

Барои админхо

Администраторҳо инчунин метавонанд маълумоти профилро барои корбарони дигар навсозӣ кунанд. Ин се майдонро дар зери сарлавҳаи INFO ИСТИФОДАБАРАНДАГОНИ ХОСТ дар бар мегирад.

  1. Ба ҳисоби администратори Case Active-и худ ворид шавед.
  2. Як барномаро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Истифодабарандагонро дар менюи боло клик кунед.
  4. Ба зерменю равед ва рӯйхати корбареро, ки мехоҳед дастрасӣ дошта бошед, интихоб кунед (ФАЪОЛ ё МАЪЮЛ). Танзимоти пешфарз Фаъол аст.
  5. Дар рӯйхат корбареро пайдо кунед ва Таҳрир -ро клик кунед.
  6. Мувофиқи талаб тағирот ворид кунед. (нигаред ба поён)
  7. Захира клик кунед.

Тағир додани маълумоти корбар

Администраторҳо метавонанд маълумоти корбарро тағир диҳанд.

  1. Навъи корбарро таъин кунед (Истифодабаранда ё Менеҷер).
  2. Ҳолати корбарро таъин кунед (Фаъол ба ғайрифаъол).
  3. ID-и беназири барчаспро ворид кунед. (ихтиёрӣ)
  4. Захира клик кунед.
