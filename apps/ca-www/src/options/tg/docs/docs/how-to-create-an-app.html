Чӣ тавр сохтани барнома

Ин мақола ба корбарони нав нишон медиҳад, ки чӣ тавр эҷод кардани ҳисоби озмоишии ройгон. Case Active ба таври худкор имтиёзҳои маъмурии корбари навро барои ин ҳисоб медиҳад.

Эҷоди ҳисоб

  1. Ба саҳифаи воридшавӣ гузаред .
  2. Тугмаи Оғозро пахш кунед.
  3. Суроғаи почтаи электрониро ворид кунед.
  4. Баъдан клик кунед.

Тасдиқи ҳисоб

  1. Ҷадвали браузери нав кушоед ва ба ҳисоби почтаи электронии худ ворид шавед.
  2. Рамзи шаш рақамро нусхабардорӣ кунед.
  3. Бозгашт ба ҷадвали Парвандаи фаъол.
  4. Рамзро ба майдонҳо гузоред. Равзанаи нав намоиш дода мешавад.

Дар равзанаи нав:

  1. Майдонҳои заруриро пур кунед.
  2. Шартҳои хидматро хонед ва барои розӣ шудан қуттиро қайд кунед.
  3. Тугмаи Эҷоди Ҳисоби ройгонро клик кунед.

Анҷоми насби барнома

Ҷараёни кор дар саҳифаи оянда намоиш дода мешавад. Барои анҷом додани танзими барномаи худ қадамҳоро иҷро кунед. Шумо ҳамеша метавонед ба ин саҳифаҳо баргардед, то маълумот ё ҳолати ҳисоби худро таҳрир ё навсозӣ кунед.

Дар саҳифаи Дар бораи барнома, номи ширкат ва майдонҳои вебсайтро пур кунед (ҳатмӣ) ва сипас клик кунед Баъдӣ.

img

Дар саҳифаи Маълумоти ҳисобдорӣ майдони Нақшаи пардохтро клик кунед ва имконотро аз назар гузаронед. Танзимоти пешфарз Free аст. ПУРСИШ ро пахш кунед.

img

Пас аз пешниҳоди шакл, Case Active саҳифаи Барномаҳоро намоиш медиҳад. Барои ҳар як барномае, ки шумо эҷод мекунед ё ба он тааллуқ доред, саҳифа нишон медиҳад:

  1. Номи барнома
  2. Навъи шумо (нақши корбар)
  3. Статус

Саҳифаи Барномаҳо нуқтаи оғоз барои бисёр вазифаҳои администратор аст:

  1. Барои муошират кардан, пур кардани ҳуҷҷатҳо, сӯҳбати видеоӣ (ва ғайра) оид ба масъалаҳои марбут ба парванда тугмаи APP-ро клик кунед.
  2. Барои кушодани панели администратор тугмаи ADMIN -ро клик кунед. Дар ин ҷо маъмурон корбарон, парвандаҳо, ҳамгироӣ ва ғайраро идора мекунанд.
  3. Барои идоракунии ҳисобҳои ҳисоб ва танзимоти огоҳинома эллипсро клик кунед.
  • Эзоҳ Case Active тугмаи ADMIN-ро барои корбарони администратор нишон медиҳад. Корбароне, ки имтиёзҳои администраторӣ надоранд, тугмаи ADMIN-ро намебинанд (тавре ки дар экрани зер нишон дода шудааст).
