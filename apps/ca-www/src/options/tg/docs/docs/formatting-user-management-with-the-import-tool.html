Формат кардани маълумоти идоракунии корбар бо воситаи воридот

Дар зер рӯйхати майдонҳои маълумот дар файлҳои CSV оварда шудааст. Ҳар як варианти воридот дорои маҷмӯи гуногуни майдонҳо мебошад.

номи корбар

  • Суроғаи почтаи электронии корбарро ворид кунед.

Ном

  • Номи корбарро ворид кунед.

насаб

  • Номи корбарро ворид кунед.

маҳаллӣ

  • Ин сутун забони дилхоҳи корбарро тафтиш мекунад.
  • Барои забони англисӣ en, барои испанӣ sp.

zoneinfo

  • Ин минтақаи вақти корбарро муқаррар мекунад. Материк ва шаҳрро бидуни хол ворид кунед (хатти зерро истифода баред).
  • Бояд интихобро дар менюи афтанда Профил > ЗАБОН ва МИНТАҚА > Минтақаи вақт истифода барад.
  • Минтақаи вақти мувофиқро истифода баред. Масалан, Бостон дар афтанда нест, аз ин рӯ мо метавонем Ню Йоркро (ё ҳар як шаҳри минтақаи вақти EST/EDT) истифода барем.

host_tag_id

  • аз, 0-9, -, / ичозат дода мешавад.
