Чӣ тавр ба таври худкор парвандаҳоро бо истиноди бақайдгирии ҷамъиятӣ эҷод кардан мумкин аст

Ин мақола ба маъмурон нишон медиҳад, ки чӣ гуна парванда эҷод кардан (ва ҳамзамон корбарро илова кардан) тавассути фиристодани як истиноди сабти ном.

Ҷойгир кардани пайванд

  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Дар панели маъмур, ҷадвали менюи Танзимотро клик кунед.
  4. Боварӣ ҳосил кунед, ки ҷадвали зерменюи СИСТЕМАИ СИСТЕМА интихоб шудааст.

Танзимоти танзимот

Ба сарлавҳаи РАЙЗИ ИСТИФОДАБАР ба поён ҳаракат кунед.

Боварӣ ҳосил кунед, ки майдони Иҷозати бақайдгирӣ ба Фаъол гузошта шудааст. (зарур) Вақте ки ин хусусият фаъол аст, Case Active истиноди беназир эҷод мекунад.

  1. Майдони худкор сохтани парвандаро клик кунед ва Фаъол -ро интихоб кунед. Майдони Нақши корбар намоиш дода мешавад.
  2. Нақшро интихоб кунед. (нигаред ба поён)

Нақшҳои корбарони парванда

Се нақши корбар вуҷуд дорад.

  • мизоҷон (тавсия дода мешавад)
  • пешфарз
  • кадрхо

Фиристодани пайванд

Истинодро бо як ё якчанд нафар мубодила кунед. Ҳар як корбаре, ки истинодро клик мекунад ва сабти номро анҷом медиҳад, платформаро барои тавлиди парвандаи нав дар саҳифаи парвандаҳо ва ба таври худкор таъин кардани корбар дар парванда ангеза медиҳад.
