Nola sortu eta aldatu erabiltzaileak UI-rekin

Artikulu honek administratzaileei erabiltzaileak sortu eta erregistroak nola eguneratu erakusten die plataformaren erabiltzaile-interfazearekin. Administratzaileek kasu berri batera erabiltzaileak gehi ditzakete eta lehendik dauden kasuak edita ditzakete.

Erabiltzaile bat sortzea

  1. Hasi saioa administratzaile-kontuan.
  2. Bilatu aplikazio bat Aplikazioen orrian eta egin klik ADMIN. botoian.
  3. Admin panelean, egin klik Erabiltzaileak menu fitxan.
  4. Egin klik BERRIA botoian. Leiho berri bat bistaratzen da.

irudia

Bete eremuak eta sakatu GORDE. Case Active-k izena gehitzen du aplikazio honekin lotutako erabiltzaileen zerrendan.

irudia

Erabiltzaile bat aldatzea

Erabiltzaile-erregistro kopuru txiki bat eguneratzeko, erabili Editatu tresna eta egin aldaketak erabiltzaile bakoitzari.

  • Oharra Erabiltzaile-erregistro ugari aldatzeko eserleku batean, kontuan hartu Inportatu tresna erabiltzea.
  1. Admin panelean, egin klik Erabiltzaileak menu fitxan.
  2. Bilatu erabiltzaile bat eta egin klik Editatu.
  3. Egin aldaketak eta sakatu GORDE.

irudia

Hurrengo urratsak

Case Activek ez dio erabiltzaileari posta elektroniko bidezko jakinarazpenik bidaltzen. Administratzaileek erabiltzailearekin harremanetan jarri beharko lukete eta Case Active orri nagusirako esteka bidali beharko lukete. Ondoren, erabiltzaileak LOGIN botoian klik egiten du eta erregistratze berrietarako lan-fluxua osatuko du.

  • Artikulu honek erabiltzaile berrientzako erregistratzeko lan-fluxua deskribatzen du.
