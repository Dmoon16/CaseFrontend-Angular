Administratzaileek nola kudeatzen dituzten erabiltzaileen esleipenak kasuetarako

Erabiltzaileak eta kasuen esleipenak kudeatzeko bost modu daude. Egin klik estekan nola ikasteko.

Metodoa Mota Ekintzak Xehetasunak

UI erabiltzen lehendik dagoen kasuetan UI gehitu eta kendu Admin botoia > Kasuak > Esleitu erabiltzaileak
