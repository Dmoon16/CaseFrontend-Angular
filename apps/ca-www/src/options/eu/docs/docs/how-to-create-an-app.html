Nola sortu aplikazio bat

Artikulu honek erabiltzaile berriei doako probako kontua nola sortu erakusten die. Case Activek automatikoki ematen dizkio erabiltzaile berriari kontu honen administratzaile-pribilegioak.

Kontu bat sortzea

  1. Joan saioa hasteko orrira .
  2. Egin klik Hasi botoian.
  3. Sartu helbide elektroniko bat.
  4. Sakatu Hurrengoa.

irudia

Kontu bat egiaztatzea

  1. Ireki arakatzailearen fitxa berri bat eta hasi saioa zure posta elektronikoko kontuan.
  2. Kopiatu sei zifrako kodea.
  3. Itzuli Kasu Aktiboa fitxara.
  4. Itsatsi kodea eremuetan. Leiho berri bat bistaratzen da.

Leiho berrian:

  1. Bete beharrezko eremuak.
  2. Irakurri zerbitzu-baldintzak eta markatu laukia adosteko.
  3. Sakatu SORTU DOAKO KONTU BAT.

Aplikazioaren konfigurazioa osatzen

Hurrengo orrian lan-fluxu bat bistaratzen da. Bete urratsak zure aplikazioa konfiguratzen amaitzeko. Orri hauetara itzul zaitezke informazioa editatzeko edo eguneratzeko edo zure kontuaren egoera.

Aplikazioari buruz orrian, bete enpresaren izena eta webgunearen eremuak (beharrezkoa) eta sakatu Hurrengoa.

Fakturazio-informazioa orrian, egin klik Ordainketa-plana eremuan eta berrikusi aukerak. Ezarpen lehenetsia Doakoa da. Sakatu BIDALI.

irudia

Inprimakia bidali ondoren, Case Activek Aplikazioen orria bistaratzen du. Sortzen duzun edo parte hartzen duzun aplikazio bakoitzeko, orrialdeak hau erakusten du:

  1. Aplikazioaren izena
  2. Zure mota (erabiltzaile rola)
  3. Egoera

irudia

Aplikazioen orria administrazio-funtzio askoren abiapuntua da:

  1. Egin klik APP botoian kasuekin erlazionatutako arazoei buruz komunikatzeko, dokumentuak betetzeko, bideo-txata (eta gehiago) egiteko.
  2. Egin klik ADMIN. botoian administrazio panela irekitzeko. Hemen administratzaileek erabiltzaileak, kasuak, integrazioak eta abar kudeatzen dituzte.
  3. Sakatu elipsia kontuaren fakturazio eta jakinarazpen ezarpenak kudeatzeko.
  • Oharra Case Active-k ADMIN botoia erakusten du administratzaileentzako. Admin pribilegiorik ez duten erabiltzaileek ez dute ADMIN botoia ikusten (beheko pantailan ikusten den bezala).
