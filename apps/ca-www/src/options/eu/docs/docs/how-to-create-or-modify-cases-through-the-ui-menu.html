Nola sortu edo aldatu kasuak UI menuaren bidez

Artikulu honek administratzaileei plataformaren erabiltzaile-interfazearekin kasu bat nola sortu erakusten die. Administratzaileek kasu berri batera erabiltzaileak gehi ditzakete eta lehendik dauden kasuak egunera ditzakete.

Kasu bat sortzea

  1. Hasi saioa administratzaile-kontuan.
  2. Bilatu aplikazio bat Aplikazioen orrian eta egin klik ADMIN. botoian.
  3. Admin panelean, egin klik Kasuak menuko fitxan.
  4. Egin klik BERRIA botoian. Leiho berri bat bistaratzen da.

Erabiltzaileak gehitzea

Aukeran, administratzaileek erabiltzaileak gehi ditzakete orrialde honetako kasura. Joan behera eta egin klik ESLEITU ERABILTZAILEAK botoian.

Kasu bat aldatzea

Kasu kopuru txiki bat eguneratzeko, erabili Editatu tresna eta egin aldaketak kasu bakoitzean.

  • Oharra Eserleku bakarrean kasu ugari eguneratzeko, kontuan hartu Inportatu tresna erabiltzea
  1. Hasi saioa administratzaile-kontuan.
  2. Bilatu aplikazio bat Aplikazioen orrian eta egin klik ADMIN. botoian.
  3. Admin panelean, egin klik Kasuak menuko fitxan.
  4. Aurkitu kasu bat eta egin klik Editatu.
  5. Egin aldaketak eta sakatu GORDE.
