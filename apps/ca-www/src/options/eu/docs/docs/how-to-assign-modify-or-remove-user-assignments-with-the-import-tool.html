Nola esleitu, aldatu edo kendu erabiltzaileen esleipenak inportazio tresnarekin

Artikulu honek administratzaileei inportazio tresnarekin kasuen erabiltzaileak nola kudeatu erakusten die.

Administratzaileek:

  • erabiltzaileak kasu bati esleitu
  • aldatu lehendik kasu bati esleitutako erabiltzaileak
  • kendu kasu bati esleitutako erabiltzaileak

Inportatu tresna irekitzea

  1. Hasi saioa administratzaile-kontuan.
  2. Bilatu aplikazio bat Aplikazioen orrian eta egin klik ADMIN. botoian.
  3. Egin klik Integrazioa fitxan eta, ondoren, INPORTAZIOAK azpimenuan.
  4. Egin klik BERRIA botoian. Gehitu inportazioak elkarrizketa-koadroa bistaratzen da.

Erregistroak sortzea

Deskargatu CSV fitxategi bat edo gehiago. Erabili fitxategi hauek kasuen erabiltzaileak gehitzeko, aldatzeko edo kentzeko. Gehitu inportazioak elkarrizketa-koadroa ireki ondoren:

  1. Egin klik Mota eremuan.
  2. Joan goitibeherako menuan eta hautatu aukera bat (ikus behean). Esteka bat bistaratzen da.
  3. Egin klik Deskargatu Lagina estekan. CSV fitxategi bat deskargatzen da zure gailuan.

irudia

Ireki fitxategia eta gehitu erregistroak. Egin klik hemen formatu-datuak CSV fitxategian nola erabiltzen diren jakiteko.

Fitxategien aukerak

Esleitu erabiltzaileak kasuei * Aukeratu aukera hau erabiltzaileak kasu batean gehitzeko. * Utzi case_role_order gelaxka hutsik. * Joan Erabiltzaileak orrira eta kopiatu pertsona honen Etiketa IDa. Etiketa ID eremua hutsik badago, idatzi ID esklusibo bat, egin klik Gorde sakatu eta kopiatu IDa CSV fitxategira.

irudia

Aldatu kasuei esleitutako erabiltzaileak * Hautatu aukera hau kasu bati esleitutako uneko erabiltzaileen datuak eguneratzeko. * Erabili kasuaren etiketa IDa case_ID zutaberako.

irudia

Kendu kasuei esleitutako erabiltzaileak * Hautatu aukera hau kasu bati esleitutako erabiltzaileak kentzeko.

irudia

Fitxategia kargatzen

Fitxategi bat berehala karga dezakezu edo Gehitu inportazioak elkarrizketa-koadrora itzuli geroago eta urrats hauek bete.

  1. Ireki Gehitu inportazioak elkarrizketa-koadroa.
  2. Idatzi izen esklusibo bat CSV fitxategirako.
  3. Egin klik plus (+) ikonoan fitxategia zure gailutik kargatzeko.
  4. Egin klik Mota eremuan eta hautatu aukera bat. (ikus goian)
  5. Sakatu GORDE.

Tramitazioa

Fitxategiko erregistro kopuruaren eta sistemaren kargaren arabera, plataformak hainbat minutu behar izan ditzake inportazioa prozesatzeko.

irudia

Inportazioa amaitzen denean, plataformak erabiltzaileen erregistroak doitzen ditu (hau da, gehitzen, aldatzen edo kentzen ditu) Kasuak orrian.

Porrota

Inportazio-eragiketak huts egiten badu, plataformak Huts egin duen mezua itzultzen du. Kasu gehienetan, hutsegitearen kausa datuen formatua da.

irudia

Porrotaren arrazoia ikusteko:

  1. Egin klik Ikusi Ekintza goiburuaren azpian.
  2. Hurrengo orrian, pasatu sagua Huts egin duen ikonoan. Tresnaren argibideak errorea erakusten du.
  3. Joan CVS fitxategira eta egokitu eta konpondu errorea.
  4. Kargatu berriro fitxategia.
