Nola sortu erabiltzaileak inportazio tresnarekin

Artikulu honek inportazio tresnarekin erabiltzaileak nola sortu erakusten die administratzaileei.

Inportatu tresna irekitzea

Administratzaileek Inportatu tresna erabil dezakete erabiltzaile berriak masiboki kargatzeko eta lehendik dauden erabiltzaileen erregistroak aldatzeko.

  1. Hasi saioa administratzaile-kontuan.
  2. Bilatu aplikazio bat Aplikazioen orrian eta egin klik ADMIN. botoian.
  3. Admin panelean, egin klik Integrazioa fitxan eta, ondoren, INPORTAZIOAK azpimenuan.
  4. Egin klik BERRIA botoian. Gehitu inportazioak elkarrizketa-koadroa arakatzaileko fitxa berri batean bistaratzen da.

irudia

Erregistroak sortzea

Gehitu inportazioak elkarrizketa-koadroa ireki ondoren:

  1. Egin klik Mota eremuan eta hautatu aukera bat (ikus behean). Esteka bat bistaratzen da.
  2. Egin klik Deskargatu Lagina estekan. CSV fitxategi bat deskargatzen da zure gailuan.

Ondoren, ireki fitxategia eta gehitu erregistroak. Egin klik hemen formatu-datuak CSV fitxategian nola erabiltzen diren jakiteko.

irudia

Mota

Mota eremuan klik egiten duzunean, goitibeherako menu bat agertzen da. Aukeratu bi aukeretako bat inportazio masiboetarako.

:exclamation: OHARRA Goitibeherako menuak aukera asko ditu, baina bi bakarrik daude lotuta Inportatu tresnarekin. Aukera bakoitzak CSV fitxategi lagin ezberdin bat du.

Gehitu erabiltzaileak

  • Aukeratu aukera hau erabiltzaileen erregistroak kargatzeko.
  • Egin klik Deskargatu Lagina estekan Gehitu erabiltzaileak Inportazio motarako zutabe-izenburuak dituen CSV fitxategi bat irekitzeko.

Erabiltzaileak aldatu

  • Aukeratu aukera hau egungo erabiltzaileen datuak eguneratzeko.
  • Egin klik Deskargatu Lagina estekan CSV fitxategi bat irekitzeko Aldatu erabiltzaileak Inportazio motarako zutabe-izenburuak dituena.

Fitxategia kargatzen

Fitxategi bat berehala karga dezakezu edo Gehitu inportazioak elkarrizketa-koadrora itzuli geroago eta urrats hauek bete.

  1. Ireki Gehitu inportazioak elkarrizketa-koadroa.
  2. Idatzi izen esklusibo bat CSV fitxategirako.
  3. Egin klik plus (+) ikonoan fitxategia zure gailutik kargatzeko.
  4. Egin klik Mota eremuan eta hautatu aukera bat. (ikus goian)
  5. Sakatu GORDE.

irudia

Tramitazioa

Fitxategiko erregistro kopuruaren eta sistemaren kargaren arabera, plataformak hainbat minutu behar izan ditzake inportazioa prozesatzeko.

irudia

Inportazioa amaitzen denean, plataformak erabiltzaileen erregistroak doitzen ditu (hau da, gehitzen, aldatzen edo desgaitzen ditu) administratzailearen Erabiltzaileen panelean.

Porrota

Inportazio-eragiketak huts egiten badu, plataformak Huts egin duen mezua itzultzen du. Kasu gehienetan, hutsegitearen kausa datuen formatua da.

irudia

Porrotaren arrazoia ikusteko:

  1. Egin klik Ikusi Ekintza goiburuaren azpian.
  2. Hurrengo orrialdean, pasatu sagua Huts egin duen ikonoan. Tresnaren argibideak errorea erakusten du.
  3. Joan CVS fitxategira eta egokitu eta konpondu errorea.
  4. Kargatu berriro fitxategia.
