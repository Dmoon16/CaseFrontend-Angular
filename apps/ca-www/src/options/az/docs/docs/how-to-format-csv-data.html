CSV məlumatlarını necə formatlaşdırmaq olar

Aşağıda CSV fayllarında məlumat sahələrinin siyahısı verilmişdir. Hər bir idxal seçimində sahələrin fərqli kombinasiyası var.

istifadəçi adı

  • İstifadəçinin e-poçt ünvanını daxil edin.

verilmiş ad

  • İstifadəçinin adını daxil edin.

ailə Adı

  • İstifadəçinin soyadını daxil edin.

yerli

  • Bu sütun istifadəçinin üstünlük verdiyi dili yoxlayır.
  • İngilis üçün en, ispan üçün sp daxil edin.

zona məlumatı

  • Bu, istifadəçinin vaxt qurşağını təyin edir. Qitəni və şəhəri boşluq qoymadan daxil edin (alt xəttdən istifadə edin).
  • Profil > DİL və BÖLGƏ > Saat qurşağı açılan menyusunda seçimdən istifadə etməlisiniz.
  • Ən uyğun vaxt qurşağından istifadə edin. Məsələn, Boston açılan menyuda deyil, ona görə də biz Nyu-Yorkdan (və ya EST/EDT saat qurşağındakı hər hansı şəhərdən) istifadə edə bilərik.

host_tag_id

  • Daimi etiket daxil edin.
  • az, 0-9, -, / icazə verilir.

case_status

  • 0 = bağlıdır
  • 1 = aktiv
