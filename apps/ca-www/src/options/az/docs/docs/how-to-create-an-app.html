Proqramı Necə Yaratmaq olar

Bu məqalə yeni istifadəçilərə pulsuz sınaq hesabı yaratmağı göstərir. Case Active avtomatik olaraq yeni istifadəçiyə bu hesab üçün admin imtiyazları verir.

Hesab yaradılması

  1. Giriş səhifəsinə keçin .
  2. Başlayın düyməsini basın.
  3. E-poçt ünvanı daxil edin.
  4. Next düyməsini basın.

Hesabın doğrulanması

  1. Yeni bir brauzer nişanı açın və e-poçt hesabınıza daxil olun.
  2. Altı rəqəmli kodu kopyalayın.
  3. Case Active tabına qayıdın.
  4. Kodu sahələrə yapışdırın. Yeni bir pəncərə göstərilir.

Yeni pəncərədə:

  1. Tələb olunan sahələri doldurun.
  2. Xidmət şərtlərini oxuyun və razılaşmaq üçün qutuyu işarələyin.
  3. PULSUZ HESAB YARAT düyməsini klikləyin.

img

Tətbiq quraşdırması tamamlanır

Növbəti səhifədə iş axını göstərilir. Tətbiqinizi qurmağı başa çatdırmaq üçün addımları tamamlayın. Məlumatı və ya hesab statusunuzu redaktə etmək və ya yeniləmək üçün həmişə bu səhifələrə qayıda bilərsiniz.

Tətbiq haqqında səhifəsində şirkət adını və veb sayt sahələrini (tələb olunur) doldurun və sonra Next düyməsini basın.

img

Ödəniş Məlumatı səhifəsində Ödəniş Planı sahəsinə klikləyin və seçimləri nəzərdən keçirin. Standart parametr Pulsuzdur. TƏQDİM ET üzərinə klikləyin.

img

Siz formanı təqdim etdikdən sonra Case Active Proqramlar səhifəsini göstərir. Yaratdığınız və ya aid olduğunuz hər bir proqram üçün səhifədə göstərilir:

  1. Proqramın adı
  2. Tipiniz (istifadəçi rolu)
  3. Vəziyyət

Proqramlar səhifəsi bir çox admin funksiyaları üçün başlanğıc nöqtəsidir:

  1. İşlə bağlı məsələlərlə bağlı ünsiyyət qurmaq, sənədləri doldurmaq, video çat (və daha çox) üçün APP düyməsini klikləyin.
  2. Admin panelini açmaq üçün ADMIN düyməsini klikləyin. Adminlərin istifadəçiləri, hadisələri, inteqrasiyaları və daha çoxunu idarə etdiyi yer budur.
  3. Hesab fakturasını və bildiriş parametrlərini idarə etmək üçün ellipsisin üzərinə klikləyin.
  • Qeyd Case Active admin istifadəçiləri üçün ADMIN düyməsini göstərir. Admin imtiyazları olmayan istifadəçilər ADMIN düyməsini görmürlər (aşağıdakı ekranda göstərildiyi kimi).
