UI ilə istifadəçiləri necə yaratmaq və dəyişdirmək olar

Bu məqalə adminlərə istifadəçiləri necə yaratmağı və platformanın istifadəçi interfeysi ilə qeydləri yeniləməyi göstərir. Adminlər həmçinin istifadəçiləri yeni işə əlavə edə və mövcud halları redaktə edə bilərlər.

İstifadəçinin yaradılması

  1. Admin hesabına daxil olun.
  2. Proqramlar səhifəsində proqram tapın və ADMIN düyməsini klikləyin.
  3. İdarəetmə panelində İstifadəçilər menyu nişanını vurun.
  4. YENİ düyməsini basın. Yeni bir pəncərə göstərilir.

Sahələri doldurun və SAVE düyməsini basın. Case Active adı bu proqramla əlaqəli istifadəçilərin siyahısına əlavə edir.

img

İstifadəçinin dəyişdirilməsi

Az sayda istifadəçi qeydlərini yeniləmək üçün Redaktə alətindən istifadə edin və hər bir istifadəçiyə dəyişiklik edin.

  • Qeyd Bir iclasda çoxlu sayda istifadəçi qeydlərini dəyişdirmək üçün İdxal alətindən istifadə etməyi düşünün.
  1. İdarəetmə panelində İstifadəçilər menyu nişanını vurun.
  2. İstifadəçini tapın və Redaktə et düyməsini basın.
  3. Dəyişikliklər edin və YAXŞI klikləyin.

Növbəti addımlar

Case Active istifadəçiyə e-poçt bildirişi göndərmir. Adminlər istifadəçi ilə əlaqə saxlamalı və onlara Case Active ana səhifəsinə keçid göndərməlidir. İstifadəçi daha sonra LOGIN düyməsini klikləyir və yeni qeydiyyatlar üçün iş prosesini tamamlayır.

  • Bu məqalə yeni istifadəçilər üçün qeydiyyatdan keçmə prosesini təsvir edir.
