UI menyusu vasitəsilə halları necə yaratmaq və ya dəyişdirmək olar

Bu məqalə adminlərə platformanın istifadəçi interfeysi ilə işin necə yaradılacağını göstərir. Adminlər həmçinin istifadəçiləri yeni işə əlavə edə və mövcud halları yeniləyə bilər.

Davanın yaradılması

  1. Admin hesabına daxil olun.
  2. Proqramlar səhifəsində proqram tapın və ADMIN düyməsini klikləyin.
  3. İdarəetmə panelində Cases menyu nişanına klikləyin.
  4. YENİ düyməsini basın. Yeni bir pəncərə göstərilir.

Tələb olunan sahələri doldurun və SAVE düyməsini basın.

İstifadəçilərin əlavə edilməsi

İstəyə görə, adminlər bu səhifədəki işə istifadəçilər əlavə edə bilər. Aşağı diyirləyin və İSTİFADƏÇİLƏRİ TƏYİN ET düyməsini klikləyin.

img

Assign User modalında sahələri doldurun. İstifadəçiyə e-poçt bildirişi göndərmək üçün qutuyu işarələyin. (isteğe bağlı)

img

İşin dəyişdirilməsi

Az sayda işi yeniləmək üçün Redaktə alətindən istifadə edin və hər bir fərdi işə dəyişiklik edin.

  • Qeyd Bir iclasda çox sayda işi yeniləmək üçün İdxal alətindən istifadə edin
  1. Admin hesabına daxil olun.
  2. Proqramlar səhifəsində proqram tapın və ADMIN düyməsini klikləyin.
  3. İdarəetmə panelində Cases menyu nişanına klikləyin.
  4. İşi tapın və Redaktə et üzərinə klikləyin.
  5. Dəyişikliklər edin və YAXŞI klikləyin.
