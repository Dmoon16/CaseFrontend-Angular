İctimai Qeydiyyat Linki ilə İstifadəçiləri Avtomatik Təyin Etmək necə

Adminlər sistem parametrlərini necə konfiqurasiya etməyi öyrənirlər ki, yeni istifadəçilər avtomatik olaraq yeni avtomatik yaradılmış işə təyin olunsunlar.

Quraşdırma səhifəsinin açılması

İlk addım Sistem Quraşdırma səhifəsini açmaqdır.

  1. Admin hesabına daxil olun.
  2. Proqramlar səhifəsində proqram tapın və ADMIN düyməsini klikləyin.
  3. Alt menyuda Sistemlər sekmesini və sonra SİSTEM SETUP üzərinə klikləyin.
  4. İSTİFADƏÇİ QEYDİYYƏSİ başlığına aşağı diyirləyin.

Sistem parametrlərinin konfiqurasiyası

  1. Qeydiyyata İcazə Ver və İşin Avtomatik Yaradılması açılan menyularını Aktiv olaraq təyin edin.
  2. Rol menyusuna klikləyin və bir seçim seçin (standart tövsiyə olunur).
  3. İctimai İstifadəçi Qeydiyyatı linkini kopyalayın və paylaşın.

Linkin göndərilməsi

Linki klikləyən və qeydiyyatı tamamlayan hər bir istifadəçi, İşlər səhifəsində yeni iş yaratmaq üçün platformanı işə salır və bu halda istifadəçini avtomatik təyin edir.
