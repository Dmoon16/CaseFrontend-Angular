Hoe behearders brûkers beheare

Behearders meitsje, wizigje of útskeakelje wa't tagong krije ta apps en gefallen. As in brûker foar it earst oanmelde by it platfoarm, dan moatte se in profylworkflow foltôgje.

  • Klik hjir om mear te learen oer de nije workflow fan it brûkersprofyl.

Behear fan in brûker

De tabel hjirûnder lit fiif manieren sjen om in brûker te behearjen. Klikje op in keppeling om te learen hoe.

Metoade Type Aksjes Details
Mei help fan UI UI tafoegje & feroarje Behearknop > Brûkers
Wylst it meitsjen fan in saak en tawize brûker oan saak UI tafoegje Behearknop > Cases > Nije knop > Case-side oanmeitsje

Notifikaasjes

It platfoarm stjoert brûkers gjin notifikaasje as se wurde tafoege oan in app. Admins kinne lykwols notifikaasjes ynskeakelje foar saakopdrachten.
