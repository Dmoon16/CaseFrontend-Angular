Hjirûnder is in list mei gegevensfjilden yn CSV-bestannen. Elke ymportopsje hat in oare kombinaasje fan fjilden.

case_ID

  • Fier de saak User of Tag ID yn.
  • Gean nei Cases side > Bewurkje > Tag ID OF Gean nei Cases side > Bewurkje > Case ID yn url.

brûker_ID

  • Fier de brûker syn ID of Tag ID.
  • Gean nei brûkers side> Bewurkje> Tag ID OF Gean nei brûkers side> Bewurkje> Brûkers ID yn url.

case_role_ID

case_notify
