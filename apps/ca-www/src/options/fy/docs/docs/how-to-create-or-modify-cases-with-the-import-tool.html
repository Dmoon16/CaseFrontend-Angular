Hoe saken oanmeitsje of wizigje mei it ymportearje-ark

Dit artikel lit admins sjen hoe't jo besteande gefallen kinne oanmeitsje of wizigje mei it ymportearje-ark.

It ymportearje-ark iepenje

  1. Oanmelde by it admin akkount.
  2. Sykje in app op 'e Apps-side en klikje op de ADMIN-knop.
  3. Klikje op it adminpaniel op it ljepblêd Yntegraasje en dan IMPORTS yn it submenu.
  4. Klikje op de NIJE knop. It dialoochfinster ymportearje tafoegje toant yn in nije blêderljepper.

img

It meitsjen fan gefallen

Nei it dialoochfinster Foegje ymporten iepent:

  1. Klikje op it fjild Type en selektearje in opsje (sjoch hjirûnder). In keppeling toant.
  2. Klikje op de keppeling Download Sample. In CSV-bestân wurdt downloade nei jo apparaat.

Folgjende, iepenje it bestân en foegje gefallen ta.

Type

As jo op it fjild Type klikke, sil in útklapmenu werjaan. Kies ien fan twa opsjes foar ymportearjen fan bulkkasten.

  • Opmerking It útklapmenu hat in protte opsjes, mar mar twa binne relatearre oan it behearen fan saak. Elke opsje hat in oare foarbyld CSV-bestân.

Foegje gefallen ta

  • Kies dizze opsje foar in ienfâldige bulk upload fan gefallen.
  • Klikje op de keppeling Download Sample om in CSV-bestân te iepenjen mei kolomkoppen foar it ymporttype fan gefallen oanmeitsje.

Feroarje gefallen

  • Kies dizze opsje om aktuele gefallen te aktualisearjen.
  • Klikje op de keppeling Download Sample om in CSV-bestân te iepenjen mei kolomkoppen foar it ymporttype fan gefallen feroarje.

It opladen fan de triem

Jo kinne direkt in bestân uploade of weromgean nei it dialoochfinster ymportearje tafoegje op in lettere datum en dizze stappen foltôgje.

  1. Iepenje it dialoochfinster ymportearje tafoegje.
  2. Fier in unike namme yn foar it CSV-bestân.
  3. Klikje op it plus (+) ikoan om it bestân fan jo apparaat te uploaden.
  4. Klikje op it fjild Type en selektearje in opsje. (Sjoch hjir boppe)
  5. Klik op SAVE.

img

Opmaak

Dit binne de fereaske fjilden en earste easken. Klik hjir om opmaakdetails te besjen.

tag_id (of case_ID) in unike identifier (brûk alfanumerike tekens, koppeltekens wurde akseptearre)

case_status 0 = sluten 1 = aktyf

date_opened month/day/year mm/dd/yyyy

oer in unike beskriuwing fan 'e saak (brûk alfanumerike tekens, koppeltekens wurde akseptearre)

img

Bewurking

Ofhinklik fan it oantal records yn 'e triem en systeemlading, kin it platfoarm ferskate minuten duorje om de ymport te ferwurkjen.

img

As de ymport is foltôge, past it platfoarm saakrecords oan (dat wol sizze foeget ta of wiziget) op it Cases-dashboard fan de admin.

Mislearring

As de ymportoperaasje mislearret, jout it platfoarm in mislearre berjocht werom. Yn 'e measte gefallen is de oarsaak fan mislearring gegevensopmaak.

Om de oarsaak fan it mislearjen te sjen:

  1. Klik op Besjoch ûnder de koptekst ACTION.
  2. Op de folgjende side, hoverje de mûs op it mislearre ikoan. De tooltip toant de flater.
  3. Gean nei it CVS-bestân en oanpasse en reparearje de flater.
  4. Upload it bestân nochris.
