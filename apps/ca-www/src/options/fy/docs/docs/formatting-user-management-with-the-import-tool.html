Opmaak fan brûkersbeheargegevens mei it ymportearje-ark

Hjirûnder is in list mei gegevensfjilden yn CSV-bestannen. Elke ymportopsje hat in oare kombinaasje fan fjilden.

brûkersnamme

  • Fier it e-postadres fan de brûker yn.

jûne namme

  • Fier de foarnamme fan de brûker yn.

famyljenamme

  • Fier de efternamme fan de brûker yn.

locale

  • Dizze kolom kontrolearret de foarkarstaal fan de brûker.
  • Fier en yn foar Ingelsk, sp foar Spaansk.

zoneinfo

  • Dit stelt de tiidsône fan 'e brûker yn. Fier it kontinint en stêd yn sûnder spaasjes (brûk in ûnderstreekje).
  • Moat in opsje brûke yn it útklapmenu Profile > TAAL & REGION > Tiidsône.
  • Brûk de bêste passende tiidsône. Bygelyks, Boston is net op it dellûk, dus wy kinne New York brûke (as elke stêd yn 'e EST / EDT-tiidsône).

host_tag_id

  • Fier in permanint label yn.
  • az, 0-9, -, / binne tastien.

case_status

  • 0 = sluten
  • 1 = aktyf
