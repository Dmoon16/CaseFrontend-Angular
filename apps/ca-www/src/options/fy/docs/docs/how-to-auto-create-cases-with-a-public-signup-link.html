Hoe kinne jo saken automatysk oanmeitsje mei in keppeling foar iepenbiere oanmelding

Dit artikel lit admins sjen hoe't jo in saak kinne oanmeitsje (en tagelyk in brûker taheakje) troch in inkelde oanmeldingskeppeling te ferstjoeren.

It lokalisearjen fan de keppeling

  1. Oanmelde by it admin akkount.
  2. Sykje in app op 'e Apps-side en klikje op de ADMIN-knop.
  3. Klikje op it adminpaniel op it ljepblêd Ynstellingsmenu.
  4. Soargje derfoar dat de SYSTEEM SETUP submenu ljepper is selektearre.

img

Ynstellings konfigurearje

Rôlje omleech nei de koptekst USER SIGNUP.

Soargje derfoar dat it fjild Oanmelding tastean is ynsteld op Oan. (ferplicht) As dizze funksje ynskeakele is, generearret Case Active in unike keppeling.

  1. Klikje op it fjild Auto oanmeitsje saak en selektearje Oan. It fjild Case User Role toant.
  2. Selektearje in rol. (Sjoch hjirûnder)

img

Case brûkersrollen

D'r binne trije saak brûkersrollen.

  • kliïnten (oanrikkemandearre)
  • standert
  • personiel

It ferstjoeren fan de keppeling

Diel de keppeling mei ien of mear minsken. Elke brûker dy't op de kepling klikt en de oanmelding foltôget, triggert it platfoarm om in nije saak te generearjen op 'e saakpagina en de brûker yn' e saak automatysk tawize.
