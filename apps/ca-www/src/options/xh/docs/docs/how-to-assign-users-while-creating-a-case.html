Ubabela njani abasebenzisi ngelixa usenza iCase

Eli nqaku libonisa indlela yokwabela umsebenzisi ngelixa usenza ityala elitsha. Ukuba umsebenzisi akabhaliswanga eqongeni, unokudala ukongeza umsebenzisi omtsha ngelixa udala ityala elitsha.

Ukudala ityala

  1. Ngena kwiakhawunti admin.
  2. Khangela usetyenziso kwiphepha leApps kwaye ucofe iqhosha elithi ADMIN.
  3. Cofa i Amatyala menu thebhu.
  4. Cofa iqhosha ENTSHA. Ifestile entsha ibonisa.

img

  • Gcwalisa iindawo ukwenza i-case kwaye emva koko uskrolele ezantsi kwisihloko sabasebenzisi abaBelweyo.

Ukwabela umsebenzisi

Umsebenzisi wangoku

Landela la manyathelo ukwabela umsebenzisi wangoku kwimeko entsha.

Cofa iqhosha elithi ASSIGN USERS kwaye ugcwalise iindawo.

  1. Cofa i-User drop-down kwaye ukhethe isazisi (umz. i-imeyile).
  2. Ngenisa ixabiso kwindawo engenanto (umzekelo, idilesi ye-imeyile).
  3. Ukuba irekhodi yomsebenzisi ikhona eqongeni, intsimi iphakamisa ngokuzenzekelayo igama.
  4. Khetha umsebenzisi ochanekileyo.
  5. Khetha indima yomsebenzisi (abaxumi, abangagqibekanga, okanye abasebenzi).
  6. Ngokuzithandela, khangela ibhokisi ukuthumela umsebenzisi isaziso se-imeyile.
  7. Cofa Gcina.

img

I-Case Active yongeza umsebenzisi kwimeko. Ngoku ii-admins zinokucofa Gcina ukuvala ifestile, okanye ukongeza umsebenzisi omtsha, okanye ukususa umsebenzisi.

Ukongeza umsebenzisi omtsha

Ukuba iqonga alikwazi ukufumana irekhodi yangoku, cofa iqhosha ukongeza umsebenzisi omtsha.

img

Gcwalisa amasimi kwaye emva koko ucofe u-GCINA ukongeza umsebenzisi eqongeni. Okulandelayo, seta indima yomsebenzisi kwaye ucofe KUGCINE kwakhona ukwabela umsebenzisi kwimeko. Okokugqibela, cofa u-GCINA elinye ixesha elingakumbi ukuze ugcine ityala.
