Uwenziwa njani okanye uhlengahlengise njani amatyala nge-UI Menu

Eli nqaku libonisa i-admins indlela yokwenza ityala kunye nojongano lomsebenzisi weqonga. Abalawuli banokongeza abasebenzisi kwimeko entsha kwaye bahlaziye iimeko ezikhoyo.

Ukudala ityala

  1. Ngena kwiakhawunti admin.
  2. Khangela usetyenziso kwiphepha leApps kwaye ucofe iqhosha elithi ADMIN.
  3. Kwiphaneli yoLawulo, cofa i-Cases menu tab.
  4. Cofa iqhosha ENTSHA. Ifestile entsha ibonisa.

Ukongeza abasebenzisi

Ngokuzikhethela, ii-admins zingongeza abasebenzisi kwimeko kweli phepha. Skrolela ezantsi kwaye ucofe iqhosha elithi ASSIGN USERS .
