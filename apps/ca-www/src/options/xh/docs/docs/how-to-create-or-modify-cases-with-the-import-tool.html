Uwenziwa njani okanye uhlengahlengise njani amatyala ngeSixhobo sokungeniswa ngaphandle

Eli nqaku libonisa ii-admins indlela yokudala okanye ukulungisa iimeko ezikhoyo ngesixhobo sokungeniswa.

Ukuvula isixhobo sokungeniswa ngaphandle

  1. Ngena kwiakhawunti admin.
  2. Khangela usetyenziso kwiphepha leApps kwaye ucofe iqhosha elithi ADMIN.
  3. Kwiphaneli yoLawulo, cofa ithebhu yoManyano kwaye emva koko U-IMPORTS kwimenyu engaphantsi.
  4. Cofa iqhosha ENTSHA. Ibhokisi yencoko yababini ethi Yongeza Urhweba ngaphandle ibonisa kumkhangeli zincwadi omtsha thebhu.

img

Ukudala iimeko

Emva kwebhokisi yencoko yababini ethi Yongeza ngaphandle ivula:

  1. Cofa indawo yodidi kwaye ukhethe ukhetho (bona ngezantsi). Ikhonkco liyabonisa.
  2. Cofa ikhonkco lokukhuphela iSample. Ifayile ye-CSV ikhutshelwa kwisixhobo sakho.

Emva koko, vula ifayile kwaye wongeze iimeko.

Uhlobo

Xa ucofa indawo yoHlobo , imenyu eyehlayo ibonisa. Khetha enye yezimbini iinketho zokurhweba ngaphandle kweemeko ezininzi.

  • Qaphela Imenyu eyehlayo inokhetho oluninzi, kodwa zimbini kuphela ezinxulumene nolawulo lotyala. Inketho nganye inesampulu eyahlukileyo yefayile ye-CSV.

Yongeza iimeko

  • Khetha olu khetho lokukhuphela isambuku esilula samatyala.
  • Cofa iSample Sample link yokukhuphela ukuvula ifayile ye-CSV enezihloko zekholamu yeYenza iimeko zokungenisa uhlobo.

Guqula iimeko

  • Khetha olu khetho ukuhlaziya iimeko zangoku.
  • Cofa i Sample Sample link yokukhuphela ukuvula ifayile ye CSV enezihloko zekholamu ye Guqula iimeko zokungenisa uhlobo.

Kunyuswa ifayile

Ungalayisha ifayile kwangoko okanye ubuyele kwibhokisi yencoko yababini yoNgenisa Izinto ngaphandle kwamva kwaye ugqibezele la manyathelo.

  1. Vula ibhokisi yencoko yababini ethi Yongeza Izinto ngaphandle.
  2. Ngenisa igama elilodwa lefayile ye-CSV.
  3. Cofa i-plus (+) icon ukulayisha ifayile kwisixhobo sakho.
  4. Cofa i-Thayipha indawo kwaye ukhethe ukhetho. (jonga ngasentla)
  5. Cofa Gcina.

Ukufomatha

Le yimimandla efunekayo kunye neemfuno zokuqala. Cofa apha ukujonga iinkcukacha zokufomatha.

i-tag_id (okanye case_ID) isichongi esisodwa (sebenzisa oonobumba bealphanumeric, iqhagamshela wamkelwe)

case_status 0 = ivaliwe 1 = iyasebenza

umhla_evuliwe inyanga/usuku/unyaka mm/dd/yyyy

malunga nenkcazo ekhethekileyo yecala (sebenzisa iimpawu zealphanumeric, ikhonkco lamkelwe)

Iyaqhuba

Ngokuxhomekeke kwinani leerekhodi kwifayile kunye nomthwalo wenkqubo, iqonga lingathatha imizuzu emininzi ukucubungula ukungenisa.

img

Xa ukungenisa elizweni kugqityiwe, iqonga lihlengahlengisa iirekhodi zetyala (oko kukuthi liyongeza okanye lilungise) kwideshibhodi yeeCasi zolawulo.

Ukusilela

Ukuba umsebenzi wokungenisa elizweni uyasilela, iqonga libuyisela umyalezo ongaphumeleliyo. Kwiimeko ezininzi, unobangela wokusilela kukufomatha idatha.

img

Ukubona unobangela wokusilela:

  1. Cofa Jonga phantsi ko ACTION isihloko.
  2. Kwiphepha elilandelayo, hambisa imouse kwi icon Yoyisakele. Inqaku lesixhobo libonisa impazamo.
  3. Yiya kwifayile yeCVS kwaye ulungise kwaye ulungise imposiso.
  4. Layisha ifayile kwakhona.
