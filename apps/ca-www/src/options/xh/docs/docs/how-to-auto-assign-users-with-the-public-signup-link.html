Ubanika njani abasebenzisi ngokuzenzekela ngekhonkco loNgeniso loLuntu

Abalawuli bafunda indlela yokumisela useto lwenkqubo ukuze abasebenzisi abatsha babelwe ngokuzenzekelayo kwimeko esanda kwenziwa ngokuzenzekelayo.

Ukuvula iphepha lokucwangcisa

Inyathelo lokuqala kukuvula i-Systems Setup page.

  1. Ngena kwiakhawunti admin.
  2. Khangela usetyenziso kwiphepha leApps kwaye ucofe iqhosha elithi ADMIN.
  3. Cofa i Iinkqubo isithuba kwaye emva koko USEKO LWENKQUBO kwimenyu engaphantsi.
  4. Skrolela ezantsi kwi-USER SIGNUP isihloko.

Ukuqwalasela useto lwesixokelelwano

  1. Cwangcisa i-Vumela ukubhalisa kwaye wenze ngokuzenzekela iimenyu zokuhla ukuya kwi-On.
  2. Cofa imenyu yendima kwaye ukhethe ukhetho (okungagqibekanga kuyacetyiswa).
  3. Khuphela kwaye wabelane ngekhonkco loBhaliso loMsebenzisi kaRhulumente.
