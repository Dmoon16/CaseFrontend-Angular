Ubenza njani abasebenzisi ngekhonkco lokubhalisa

Eli nqaku libonisa ii-admins indlela yokwenza abasebenzisi ngokuthumela ikhonkco lokubhalisa.

Ukuqwalasela ikhonkco lokubhalisa

  1. Cofa i Bathumele ikhonkco lokubhalisa ezantsi kwe Yongeza ibhokisi yencoko yababini yomsebenzisi. Iphepha elitsha liyaboniswa.
  2. Qwalasela iinketho zokubhalisa. (bona ngezantsi)

img

Qinisekisa indawo yoVumela uBhalo ukuseta ku Vula. (kuyafuneka) Xa yenziwe, i-Case Active yenza ikhonkco elilodwa.

Jonga ezi ndlela zingezantsi kwaye emva koko ucofe i icon ukukopa ikhonkco. Ungabeka ikhonkco kwiwebhusayithi yakho okanye uthumele ikhonkco kumntu omnye okanye ngaphezulu nge-imeyile okanye naliphi na ijelo lonxibelelwano.

Ukudala ityala

Ngokuzithandela, unokwenza ngokuzenzekelayo ityala kumsebenzisi ngamnye omtsha kwaye unike indima yomsebenzisi.

  1. Cofa i-Auto Create Case field kwaye ukhethe Vula. I-Case User Role fields ibonisa.
  2. Cofa iNdawo yoMsebenzisi weCase kwaye ukhethe ukhetho. (bona ngezantsi)

Ukuthumela ikhonkco

Yabelana ngekhonkco nomntu omnye okanye ngaphezulu. Umntu ngamnye ucofa ikhonkco kwaye agqibezele ukuhamba komsebenzi ukuze avule i-akhawunti yakhe ye-Case Active. Nje ukuba umsebenzisi agqibezele ukubhaliswa komsebenzi, i-Case Active yongeza umsebenzisi kuluhlu lwabasebenzisi.

img

Amanyathelo alandelayo

I-Case Active ithumela umsebenzisi isaziso se-imeyile ngekhonkco lokubhalisa. Umsebenzisi ucofa ikhonkco kwaye agqibezele ukuhamba komsebenzi kubhaliso olutsha.

  • Eli nqaku lichaza ukuhamba komsebenzi wokubhalisa kubasebenzisi abatsha.
