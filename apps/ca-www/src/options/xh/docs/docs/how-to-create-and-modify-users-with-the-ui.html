Uyenza njani kwaye uguqule Abasebenzisi nge-UI

Eli nqaku libonisa indlela yokwenza abasebenzisi kunye nokuhlaziya iirekhodi kunye nojongano lomsebenzisi weqonga. Abalawuli banokongeza abasebenzisi kwimeko entsha kwaye bahlele iimeko ezikhoyo.

Ukudala umsebenzisi

  1. Ngena kwiakhawunti admin.
  2. Khangela usetyenziso kwiphepha leApps kwaye ucofe iqhosha elithi ADMIN.
  3. Kwiphaneli yoLawulo, cofa i Abasebenzisi menu tab.
  4. Cofa iqhosha ENTSHA. Ifestile entsha ibonisa.

img

Gcwalisa iindawo kwaye ucofe KUGCINE. I-Case Active yongeza igama kuluhlu lwabasebenzisi abanxulumene nale app.

img

Ukuguqula umsebenzisi

Ukuhlaziya inani elincinci leerekhodi zomsebenzisi, sebenzisa isixhobo sokuHlela kwaye wenze utshintsho kumsebenzisi ngamnye.

  • Qaphela Ukulungisa inani elikhulu leerekhodi zomsebenzisi kwindawo enye, cinga ukusebenzisa isixhobo sokungeniswa ngaphandle .
  1. Kwiphaneli yoLawulo, cofa i Abasebenzisi menu tab.
  2. Khangela umsebenzisi kwaye ucofe Hlela.
  3. Yenza utshintsho kwaye ucofe KUGCINE.

img

Amanyathelo alandelayo

I-Case Active ayithumeli umsebenzisi isaziso se-imeyile. Abalawuli kufuneka baqhagamshelane nomsebenzisi kwaye babathumele ikhonkco kwiphepha lasekhaya leCase Active. Umsebenzisi emva koko ucofa i-LOGIN iqhosha kwaye agqibezele ukuhamba komsebenzi kubhaliso olutsha.

  • Eli nqaku lichaza ukuhamba komsebenzi wokubhalisa kubasebenzisi abatsha.
