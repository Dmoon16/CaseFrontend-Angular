עיצוב נתוני הקצאת מקרה משתמש עם כלי הייבוא

להלן רשימה של שדות נתונים בקובצי CSV. לכל אפשרות ייבוא יש שילוב שונה של שדות.

case_ID

  • הזן את ה-Case User או Tag ID.
  • עבור לדף מקרים > ערוך > מזהה תג או עבור לדף מקרים > ערוך > מזהה מקרה בכתובת האתר.

תעודת זהות של משתמש

  • הזן את מזהה המשתמש או תג מזהה.
  • עבור לדף משתמשים > עריכה > מזהה תג או עבור לדף משתמשים > ערוך > מזהה משתמש בכתובת האתר.

סדר_תפקיד_מקרה

  • השאר את התא ריק.

case_role_ID

  • הזן: לקוחות, צוות או ברירת מחדל.

case_notify

  • 0 = לא
  • 1 = כן, שלח למשתמש הודעת דוא"ל.
