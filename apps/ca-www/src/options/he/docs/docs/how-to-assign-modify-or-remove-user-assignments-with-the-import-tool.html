כיצד להקצות, לשנות או להסיר הקצאות משתמש עם כלי הייבוא

מאמר זה מראה למנהלי מערכת כיצד לנהל משתמשי מקרה עם כלי הייבוא.

מנהלי מערכת יכולים:

  • להקצות משתמשים למקרה
  • לשנות משתמשים שכבר הוקצו למקרה
  • להסיר משתמשים שהוקצו למקרה

פתיחת כלי הייבוא

  1. היכנס לחשבון הניהול.
  2. אתר אפליקציה בדף האפליקציות ולחץ על הלחצן ADMIN.
  3. לחץ על הכרטיסייה אינטגרציה ולאחר מכן יבוא בתפריט המשנה.
  4. לחץ על כפתור חדש. תיבת הדו-שיח הוסף יבוא מוצגת.

יצירת רשומות

הורד קובץ CSV אחד או יותר. השתמש בקבצים אלה כדי להוסיף, לשנות או להסיר משתמשי מקרה. לאחר שתיבת הדו-שיח הוסף יבוא נפתחת:

  1. לחץ על השדה סוג.
  2. גלול דרך התפריט הנפתח ובחר אפשרות (ראה להלן). מופיע קישור.
  3. לחץ על הקישור הורדה לדוגמה. קובץ CSV מוריד למכשיר שלך.

פתח את הקובץ והוסף רשומות. לחץ כאן כדי ללמוד כיצד להשתמש בנתוני פורמט בקובץ ה-CSV.

אפשרויות קובץ

הקצה משתמשים למקרים * בחר באפשרות זו כדי להוסיף משתמשים למקרה. * השאר את התא case_role_order ריק. * עבור לדף המשתמשים והעתק את מזהה התגית של אדם זה. אם השדה מזהה תג ריק, הזן מזהה ייחודי, לחץ על שמור ולאחר מכן העתק את המזהה לקובץ ה-CSV.

img

שנה משתמשים שהוקצו למקרים * בחר באפשרות זו כדי לעדכן נתונים עבור משתמשים נוכחיים שהוקצו למקרה. * השתמש במזהה תג מקרה עבור העמודה case_ID.

הסרת משתמשים שהוקצו למקרה * בחר באפשרות זו כדי להסיר משתמשים שהוקצו למקרה.

img

מעלה את הקובץ

אתה יכול להעלות קובץ מיד או לחזור לתיבת הדו-שיח הוסף יבוא במועד מאוחר יותר ולהשלים את השלבים האלה.

  1. פתח את תיבת הדו-שיח הוסף יבוא.
  2. הזן שם ייחודי עבור קובץ ה-CSV.
  3. לחץ על סמל הפלוס (+) כדי להעלות את הקובץ מהמכשיר שלך.
  4. לחץ על השדה סוג ובחר אפשרות. (ראה לעיל)
  5. לחץ על שמור.

מעבד

בהתאם למספר הרשומות בקובץ ולטעינת המערכת, הפלטפורמה יכולה לקחת מספר דקות לעבד את הייבוא.

img

כאשר הייבוא מסתיים, הפלטפורמה מתאימה את רשומות המשתמש (כלומר מוסיפה, משנה או מסירה) בדף המקרים.

כישלון

אם פעולת הייבוא נכשלת, הפלטפורמה מחזירה הודעה נכשל. ברוב המקרים, הסיבה לכשל היא עיצוב נתונים.

img

כדי לראות את הסיבה לכשל:

  1. לחץ על הצג תחת הכותרת פעולה.
  2. בעמוד הבא, העבר את העכבר על הסמל 'נכשל'. הסבר הכלי מציג את השגיאה.
  3. עבור לקובץ CVS והתאם ותקן את השגיאה.
  4. העלה את הקובץ שוב.
