כיצד לעדכן פרופיל משתמש

מאמר זה מראה למשתמשים כיצד לעדכן מידע בדף הפרופיל שלהם.

  • משתמשים יכולים לעדכן את הפרופילים שלהם.
  • מנהלי מערכת יכולים לעדכן את דף הפרופיל של כל משתמש.

עבור משתמשים

השלב הראשון הוא פתיחת דף הפרופיל. משתמשים ומנהלי מערכת בצעו את השלבים הבאים כדי לעדכן את דף הפרופיל שלהם.

  1. היכנס לחשבון Case Active שלך.
  2. לחץ על הכרטיסייה פרופיל בראש הדף.
  3. עושה שינויים.
  4. לחץ על שמור.

רוב השדות הם אופציונליים, וחלק מהשדות נדרשים. השדות הנדרשים הם:

  • שם פרטי
  • שם משפחה
  • כתובת דוא"ל
  • שפה
  • אזור זמן

למנהלים

מנהלי מערכת יכולים גם לעדכן את פרטי הפרופיל עבור משתמשים אחרים. זה כולל שלושה שדות תחת הכותרת HOST USER INFO.

  1. היכנס לחשבון הניהול שלך ב-Case Active.
  2. אתר אפליקציה ולחץ על כפתור ADMIN.
  3. לחץ על משתמשים בתפריט העליון.
  4. עבור אל תת תפריט ובחר את רשימת המשתמשים שאליה ברצונך לגשת (פעיל או מושבת). הגדרת ברירת המחדל היא פעיל.
  5. אתר משתמש ברשימה ולחץ על ערוך.
  6. בצע שינויים לפי הצורך. (ראה למטה)
  7. לחץ על שמור.

שנה את פרטי המשתמש

מנהלי מערכת יכולים לשנות את פרטי המשתמש.

  1. הגדר את סוג המשתמש (משתמש או מנהל).
  2. הגדר את סטטוס המשתמש (פעיל לנטרול).
  3. הזן מזהה תג ייחודי. (אופציונאלי)
  4. לחץ על שמור.
