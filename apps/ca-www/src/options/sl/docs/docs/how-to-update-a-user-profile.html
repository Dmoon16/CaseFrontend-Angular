Kako posodobiti uporabniški profil

Ta članek uporabnikom prikazuje, kako posodobiti podatke na strani svojega profila.

  • Uporabniki lahko posodobijo svoje profile.
  • Administratorji lahko posodobijo stran profila katerega koli uporabnika.

Za uporabnike

Prvi korak je, da odprete stran s profilom. Uporabniki in skrbniki sledijo tem korakom, da posodobijo lastno stran s profilom.

  1. Prijavite se v svoj račun Case Active.
  2. Kliknite zavihek Profil na vrhu strani.
  3. Naredite spremembe.
  4. Kliknite Shrani.

Večina polj je neobveznih, nekatera pa so obvezna. Zahtevana polja so:

  • Ime
  • Priimek
  • Email naslov
  • Jezik
  • Časovni pas

Za admine

Skrbniki lahko tudi posodobijo podatke o profilu za druge uporabnike. To vključuje tri polja pod naslovom HOST USER INFO.

  1. Prijavite se v svoj skrbniški račun Case Active.
  2. Poiščite aplikacijo in kliknite gumb SKRBNIK.
  3. V zgornjem meniju kliknite Uporabniki.
  4. Pojdite v podmeni in izberite seznam uporabnikov, do katerega želite dostopati (AKTIVNO ali ONEMOGOČENO). Privzeta nastavitev je Aktivno.
  5. Poiščite uporabnika na seznamu in kliknite Uredi.
  6. Po potrebi spremenite. (glej spodaj)
  7. Kliknite Shrani.

Spremenite podatke o uporabniku

Administratorji lahko spremenijo uporabniške podatke.

  1. Nastavite vrsto uporabnika (uporabnik ali upravitelj).
  2. Nastavite status uporabnika (od aktivnega do onemogočenega).
  3. Vnesite enolični ID oznake. (neobvezno)
  4. Kliknite Shrani.
