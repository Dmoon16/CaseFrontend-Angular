Oblikovanje podatkov o dodelitvah uporabniških primerov z orodjem za uvoz

Spodaj je seznam podatkovnih polj v datotekah CSV. Vsaka možnost uvoza ima drugačno kombinacijo polj.

case_ID

  • Vnesite ID uporabnika ali oznake v primeru.
  • Pojdite na stran Primeri > Uredi > ID oznake ALI Pojdite na stran Primeri > Uredi > ID primera v URL-ju.

Uporabniško ime

  • Vnesite ID uporabnika ali ID oznake.
  • Pojdite na stran Uporabniki > Uredi > ID oznake ALI Pojdite na stran Uporabniki > Uredi > ID uporabnika v URL-ju.

case_role_order

  • Pustite celico prazno.

case_role_ID

  • Vnesite: stranke, osebje ali privzeto.

case_notify

  • 0 = št
  • 1 = Da, pošlji uporabniku e-poštno obvestilo.
