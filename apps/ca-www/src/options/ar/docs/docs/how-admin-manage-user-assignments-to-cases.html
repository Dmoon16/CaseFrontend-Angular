هناك خمس طرق لإدارة المستخدمين وتخصيصات الحالة. انقر فوق ارتباط لمعرفة كيف.

طريقة يكتب أجراءات تفاصيل
أثناء إنشاء الحالة واجهة المستخدم يضيف زر المسؤول> الحالات> زر جديد> زر تعيين المستخدمين
تعيين المستخدم تلقائيًا باستخدام رابط الاشتراك العام عمل المستخدم يضيف زر المسؤول> الإعدادات> صفحة إعداد النظام
استخدام واجهة المستخدم في حالة موجودة واجهة المستخدم اضف احذف زر المسؤول> الحالات> تعيين المستخدمين
أداة الاستيراد واجهة المستخدم الإضافة والتعديل والإزالة زر المسؤول> عمليات التكامل> الواردات
