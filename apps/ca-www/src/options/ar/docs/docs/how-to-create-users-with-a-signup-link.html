كيفية إنشاء مستخدمين باستخدام رابط التسجيل

توضح هذه المقالة للمسؤولين كيفية إنشاء المستخدمين عن طريق إرسال ارتباط تسجيل.

تكوين ارتباط التسجيل

  1. انقر فوق ارتباط إرسال تسجيل لهم في الجزء السفلي من مربع الحوار إضافة مستخدم. يتم عرض صفحة جديدة.
  2. تكوين خيارات التسجيل. (انظر أدناه)

IMG

تأكد من تعيين حقل السماح بالتسجيل على تشغيل. (مطلوب) عند التمكين ، يقوم Case Active بإنشاء ارتباط فريد.

راجع الخيارات أدناه ثم انقر فوق الرمز لنسخ الارتباط. يمكنك وضع الرابط على موقع الويب الخاص بك أو إرسال الرابط إلى شخص أو أكثر عبر البريد الإلكتروني أو أي قناة اتصال.

IMG

خلق حالة

اختياريًا ، يمكنك إنشاء حالة تلقائيًا لكل مستخدم جديد وتعيين دور للمستخدم.

  1. انقر فوق حقل إنشاء الحالة تلقائيًا وحدد تشغيل. يتم عرض مجالات دور مستخدم الحالة.
  2. انقر فوق Case User Role وحدد خيارًا. (انظر أدناه)

  • العاملين

إرسال الارتباط

شارك الرابط مع شخص واحد أو أكثر. ينقر كل شخص على الارتباط ويكمل سير العمل لتنشيط حساب Case Active الخاص به. بمجرد أن يكمل المستخدم سير عمل التسجيل ، تضيف Case Active المستخدم إلى قائمة المستخدمين.

IMG

الخطوات التالية

يرسل Case Active للمستخدم إشعارًا بالبريد الإلكتروني مع ارتباط التسجيل. ينقر المستخدم فوق الارتباط ويكمل سير العمل للاشتراكات الجديدة.
