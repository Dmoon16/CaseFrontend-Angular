كيفية تحديث ملف تعريف المستخدم

توضح هذه المقالة للمستخدمين كيفية تحديث المعلومات على صفحة ملفهم الشخصي.

  • يمكن للمستخدمين تحديث ملفات التعريف الخاصة بهم.
  • يمكن للمسؤولين تحديث صفحة الملف الشخصي لأي مستخدم.

للمستخدمين

الخطوة الأولى هي فتح صفحة الملف الشخصي. يتبع المستخدمون والمسؤولون هذه الخطوات لتحديث صفحة ملف التعريف الخاصة بهم.

  1. قم بتسجيل الدخول إلى حساب Case Active الخاص بك.
  2. انقر فوق علامة التبويب ملف التعريف في الجزء العلوي من الصفحة.
  3. احدث تغيير.
  4. انقر فوق حفظ.

معظم الحقول اختيارية وبعض الحقول مطلوبة. الحقول المطلوبة هي:

  • الاسم الاول
  • اللقب
  • عنوان البريد الالكترونى
  • لغة
  • وحدة زمنية

للمسؤولين

يمكن للمسؤولين أيضًا تحديث معلومات الملف الشخصي للمستخدمين الآخرين. يتضمن هذا ثلاثة حقول تحت عنوان معلومات المستخدم المضيف.

  1. سجّل الدخول إلى حساب مشرف Case Active الخاص بك.
  2. حدد موقع التطبيق وانقر فوق الزر ADMIN.
  3. انقر فوق المستخدمون في القائمة العلوية.
  4. انتقل إلى القائمة الفرعية وحدد قائمة المستخدمين التي تريد الوصول إليها (نشطة أو معطلة). الإعداد الافتراضي هو نشط.
  5. حدد موقع مستخدم في القائمة وانقر فوق تحرير.
  6. قم بإجراء التغييرات على النحو المطلوب. (انظر أدناه)
  7. انقر فوق حفظ.

تغيير معلومات المستخدم

يمكن للمسؤولين تغيير معلومات المستخدم.

  1. اضبط نوع المستخدم (مستخدم أو مدير).
  2. اضبط حالة المستخدم (نشط إلى معطل).
  3. أدخل معرف علامة فريدًا. (اختياري)
  4. انقر فوق حفظ.
