كيفية إنشاء المستخدمين وتعديلهم باستخدام واجهة المستخدم

توضح هذه المقالة للمسؤولين كيفية إنشاء المستخدمين وتحديث السجلات بواجهة مستخدم النظام الأساسي. يمكن للمسؤولين أيضًا إضافة مستخدمين إلى حالة جديدة وتعديل الحالات الموجودة.

إنشاء مستخدم

  1. قم بتسجيل الدخول إلى حساب المسؤول.
  2. حدد موقع تطبيق على صفحة التطبيقات وانقر فوق الزر ADMIN.
  3. في لوحة المسؤول ، انقر فوق علامة تبويب قائمة المستخدمين.
  4. انقر فوق الزر جديد. تظهر نافذة جديدة.

IMG

املأ الحقول وانقر على حفظ. يضيف Case Active الاسم إلى قائمة المستخدمين المرتبطين بهذا التطبيق.

تعديل مستخدم

لتحديث عدد صغير من سجلات المستخدم ، استخدم أداة التحرير وقم بإجراء تغييرات على كل مستخدم على حدة.

  • ملاحظة لتعديل عدد كبير من سجلات المستخدم في جلسة واحدة ، ضع في اعتبارك استخدام أداة الاستيراد .
  1. في لوحة المسؤول ، انقر فوق علامة تبويب قائمة المستخدمين.
  2. حدد موقع مستخدم وانقر فوق تحرير.
  3. قم بإجراء التغييرات وانقر فوق حفظ.

IMG

الخطوات التالية

لا يرسل Case Active المستخدم إشعارًا بالبريد الإلكتروني. يجب على المسؤولين الاتصال بالمستخدم وإرسال ارتباط إلى الصفحة الرئيسية لـ Case Active. ينقر المستخدم بعد ذلك على زر تسجيل الدخول ويكمل سير العمل للاشتراكات الجديدة.

  • توضح هذه المقالة سير عمل التسجيل للمستخدمين الجدد.
