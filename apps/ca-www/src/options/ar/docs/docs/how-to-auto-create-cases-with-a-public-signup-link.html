كيفية إنشاء الحالات تلقائيًا باستخدام رابط اشتراك عام

توضح هذه المقالة للمسؤولين كيفية إنشاء حالة (وإضافة مستخدم في نفس الوقت) عن طريق إرسال رابط اشتراك واحد.

تحديد موقع الارتباط

  1. قم بتسجيل الدخول إلى حساب المسؤول.
  2. حدد موقع تطبيق على صفحة التطبيقات وانقر فوق الزر ADMIN.
  3. في لوحة المسؤول ، انقر فوق علامة تبويب قائمة الإعدادات.
  4. تأكد من تحديد علامة تبويب القائمة الفرعية لإعداد النظام.

تكوين الإعدادات

قم بالتمرير لأسفل إلى عنوان تسجيل المستخدم.

تأكد من تعيين حقل السماح بالتسجيل على تشغيل. (مطلوب) عند تمكين هذه الميزة ، يقوم Case Active بإنشاء ارتباط فريد.

  1. انقر فوق حقل إنشاء الحالة تلقائيًا وحدد تشغيل. يتم عرض حقل دور مستخدم الحالة.
  2. حدد دورًا. (انظر أدناه)

هناك ثلاثة أدوار للمستخدم.

  • عملاء (موصى به)
  • إفتراضي
  • العاملين

إرسال الارتباط

شارك الرابط مع شخص واحد أو أكثر. يقوم كل مستخدم ينقر فوق الارتباط ويكمل التسجيل بتشغيل النظام الأساسي لإنشاء حالة جديدة على صفحة الحالات والتعيين التلقائي للمستخدم في الحالة.
