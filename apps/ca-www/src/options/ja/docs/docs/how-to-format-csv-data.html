CSV データのフォーマット方法

以下は、CSV ファイルのデータ フィールドのリストです。各インポート オプションには、異なるフィールドの組み合わせがあります。

ユーザー名

  • ユーザーの電子メール アドレスを入力します。

名

  • ユーザーの名を入力します。

苗字

  • ユーザーの姓を入力します。

ロケール

  • この列は、ユーザーの優先言語をチェックします。
  • 英語の場合は en、スペイン語の場合は sp を入力します。

ゾーン情報

  • これにより、ユーザーのタイムゾーンが設定されます。大陸と都市をスペースなしで入力します (アンダースコアを使用)。
  • [プロファイル] > [言語と地域] > [タイムゾーン] ドロップダウン メニューのオプションを使用する必要があります。
  • 最適なタイム ゾーンを使用します。たとえば、ボストンはドロップダウンにないため、ニューヨーク (または EST/EDT タイム ゾーンの任意の都市) を使用できます。

host_tag_id

  • 恒久的なラベルを入力してください。
  • az、0-9、-、/ を使用できます。

case_status

  • 0 = クローズ
  • 1 = アクティブ
