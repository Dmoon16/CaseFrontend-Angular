パブリック サインアップ リンクを使用してユーザーを自動割り当てする方法

管理者は、新しく自動作成されたケースに新しいユーザーが自動的に割り当てられるようにシステム設定を構成する方法を学びます。

設定ページを開く

最初のステップは、Systems Setup ページを開くことです。

  1. 管理者アカウントにログインします。
  2. [アプリ] ページでアプリを見つけて、[ADMIN] ボタンをクリックします。
  3. Systems タブをクリックし、サブメニューの SYSTEM SETUP をクリックします。
  4. USER SIGNUP の見出しまで下にスクロールします。

システム設定の構成

  1. [サインアップを許可] および [ケースの自動作成] ドロップダウン メニューを [オン] に設定します。
  2. 役割メニューをクリックして、オプションを選択します (デフォルトが推奨されます)。
  3. パブリック ユーザー サインアップ リンクをコピーして共有します。

リンクの送信

リンクをクリックしてサインアップを完了する各ユーザーは、プラットフォームをトリガーしてケースページに新しいケースを生成し、その場合にユーザーを自動割り当てします。
