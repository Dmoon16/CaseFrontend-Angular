パブリック サインアップ リンクを使用してケースを自動作成する方法

この記事では、単一のサインアップ リンクを送信してケースを作成する方法 (および同時にユーザーを追加する方法) を管理者に示します。

リンクの検索

  1. 管理者アカウントにログインします。
  2. [アプリ] ページでアプリを見つけて、[ADMIN] ボタンをクリックします。
  3. 管理パネルで、[設定] メニュー タブをクリックします。
  4. SYSTEM SETUP サブメニュー タブが選択されていることを確認します。

画像

設定の構成

USER SIGNUP の見出しまでスクロールします。

[サインアップを許可] フィールドが [オン] に設定されていることを確認します。 (必須) この機能が有効な場合、Case Active は一意のリンクを生成します。

  1. [ケースの自動作成] フィールドをクリックし、[オン] を選択します。 [ケース ユーザー ロール] フィールドが表示されます。
  2. 役割を選択します。 （下記参照）

画像

ケース ユーザーの役割

3 つのケース ユーザー ロールがあります。

  • クライアント (推奨)
  • デフォルト
  • スタッフ

リンクの送信

1 人以上のユーザーとリンクを共有します。リンクをクリックしてサインアップを完了する各ユーザーは、プラットフォームをトリガーしてケースページに新しいケースを生成し、ケースにユーザーを自動割り当てます。
