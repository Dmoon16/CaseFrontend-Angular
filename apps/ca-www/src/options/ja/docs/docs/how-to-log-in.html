ログイン方法

既存の Case Active アカウントにログインする方法を学びます。

ログイン中

  1. ログインページに移動します。
  2. ログインボタンをクリックします。
  3. メールアドレスを入力。
  4. [次へ] をクリックします。

アカウントの確認

  1. 新しいブラウザー タブを開き、メール アカウントにログインします。
  2. 6 桁のコードをコピーします。
  3. Case Active タブに戻ります。
  4. コードをフィールドに貼り付けます。新しいウィンドウが表示されます。

Case Active に [Apps] ページが表示されます。

  • 注プラットフォームでは、管理者権限を持つユーザーの場合、APP ボタンの横に ADMIN ボタンが表示されます。
