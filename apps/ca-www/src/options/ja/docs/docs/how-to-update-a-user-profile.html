ユーザー プロファイルを更新する方法

この記事では、ユーザーがプロファイル ページの情報を更新する方法を説明します。

  • ユーザーは自分のプロファイルを更新できます。
  • 管理者は、任意のユーザーのプロフィール ページを更新できます。

ユーザー向け

最初のステップは、プロファイル ページを開くことです。ユーザーと管理者は、次の手順に従って自分のプロファイル ページを更新します。

  1. Case Active アカウントにログインします。
  2. ページの上部にある [プロファイル] タブをクリックします。
  3. 変更を加えます。
  4. [保存] をクリックします。

ほとんどのフィールドはオプションで、一部のフィールドは必須です。必須フィールドは次のとおりです。

  • ファーストネーム
  • 苗字
  • 電子メールアドレス
  • 言語
  • タイムゾーン

管理者向け

管理者は、他のユーザーのプロフィール情報を更新することもできます。これには、HOST USER INFO 見出しの下にある 3 つのフィールドが含まれます。

  1. Case Active 管理者アカウントにログインします。
  2. アプリを見つけて、[ADMIN] ボタンをクリックします。
  3. 上部のメニューで [ユーザー] をクリックします。
  4. サブメニューに移動し、アクセスしたいユーザー リストを選択します (ACTIVE または DISABLED)。デフォルト設定はアクティブです。
  5. リストでユーザーを見つけて、[編集] をクリックします。
  6. 必要に応じて変更します。 （下記参照）
  7. [保存] をクリックします。

ユーザー情報を変更する

管理者はユーザー情報を変更できます。

  1. ユーザー タイプ (ユーザーまたはマネージャー) を設定します。
  2. ユーザー ステータスを設定します (アクティブから無効)。
  3. 一意のタグ ID を入力します。 (オプション)
  4. [保存] をクリックします。
