システム管理者がケースへのユーザー割り当てを管理する方法

ユーザーとケースの割り当てを管理するには、5 つの方法があります。方法については、リンクをクリックしてください。

方法 タイプ 行動 詳細
ケースの作成中 UI 追加 [管理] ボタン > [ケース] > [新規] ボタン > [ユーザーの割り当て] ボタン
公開サインアップ リンクを使用してユーザーを自動割り当て ユーザー アクション 追加 管理者ボタン > 設定 > システム設定ページ
既存のケースで UI を使用する UI 削除を追加 管理ボタン > ケース > ユーザーの割り当て
インポートツール UI 追加、変更、および削除 管理ボタン > 統合 > インポート
API の使用 (準備中) API 追加、変更、および削除 なし
