既存のケースでユーザー割り当てを割り当てるまたは削除する方法

この記事では、管理者がケース ユーザーを割り当ておよび削除する方法を示します。また、ユーザー データを変更する方法も示します。

既存のケースを開く

  1. 管理者アカウントにログインします。
  2. [アプリ] ページでアプリを見つけて、[ADMIN] ボタンをクリックします。
  3. [ケース] メニュー タブをクリックします。
  4. サブメニューで ACTIVE が選択されていることを確認します。
  5. ACTION 列まで下にスクロールします。

割り当てまたは削除

[ユーザーの割り当て] をクリックして、ケースにユーザーを割り当てるか、ケースからユーザーを削除します (ただし、プラットフォームは削除しません)。次のページ: [追加] をクリックして、ワークフローを完了します。ユーザーを削除するには、削除をクリックします。

画像

変更

ケースページでユーザーデータを編集または更新することはできません。代わりに、次の手順に従います。

  1. [ユーザー] メニュー タブをクリックします。
  2. ユーザー名を見つけて、[編集] をクリックします。
  3. 変更を行い、[保存] をクリックします。
