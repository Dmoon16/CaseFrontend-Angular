ケースの作成中にユーザーを割り当てる方法

この記事では、管理者が新しいケースを作成する際にユーザーを割り当てる方法を示します。ユーザーがまだプラットフォームに登録されていない場合は、新しいケースの作成中に新しいユーザーを作成して追加できます。

ケースの作成

  1. 管理者アカウントにログインします。
  2. [アプリ] ページでアプリを見つけて、[ADMIN] ボタンをクリックします。
  3. [ケース] メニュー タブをクリックします。
  4. NEW ボタンをクリックします。新しいウィンドウが表示されます。

画像

  • フィールドに入力してケースを作成し、割り当てられたユーザーの見出しまで下にスクロールします。

ユーザーの割り当て

現在の使用者

次の手順に従って、現在のユーザーを新しいケースに割り当てます。

ASSIGN USERS ボタンをクリックし、フィールドに入力します。

  1. [ユーザー] ドロップダウンをクリックし、識別子 (電子メールなど) を選択します。
  2. 空白のフィールドに値を入力します (例: 電子メール アドレス)。
  3. ユーザー レコードがプラットフォームに存在する場合、フィールドは名前を自動提案します。
  4. 正しいユーザーを選択してください。
  5. ユーザーの役割 (クライアント、デフォルト、またはスタッフ) を選択します。
  6. 必要に応じて、ボックスをオンにしてユーザーに電子メール通知を送信します。
  7. [保存] をクリックします。

画像

Case Active は、ユーザーをケースに追加します。管理者は [保存] をクリックしてウィンドウを閉じたり、新しいユーザーを追加したり、ユーザーを削除したりできるようになりました。

新しいユーザーの追加

プラットフォームが現在のレコードを見つけられない場合は、ボタンをクリックして新しいユーザーを追加します。

画像

フィールドに入力し、[保存] をクリックしてユーザーをプラットフォームに追加します。次に、ユーザーの役割を設定し、もう一度 [保存] をクリックしてユーザーをケースに割り当てます。最後に、SAVE をもう一度クリックしてケースを保存します。
