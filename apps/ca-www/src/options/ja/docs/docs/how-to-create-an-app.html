アプリの作成方法

この記事では、新しいユーザーが無料試用版アカウントを作成する方法を示します。 Case Active は、このアカウントの新しいユーザー管理者権限を自動的に付与します。

アカウントの作成

  1. ログインページに移動します。
  2. [開始する] ボタンをクリックします。
  3. メールアドレスを入力します。
  4. [次へ] をクリックします。

画像

アカウントの確認

  1. 新しいブラウザー タブを開き、メール アカウントにログインします。
  2. 6 桁のコードをコピーします。
  3. Case Active タブに戻ります。
  4. コードをフィールドに貼り付けます。新しいウィンドウが表示されます。

新しいウィンドウで:

  1. 必須フィールドに入力します。
  2. 利用規約を読み、同意するボックスにチェックを入れます。
  3. [無料アカウントを作成] をクリックします。

画像

アプリのセットアップの完了

次のページにワークフローが表示されます。手順を完了して、アプリのセットアップを完了します。これらのページにいつでも戻って、情報やアカウントのステータスを編集または更新できます。

[アプリについて] ページで、会社名と Web サイトのフィールド (必須) に入力し、[次へ] をクリックします。

画像

[請求情報] ページで、[支払いプラン] フィールドをクリックし、オプションを確認します。デフォルト設定は無料です。 [送信] をクリックします。

画像

フォームを送信すると、Case Active に [アプリ] ページが表示されます。作成または所属するアプリごとに、ページに次の情報が表示されます。

  1. アプリ名
  2. あなたのタイプ (ユーザーの役割)
  3. 状態

画像

アプリ ページは、多くの管理機能の出発点です。

  1. APP ボタンをクリックして、ケース関連の問題についてコミュニケーション、ドキュメントへの記入、ビデオ チャットなどを行います。
  2. [ADMIN] ボタンをクリックして、管理パネルを開きます。これは、管理者がユーザー、ケース、統合などを管理する場所です。
  3. 省略記号をクリックして、アカウントの請求と通知の設定を管理します。
  • 注Case Active は、管理者ユーザーの ADMIN ボタンを表示します。管理者権限を持たないユーザーには、[ADMIN] ボタンは表示されません (下の画面を参照)。
