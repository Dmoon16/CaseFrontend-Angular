UI メニューからケースを作成または変更する方法

この記事では、プラットフォームのユーザー インターフェイスを使用してケースを作成する方法を管理者に示します。管理者は、ユーザーを新しいケースに追加したり、既存のケースを更新したりすることもできます。

ケースの作成

  1. 管理者アカウントにログインします。
  2. [アプリ] ページでアプリを見つけて、[ADMIN] ボタンをクリックします。
  3. 管理パネルで、[ケース] メニュー タブをクリックします。
  4. NEW ボタンをクリックします。新しいウィンドウが表示されます。

必須フィールドに入力し、[保存] をクリックします。

ユーザーの追加

必要に応じて、管理者はこのページでケースにユーザーを追加できます。下にスクロールして、[ユーザーの割り当て] ボタンをクリックします。

画像

ユーザーの割り当てモーダルで、フィールドに入力します。チェックボックスをオンにして、ユーザーに電子メール通知を送信します。 (オプション)

画像

ケースの変更

少数のケースを更新するには、編集ツールを使用して個々のケースを変更します。

  • 注一度に多数のケースを更新するには、インポート ツールの使用を検討してください。
  1. 管理者アカウントにログインします。
  2. [アプリ] ページでアプリを見つけて、[ADMIN] ボタンをクリックします。
  3. 管理パネルで、[ケース] メニュー タブをクリックします。
  4. ケースを見つけて、[編集] をクリックします。
  5. 変更を行い、[保存] をクリックします。
