Paano Magtalaga, Magbago o Mag-alis ng Mga Assignment ng User gamit ang Tool sa Pag-import

Ipinapakita ng artikulong ito sa mga admin kung paano pamahalaan ang mga user ng kaso gamit ang tool sa Pag-import.

Ang mga admin ay maaaring:

  • magtalaga ng mga user sa isang case
  • baguhin ang mga user na nakatalaga na sa isang case
  • alisin ang mga user na nakatalaga sa isang case

Binubuksan ang tool sa Pag-import

  1. Mag-log in sa admin account.
  2. Maghanap ng app sa page ng Apps at i-click ang ADMIN button.
  3. I-click ang tab na Integration at pagkatapos ay IMPORTS sa submenu.
  4. I-click ang BAGONG button. Nagpapakita ang dialog box ng Add Imports.

Paglikha ng mga talaan

Mag-download ng isa o higit pang CSV file. Gamitin ang mga file na ito upang magdagdag, magbago, o mag-alis ng mga user ng case. Pagkatapos magbukas ang dialog box ng Add Imports:

  1. I-click ang Uri ng field.
  2. Mag-scroll sa drop-down na menu at pumili ng opsyon (tingnan sa ibaba). Nagpapakita ang isang link.
  3. I-click ang link na I-download ang Sample. Nagda-download ang isang CSV file sa iyong device.

Buksan ang file at magdagdag ng mga tala. Mag-click dito upang matutunan kung paano gamitin ang data ng format sa CSV file.

Mga pagpipilian sa file

Magtalaga ng mga user sa mga case * Piliin ang opsyong ito para magdagdag ng mga user sa isang case. * Iwanang blangko ang case_role_order cell. * Pumunta sa page ng Mga User at kopyahin ang Tag ID para sa taong ito. Kung blangko ang field ng Tag ID, maglagay ng natatanging ID, i-click ang I-save, at pagkatapos ay kopyahin ang ID sa CSV file.

Baguhin ang mga user na nakatalaga sa mga case * Piliin ang opsyong ito para i-update ang data para sa mga kasalukuyang user na nakatalaga sa isang case. * Gamitin ang Case Tag ID para sa case_ID column.

Alisin ang mga user na nakatalaga sa mga case * Piliin ang opsyong ito para alisin ang mga user na nakatalaga sa isang case.

Pag-upload ng file

Maaari kang mag-upload kaagad ng file o bumalik sa dialog box na Magdagdag ng Mga Pag-import sa ibang araw at kumpletuhin ang mga hakbang na ito.

  1. Buksan ang dialog box na Magdagdag ng Mga Import.
  2. Maglagay ng natatanging pangalan para sa CSV file.
  3. I-click ang icon na plus (+) upang i-upload ang file mula sa iyong device.
  4. I-click ang field na Uri at pumili ng opsyon. (tingnan sa itaas)
  5. I-click ang I-SAVE.

Pinoproseso

Depende sa bilang ng mga tala sa pag-load ng file at system, maaaring tumagal ng ilang minuto ang platform upang maproseso ang pag-import.

Kapag nakumpleto ang pag-import, inaayos ng platform ang mga talaan ng user (ibig sabihin, nagdaragdag, nagbabago, o nag-aalis) sa pahina ng Mga Kaso.

Kabiguan

Kung nabigo ang pagpapatakbo ng pag-import, magbabalik ang platform ng isang Nabigong mensahe. Sa karamihan ng mga kaso, ang sanhi ng pagkabigo ay ang pag-format ng data.

Upang makita ang sanhi ng pagkabigo:

  1. I-click ang View sa ilalim ng Action heading.
  2. Sa susunod na pahina, i-hover ang mouse sa icon na Nabigo. Ipinapakita ng tooltip ang error.
  3. Pumunta sa CVS file at ayusin at ayusin ang error.
  4. I-upload muli ang file.
