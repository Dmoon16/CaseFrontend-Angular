Paano Mag-log In

Matutunan kung paano mag-log in sa iyong kasalukuyang Case Active na account.

Nagla-log in

  1. Pumunta sa login page .
  2. I-click ang LOGIN button.
  3. Ilagay ang iyong email.
  4. I-click ang NEXT.

Pag-verify ng account

  1. Magbukas ng bagong tab ng browser at mag-log in sa iyong email account.
  2. Kopyahin ang anim na digit na code.
  3. Bumalik sa tab na Case Active.
  4. I-paste ang code sa mga field. May lalabas na bagong window.

Ipinapakita ng Case Active ang pahina ng Apps.

  • Tandaan Ang platform ay nagpapakita ng ADMIN button sa tabi ng APP button para sa mga user na may mga pribilehiyo ng admin.
