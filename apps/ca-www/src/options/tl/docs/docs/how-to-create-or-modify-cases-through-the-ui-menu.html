Paano Gumawa o Magbago ng Mga Case sa pamamagitan ng Menu ng UI

Ipinapakita ng artikulong ito sa mga admin kung paano gumawa ng case gamit ang user interface ng platform. Ang mga admin ay maaari ding magdagdag ng mga user sa isang bagong kaso at i-update ang mga kasalukuyang kaso.

Paggawa ng kaso

  1. Mag-log in sa admin account.
  2. Maghanap ng app sa page ng Apps at i-click ang ADMIN button.
  3. Sa Admin panel, i-click ang tab na menu ng Cases.
  4. I-click ang BAGONG button. May lalabas na bagong window.

Punan ang mga kinakailangang field at i-click ang SAVE.

img

Pagdaragdag ng mga user

Opsyonal, maaaring magdagdag ang mga admin ng mga user sa case sa page na ito. Mag-scroll pababa at i-click ang ASSIGN USERS button.

Sa Assign User modal, punan ang mga field. Lagyan ng check ang kahon upang magpadala sa user ng isang abiso sa email. (opsyonal)

img

Pagbabago ng isang kaso

Upang i-update ang isang maliit na bilang ng mga kaso, gamitin ang tool sa Pag-edit at gumawa ng mga pagbabago sa bawat indibidwal na kaso.

  • Tandaan Upang i-update ang isang malaking bilang ng mga kaso sa isang upuan, isaalang-alang ang paggamit ng tool sa Pag-import
  1. Mag-log in sa admin account.
  2. Maghanap ng app sa page ng Apps at i-click ang ADMIN button.
  3. Sa Admin panel, i-click ang tab na menu ng Cases.
  4. Maghanap ng case at i-click ang I-edit.
  5. Gumawa ng mga pagbabago at i-click ang I-SAVE.
