Paano Magtalaga o Mag-alis ng Mga Assignment ng User na may Mga Umiiral na Kaso

Ipinapakita ng artikulong ito sa mga admin kung paano magtalaga at mag-alis ng mga user ng case. Ipinapakita rin namin sa iyo kung paano baguhin ang data ng user.

Pagbubukas ng kasalukuyang kaso

  1. Mag-log in sa admin account.
  2. Maghanap ng app sa page ng Apps at i-click ang ADMIN button.
  3. I-click ang tab na menu ng Cases .
  4. Tiyaking ACTIVE ang napili sa submenu.
  5. Mag-scroll pababa sa column na ACTION.

img

Italaga o alisin

I-click ang Magtalaga ng Mga User upang magtalaga ng user sa case o alisin ang user mula sa case (ngunit hindi ang platform). Sa susunod na page: I-click ang ADD at kumpletuhin ang workflow. I-click ang alisin upang magtanggal ng user.

img

Baguhin

Hindi posibleng i-edit o i-update ang data ng user sa page ng Cases. Sa halip, sundin ang mga hakbang na ito:

  1. I-click ang tab na menu ng Mga User .
  2. Hanapin ang user name at i-click ang I- edit .
  3. Gumawa ng mga pagbabago at i-click ang I-SAVE.
