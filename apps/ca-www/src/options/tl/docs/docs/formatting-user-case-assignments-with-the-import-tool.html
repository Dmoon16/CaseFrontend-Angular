Pag-format ng Data ng Mga Pagtatalaga sa Kaso ng User gamit ang Tool sa Pag-import

Nasa ibaba ang isang listahan ng mga field ng data sa mga CSV file. Ang bawat opsyon sa pag-import ay may iba't ibang kumbinasyon ng mga field.

Case ID

  • Ilagay ang case User o Tag ID.
  • Pumunta sa page ng Cases > Edit > Tag ID O Pumunta sa Cases page > Edit > Case ID sa url.

user_ID

  • Ilagay ang user ID o Tag ID.
  • Pumunta sa page ng Users > Edit > Tag ID O Pumunta sa Users page > Edit > User ID sa url.

case_role_order

  • Iwanang blangko ang cell.

case_role_ID

  • Enter: mga kliyente, staff, o default.

case_notify

  • 0 = hindi
  • 1 = Oo, magpadala sa user ng isang abiso sa email.
