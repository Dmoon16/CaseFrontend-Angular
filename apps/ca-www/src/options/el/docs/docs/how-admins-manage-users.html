Πώς οι διαχειριστές διαχειρίζονται τους χρήστες

Οι διαχειριστές δημιουργούν, τροποποιούν ή απενεργοποιούν τα άτομα που έχουν πρόσβαση σε εφαρμογές και υποθέσεις. Όταν ένας χρήστης συνδέεται στην πλατφόρμα για πρώτη φορά, πρέπει να ολοκληρώσει μια ροή εργασίας προφίλ.

  • Κάντε κλικ εδώ για να μάθετε περισσότερα σχετικά με τη ροή εργασίας του νέου προφίλ χρήστη.

Διαχείριση χρήστη

Ο παρακάτω πίνακας δείχνει πέντε τρόπους διαχείρισης ενός χρήστη. Κάντε κλικ σε έναν σύνδεσμο για να μάθετε πώς.

Ειδοποιήσεις

Η πλατφόρμα δεν στέλνει στους χρήστες ειδοποίηση όταν προστίθενται σε μια εφαρμογή. Ωστόσο, οι διαχειριστές μπορούν να ενεργοποιήσουν τις ειδοποιήσεις για αναθέσεις υποθέσεων.
