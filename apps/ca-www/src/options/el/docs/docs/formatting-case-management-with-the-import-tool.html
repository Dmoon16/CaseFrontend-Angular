Μορφοποίηση δεδομένων διαχείρισης υποθέσεων με το εργαλείο εισαγωγής

Ακολουθεί μια λίστα με τα απαιτούμενα πεδία σε αρχεία CSV για τη διαχείριση υποθέσεων με το εργαλείο Εισαγωγή.

όνομα χρήστη

  • Εισαγάγετε τη διεύθυνση email του χρήστη.

tag_id (ένα αναγνωριστικό στο οποίο μπορείτε να αναφέρετε αργότερα)

  • ένα μοναδικό αναγνωριστικό (χρησιμοποιήστε αλφαριθμητικούς χαρακτήρες, οι παύλες γίνονται δεκτές)

περίπτωση_κατάσταση

  • 0 = κλειστό
  • 1 = ενεργό

date_opened

  • Μηνας μερα χρονος
  • μμ/ηη/εεεε

σχετικά με

  • μια μοναδική περιγραφή της υπόθεσης (χρησιμοποιήστε αλφαριθμητικούς χαρακτήρες, γίνονται δεκτές παύλες)
