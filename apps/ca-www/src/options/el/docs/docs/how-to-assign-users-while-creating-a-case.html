Πώς να ορίσετε χρήστες κατά τη δημιουργία μιας θήκης

Αυτό το άρθρο δείχνει στους διαχειριστές πώς να εκχωρούν έναν χρήστη κατά τη δημιουργία μιας νέας υπόθεσης. Εάν ο χρήστης δεν είναι ήδη εγγεγραμμένος στην πλατφόρμα, μπορείτε να δημιουργήσετε προσθήκη νέου χρήστη ενώ δημιουργείτε μια νέα περίπτωση.

Δημιουργία θήκης

  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Κάντε κλικ στην καρτέλα μενού Υποθέσεων.
  4. Κάντε κλικ στο κουμπί ΝΕΟ. Εμφανίζεται ένα νέο παράθυρο.
  • Συμπληρώστε τα πεδία για να δημιουργήσετε μια θήκη και, στη συνέχεια, κάντε κύλιση προς τα κάτω στην επικεφαλίδα Εκχωρημένοι χρήστες.

Εκχώρηση χρήστη

Τρέχων χρήστης

Ακολουθήστε αυτά τα βήματα για να αντιστοιχίσετε έναν τρέχοντα χρήστη στη νέα υπόθεση.

Κάντε κλικ στο κουμπί ASIGN USERS και συμπληρώστε τα πεδία.

  1. Κάντε κλικ στο αναπτυσσόμενο μενού Χρήστης και επιλέξτε ένα αναγνωριστικό (π.χ. email).
  2. Εισαγάγετε μια τιμή στο κενό πεδίο (π.χ. διεύθυνση email).
  3. Εάν η εγγραφή χρήστη υπάρχει στην πλατφόρμα, το πεδίο προτείνει αυτόματα ένα όνομα.
  4. Επιλέξτε τον σωστό χρήστη.
  5. Επιλέξτε έναν ρόλο χρήστη (πελάτες, προεπιλογή ή προσωπικό).
  6. Προαιρετικά, επιλέξτε το πλαίσιο για να στείλετε στον χρήστη μια ειδοποίηση μέσω email.
  7. Κάντε κλικ στο ΑΠΟΘΗΚΕΥΣΗ.

Το Case Active προσθέτει τον χρήστη στη θήκη. Τώρα οι διαχειριστές μπορούν να κάνουν κλικ στην Αποθήκευση για να κλείσουν το παράθυρο ή να προσθέσουν έναν νέο χρήστη ή να αφαιρέσουν έναν χρήστη.

Προσθήκη νέου χρήστη

Εάν η πλατφόρμα δεν μπορεί να βρει μια τρέχουσα εγγραφή, κάντε κλικ στο κουμπί για να προσθέσετε έναν νέο χρήστη.

Συμπληρώστε τα πεδία και μετά κάντε κλικ στην ΑΠΟΘΗΚΕΥΣΗ για να προσθέσετε τον χρήστη στην πλατφόρμα. Στη συνέχεια, ορίστε έναν ρόλο χρήστη και κάντε ξανά κλικ στο SAVE για να εκχωρήσετε τον χρήστη στην υπόθεση. Τέλος, κάντε κλικ στο ΑΠΟΘΗΚΕΥΣΗ άλλη μια φορά για να αποθηκεύσετε την υπόθεση.
