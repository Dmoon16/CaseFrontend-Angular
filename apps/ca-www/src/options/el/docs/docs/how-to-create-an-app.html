Πώς να δημιουργήσετε μια εφαρμογή

Αυτό το άρθρο δείχνει στους νέους χρήστες πώς να δημιουργήσουν έναν δωρεάν δοκιμαστικό λογαριασμό. Το Case Active εκχωρεί αυτόματα στον νέο χρήστη δικαιώματα διαχειριστή για αυτόν τον λογαριασμό.

Δημιουργία λογαριασμού

  1. Μεταβείτε στη σελίδα σύνδεσης .
  2. Κάντε κλικ στο κουμπί Ξεκινήστε.
  3. Πληκτρολογήστε μια διεύθυνση ηλεκτρονικού ταχυδρομείου.
  4. Κάντε κλικ στο Επόμενο.

Επαλήθευση λογαριασμού

  1. Ανοίξτε μια νέα καρτέλα προγράμματος περιήγησης και συνδεθείτε στον λογαριασμό email σας.
  2. Αντιγράψτε τον εξαψήφιο κωδικό.
  3. Επιστρέψτε στην καρτέλα Case Active.
  4. Επικολλήστε τον κωδικό στα πεδία. Εμφανίζεται ένα νέο παράθυρο.

Στο νέο παράθυρο:

  1. Συμπληρώστε τα απαιτούμενα πεδία.
  2. Διαβάστε τους όρους παροχής υπηρεσιών και επιλέξτε το πλαίσιο για να συμφωνήσετε.
  3. Κάντε κλικ στο ΔΗΜΙΟΥΡΓΙΑ ΔΩΡΕΑΝ ΛΟΓΑΡΙΑΣΜΟΥ.

img

Ολοκλήρωση της ρύθμισης της εφαρμογής

Μια ροή εργασίας εμφανίζεται στην επόμενη σελίδα. Ολοκληρώστε τα βήματα για να ολοκληρώσετε τη ρύθμιση της εφαρμογής σας. Μπορείτε πάντα να επιστρέψετε σε αυτές τις σελίδες για να επεξεργαστείτε ή να ενημερώσετε τις πληροφορίες ή την κατάσταση του λογαριασμού σας.

Στη σελίδα Σχετικά με την εφαρμογή, συμπληρώστε τα πεδία ονόματος εταιρείας και ιστότοπου (απαιτείται) και, στη συνέχεια, κάντε κλικ στο Επόμενο.

Στη σελίδα Πληροφορίες χρέωσης, κάντε κλικ στο πεδίο Σχέδιο πληρωμών και ελέγξτε τις επιλογές. Η προεπιλεγμένη ρύθμιση είναι Δωρεάν. Κάντε κλικ στο SUBMIT.

img

Αφού υποβάλετε τη φόρμα, το Case Active εμφανίζει τη σελίδα Εφαρμογές. Για κάθε εφαρμογή που δημιουργείτε ή στην οποία ανήκετε, η σελίδα εμφανίζει:

  1. Το όνομα της εφαρμογής
  2. Ο τύπος σας (ρόλος χρήστη)
  3. Κατάσταση

img

Η σελίδα Εφαρμογές είναι ένα σημείο εκκίνησης για πολλές λειτουργίες διαχειριστή:

  1. Κάντε κλικ στο κουμπί APP για να επικοινωνήσετε, να συμπληρώσετε έγγραφα, να συνομιλήσετε μέσω βίντεο (και πολλά άλλα) για ζητήματα που σχετίζονται με υποθέσεις.
  2. Κάντε κλικ στο κουμπί ADMIN για να ανοίξετε τον πίνακα διαχείρισης. Εδώ οι διαχειριστές διαχειρίζονται χρήστες, υποθέσεις, ενσωματώσεις και πολλά άλλα.
  3. Κάντε κλικ στην έλλειψη για να διαχειριστείτε τις ρυθμίσεις χρέωσης λογαριασμού και ειδοποιήσεων.
  • Σημείωση Το Case Active εμφανίζει το κουμπί ADMIN για χρήστες διαχειριστή. Οι χρήστες που δεν έχουν δικαιώματα διαχειριστή δεν βλέπουν το κουμπί ADMIN (όπως φαίνεται στην παρακάτω οθόνη).
