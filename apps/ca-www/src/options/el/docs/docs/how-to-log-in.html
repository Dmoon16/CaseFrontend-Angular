Πώς να συνδεθείτε

Μάθετε πώς να συνδεθείτε στον υπάρχοντα λογαριασμό σας Case Active.

Σύνδεση

  1. Μεταβείτε στη σελίδα σύνδεσης .
  2. Κάντε κλικ στο κουμπί LOGIN.
  3. Συμπληρώστε το email σας.
  4. Κάντε κλικ στο ΕΠΟΜΕΝΟ.

Επαλήθευση λογαριασμού

  1. Ανοίξτε μια νέα καρτέλα προγράμματος περιήγησης και συνδεθείτε στον λογαριασμό email σας.
  2. Αντιγράψτε τον εξαψήφιο κωδικό.
  3. Επιστρέψτε στην καρτέλα Case Active.
  4. Επικολλήστε τον κωδικό στα πεδία. Εμφανίζεται ένα νέο παράθυρο.

Το Case Active εμφανίζει τη σελίδα Εφαρμογές.

  • Σημείωση Η πλατφόρμα εμφανίζει ένα κουμπί ADMIN δίπλα στο κουμπί APP για χρήστες με δικαιώματα διαχειριστή.
