Μορφοποίηση δεδομένων αναθέσεων υποθέσεων χρήστη με το εργαλείο εισαγωγής

Παρακάτω είναι μια λίστα με πεδία δεδομένων σε αρχεία CSV. Κάθε επιλογή εισαγωγής έχει διαφορετικό συνδυασμό πεδίων.

case_ID

  • Εισαγάγετε το αναγνωριστικό χρήστη ή ετικέτας περίπτωσης.
  • Μεταβείτε στη σελίδα Υποθέσεις > Επεξεργασία > Αναγνωριστικό ετικέτας Ή Μεταβείτε στη σελίδα Υποθέσεις > Επεξεργασία > Αναγνωριστικό υπόθεσης στη διεύθυνση url.

ταυτότητα χρήστη

  • Εισαγάγετε το αναγνωριστικό χρήστη ή το αναγνωριστικό ετικέτας.
  • Μεταβείτε στη σελίδα Χρήστες > Επεξεργασία > Αναγνωριστικό ετικέτας Ή Μεταβείτε στη σελίδα Χρήστες > Επεξεργασία > Αναγνωριστικό χρήστη στη διεύθυνση url.

υπόθεση_ρόλος_παραγγελία

  • Αφήστε το κελί κενό.

Αναγνωριστικό_ρόλου_υπόθεσης

  • Εισαγάγετε: πελάτες, προσωπικό ή προεπιλογή.

περίπτωση_ειδοποίηση

  • 0 = όχι
  • 1 = Ναι, στείλτε στον χρήστη μια ειδοποίηση μέσω email.
