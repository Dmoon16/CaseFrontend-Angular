Πώς να δημιουργήσετε χρήστες με το εργαλείο εισαγωγής

Αυτό το άρθρο δείχνει στους διαχειριστές πώς να δημιουργούν χρήστες με το εργαλείο Εισαγωγή.

Άνοιγμα του εργαλείου Εισαγωγή

Οι διαχειριστές μπορούν να χρησιμοποιήσουν το εργαλείο Εισαγωγή για να ανεβάσουν μαζικά νέους χρήστες και να τροποποιήσουν τις υπάρχουσες εγγραφές χρηστών.

  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Στον πίνακα Διαχειριστής, κάντε κλικ στην καρτέλα Ενσωμάτωση και, στη συνέχεια, ΕΙΣΑΓΩΓΕΣ στο υπομενού.
  4. Κάντε κλικ στο κουμπί ΝΕΟ. Το παράθυρο διαλόγου Προσθήκη εισαγωγών εμφανίζεται σε μια νέα καρτέλα προγράμματος περιήγησης.

Δημιουργία εγγραφών

Αφού ανοίξει το παράθυρο διαλόγου Προσθήκη εισαγωγών:

  1. Κάντε κλικ στο πεδίο Τύπος και επιλέξτε μια επιλογή (δείτε παρακάτω). Εμφανίζεται ένας σύνδεσμος.
  2. Κάντε κλικ στο σύνδεσμο Λήψη δείγματος . Ένα αρχείο CSV λαμβάνει στη συσκευή σας.

Στη συνέχεια, ανοίξτε το αρχείο και προσθέστε εγγραφές. Κάντε κλικ εδώ για να μάθετε πώς να χρησιμοποιείτε δεδομένα μορφής στο αρχείο CSV.

Τύπος

Όταν κάνετε κλικ στο πεδίο Τύπος, εμφανίζεται ένα αναπτυσσόμενο μενού. Επιλέξτε μία από τις δύο επιλογές για μαζικές εισαγωγές.

:exclamation: ΣΗΜΕΙΩΣΗ Το αναπτυσσόμενο μενού έχει πολλές επιλογές, αλλά μόνο δύο σχετίζονται με το εργαλείο Εισαγωγή. Κάθε επιλογή έχει διαφορετικό δείγμα αρχείου CSV.

Προσθήκη χρηστών

  • Ορίστε αυτήν την επιλογή για μια απλή μαζική μεταφόρτωση εγγραφών χρηστών.
  • Κάντε κλικ στον σύνδεσμο Λήψη δείγματος για να ανοίξετε ένα αρχείο CSV με επικεφαλίδες στηλών για τον τύπο Εισαγωγής Προσθήκη χρηστών.

Τροποποίηση χρηστών

  • Ορίστε αυτήν την επιλογή για ενημέρωση δεδομένων για τρέχοντες χρήστες.
  • Κάντε κλικ στον σύνδεσμο Λήψη δείγματος για να ανοίξετε ένα αρχείο CSV με επικεφαλίδες στηλών για τον τύπο Τροποποίηση χρηστών Εισαγωγή.

Μεταφόρτωση του αρχείου

Μπορείτε να ανεβάσετε ένα αρχείο αμέσως ή να επιστρέψετε στο παράθυρο διαλόγου Προσθήκη εισαγωγών αργότερα και να ολοκληρώσετε αυτά τα βήματα.

  1. Ανοίξτε το παράθυρο διαλόγου Προσθήκη εισαγωγών.
  2. Εισαγάγετε ένα μοναδικό όνομα για το αρχείο CSV.
  3. Κάντε κλικ στο εικονίδιο συν (+) για να ανεβάσετε το αρχείο από τη συσκευή σας.
  4. Κάντε κλικ στο πεδίο Τύπος και επιλέξτε μια επιλογή. (βλέπε παραπάνω)
  5. Κάντε κλικ στην ΑΠΟΘΗΚΕΥΣΗ.

Επεξεργασία

Ανάλογα με τον αριθμό των εγγραφών στο αρχείο και τη φόρτωση του συστήματος, η πλατφόρμα μπορεί να διαρκέσει αρκετά λεπτά για να επεξεργαστεί την εισαγωγή.

Όταν ολοκληρωθεί η εισαγωγή, η πλατφόρμα προσαρμόζει τις εγγραφές χρηστών (δηλαδή προσθέτει, τροποποιεί ή απενεργοποιεί) στον πίνακα ελέγχου χρηστών του διαχειριστή.

Αποτυχία

Εάν η λειτουργία εισαγωγής αποτύχει, η πλατφόρμα επιστρέφει ένα μήνυμα Απέτυχε. Στις περισσότερες περιπτώσεις, η αιτία της αποτυχίας είναι η μορφοποίηση δεδομένων.

Για να δείτε την αιτία της αποτυχίας:

  1. Κάντε κλικ στην Προβολή κάτω από την επικεφαλίδα Ενέργεια.
  2. Στην επόμενη σελίδα, τοποθετήστε το ποντίκι στο εικονίδιο Αποτυχία. Η επεξήγηση εργαλείου εμφανίζει το σφάλμα.
  3. Μεταβείτε στο αρχείο CVS και προσαρμόστε και διορθώστε το σφάλμα.
  4. Ανεβάστε ξανά το αρχείο.
