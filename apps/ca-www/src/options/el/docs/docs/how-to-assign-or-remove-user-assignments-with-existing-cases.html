Πώς να εκχωρήσετε ή να αφαιρέσετε εργασίες χρηστών με υπάρχουσες θήκες

Αυτό το άρθρο δείχνει στους διαχειριστές πώς να εκχωρούν και να καταργούν χρήστες υποθέσεων. Σας δείχνουμε επίσης πώς να τροποποιήσετε τα δεδομένα χρήστη.

Άνοιγμα υπάρχουσας υπόθεσης

  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Κάντε κλικ στην καρτέλα μενού Υποθέσεων .
  4. Βεβαιωθείτε ότι το ACTIVE είναι επιλεγμένο στο υπομενού.
  5. Κάντε κύλιση προς τα κάτω στη στήλη ACTION.

Εκχώρηση ή αφαίρεση

Κάντε κλικ στην επιλογή Εκχώρηση χρηστών για να εκχωρήσετε έναν χρήστη στην υπόθεση ή να αφαιρέσετε τον χρήστη από τη θήκη (αλλά όχι την πλατφόρμα). Στην επόμενη σελίδα: Κάντε κλικ στο ADD και ολοκληρώστε τη ροή εργασίας. Κάντε κλικ στην κατάργηση για να διαγράψετε έναν χρήστη.

img

Τροποποιώ

Δεν είναι δυνατή η επεξεργασία ή η ενημέρωση των δεδομένων χρήστη στη σελίδα Υποθέσεων. Αντίθετα, ακολουθήστε αυτά τα βήματα:

  1. Κάντε κλικ στην καρτέλα μενού Χρήστες .
  2. Εντοπίστε το όνομα χρήστη και κάντε κλικ στην Επεξεργασία .
  3. Κάντε αλλαγές και κάντε κλικ στο ΑΠΟΘΗΚΕΥΣΗ.
