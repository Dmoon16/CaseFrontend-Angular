Πώς να δημιουργήσετε ή να τροποποιήσετε θήκες μέσω του μενού διεπαφής χρήστη

Αυτό το άρθρο δείχνει στους διαχειριστές πώς να δημιουργήσουν μια θήκη με τη διεπαφή χρήστη της πλατφόρμας. Οι διαχειριστές μπορούν επίσης να προσθέσουν χρήστες σε μια νέα υπόθεση και να ενημερώσουν τις υπάρχουσες περιπτώσεις.

Δημιουργία θήκης

  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Στον πίνακα "Διαχειριστής", κάντε κλικ στην καρτέλα μενού "Υποθέσεις".
  4. Κάντε κλικ στο κουμπί ΝΕΟ. Εμφανίζεται ένα νέο παράθυρο.

Συμπληρώστε τα απαιτούμενα πεδία και κάντε κλικ στο ΑΠΟΘΗΚΕΥΣΗ.

img

Προσθήκη χρηστών

Προαιρετικά, οι διαχειριστές μπορούν να προσθέσουν χρήστες στην υπόθεση σε αυτήν τη σελίδα. Κάντε κύλιση προς τα κάτω και κάντε κλικ στο κουμπί ASIGN USERS .

img

Στην επιλογή Εκχώρηση χρήστη, συμπληρώστε τα πεδία. Επιλέξτε το πλαίσιο για να στείλετε στον χρήστη μια ειδοποίηση μέσω email. (προαιρετικός)

Τροποποίηση υπόθεσης

Για να ενημερώσετε έναν μικρό αριθμό περιπτώσεων, χρησιμοποιήστε το εργαλείο Επεξεργασία και κάντε αλλαγές σε κάθε μεμονωμένη περίπτωση.

  • Σημείωση Για να ενημερώσετε έναν μεγάλο αριθμό περιπτώσεων σε μία συνεδρίαση, εξετάστε το ενδεχόμενο να χρησιμοποιήσετε το εργαλείο Εισαγωγή
  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Στον πίνακα "Διαχειριστής", κάντε κλικ στην καρτέλα μενού "Υποθέσεις".
  4. Εντοπίστε μια υπόθεση και κάντε κλικ στην Επεξεργασία.
  5. Κάντε αλλαγές και κάντε κλικ στο ΑΠΟΘΗΚΕΥΣΗ.
