Τρόπος εκχώρησης, τροποποίησης ή κατάργησης εργασιών χρηστών με το εργαλείο εισαγωγής

Αυτό το άρθρο δείχνει στους διαχειριστές πώς να διαχειρίζονται χρήστες υποθέσεων με το εργαλείο Εισαγωγή.

Οι διαχειριστές μπορούν:

  • εκχώρηση χρηστών σε μια υπόθεση
  • τροποποίηση χρηστών που έχουν ήδη ανατεθεί σε μια υπόθεση
  • καταργήστε χρήστες που έχουν ανατεθεί σε μια υπόθεση

Άνοιγμα του εργαλείου Εισαγωγή

  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Κάντε κλικ στην καρτέλα Ενσωμάτωση και, στη συνέχεια, ΕΙΣΑΓΩΓΕΣ στο υπομενού.
  4. Κάντε κλικ στο κουμπί ΝΕΟ. Εμφανίζεται το παράθυρο διαλόγου Προσθήκη εισαγωγών.

Δημιουργία εγγραφών

Κατεβάστε ένα ή περισσότερα αρχεία CSV. Χρησιμοποιήστε αυτά τα αρχεία για να προσθέσετε, να τροποποιήσετε ή να αφαιρέσετε χρήστες περιπτώσεων. Αφού ανοίξει το παράθυρο διαλόγου Προσθήκη εισαγωγών:

  1. Κάντε κλικ στο πεδίο Τύπος.
  2. Κάντε κύλιση στο αναπτυσσόμενο μενού και επιλέξτε μια επιλογή (δείτε παρακάτω). Εμφανίζεται ένας σύνδεσμος.
  3. Κάντε κλικ στο σύνδεσμο Λήψη δείγματος. Ένα αρχείο CSV λαμβάνει στη συσκευή σας.

Επιλογές αρχείου

Αντιστοίχιση χρηστών σε περιπτώσεις * Ορίστε αυτήν την επιλογή για να προσθέσετε χρήστες σε μια υπόθεση. * Αφήστε το κελί case_role_order κενό. * Μεταβείτε στη σελίδα Χρήστες και αντιγράψτε το αναγνωριστικό ετικέτας για αυτό το άτομο. Εάν το πεδίο Αναγνωριστικό ετικέτας είναι κενό, εισαγάγετε ένα μοναδικό αναγνωριστικό, κάντε κλικ στην επιλογή Αποθήκευση και, στη συνέχεια, αντιγράψτε το αναγνωριστικό στο αρχείο CSV.

Μεταφόρτωση του αρχείου

Μπορείτε να ανεβάσετε ένα αρχείο αμέσως ή να επιστρέψετε στο παράθυρο διαλόγου Προσθήκη εισαγωγών αργότερα και να ολοκληρώσετε αυτά τα βήματα.

  1. Ανοίξτε το παράθυρο διαλόγου Προσθήκη εισαγωγών.
  2. Εισαγάγετε ένα μοναδικό όνομα για το αρχείο CSV.
  3. Κάντε κλικ στο εικονίδιο συν (+) για να ανεβάσετε το αρχείο από τη συσκευή σας.
  4. Κάντε κλικ στο πεδίο Τύπος και επιλέξτε μια επιλογή. (βλέπε παραπάνω)
  5. Κάντε κλικ στην ΑΠΟΘΗΚΕΥΣΗ.

Επεξεργασία

Ανάλογα με τον αριθμό των εγγραφών στο αρχείο και τη φόρτωση του συστήματος, η πλατφόρμα μπορεί να διαρκέσει αρκετά λεπτά για να επεξεργαστεί την εισαγωγή.

Όταν ολοκληρωθεί η εισαγωγή, η πλατφόρμα προσαρμόζει τις εγγραφές χρήστη (δηλαδή προσθέτει, τροποποιεί ή αφαιρεί) στη σελίδα Υποθέσεις.

Αποτυχία

Εάν η λειτουργία εισαγωγής αποτύχει, η πλατφόρμα επιστρέφει ένα μήνυμα Απέτυχε. Στις περισσότερες περιπτώσεις, η αιτία της αποτυχίας είναι η μορφοποίηση δεδομένων.

Για να δείτε την αιτία της αποτυχίας:

  1. Κάντε κλικ στην Προβολή κάτω από την επικεφαλίδα Ενέργεια.
  2. Στην επόμενη σελίδα, τοποθετήστε το ποντίκι στο εικονίδιο Αποτυχία. Η επεξήγηση εργαλείου εμφανίζει το σφάλμα.
  3. Μεταβείτε στο αρχείο CVS και προσαρμόστε και διορθώστε το σφάλμα.
  4. Ανεβάστε ξανά το αρχείο.
