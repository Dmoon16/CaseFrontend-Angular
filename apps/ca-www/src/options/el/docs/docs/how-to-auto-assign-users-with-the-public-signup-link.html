Τρόπος αυτόματης ανάθεσης χρηστών με τον σύνδεσμο δημόσιας εγγραφής

Οι διαχειριστές μαθαίνουν πώς να διαμορφώνουν τις ρυθμίσεις συστήματος έτσι ώστε οι νέοι χρήστες να αντιστοιχίζονται αυτόματα σε μια θήκη που δημιουργήθηκε πρόσφατα αυτόματα.

Άνοιγμα της σελίδας εγκατάστασης

Το πρώτο βήμα είναι να ανοίξετε τη σελίδα Systems Setup.

  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Κάντε κλικ στην καρτέλα Systems και, στη συνέχεια, SYSTEM SETUP στο υπομενού.
  4. Κάντε κύλιση προς τα κάτω στην επικεφαλίδα ΕΓΓΡΑΦΗ ΧΡΗΣΤΗ.

Διαμόρφωση ρυθμίσεων συστήματος

  1. Ορίστε τα αναπτυσσόμενα μενού Allow Signup και Auto Create Case σε On.
  2. Κάντε κλικ στο μενού ρόλων και επιλέξτε μια επιλογή (συνιστάται η προεπιλογή).
  3. Αντιγράψτε και μοιραστείτε τον σύνδεσμο Δημόσια Εγγραφή χρήστη.

Αποστολή του συνδέσμου

Κάθε χρήστης που κάνει κλικ στον σύνδεσμο και ολοκληρώνει την εγγραφή ενεργοποιεί την πλατφόρμα για να δημιουργήσει μια νέα υπόθεση στη σελίδα Υποθέσεων και εκχωρεί αυτόματα τον χρήστη σε αυτήν την περίπτωση.
