Πώς να μορφοποιήσετε δεδομένα CSV

Ακολουθεί μια λίστα με πεδία δεδομένων σε αρχεία CSV. Κάθε επιλογή εισαγωγής έχει διαφορετικό συνδυασμό πεδίων.

όνομα χρήστη

  • Εισαγάγετε τη διεύθυνση email του χρήστη.

δεδομένο_όνομα

  • Εισαγάγετε το όνομα του χρήστη.

οικογενειακό_όνομα

  • Εισαγάγετε το επώνυμο του χρήστη.

μικρός λοβός

  • Αυτή η στήλη ελέγχει τη γλώσσα που προτιμά ο χρήστης.
  • Εισαγάγετε en για τα αγγλικά, sp για τα ισπανικά.

zoneinfo

  • Αυτό ορίζει τη ζώνη ώρας του χρήστη. Εισαγάγετε την ήπειρο και την πόλη χωρίς κενά (χρησιμοποιήστε υπογράμμιση).
  • Πρέπει να χρησιμοποιήσετε μια επιλογή στο αναπτυσσόμενο μενού Προφίλ > ΓΛΩΣΣΑ & ΠΕΡΙΟΧΗ > Ζώνη ώρας.
  • Χρησιμοποιήστε τη ζώνη ώρας που ταιριάζει καλύτερα. Για παράδειγμα, η Βοστώνη δεν βρίσκεται στο αναπτυσσόμενο μενού, επομένως μπορούμε να χρησιμοποιήσουμε τη Νέα Υόρκη (ή οποιαδήποτε πόλη στη ζώνη ώρας EST/EDT).

host_tag_id

  • Εισαγάγετε μια μόνιμη ετικέτα.
  • az, 0-9, -, / επιτρέπονται.

περίπτωση_κατάσταση

  • 0 = κλειστό
  • 1 = ενεργό
