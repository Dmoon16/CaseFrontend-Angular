Πώς να δημιουργήσετε ή να τροποποιήσετε θήκες με το εργαλείο εισαγωγής

Αυτό το άρθρο δείχνει στους διαχειριστές πώς να δημιουργούν ή να τροποποιούν υπάρχουσες περιπτώσεις με το εργαλείο Εισαγωγή.

Άνοιγμα του εργαλείου Εισαγωγή

  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Στον πίνακα Διαχειριστής, κάντε κλικ στην καρτέλα Ενσωμάτωση και, στη συνέχεια, ΕΙΣΑΓΩΓΕΣ στο υπομενού.
  4. Κάντε κλικ στο κουμπί ΝΕΟ. Το παράθυρο διαλόγου Προσθήκη εισαγωγών εμφανίζεται σε μια νέα καρτέλα προγράμματος περιήγησης.

img

Δημιουργία υποθέσεων

Αφού ανοίξει το παράθυρο διαλόγου Προσθήκη εισαγωγών:

  1. Κάντε κλικ στο πεδίο Τύπος και επιλέξτε μια επιλογή (δείτε παρακάτω). Εμφανίζεται ένας σύνδεσμος.
  2. Κάντε κλικ στο σύνδεσμο Λήψη δείγματος. Ένα αρχείο CSV λαμβάνει στη συσκευή σας.

Στη συνέχεια, ανοίξτε το αρχείο και προσθέστε θήκες.

Τύπος

Όταν κάνετε κλικ στο πεδίο Τύπος , εμφανίζεται ένα αναπτυσσόμενο μενού. Επιλέξτε μία από τις δύο επιλογές για τις μαζικές εισαγωγές θήκης.

  • Σημείωση Το αναπτυσσόμενο μενού έχει πολλές επιλογές, αλλά μόνο δύο σχετίζονται με τη διαχείριση υπόθεσης. Κάθε επιλογή έχει διαφορετικό δείγμα αρχείου CSV.

Προσθέστε θήκες

  • Ορίστε αυτήν την επιλογή για μια απλή μαζική μεταφόρτωση περιπτώσεων.
  • Κάντε κλικ στον σύνδεσμο Λήψη δείγματος για να ανοίξετε ένα αρχείο CSV με επικεφαλίδες στηλών για τον τύπο εισαγωγής δημιουργίας υποθέσεων.

Τροποποίηση περιπτώσεων

  • Ορίστε αυτήν την επιλογή για να ενημερώσετε τις τρέχουσες περιπτώσεις.
  • Κάντε κλικ στον σύνδεσμο Λήψη δείγματος για να ανοίξετε ένα αρχείο CSV με επικεφαλίδες στηλών για τον τύπο εισαγωγής Τροποποίηση υποθέσεων.

Μεταφόρτωση του αρχείου

Μπορείτε να ανεβάσετε ένα αρχείο αμέσως ή να επιστρέψετε στο παράθυρο διαλόγου Προσθήκη εισαγωγών αργότερα και να ολοκληρώσετε αυτά τα βήματα.

  1. Ανοίξτε το παράθυρο διαλόγου Προσθήκη εισαγωγών.
  2. Εισαγάγετε ένα μοναδικό όνομα για το αρχείο CSV.
  3. Κάντε κλικ στο εικονίδιο συν (+) για να ανεβάσετε το αρχείο από τη συσκευή σας.
  4. Κάντε κλικ στο πεδίο Τύπος και επιλέξτε μια επιλογή. (βλέπε παραπάνω)
  5. Κάντε κλικ στην ΑΠΟΘΗΚΕΥΣΗ.

Μορφοποίηση

Αυτά είναι τα απαιτούμενα πεδία και οι πρώτες απαιτήσεις. Κάντε κλικ εδώ για να ελέγξετε τις λεπτομέρειες μορφοποίησης.

tag_id (ή case_ID) ένα μοναδικό αναγνωριστικό (χρησιμοποιήστε αλφαριθμητικούς χαρακτήρες, οι παύλες γίνονται δεκτές)

case_status 0 = κλειστό 1 = ενεργό

ημερομηνία_ανοίγματος μήνας/ημέρα/έτος μμ/ηη/εεεε

σχετικά με μια μοναδική περιγραφή της υπόθεσης (χρησιμοποιήστε αλφαριθμητικούς χαρακτήρες, γίνονται δεκτές παύλες)

Επεξεργασία

Ανάλογα με τον αριθμό των εγγραφών στο αρχείο και τη φόρτωση του συστήματος, η πλατφόρμα μπορεί να διαρκέσει αρκετά λεπτά για να επεξεργαστεί την εισαγωγή.

img

Όταν ολοκληρωθεί η εισαγωγή, η πλατφόρμα προσαρμόζει τις εγγραφές υποθέσεων (δηλαδή προσθέτει ή τροποποιεί) στον πίνακα ελέγχου Υποθέσεων του διαχειριστή.

Αποτυχία

Εάν η λειτουργία εισαγωγής αποτύχει, η πλατφόρμα επιστρέφει ένα μήνυμα Απέτυχε. Στις περισσότερες περιπτώσεις, η αιτία της αποτυχίας είναι η μορφοποίηση δεδομένων.

Για να δείτε την αιτία της αποτυχίας:

  1. Κάντε κλικ στην Προβολή κάτω από την επικεφαλίδα ACTION.
  2. Στην επόμενη σελίδα, τοποθετήστε το ποντίκι στο εικονίδιο Αποτυχία. Η επεξήγηση εργαλείου εμφανίζει το σφάλμα.
  3. Μεταβείτε στο αρχείο CVS και προσαρμόστε και διορθώστε το σφάλμα.
  4. Ανεβάστε ξανά το αρχείο.
