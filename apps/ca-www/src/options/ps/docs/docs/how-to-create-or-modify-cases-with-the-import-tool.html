د وارداتو وسیلې سره د قضیې رامینځته کولو یا بدلولو څرنګوالی

دا مقاله مدیرانو ته ښیې چې څنګه د وارداتو وسیلې سره موجوده قضیې رامینځته یا بدل کړي.

د وارداتو وسیله پرانیستل

  1. د اډمین حساب ته ننوځئ.
  2. د ایپس پاڼې کې یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. په اډمین پینل کې، د ادغام ټب کلیک وکړئ او بیا په فرعي مینو کې واردات.
  4. په نوي تڼۍ کلیک وکړئ. د وارداتو اضافه کولو ډیالوګ بکس په نوي براوزر ټب کې څرګندیږي.

img

د قضیو جوړول

وروسته له دې چې د وارداتو اضافه کولو ډیالوګ بکس خلاصیږي:

  1. د ډول په ساحه کې کلیک وکړئ او یو اختیار غوره کړئ (لاندې وګورئ). یو لینک ښکاره کیږي.
  2. د ډاونلوډ نمونې لینک باندې کلیک وکړئ. ستاسو وسیله ته د CSV فایل ډاونلوډ کیږي.

بیا، فایل خلاص کړئ او قضیې اضافه کړئ.

ډول

کله چې تاسو د ډول په ساحه کې کلیک وکړئ، یو ډراپ-ډاون مینو ښکاره کیږي. د لویو قضیو وارداتو لپاره له دوو انتخابونو څخه یو غوره کړئ.

  • یادونه د ډراپ ډاون مینو ډیری اختیارونه لري، مګر یوازې دوه یې د قضیې اداره کولو پورې تړاو لري. هر اختیار د مختلف نمونې CSV فایل لري.

قضیې اضافه کړئ

  • د قضیو د ساده لوی اپلوډ لپاره دا اختیار غوره کړئ.
  • د نمونې ډاونلوډ کولو لینک باندې کلیک وکړئ ترڅو د CSV فایل خلاصولو لپاره د کالم سرلیکونو سره د قضیې وارداتو ډول رامینځته کړئ.

قضیې تعدیل کړئ

  • د اوسني قضیې تازه کولو لپاره دا اختیار غوره کړئ.
  • د نمونې ډاونلوډ کولو لینک باندې کلیک وکړئ ترڅو د CSV فایل خلاصولو لپاره د کالم سرلیکونو سره د قضیې وارداتو ډول بدل کړئ.

د فایل پورته کول

تاسو کولی شئ سمدلاسه فایل اپلوډ کړئ یا په وروستي نیټه کې د وارداتو اضافه کولو ډیالوګ بکس ته راستون شئ او دا مرحلې بشپړ کړئ.

  1. د وارداتو اضافه کولو ډیالوګ بکس خلاص کړئ.
  2. د CSV فایل لپاره یو ځانګړی نوم دننه کړئ.
  3. د خپل وسیله څخه فایل اپلوډ کولو لپاره په پلس (+) آیکون کلیک وکړئ.
  4. د ډول په ساحه کې کلیک وکړئ او یو اختیار غوره کړئ. (پورته وګورئ)
  5. خوندي کړئ کلیک وکړئ.

شکل ورکول

دا اړین ساحې او لومړۍ اړتیاوې دي. د فارمیټینګ توضیحاتو بیاکتنې لپاره دلته کلیک وکړئ.

tag_id (یا case_ID) یو ځانګړی پیژندونکی (د الفانومیریک حروف وکاروئ، هایفنز منل شوي)

case_status 0 = تړل شوی 1 = فعال

نیټه_پرانستل شوې میاشت/ورځ/کال mm/dd/yyyy

د قضیې د ځانګړي توضیحاتو په اړه (د الفانومیریک حروف وکاروئ، هایفینونه منل شوي)

img

پروسس کول

په فایل او سیسټم بار کې د ریکارډونو شمیر پورې اړه لري ، پلیټ فارم کولی شي د وارداتو پروسس کولو لپاره څو دقیقې وخت ونیسي.

img

کله چې واردات بشپړ شي، پلیټ فارم د مدیر قضیې ډشبورډ کې د قضیې ریکارډونه (لکه اضافه یا تعدیل) تنظیموي.

ناکامي

که د وارداتو عملیات ناکام شي، پلیټ فارم یو ناکام پیغام بیرته راولي. په ډیری مواردو کې، د ناکامۍ لامل د معلوماتو بڼه کول دي.

img

د ناکامۍ علت معلومولو لپاره:

  1. د عمل سرلیک لاندې لید باندې کلیک وکړئ.
  2. په بله پاڼه کې، موږک په ناکامه نښه کې واچوئ. وسیلې ټایپ خطا ښیې.
  3. د CVS فایل ته لاړ شئ او غلطي تنظیم او حل کړئ.
  4. فایل بیا پورته کړئ.
