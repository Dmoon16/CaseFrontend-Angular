د UI مینو له لارې د قضیې رامینځته کولو یا بدلولو څرنګوالی

دا مقاله مدیرانو ته ښیې چې څنګه د پلیټ فارم کارونکي انٹرفیس سره قضیه رامینځته کړي. اډمین کولی شي کاروونکي په نوي قضیه کې اضافه کړي او موجوده قضیې تازه کړي.

د قضیې جوړول

  1. د اډمین حساب ته ننوتل.
  2. د ایپس پاڼې کې یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. په اډمین پینل کې، د قضیې مینو ټب باندې کلیک وکړئ.
  4. په نوي تڼۍ کلیک وکړئ. یوه نوې کړکۍ به ښکاره شي.

د کاروونکو اضافه کول

په اختیاري توګه، مدیران کولی شي کاروونکي په دې پاڼه کې قضیه کې اضافه کړي. لاندې سکرول کړئ او د ASSIN USERS تڼۍ کلیک وکړئ.

د کارونکي موډل په ټاکلو کې، ساحې ډک کړئ. کارونکي ته د بریښنالیک خبرتیا لیږلو لپاره بکس چیک کړئ. (اختیاری)

د قضیې تعدیل

د لږ شمیر قضیو تازه کولو لپاره ، د ترمیم وسیله وکاروئ او په هر انفرادي قضیه کې بدلونونه رامینځته کړئ.

  • یادونه په یوه ناسته کې د ډیری قضیو تازه کولو لپاره ، د وارداتو وسیلې کارولو ته پام وکړئ
  1. د اډمین حساب ته ننوتل.
  2. د ایپس پاڼې کې یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. په اډمین پینل کې، د قضیې مینو ټب باندې کلیک وکړئ.
  4. یوه قضیه ومومئ او په ایډیټ کلیک وکړئ.
  5. بدلونونه وکړئ او خوندي کړئ کلیک وکړئ.
