د واردولو وسیلې سره د کارونکي قضیې دندې ډیټا فارمیټ کول

لاندې د CSV فایلونو کې د معلوماتو ساحو لیست دی. د وارداتو هر اختیار د ساحو مختلف ترکیب لري.

قضیه_ID

  • د قضیې کارن یا ټاګ ID دننه کړئ.
  • د قضیو پاڼې ته لاړ شئ> ایډیټ> د ټاګ ID یا د قضیې پاڼې ته لاړ شئ> ترمیم> په url کې د قضیې ID.

کارن نوم

  • د کارونکي ID یا Tag ID دننه کړئ.
  • د کاروونکو پاڼې ته لاړ شئ> ایډیټ> د ټاګ ID یا د کاروونکو پاڼې ته لاړ شئ> ترمیم> په url کې د کارن ID.
