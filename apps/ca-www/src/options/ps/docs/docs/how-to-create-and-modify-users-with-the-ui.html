د UI سره د کاروونکو رامینځته کولو او بدلولو څرنګوالی

دا مقاله مدیرانو ته ښیې چې څنګه کارونکي رامینځته کړي او د پلیټ فارم کارونکي انٹرفیس سره ریکارډونه تازه کړي. اډمین کولی شي کاروونکي په نوي قضیه کې اضافه کړي او موجوده قضیې ترمیم کړي.

د کاروونکي جوړول

  1. د اډمین حساب ته ننوځئ.
  2. د ایپس پاڼې کې یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. په اډمین پینل کې، د کاروونکو مینو ټب باندې کلیک وکړئ.
  4. په نوي تڼۍ کلیک وکړئ. یوه نوې کړکۍ به ښکاره شي.

د کاروونکي بدلول

د لږ شمیر کاروونکو ریکارډونو تازه کولو لپاره، د ترمیم وسیله وکاروئ او هر یو کارونکي ته بدلون ورکړئ.

  • یادونه په یوه ناسته کې د ډیری کاروونکو ریکارډونو بدلولو لپاره، د وارداتو وسیلې کارولو ته پام وکړئ.
  1. په اډمین پینل کې، د کاروونکو مینو ټب باندې کلیک وکړئ.
  2. یو کارن ومومئ او په ایډیټ کلیک وکړئ.
  3. بدلونونه وکړئ او خوندي کړئ کلیک وکړئ.

img

راتلونکی ګامونه

فعاله قضیه کارونکي ته د بریښنالیک خبرتیا نه لیږي. اډمین باید د کارونکي سره اړیکه ونیسي او دوی ته د Case Active کور پاڼې ته یو لینک واستوي. کارن بیا د LOGIN تڼۍ کلیک کوي او د نوي لاسلیکونو لپاره کاري جریان بشپړوي.

  • دا مقاله د نوي کاروونکو لپاره د لاسلیک کاري فلو تشریح کوي.
