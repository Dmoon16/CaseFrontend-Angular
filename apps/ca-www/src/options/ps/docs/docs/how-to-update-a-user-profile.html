د یو کارن پروفایل تازه کولو څرنګوالی

دا مقاله کاروونکو ته ښیې چې څنګه د دوی پروفایل پاڼې کې معلومات تازه کړي.

  • کاروونکي کولی شي خپل پروفایل تازه کړي.
  • اډمین کولی شي د هر کارونکي پروفایل پاڼه تازه کړي.

د کاروونکو لپاره

لومړی ګام د پروفایل پاڼې خلاصول دي. کاروونکي او اډمین د خپل پروفایل پاڼې تازه کولو لپاره دا مرحلې تعقیبوي.

  1. خپل د قضیې فعال حساب ته ننوځئ.
  2. د پاڼې په سر کې د پروفایل ټب باندې کلیک وکړئ.
  3. بدلونونه وکړئ.
  4. په خوندي کولو کلیک وکړئ.

ډیری ساحې اختیاري دي، او ځینې ساحې اړین دي. اړین ساحې دي:

  • لومړی نوم
  • تخلص
  • بریښنا‌لیک
  • ژبه
  • د وخت زون

د مدیرانو لپاره

اډمین کولی شي د نورو کاروونکو لپاره د پروفایل معلومات هم تازه کړي. پدې کې د HOST USER INFO سرلیک لاندې درې ساحې شاملې دي.

  1. خپل د قضیې فعال مدیر حساب ته ننوځئ.
  2. یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. په پورتنۍ مینو کې د کاروونکو کلیک وکړئ.
  4. فرعي مینو ته لاړ شئ او د کارونکي لیست غوره کړئ چې تاسو یې لاسرسی غواړئ (فعال یا غیر فعال شوی). ډیفالټ ترتیب فعال دی.
  5. په لیست کې یو کارن ومومئ او په ایډیټ کلیک وکړئ.
  6. د اړتیا سره سم بدلونونه وکړئ. (لاندې وګورئ)
  7. په خوندي کولو کلیک وکړئ.

د کارونکي معلومات بدل کړئ

اډمین کولی شي د کارونکي معلومات بدل کړي.

  1. د کارونکي ډول تنظیم کړئ (کارونکي یا مدیر).
  2. د کارونکي حالت تنظیم کړئ (فعال غیر فعال).
  3. یو ځانګړی ټاګ ID دننه کړئ. (اختیاری)
  4. په خوندي کولو کلیک وکړئ.
