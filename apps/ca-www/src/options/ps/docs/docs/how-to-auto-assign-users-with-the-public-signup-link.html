څنګه د عامه لاسلیک لینک سره کارونکي په اتومات ډول وټاکئ

مدیران زده کوي چې څنګه د سیسټم تنظیمات تنظیم کړي ترڅو نوي کاروونکي په اوتومات ډول نوي رامینځته شوي قضیې ته وګمارل شي.

د تنظیم کولو پاڼه پرانیستل

لومړی ګام دا دی چې د سیسټمونو تنظیم پاڼه پرانیزي.

  1. د اډمین حساب ته ننوتل.
  2. د ایپس پاڼې کې یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. په فرعي مینو کې د سیسټمونو ټب او بیا د سیسټم تنظیم کلیک وکړئ.
  4. د USER SIGNUP سرلیک ته لاندې سکرول کړئ.

د سیسټم ترتیبات تنظیم کول

  1. د لاسلیک کولو اجازه ورکړئ او په اتوماتيک ډول د قضیې ډراپ-ډاون مینو فعال کړئ.
  2. د رول مینو کې کلیک وکړئ او یو اختیار غوره کړئ (ډیفالټ وړاندیز شوی).
  3. د عامه کارونکي لاسلیک لینک کاپي او شریک کړئ.
