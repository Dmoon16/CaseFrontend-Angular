د واردولو وسیلې سره د کارونکي دندې ګمارلو ، بدلولو یا لرې کولو څرنګوالی

دا مقاله مدیران ښیې چې څنګه د وارداتو وسیلې سره د قضیې کاروونکي اداره کړي.

اډمین کولی شي:

  • کاروونکي یوې قضیې ته وټاکئ
  • کاروونکي تعدیل کړئ چې دمخه یې قضیې ته ګمارل شوي
  • د قضیې لپاره ګمارل شوي کاروونکي لرې کړئ

د وارداتو وسیله پرانیستل

  1. د اډمین حساب ته ننوځئ.
  2. د ایپس پاڼې کې یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. د ادغام ټب باندې کلیک وکړئ او بیا په فرعي مینو کې واردات.
  4. په نوي تڼۍ کلیک وکړئ. د وارداتو اضافه کولو ډیالوګ بکس څرګندیږي.

د ریکارډونو جوړول

یو یا څو CSV فایلونه ډاونلوډ کړئ. دا فایلونه د قضیې کاروونکو اضافه کولو، بدلولو یا لرې کولو لپاره وکاروئ. وروسته له دې چې د وارداتو اضافه کولو ډیالوګ بکس خلاصیږي:

  1. د ډول په ساحه کې کلیک وکړئ.
  2. د ډراپ-ډاون مینو له لارې سکرول وکړئ او یو اختیار غوره کړئ (لاندې وګورئ). یو لینک ښکاره کیږي.
  3. د ډاونلوډ نمونې لینک باندې کلیک وکړئ. ستاسو وسیله ته د CSV فایل ډاونلوډ کیږي.

img

فایل خلاص کړئ او ریکارډونه اضافه کړئ. د CSV فایل کې د فارمیټ ډیټا کارولو څرنګوالي زده کولو لپاره دلته کلیک وکړئ.

د فایل انتخابونه

قضیو ته کاروونکي وټاکئ * په قضیه کې د کاروونکو اضافه کولو لپاره دا اختیار غوره کړئ. * د قضیې_رول_آرډر حجره خالي پریږدئ. * د کاروونکو پاڼې ته لاړ شئ او د دې کس لپاره د ټاګ ID کاپي کړئ. که د ټاګ ID ساحه خالي وي، یو ځانګړی ID داخل کړئ، خوندي کړئ کلیک وکړئ، او بیا ID CSV فایل ته کاپي کړئ.

img

قضیو ته ګمارل شوي کارونکي بدل کړئ * دا اختیار غوره کړئ ترڅو د اوسني کاروونکو لپاره ډیټا تازه کړئ چې قضیې ته ګمارل شوي. * د قضیې_ID کالم لپاره د قضیې ټاګ ID وکاروئ.

img

قضیو ته ګمارل شوي کارونکي لرې کړئ * د قضیې لپاره ټاکل شوي کارونکي لرې کولو لپاره دا اختیار غوره کړئ.

img

د فایل پورته کول

تاسو کولی شئ سمدلاسه فایل اپلوډ کړئ یا په وروستي نیټه کې د وارداتو اضافه کولو ډیالوګ بکس ته راستون شئ او دا مرحلې بشپړ کړئ.

  1. د وارداتو اضافه کولو ډیالوګ بکس خلاص کړئ.
  2. د CSV فایل لپاره یو ځانګړی نوم دننه کړئ.
  3. د خپل وسیله څخه فایل اپلوډ کولو لپاره په پلس (+) آیکون کلیک وکړئ.
  4. د ډول په ساحه کې کلیک وکړئ او یو اختیار غوره کړئ. (پورته وګورئ)
  5. خوندي کړئ کلیک وکړئ.

پروسس کول

په فایل او سیسټم بار کې د ریکارډونو شمیر پورې اړه لري ، پلیټ فارم کولی شي د وارداتو پروسس کولو لپاره څو دقیقې وخت ونیسي.

img

کله چې واردات بشپړ شي، پلیټ فارم د قضیې په پاڼه کې د کاروونکي ریکارډونه تنظیموي (د بیلګې په توګه اضافه کول، تعدیل کول، یا لرې کول).

ناکامي

که د وارداتو عملیات ناکام شي، پلیټ فارم یو ناکام پیغام بیرته راولي. په ډیری مواردو کې، د ناکامۍ لامل د معلوماتو بڼه کول دي.

img

د ناکامۍ علت معلومولو لپاره:

  1. د عمل سرلیک لاندې لید کلیک وکړئ.
  2. په بله پاڼه کې، موږک په ناکامه نښه کې واچوئ. وسیلې ټایپ خطا ښیې.
  3. د CVS فایل ته لاړ شئ او غلطي تنظیم او حل کړئ.
  4. فایل بیا پورته کړئ.
