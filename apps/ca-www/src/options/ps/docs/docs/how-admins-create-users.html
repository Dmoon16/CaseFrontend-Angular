څنګه اډمین کاروونکي جوړوي

اډمینونه رامینځته کوي، بدلوي یا غیر فعالوي څوک چې ایپسونو او قضیو ته لاسرسی ولري. کله چې یو کارن د لومړي ځل لپاره پلیټ فارم ته ننوځي، دوی اړتیا لري د پروفایل کاري فلو بشپړ کړي.

  • د نوي کارونکي پروفایل کاري فلو په اړه د نورو معلوماتو لپاره دلته کلیک وکړئ.

د کاروونکي جوړول

لاندې جدول د کاروونکي رامینځته کولو پنځه لارې ښیې. د څرنګوالي زده کولو لپاره لینک کلیک وکړئ.

خبرتیاوې

پلیټ فارم کاروونکو ته خبرتیا نه لیږي کله چې دوی په اپلیکیشن کې اضافه شي. په هرصورت، مدیران کولی شي د قضیې دندې لپاره خبرتیاوې فعالې کړي.
