Formatar dades de gestió d'usuaris amb l'eina d'importació

A continuació es mostra una llista de camps de dades en fitxers CSV. Cada opció d'importació té una combinació diferent de camps.

nom d'usuari

  • Introduïu l'adreça de correu electrònic de l'usuari.

nom_prestat

  • Introduïu el nom de l'usuari.

nom de familia

  • Introduïu el cognom de l'usuari.

local

  • Aquesta columna comprova l'idioma preferit de l'usuari.
  • Introduïu en per a l'anglès, sp per a l'espanyol.

info de zona

  • Això estableix la zona horària de l'usuari. Introduïu el continent i la ciutat sense espais (utilitza un guió baix).
  • Heu d'utilitzar una opció al menú desplegable Perfil > IDIOMA I REGIÓ > Zona horària.
  • Utilitzeu la zona horària més adequada. Per exemple, Boston no és al menú desplegable, de manera que podem utilitzar Nova York (o qualsevol ciutat de la zona horària EST/EDT).

host_tag_id

  • Introduïu una etiqueta permanent.
  • az, 0-9, -, / estan permesos.

cas_estat

  • 0 = tancat
  • 1 = actiu
