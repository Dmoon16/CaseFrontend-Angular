Com els administradors gestionen les assignacions d'usuari als casos

Hi ha cinc maneres de gestionar els usuaris i les assignacions de casos. Feu clic a un enllaç per saber com fer-ho.

Mètode Tipus Accions Detalls
Mentre es crea un cas IU afegir Botó Administrador > Casos > Botó Nou > Botó Assigna usuaris
Assigna automàticament l'usuari mitjançant l'enllaç de registre públic Acció de l'usuari afegir Botó Administrador > Configuració > Pàgina de configuració del sistema
S'utilitza la interfície d'usuari en el cas existent IU afegir i eliminar Botó Administrador > Casos > Assigna usuaris
Eina d'importació IU afegir, modificar i eliminar Botó Administrador > Integracions > Importacions
Ús de l'API (properament) API afegir, modificar i eliminar N/A
