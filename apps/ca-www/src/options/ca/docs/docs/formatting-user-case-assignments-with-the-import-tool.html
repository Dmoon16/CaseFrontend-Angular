Format de dades d'assignació de casos d'usuari amb l'eina d'importació

A continuació es mostra una llista de camps de dades en fitxers CSV. Cada opció d'importació té una combinació diferent de camps.

ID_cas

  • Introduïu l'identificador d'usuari o etiqueta del cas.
  • Aneu a la pàgina Casos > Edita > Identificador de l'etiqueta O Vés a la pàgina Casos > Edita > Identificador de cas a l'URL.

ID d'usuari

  • Introduïu l'identificador de l'usuari o l'identificador de l'etiqueta.
  • Aneu a la pàgina Usuaris > Edita > ID de l'etiqueta O Vés a la pàgina Usuaris > Edita > ID d'usuari a l'url.

cas_ordre_rol

  • Deixa la cel·la en blanc.

ID_funció_cas

  • Introduïu: clients, personal o predeterminat.

cas_notificar

  • 0 = no
  • 1 = Sí, envieu a l'usuari una notificació per correu electrònic.
