Com crear una aplicació

Aquest article mostra als usuaris nous com crear un compte de prova gratuït. Case Active concedeix automàticament al nou usuari privilegis d'administrador per a aquest compte.

Creació d'un compte

  1. Aneu a la pàgina d'inici de sessió.
  2. Feu clic al botó Comença.
  3. Introduïu una adreça de correu electrònic.
  4. Feu clic a Següent.

img

Verificació d'un compte

  1. Obriu una nova pestanya del navegador i inicieu sessió al vostre compte de correu electrònic.
  2. Copieu el codi de sis dígits.
  3. Torna a la pestanya Cas actiu.
  4. Enganxeu el codi als camps. Es mostra una finestra nova.

A la nova finestra:

  1. Ompliu els camps obligatoris.
  2. Llegeix les condicions del servei i marca la casella per acceptar.
  3. Feu clic a CREAR UN COMPTE GRATIS.

img

S'està completant la configuració de l'aplicació

A la pàgina següent es mostra un flux de treball. Completa els passos per acabar de configurar l'aplicació. Sempre podeu tornar a aquestes pàgines per editar o actualitzar la informació o l'estat del vostre compte.

A la pàgina Quant a l'aplicació, ompliu el nom de l'empresa i els camps del lloc web (obligatoris) i feu clic a Següent.

img

A la pàgina Informació de facturació, feu clic al camp Pla de pagament i reviseu les opcions. La configuració predeterminada és Gratuït. Feu clic a ENVIAR.

img

Després d'enviar el formulari, Case Active mostra la pàgina d'aplicacions. Per a cada aplicació que creeu o a la qual pertanyeu, la pàgina mostra:

  1. El nom de l'aplicació
  2. El vostre tipus (rol d'usuari)
  3. Estat

img

La pàgina d'aplicacions és un punt de llançament per a moltes funcions d'administració:

  1. Feu clic al botó APP per comunicar-vos, omplir documents, xat de vídeo (i més) sobre problemes relacionats amb casos.
  2. Feu clic al botó ADMINISTRADOR per obrir el tauler d'administració. Aquí és on els administradors gestionen usuaris, casos, integracions i molt més.
  3. Feu clic als punts suspensius per gestionar la facturació del compte i la configuració de notificacions.
  • Nota Case Active mostra el botó ADMIN per als usuaris administradors. Els usuaris que no tenen privilegis d'administrador no veuen el botó ADMINISTRADOR (com es mostra a la pantalla següent).
