درآمدي اوزار سان ڪيس ڪيئن ٺاھيو يا تبديل ڪجي

ھي آرٽيڪل ڏيکاري ٿو منتظمين کي ڪيئن ٺاھيو يا تبديل ڪجي موجوده ڪيسن کي درآمد واري اوزار سان.

درآمد وارو اوزار کولڻ

  1. منتظم کاتي ۾ لاگ ان ٿيو.
  2. ايپس پيج تي ائپ ڳولھيو ۽ ايڊمن بٽڻ تي ڪلڪ ڪريو.
  3. ايڊمن پينل تي، انٽيگريشن ٽيب تي ڪلڪ ڪريو ۽ پوءِ سب مينيو تي امپورٽس.
  4. NEW بٽڻ تي ڪلڪ ڪريو. شامل ڪريو درآمدات ڊائلاگ باڪس نئين برائوزر ٽيب ۾ ڏيکاري ٿو.

img

ڪيس ٺاهڻ

ان کان پوءِ شامل ڪريو درآمدات ڊائلاگ باڪس کلي ٿو:

  1. ٽائپ فيلڊ تي ڪلڪ ڪريو ۽ اختيار چونڊيو (هيٺ ڏسو). هڪ لنڪ ڏيکاري ٿو.
  2. ڊائون لوڊ نموني لنڪ تي ڪلڪ ڪريو. هڪ CSV فائل توهان جي ڊوائس تي ڊائون لوڊ.

اڳيون، فائل کوليو ۽ ڪيس شامل ڪريو.

قسم

جڏهن توهان ٽائپ فيلڊ تي ڪلڪ ڪريو، هڪ ڊراپ-ڊائون مينيو ڏيکاري ٿو. بلڪ ڪيس جي درآمد لاءِ ٻن اختيارن مان ھڪڙو چونڊيو.

  • نوٽ ڊراپ-ڊائون مينيو ۾ ڪيترائي آپشن آھن، پر صرف ٻه آھن ڪيس انتظام ڪرڻ سان. هر اختيار ۾ هڪ مختلف نموني CSV فائل آهي.

ڪيس شامل ڪريو

  • هي اختيار چونڊيو هڪ سادي بلڪ ڪيسن جي اپلوڊ لاءِ.
  • ڪلڪ ڪريو ڊائون لوڊ نمونو لنڪ کولڻ لاءِ CSV فائل کولڻ لاءِ ڪالمن جي عنوانن سان ٺاهيو ڪيس درآمد جي قسم لاءِ.

ڪيسن کي تبديل ڪريو

  • موجوده ڪيسن کي اپڊيٽ ڪرڻ لاء هي اختيار چونڊيو.
  • ڪلڪ ڪريو ڊائون لوڊ نمونو لنڪ کولڻ لاءِ CSV فائل کولڻ لاءِ ڪالمن جي عنوانن سان تبديل ٿيل ڪيسن جي درآمد جي قسم لاءِ.

فائل اپ لوڊ ڪندي

توھان فوري طور تي ھڪڙي فائل اپلوڊ ڪري سگھو ٿا يا واپسي ڪري سگھوٿا شامل ڪريو درآمدات ڊائلاگ باڪس بعد جي تاريخ ۾ ۽ انھن مرحلن کي مڪمل ڪريو.

  1. درآمدات شامل ڪريو ڊائلاگ باڪس کوليو.
  2. CSV فائل لاءِ منفرد نالو داخل ڪريو.
  3. پنهنجي ڊوائيس تان فائل اپلوڊ ڪرڻ لاءِ پلس (+) آئڪن تي ڪلڪ ڪريو.
  4. ٽائپ فيلڊ تي ڪلڪ ڪريو ۽ اختيار چونڊيو. (مٿي ڏسو)
  5. ڪلڪ ڪريو محفوظ ڪريو.

فارميٽ ڪرڻ

اهي گهربل فيلڊ ۽ پهرين گهربل آهن. فارميٽنگ جي تفصيلن جو جائزو وٺڻ لاءِ هتي ڪلڪ ڪريو.

tag_id (يا case_ID) هڪ منفرد سڃاڻپ ڪندڙ (الفانيمري اکر استعمال ڪريو، هائفن قبول ڪيا ويا آهن)

case_status 0 = بند ٿيل 1 = فعال

تاريخ_کليل مهينو/ڏينهن/سال mm/dd/yyyy

ڪيس جي هڪ منفرد وضاحت بابت (الفانيمري اکر استعمال ڪريو، هائفن قبول ڪيا ويا آهن)

پروسيسنگ

فائل ۽ سسٽم جي لوڊ ۾ رڪارڊ جي تعداد تي مدار رکندي، پليٽ فارم درآمد کي عمل ڪرڻ لاء ڪيترائي منٽ وٺي سگھي ٿو.

img

جڏهن درآمد مڪمل ٿئي ٿي، پليٽ فارم منتظم جي ڪيس ڊيش بورڊ تي ڪيس رڪارڊز (يعني شامل ڪري ٿو يا تبديل ڪري ٿو) ترتيب ڏئي ٿو.

ناڪامي

جيڪڏهن درآمد آپريشن ناڪام ٿئي ٿي، پليٽ فارم هڪ ناڪام پيغام موٽائي ٿو. اڪثر ڪيسن ۾، ناڪامي جو سبب ڊيٽا فارميٽنگ آهي.

img

ناڪامي جو سبب ڏسڻ لاء:

  1. ACTION عنوان هيٺ ڏسو تي ڪلڪ ڪريو.
  2. ايندڙ صفحي تي، مائوس کي هور ڪريو ناڪامي آئڪن تي. ٽول ٽائپ غلطي ڏيکاري ٿو.
  3. CVS فائل ڏانھن وڃو ۽ ترتيب ڏيو ۽ غلطي کي درست ڪريو.
  4. فائل کي ٻيهر لوڊ ڪريو.
