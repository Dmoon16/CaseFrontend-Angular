هڪ ڪيس ٺاهڻ دوران صارفين کي ڪيئن تفويض ڪجي

هي آرٽيڪل منتظمين کي ڏيکاري ٿو ته هڪ نئون ڪيس ٺاهڻ دوران صارف کي ڪيئن تفويض ڪجي. جيڪڏهن صارف پليٽ فارم تي اڳ ۾ ئي رجسٽر ٿيل نه آهي، توهان ٺاهي سگهو ٿا هڪ نئون صارف شامل ڪريو جڏهن هڪ نئون ڪيس ٺاهي.

ڪيس ٺاهڻ

  1. منتظم کاتي ۾ لاگ ان ٿيو.
  2. ايپس پيج تي ائپ ڳولھيو ۽ ايڊمن بٽڻ تي ڪلڪ ڪريو.
  3. ڪلڪ ڪريو ڪيس مينيو ٽيب.
  4. NEW بٽڻ تي ڪلڪ ڪريو. هڪ نئين ونڊو ڏيکاري ٿي.

استعمال ڪندڙ کي مقرر ڪرڻ

موجوده استعمال ڪندڙ

موجوده صارف کي نئين صورت ۾ تفويض ڪرڻ لاء انهن قدمن تي عمل ڪريو.

ASSIN USERS بٽڻ تي ڪلڪ ڪريو ۽ فيلڊ ۾ ڀريو.

  1. يوزر ڊراپ ڊائون تي ڪلڪ ڪريو ۽ سڃاڻپ ڪندڙ چونڊيو (مثال طور اي ميل).
  2. خالي خاني ۾ قدر داخل ڪريو (مثال طور اي ميل پتو).
  3. جيڪڏهن صارف جو رڪارڊ پليٽ فارم ۾ موجود آهي، فيلڊ خودڪار طور تي نالو تجويز ڪري ٿو.
  4. صحيح استعمال ڪندڙ چونڊيو.
  5. صارف جو ڪردار چونڊيو (ڪلائنٽ، ڊفالٽ، يا اسٽاف).
  6. اختياري طور، باڪس چيڪ ڪريو صارف کي اي ميل نوٽيفڪيشن موڪلڻ لاءِ.
  7. ڪلڪ ڪريو محفوظ ڪريو.

جيڪڏهن پليٽ فارم موجوده رڪارڊ نه ڳولي سگهي، بٽڻ تي ڪلڪ ڪريو نئون استعمال ڪندڙ شامل ڪرڻ لاء.

فيلڊ ۾ ڀريو ۽ پوءِ ڪلڪ ڪريو محفوظ ڪريو صارف کي پليٽ فارم تي شامل ڪرڻ لاءِ. اڳيون، صارف جو ڪردار مقرر ڪريو ۽ ڪلڪ ڪريو محفوظ ڪريو ٻيهر صارف کي ڪيس ۾ تفويض ڪرڻ لاءِ. آخرڪار، ڪيس کي بچائڻ لاءِ هڪ وڌيڪ وقت محفوظ ڪريو تي ڪلڪ ڪريو.
