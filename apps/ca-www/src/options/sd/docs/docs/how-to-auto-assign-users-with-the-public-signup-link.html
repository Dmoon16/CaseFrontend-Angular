پبلڪ سائن اپ لنڪ سان صارفين کي خودڪار طور تي ڪيئن تفويض ڪجي

منتظمين سکو ته ڪيئن سسٽم سيٽنگون ترتيب ڏيو ته جيئن نوان استعمال ڪندڙ خودڪار طور تي نئين خودڪار ٺاهيل ڪيس ۾ مقرر ڪيا وڃن.

سيٽ اپ صفحي کي کولڻ

پهريون قدم کولڻ آهي سسٽم سيٽ اپ صفحو.

  1. منتظم کاتي ۾ لاگ ان ٿيو.
  2. ايپس پيج تي ائپ ڳولھيو ۽ ايڊمن بٽڻ تي ڪلڪ ڪريو.
  3. ڪلڪ ڪريو سسٽم ٽئب ۽ پوءِ سسٽم سيٽ اپ سب مينيو تي.
  4. USER SIGNUP عنوان ڏانھن ھيٺ لٿو.

سسٽم سيٽنگون ترتيب ڏيڻ

  1. سيٽ ڪريو سائن اپ جي اجازت ڏيو ۽ آٽو ٺاهيو ڪيس ڊراپ ڊائون مينيو تي.
  2. رول مينيو تي ڪلڪ ڪريو ۽ اختيار چونڊيو (ڊفالٽ سفارش ڪئي وئي آهي).
  3. ڪاپي ڪريو ۽ حصيداري ڪريو عوامي صارف سائن اپ لنڪ.

لنڪ موڪلڻ

هر صارف جيڪو لنڪ تي ڪلڪ ڪري ٿو ۽ سائن اپ مڪمل ڪري ٿو پليٽ فارم کي ٽرگر ڪري ٿو هڪ نئون ڪيس پيدا ڪرڻ لاءِ ڪيس پيج تي ۽ صارف کي ان صورت ۾ خودڪار تفويض ڪري ٿو.
