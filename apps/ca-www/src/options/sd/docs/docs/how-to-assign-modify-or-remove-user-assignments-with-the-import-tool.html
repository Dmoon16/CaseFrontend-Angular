درآمدي اوزار سان صارف جي اسائنمنٽس کي ڪيئن تفويض، تبديل ڪرڻ يا ختم ڪرڻ

هي آرٽيڪل ڏيکاري ٿو منتظمين کي ڪيئن منظم ڪجي ڪيس استعمال ڪندڙن کي درآمد واري اوزار سان.

منتظم ڪري سگھن ٿا:

  • صارفين کي هڪ ڪيس ۾ تفويض ڪريو
  • تبديل ڪريو صارفين کي اڳ ۾ ئي هڪ ڪيس ۾ مقرر ڪيو ويو آهي
  • هٽايو صارفين کي هڪ ڪيس ۾ لڳايو ويو آهي

درآمد وارو اوزار کولڻ

  1. منتظم کاتي ۾ لاگ ان ٿيو.
  2. ايپس پيج تي ائپ ڳولھيو ۽ ايڊمن بٽڻ تي ڪلڪ ڪريو.
  3. انٽيگريشن ٽيب تي ڪلڪ ڪريو ۽ پوءِ سب مينيو تي درآمد ڪريو.
  4. NEW بٽڻ تي ڪلڪ ڪريو. شامل ڪريو درآمدات ڊائلاگ باڪس ڏيکاري ٿو.

رڪارڊ ٺاهڻ

هڪ يا وڌيڪ CSV فائلون ڊائون لوڊ ڪريو. استعمال ڪندڙن کي شامل ڪرڻ، تبديل ڪرڻ يا ختم ڪرڻ لاءِ ھي فائلون استعمال ڪريو. ان کان پوءِ شامل ڪريو درآمدات ڊائلاگ باڪس کلي ٿو:

  1. ٽائپ فيلڊ تي ڪلڪ ڪريو.
  2. ڊراپ-ڊائون مينيو ذريعي سکرول ڪريو ۽ اختيار چونڊيو (هيٺ ڏسو). هڪ لنڪ ڏيکاري ٿو.
  3. ڊائون لوڊ نموني لنڪ تي ڪلڪ ڪريو. هڪ CSV فائل توهان جي ڊوائس تي ڊائون لوڊ.

فائل کوليو ۽ رڪارڊ شامل ڪريو. CSV فائل ۾ فارميٽ ڊيٽا ڪيئن استعمال ڪجي سکڻ لاءِ هتي ڪلڪ ڪريو.

فائل جا اختيار

صارفين کي ڪيسن ۾ تفويض ڪريو * استعمال ڪندڙن کي ڪيس ۾ شامل ڪرڻ لاءِ ھي اختيار چونڊيو. * ڪيس_رول_آرڊر سيل کي خالي ڇڏي ڏيو. * وڃو يوزر پيج ۽ ڪاپي ڪريو ٽيگ ID هن شخص لاءِ. جيڪڏهن ٽيگ ID جو ميدان خالي آهي، هڪ منفرد ID داخل ڪريو، محفوظ ڪريو تي ڪلڪ ڪريو، ۽ پوءِ ID کي CSV فائل ۾ نقل ڪريو.

img

تبديل ڪريو صارفين کي ڪيسن ۾ لڳايو ويو * هي اختيار چونڊيو موجوده استعمال ڪندڙن لاءِ ڊيٽا کي اپڊيٽ ڪرڻ لاءِ جيڪو ڪيس ۾ لڳايو ويو آهي. * ڪيس ٽيگ ID استعمال ڪريو ڪيس_ID ڪالمن لاءِ.

img

هٽايو استعمال ڪندڙن کي ڪيسن ۾ لڳايو ويو * هي اختيار چونڊيو صارفين کي هٽائڻ لاءِ هڪ ڪيس ۾ لڳايو ويو.

فائل اپ لوڊ ڪندي

توھان فوري طور تي ھڪڙي فائل اپلوڊ ڪري سگھو ٿا يا واپسي ڪري سگھوٿا شامل ڪريو درآمدات ڊائلاگ باڪس بعد جي تاريخ ۾ ۽ انھن مرحلن کي مڪمل ڪريو.

  1. درآمدات شامل ڪريو ڊائلاگ باڪس کوليو.
  2. CSV فائل لاءِ منفرد نالو داخل ڪريو.
  3. پنهنجي ڊوائيس تان فائل اپلوڊ ڪرڻ لاءِ پلس (+) آئڪن تي ڪلڪ ڪريو.
  4. ٽائپ فيلڊ تي ڪلڪ ڪريو ۽ اختيار چونڊيو. (مٿي ڏسو)
  5. ڪلڪ ڪريو محفوظ ڪريو.

پروسيسنگ

فائل ۽ سسٽم لوڊ ۾ رڪارڊ جي تعداد تي منحصر ڪري، پليٽ فارم کي درآمد ڪرڻ لاء ڪيترن ئي منٽن وٺي سگھي ٿو.

img

جڏهن درآمد مڪمل ٿئي ٿي، پليٽ فارم صارف جي رڪارڊ کي ترتيب ڏئي ٿو (يعني شامل ڪري ٿو، تبديل ڪري ٿو، يا هٽائي ٿو).

ناڪامي

جيڪڏهن درآمد آپريشن ناڪام ٿئي ٿي، پليٽ فارم هڪ ناڪام پيغام موٽائي ٿو. اڪثر ڪيسن ۾، ناڪامي جو سبب ڊيٽا فارميٽنگ آهي.

img

ناڪامي جو سبب ڏسڻ لاء:

  1. ڪلڪ ڪريو ڏسو عمل جي عنوان هيٺ.
  2. ايندڙ صفحي تي، مائوس کي هور ڪريو ناڪامي آئڪن تي. ٽول ٽائپ غلطي ڏيکاري ٿو.
  3. CVS فائل ڏانھن وڃو ۽ ترتيب ڏيو ۽ غلطي کي درست ڪريو.
  4. فائل کي ٻيهر لوڊ ڪريو.
