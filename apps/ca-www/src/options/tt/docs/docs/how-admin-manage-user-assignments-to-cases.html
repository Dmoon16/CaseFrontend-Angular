Администратор очракларны ничек куллана

Кулланучылар белән идарә итүнең биш ысулы бар. Ничек белү өчен сылтамага басыгыз.

Метод Тип Эшләр Детальләр
Эш ясаганда UI өстәргә Администратор төймәсе> Эшләр> Яңа төймә> Кулланучыларны билгеләү төймәсе
Иҗтимагый теркәлү сылтамасын кулланып, кулланучыны автоматик рәвештә билгеләгез Кулланучының эше өстәргә Администратор төймәсе> көйләүләр> Система урнаштыру бите
