Админнар кулланучыларны ничек идарә итә

Администраторлар кушымталарга һәм очракларга кем керә ала, үзгәртә яки сүндерә. Кулланучы платформага беренче тапкыр кергәч, аларга профиль эш процессын тәмамларга кирәк.

  • Яңа кулланучы профиле эш процессы турында күбрәк белү өчен монда басыгыз.

Кулланучы белән идарә итү

Түбәндәге таблицада кулланучы белән идарә итүнең биш ысулы күрсәтелгән. Ничек белү өчен сылтамага басыгыз.
