Кушымта ничек ясарга

Бу мәкалә яңа кулланучыларга бушлай сынау счетын ничек ясарга икәнен күрсәтә. Case Active бу хисап өчен яңа кулланучы администратор өстенлекләрен автоматик рәвештә бирә.

Хисап ясау

  1. Керү битенә керегез.
  2. Башлау төймәсенә басыгыз.
  3. Электрон почта адресын кертегез.
  4. Киләсе төймәгә басыгыз.

img

Хисапны тикшерү

  1. Яңа браузер салынмасын ачып, электрон почта счетына керегез.
  2. Алты санлы кодны күчерегез.
  3. Case Active салынмасына кире кайту.
  4. Кодны кырларга куегыз. Яңа тәрәзә күрсәтелә.

Яңа тәрәзәдә:

  1. Кирәкле кырларны тутырыгыз.
  2. Хезмәт шартларын укыгыз һәм ризалашу өчен рамканы тикшерегез.
  3. Түләүсез хисап ясау өчен басыгыз.

Кушымта урнаштыруны тәмамлау

Киләсе биттә эш процессы күрсәтелә. Кушымтагызны урнаштыру өчен адымнарны тәмамлагыз. Сез мәгълүматны яки хисап статусын үзгәртү яки яңарту өчен сез һәрвакыт бу битләргә кире кайта аласыз.

Кушымта турында биттә компания исемен һәм вебсайт кырларын тутырыгыз (кирәк), аннары Киләсе төймәгә басыгыз.

Хисап мәгълүматы битендә Түләү планы кырына басыгыз һәм вариантларны карагыз. Килешенгән көйләү бушлай. SUBMIT басыгыз.

Форманы тапшырганнан соң, Case Active кушымталар битен күрсәтә. Сез ясаган яки булган һәр кушымта өчен бит күрсәтә:

  1. Кушымта исеме
  2. Сезнең тип (кулланучы роле)
  3. Статус

img

Кушымталар бите - күп администратор функцияләре өчен башлангыч нокта:

  1. Аралашу, документлар тутыру, видео чат (һәм башкалар) белән бәйле сораулар буенча APP төймәсенә басыгыз.
  2. Администратор панелен ачу өчен ADMIN төймәсенә басыгыз. Монда администраторлар кулланучылар, очраклар, интеграцияләр һәм башкалар белән идарә итәләр.
  3. Хисапны исәпләү һәм хәбәр итү көйләүләре белән идарә итү өчен эллипска басыгыз.
  • Искәрмә Case Active администратор кулланучылары өчен ADMIN төймәсен күрсәтә. Административ өстенлекләре булмаган кулланучылар ADMIN төймәсен күрмиләр (астагы экранда күрсәтелгәнчә).
