Импорт коралы белән очраклар белән идарә итү мәгълүматларын форматлау

Түбәндә импорт коралы белән эшләрне идарә итү өчен CSV файлларында кирәкле кырлар исемлеге китерелгән.

кулланучы исеме

  • Кулланучының электрон почта адресын кертегез.

tag_id (идентификацияне соңрак күрсәтә аласыз)

  • уникаль идентификатор (хәреф саннарын кулланыгыз, сызыклар кабул ителә)

очрак

  • 0 = ябык
  • 1 = актив

дата_ ачылды

  • ай / көн / ел
  • мм / дд / ййй

турында

  • эшнең уникаль тасвирламасы (хәреф саннарын кулланыгыз, сызыклар кабул ителә)
