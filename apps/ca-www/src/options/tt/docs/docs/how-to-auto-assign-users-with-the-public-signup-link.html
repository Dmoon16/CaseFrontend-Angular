Иҗтимагый теркәлү сылтамасы белән кулланучыларны ничек автоматик рәвештә билгеләргә

Администраторлар система көйләүләрен ничек конфигурацияләргә өйрәнәләр, шулай итеп яңа кулланучылар автоматик рәвештә яңа автомобильгә корылган.

Урнаштыру битен ачу

Беренче адым - системаларны урнаштыру битен ачу.

  1. Администратор счетына керегез.
  2. Кушымталар битендә кушымтаны табыгыз һәм ADMIN төймәсенә басыгыз.
  3. Системалар салынмасына басыгыз, аннары субменюдагы SYSTEM SETUP.
  4. USER SIGNUP рубрикасына асылыгыз.

Система көйләнмәләрен конфигурацияләү

  1. Керү рөхсәтен һәм Автомобиль ясау очракларын ачу менюларын кабызыгыз.
  2. Роль менюсына басыгыз һәм вариантны сайлагыз (килешү тәкъдим ителә).
  3. Иҗтимагый кулланучыга язылу сылтамасын күчереп алыгыз.
