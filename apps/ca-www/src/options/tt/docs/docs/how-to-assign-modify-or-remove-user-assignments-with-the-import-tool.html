Импорт коралы белән кулланучының биремнәрен ничек билгеләү, үзгәртү яки бетерү

Бу мәкалә администраторларны импорт коралы белән ничек кулланырга икәнен күрсәтә.

Администраторлар:

  • кулланучыларны эшкә билгеләгез
  • эшкә билгеләнгән кулланучыларны үзгәртү
  • эшкә билгеләнгән кулланучыларны бетерегез

Импорт коралын ачу

  1. Администратор счетына керегез.
  2. Кушымталар битендә кушымтаны табыгыз һәм ADMIN төймәсенә басыгыз.
  3. Интеграция салынмасына басыгыз, аннары асты менюдагы ИМПОРТ.
  4. Я NEWА төймәгә басыгыз. Импортны өстәү диалог тартмасы күрсәтелә.

Язмалар ясау

Бер яки берничә CSV файлны йөкләү. Кулланучыларны өстәү, үзгәртү яки бетерү өчен бу файлларны кулланыгыз. Импортны өстәү диалог рамкасы ачылганнан соң:

  1. Тип кырына басыгыз.
  2. Ачылучы менюны әйләндереп алыгыз һәм вариантны сайлагыз (аста карагыз). Ссылка күрсәтелә.
  3. Йөкләү үрнәге сылтамасына басыгыз. Сезнең җайланмага CSV файл йөкләү.

Файлны ачыгыз һәм язмалар өстәгез. CSV файлында формат мәгълүматларын ничек кулланырга икәнен белү өчен монда басыгыз.

Файл вариантлары

Кулланучыларны очракларга билгеләгез * Кулланучыларны эшкә өстәү өчен бу вариантны сайлагыз. * Case_role_order шакмакларын буш калдырыгыз. * Кулланучылар битенә керегез һәм бу кеше өчен Тэг ID-ны күчерегез. Тэг ID кыры буш булса, уникаль ID кертегез, Саклау төймәсенә басыгыз, һәм ID-ны CSV файлына күчерегез.

Эшләргә билгеләнгән кулланучыларны үзгәртегез * Эшкә билгеләнгән хәзерге кулланучылар өчен мәгълүматны яңарту өчен бу параметрны сайлагыз. * Case_ID баганасы өчен Case Tag ID кулланыгыз.

Эшләргә билгеләнгән кулланучыларны бетерегез * Эшкә билгеләнгән кулланучыларны бетерү өчен бу параметрны сайлагыз.

Файлны йөкләү

Сез шунда ук файлны йөкли аласыз яки соңрак импорт өстәү диалог рамкасына кире кайта аласыз һәм бу адымнарны тәмамлый аласыз.

  1. Импорт өстәү диалог рамкасын ачыгыз.
  2. CSV файл өчен уникаль исем кертегез.
  3. Файлны җайланмадан йөкләү өчен плюс (+) иконасына басыгыз.
  4. Тип кырына басыгыз һәм вариантны сайлагыз. (өстә карагыз)
  5. Саклау төймәсенә басыгыз.

Эшкәртү

Файлдагы язмалар санына һәм система йөкләнешенә карап, платформа импортны эшкәртү өчен берничә минут вакыт ала ала.

Импорт тәмамлангач, платформа кулланучылар язмаларын көйли (ягъни өсти, үзгәртә яки бетерә) Эшләр битендә.

Уңышсызлык

Импорт операциясе уңышсыз булса, платформа уңышсыз хәбәрне кире кайтара. Күпчелек очракта, уңышсызлыкның сәбәбе - мәгълүмат форматлау.

Уңышсызлыкның сәбәбен күрү өчен:

  1. Эш башы астында Күрү төймәсенә басыгыз.
  2. Киләсе биттә тычканны Уңышсыз иконасына өстәгез. Корал киңәше хатаны күрсәтә.
  3. CVS файлына керегез һәм хатаны көйләгез һәм төзәтегез.
  4. Файлны яңадан йөкләгез.
