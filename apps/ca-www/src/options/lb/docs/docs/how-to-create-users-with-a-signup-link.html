Wéi erstellen ech Benotzer mat engem Umeldungslink

Dësen Artikel weist Administrateuren wéi een Benotzer erstellt andeems en Umeldungslink schéckt.

Konfiguratioun vum Umeldungslink

  1. Klickt op de Schéckt hinnen eng Umeldungslink um Enn vum Dialog Füügt Benotzer. Eng nei Säit gëtt ugewisen.
  2. Konfiguréiert d'Umeldungsoptiounen. (kuckt ënnen)

Vergewëssert Iech datt d' Umeldungsfeld Erlaabt fir op On ze setzen. (obligatoresch) Wann aktivéiert, generéiert Case Active en eenzegaartege Link.

Iwwerpréift d'Optiounen hei ënnen a klickt dann op d'Ikon fir de Link ze kopéieren. Dir kënnt de Link op Ärer Websäit setzen oder de Link un een oder méi Leit per E-Mail oder all Kommunikatiounskanal schécken.

img

Schafen engem Fall

Optional kënnt Dir automatesch e Fall fir all neie Benotzer erstellen an eng Benotzerroll zouginn.

  1. Klickt op de Feld Auto erstellen Fall a wielt Op. De Case User Roll Felder ginn ugewisen.
  2. Klickt op de Case User Roll a wielt eng Optioun. (kuckt ënnen)

Fall Benotzer Rollen

Et ginn dräi Fall Benotzer Rollen.

  • Clienten
  • Default
  • Personal

Schéckt de Link

Deelt de Link mat engem oder méi Leit. Jiddereen klickt op de Link a fäerdeg de Workflow fir hire Case Active Kont ze aktivéieren. Wann de Benotzer den Umeldungsworkflow ofgeschloss huet, füügt Case Active de Benotzer un d'Benotzerlëscht.

img

Nächst Schrëtt

Case Active schéckt dem Benotzer eng E-Mail Notifikatioun mam Umeldungslink. De Benotzer klickt op de Link a fäerdeg de Workflow fir nei Umeldungen.
