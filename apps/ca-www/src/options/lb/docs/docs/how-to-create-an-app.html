Wéi eng App erstellen

Dësen Artikel weist nei Benotzer wéi een e gratis Testkonto erstellt. Case Active gëtt automatesch den neie Benotzer Admin Privilegien fir dëse Kont.

Schafen e Kont

  1. Gitt op d' Login Säit .
  2. Klickt op de Start Knäppchen.
  3. Gitt eng E-Mailadress un.
  4. Klickt Next.

img

Verifizéiert e Kont

  1. Öffnen eng nei Browser Tab a loggt Iech op Ären E-Mail Kont un.
  2. Kopie de sechs-Zifferen Code.
  3. Zréck op d'Tab Case Active.
  4. Paste de Code an d'Felder. Eng nei Fënster weist.

An der neier Fënster:

  1. Fëllt déi néideg Felder aus.
  2. Liest d'Konditioune vum Service a kontrolléiert d'Këscht fir averstanen.
  3. Klickt CREATE E GRATIS KONT.

img

App Setup ofgeschloss

E Workflow weist op der nächster Säit. Fëllt d'Schrëtt aus fir Är App opzestellen. Dir kënnt ëmmer op dës Säiten zréckkommen fir Informatiounen oder Äre Kontstatus z'änneren oder ze aktualiséieren.

Op der Iwwer App Säit, fëllt de Firmennumm a Websäit Felder aus (obligatoresch) a klickt duerno op Next.

img

Op der Rechnungsinformatioun Säit, klickt op de Bezuelungsplang Feld an iwwerpréift d'Optiounen. De Standardastellung ass gratis. Klickt SUBMIT.

img

Nodeems Dir de Formulaire ofginn hutt, weist Case Active d'Apps Säit. Fir all App, déi Dir erstellt oder gehéiert, weist d'Säit:

  1. Den Numm vun der App
  2. Ären Typ (Benotzer Roll)
  3. Status

img

D'Apps Säit ass e Startpunkt fir vill Adminfunktiounen:

  1. Klickt op den APP Knäppchen fir ze kommunizéieren, Dokumenter auszefëllen, Videochat (a méi) iwwer Fall-relatéiert Themen.
  2. Klickt op den ADMIN Knäppchen fir den Admin Panel opzemaachen. Dëst ass wou Admins Benotzer, Fäll, Integratiounen a méi verwalten.
  3. Klickt op d'Ellipsis fir d'Rechnung an d'Notifikatiounsastellungen ze managen.
  • Note Case Active weist den ADMIN Knäppchen fir Admin Benotzer. Benotzer déi keng Admin Privilegien hunn gesinn net den ADMIN Knäppchen (wéi um Écran hei ënnendrënner).
