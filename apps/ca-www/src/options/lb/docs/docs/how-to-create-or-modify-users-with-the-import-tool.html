Wéi erstellen oder änneren Benotzer mam Import Tool

Dësen Artikel weist Administrateuren wéi een Benotzer mam Import Tool erstellt.

Öffnen vum Import Tool

Administrateuren kënnen den Import-Tool benotze fir nei Benotzer eropzelueden an existéierend Benotzerrecords z'änneren.

  1. Loggt Iech op den Admin Kont.
  2. Fannt eng App op der Apps Säit a klickt op den ADMIN Knäppchen.
  3. Op der Admin Panel, klickt op d'Integratioun Tab an dann IMPORTS am Ënnermenü.
  4. Klickt op den NEW Knäppchen. D'Füügt Importer Dialogbox weist an engem neie Browser Tab.

img

Schafen records

Nodeems d'Füügt Importer Dialogbox opmaacht:

  1. Klickt op d' Typ Feld a wielt eng Optioun (kuckt hei ënnen). E Link weist.
  2. Klickt op den Download Sample Link. Eng CSV Datei gëtt op Ärem Apparat erofgelueden.

Als nächst, öffnen d'Datei a füügt records derbäi. Klickt hei fir ze léieren wéi Dir Formatdaten an der CSV Datei benotzt.

img

Typ

Wann Dir op d'Typ Feld klickt, gëtt en Dropdown-Menü ugewisen. Wielt eng vun zwou Optiounen fir Bulkimport.

: Ausrufezeeche: NOTÉIERT Den Dropdown-Menü huet vill Méiglechkeeten, awer nëmmen zwee si mam Importinstrument verbonnen. All Optioun huet eng aner Probe CSV Datei.

Benotzer addéieren

  • Wielt dës Optioun fir eng einfach bulk eropluede vun Benotzer records.
  • Klickt op den Download Sample Link fir eng CSV Datei opzemaachen mat Kolonn Rubriken fir den Add users Import Typ.

Benotzer änneren

  • Wielt dës Optioun fir Daten fir aktuell Benotzer ze aktualiséieren.
  • Klickt op den Download Sample Link fir eng CSV Datei opzemaachen mat Spaltrubriken fir den Änneren Benotzer Import Typ.

Eroplueden vun der Datei

Dir kënnt e Fichier direkt eroplueden oder zréck an d'Dialogbox Importer addéieren zu engem spéideren Datum an dës Schrëtt ausfëllen.

  1. Öffnen d'Füügt Importer Dialogbox.
  2. Gitt en eenzegaartegen Numm fir d'CSV Datei.
  3. Klickt op de Plus (+) Ikon fir d'Datei vun Ärem Apparat eropzelueden.
  4. Klickt op d' Typ Feld a wielt eng Optioun. (kuckt uewen)
  5. Klickt SAVE.

img

Veraarbechtung

Ofhängeg vun der Unzuel vun den Opzeechnungen an der Datei an der Systembelaaschtung, kann d'Plattform e puer Minutten daueren fir den Import ze veraarbecht.

Wann den Import fäerdeg ass, passt d'Plattform d'Benotzerrecords un (dh bäidréit, ännert oder deaktivéiert) um Dashboard vum Admin Benotzer .

Echec

Wann d'Importoperatioun feelt, gëtt d'Plattform e Failed Message zréck. An deene meeschte Fäll ass d'Ursaach vum Feeler d'Dateformatéierung.

img

Fir d'Ursaach vum Echec ze gesinn:

  1. Klickt View ënner der Aktioun Rubrik.
  2. Op der nächster Säit, hover d'Maus op der Failed Ikon. Den Tooltip weist de Feeler.
  3. Gitt op d'CVS-Datei an ajustéiert a fixéiert de Feeler.
  4. Eroplueden de Fichier erëm.
