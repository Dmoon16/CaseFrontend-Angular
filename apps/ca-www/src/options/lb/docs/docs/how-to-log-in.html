Wéi aloggen

Léiert wéi Dir Iech op Äre existente Case Active Kont aloggen.

Aloggen

  1. Gitt op d' Login Säit .
  2. Klickt op de LOGIN Knäppchen.
  3. Gitt Är Email.
  4. Klickt NEXT.

Verifizéiert e Kont

  1. Öffnen eng nei Browser Tab a loggt Iech op Ären E-Mail Kont un.
  2. Kopie de sechs-Zifferen Code.
  3. Zréck op d'Tab Case Active.
  4. Paste de Code an d'Felder. Eng nei Fënster weist.

Case Active weist d'Apps Säit.

  • Notiz D'Plattform weist en ADMIN Knäppchen nieft dem APP Knäppchen fir Benotzer mat Admin Privilegien.
