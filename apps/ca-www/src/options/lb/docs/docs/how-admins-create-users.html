Wéi Administrateuren Benotzer erstellen

Administrateuren erstellen, änneren oder deaktivéieren wien Zougang zu Apps a Fäll kënnen hunn. Wann e Benotzer fir d'éischte Kéier op d'Plattform aloggen, musse se e Profil Workflow ofgeschloss hunn.

  • Klickt hei fir méi iwwer den neie Benotzerprofil Workflow ze léieren.

E Benotzer erstellen

D'Tabell hei ënnen weist fënnef Weeër fir e Benotzer ze kreéieren. Klickt op e Link fir ze léieren wéi.

| Methode | Typ | Aktiounen | Detailer | Neie Benotzer kritt eng Notifikatioun? |
|------------------------|------------------- ------|-------------------------- --------------------------|

Benotzt UI | UI | dobäizemaachen & änneren | Admin Button > Benotzer | Nee |

Wärend e Fall erstellt an de Benotzer un de Fall zougewisen | UI | addéieren | Admin Knäppchen> Fäll> Neie Knäppchen> Erstelle Fall Säit | Jo, wann aktivéiert |

Erstellt automatesch Benotzer mat ëffentlechen Umeldungslink | Benotzer Aktioun | addéieren | Admin Button > Settings > System Setup Säit | Jo, wann automatesch erstellen Fall aktivéiert ass |

Import Tool | UI | dobäizemaachen & änneren | Admin Button > Integratiounen > Importer | Nee |

Benotzt API (kënnt geschwënn) | API | dobäizemaachen & änneren | na | na |

Notifikatiounen

D'Plattform schéckt d'Benotzer keng Notifikatioun wann se an eng App bäigefüügt ginn. Wéi och ëmmer, Administrateuren kënnen Notifikatioune fir Fall Uerderen aktivéieren.
