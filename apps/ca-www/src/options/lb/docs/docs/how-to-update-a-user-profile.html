Wéi aktualiséieren ech e Benotzerprofil

Dësen Artikel weist d'Benotzer wéi d'Informatioun op hirer Profilsäit aktualiséieren.

  • Benotzer kënnen hir eege Profiler update.
  • Administrateuren kënnen d'Profil Säit vun all Benotzer aktualiséieren.

Fir Benotzer

Den éischte Schrëtt ass d'Profil Säit opzemaachen. D'Benotzer an d'Administrateuren verfollegen dës Schrëtt fir hir eege Profilsäit ze aktualiséieren.

  1. Mellt Iech op Äre Case Active Kont un.
  2. Klickt op de Profil Tab uewen op der Säit.
  3. Maacht Ännerungen.
  4. Klickt op Späicheren.

Déi meescht Felder sinn fakultativ, an e puer Felder sinn erfuerderlech. Déi erfuerderlech Felder sinn:

  • Virnumm
  • Familljennumm
  • E-Mailadress
  • Sprooch
  • Zäitzone

Fir Administrateuren

Administrateuren kënnen och d'Profilinformatioun fir aner Benotzer aktualiséieren. Dëst enthält dräi Felder ënner der Rubrik HOST USER INFO.

  1. Loggt Iech op Äre Case Active Admin Kont.
  2. Fannt eng App a klickt op den ADMIN Knäppchen.
  3. Klickt op Benotzer am Topmenü.
  4. Gitt an den Ënnermenü a wielt d'Benotzerlëscht op déi Dir wëllt zougräifen (AKTIV oder BEHËNNERT). De Standardastellung ass Aktiv.
  5. Situéiert e Benotzer op der Lëscht a klickt Edit.
  6. Maacht Ännerungen wéi néideg. (kuckt ënnen)
  7. Klickt op Späicheren.

Benotzerinformatioun änneren

Administrateuren kënnen d'Benotzerinformatioun änneren.

  1. Setzt de Benotzertyp (Benotzer oder Manager).
  2. Setzt de Benotzerstatus (Aktiv op Behënnert).
  3. Gitt eng eenzegaarteg Tag ID. (optional)
  4. Klickt op Späicheren.
