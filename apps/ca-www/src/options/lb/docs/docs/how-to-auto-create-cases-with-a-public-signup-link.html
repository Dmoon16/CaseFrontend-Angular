Wéi automatesch Fäll mat engem Ëffentlechen Umeldungslink erstellen

Dësen Artikel weist Administrateuren wéi een e Fall erstellt (a gläichzäiteg e Benotzer derbäigesat) andeems een en eenzegen Umeldungslink schéckt.

De Link lokaliséieren

  1. Loggt Iech op den Admin Kont.
  2. Fannt eng App op der Apps Säit a klickt op den ADMIN Knäppchen.
  3. Op der Admin Panel klickt op d'Astellungsmenü Tab.
  4. Vergewëssert Iech datt de SYSTEM SETUP Submenu Tab ausgewielt ass.

Astellunge konfiguréieren

Scroll down to the USER SIGNUP heading.

Vergewëssert Iech datt d'Aschreiwungsfeld erlaabt ass op On gesat. (obligatoresch) Wann dës Fonktioun aktivéiert ass, generéiert Case Active en eenzegaartege Link.

  1. Klickt op de Feld Auto erstellen Fall a wielt Op. De Fall Benotzer Roll Feld weist.
  2. Wielt eng Roll. (kuckt ënnen)

Fall Benotzer Rollen

Et ginn dräi Fall Benotzer Rollen.

  • Clienten (recommandéiert)
  • Default
  • Personal

Schéckt de Link

Deelt de Link mat engem oder méi Leit. All Benotzer, deen op de Link klickt an d'Umeldung ofgeschloss huet, léist d'Plattform aus fir en neie Fall op der Case Säit ze generéieren an de Benotzer am Fall automatesch ze ginn.
