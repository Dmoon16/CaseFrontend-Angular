Wéi Assignéieren, Änneren oder Ewechmaachen Benotzer Uerderen mam Import Tool

Dësen Artikel weist Administrateuren wéi een Case Benotzer mam Import Tool verwalten.

Administrateuren kënnen:

  • Benotzer zu engem Fall zougewisen
  • Ännerung Benotzer schonn un engem Fall zougewisen
  • ewechzehuelen Benotzer zu engem Fall zougewisen

Öffnen vum Import Tool

  1. Loggt Iech op den Admin Kont.
  2. Fannt eng App op der Apps Säit a klickt op den ADMIN Knäppchen.
  3. Klickt op d'Integratioun Reiter an dann IMPORTS am Ënnermenü.
  4. Klickt op den NEW Knäppchen. D'Dialogbox Importer addéieren erschéngt.

Schafen records

Luet een oder méi CSV Dateien erof. Benotzt dës Dateie fir Benotzer ze addéieren, z'änneren oder ze läschen. Nodeems d'Füügt Importer Dialogbox opmaacht:

  1. Klickt op d'Typ Feld.
  2. Scroll duerch den Dropdown-Menü a wielt eng Optioun (kuckt hei ënnen). E Link weist.
  3. Klickt op den Download Sample Link. Eng CSV Datei gëtt op Ärem Apparat erofgelueden.

Öffnen d'Datei a füügt records derbäi. Klickt hei fir ze léieren wéi Dir Formatdaten an der CSV Datei benotzt.

Fichier Optiounen

Assignéiert Benotzer op Fäll * Wielt dës Optioun fir Benotzer zu engem Fall ze addéieren. * Loosst d'Case_role_order Zell eidel. * Gitt op d'Benotzer Säit a kopéiert d'Tag ID fir dës Persoun. Wann d'Tag ID Feld eidel ass, gitt eng eenzegaarteg ID, klickt Späicheren, a kopéiert dann d'ID an d'CSV Datei.

Änneren Benotzer zu Fäll zougewisen * Wielt dës Optioun fir eng Aktualiséierung opgetrueden Daten fir aktuell Benotzer zu engem Fall zougewisen. * Benotzt de Case Tag ID fir d'Case_ID Kolonn.

Eroplueden vun der Datei

Dir kënnt e Fichier direkt eroplueden oder zréck an d'Dialogbox Importer addéieren zu engem spéideren Datum an dës Schrëtt ausfëllen.

  1. Öffnen d'Füügt Importer Dialogbox.
  2. Gitt en eenzegaartegen Numm fir d'CSV Datei.
  3. Klickt op de Plus (+) Ikon fir d'Datei vun Ärem Apparat eropzelueden.
  4. Klickt op d'Typ Feld a wielt eng Optioun. (kuckt uewen)
  5. Klickt SAVE.

Veraarbechtung

Ofhängeg vun der Unzuel vun den Opzeechnungen an der Datei an der Systembelaaschtung, kann d'Plattform e puer Minutten daueren fir den Import ze veraarbecht.

Wann den Import fäerdeg ass, passt d'Plattform Benotzerrecords un (dh addéiert, ännert oder läscht) op der Cases Säit.

Echec

Wann d'Importoperatioun feelt, gëtt d'Plattform e Failed Message zréck. An deene meeschte Fäll ass d'Ursaach vum Feeler d'Dateformatéierung.

Fir d'Ursaach vum Echec ze gesinn:

  1. Klickt View ënner der Aktioun Rubrik.
  2. Op der nächster Säit, hover d'Maus op der Failed Ikon. Den Tooltip weist de Feeler.
  3. Gitt op d'CVS-Datei a passt de Feeler un a fixéiert.
  4. Eroplueden de Fichier erëm.
