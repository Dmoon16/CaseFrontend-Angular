Как обновить профиль пользователя

В этой статье показано, как пользователям обновлять информацию на странице своего профиля.

  • Пользователи могут обновлять свои собственные профили.
  • Администраторы могут обновлять страницу профиля любого пользователя.

Для пользователей

Первый шаг — открыть страницу профиля. Пользователи и администраторы выполняют следующие действия, чтобы обновить свою страницу профиля.

  1. Войдите в свою учетную запись Case Active.
  2. Щелкните вкладку «Профиль» в верхней части страницы.
  3. Вносить изменения.
  4. Щелкните Сохранить.

Большинство полей являются необязательными, а некоторые обязательны для заполнения. Обязательные поля:

  • Имя
  • Фамилия
  • Адрес электронной почты
  • Язык
  • Часовой пояс

Для администраторов

Администраторы также могут обновлять информацию профиля для других пользователей. Сюда входят три поля под заголовком HOST USER INFO.

  1. Войдите в свою учетную запись администратора Case Active.
  2. Найдите приложение и нажмите кнопку ADMIN.
  3. Щелкните Пользователи в верхнем меню.
  4. Перейдите в подменю и выберите список пользователей, к которому вы хотите получить доступ (АКТИВНЫЙ или ОТКЛЮЧЕННЫЙ). Значение по умолчанию — Активный.
  5. Найдите пользователя в списке и нажмите «Изменить».
  6. Внесите необходимые изменения. (Смотри ниже)
  7. Щелкните Сохранить.

Изменить информацию о пользователе

Администраторы могут изменять информацию о пользователе.

  1. Установите тип пользователя (пользователь или менеджер).
  2. Установите статус пользователя (от активного до отключенного).
  3. Введите уникальный идентификатор тега. (по желанию)
  4. Щелкните Сохранить.
