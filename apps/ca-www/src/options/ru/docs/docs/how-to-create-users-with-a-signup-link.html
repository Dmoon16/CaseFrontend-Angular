Как создать пользователей с помощью ссылки для регистрации

В этой статье администраторам показано, как создавать пользователей, отправляя ссылку для регистрации.

Настройка ссылки для регистрации

  1. Щелкните ссылку «Отправить им регистрацию » в нижней части диалогового окна «Добавить пользователя». Отобразится новая страница.
  2. Настройте параметры регистрации. (Смотри ниже)

изображение

Убедитесь, что в поле Разрешить регистрацию установлено значение Вкл. (обязательно) Когда включено, Case Active генерирует уникальную ссылку.

Просмотрите параметры ниже, а затем щелкните значок, чтобы скопировать ссылку. Вы можете разместить ссылку на своем веб-сайте или отправить ссылку одному или нескольким людям по электронной почте или любому каналу связи.

изображение

Создание дела

При желании вы можете автоматически создать кейс для каждого нового пользователя и назначить роль пользователя.

  1. Щелкните поле «Автоматическое создание обращения» и выберите «Вкл.». Отобразятся поля Роль пользователя дела.
  2. Щелкните Роль пользователя обращения и выберите параметр. (Смотри ниже)

изображение

Роли пользователей дела

Существует три роли пользователя дела.

  • клиенты
  • дефолт
  • сотрудники

Отправка ссылки

Поделитесь ссылкой с одним или несколькими людьми. Каждый человек щелкает ссылку и завершает рабочий процесс, чтобы активировать свою учетную запись Case Active. Как только пользователь завершает рабочий процесс регистрации, Case Active добавляет пользователя в список пользователей.

Следующие шаги

Case Active отправляет пользователю уведомление по электронной почте со ссылкой для регистрации. Пользователь щелкает ссылку и завершает рабочий процесс для новых регистраций.
