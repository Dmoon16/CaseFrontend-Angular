Как автоматически назначать пользователей с помощью общедоступной ссылки для регистрации

Администраторы узнают, как настроить параметры системы, чтобы новые пользователи автоматически назначались для вновь созданного дела.

Открытие страницы настройки

Первый шаг — открыть страницу настройки системы.

  1. Войдите в учетную запись администратора.
  2. Найдите приложение на странице приложений и нажмите кнопку ADMIN.
  3. Перейдите на вкладку «Системы», а затем в подменю «НАСТРОЙКА СИСТЕМЫ».
  4. Прокрутите вниз до заголовка USER SIGNUP.

Настройка параметров системы

  1. Установите для раскрывающихся меню «Разрешить регистрацию» и «Автоматическое создание обращения» значение «Вкл.».
  2. Щелкните меню ролей и выберите вариант (рекомендуется значение по умолчанию).
  3. Скопируйте и поделитесь общедоступной ссылкой для регистрации пользователей.

Отправка ссылки

Каждый пользователь, который щелкает ссылку и завершает регистрацию, запускает платформу для создания нового обращения на странице «Обращения» и автоматически назначает пользователя в этом деле.
