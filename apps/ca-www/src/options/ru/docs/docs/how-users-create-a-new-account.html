Как пользователи создают новую учетную запись

После того, как администратор добавит пользователя в дело: * если пользователь является новым в caseactive, пользователь должен выполнить рабочий процесс, чтобы создать учетную запись и принять приглашение для доступа к делу. * если у пользователя есть активная учетная запись кейса, пользователь должен принять приглашение для доступа к кейсу.

В этой статье описываются шаги, которые должен выполнить пользователь, чтобы создать свою учетную запись.

Начиная

Есть два способа добавить пользователя в caseactive приложение.

  • Администратор: если администратор создает пользователя с помощью пользовательского интерфейса платформы или инструмента импорта , он должен отправить пользователю ссылку на account.caseactive.com, чтобы перейти к входу в систему.
  • По пользователю: если пользователь создал учетную запись, используя ссылку для регистрации , он будет автоматически перенаправлен на account.caseactive.com для входа в систему.

Вход в систему

Когда пользователь входит в свою учетную запись CaseActive в первый раз, платформа предлагает пользователю:

  • откройте свою учетную запись электронной почты, а затем скопируйте / вставьте шестизначный код безопасности
  • заполнить профиль учетной записи

изображение

После того, как пользователь нажмет ОБНОВИТЬ, отобразится страница приложений. На странице отображается название приложения (название компании), тип пользователя (роль), статус и кнопка ПРИГЛАШЕНИЕ.

изображение

Чтобы получить доступ к приложению, пользователь должен принять приглашение. Пользователь должен:

  1. Нажмите кнопку ПРИГЛАШЕНИЕ.
  2. Просмотрите разрешения.
  3. Перейдите на вкладку «Уведомления», чтобы просмотреть и изменить (необязательно) условия.
  4. Нажмите ПОДТВЕРДИТЬ ДОСТУП.

изображение

Когда пользователь подтверждает, платформа отображает кнопку APP на странице пользователя. Эта кнопка дает пользователю доступ к назначенным случаям.

Чтобы изменить пользовательские настройки, пользователь щелкает многоточие.

изображение

Пользователь может: * удалить доступ к приложению * изменить настройки уведомлений

изображение

После того, как пользователь завершит рабочий процесс, имя пользователя появится в списке пользователей администратора. Список показывает всех пользователей, вошедших в систему или нет.
