Как создавать и изменять пользователей с помощью пользовательского интерфейса

В этой статье администраторам показано, как создавать пользователей и обновлять записи с помощью пользовательского интерфейса платформы. Администраторы также могут добавлять пользователей в новое дело и редактировать существующие дела.

Создание пользователя

  1. Войдите в учетную запись администратора.
  2. Найдите приложение на странице приложений и нажмите кнопку ADMIN.
  3. На панели администратора щелкните вкладку меню «Пользователи».
  4. Нажмите кнопку НОВЫЙ. Появится новое окно.

изображение

Заполните поля и нажмите СОХРАНИТЬ. Case Active добавляет имя в список пользователей, связанных с этим приложением.

изображение

Изменение пользователя

Чтобы обновить небольшое количество пользовательских записей, используйте инструмент «Редактировать» и внесите изменения для каждого отдельного пользователя.

  • Примечание . Чтобы изменить большое количество пользовательских записей за один раз, рассмотрите возможность использования инструмента «Импорт ».
  1. На панели администратора щелкните вкладку меню «Пользователи».
  2. Найдите пользователя и нажмите «Изменить».
  3. Внесите изменения и нажмите СОХРАНИТЬ.

Следующие шаги

Case Active не отправляет пользователю уведомление по электронной почте. Администраторы должны связаться с пользователем и отправить ему ссылку на домашнюю страницу Case Active. Затем пользователь нажимает кнопку ВХОД и завершает рабочий процесс для новых регистраций.
