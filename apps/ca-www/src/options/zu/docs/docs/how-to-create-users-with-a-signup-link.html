Ubakha Kanjani Abasebenzisi Ngesixhumanisi Sokubhalisa

Lesi sihloko sibonisa abaphathi ukuthi babadala kanjani abasebenzisi ngokuthumela isixhumanisi sokubhalisa.

Ilungiselela isixhumanisi sokubhalisa

  1. Chofoza okuthi Bathumele isixhumanisi sokubhalisa ezansi kwebhokisi lengxoxo elithi Faka umsebenzisi. Ikhasi elisha liyaboniswa.
  2. Lungiselela izinketho zokubhalisa. (bona ngezansi)

Qiniseka ukuthi inkambu ethi Vumela Ukubhalisa ukuze isethelwe kokuthi Vuliwe. (kuyadingeka) Uma inikwe amandla, i-Case Active ikhiqiza isixhumanisi esiyingqayizivele.

Buyekeza izinketho ezingezansi bese uchofoza isithonjana ukuze ukopishe isixhumanisi. Ungabeka isixhumanisi kuwebhusayithi yakho noma uthumele isixhumanisi kumuntu oyedwa noma ngaphezulu nge-imeyili nanoma yisiphi isiteshi sokuxhumana.

Ukudala icala

Ongakukhetha, ungadala ngokuzenzakalela icala lomsebenzisi ngamunye omusha futhi unikeze indima yomsebenzisi.

  1. Chofoza inkambu ethi Dala Ngokuzenzakalelayo Ikesi bese ukhetha Vuliwe. Izinkambu zendima yomsebenzisi wekesi ziyaboniswa.
  2. Chofoza Indima Yomsebenzisi Wecala bese ukhetha inketho. (bona ngezansi)

Ithumela isixhumanisi

Yabelana ngesixhumanisi nomuntu oyedwa noma ngaphezulu. Umuntu ngamunye uchofoza isixhumanisi futhi aqedele ukugeleza komsebenzi ukuze enze kusebenze i-akhawunti yakhe ethi Case Active. Uma umsebenzisi eseqedile ukugeleza komsebenzi wokubhalisa, i-Case Active yengeza umsebenzisi ohlwini lwabasebenzisi.

img

Izinyathelo ezilandelayo

I-Case Active ithumela umsebenzisi isaziso se-imeyili esinesixhumanisi sokubhalisa. Umsebenzisi uchofoza isixhumanisi futhi aqedele ukuhamba komsebenzi kokubhalisa okusha.

  • Lesi sihloko sichaza ukuhamba komsebenzi wokubhalisa kubasebenzisi abasha.
