Ungangena kanjani

Funda ukuthi ungangena kanjani ku-akhawunti yakho ekhona esebenzayo.

Ngena ngemvume

  1. Iya ekhasini lokungena .
  2. Chofoza inkinobho ethi LOGIN.
  3. Faka i-imeyili yakho.
  4. Chofoza OKULANDELAYO.

Iqinisekisa i-akhawunti

  1. Vula ithebhu yesiphequluli esisha bese ungena ku-akhawunti yakho ye-imeyli.
  2. Kopisha ikhodi enamadijithi ayisithupha.
  3. Buyela kuthebhu ethi Ikesi Liyasebenza.
  4. Namathisela ikhodi ezinkambini. Iwindi elisha liyavela.

I-Case Active ibonisa ikhasi lezinhlelo zokusebenza.

  • Qaphela Inkundla ibonisa inkinobho ethi ADMIN eduze kwenkinobho ye-APP kubasebenzisi abanamalungelo okuphatha.
