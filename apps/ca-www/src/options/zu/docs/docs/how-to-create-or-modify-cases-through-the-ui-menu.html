Ungawakha kanjani noma uwaguqule kanjani amacala ngemenyu ye-UI

Lesi sihloko sibonisa abalawuli ukuthi lingadala kanjani icala ngesixhumi esibonakalayo somsebenzisi wenkundla. Abaphathi bangakwazi futhi ukwengeza abasebenzisi ecaleni elisha futhi babuyekeze amacala akhona.

Ukudala icala

  1. Ngena ngemvume ku-akhawunti yomqondisi.
  2. Thola uhlelo lokusebenza ekhasini lezinhlelo zokusebenza bese uchofoza inkinobho ethi ADMIN.
  3. Kuphaneli Yokuphatha, chofoza ithebhu yemenyu Yamacala.
  4. Chofoza inkinobho ENTSHA. Iwindi elisha liyavela.

img

Gcwalisa izinkambu ezidingekayo bese uchofoza LINDISA.

Yengeza abasebenzisi

Ngokuzithandela, abalawuli bangakwazi ukwengeza abasebenzisi ecaleni kuleli khasi. Skrolela phansi bese uchofoza inkinobho ethi ASSIGN USERS .

img

Kumodi Yabela Umsebenzisi, gcwalisa izinkambu. Thikha ibhokisi ukuze uthumele umsebenzisi isaziso se-imeyili. (uyazikhethela)

Ukulungisa icala

Ukuze ubuyekeze inani elincane lamakesi, sebenzisa ithuluzi lokuhlela futhi wenze izinguquko esimweni ngasinye.

  • Qaphela Ukuze ubuyekeze inani elikhulu lamakesi ngesikhathi esisodwa, cabanga ukusebenzisa ithuluzi Lokungenisa
  1. Ngena ngemvume ku-akhawunti yomqondisi.
  2. Thola uhlelo lokusebenza ekhasini lezinhlelo zokusebenza bese uchofoza inkinobho ethi ADMIN.
  3. Kuphaneli Yokuphatha, chofoza ithebhu yemenyu Yamacala.
  4. Thola ikesi bese uchofoza Hlela.
  5. Yenza izinguquko bese uchofoza LINDISA.
