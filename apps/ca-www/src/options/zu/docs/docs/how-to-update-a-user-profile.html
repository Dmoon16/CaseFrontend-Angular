Indlela Yokuvuselela Iphrofayela Yomsebenzisi

Lesi sihloko sibonisa abasebenzisi ukuthi balubuyekeza kanjani ulwazi ekhasini labo Lephrofayela.

  • Abasebenzisi bangabuyekeza amaphrofayili abo.
  • Abaphathi bangabuyekeza ikhasi Lephrofayela lanoma yimuphi umsebenzisi.

Okwabasebenzisi

Isinyathelo sokuqala ukuvula ikhasi lephrofayela. Abasebenzisi nabalawuli balandela lezi zinyathelo ukuze babuyekeze ikhasi labo Lephrofayela.

  1. Ngena ngemvume ku-akhawunti yakho ye-Case Active.
  2. Chofoza ithebhu Yephrofayela phezulu ekhasini.
  3. Yenza izinguquko.
  4. Chofoza Londoloza.

Izinkambu eziningi ziyazikhethela, futhi ezinye izinkambu ziyadingeka. Izinkambu ezidingekayo yilezi:

  • Igama
  • Isibongo
  • Ikheli le-imeyili
  • Ulimi
  • Indawo yesikhathi

Okwabaphathi

Abaphathi bangakwazi futhi ukubuyekeza ulwazi lwephrofayela kwabanye abasebenzisi. Lokhu kuhlanganisa izinkambu ezintathu ngaphansi kwesihloko esithi HOST USER INFO.

  1. Ngena ngemvume ku-akhawunti yakho ye-Case Active admin.
  2. Thola uhlelo lokusebenza bese uchofoza inkinobho ethi ADMIN.
  3. Chofoza Abasebenzisi kumenyu ephezulu.
  4. Yiya ku-sub-menu bese ukhetha uhlu lwabasebenzisi ofuna ukufinyelela kulo (OKUSEBENZA noma KUKHUBAZIWE). Ukusetha okuzenzakalelayo kuyasebenza.
  5. Thola umsebenzisi ohlwini bese uchofoza Hlela.
  6. Yenza izinguquko njengoba kudingeka. (bona ngezansi)
  7. Chofoza Londoloza.

Shintsha ulwazi lomsebenzisi

Abaphathi bangashintsha ulwazi lomsebenzisi.

  1. Setha Uhlobo Lomsebenzisi (Umsebenzisi noma Umphathi).
  2. Setha Isimo Somsebenzisi (Siyasebenza Ukuze Sikhutshazwe).
  3. Faka i-ID ye-Tag ehlukile. (uyazikhethela)
  4. Chofoza Londoloza.
