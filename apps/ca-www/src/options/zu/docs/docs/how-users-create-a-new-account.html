Indlela Abasebenzisi Abadala Ngayo I-akhawunti Entsha

Ngemva kokuthi umlawuli engeze umsebenzisi ecaleni: * uma umsebenzisi emusha ekusebenzeni, umsebenzisi kufanele aqedele ukuhamba komsebenzi ukuze adale i-akhawunti futhi amukele isimemo sokufinyelela icala. * uma umsebenzisi ene-caseactive acounnt ekhona, umsebenzisi kufanele amukele isimemo sokufinyelela icala.

Lesi sihloko sichaza izinyathelo umsebenzisi okufanele azigcwalise ukuze adale i-akhawunti yakhe.

Ukuqalisa

Kunezindlela ezimbili umsebenzisi angengeza ngazo kuhlelo lokusebenza olusebenzayo

  • Ngomphathi: Uma umlawuli edala umsebenzisi esebenzisa i- UI yenkundla noma ithuluzi Lokungenisa , umlawuli kufanele athumele umsebenzisi isixhumanisi ku-account.caseactive.com ukuze aqhubeke nokungena.
  • Ngomsebenzisi: Uma umsebenzisi edale i-akhawunti esebenzisa isixhumanisi sokubhalisa , umsebenzisi uzoqondiswa kabusha ngokuzenzakalela ku-account.caseactive.com ukuze aqhubeke nokungena.

Ngena ngemvume

Uma umsebenzisi engena ngemvume ku-akhawunti yakhe ye-CaseActive okokuqala, inkundla yazisa umsebenzisi ukuthi:

  • bavule i-akhawunti yabo ye-imeyili bese ukopisha/unamathisele ikhodi yokuphepha enezinhlamvu eziyisithupha
  • qedela iphrofayili ye-akhawunti

Ngemuva kokuthi umsebenzisi achofoze okuthi BUYISA, ikhasi lezinhlelo zokusebenza liyavela. Ikhasi libonisa igama lohlelo lokusebenza (igama lenkampani), uhlobo lomsebenzisi (indima), isimo, kanye nenkinobho ethi INVITATION.

Ukuze ufinyelele uhlelo lokusebenza, umsebenzisi kufanele amukele isimemo. Umsebenzisi kufanele:

  1. Chofoza inkinobho ethi INVITATION.
  2. Buyekeza izimvume.
  3. Chofoza ithebhu Yezaziso ukuze ubuyekeze futhi uhlele (uma uthanda) imigomo.
  4. Chofoza okuthi VUMISA UKUFINYELELA.

img

Uma umsebenzisi evuma, inkundla ibonisa inkinobho ye-APP ekhasini lomsebenzisi. Le nkinobho inika umsebenzisi ukufinyelela kumacala abelwe wona.

Ukuze uhlele izilungiselelo zomsebenzisi, umsebenzisi uchofoza i-ellipsis.

Umsebenzisi angakwazi: * ukususa ukufinyelela kohlelo lokusebenza * aguqule izilungiselelo zesaziso

Ngemva kokuba umsebenzisi eqedele ukugeleza komsebenzi, igama lomsebenzisi liyavela ohlwini lwabasebenzisi bomlawuli. Uhlu lubonisa bonke abasebenzisi, abangenile noma cha.
