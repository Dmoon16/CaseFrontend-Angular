Uludala kanjani uhlelo lokusebenza

Lesi sihloko sibonisa abasebenzisi abasha indlela yokudala i-akhawunti yesilingo samahhala. I-Case Active inikeza ngokuzenzakalelayo amalungelo okuphatha ale akhawunti.

Ukudala i-akhawunti

  1. Iya ekhasini lokungena .
  2. Chofoza inkinobho ethi Qalisa.
  3. Faka ikheli le-imeyili.
  4. Chofoza Okulandelayo.

Iqinisekisa i-akhawunti

  1. Vula ithebhu yesiphequluli esisha bese ungena ku-akhawunti yakho ye-imeyli.
  2. Kopisha ikhodi enamadijithi ayisithupha.
  3. Buyela kuthebhu ethi Ikesi Liyasebenza.
  4. Namathisela ikhodi ezinkambini. Iwindi elisha liyavela.

Ewindini elisha:

  1. Gcwalisa izinkambu ezidingekayo.
  2. Funda imigomo yesevisi bese uqoka ibhokisi ukuze uvume.
  3. Chofoza okuthi DALA I-AKHAWUNTI YAMAHHALA.

Iqedela ukumisa uhlelo lokusebenza

Ukugeleza komsebenzi kuboniswa ekhasini elilandelayo. Qedela izinyathelo ukuze uqedele ukusetha uhlelo lwakho lokusebenza. Ungakwazi njalo ukubuyela kulawa makhasi ukuze uhlele noma ubuyekeze ulwazi noma isimo se-akhawunti yakho.

Ekhasini elithi Mayelana nohlelo lokusebenza, gcwalisa igama lenkampani nezinkambu zewebhusayithi (kuyadingeka) bese uchofoza Okulandelayo.

Ekhasini Lolwazi Lwenkokhelo, chofoza inkambu Yohlelo Lokukhokha futhi ubuyekeze izinketho. Ukusetha okuzenzakalelayo kumahhala. Chofoza okuthi THUMELA.

Ngemva kokuhambisa ifomu, i-Case Active ibonisa ikhasi lezinhlelo zokusebenza. Kuhlelo lokusebenza ngalunye oludalayo noma oluyingxenye yalo, ikhasi liyabonisa:

  1. Igama lohlelo lokusebenza
  2. Uhlobo lwakho (indima yomsebenzisi)
  3. Isimo

img

Ikhasi lezinhlelo zokusebenza liyindawo yokwethulwa kwemisebenzi eminingi yokuphatha:

  1. Chofoza inkinobho ye-APP ukuze uxhumane, ugcwalise amadokhumenti, ingxoxo yevidiyo (nokwengeziwe) ezindabeni ezihlobene namacala.
  2. Chofoza inkinobho ethi ADMIN ukuze uvule iphaneli yokulawula. Lapha yilapho Abalawuli baphatha khona abasebenzisi, izimo, ukuhlanganiswa, nokunye okwengeziwe.
  3. Chofoza i-ellipsis ukuze uphathe ukukhokhiswa kwe-akhawunti nezilungiselelo zezaziso.
  • I- Note Case Active ibonisa inkinobho ethi ADMIN kubasebenzisi abangabaphathi. Abasebenzisi abangenawo amalungelo okuphatha abayiboni inkinobho ethi ADMIN (njengoba kuboniswe kusikrini esingezansi).
