Форматування даних призначень іменів користувачів за допомогою інструмента імпорту

Нижче наведено список полів даних у файлах CSV. Кожен параметр імпорту має різну комбінацію полів.

case_ID

  • Введіть ідентифікатор користувача або тегу в регістрі.
  • Перейдіть на сторінку запитів > Редагувати > Ідентифікатор тегу АБО Перейдіть на сторінку запитів > Редагувати > Ідентифікатор запиту в URL-адресі.

ідентифікатор користувача

  • Введіть ідентифікатор користувача або ідентифікатор тегу.
  • Перейдіть на сторінку Користувачі > Редагувати > Ідентифікатор тегу АБО Перейдіть на сторінку Користувачі > Редагувати > Ідентифікатор користувача в URL-адресі.

case_role_order

  • Залиште клітинку порожньою.

case_role_ID

  • Введіть: клієнти, персонал або за замовчуванням.

case_notify

  • 0 = ні
  • 1 = Так, надіслати користувачеві сповіщення електронною поштою.
