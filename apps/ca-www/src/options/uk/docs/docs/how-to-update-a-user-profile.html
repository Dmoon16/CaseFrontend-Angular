Як оновити профіль користувача

У цій статті користувачам показано, як оновлювати інформацію на сторінці свого профілю.

  • Користувачі можуть оновлювати власні профілі.
  • Адміністратори можуть оновлювати сторінку профілю будь-якого користувача.

Для користувачів

Першим кроком є відкриття сторінки профілю. Користувачі та адміністратори виконають ці дії, щоб оновити власну сторінку профілю.

  1. Увійдіть у свій обліковий запис Case Active.
  2. Натисніть вкладку Профіль у верхній частині сторінки.
  3. Внесіть зміни.
  4. Натисніть Зберегти.

Більшість полів необов’язкові, а деякі обов’язкові. Необхідні поля:

  • Ім'я
  • Прізвище
  • Адреса електронної пошти
  • Мову
  • Часовий пояс

Для адмінів

Адміністратори також можуть оновлювати інформацію профілю для інших користувачів. Це включає три поля під заголовком HOST USER INFO.

  1. Увійдіть у свій обліковий запис адміністратора Case Active.
  2. Знайдіть програму та натисніть кнопку АДМІНІСТРАТОР.
  3. Натисніть Користувачі у верхньому меню.
  4. Перейдіть до підменю та виберіть список користувачів, до якого ви хочете отримати доступ (АКТИВНИЙ або ВИМКНЕНИЙ). За замовчуванням встановлено значення «Активний».
  5. Знайдіть користувача в списку та натисніть Редагувати.
  6. Внесіть необхідні зміни. (Дивіться нижче)
  7. Натисніть Зберегти.

Змінити інформацію про користувача

Адміністратори можуть змінювати інформацію про користувача.

  1. Встановіть тип користувача (користувач або менеджер).
  2. Встановіть статус користувача (Активний на Вимкнено).
  3. Введіть унікальний ідентифікатор тегу. (необов'язково)
  4. Натисніть Зберегти.
