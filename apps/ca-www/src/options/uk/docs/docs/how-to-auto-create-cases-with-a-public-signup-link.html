Як автоматично створювати випадки за допомогою загальнодоступного посилання для реєстрації

У цій статті адміністраторам показано, як створити запит (і одночасно додати користувача), надіславши одне посилання для реєстрації.

Пошук посилання

  1. Увійдіть в обліковий запис адміністратора.
  2. Знайдіть програму на сторінці програм і натисніть кнопку АДМІНІСТРАТОР.
  3. На панелі адміністратора натисніть вкладку меню «Налаштування».
  4. Переконайтеся, що вибрано вкладку підменю НАЛАШТУВАННЯ СИСТЕМИ.

Налаштування параметрів

Прокрутіть вниз до заголовка РЕЄСТРАЦІЯ КОРИСТУВАЧА.

Переконайтеся, що для поля «Дозволити реєстрацію» встановлено значення «Увімк.». (обов’язково) Якщо цю функцію ввімкнено, Case Active генерує унікальне посилання.

  1. Клацніть поле Auto Create Case і виберіть On. Відобразиться поле «Роль користувача справи».
  2. Виберіть роль. (Дивіться нижче)

Ролі користувачів Case

Є три ролі користувача case.

  • клієнтів (рекомендовано)
  • за замовчуванням
  • персонал

Надсилання посилання

Поділіться посиланням з одним або кількома людьми. Кожен користувач, який натискає посилання та завершує реєстрацію, запускає платформу для створення нового запиту на сторінці запитів і автоматичного призначення користувача у випадку.
