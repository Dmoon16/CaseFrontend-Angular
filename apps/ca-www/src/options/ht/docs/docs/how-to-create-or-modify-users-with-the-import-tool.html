Ki jan yo kreye oswa modifye itilizatè yo ak zouti enpòte

Atik sa a montre administratè yo ki jan yo kreye itilizatè yo ak zouti Import la.

Louvri zouti enpòte

Administratè yo ka itilize zouti Import pou yo telechaje nouvo itilizatè yo ak modifye dosye itilizatè ki egziste deja yo.

  1. Konekte nan kont admin la.
  2. Jwenn yon aplikasyon sou paj Aplikasyon an epi klike sou bouton ADMIN.
  3. Sou panèl la Admin, klike sou tab Entegrasyon an ak Lè sa a, ENPÒTATION sou submenu a.
  4. Klike sou bouton NOUVO. Bwat dyalòg Add Imports la parèt nan yon nouvo tab navigatè.

Kreye dosye

Apre bwat dyalòg Add Imports la louvri:

  1. Klike sou jaden Kalite a epi chwazi yon opsyon (gade anba a). Yon lyen ap parèt.
  2. Klike sou lyen Download Sample la. Yon dosye CSV telechaje sou aparèy ou an.

Apre sa, louvri dosye a epi ajoute dosye yo. Klike la a pou aprann kijan pou itilize done fòma nan dosye CSV la.

Kalite

Lè ou klike sou jaden Kalite a, yon meni dewoulman parèt. Chwazi youn nan de opsyon pou enpòtasyon en.

:exclamation: REMAK Meni dewoulman an gen anpil opsyon, men se sèlman de ki gen rapò ak zouti Import la. Chak opsyon gen yon diferan echantiyon dosye CSV.

Ajoute itilizatè yo

  • Chwazi opsyon sa a pou yon senp upload dosye itilizatè yo.
  • Klike sou lyen Download Sample pou ouvri yon fichye CSV ki gen tit kolòn pou kalite Add User Import.

Modifye itilizatè yo

  • Chwazi opsyon sa a pou mete ajou done pou itilizatè aktyèl yo.
  • Klike sou lyen Download Egzanp pou ouvri yon fichye CSV ki gen tit kolòn pou Modify User Import type.

Téléchargement fichye a

Ou ka telechaje yon fichye touswit oswa retounen nan bwat dyalòg Ajoute enpòtasyon an nan yon dat apre epi ranpli etap sa yo.

  1. Louvri bwat dyalòg Add Imports la.
  2. Antre yon non inik pou dosye CSV la.
  3. Klike sou ikòn plis (+) pou w telechaje fichye a nan aparèy ou an.
  4. Klike sou jaden Kalite a epi chwazi yon opsyon. (gade pi wo a)
  5. Klike sou SAVE.

img

Pwosesis

Tou depan de kantite dosye nan dosye a ak chaj sistèm, platfòm la ka pran plizyè minit pou trete enpòte a.

img

Lè enpòte a fini, platfòm la ajiste dosye itilizatè yo (sa vle di ajoute, modifye, oswa enfim) sou tablodbò Itilizatè admin la.

Echèk

Si operasyon enpòte a echwe, platfòm la retounen yon mesaj ki echwe. Nan pifò ka yo, kòz echèk la se fòma done yo.

Pou wè kòz echèk la:

  1. Klike sou View anba tit Aksyon an.
  2. Nan pwochen paj la, pase sourit la sou ikòn Echwe a. Ti konsèy la montre erè a.
  3. Ale nan dosye CVS la epi ajiste epi ranje erè a.
  4. Telechaje fichye a ankò.
