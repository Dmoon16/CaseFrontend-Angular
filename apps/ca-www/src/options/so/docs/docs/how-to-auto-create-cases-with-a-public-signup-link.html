Sida otomaatig ah loogu Sameeyo Kiisaska Xiriirinta Is-diiwaangelinta Dadweynaha

Maqaalkani wuxuu muujinayaa maamulayaasha sida loo abuuro kiis (oo ku dara isticmaalaha isku mar) adoo soo diraya hal xiriiriye.

Helitaanka isku xirka

  1. Gal koontada maamulka
  2. Ka hel app bogga Apps oo dhagsii badhanka ADMIN.
  3. On the admin panel, guji Settings menu tab.
  4. Hubi in tab menu-hoosaadka SYSTEM SETUP la doortay.

Dejinta dejinta

Hoos ugu dhaadhac ciwaanka USER SIGNUP.

Hubi in goobta Oggolow Is-diiwaangelinta ay daaran tahay. (loo baahan yahay) Marka sifadan la furo, Case Active waxa ay abuurtaa xidhiidh gaar ah.

  1. Guji goobta Keeska Auto Create oo dooro On. Goobta Doorka Isticmaalaha Kiiska ayaa soo bandhigaya.
  2. Dooro door (hoos eeg)

Doorarka isticmaalaha kiiska

Waxaa jira saddex door isticmaale kiis.

  • macaamiisha (lagu taliyay)
  • default
  • shaqaalaha

Diritaanka xiriirka

La wadaag xiriirka hal ama ka badan oo qof. Isticmaale kasta oo gujinaya isku xirka oo dhammaystiray is-diiwaangelinta waxa uu kiciyaa goobta si uu kiis cusub uga dhaliyo bogga kiisaska oo si toos ah ugu meeleeyo isticmaalaha kiiska.
