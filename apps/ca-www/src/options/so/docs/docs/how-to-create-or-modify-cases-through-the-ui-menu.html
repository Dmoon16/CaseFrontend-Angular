Sida loo abuuro ama wax looga beddelo Kiisaska iyada oo loo marayo Menu UI

Maqaalkani waxa uu muujinayaa admins sida loo abuuro kiis leh interface interface ee madal. Maamulayaashu waxay sidoo kale ku dari karaan isticmaalayaasha kiis cusub waxayna cusbooneysiin karaan kiisaska jira.

Abuuritaanka kiis

  1. Gal koontada maamulka
  2. Ka hel app bogga Apps oo dhagsii badhanka ADMIN.
  3. On the admin panel, guji kiisaska menu tab.
  4. Guji badhanka CUSUB Daaqad cusub ayaa soo bandhigaysa.

Buuxi meelaha loo baahan yahay oo guji SAVE.

img

Ku darida isticmaalayaasha

Ikhtiyaar ahaan, maamulayaashu waxay ku dari karaan isticmaalayaasha kiiska boggan. Hoos u deg oo guji badhanka ASSIGN Users .

img

Habka Isticmaalaha ku meelee, buuxi meelaha. Calaamadee sanduuqa si aad ugu soo dirto isticmaalaha ogeysiiska iimaylka. (ikhtiyaar)

Wax ka beddelka kiis

Si aad u cusboonaysiiso tiro yar oo kiisas ah, isticmaal qalabka wax ka beddelka oo kiis kasta ku samee.

  • Xusuusin Si aad u cusboonaysiiso tiro badan oo kiisas hal fadhi ah, tixgeli isticmaalka qalabka soo dejinta
  1. Gal koontada maamulka
  2. Ka hel app bogga Apps oo dhagsii badhanka ADMIN.
  3. On the admin panel, guji kiisaska menu tab.
  4. Hel kiis oo guji Edit.
  5. Isbeddel samee oo guji SAVE.
