Sida loo qoondeeyo, wax ka beddelo ama looga saaro Meelaynta Isticmaalaha ee Qalabka Soo Dejinta

Maqaalkani waxa uu tusayaa maamulayaasha sida loo maareeyo isticmaalayaasha kiisaska qalabka soo dejinta.

Maamulayaashu waxay awoodaan:

  • ku meeleey isticmaalayaasha kiis
  • wax ka beddel isticmaalayaasha hore loogu qoondeeyay kiis
  • ka saar isticmaalayaasha loo xilsaaray kiis

Furitaanka qalabka soo dejinta

  1. Gal koontada maamulka
  2. Ka hel app bogga Apps oo dhagsii badhanka ADMIN.
  3. Guji tab isdhexgalka ka dibna IMPORTS ee liiska hoosaadka.
  4. Guji badhanka CUSUB Sanduuqa wada hadalka ku dar Imports.

Samaynta diiwaannada

Soo deji hal ama kabadan faylal CSV Isticmaal faylashan si aad ugu darto, wax uga bedesho, ama uga saarto isticmaalayaasha kiiska. Kadib marka sanduuqa wada hadalka ku dar Imports uu furmo:

  1. Guji goobta Nooca
  2. Guji liiska hoos-u-hoosaadka oo dooro ikhtiyaar (hoos eeg). Isku xirka ayaa muujinaya.
  3. Guji isku xirka Tusaalaha soo dejinta. Faylka CSV wuxuu ku soo dejinayaa qalabkaaga.

Fur faylka oo ku dar diiwaannada. Riix halkan si aad u barato sida loo isticmaalo xogta qaabka faylka CSV

Xulashada faylka

Ku qor isticmaalayaasha kiisaska * Dooro ikhtiyaarkan si aad ugu darto isticmaalayaasha kiis. * Kaga tag qolka_door_dalabka meel bannaan. * Tag bogga Isticmaalayaasha oo koobi ka geli qofkan Tag ID. Haddii goobta aqoonsiga Tag ay bannaan tahay, geli aqoonsi gaar ah, dhagsii Save, ka dibna ku koobi aqoonsiga faylka CSV.

img

Wax ka beddel isticmaalayaasha loo qoondeeyay kiisaska * Dooro ikhtiyaarkan si aad u cusboonaysiiso xogta isticmaalayaasha hadda loo xilsaaray kiis. * U adeegso Aqoonsiga Case ID tiirka case_ID.

Ka saar isticmaalayaasha loo qoondeeyay kiisaska * Dooro ikhtiyaarkan si aad meesha uga saarto isticmaalayaasha loo qoondeeyay kiis.

Soo dejinta faylka

Waxaad isla markaaba geli kartaa fayl ama ku soo celi sanduuqa wada hadalka ku dar Imports taariikh dambe oo dhamaystir talaabooyinkan.

  1. Fur sanduuqa wada hadalka ku dar Imports
  2. Geli magac gaar ah faylka CSV
  3. Guji summada lagu daray (+) si aad faylka uga soo dejiso qalabkaaga.
  4. Guji goobta Nooca oo dooro ikhtiyaar. (kor eeg)
  5. Guji SAVE

Habaynta

Iyada oo ku xidhan tirada diiwaanada ee faylka iyo culayska nidaamka, madalku waxay qaadan kartaa dhawr daqiiqo si ay uga baaraandegto soo dejinta.

Marka soodejintu dhammaato, madalku waxay hagaajisaa diiwaanka isticmaalaha (sida ku dara, wax ka beddelo, ama ka saartaa) bogga kiisaska.

Fashilan

Haddii hawlgalka soo dejinta uu ku guuldareysto, madalku waxay soo celinaysaa fariin guuldareystay. Inta badan, sababta guul-darradu waa habaynta xogta.

img

Si loo arko sababta fashilka:

  1. Guji View hoostiisa cinwaanka Ficil
  2. Bogga xiga, dul bood mouse-ka summada Failed. Tilmaamaha qalabka ayaa muujinaya qaladka.
  3. Aad faylka CVS oo hagaaji oo hagaaji khaladka.
  4. Soo rar faylka mar kale
