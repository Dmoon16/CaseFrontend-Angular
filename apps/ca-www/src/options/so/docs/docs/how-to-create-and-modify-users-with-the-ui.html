Sida loo Abuuro oo wax looga beddelo Isticmaalayaasha UI

Maqaalkani waxa uu tusayaa maamulayaasha sida loo abuuro isticmaalayaasha oo loo cusboonaysiiyo diiwaanka is dhexgalka isticmaale ee madal. Maamulayaashu waxay sidoo kale ku dari karaan isticmaalayaasha kiis cusub oo ay wax ka beddelaan kiisaska jira.

Abuuritaanka isticmaale

  1. Gal koontada maamulka
  2. Ka hel app bogga Apps oo dhagsii badhanka ADMIN.
  3. On the admin panel, guji Users menu tab.
  4. Guji badhanka CUSUB Daaqad cusub ayaa soo bandhigaysa.

Buuxi meelaha oo guji SAVE. Case Active waxa ay ku daraa magaca liiska isticmaalayaasha la xidhiidha abkan.

Wax ka beddelka isticmaalaha

Si aad u cusboonaysiiso tiro yar oo diiwaanka isticmaalaha, isticmaal aalada Tafatirka oo isbedel ku samee isticmaale kasta.

  • Xusuusin Si aad wax uga beddesho tiro badan oo diiwaannada isticmaalaha hal fadhi, tixgeli isticmaalka qalabka soo dejinta .
  1. On the admin panel, guji Users menu tab.
  2. Hel isticmaale oo guji Edit.
  3. Isbeddel samee oo guji SAVE.

Tallaabooyinka xiga

Kiis Active uma soo diro isticmaalaha ogeysiis iimayl. Maamulayaashu waa inay la xiriiraan isticmaalaha oo ay u soo diraan isku xirka bogga guriga Case Active. Isticmaaluhu wuxuu markaa riixayaa badhanka LOGIN oo wuxuu dhammaystiraa socodka shaqada ee saxiixayaasha cusub.

  • Maqaalkani waxa uu sharaxayaa habka isdiiwaangelinta ee isticmaalayaasha cusub.
