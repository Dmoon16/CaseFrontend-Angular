Sida otomaatig ah loogu meeleeyo Isticmaalayaasha isku xidhka Is-diiwaangelinta Dadweynaha

Maamulayaashu waxay bartaan sida loo habeeyo nidaamka nidaamka si isticmaalayaasha cusub si toos ah loogu meeleeyo kiis cusub oo otomaatiga ah.

Furitaanka bogga habaynta

Tallaabada ugu horreysa waa in la furo bogga Setup System.

  1. Gal koontada maamulka
  2. Ka hel app bogga Apps oo dhagsii badhanka ADMIN.
  3. Guji tab Nidaamyada ka dibna SYSTEM SETUP ee liiska hoosaadka.
  4. Hoos ugu dhaadhac ciwaanka USER SIGNUP.

Dejinta habaynta nidaamka

  1. Deji Oggol Saxiixa iyo Si Atoomaatig ah u Samee liiska hoos-u-dhaca ee menu si uu shido.
  2. Guji liiska doorka oo dooro ikhtiyaar (default waa lagu talinayaa).
  3. Nuqul ka samee oo wadaag isku xirka Isticmaalaha Dadweynaha.

Diritaanka xiriirka

Isticmaale kasta oo gujinaya isku xirka oo dhammaystiray is-diiwaangelinta ayaa kicisa goobta si uu kiis cusub uga soo saaro bogga Kiisas oo si toos ah ugu meeleeyaa isticmaalaha kiiskaas.
