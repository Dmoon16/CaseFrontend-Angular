Kako administratori upravljaju korisničkim dodjelama u slučajevima

Postoji pet načina za upravljanje korisnicima i dodjeljivanjem predmeta. Kliknite na link da saznate kako.

Metoda Tip Akcije Detalji
Prilikom kreiranja slučaja UI dodati Dugme Administrator > Predmeti > Dugme Novo > Dugme Dodijeli korisnike
Automatski dodijelite korisnika koristeći javni link za registraciju Akcija korisnika dodati Dugme Administrator > Postavke > Stranica za podešavanje sistema
Korištenje korisničkog sučelja na postojećem slučaju UI dodati i ukloniti Administratorsko dugme > Predmeti > Dodeli korisnike
Alat za uvoz UI dodati, modificirati i ukloniti Administratorsko dugme > Integracije > Uvozi
Korištenje API-ja (uskoro) API dodati, modificirati i ukloniti N / A
