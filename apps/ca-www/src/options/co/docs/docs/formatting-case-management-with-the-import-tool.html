Formating Case Management Data cù l'Import Tool

Quì sottu hè una lista di campi richiesti in i fugliali CSV per gestisce i casi cù u strumentu Import.

nome d'utilizatore

  • Inserite l'indirizzu email di l'utilizatore.

tag_id (un id chì pudete riferite dopu)

  • un identificatore unicu (utilizate caratteri alfanumerici, i trattini sò accettati)

casu_statu

  • 0 = chjusu
  • 1 = attivu

data_apertura

  • mese/ghjornu/annu
  • mm/gg/aaaa

circa

  • una descrizzione unica di u casu (utilizate caratteri alfanumerici, i trattini sò accettati)
