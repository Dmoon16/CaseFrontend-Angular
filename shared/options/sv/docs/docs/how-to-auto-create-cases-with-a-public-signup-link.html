Hur man skapar ärenden automatiskt med en offentlig registreringslänk

Den här artikeln visar administratörer hur man skapar ett ärende (och lägger till en användare samtidigt) genom att skicka en enda registreringslänk.

Hitta länken

  1. Logga in på administratörskontot.
  2. Leta reda på en app på appsidan och klicka på ADMIN-knappen.
  3. På adminpanelen klickar du på fliken Inställningar.
  4. Se till att undermenyfliken SYSTEMINSTÄLLNING är vald.

img

Konfigurera inställningar

Scrolla ner till rubriken ANVÄNDARANMÄLAN.

Se till att fältet Tillåt registrering är inställt på På. (obligatoriskt) När den här funktionen är aktiverad genererar Case Active en unik länk.

  1. Klicka på fältet Skapa ärende automatiskt och välj På. Fältet Case User Roll visas.
  2. Välj en roll. (se nedan)

Case användarroller

Det finns tre fallanvändarroller.

  • kunder (rekommenderas)
  • standard
  • personal

Skickar länken

Dela länken med en eller flera personer. Varje användare som klickar på länken och slutför registreringen utlöser plattformen för att generera ett nytt ärende på ärendesidan och automatiskt tilldela användaren i ärendet.
