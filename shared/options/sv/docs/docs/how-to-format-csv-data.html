Hur man formaterar CSV-data

Nedan finns en lista över datafält i CSV-filer. Varje importalternativ har olika kombination av fält.

Användarnamn

  • Ange användarens e-postadress.

förnamn

  • Ange användarens förnamn.

efternamn

  • Ange användarens efternamn.

plats

  • Denna kolumn söker efter användarens föredragna språk.
  • Ange en för engelska, sp för spanska.

zoninfo

  • Detta ställer in användarens tidszon. Gå in på kontinenten och staden utan mellanslag (använd ett understreck).
  • Måste använda ett alternativ i rullgardinsmenyn Profil > SPRÅK & REGION > Tidszon.
  • Använd den tidszon som passar bäst. Boston finns till exempel inte på rullgardinsmenyn, så vi kan använda New York (eller någon stad i EST/EDT-tidszonen).

host_tag_id

  • Ange en permanent etikett.
  • az, 0-9, -, / är tillåtna.

case_status

  • 0 = stängd
  • 1 = aktiv
