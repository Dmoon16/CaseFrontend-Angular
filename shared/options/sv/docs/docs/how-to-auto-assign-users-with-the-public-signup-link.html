Hur man automatiskt tilldelar användare med den offentliga registreringslänken

Administratörer lär sig hur man konfigurerar systeminställningar så att nya användare automatiskt tilldelas ett nyligen automatiskt skapat ärende.

Öppnar inställningssidan

Det första steget är att öppna sidan Systeminställningar.

  1. Logga in på administratörskontot.
  2. Leta upp en app på appsidan och klicka på ADMIN-knappen.
  3. Klicka på fliken System och sedan SYSTEMINSTÄLLNINGAR på undermenyn.
  4. Scrolla ner till rubriken ANVÄNDARANMÄLAN.

Konfigurera systeminställningar

  1. Ställ in rullgardinsmenyerna Tillåt registrering och Skapa ärende automatiskt till På.
  2. Klicka på rollmenyn och välj ett alternativ (standard rekommenderas).
  3. Kopiera och dela länken Public User Signup.

Skickar länken

Varje användare som klickar på länken och slutför registreringen utlöser plattformen att generera ett nytt ärende på sidan Ärende och automatiskt tilldelar användaren i det fallet.
