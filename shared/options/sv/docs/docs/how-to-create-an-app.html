Hur man skapar en app

Den här artikeln visar nya användare hur man skapar ett gratis testkonto. Case Active ger automatiskt den nya användarens administratörsbehörighet för detta konto.

Skapa ett konto

  1. Gå till inloggningssidan .
  2. Klicka på knappen Kom igång.
  3. Ange en e-postadress.
  4. Klicka på Nästa.

Verifierar ett konto

  1. Öppna en ny webbläsarflik och logga in på ditt e-postkonto.
  2. Kopiera den sexsiffriga koden.
  3. Återgå till fliken Case Active.
  4. Klistra in koden i fälten. Ett nytt fönster visas.

I det nya fönstret:

  1. Fyll i de obligatoriska fälten.
  2. Läs användarvillkoren och markera rutan för att godkänna.
  3. Klicka på SKAPA ETT GRATIS KONTO.

img

Slutför appinstallationen

Ett arbetsflöde visas på nästa sida. Slutför stegen för att slutföra konfigureringen av din app. Du kan alltid gå tillbaka till dessa sidor för att redigera eller uppdatera information eller din kontostatus.

På sidan Om appen fyller du i fälten för företagsnamn och webbplats (obligatoriskt) och klickar sedan på Nästa.

På sidan Faktureringsinformation klickar du på fältet Betalningsplan och granskar alternativen. Standardinställningen är Gratis. Klicka på SKICK.

img

När du har skickat in formuläret visar Case Active appsidan. För varje app du skapar eller tillhör visar sidan:

  1. Appens namn
  2. Din typ (användarroll)
  3. Status

img

Appsidan är en startpunkt för många administratörsfunktioner:

  1. Klicka på APP-knappen för att kommunicera, fylla i dokument, videochatta (och mer) om ärenderelaterade frågor.
  2. Klicka på ADMIN-knappen för att öppna adminpanelen. Det är här administratörer hanterar användare, ärenden, integrationer och mer.
  3. Klicka på ellipsen för att hantera kontofakturerings- och aviseringsinställningar.
  • Obs Case Active visar ADMIN-knappen för adminanvändare. Användare som inte har administratörsbehörighet ser inte ADMIN-knappen (som visas på skärmen nedan).
