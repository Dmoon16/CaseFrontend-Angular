Hur man skapar eller ändrar användare med importverktyget

Den här artikeln visar administratörer hur man skapar användare med importverktyget.

Öppnar importverktyget

Administratörer kan använda importverktyget för att massöverföra nya användare och ändra befintliga användarposter.

  1. Logga in på administratörskontot.
  2. Leta upp en app på appsidan och klicka på ADMIN-knappen.
  3. På adminpanelen klickar du på fliken Integration och sedan på IMPORTER på undermenyn.
  4. Klicka på knappen NY. Dialogrutan Lägg till importer visas på en ny webbläsarflik.

img

Skapar poster

När dialogrutan Lägg till importer öppnas:

  1. Klicka på fältet Typ och välj ett alternativ (se nedan). En länk visas.
  2. Klicka på länken Ladda ner exempel . En CSV-fil laddas ner till din enhet.

Öppna sedan filen och lägg till poster. Klicka här för att lära dig hur du använder formatdata i CSV-filen.

Typ

När du klickar på fältet Typ visas en rullgardinsmeny. Välj ett av två alternativ för massimport.

:utrop: OBS Rullgardinsmenyn har många alternativ, men bara två är relaterade till importverktyget. Varje alternativ har olika exempel på CSV-fil.

Lägg till användare

  • Välj det här alternativet för en enkel massuppladdning av användarposter.
  • Klicka på länken Ladda ner exempel för att öppna en CSV-fil med kolumnrubriker för importtypen Lägg till användare.

Ändra användare

  • Välj det här alternativet för att uppdatera data för nuvarande användare.
  • Klicka på länken Ladda ner exempel för att öppna en CSV-fil med kolumnrubriker för typen Ändra användares import.

Laddar upp filen

Du kan ladda upp en fil direkt eller gå tillbaka till dialogrutan Lägg till importer vid ett senare tillfälle och slutföra dessa steg.

  1. Öppna dialogrutan Lägg till importer.
  2. Ange ett unikt namn för CSV-filen.
  3. Klicka på plusikonen (+) för att ladda upp filen från din enhet.
  4. Klicka på fältet Typ och välj ett alternativ. (se ovan)
  5. Klicka på SPARA.

img

Bearbetning

Beroende på antalet poster i filen och systemladdningen kan plattformen ta flera minuter att bearbeta importen.

img

När importen är klar justerar plattformen användarposter (dvs. lägger till, modifierar eller inaktiverar) på administratörens användarpanel .

Fel

Om importen misslyckas returnerar plattformen ett misslyckat meddelande. I de flesta fall är orsaken till felet dataformatering.

img

Så här ser du orsaken till felet:

  1. Klicka på Visa under rubriken Åtgärd.
  2. På nästa sida håller du musen över ikonen Misslyckades. Verktygstipset visar felet.
  3. Gå till CVS-filen och justera och åtgärda felet.
  4. Ladda upp filen igen.
