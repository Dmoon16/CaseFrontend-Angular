Hur man loggar in

Lär dig hur du loggar in på ditt befintliga Case Active-konto.

Loggar in

  1. Gå till inloggningssidan .
  2. Klicka på knappen LOGGA IN.
  3. Skriv in din e-postadress.
  4. Klicka på NÄSTA.

Verifierar ett konto

  1. Öppna en ny webbläsarflik och logga in på ditt e-postkonto.
  2. Kopiera den sexsiffriga koden.
  3. Återgå till fliken Case Active.
  4. Klistra in koden i fälten. Ett nytt fönster visas.

Case Active visar appsidan.

  • Obs! Plattformen visar en ADMIN-knapp bredvid APP-knappen för användare med administratörsbehörighet.
