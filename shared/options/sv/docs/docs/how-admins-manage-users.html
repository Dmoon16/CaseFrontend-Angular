Hur administratörer hanterar användare

Administratörer skapar, ändrar eller inaktiverar vem som kan komma åt appar och fall. När en användare loggar in på plattformen för första gången måste de slutföra ett profilarbetsflöde.

  • Klicka här för att lära dig mer om det nya arbetsflödet för användarprofiler.

Hantera en användare

Tabellen nedan visar fem sätt att hantera en användare. Klicka på en länk för att lära dig hur.

Metod Typ Insatser Detaljer
Använder UI UI lägga till & ändra Admin-knapp > Användare
När du skapar ett ärende och tilldelar användare till ärende UI Lägg till Admin-knapp > Ärenden > Ny knapp > Skapa ärendesida
Skapa användare automatiskt med en offentlig registreringslänk Användaråtgärd Lägg till Admin-knapp > Inställningar > Systeminställningar
Importverktyg UI lägga till & ändra Admin-knapp > Integrationer > Importer
Använder API (kommer snart) API lägga till & ändra N/A

Aviseringar

Plattformen skickar inte användarna ett meddelande när de läggs till i en app. Administratörer kan dock aktivera aviseringar för ärendetilldelningar.
