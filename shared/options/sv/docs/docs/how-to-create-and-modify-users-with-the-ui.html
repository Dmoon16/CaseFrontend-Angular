Hur man skapar och ändrar användare med användargränssnittet

Den här artikeln visar administratörer hur man skapar användare och uppdaterar poster med plattformens användargränssnitt. Administratörer kan också lägga till användare i ett nytt ärende och redigera befintliga ärenden.

Skapa en användare

  1. Logga in på administratörskontot.
  2. Leta upp en app på appsidan och klicka på ADMIN-knappen.
  3. På adminpanelen klickar du på fliken Användare.
  4. Klicka på knappen NY. Ett nytt fönster visas.

Fyll i fälten och klicka på SPARA. Case Active lägger till namnet i listan över användare som är kopplade till den här appen.

img

Ändra en användare

För att uppdatera ett litet antal användarposter, använd redigeringsverktyget och gör ändringar för varje enskild användare.

  • Obs! Om du vill ändra ett stort antal användarposter på en gång, överväg att använda importverktyget .
  1. På adminpanelen klickar du på fliken Användare.
  2. Hitta en användare och klicka på Redigera.
  3. Gör ändringar och klicka på SPARA.

img

Nästa steg

Case Active skickar inte ett e-postmeddelande till användaren. Administratörer bör kontakta användaren och skicka en länk till Case Active-hemsidan. Användaren klickar sedan på LOGGA IN-knappen och slutför arbetsflödet för nya registreringar.

  • Den här artikeln beskriver registreringsarbetsflödet för nya användare.
