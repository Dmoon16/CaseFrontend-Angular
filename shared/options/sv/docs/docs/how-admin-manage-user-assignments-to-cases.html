Hur administratörer hanterar användartilldelningar till ärenden

Det finns fem sätt att hantera användare och ärendetilldelningar. Klicka på en länk för att lära dig hur.

Metod Typ Insatser Detaljer
När du skapar ett ärende UI Lägg till Admin-knapp > Fall > Ny knapp > Tilldela användare
Tilldela användare automatiskt med en offentlig registreringslänk Användaråtgärd Lägg till Admin-knapp > Inställningar > Systeminställningar
Använder UI på befintliga fall UI Lägg till ta bort Admin-knapp > Fall > Tilldela användare
Importverktyg UI lägga till, ändra och ta bort Admin-knapp > Integrationer > Importer
Använder API (kommer snart) API lägga till, ändra och ta bort N/A
