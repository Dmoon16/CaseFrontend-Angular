يوزر پروفائل کي ڪيئن اپڊيٽ ڪجي

هي آرٽيڪل ڏيکاري ٿو صارفين کي ڪيئن تازه ڪاري ڪجي انهن جي پروفائل صفحي تي معلومات.

  • استعمال ڪندڙ پنھنجي پروفائل کي اپڊيٽ ڪري سگھن ٿا.
  • منتظمين ڪنهن به صارف جي پروفائل صفحي کي اپڊيٽ ڪري سگھن ٿا.

صارفين لاء

پهريون قدم پروفائل صفحي کي کولڻ آھي. صارفين ۽ منتظمين انهن قدمن تي عمل ڪريو انهن جي پنهنجي پروفائل صفحي کي اپڊيٽ ڪرڻ لاء.

  1. پنهنجي ڪيس ايڪٽو اڪائونٽ ۾ لاگ ان ٿيو.
  2. صفحي جي چوٽي تي پروفائل ٽيب تي ڪلڪ ڪريو.
  3. تبديليون آڻيو.
  4. ڪلڪ محفوظ ڪريو.

گھڻا شعبا اختياري آھن، ۽ ڪجھ فيلڊ گھربل آھن. گهربل فيلڊ آهن:

  • پهريون نالو
  • آخري نالو
  • اي ميل پتو
  • ٻولي
  • ٽائيم زون

منتظمين لاءِ

منتظمين پڻ ٻين استعمال ڪندڙن لاءِ پروفائل جي معلومات کي اپڊيٽ ڪري سگھن ٿا. ھن ۾ HOST USER INFO عنوان ھيٺ ٽي شعبا شامل آھن.

  1. پنهنجي Case Active ايڊمن اڪائونٽ ۾ لاگ ان ٿيو.
  2. هڪ ايپ ڳوليو ۽ ڪلڪ ڪريو ADMIN بٽڻ.
  3. ڪلڪ ڪريو صارفين جي مٿين مينيو تي.
  4. ذيلي مينيو ڏانھن وڃو ۽ يوزر لسٽ چونڊيو جنھن کي توھان رسائي ڪرڻ چاھيو ٿا (فعال يا غير فعال). ڊفالٽ سيٽنگ فعال آهي.
  5. لسٽ تي استعمال ڪندڙ کي ڳولھيو ۽ ڪلڪ ڪريو تبديلي آڻيو.
  6. ضرورت مطابق تبديليون ڪريو. (هيٺ ڏسو)
  7. ڪلڪ محفوظ ڪريو.

تبديل ڪريو استعمال ڪندڙ جي معلومات

منتظمين صارف جي معلومات کي تبديل ڪري سگھن ٿا.

  1. استعمال ڪندڙ جو قسم مقرر ڪريو (يوزر يا مئنيجر).
  2. يوزر اسٽيٽس سيٽ ڪريو (فعال کان معذور).
  3. هڪ منفرد ٽيگ ID داخل ڪريو. (اختياري)
  4. ڪلڪ محفوظ ڪريو.
