درآمدي اوزار سان صارفين کي ڪيئن ٺاهيو يا تبديل ڪيو وڃي

هي آرٽيڪل ڏيکاري ٿو منتظمين کي ڪيئن ٺاهيو صارفين کي درآمد واري اوزار سان.

درآمد وارو اوزار کولڻ

منتظم استعمال ڪري سگھن ٿا درآمد وارو اوزار بلڪ اپلوڊ ڪرڻ لاءِ نوان استعمال ڪندڙ ۽ موجوده صارف رڪارڊ کي تبديل ڪرڻ لاءِ.

  1. منتظم کاتي ۾ لاگ ان ٿيو.
  2. ايپس پيج تي ائپ ڳولھيو ۽ ايڊمن بٽڻ تي ڪلڪ ڪريو.
  3. ايڊمن پينل تي، انٽيگريشن ٽيب تي ڪلڪ ڪريو ۽ پوءِ سب مينيو تي امپورٽس.
  4. NEW بٽڻ تي ڪلڪ ڪريو. شامل ڪريو درآمدات ڊائلاگ باڪس نئين برائوزر ٽيب ۾ ڏيکاري ٿو.

img

رڪارڊ ٺاهڻ

ان کان پوءِ شامل ڪريو درآمدات ڊائلاگ باڪس کلي ٿو:

  1. ٽائپ فيلڊ تي ڪلڪ ڪريو ۽ اختيار چونڊيو (هيٺ ڏسو). هڪ لنڪ ڏيکاري ٿو.
  2. ڊائون لوڊ نموني لنڪ تي ڪلڪ ڪريو. هڪ CSV فائل توهان جي ڊوائس تي ڊائون لوڊ.

اڳيون، فائل کوليو ۽ رڪارڊ شامل ڪريو. CSV فائل ۾ فارميٽ ڊيٽا ڪيئن استعمال ڪجي سکڻ لاءِ هتي ڪلڪ ڪريو.

img

قسم

جڏهن توهان ٽائپ فيلڊ تي ڪلڪ ڪريو، هڪ ڊراپ-ڊائون مينيو ڏيکاري ٿو. بلڪ درآمد لاءِ ٻن اختيارن مان ھڪڙو چونڊيو.

: exclamation: نوٽ ڊراپ-ڊائون مينيو ۾ ڪيترائي آپشن آھن، پر صرف ٻه آھن درآمدي ٽول سان لاڳاپيل. هر اختيار ۾ هڪ مختلف نموني CSV فائل آهي.

استعمال ڪندڙ شامل ڪريو

  • هي اختيار چونڊيو صارف جي رڪارڊ جي سادي بلڪ اپلوڊ لاءِ.
  • ڪلڪ ڪريو ڊائون لوڊ نمونو لنڪ کولڻ لاءِ CSV فائل کولڻ لاءِ ڪالمن جي عنوانن سان شامل ڪريو استعمال ڪندڙن جي درآمد جي قسم لاءِ.

صارفين کي تبديل ڪريو

  • موجوده استعمال ڪندڙن لاءِ ڊيٽا کي اپڊيٽ ڪرڻ لاءِ هي اختيار چونڊيو.
  • ڪلڪ ڪريو ڊائون لوڊ نمونو لنڪ کولڻ لاءِ CSV فائل کولڻ لاءِ ڪالمن جي عنوانن سان تبديل ڪريو صارفين جي درآمد جي قسم لاءِ.

فائل اپ لوڊ ڪندي

توھان فوري طور تي ھڪڙي فائل اپلوڊ ڪري سگھو ٿا يا واپسي ڪري سگھوٿا شامل ڪريو درآمدات ڊائلاگ باڪس بعد جي تاريخ ۾ ۽ انھن مرحلن کي مڪمل ڪريو.

  1. درآمدات شامل ڪريو ڊائلاگ باڪس کوليو.
  2. CSV فائل لاءِ منفرد نالو داخل ڪريو.
  3. پنهنجي ڊوائيس تان فائل اپلوڊ ڪرڻ لاءِ پلس (+) آئڪن تي ڪلڪ ڪريو.
  4. ٽائپ فيلڊ تي ڪلڪ ڪريو ۽ اختيار چونڊيو. (مٿي ڏسو)
  5. ڪلڪ ڪريو محفوظ ڪريو.

پروسيسنگ

فائل ۽ سسٽم جي لوڊ ۾ رڪارڊ جي تعداد تي مدار رکندي، پليٽ فارم درآمد کي عمل ڪرڻ لاء ڪيترائي منٽ وٺي سگھي ٿو.

img

جڏهن درآمد مڪمل ٿئي ٿي، پليٽ فارم صارف جي رڪارڊ کي ترتيب ڏئي ٿو (يعني شامل ڪري ٿو، تبديل ڪري ٿو، يا غير فعال ڪري ٿو) منتظم جي صارفين جي ڊيش بورڊ تي.

ناڪامي

جيڪڏهن درآمد آپريشن ناڪام ٿئي ٿي، پليٽ فارم هڪ ناڪام پيغام موٽائي ٿو. اڪثر ڪيسن ۾، ناڪامي جو سبب ڊيٽا فارميٽنگ آهي.

img

ناڪامي جو سبب ڏسڻ لاء:

  1. ڪلڪ ڪريو ڏسو عمل جي عنوان هيٺ.
  2. ايندڙ صفحي تي، مائوس کي هور ڪريو ناڪامي آئڪن تي. ٽول ٽائپ غلطي ڏيکاري ٿو.
  3. CVS فائل ڏانھن وڃو ۽ ترتيب ڏيو ۽ غلطي کي درست ڪريو.
  4. فائل کي ٻيهر لوڊ ڪريو.
