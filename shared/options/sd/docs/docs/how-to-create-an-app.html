هڪ ايپ ڪيئن ٺاهيو

هي آرٽيڪل ڏيکاري ٿو نون صارفين کي ڪيئن ٺاهيو مفت آزمائشي اڪائونٽ. Case Active پاڻمرادو ڏئي ٿو نئين يوزر کي ايڊمن استحقاق هن اڪائونٽ لاءِ.

اڪائونٽ ٺاهڻ

  1. لاگ ان صفحي ڏانھن وڃو.
  2. شروع ڪريو بٽڻ تي ڪلڪ ڪريو.
  3. اي ميل پتو داخل ڪريو.
  4. ڪلڪ اڳيون.

اڪائونٽ جي تصديق ڪندي

  1. هڪ نئون برائوزر ٽيب کوليو ۽ پنهنجي اي ميل اڪائونٽ ۾ لاگ ان ٿيو.
  2. ڇهه عددي ڪوڊ نقل ڪريو.
  3. Case Active ٽيب ڏانھن واپس وڃو.
  4. ڪوڊ کي فيلڊ ۾ پيسٽ ڪريو. هڪ نئين ونڊو ڏيکاري ٿي.

نئين ونڊو ۾:

  1. گھربل فيلڊ ۾ ڀريو.
  2. خدمت جون شرطون پڙهو ۽ دٻي کي چيڪ ڪريو متفق ٿيڻ لاءِ.
  3. ڪلڪ ڪريو هڪ مفت اڪائونٽ ٺاهيو.

ايپ سيٽ اپ مڪمل ڪرڻ

هڪ ڪم فلو ايندڙ صفحي تي ڏيکاري ٿو. پنھنجي ايپ کي سيٽ ڪرڻ لاءِ قدم مڪمل ڪريو. توھان ھميشه انھن صفحن تي واپس ڪري سگھوٿا معلومات يا پنھنجي کاتي جي حالت کي ايڊٽ ڪرڻ يا تازه ڪاري ڪرڻ لاءِ.

ايپ بابت صفحي تي، ڪمپني جو نالو ۽ ويب سائيٽ جي فيلڊ ۾ ڀريو (گهربل) ۽ پوء ڪلڪ ڪريو اڳيون.

img

بلنگ جي معلومات واري صفحي تي، ادائگي جي منصوبي جي فيلڊ تي ڪلڪ ڪريو ۽ اختيارن جو جائزو وٺو. ڊفالٽ سيٽنگ مفت آهي. SUBMIT تي ڪلڪ ڪريو.

فارم جمع ڪرائڻ کان پوءِ، Case Active ڏيکاري ٿو ائپس جو صفحو. هر ايپ لاءِ جيڪو توهان ٺاهيو يا ان سان واسطو رکي، صفحو ڏيکاري ٿو:

  1. ايپ جو نالو
  2. توهان جو قسم (استعمال ڪندڙ ڪردار)
  3. حيثيت

ائپس جو صفحو ڪيترن ئي منتظم جي ڪمن لاءِ لانچنگ پوائنٽ آھي:

  1. APP بٽڻ تي ڪلڪ ڪريو گفتگو ڪرڻ لاءِ، دستاويز ڀريو، وڊيو چيٽ (۽ وڌيڪ) ڪيس سان لاڳاپيل مسئلن تي.
  2. منتظم پينل کولڻ لاءِ ADMIN بٽڻ تي ڪلڪ ڪريو. هي آهي جتي منتظمين صارفين، ڪيس، انضمام، ۽ وڌيڪ انتظام ڪن ٿا.
  3. اڪائونٽ بلنگ ۽ نوٽيفڪيشن سيٽنگون منظم ڪرڻ لاءِ ايلپسس تي ڪلڪ ڪريو.
  • نوٽ Case Active منتظم استعمال ڪندڙن لاءِ ADMIN بٽڻ ڏيکاري ٿو. اهي صارف جن وٽ انتظامي استحقاق نه آهي اهي ADMIN بٽڻ نه ڏسندا (جيئن هيٺ ڏنل اسڪرين ۾ ڏيکاريل آهي).
