عوامي سائن اپ لنڪ سان ڪيس ڪيئن ٺاهيا وڃن

هي آرٽيڪل ڏيکاري ٿو منتظمين کي ڪيئن ٺاهيو ڪيس ٺاهيو (۽ ساڳئي وقت هڪ صارف شامل ڪريو) هڪ واحد سائن اپ لنڪ موڪلڻ سان.

لنڪ ڳولڻ

  1. منتظم کاتي ۾ لاگ ان ٿيو.
  2. ايپس پيج تي ائپ ڳولھيو ۽ ايڊمن بٽڻ تي ڪلڪ ڪريو.
  3. منتظم پينل تي، ڪلڪ ڪريو سيٽنگون مينيو ٽيب.
  4. پڪ ڪريو ته سسٽم سيٽ اپ ذيلي مينيو ٽيب چونڊيو ويو آهي.

  1. ڪلڪ ڪريو خودڪار ٺاهيو ڪيس فيلڊ ۽ چونڊيو آن. ڪيس يوزر رول فيلڊ ڏيکاري ٿو.
  2. ڪردار چونڊيو. (هيٺ ڏسو)

لنڪ موڪلڻ

ھڪڙي يا وڌيڪ ماڻھن سان لنڪ شيئر ڪريو. هر صارف جيڪو لنڪ تي ڪلڪ ڪري ٿو ۽ سائن اپ مڪمل ڪري ٿو پليٽ فارم کي ٽرگر ڪري ٿو هڪ نئون ڪيس پيدا ڪرڻ لاءِ ڪيس جي صفحي تي ۽ خودڪار طور تي صارف کي ڪيس ۾ تفويض ڪري ٿو.
