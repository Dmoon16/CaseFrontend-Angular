موجوده ڪيسن سان صارف جي اسائنمنٽس کي ڪيئن تفويض يا ختم ڪيو وڃي

هي آرٽيڪل ڏيکاري ٿو منتظمين کي ڪيئن تفويض ڪرڻ ۽ ڪيس استعمال ڪندڙن کي هٽائڻ. اسان توهان کي ڏيکاريون ٿا ته صارف ڊيٽا کي ڪيئن تبديل ڪجي.

هڪ موجوده ڪيس کولڻ

  1. منتظم کاتي ۾ لاگ ان ٿيو.
  2. ايپس پيج تي ائپ ڳولھيو ۽ ايڊمن بٽڻ تي ڪلڪ ڪريو.
  3. ڪلڪ ڪريو ڪيس مينيو ٽيب.
  4. پڪ ڪريو ته ACTIVE ذيلي مينيو ۾ چونڊيو ويو آهي.
  5. ھيٺ لٿو ACTION ڪالمن ڏانھن.

مقرر ڪريو يا ختم ڪريو

ڪلڪ ڪريو صارفين کي تفويض ڪرڻ لاءِ صارف کي ڪيس ۾ تفويض ڪرڻ يا صارف کي ڪيس مان هٽايو (پر پليٽ فارم نه). ايندڙ صفحي تي: ADD تي ڪلڪ ڪريو ۽ ڪم فلو مڪمل ڪريو. ڪلڪ ڪريو هٽايو صارف کي ختم ڪرڻ لاء.

img

ترميم ڪريو

ڪيسن جي صفحي تي صارف جي ڊيٽا کي تبديل ڪرڻ يا تازه ڪاري ڪرڻ ممڪن ناهي. ان جي بدران، انهن قدمن تي عمل ڪريو:

  1. صارفين مينيو ٽيب تي ڪلڪ ڪريو.
  2. استعمال ڪندڙ جو نالو ڳولھيو ۽ ڪلڪ ڪريو تبديلي آڻيو .
  3. تبديليون ڪريو ۽ محفوظ ڪريو تي ڪلڪ ڪريو.
