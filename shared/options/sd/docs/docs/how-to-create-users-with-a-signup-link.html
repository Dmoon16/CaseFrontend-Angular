سائن اپ لنڪ سان صارفين کي ڪيئن ٺاهيو

هي آرٽيڪل ڏيکاري ٿو منتظمين کي ڪيئن ٺاهيو صارفين کي سائن اپ لنڪ موڪلڻ سان.

سائن اپ لنڪ کي ترتيب ڏيڻ

  1. ڪلڪ ڪريو انھن کي سائن اپ لنڪ موڪليو صارف شامل ڪريو ڊائلاگ باڪس جي ھيٺان. ھڪڙو نئون صفحو ڏيکاري ٿو.
  2. سائن اپ اختيارن کي ترتيب ڏيو. (هيٺ ڏسو)

img

پڪ ڪريو ته سائن اپ فيلڊ کي سيٽ ڪرڻ جي اجازت ڏيو آن. (گهربل) جڏهن فعال ٿئي ٿي، Case Active هڪ منفرد لنڪ ٺاهي ٿو.

ھيٺ ڏنل اختيارن جو جائزو وٺو ۽ پوءِ لنڪ کي نقل ڪرڻ لاءِ آئڪن تي ڪلڪ ڪريو. توهان لنڪ کي پنهنجي ويب سائيٽ تي رکي سگهو ٿا يا لنڪ موڪلي سگهو ٿا هڪ يا وڌيڪ ماڻهن کي اي ميل يا ڪنهن ڪميونيڪيشن چينل ذريعي.

img

ڪيس ٺاهڻ

اختياري طور تي، توهان خودڪار طور تي هر نئين صارف لاء هڪ ڪيس ٺاهي سگهو ٿا ۽ هڪ صارف جو ڪردار تفويض ڪري سگهو ٿا.

  1. ڪلڪ ڪريو خودڪار ٺاهيو ڪيس فيلڊ ۽ چونڊيو آن. ڪيس يوزر رول فيلڊ ڏيکاري ٿو.
  2. ڪيس يوزر رول تي ڪلڪ ڪريو ۽ اختيار چونڊيو. (هيٺ ڏسو)

لنڪ موڪلڻ

ھڪڙي يا وڌيڪ ماڻھن سان لنڪ شيئر ڪريو. هر ماڻهو لنڪ تي ڪلڪ ڪندو آهي ۽ ڪم فلو مڪمل ڪري ٿو پنهنجي ڪيس ايڪٽو اڪائونٽ کي چالو ڪرڻ لاءِ. هڪ دفعو صارف سائن اپ ورڪ فلو مڪمل ڪري ٿو، ڪيس ايڪٽو صارف کي صارفين جي فهرست ۾ شامل ڪري ٿو.

img

ايندڙ قدم

Case Active صارف کي سائن اپ لنڪ سان اي ميل نوٽيفڪيشن موڪلي ٿو. صارف لنڪ تي ڪلڪ ڪري ٿو ۽ نئين سائن اپ لاء ڪم فلو مڪمل ڪري ٿو.

  • هي آرٽيڪل بيان ڪري ٿو سائن اپ ورڪ فلو نون استعمال ڪندڙن لاءِ.
