Cara Menetapkan, Mengubah, atau Menghapus Tugas Pengguna dengan Alat Impor

Artikel ini menunjukkan kepada admin cara mengelola pengguna kasus dengan alat Impor.

Admin dapat:

  • menetapkan pengguna ke suatu kasus
  • memodifikasi pengguna yang sudah ditugaskan ke suatu kasus
  • menghapus pengguna yang ditetapkan untuk suatu kasus

Membuka alat Impor

  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Klik tab Integrasi lalu IMPOR pada submenu.
  4. Klik tombol BARU. Kotak dialog Tambah Impor ditampilkan.

Membuat rekaman

Unduh satu atau lebih file CSV. Gunakan file ini untuk menambahkan, mengubah, atau menghapus pengguna kasus. Setelah kotak dialog Tambah Impor terbuka:

  1. Klik bidang Jenis.
  2. Gulir menu tarik-turun dan pilih opsi (lihat di bawah). Tautan ditampilkan.
  3. Klik tautan Unduh Sampel. File CSV diunduh ke perangkat Anda.

Buka file dan tambahkan catatan. Klik di sini untuk mempelajari cara menggunakan format data dalam file CSV.

Opsi berkas

Mengunggah file

Anda dapat langsung mengunggah file atau kembali ke kotak dialog Tambahkan Impor di kemudian hari dan selesaikan langkah-langkah ini.

  1. Buka kotak dialog Tambah Impor.
  2. Masukkan nama unik untuk file CSV.
  3. Klik ikon tambah (+) untuk mengunggah file dari perangkat Anda.
  4. Klik bidang Jenis dan pilih opsi. (Lihat di atas)
  5. Klik SIMPAN.

Pengolahan

Bergantung pada jumlah catatan dalam file dan beban sistem, platform dapat memerlukan waktu beberapa menit untuk memproses impor.

Saat impor selesai, platform menyesuaikan catatan pengguna (yaitu menambahkan, mengubah, atau menghapus) pada halaman Kasus.

Kegagalan

Jika operasi impor gagal, platform mengembalikan pesan Gagal. Dalam kebanyakan kasus, penyebab kegagalan adalah pemformatan data.

Untuk melihat penyebab kegagalan:

  1. Klik Lihat di bawah judul Tindakan.
  2. Di halaman berikutnya, arahkan mouse ke ikon Gagal. Tooltip menampilkan kesalahan.
  3. Buka file CVS dan sesuaikan dan perbaiki kesalahannya.
  4. Unggah file lagi.
