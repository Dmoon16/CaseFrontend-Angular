Bagaimana Pengguna Membuat Akun Baru

Setelah admin menambahkan pengguna ke kasus: * jika pengguna baru dalam kasus aktif, pengguna harus menyelesaikan alur kerja untuk membuat akun dan menerima undangan untuk mengakses kasus. * jika pengguna memiliki akun caseactive, pengguna harus menerima undangan untuk mengakses kasus.

Artikel ini menjelaskan langkah-langkah yang harus diselesaikan pengguna untuk membuat akun mereka.

Mulai

Ada dua cara pengguna dapat ditambahkan ke aplikasi caseactive

  • Oleh admin: Jika admin membuat pengguna melalui UI platform atau alat Impor , admin harus mengirimkan tautan ke account.caseactive.com kepada pengguna untuk melanjutkan masuk.
  • Oleh pengguna: Jika pengguna membuat akun menggunakan tautan pendaftaran , pengguna akan dialihkan secara otomatis ke account.caseactive.com untuk melanjutkan masuk.

Masuk

Saat pengguna masuk ke akun CaseActive mereka untuk pertama kalinya, platform meminta pengguna untuk:

  • buka akun email mereka lalu salin/tempel kode keamanan enam digit
  • melengkapi profil akun

img

Setelah pengguna mengklik UPDATE, halaman Aplikasi akan ditampilkan. Laman menampilkan nama aplikasi (nama perusahaan), jenis pengguna (peran), status, dan tombol UNDANGAN.

img

Untuk mengakses aplikasi, pengguna harus menerima undangan. Pengguna harus:

  1. Klik tombol UNDANGAN.
  2. Tinjau izin.
  3. Klik tab Notifikasi untuk meninjau dan mengedit (opsional) persyaratan.
  4. Klik SETUJU AKSES.

Saat pengguna menyetujui, platform menampilkan tombol APP di halaman pengguna. Tombol ini memberi pengguna akses ke kasus yang ditetapkan.

Untuk mengedit pengaturan pengguna, pengguna mengklik elipsis.

img

Pengguna dapat: * menghapus akses aplikasi * mengubah pengaturan notifikasi

img

Setelah pengguna menyelesaikan alur kerja, nama pengguna muncul di daftar Pengguna admin. Daftar menunjukkan semua pengguna, masuk atau tidak.
