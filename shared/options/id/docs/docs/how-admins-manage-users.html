Bagaimana Admin Mengelola Pengguna

Admin membuat, mengubah, atau menonaktifkan siapa yang dapat mengakses aplikasi dan kasus. Saat pengguna masuk ke platform untuk pertama kalinya, mereka harus menyelesaikan alur kerja profil.

  • Klik di sini untuk mempelajari lebih lanjut tentang alur kerja profil pengguna baru.

Mengelola pengguna

Tabel di bawah menunjukkan lima cara untuk mengelola pengguna. Klik tautan untuk mempelajari caranya.

metode Jenis Tindakan Detail
Menggunakan UI UI tambah & ubah Tombol Admin > Pengguna
Saat membuat kasing dan tetapkan pengguna ke kasing UI menambahkan Tombol Admin > Kasus > Tombol baru > Halaman Buat Kasus
Otomatis buat pengguna menggunakan tautan pendaftaran publik Tindakan pengguna menambahkan Tombol Admin > Pengaturan > halaman Pengaturan Sistem
Alat impor UI tambah & ubah Tombol Admin > Integrasi > Impor
Menggunakan API (segera hadir) API tambah & ubah T/A

Notifikasi

Platform tidak mengirimkan pemberitahuan kepada pengguna saat mereka ditambahkan ke aplikasi. Namun, admin dapat mengaktifkan notifikasi untuk penetapan kasus.
