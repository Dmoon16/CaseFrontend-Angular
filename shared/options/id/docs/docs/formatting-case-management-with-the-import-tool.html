Memformat Data Manajemen Kasus dengan Alat Impor

Di bawah ini adalah daftar bidang wajib dalam file CSV untuk mengelola kasus dengan alat Impor.

nama belakang

  • Masukkan alamat email pengguna.

tag_id (id yang bisa Anda rujuk nanti)

  • pengidentifikasi unik (menggunakan karakter alfanumerik, tanda hubung diterima)

kasus_status

  • 0 = tertutup
  • 1 = aktif

tanggal_dibuka

  • bulan hari Tahun
  • mm/dd/yyyy

tentang

  • deskripsi unik kasus (gunakan karakter alfanumerik, tanda hubung diterima)
