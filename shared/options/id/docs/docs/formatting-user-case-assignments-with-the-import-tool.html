Memformat Data Tugas Kasus Pengguna dengan Alat Impor

Di bawah ini adalah daftar bidang data dalam file CSV. Setiap opsi impor memiliki kombinasi bidang yang berbeda.

case_ID

  • Masukkan kasus Pengguna atau Tag ID.
  • Buka halaman Kasus > Edit > Tag ID ATAU Buka halaman Kasus > Edit > ID Kasus di url.

identitas pengguna

  • Masukkan ID pengguna atau ID Tag.
  • Buka halaman Pengguna > Edit > Tag ID ATAU Buka halaman Pengguna > Edit > ID Pengguna di url.

case_role_order

  • Biarkan sel kosong.

case_role_ID

  • Masukkan: klien, staf, atau default.

case_notify

  • 0 = tidak
  • 1 = Ya, kirimkan pemberitahuan email kepada pengguna.
