Cara Membuat dan Memodifikasi Pengguna dengan UI

Artikel ini menunjukkan kepada admin cara membuat pengguna dan memperbarui rekaman dengan antarmuka pengguna platform. Admin juga dapat menambahkan pengguna ke kasus baru dan mengedit kasus yang ada.

Membuat pengguna

  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Pada panel Admin, klik tab menu Pengguna.
  4. Klik tombol BARU. Jendela baru ditampilkan.

Isi kolom dan klik SIMPAN. Case Active menambahkan nama ke daftar pengguna yang terkait dengan aplikasi ini.

Memodifikasi pengguna

Untuk memperbarui sejumlah kecil catatan pengguna, gunakan alat Edit dan buat perubahan pada setiap pengguna individu.

  • Catatan Untuk mengubah sejumlah besar rekaman pengguna sekaligus, pertimbangkan untuk menggunakan alat Impor .
  1. Pada panel Admin, klik tab menu Pengguna.
  2. Temukan pengguna dan klik Edit.
  3. Lakukan perubahan dan klik SIMPAN.

Langkah selanjutnya

Kasus Aktif tidak mengirimkan pemberitahuan email kepada pengguna. Admin harus menghubungi pengguna dan mengirimkan link ke halaman beranda Case Active. Pengguna kemudian mengklik tombol LOGIN dan menyelesaikan alur kerja untuk pendaftaran baru.

  • Artikel ini menjelaskan alur kerja pendaftaran untuk pengguna baru.
