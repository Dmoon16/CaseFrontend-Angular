Cara Membuat Aplikasi

Artikel ini menunjukkan kepada pengguna baru cara membuat akun uji coba gratis. Case Active secara otomatis memberikan hak istimewa admin pengguna baru untuk akun ini.

Membuat akun

  1. Pergi ke halaman login .
  2. Klik tombol Mulai.
  3. Masukan alamat email.
  4. Klik Berikutnya.

img

Memverifikasi akun

  1. Buka tab browser baru dan masuk ke akun email Anda.
  2. Salin kode enam digit.
  3. Kembali ke tab Kasus Aktif.
  4. Rekatkan kode ke dalam bidang. Jendela baru ditampilkan.

Di jendela baru:

  1. Isi kolom yang wajib diisi.
  2. Baca persyaratan layanan dan centang kotak untuk setuju.
  3. Klik BUAT AKUN GRATIS.

Menyelesaikan penyiapan aplikasi

Alur kerja ditampilkan di halaman berikutnya. Selesaikan langkah-langkah untuk menyelesaikan penyiapan aplikasi Anda. Anda selalu dapat kembali ke halaman ini untuk mengedit atau memperbarui informasi atau status akun Anda.

Pada halaman Tentang Aplikasi, isi bidang nama perusahaan dan situs web (wajib) lalu klik Berikutnya.

img

Di laman Informasi Penagihan, klik bidang Paket Pembayaran dan tinjau opsinya. Pengaturan standarnya adalah Gratis. Klik KIRIM.

img

Setelah Anda mengirimkan formulir, Case Active menampilkan halaman Aplikasi. Untuk setiap aplikasi yang Anda buat atau miliki, halaman tersebut menampilkan:

  1. Nama aplikasi
  2. Jenis Anda (peran pengguna)
  3. Status

Halaman Aplikasi adalah titik peluncuran untuk banyak fungsi admin:

  1. Klik tombol APP untuk berkomunikasi, mengisi dokumen, obrolan video (dan lainnya) tentang masalah terkait kasus.
  2. Klik tombol ADMIN untuk membuka panel admin. Di sinilah Admin mengelola pengguna, kasus, integrasi, dan lainnya.
  3. Klik elipsis untuk mengelola penagihan akun dan pengaturan notifikasi.
  • Note Case Active menampilkan tombol ADMIN untuk pengguna admin. Pengguna yang tidak memiliki hak admin tidak akan melihat tombol ADMIN (seperti yang ditunjukkan pada layar di bawah).
