Cara Memperbarui Profil Pengguna

Artikel ini menunjukkan kepada pengguna cara memperbarui informasi di halaman Profil mereka.

  • Pengguna dapat memperbarui profil mereka sendiri.
  • Admin dapat memperbarui halaman Profil pengguna mana pun.

Untuk pengguna

Langkah pertama adalah membuka halaman Profil. Pengguna dan admin mengikuti langkah-langkah ini untuk memperbarui halaman Profil mereka sendiri.

  1. Masuk ke akun Case Active Anda.
  2. Klik tab Profil di bagian atas halaman.
  3. Membuat perubahan.
  4. Klik Simpan.

Sebagian besar bidang bersifat opsional, dan beberapa bidang wajib diisi. Bidang yang diperlukan adalah:

  • Nama depan
  • nama keluarga
  • Alamat email
  • Bahasa
  • Zona waktu

Untuk admin

Admin juga dapat memperbarui informasi profil untuk pengguna lain. Ini mencakup tiga bidang di bawah tajuk HOST USER INFO.

  1. Masuk ke akun admin Case Active Anda.
  2. Temukan aplikasi dan klik tombol ADMIN.
  3. Klik Pengguna di menu atas.
  4. Buka sub-menu dan pilih daftar pengguna yang ingin Anda akses (AKTIF atau NONAKTIF). Pengaturan standarnya adalah Aktif.
  5. Temukan pengguna di daftar dan klik Edit.
  6. Lakukan perubahan sesuai kebutuhan. (Lihat di bawah)
  7. Klik Simpan.

Ubah info pengguna

Admin dapat mengubah informasi pengguna.

  1. Tetapkan Jenis Pengguna (Pengguna atau Manajer).
  2. Atur Status Pengguna (Aktif menjadi Nonaktif).
  3. Masukkan ID Tag unik. (opsional)
  4. Klik Simpan.
