Cara Membuat Pengguna dengan Tautan Pendaftaran

Artikel ini menunjukkan kepada admin cara membuat pengguna dengan mengirimkan tautan pendaftaran.

Mengkonfigurasi tautan pendaftaran

  1. Klik tautan Kirimi mereka pendaftaran di bagian bawah kotak dialog Tambahkan pengguna. Halaman baru ditampilkan.
  2. Konfigurasikan opsi pendaftaran. (Lihat di bawah)

img

Pastikan bidang Izinkan Pendaftaran disetel ke Aktif. (wajib) Saat diaktifkan, Case Active menghasilkan tautan unik.

Tinjau opsi di bawah, lalu klik ikon untuk menyalin tautan. Anda dapat menempatkan tautan di situs web Anda atau mengirim tautan ke satu atau lebih orang melalui email atau saluran komunikasi apa pun.

img

Membuat kasus

Secara opsional, Anda dapat secara otomatis membuat kasus untuk setiap pengguna baru dan menetapkan peran pengguna.

  1. Klik bidang Buat Kasus Otomatis dan pilih Aktif. Bidang Peran Pengguna Kasus ditampilkan.
  2. Klik Kasus Peran Pengguna dan pilih opsi. (Lihat di bawah)

img

Kasus peran pengguna

Ada tiga peran pengguna kasus.

  • klien
  • bawaan
  • staf

Mengirim tautan

Bagikan tautan dengan satu atau beberapa orang. Setiap orang mengklik link dan menyelesaikan alur kerja untuk mengaktifkan akun Case Active mereka. Setelah pengguna menyelesaikan alur kerja pendaftaran, Case Active menambahkan pengguna ke daftar Pengguna.

img

Langkah selanjutnya

Case Active mengirimkan pemberitahuan email kepada pengguna dengan tautan pendaftaran. Pengguna mengklik tautan dan menyelesaikan alur kerja untuk pendaftaran baru.

  • Artikel ini menjelaskan alur kerja pendaftaran untuk pengguna baru.
