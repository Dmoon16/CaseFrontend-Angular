Cara Menetapkan Pengguna Secara Otomatis dengan Tautan Pendaftaran Publik

Admin mempelajari cara mengonfigurasi pengaturan sistem sehingga pengguna baru secara otomatis ditetapkan ke kasus yang baru dibuat secara otomatis.

Membuka halaman pengaturan

Langkah pertama adalah membuka halaman Pengaturan Sistem.

  1. Masuk ke akun admin.
  2. Temukan aplikasi di halaman Aplikasi dan klik tombol ADMIN.
  3. Klik tab Sistem dan kemudian PENGATURAN SISTEM pada submenu.
  4. Gulir ke bawah ke tajuk USER SIGNUP.

Mengonfigurasi pengaturan sistem

  1. Setel menu tarik-turun Izinkan Pendaftaran dan Buat Kasus Otomatis ke Aktif.
  2. Klik menu peran dan pilih opsi (disarankan default).
  3. Salin dan bagikan tautan Pendaftaran Pengguna Publik.

img

Mengirim tautan

Setiap pengguna yang mengeklik tautan dan menyelesaikan pendaftaran memicu platform untuk membuat kasus baru di halaman Kasus dan menetapkan pengguna secara otomatis dalam kasus tersebut.
