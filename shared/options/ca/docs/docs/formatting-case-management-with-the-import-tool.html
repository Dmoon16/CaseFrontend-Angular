Format de dades de gestió de casos amb l'eina d'importació

A continuació es mostra una llista de camps obligatoris als fitxers CSV per gestionar casos amb l'eina d'importació.

nom d'usuari

  • Introduïu l'adreça de correu electrònic de l'usuari.

tag_id (un identificador que podeu fer referència més endavant)

  • un identificador únic (utilitza caràcters alfanumèrics, s'accepten guions)

cas_estat

  • 0 = tancat
  • 1 = actiu

data_obert

  • mes/dia/any
  • mm/dd/aaaa

Sobre

  • una descripció única del cas (utilitzeu caràcters alfanumèrics, s'accepten guions)
