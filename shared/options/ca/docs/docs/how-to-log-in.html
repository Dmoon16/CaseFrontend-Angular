Com iniciar sessió

Obteniu informació sobre com iniciar sessió al vostre compte de Case Active existent.

Inicia sesió

  1. Aneu a la pàgina d'inici de sessió.
  2. Feu clic al botó LOGIN.
  3. Introdueixi la seva adreça de correu electrònic.
  4. Feu clic a SEGÜENT.

Verificació d'un compte

  1. Obriu una nova pestanya del navegador i inicieu sessió al vostre compte de correu electrònic.
  2. Copieu el codi de sis dígits.
  3. Torna a la pestanya Cas actiu.
  4. Enganxeu el codi als camps. Es mostra una finestra nova.

Case Active mostra la pàgina d'aplicacions.

  • Nota La plataforma mostra un botó ADMIN al costat del botó APP per als usuaris amb privilegis d'administrador.
