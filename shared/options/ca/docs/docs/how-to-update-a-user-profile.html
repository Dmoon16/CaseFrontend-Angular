Com actualitzar un perfil d'usuari

Aquest article mostra als usuaris com actualitzar la informació a la seva pàgina de perfil.

  • Els usuaris poden actualitzar els seus propis perfils.
  • Els administradors poden actualitzar la pàgina de perfil de qualsevol usuari.

Per als usuaris

El primer pas és obrir la pàgina de perfil. Els usuaris i els administradors segueixen aquests passos per actualitzar la seva pròpia pàgina de perfil.

  1. Inicieu la sessió al vostre compte de Case Active.
  2. Feu clic a la pestanya Perfil a la part superior de la pàgina.
  3. Feu canvis.
  4. Feu clic a Desa.

La majoria dels camps són opcionals i alguns són obligatoris. Els camps obligatoris són:

  • Nom
  • Cognom
  • Correu electrònic
  • Llenguatge
  • Fus horari

Per a administradors

Els administradors també poden actualitzar la informació del perfil d'altres usuaris. Això inclou tres camps sota l'encapçalament HOST USER INFO.

  1. Inicieu sessió al vostre compte d'administrador de Case Active.
  2. Localitzeu una aplicació i feu clic al botó ADMINISTRADOR.
  3. Feu clic a Usuaris al menú superior.
  4. Aneu al submenú i seleccioneu la llista d'usuaris a la qual voleu accedir (ACTIVA o DESACTIVADA). La configuració predeterminada és Active.
  5. Localitzeu un usuari a la llista i feu clic a Edita.
  6. Feu els canvis necessaris. (mirar abaix)
  7. Feu clic a Desa.

img

Canvia la informació de l'usuari

Els administradors poden canviar la informació de l'usuari.

  1. Definiu el tipus d'usuari (usuari o gestor).
  2. Estableix l'estat de l'usuari (actiu a desactivat).
  3. Introduïu un identificador d'etiqueta únic. (opcional)
  4. Feu clic a Desa.
