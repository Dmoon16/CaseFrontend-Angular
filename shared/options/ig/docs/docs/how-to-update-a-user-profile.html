Otu esi emelite profaịlụ onye ọrụ

Edemede a na-egosi ndị ọrụ otu esi emelite ozi na ibe profaịlụ ha.

  • Ndị ọrụ nwere ike imelite profaịlụ nke ha.
  • Ndị nlekọta nwere ike imelite ibe profaịlụ nke onye ọrụ ọ bụla.

Maka ndị ọrụ

Nzọụkwụ mbụ bụ imepe ibe profaịlụ. Ndị ọrụ na ndị nchịkwa na-agbaso usoro ndị a iji melite ibe profaịlụ nke ha.

  1. Banye na akaụntụ Case Active gị.
  2. Pịa taabụ profaịlụ dị n'elu ibe ahụ.
  3. Mee mgbanwe.
  4. Pịa Chekwa.

Ọtụtụ mpaghara bụ nhọrọ, yana ụfọdụ ubi achọrọ. Ala achọrọ bụ:

  • Aha mbụ
  • Aha ikpeazụ
  • Adreesị ozi ịntanetị
  • Asụsụ
  • Mpaghara oge

Maka ndị admins

Ndị nchịkwa nwekwara ike imelite ozi profaịlụ maka ndị ọrụ ndị ọzọ. Nke a gụnyere mpaghara atọ n'okpuru isi ozi HOST USER.

  1. Banye na akaụntụ nchịkwa Case Active gị.
  2. Chọta ngwa wee pịa bọtịnụ ADMIN.
  3. Pịa Ndị ọrụ n'elu menu.
  4. Gaa na sub-menu wee họrọ ndetu onye ọrụ ịchọrọ ịnweta (AKWỤKWỌ ma ọ bụ Nwepu). Ntọala ndabara na-arụ ọrụ.
  5. Chọta onye ọrụ na ndepụta wee pịa Dezie.
  6. Mee mgbanwe dị ka achọrọ. (hụ okpuru)
  7. Pịa Chekwa.

Gbanwee ozi onye ọrụ

Ndị nchịkwa nwere ike ịgbanwe ozi onye ọrụ.

  1. Tọọ ụdị onye ọrụ (onye ọrụ ma ọ bụ njikwa).
  2. Tọọ ọnọdụ onye ọrụ (Ọ na-arụ ọrụ iji gbanyụọ).
  3. Tinye ID mkpado pụrụ iche. (nhọrọ)
  4. Pịa Chekwa.
