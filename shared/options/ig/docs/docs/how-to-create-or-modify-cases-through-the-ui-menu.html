Otu esi emepụta ma ọ bụ gbanwee okwu site na menu UI

Edemede a na-egosi admins otu esi emepụta ikpe na interface onye ọrụ nke ikpo okwu. Ndị nchịkwa nwekwara ike ịgbakwunye ndị ọrụ na ikpe ọhụrụ wee melite ikpe ndị dị adị.

Ịmepụta ikpe

  1. Banye na akaụntụ nchịkwa.
  2. Chọta ngwa na ibe ngwa wee pịa bọtịnụ ADMIN.
  3. Na panel Admin, pịa taabụ menu Okwu.
  4. Pịa bọtịnụ NEW. Ngosipụta windo ọhụrụ.

Ịgbanwe okwu

Ka imelite ọnụ ọgụgụ dị nta nke ikpe, jiri ngwa Dezie ma mee mgbanwe na ikpe nke ọ bụla.
