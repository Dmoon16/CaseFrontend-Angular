App တစ်ခုဖန်တီးနည်း

ဤဆောင်းပါးတွင် သုံးစွဲသူအသစ်များကို အခမဲ့အစမ်းသုံးအကောင့်ဖန်တီးနည်းကို ပြသထားသည်။ Case Active သည် ဤအကောင့်အတွက် အသုံးပြုသူ စီမံခန့်ခွဲခွင့်အသစ်များကို အလိုအလျောက် ပေးပါသည်။

အကောင့်တစ်ခုဖန်တီးခြင်း။

  1. လော့ဂ်အင်စာမျက်နှာ သို့သွားပါ။
  2. Get Started ခလုတ်ကိုနှိပ်ပါ။
  3. အီးမေးလ်လိပ်စာကို ထည့်ပါ။
  4. Next ကိုနှိပ်ပါ။

အကောင့်တစ်ခုစစ်ဆေးခြင်း။

  1. ဘရောက်ဆာ တက်ဘ်အသစ်တစ်ခုဖွင့်ပြီး သင့်အီးမေးလ်အကောင့်သို့ လော့ဂ်အင်ဝင်ပါ။
  2. ဂဏန်းခြောက်လုံးကုဒ်ကို ကူးယူပါ။
  3. Case Active တက်ဘ်သို့ ပြန်သွားပါ။
  4. ကုဒ်ကို အကွက်များထဲသို့ ကူးထည့်ပါ။ ဝင်းဒိုးအသစ်တစ်ခုပေါ်လာသည်။

ဝင်းဒိုးအသစ်တွင်-

  1. လိုအပ်သောအကွက်များကိုဖြည့်ပါ။
  2. ဝန်ဆောင်မှု စည်းကမ်းချက်များကို ဖတ်ပြီး သဘောတူရန် အကွက်ကို အမှန်ခြစ်ပါ။
  3. အခမဲ့အကောင့်ဖန်တီးရန် နှိပ်ပါ။

အက်ပ်စဖွင့်သတ်မှတ်မှု အပြီးသတ်နေပါသည်။

အလုပ်အသွားအလာကို နောက်စာမျက်နှာတွင် ပြသသည်။ သင့်အက်ပ်ကို စဖွင့်သတ်မှတ်ခြင်း အပြီးသတ်ရန် အဆင့်များကို ဖြည့်စွက်ပါ။ အချက်အလက် သို့မဟုတ် သင့်အကောင့်အခြေအနေကို တည်းဖြတ်ရန် သို့မဟုတ် အပ်ဒိတ်လုပ်ရန် ဤစာမျက်နှာများသို့ သင်အမြဲပြန်လာနိုင်သည်။

About App စာမျက်နှာတွင်၊ ကုမ္ပဏီအမည်နှင့် ဝဘ်ဆိုဒ်ကွက်လပ်များကိုဖြည့်ပါ (လိုအပ်သည်) ပြီးနောက် Next ကိုနှိပ်ပါ။

ငွေတောင်းခံခြင်းဆိုင်ရာ အချက်အလက်စာမျက်နှာတွင်၊ ငွေပေးချေမှုအစီအစဉ်အကွက်ကို နှိပ်ပြီး ရွေးချယ်စရာများကို ပြန်လည်သုံးသပ်ပါ။ မူရင်းဆက်တင်သည် အခမဲ့ဖြစ်သည်။ SUBMIT ကိုနှိပ်ပါ။

ဖောင်ကို သင်တင်ပြပြီးနောက်၊ Case Active သည် အက်ပ်များစာမျက်နှာကို ပြသသည်။ သင်ဖန်တီးသော သို့မဟုတ် ပိုင်ဆိုင်သည့်အက်ပ်တစ်ခုစီအတွက်၊ စာမျက်နှာတွင် ဖော်ပြသည်-

  1. အက်ပ်အမည်
  2. သင့်အမျိုးအစား (အသုံးပြုသူအခန်းကဏ္ဍ)
  3. အဆင့်အတန်း

img

အက်ပ်များ စာမျက်နှာသည် စီမံခန့်ခွဲသူ လုပ်ဆောင်ချက်များစွာအတွက် စတင်သည့်အချက်ဖြစ်သည်။

  1. ဆက်သွယ်ရန်၊ စာရွက်စာတမ်းများဖြည့်ရန်၊ ဗီဒီယိုချတ် (နှင့် အခြားအရာများ) နှင့်ပတ်သက်သည့် ကိစ္စများအတွက် APP ခလုတ်ကို နှိပ်ပါ။
  2. စီမံခန့်ခွဲသူအကန့်ကိုဖွင့်ရန် ADMIN ခလုတ်ကိုနှိပ်ပါ။ ဤနေရာတွင် အက်ဒ်မင်များသည် သုံးစွဲသူများ၊ ကိစ္စများ၊ ပေါင်းစည်းမှုများနှင့် အခြားအရာများကို စီမံခန့်ခွဲသည့်နေရာဖြစ်သည်။
  3. အကောင့်ငွေတောင်းခံမှုနှင့် အကြောင်းကြားချက်ဆက်တင်များကို စီမံခန့်ခွဲရန် ellipsis ကို နှိပ်ပါ။
  • မှတ်ချက် Case Active သည် စီမံခန့်ခွဲသူအသုံးပြုသူများအတွက် ADMIN ခလုတ်ကို ပြသသည်။ စီမံခန့်ခွဲရေးဆိုင်ရာအခွင့်ထူးများမရှိသောအသုံးပြုသူများသည် ADMIN ခလုတ်ကိုမတွေ့ပါ (အောက်ပါမျက်နှာပြင်တွင်ပြထားသည့်အတိုင်း)။
