သွင်းကုန်တူးလ်ဖြင့် ဖြစ်ရပ်စီမံခန့်ခွဲမှုဒေတာကို ပုံစံချခြင်း။

အောက်တွင်ဖော်ပြထားသည်မှာ Import tool ဖြင့် ကိစ္စများကို စီမံခန့်ခွဲရန် CSV ဖိုင်များတွင် လိုအပ်သော အကွက်များစာရင်းဖြစ်သည်။

အသုံးပြုသူအမည်

  • အသုံးပြုသူ၏ အီးမေးလ်လိပ်စာကို ထည့်သွင်းပါ။

tag_id (နောက်မှကိုးကားနိုင်သော ID တစ်ခု)

  • တစ်မူထူးခြားသော အမှတ်အသားတစ်ခု (အက္ခရာစဉ်အလိုက် ဂဏန်းအက္ခရာများကို အသုံးပြုပါ၊ တုံးတိုများကို လက်ခံပါသည်)

ကိစ္စ_အခြေအနေ

  • 0 = ပိတ်သည်။
  • 1 = တက်ကြွသော

ရက်စွဲ_ဖွင့်သည်။

  • လ/ရက်/နှစ်
  • mm/dd/yyyy

အကြောင်း

  • ကိစ္စ၏ထူးခြားသောဖော်ပြချက် (အက္ခရာဂဏန်းစာလုံးများကိုသုံးပါ၊ တုံးတိုများကိုလက်ခံသည်)
