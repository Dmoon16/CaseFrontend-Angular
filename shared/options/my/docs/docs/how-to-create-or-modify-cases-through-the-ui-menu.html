UI Menu မှတဆင့် Cases များကို ဖန်တီးရန် သို့မဟုတ် ပြင်ဆင်နည်း

ဤဆောင်းပါးသည် ပလပ်ဖောင်း၏ အသုံးပြုသူမျက်နှာပြင်ဖြင့် အက်ဒ်မင်များကို မည်သို့ဖန်တီးရမည်ကို ပြသထားသည်။ အက်ဒ်မင်များသည် သုံးစွဲသူများကို အမှုအသစ်တစ်ခုသို့ ပေါင်းထည့်နိုင်ပြီး ရှိပြီးသားကိစ္စများကို အပ်ဒိတ်လုပ်နိုင်သည်။

အမှုတစ်ခုဖန်တီးခြင်း။

  1. စီမံခန့်ခွဲသူအကောင့်သို့ ဝင်ရောက်ပါ။
  2. အက်ပ်များ စာမျက်နှာတွင် အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. Admin panel တွင် Cases menu tab ကို နှိပ်ပါ။
  4. NEW ခလုတ်ကိုနှိပ်ပါ။ ဝင်းဒိုးအသစ်တစ်ခုပေါ်လာသည်။

img

လိုအပ်သောကွက်လပ်များကိုဖြည့်ပြီး SAVE ကိုနှိပ်ပါ။

img

အသုံးပြုသူများကို ထည့်သွင်းခြင်း။

ရွေးချယ်နိုင်သောအားဖြင့်၊ စီမံခန့်ခွဲသူများသည် ဤစာမျက်နှာရှိ အမှုကိစ္စတွင် အသုံးပြုသူများကို ထည့်သွင်းနိုင်သည်။ အောက်ကိုဆင်းပြီး ASSIGN USERS ခလုတ်ကို နှိပ်ပါ။

img

Assign User modal တွင်၊ အကွက်များကိုဖြည့်ပါ။ သုံးစွဲသူထံ အီးမေးလ်အကြောင်းကြားချက် ပေးပို့ရန် အကွက်ကို စစ်ဆေးပါ။ (ချန်လှပ်ထားနိုင်သည်)

img

ကိစ္စတစ်ခုကို ပြင်ဆင်ခြင်း။

အမှုအနည်းစုကို အပ်ဒိတ်လုပ်ရန်၊ တည်းဖြတ်ကိရိယာကို အသုံးပြုပြီး အမှုတစ်ခုချင်းစီအတွက် အပြောင်းအလဲများ ပြုလုပ်ပါ။

  • မှတ်ချက် အများအပြားကို တစ်ထိုင်တည်း အပ်ဒိတ်လုပ်ရန်၊ သွင်းကုန်တူး လ်ကို အသုံးပြု၍ စဉ်းစားပါ။
  1. စီမံခန့်ခွဲသူအကောင့်သို့ ဝင်ရောက်ပါ။
  2. အက်ပ်များ စာမျက်နှာတွင် အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. Admin panel တွင် Cases menu tab ကို နှိပ်ပါ။
  4. ကိစ္စတစ်ခုကို ရှာပြီး Edit ကိုနှိပ်ပါ။
  5. အပြောင်းအလဲများပြုလုပ်ပြီး SAVE ကိုနှိပ်ပါ။
