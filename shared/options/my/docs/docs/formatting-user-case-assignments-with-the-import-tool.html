Import Tool ဖြင့် အသုံးပြုသူ Case Assignments ဒေတာကို ပုံစံချခြင်း။

အောက်တွင် CSV ဖိုင်များတွင် ဒေတာအကွက်များစာရင်းဖြစ်သည်။ တင်သွင်းမှုရွေးချယ်မှုတစ်ခုစီတွင် မတူညီသောအကွက်များ ပေါင်းစပ်မှုရှိသည်။

case_ID

  • အသုံးပြုသူ သို့မဟုတ် Tag ID ကို ထည့်ပါ။
  • Cases စာမျက်နှာသို့ သွားရန် > တည်းဖြတ် > Tag ID သို့မဟုတ် Cases စာမျက်နှာသို့ သွားရန် > Edit > Case ID ကို url တွင် သွားပါ။

သုံးစွဲသူအိုင်ဒီ

  • အသုံးပြုသူ၏ ID သို့မဟုတ် Tag ID ကိုထည့်ပါ။
  • အသုံးပြုသူများ စာမျက်နှာသို့ သွားရန် > တည်းဖြတ် > Tag ID သို့မဟုတ် အသုံးပြုသူများ စာမျက်နှာသို့ သွားပါ > Edit > User ID ကို url တွင် သွားပါ။

case_role_order

  • ဆဲလ်ကို ကွက်လပ်ထားပါ။

case_role_ID

  • ထည့်ပါ- ဖောက်သည်များ၊ ဝန်ထမ်းများ သို့မဟုတ် မူရင်း။

case_notify

  • 0 = မရှိပါ။
  • 1 = ဟုတ်ကဲ့၊ သုံးစွဲသူထံ အီးမေးလ် အကြောင်းကြားချက် ပေးပို့ပါ။
