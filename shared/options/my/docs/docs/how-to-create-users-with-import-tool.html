Import Tool ဖြင့် အသုံးပြုသူများကို ဖန်တီးနည်း

ဤဆောင်းပါးသည် သွင်းကုန်တူးလ်ဖြင့် အသုံးပြုသူများကို စီမံခန့်ခွဲသူများကို မည်သို့ဖန်တီးရမည်ကို ပြသထားသည်။

သွင်းကုန်တူးလ်ကိုဖွင့်ခြင်း။

အက်ဒ်မင်များသည် သုံးစွဲသူအသစ်များ အစုလိုက် အပ်လုဒ်လုပ်ပြီး လက်ရှိအသုံးပြုသူ မှတ်တမ်းများကို မွမ်းမံရန် သွင်းကုန်တူးလ်ကို အသုံးပြုနိုင်သည်။

  1. စီမံခန့်ခွဲသူအကောင့်သို့ ဝင်ရောက်ပါ။
  2. အက်ပ်များ စာမျက်နှာတွင် အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. စီမံခန့်ခွဲသူအကန့်တွင်၊ ပေါင်းစည်းခြင်းတက်ဘ်ကို နှိပ်ပြီး မီနူးတွင် တင်သွင်းခြင်းများကို နှိပ်ပါ။
  4. NEW ခလုတ်ကိုနှိပ်ပါ။ Add Imports dialog box သည် browser တက်ဘ်အသစ်တစ်ခုတွင် ပြသသည်။

မှတ်တမ်းများဖန်တီးခြင်း။

Add Imports dialog box ပွင့်ပြီးနောက်-

  1. Type အကွက်ကို နှိပ်ပြီး ရွေးချယ်စရာတစ်ခုကို ရွေးပါ (အောက်ပါပုံကိုကြည့်ပါ)။ လင့်ခ်တစ်ခု ပေါ်လာသည်။
  2. Download နမူနာ လင့်ခ်ကို နှိပ်ပါ။ CSV ဖိုင်သည် သင့်စက်ထဲသို့ ဒေါင်းလုဒ်လုပ်သည်။

ထို့နောက် ဖိုင်ကိုဖွင့်ပြီး မှတ်တမ်းများထည့်ပါ။ CSV ဖိုင်တွင် ဖော်မတ်ဒေတာ အသုံးပြုနည်းကို လေ့လာရန် ဤနေရာကို နှိပ်ပါ။

ရိုက်ပါ။

Type အကွက်ကို နှိပ်လိုက်သောအခါတွင် drop-down menu ပေါ်လာသည်။ အများအပြားတင်သွင်းမှုအတွက် ရွေးချယ်စရာနှစ်ခုထဲမှ တစ်ခုကို ရွေးပါ။

:exclamation- မှတ်ချက် - drop-down menu တွင် ရွေးချယ်စရာများစွာရှိသော်လည်း တင်သွင်းသည့်ကိရိယာနှင့် ပတ်သက်သည့် နှစ်ခုသာရှိသည်။ ရွေးချယ်မှုတစ်ခုစီတွင် မတူညီသောနမူနာ CSV ဖိုင်တစ်ခုရှိသည်။

အသုံးပြုသူများ ထည့်ပါ။

  • အသုံးပြုသူမှတ်တမ်းများ၏ ရိုးရှင်းသောအစုလိုက်အပ်လုဒ်အတွက် ဤရွေးချယ်မှုကို ရွေးချယ်ပါ။
  • Add users Import type အတွက် ကော်လံခေါင်းစီးများပါသော CSV ဖိုင်ကိုဖွင့်ရန် နမူနာဒေါင်းလုဒ်လင့်ခ်ကို နှိပ်ပါ။

အသုံးပြုသူများကို ပြင်ဆင်ပါ။

  • လက်ရှိအသုံးပြုသူများအတွက် ဒေတာမွမ်းမံရန် ဤရွေးချယ်မှုကို ရွေးချယ်ပါ။
  • Modify users Import type အတွက် ကော်လံခေါင်းစီးများပါသော CSV ဖိုင်ကိုဖွင့်ရန် နမူနာဒေါင်းလုဒ်လင့်ခ်ကို နှိပ်ပါ။

ဖိုင်ကို တင်ခြင်း။

သင်သည် ဖိုင်တစ်ခုကို ချက်ချင်း အပ်လုဒ်တင်နိုင်သည် သို့မဟုတ် နောက်ပိုင်းရက်စွဲတွင် Add Imports dialog box သို့ ပြန်သွားပြီး ဤအဆင့်များကို ပြီးအောင်လုပ်ပါ။

  1. Add Imports dialog box ကိုဖွင့်ပါ။
  2. CSV ဖိုင်အတွက် သီးသန့်အမည်တစ်ခု ထည့်ပါ။
  3. သင့်စက်မှ ဖိုင်ကို အပ်လုဒ်လုပ်ရန် အပေါင်း (+) အိုင်ကွန်ကို နှိပ်ပါ။
  4. Type အကွက်ကို နှိပ်ပြီး ရွေးချယ်မှုတစ်ခုကို ရွေးချယ်ပါ။ (အပေါ်ကိုကြည့်ပါ)
  5. SAVE ကိုနှိပ်ပါ။

လုပ်ဆောင်နေသည်

ဖိုင်နှင့် စနစ်ဝန်ရှိ မှတ်တမ်းအရေအတွက်ပေါ်မူတည်၍ ပလပ်ဖောင်းသည် တင်သွင်းမှုကို လုပ်ဆောင်ရန် မိနစ်များစွာ ကြာနိုင်သည်။

တင်သွင်းမှုပြီးသွားသောအခါ၊ ပလပ်ဖောင်းသည် စီမံခန့်ခွဲသူ၏ အသုံးပြုသူများ ဒက်ရှ်ဘုတ်တွင် အသုံးပြုသူမှတ်တမ်းများ (ဆိုလိုသည်မှာ ထပ်ထည့်ခြင်း၊ ပြင်ဆင်ခြင်း သို့မဟုတ် ပိတ်ခြင်း) ကို ချိန်ညှိပေးပါသည်။

ပျက်ကွက်ခြင်း။

တင်သွင်းမှုလုပ်ဆောင်မှု မအောင်မြင်ပါက၊ ပလပ်ဖောင်းသည် မအောင်မြင်သော မက်ဆေ့ဂျ်ကို ပြန်ပေးသည်။ ကိစ္စအများစုတွင် ပျက်ကွက်ရခြင်းအကြောင်းရင်းမှာ ဒေတာဖော်မတ်ချခြင်း ဖြစ်သည်။

ရှုံးနိမ့်ရခြင်းအကြောင်းရင်းကို ကြည့်ရန်-

  1. လုပ်ဆောင်ချက်ခေါင်းစဉ်အောက်ရှိ View ကို နှိပ်ပါ။
  2. နောက်စာမျက်နှာတွင် မအောင်မြင်သောအိုင်ကွန်ပေါ်တွင် မောက်စ်ကို ရွှေ့ပါ။ tooltip သည် error ကိုပြသသည်။
  3. CVS ဖိုင်ကိုသွားပြီး အမှားကို ချိန်ညှိပြီး ပြင်ဆင်ပါ။
  4. ဖိုင်ကို ပြန်တင်ပါ။
