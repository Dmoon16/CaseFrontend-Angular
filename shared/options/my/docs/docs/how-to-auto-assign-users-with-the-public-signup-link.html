အများသူငှာ အကောင့်ဖွင့်ခြင်းလင့်ခ်ဖြင့် သုံးစွဲသူများကို အလိုအလျောက် သတ်မှတ်နည်း

အက်ဒ်မင်များသည် အသစ်စက်စက် အလိုအလျောက်ဖန်တီးထားသော ကိစ္စတစ်ခုတွင် အသုံးပြုသူအသစ်များကို အလိုအလျောက်သတ်မှတ်ပေးနိုင်ရန် စနစ်ဆက်တင်များကို မည်သို့စီစဉ်သတ်မှတ်ရမည်ကို လေ့လာပါ။

တပ်ဆင်မှုစာမျက်နှာကိုဖွင့်ခြင်း။

ပထမအဆင့်မှာ Systems Setup စာမျက်နှာကိုဖွင့်ရန်ဖြစ်သည်။

  1. admin အကောင့်သို့ဝင်ရောက်ပါ။
  2. အက်ပ်များ စာမျက်နှာတွင် အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. စနစ်များတက်ဘ်ကို နှိပ်ပြီး မီနူးတွင် SYSTEM SETUP ကိုနှိပ်ပါ။
  4. USER SIGNUP ခေါင်းစဉ်သို့ ဆင်းပါ။

စနစ်ဆက်တင်များကို ပြင်ဆင်သတ်မှတ်ခြင်း။

  1. အကောင့်ဖွင့်ခွင့်ပြုရန်နှင့် အလိုအလျောက်ဖန်တီးရန် Case drop-down မီနူးများကို ဖွင့်ရန် သတ်မှတ်ပါ။
  2. အခန်းကဏ္ဍမီနူးကို နှိပ်ပြီး ရွေးချယ်မှုတစ်ခုကို ရွေးချယ်ပါ (မူလက အကြံပြုထားသည်)။
  3. အများသူငှာ အသုံးပြုသူ အကောင့်ဖွင့်ခြင်းလင့်ခ်ကို ကူးယူပြီး မျှဝေပါ။

လင့်ခ် ပေးပို့ခြင်း။

လင့်ခ်ကို နှိပ်ပြီး အကောင့်ဖွင့်ခြင်းကို အပြီးသတ်သော အသုံးပြုသူတိုင်းသည် Cases စာမျက်နှာတွင် အမှုအသစ်တစ်ခု ဖန်တီးရန် ပလပ်ဖောင်းကို အစပျိုးပေးပြီး ထိုကိစ္စတွင် အသုံးပြုသူကို အလိုအလျောက် သတ်မှတ်ပေးသည်။
