Case တစ်ခုကို ဖန်တီးနေစဉ် အသုံးပြုသူများအား သတ်မှတ်ပုံ

ဤဆောင်းပါးတွင် အမှုတွဲအသစ်တစ်ခုကို ဖန်တီးစဉ်တွင် အသုံးပြုသူတစ်ဦးအား စီမံခန့်ခွဲသူများကို မည်သို့တာဝန်ပေးရမည်ကို ပြသထားသည်။ အကယ်၍ အသုံးပြုသူသည် ပလပ်ဖောင်းပေါ်တွင် စာရင်းမသွင်းရသေးပါက၊ အမှုတွဲအသစ်ကို ဖန်တီးနေစဉ်တွင် အသုံးပြုသူအသစ်ကို ထည့်သွင်းဖန်တီးနိုင်ပါသည်။

အမှုတစ်ခုဖန်တီးခြင်း။

  1. admin အကောင့်သို့ဝင်ရောက်ပါ။
  2. အက်ပ်များ စာမျက်နှာတွင် အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. Cases menu tab ကိုနှိပ်ပါ။
  4. NEW ခလုတ်ကိုနှိပ်ပါ။ ဝင်းဒိုးအသစ်တစ်ခုပေါ်လာသည်။
  • ကိစ္စတစ်ခုဖန်တီးရန် ကွက်လပ်များကို ဖြည့်ပြီး ပေးအပ်ထားသည့် အသုံးပြုသူများ ခေါင်းစဉ်သို့ အောက်သို့ ဆင်းပါ။

အသုံးပြုသူအား တာဝန်ပေးခြင်း

လက်ရှိအသုံးပြုသူ

လက်ရှိအသုံးပြုသူတစ်ဦးအား အမှုအသစ်အတွက် သတ်မှတ်ရန် ဤအဆင့်များကို လိုက်နာပါ။

ASSIGN USERS ခလုတ်ကို နှိပ်ပြီး ကွက်လပ်များကို ဖြည့်ပါ။

  1. အသုံးပြုသူ drop-down ကိုနှိပ်ပြီး identifier (ဥပမာ အီးမေးလ်) ကို ရွေးချယ်ပါ။
  2. အကွက်အလွတ်တွင် တန်ဖိုးတစ်ခု ထည့်ပါ (ဥပမာ အီးမေးလ်လိပ်စာ)။
  3. အသုံးပြုသူမှတ်တမ်းသည် ပလပ်ဖောင်းတွင် ရှိနေပါက၊ အကွက်သည် အမည်တစ်ခုကို အလိုအလျောက် အကြံပြုသည်။
  4. မှန်ကန်သောအသုံးပြုသူကို ရွေးပါ။
  5. အသုံးပြုသူအခန်းကဏ္ဍ (ဖောက်သည်များ၊ မူရင်း သို့မဟုတ် ဝန်ထမ်းများ) ကို ရွေးချယ်ပါ။
  6. ရွေးချယ်နိုင်သည်၊ အသုံးပြုသူကို အီးမေးလ်အကြောင်းကြားချက်ပေးပို့ရန် အကွက်ကို အမှန်ခြစ်ပါ။
  7. SAVE ကိုနှိပ်ပါ။

img

Case Active သည် အသုံးပြုသူကို အမှုတွဲထဲသို့ ပေါင်းထည့်သည်။ ယခုအခါ စီမံခန့်ခွဲသူများသည် ဝင်းဒိုးကိုပိတ်ရန် သိမ်းဆည်းရန်၊ သို့မဟုတ် အသုံးပြုသူအသစ်ထည့်ရန် သို့မဟုတ် အသုံးပြုသူကို ဖယ်ရှားရန် နှိပ်နိုင်သည်။

အသုံးပြုသူအသစ်ကို ထည့်သွင်းခြင်း။

ပလပ်ဖောင်းတွင် လက်ရှိမှတ်တမ်းကို ရှာမတွေ့ပါက၊ အသုံးပြုသူအသစ်ထည့်ရန် ခလုတ်ကို နှိပ်ပါ။

ကွက်လပ်များကိုဖြည့်ပြီး အသုံးပြုသူကို ပလပ်ဖောင်းသို့ထည့်ရန် SAVE ကိုနှိပ်ပါ။ ထို့နောက်၊ အသုံးပြုသူအခန်းကဏ္ဍကို သတ်မှတ်ပြီး အမှုကိစ္စအတွက် အသုံးပြုသူကို သတ်မှတ်ရန် ထပ်မံ၍ သိမ်းဆည်းရန် ထပ်မံနှိပ်ပါ။ နောက်ဆုံးတွင် အမှုသိမ်းရန် နောက်ထပ်တစ်ကြိမ် SAVE ကိုနှိပ်ပါ။
