အကောင့်ဖွင့်လင့်ခ်ဖြင့် အသုံးပြုသူများကို ဖန်တီးနည်း

ဤဆောင်းပါးတွင် အကောင့်ဖွင့်သည့်လင့်ခ်ကို ပေးပို့ခြင်းဖြင့် သုံးစွဲသူများအား စီမံခန့်ခွဲသူများကို မည်သို့ဖန်တီးရမည်ကို ပြသထားသည်။

အကောင့်ဖွင့်ခြင်းလင့်ခ်ကို ပြင်ဆင်သတ်မှတ်ခြင်း။

  1. Add user dialog box ၏အောက်ခြေရှိ Send them a signup link ကိုနှိပ်ပါ။ စာမျက်နှာအသစ်တစ်ခုပေါ်လာသည်။
  2. အကောင့်ဖွင့်ခြင်းရွေးချယ်စရာများကို စီစဉ်သတ်မှတ်ပါ။ (အောက်တွင်ကြည့်ပါ)

img

အကောင့်ဖွင့် ခွင့်ပြုရန် အကွက်ကို On ဟု သတ်မှတ်ကြောင်း သေချာပါစေ။ (လိုအပ်သည်) ဖွင့်ထားသောအခါ၊ Case Active သည် ထူးခြားသောလင့်ခ်တစ်ခုကို ထုတ်ပေးသည်။

အောက်ပါရွေးချယ်စရာများကို ပြန်လည်သုံးသပ်ပြီးနောက် လင့်ခ်ကိုကူးယူရန် အိုင်ကွန်ကိုနှိပ်ပါ။ သင့်ဝဘ်ဆိုဒ်တွင် လင့်ခ်ကို ထားနိုင်သည် သို့မဟုတ် လင့်ခ်ကို အီးမေးလ် သို့မဟုတ် ဆက်သွယ်ရေးချန်နယ်တစ်ခုမှတစ်ဆင့် တစ်ဦး သို့မဟုတ် တစ်ဦးထက်ပိုသောလူထံ ပေးပို့နိုင်သည်။

အမှုတစ်ခုဖန်တီးခြင်း။

ရွေးချယ်နိုင်သောအားဖြင့်၊ သင်သည် သုံးစွဲသူအသစ်တစ်ခုစီအတွက် ဖြစ်ရပ်တစ်ခုကို အလိုအလျောက်ဖန်တီးနိုင်ပြီး အသုံးပြုသူအခန်းကဏ္ဍကို သတ်မှတ်ပေးနိုင်သည်။

  1. Auto Create Case ကိုနှိပ်ပြီး On ကိုရွေးချယ်ပါ။ Case User Role အကွက်များကို ဖော်ပြသည်။
  2. Case User Role ကို နှိပ်ပြီး ရွေးချယ်မှုတစ်ခုကို ရွေးချယ်ပါ။ (အောက်တွင်ကြည့်ပါ)

လင့်ခ် ပေးပို့ခြင်း။

လင့်ခ်ကို တစ်ဦး သို့မဟုတ် တစ်ဦးထက်ပိုသောသူများနှင့် မျှဝေပါ။ လူတစ်ဦးစီသည် လင့်ခ်ကို နှိပ်ပြီး ၎င်းတို့၏ Case Active အကောင့်ကို အသက်သွင်းရန်အတွက် အလုပ်အသွားအလာကို အပြီးသတ်ပါ။ အသုံးပြုသူသည် စာရင်းသွင်းခြင်းလုပ်ငန်းအသွားအလာကို ပြီးသည်နှင့်၊ Case Active သည် အသုံးပြုသူကို အသုံးပြုသူများစာရင်းသို့ ပေါင်းထည့်သည်။

img

နောက်တစ်ဆင့်

Case Active သည် အကောင့်ဖွင့်ခြင်းလင့်ခ်ဖြင့် သုံးစွဲသူအား အီးမေးလ်အကြောင်းကြားချက်တစ်ခု ပေးပို့သည်။ အသုံးပြုသူသည် လင့်ခ်ကို နှိပ်ပြီး စာရင်းသွင်းမှုအသစ်များအတွက် အလုပ်အသွားအလာကို အပြီးသတ်သည်။

  • ဤ ဆောင်းပါး သည် သုံးစွဲသူအသစ်များအတွက် အကောင့်ဖွင့်ခြင်းလုပ်ငန်းစဉ်ကို ဖော်ပြသည်။
