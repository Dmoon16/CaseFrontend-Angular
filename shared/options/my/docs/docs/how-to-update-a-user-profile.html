အသုံးပြုသူပရိုဖိုင်ကို အပ်ဒိတ်လုပ်နည်း

ဤဆောင်းပါးတွင် အသုံးပြုသူများအား ၎င်းတို့၏ Profile စာမျက်နှာရှိ အချက်အလက်များကို အပ်ဒိတ်လုပ်နည်းကို ပြသထားသည်။

  • အသုံးပြုသူများသည် ၎င်းတို့၏ကိုယ်ပိုင်ပရိုဖိုင်များကို အပ်ဒိတ်လုပ်နိုင်ပါသည်။
  • အက်ဒ်မင်များသည် အသုံးပြုသူတိုင်း၏ Profile စာမျက်နှာကို အပ်ဒိတ်လုပ်နိုင်ပါသည်။

အသုံးပြုသူများအတွက်

ပထမအဆင့်အနေနဲ့ Profile စာမျက်နှာကိုဖွင့်ပါ။ အသုံးပြုသူများနှင့် စီမံခန့်ခွဲသူများသည် ၎င်းတို့၏ကိုယ်ပိုင် Profile စာမျက်နှာကို အပ်ဒိတ်လုပ်ရန် ဤအဆင့်များကို လိုက်နာပါ။

  1. သင်၏ Case Active အကောင့်သို့ ဝင်ရောက်ပါ။
  2. စာမျက်နှာ၏ထိပ်ရှိ Profile တက်ဘ်ကို နှိပ်ပါ။
  3. အပြောင်းအလဲလုပ်ပါ။
  4. Save ကိုနှိပ်ပါ။

အကွက်အများစုသည် စိတ်ကြိုက်ရွေးချယ်နိုင်ပြီး အချို့အကွက်များ လိုအပ်သည်။ လိုအပ်သောအကွက်များမှာ-

  • နာမည်
  • မျိုးနွယ်အမည်
  • အီးမေးလ်လိပ်စာ
  • ဘာသာစကား
  • အချိန်ဇုန်

စီမံခန့်ခွဲသူများအတွက်

အက်ဒ်မင်များသည် အခြားအသုံးပြုသူများအတွက် ပရိုဖိုင်အချက်အလက်ကိုလည်း အပ်ဒိတ်လုပ်နိုင်ပါသည်။ ၎င်းတွင် HOST USER INFO ခေါင်းစဉ်အောက်ရှိ နယ်ပယ်သုံးခုပါဝင်သည်။

  1. သင်၏ Case Active စီမံခန့်ခွဲသူအကောင့်သို့ ဝင်ရောက်ပါ။
  2. အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. ထိပ်တန်းမီနူးရှိ သုံးစွဲသူများကို နှိပ်ပါ။
  4. မီနူးခွဲသို့သွားပြီး သင်ဝင်ရောက်လိုသော အသုံးပြုသူစာရင်းကို ရွေးပါ (တက်ကြွ သို့မဟုတ် ပိတ်ထားသည်)။ မူရင်းဆက်တင်သည် Active ဖြစ်သည်။
  5. စာရင်းတွင် အသုံးပြုသူတစ်ဦးကို ရှာဖွေပြီး တည်းဖြတ်မည်ကို နှိပ်ပါ။
  6. လိုအပ်သလို အပြောင်းအလဲလုပ်ပါ။ (အောက်တွင်ကြည့်ပါ)
  7. Save ကိုနှိပ်ပါ။

အသုံးပြုသူအချက်အလက်ကို ပြောင်းလဲပါ။

အက်ဒ်မင်များသည် အသုံးပြုသူအချက်အလက်ကို ပြောင်းလဲနိုင်သည်။

  1. အသုံးပြုသူအမျိုးအစား (အသုံးပြုသူ သို့မဟုတ် မန်နေဂျာ) သတ်မှတ်ပါ။
  2. အသုံးပြုသူ အခြေအနေကို သတ်မှတ်ပါ (Active to Disabled)။
  3. ထူးခြားသော Tag ID ကိုထည့်ပါ။ (ချန်လှပ်ထားနိုင်သည်)
  4. Save ကိုနှိပ်ပါ။
