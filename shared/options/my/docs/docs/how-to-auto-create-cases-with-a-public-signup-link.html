အများသူငှာ အကောင့်ဖွင့်ခြင်းလင့်ခ်ဖြင့် ဖြစ်ရပ်များကို အလိုအလျောက်ဖန်တီးနည်း

ဤဆောင်းပါးတွင် အကောင့်ဖွင့်ရန်လင့်ခ်တစ်ခု ပေးပို့ခြင်းဖြင့် အက်ဒ်မင်များအား အမှုတစ်ခုဖန်တီးနည်း (အသုံးပြုသူတစ်ဦးကို တစ်ချိန်တည်းတွင် ထည့်ရန်) ကိုပြသထားသည်။

လင့်ခ်ကိုရှာဖွေခြင်း။

  1. စီမံခန့်ခွဲသူအကောင့်သို့ ဝင်ရောက်ပါ။
  2. Apps စာမျက်နှာတွင် အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. စီမံခန့်ခွဲသူ အကန့်တွင်၊ ဆက်တင်များ မီနူး တက်ဘ်ကို နှိပ်ပါ။
  4. စနစ် SETUP မီနူးတဘ်ကို ရွေးထားကြောင်း သေချာပါစေ။

ဆက်တင်များကို ပြင်ဆင်ခြင်း။

USER SIGNUP ခေါင်းစဉ်သို့ ဆင်းပါ။

အကောင့်ဖွင့်ခွင့်ပြုရန်အကွက်ကို On ဟုသတ်မှတ်ထားကြောင်း သေချာပါစေ။ (လိုအပ်သည်) ဤအင်္ဂါရပ်ကို ဖွင့်ထားသောအခါ၊ Case Active သည် ထူးခြားသောလင့်ခ်တစ်ခုကို ထုတ်ပေးသည်။

  1. Auto Create Case ကိုနှိပ်ပြီး On ကိုရွေးချယ်ပါ။ Case User Role အကွက်ကို ပြသသည်။
  2. အခန်းကဏ္ဍတစ်ခုကို ရွေးချယ်ပါ။ (အောက်တွင်ကြည့်ပါ)

ဖြစ်ရပ်မှန်အသုံးပြုသူအခန်းကဏ္ဍ

အသုံးပြုသူ အခန်းကဏ္ဍ သုံးခုရှိသည်။

  • ဖောက်သည်များ (အကြံပြုထားသည်)
  • ပုံသေ
  • ဝန်ထမ်း

လင့်ခ် ပေးပို့ခြင်း။

လင့်ခ်ကို တစ်ဦး သို့မဟုတ် တစ်ဦးထက်ပိုသောလူများနှင့် မျှဝေပါ။ လင့်ခ်ကို နှိပ်ပြီး အကောင့်ဖွင့်ခြင်းကို အပြီးသတ်သော အသုံးပြုသူတိုင်းသည် အမှုတွဲစာမျက်နှာတွင် အမှုတွဲအသစ်တစ်ခု ဖန်တီးရန် ပလပ်ဖောင်းကို အစပျိုးပေးပြီး အမှုတွင် အသုံးပြုသူကို အလိုအလျောက် သတ်မှတ်ပေးသည်။
