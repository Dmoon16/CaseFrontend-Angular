ရှိရင်းစွဲကိစ္စများဖြင့် အသုံးပြုသူ၏တာဝန်များကို သတ်မှတ်ခြင်း သို့မဟုတ် ဖယ်ရှားနည်း

ဤဆောင်းပါးသည် အက်ဒ်မင်များအား သုံးစွဲသူများကို မည်ကဲ့သို့ တာဝန်ပေးပြီး ဖယ်ရှားရမည်ကို ပြသထားသည်။ သုံးစွဲသူဒေတာကို မွမ်းမံနည်းကိုလည်း သင်ပြပေးပါသည်။

ရှိပြီးသားအမှုဖွင့်တယ်။

  1. စီမံခန့်ခွဲသူအကောင့်သို့ ဝင်ရောက်ပါ။
  2. Apps စာမျက်နှာတွင် အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. Cases menu tab ကိုနှိပ်ပါ။
  4. submenu တွင် ACTIVE ကို ရွေးထားကြောင်း သေချာပါစေ။
  5. ACTION ကော်လံသို့ ဆင်းပါ။

img

သတ်မှတ်ပါ သို့မဟုတ် ဖယ်ရှားပါ။

Assign Users ကို ကလစ်နှိပ်ပါ (သို့သော် ပလက်ဖောင်းမဟုတ်ပါ)၊ နောက်စာမျက်နှာတွင်- ADD ကိုနှိပ်ပြီး အလုပ်အသွားအလာကို အပြီးသတ်ပါ။ အသုံးပြုသူတစ်ဦးကို ဖျက်ရန် ဖယ်ရှားရန် နှိပ်ပါ။

ပြင်ဆင်ပါ။

Cases စာမျက်နှာရှိ သုံးစွဲသူဒေတာကို တည်းဖြတ်ရန် သို့မဟုတ် အပ်ဒိတ်လုပ်ရန် မဖြစ်နိုင်ပါ။ ယင်းအစား၊ ဤအဆင့်များကို လိုက်နာပါ-

  1. Users menu tab ကိုနှိပ်ပါ။
  2. အသုံးပြုသူအမည်ကိုရှာပြီး Edit ကိုနှိပ်ပါ။
  3. အပြောင်းအလဲများပြုလုပ်ပြီး SAVE ကိုနှိပ်ပါ။
