Import Tool ဖြင့် အသုံးပြုသူ Assignments များကို သတ်မှတ်ခြင်း၊ ပြင်ဆင်ခြင်း သို့မဟုတ် ဖယ်ရှားနည်း

ဤဆောင်းပါးတွင် အက်ဒမင်များကို တင်သွင်းသည့်တူးလ်ဖြင့် အမှုတွဲအသုံးပြုသူများကို စီမံခန့်ခွဲနည်းကို စီမံခန့်ခွဲသူများကို ပြသထားသည်။

စီမံခန့်ခွဲသူများ လုပ်နိုင်သည်-

  • အသုံးပြုသူများကို ကိစ္စတစ်ခုတွင် သတ်မှတ်ပေးသည်။
  • ကိစ္စတစ်ခုအတွက် သတ်မှတ်ထားပြီးသား အသုံးပြုသူများကို ပြုပြင်မွမ်းမံပါ။
  • ကိစ္စတစ်ခုအတွက် သတ်မှတ်ထားသော အသုံးပြုသူများကို ဖယ်ရှားပါ။

သွင်းကုန်တူးလ်ကိုဖွင့်ခြင်း။

  1. စီမံခန့်ခွဲသူအကောင့်သို့ ဝင်ရောက်ပါ။
  2. အက်ပ်များ စာမျက်နှာတွင် အက်ပ်တစ်ခုကို ရှာဖွေပြီး ADMIN ခလုတ်ကို နှိပ်ပါ။
  3. ပေါင်းစည်းခြင်းတက်ဘ်ကို နှိပ်ပြီး ဖိုင်မီနူးတွင် တင်သွင်းပါ။
  4. NEW ခလုတ်ကိုနှိပ်ပါ။ Add Imports dialog box တွင် ပြထားသည်။

မှတ်တမ်းများဖန်တီးခြင်း။

တစ်ခု သို့မဟုတ် တစ်ခုထက်ပိုသော CSV ဖိုင်များကို ဒေါင်းလုဒ်လုပ်ပါ။ အသုံးပြုသူများ ထည့်ရန်၊ ပြင်ဆင်ရန် သို့မဟုတ် ဖယ်ရှားရန် ဤဖိုင်များကို အသုံးပြုပါ။ Add Imports dialog box ပွင့်ပြီးနောက်-

  1. Type field ကိုနှိပ်ပါ။
  2. drop-down menu မှတဆင့် ဆင်းပြီး ရွေးချယ်စရာတစ်ခုကို ရွေးပါ (အောက်တွင်ကြည့်ပါ)။ လင့်ခ်တစ်ခု ပေါ်လာသည်။
  3. Download နမူနာလင့်ခ်ကို နှိပ်ပါ။ CSV ဖိုင်သည် သင့်စက်ပစ္စည်းသို့ ဒေါင်းလုဒ်လုပ်သည်။

img

ဖိုင်ကိုဖွင့်ပြီး မှတ်တမ်းများထည့်ပါ။ CSV ဖိုင်တွင် ဖော်မတ်ဒေတာ အသုံးပြုနည်းကို လေ့လာရန် ဤနေရာကို နှိပ်ပါ။

ဖိုင်ရွေးချယ်မှုများ

အသုံးပြုသူများအား အမှုတွဲများအဖြစ် သတ်မှတ်ခြင်း * အမှုတွဲတစ်ခုသို့ အသုံးပြုသူများကို ထည့်ရန် ဤရွေးချယ်မှုကို ရွေးချယ်ပါ။ * case_role_order cell ကို ကွက်လပ်ထားပါ။ * အသုံးပြုသူများ စာမျက်နှာသို့ သွားပြီး ဤပုဂ္ဂိုလ်အတွက် Tag ID ကို ကူးယူပါ။ Tag ID အကွက်သည် ကွက်လပ်ဖြစ်နေပါက၊ သီးခြား ID တစ်ခုထည့်ပါ၊ Save ကိုနှိပ်ပါ၊ ထို့နောက် ID ကို CSV ဖိုင်သို့ ကူးယူပါ။

img

ကိစ္စများတွင် သတ်မှတ်ထားသော အသုံးပြုသူများကို မွမ်းမံပြင်ဆင်ပါ * အမှုတွဲ တစ်ခုအတွက် သတ်မှတ်ထားသော လက်ရှိအသုံးပြုသူများအတွက် ဒေတာကို အပ်ဒိတ်လုပ်ရန် ဤရွေးချယ်မှုကို ရွေးချယ်ပါ။ * case_ID ကော်လံအတွက် Case Tag ID ကိုသုံးပါ။

img

အမှုတွဲများတွင် သတ်မှတ်ထားသော အသုံးပြုသူများကို ဖယ်ရှားရန် * အမှုကိစ္စတစ်ခုအတွက် သတ်မှတ်ထားသော အသုံးပြုသူများကို ဖယ်ရှားရန် ဤရွေးချယ်မှုကို ရွေးချယ်ပါ။

img

ဖိုင်ကို တင်ခြင်း။

သင်သည် ဖိုင်တစ်ခုကို ချက်ချင်း အပ်လုဒ်တင်နိုင်သည် သို့မဟုတ် နောက်ပိုင်းရက်စွဲတွင် Add Imports dialog box သို့ ပြန်သွားပြီး ဤအဆင့်များကို ပြီးအောင်လုပ်ပါ။

  1. Add Imports dialog box ကိုဖွင့်ပါ။
  2. CSV ဖိုင်အတွက် သီးသန့်အမည်တစ်ခု ထည့်ပါ။
  3. သင့်စက်မှ ဖိုင်ကို အပ်လုဒ်လုပ်ရန် အပေါင်း (+) အိုင်ကွန်ကို နှိပ်ပါ။
  4. Type အကွက်ကို နှိပ်ပြီး ရွေးချယ်မှုတစ်ခုကို ရွေးချယ်ပါ။ (အပေါ်ကိုကြည့်ပါ)
  5. SAVE ကိုနှိပ်ပါ။

လုပ်ဆောင်နေသည်

ဖိုင်နှင့် စနစ်ဝန်ရှိ မှတ်တမ်းအရေအတွက်ပေါ်မူတည်၍ ပလပ်ဖောင်းသည် တင်သွင်းမှုကို လုပ်ဆောင်ရန် မိနစ်များစွာ ကြာနိုင်သည်။

တင်သွင်းမှုပြီးသွားသောအခါ၊ ပလပ်ဖောင်းသည် Cases စာမျက်နှာရှိ သုံးစွဲသူမှတ်တမ်းများ (ဆိုလိုသည်မှာ ထပ်ထည့်ခြင်း၊ ပြင်ဆင်ခြင်း သို့မဟုတ် ဖယ်ရှားခြင်း) ကို ချိန်ညှိပေးပါသည်။

ပျက်ကွက်ခြင်း။

တင်သွင်းမှုလုပ်ဆောင်မှု မအောင်မြင်ပါက၊ ပလပ်ဖောင်းသည် မအောင်မြင်သော မက်ဆေ့ဂျ်ကို ပြန်ပေးသည်။ ကိစ္စအများစုတွင် ပျက်ကွက်ရခြင်းအကြောင်းရင်းမှာ ဒေတာဖော်မတ်ချခြင်း ဖြစ်သည်။

img

ရှုံးနိမ့်ရခြင်းအကြောင်းရင်းကို ကြည့်ရန်-

  1. လုပ်ဆောင်ချက်ခေါင်းစဉ်အောက်ရှိ View ကိုနှိပ်ပါ။
  2. နောက်စာမျက်နှာတွင် မအောင်မြင်သောအိုင်ကွန်ပေါ်တွင် မောက်စ်ကို ရွှေ့ပါ။ tooltip သည် error ကိုပြသသည်။
  3. CVS ဖိုင်ကိုသွားပြီး အမှားကို ချိန်ညှိပြီး ပြင်ဆင်ပါ။
  4. ဖိုင်ကို ပြန်တင်ပါ။
