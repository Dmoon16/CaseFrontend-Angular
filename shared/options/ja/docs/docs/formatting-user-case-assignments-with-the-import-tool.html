インポート ツールを使用したユーザー ケース割り当てデータのフォーマット

以下は、CSV ファイルのデータ フィールドのリストです。各インポート オプションには、異なるフィールドの組み合わせがあります。

ケースID

  • ケースのユーザーまたはタグ ID を入力します。
  • [ケース] ページ > [編集] > [タグ ID] に移動するか、[ケース] ページ > [編集] > [URL のケース ID] に移動します。

ユーザーID

  • ユーザーの ID またはタグ ID を入力します。
  • [ユーザー] ページ > [編集] > [タグ ID] に移動するか、[ユーザー] ページ > [編集] > [URL のユーザー ID] に移動します。

case_role_order

  • セルを空白のままにします。

case_role_ID

  • 入力: クライアント、スタッフ、またはデフォルト。

case_notify

  • 0 = いいえ
  • 1 = はい、ユーザーに電子メール通知を送信します。
