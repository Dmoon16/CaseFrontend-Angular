インポート ツールを使用してユーザー割り当てを割り当て、変更、または削除する方法

この記事では、管理者がインポート ツールを使用してケース ユーザーを管理する方法について説明します。

管理者は次のことができます。

  • ユーザーをケースに割り当てる
  • ケースにすでに割り当てられているユーザーを変更する
  • ケースに割り当てられたユーザーを削除する

インポート ツールを開く

  1. 管理者アカウントにログインします。
  2. [アプリ] ページでアプリを見つけて、[ADMIN] ボタンをクリックします。
  3. Integration タブをクリックし、サブメニューの IMPORTS をクリックします。
  4. NEW ボタンをクリックします。 [インポートの追加] ダイアログ ボックスが表示されます。

記録の作成

1 つ以上の CSV ファイルをダウンロードします。これらのファイルを使用して、ケース ユーザーを追加、変更、または削除します。 [インポートの追加] ダイアログ ボックスが開いたら、次の手順を実行します。

  1. [タイプ] フィールドをクリックします。
  2. ドロップダウン メニューをスクロールして、オプションを選択します (以下を参照)。リンクが表示されます。
  3. [サンプルのダウンロード] リンクをクリックします。 CSV ファイルがデバイスにダウンロードされます。

ファイルを開き、レコードを追加します。 CSV ファイルでフォーマット データを使用する方法については、ここをクリックしてください。

ファイル オプション

ユーザーをケースに割り当てる * ユーザーをケースに追加するには、このオプションを選択します。 * case_role_order セルは空白のままにします。 * ユーザー ページに移動し、この人物のタグ ID をコピーします。 [タグ ID] フィールドが空白の場合は、一意の ID を入力し、[保存] をクリックして、ID を CSV ファイルにコピーします。

画像

ケースに割り当てられたユーザーの変更 * ケースに割り当てられた現在のユーザーのデータを更新するには、このオプションを選択します。 * case_ID 列にはケース タグ ID を使用します。

ファイルのアップロード

ファイルをすぐにアップロードすることも、後で [インポートの追加] ダイアログ ボックスに戻ってこれらの手順を完了することもできます。

  1. [インポートの追加] ダイアログ ボックスを開きます。
  2. CSV ファイルの一意の名前を入力します。
  3. プラス (+) アイコンをクリックして、デバイスからファイルをアップロードします。
  4. [タイプ] フィールドをクリックして、オプションを選択します。 （上記を参照）
  5. [保存] をクリックします。

処理

ファイル内のレコード数とシステム負荷によっては、プラットフォームがインポートを処理するのに数分かかる場合があります。

画像

インポートが完了すると、プラットフォームは [ケース] ページでユーザー レコードを調整 (つまり、追加、変更、または削除) します。

失敗

インポート操作が失敗した場合、プラットフォームは Failed メッセージを返します。ほとんどの場合、失敗の原因はデータのフォーマットです。

画像

失敗の原因を確認するには:

  1. [アクション] 見出しの下にある [表示] をクリックします。
  2. 次のページで、[Failed] アイコンにマウスを合わせます。ツールチップにエラーが表示されます。
  3. CVS ファイルに移動し、エラーを調整して修正します。
  4. ファイルを再度アップロードします。
