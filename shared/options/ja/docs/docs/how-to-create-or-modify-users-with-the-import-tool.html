インポート ツールを使用してユーザーを作成または変更する方法

この記事では、インポート ツールを使用してユーザーを作成する方法を管理者に示します。

インポート ツールを開く

管理者はインポート ツールを使用して、新しいユーザーを一括アップロードし、既存のユーザー レコードを変更できます。

  1. 管理者アカウントにログインします。
  2. [アプリ] ページでアプリを見つけて、[ADMIN] ボタンをクリックします。
  3. 管理パネルで、[統合] タブをクリックし、サブメニューの [インポート] をクリックします。
  4. NEW ボタンをクリックします。 [インポートの追加] ダイアログ ボックスが新しいブラウザ タブに表示されます。

画像

記録の作成

[インポートの追加] ダイアログ ボックスが開いたら、次の手順を実行します。

  1. [タイプ] フィールドをクリックし、オプションを選択します (以下を参照)。リンクが表示されます。
  2. [サンプルのダウンロード]リンクをクリックします。 CSV ファイルがデバイスにダウンロードされます。

次に、ファイルを開いてレコードを追加します。 CSV ファイルでフォーマット データを使用する方法については、ここをクリックしてください。

画像

タイプ

[タイプ] フィールドをクリックすると、ドロップダウン メニューが表示されます。一括インポートの 2 つのオプションのいずれかを選択します。

:exclamation:注意: ドロップダウン メニューには多くのオプションがありますが、インポート ツールに関連するのは 2 つだけです。各オプションには、異なるサンプル CSV ファイルがあります。

ユーザーを追加

  • ユーザー レコードを簡単に一括アップロードするには、このオプションを選択します。
  • [サンプルのダウンロード] リンクをクリックして、[ユーザーの追加] インポート タイプの列見出しを含む CSV ファイルを開きます。

ユーザーを変更する

  • 現在のユーザーのデータを更新するには、このオプションを選択します。
  • [サンプルのダウンロード] リンクをクリックして、[ユーザーの変更] インポート タイプの列見出しを含む CSV ファイルを開きます。

ファイルのアップロード

ファイルをすぐにアップロードすることも、後で [インポートの追加] ダイアログ ボックスに戻ってこれらの手順を完了することもできます。

  1. [インポートの追加] ダイアログ ボックスを開きます。
  2. CSV ファイルの一意の名前を入力します。
  3. プラス (+) アイコンをクリックして、デバイスからファイルをアップロードします。
  4. [タイプ] フィールドをクリックして、オプションを選択します。 （上記を参照）
  5. [保存] をクリックします。

画像

処理

ファイル内のレコード数とシステム負荷によっては、プラットフォームがインポートを処理するのに数分かかる場合があります。

画像

インポートが完了すると、プラットフォームは管理者のユーザーダッシュボードでユーザー レコードを調整します (つまり、追加、変更、または無効化します)。

失敗

インポート操作が失敗した場合、プラットフォームは Failed メッセージを返します。ほとんどの場合、失敗の原因はデータのフォーマットです。

画像

失敗の原因を確認するには:

  1. [アクション] 見出しの下にある [表示]をクリックします。
  2. 次のページで、[Failed] アイコンにマウスを合わせます。ツールチップにエラーが表示されます。
  3. CVS ファイルに移動し、エラーを調整して修正します。
  4. ファイルを再度アップロードします。
