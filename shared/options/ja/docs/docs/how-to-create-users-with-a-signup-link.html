サインアップ リンクを使用してユーザーを作成する方法

この記事では、サインアップ リンクを送信してユーザーを作成する方法を管理者に示します。

サインアップ リンクの構成

  1. [ユーザーの追加] ダイアログ ボックスの下部にある [サインアップを送信] リンクをクリックします。新しいページが表示されます。
  2. サインアップ オプションを構成します。 （下記参照）

画像

[サインアップを許可]フィールドが [オン] に設定されていることを確認します。 (必須) 有効にすると、Case Active は一意のリンクを生成します。

以下のオプションを確認し、アイコンをクリックしてリンクをコピーします。リンクを Web サイトに配置するか、電子メールまたは任意の通信チャネルを介して 1 人または複数の人にリンクを送信できます。

画像

ケースの作成

必要に応じて、新規ユーザーごとにケースを自動的に作成し、ユーザー ロールを割り当てることができます。

  1. [ケースの自動作成] フィールドをクリックし、[オン] を選択します。 Case User Role フィールドが表示されます。
  2. Case User Role をクリックし、オプションを選択します。 （下記参照）

画像

ケース ユーザーの役割

3 つのケース ユーザー ロールがあります。

  • クライアント
  • デフォルト
  • スタッフ

リンクの送信

1 人以上のユーザーとリンクを共有します。各ユーザーはリンクをクリックしてワークフローを完了し、Case Active アカウントを有効にします。ユーザーがサインアップ ワークフローを完了すると、Case Active はユーザーをユーザー リストに追加します。

画像

次のステップ

Case Active は、サインアップ リンクを含む電子メール通知をユーザーに送信します。ユーザーはリンクをクリックして、新規サインアップのワークフローを完了します。

  • この記事では、新規ユーザーのサインアップ ワークフローについて説明します。
