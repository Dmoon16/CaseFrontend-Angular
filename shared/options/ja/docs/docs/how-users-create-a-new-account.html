ユーザーが新しいアカウントを作成する方法

管理者がユーザーをケースに追加した後: * ユーザーが caseactive を初めて使用する場合、ユーザーはワークフローを完了してアカウントを作成し、ケースにアクセスするための招待を受け入れる必要があります。 * ユーザーが既存のケースアクティブ アカウントを持っている場合、ユーザーはケースにアクセスするための招待を受け入れる必要があります。

この記事では、ユーザーがアカウントを作成するために完了する必要がある手順について説明します。

入門

ケースアクティブ アプリにユーザーを追加する方法は 2 つあります。

  • 管理者による: 管理者がプラットフォーム UIまたはインポート ツールを使用してユーザーを作成する場合、管理者はユーザーに account.caseactive.com へのリンクを送信してログインに進む必要があります。
  • ユーザーごと: ユーザーがサインアップ リンクを使用してアカウントを作成した場合、ユーザーは自動的に account.caseactive.com にリダイレクトされ、ログインに進みます。

ログイン中

ユーザーが CaseActive アカウントに初めてログインすると、プラットフォームはユーザーに次のように求めます。

  • メール アカウントを開き、6 桁のセキュリティ コードをコピーして貼り付けます
  • アカウントのプロフィールを完成させる

画像

ユーザーが [更新] をクリックすると、[アプリ] ページが表示されます。このページには、アプリの名前 (会社名)、ユーザーの種類 (役割)、ステータス、および [招待] ボタンが表示されます。

画像

アプリにアクセスするには、ユーザーは招待を受け入れる必要があります。ユーザーは次のことを行う必要があります。

  1. [招待] ボタンをクリックします。
  2. 権限を確認します。
  3. [通知] タブをクリックして、条件を確認および編集 (オプション) します。
  4. [アクセスを承認] をクリックします。

画像

ユーザーが承認すると、プラットフォームはユーザー ページに APP ボタンを表示します。このボタンにより、ユーザーは割り当てられたケースにアクセスできます。

ユーザー設定を編集するには、ユーザーは省略記号をクリックします。

画像

ユーザーは次のことができます: * アプリへのアクセスを削除する * 通知設定を変更する

画像

ユーザーがワークフローを完了すると、ユーザー名が管理者のユーザー リストに表示されます。リストには、ログインしているかどうかにかかわらず、すべてのユーザーが表示されます。
