インポート ツールを使用してケースを作成または変更する方法

この記事では、インポート ツールを使用して既存のケースを作成または変更する方法を管理者に示します。

インポート ツールを開く

  1. 管理者アカウントにログインします。
  2. [アプリ] ページでアプリを見つけて、[ADMIN] ボタンをクリックします。
  3. 管理パネルで、[統合] タブをクリックし、サブメニューの [インポート] をクリックします。
  4. NEW ボタンをクリックします。 [インポートの追加] ダイアログ ボックスが新しいブラウザ タブに表示されます。

ケースの作成

[インポートの追加] ダイアログ ボックスが開いたら、次の手順を実行します。

  1. [タイプ] フィールドをクリックし、オプションを選択します (以下を参照)。リンクが表示されます。
  2. [サンプルのダウンロード] リンクをクリックします。 CSV ファイルがデバイスにダウンロードされます。

次に、ファイルを開いてケースを追加します。

タイプ

[タイプ] フィールドをクリックすると、ドロップダウン メニューが表示されます。ケースの一括インポートの 2 つのオプションのいずれかを選択します。

  • 注ドロップダウン メニューには多くのオプションがありますが、ケースの管理に関連するオプションは 2 つだけです。各オプションには、異なるサンプル CSV ファイルがあります。

ケースを追加

  • ケースを簡単に一括アップロードするには、このオプションを選択します。
  • [サンプルのダウンロード] リンクをクリックして、[ケースの作成] インポート タイプの列見出しを含む CSV ファイルを開きます。

ケースの変更

  • 現在のケースを更新するには、このオプションを選択します。
  • [サンプルのダウンロード] リンクをクリックして、[ケースの変更] インポート タイプの列見出しを含む CSV ファイルを開きます。

ファイルのアップロード

ファイルをすぐにアップロードすることも、後で [インポートの追加] ダイアログ ボックスに戻ってこれらの手順を完了することもできます。

  1. [インポートの追加] ダイアログ ボックスを開きます。
  2. CSV ファイルの一意の名前を入力します。
  3. プラス (+) アイコンをクリックして、デバイスからファイルをアップロードします。
  4. [タイプ] フィールドをクリックして、オプションを選択します。 （上記を参照）
  5. [保存] をクリックします。

書式設定

これらは必須フィールドであり、最初の要件です。フォーマットの詳細を確認するには、ここをクリックしてください。

tag_id (または case_ID)一意の識別子 (英数字を使用します。ハイフンを使用できます)

case_status 0 = クローズ 1 = アクティブ

date_opened月/日/年 mm/dd/yyyy

ケースの固有の説明について(英数字を使用し、ハイフンを使用できます)

処理

ファイル内のレコード数とシステム負荷によっては、プラットフォームがインポートを処理するのに数分かかる場合があります。

画像

インポートが完了すると、プラットフォームは管理者の Cases ダッシュボードでケース レコードを調整 (つまり、追加または変更) します。

失敗

インポート操作が失敗した場合、プラットフォームは Failed メッセージを返します。ほとんどの場合、失敗の原因はデータのフォーマットです。

画像

失敗の原因を確認するには:

  1. [アクション] 見出しの下にある [表示] をクリックします。
  2. 次のページで、[Failed] アイコンにマウスを合わせます。ツールチップにエラーが表示されます。
  3. CVS ファイルに移動し、エラーを調整して修正します。
  4. ファイルを再度アップロードします。
