Qhib cov cuab yeej Import

  1. Nkag mus rau hauv tus account admin.
  2. Nrhiav ib qho app ntawm nplooj ntawv Apps thiab nias lub pob ADMIN.
  3. Nyob rau ntawm Admin vaj huam sib luag, nyem rau ntawm Integration tab thiab ces IMPORTS ntawm lub submenu.
  4. Nyem lub pob tshiab. Qhov Add Imports dialog box qhia hauv qhov browser tshiab.

Tsim rooj plaub

Tom qab Add Imports dialog box qhib:

  1. Nyem rau hom teb thiab xaiv ib qho kev xaiv (saib hauv qab). Ib qhov txuas qhia.
  2. Nyem qhov Download Sample link. CSV cov ntaub ntawv rub tawm rau koj lub cuab yeej.

Tom ntej no, qhib cov ntaub ntawv thiab ntxiv rooj plaub.

Hom

Thaum koj nyem rau hom teb, ib daim ntawv qhia nco-down qhia. Xaiv ib qho ntawm ob txoj kev xaiv rau cov ntaub ntawv loj tuaj.

  • Nco ntsoov Cov ntawv qhia zaub mov muaj ntau txoj kev xaiv, tab sis tsuas yog ob qho cuam tshuam nrog kev tswj hwm rooj plaub. Txhua qhov kev xaiv muaj cov qauv CSV sib txawv.

Ntxiv rooj plaub

  • Xaiv qhov kev xaiv no rau ib qho yooj yim bulk upload ntawm rooj plaub.
  • Nyem qhov Download Sample txuas mus qhib CSV cov ntaub ntawv nrog kab ntawv kab lus rau Tsim cov ntaub ntawv ntshuam hom.

Hloov cov rooj plaub

  • Xaiv qhov kev xaiv no los hloov kho cov xwm txheej tam sim no.
  • Nyem qhov Download Sample txuas mus qhib CSV cov ntaub ntawv nrog kab ntawv kab lus rau Hloov kho cov ntaub ntawv ntshuam hom.

Uploading cov ntaub ntawv

Koj tuaj yeem xa cov ntaub ntawv tam sim ntawd lossis rov qab mus rau Add Imports dialog box ntawm hnub tom qab thiab ua tiav cov kauj ruam no.

  1. Qhib Add Imports dialog box.
  2. Sau ib lub npe tshwj xeeb rau cov ntaub ntawv CSV.
  3. Nyem qhov ntxiv (+) icon kom upload cov ntaub ntawv los ntawm koj lub cuab yeej.
  4. Nyem rau hom teb thiab xaiv ib qho kev xaiv. (saib saum toj no)
  5. Nyem SAVE.
