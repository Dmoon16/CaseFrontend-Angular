របៀបចាត់តាំង ឬលុបការចាត់តាំងរបស់អ្នកប្រើជាមួយករណីដែលមានស្រាប់

អត្ថបទនេះបង្ហាញអ្នកគ្រប់គ្រងពីរបៀបចាត់តាំង និងលុបអ្នកប្រើប្រាស់ករណីចេញ។ យើងក៏បង្ហាញអ្នកពីរបៀបកែប្រែទិន្នន័យអ្នកប្រើប្រាស់ផងដែរ។

ការបើកករណីដែលមានស្រាប់

  1. ចូលទៅគណនីអ្នកគ្រប់គ្រង។
  2. ស្វែងរកទីតាំងកម្មវិធីនៅលើទំព័រកម្មវិធី ហើយចុចប៊ូតុង ADMIN ។
  3. ចុចលើផ្ទាំងម៉ឺនុយ ករណី ។
  4. សូមប្រាកដថា ACTIVE ត្រូវបានជ្រើសរើសនៅក្នុងម៉ឺនុយរង។
  5. រំកិលចុះក្រោមទៅជួរ ACTION ។

ចាត់តាំង ឬដកចេញ

ចុច កំណត់អ្នកប្រើប្រាស់ ដើម្បីចាត់ឱ្យអ្នកប្រើប្រាស់ទៅករណី ឬដកអ្នកប្រើប្រាស់ចេញពីករណី (ប៉ុន្តែមិនមែនវេទិកា)។ នៅទំព័របន្ទាប់៖ ចុច បន្ថែម ហើយបញ្ចប់ដំណើរការការងារ។ ចុច លុប ដើម្បីលុបអ្នកប្រើប្រាស់។

កែប្រែ

វាមិនអាចកែសម្រួល ឬធ្វើបច្ចុប្បន្នភាពទិន្នន័យអ្នកប្រើប្រាស់នៅលើទំព័រករណីបានទេ។ ជំនួសមកវិញ សូមអនុវត្តតាមជំហានទាំងនេះ៖

  1. ចុចផ្ទាំងម៉ឺនុយ អ្នកប្រើប្រាស់ ។
  2. ស្វែងរកឈ្មោះអ្នកប្រើប្រាស់ ហើយចុច កែសម្រួល ។
  3. ធ្វើការផ្លាស់ប្តូរ ហើយចុច រក្សាទុក។
