របៀបបង្កើតករណីដោយស្វ័យប្រវត្តិជាមួយនឹងតំណភ្ជាប់ចុះឈ្មោះសាធារណៈ

អត្ថបទនេះបង្ហាញអ្នកគ្រប់គ្រងពីរបៀបបង្កើតករណី (និងបន្ថែមអ្នកប្រើប្រាស់ក្នុងពេលតែមួយ) ដោយផ្ញើតំណចុះឈ្មោះតែមួយ។

កំណត់ទីតាំងតំណ

  1. ចូលទៅគណនីអ្នកគ្រប់គ្រង។
  2. ស្វែងរកទីតាំងកម្មវិធីនៅលើទំព័រកម្មវិធី ហើយចុចប៊ូតុង ADMIN ។
  3. នៅលើផ្ទាំងគ្រប់គ្រង ចុចលើផ្ទាំងម៉ឺនុយការកំណត់។
  4. ត្រូវប្រាកដថាផ្ទាំងម៉ឺនុយរង SETUP ប្រព័ន្ធត្រូវបានជ្រើសរើស។

រំកិលចុះក្រោមទៅផ្នែក USER SIGNUP ។

ត្រូវប្រាកដថាវាលអនុញ្ញាតការចុះឈ្មោះត្រូវបានកំណត់ទៅបើក។ (ទាមទារ) នៅពេលដែលមុខងារនេះត្រូវបានបើក Case Active បង្កើតតំណភ្ជាប់តែមួយគត់។

  1. ចុចលើវាល បង្កើតករណីដោយស្វ័យប្រវត្តិ ហើយជ្រើសរើស បើក។ ប្រអប់តួនាទីអ្នកប្រើប្រាស់ករណីបង្ហាញ។
  2. ជ្រើសរើសតួនាទីមួយ។ (មើល​ខាង​ក្រោម)
