ວິທີການມອບຫມາຍຜູ້ໃຊ້ໃນຂະນະທີ່ສ້າງກໍລະນີ

ບົດຄວາມນີ້ສະແດງໃຫ້ເຫັນຜູ້ເບິ່ງແຍງວິທີການມອບຫມາຍຜູ້ໃຊ້ໃນຂະນະທີ່ສ້າງກໍລະນີໃຫມ່. ຖ້າຜູ້ໃຊ້ຍັງບໍ່ໄດ້ລົງທະບຽນຢູ່ໃນເວທີ, ທ່ານສາມາດສ້າງເພີ່ມຜູ້ໃຊ້ໃຫມ່ໃນຂະນະທີ່ສ້າງກໍລະນີໃຫມ່.

ການສ້າງກໍລະນີ

  1. ເຂົ້າສູ່ລະບົບບັນຊີ admin.
  2. ຊອກຫາແອັບຯໃນຫນ້າ Apps ແລະຄລິກໃສ່ປຸ່ມ ADMIN.
  3. ຄລິກແຖບເມນູກໍລະນີ.
  4. ໃຫ້ຄລິກໃສ່ປຸ່ມ NEW. ໜ້າຈໍໃໝ່ສະແດງ.
  • ຕື່ມຂໍ້ມູນໃສ່ໃນຊ່ອງຂໍ້ມູນເພື່ອສ້າງກໍລະນີ ແລະຈາກນັ້ນເລື່ອນລົງໄປຫາຫົວຂໍ້ຜູ້ໃຊ້ທີ່ໄດ້ຮັບມອບໝາຍ.

ການແຕ່ງຕັ້ງຜູ້ໃຊ້

ຜູ້ໃຊ້ປະຈຸບັນ

ປະຕິບັດຕາມຂັ້ນຕອນເຫຼົ່ານີ້ເພື່ອມອບຫມາຍຜູ້ໃຊ້ປະຈຸບັນໃຫ້ກັບກໍລະນີໃຫມ່.

ຄລິກທີ່ປຸ່ມ ASSIGN USERS ແລະຕື່ມຂໍ້ມູນໃສ່ໃນຊ່ອງຂໍ້ມູນ.

  1. ຄລິກທີ່ເມນູເລື່ອນລົງຜູ້ໃຊ້ ແລະເລືອກຕົວລະບຸຕົວຕົນ (ເຊັ່ນ: ອີເມວ).
  2. ໃສ່ຄ່າໃນຊ່ອງຫວ່າງ (ເຊັ່ນ: ທີ່ຢູ່ອີເມວ).
  3. ຖ້າບັນທຶກຜູ້ໃຊ້ຢູ່ໃນເວທີ, ພາກສະຫນາມຈະແນະນໍາຊື່ອັດຕະໂນມັດ.
  4. ເລືອກຜູ້ໃຊ້ທີ່ຖືກຕ້ອງ.
  5. ເລືອກບົດບາດຂອງຜູ້ໃຊ້ (ລູກຄ້າ, ຄ່າເລີ່ມຕົ້ນ, ຫຼືພະນັກງານ).
  6. ທາງເລືອກອື່ນ, ໃຫ້ກວດເບິ່ງກ່ອງເພື່ອສົ່ງອີເມວແຈ້ງເຕືອນຜູ້ໃຊ້.
  7. ກົດ SAVE.

Case Active ເພີ່ມຜູ້ໃຊ້ເຂົ້າໃນກໍລະນີ. ຕອນນີ້ຜູ້ເບິ່ງແຍງລະບົບສາມາດຄລິກບັນທຶກເພື່ອປິດໜ້າຈໍ ຫຼືເພີ່ມຜູ້ໃຊ້ໃໝ່ ຫຼືລຶບຜູ້ໃຊ້ອອກໄດ້.

ກຳລັງເພີ່ມຜູ້ໃຊ້ໃໝ່

ຖ້າເວທີບໍ່ສາມາດຊອກຫາບັນທຶກປະຈຸບັນ, ໃຫ້ຄລິກໃສ່ປຸ່ມເພື່ອເພີ່ມຜູ້ໃຊ້ໃຫມ່.

img

ຕື່ມຂໍ້ມູນໃສ່ໃນຊ່ອງຂໍ້ມູນແລະຫຼັງຈາກນັ້ນໃຫ້ຄລິກໃສ່ SAVE ເພື່ອເພີ່ມຜູ້ໃຊ້ເຂົ້າໃນເວທີ. ຕໍ່ໄປ, ກໍານົດບົດບາດຂອງຜູ້ໃຊ້ແລະຄລິກ SAVE ອີກເທື່ອຫນຶ່ງເພື່ອມອບຫມາຍຜູ້ໃຊ້ໃຫ້ກັບກໍລະນີ. ສຸດທ້າຍ, ຄລິກ SAVE ອີກຄັ້ງໜຶ່ງເພື່ອບັນທຶກກໍລະນີ.
