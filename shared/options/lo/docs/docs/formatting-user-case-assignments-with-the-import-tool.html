ການຈັດຮູບແບບຂໍ້ມູນການມອບໝາຍກໍລະນີຜູ້ໃຊ້ດ້ວຍເຄື່ອງມືນຳເຂົ້າ

ຂ້າງລຸ່ມນີ້ແມ່ນບັນຊີລາຍຊື່ຂອງຊ່ອງຂໍ້ມູນໃນໄຟລ໌ CSV. ແຕ່ລະທາງເລືອກການນໍາເຂົ້າມີການປະສົມປະສານທີ່ແຕກຕ່າງກັນຂອງພາກສະຫນາມ.

case_ID

  • ໃສ່ກໍລະນີ User ຫຼື Tag ID.
  • ໄປ​ທີ່​ຫນ້າ​ກໍ​ລະ​ນີ​> ແກ້​ໄຂ​> Tag ID ຫຼື​ໄປ​ທີ່​ຫນ້າ​ກໍ​ລະ​ນີ​> ແກ້​ໄຂ​> Case ID ໃນ url​.

user_ID

  • ໃສ່ ID ຫຼື Tag ID ຂອງຜູ້ໃຊ້.
  • ໄປ​ທີ່​ຫນ້າ​ຜູ້​ໃຊ້​> ແກ້​ໄຂ​> Tag ID ຫຼື​ໄປ​ທີ່​ຫນ້າ​ຜູ້​ໃຊ້​> ແກ້​ໄຂ​> User ID ໃນ url​.

case_role_ID
