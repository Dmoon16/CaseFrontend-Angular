Admins ຈັດການຜູ້ໃຊ້ແນວໃດ

ຜູ້ເບິ່ງແຍງສ້າງ, ແກ້ໄຂ ຫຼືປິດນຳໃຊ້ຜູ້ທີ່ສາມາດເຂົ້າເຖິງແອັບ ແລະກໍລະນີ. ເມື່ອຜູ້ໃຊ້ເຂົ້າສູ່ລະບົບໃນເວທີຄັ້ງທໍາອິດ, ພວກເຂົາຈໍາເປັນຕ້ອງເຮັດສໍາເລັດຂະບວນການເຮັດວຽກຂອງໂປຣໄຟລ໌.

  • ຄລິກ ບ່ອນນີ້ ເພື່ອສຶກສາເພີ່ມເຕີມກ່ຽວກັບຂັ້ນຕອນການເຮັດວຽກໂປຣໄຟລ໌ຜູ້ໃຊ້ໃໝ່.

ການຄຸ້ມຄອງຜູ້ໃຊ້

ຕາຕະລາງຂ້າງລຸ່ມນີ້ສະແດງໃຫ້ເຫັນຫ້າວິທີໃນການຄຸ້ມຄອງຜູ້ໃຊ້. ຄລິກລິ້ງເພື່ອຮຽນຮູ້ວິທີ.

ການແຈ້ງເຕືອນ

ແພລະຕະຟອມບໍ່ໄດ້ສົ່ງການແຈ້ງເຕືອນຜູ້ໃຊ້ເມື່ອພວກເຂົາຖືກເພີ່ມເຂົ້າໃນແອັບຯ. ຢ່າງໃດກໍຕາມ, ຜູ້ເບິ່ງແຍງສາມາດເປີດໃຊ້ການແຈ້ງເຕືອນສໍາລັບການມອບຫມາຍກໍລະນີ.
