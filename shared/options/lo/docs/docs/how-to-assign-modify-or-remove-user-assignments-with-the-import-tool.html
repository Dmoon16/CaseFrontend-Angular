ວິທີການມອບຫມາຍ, ດັດແກ້ຫຼືເອົາການມອບຫມາຍຂອງຜູ້ໃຊ້ອອກດ້ວຍເຄື່ອງມືນໍາເຂົ້າ

ບົດຄວາມນີ້ສະແດງໃຫ້ເຫັນຜູ້ເບິ່ງແຍງວິທີການຈັດການຜູ້ໃຊ້ກໍລະນີດ້ວຍເຄື່ອງມືນໍາເຂົ້າ.

ຜູ້ເບິ່ງແຍງສາມາດ:

  • ມອບໝາຍໃຫ້ຜູ້ໃຊ້ເປັນກໍລະນີ
  • ປັບປຸງແກ້ໄຂຜູ້ໃຊ້ທີ່ຖືກມອບຫມາຍໃຫ້ກໍລະນີ
  • ເອົາຜູ້ໃຊ້ທີ່ຖືກມອບຫມາຍໃຫ້ກັບກໍລະນີ

ເປີດ​ເຄື່ອງ​ມື​ການ​ນໍາ​ເຂົ້າ​

  1. ເຂົ້າສູ່ລະບົບບັນຊີ admin.
  2. ຊອກຫາແອັບຯໃນຫນ້າ Apps ແລະຄລິກໃສ່ປຸ່ມ ADMIN.
  3. ຄລິກແຖບການລວມເຂົ້າກັນ ແລະຫຼັງຈາກນັ້ນ IMPORTS ໃນເມນູຍ່ອຍ.
  4. ໃຫ້ຄລິກໃສ່ປຸ່ມ NEW. ກ່ອງໂຕ້ຕອບ Add Imports ສະແດງ.

ການສ້າງບັນທຶກ

ດາວໂຫຼດໜຶ່ງ ຫຼືຫຼາຍໄຟລ໌ CSV. ໃຊ້ໄຟລ໌ເຫຼົ່ານີ້ເພື່ອເພີ່ມ, ແກ້ໄຂ, ຫຼືລຶບຜູ້ໃຊ້ກໍລະນີ. ຫຼັງຈາກກ່ອງໂຕ້ຕອບ Add Imports ເປີດຂຶ້ນ:

  1. ໃຫ້ຄລິກໃສ່ປະເພດພາກສະຫນາມ.
  2. ເລື່ອນຜ່ານເມນູເລື່ອນລົງແລະເລືອກທາງເລືອກ (ເບິ່ງຂ້າງລຸ່ມນີ້). ການເຊື່ອມຕໍ່ສະແດງ.
  3. ຄລິກທີ່ລິ້ງດາວໂຫຼດຕົວຢ່າງ. ໄຟລ໌ CSV ດາວໂຫຼດໃສ່ອຸປະກອນຂອງທ່ານ.

ກຳລັງອັບໂຫລດໄຟລ໌

ທ່ານສາມາດອັບໂຫລດໄຟລ໌ໄດ້ທັນທີ ຫຼືກັບຄືນໄປທີ່ກ່ອງໂຕ້ຕອບເພີ່ມການນໍາເຂົ້າໃນພາຍຫຼັງ ແລະເຮັດສໍາເລັດຂັ້ນຕອນເຫຼົ່ານີ້.

  1. ເປີດກ່ອງໂຕ້ຕອບເພີ່ມການນໍາເຂົ້າ.
  2. ໃສ່ຊື່ສະເພາະສໍາລັບໄຟລ໌ CSV.
  3. ຄລິກໄອຄອນບວກ (+) ເພື່ອອັບໂຫລດໄຟລ໌ຈາກອຸປະກອນຂອງທ່ານ.
  4. ໃຫ້ຄລິກໃສ່ປະເພດພາກສະຫນາມແລະເລືອກເອົາທາງເລືອກ. (ເບິ່ງ​ຢູ່​ຂ້າງ​ເທິງ)
  5. ກົດ SAVE.

ການປຸງແຕ່ງ

ອີງຕາມຈໍານວນບັນທຶກໃນໄຟລ໌ແລະການໂຫຼດຂອງລະບົບ, ເວທີສາມາດໃຊ້ເວລາຫຼາຍນາທີເພື່ອປະມວນຜົນການນໍາເຂົ້າ.

ເມື່ອການນໍາເຂົ້າສໍາເລັດ, ເວທີຈະປັບບັນທຶກຜູ້ໃຊ້ (ເຊັ່ນ: ເພີ່ມ, ແກ້ໄຂ, ຫຼືເອົາອອກ) ໃນຫນ້າກໍລະນີ.

ຄວາມລົ້ມເຫຼວ

ຖ້າການປະຕິບັດການນໍາເຂົ້າລົ້ມເຫລວ, ເວທີຈະສົ່ງຄືນຂໍ້ຄວາມທີ່ລົ້ມເຫລວ. ໃນກໍລະນີຫຼາຍທີ່ສຸດ, ສາເຫດຂອງຄວາມລົ້ມເຫຼວແມ່ນການຈັດຮູບແບບຂໍ້ມູນ.

img

ເພື່ອເບິ່ງສາເຫດຂອງຄວາມລົ້ມເຫລວ:

  1. ກົດເບິ່ງພາຍໃຕ້ຫົວຂໍ້ການປະຕິບັດ.
  2. ໃນໜ້າຕໍ່ໄປ, ໃຫ້ເລື່ອນເມົ້າໄປທີ່ໄອຄອນທີ່ລົ້ມເຫລວ. ຄໍາແນະນໍາເຄື່ອງມືສະແດງຂໍ້ຜິດພາດ.
  3. ໄປທີ່ໄຟລ໌ CVS ແລະປັບແລະແກ້ໄຂຂໍ້ຜິດພາດ.
  4. ອັບໂຫຼດໄຟລ໌ອີກຄັ້ງ.
