ວິທີການສ້າງຫຼືແກ້ໄຂກໍລະນີດ້ວຍເຄື່ອງມືນໍາເຂົ້າ

ບົດຄວາມນີ້ສະແດງໃຫ້ເຫັນຜູ້ເບິ່ງແຍງວິທີການສ້າງຫຼືແກ້ໄຂກໍລະນີທີ່ມີຢູ່ແລ້ວດ້ວຍເຄື່ອງມືນໍາເຂົ້າ.

ເປີດ​ເຄື່ອງ​ມື​ການ​ນໍາ​ເຂົ້າ​

  1. ເຂົ້າສູ່ລະບົບບັນຊີ admin.
  2. ຊອກຫາແອັບຯໃນຫນ້າ Apps ແລະຄລິກໃສ່ປຸ່ມ ADMIN.
  3. ໃນກະດານ Admin, ໃຫ້ຄລິກໃສ່ແຖບການລວມເຂົ້າກັນແລະຫຼັງຈາກນັ້ນ IMPORTS ໃນເມນູຍ່ອຍ.
  4. ໃຫ້ຄລິກໃສ່ປຸ່ມ NEW. ກ່ອງໂຕ້ຕອບ Add Imports ສະແດງຢູ່ໃນແຖບຕົວທ່ອງເວັບໃຫມ່.

ການສ້າງກໍລະນີ

ຫຼັງຈາກກ່ອງໂຕ້ຕອບ Add Imports ເປີດ:

  1. ໃຫ້ຄລິກໃສ່ປະເພດພາກສະຫນາມແລະເລືອກທາງເລືອກ (ເບິ່ງຂ້າງລຸ່ມນີ້). ການເຊື່ອມຕໍ່ສະແດງ.
  2. ຄລິກທີ່ລິ້ງດາວໂຫຼດຕົວຢ່າງ. ໄຟລ໌ CSV ດາວໂຫຼດໃສ່ອຸປະກອນຂອງທ່ານ.

ຕໍ່ໄປ, ເປີດໄຟລ໌ແລະເພີ່ມກໍລະນີ.

ປະເພດ

ເມື່ອ​ທ່ານ​ຄລິກ​ໃສ່​ພາກ​ສະ​ຫນາມ ​ປະ​ເພດ ​, ເປັນ​ເມ​ນູ​ເລື່ອນ​ລົງ​ສະ​ແດງ​. ເລືອກຫນຶ່ງໃນສອງທາງເລືອກສໍາລັບການນໍາເຂົ້າກໍລະນີຫຼາຍ.

  • ຫມາຍເຫດ ເມນູເລື່ອນລົງມີຫຼາຍທາງເລືອກ, ແຕ່ວ່າມີພຽງແຕ່ສອງອັນທີ່ກ່ຽວຂ້ອງກັບການຈັດການກໍລະນີ. ແຕ່ລະທາງເລືອກມີໄຟລ໌ CSV ຕົວຢ່າງທີ່ແຕກຕ່າງກັນ.

ເພີ່ມກໍລະນີ

  • ເລືອກຕົວເລືອກນີ້ສຳລັບການອັບໂຫລດກໍລະນີທີ່ງ່າຍດາຍ.
  • ຄລິກທີ່ລິ້ງດາວໂຫຼດຕົວຢ່າງເພື່ອເປີດໄຟລ໌ CSV ທີ່ມີຫົວຂໍ້ຖັນສໍາລັບປະເພດການນໍາເຂົ້າກໍລະນີສ້າງ.

ແກ້ໄຂກໍລະນີ

  • ເລືອກຕົວເລືອກນີ້ເພື່ອອັບເດດກໍລະນີປັດຈຸບັນ.
  • ຄລິກທີ່ລິ້ງດາວໂຫຼດຕົວຢ່າງເພື່ອເປີດໄຟລ໌ CSV ທີ່ມີຫົວຂໍ້ຖັນສໍາລັບປະເພດການນໍາເຂົ້າການດັດແກ້ກໍລະນີ.

ກຳລັງອັບໂຫລດໄຟລ໌

ທ່ານສາມາດອັບໂຫລດໄຟລ໌ໄດ້ທັນທີ ຫຼືກັບຄືນໄປທີ່ກ່ອງໂຕ້ຕອບເພີ່ມການນໍາເຂົ້າໃນພາຍຫຼັງ ແລະເຮັດສໍາເລັດຂັ້ນຕອນເຫຼົ່ານີ້.

  1. ເປີດກ່ອງໂຕ້ຕອບເພີ່ມການນໍາເຂົ້າ.
  2. ໃສ່ຊື່ສະເພາະສໍາລັບໄຟລ໌ CSV.
  3. ຄລິກໄອຄອນບວກ (+) ເພື່ອອັບໂຫລດໄຟລ໌ຈາກອຸປະກອນຂອງທ່ານ.
  4. ໃຫ້ຄລິກໃສ່ປະເພດພາກສະຫນາມແລະເລືອກເອົາທາງເລືອກ. (ເບິ່ງ​ຢູ່​ຂ້າງ​ເທິງ)
  5. ກົດ SAVE.

ການປຸງແຕ່ງ

ອີງຕາມຈໍານວນບັນທຶກໃນໄຟລ໌ແລະການໂຫຼດຂອງລະບົບ, ເວທີສາມາດໃຊ້ເວລາຫຼາຍນາທີເພື່ອດໍາເນີນການນໍາເຂົ້າ.

img

ເມື່ອການນໍາເຂົ້າສໍາເລັດ, ເວທີຈະປັບບັນທຶກກໍລະນີ (ເຊັ່ນ: ເພີ່ມຫຼືດັດແກ້) ໃນ dashboard Cases ຂອງຜູ້ເບິ່ງແຍງ.

ຄວາມລົ້ມເຫຼວ

ຖ້າການປະຕິບັດການນໍາເຂົ້າລົ້ມເຫລວ, ເວທີຈະສົ່ງຄືນຂໍ້ຄວາມທີ່ລົ້ມເຫລວ. ໃນກໍລະນີຫຼາຍທີ່ສຸດ, ສາເຫດຂອງຄວາມລົ້ມເຫຼວແມ່ນການຈັດຮູບແບບຂໍ້ມູນ.

img

ເພື່ອເບິ່ງສາເຫດຂອງຄວາມລົ້ມເຫລວ:

  1. ຄລິກເບິ່ງພາຍໃຕ້ຫົວຂໍ້ ACTION.
  2. ໃນໜ້າຕໍ່ໄປ, ໃຫ້ເລື່ອນເມົ້າໄປທີ່ໄອຄອນທີ່ລົ້ມເຫລວ. ຄໍາແນະນໍາເຄື່ອງມືສະແດງຂໍ້ຜິດພາດ.
  3. ໄປທີ່ໄຟລ໌ CVS ແລະປັບແລະແກ້ໄຂຂໍ້ຜິດພາດ.
  4. ອັບໂຫຼດໄຟລ໌ອີກຄັ້ງ.
