ວິທີການສ້າງ App

ບົດຄວາມນີ້ສະແດງໃຫ້ເຫັນຜູ້ໃຊ້ໃຫມ່ວິທີການສ້າງບັນຊີທົດລອງຟຣີ. Case Active ໃຫ້ສິດທິຜູ້ເບິ່ງແຍງຜູ້ໃຊ້ໃໝ່ສຳລັບບັນຊີນີ້ໂດຍອັດຕະໂນມັດ.

ການສ້າງບັນຊີ

  1. ໄປ​ທີ່ ​ຫນ້າ​ເຂົ້າ​ສູ່​ລະ​ບົບ .
  2. ໃຫ້ຄລິກໃສ່ປຸ່ມເລີ່ມຕົ້ນ.
  3. ໃສ່ທີ່ຢູ່ອີເມວ.
  4. ກົດຕໍ່ໄປ.

ຢືນຢັນບັນຊີ

  1. ເປີດແຖບຕົວທ່ອງເວັບໃຫມ່ແລະເຂົ້າສູ່ລະບົບບັນຊີອີເມວຂອງທ່ານ.
  2. ສຳເນົາລະຫັດຫົກຕົວເລກ.
  3. ກັບໄປທີ່ແຖບ Case Active.
  4. ວາງລະຫັດໃສ່ໃນຊ່ອງຂໍ້ມູນ. ໜ້າຈໍໃໝ່ສະແດງ.

ໃນ​ປ່ອງ​ຢ້ຽມ​ໃຫມ່​:

  1. ຕື່ມຂໍ້ມູນໃສ່ໃນຊ່ອງຂໍ້ມູນທີ່ຕ້ອງການ.
  2. ອ່ານເງື່ອນໄຂການໃຫ້ບໍລິການແລະກວດເບິ່ງກ່ອງເພື່ອຕົກລົງ.
  3. ຄລິກສ້າງບັນຊີຟຣີ.

ກຳລັງສຳເລັດການຕັ້ງຄ່າແອັບ

ຂັ້ນຕອນການເຮັດວຽກຈະສະແດງຢູ່ໃນໜ້າຕໍ່ໄປ. ເຮັດຂັ້ນຕອນຕ່າງໆເພື່ອສຳເລັດການຕັ້ງຄ່າແອັບຯຂອງທ່ານ. ທ່ານສາມາດກັບຄືນໄປຫາໜ້າເຫຼົ່ານີ້ເພື່ອແກ້ໄຂ ຫຼືອັບເດດຂໍ້ມູນ ຫຼືສະຖານະບັນຊີຂອງທ່ານໄດ້ສະເໝີ.

ໃນຫນ້າກ່ຽວກັບ App, ຕື່ມຂໍ້ມູນໃສ່ໃນຊື່ບໍລິສັດແລະເວັບໄຊທ໌ (ຕ້ອງການ) ແລະຫຼັງຈາກນັ້ນໃຫ້ຄລິກໃສ່ຕໍ່ໄປ.

ໃນຫນ້າຂໍ້ມູນການເອີ້ນເກັບເງິນ, ໃຫ້ຄລິກໃສ່ຊ່ອງ Plan Payment ແລະທົບທວນຄືນທາງເລືອກ. ການຕັ້ງຄ່າເລີ່ມຕົ້ນແມ່ນບໍ່ເສຍຄ່າ. ກົດ SUBMIT.

img

ຫຼັງຈາກທີ່ທ່ານສົ່ງແບບຟອມ, Case Active ຈະສະແດງໜ້າແອັບ. ສຳລັບແຕ່ລະແອັບທີ່ທ່ານສ້າງ ຫຼືຂຶ້ນກັບ, ໜ້າຈະສະແດງ:

  1. ຊື່ app
  2. ປະເພດຂອງທ່ານ (ບົດບາດຂອງຜູ້ໃຊ້)
  3. ສະຖານະ

img

ຫນ້າ Apps ເປັນຈຸດເປີດຕົວສໍາລັບຫນ້າທີ່ admin ຫຼາຍ:

  1. ຄລິກປຸ່ມ APP ເພື່ອຕິດຕໍ່ສື່ສານ, ຕື່ມຂໍ້ມູນໃສ່ເອກະສານ, ວິດີໂອສົນທະນາ (ແລະອື່ນໆອີກ) ກ່ຽວກັບບັນຫາທີ່ກ່ຽວຂ້ອງກັບກໍລະນີ.
  2. ຄລິກປຸ່ມ ADMIN ເພື່ອເປີດແຜງບໍລິຫານ. ນີ້ແມ່ນບ່ອນທີ່ Admins ຈັດການຜູ້ໃຊ້, ກໍລະນີ, ການເຊື່ອມໂຍງ, ແລະອື່ນໆ.
  3. ຄລິກທີ່ຮູບໄຂ່ເພື່ອຈັດການການຮຽກເກັບເງິນບັນຊີ ແລະການຕັ້ງຄ່າການແຈ້ງເຕືອນ.
  • Note Case Active ສະແດງປຸ່ມ ADMIN ສໍາລັບຜູ້ໃຊ້ admin. ຜູ້ໃຊ້ທີ່ບໍ່ມີສິດທິ admin ບໍ່ເຫັນປຸ່ມ ADMIN (ດັ່ງທີ່ສະແດງຢູ່ໃນຫນ້າຈໍຂ້າງລຸ່ມນີ້).
