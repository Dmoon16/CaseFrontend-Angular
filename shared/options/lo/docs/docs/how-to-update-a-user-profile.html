ວິທີການອັບເດດໂປຣໄຟລ໌ຜູ້ໃຊ້

ບົດຄວາມນີ້ສະແດງໃຫ້ເຫັນຜູ້ໃຊ້ວິທີການປັບປຸງຂໍ້ມູນໃນຫນ້າໂປຣໄຟລ໌ຂອງເຂົາເຈົ້າ.

  • ຜູ້ໃຊ້ສາມາດປັບປຸງໂປຣໄຟລ໌ຂອງຕົນເອງໄດ້.
  • ຜູ້ເບິ່ງແຍງສາມາດປັບປຸງຫນ້າໂປຣໄຟລ໌ຂອງຜູ້ໃຊ້ໃດໆ.

ສໍາລັບຜູ້ໃຊ້

ຂັ້ນຕອນທໍາອິດແມ່ນເພື່ອເປີດຫນ້າໂປຣໄຟລ໌. ຜູ້ໃຊ້ ແລະຜູ້ເບິ່ງແຍງລະບົບປະຕິບັດຕາມຂັ້ນຕອນເຫຼົ່ານີ້ເພື່ອອັບເດດໜ້າໂປຣໄຟລ໌ຂອງຕົນເອງ.

  1. ເຂົ້າສູ່ລະບົບບັນຊີ Case Active ຂອງທ່ານ.
  2. ຄລິກແຖບໂປຣໄຟລ໌ຢູ່ເທິງສຸດຂອງໜ້າ.
  3. ເຮັດການປ່ຽນແປງ.
  4. ກົດ Save.

ສໍາລັບຜູ້ບໍລິຫານ

ຜູ້ເບິ່ງແຍງລະບົບຍັງສາມາດອັບເດດຂໍ້ມູນໂປຣໄຟລ໌ໃຫ້ກັບຜູ້ໃຊ້ອື່ນໄດ້. ນີ້ປະກອບມີສາມຊ່ອງຂໍ້ມູນພາຍໃຕ້ຫົວຂໍ້ HOST USER INFO.

  1. ເຂົ້າສູ່ລະບົບບັນຊີຜູ້ເບິ່ງແຍງ Case Active ຂອງທ່ານ.
  2. ຊອກຫາ app ແລະຄລິກໃສ່ປຸ່ມ ADMIN.
  3. ກົດ​ຜູ້​ໃຊ້​ຢູ່​ໃນ​ເມ​ນູ​ທາງ​ເທິງ​.
  4. ໄປ​ທີ່​ເມ​ນູ​ຍ່ອຍ​ແລະ​ເລືອກ​ບັນ​ຊີ​ລາຍ​ຊື່​ຜູ້​ໃຊ້​ທີ່​ທ່ານ​ຕ້ອງ​ການ​ເຂົ້າ​ເຖິງ (ACTIVE ຫຼື​ປິດ​)​. ການຕັ້ງຄ່າເລີ່ມຕົ້ນແມ່ນ Active.
  5. ຊອກຫາຜູ້ໃຊ້ຢູ່ໃນບັນຊີລາຍຊື່ແລະຄລິກແກ້ໄຂ.
  6. ປ່ຽນແປງຕາມຄວາມຕ້ອງການ. (ເບິ່ງຂ້າງລຸ່ມນີ້)
  7. ກົດ Save.
