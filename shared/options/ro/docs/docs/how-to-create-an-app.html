Cum se creează o aplicație

Acest articol arată utilizatorilor noi cum să creeze un cont de probă gratuită. Case Active acordă automat noului utilizator privilegii de administrator pentru acest cont.

Crearea unui cont

  1. Accesați pagina de conectare .
  2. Faceți clic pe butonul Începeți.
  3. Introduceți o adresă de email.
  4. Faceți clic pe Următorul.

Verificarea unui cont

  1. Deschideți o nouă filă de browser și conectați-vă la contul dvs. de e-mail.
  2. Copiați codul din șase cifre.
  3. Reveniți la fila Case Active.
  4. Lipiți codul în câmpuri. Se afișează o nouă fereastră.

În fereastra nouă:

  1. Completați câmpurile obligatorii.
  2. Citiți termenii și condițiile și bifați caseta pentru a fi de acord.
  3. Faceți clic pe CREAȚI UN CONT GRATUIT.

img

Se finalizează configurarea aplicației

Un flux de lucru este afișat pe pagina următoare. Parcurgeți pașii pentru a finaliza configurarea aplicației. Puteți reveni oricând la aceste pagini pentru a edita sau actualiza informațiile sau starea contului dvs.

Pe pagina Despre aplicație, completați câmpurile pentru numele companiei și site-ul web (obligatoriu), apoi faceți clic pe Următorul.

În pagina Informații de facturare, faceți clic pe câmpul Plan de plată și examinați opțiunile. Setarea implicită este Gratuită. Faceți clic pe TRIMITE.

img

După ce trimiteți formularul, Case Active afișează pagina Aplicații. Pentru fiecare aplicație pe care o creați sau căreia îi aparțineți, pagina arată:

  1. Numele aplicației
  2. Tipul dvs. (rol de utilizator)
  3. stare

img

Pagina Aplicații este un punct de lansare pentru multe funcții de administrare:

  1. Faceți clic pe butonul APP pentru a comunica, completa documente, chat video (și multe altele) despre probleme legate de cazuri.
  2. Faceți clic pe butonul ADMIN pentru a deschide panoul de administrare. Aici administratorii gestionează utilizatorii, cazurile, integrările și multe altele.
  3. Faceți clic pe punctele de suspensie pentru a gestiona setările de facturare și notificări ale contului.
  • Notă Case Active afișează butonul ADMIN pentru utilizatorii admin. Utilizatorii care nu au privilegii de administrator nu văd butonul ADMIN (așa cum se arată în ecranul de mai jos).
