Cum să creați utilizatori cu un link de înregistrare

Acest articol arată administratorilor cum să creeze utilizatori trimițând un link de înscriere.

Configurarea linkului de înscriere

  1. Faceți clic pe linkul Trimiteți-le o înscriere din partea de jos a casetei de dialog Adăugați utilizator. Se afișează o pagină nouă.
  2. Configurați opțiunile de înscriere. (vezi mai jos)

img

Asigurați-vă că câmpul Permitere înscriere este setat la Activat. (obligatoriu) Când este activat, Case Active generează o legătură unică.

Examinați opțiunile de mai jos și apoi faceți clic pe pictogramă pentru a copia linkul. Puteți plasa linkul pe site-ul dvs. sau trimite linkul către una sau mai multe persoane prin e-mail sau prin orice canal de comunicare.

img

Crearea unui caz

Opțional, puteți crea automat un caz pentru fiecare utilizator nou și puteți atribui un rol de utilizator.

  1. Faceți clic pe câmpul Creare automată a cazului și selectați Activat. Se afișează câmpurile Rol utilizator al cazului.
  2. Faceți clic pe Rolul utilizator al cazului și selectați o opțiune. (vezi mai jos)

Roluri de utilizator de caz

Există trei roluri de utilizator de caz.

  • clientii
  • Mod implicit
  • personal

Se trimite link-ul

Distribuiți linkul cu una sau mai multe persoane. Fiecare persoană face clic pe link și finalizează fluxul de lucru pentru a-și activa contul Case Active. Odată ce utilizatorul finalizează fluxul de lucru de înscriere, Case Active îl adaugă la lista de utilizatori.

img

Pasii urmatori

Case Active trimite utilizatorului o notificare prin e-mail cu linkul de înscriere. Utilizatorul face clic pe link și finalizează fluxul de lucru pentru înscrieri noi.
