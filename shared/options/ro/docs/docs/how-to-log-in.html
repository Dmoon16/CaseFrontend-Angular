Cum să vă autentificați

Aflați cum să vă conectați la contul dvs. Case Active existent.

Autentificare

  1. Accesați pagina de conectare .
  2. Faceți clic pe butonul LOGIN.
  3. Introduceți adresa dvs. de email.
  4. Faceți clic pe NEXT.

Verificarea unui cont

  1. Deschideți o nouă filă de browser și conectați-vă la contul dvs. de e-mail.
  2. Copiați codul din șase cifre.
  3. Reveniți la fila Case Active.
  4. Lipiți codul în câmpuri. Se afișează o nouă fereastră.

Case Active afișează pagina Aplicații.

  • Notă Platforma afișează un buton ADMIN lângă butonul APP pentru utilizatorii cu privilegii de administrator.
