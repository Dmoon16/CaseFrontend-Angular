Formatarea datelor privind atribuirea cazurilor utilizatorului cu Instrumentul de importare

Mai jos este o listă de câmpuri de date din fișierele CSV. Fiecare opțiune de import are o combinație diferită de câmpuri.

case_ID

  • Introduceți ID-ul utilizatorului sau al etichetei.
  • Accesați pagina Cazuri > Editați > ID-ul etichetei SAU Accesați pagina Cazuri > Editați > ID-ul cazului în url.

ID-ul de utilizator

  • Introduceți ID-ul utilizatorului sau ID-ul etichetei.
  • Accesați pagina Utilizatori > Editare > ID etichetă SAU Accesați pagina Utilizatori > Editare > ID utilizator în url.

case_role_order

  • Lăsați celula goală.

case_role_ID

  • Introduceți: clienți, personal sau implicit.

case_notify

  • 0 = nu
  • 1 = Da, trimiteți utilizatorului o notificare prin e-mail.
