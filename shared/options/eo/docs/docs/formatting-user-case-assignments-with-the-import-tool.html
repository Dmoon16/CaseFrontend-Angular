Formatado de Uzantkazaj Taskoj Datumoj kun la Importilo

Malsupre estas listo de datumkampoj en CSV-dosieroj. Ĉiu importopcio havas malsaman kombinaĵon de kampoj.

case_ID

  • Enigu la kazon de Uzanto aŭ Tag ID.
  • Iru al Kazoj-paĝo > Redaktu > Etikedo ID AŬ Iru al Kazoj-paĝo > Redakti > Kaza ID en url.

uzanto_ID

  • Enigu la ID aŭ Tag ID de la uzanto.
  • Iru al Uzantoj-paĝo > Redaktu > Etikedo ID AŬ Iru al Uzantoj-paĝo > Redaktu > Uzanto-ID en url.

case_role_order

  • Lasu ĉelon malplena.

case_role_ID

  • Enigu: klientoj, dungitaro aŭ defaŭlta.

case_notify

  • 0 = ne
  • 1 = Jes, sendu al uzanto retpoŝtan sciigon.
