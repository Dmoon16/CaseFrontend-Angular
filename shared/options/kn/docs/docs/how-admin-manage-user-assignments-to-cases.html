ನಿರ್ವಾಹಕರು ಪ್ರಕರಣಗಳಿಗೆ ಬಳಕೆದಾರರ ನಿಯೋಜನೆಗಳನ್ನು ಹೇಗೆ ನಿರ್ವಹಿಸುತ್ತಾರೆ

ಬಳಕೆದಾರರು ಮತ್ತು ಕೇಸ್ ಅಸೈನ್‌ಮೆಂಟ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಐದು ಮಾರ್ಗಗಳಿವೆ. ಹೇಗೆ ಎಂದು ತಿಳಿಯಲು ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.

ವಿಧಾನ ಮಾದರಿ ಕ್ರಿಯೆಗಳು ವಿವರಗಳು
ಪ್ರಕರಣವನ್ನು ರಚಿಸುವಾಗ UI ಸೇರಿಸಿ ನಿರ್ವಹಣೆ ಬಟನ್ > ಪ್ರಕರಣಗಳು > ಹೊಸ ಬಟನ್ > ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸು ಬಟನ್
ಸಾರ್ವಜನಿಕ ಸೈನ್ ಅಪ್ ಲಿಂಕ್ ಬಳಸಿಕೊಂಡು ಬಳಕೆದಾರರನ್ನು ಸ್ವಯಂ ನಿಯೋಜಿಸಿ ಬಳಕೆದಾರರ ಕ್ರಿಯೆ ಸೇರಿಸಿ ನಿರ್ವಹಣೆ ಬಟನ್> ಸೆಟ್ಟಿಂಗ್‌ಗಳು> ಸಿಸ್ಟಮ್ ಸೆಟಪ್ ಪುಟ
ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಪ್ರಕರಣದಲ್ಲಿ UI ಅನ್ನು ಬಳಸುವುದು UI ಸೇರಿಸಿ ಮತ್ತು ತೆಗೆದುಹಾಕಿ ನಿರ್ವಹಣೆ ಬಟನ್ > ಪ್ರಕರಣಗಳು > ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಿ
ಆಮದು ಸಾಧನ UI ಸೇರಿಸಿ, ಮಾರ್ಪಡಿಸಿ ಮತ್ತು ತೆಗೆದುಹಾಕಿ ನಿರ್ವಹಣೆ ಬಟನ್> ಇಂಟಿಗ್ರೇಷನ್‌ಗಳು> ಆಮದುಗಳು
