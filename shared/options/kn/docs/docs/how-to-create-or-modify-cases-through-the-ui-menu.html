UI ಮೆನು ಮೂಲಕ ಪ್ರಕರಣಗಳನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ಅಥವಾ ಮಾರ್ಪಡಿಸುವುದು

ಈ ಲೇಖನವು ಪ್ಲಾಟ್‌ಫಾರ್ಮ್‌ನ ಬಳಕೆದಾರ ಇಂಟರ್ಫೇಸ್‌ನೊಂದಿಗೆ ಕೇಸ್ ಅನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ಎಂಬುದನ್ನು ನಿರ್ವಾಹಕರಿಗೆ ತೋರಿಸುತ್ತದೆ. ನಿರ್ವಾಹಕರು ಹೊಸ ಪ್ರಕರಣಕ್ಕೆ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಬಹುದು ಮತ್ತು ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಪ್ರಕರಣಗಳನ್ನು ನವೀಕರಿಸಬಹುದು.

ಪ್ರಕರಣವನ್ನು ರಚಿಸುವುದು

  1. ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ನಿರ್ವಾಹಕ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ನಿರ್ವಾಹಕ ಫಲಕದಲ್ಲಿ, ಪ್ರಕರಣಗಳ ಮೆನು ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಹೊಸ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ಹೊಸ ವಿಂಡೋ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ.

ಅಗತ್ಯವಿರುವ ಕ್ಷೇತ್ರಗಳನ್ನು ಭರ್ತಿ ಮಾಡಿ ಮತ್ತು ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.

ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಲಾಗುತ್ತಿದೆ

ಐಚ್ಛಿಕವಾಗಿ, ನಿರ್ವಾಹಕರು ಈ ಪುಟದಲ್ಲಿ ಪ್ರಕರಣಕ್ಕೆ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಬಹುದು. ಕೆಳಗೆ ಸ್ಕ್ರಾಲ್ ಮಾಡಿ ಮತ್ತು ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸು ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.

img

ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಿ ಮಾದರಿಯಲ್ಲಿ, ಕ್ಷೇತ್ರಗಳನ್ನು ಭರ್ತಿ ಮಾಡಿ. ಬಳಕೆದಾರರಿಗೆ ಇಮೇಲ್ ಅಧಿಸೂಚನೆಯನ್ನು ಕಳುಹಿಸಲು ಬಾಕ್ಸ್ ಅನ್ನು ಪರಿಶೀಲಿಸಿ. (ಐಚ್ಛಿಕ)

ಪ್ರಕರಣವನ್ನು ಮಾರ್ಪಡಿಸುವುದು

ಕಡಿಮೆ ಸಂಖ್ಯೆಯ ಪ್ರಕರಣಗಳನ್ನು ನವೀಕರಿಸಲು, ಎಡಿಟ್ ಪರಿಕರವನ್ನು ಬಳಸಿ ಮತ್ತು ಪ್ರತಿಯೊಂದು ಪ್ರಕರಣಕ್ಕೂ ಬದಲಾವಣೆಗಳನ್ನು ಮಾಡಿ.

  • ಗಮನಿಸಿ ಒಂದೇ ಸಿಟ್ಟಿಂಗ್‌ನಲ್ಲಿ ಹೆಚ್ಚಿನ ಸಂಖ್ಯೆಯ ಪ್ರಕರಣಗಳನ್ನು ನವೀಕರಿಸಲು, ಆಮದು ಪರಿಕರವನ್ನು ಬಳಸುವುದನ್ನು ಪರಿಗಣಿಸಿ
  1. ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ನಿರ್ವಾಹಕ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ನಿರ್ವಾಹಕ ಫಲಕದಲ್ಲಿ, ಪ್ರಕರಣಗಳ ಮೆನು ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಪ್ರಕರಣವನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ಸಂಪಾದಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.
  5. ಬದಲಾವಣೆಗಳನ್ನು ಮಾಡಿ ಮತ್ತು ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.
