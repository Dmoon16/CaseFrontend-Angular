ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಹೇಗೆ ರಚಿಸುವುದು

ಈ ಲೇಖನವು ಹೊಸ ಬಳಕೆದಾರರಿಗೆ ಉಚಿತ ಪ್ರಯೋಗ ಖಾತೆಯನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ಎಂಬುದನ್ನು ತೋರಿಸುತ್ತದೆ. ಕೇಸ್ ಆಕ್ಟಿವ್ ಈ ಖಾತೆಗಾಗಿ ಹೊಸ ಬಳಕೆದಾರ ನಿರ್ವಾಹಕ ಸವಲತ್ತುಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನೀಡುತ್ತದೆ.

ಖಾತೆಯನ್ನು ರಚಿಸಲಾಗುತ್ತಿದೆ

  1. ಲಾಗಿನ್ ಪುಟಕ್ಕೆ ಹೋಗಿ.
  2. ಗೆಟ್ ಸ್ಟಾರ್ಟ್ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ಇಮೇಲ್ ವಿಳಾಸವನ್ನು ನಮೂದಿಸಿ.
  4. ಮುಂದೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಖಾತೆಯನ್ನು ಪರಿಶೀಲಿಸಲಾಗುತ್ತಿದೆ

  1. ಹೊಸ ಬ್ರೌಸರ್ ಟ್ಯಾಬ್ ತೆರೆಯಿರಿ ಮತ್ತು ನಿಮ್ಮ ಇಮೇಲ್ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಆರು-ಅಂಕಿಯ ಕೋಡ್ ಅನ್ನು ನಕಲಿಸಿ.
  3. ಕೇಸ್ ಆಕ್ಟಿವ್ ಟ್ಯಾಬ್‌ಗೆ ಹಿಂತಿರುಗಿ.
  4. ಕ್ಷೇತ್ರಗಳಲ್ಲಿ ಕೋಡ್ ಅನ್ನು ಅಂಟಿಸಿ. ಹೊಸ ವಿಂಡೋ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ.

ಹೊಸ ವಿಂಡೋದಲ್ಲಿ:

  1. ಅಗತ್ಯವಿರುವ ಕ್ಷೇತ್ರಗಳನ್ನು ಭರ್ತಿ ಮಾಡಿ.
  2. ಸೇವಾ ನಿಯಮಗಳನ್ನು ಓದಿ ಮತ್ತು ಒಪ್ಪಿಕೊಳ್ಳಲು ಬಾಕ್ಸ್ ಅನ್ನು ಪರಿಶೀಲಿಸಿ.
  3. ಉಚಿತ ಖಾತೆಯನ್ನು ರಚಿಸಿ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಅಪ್ಲಿಕೇಶನ್ ಸೆಟಪ್ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸಲಾಗುತ್ತಿದೆ

ಮುಂದಿನ ಪುಟದಲ್ಲಿ ಕೆಲಸದ ಹರಿವು ತೋರಿಸುತ್ತದೆ. ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಹೊಂದಿಸುವುದನ್ನು ಪೂರ್ಣಗೊಳಿಸಲು ಹಂತಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿ. ಮಾಹಿತಿಯನ್ನು ಅಥವಾ ನಿಮ್ಮ ಖಾತೆಯ ಸ್ಥಿತಿಯನ್ನು ಸಂಪಾದಿಸಲು ಅಥವಾ ನವೀಕರಿಸಲು ನೀವು ಯಾವಾಗಲೂ ಈ ಪುಟಗಳಿಗೆ ಹಿಂತಿರುಗಬಹುದು.

ಅಪ್ಲಿಕೇಶನ್ ಕುರಿತು ಪುಟದಲ್ಲಿ, ಕಂಪನಿಯ ಹೆಸರು ಮತ್ತು ವೆಬ್‌ಸೈಟ್ ಕ್ಷೇತ್ರಗಳನ್ನು ಭರ್ತಿ ಮಾಡಿ (ಅಗತ್ಯವಿದೆ) ತದನಂತರ ಮುಂದೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಬಿಲ್ಲಿಂಗ್ ಮಾಹಿತಿ ಪುಟದಲ್ಲಿ, ಪಾವತಿ ಯೋಜನೆ ಕ್ಷೇತ್ರವನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಆಯ್ಕೆಗಳನ್ನು ಪರಿಶೀಲಿಸಿ. ಡೀಫಾಲ್ಟ್ ಸೆಟ್ಟಿಂಗ್ ಉಚಿತವಾಗಿದೆ. ಸಲ್ಲಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.

img

ನೀವು ಫಾರ್ಮ್ ಅನ್ನು ಸಲ್ಲಿಸಿದ ನಂತರ, ಕೇಸ್ ಆಕ್ಟಿವ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟವನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ. ನೀವು ರಚಿಸುವ ಅಥವಾ ಸೇರಿರುವ ಪ್ರತಿಯೊಂದು ಅಪ್ಲಿಕೇಶನ್‌ಗಾಗಿ, ಪುಟವು ತೋರಿಸುತ್ತದೆ:

  1. ಅಪ್ಲಿಕೇಶನ್ ಹೆಸರು
  2. ನಿಮ್ಮ ಪ್ರಕಾರ (ಬಳಕೆದಾರರ ಪಾತ್ರ)
  3. ಸ್ಥಿತಿ

ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟವು ಅನೇಕ ನಿರ್ವಾಹಕ ಕಾರ್ಯಗಳಿಗೆ ಉಡಾವಣಾ ಕೇಂದ್ರವಾಗಿದೆ:

  1. ಕೇಸ್-ಸಂಬಂಧಿತ ಸಮಸ್ಯೆಗಳ ಕುರಿತು ಸಂವಹನ ಮಾಡಲು, ಡಾಕ್ಸ್ ತುಂಬಲು, ವೀಡಿಯೊ ಚಾಟ್ (ಮತ್ತು ಇನ್ನಷ್ಟು) ಮಾಡಲು APP ಬಟನ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  2. ನಿರ್ವಾಹಕ ಫಲಕವನ್ನು ತೆರೆಯಲು ನಿರ್ವಾಹಕ ಬಟನ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ. ಇಲ್ಲಿಯೇ ನಿರ್ವಾಹಕರು ಬಳಕೆದಾರರು, ಪ್ರಕರಣಗಳು, ಏಕೀಕರಣಗಳು ಮತ್ತು ಹೆಚ್ಚಿನದನ್ನು ನಿರ್ವಹಿಸುತ್ತಾರೆ.
  3. ಖಾತೆ ಬಿಲ್ಲಿಂಗ್ ಮತ್ತು ಅಧಿಸೂಚನೆ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಎಲಿಪ್ಸಿಸ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  • ನೋಟ್ ಕೇಸ್ ಆಕ್ಟಿವ್ ನಿರ್ವಾಹಕ ಬಳಕೆದಾರರಿಗಾಗಿ ನಿರ್ವಾಹಕ ಬಟನ್ ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ. ನಿರ್ವಾಹಕ ಸವಲತ್ತುಗಳನ್ನು ಹೊಂದಿರದ ಬಳಕೆದಾರರು ADMIN ಬಟನ್ ಅನ್ನು ನೋಡುವುದಿಲ್ಲ (ಕೆಳಗಿನ ಪರದೆಯಲ್ಲಿ ತೋರಿಸಿರುವಂತೆ).
