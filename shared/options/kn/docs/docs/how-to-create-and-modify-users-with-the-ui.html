UI ನೊಂದಿಗೆ ಬಳಕೆದಾರರನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ಮತ್ತು ಮಾರ್ಪಡಿಸುವುದು

ಪ್ಲಾಟ್‌ಫಾರ್ಮ್‌ನ ಬಳಕೆದಾರ ಇಂಟರ್ಫೇಸ್‌ನೊಂದಿಗೆ ಬಳಕೆದಾರರನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ಮತ್ತು ದಾಖಲೆಗಳನ್ನು ನವೀಕರಿಸುವುದು ಹೇಗೆ ಎಂಬುದನ್ನು ಈ ಲೇಖನವು ನಿರ್ವಾಹಕರಿಗೆ ತೋರಿಸುತ್ತದೆ. ನಿರ್ವಾಹಕರು ಹೊಸ ಪ್ರಕರಣಕ್ಕೆ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಬಹುದು ಮತ್ತು ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಪ್ರಕರಣಗಳನ್ನು ಸಂಪಾದಿಸಬಹುದು.

ಬಳಕೆದಾರರನ್ನು ರಚಿಸಲಾಗುತ್ತಿದೆ

  1. ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ನಿರ್ವಾಹಕ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ನಿರ್ವಾಹಕ ಫಲಕದಲ್ಲಿ, ಬಳಕೆದಾರರ ಮೆನು ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಹೊಸ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ಹೊಸ ವಿಂಡೋ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ.

ಕ್ಷೇತ್ರಗಳನ್ನು ಭರ್ತಿ ಮಾಡಿ ಮತ್ತು ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ. ಕೇಸ್ ಆಕ್ಟಿವ್ ಈ ಅಪ್ಲಿಕೇಶನ್‌ನೊಂದಿಗೆ ಸಂಯೋಜಿತವಾಗಿರುವ ಬಳಕೆದಾರರ ಪಟ್ಟಿಗೆ ಹೆಸರನ್ನು ಸೇರಿಸುತ್ತದೆ.

ಬಳಕೆದಾರರನ್ನು ಮಾರ್ಪಡಿಸಲಾಗುತ್ತಿದೆ

ಕಡಿಮೆ ಸಂಖ್ಯೆಯ ಬಳಕೆದಾರರ ದಾಖಲೆಗಳನ್ನು ನವೀಕರಿಸಲು, ಸಂಪಾದನೆ ಉಪಕರಣವನ್ನು ಬಳಸಿ ಮತ್ತು ಪ್ರತಿಯೊಬ್ಬ ಬಳಕೆದಾರರಿಗೆ ಬದಲಾವಣೆಗಳನ್ನು ಮಾಡಿ.

  • ಗಮನಿಸಿ ಒಂದೇ ಸಿಟ್ಟಿಂಗ್‌ನಲ್ಲಿ ಹೆಚ್ಚಿನ ಸಂಖ್ಯೆಯ ಬಳಕೆದಾರರ ದಾಖಲೆಗಳನ್ನು ಮಾರ್ಪಡಿಸಲು, ಆಮದು ಉಪಕರಣವನ್ನು ಬಳಸಿ ಪರಿಗಣಿಸಿ.
  1. ನಿರ್ವಾಹಕ ಫಲಕದಲ್ಲಿ, ಬಳಕೆದಾರರ ಮೆನು ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  2. ಬಳಕೆದಾರರನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ಸಂಪಾದಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ಬದಲಾವಣೆಗಳನ್ನು ಮಾಡಿ ಮತ್ತು ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.

ಕೇಸ್ ಆಕ್ಟಿವ್ ಬಳಕೆದಾರರಿಗೆ ಇಮೇಲ್ ಅಧಿಸೂಚನೆಯನ್ನು ಕಳುಹಿಸುವುದಿಲ್ಲ. ನಿರ್ವಾಹಕರು ಬಳಕೆದಾರರನ್ನು ಸಂಪರ್ಕಿಸಬೇಕು ಮತ್ತು ಅವರಿಗೆ ಕೇಸ್ ಆಕ್ಟಿವ್ ಮುಖಪುಟಕ್ಕೆ ಲಿಂಕ್ ಕಳುಹಿಸಬೇಕು. ಬಳಕೆದಾರರು ನಂತರ LOGIN ಬಟನ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡುತ್ತಾರೆ ಮತ್ತು ಹೊಸ ಸೈನ್‌ಅಪ್‌ಗಳಿಗಾಗಿ ವರ್ಕ್‌ಫ್ಲೋ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸುತ್ತಾರೆ.

  • ಈ ಲೇಖನವು ಹೊಸ ಬಳಕೆದಾರರಿಗಾಗಿ ಸೈನ್ ಅಪ್ ವರ್ಕ್‌ಫ್ಲೋ ಅನ್ನು ವಿವರಿಸುತ್ತದೆ.
