ಆಮದು ಉಪಕರಣದೊಂದಿಗೆ ಪ್ರಕರಣಗಳನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ಅಥವಾ ಮಾರ್ಪಡಿಸುವುದು

ಆಮದು ಪರಿಕರದೊಂದಿಗೆ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಪ್ರಕರಣಗಳನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ಅಥವಾ ಮಾರ್ಪಡಿಸುವುದು ಎಂಬುದನ್ನು ಈ ಲೇಖನವು ನಿರ್ವಾಹಕರಿಗೆ ತೋರಿಸುತ್ತದೆ.

ಆಮದು ಉಪಕರಣವನ್ನು ತೆರೆಯಲಾಗುತ್ತಿದೆ

  1. ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ನಿರ್ವಾಹಕ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ನಿರ್ವಾಹಕ ಫಲಕದಲ್ಲಿ, ಇಂಟಿಗ್ರೇಷನ್ ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ನಂತರ ಉಪಮೆನುವಿನಲ್ಲಿ ಆಮದು ಮಾಡಿ.
  4. ಹೊಸ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ಆಮದು ಸೇರಿಸಿ ಸಂವಾದ ಪೆಟ್ಟಿಗೆ ಹೊಸ ಬ್ರೌಸರ್ ಟ್ಯಾಬ್‌ನಲ್ಲಿ ಪ್ರದರ್ಶಿಸುತ್ತದೆ.

img

ಪ್ರಕರಣಗಳನ್ನು ರಚಿಸುವುದು

ಆಮದು ಸೇರಿಸಿ ಸಂವಾದ ಪೆಟ್ಟಿಗೆ ತೆರೆದ ನಂತರ:

  1. ಟೈಪ್ ಫೀಲ್ಡ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ (ಕೆಳಗೆ ನೋಡಿ). ಲಿಂಕ್ ಪ್ರದರ್ಶಿಸುತ್ತದೆ.
  2. ಡೌನ್‌ಲೋಡ್ ಮಾದರಿ ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ. ನಿಮ್ಮ ಸಾಧನಕ್ಕೆ CSV ಫೈಲ್ ಡೌನ್‌ಲೋಡ್ ಆಗುತ್ತದೆ.

ಮುಂದೆ, ಫೈಲ್ ತೆರೆಯಿರಿ ಮತ್ತು ಪ್ರಕರಣಗಳನ್ನು ಸೇರಿಸಿ.

ಮಾದರಿ

ನೀವು ಟೈಪ್ ಫೀಲ್ಡ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿದಾಗ, ಡ್ರಾಪ್-ಡೌನ್ ಮೆನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ. ಬಲ್ಕ್ ಕೇಸ್ ಆಮದುಗಳಿಗಾಗಿ ಎರಡು ಆಯ್ಕೆಗಳಲ್ಲಿ ಒಂದನ್ನು ಆರಿಸಿ.

  • ಗಮನಿಸಿ ಡ್ರಾಪ್-ಡೌನ್ ಮೆನು ಅನೇಕ ಆಯ್ಕೆಗಳನ್ನು ಹೊಂದಿದೆ, ಆದರೆ ಕೇವಲ ಎರಡು ಮಾತ್ರ ಕೇಸ್ ನಿರ್ವಹಣೆಗೆ ಸಂಬಂಧಿಸಿದೆ. ಪ್ರತಿಯೊಂದು ಆಯ್ಕೆಯು ವಿಭಿನ್ನ ಮಾದರಿ CSV ಫೈಲ್ ಅನ್ನು ಹೊಂದಿದೆ.

ಪ್ರಕರಣಗಳನ್ನು ಸೇರಿಸಿ

  • ಪ್ರಕರಣಗಳ ಸರಳ ಬೃಹತ್ ಅಪ್‌ಲೋಡ್‌ಗಾಗಿ ಈ ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ.
  • ಪ್ರಕರಣಗಳನ್ನು ರಚಿಸಿ ಆಮದು ಪ್ರಕಾರಕ್ಕಾಗಿ ಕಾಲಮ್ ಶೀರ್ಷಿಕೆಗಳೊಂದಿಗೆ CSV ಫೈಲ್ ಅನ್ನು ತೆರೆಯಲು ಡೌನ್‌ಲೋಡ್ ಮಾದರಿ ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.

ಪ್ರಕರಣಗಳನ್ನು ಮಾರ್ಪಡಿಸಿ

  • ಪ್ರಸ್ತುತ ಪ್ರಕರಣಗಳನ್ನು ನವೀಕರಿಸಲು ಈ ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ.
  • ಮಾರ್ಪಡಿಸಿ ಪ್ರಕರಣಗಳ ಆಮದು ಪ್ರಕಾರಕ್ಕಾಗಿ ಕಾಲಮ್ ಶೀರ್ಷಿಕೆಗಳೊಂದಿಗೆ CSV ಫೈಲ್ ಅನ್ನು ತೆರೆಯಲು ಡೌನ್‌ಲೋಡ್ ಮಾದರಿ ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.

ಫೈಲ್ ಅನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಲಾಗುತ್ತಿದೆ

ನೀವು ಈಗಿನಿಂದಲೇ ಫೈಲ್ ಅನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಬಹುದು ಅಥವಾ ನಂತರದ ದಿನಾಂಕದಲ್ಲಿ ಆಮದು ಸೇರಿಸಿ ಸಂವಾದ ಪೆಟ್ಟಿಗೆಗೆ ಹಿಂತಿರುಗಿ ಮತ್ತು ಈ ಹಂತಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿ.

  1. ಆಮದು ಸೇರಿಸಿ ಸಂವಾದ ಪೆಟ್ಟಿಗೆಯನ್ನು ತೆರೆಯಿರಿ.
  2. CSV ಫೈಲ್‌ಗಾಗಿ ಅನನ್ಯ ಹೆಸರನ್ನು ನಮೂದಿಸಿ.
  3. ನಿಮ್ಮ ಸಾಧನದಿಂದ ಫೈಲ್ ಅನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಲು ಪ್ಲಸ್ (+) ಐಕಾನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಟೈಪ್ ಫೀಲ್ಡ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ. (ಮೇಲೆ ನೋಡು)
  5. ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.

ವೈಫಲ್ಯ

ಆಮದು ಕಾರ್ಯಾಚರಣೆ ವಿಫಲವಾದರೆ, ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ವಿಫಲ ಸಂದೇಶವನ್ನು ನೀಡುತ್ತದೆ. ಹೆಚ್ಚಿನ ಸಂದರ್ಭಗಳಲ್ಲಿ, ವೈಫಲ್ಯದ ಕಾರಣ ಡೇಟಾ ಫಾರ್ಮ್ಯಾಟಿಂಗ್ ಆಗಿದೆ.

img

ವೈಫಲ್ಯದ ಕಾರಣವನ್ನು ನೋಡಲು:

  1. ಕ್ರಿಯೆ ಶೀರ್ಷಿಕೆಯ ಅಡಿಯಲ್ಲಿ ವೀಕ್ಷಿಸಿ ಕ್ಲಿಕ್ ಮಾಡಿ.
  2. ಮುಂದಿನ ಪುಟದಲ್ಲಿ, ವಿಫಲ ಐಕಾನ್ ಮೇಲೆ ಮೌಸ್ ಅನ್ನು ಸುಳಿದಾಡಿ. ಟೂಲ್ಟಿಪ್ ದೋಷವನ್ನು ತೋರಿಸುತ್ತದೆ.
  3. CVS ಫೈಲ್‌ಗೆ ಹೋಗಿ ಮತ್ತು ದೋಷವನ್ನು ಸರಿಹೊಂದಿಸಿ ಮತ್ತು ಸರಿಪಡಿಸಿ.
  4. ಫೈಲ್ ಅನ್ನು ಮತ್ತೆ ಅಪ್ಲೋಡ್ ಮಾಡಿ.
