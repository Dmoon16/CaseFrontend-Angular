CSV ಡೇಟಾವನ್ನು ಫಾರ್ಮ್ಯಾಟ್ ಮಾಡುವುದು ಹೇಗೆ

CSV ಫೈಲ್‌ಗಳಲ್ಲಿನ ಡೇಟಾ ಕ್ಷೇತ್ರಗಳ ಪಟ್ಟಿಯನ್ನು ಕೆಳಗೆ ನೀಡಲಾಗಿದೆ. ಪ್ರತಿಯೊಂದು ಆಮದು ಆಯ್ಕೆಯು ಕ್ಷೇತ್ರಗಳ ವಿಭಿನ್ನ ಸಂಯೋಜನೆಯನ್ನು ಹೊಂದಿದೆ.

ಬಳಕೆದಾರ ಹೆಸರು

  • ಬಳಕೆದಾರರ ಇಮೇಲ್ ವಿಳಾಸವನ್ನು ನಮೂದಿಸಿ.

ಕೊಟ್ಟ ಹೆಸರು

  • ಬಳಕೆದಾರರ ಮೊದಲ ಹೆಸರನ್ನು ನಮೂದಿಸಿ.

ಕೌಟುಂಬಿಕ ಹೆಸರು

  • ಬಳಕೆದಾರರ ಕೊನೆಯ ಹೆಸರನ್ನು ನಮೂದಿಸಿ.

ಸ್ಥಳ

  • ಈ ಕಾಲಮ್ ಬಳಕೆದಾರರ ಆದ್ಯತೆಯ ಭಾಷೆಯನ್ನು ಪರಿಶೀಲಿಸುತ್ತದೆ.
  • ಇಂಗ್ಲಿಷ್‌ಗಾಗಿ en ನಮೂದಿಸಿ, ಸ್ಪ್ಯಾನಿಷ್‌ಗಾಗಿ sp.

ವಲಯ ಮಾಹಿತಿ

  • ಇದು ಬಳಕೆದಾರರ ಸಮಯ ವಲಯವನ್ನು ಹೊಂದಿಸುತ್ತದೆ. ಯಾವುದೇ ಸ್ಥಳಗಳಿಲ್ಲದೆ ಖಂಡ ಮತ್ತು ನಗರವನ್ನು ನಮೂದಿಸಿ (ಅಂಡರ್ಸ್ಕೋರ್ ಬಳಸಿ).
  • ಪ್ರೊಫೈಲ್ > ಭಾಷೆ ಮತ್ತು ಪ್ರದೇಶ > ಸಮಯವಲಯ ಡ್ರಾಪ್-ಡೌನ್ ಮೆನುವಿನಲ್ಲಿ ಆಯ್ಕೆಯನ್ನು ಬಳಸಬೇಕು.
  • ಅತ್ಯುತ್ತಮವಾದ ಸಮಯ ವಲಯವನ್ನು ಬಳಸಿ. ಉದಾಹರಣೆಗೆ, ಬೋಸ್ಟನ್ ಡ್ರಾಪ್-ಡೌನ್‌ನಲ್ಲಿಲ್ಲ, ಆದ್ದರಿಂದ ನಾವು ನ್ಯೂಯಾರ್ಕ್ ಅನ್ನು ಬಳಸಬಹುದು (ಅಥವಾ EST/EDT ಸಮಯ ವಲಯದಲ್ಲಿರುವ ಯಾವುದೇ ನಗರ).

ಹೋಸ್ಟ್_ಟ್ಯಾಗ್_ಐಡಿ

  • ಶಾಶ್ವತ ಲೇಬಲ್ ಅನ್ನು ನಮೂದಿಸಿ.
  • az, 0-9, -, / ಅನ್ನು ಅನುಮತಿಸಲಾಗಿದೆ.

ಪ್ರಕರಣ_ಸ್ಥಿತಿ

  • 0 = ಮುಚ್ಚಲಾಗಿದೆ
  • 1 = ಸಕ್ರಿಯ
