ಬಳಕೆದಾರರ ಪ್ರೊಫೈಲ್ ಅನ್ನು ಹೇಗೆ ನವೀಕರಿಸುವುದು

ಈ ಲೇಖನವು ಬಳಕೆದಾರರಿಗೆ ಅವರ ಪ್ರೊಫೈಲ್ ಪುಟದಲ್ಲಿ ಮಾಹಿತಿಯನ್ನು ಹೇಗೆ ನವೀಕರಿಸಬೇಕು ಎಂಬುದನ್ನು ತೋರಿಸುತ್ತದೆ.

  • ಬಳಕೆದಾರರು ತಮ್ಮ ಸ್ವಂತ ಪ್ರೊಫೈಲ್‌ಗಳನ್ನು ನವೀಕರಿಸಬಹುದು.
  • ನಿರ್ವಾಹಕರು ಯಾವುದೇ ಬಳಕೆದಾರರ ಪ್ರೊಫೈಲ್ ಪುಟವನ್ನು ನವೀಕರಿಸಬಹುದು.

ಬಳಕೆದಾರರಿಗೆ

ಪ್ರೊಫೈಲ್ ಪುಟವನ್ನು ತೆರೆಯುವುದು ಮೊದಲ ಹಂತವಾಗಿದೆ. ಬಳಕೆದಾರರು ಮತ್ತು ನಿರ್ವಾಹಕರು ತಮ್ಮದೇ ಆದ ಪ್ರೊಫೈಲ್ ಪುಟವನ್ನು ನವೀಕರಿಸಲು ಈ ಹಂತಗಳನ್ನು ಅನುಸರಿಸುತ್ತಾರೆ.

  1. ನಿಮ್ಮ ಕೇಸ್ ಆಕ್ಟಿವ್ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಪುಟದ ಮೇಲ್ಭಾಗದಲ್ಲಿರುವ ಪ್ರೊಫೈಲ್ ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ಬದಲಾವಣೆಗಳನ್ನು ಮಾಡಿ.
  4. ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.

  1. ನಿಮ್ಮ ಕೇಸ್ ಆಕ್ಟಿವ್ ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ADMIN ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ಮೇಲಿನ ಮೆನುವಿನಲ್ಲಿ ಬಳಕೆದಾರರನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಉಪ-ಮೆನುವಿಗೆ ಹೋಗಿ ಮತ್ತು ನೀವು ಪ್ರವೇಶಿಸಲು ಬಯಸುವ ಬಳಕೆದಾರರ ಪಟ್ಟಿಯನ್ನು ಆಯ್ಕೆ ಮಾಡಿ (ಸಕ್ರಿಯ ಅಥವಾ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಲಾಗಿದೆ). ಡೀಫಾಲ್ಟ್ ಸೆಟ್ಟಿಂಗ್ ಸಕ್ರಿಯವಾಗಿದೆ.

  7. ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.
