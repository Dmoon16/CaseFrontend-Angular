ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಪ್ರಕರಣಗಳೊಂದಿಗೆ ಬಳಕೆದಾರರ ನಿಯೋಜನೆಗಳನ್ನು ನಿಯೋಜಿಸುವುದು ಅಥವಾ ತೆಗೆದುಹಾಕುವುದು ಹೇಗೆ

ಈ ಲೇಖನವು ನಿರ್ವಾಹಕರಿಗೆ ಕೇಸ್ ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸುವುದು ಮತ್ತು ತೆಗೆದುಹಾಕುವುದು ಹೇಗೆ ಎಂಬುದನ್ನು ತೋರಿಸುತ್ತದೆ. ಬಳಕೆದಾರರ ಡೇಟಾವನ್ನು ಹೇಗೆ ಮಾರ್ಪಡಿಸುವುದು ಎಂಬುದನ್ನು ಸಹ ನಾವು ನಿಮಗೆ ತೋರಿಸುತ್ತೇವೆ.

ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಪ್ರಕರಣವನ್ನು ತೆರೆಯಲಾಗುತ್ತಿದೆ

  1. ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ನಿರ್ವಾಹಕ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ಪ್ರಕರಣಗಳ ಮೆನು ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಉಪಮೆನುವಿನಲ್ಲಿ ACTIVE ಅನ್ನು ಆಯ್ಕೆ ಮಾಡಲಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.
  5. ACTION ಕಾಲಮ್‌ಗೆ ಕೆಳಗೆ ಸ್ಕ್ರಾಲ್ ಮಾಡಿ.

ನಿಯೋಜಿಸಿ ಅಥವಾ ತೆಗೆದುಹಾಕಿ

ಪ್ರಕರಣಕ್ಕೆ ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಲು ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಿ ಕ್ಲಿಕ್ ಮಾಡಿ ಅಥವಾ ಪ್ರಕರಣದಿಂದ ಬಳಕೆದಾರರನ್ನು ತೆಗೆದುಹಾಕಿ (ಆದರೆ ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಅಲ್ಲ). ಮುಂದಿನ ಪುಟದಲ್ಲಿ: ಸೇರಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಕೆಲಸದ ಹರಿವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿ. ಬಳಕೆದಾರರನ್ನು ಅಳಿಸಲು ತೆಗೆದುಹಾಕಿ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಮಾರ್ಪಡಿಸಿ

ಪ್ರಕರಣಗಳ ಪುಟದಲ್ಲಿ ಬಳಕೆದಾರರ ಡೇಟಾವನ್ನು ಸಂಪಾದಿಸಲು ಅಥವಾ ನವೀಕರಿಸಲು ಸಾಧ್ಯವಿಲ್ಲ. ಬದಲಾಗಿ, ಈ ಹಂತಗಳನ್ನು ಅನುಸರಿಸಿ:

  1. ಬಳಕೆದಾರರ ಮೆನು ಟ್ಯಾಬ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  2. ಬಳಕೆದಾರ ಹೆಸರನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ಸಂಪಾದಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ಬದಲಾವಣೆಗಳನ್ನು ಮಾಡಿ ಮತ್ತು ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.
