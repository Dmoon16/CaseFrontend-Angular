ಸೈನ್‌ಅಪ್ ಲಿಂಕ್‌ನೊಂದಿಗೆ ಬಳಕೆದಾರರನ್ನು ಹೇಗೆ ರಚಿಸುವುದು

ಸೈನ್‌ಅಪ್ ಲಿಂಕ್ ಅನ್ನು ಕಳುಹಿಸುವ ಮೂಲಕ ಬಳಕೆದಾರರನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ಎಂಬುದನ್ನು ಈ ಲೇಖನವು ನಿರ್ವಾಹಕರಿಗೆ ತೋರಿಸುತ್ತದೆ.

ಸೈನ್ ಅಪ್ ಲಿಂಕ್ ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಲಾಗುತ್ತಿದೆ

  1. ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸು ಸಂವಾದ ಪೆಟ್ಟಿಗೆಯ ಕೆಳಭಾಗದಲ್ಲಿ ಅವರಿಗೆ ಸೈನ್ ಅಪ್ ಲಿಂಕ್ ಕಳುಹಿಸಿ ಕ್ಲಿಕ್ ಮಾಡಿ. ಹೊಸ ಪುಟವನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ.
  2. ಸೈನ್ ಅಪ್ ಆಯ್ಕೆಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ. (ಕೆಳಗೆ ನೋಡಿ)

img

ಸೈನ್‌ಅಪ್ ಕ್ಷೇತ್ರವನ್ನು ಆನ್‌ಗೆ ಹೊಂದಿಸಲು ಅನುಮತಿಸಿ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ. (ಅಗತ್ಯವಿದೆ) ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ, ಕೇಸ್ ಆಕ್ಟಿವ್ ಅನನ್ಯ ಲಿಂಕ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ.

ಕೆಳಗಿನ ಆಯ್ಕೆಗಳನ್ನು ಪರಿಶೀಲಿಸಿ ಮತ್ತು ನಂತರ ಲಿಂಕ್ ಅನ್ನು ನಕಲಿಸಲು ಐಕಾನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ನಿಮ್ಮ ವೆಬ್‌ಸೈಟ್‌ನಲ್ಲಿ ನೀವು ಲಿಂಕ್ ಅನ್ನು ಇರಿಸಬಹುದು ಅಥವಾ ಇಮೇಲ್ ಅಥವಾ ಯಾವುದೇ ಸಂವಹನ ಚಾನಲ್ ಮೂಲಕ ಒಬ್ಬರು ಅಥವಾ ಹೆಚ್ಚಿನ ಜನರಿಗೆ ಲಿಂಕ್ ಅನ್ನು ಕಳುಹಿಸಬಹುದು.

img

ಪ್ರಕರಣವನ್ನು ರಚಿಸುವುದು

ಐಚ್ಛಿಕವಾಗಿ, ನೀವು ಪ್ರತಿ ಹೊಸ ಬಳಕೆದಾರರಿಗಾಗಿ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಒಂದು ಪ್ರಕರಣವನ್ನು ರಚಿಸಬಹುದು ಮತ್ತು ಬಳಕೆದಾರರ ಪಾತ್ರವನ್ನು ನಿಯೋಜಿಸಬಹುದು.

  1. ಸ್ವಯಂ ರಚಿಸಿ ಕೇಸ್ ಕ್ಷೇತ್ರವನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಆನ್ ಆಯ್ಕೆಮಾಡಿ. ಕೇಸ್ ಬಳಕೆದಾರ ಪಾತ್ರದ ಕ್ಷೇತ್ರಗಳನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ.
  2. ಕೇಸ್ ಯೂಸರ್ ರೋಲ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ. (ಕೆಳಗೆ ನೋಡಿ)

img

ಕೇಸ್ ಬಳಕೆದಾರರ ಪಾತ್ರಗಳು

ಮೂರು ಕೇಸ್ ಬಳಕೆದಾರರ ಪಾತ್ರಗಳಿವೆ.

  • ಗ್ರಾಹಕರು
  • ಪೂರ್ವನಿಯೋಜಿತ
  • ಸಿಬ್ಬಂದಿ

ಲಿಂಕ್ ಕಳುಹಿಸಲಾಗುತ್ತಿದೆ

ಒಬ್ಬರು ಅಥವಾ ಹೆಚ್ಚಿನ ಜನರೊಂದಿಗೆ ಲಿಂಕ್ ಅನ್ನು ಹಂಚಿಕೊಳ್ಳಿ. ಪ್ರತಿಯೊಬ್ಬ ವ್ಯಕ್ತಿಯು ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡುತ್ತಾರೆ ಮತ್ತು ಅವರ ಕೇಸ್ ಆಕ್ಟಿವ್ ಖಾತೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಲು ವರ್ಕ್‌ಫ್ಲೋ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸುತ್ತಾರೆ. ಒಮ್ಮೆ ಬಳಕೆದಾರರು ಸೈನ್‌ಅಪ್ ವರ್ಕ್‌ಫ್ಲೋ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದರೆ, ಕೇಸ್ ಆಕ್ಟಿವ್ ಬಳಕೆದಾರರನ್ನು ಬಳಕೆದಾರರ ಪಟ್ಟಿಗೆ ಸೇರಿಸುತ್ತದೆ.

img

ಮುಂದಿನ ಹೆಜ್ಜೆಗಳು

ಕೇಸ್ ಆಕ್ಟಿವ್ ಬಳಕೆದಾರರಿಗೆ ಸೈನ್ ಅಪ್ ಲಿಂಕ್‌ನೊಂದಿಗೆ ಇಮೇಲ್ ಅಧಿಸೂಚನೆಯನ್ನು ಕಳುಹಿಸುತ್ತದೆ. ಬಳಕೆದಾರರು ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡುತ್ತಾರೆ ಮತ್ತು ಹೊಸ ಸೈನ್‌ಅಪ್‌ಗಳಿಗಾಗಿ ವರ್ಕ್‌ಫ್ಲೋ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸುತ್ತಾರೆ.

  • ಈ ಲೇಖನವು ಹೊಸ ಬಳಕೆದಾರರಿಗಾಗಿ ಸೈನ್ ಅಪ್ ವರ್ಕ್‌ಫ್ಲೋ ಅನ್ನು ವಿವರಿಸುತ್ತದೆ.
