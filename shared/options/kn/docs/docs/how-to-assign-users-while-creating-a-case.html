ಪ್ರಕರಣವನ್ನು ರಚಿಸುವಾಗ ಬಳಕೆದಾರರನ್ನು ಹೇಗೆ ನಿಯೋಜಿಸುವುದು

ಹೊಸ ಪ್ರಕರಣವನ್ನು ರಚಿಸುವಾಗ ಬಳಕೆದಾರರನ್ನು ಹೇಗೆ ನಿಯೋಜಿಸಬೇಕು ಎಂಬುದನ್ನು ಈ ಲೇಖನವು ನಿರ್ವಾಹಕರಿಗೆ ತೋರಿಸುತ್ತದೆ. ಬಳಕೆದಾರರು ಈಗಾಗಲೇ ಪ್ಲಾಟ್‌ಫಾರ್ಮ್‌ನಲ್ಲಿ ನೋಂದಾಯಿಸದಿದ್ದರೆ, ಹೊಸ ಪ್ರಕರಣವನ್ನು ರಚಿಸುವಾಗ ನೀವು ಹೊಸ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಬಹುದು.

ಪ್ರಕರಣವನ್ನು ರಚಿಸುವುದು

  1. ನಿರ್ವಾಹಕ ಖಾತೆಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ.
  2. ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ಪುಟದಲ್ಲಿ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಪತ್ತೆ ಮಾಡಿ ಮತ್ತು ನಿರ್ವಾಹಕ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.
  3. ಪ್ರಕರಣಗಳ ಮೆನು ಟ್ಯಾಬ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ.
  4. ಹೊಸ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ಹೊಸ ವಿಂಡೋ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ.
  • ಪ್ರಕರಣವನ್ನು ರಚಿಸಲು ಕ್ಷೇತ್ರಗಳನ್ನು ಭರ್ತಿ ಮಾಡಿ ಮತ್ತು ನಂತರ ನಿಯೋಜಿತ ಬಳಕೆದಾರರ ಶೀರ್ಷಿಕೆಗೆ ಸ್ಕ್ರಾಲ್ ಮಾಡಿ.

ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಲಾಗುತ್ತಿದೆ

ಪ್ರಸ್ತುತ ಬಳಕೆದಾರ

ಹೊಸ ಪ್ರಕರಣಕ್ಕೆ ಪ್ರಸ್ತುತ ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಲು ಈ ಹಂತಗಳನ್ನು ಅನುಸರಿಸಿ.

ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಿ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಕ್ಷೇತ್ರಗಳನ್ನು ಭರ್ತಿ ಮಾಡಿ.

  1. ಬಳಕೆದಾರರ ಡ್ರಾಪ್-ಡೌನ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಗುರುತಿಸುವಿಕೆಯನ್ನು ಆಯ್ಕೆ ಮಾಡಿ (ಉದಾ ಇಮೇಲ್).
  2. ಖಾಲಿ ಕ್ಷೇತ್ರದಲ್ಲಿ ಮೌಲ್ಯವನ್ನು ನಮೂದಿಸಿ (ಉದಾ ಇಮೇಲ್ ವಿಳಾಸ).
  3. ಬಳಕೆದಾರರ ದಾಖಲೆಯು ಪ್ಲಾಟ್‌ಫಾರ್ಮ್‌ನಲ್ಲಿ ಅಸ್ತಿತ್ವದಲ್ಲಿದ್ದರೆ, ಕ್ಷೇತ್ರವು ಹೆಸರನ್ನು ಸ್ವಯಂ-ಸೂಚಿಸುತ್ತದೆ.
  4. ಸರಿಯಾದ ಬಳಕೆದಾರರನ್ನು ಆಯ್ಕೆಮಾಡಿ.
  5. ಬಳಕೆದಾರರ ಪಾತ್ರವನ್ನು ಆಯ್ಕೆಮಾಡಿ (ಕ್ಲೈಂಟ್, ಡೀಫಾಲ್ಟ್ ಅಥವಾ ಸಿಬ್ಬಂದಿ).
  6. ಐಚ್ಛಿಕವಾಗಿ, ಬಳಕೆದಾರರಿಗೆ ಇಮೇಲ್ ಅಧಿಸೂಚನೆಯನ್ನು ಕಳುಹಿಸಲು ಬಾಕ್ಸ್ ಅನ್ನು ಪರಿಶೀಲಿಸಿ.
  7. ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.

ಕೇಸ್ ಆಕ್ಟಿವ್ ಬಳಕೆದಾರರನ್ನು ಕೇಸ್‌ಗೆ ಸೇರಿಸುತ್ತದೆ. ಈಗ ನಿರ್ವಾಹಕರು ವಿಂಡೋವನ್ನು ಮುಚ್ಚಲು ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಬಹುದು ಅಥವಾ ಹೊಸ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಬಹುದು ಅಥವಾ ಬಳಕೆದಾರರನ್ನು ತೆಗೆದುಹಾಕಬಹುದು.

ಹೊಸ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಲಾಗುತ್ತಿದೆ

ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಪ್ರಸ್ತುತ ದಾಖಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯಲಾಗದಿದ್ದರೆ, ಹೊಸ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಲು ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಪ್ಲಾಟ್‌ಫಾರ್ಮ್‌ಗೆ ಬಳಕೆದಾರರನ್ನು ಸೇರಿಸಲು ಕ್ಷೇತ್ರಗಳನ್ನು ಭರ್ತಿ ಮಾಡಿ ಮತ್ತು ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ. ಮುಂದೆ, ಬಳಕೆದಾರರ ಪಾತ್ರವನ್ನು ಹೊಂದಿಸಿ ಮತ್ತು ಪ್ರಕರಣಕ್ಕೆ ಬಳಕೆದಾರರನ್ನು ನಿಯೋಜಿಸಲು ಮತ್ತೆ ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ. ಅಂತಿಮವಾಗಿ, ಪ್ರಕರಣವನ್ನು ಉಳಿಸಲು ಮತ್ತೊಂದು ಬಾರಿ ಉಳಿಸು ಕ್ಲಿಕ್ ಮಾಡಿ.
