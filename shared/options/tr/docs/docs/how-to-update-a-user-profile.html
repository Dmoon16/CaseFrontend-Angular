Kullanıcı Profili Nasıl Güncellenir?

Bu makale, kullanıcılara Profil sayfalarındaki bilgileri nasıl güncelleyeceklerini gösterir.

  • Kullanıcılar kendi profillerini güncelleyebilir.
  • Yöneticiler herhangi bir kullanıcının Profil sayfasını güncelleyebilir.

kullanıcılar için

İlk adım, Profil sayfasını açmaktır. Kullanıcılar ve yöneticiler kendi Profil sayfalarını güncellemek için bu adımları takip eder.

  1. Case Active hesabınızda oturum açın.
  2. Sayfanın üst kısmındaki Profil sekmesine tıklayın.
  3. Değişiklik yapmak.
  4. Kaydet'i tıklayın.

Çoğu alan isteğe bağlıdır ve bazı alanlar zorunludur. Zorunlu alanlar şunlardır:

  • İlk adı
  • Soyadı
  • E-posta adresi
  • Dil
  • Saat dilimi

yöneticiler için

Yöneticiler, diğer kullanıcıların profil bilgilerini de güncelleyebilir. Bu, HOST USER INFO başlığı altındaki üç alanı içerir.

  1. Case Active yönetici hesabınızda oturum açın.
  2. Bir uygulama bulun ve YÖNETİCİ düğmesine tıklayın.
  3. Üst menüde Kullanıcılar'ı tıklayın.
  4. Alt menüye gidin ve erişmek istediğiniz kullanıcı listesini seçin (AKTİF veya DEVRE DIŞI). Varsayılan ayar Etkin'dir.
  5. Listede bir kullanıcı bulun ve Düzenle'ye tıklayın.
  6. Gerekli değişiklikleri yapın. (aşağıya bakınız)
  7. Kaydet'i tıklayın.

Kullanıcı bilgilerini değiştir

Yöneticiler kullanıcı bilgilerini değiştirebilir.

  1. Kullanıcı Türünü (Kullanıcı veya Yönetici) ayarlayın.
  2. Kullanıcı Durumunu ayarlayın (Etkin'den Devre Dışı'na).
  3. Benzersiz bir Etiket Kimliği girin. (isteğe bağlı)
  4. Kaydet'i tıklayın.
