Vaka Yönetimi Verilerini İçe Aktarma Aracıyla Biçimlendirme

Aşağıda, İçe Aktarma aracıyla vakaları yönetmek için CSV dosyalarındaki gerekli alanların bir listesi bulunmaktadır.

Kullanıcı adı

  • Kullanıcının e-posta adresini girin.

tag_id (daha sonra başvurabileceğiniz bir kimlik)

  • benzersiz bir tanımlayıcı (alfasayısal karakterler kullanın, kısa çizgiler kabul edilir)

vaka_durumu

  • 0 = kapalı
  • 1 = aktif

tarih_açıldı

  • ay gün yıl
  • aa/gg/yyyy

hakkında

  • vakanın benzersiz bir açıklaması (alfanümerik karakterler kullanın, kısa çizgiler kabul edilir)
