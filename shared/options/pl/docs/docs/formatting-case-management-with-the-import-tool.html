Formatowanie danych zarządzania sprawami za pomocą narzędzia do importowania

Poniżej znajduje się lista wymaganych pól w plikach CSV do zarządzania sprawami za pomocą narzędzia Importuj.

Nazwa Użytkownika

  • Wprowadź adres e-mail użytkownika.

tag_id (identyfikator, do którego możesz się odwołać później)

  • unikalny identyfikator (użyj znaków alfanumerycznych, myślniki są akceptowane)

status sprawy

  • 0 = zamknięte
  • 1 = aktywny

data_otwarta

  • miesiąc dzień rok
  • mm/dd/rrrr

o

  • unikalny opis sprawy (użyj znaków alfanumerycznych, łączniki są akceptowane)
