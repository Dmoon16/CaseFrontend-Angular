Jak przypisywać użytkowników podczas tworzenia sprawy

Ten artykuł pokazuje administratorom, jak przypisać użytkownika podczas tworzenia nowej sprawy. Jeśli użytkownik nie jest jeszcze zarejestrowany na platformie, możesz dodać nowego użytkownika podczas tworzenia nowej sprawy.

Tworzenie sprawy

  1. Zaloguj się na konto administratora.
  2. Znajdź aplikację na stronie Aplikacje i kliknij przycisk ADMIN.
  3. Kliknij kartę menu Sprawy.
  4. Kliknij przycisk NOWY. Zostanie wyświetlone nowe okno.
  • Wypełnij pola, aby utworzyć zgłoszenie, a następnie przewiń w dół do nagłówka Przypisani użytkownicy.

Przypisywanie użytkownika

Aktualny użytkownik

Wykonaj poniższe kroki, aby przypisać bieżącego użytkownika do nowej sprawy.

Kliknij przycisk PRZYPISZ UŻYTKOWNIKÓW i wypełnij pola.

  1. Kliknij listę rozwijaną Użytkownik i wybierz identyfikator (np. adres e-mail).
  2. Wprowadź wartość w pustym polu (np. adres e-mail).
  3. Jeśli rekord użytkownika istnieje na platformie, pole automatycznie sugeruje nazwę.
  4. Wybierz właściwego użytkownika.
  5. Wybierz rolę użytkownika (klienci, domyślna lub personel).
  6. Opcjonalnie zaznacz pole, aby wysłać użytkownikowi powiadomienie e-mail.
  7. Kliknij ZAPISZ.

Case Active dodaje użytkownika do sprawy. Teraz administratorzy mogą kliknąć Zapisz, aby zamknąć okno, dodać nowego użytkownika lub usunąć użytkownika.

Dodanie nowego użytkownika

Jeśli platforma nie może znaleźć bieżącego rekordu, kliknij przycisk, aby dodać nowego użytkownika.

Wypełnij pola, a następnie kliknij ZAPISZ, aby dodać użytkownika do platformy. Następnie ustaw rolę użytkownika i ponownie kliknij ZAPISZ, aby przypisać użytkownika do sprawy. Na koniec jeszcze raz kliknij ZAPISZ, aby zapisać sprawę.
