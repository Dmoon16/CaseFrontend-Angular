Kaip administratoriai tvarko vartotojus

Administratoriai kuria, modifikuoja arba išjungia, kas gali pasiekti programas ir dėklus. Kai vartotojas pirmą kartą prisijungia prie platformos, jis turi užbaigti profilio darbo eigą.

  • Spustelėkite čia , kad sužinotumėte daugiau apie naujo vartotojo profilio darbo eigą.

Vartotojo valdymas

Žemiau esančioje lentelėje pateikti penki naudotojo valdymo būdai. Spustelėkite nuorodą, kad sužinotumėte, kaip tai padaryti.

Metodas Tipas Veiksmai Detalės
Naudojant vartotojo sąsają UI pridėti ir keisti Administratoriaus mygtukas > Vartotojai
Kurdami atvejį ir priskirdami vartotoją prie atvejo UI papildyti Administratoriaus mygtukas > Atvejai > Naujas mygtukas > Puslapis Kurti atvejį
Automatiškai sukurkite vartotoją naudodami viešą registracijos nuorodą Vartotojo veiksmas papildyti Administratoriaus mygtukas > Nustatymai > Sistemos sąrankos puslapis
Importo įrankis UI pridėti ir keisti Administratoriaus mygtukas > Integracijos > Importavimas
API naudojimas (netrukus) API pridėti ir keisti N/A

Pranešimai

Platforma nesiunčia vartotojams pranešimo, kai jie pridedami prie programos. Tačiau administratoriai gali įjungti pranešimus apie bylų priskyrimą.
