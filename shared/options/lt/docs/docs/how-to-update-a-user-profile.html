Kaip atnaujinti vartotojo profilį

Šiame straipsnyje naudotojai parodo, kaip atnaujinti informaciją savo profilio puslapyje.

  • Vartotojai gali atnaujinti savo profilius.
  • Administratoriai gali atnaujinti bet kurio vartotojo profilio puslapį.

Vartotojams

Pirmas žingsnis yra atidaryti profilio puslapį. Naudotojai ir administratoriai atlieka šiuos veiksmus norėdami atnaujinti savo profilio puslapį.

  1. Prisijunkite prie savo Case Active paskyros.
  2. Puslapio viršuje spustelėkite skirtuką Profilis.
  3. Padaryti pakeitimų.
  4. Spustelėkite Išsaugoti.

Dauguma laukų yra neprivalomi, o kai kurie laukai yra būtini. Privalomi laukai yra:

  • Pirmas vardas
  • Pavardė
  • Elektroninio pašto adresas
  • Kalba
  • Laiko zona

Adminams

Administratoriai taip pat gali atnaujinti profilio informaciją kitiems vartotojams. Tai apima tris laukus po antrašte HOST USER INFO.

  1. Prisijunkite prie Case Active administratoriaus paskyros.
  2. Raskite programą ir spustelėkite mygtuką ADMINISTRATORIUS.
  3. Viršutiniame meniu spustelėkite Vartotojai.
  4. Eikite į submeniu ir pasirinkite naudotojų sąrašą, kurį norite pasiekti (AKTYVUS arba IŠJUNGTAS). Numatytasis nustatymas yra Aktyvus.
  5. Sąraše suraskite vartotoją ir spustelėkite Redaguoti.
  6. Jei reikia, atlikite pakeitimus. (žr. žemiau)
  7. Spustelėkite Išsaugoti.

Keisti vartotojo informaciją

Administratoriai gali keisti vartotojo informaciją.

  1. Nustatykite vartotojo tipą (vartotojas arba valdytojas).
  2. Nustatykite vartotojo būseną (aktyvus į išjungtas).
  3. Įveskite unikalų žymos ID. (neprivaloma)
  4. Spustelėkite Išsaugoti.
