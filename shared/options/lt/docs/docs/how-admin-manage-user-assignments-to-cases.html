Kaip administratoriai tvarko naudotojų priskyrimus atvejams

Yra penki būdai valdyti vartotojus ir atvejų priskyrimą. Spustelėkite nuorodą, kad sužinotumėte, kaip tai padaryti.

Metodas Tipas Veiksmai Detalės
Kuriant bylą UI papildyti Administratoriaus mygtukas > Atvejai > Naujas mygtukas > Mygtukas Priskirti vartotojus
Automatiškai priskirti vartotoją naudojant viešą registracijos nuorodą Vartotojo veiksmas papildyti Administratoriaus mygtukas > Nustatymai > Sistemos sąrankos puslapis
Naudojame vartotojo sąsają esamoje byloje UI pridėti ir pašalinti Administratoriaus mygtukas > Atvejai > Priskirti naudotojus
Importo įrankis UI pridėti, keisti ir pašalinti Administratoriaus mygtukas > Integracijos > Importavimas
API naudojimas (netrukus) API pridėti, keisti ir pašalinti N/A
