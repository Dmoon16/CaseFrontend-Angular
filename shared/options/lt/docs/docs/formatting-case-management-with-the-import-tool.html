Bylų tvarkymo duomenų formatavimas naudojant importavimo įrankį

Toliau pateikiamas būtinų CSV failų laukų sąrašas, norint tvarkyti atvejus naudojant importavimo įrankį.

Vartotojo vardas

  • Įveskite vartotojo el. pašto adresą.

tag_id (ID, kurį galėsite nurodyti vėliau)

  • unikalus identifikatorius (naudokite raidinius ir skaitinius simbolius, priimami brūkšneliai)

case_status

  • 0 = uždaryta
  • 1 = aktyvus

data_atidaryta

  • mėnuo Diena Metai
  • mm/dd/yyyy

apie

  • unikalus didžiosios ir mažosios raidės aprašymas (naudokite raidinius ir skaitinius simbolius, priimtini brūkšneliai)
