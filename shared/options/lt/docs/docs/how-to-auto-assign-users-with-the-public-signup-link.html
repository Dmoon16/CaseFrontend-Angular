Kaip automatiškai priskirti vartotojus naudojant viešosios registracijos nuorodą

Administratoriai sužino, kaip sukonfigūruoti sistemos nustatymus, kad nauji vartotojai būtų automatiškai priskirti naujai automatiškai sukurtai bylai.

Atidaromas sąrankos puslapis

Pirmas žingsnis yra atidaryti sistemos sąrankos puslapį.

  1. Prisijunkite prie administratoriaus paskyros.
  2. Programų puslapyje suraskite programą ir spustelėkite mygtuką ADMINISTRATORIUS.
  3. Spustelėkite skirtuką Sistemos, tada submeniu SISTEMOS NUSTATYMAS.
  4. Slinkite žemyn iki antraštės USER SIGNUP.

Sistemos nustatymų konfigūravimas

  1. Nustatykite išskleidžiamuosius meniu Leisti prisiregistruoti ir automatiškai sukurti atvejį į Įjungta.
  2. Spustelėkite vaidmenų meniu ir pasirinkite parinktį (rekomenduojama pagal nutylėjimą).
  3. Nukopijuokite ir bendrinkite viešosios vartotojo registracijos nuorodą.

Siunčiu nuorodą

Kiekvienas vartotojas, spustelėjęs nuorodą ir užbaigęs prisiregistravimą, suaktyvina platformą, kad sugeneruotų naują atvejį puslapyje „Atvejai“ ir tokiu atveju automatiškai priskiriamas naudotojas.
