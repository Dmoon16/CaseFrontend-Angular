Jak aktualizovat uživatelský profil

Tento článek ukazuje uživatelům, jak aktualizovat informace na stránce profilu.

  • Uživatelé mohou aktualizovat své vlastní profily.
  • Správci mohou aktualizovat stránku profilu libovolného uživatele.

Pro uživatele

Prvním krokem je otevření stránky Profil. Uživatelé a administrátoři podle těchto kroků aktualizují svou vlastní stránku profilu.

  1. Přihlaste se ke svému účtu Case Active.
  2. Klikněte na kartu Profil v horní části stránky.
  3. Udělej změny.
  4. Klikněte na Uložit.

Většina polí je volitelná a některá jsou povinná. Povinná pole jsou:

  • Jméno
  • Příjmení
  • Emailová adresa
  • Jazyk
  • Časové pásmo

Pro adminy

Správci mohou také aktualizovat informace o profilu pro ostatní uživatele. To zahrnuje tři pole pod nadpisem HOST USER INFO.

  1. Přihlaste se ke svému účtu správce Case Active.
  2. Vyhledejte aplikaci a klikněte na tlačítko ADMIN.
  3. V horní nabídce klikněte na Uživatelé.
  4. Přejděte do podnabídky a vyberte seznam uživatelů, ke kterému chcete získat přístup (AKTIVNÍ nebo ZAKÁZANÉ). Výchozí nastavení je Aktivní.
  5. Najděte uživatele v seznamu a klikněte na Upravit.
  6. Proveďte požadované změny. (viz. níže)
  7. Klikněte na Uložit.

Změňte informace o uživateli

Správci mohou změnit informace o uživateli.

  1. Nastavte Typ uživatele (Uživatel nebo Správce).
  2. Nastavte Stav uživatele (Aktivní na Zakázáno).
  3. Zadejte jedinečné ID značky. (volitelný)
  4. Klikněte na Uložit.
