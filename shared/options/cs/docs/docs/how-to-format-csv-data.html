Jak formátovat data CSV

Níže je uveden seznam datových polí v souborech CSV. Každá možnost importu má jinou kombinaci polí.

uživatelské jméno

  • Zadejte e-mailovou adresu uživatele.

křestní jméno

  • Zadejte křestní jméno uživatele.

rodinné jméno

  • Zadejte příjmení uživatele.

národní prostředí

  • Tento sloupec kontroluje preferovaný jazyk uživatele.
  • Zadejte en pro angličtinu, sp pro španělštinu.

info o zóně

  • Tím se nastaví časové pásmo uživatele. Kontinent a město zadejte bez mezer (použijte podtržítko).
  • Musíte použít možnost v rozevírací nabídce Profil > JAZYK A REGION > Časové pásmo.
  • Použijte nejvhodnější časové pásmo. Například Boston není v rozevíracím seznamu, takže můžeme použít New York (nebo jakékoli město v časovém pásmu EST/EDT).

host_tag_id

  • Zadejte trvalý štítek.
  • az, 0-9, -, / jsou povoleny.

case_status

  • 0 = uzavřeno
  • 1 = aktivní
