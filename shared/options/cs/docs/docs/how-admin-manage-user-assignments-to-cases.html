Jak administrátoři spravují přiřazení uživatelů k případům

Existuje pět způsobů, jak spravovat uživatele a přiřazení případů. Kliknutím na odkaz zjistíte, jak na to.

Metoda Typ Akce Podrobnosti
Při vytváření případu UI přidat tlačítko Správce > Případy > tlačítko Nový > tlačítko Přiřadit uživatele
Automaticky přiřadit uživatele pomocí veřejného registračního odkazu Akce uživatele přidat Tlačítko Správce > Nastavení > Stránka Nastavení systému
Použití uživatelského rozhraní na stávajícím případu UI přidat a odebrat Tlačítko Správce > Případy > Přiřadit uživatele
Nástroj pro import UI přidat, upravit a odebrat Tlačítko Správce > Integrace > Importy
Používání API (již brzy) API přidat, upravit a odebrat N/A
