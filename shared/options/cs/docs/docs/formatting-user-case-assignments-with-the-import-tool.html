Formátování dat přiřazení případu uživatele pomocí nástroje pro import

Níže je uveden seznam datových polí v souborech CSV. Každá možnost importu má jinou kombinaci polí.

ID případu

  • Zadejte ID uživatele nebo značky případu.
  • Přejděte na stránku Případy > Upravit > ID značky NEBO přejděte na stránku Případy > Upravit > ID případu v adrese URL.

uživatelské ID

  • Zadejte ID uživatele nebo ID značky.
  • Přejděte na stránku Uživatelé > Upravit > ID značky NEBO přejděte na stránku Uživatelé > Upravit > ID uživatele v adrese URL.

case_role_order

  • Ponechte buňku prázdnou.

case_role_ID

  • Zadejte: klienti, zaměstnanci nebo výchozí.

case_notify

  • 0 = ne
  • 1 = Ano, odeslat uživateli upozornění e-mailem.
