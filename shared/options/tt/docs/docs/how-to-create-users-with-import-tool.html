Импорт коралы белән кулланучыларны ничек булдырырга

Бу мәкалә администраторларны импорт коралы белән ничек ясарга икәнен күрсәтә.

Импорт коралын ачу

Администраторлар импорт коралын яңа кулланучыларны күпләп йөкләү һәм булган кулланучы язмаларын үзгәртү өчен куллана ала.

  1. Администратор счетына керегез.
  2. Кушымталар битендә кушымтаны табыгыз һәм ADMIN төймәсенә басыгыз.
  3. Администратор панелендә Интеграция салынмасына басыгыз, аннары асты менюдагы ИМПОРТ.
  4. Я NEWА төймәгә басыгыз. Импортны өстәү диалог рамкасы яңа браузер салынмасында күрсәтелә.

Язмалар ясау

Импортны өстәү диалог рамкасы ачылганнан соң:

  1. Тип кырына басыгыз һәм вариантны сайлагыз (аста карагыз). Ссылка күрсәтелә.
  2. Йөкләү үрнәге сылтамасына басыгыз. Сезнең җайланмага CSV файл йөкләү.

Аннары, файлны ачып, язмалар өстәгез. CSV файлында формат мәгълүматларын ничек кулланырга икәнен белү өчен монда басыгыз.

Тип

Тип кырына баскач, тамчы меню күрсәтелә. Күпчелек импорт өчен ике вариантның берсен сайлагыз.

: кычкыру: Искәрмә Ачылучы менюның бик күп вариантлары бар, ләкин икесе генә Импорт коралы белән бәйле. Eachәр вариантның төрле CSV файллары бар.

Кулланучыларны өстәгез

  • Кулланучының язмаларын гади йөкләү өчен бу вариантны сайлагыз.
  • Кулланучыларны импортлау төре өчен багана башлы CSV файлын ачу өчен Йөкләү үрнәге сылтамасына басыгыз.

Кулланучыларны үзгәртү

  • Хәзерге кулланучылар өчен мәгълүматны яңарту өчен бу вариантны сайлагыз.
  • Кулланучыларны импортлау төре өчен багана башлы CSV файлын ачу өчен Йөкләү үрнәге сылтамасына басыгыз.

Файлны йөкләү

Сез шунда ук файлны йөкли аласыз яки соңрак импорт өстәү диалог рамкасына кире кайта аласыз һәм бу адымнарны тәмамлый аласыз.

  1. Импорт өстәү диалог рамкасын ачыгыз.
  2. CSV файл өчен уникаль исем кертегез.
  3. Файлны җайланмадан йөкләү өчен плюс (+) иконасына басыгыз.
  4. Тип кырына басыгыз һәм вариантны сайлагыз. (өстә карагыз)
  5. Саклау төймәсенә басыгыз.

Эшкәртү

Файлдагы язмалар санына һәм система йөкләнешенә карап, платформа импортны эшкәртү өчен берничә минут вакыт ала ала.

Импорт тәмамлангач, платформа кулланучы язмаларын көйли (ягъни өсти, үзгәртә яки сүндерә) администратор кулланучылар тактасында.

Уңышсызлык

Импорт операциясе уңышсыз булса, платформа уңышсыз хәбәрне кире кайтара. Күпчелек очракта, уңышсызлыкның сәбәбе - мәгълүмат форматлау.

Уңышсызлыкның сәбәбен күрү өчен:

  1. Эш башы астында Күрү төймәсенә басыгыз.
  2. Киләсе биттә тычканны Уңышсыз иконасына өстәгез. Корал киңәше хатаны күрсәтә.
  3. CVS файлына керегез һәм хатаны көйләгез һәм төзәтегез.
  4. Файлны яңадан йөкләгез.
