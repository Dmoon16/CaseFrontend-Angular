Булган очраклар белән кулланучының биремнәрен ничек билгеләргә яки бетерергә

Бу мәкалә администраторларга очрак кулланучыларны ничек билгеләргә һәм бетерергә күрсәтә. Без сезгә шулай ук кулланучы мәгълүматларын ничек үзгәртүне күрсәтәбез.

Мөмкин булган эшне ачу

  1. Администратор счетына керегез.
  2. Кушымталар битендә кушымтаны табыгыз һәм ADMIN төймәсенә басыгыз.
  3. Эшләр меню салынмасына басыгыз.
  4. ACTIVE субменюда сайланганына инаныгыз.
  5. ACTION баганасына аска борылыгыз.

Билгеләгез яки бетерегез

Кулланучыны эшкә басыгыз яки кулланучыны эштән чыгару өчен (ләкин платформа түгел). Киләсе биттә: ADD төймәсенә басыгыз һәм эш процессын тәмамлагыз. Кулланучыны бетерү өчен бетерүне басыгыз.

.Згәртү

Эшләр битендә кулланучы мәгълүматларын редакцияләү яки яңарту мөмкин түгел. Киресенчә, бу адымнарны үтәгез:

  1. Кулланучылар меню салынмасына басыгыз.
  2. Кулланучының исемен табыгыз һәм Төзәтүгә басыгыз .
  3. Changesзгәрешләр кертегез һәм Саклагыз.
