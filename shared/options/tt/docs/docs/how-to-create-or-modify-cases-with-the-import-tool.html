Импорт коралы белән очракларны ничек ясарга яки үзгәртергә

Бу мәкалә администраторларны импорт коралы ярдәмендә булган очракларны ничек ясарга яки үзгәртергә кирәклеген күрсәтә.

Импорт коралын ачу

  1. Администратор счетына керегез.
  2. Кушымталар битендә кушымтаны табыгыз һәм ADMIN төймәсенә басыгыз.
  3. Администратор панелендә Интеграция салынмасына басыгыз, аннары асты менюдагы ИМПОРТ.
  4. Я NEWА төймәгә басыгыз. Импортны өстәү диалог рамкасы яңа браузер салынмасында күрсәтелә.

Эшләр булдыру

Импортны өстәү диалог рамкасы ачылганнан соң:

  1. Тип кырына басыгыз һәм вариантны сайлагыз (аста карагыз). Ссылка күрсәтелә.
  2. Йөкләү үрнәге сылтамасына басыгыз. Сезнең җайланмага CSV файл йөкләү.

Аннары, файлны ачып, очраклар өстәгез.

Тип

Тип кырына баскач, тамчы меню күрсәтелә. Күпчелек импорт өчен ике вариантның берсен сайлагыз.

  • Искәрмә Ачылучы менюның бик күп вариантлары бар, ләкин икесе генә идарә итү белән бәйле. Eachәр вариантның төрле CSV файллары бар.

Эшләрне өстәгез

  • Гади күп йөкләү өчен бу вариантны сайлагыз.
  • Йөкләү үрнәге сылтамасына басыгыз, CSV файлын ачу өчен багана башлыклары булган очракларны импорт төре өчен.

Эшләрне үзгәртү

  • Хәзерге очракларны яңарту өчен бу вариантны сайлагыз.
  • Йөкләү үрнәге сылтамасына басыгыз, импорт төрен үзгәртү өчен багана башлы CSV файлын ачу өчен.

Файлны йөкләү

Сез шунда ук файлны йөкли аласыз яки соңрак импорт өстәү диалог рамкасына кире кайта аласыз һәм бу адымнарны тәмамлый аласыз.

  1. Импорт өстәү диалог рамкасын ачыгыз.
  2. CSV файл өчен уникаль исем кертегез.
  3. Файлны җайланмадан йөкләү өчен плюс (+) иконасына басыгыз.
  4. Тип кырына басыгыз һәм вариантны сайлагыз. (өстә карагыз)
  5. Саклау төймәсенә басыгыз.

Форматлау

Бу кирәкле кырлар һәм беренче таләпләр. Форматлау детальләрен карау өчен монда басыгыз.

tag_id (яки case_ID) уникаль идентификатор (хәреф саннарын кулланыгыз, сызыклар кабул ителә)

case_status 0 = ябык 1 = актив

дата_ ачылган ай / көн / ел мм / dd / yyyy

эшнең уникаль тасвирламасы турында (хәреф саннарын кулланыгыз, сызыклар кабул ителә)

Эшкәртү

Файлдагы язмалар санына һәм система йөгенә карап, платформа импортны эшкәртү өчен берничә минут вакыт ала ала.

Импорт тәмамлангач, платформа администратор очраклары тактасында очрак язмаларын көйли (ягъни өсти яки үзгәртә).

Уңышсызлык

Импорт операциясе уңышсыз булса, платформа уңышсыз хәбәрне кире кайтара. Күпчелек очракта, уңышсызлыкның сәбәбе - мәгълүмат форматлау.

img

Уңышсызлыкның сәбәбен күрү өчен:

  1. ACTION рубрикасы астында View төймәсенә басыгыз.
  2. Киләсе биттә тычканны Уңышсыз иконасына өстәгез. Корал киңәше хатаны күрсәтә.
  3. CVS файлына керегез һәм хатаны көйләгез һәм төзәтегез.
  4. Файлны яңадан йөкләгез.
