Администраторлар ничек кулланучылар булдыралар

Администраторлар кушымталарга һәм очракларга кем керә ала, үзгәртә яки сүндерә. Кулланучы платформага беренче тапкыр кергәч, аларга профиль эш процессын тәмамларга кирәк.

  • Яңа кулланучы профиле эш процессы турында күбрәк белү өчен монда басыгыз.

Кулланучы булдыру

Түбәндәге таблицада кулланучыны булдыруның биш ысулы күрсәтелгән. Ничек белү өчен сылтамага басыгыз.

Хәбәрләр

Платформа кулланучыларга кушымтага өстәлгәндә хәбәр җибәрми. Ләкин, админнар эш биремнәре өчен хәбәрләр җибәрә ала.
