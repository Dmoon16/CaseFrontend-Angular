UI менюсы аша очракларны ничек ясарга яки үзгәртергә

Бу мәкалә администраторларга платформаның кулланучы интерфейсы белән эш ничек ясарга икәнен күрсәтә. Администратор шулай ук кулланучыларны яңа очракка өсти һәм булган очракларны яңарта ала.

Эш ясау

  1. Администратор счетына керегез.
  2. Кушымталар битендә кушымтаны табыгыз һәм ADMIN төймәсенә басыгыз.
  3. Администратор панелендә очраклар меню салынмасына басыгыз.
  4. Я NEWА төймәгә басыгыз. Яңа тәрәзә күрсәтелә.

Кулланучылар өстәү

Ихтимал, администраторлар бу биттәге очракны кулланучыларга өсти алалар. Аска әйләндерегез һәм ASSIGN USERS төймәсенә басыгыз.

Кулланучыны билгеләү модалында кырларны тутырыгыз. Кулланучының электрон почта хәбәрен җибәрү өчен рамканы тикшерегез. (өстәмә)

Эшне үзгәртү

Аз санлы очракларны яңарту өчен, үзгәртү коралын кулланыгыз һәм һәрбер очракта үзгәрешләр кертегез.

  • Искәрмә Бер утырышта күп санлы очракларны яңарту өчен, Импорт коралын кулланырга уйлагыз
  1. Администратор счетына керегез.
  2. Кушымталар битендә кушымтаны табыгыз һәм ADMIN төймәсенә басыгыз.
  3. Администратор панелендә очраклар меню салынмасына басыгыз.
  4. Эшне табыгыз һәм Төзәтү төймәсенә басыгыз.
  5. Changesзгәрешләр кертегез һәм Саклагыз.
