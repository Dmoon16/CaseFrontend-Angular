Импорт коралы белән кулланучы очракларын билгеләү мәгълүматларын форматлау

Түбәндә CSV файлларындагы мәгълүмат кырлары исемлеге китерелгән. Eachәрбер импорт варианты кырларның төрле комбинациясенә ия.

case_ID

  • Кулланучыны яки Тэг ID-ны кертегез.
  • Эш битенә> Редакцияләү> Тэг ID яки Яисә Эшләр битенә керегез> Edit> Case ID url.

user_ID

  • Кулланучының ID яки Тэг ID кертегез.
  • Кулланучылар битенә> Редакцияләү> Тэг ID яки Кулланучылар битенә керегез> Редакцияләү> Кулланучының ID-ны url.

case_role_order

  • Шакмакны буш калдырыгыз.

case_role_ID

  • Керегез: клиентлар, персонал, яисә килешү.

очрак

  • 0 = юк
  • 1 = Әйе, кулланучыга электрон почта хәбәрен җибәрегез.
