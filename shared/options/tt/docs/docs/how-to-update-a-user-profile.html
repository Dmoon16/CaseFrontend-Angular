Кулланучының профилен ничек яңартырга

Бу мәкалә кулланучыларга Профиль битендәге мәгълүматны ничек яңартырга икәнен күрсәтә.

  • Кулланучылар үз профильләрен яңарта алалар.
  • Администраторлар теләсә нинди кулланучының Профиль битен яңартырга мөмкин.

Кулланучылар өчен

Беренче адым - Профиль битен ачу. Кулланучылар һәм администраторлар үзләренең профиль битен яңарту өчен бу адымнарны ясыйлар.

  1. Сезнең Актив счетка керегез.
  2. Битнең башындагы Профиль салынмасына басыгыз.
  3. Changesзгәрешләр кертегез.
  4. Саклау төймәсенә басыгыз.

Күпчелек кырлар өстәмә, һәм кайбер кырлар кирәк. Кирәкле кырлар:

  • Исем
  • Фамилия
  • Мэйл адресы
  • Тел
  • Вакыт зонасы

Администраторлар өчен

Администраторлар шулай ук бүтән кулланучылар өчен профиль мәгълүматларын яңартырга мөмкин. Бу HOST USER INFO рубрикасы астында өч кырны үз эченә ала.

  1. Сезнең Актив администратор счетына керегез.
  2. Кушымтаны табыгыз һәм ADMIN төймәсенә басыгыз.
  3. Usersгары менюдагы кулланучыларга басыгыз.
  4. Суб-менюга керегез һәм сез керергә теләгән кулланучылар исемлеген сайлагыз (АКТИВ яки ДИСАБЛЕД). Килешенгән көйләү Актив.
  5. Исемлектә кулланучыны табыгыз һәм Төзәтү төймәсенә басыгыз.
  6. Кирәк булганча үзгәрешләр кертегез. (астарак карагыз)
  7. Саклау төймәсенә басыгыз.

Кулланучының мәгълүматын үзгәртү

Администратор кулланучы мәгълүматын үзгәртә ала.

  1. Кулланучының төрен (кулланучы яки менеджер) куегыз.
  2. Кулланучының статусын куегыз (Инвалидларга актив).
  3. Уникаль Тэг ID кертегез. (өстәмә)
  4. Саклау төймәсенә басыгыз.
