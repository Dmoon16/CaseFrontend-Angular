Эш ясаганда кулланучыларны ничек билгеләргә

Бу мәкалә администраторларга яңа очрак төзегәндә ничек билгеләргә икәнен күрсәтә. Әгәр дә кулланучы платформада теркәлмәгән булса, сез яңа очрак төзегәндә яңа кулланучы өсти аласыз.

Эш ясау

  1. Администратор счетына керегез.
  2. Кушымталар битендә кушымтаны табыгыз һәм ADMIN төймәсенә басыгыз.
  3. Эшләр меню салынмасына басыгыз.
  4. Я NEWА төймәгә басыгыз. Яңа тәрәзә күрсәтелә.
  • Эш ясау өчен кырларны тутырыгыз, аннары билгеләнгән кулланучылар рубрикасына асылыгыз.

Кулланучыны билгеләү

Хәзерге кулланучы

Хәзерге кулланучыны яңа очракта билгеләү өчен бу адымнарны үтәгез.

ASSIGN USERS төймәсенә басыгыз һәм кырларны тутырыгыз.

  1. Кулланучының тамчысына басыгыз һәм идентификаторны сайлагыз (мәсәлән, электрон почта).
  2. Буш мәйданга кыйммәт кертегез (мәсәлән, электрон почта адресы).
  3. Әгәр дә платформада кулланучы рекорды бар икән, кыр исемне автоматик рәвештә тәкъдим итә.
  4. Дөрес кулланучыны сайлагыз.
  5. Кулланучының ролен сайлагыз (клиентлар, килешү, яки персонал).
  6. Ихтимал, кулланучыга электрон почта хәбәрен җибәрү өчен тартманы тикшерегез.
  7. Саклау төймәсенә басыгыз.

Case Active кулланучыны эшкә өсти. Хәзер администраторлар тәрәзәне ябу өчен Саклау төймәсенә басыгыз, яисә яңа кулланучы өсти аласыз, яисә кулланучыны бетерә аласыз.

Яңа кулланучы өстәү

Платформа хәзерге язуны таба алмаса, яңа кулланучы өстәү өчен төймәгә басыгыз.
