Ungawakha noma uwalungise kanjani amacala ngethuluzi lokungenisa

Lesi sihloko sibonisa abalawuli ukuthi bawadala kanjani noma bawashintsha kanjani amacala akhona ngethuluzi Lokungenisa.

Ivula ithuluzi Lokungenisa

  1. Ngena ngemvume ku-akhawunti yomqondisi.
  2. Thola uhlelo lokusebenza ekhasini lezinhlelo zokusebenza bese uchofoza inkinobho ethi ADMIN.
  3. Kuphaneli Yokuphatha, chofoza ithebhu yokuhlanganisa bese kuthi OKUNGANGISA kumenyu encane.
  4. Chofoza inkinobho ENTSHA. Ibhokisi lengxoxo elithi Engeza Okungeniswayo liboniswa kuthebhu entsha yesiphequluli.

img

Ukudala amacala

Ngemuva kokuthi ibhokisi lengxoxo le-Add Imports livuleka:

  1. Chofoza inkambu Yohlobo bese ukhetha inketho (bona ngezansi). Isixhumanisi siyaboniswa.
  2. Chofoza isixhumanisi Sesampula Sokulanda. Ifayela le-CSV lilandwa kudivayisi yakho.

Okulandelayo, vula ifayela bese wengeza amacala.

Uhlobo

Uma uchofoza inkambu yoHlobo , imenyu yokudonsela phansi iyaboniswa. Khetha inketho eyodwa kwezimbili zokungeniswa kwenqwaba yenqwaba.

  • Qaphela Imenyu yokudonsela phansi inezinketho eziningi, kodwa ezimbili kuphela ezihlobene nokuphatha icala. Inketho ngayinye inesampula ehlukile yefayela le-CSV.

Engeza amacala

  • Khetha le nketho ukuze uthole ukulayishwa kwenqwaba yamacala.
  • Chofoza isixhumanisi Sokulanda Isampula ukuze uvule ifayela le-CSV elinezihloko zekholomu zohlobo lokungenisa lwe-Dala amacala.

Lungisa amacala

  • Khetha le nketho ukuze ubuyekeze izimo zamanje.
  • Chofoza isixhumanisi Sokulanda Isampula ukuze uvule ifayela le-CSV elinezihloko zekholomu zohlobo lokungenisa izimo ezishintshayo.

Ilayisha ifayela

Ungakwazi ukulayisha ifayela ngokushesha noma ubuyele ebhokisini lengxoxo elithi Engeza Okungeniswayo ngedethi ekamuva futhi uqedele lezi zinyathelo.

  1. Vula ibhokisi lengxoxo elithi Engeza Okungeniswayo.
  2. Faka igama elihlukile lefayela le-CSV.
  3. Chofoza isithonjana sokuhlanganisa (+) ukuze ulayishe ifayela kusuka kudivayisi yakho.
  4. Chofoza okuthi Thayipha inkambu bese ukhetha inketho. (bona ngenhla)
  5. Chofoza okuthi LINDISA.

Ukufometha

Lezi izinkambu ezidingekayo kanye nezidingo zokuqala. Chofoza lapha ukuze ubuyekeze imininingwane yokufometha.

tag_id (noma i-case_ID) isihlonzi esiyingqayizivele (sebenzisa izinhlamvu ze-alphanumeric, amakhonco ayamukelwa)

case_status 0 = ivaliwe 1 = iyasebenza

idethi_evuliwe inyanga/usuku/unyaka mm/dd/yyyy

mayelana nencazelo eyingqayizivele yekesi (sebenzisa izinhlamvu ze-alphanumeric, amakhonco ayamukelwa)

Iyacubungula

Kuye ngenani lamarekhodi kufayela nomthwalo wesistimu, inkundla ingathatha imizuzu embalwa ukucubungula ukungenisa.

img

Lapho ukungenisa kuqedwa, inkundla ilungisa amarekhodi amacala (okungukuthi, iyanezela noma ilungise) kudeshibhodi Yamacala omphathi.

Ukwehluleka

Uma umsebenzi wokungenisa uhluleka, inkundla ibuyisela umlayezo Ohlulekile. Ezimweni eziningi, imbangela yokwehluleka ukufometha idatha.

img

Ukubona imbangela yokwehluleka:

  1. Chofoza okuthi Buka ngaphansi kwesihloko esithi ACTION.
  2. Ekhasini elilandelayo, hambisa igundane kusithonjana Sehlulekile. Ithiphu yamathuluzi ibonisa iphutha.
  3. Iya kufayela le-CVS bese ulungisa futhi ulungise iphutha.
  4. Layisha ifayela futhi.
