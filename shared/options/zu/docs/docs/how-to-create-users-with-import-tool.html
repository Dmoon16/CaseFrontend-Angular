Ubadala Kanjani Abasebenzisi Ngethuluzi Lokungenisa

Lesi sihloko sibonisa abalawuli ukuthi babadala kanjani abasebenzisi ngethuluzi Lokungenisa.

Ivula ithuluzi Lokungenisa

Abalawuli bangasebenzisa ithuluzi Lokungenisa ukuze balayishe ngenqwaba abasebenzisi abasha futhi balungise amarekhodi akhona abasebenzisi.

  1. Ngena ngemvume ku-akhawunti yomqondisi.
  2. Thola uhlelo lokusebenza ekhasini lezinhlelo zokusebenza bese uchofoza inkinobho ethi ADMIN.
  3. Kuphaneli Yokuphatha, chofoza ithebhu yokuhlanganisa bese kuthi OKUNGANGISA kumenyu encane.
  4. Chofoza inkinobho ENTSHA. Ibhokisi lengxoxo elithi Engeza Okungeniswayo liboniswa kuthebhu entsha yesiphequluli.

Ukudala amarekhodi

Ngemuva kokuthi ibhokisi lengxoxo le-Add Imports livuleka:

  1. Chofoza inkambu Yohlobo bese ukhetha inketho (bona ngezansi). Isixhumanisi siyaboniswa.
  2. Chofoza isixhumanisi Sesampula Sokulanda. Ifayela le-CSV lilandwa kudivayisi yakho.

Okulandelayo, vula ifayela bese wengeza amarekhodi. Chofoza lapha ukuze ufunde indlela yokusebenzisa idatha yefomethi kufayela le-CSV.

Uhlobo

Uma uchofoza inkambu yoHlobo, imenyu yokudonsela phansi iyaboniswa. Khetha okukodwa kokubili ongakhetha kukho kokungeniswa kwenqwaba.

:isibabazo: QAPHELA Imenyu yokudonsela phansi inezinketho eziningi, kodwa ezimbili kuphela ezihlobene nethuluzi Lokungenisa. Inketho ngayinye inesampula ehlukile yefayela le-CSV.

Engeza abasebenzisi

  • Khetha le nketho ukuze uthole ukulayishwa kwenqwaba okulula kwamarekhodi omsebenzisi.
  • Chofoza isixhumanisi Sesampula Sokulanda ukuze uvule ifayela le-CSV elinezihloko zekholomu zohlobo Lokungenisa abasebenzisi.

Shintsha abasebenzisi

  • Khetha le nketho ukuze ubuyekeze idatha yabasebenzisi bamanje.
  • Chofoza isixhumanisi sesampula yokulanda ukuze uvule ifayela le-CSV elinezihloko zekholomu zohlobo lokungenisa abasebenzisi elithi Guqula.

Ilayisha ifayela

Ungakwazi ukulayisha ifayela ngokushesha noma ubuyele ebhokisini lengxoxo elithi Engeza Okungeniswayo ngedethi ekamuva futhi uqedele lezi zinyathelo.

  1. Vula ibhokisi lengxoxo elithi Engeza Okungeniswayo.
  2. Faka igama elihlukile lefayela le-CSV.
  3. Chofoza isithonjana sokuhlanganisa (+) ukuze ulayishe ifayela kusuka kudivayisi yakho.
  4. Chofoza okuthi Thayipha inkambu bese ukhetha inketho. (bona ngenhla)
  5. Chofoza okuthi LINDISA.

Iyacubungula

Kuye ngenani lamarekhodi kufayela nomthwalo wesistimu, inkundla ingathatha imizuzu embalwa ukucubungula ukungenisa.

Lapho ukungenisa kuqedwa, inkundla ilungisa amarekhodi omsebenzisi (okungukuthi, iyanezela, ilungise, noma ikhubaze) kudeshibhodi Yabasebenzisi bomphathi.

Ukwehluleka

Uma umsebenzi wokungenisa uhluleka, inkundla ibuyisela umlayezo Ohlulekile. Ezimweni eziningi, imbangela yokwehluleka ukufometha idatha.

Ukubona imbangela yokwehluleka:

  1. Chofoza okuthi Buka ngaphansi kwesihloko esithi Isenzo.
  2. Ekhasini elilandelayo, hambisa igundane kusithonjana Sehlulekile. Ithiphu yamathuluzi ibonisa iphutha.
  3. Iya kufayela le-CVS bese ulungisa futhi ulungise iphutha.
  4. Layisha ifayela futhi.
