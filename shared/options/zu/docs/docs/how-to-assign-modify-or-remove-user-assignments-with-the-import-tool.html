Indlela Yokwabela, Yokushintsha noma Yokukhipha Umsebenzisi Ngethuluzi Lokungenisa

Lesi sihloko sibonisa abaphathi ukuthi babaphatha kanjani abasebenzisi becala ngethuluzi Lokungenisa.

Abaphathi bangakwazi:

  • yabela abasebenzisi ecaleni
  • guqula abasebenzisi asebevele banikezwe icala
  • susa abasebenzisi abanikezwe icala

Ivula ithuluzi Lokungenisa

  1. Ngena ngemvume ku-akhawunti yomqondisi.
  2. Thola uhlelo lokusebenza ekhasini lezinhlelo zokusebenza bese uchofoza inkinobho ethi ADMIN.
  3. Chofoza ithebhu yokuhlanganisa bese kuthi IMPORTS kumenyu encane.
  4. Chofoza inkinobho ENTSHA. Ibhokisi lengxoxo elithi Engeza Ukungenisa liyaboniswa.

Ukudala amarekhodi

Landa ifayela le-CSV elilodwa noma amaningi. Sebenzisa lawa mafayela ukuze wengeze, uguqule, noma ususe abasebenzisi becala. Ngemuva kokuthi ibhokisi lengxoxo le-Add Imports livuleka:

  1. Chofoza inkambu ethi Thayipha.
  2. Skrola kumenyu yokudonsela phansi bese ukhetha inketho (bona ngezansi). Isixhumanisi siyaboniswa.
  3. Chofoza isixhumanisi Sesampula Sokulanda. Ifayela le-CSV lilandwa kudivayisi yakho.

Vula ifayela bese wengeza amarekhodi. Chofoza lapha ukuze ufunde indlela yokusebenzisa idatha yefomethi kufayela le-CSV.

Izinketho zefayela

Yabela abasebenzisi amacala * Khetha le nketho ukuze wengeze abasebenzisi ecaleni. * Shiya case_role_order cell lingenalutho. * Iya ekhasini labasebenzisi bese ukopisha i-ID yethegi yalo muntu. Uma inkambu ye-Tag ID ingenalutho, faka i-ID ehlukile, chofoza okuthi Londoloza, bese ukopisha i-ID kufayela le-CSV.

Guqula abasebenzisi abanikezwe amacala * Khetha le nketho ukuze ubuyekeze idatha yabasebenzisi bamanje abanikezwe icala. * Sebenzisa i-Case Tag ID kukholomu ye-case_ID.

Susa abasebenzisi abanikezwe amacala * Khetha le nketho ukuze ususe abasebenzisi abanikezwe icala.

Ilayisha ifayela

Ungakwazi ukulayisha ifayela ngokushesha noma ubuyele ebhokisini lengxoxo elithi Engeza Okungeniswayo ngedethi ekamuva futhi uqedele lezi zinyathelo.

  1. Vula ibhokisi lengxoxo elithi Engeza Okungeniswayo.
  2. Faka igama elihlukile lefayela le-CSV.
  3. Chofoza isithonjana sokuhlanganisa (+) ukuze ulayishe ifayela kusuka kudivayisi yakho.
  4. Chofoza okuthi Thayipha inkambu bese ukhetha inketho. (bona ngenhla)
  5. Chofoza okuthi LINDISA.

Iyacubungula

Kuye ngenani lamarekhodi kufayela nomthwalo wesistimu, inkundla ingathatha imizuzu embalwa ukucubungula ukungenisa.

img

Lapho ukungenisa kuqedwa, inkundla ilungisa amarekhodi omsebenzisi (okungukuthi, iyanezela, ilungise, noma isuse) ekhasini Lamacala.

Ukwehluleka

Uma umsebenzi wokungenisa uhluleka, inkundla ibuyisela umlayezo Ohlulekile. Ezimweni eziningi, imbangela yokwehluleka ukufometha idatha.

Ukubona imbangela yokwehluleka:

  1. Chofoza okuthi Buka ngaphansi kwesihloko esithi Isenzo.
  2. Ekhasini elilandelayo, hambisa igundane kusithonjana Sehlulekile. Ithiphu yamathuluzi ibonisa iphutha.
  3. Iya kufayela le-CVS bese ulungisa futhi ulungise iphutha.
  4. Layisha ifayela futhi.
