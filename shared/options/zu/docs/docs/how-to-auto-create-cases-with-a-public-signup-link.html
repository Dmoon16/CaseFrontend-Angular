Ungawakha Kanjani Amacala Ngokuzenzakalelayo Ngesixhumanisi Sokubhalisa Somphakathi

Lesi sihloko sibonisa abaphathi ukuthi bangadala kanjani icala (futhi bengeze umsebenzisi ngesikhathi esifanayo) ngokuthumela isixhumanisi sokubhalisa esisodwa.

Ithola isixhumanisi

  1. Ngena ngemvume ku-akhawunti yomqondisi.
  2. Thola uhlelo lokusebenza ekhasini lezinhlelo zokusebenza bese uchofoza inkinobho ethi ADMIN.
  3. Kuphaneli Yokulawula, chofoza ithebhu yemenyu Yezilungiselelo.
  4. Qiniseka ukuthi ithebhu yemenyu engaphansi YE-SYSTEM SETUP ikhethiwe.

Ilungiselela izilungiselelo

Skrolela phansi ku-USER SIGNUP isihloko.

Qiniseka ukuthi inkambu ethi Vumela Ukubhalisa isethwe kokuthi Vuliwe. (kuyadingeka) Uma lesi sici sinikwe amandla, i-Case Active ikhiqiza isixhumanisi esiyingqayizivele.

  1. Chofoza inkambu ethi Dala Ngokuzenzakalelayo Ikesi bese ukhetha Vuliwe. I-Case User Role field displays.
  2. Khetha indima. (bona ngezansi)

Izindima zabasebenzisi

Kunezindima ezintathu zabasebenzisi.

  • amaklayenti (kunconyiwe)
  • okuzenzakalelayo
  • abasebenzi

Ithumela isixhumanisi

Yabelana ngesixhumanisi nomuntu oyedwa noma ngaphezulu. Umsebenzisi ngamunye ochofoza isixhumanisi futhi aqedele ukubhalisa ucupha inkundla ukuthi ikhiqize ikesi elisha ekhasini lamacala futhi inikeze ngokuzenzakalelayo umsebenzisi esimweni.
