Ukufometha Idatha Yezabelo Zomsebenzisi Ngethuluzi Lokungenisa

Ngezansi kunohlu lwezinkambu zedatha kumafayela e-CSV. Inketho ngayinye yokungenisa inenhlanganisela ehlukile yezinkambu.

case_ID

  • Faka icala lomsebenzisi noma i-ID yomaka.
  • Iya ekhasini elithi Izimo > Hlela > Umazisi womaka NOMA Yiya ekhasini elithi Izimo > Hlela > I-ID yekesi ku-url.

I-ID Yomsebenzisi

  • Faka i-ID yomsebenzisi noma i-ID yethegi.
  • Iya ekhasini labasebenzisi > Hlela > Umazisi womaka NOMA Yiya ekhasini labasebenzisi > Hlela > I-ID yomsebenzisi ku-url.

i-oda_lendima_yecala

  • Shiya iseli lingenalutho.

case_role_ID

  • Faka: amaklayenti, abasebenzi, noma okuzenzakalelayo.

case_notify

  • 0 = no
  • 1 = Yebo, thumela umsebenzisi isaziso se-imeyili.
