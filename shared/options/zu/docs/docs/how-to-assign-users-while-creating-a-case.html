Ubabela Kanjani Abasebenzisi Ngenkathi Udala Ikesi

Lesi sihloko sibonisa abalawuli ukuthi bamnika kanjani umsebenzisi ngenkathi bedala ikesi elisha. Uma umsebenzisi engabhalisiwe epulatifomu, ungakha ukwengeza umsebenzisi omusha ngenkathi udala icala elisha.

Ukudala icala

  1. Ngena ngemvume ku-akhawunti yomqondisi.
  2. Thola uhlelo lokusebenza ekhasini lezinhlelo zokusebenza bese uchofoza inkinobho ethi ADMIN.
  3. Chofoza ithebhu yemenyu ye-Cases.
  4. Chofoza inkinobho ENTSHA. Iwindi elisha liyavela.
  • Gcwalisa izinkambu ukuze udale ikesi bese uskrolela phansi esihlokweni esithi Abasebenzisi Ababelwe.

Yabela umsebenzisi

Umsebenzisi wamanje

Landela lezi zinyathelo ukuze unikeze umsebenzisi wamanje esimweni esisha.

Chofoza inkinobho ethi ASSIGN USERS bese ugcwalisa izinkambu.

  1. Chofoza okokudonsela phansi komsebenzisi bese ukhetha isihlonzi (isb i-imeyili).
  2. Faka inani endaweni engenalutho (isb. ikheli le-imeyili).
  3. Uma irekhodi lomsebenzisi likhona endaweni, inkambu iphakamisa ngokuzenzakalelayo igama.
  4. Khetha umsebenzisi olungile.
  5. Khetha indima yomsebenzisi (amaklayenti, okuzenzakalelayo, noma abasebenzi).
  6. Uma uthanda, khetha ibhokisi ukuze uthumele umsebenzisi isaziso se-imeyili.
  7. Chofoza okuthi LINDISA.

I-Case Active ingeza umsebenzisi kukesi. Manje abalawuli bangachofoza Londoloza ukuze bavale iwindi, noma bengeze umsebenzisi omusha, noma basuse umsebenzisi.

Ingeza umsebenzisi omusha

Uma inkundla ingakwazi ukuthola irekhodi lamanje, chofoza inkinobho ukuze ungeze umsebenzisi omusha.

Gcwalisa izinkambu bese uchofoza LINGA ukuze wengeze umsebenzisi endaweni. Okulandelayo, setha indima yomsebenzisi bese uchofoza LINGA futhi ukuze unikeze umsebenzisi icala. Ekugcineni, chofoza okuthi LINGA isikhathi esisodwa ukuze ulondoloze ikesi.
