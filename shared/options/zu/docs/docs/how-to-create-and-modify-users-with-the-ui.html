Indlela Yokudala Nokuguqula Abasebenzisi nge-UI

Lesi sihloko sibonisa abalawuli ukuthi babadala kanjani abasebenzisi futhi babuyekeze amarekhodi ngesixhumi esibonakalayo somsebenzisi wenkundla. Abaphathi bangakwazi futhi ukwengeza abasebenzisi esimweni esisha futhi bahlele amacala akhona.

Ukudala umsebenzisi

  1. Ngena ngemvume ku-akhawunti yomqondisi.
  2. Thola uhlelo lokusebenza ekhasini lezinhlelo zokusebenza bese uchofoza inkinobho ethi ADMIN.
  3. Kuphaneli Yokulawula, chofoza ithebhu yemenyu Abasebenzisi.
  4. Chofoza inkinobho ENTSHA. Iwindi elisha liyavela.

Gcwalisa izinkambu bese uchofoza LINDISA. I-Case Active ingeza igama kuhlu lwabasebenzisi abahlotshaniswa nalolu hlelo lokusebenza.

Ukushintsha umsebenzisi

Ukuze ubuyekeze inombolo encane yamarekhodi omsebenzisi, sebenzisa ithuluzi lokuhlela futhi wenze izinguquko kumsebenzisi ngamunye.

  • Qaphela Ukuze ulungise inombolo enkulu yamarekhodi omsebenzisi ngesikhathi esisodwa, cabanga ukusebenzisa ithuluzi Lokungenisa .
  1. Kuphaneli Yokulawula, chofoza ithebhu yemenyu Abasebenzisi.
  2. Thola umsebenzisi bese uchofoza Hlela.
  3. Yenza izinguquko bese uchofoza LINDISA.

Izinyathelo ezilandelayo

I-Case Active ayithumeli umsebenzisi isaziso se-imeyili. Abaphathi kufanele bathinte umsebenzisi futhi bamthumele isixhumanisi ekhasini lasekhaya le-Case Active. Umsebenzisi ube esechofoza inkinobho ethi NGENA futhi aqedele ukuhamba komsebenzi kokubhalisa okusha.

  • Lesi sihloko sichaza ukuhamba komsebenzi wokubhalisa kubasebenzisi abasha.
