Hoe beheerders gebruikerstoewijzingen aan cases beheren

Er zijn vijf manieren om gebruikers en casetoewijzingen te beheren. Klik op een koppeling voor meer informatie.

Methode Type Acties Details
Bij het maken van een zaak gebruikersinterface toevoegen Knop Beheer > Aanvragen > Knop Nieuw > Knop Gebruikers toewijzen
Gebruiker automatisch toewijzen met openbare aanmeldingslink Actie van de gebruiker toevoegen Admin-knop > Instellingen > Systeeminstellingen pagina
UI gebruiken op bestaande case gebruikersinterface toevoegen verwijderen Beheerknop > Cases > Gebruikers toewijzen
Importeer hulpmiddel gebruikersinterface toevoegen, wijzigen en verwijderen Beheerknop > Integraties > Importeren
API gebruiken (binnenkort beschikbaar) API toevoegen, wijzigen en verwijderen NVT
