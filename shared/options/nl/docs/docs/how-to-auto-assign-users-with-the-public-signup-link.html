Gebruikers automatisch toewijzen met de openbare aanmeldingslink

Beheerders leren hoe ze systeeminstellingen kunnen configureren zodat nieuwe gebruikers automatisch worden toegewezen aan een nieuw automatisch aangemaakte case.

De instellingenpagina openen

De eerste stap is het openen van de pagina Systeeminstellingen.

  1. Log in op het beheerdersaccount.
  2. Zoek een app op de Apps-pagina en klik op de ADMIN-knop.
  3. Klik op het tabblad Systemen en vervolgens op SYSTEM SETUP in het submenu.
  4. Scrol omlaag naar de kop GEBRUIKER AANMELDEN.

Systeeminstellingen configureren

  1. Stel de vervolgkeuzemenu's Aanmelding toestaan en Case automatisch maken in op Aan.
  2. Klik op het rolmenu en kies een optie (standaard wordt aanbevolen).
  3. Kopieer en deel de aanmeldingslink voor openbare gebruikers.

img

De link verzenden

Elke gebruiker die op de link klikt en de aanmelding voltooit, activeert het platform om een nieuwe case te genereren op de Cases-pagina en wijst de gebruiker automatisch toe aan die case.
