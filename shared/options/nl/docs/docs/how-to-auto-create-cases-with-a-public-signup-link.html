Hoe u automatisch cases maakt met een openbare aanmeldingslink

Dit artikel laat beheerders zien hoe ze een case kunnen maken (en tegelijkertijd een gebruiker kunnen toevoegen) door een enkele aanmeldingslink te sturen.

Het lokaliseren van de koppeling

  1. Log in op het beheerdersaccount.
  2. Zoek een app op de Apps-pagina en klik op de ADMIN-knop.
  3. Klik in het beheerdersdashboard op het menutabblad Instellingen.
  4. Zorg ervoor dat de submenutab SYSTEM SETUP is geselecteerd.

Instellingen configureren

Scrol omlaag naar de kop GEBRUIKER AANMELDEN.

Zorg ervoor dat het veld Aanmelden toestaan is ingesteld op Aan. (vereist) Wanneer deze functie is ingeschakeld, genereert Case Active een unieke link.

  1. Klik op het veld Auto Create Case en selecteer Aan. Het veld Casegebruikersrol wordt weergegeven.
  2. Selecteer een rol. (zie onder)

Case gebruikersrollen

Er zijn drie casusgebruikersrollen.

  • klanten (aanbevolen)
  • standaard
  • personeel

De link verzenden

Deel de link met een of meer mensen. Elke gebruiker die op de link klikt en de aanmelding voltooit, activeert het platform om een nieuwe casus op de casuspagina te genereren en automatisch de gebruiker aan de casus toe te wijzen.
