Форматирование данных о назначении обращений пользователей с помощью инструмента импорта

Ниже приведен список полей данных в файлах CSV. Каждый вариант импорта имеет разную комбинацию полей.

Идентификатор дела

  • Введите идентификатор пользователя или тега дела.
  • Перейдите на страницу «Обращения» > «Правка» > «Идентификатор тега» ИЛИ перейдите на страницу «Обращения» > «Правка» > «Идентификатор обращения в URL-адресе».

Логин пользователя

  • Введите идентификатор пользователя или идентификатор тега.
  • Перейдите на страницу «Пользователи» > «Правка» > «Идентификатор тега» ИЛИ перейдите на страницу «Пользователи» > «Правка» > «Идентификатор пользователя в URL-адресе».

case_role_order

  • Оставьте ячейку пустой.

case_role_ID

  • Введите: клиенты, персонал или по умолчанию.

case_notify

  • 0 = нет
  • 1 = Да, отправить пользователю уведомление по электронной почте.
