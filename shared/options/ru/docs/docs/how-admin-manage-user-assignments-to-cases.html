Как администраторы управляют назначениями пользователей для обращений

Существует пять способов управления пользователями и назначениями дел. Щелкните ссылку, чтобы узнать, как это сделать.

Метод Тип Действия Подробности
При создании кейса интерфейс Добавлять Кнопка «Администратор» > «Обращения» > кнопка «Создать» > кнопка «Назначить пользователей»
Автоматически назначать пользователя по общедоступной ссылке для регистрации Действия пользователя Добавлять Кнопка «Администратор» > «Настройки» > страница «Настройка системы»
Использование пользовательского интерфейса в существующем случае интерфейс добавить и удалить Кнопка администратора > Обращения > Назначить пользователей
Инструмент импорта интерфейс добавлять, изменять и удалять Кнопка администратора > Интеграции > Импорт
Использование API (скоро) API добавлять, изменять и удалять Н/Д
