Как автоматически создавать обращения с общедоступной ссылкой для регистрации

В этой статье администраторам показано, как создать обращение (и одновременно добавить пользователя), отправив одну ссылку для регистрации.

Расположение ссылки

  1. Войдите в учетную запись администратора.
  2. Найдите приложение на странице приложений и нажмите кнопку ADMIN.
  3. В панели администратора перейдите на вкладку меню «Настройки».
  4. Убедитесь, что выбрана вкладка подменю SYSTEM SETUP.

изображение

Настройка параметров

Прокрутите вниз до заголовка USER SIGNUP.

Убедитесь, что в поле Разрешить регистрацию установлено значение Вкл. (обязательно) Когда эта функция включена, Case Active создает уникальную ссылку.

  1. Щелкните поле «Автоматическое создание обращения» и выберите «Вкл.». Отображается поле Роль пользователя дела.
  2. Выберите роль. (Смотри ниже)

изображение

Роли пользователей дела

Существует три роли пользователя дела.

  • клиенты (рекомендуется)
  • дефолт
  • сотрудники

Отправка ссылки

Поделитесь ссылкой с одним или несколькими людьми. Каждый пользователь, который щелкает ссылку и завершает регистрацию, запускает платформу для создания нового обращения на странице обращений и автоматического назначения пользователя в деле.
