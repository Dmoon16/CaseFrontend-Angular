Как форматировать данные CSV

Ниже приведен список полей данных в файлах CSV. Каждый вариант импорта имеет разную комбинацию полей.

имя пользователя

  • Введите адрес электронной почты пользователя.

собственное имя

  • Введите имя пользователя.

фамилия

  • Введите фамилию пользователя.

место действия

  • В этом столбце проверяется предпочтительный язык пользователя.
  • Введите en для английского языка, sp для испанского.

зонаинформация

  • Это устанавливает часовой пояс пользователя. Введите континент и город без пробелов (используйте подчеркивание).
  • Необходимо использовать параметр в раскрывающемся меню «Профиль» > «ЯЗЫК И РЕГИОН» > «Часовой пояс».
  • Используйте наиболее подходящий часовой пояс. Например, Бостон отсутствует в раскрывающемся списке, поэтому мы можем использовать Нью-Йорк (или любой город в часовом поясе EST/EDT).

host_tag_id

  • Введите постоянный ярлык.
  • az, 0-9, -, / допускаются.

case_status

  • 0 = закрыто
  • 1 = активен
