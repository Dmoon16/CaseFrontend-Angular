Форматирование данных управления делами с помощью инструмента импорта

Ниже приведен список обязательных полей в файлах CSV для управления обращениями с помощью инструмента импорта.

имя пользователя

  • Введите адрес электронной почты пользователя.

tag_id (идентификатор, на который вы можете сослаться позже)

  • уникальный идентификатор (используйте буквенно-цифровые символы, допускаются дефисы)

case_status

  • 0 = закрыто
  • 1 = активен

date_opened

  • месяц день год
  • мм/дд/гггг

о

  • уникальное описание дела (используйте буквенно-цифровые символы, допускаются дефисы)

изображение
