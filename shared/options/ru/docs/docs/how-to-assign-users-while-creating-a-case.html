Как назначать пользователей при создании дела

В этой статье администраторам показано, как назначить пользователя при создании нового обращения. Если пользователь еще не зарегистрирован на платформе, вы можете добавить нового пользователя при создании нового обращения.

Создание дела

  1. Войдите в учетную запись администратора.
  2. Найдите приложение на странице приложений и нажмите кнопку ADMIN.
  3. Щелкните вкладку меню «Случаи».
  4. Нажмите кнопку НОВЫЙ. Появится новое окно.

изображение

  • Заполните поля, чтобы создать обращение, а затем прокрутите вниз до заголовка «Назначенные пользователи».

Назначение пользователя

Текущий пользователь

Выполните следующие действия, чтобы назначить текущего пользователя новому делу.

Нажмите кнопку НАЗНАЧИТЬ ПОЛЬЗОВАТЕЛЕЙ и заполните поля.

  1. Щелкните раскрывающийся список «Пользователь» и выберите идентификатор (например, адрес электронной почты).
  2. Введите значение в пустое поле (например, адрес электронной почты).
  3. Если запись пользователя существует на платформе, поле автоматически предлагает имя.
  4. Выберите правильного пользователя.
  5. Выберите роль пользователя (клиенты, по умолчанию или персонал).
  6. При необходимости установите флажок, чтобы отправить пользователю уведомление по электронной почте.
  7. Нажмите СОХРАНИТЬ.

изображение

Case Active добавляет пользователя в дело. Теперь администраторы могут нажать «Сохранить», чтобы закрыть окно, добавить нового пользователя или удалить пользователя.

изображение

Добавление нового пользователя

Если платформа не может найти текущую запись, нажмите кнопку, чтобы добавить нового пользователя.

изображение

Заполните поля и нажмите СОХРАНИТЬ, чтобы добавить пользователя на платформу. Затем установите роль пользователя и снова нажмите СОХРАНИТЬ, чтобы назначить пользователя делу. Наконец, нажмите СОХРАНИТЬ еще раз, чтобы сохранить дело.
