Как создать приложение

В этой статье показано, как новым пользователям создать бесплатную пробную учетную запись. Case Active автоматически предоставляет новому пользователю права администратора для этой учетной записи.

Создание учетной записи

  1. Перейдите на страницу входа .
  2. Нажмите кнопку «Начать».
  3. Введите адрес электронной почты.
  4. Нажмите "Далее.

Проверка учетной записи

  1. Откройте новую вкладку браузера и войдите в свою учетную запись электронной почты.
  2. Скопируйте шестизначный код.
  3. Вернитесь на вкладку «Активное дело».
  4. Вставьте код в поля. Появится новое окно.

В новом окне:

  1. Заполните необходимые поля.
  2. Прочтите условия обслуживания и поставьте галочку, чтобы согласиться.
  3. Нажмите СОЗДАТЬ БЕСПЛАТНУЮ АККАУНТ.

Завершение настройки приложения

Рабочий процесс отображается на следующей странице. Выполните шаги, чтобы завершить настройку приложения. Вы всегда можете вернуться на эти страницы, чтобы отредактировать или обновить информацию или статус своей учетной записи.

На странице «О приложении» заполните поля с названием компании и веб-сайтом (обязательно), а затем нажмите «Далее».

На странице «Информация для выставления счетов» щелкните поле «План платежей» и просмотрите параметры. Значение по умолчанию — бесплатно. Нажмите ОТПРАВИТЬ.

После отправки формы Case Active отобразит страницу приложений. Для каждого приложения, которое вы создаете или к которому принадлежите, на странице отображаются:

  1. Имя приложения
  2. Ваш тип (роль пользователя)
  3. Статус

изображение

Страница приложений является отправной точкой для многих функций администратора:

  1. Нажмите кнопку APP, чтобы общаться, заполнять документы, общаться в видеочате (и т. д.) по вопросам, связанным с делом.
  2. Нажмите кнопку ADMIN, чтобы открыть панель администратора. Здесь администраторы управляют пользователями, делами, интеграциями и многим другим.
  3. Щелкните многоточие, чтобы управлять настройками выставления счетов и уведомлений.
  • Примечание . Case Active отображает кнопку ADMIN для пользователей с правами администратора. Пользователи, не имеющие прав администратора, не видят кнопку ADMIN (как показано на экране ниже).
