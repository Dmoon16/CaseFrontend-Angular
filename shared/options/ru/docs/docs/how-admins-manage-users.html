Как администраторы управляют пользователями

Администраторы создают, изменяют или отключают доступ к приложениям и делам. Когда пользователь входит на платформу в первый раз, ему необходимо выполнить рабочий процесс профиля.

  • Нажмите здесь, чтобы узнать больше о новом рабочем процессе профиля пользователя.

Управление пользователем

В таблице ниже показаны пять способов управления пользователем. Щелкните ссылку, чтобы узнать, как это сделать.

Метод Тип Действия Подробности
Использование пользовательского интерфейса интерфейс добавить и изменить Кнопка администратора > Пользователи
При создании дела и назначении пользователя на дело интерфейс Добавлять Кнопка администратора > Обращения > Новая кнопка > Создать страницу обращения
Автоматическое создание пользователя по общедоступной ссылке для регистрации Действие пользователя Добавлять Кнопка «Администратор» > «Настройки» > страница «Настройка системы»
Инструмент импорта интерфейс добавить и изменить Кнопка администратора > Интеграции > Импорт
Использование API (скоро) API добавить и изменить Н/Д

Уведомления

Платформа не отправляет пользователям уведомление, когда они добавляются в приложение. Однако администраторы могут включить уведомления для назначений дел.
