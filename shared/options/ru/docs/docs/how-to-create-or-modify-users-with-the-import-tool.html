Как создать или изменить пользователей с помощью инструмента импорта

В этой статье администраторам показано, как создавать пользователей с помощью инструмента импорта.

Открытие инструмента импорта

Администраторы могут использовать инструмент импорта для массовой загрузки новых пользователей и изменения существующих записей пользователей.

  1. Войдите в учетную запись администратора.
  2. Найдите приложение на странице приложений и нажмите кнопку ADMIN.
  3. На панели администратора щелкните вкладку «Интеграция», а затем «ИМПОРТ» в подменю.
  4. Нажмите кнопку НОВЫЙ. Диалоговое окно «Добавить импорт» отображается на новой вкладке браузера.

Создание записей

После открытия диалогового окна «Добавить импорт»:

  1. Щелкните поле Тип и выберите параметр (см. ниже). Отображается ссылка.
  2. Щелкните ссылку Загрузить образец . CSV-файл загружается на ваше устройство.

Затем откройте файл и добавьте записи. Щелкните здесь, чтобы узнать, как использовать данные формата в файле CSV.

изображение

Тип

Если щелкнуть поле Тип, появится раскрывающееся меню. Выберите один из двух вариантов массового импорта.

:exclamation: ПРИМЕЧАНИЕ В раскрывающемся меню есть много параметров, но только два из них относятся к инструменту импорта. Каждый вариант имеет отдельный образец CSV-файла.

Добавить пользователей

  • Выберите этот вариант для простой массовой загрузки записей пользователей.
  • Щелкните ссылку «Загрузить образец», чтобы открыть CSV-файл с заголовками столбцов для типа импорта «Добавить пользователей».

Изменить пользователей

  • Выберите этот параметр, чтобы обновить данные для текущих пользователей.
  • Щелкните ссылку «Загрузить образец», чтобы открыть CSV-файл с заголовками столбцов для типа импорта «Изменить пользователей».

Загрузка файла

Вы можете загрузить файл сразу или вернуться к диалоговому окну «Добавить импорт» позже и выполнить эти шаги.

  1. Откройте диалоговое окно «Добавить импорт».
  2. Введите уникальное имя для CSV-файла.
  3. Нажмите значок плюса (+), чтобы загрузить файл с вашего устройства.
  4. Щелкните поле Тип и выберите параметр. (см. выше)
  5. Нажмите СОХРАНИТЬ.

Обработка

В зависимости от количества записей в файле и загрузки системы платформе может потребоваться несколько минут для обработки импорта.

изображение

По завершении импорта платформа настраивает записи пользователей (т. е. добавляет, изменяет или отключает) на панели управления « Пользователи » администратора.

Отказ

Если операция импорта не удалась, платформа возвращает сообщение Failed. В большинстве случаев причиной сбоя является форматирование данных.

изображение

Чтобы увидеть причину сбоя:

  1. Нажмите « Просмотр » под заголовком «Действие».
  2. На следующей странице наведите указатель мыши на значок «Не удалось». Во всплывающей подсказке отображается ошибка.
  3. Перейдите к файлу CVS, настройте и исправьте ошибку.
  4. Загрузите файл еще раз.
