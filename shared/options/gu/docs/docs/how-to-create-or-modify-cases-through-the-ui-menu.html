UI મેનૂ દ્વારા કેસ કેવી રીતે બનાવવું અથવા સંશોધિત કરવું

આ લેખ સંચાલકોને બતાવે છે કે પ્લેટફોર્મના યુઝર ઇન્ટરફેસ સાથે કેસ કેવી રીતે બનાવવો. એડમિન્સ વપરાશકર્તાઓને નવા કેસમાં ઉમેરી શકે છે અને હાલના કેસોને અપડેટ કરી શકે છે.

કેસ બનાવી રહ્યા છે

  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્લિકેશન પૃષ્ઠ પર એક એપ્લિકેશન શોધો અને એડમિન બટનને ક્લિક કરો.
  3. એડમિન પેનલ પર, કેસ મેનુ ટેબ પર ક્લિક કરો.
  4. નવું બટન ક્લિક કરો. એક નવી વિન્ડો દેખાય છે.

જરૂરી ફીલ્ડ્સ ભરો અને સેવ પર ક્લિક કરો.

વપરાશકર્તાઓ ઉમેરી રહ્યા છે

વૈકલ્પિક રીતે, સંચાલકો આ પૃષ્ઠ પરના કેસમાં વપરાશકર્તાઓને ઉમેરી શકે છે. નીચે સ્ક્રોલ કરો અને વપરાશકર્તાઓને સોંપો બટનને ક્લિક કરો.

અસાઇન યુઝર મોડલમાં, ફીલ્ડ્સ ભરો. વપરાશકર્તાને ઇમેઇલ સૂચના મોકલવા માટે બોક્સને ચેક કરો. (વૈકલ્પિક)

કેસમાં ફેરફાર કરવો

નાની સંખ્યામાં કેસોને અપડેટ કરવા માટે, એડિટ ટૂલનો ઉપયોગ કરો અને દરેક વ્યક્તિગત કેસમાં ફેરફાર કરો.

  • નોંધ એક બેઠકમાં મોટી સંખ્યામાં કેસ અપડેટ કરવા માટે, આયાત સાધનનો ઉપયોગ કરવાનું વિચારો
  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્લિકેશન પૃષ્ઠ પર એક એપ્લિકેશન શોધો અને એડમિન બટનને ક્લિક કરો.
  3. એડમિન પેનલ પર, કેસ મેનુ ટેબ પર ક્લિક કરો.
  4. કેસ શોધો અને સંપાદિત કરો ક્લિક કરો.
  5. ફેરફારો કરો અને સાચવો પર ક્લિક કરો.
