વપરાશકર્તા પ્રોફાઇલ કેવી રીતે અપડેટ કરવી

આ લેખ વપરાશકર્તાઓને તેમના પ્રોફાઇલ પૃષ્ઠ પર માહિતી કેવી રીતે અપડેટ કરવી તે બતાવે છે.

  • વપરાશકર્તાઓ તેમની પોતાની પ્રોફાઇલ અપડેટ કરી શકે છે.
  • સંચાલકો કોઈપણ વપરાશકર્તાના પ્રોફાઇલ પૃષ્ઠને અપડેટ કરી શકે છે.

વપરાશકર્તાઓ માટે

પ્રથમ પગલું પ્રોફાઇલ પૃષ્ઠ ખોલવાનું છે. વપરાશકર્તાઓ અને સંચાલકો તેમના પોતાના પ્રોફાઇલ પૃષ્ઠને અપડેટ કરવા માટે આ પગલાંને અનુસરો.

  1. તમારા કેસ એક્ટિવ એકાઉન્ટમાં લૉગ ઇન કરો.
  2. પૃષ્ઠની ટોચ પર પ્રોફાઇલ ટેબ પર ક્લિક કરો.
  3. ફેરફારો કરો.
  4. સેવ પર ક્લિક કરો.

મોટાભાગના ક્ષેત્રો વૈકલ્પિક છે, અને કેટલાક ક્ષેત્રો આવશ્યક છે. આવશ્યક ક્ષેત્રો છે:

  • પ્રથમ નામ
  • છેલ્લું નામ
  • ઈ - મેઈલ સરનામું
  • ભાષા
  • સમય ઝોન

એડમિન માટે

એડમિન્સ અન્ય વપરાશકર્તાઓ માટે પ્રોફાઇલ માહિતી પણ અપડેટ કરી શકે છે. આમાં HOST USER INFO શીર્ષક હેઠળ ત્રણ ફીલ્ડનો સમાવેશ થાય છે.

  1. તમારા કેસ એક્ટિવ એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્લિકેશન શોધો અને એડમિન બટન પર ક્લિક કરો.
  3. ટોચના મેનૂ પર વપરાશકર્તાઓને ક્લિક કરો.
  4. સબ-મેનૂ પર જાઓ અને તમે ઍક્સેસ કરવા માંગો છો તે વપરાશકર્તા સૂચિ પસંદ કરો (સક્રિય અથવા અક્ષમ). ડિફૉલ્ટ સેટિંગ સક્રિય છે.
  5. સૂચિમાં વપરાશકર્તાને શોધો અને સંપાદિત કરો પર ક્લિક કરો.
  6. જરૂર મુજબ ફેરફારો કરો. (નીચે જુઓ)
  7. સેવ પર ક્લિક કરો.

વપરાશકર્તા માહિતી બદલો

સંચાલકો વપરાશકર્તાની માહિતી બદલી શકે છે.

  2. વપરાશકર્તા સ્થિતિ સેટ કરો (નિષ્ક્રિય કરવા માટે સક્રિય).
  3. એક અનન્ય ટેગ ID દાખલ કરો. (વૈકલ્પિક)
  4. સેવ પર ક્લિક કરો.
