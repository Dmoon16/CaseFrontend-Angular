એડમિન્સ કેસ કેવી રીતે મેનેજ કરે છે

કેસ મેનેજ કરવાની ચાર રીતો છે. કેવી રીતે જાણવા માટે એક લિંક પર ક્લિક કરો.

પદ્ધતિ પ્રકાર ક્રિયાઓ વિગતો
UI નો ઉપયોગ કરીને UI ઉમેરો અને સંશોધિત કરો એડમિન બટન > કેસ
પબ્લિક સાઇનઅપ લિંકનો ઉપયોગ કરીને ઑટોમૅટિક રીતે કેસ બનાવો વપરાશકર્તા ક્રિયા ઉમેરો એડમિન બટન > સેટિંગ્સ > સિસ્ટમ સેટઅપ પેજ
આયાત સાધન UI ઉમેરો અને સંશોધિત કરો એડમિન બટન > એકીકરણ > આયાત
