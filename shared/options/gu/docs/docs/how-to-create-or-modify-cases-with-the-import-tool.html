આયાત ટૂલ વડે કેસ કેવી રીતે બનાવવું અથવા સંશોધિત કરવું

આ લેખ આયાત ટૂલ વડે હાલના કેસ કેવી રીતે બનાવવું અથવા સંશોધિત કરવું તે વ્યવસ્થાપકોને બતાવે છે.

આયાત સાધન ખોલી રહ્યું છે

  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્લિકેશન પૃષ્ઠ પર એક એપ્લિકેશન શોધો અને એડમિન બટનને ક્લિક કરો.
  3. એડમિન પેનલ પર, એકીકરણ ટેબ પર ક્લિક કરો અને પછી સબમેનુ પર આયાત કરો.
  4. નવું બટન ક્લિક કરો. નવા બ્રાઉઝર ટેબમાં આયાત ઉમેરો સંવાદ બોક્સ પ્રદર્શિત થાય છે.

img

કેસો બનાવી રહ્યા છે

આયાત ઉમેરો સંવાદ બોક્સ ખુલે તે પછી:

  1. પ્રકાર ફીલ્ડ પર ક્લિક કરો અને એક વિકલ્પ પસંદ કરો (નીચે જુઓ). એક લિંક પ્રદર્શિત થાય છે.
  2. ડાઉનલોડ નમૂના લિંક પર ક્લિક કરો. તમારા ઉપકરણ પર CSV ફાઇલ ડાઉનલોડ થાય છે.

આગળ, ફાઇલ ખોલો અને કેસ ઉમેરો.

પ્રકાર

જ્યારે તમે પ્રકાર ફીલ્ડ પર ક્લિક કરો છો, ત્યારે ડ્રોપ-ડાઉન મેનૂ પ્રદર્શિત થાય છે. બલ્ક કેસ આયાત માટે બેમાંથી એક વિકલ્પ પસંદ કરો.

  • નોંધ ડ્રોપ-ડાઉન મેનૂમાં ઘણા વિકલ્પો છે, પરંતુ માત્ર બે જ કેસ મેનેજ કરવા સંબંધિત છે. દરેક વિકલ્પમાં અલગ નમૂનાની CSV ફાઇલ હોય છે.

કેસો ઉમેરો

  • કેસોના સરળ બલ્ક અપલોડ માટે આ વિકલ્પ પસંદ કરો.
  • કેસ આયાત પ્રકાર બનાવો માટે કૉલમ હેડિંગ સાથે CSV ફાઇલ ખોલવા માટે નમૂના ડાઉનલોડ કરો લિંક પર ક્લિક કરો.

કેસોમાં ફેરફાર કરો

  • વર્તમાન કેસ અપડેટ કરવા માટે આ વિકલ્પ પસંદ કરો.
  • સંશોધિત કેસ આયાત પ્રકાર માટે કૉલમ હેડિંગ સાથે CSV ફાઇલ ખોલવા માટે નમૂના ડાઉનલોડ કરો લિંક પર ક્લિક કરો.

ફાઈલ અપલોડ કરી રહ્યા છીએ

તમે તરત જ ફાઇલ અપલોડ કરી શકો છો અથવા પછીની તારીખે આયાત ઉમેરો સંવાદ બોક્સ પર પાછા આવી શકો છો અને આ પગલાંઓ પૂર્ણ કરી શકો છો.

  1. આયાત ઉમેરો સંવાદ બોક્સ ખોલો.
  2. CSV ફાઇલ માટે અનન્ય નામ દાખલ કરો.
  3. તમારા ઉપકરણમાંથી ફાઇલ અપલોડ કરવા માટે વત્તા (+) આયકન પર ક્લિક કરો.
  4. પ્રકાર ફીલ્ડ પર ક્લિક કરો અને એક વિકલ્પ પસંદ કરો. (ઉપર જુવો)
  5. સેવ પર ક્લિક કરો.

પ્રક્રિયા

ફાઇલ અને સિસ્ટમ લોડમાં રેકોર્ડ્સની સંખ્યાના આધારે, પ્લેટફોર્મ આયાતની પ્રક્રિયા કરવામાં ઘણી મિનિટો લઈ શકે છે.

જ્યારે આયાત પૂર્ણ થાય છે, ત્યારે પ્લેટફોર્મ એડમિનનાં કેસ ડેશબોર્ડ પર કેસ રેકોર્ડ્સ (એટલે કે ઉમેરે છે અથવા સુધારે છે) ગોઠવે છે.

નિષ્ફળતા

જો આયાત કામગીરી નિષ્ફળ જાય, તો પ્લેટફોર્મ નિષ્ફળ સંદેશ પરત કરે છે. મોટાભાગના કિસ્સાઓમાં, નિષ્ફળતાનું કારણ ડેટા ફોર્મેટિંગ છે.

img

નિષ્ફળતાનું કારણ જોવા માટે:

  1. ACTION શીર્ષક હેઠળ વ્યૂ પર ક્લિક કરો.
  2. આગલા પૃષ્ઠ પર, નિષ્ફળ ચિહ્ન પર માઉસને હોવર કરો. ટૂલટીપ ભૂલ દર્શાવે છે.
  3. CVS ફાઇલ પર જાઓ અને એડજસ્ટ કરો અને ભૂલને ઠીક કરો.
  4. ફાઇલ ફરીથી અપલોડ કરો.
