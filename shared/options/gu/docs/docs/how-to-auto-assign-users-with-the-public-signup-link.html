સાર્વજનિક સાઇનઅપ લિંક સાથે વપરાશકર્તાઓને સ્વતઃ સોંપણી કેવી રીતે કરવી

એડમિન્સ સિસ્ટમ સેટિંગ્સને કેવી રીતે ગોઠવવી તે શીખે છે જેથી નવા વપરાશકર્તાઓને નવા સ્વતઃ બનાવેલ કેસમાં આપમેળે સોંપવામાં આવે.

સેટઅપ પેજ ખોલી રહ્યા છીએ

પ્રથમ પગલું એ સિસ્ટમ્સ સેટઅપ પૃષ્ઠ ખોલવાનું છે.

  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્સ પેજ પર એક એપ શોધો અને એડમિન બટન પર ક્લિક કરો.
  3. સબમેનુ પર સિસ્ટમ્સ ટેબ અને પછી સિસ્ટમ સેટઅપ પર ક્લિક કરો.
  4. USER SIGNUP મથાળા સુધી નીચે સ્ક્રોલ કરો.

સિસ્ટમ સેટિંગ્સને ગોઠવી રહ્યું છે

  1. સાઇનઅપને મંજૂરી આપો અને કેસ સ્વતઃ બનાવો ડ્રોપ-ડાઉન મેનુને ચાલુ પર સેટ કરો.
  2. રોલ મેનૂ પર ક્લિક કરો અને એક વિકલ્પ પસંદ કરો (ડિફૉલ્ટની ભલામણ કરવામાં આવે છે).
  3. સાર્વજનિક વપરાશકર્તા સાઇનઅપ લિંક કૉપિ કરો અને શેર કરો.

લિંક મોકલી રહ્યું છે

દરેક વપરાશકર્તા જે લિંકને ક્લિક કરે છે અને સાઇનઅપ પૂર્ણ કરે છે તે કેસ પેજ પર નવો કેસ જનરેટ કરવા પ્લેટફોર્મને ટ્રિગર કરે છે અને તે કિસ્સામાં વપરાશકર્તાને સ્વતઃ-સોંપણી કરે છે.
