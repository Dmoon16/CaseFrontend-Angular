કેસ બનાવતી વખતે વપરાશકર્તાઓને કેવી રીતે સોંપવું

આ લેખ એડમિન્સને બતાવે છે કે નવો કેસ બનાવતી વખતે વપરાશકર્તાને કેવી રીતે સોંપવું. જો વપરાશકર્તા પ્લેટફોર્મ પર પહેલાથી નોંધાયેલ નથી, તો તમે નવો કેસ બનાવતી વખતે એક નવો વપરાશકર્તા ઉમેરી શકો છો.

કેસ બનાવી રહ્યા છે

  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્લિકેશન પૃષ્ઠ પર એક એપ્લિકેશન શોધો અને એડમિન બટનને ક્લિક કરો.
  3. કેસ મેનુ ટેબ પર ક્લિક કરો.
  4. નવું બટન ક્લિક કરો. એક નવી વિન્ડો દેખાય છે.
  • કેસ બનાવવા માટે ફીલ્ડ્સ ભરો અને પછી અસાઇન્ડ યુઝર્સ હેડિંગ પર નીચે સ્ક્રોલ કરો.

વપરાશકર્તાને સોંપવું

વર્તમાન વપરાશકર્તા

વર્તમાન વપરાશકર્તાને નવા કેસમાં સોંપવા માટે આ પગલાં અનુસરો.

ASSIN USERS બટન પર ક્લિક કરો અને ફીલ્ડ્સ ભરો.

  1. વપરાશકર્તા ડ્રોપ-ડાઉન પર ક્લિક કરો અને ઓળખકર્તા (દા.ત. ઈમેલ) પસંદ કરો.
  2. ખાલી ફીલ્ડમાં મૂલ્ય દાખલ કરો (દા.ત. ઈમેલ સરનામું).
  3. જો પ્લેટફોર્મમાં વપરાશકર્તા રેકોર્ડ અસ્તિત્વમાં છે, તો ફીલ્ડ સ્વતઃ નામ સૂચવે છે.
  4. યોગ્ય વપરાશકર્તા પસંદ કરો.
  5. વપરાશકર્તાની ભૂમિકા (ક્લાયન્ટ, ડિફોલ્ટ અથવા સ્ટાફ) પસંદ કરો.
  6. વૈકલ્પિક રીતે, વપરાશકર્તાને ઇમેઇલ સૂચના મોકલવા માટે બૉક્સને ચેક કરો.
  7. સેવ પર ક્લિક કરો.

img

કેસ એક્ટિવ વપરાશકર્તાને કેસમાં ઉમેરે છે. હવે એડમિન્સ વિન્ડો બંધ કરવા માટે સેવ પર ક્લિક કરી શકે છે, અથવા નવો વપરાશકર્તા ઉમેરી શકે છે અથવા વપરાશકર્તાને દૂર કરી શકે છે.

img

નવો વપરાશકર્તા ઉમેરી રહ્યા છીએ

જો પ્લેટફોર્મ વર્તમાન રેકોર્ડ શોધી શકતું નથી, તો નવા વપરાશકર્તાને ઉમેરવા માટે બટનને ક્લિક કરો.

img

ફીલ્ડ્સ ભરો અને પછી વપરાશકર્તાને પ્લેટફોર્મ પર ઉમેરવા માટે સાચવો પર ક્લિક કરો. આગળ, વપરાશકર્તાની ભૂમિકા સેટ કરો અને વપરાશકર્તાને કેસ સોંપવા માટે ફરીથી સાચવો પર ક્લિક કરો. છેલ્લે, કેસ સાચવવા માટે વધુ એક વખત સાચવો પર ક્લિક કરો.
