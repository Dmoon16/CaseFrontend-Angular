UI વડે વપરાશકર્તાઓને કેવી રીતે બનાવવું અને સંશોધિત કરવું

આ લેખ સંચાલકોને બતાવે છે કે કેવી રીતે વપરાશકર્તાઓ બનાવવા અને પ્લેટફોર્મના વપરાશકર્તા ઇન્ટરફેસ સાથે રેકોર્ડ અપડેટ કરવા. એડમિન્સ વપરાશકર્તાઓને નવા કેસમાં ઉમેરી શકે છે અને હાલના કેસમાં ફેરફાર કરી શકે છે.

વપરાશકર્તા બનાવી રહ્યા છીએ

  1. એડમિન એકાઉન્ટમાં લોગ ઇન કરો.
  2. એપ્લિકેશન પૃષ્ઠ પર એક એપ્લિકેશન શોધો અને એડમિન બટનને ક્લિક કરો.
  3. એડમિન પેનલ પર, વપરાશકર્તાઓ મેનૂ ટેબ પર ક્લિક કરો.
  4. નવું બટન ક્લિક કરો. એક નવી વિન્ડો દેખાય છે.

ફીલ્ડ્સ ભરો અને સેવ પર ક્લિક કરો. કેસ એક્ટિવ આ એપ સાથે સંકળાયેલા યુઝર્સની યાદીમાં નામ ઉમેરે છે.

વપરાશકર્તાને સંશોધિત કરી રહ્યા છીએ

ઓછી સંખ્યામાં વપરાશકર્તા રેકોર્ડ અપડેટ કરવા માટે, સંપાદન સાધનનો ઉપયોગ કરો અને દરેક વ્યક્તિગત વપરાશકર્તામાં ફેરફાર કરો.

  • નોંધ એક બેઠકમાં મોટી સંખ્યામાં વપરાશકર્તા રેકોર્ડ્સને સંશોધિત કરવા માટે, આયાત સાધનનો ઉપયોગ કરવાનું વિચારો.
  1. એડમિન પેનલ પર, વપરાશકર્તાઓ મેનૂ ટેબ પર ક્લિક કરો.
  2. વપરાશકર્તાને શોધો અને સંપાદિત કરો પર ક્લિક કરો.
  3. ફેરફારો કરો અને સાચવો પર ક્લિક કરો.

આગામી પગલાં

કેસ એક્ટિવ વપરાશકર્તાને ઈમેલ સૂચના મોકલતું નથી. એડમિને વપરાશકર્તાનો સંપર્ક કરવો જોઈએ અને તેમને કેસ એક્ટિવ હોમ પેજ પર એક લિંક મોકલવી જોઈએ. પછી વપરાશકર્તા LOGIN બટનને ક્લિક કરે છે અને નવા સાઇનઅપ માટે વર્કફ્લો પૂર્ણ કરે છે.

  • આ લેખ નવા વપરાશકર્તાઓ માટે સાઇનઅપ વર્કફ્લોનું વર્ણન કરે છે.
