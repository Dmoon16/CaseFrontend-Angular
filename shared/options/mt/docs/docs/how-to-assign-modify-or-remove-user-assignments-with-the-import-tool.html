Kif Tassenja, Immodifika jew Neħħi Assenji tal-Utenti bl-Għodda tal-Importazzjoni

Dan l-artikolu juri lill-amministraturi kif jimmaniġġjaw l-utenti tal-każ bl-għodda tal-Importazzjoni.

L-amministraturi jistgħu:

  • tassenja utenti għal każ
  • timmodifika utenti diġà assenjati għal każ
  • neħħi utenti assenjati għal każ

Ftuħ l-għodda ta 'importazzjoni

  1. Idħol fil-kont tal-amministratur.
  2. Sib app fuq il-paġna Apps u kklikkja l-buttuna ADMIN.
  3. Ikklikkja t-tab Integrazzjoni u mbagħad IMPORTAZZJONIJIET fuq is-submenu.
  4. Ikklikkja l-buttuna ĠDIDA. Il-kaxxa tad-djalogu Żid Importazzjonijiet tidher.

Ħolqien ta' rekords

Niżżel fajl CSV wieħed jew aktar. Uża dawn il-fajls biex iżżid, timmodifika jew tneħħi utenti tal-każ. Wara li tiftaħ il-kaxxa tad-djalogu Żid Importazzjonijiet:

  1. Ikklikkja l-qasam Tip.
  2. Skrollja fil-menu drop-down u agħżel għażla (ara hawn taħt). Wirjiet link.
  3. Ikklikkja l-link Niżżel il-Kampjun. Fajl CSV jitniżżel fuq it-tagħmir tiegħek.

Iftaħ il-fajl u żid ir-rekords. Ikklikkja hawn biex titgħallem kif tuża data tal-format fil-fajl CSV.

Għażliet tal-fajl

Assenja utenti għal każi * Agħżel din l-għażla biex iżżid utenti f'każ. * Ħalli ċ-ċellula case_role_order vojta. * Mur fil-paġna Utenti u kkopja l-ID tat-Tag għal din il-persuna. Jekk il-qasam Tag ID huwa vojt, daħħal ID unika, ikklikkja Save, u mbagħad ikkopja l-ID fil-fajl CSV.

Immodifika l-utenti assenjati għal każijiet * Agħżel din l-għażla biex taġġorna d-dejta għall-utenti attwali assenjati għal każ. * Uża l-ID tat-Tag tal-Każ għall-kolonna case_ID.

Neħħi utenti assenjati għal każi * Agħżel din l-għażla biex tneħħi utenti assenjati għal każ.

Tlugħ tal-fajl

Tista 'ttella' fajl mill-ewwel jew terġa 'lura fil-kaxxa ta' dialog Żid Importazzjonijiet f'data aktar tard u tlesti dawn il-passi.

  1. Iftaħ il-kaxxa ta 'dialog Żid Importazzjonijiet.
  2. Daħħal isem uniku għall-fajl CSV.
  3. Ikklikkja l-ikona plus (+) biex ittella' l-fajl mit-tagħmir tiegħek.
  4. Ikklikkja l-qasam Tip u agħżel għażla. (ara hawn fuq)
  5. Ikklikkja SAVE.

Ipproċessar

Skont in-numru ta 'rekords fil-fajl u t-tagħbija tas-sistema, il-pjattaforma tista' tieħu diversi minuti biex tipproċessa l-importazzjoni.

Meta l-importazzjoni titlesta, il-pjattaforma taġġusta r-rekords tal-utent (jiġifieri żżid, timmodifika, jew tneħħi) fuq il-paġna Każijiet.

Nuqqas

Jekk l-operazzjoni ta' importazzjoni tfalli, il-pjattaforma tirritorna messaġġ Faled. Fil-biċċa l-kbira tal-każijiet, il-kawża tal-falliment hija l-ifformattjar tad-dejta.

Biex tara l-kawża tan-nuqqas:

  1. Ikklikkja Ara taħt l-intestatura Azzjoni.
  2. Fil-paġna li jmiss, poġġi l-maws fuq l-ikona Falliet. It-tooltip juri l-iżball.
  3. Mur fil-fajl CVS u aġġusta u rranġa l-iżball.
  4. Erġa' ttella' l-fajl.
