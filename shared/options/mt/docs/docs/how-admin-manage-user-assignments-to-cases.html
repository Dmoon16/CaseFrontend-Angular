Kif l-Amministraturi jimmaniġġjaw l-Assenjamenti tal-Utenti għall-Każijiet

Hemm ħames modi kif timmaniġġja l-utenti u l-assenjazzjonijiet tal-każijiet. Ikklikkja link biex titgħallem kif.

Metodu Tip Azzjonijiet Dettalji
Filwaqt li toħloq każ UI żid Buttuna Admin > Każijiet > buttuna ġdida > buttuna Assenja Utenti
Assenja l-utent awtomatikament billi tuża link ta' reġistrazzjoni pubblika Azzjoni tal-Utent żid Buttuna Admin > Settings > paġna Setup tas-Sistema
Uża UI fuq każ eżistenti UI żid & neħħi Buttuna Amministratur > Każijiet > Assenja Utenti
Għodda ta 'importazzjoni UI żid, immodifika, & neħħi Buttuna Admin > Integrazzjonijiet > Importazzjonijiet
