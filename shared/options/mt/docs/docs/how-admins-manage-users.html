Kif l-Amministraturi Jiġġestixxu l-Utenti

L-amministraturi joħolqu, jimmodifikaw jew jiskonnettjaw min jista' jaċċessa apps u każijiet. Meta utent jidħol fil-pjattaforma għall-ewwel darba, jeħtieġ li jlesti fluss tax-xogħol tal-profil.

  • Ikklikkja hawn biex titgħallem aktar dwar il-fluss tax-xogħol il-ġdid tal-profil tal-utent.

Ġestjoni ta' utent

It-tabella hawn taħt turi ħames modi kif timmaniġġja utent. Ikklikkja link biex titgħallem kif.

Notifiki

Il-pjattaforma ma tibgħatx notifika lill-utenti meta jiġu miżjuda ma 'app. Madankollu, l-amministraturi jistgħu jippermettu notifiki għall-assenjazzjonijiet tal-każijiet.
