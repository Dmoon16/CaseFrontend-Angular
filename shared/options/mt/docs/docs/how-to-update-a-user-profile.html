Kif Taġġorna Profil tal-Utent

Dan l-artikolu juri lill-utenti kif jaġġornaw l-informazzjoni fuq il-paġna tal-Profil tagħhom.

  • L-utenti jistgħu jaġġornaw il-profili tagħhom stess.
  • L-amministraturi jistgħu jaġġornaw il-paġna tal-Profil ta' kwalunkwe utent.

Għall-utenti

L-ewwel pass huwa li tiftaħ il-paġna tal-Profil. L-utenti u l-amministraturi jsegwu dawn il-passi biex jaġġornaw il-paġna tal-Profil tagħhom stess.

  1. Idħol fil-kont tiegħek Case Active.
  2. Ikklikkja t-tab Profil fin-naħa ta' fuq tal-paġna.
  3. Agħmel bidliet.
  4. Ikklikkja Save.

Ħafna mill-oqsma huma fakultattivi, u xi oqsma huma meħtieġa. L-oqsma meħtieġa huma:

  • Isem
  • Kunjom
  • Indirizz elettroniku
  • Lingwa
  • Żona tal-ħin

Għall-amministraturi

L-amministraturi jistgħu wkoll jaġġornaw l-informazzjoni tal-profil għal utenti oħra. Dan jinkludi tliet oqsma taħt l-intestatura INFORMAZZJONI DWAR L-UTENTI HOST.

  1. Idħol fil-kont amministratur tiegħek ta' Case Active.
  2. Sib app u kklikkja l-buttuna ADMIN.
  3. Ikklikkja Utenti fuq il-menu ta 'fuq.
  4. Mur fis-submenu u agħżel il-lista tal-utenti li trid taċċessa (ATTIVA jew B'DIŻABILITÀ). Is-setting default huwa Attiv.
  5. Sib utent fuq il-lista u kklikkja Editja.
  6. Agħmel bidliet kif meħtieġ. (ara isfel)
  7. Ikklikkja Save.

Ibdel l-informazzjoni tal-utent

L-amministraturi jistgħu jibdlu l-informazzjoni tal-utent.

  1. Issettja t-Tip ta' Utent (Utent jew Maniġer).
  2. Issettja l-Istatus tal-Utent (Attiv għal Disabilitat).
  3. Daħħal Tag ID unika. (mhux obbligatorju)
  4. Ikklikkja Save.
