როგორ განაახლოთ მომხმარებლის პროფილი

ეს სტატია აჩვენებს მომხმარებლებს, თუ როგორ განაახლონ ინფორმაცია მათი პროფილის გვერდზე.

  • მომხმარებლებს შეუძლიათ განაახლონ საკუთარი პროფილები.
  • ადმინისტრატორებს შეუძლიათ ნებისმიერი მომხმარებლის პროფილის გვერდის განახლება.

მომხმარებლებისთვის

პირველი ნაბიჯი არის პროფილის გვერდის გახსნა. მომხმარებლები და ადმინისტრატორები მიჰყვებიან ამ ნაბიჯებს საკუთარი პროფილის გვერდის განახლებისთვის.

  1. შედით თქვენს Case Active ანგარიშში.
  2. დააწკაპუნეთ პროფილის ჩანართზე გვერდის ზედა ნაწილში.
  3. Ცვლილებების შეტანა.
  4. დააწკაპუნეთ შენახვა.

ველების უმეტესობა არჩევითია და ზოგიერთი ველი სავალდებულოა. აუცილებელი ველებია:

  • Სახელი
  • Გვარი
  • Ელექტრონული მისამართი
  • Ენა
  • Დროის სარტყელი

ადმინებისთვის

ადმინისტრატორებს ასევე შეუძლიათ განაახლონ პროფილის ინფორმაცია სხვა მომხმარებლებისთვის. ეს მოიცავს სამ ველს მასპინძელი მომხმარებლის ინფორმაციის სათაურის ქვეშ.

  1. შედით თქვენს Case Active ადმინისტრატორის ანგარიშში.
  2. იპოვნეთ აპლიკაცია და დააჭირეთ ღილაკს ADMIN.
  3. დააჭირეთ მომხმარებლებს ზედა მენიუში.
  4. გადადით ქვემენიუში და აირჩიეთ მომხმარებელთა სია, რომელზეც გსურთ წვდომა (ACTIVE ან DISABLED). ნაგულისხმევი პარამეტრი არის აქტიური.
  5. იპოვნეთ მომხმარებელი სიაში და დააჭირეთ რედაქტირებას.
  6. შეიტანეთ ცვლილებები საჭიროებისამებრ. (იხილეთ ქვემოთ)
  7. დააწკაპუნეთ შენახვა.

მომხმარებლის ინფორმაციის შეცვლა

ადმინისტრატორებს შეუძლიათ მომხმარებლის ინფორმაციის შეცვლა.

  1. დააყენეთ მომხმარებლის ტიპი (მომხმარებელი ან მენეჯერი).
  2. დააყენეთ მომხმარებლის სტატუსი (აქტიური გამორთულია).
  3. შეიყვანეთ უნიკალური ტეგის ID. (სურვილისამებრ)
  4. დააწკაპუნეთ შენახვა.
