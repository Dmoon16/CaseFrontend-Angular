როგორ შევქმნათ და შეცვალოთ მომხმარებლები UI-ით

ეს სტატია გვიჩვენებს ადმინისტრატორებს, როგორ შექმნან მომხმარებლები და განაახლონ ჩანაწერები პლატფორმის მომხმარებლის ინტერფეისით. ადმინისტრატორებს ასევე შეუძლიათ მომხმარებლების დამატება ახალ საქმეში და არსებული შემთხვევების რედაქტირება.

მომხმარებლის შექმნა

  1. შედით ადმინისტრატორის ანგარიშზე.
  2. იპოვნეთ აპლიკაცია აპების გვერდზე და დააჭირეთ ღილაკს ADMIN.
  3. ადმინისტრაციულ პანელზე დააწკაპუნეთ მომხმარებლის მენიუს ჩანართზე.
  4. დააჭირეთ ღილაკს NEW. გამოჩნდება ახალი ფანჯარა.

შეავსეთ ველები და დააჭირეთ შენახვას. Case Active ამატებს სახელს ამ აპთან ასოცირებულ მომხმარებელთა სიაში.

მომხმარებლის შეცვლა

მცირე რაოდენობის მომხმარებლის ჩანაწერების გასაახლებლად გამოიყენეთ რედაქტირების ინსტრუმენტი და შეიტანეთ ცვლილებები თითოეულ ინდივიდუალურ მომხმარებელს.

  • შენიშვნა იმისათვის, რომ შეცვალოთ მომხმარებლის ჩანაწერების დიდი რაოდენობა ერთ სხდომაზე, განიხილეთ იმპორტის ინსტრუმენტის გამოყენება.
  1. ადმინისტრაციულ პანელზე დააწკაპუნეთ მომხმარებლის მენიუს ჩანართზე.
  2. იპოვნეთ მომხმარებელი და დააჭირეთ რედაქტირებას.
  3. შეიტანეთ ცვლილებები და დააჭირეთ შენახვას.
