როგორ შევქმნათ ან შეცვალოთ მომხმარებლები იმპორტის ხელსაწყოთი

ეს სტატია აჩვენებს ადმინისტრატორებს, თუ როგორ შექმნან მომხმარებლები იმპორტის ხელსაწყოთი.

იმპორტის ხელსაწყოს გახსნა

ადმინისტრატორებს შეუძლიათ გამოიყენონ იმპორტის ინსტრუმენტი ახალი მომხმარებლების ნაყარი ატვირთვისა და არსებული მომხმარებლის ჩანაწერების შესაცვლელად.

  1. შედით ადმინისტრატორის ანგარიშზე.
  2. იპოვნეთ აპლიკაცია აპების გვერდზე და დააჭირეთ ღილაკს ADMIN.
  3. ადმინისტრაციულ პანელზე დააწკაპუნეთ ჩანართზე ინტეგრაცია და შემდეგ იმპორტი ქვემენიუში.
  4. დააჭირეთ ღილაკს NEW. იმპორტის დამატების დიალოგური ფანჯარა გამოჩნდება ბრაუზერის ახალ ჩანართში.

ჩანაწერების შექმნა

იმპორტის დამატების დიალოგური ფანჯრის გახსნის შემდეგ:

  1. დააწკაპუნეთ ველზე Type და აირჩიეთ ვარიანტი (იხ. ქვემოთ). გამოჩნდება ბმული.
  2. დააჭირეთ ჩამოტვირთვა Sample ბმულს. CSV ფაილი ჩამოიტვირთება თქვენს მოწყობილობაში.

შემდეგი, გახსენით ფაილი და დაამატეთ ჩანაწერები. დააწკაპუნეთ აქ , რომ გაიგოთ, როგორ გამოიყენოთ ფორმატის მონაცემები CSV ფაილში.

ტიპი

როდესაც დააწკაპუნებთ Type ველზე, გამოჩნდება ჩამოსაშლელი მენიუ. აირჩიეთ ორი ვარიანტიდან ერთი ნაყარი იმპორტისთვის.

:exclamation: შენიშვნა ჩამოსაშლელ მენიუს აქვს მრავალი ვარიანტი, მაგრამ მხოლოდ ორია დაკავშირებული იმპორტის ხელსაწყოთან. თითოეულ ვარიანტს აქვს სხვადასხვა ნიმუში CSV ფაილი.

მომხმარებლების დამატება

  • აირჩიეთ ეს ვარიანტი მომხმარებლის ჩანაწერების მარტივი ნაყარი ატვირთვისთვის.
  • დააწკაპუნეთ ჩამოტვირთვის ნიმუშის ბმულზე, რათა გახსნათ CSV ფაილი სვეტების სათაურებით მომხმარებლის დამატება იმპორტის ტიპისთვის.

მომხმარებლების შეცვლა

  • აირჩიეთ ეს ვარიანტი მიმდინარე მომხმარებლების მონაცემების გასაახლებლად.
  • დააწკაპუნეთ ბმულზე ჩამოტვირთვის ნიმუშის გასახსნელად CSV ფაილი სვეტების სათაურებით Modify users Import ტიპისთვის.

ფაილის ატვირთვა

შეგიძლიათ დაუყოვნებლივ ატვირთოთ ფაილი ან მოგვიანებით დაუბრუნდეთ იმპორტის დამატების დიალოგურ ფანჯარას და დაასრულოთ ეს ნაბიჯები.

  1. გახსენით იმპორტის დამატების დიალოგური ფანჯარა.
  2. შეიყვანეთ უნიკალური სახელი CSV ფაილისთვის.
  3. დააწკაპუნეთ პლუს (+) ხატულაზე, რომ ატვირთოთ ფაილი თქვენი მოწყობილობიდან.
  4. დააწკაპუნეთ ველზე Type და აირჩიეთ ვარიანტი. (იხილეთ ზემოთ)
  5. დააწკაპუნეთ შენახვა.

დამუშავება

ფაილში ჩანაწერების რაოდენობისა და სისტემის დატვირთვის მიხედვით, პლატფორმას შეიძლება რამდენიმე წუთი დასჭირდეს იმპორტის დამუშავებას.

img

როდესაც იმპორტი დასრულდება, პლატფორმა არეგულირებს მომხმარებლის ჩანაწერებს (ანუ ამატებს, ცვლის ან გამორთავს) ადმინისტრატორის მომხმარებლების დაფაზე.

წარუმატებლობა

თუ იმპორტის ოპერაცია ვერ მოხერხდა, პლატფორმა უბრუნებს შეტყობინებას წარუმატებელი. უმეტეს შემთხვევაში, მარცხის მიზეზი არის მონაცემთა ფორმატირება.

img

წარუმატებლობის მიზეზის სანახავად:

  1. დააწკაპუნეთ ხედვა მოქმედების სათაურის ქვეშ.
  2. შემდეგ გვერდზე გადაიტანეთ მაუსი წარუმატებელი ხატულაზე. ინსტრუმენტის მინიშნება აჩვენებს შეცდომას.
  3. გადადით CVS ფაილში და შეცვალეთ და გამოასწორეთ შეცდომა.
  4. ისევ ატვირთეთ ფაილი.
