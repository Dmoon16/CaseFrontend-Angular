ადმინისტრატორები სწავლობენ, როგორ დააკონფიგურირონ სისტემის პარამეტრები ისე, რომ ახალი მომხმარებლები ავტომატურად მიენიჭონ ახლად შექმნილ ქეისს.

დაყენების გვერდის გახსნა

პირველი ნაბიჯი არის Systems Setup გვერდის გახსნა.

  1. შედით ადმინისტრატორის ანგარიშზე.
  2. იპოვნეთ აპლიკაცია აპების გვერდზე და დააჭირეთ ღილაკს ADMIN.
  3. დააწკაპუნეთ ჩანართზე Systems და შემდეგ SYSTEM SETUP ქვემენიუში.
  4. გადადით ქვემოთ USER SIGNUP-ის სათაურზე.

სისტემის პარამეტრების კონფიგურაცია

  1. დააყენეთ Allow Signup და Auto Create Case ჩამოსაშლელი მენიუები ჩართვაზე.
  2. დააწკაპუნეთ როლების მენიუზე და აირჩიეთ ვარიანტი (ნაგულისხმევი რეკომენდირებულია).
  3. დააკოპირეთ და გააზიარეთ მომხმარებლის საჯარო რეგისტრაციის ბმული.
