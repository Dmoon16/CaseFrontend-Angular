როგორ დავანიშნოთ მომხმარებლები საქმის შექმნისას

ეს სტატია აჩვენებს ადმინისტრატორებს, თუ როგორ დაავალონ მომხმარებელი ახალი ქეისის შექმნისას. თუ მომხმარებელი უკვე არ არის დარეგისტრირებული პლატფორმაზე, შეგიძლიათ შექმნათ ახალი მომხმარებლის დამატება ახალი ქეისის შექმნისას.

საქმის შექმნა

  1. შედით ადმინისტრატორის ანგარიშზე.
  2. იპოვნეთ აპლიკაცია აპების გვერდზე და დააჭირეთ ღილაკს ADMIN.
  3. დააწკაპუნეთ საქმეების მენიუს ჩანართზე.
  4. დააჭირეთ ღილაკს NEW. გამოჩნდება ახალი ფანჯარა.

მომხმარებლის დანიშვნა

Მიმდინარე მომხმარებელი

მიჰყევით ამ ნაბიჯებს ამჟამინდელი მომხმარებლის ახალ საქმეზე დასანიშნად.

დააწკაპუნეთ ღილაკზე ASSIGN USERS და შეავსეთ ველები.

  1. დააწკაპუნეთ მომხმარებლის ჩამოსაშლელ მენიუზე და აირჩიეთ იდენტიფიკატორი (მაგ. ელფოსტა).
  2. შეიყვანეთ მნიშვნელობა ცარიელ ველში (მაგ. ელექტრონული ფოსტის მისამართი).
  3. თუ მომხმარებლის ჩანაწერი არსებობს პლატფორმაზე, ველი ავტომატურად გთავაზობთ სახელს.
  4. აირჩიეთ სწორი მომხმარებელი.
  5. აირჩიეთ მომხმარებლის როლი (კლიენტები, ნაგულისხმევი ან პერსონალი).
  6. სურვილისამებრ, მონიშნეთ ყუთი მომხმარებლის ელ.ფოსტის შეტყობინების გაგზავნისთვის.
  7. დააწკაპუნეთ შენახვა.

Case Active ამატებს მომხმარებელს საქმეს. ახლა ადმინისტრატორებს შეუძლიათ დააწკაპუნონ შენახვაზე ფანჯრის დასახურად, ან ახალი მომხმარებლის დასამატებლად, ან მომხმარებლის წაშლა.

ახალი მომხმარებლის დამატება

თუ პლატფორმა ვერ პოულობს მიმდინარე ჩანაწერს, დააჭირეთ ღილაკს ახალი მომხმარებლის დასამატებლად.

შეავსეთ ველები და შემდეგ დააწკაპუნეთ SAVE-ზე მომხმარებლის პლატფორმაზე დასამატებლად. შემდეგი, დააყენეთ მომხმარებლის როლი და კვლავ დააწკაპუნეთ SAVE-ზე, რათა მომხმარებელი მიაკუთვნოს საქმეს. ბოლოს დააწკაპუნეთ შენახვაზე კიდევ ერთხელ საქმის შესანახად.
