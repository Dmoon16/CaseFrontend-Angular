როგორ შევქმნათ აპლიკაცია

ეს სტატია აჩვენებს ახალ მომხმარებლებს როგორ შექმნან უფასო საცდელი ანგარიში. Case Active ავტომატურად ანიჭებს ახალ მომხმარებელს ადმინისტრატორის პრივილეგიებს ამ ანგარიშისთვის.

ანგარიშის შექმნა

  1. გადადით შესვლის გვერდზე .
  2. დააჭირეთ ღილაკს დაწყება.
  3. შეიყვანეთ ელ.ფოსტის მისამართი.
  4. დააწკაპუნეთ შემდეგი.

ანგარიშის გადამოწმება

  1. გახსენით ბრაუზერის ახალი ჩანართი და შედით თქვენს ელ.ფოსტის ანგარიშზე.
  2. დააკოპირეთ ექვსნიშნა კოდი.
  3. დაუბრუნდით Case Active ჩანართს.
  4. ჩასვით კოდი ველებში. გამოჩნდება ახალი ფანჯარა.

ახალ ფანჯარაში:

  1. შეავსეთ საჭირო ველები.
  2. წაიკითხეთ მომსახურების პირობები და მონიშნეთ ველი დაეთანხმოთ.
  3. დააწკაპუნეთ შექმენით უფასო ანგარიში.

აპლიკაციის დაყენების დასრულება

სამუშაო ნაკადი გამოჩნდება შემდეგ გვერდზე. დაასრულეთ ნაბიჯები თქვენი აპის დაყენების დასასრულებლად. თქვენ ყოველთვის შეგიძლიათ დაბრუნდეთ ამ გვერდებზე ინფორმაციის ან თქვენი ანგარიშის სტატუსის რედაქტირებისთვის ან განახლებისთვის.

აპლიკაციის შესახებ გვერდზე, შეავსეთ კომპანიის სახელი და ვებსაიტის ველები (აუცილებელია) და შემდეგ დააწკაპუნეთ შემდეგი.

  1. აპლიკაციის სახელი
  2. თქვენი ტიპი (მომხმარებლის როლი)
  3. სტატუსი
