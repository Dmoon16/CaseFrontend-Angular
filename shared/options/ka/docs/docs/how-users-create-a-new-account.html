როგორ ქმნიან მომხმარებლები ახალ ანგარიშს

მას შემდეგ, რაც ადმინი დაამატებს მომხმარებელს საქმეს: * თუ მომხმარებელი ახალია ქეისზე აქტიური, მომხმარებელმა უნდა დაასრულოს სამუშაო პროცესი ანგარიშის შესაქმნელად და მიიღოს მოწვევა საქმეზე წვდომისთვის. * თუ მომხმარებელს აქვს არსებული ქეისის აქტიური ანგარიში, მომხმარებელმა უნდა მიიღოს მოწვევა საქმეზე წვდომისთვის.

ეს სტატია აღწერს ნაბიჯებს, რომლებიც მომხმარებელმა უნდა შეასრულოს თავისი ანგარიშის შესაქმნელად.

Ვიწყებთ

არსებობს ორი გზა, რომლითაც მომხმარებელი შეიძლება დაემატოს caseactive აპლიკაციას

  • ადმინისტრატორის მიერ: თუ ადმინისტრატორი ქმნის მომხმარებელს პლატფორმის ინტერფეისის ან იმპორტის ხელსაწყოს მეშვეობით, ადმინისტრატორმა უნდა გაუგზავნოს მომხმარებელს ბმული account.caseactive.com-ზე, რათა გააგრძელოს შესვლა.
  • მომხმარებლის მიერ: თუ მომხმარებელმა შექმნა ანგარიში რეგისტრაციის ბმულის გამოყენებით, მომხმარებელი ავტომატურად გადამისამართდება account.caseactive.com-ზე შესვლის გასაგრძელებლად.

Შესვლა

როდესაც მომხმარებელი პირველად შედის თავის CaseActive ანგარიშში, პლატფორმა მოუწოდებს მომხმარებელს:

  • გახსენით მათი ელ.ფოსტის ანგარიში და შემდეგ დააკოპირეთ/ჩასვით უსაფრთხოების ექვსნიშნა კოდი
  • შეავსეთ ანგარიშის პროფილი

მას შემდეგ, რაც მომხმარებელი დააწკაპუნებს UPDATE-ზე, გამოჩნდება აპების გვერდი. გვერდზე ნაჩვენებია აპლიკაციის სახელი (კომპანიის სახელი), მომხმარებლის ტიპი (როლი), სტატუსი და INVITATION ღილაკი.
