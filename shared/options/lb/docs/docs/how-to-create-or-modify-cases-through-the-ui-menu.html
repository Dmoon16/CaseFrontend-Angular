Wéi erstellen oder änneren Fäll duerch den UI Menu

Dësen Artikel weist Administrateuren wéi een e Fall mat der Plattform User Interface erstellt. Administrateuren kënnen och Benotzer op en neie Fall addéieren an existéierend Fäll aktualiséieren.

Schafen engem Fall

  1. Loggt Iech op den Admin Kont.
  2. Fannt eng App op der Apps Säit a klickt op den ADMIN Knäppchen.
  3. Op der Admin Panel, klickt op de Fall Menü Tab.
  4. Klickt op den NEW Knäppchen. Eng nei Fënster weist.

img

Fëllt déi erfuerderlech Felder aus a klickt SAVE.

Benotzer dobäizemaachen

Optional kënnen d'Administrateuren d'Benotzer an de Fall op dëser Säit addéieren. Scroll down a klickt op ASSIGN USERS Knäppchen.

img

Am Assign User Modal fëllt d'Felder aus. Kuckt d'Këscht fir dem Benotzer eng E-Mail Notifikatioun ze schécken. (optional)

Ännere vun engem Fall

Fir eng kleng Unzuel vu Fäll ze aktualiséieren, benotzt d'Edit Tool a maacht Ännerunge fir all eenzel Fall.

  • Notiz Fir eng grouss Zuel vu Fäll an enger Sëtzung ze aktualiséieren, betruecht d' Importinstrument ze benotzen
  1. Loggt Iech op den Admin Kont.
  2. Fannt eng App op der Apps Säit a klickt op den ADMIN Knäppchen.
  3. Op der Admin Panel, klickt op de Fall Menü Tab.
  4. Situéiert e Fall a klickt Edit.
  5. Maacht Ännerungen a klickt SAVE.
