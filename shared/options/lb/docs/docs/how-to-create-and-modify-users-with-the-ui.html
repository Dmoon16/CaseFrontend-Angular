Wéi erstellen an änneren Benotzer mat der UI

Dësen Artikel weist Administrateuren wéi d'Benotzer erstellen an d'Records aktualiséieren mat der User-Interface vun der Plattform. Administrateuren kënnen och Benotzer op en neie Fall addéieren an existent Fäll änneren.

E Benotzer erstellen

  1. Loggt Iech op den Admin Kont.
  2. Fannt eng App op der Apps Säit a klickt op den ADMIN Knäppchen.
  3. Op der Admin Panel klickt op d'Benotzermenü Tab.
  4. Klickt op den NEW Knäppchen. Eng nei Fënster weist.

Fëllt d'Felder aus a klickt SAVE. Case Active füügt den Numm un d'Lëscht vun de Benotzer, déi mat dëser App verbonne sinn.

img

Änneren vun engem Benotzer

Fir eng kleng Unzuel vu Benotzerrecords ze aktualiséieren, benotzt den Edit-Tool a maacht Ännerunge fir all eenzelne Benotzer.

  • Notiz Fir eng grouss Zuel vu Benotzerrecords an enger Sëtzung z'änneren, betruecht d' Importinstrument ze benotzen .
  1. Op der Admin Panel klickt op d'Benotzermenü Tab.
  2. Situéiert e Benotzer a klickt Edit.
  3. Maacht Ännerungen a klickt SAVE.

img

Nächst Schrëtt

Case Active schéckt dem Benotzer keng E-Mail Notifikatioun. Administrateuren sollen de Benotzer kontaktéieren an hinnen e Link op d'Case Active Homepage schécken. De Benotzer klickt dann op de LOGIN Knäppchen a fäerdeg de Workflow fir nei Umeldungen.
