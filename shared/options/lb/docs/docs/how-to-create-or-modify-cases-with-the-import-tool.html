Wéi erstellen oder ännere Fäll mat dem Import Tool

Dësen Artikel weist Administrateuren wéi een existéierend Fäll erstellt oder ännert mam Import Tool.

Öffnen vum Import Tool

  1. Loggt Iech op den Admin Kont.
  2. Fannt eng App op der Apps Säit a klickt op den ADMIN Knäppchen.
  3. Op der Admin Panel, klickt op d'Integratioun Tab an dann IMPORTS am Ënnermenü.
  4. Klickt op den NEW Knäppchen. D'Füügt Importer Dialogbox weist an engem neie Browser Tab.

img

Schafen Fäll

Nodeems d'Füügt Importer Dialogbox opmaacht:

  1. Klickt op d'Typ Feld a wielt eng Optioun (kuckt hei ënnen). E Link weist.
  2. Klickt op den Download Sample Link. Eng CSV Datei gëtt op Ärem Apparat erofgelueden.

Als nächst, öffnen d'Datei a füügt Fäll un.

Typ

Wann Dir op d' Typ Feld klickt, gëtt en Dropdown-Menü ugewisen. Wielt eng vun zwou Optiounen fir bulk Fall Importer.

  • Notiz Den Dropdown-Menü huet vill Optiounen, awer nëmmen zwee si mat der Gestioun vum Fall verbonnen. All Optioun huet eng aner Probe CSV Datei.

Dobäizemaachen Fäll

  • Wielt dës Optioun fir en einfache Bulk-Upload vu Fäll.
  • Klickt op den Download Sample Link fir eng CSV-Datei opzemaachen mat Spaltrubriken fir den Erstelle Fäll Import Typ.

Änneren Fäll

  • Wielt dës Optioun fir aktuell Fäll ze aktualiséieren.
  • Klickt op den Download Sample Link fir eng CSV Datei opzemaachen mat Kolonnrubriken fir den Änneren Fäll Import Typ.

Eroplueden vun der Datei

Dir kënnt e Fichier direkt eroplueden oder zréck an d'Dialogbox Importer addéieren zu engem spéideren Datum an dës Schrëtt ausfëllen.

  1. Öffnen d'Füügt Importer Dialogbox.
  2. Gitt en eenzegaartegen Numm fir d'CSV-Datei un.
  3. Klickt op de Plus (+) Ikon fir d'Datei vun Ärem Apparat eropzelueden.
  4. Klickt op d'Typ Feld a wielt eng Optioun. (kuckt uewen)
  5. Klickt SAVE.

img

Formatéieren

Dëst sinn déi erfuerderlech Felder an déi éischt Ufuerderungen. Klickt hei fir d'Formatéierungsdetailer ze iwwerpréiwen.

tag_id (oder case_ID) en eenzegaartegen Identifizéierer (benotzt alphanumeresch Zeechen, Bindestrecken ginn ugeholl)

case_status 0 = zougemaach 1 = aktiv

date_opened month/day/year mm/dd/yyyy

iwwer eng eenzegaarteg Beschreiwung vum Fall (benotzt alphanumeresch Zeechen, Bindestrecken ginn ugeholl)

img

Veraarbechtung

Ofhängeg vun der Unzuel vun den Opzeechnungen an der Datei an der Systembelaaschtung, kann d'Plattform e puer Minutten daueren fir den Import ze veraarbecht.

img

Wann den Import fäerdeg ass, passt d'Plattform Case records un (dh addt oder ännert) op dem Administrateur Cases Dashboard.

Echec

Wann d'Importoperatioun feelt, gëtt d'Plattform e Failed Message zréck. An deene meeschte Fäll ass d'Ursaach vum Feeler d'Dateformatéierung.

img

Fir d'Ursaach vum Echec ze gesinn:

  1. Klickt View ënner der ACTION Rubrik.
  2. Op der nächster Säit, hover d'Maus op der Failed Ikon. Den Tooltip weist de Feeler.
  3. Gitt op d'CVS-Datei a passt de Feeler un a fixéiert.
  4. Eroplueden de Fichier erëm.
