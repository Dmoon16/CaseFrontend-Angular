Hoe kinne jo in brûkersprofyl bywurkje

Dit artikel lit brûkers sjen hoe't se ynformaasje op har profylside bywurkje.

  • Brûkers kinne har eigen profilen bywurkje.
  • Behearders kinne de profylside fan elke brûker bywurkje.

Foar brûkers

De earste stap is om de profyl side te iepenjen. Brûkers en admins folgje dizze stappen om har eigen profylside te aktualisearjen.

  1. Oanmelde by jo Case Active akkount.
  2. Klikje op it ljepblêd Profile boppe oan 'e side.
  3. Meitsje feroarings.
  4. Klik op Bewarje.

De measte fjilden binne opsjoneel, en guon fjilden binne ferplicht. De fereaske fjilden binne:

  • Foarnamme
  • Achternamme
  • E-postadres
  • Taal
  • Tiidsône

Foar admins

Admins kinne ek de profylynformaasje foar oare brûkers bywurkje. Dit omfettet trije fjilden ûnder de koptekst HOST USER INFO.

  1. Oanmelde by jo Case Active admin akkount.
  2. Sykje in app en klikje op de ADMIN knop.
  3. Klikje op Brûkers yn it boppeste menu.
  4. Gean nei it submenu en selektearje de brûkerslist wêr't jo tagong ta wolle (AKTYF of útskeakele). De standertynstelling is Aktyf.
  5. Sykje in brûker op 'e list en klikje op Bewurkje.
  6. Meitsje wizigingen as nedich. (Sjoch hjirûnder)
  7. Klik op Bewarje.

Feroarje brûkersynfo

Admins kinne brûkersynformaasje feroarje.

  1. Stel it brûkerstype yn (brûker of behearder).
  2. Stel de brûkersstatus yn (aktyf nei útskeakele).
  3. Fier in unike tag-ID yn. (fakultatyf)
  4. Klik op Bewarje.
