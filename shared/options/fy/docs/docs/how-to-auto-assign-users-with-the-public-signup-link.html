Hoe kinne jo brûkers automatysk tawize mei de publike oanmeldingslink

Behearders leare hoe't se systeemynstellingen konfigurearje sadat nije brûkers automatysk wurde tawiisd oan in nij automatysk oanmakke saak.

It iepenjen fan de opset side

De earste stap is om de side Systems Setup te iepenjen.

  1. Oanmelde by it admin akkount.
  2. Sykje in app op 'e Apps-side en klikje op de ADMIN-knop.
  3. Klikje op it ljepblêd Systemen en dan SYSTEEM SETUP yn it submenu.
  4. Rôlje omleech nei de koptekst USER SIGNUP.

Systeemynstellingen konfigurearje

  1. Stel de drop-down menu's Tastean oanmelding en saak automatysk oanmeitsje op Oan.
  2. Klikje op it rolmenu en kies in opsje (standert wurdt oanrikkemandearre).
  3. Kopiearje en diel de keppeling foar oanmelding foar iepenbiere brûkers.

It ferstjoeren fan de keppeling

Elke brûker dy't op de keppeling klikt en de oanmelding foltôget, triggert it platfoarm om in nije saak te generearjen op 'e Cases-side en jout de brûker yn dat gefal automatysk ta.
