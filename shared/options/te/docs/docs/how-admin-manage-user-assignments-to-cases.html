వినియోగదారులు మరియు కేస్ అసైన్‌మెంట్‌లను నిర్వహించడానికి ఐదు మార్గాలు ఉన్నాయి. ఎలాగో తెలుసుకోవడానికి లింక్‌ను క్లిక్ చేయండి.

పద్ధతి టైప్ చేయండి చర్యలు వివరాలు
ఒక కేసును సృష్టిస్తున్నప్పుడు UI జోడించు అడ్మిన్ బటన్ > కేసులు > కొత్త బటన్ > వినియోగదారులను అప్పగించు బటన్
పబ్లిక్ సైన్అప్ లింక్‌ని ఉపయోగించి వినియోగదారుని స్వయంచాలకంగా కేటాయించండి వినియోగదారు చర్య జోడించు అడ్మిన్ బటన్ > సెట్టింగ్‌లు > సిస్టమ్ సెటప్ పేజీ
ఇప్పటికే ఉన్న కేసులో UIని ఉపయోగించడం UI జోడించు & తీసివేయు అడ్మిన్ బటన్ > కేసులు > వినియోగదారులను కేటాయించండి
