కేసును సృష్టించేటప్పుడు వినియోగదారులను ఎలా కేటాయించాలి

కొత్త కేసును సృష్టించేటప్పుడు వినియోగదారుని ఎలా కేటాయించాలో ఈ కథనం నిర్వాహకులకు చూపుతుంది. వినియోగదారు ప్లాట్‌ఫారమ్‌లో ఇప్పటికే నమోదు కానట్లయితే, మీరు కొత్త కేసును సృష్టించేటప్పుడు కొత్త వినియోగదారుని జోడించడాన్ని సృష్టించవచ్చు.

ఒక కేసును సృష్టిస్తోంది

  1. నిర్వాహక ఖాతాకు లాగిన్ చేయండి.
  2. యాప్‌ల పేజీలో యాప్‌ను గుర్తించి, అడ్మిన్ బటన్‌ను క్లిక్ చేయండి.
  3. కేసుల మెను ట్యాబ్‌ని క్లిక్ చేయండి.
  4. కొత్త బటన్‌ను క్లిక్ చేయండి. కొత్త విండో కనిపిస్తుంది.
  • కేసును సృష్టించడానికి ఫీల్డ్‌లను పూరించండి, ఆపై అసైన్డ్ యూజర్స్ హెడ్డింగ్‌కి క్రిందికి స్క్రోల్ చేయండి.

వినియోగదారుని కేటాయించడం

ప్రస్తుత వినియోగదారుడు

కొత్త కేసుకు ప్రస్తుత వినియోగదారుని కేటాయించడానికి ఈ దశలను అనుసరించండి.

వినియోగదారులను కేటాయించు బటన్‌ను క్లిక్ చేసి, ఫీల్డ్‌లను పూరించండి.

  1. వినియోగదారు డ్రాప్-డౌన్ క్లిక్ చేసి, ఐడెంటిఫైయర్‌ను ఎంచుకోండి (ఉదా. ఇమెయిల్).
  2. ఖాళీ ఫీల్డ్‌లో విలువను నమోదు చేయండి (ఉదా. ఇమెయిల్ చిరునామా).
  3. ప్లాట్‌ఫారమ్‌లో వినియోగదారు రికార్డ్ ఉన్నట్లయితే, ఫీల్డ్ స్వయంచాలకంగా పేరును సూచిస్తుంది.
  4. సరైన వినియోగదారుని ఎంచుకోండి.
  5. వినియోగదారు పాత్రను ఎంచుకోండి (క్లయింట్లు, డిఫాల్ట్ లేదా సిబ్బంది).
  6. ఐచ్ఛికంగా, వినియోగదారుకు ఇమెయిల్ నోటిఫికేషన్ పంపడానికి పెట్టెను ఎంచుకోండి.
  7. సేవ్ చేయి క్లిక్ చేయండి.

కేస్ యాక్టివ్ వినియోగదారుని కేసుకు జోడిస్తుంది. ఇప్పుడు నిర్వాహకులు విండోను మూసివేయడానికి సేవ్ చేయి క్లిక్ చేయవచ్చు లేదా కొత్త వినియోగదారుని జోడించవచ్చు లేదా వినియోగదారుని తీసివేయవచ్చు.

కొత్త వినియోగదారుని జోడిస్తోంది

ప్లాట్‌ఫారమ్ ప్రస్తుత రికార్డ్‌ను కనుగొనలేకపోతే, కొత్త వినియోగదారుని జోడించడానికి బటన్‌ను క్లిక్ చేయండి.
