పబ్లిక్ సైన్అప్ లింక్‌తో వినియోగదారులను స్వయంచాలకంగా ఎలా కేటాయించాలి

సిస్టమ్ సెట్టింగ్‌లను ఎలా కాన్ఫిగర్ చేయాలో నిర్వాహకులు నేర్చుకుంటారు, తద్వారా కొత్త వినియోగదారులు స్వయంచాలకంగా కొత్తగా సృష్టించబడిన కేసుకు కేటాయించబడతారు.

సెటప్ పేజీని తెరుస్తోంది

సిస్టమ్స్ సెటప్ పేజీని తెరవడం మొదటి దశ.

  1. నిర్వాహక ఖాతాకు లాగిన్ చేయండి.
  2. యాప్‌ల పేజీలో యాప్‌ను గుర్తించి, అడ్మిన్ బటన్‌ను క్లిక్ చేయండి.
  3. సిస్టమ్‌ల ట్యాబ్‌పై క్లిక్ చేసి, ఆపై ఉపమెనులో SYSTEM SETUPని క్లిక్ చేయండి.
  4. USER SIGNUP శీర్షికకు క్రిందికి స్క్రోల్ చేయండి.

సిస్టమ్ సెట్టింగ్‌లను కాన్ఫిగర్ చేస్తోంది

  1. అనుమతించు సైన్అప్ మరియు ఆటో క్రియేట్ కేస్ డ్రాప్-డౌన్ మెనులను ఆన్‌కి సెట్ చేయండి.
  2. పాత్ర మెనుని క్లిక్ చేసి, ఒక ఎంపికను ఎంచుకోండి (డిఫాల్ట్ సిఫార్సు చేయబడింది).
  3. పబ్లిక్ యూజర్ సైన్అప్ లింక్‌ని కాపీ చేసి షేర్ చేయండి.

లింక్ పంపుతోంది

లింక్‌ను క్లిక్ చేసి, సైన్అప్‌ను పూర్తి చేసిన ప్రతి వినియోగదారు ప్లాట్‌ఫారమ్‌ను కేసుల పేజీలో కొత్త కేసును రూపొందించడానికి ట్రిగ్గర్ చేస్తుంది మరియు ఆ సందర్భంలో వినియోగదారుని స్వయంచాలకంగా కేటాయిస్తుంది.
