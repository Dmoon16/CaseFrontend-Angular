Қалай кіруге болады

Қолданыстағы Case Active тіркелгіңізге қалай кіру керектігін біліңіз.

Жүйеге кіру

  1. Жүйеге кіру бетіне өтіңіз.
  2. КІРУ түймесін басыңыз.
  3. Электрондық поштаңызды енгізіңіз.
  4. КЕЛЕСІ түймесін басыңыз.

Есептік жазбаны растау

  1. Жаңа шолғыш қойындысын ашып, электрондық пошта тіркелгіңізге кіріңіз.
  2. Алты таңбалы кодты көшіріңіз.
  3. Іс белсенді қойындысына оралыңыз.
  4. Кодты өрістерге қойыңыз. Жаңа терезе көрсетіледі.

Case Active қолданбалар бетін көрсетеді.

  • Ескертпе Платформа әкімші артықшылықтары бар пайдаланушылар үшін APP түймешігінің жанындағы ADMIN түймесін көрсетеді.
