Тіркелу сілтемесі бар пайдаланушыларды қалай жасауға болады

Бұл мақала әкімшілерге тіркелу сілтемесін жіберу арқылы пайдаланушыларды қалай жасау керектігін көрсетеді.

Тіркелу сілтемесі конфигурациялануда

  1. Пайдаланушы қосу тілқатысу терезесінің төменгі жағындағы Оларға тіркелуді жіберу сілтемесін басыңыз. Жаңа бет көрсетіледі.
  2. Тіркелу опцияларын конфигурациялаңыз. (төменде қараңыз)

img

Тіркелуге рұқсат ету өрісінің Қосулы күйіне орнатылғанын тексеріңіз. (міндетті) Қосылған кезде Case Active бірегей сілтеме жасайды.

Төмендегі опцияларды қарап шығыңыз, содан кейін сілтемені көшіру үшін белгішені басыңыз. Сілтемені веб-сайтыңызға орналастыруға немесе сілтемені электрондық пошта немесе кез келген байланыс арнасы арқылы бір немесе бірнеше адамға жіберуге болады.

Іс құру

Қосымша түрде әрбір жаңа пайдаланушы үшін іс жасап, пайдаланушы рөлін тағайындай аласыз.

  1. Істі автоматты түрде жасау өрісін басып, Қосу опциясын таңдаңыз. Case User Role өрістері көрсетіледі.
  2. Case User Role түймесін басып, опцияны таңдаңыз. (төменде қараңыз)

Сілтеме жіберу

Сілтемені бір немесе бірнеше адаммен бөлісіңіз. Әрбір адам сілтемені басып, Case Active тіркелгісін белсендіру үшін жұмыс процесін аяқтайды. Пайдаланушы тіркелу жұмыс процесін аяқтағаннан кейін, Case Active пайдаланушыны Пайдаланушылар тізіміне қосады.

img

Келесі қадамдар

Case Active пайдаланушыға тіркелу сілтемесі бар электрондық пошта хабарландыруын жібереді. Пайдаланушы сілтемені басып, жаңа тіркелулер үшін жұмыс процесін аяқтайды.

  • Бұл мақалада жаңа пайдаланушылар үшін тіркелу жұмыс процесі сипатталады.
