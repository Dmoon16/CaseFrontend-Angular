Қолданыстағы жағдайлармен пайдаланушы тағайындауларын қалай тағайындауға немесе жоюға болады

Бұл мақала әкімшілерге іс пайдаланушыларын тағайындау және жою жолын көрсетеді. Сондай-ақ пайдаланушы деректерін өзгерту жолын көрсетеміз.

Бар істі ашу

  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Істер мәзірі қойындысын басыңыз.
  4. Ішкі мәзірде ACTIVE таңдалғанын тексеріңіз.
  5. ӘРЕКЕТ бағанына төмен жылжыңыз.

Тағайындау немесе жою

Пайдаланушыны іске тағайындау немесе пайдаланушыны корпустан (бірақ платформадан емес) жою үшін Пайдаланушыларды тағайындау түймесін басыңыз. Келесі бетте: ҚОСУ түймесін басып, жұмыс процесін аяқтаңыз. Пайдаланушыны жою үшін жою түймесін басыңыз.

img

Өзгерту

Істер бетінде пайдаланушы деректерін өңдеу немесе жаңарту мүмкін емес. Оның орнына мына қадамдарды орындаңыз:

  1. Пайдаланушылар мәзірі қойындысын басыңыз.
  2. Пайдаланушы атын тауып, Өңдеу түймесін басыңыз.
  3. Өзгерістерді енгізіп, САҚТАУ түймесін басыңыз.
