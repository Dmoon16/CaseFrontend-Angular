Импорттау құралымен Істі басқару деректерін пішімдеу

Төменде CSV файлдарындағы істерді Импорттау құралымен басқару үшін қажетті өрістердің тізімі берілген.

пайдаланушы аты

  • Пайдаланушының электрондық пошта мекенжайын енгізіңіз.

tag_id (идентификаторға кейінірек сілтеме жасай аласыз)

  • бірегей идентификатор (әріптік-цифрлық таңбаларды пайдаланыңыз, сызықшалар қабылданады)

case_status

  • 0 = жабық
  • 1 = белсенді

күні_ашылған

  • ай/күн/жыл
  • мм/кк/жжж

туралы

  • істің бірегей сипаттамасы (әріптік-цифрлық таңбаларды пайдаланыңыз, сызықшалар қабылданады)
