Импорт құралымен пайдаланушы тағайындауларын қалай тағайындауға, өзгертуге немесе жоюға болады

Бұл мақала әкімшілерге Импорттау құралы арқылы іс пайдаланушыларын басқару жолын көрсетеді.

Әкімшілер:

  • пайдаланушыларды кейске тағайындаңыз
  • іске тағайындалған пайдаланушыларды өзгерту
  • іске тағайындалған пайдаланушыларды жою

Импорттау құралын ашу

  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Интеграция қойындысын, содан кейін ішкі мәзірде ИМПОРТТАР түймесін басыңыз.
  4. ЖАҢА түймесін басыңыз. Импорттарды қосу тілқатысу терезесі көрсетіледі.

Жазбаларды құру

Бір немесе бірнеше CSV файлдарын жүктеп алыңыз. Іс пайдаланушыларын қосу, өзгерту немесе жою үшін осы файлдарды пайдаланыңыз. Импорттарды қосу тілқатысу терезесі ашылғаннан кейін:

  1. Түр өрісін басыңыз.
  2. Ашылмалы мәзірді айналдырып, опцияны таңдаңыз (төменде қараңыз). Сілтеме көрсетіледі.
  3. Үлгіні жүктеп алу сілтемесін басыңыз. CSV файлы құрылғыңызға жүктеледі.

img

Файлды ашып, жазбаларды қосыңыз. CSV файлында пішім деректерін пайдалану жолын білу үшін осы жерді басыңыз.

Файл опциялары

Пайдаланушыларды істерге тағайындау * Пайдаланушыларды іске қосу үшін осы опцияны таңдаңыз. * Case_role_order ұяшығын бос қалдырыңыз. * Пайдаланушылар бетіне өтіп, осы адамға арналған тег идентификаторын көшіріңіз. Тег идентификаторы өрісі бос болса, бірегей идентификаторды енгізіңіз, Сақтау түймесін басыңыз, содан кейін идентификаторды CSV файлына көшіріңіз.

img

Істерге тағайындалған пайдаланушыларды өзгерту * Іске тағайындалған ағымдағы пайдаланушылар үшін деректерді жаңарту үшін осы опцияны таңдаңыз. * Case_ID бағаны үшін Case Tag ID пайдаланыңыз.

Істерге тағайындалған пайдаланушыларды жою * Іске тағайындалған пайдаланушыларды жою үшін осы опцияны таңдаңыз.

Файлды жүктеп салу

Файлды бірден жүктеп салуыңызға немесе кейінірек импортты қосу тілқатысу терезесіне оралып, осы қадамдарды орындауға болады.

  1. Импорттарды қосу тілқатысу терезесін ашыңыз.
  2. CSV файлы үшін бірегей атау енгізіңіз.
  3. Файлды құрылғыдан жүктеп салу үшін плюс (+) белгішесін басыңыз.
  4. Түр өрісін басып, опцияны таңдаңыз. (жоғарыдан қараңыз)
  5. САҚТАУ түймесін басыңыз.

Өңдеу

Файлдағы жазбалар санына және жүйе жүктемесіне байланысты, платформаға импортты өңдеуге бірнеше минут кетуі мүмкін.

Импорттау аяқталған кезде платформа Істер бетінде пайдаланушы жазбаларын реттейді (яғни қосады, өзгертеді немесе жояды).

Сәтсіздік

Импорттау әрекеті сәтсіз болса, платформа сәтсіз хабарды қайтарады. Көп жағдайда сәтсіздіктің себебі деректерді пішімдеу болып табылады.

img

Ақаулықтың себебін көру үшін:

  1. Әрекет тақырыбының астындағы Көрініс түймесін басыңыз.
  2. Келесі бетте тінтуірді «Сәтсіз» белгішесіне апарыңыз. Құрал кеңесі қатені көрсетеді.
  3. CVS файлына өтіп, қатені реттеңіз және түзетіңіз.
  4. Файлды қайта жүктеп салыңыз.
