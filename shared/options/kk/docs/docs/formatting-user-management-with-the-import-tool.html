Импорт құралымен пайдаланушыны басқару деректерін пішімдеу

Төменде CSV файлдарындағы деректер өрістерінің тізімі берілген. Әрбір импорттау опциясында өрістердің әртүрлі тіркесімі бар.

пайдаланушы аты

  • Пайдаланушының электрондық пошта мекенжайын енгізіңіз.

есім

  • Пайдаланушының атын енгізіңіз.

тек

  • Пайдаланушының тегін енгізіңіз.

жергілікті тіл

  • Бұл баған пайдаланушының қалаған тілін тексереді.
  • Ағылшын үшін en, испан үшін sp енгізіңіз.

аймақ ақпараты

  • Бұл пайдаланушының уақыт белдеуін орнатады. Материк пен қаланы бос орынсыз енгізіңіз (астын сызуды пайдаланыңыз).
  • Профиль > ТІЛ және АЙМАҚ > Уақыт белдеуі ашылмалы мәзіріндегі опцияны пайдалану керек.
  • Ең қолайлы уақыт белдеуін пайдаланыңыз. Мысалы, Бостон ашылмалы тізімде жоқ, сондықтан біз Нью-Йоркті (немесе EST/EDT уақыт белдеуіндегі кез келген қаланы) пайдалана аламыз.

хост_тегі_идентификаторы

  • Тұрақты белгіні енгізіңіз.
  • az, 0-9, -, / рұқсат етіледі.

case_status

  • 0 = жабық
  • 1 = белсенді
