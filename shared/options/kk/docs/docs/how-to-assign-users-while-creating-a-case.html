Істі жасау кезінде пайдаланушыларды қалай тағайындауға болады

Бұл мақала әкімшілерге жаңа іс жасау кезінде пайдаланушыны қалай тағайындау керектігін көрсетеді. Егер пайдаланушы платформада әлі тіркелмеген болса, жаңа іс жасау кезінде жаңа пайдаланушыны қосуға болады.

Іс құру

  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Істер мәзірі қойындысын басыңыз.
  4. ЖАҢА түймесін басыңыз. Жаңа терезе көрсетіледі.

  • Іс жасау үшін өрістерді толтырыңыз, содан кейін «Тағайындалған пайдаланушылар» тақырыбына төмен жылжыңыз.

Пайдаланушыны тағайындау

Ағымдағы пайдаланушы

Ағымдағы пайдаланушыны жаңа іске тағайындау үшін мына қадамдарды орындаңыз.

ПАЙДАЛАНУШЫЛАРДЫ ТАҢДАУ түймесін басып, өрістерді толтырыңыз.

  1. Пайдаланушы ашылмалы тізімін басып, идентификаторды таңдаңыз (мысалы, электрондық пошта).
  2. Бос өріске мәнді енгізіңіз (мысалы, электрондық пошта мекенжайы).
  3. Егер пайдаланушы жазбасы платформада болса, өріс автоматты түрде атауды ұсынады.
  4. Дұрыс пайдаланушыны таңдаңыз.
  5. Пайдаланушы рөлін таңдаңыз (клиенттер, әдепкі немесе қызметкерлер).
  6. Қажет болса, пайдаланушыға электрондық пошта хабарландыруын жіберу үшін құсбелгіні қойыңыз.
  7. САҚТАУ түймесін басыңыз.

Case Active пайдаланушыны іске қосады. Енді әкімшілер терезені жабу немесе жаңа пайдаланушы қосу немесе пайдаланушыны жою үшін Сақтау түймесін баса алады.

Жаңа пайдаланушы қосу

Егер платформа ағымдағы жазбаны таба алмаса, жаңа пайдаланушыны қосу үшін түймені басыңыз.

Өрістерді толтырып, пайдаланушыны платформаға қосу үшін САҚТАУ түймесін басыңыз. Содан кейін пайдаланушы рөлін орнатыңыз және пайдаланушыны іске тағайындау үшін САҚТАУ түймесін қайта басыңыз. Соңында, істі сақтау үшін САҚТАУ түймесін тағы бір рет басыңыз.
