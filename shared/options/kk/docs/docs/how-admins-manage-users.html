Әкімшілер пайдаланушыларды қалай басқарады

Әкімшілер қолданбалар мен істерге кім кіре алатынын жасайды, өзгертеді немесе өшіреді. Пайдаланушы платформаға бірінші рет кірген кезде профильдің жұмыс процесін аяқтауы керек.

  • Жаңа пайдаланушы профилінің жұмыс процесі туралы қосымша ақпарат алу үшін осы жерді басыңыз.

Пайдаланушыны басқару

Төмендегі кесте пайдаланушыны басқарудың бес әдісін көрсетеді. Қалай білу үшін сілтемені басыңыз.

Әдіс Түр Әрекеттер Егжей
UI пайдалану UI қосу және өзгерту Әкімші түймесі > Пайдаланушылар
Істі жасау және пайдаланушыны кейске тағайындау кезінде UI қосу Әкімші түймесі > Істер > Жаңа түйме > Іс бетін жасау
Қоғамдық тіркелу сілтемесі арқылы пайдаланушыны автоматты түрде жасау Пайдаланушы әрекеті қосу Әкімші түймесі > Параметрлер > Жүйені орнату беті
Импорттау құралы UI қосу және өзгерту Әкімші түймесі > Интеграциялар > Импорттар
API пайдалану (жақында) API қосу және өзгерту Жоқ

Хабарландырулар

Платформа пайдаланушыларға қолданбаға қосылған кезде хабарландыру жібермейді. Дегенмен, әкімшілер іс тапсырмалары үшін хабарландыруларды қоса алады.
