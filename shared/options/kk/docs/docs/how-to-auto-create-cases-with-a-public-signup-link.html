Жалпы тіркелу сілтемесі бар істерді автоматты түрде қалай жасауға болады

Бұл мақала әкімшілерге бір тіркелу сілтемесін жіберу арқылы істі жасау (және бір уақытта пайдаланушыны қосу) жолын көрсетеді.

Сілтемені табу

  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Әкімші тақтасында Параметрлер мәзір қойындысын басыңыз.
  4. ЖҮЙені ОРНАТУ ішкі мәзір қойындысының таңдалғанын тексеріңіз.

Параметрлерді конфигурациялау

ПАЙДАЛАНУШЫНЫҢ ТІРКЕЛУ тақырыбына төмен жылжыңыз.

Тіркелуге рұқсат беру өрісі Қосулы күйіне орнатылғанын тексеріңіз. (міндетті) Бұл мүмкіндік қосылғанда, Case Active бірегей сілтеме жасайды.

  1. Істі автоматты түрде жасау өрісін басып, Қосу опциясын таңдаңыз. Case User Role өрісі көрсетіледі.
  2. Рөлді таңдаңыз. (төменде қараңыз)

Үш жағдай пайдаланушы рөлі бар.

  • клиенттер (ұсынылады)
  • әдепкі
  • персонал

Сілтеме жіберу

Сілтемені бір немесе бірнеше адаммен бөлісіңіз. Сілтемені басқан және тіркелуді аяқтаған әрбір пайдаланушы істер бетінде жаңа істі жасау және іске пайдаланушыны автоматты түрде тағайындау үшін платформаны іске қосады.
