UI мәзірі арқылы істерді қалай жасауға немесе өзгертуге болады

Бұл мақала әкімшілерге платформаның пайдаланушы интерфейсімен іс жасау жолын көрсетеді. Сондай-ақ әкімшілер пайдаланушыларды жаңа іске қосып, бар істерді жаңарта алады.

Іс құру

  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Әкімші тақтасында Істер мәзірі қойындысын басыңыз.
  4. ЖАҢА түймесін басыңыз. Жаңа терезе көрсетіледі.

Қажетті өрістерді толтырып, САҚТАУ түймесін басыңыз.

Пайдаланушыларды қосу

Қажет болса, әкімшілер осы беттегі іске пайдаланушыларды қоса алады. Төмен жылжып, ПАЙДАЛАНУШЫЛАРДЫ ТАҢДАУ түймесін басыңыз.

img

Қолданушыны тағайындау модальында өрістерді толтырыңыз. Пайдаланушыға электрондық пошта хабарландыруын жіберу үшін құсбелгіні қойыңыз. (міндетті емес)

img

Істі өзгерту

Істердің аз санын жаңарту үшін Өңдеу құралын пайдаланыңыз және әрбір жеке іске өзгертулер енгізіңіз.

  • Ескерту Бір отырыста істердің көп санын жаңарту үшін Импорттау құралын пайдалануды қарастырыңыз
  1. Әкімші тіркелгісіне кіріңіз.
  2. Қолданбалар бетінде қолданбаны тауып, ADMIN түймесін басыңыз.
  3. Әкімші тақтасында Істер мәзірі қойындысын басыңыз.
  4. Істі тауып, Өңдеу түймесін басыңыз.
  5. Өзгерістерді енгізіп, САҚТАУ түймесін басыңыз.
