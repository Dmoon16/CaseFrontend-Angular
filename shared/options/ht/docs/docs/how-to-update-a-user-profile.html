Ki jan yo mete ajou yon pwofil itilizatè

Atik sa a montre itilizatè yo ki jan yo mete ajou enfòmasyon sou paj Profile yo.

  • Itilizatè yo ka mete ajou pwòp pwofil yo.
  • Admin yo ka mete ajou paj Pwofil nenpòt itilizatè.

Pou itilizatè yo

Premye etap la se louvri paj Profile a. Itilizatè yo ak administratè yo swiv etap sa yo pou mete ajou pwòp paj Profil yo.

  1. Konekte nan kont Case Active ou a.
  2. Klike sou tab Profile ki anlè paj la.
  3. Fè chanjman.
  4. Klike sou Save.

Pifò jaden yo opsyonèl, epi kèk jaden yo obligatwa. Jaden obligatwa yo se:

  • Premye non
  • Siyati
  • Adrès imel
  • Lang
  • Timezone

Pou admin

Admin yo kapab tou mete ajou enfòmasyon pwofil yo pou lòt itilizatè yo. Sa a gen ladan twa jaden anba tit la ENFO ITIlizatè HOST.

  1. Konekte nan kont administratè Case Active ou a.
  2. Jwenn yon aplikasyon epi klike sou bouton ADMIN.
  3. Klike sou Itilizatè yo nan meni an tèt la.
  4. Ale nan sou-meni a epi chwazi lis itilizatè ou vle jwenn aksè (AKTIF oswa ANDIKAP). Anviwònman default la se Aktif.
  5. Jwenn yon itilizatè sou lis la epi klike sou Edit.
  6. Fè chanjman jan sa nesesè. (gade anba a)
  7. Klike sou Save.

Chanje enfòmasyon itilizatè

Admin yo ka chanje enfòmasyon itilizatè yo.

  1. Mete Kalite itilizatè a (Itilizatè oswa Manadjè).
  2. Mete Estati Itilizatè a (Aktif pou Enfim).
  3. Antre yon ID inik Tag. (opsyonèl)
  4. Klike sou Save.
