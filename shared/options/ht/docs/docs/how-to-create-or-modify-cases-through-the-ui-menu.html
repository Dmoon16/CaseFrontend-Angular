Kijan pou kreye oswa modifye ka atravè meni UI

Atik sa a montre administratè yo ki jan yo kreye yon ka ak koòdone itilizatè platfòm la. Administratè yo ka ajoute itilizatè tou nan yon nouvo ka epi mete ajou ka ki deja egziste yo.

Kreye yon ka

  1. Konekte nan kont admin la.
  2. Jwenn yon aplikasyon sou paj Aplikasyon an epi klike sou bouton ADMIN.
  3. Sou panèl Admin, klike sou tab meni Ka yo.
  4. Klike sou bouton NOUVO. Yon nouvo fenèt ap parèt.

Ranpli jaden obligatwa yo epi klike sou SAVE.

img

Ajoute itilizatè yo

Opsyonèlman, admin yo ka ajoute itilizatè yo nan ka a sou paj sa a. Scroll down epi klike sou bouton ASIGN USERS la.

img

Nan modal Assign User, ranpli jaden yo. Tcheke kare a pou voye itilizatè a yon notifikasyon imel. (opsyonèl)

img

Modifye yon ka

Pou mete ajou yon ti kantite ka, sèvi ak zouti Edit epi fè chanjman nan chak ka endividyèl.

  • Remak Pou mete ajou yon gwo kantite ka nan yon sèl chita, konsidere itilize zouti Import la
  1. Konekte nan kont admin la.
  2. Jwenn yon aplikasyon sou paj Aplikasyon an epi klike sou bouton ADMIN.
  3. Sou panèl Admin, klike sou tab meni Ka yo.
  4. Jwenn yon ka epi klike sou Edit.
  5. Fè chanjman epi klike sou SAVE.
