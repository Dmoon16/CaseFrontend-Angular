Fòma Done devwa ka itilizatè yo ak zouti enpòte

Anba a se yon lis jaden done nan dosye CSV. Chak opsyon enpòte gen yon konbinezon diferan nan jaden.

case_ID

  • Antre ka a Itilizatè oswa Tag ID.
  • Ale nan paj Ka > Edit > ID Tag OSWA Ale nan paj Ka > Edit > ID Ka nan url.

user_ID

  • Antre ID itilizatè a oswa ID Tag.
  • Ale nan paj Itilizatè yo > Edit > ID Tag OSWA Ale nan paj Itilizatè yo > Edit > ID Itilizatè nan url.

case_role_order

  • Kite selil vid.

case_role_ID

  • Antre: kliyan, anplwaye, oswa default.

case_notify

  • 0 = non
  • 1 = Wi, voye itilizatè yon imel notifikasyon.
