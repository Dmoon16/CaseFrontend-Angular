Ki jan admin yo jere itilizatè yo

Admin yo kreye, modifye oswa enfim ki moun ki ka jwenn aksè nan aplikasyon yo ak ka yo. Lè yon itilizatè konekte sou platfòm la pou premye fwa, li bezwen ranpli yon workflow pwofil.

  • Klike la a pou aprann plis sou nouvo workflow pwofil itilizatè a.

Jere yon itilizatè

Tablo ki anba a montre senk fason pou jere yon itilizatè. Klike sou yon lyen pou aprann kijan.

Metòd Kalite Aksyon Detay yo
Sèvi ak UI UI ajoute & modifye Bouton Admin > Itilizatè yo
Pandan y ap kreye yon ka epi bay itilizatè a ka UI ajoute Bouton Admin > Ka > Nouvo bouton > paj Kreye Ka
Oto kreye itilizatè lè l sèvi avèk lyen enskripsyon piblik la Aksyon itilizatè ajoute Bouton Admin > Anviwònman > Paj Enstalasyon Sistèm
Enpòte zouti UI ajoute & modifye Admin bouton > Entegrasyon > Enpòtasyon
Sèvi ak API (ap vini byento) API ajoute & modifye N/A

Notifikasyon

Platfòm nan pa voye itilizatè yo yon notifikasyon lè yo ajoute yo nan yon aplikasyon. Sepandan, administratè yo ka pèmèt notifikasyon pou devwa ka yo.
