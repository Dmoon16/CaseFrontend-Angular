Ki jan yo konekte

Aprann kijan pou w konekte ak kont Case Active ou ki egziste deja.

Konekte

  1. Ale nan paj login la.
  2. Klike sou bouton LOGIN.
  3. Antre imel ou.
  4. Klike sou PWOCHÈ.

Verifye yon kont

  1. Louvri yon nouvo tab navigatè epi konekte nan kont imel ou.
  2. Kopi kòd sis chif la.
  3. Retounen nan ka aktif tab la.
  4. Kole kòd la nan jaden yo. Yon nouvo fenèt ap parèt.

Case Active montre paj Aplikasyon an.

  • Remak Platfòm nan montre yon bouton ADMIN akote bouton APP a pou itilizatè ki gen privilèj admin.
