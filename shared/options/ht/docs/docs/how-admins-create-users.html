Ki jan admin yo kreye itilizatè yo

Admin yo kreye, modifye oswa enfim ki moun ki ka jwenn aksè nan aplikasyon yo ak ka yo. Lè yon itilizatè konekte sou platfòm la pou premye fwa, yo bezwen ranpli yon workflow pwofil.

  • Klike la a pou aprann plis sou nouvo workflow pwofil itilizatè a.

Kreye yon itilizatè

Tablo ki anba a montre senk fason pou kreye yon itilizatè. Klike sou yon lyen pou aprann kijan.

| Metòd | Kalite | Aksyon | Detay | Nouvo itilizatè resevwa yon notifikasyon? |
|-------------------------|---------------|------- ------|-------------------------------------------- ------------------------|

Sèvi ak UI | UI | ajoute & modifye | bouton Admin > Itilizatè | Non |

Pandan w ap kreye yon ka epi bay itilizatè a ka | UI | ajoute | Bouton Admin > Ka > Nouvo bouton > Kreye paj Ka | Wi, si li aktive |

Oto kreye itilizatè lè l sèvi avèk lyen enskripsyon piblik | Aksyon itilizatè | ajoute | Bouton Admin > Anviwònman > Paj Enstalasyon Sistèm | Wi, si oto kreye ka pèmèt |

Enpòte zouti | UI | ajoute & modifye | Admin bouton > Entegrasyon > Enpòtasyon | Non |

Sèvi ak API (ap vini byento) | API | ajoute & modifye | nan | nan |

Notifikasyon

Platfòm nan pa voye itilizatè yo yon notifikasyon lè yo ajoute yo nan yon aplikasyon. Sepandan, administratè yo ka pèmèt notifikasyon pou devwa ka yo.
