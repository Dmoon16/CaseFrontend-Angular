Comment les administrateurs gèrent les affectations d'utilisateurs aux requêtes

Il existe cinq façons de gérer les utilisateurs et les affectations de cas. Cliquez sur un lien pour savoir comment.

Méthode Taper Actions Détails
Lors de la création d'un dossier interface utilisateur ajouter Bouton Admin > Requêtes > Nouveau bouton > Bouton Attribuer des utilisateurs
Attribuer automatiquement un utilisateur à l'aide d'un lien d'inscription public Action de l'utilisateur ajouter Bouton Admin > Paramètres > page Configuration du système
Utilisation de l'interface utilisateur sur un cas existant interface utilisateur Ajouter enlever Bouton Admin > Requêtes > Attribuer des utilisateurs
Outil d'importation interface utilisateur ajouter, modifier et supprimer Bouton Admin > Intégrations > Importations
