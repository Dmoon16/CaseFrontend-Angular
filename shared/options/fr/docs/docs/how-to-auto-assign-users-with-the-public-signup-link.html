Comment attribuer automatiquement des utilisateurs avec le lien d'inscription public

Les administrateurs apprennent à configurer les paramètres système afin que les nouveaux utilisateurs soient automatiquement affectés à un cas nouvellement créé automatiquement.

Ouverture de la page de configuration

La première étape consiste à ouvrir la page de configuration des systèmes.

  1. Connectez-vous au compte administrateur.
  2. Localisez une application sur la page Applications et cliquez sur le bouton ADMIN.
  3. Cliquez sur l'onglet Systèmes puis sur CONFIGURATION DU SYSTÈME dans le sous-menu.
  4. Faites défiler jusqu'à l'en-tête INSCRIPTION UTILISATEUR.

Configuration des paramètres système

  1. Définissez les menus déroulants Autoriser l'inscription et Créer automatiquement un ticket sur Activé.
  2. Cliquez sur le menu du rôle et choisissez une option (par défaut est recommandé).
  3. Copiez et partagez le lien Public User Signup.

image

Envoi du lien

Chaque utilisateur qui clique sur le lien et termine l'inscription déclenche la plate-forme pour générer un nouveau cas sur la page Cas et attribue automatiquement l'utilisateur dans ce cas.
