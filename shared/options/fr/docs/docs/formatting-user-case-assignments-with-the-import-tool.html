Formatage des données des affectations de cas utilisateur avec l'outil d'importation

Vous trouverez ci-dessous une liste des champs de données dans les fichiers CSV. Chaque option d'importation a une combinaison différente de champs.

ID_cas

  • Saisissez l'identifiant de l'utilisateur ou de la balise.
  • Accédez à la page Cas > Modifier > ID de balise OU Accédez à la page Cas > Modifier > ID de cas dans l'url.

ID de l'utilisateur

  • Entrez l'ID de l'utilisateur ou l'ID de balise.
  • Accédez à la page Utilisateurs > Modifier > Tag ID OU Accédez à la page Utilisateurs > Modifier > ID utilisateur dans l'url.

case_role_order

  • Laissez la cellule vide.

case_role_ID

  • Entrez : clients, personnel ou par défaut.

cas_notifier

  • 0 = non
  • 1 = Oui, envoyer à l'utilisateur une notification par e-mail.
