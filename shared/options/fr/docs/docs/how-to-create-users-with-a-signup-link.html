Comment créer des utilisateurs avec un lien d'inscription

Cet article montre aux administrateurs comment créer des utilisateurs en envoyant un lien d'inscription.

Configuration du lien d'inscription

  1. Cliquez sur le lien Envoyez-leur une inscription en bas de la boîte de dialogue Ajouter un utilisateur. Une nouvelle page s'affiche.
  2. Configurez les options d'inscription. (voir ci-dessous)

image

Assurez-vous que le champ Autoriser l'inscription est défini sur Activé. (obligatoire) Lorsqu'il est activé, Case Active génère un lien unique.

Passez en revue les options ci-dessous, puis cliquez sur l'icône pour copier le lien. Vous pouvez placer le lien sur votre site Web ou envoyer le lien à une ou plusieurs personnes par e-mail ou tout autre canal de communication.

image

Création d'un cas

Facultativement, vous pouvez créer automatiquement un cas pour chaque nouvel utilisateur et attribuer un rôle d'utilisateur.

  1. Cliquez sur le champ Créer automatiquement un cas et sélectionnez Activé. Les champs Rôle de l'utilisateur du dossier s'affichent.
  2. Cliquez sur le rôle d'utilisateur de dossier et sélectionnez une option. (voir ci-dessous)

image

Rôles d'utilisateur de cas

Il existe trois rôles d'utilisateur de dossier.

  • clients
  • défaut
  • Personnel

Envoi du lien

Partagez le lien avec une ou plusieurs personnes. Chaque personne clique sur le lien et termine le workflow pour activer son compte Case Active. Une fois que l'utilisateur a terminé le processus d'inscription, Case Active ajoute l'utilisateur à la liste des utilisateurs.

image

Prochaines étapes

Case Active envoie à l'utilisateur une notification par e-mail avec le lien d'inscription. L'utilisateur clique sur le lien et termine le flux de travail pour les nouvelles inscriptions.
