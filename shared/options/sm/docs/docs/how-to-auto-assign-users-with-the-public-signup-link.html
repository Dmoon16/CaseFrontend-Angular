Fa'afefea ona Toto'a Aunoa Tagata fa'aoga i le So'otaga Fa'asinomaga Lautele

E aʻoaʻoina e le au pulega le faʻaogaina o faʻatonuga o le faiga ina ia otometi le tuʻuina atu o tagata faʻaoga fou i se mea fou na faia.

Tatala le itulau seti

O le laasaga muamua o le tatalaina lea o le System Setup page.

  1. Ulufale i le admin account.
  2. Su'e se app ile itulau Apps ma kiliki le ADMIN button.
  3. Kiliki le fa'ailoga Fa'atonu ona sosoo ai lea ma le SYSTEM SETUP i le submenu.
  4. Fa'asolo i lalo ile USER SIGNUP ulutala.

Fa'atulagaina o fa'atonuga o faiga

  1. Seti le Allow Signup ma Auto Create Case dropdown menus i luga.
  2. Kiliki le lisi o matafaioi ma filifili se filifiliga (ua fautuaina le fa'aletonu).
  3. Kopi ma fa'asoa le feso'ota'iga Fa'asinomaga a Tagata Lautele.

Tuuina atu le sootaga

O tagata taʻitasi e kiliki le fesoʻotaʻiga ma faʻamaeʻaina le saini e faʻaosoina le faʻavae e faʻatupu se mataupu fou i luga o le itulau Cases ma tuʻuina atu le tagata faʻaoga i lena tulaga.
