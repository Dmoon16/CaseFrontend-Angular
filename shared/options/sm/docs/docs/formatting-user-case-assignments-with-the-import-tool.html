Fa'asologa o Fa'amatalaga Tofiga a Tagata Fa'aaoga ma le Meafaigaluega Fa'aulufale mai

O lo'o i lalo le lisi o fa'amaumauga i faila CSV. O filifiliga fa'aulufale ta'itasi e iai le tu'ufa'atasiga o fanua.

case_ID

  • Ulufale i le mataupu User poʻo Tag ID.
  • Alu ile Itulau itulau > Fa'asa'o > Tag ID PO'O le Alu ile Itulau itulau > Fa'asa'o > Fa'ailoga ID ile url.

user_ID

  • Ulufale ile ID ole tagata fa'aoga po'o le Tag ID.
  • Alu i Tagata Fa'aoga itulau> Fa'asa'o> Tag ID PO'O Alu i Tagata Fa'aoga itulau> Fa'atonu> Fa'aoga ID ile url.

case_role_order

  • Tuu le sela e avanoa.

case_role_ID

  • Ulufale: tagata fa'atau, tagata faigaluega, po'o fa'aletonu.

case_notify

  • 0 = leai
  • 1 = Ioe, auina atu i le tagata faʻaoga se faʻamatalaga imeli.
