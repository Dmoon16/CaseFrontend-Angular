Fa'afefea ona Tofi, Suia pe Ave'ese Tofiga a Tagata Fa'aaoga ma le Meafaigaluega Fa'aulufale mai

O le tala lea e fa'aali atu ai i le au pulega le fa'afoeina o tagata fa'aoga mataupu i le mea faigaluega Fa'aulufale mai.

E mafai e le pulega ona:

  • tofia tagata fa'aoga i se mataupu
  • sui tagata fa'aoga ua uma ona tofia i se mataupu
  • aveese tagata fa'aoga ua tofia i se mataupu

Tatala le meafaigaluega Fa'aulufale mai

  1. Ulufale i le admin account.
  2. Su'e se app ile itulau Apps ma kiliki le ADMIN button.
  3. Kiliki le Fa'atasi fa'atasi ona fa'aulufale mai lea i luga ole la'au lisi.
  4. Kiliki le faamau FOU. O lo'o fa'aalia le pusa fa'atalanoaga Add Imports.

Fausia faamaumauga

La'u mai se tasi pe sili atu faila CSV. Fa'aoga faila nei e fa'aopoopo, sui, pe aveese tagata fa'aoga mataupu. A uma ona tatala le pusa fa'atalanoaga Add Imports:

  1. Kiliki le Ituaiga fanua.
  2. Fa'asolo i totonu o le lisi o le pa'u i lalo ma filifili se filifiliga (silasila i lalo). O lo'o fa'aalia se so'oga.
  3. Kiliki le download Fa'ata'ita'iga so'oga. O se faila CSV e la'u mai i lau masini.

Tatala le faila ma faʻaopoopo faʻamaumauga. Kiliki iinei e a'oa'o ai pe fa'apefea ona fa'aoga fa'amaumauga fa'asologa i le faila CSV.

Filifiliga faila

Tofi tagata fa'aoga i mataupu * Filifili lenei filifiliga e fa'aopoopo ai tagata fa'aoga i se mataupu. * Tuu le case_role_order cell e avanoa. * Alu i le itulau Users ma kopi le Tag ID mo lenei tagata. Afai e avanoa le Fa'ailoga ID ID, tu'u se ID tulaga ese, kiliki Save, ona kopi lea o le ID ile faila CSV.

img

Suia tagata fa'aoga ua tofia i mataupu * Filifili le filifiliga lenei e fa'afou ai fa'amaumauga mo tagata o lo'o iai nei ua tofia i se mataupu. * Fa'aaoga le Case Tag ID mo le case_ID koluma.

Aveese tagata fa'aoga ua tofia i mataupu * Filifili le filifiliga lenei e aveese ai tagata fa'aoga ua tofia i se mataupu.

Tuuina atu o le faila

E mafai ona e tu'u i luga se faila i le taimi lava lena pe toe fo'i i le Add Imports dialog box i se aso mulimuli ane ma fa'auma nei la'asaga.

  1. Tatala le pusa fa'atalanoaga Add Imports.
  2. Ulufale se igoa tulaga ese mo le faila CSV.
  3. Kiliki le fa'aopoopo (+) icon e lafo ai le faila mai lau masini.
  4. Kiliki le Ituaiga fanua ma filifili se filifiliga. (silasila i luga)
  5. Kiliki SAVE.

Fa'agasologa

Faʻalagolago i le numera o faʻamaumauga i le faila ma le utaina o le polokalama, e mafai e le faʻavae ona faʻaalu ni nai minute e faʻagasolo ai le faʻaulufale mai.

img

A mae'a le fa'aulufale mai, e fetu'una'i e le tulaga fa'amaumauga a tagata fa'aoga (fa'aopoopo, suia, pe aveese) i luga o le itulau Cases.

To'ilalo

Afai e le manuia le fa'aulufaleina mai, e toe fa'afo'i mai e le tulaga se fe'au Fa'aletonu. I le tele o tulaga, o le mafuaʻaga o le toilalo o le faʻatulagaina o faʻamaumauga.

Ina ia iloa le mafuaaga o le faaletonu:

  1. Kiliki Va'ai i lalo o le Fa'aulutala Fa'atino.
  2. I le itulau e sosoo ai, fa'ae'e le isumu ile fa'ailoga Fai. O lo'o fa'aalia e le tooltip le mea sese.
  3. Alu i le faila CVS ma fetuunai ma faʻaleleia le mea sese.
  4. Toe lafo le faila.
