Fa'afefea ona Fausia Fa'amata'u Fa'amatalaga ma se So'oga Fa'asinomaga Lautele

O lenei tusiga o loʻo faʻaalia ai le au pulega pe faʻafefea ona fai se mataupu (ma faʻaopoopo se tagata faʻaoga i le taimi e tasi) e ala i le lafoina o se soʻotaga saini e tasi.

Su'eina le so'oga

  1. Ulufale i le admin account.
  2. Su'e se app ile itulau Apps ma kiliki le ADMIN button.
  3. I luga o le Admin panel, kiliki le Seti menu tab.
  4. Ia mautinoa ua filifilia le SYSTEM SETUP submenu tab.

Fa'atonu tulaga

Fa'asolo i lalo ile USER SIGNUP ulutala.

Ia mautinoa ua seti le Fa'ataga Signup fanua i luga. (mana'omia) A fa'agaoioia lenei vaega, e fa'atupuina e Case Active se so'oga tulaga ese.

  1. Kiliki le Auto Create Case ma filifili I luga. O lo'o fa'aalia le fanua o le Matafaioi a le Tagata Fa'aaoga.
  2. Filifili se matafaioi. (silasila i lalo)

Matafaioi tagata faaaoga

E tolu mata'upu fa'aoga matafaioi.

  • tagata fa'atau (fautua)
  • faaletonu
  • aufaigaluega

Tuuina atu le sootaga

Faasoa atu le sootaga i se tasi pe sili atu tagata. O tagata faʻaoga taʻitasi e kiliki le fesoʻotaʻiga ma faʻamaeʻaina le saini e faʻaosoina le tulaga e faʻatupu ai se mataupu fou i luga o le itulau mataupu ma tuʻuina atu le tagata faʻaoga i le mataupu.
