Auala e saini ai

A'oa'o pe fa'afefea ona e ulufale i lau Case Active account.

Ulufale i totonu

  1. Alu i le itulau e saini ai .
  2. Kiliki le faamau LOGIN.
  3. Ulufale lau imeli.
  4. Kiliki le NEXT.

Fa'amaonia se tala

  1. Tatala se tab browser fou ma saini i lau imeli tala.
  2. Kopi le numera ono numera.
  3. Toe fo'i i le Case Active tab.
  4. Fa'amau le code i totonu o fanua. Ua fa'aalia se faamalama fou.

Case Active fa'aalia le itulau Apps.

  • Fa'aaliga O lo'o fa'aalia e le fa'ailoga se fa'amau ADMIN i tafatafa o le fa'amau APP mo tagata fa'aoga e iai fa'amanuiaga fa'aau.
