Bii o ṣe le Fi Awọn olumulo ṣiṣẹ Lakoko Ṣiṣẹda Ọran kan

Nkan yii fihan awọn admins bi o ṣe le fi olumulo ranṣẹ lakoko ṣiṣẹda ọran tuntun kan. Ti olumulo ko ba ti forukọsilẹ tẹlẹ lori pẹpẹ, o le ṣẹda ṣafikun olumulo tuntun lakoko ṣiṣẹda ọran tuntun kan.

Ṣiṣẹda ọran kan

  1. Wọle si akọọlẹ abojuto.
  2. Wa ohun elo kan lori oju-iwe Awọn ohun elo ki o tẹ bọtini ADMIN naa.
  3. Tẹ awọn Awọn ọran taabu taabu.
  4. Tẹ bọtini TITUN. A titun window han.
  • Fọwọsi awọn aaye lati ṣẹda ọran kan lẹhinna yi lọ si isalẹ si akọle Awọn olumulo ti a sọtọ.

Ifiranṣẹ olumulo kan

Olumulo lọwọlọwọ

Tẹle awọn igbesẹ wọnyi lati fi olumulo lọwọlọwọ si ọran tuntun.

Tẹ bọtini ASSIGN olumulo ati fọwọsi awọn aaye.

  1. Tẹ jabọ-silẹ Olumulo ko si yan idanimọ kan (fun apẹẹrẹ imeeli).
  2. Tẹ iye kan sii ni aaye òfo (fun apẹẹrẹ adirẹsi imeeli).
  3. Ti igbasilẹ olumulo ba wa ni pẹpẹ, aaye naa ni adaṣe ni imọran orukọ kan.
  4. Yan olumulo to tọ.
  5. Yan ipa olumulo kan (awọn alabara, aiyipada, tabi oṣiṣẹ).
  6. Ni yiyan, ṣayẹwo apoti lati fi ifitonileti imeeli ranṣẹ si olumulo.
  7. Tẹ Fipamọ.

Nṣiṣẹ Nṣiṣẹ ṣe afikun olumulo si ọran naa. Bayi awọn alabojuto le tẹ Fipamọ lati tii window, tabi ṣafikun olumulo tuntun kan, tabi yọ olumulo kan kuro.

Fifi olumulo titun kan kun

Ti pẹpẹ ko ba le rii igbasilẹ lọwọlọwọ, tẹ bọtini lati ṣafikun olumulo tuntun kan.

Fọwọsi awọn aaye ati lẹhinna tẹ Fipamọ lati ṣafikun olumulo si pẹpẹ. Nigbamii, ṣeto ipa olumulo kan ki o tẹ Fipamọ lẹẹkansi lati fi olumulo si ọran naa. Ni ipari, tẹ Fipamọ ni akoko diẹ sii lati fi ọran naa pamọ.
