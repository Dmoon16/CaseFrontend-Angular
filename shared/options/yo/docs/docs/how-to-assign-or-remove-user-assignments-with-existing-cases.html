Bii o ṣe le Fi tabi Yọ Awọn iṣẹ iyansilẹ Olumulo pẹlu Awọn ọran ti o wa tẹlẹ

Nkan yii fihan awọn admins bi o ṣe le yan ati yọ awọn olumulo ọran kuro. A tun fihan ọ bi o ṣe le yipada data olumulo.

Nsii ọran ti o wa tẹlẹ

  1. Wọle si akọọlẹ abojuto.
  2. Wa ohun elo kan lori oju-iwe Awọn ohun elo ki o tẹ bọtini ADMIN naa.
  3. Tẹ awọn Awọn ọran taabu taabu.
  4. Rii daju pe A ti yan ACTIVE ninu akojọ aṣayan.
  5. Yi lọ si isalẹ si iwe ACTION.

Sọtọ tabi yọ kuro

Tẹ Fi Awọn olumulo ranṣẹ lati fi olumulo kan si ọran naa tabi yọ olumulo kuro ninu ọran naa (ṣugbọn kii ṣe pẹpẹ). Lori oju-iwe ti o tẹle: Tẹ ADD ki o pari iṣiṣẹ iṣẹ. Tẹ yọ kuro lati pa olumulo kan rẹ.

Ṣatunṣe

Ko ṣee ṣe lati ṣatunkọ tabi imudojuiwọn data olumulo lori oju-iwe Awọn ọran. Dipo, tẹle awọn igbesẹ wọnyi:

  1. Tẹ awọn olumulo taabu taabu.
  2. Wa orukọ olumulo ki o tẹ Ṣatunkọ .
  3. Ṣe awọn ayipada ki o tẹ Fipamọ.
