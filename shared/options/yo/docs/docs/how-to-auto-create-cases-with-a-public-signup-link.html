Bii o ṣe le Ṣẹda Awọn ọran Aifọwọyi pẹlu Ọna asopọ Iforukọsilẹ Gbogbo eniyan

Nkan yii fihan awọn admins bi o ṣe le ṣẹda ọran kan (ki o ṣafikun olumulo ni akoko kanna) nipa fifiranṣẹ ọna asopọ iforukọsilẹ ẹyọkan.

Wiwa ọna asopọ

  1. Wọle si akọọlẹ abojuto.
  2. Wa ohun elo kan lori oju-iwe Awọn ohun elo ki o tẹ bọtini ADMIN naa.
  3. Lori awọn Abojuto nronu, tẹ awọn Eto taabu taabu.
  4. Rii daju pe o ti yan taabu inu eto SYSTEM SETUP.

Awọn eto atunto

Yi lọ si isalẹ si akọle SIGNUP USER.

Rii daju pe aaye Gba Iforukọsilẹ ti ṣeto si Tan-an. (beere fun) Nigba ti ẹya ara ẹrọ yi wa ni sise, Case Active gbogbo a oto ọna asopọ.

  1. Tẹ aaye Aifọwọyi Ṣẹda Case ko si yan Tan-an. Awọn ifihan aaye ipa User Case.
  2. Yan ipa kan. (wo isalẹ)

Awọn ipa olumulo ọran

Awọn ipa olumulo ọran mẹta wa.

  • awọn onibara (niyanju)
  • aiyipada
  • osise

Fifiranṣẹ ọna asopọ

Pin ọna asopọ pẹlu eniyan kan tabi diẹ sii. Olumulo kọọkan ti o tẹ ọna asopọ ti o pari iforukọsilẹ nfa pẹpẹ lati ṣe agbekalẹ ọran tuntun lori oju-iwe awọn ọran ati fi olumulo le laifọwọyi ninu ọran naa.
