Bii o ṣe le Ṣẹda tabi Ṣatunṣe Awọn ọran pẹlu Irinṣẹ Wọwọle

Nkan yii fihan awọn alabojuto bi o ṣe le ṣẹda tabi yipada awọn ọran ti o wa pẹlu irinṣẹ agbewọle.

Nsii ohun elo agbewọle

  1. Wọle si akọọlẹ abojuto.
  2. Wa ohun elo kan lori oju-iwe Awọn ohun elo ki o tẹ bọtini ADMIN naa.
  3. Lori igbimọ abojuto, tẹ taabu Integration ati lẹhinna gbe wọle lori akojọ aṣayan.
  4. Tẹ bọtini TITUN. Awọn Fikun Awọn agbewọle lati inu apoti ibanisọrọ han ni taabu aṣawakiri tuntun kan.

Ṣiṣẹda awọn ọran

Lẹhin apoti ibaraẹnisọrọ Fikun Awọn agbewọle lati ilu okeere ṣii:

  1. Tẹ aaye Iru ki o yan aṣayan kan (wo isalẹ). A ọna asopọ han.
  2. Tẹ ọna asopọ Ayẹwo Gbigba lati ayelujara. Awọn igbasilẹ faili CSV si ẹrọ rẹ.

Nigbamii, ṣii faili naa ki o ṣafikun awọn ọran.

Iru

Nigbati o ba tẹ aaye Iru , akojọ aṣayan-silẹ yoo han. Yan ọkan ninu awọn aṣayan meji fun awọn agbewọle nla nla.

  • Akiyesi Akojọ aṣayan-silẹ ni ọpọlọpọ awọn aṣayan, ṣugbọn meji nikan ni o ni ibatan si iṣakoso ọran. Aṣayan kọọkan ni faili CSV ti o yatọ.

Fi awọn igba kun

  • Yan aṣayan yii fun ikojọpọ olopobobo ti o rọrun ti awọn ọran.
  • Tẹ ọna asopọ Ayẹwo Gbigba lati ayelujara lati ṣii faili CSV pẹlu awọn akọle iwe fun Ṣẹda awọn ọran gbe wọle iru.

Ṣatunṣe awọn ọran

  • Yan aṣayan yii lati ṣe imudojuiwọn awọn ọran lọwọlọwọ.
  • Tẹ ọna asopọ Ayẹwo Gbigba lati ayelujara lati ṣii faili CSV pẹlu awọn akọle iwe fun Yipada awọn ọran gbe wọle iru.

Ikojọpọ faili

O le gbe faili kan silẹ lẹsẹkẹsẹ tabi pada si Fikun apoti ibanisọrọ Awọn agbewọle ni ọjọ ti o tẹle ki o pari awọn igbesẹ wọnyi.

  1. Ṣii apoti ibaraẹnisọrọ Fikun Awọn agbewọle lati ilu okeere.
  2. Tẹ orukọ alailẹgbẹ sii fun faili CSV.
  3. Tẹ aami afikun (+) lati gbe faili naa sori ẹrọ rẹ.
  4. Tẹ aaye Iru ki o yan aṣayan kan. (wo loke)
  5. Tẹ Fipamọ.

Tito kika

Iwọnyi jẹ awọn aaye ti a beere ati awọn ibeere akọkọ. Tẹ ibi lati ṣe ayẹwo awọn alaye kika.

tag_id (tabi case_ID) idamọ ara oto (lo awọn ohun kikọ alphanumeric, awọn hyphen ti gba)

case_status 0 = pipade 1 = lọwọ

date_oṣu /ojo/odun mm/dd/yyyy

nipa apejuwe alailẹgbẹ ti ọran naa (lo awọn ohun kikọ alphanumeric, a gba awọn hyphens)

Ṣiṣẹda

Ti o da lori nọmba awọn igbasilẹ ninu faili ati fifuye eto, pẹpẹ le gba awọn iṣẹju pupọ lati ṣe ilana gbigbe wọle.

Nigbati agbewọle ba pari, pẹpẹ n ṣatunṣe awọn igbasilẹ ọran (ie ṣafikun tabi ṣe atunṣe) lori Dasibodu Awọn ọran abojuto.

Ikuna

Ti iṣẹ agbewọle ba kuna, pẹpẹ naa da ifiranṣẹ ti o kuna pada. Ni ọpọlọpọ igba, idi ti ikuna jẹ kika data.

img

Lati wo idi ti ikuna:

  1. Tẹ Wo labẹ akọle ACTION.
  2. Ni oju-iwe ti o tẹle, gbe eku lori aami ti o kuna. Ohun elo irinṣẹ ṣe afihan aṣiṣe naa.
  3. Lọ si faili CVS ki o ṣatunṣe ati ṣatunṣe aṣiṣe naa.
  4. Po si faili lẹẹkansi.
