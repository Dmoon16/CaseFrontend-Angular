Bii o ṣe le Ṣẹda tabi Ṣatunṣe Awọn ọran Nipasẹ Akojọ UI

Nkan yii fihan awọn admins bii o ṣe le ṣẹda ọran pẹlu wiwo olumulo ti pẹpẹ. Awọn alabojuto tun le ṣafikun awọn olumulo si ọran tuntun ki o ṣe imudojuiwọn awọn ọran ti o wa tẹlẹ.

Ṣiṣẹda ọran kan

  1. Wọle si akọọlẹ abojuto.
  2. Wa ohun elo kan lori oju-iwe Awọn ohun elo ki o tẹ bọtini ADMIN naa.
  3. Lori awọn Abojuto nronu, tẹ awọn ọran taabu taabu.
  4. Tẹ bọtini TITUN. A titun window han.

Fọwọsi awọn aaye ti a beere ki o tẹ FIPAMỌ.

Fifi awọn olumulo

Ni yiyan, awọn alabojuto le ṣafikun awọn olumulo si ọran naa ni oju-iwe yii. Yi lọ si isalẹ ki o tẹ bọtini ASSIGN olumulo .

img

Ni Ilana Olumulo Firanṣẹ, fọwọsi awọn aaye. Ṣayẹwo apoti lati fi ifitonileti imeeli ranṣẹ si olumulo. (aṣayan)

Iyipada a irú

Lati ṣe imudojuiwọn nọmba kekere ti awọn ọran, lo ohun elo Ṣatunkọ ati ṣe awọn ayipada si ọran kọọkan.

  • Akiyesi Lati ṣe imudojuiwọn nọmba nla ti awọn ọran ni ijoko kan, ronu nipa lilo irinṣẹ agbewọle
  1. Wọle si akọọlẹ abojuto.
  2. Wa ohun elo kan lori oju-iwe Awọn ohun elo ki o tẹ bọtini ADMIN naa.
  3. Lori awọn Abojuto nronu, tẹ awọn ọran taabu taabu.
  4. Wa ọran kan ki o tẹ Ṣatunkọ.
  5. Ṣe awọn ayipada ki o tẹ Fipamọ.
