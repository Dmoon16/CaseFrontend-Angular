Como crear usuarios cunha ligazón de rexistro

Este artigo mostra aos administradores como crear usuarios enviando unha ligazón de rexistro.

Configurando a ligazón de rexistro

  1. Fai clic na ligazón Enviarlles un rexistro na parte inferior do cadro de diálogo Engadir usuario. Aparece unha nova páxina.
  2. Configure as opcións de rexistro. (Ver abaixo)

Asegúrate de que o campo Permitir rexistro estea activado. (obrigatorio) Cando está activado, Case Active xera unha ligazón única.

Revisa as opcións seguintes e fai clic na icona para copiar a ligazón. Podes colocar a ligazón no teu sitio web ou enviar a ligazón a unha ou varias persoas por correo electrónico ou calquera canle de comunicación.

img

Creando un caso

Opcionalmente, pode crear automaticamente un caso para cada novo usuario e asignar un rol de usuario.

  1. Fai clic no campo Crear caso automaticamente e selecciona Activado. Aparecen os campos Rol de usuario do caso.
  2. Fai clic na función de usuario do caso e selecciona unha opción. (Ver abaixo)

Roles de usuario do caso

Hai tres roles de usuario de caso.

  • clientes
  • por defecto
  • persoal

Enviando a ligazón

Comparte a ligazón cunha ou máis persoas. Cada persoa fai clic na ligazón e completa o fluxo de traballo para activar a súa conta Case Active. Unha vez que o usuario complete o fluxo de traballo de rexistro, Case Active engade o usuario á lista de Usuarios.

img

Próximos pasos

Case Active envía ao usuario unha notificación por correo electrónico coa ligazón de rexistro. O usuario fai clic na ligazón e completa o fluxo de traballo para novos rexistros.
