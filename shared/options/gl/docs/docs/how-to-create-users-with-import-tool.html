Como crear usuarios coa ferramenta de importación

Este artigo mostra aos administradores como crear usuarios coa ferramenta Importar.

Abrir a ferramenta Importar

Os administradores poden usar a ferramenta Importar para cargar novos usuarios en masa e modificar os rexistros de usuarios existentes.

  1. Inicie sesión na conta de administrador.
  2. Localiza unha aplicación na páxina Aplicacións e fai clic no botón ADMINISTRADOR.
  3. No panel de administración, faga clic na pestana Integración e despois IMPORTACIÓNS no submenú.
  4. Fai clic no botón NOVO. O cadro de diálogo Engadir importacións móstrase nunha nova pestana do navegador.

Creando rexistros

Despois de abrir a caixa de diálogo Engadir importacións:

  1. Fai clic no campo Tipo e selecciona unha opción (ver abaixo). Móstrase unha ligazón.
  2. Fai clic na ligazón Descargar mostra . Descárgase un ficheiro CSV no teu dispositivo.

A continuación, abra o ficheiro e engada rexistros. Fai clic aquí para saber como usar os datos de formato no ficheiro CSV.

Tipo

Cando fai clic no campo Tipo, aparece un menú despregable. Escolla unha das dúas opcións para importacións masivas.

:exclamation: NOTA O menú despregable ten moitas opcións, pero só dúas están relacionadas coa ferramenta Importar. Cada opción ten un ficheiro CSV de mostra diferente.

Engadir usuarios

  • Escolla esta opción para realizar unha simple carga masiva de rexistros de usuarios.
  • Fai clic na ligazón Descargar mostra para abrir un ficheiro CSV con títulos de columna para o tipo de importación Engadir usuarios.

Modificar usuarios

  • Escolla esta opción para actualizar os datos dos usuarios actuais.
  • Fai clic na ligazón Descargar mostra para abrir un ficheiro CSV con títulos de columna para o tipo de importación Modificar usuarios.

Cargando o ficheiro

Podes cargar un ficheiro de inmediato ou volver á caixa de diálogo Engadir importacións nunha data posterior e completar estes pasos.

  1. Abre o cadro de diálogo Engadir importacións.
  2. Introduza un nome único para o ficheiro CSV.
  3. Fai clic na icona máis (+) para cargar o ficheiro desde o teu dispositivo.
  4. Fai clic no campo Tipo e selecciona unha opción. (ver arriba)
  5. Fai clic en GARDAR.

Procesamento

Dependendo do número de rexistros no ficheiro e da carga do sistema, a plataforma pode tardar varios minutos en procesar a importación.

Cando se completa a importación, a plataforma axusta os rexistros de usuarios (é dicir, engade, modifica ou desactiva) no panel de usuarios do administrador.

Fracaso

Se a operación de importación falla, a plataforma devolve unha mensaxe de fallo. Na maioría dos casos, a causa do fallo é o formato dos datos.

Para ver a causa do fallo:

  1. Fai clic en Ver baixo o título Acción.
  2. Na páxina seguinte, pase o rato sobre a icona Fallou. A información sobre ferramentas mostra o erro.
  3. Vaia ao ficheiro CVS e axuste e corrixa o erro.
  4. Carga o ficheiro de novo.
