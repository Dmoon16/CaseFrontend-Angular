Formato de datos de xestión de casos coa ferramenta de importación

A continuación móstrase unha lista de campos obrigatorios en ficheiros CSV para xestionar casos coa ferramenta Importar.

nome de usuario

  • Introduza o enderezo de correo electrónico do usuario.

tag_id (un id que podes consultar máis tarde)

  • un identificador único (usar caracteres alfanuméricos, se aceptan guións)

caso_estado

  • 0 = pechado
  • 1 = activo

data_aberto

  • mes/día/ano
  • mm/dd/aaaa

sobre

  • unha descrición única do caso (utilice caracteres alfanuméricos, acéptanse guións)
