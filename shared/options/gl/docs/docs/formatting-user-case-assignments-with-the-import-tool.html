Formato de datos de asignacións de casos de usuario coa ferramenta de importación

A continuación móstrase unha lista de campos de datos en ficheiros CSV. Cada opción de importación ten unha combinación diferente de campos.

ID_case

  • Introduza o ID de usuario ou etiqueta do caso.
  • Vaia á páxina Casos > Editar > ID da etiqueta OU Vaia á páxina Casos > Editar > ID do caso en url.

ID do usuario

  • Introduza o ID do usuario ou o ID da etiqueta.
  • Vaia á páxina Usuarios > Editar > ID de etiqueta OU Vaia á páxina Usuarios > Editar > ID de usuario en url.

orde_papel_caso

  • Deixa a cela en branco.

ID_función_caso

  • Introduza: clientes, persoal ou predeterminado.

caso_notificar

  • 0 = non
  • 1 = Si, envíe ao usuario unha notificación por correo electrónico.
