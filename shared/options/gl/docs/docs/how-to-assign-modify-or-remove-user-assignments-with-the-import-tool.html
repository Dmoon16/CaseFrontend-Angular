Como asignar, modificar ou eliminar asignacións de usuarios coa ferramenta de importación

Este artigo mostra aos administradores como xestionar os usuarios de casos coa ferramenta de importación.

Os administradores poden:

  • asignar usuarios a un caso
  • modificar usuarios xa asignados a un caso
  • eliminar usuarios asignados a un caso

Abrir a ferramenta Importar

  1. Inicie sesión na conta de administrador.
  2. Localiza unha aplicación na páxina Aplicacións e fai clic no botón ADMINISTRADOR.
  3. Fai clic na pestana Integración e, a continuación, en IMPORTACIÓNS no submenú.
  4. Fai clic no botón NOVO. Aparece o cadro de diálogo Engadir importacións.

Creando rexistros

Descarga un ou máis ficheiros CSV. Use estes ficheiros para engadir, modificar ou eliminar usuarios de casos. Despois de abrir a caixa de diálogo Engadir importacións:

  1. Fai clic no campo Tipo.
  2. Desprácese polo menú despregable e seleccione unha opción (ver abaixo). Móstrase unha ligazón.
  3. Fai clic na ligazón Descargar mostra. Descárgase un ficheiro CSV no teu dispositivo.

img

Abre o ficheiro e engade rexistros. Fai clic aquí para saber como usar os datos de formato no ficheiro CSV.

Opcións de ficheiro

Asignar usuarios a casos * Escolla esta opción para engadir usuarios a un caso. * Deixe a cela case_role_order en branco. * Vaia á páxina Usuarios e copie o ID de etiqueta desta persoa. Se o campo ID da etiqueta está en branco, introduza un ID único, faga clic en Gardar e, a continuación, copie o ID no ficheiro CSV.

Modificar usuarios asignados a casos * Escolla esta opción para actualizar os datos dos usuarios actuais asignados a un caso. * Use o ID de etiqueta de caso para a columna case_ID.

img

Eliminar usuarios asignados a casos * Escolla esta opción para eliminar usuarios asignados a un caso.

Cargando o ficheiro

Podes cargar un ficheiro de inmediato ou volver á caixa de diálogo Engadir importacións nunha data posterior e completar estes pasos.

  1. Abre o cadro de diálogo Engadir importacións.
  2. Introduza un nome único para o ficheiro CSV.
  3. Fai clic na icona máis (+) para cargar o ficheiro desde o teu dispositivo.
  4. Fai clic no campo Tipo e selecciona unha opción. (ver arriba)
  5. Fai clic en GARDAR.

Procesamento

Dependendo do número de rexistros no ficheiro e da carga do sistema, a plataforma pode tardar varios minutos en procesar a importación.

Cando se completa a importación, a plataforma axusta os rexistros de usuarios (é dicir, engade, modifica ou elimina) na páxina Casos.

Fracaso

Se a operación de importación falla, a plataforma devolve unha mensaxe de fallo. Na maioría dos casos, a causa do fallo é o formato dos datos.

img

Para ver a causa do fallo:

  1. Fai clic en Ver baixo o título Acción.
  2. Na páxina seguinte, pase o rato sobre a icona Fallou. A información sobre ferramentas mostra o erro.
  3. Vaia ao ficheiro CVS e axuste e corrixa o erro.
  4. Carga o ficheiro de novo.
