Como crear automaticamente casos cunha ligazón de rexistro público

Este artigo mostra aos administradores como crear un caso (e engadir un usuario ao mesmo tempo) enviando unha única ligazón de rexistro.

Localizando a ligazón

  1. Inicie sesión na conta de administrador.
  2. Localiza unha aplicación na páxina Aplicacións e fai clic no botón ADMINISTRADOR.
  3. No panel de administración, faga clic na pestana do menú Configuración.
  4. Asegúrese de que a pestana do submenú CONFIGURACIÓN DO SISTEMA estea seleccionada.

img

Configurando axustes

Desprácese cara abaixo ata o título REGISTRO DE USUARIO.

Asegúrate de que o campo Permitir rexistro estea activado. (obrigatorio) Cando esta función está activada, Case Active xera unha ligazón única.

  1. Fai clic no campo Crear caso automaticamente e selecciona Activado. Aparece o campo Rol de usuario do caso.
  2. Seleccione un rol. (Ver abaixo)

img

Roles de usuario do caso

Hai tres roles de usuario de caso.

  • clientes (recomendado)
  • por defecto
  • persoal

Enviando a ligazón

Comparte a ligazón cunha ou máis persoas. Cada usuario que fai clic na ligazón e completa o rexistro activa a plataforma para xerar un novo caso na páxina de casos e asignar automaticamente o usuario no caso.
