Como xestionan os administradores as asignacións de usuarios aos casos

Hai cinco formas de xestionar usuarios e asignacións de casos. Fai clic nunha ligazón para saber como.

Método Tipo Accións Detalles
Mentres se crea un caso IU engadir Botón Administrador > Casos > Botón Novo > Botón Asignar usuarios
Asignación automática de usuario mediante a ligazón de rexistro público Acción do usuario engadir Botón de administración > Configuración > páxina Configuración do sistema
Usando a IU no caso existente IU engadir e eliminar Botón de administración > Casos > Asignar usuarios
Ferramenta de importación IU engadir, modificar e eliminar Botón de administración > Integracións > Importacións
Usando API (próximamente) API engadir, modificar e eliminar N / A
