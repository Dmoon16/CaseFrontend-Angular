ایڈمن کس طرح صارفین بناتے ہیں۔

ایڈمنز تخلیق، ترمیم یا غیر فعال کرتے ہیں جو ایپس اور کیسز تک رسائی حاصل کر سکتے ہیں۔ جب کوئی صارف پہلی بار پلیٹ فارم میں لاگ ان ہوتا ہے، تو اسے پروفائل ورک فلو کو مکمل کرنے کی ضرورت ہوتی ہے۔

  • نئے صارف پروفائل ورک فلو کے بارے میں مزید جاننے کے لیے یہاں کلک کریں۔

صارف بنانا

نیچے دی گئی جدول صارف بنانے کے پانچ طریقے دکھاتی ہے۔ طریقہ جاننے کے لیے لنک پر کلک کریں۔

اطلاعات

پلیٹ فارم صارفین کو کسی ایپ میں شامل ہونے پر اطلاع نہیں بھیجتا ہے۔ تاہم، منتظمین کیس اسائنمنٹس کے لیے اطلاعات کو فعال کر سکتے ہیں۔
