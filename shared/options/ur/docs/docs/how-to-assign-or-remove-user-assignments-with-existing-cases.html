موجودہ کیسز کے ساتھ یوزر اسائنمنٹس کو کیسے تفویض یا ہٹایا جائے۔

یہ مضمون منتظمین کو دکھاتا ہے کہ کیس صارفین کو تفویض اور ہٹانے کا طریقہ۔ ہم آپ کو یہ بھی دکھاتے ہیں کہ صارف کے ڈیٹا کو کیسے تبدیل کیا جائے۔

موجودہ کیس کھولنا

  1. ایڈمن اکاؤنٹ میں لاگ ان کریں۔
  2. ایپس پیج پر ایک ایپ تلاش کریں اور ایڈمن بٹن پر کلک کریں۔
  3. کیسز مینو ٹیب پر کلک کریں۔
  4. یقینی بنائیں کہ ذیلی مینیو میں ACTIVE منتخب کیا گیا ہے۔
  5. نیچے ACTION کالم تک سکرول کریں۔

تفویض کریں یا ہٹا دیں۔

صارف کو کیس میں تفویض کرنے یا صارف کو کیس سے ہٹانے کے لیے صارفین کو تفویض کریں پر کلک کریں (لیکن پلیٹ فارم سے نہیں)۔ اگلے صفحے پر: ADD پر کلک کریں اور ورک فلو کو مکمل کریں۔ صارف کو حذف کرنے کے لیے ہٹائیں پر کلک کریں۔

img

ترمیم کریں۔

کیسز کے صفحہ پر صارف کے ڈیٹا میں ترمیم یا اپ ڈیٹ کرنا ممکن نہیں ہے۔ اس کے بجائے، ان اقدامات پر عمل کریں:

  1. یوزر مینو ٹیب پر کلک کریں۔
  2. صارف کا نام تلاش کریں اور ترمیم پر کلک کریں۔
  3. تبدیلیاں کریں اور محفوظ کریں پر کلک کریں۔
