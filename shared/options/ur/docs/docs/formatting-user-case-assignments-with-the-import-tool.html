case_ID

  • کیس صارف یا ٹیگ ID درج کریں۔
  • کیسز صفحہ > ترمیم > ٹیگ ID پر جائیں یا url میں کیسز صفحہ > ترمیم > کیس ID پر جائیں۔

صارف کی شناخت

  • صارف کی شناخت یا ٹیگ ID درج کریں۔
  • یوزر پیج> ایڈیٹ> ٹیگ آئی ڈی پر جائیں یا یو آر ایل میں یوزر پیج> ایڈیٹ> یوزر آئی ڈی پر جائیں۔
