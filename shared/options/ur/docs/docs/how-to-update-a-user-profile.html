صارف پروفائل کو کیسے اپ ڈیٹ کریں۔

یہ مضمون صارفین کو اپنے پروفائل کے صفحہ پر معلومات کو اپ ڈیٹ کرنے کا طریقہ دکھاتا ہے۔

  • صارفین اپنے پروفائلز کو اپ ڈیٹ کر سکتے ہیں۔
  • منتظمین کسی بھی صارف کے پروفائل پیج کو اپ ڈیٹ کر سکتے ہیں۔

صارفین کے لیے

پہلا قدم پروفائل کا صفحہ کھولنا ہے۔ صارفین اور منتظمین اپنے پروفائل پیج کو اپ ڈیٹ کرنے کے لیے ان اقدامات پر عمل کریں۔

  1. اپنے کیس ایکٹو اکاؤنٹ میں لاگ ان کریں۔
  2. صفحہ کے اوپری حصے میں پروفائل ٹیب پر کلک کریں۔
  3. تبدیلیاں کرنا.
  4. محفوظ کریں پر کلک کریں۔

زیادہ تر فیلڈز اختیاری ہیں، اور کچھ فیلڈز درکار ہیں۔ مطلوبہ فیلڈز ہیں:

  • پہلا نام
  • آخری نام
  • ای میل اڈریس
  • زبان
  • ٹائم زون

ایڈمنز کے لیے

منتظمین دوسرے صارفین کے لیے پروفائل کی معلومات کو بھی اپ ڈیٹ کر سکتے ہیں۔ اس میں HOST USER INFO عنوان کے تحت تین فیلڈز شامل ہیں۔

  1. اپنے کیس ایکٹو ایڈمن اکاؤنٹ میں لاگ ان کریں۔
  2. ایک ایپ تلاش کریں اور ایڈمن بٹن پر کلک کریں۔
  3. اوپر والے مینو میں صارفین پر کلک کریں۔
  4. ذیلی مینو پر جائیں اور اس صارف کی فہرست کو منتخب کریں جس تک آپ رسائی حاصل کرنا چاہتے ہیں (فعال یا غیر فعال)۔ پہلے سے طے شدہ ترتیب ایکٹو ہے۔
  5. فہرست میں صارف کو تلاش کریں اور ترمیم پر کلک کریں۔
  6. ضرورت کے مطابق تبدیلیاں کریں۔ (ذیل میں دیکھیں)
  7. محفوظ کریں پر کلک کریں۔

صارف کی معلومات کو تبدیل کریں۔

منتظمین صارف کی معلومات کو تبدیل کر سکتے ہیں۔

  1. صارف کی قسم (صارف یا مینیجر) مقرر کریں۔
  2. یوزر اسٹیٹس سیٹ کریں (ایکٹو ٹو ڈس ایبلڈ)۔
  3. ایک منفرد ٹیگ ID درج کریں۔ (اختیاری)
  4. محفوظ کریں پر کلک کریں۔
