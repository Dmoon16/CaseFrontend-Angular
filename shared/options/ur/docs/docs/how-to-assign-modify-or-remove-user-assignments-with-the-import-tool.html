درآمدی ٹول کے ساتھ صارف کی اسائنمنٹس کو تفویض، ترمیم یا ہٹانے کا طریقہ

یہ مضمون منتظمین کو دکھاتا ہے کہ درآمدی ٹول کے ساتھ کیس صارفین کا نظم کیسے کریں۔

منتظمین کر سکتے ہیں:

  • صارفین کو کیس میں تفویض کریں۔
  • ایک کیس کے لیے پہلے سے تفویض کردہ صارفین میں ترمیم کریں۔
  • کیس میں تفویض کردہ صارفین کو ہٹا دیں۔

امپورٹ ٹول کھولنا

  1. ایڈمن اکاؤنٹ میں لاگ ان کریں۔
  2. ایپس پیج پر ایک ایپ تلاش کریں اور ایڈمن بٹن پر کلک کریں۔
  3. انٹیگریشن ٹیب پر کلک کریں اور پھر سب مینیو پر امپورٹس پر کلک کریں۔
  4. نیو بٹن پر کلک کریں۔ درآمدات شامل کریں ڈائیلاگ باکس دکھاتا ہے۔

ریکارڈ بنانا

ایک یا زیادہ CSV فائلیں ڈاؤن لوڈ کریں۔ کیس صارفین کو شامل کرنے، ترمیم کرنے یا ہٹانے کے لیے ان فائلوں کا استعمال کریں۔ درآمدات شامل کریں ڈائیلاگ باکس کھلنے کے بعد:

  1. ٹائپ فیلڈ پر کلک کریں۔
  2. ڈراپ ڈاؤن مینو میں اسکرول کریں اور ایک آپشن منتخب کریں (نیچے دیکھیں)۔ ایک لنک دکھاتا ہے۔
  3. ڈاؤن لوڈ نمونہ لنک پر کلک کریں۔ ایک CSV فائل آپ کے آلے پر ڈاؤن لوڈ ہوتی ہے۔

img

فائل کھولیں اور ریکارڈز شامل کریں۔ CSV فائل میں فارمیٹ ڈیٹا استعمال کرنے کا طریقہ جاننے کے لیے یہاں کلک کریں۔

فائل کے اختیارات

صارفین کو کیسز کے لیے تفویض کریں * صارفین کو کیس میں شامل کرنے کے لیے اس آپشن کا انتخاب کریں۔ * کیس_رول_آرڈر سیل کو خالی چھوڑ دیں۔ * یوزر پیج پر جائیں اور اس شخص کے لیے ٹیگ آئی ڈی کاپی کریں۔ اگر ٹیگ ID کا فیلڈ خالی ہے تو، ایک منفرد ID درج کریں، محفوظ کریں پر کلک کریں، اور پھر ID کو CSV فائل میں کاپی کریں۔

کیسز کے لیے تفویض کردہ صارفین میں ترمیم کریں * کیس کے لیے تفویض کردہ موجودہ صارفین کے ڈیٹا کو اپ ڈیٹ کرنے کے لیے اس آپشن کا انتخاب کریں۔ * کیس_ID کالم کے لیے کیس ٹیگ ID استعمال کریں۔

فائل اپ لوڈ ہو رہی ہے۔

آپ فوراً فائل اپ لوڈ کر سکتے ہیں یا بعد کی تاریخ میں ایڈ امپورٹس ڈائیلاگ باکس میں واپس جا کر ان مراحل کو مکمل کر سکتے ہیں۔

  1. درآمدات شامل کریں ڈائیلاگ باکس کھولیں۔
  2. CSV فائل کے لیے ایک منفرد نام درج کریں۔
  3. اپنے آلے سے فائل اپ لوڈ کرنے کے لیے پلس (+) آئیکن پر کلک کریں۔
  4. ٹائپ فیلڈ پر کلک کریں اور ایک آپشن منتخب کریں۔ (اوپر ملاحظہ کریں)
  5. محفوظ کریں پر کلک کریں۔

پروسیسنگ

فائل اور سسٹم لوڈ میں ریکارڈز کی تعداد پر منحصر ہے، پلیٹ فارم کو درآمد پر کارروائی کرنے میں کئی منٹ لگ سکتے ہیں۔

img

جب درآمد مکمل ہو جاتی ہے، تو پلیٹ فارم کیسز کے صفحہ پر صارف کے ریکارڈز کو ایڈجسٹ کرتا ہے (یعنی جوڑتا ہے، ترمیم کرتا ہے یا ہٹاتا ہے)۔

ناکامی

اگر امپورٹ آپریشن ناکام ہوجاتا ہے، تو پلیٹ فارم ایک ناکام پیغام لوٹاتا ہے۔ زیادہ تر معاملات میں، ناکامی کی وجہ ڈیٹا فارمیٹنگ ہے۔

img

ناکامی کی وجہ جاننے کے لیے:

  1. ایکشن کے عنوان کے تحت دیکھیں پر کلک کریں۔
  2. اگلے صفحے پر، ناکام آئیکن پر ماؤس ہوور کریں۔ ٹول ٹپ غلطی کو ظاہر کرتا ہے۔
  3. CVS فائل پر جائیں اور غلطی کو ایڈجسٹ اور ٹھیک کریں۔
  4. فائل کو دوبارہ اپ لوڈ کریں۔
