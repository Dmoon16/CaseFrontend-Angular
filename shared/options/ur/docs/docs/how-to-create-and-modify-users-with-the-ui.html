UI کے ساتھ صارفین کو کیسے بنائیں اور ان میں ترمیم کریں۔

یہ مضمون منتظمین کو دکھاتا ہے کہ پلیٹ فارم کے صارف انٹرفیس کے ساتھ صارفین کو کیسے بنایا جائے اور ریکارڈ کو اپ ڈیٹ کیا جائے۔ ایڈمنز صارفین کو نئے کیس میں شامل کر سکتے ہیں اور موجودہ کیسز میں ترمیم کر سکتے ہیں۔

صارف بنانا

  1. ایڈمن اکاؤنٹ میں لاگ ان کریں۔
  2. ایپس پیج پر ایک ایپ تلاش کریں اور ایڈمن بٹن پر کلک کریں۔
  3. ایڈمن پینل پر، یوزرز مینو ٹیب پر کلک کریں۔
  4. نیو بٹن پر کلک کریں۔ ایک نئی ونڈو دکھائی دیتی ہے۔

صارف کو تبدیل کرنا

صارف کے ریکارڈ کی ایک چھوٹی سی تعداد کو اپ ڈیٹ کرنے کے لیے، ترمیم کا ٹول استعمال کریں اور ہر ایک صارف میں تبدیلیاں کریں۔

  • نوٹ ایک ہی نشست میں صارف کے ریکارڈ کی ایک بڑی تعداد میں ترمیم کرنے کے لیے امپورٹ ٹول استعمال کرنے پر غور کریں۔
  1. ایڈمن پینل پر، یوزرز مینو ٹیب پر کلک کریں۔
  2. صارف کو تلاش کریں اور ترمیم پر کلک کریں۔
  3. تبدیلیاں کریں اور محفوظ کریں پر کلک کریں۔

اگلے مراحل

کیس ایکٹو صارف کو ای میل اطلاع نہیں بھیجتا ہے۔ منتظمین کو صارف سے رابطہ کرنا چاہیے اور انہیں کیس ایکٹو ہوم پیج پر ایک لنک بھیجنا چاہیے۔ اس کے بعد صارف LOGIN بٹن پر کلک کرتا ہے اور نئے سائن اپس کے لیے ورک فلو کو مکمل کرتا ہے۔

  • یہ مضمون نئے صارفین کے لیے سائن اپ ورک فلو کی وضاحت کرتا ہے۔
