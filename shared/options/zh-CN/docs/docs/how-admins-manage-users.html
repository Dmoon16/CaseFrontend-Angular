管理员如何管理用户

管理员创建、修改或禁用谁可以访问应用程序和案例。当用户首次登录平台时，他们需要完成一个配置文件工作流程。

  • 单击此处了解有关新用户配置文件工作流程的更多信息。

管理用户

下表显示了五种管理用户的方法。单击链接了解操作方法。

方法类型动作细节
使用界面用户界面添加和修改管理按钮 > 用户
创建案例并将用户分配给案例时用户界面添加管理按钮 > 案例 > 新建按钮 > 创建案例页面
使用公共注册链接自动创建用户用户操作添加管理按钮 > 设置 > 系统设置页面
导入工具用户界面添加和修改管理按钮 > 集成 > 导入
使用 API（即将推出）应用程序接口添加和修改不适用

通知

当用户被添加到应用程序时，该平台不会向用户发送通知。但是，管理员可以为个案分配启用通知。
