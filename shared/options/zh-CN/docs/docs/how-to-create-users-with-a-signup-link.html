如何使用注册链接创建用户

本文向管理员展示了如何通过发送注册链接来创建用户。

配置注册链接

  1. 单击添加用户对话框底部的向他们发送注册链接。显示一个新页面。
  2. 配置注册选项。 （见下文）

图片

确保允许注册字段设置为开。 （必需）启用后，Case Active 会生成一个唯一链接。

查看以下选项，然后单击图标以复制链接。您可以将链接放在您的网站上，或通过电子邮件或任何通信渠道将链接发送给一个或多个人。

图片

创建案例

或者，您可以为每个新用户自动创建案例并分配用户角色。

  1. 单击“自动创建案例”字段并选择“打开”。个案用户角色字段显示。
  2. 单击案例用户角色并选择一个选项。 （见下文）

图片

个案用户角色

存在三种案例用户角色。

  • 客户
  • 默认
  • 职员

发送链接

与一个或多个人共享链接。每个人都单击链接并完成工作流程以激活他们的 Case Active 帐户。用户完成注册工作流程后，Case Active 会将用户添加到用户列表中。

图片

下一步

Case Active 向用户发送带有注册链接的电子邮件通知。用户单击链接并完成新注册的工作流程。
