如何使用导入工具创建用户

本文向管理员展示了如何使用导入工具创建用户。

打开导入工具

管理员可以使用导入工具批量上传新用户和修改现有用户记录。

  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 在管理面板上，单击集成选项卡，然后单击子菜单上的导入。
  4. 单击新建按钮。添加导入对话框显示在新的浏览器选项卡中。

图片

创建记录

添加导入对话框打开后：

  1. 单击类型字段并选择一个选项（见下文）。显示一个链接。
  2. 单击下载示例链接。 CSV 文件将下载到您的设备。

接下来，打开文件并添加记录。单击此处了解如何在 CSV 文件中使用格式数据。

图片

类型

单击“类型”字段时，将显示一个下拉菜单。选择批量导入的两个选项之一。

:exclamation:注意下拉菜单有很多选项，但只有两个与导入工具相关。每个选项都有一个不同的示例 CSV 文件。

添加用户

  • 选择此选项可简单地批量上传用户记录。
  • 单击“下载示例”链接以打开一个 CSV 文件，其中包含“添加用户”导入类型的列标题。

修改用户

  • 选择此选项可为当前用户更新数据。
  • 单击下载示例链接打开一个 CSV 文件，其中列标题为修改用户导入类型。

上传文件

您可以立即上传文件或稍后返回添加导入对话框并完成这些步骤。

  1. 打开添加导入对话框。
  2. 输入 CSV 文件的唯一名称。
  3. 单击加号 (+) 图标从您的设备上传文件。
  4. 单击类型字段并选择一个选项。 （看上面）
  5. 单击保存。

加工

根据文件中记录的数量和系统负载，平台可能需要几分钟时间来处理导入。

图片

导入完成后，平台会在管理员的用户仪表板上调整用户记录（即添加、修改或禁用）。

失败

如果导入操作失败，平台将返回失败消息。在大多数情况下，失败的原因是数据格式化。

图片

查看失败原因：

  1. 单击“操作”标题下的“查看”。
  2. 在下一页上，将鼠标悬停在“失败”图标上。工具提示显示错误。
  3. 转到 CVS 文件并调整并修复错误。
  4. 再次上传文件。
