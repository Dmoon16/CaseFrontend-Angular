使用导入工具格式化用户管理数据

以下是 CSV 文件中的数据字段列表。每个导入选项都有不同的字段组合。

用户名

  • 输入用户的电子邮件地址。

给定的名称

  • 输入用户的名字。

姓

  • 输入用户的姓氏。

语言环境

  • 此列检查用户的首选语言。
  • 输入 en 表示英语，sp 表示西班牙语。

区域信息

  • 这将设置用户时区。输入没有空格的大陆和城市（使用下划线）。
  • 必须使用配置文件 > 语言和地区 > 时区下拉菜单中的选项。
  • 使用最合适的时区。例如，波士顿不在下拉列表中，因此我们可以使用纽约（或 EST/EDT 时区中的任何城市）。

host_tag_id

  • 输入永久标签。
  • 允许使用 az、0-9、-、/。

案例状态

  • 0 = 关闭
  • 1 = 活跃
