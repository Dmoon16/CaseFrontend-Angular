case_ID

  • Shigar da mai amfani ko Tag ID.
  • Je zuwa shafi > Shirya > Tag ID KO Je zuwa shafi > Shirya > ID na harka a url.

mai amfani_ID

  • Shigar da ID na mai amfani ko Tag ID.
  • Je zuwa shafin Masu amfani > Shirya > Tag ID KO Je zuwa shafin Masu amfani > Shirya > ID na mai amfani a url.
