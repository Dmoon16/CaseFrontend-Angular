Yadda ake Ƙirƙiri ko Gyara Harkoki Ta hanyar Menu na UI

Wannan labarin yana nuna admins yadda ake ƙirƙirar shari'a tare da mai amfani da dandamali. Admins kuma na iya ƙara masu amfani zuwa sabon harka kuma su sabunta shari'o'in da ke akwai.

Ƙirƙirar harka

  1. Shiga cikin asusun admin.
  2. Nemo app akan shafin Apps kuma danna maɓallin ADMIN.
  3. A cikin Admin panel, danna maballin menu na Cases.
  4. Danna NEW maballin. Wani sabon taga yana nuni.
