In che modo gli amministratori gestiscono le assegnazioni degli utenti ai casi

Esistono cinque modi per gestire gli utenti e le assegnazioni dei casi. Fare clic su un collegamento per sapere come.

Metodo Tipo Azioni Dettagli
Durante la creazione di un caso interfaccia utente Inserisci Pulsante amministratore > Casi > pulsante Nuovo > pulsante Assegna utenti
Assegna automaticamente l'utente utilizzando il link di registrazione pubblico Azione dell'utente Inserisci Pulsante amministratore > Impostazioni > pagina Configurazione di sistema
Utilizzo dell'interfaccia utente su un caso esistente interfaccia utente Aggiungi Rimuovi Pulsante amministratore > Casi > Assegna utenti
Strumento di importazione interfaccia utente aggiungi, modifica e rimuovi Pulsante amministratore > Integrazioni > Importazioni
Utilizzo dell'API (disponibile a breve) API aggiungi, modifica e rimuovi N / A
