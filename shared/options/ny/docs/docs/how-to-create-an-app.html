Momwe Mungapangire App

Nkhaniyi ikuwonetsa ogwiritsa ntchito atsopano momwe angapangire akaunti yoyeserera yaulere. Case Active imangopereka mwayi kwa wogwiritsa ntchito watsopano pa akauntiyi.

Kupanga akaunti

  1. Pitani patsamba lolowera .
  2. Dinani batani loyambira.
  3. Lowetsani imelo adilesi.
  4. Dinani Kenako.

Kutsimikizira akaunti

  1. Tsegulani tabu yatsopano ya msakatuli ndikulowa muakaunti yanu ya imelo.
  2. Koperani manambala asanu ndi limodzi.
  3. Bwererani ku tsamba la Case Active.
  4. Matani kachidindo m'minda. Zenera latsopano likuwonekera.

Pawindo latsopano:

  1. Lembani minda yofunikira.
  2. Werengani malamulo a kagwiritsidwe ntchito ndikusankha bokosi kuti muvomereze.
  3. Dinani PANGANI AKAUNTI YAULERE.

img

Kumaliza kukhazikitsa pulogalamu

Njira yogwirira ntchito ikuwonetsedwa patsamba lotsatira. Malizitsani masitepe kuti mumalize kukhazikitsa pulogalamu yanu. Mutha kubwereranso kumasamba awa nthawi zonse kuti musinthe kapena kusintha zambiri kapena mbiri yanu ya akaunti.

Patsamba la About App, lembani dzina la kampani ndi masamba awebusayiti (zofunikira) ndikudina Kenako.

Patsamba la Information Billing, dinani gawo la Mapulani a Malipiro ndikuwonanso zomwe mungasankhe. Zosintha zokhazikika ndi Zaulere. Dinani SUBMIT.

img

Mukatumiza fomu, Case Active imawonetsa tsamba la Mapulogalamu. Pa pulogalamu iliyonse yomwe mudapanga kapena kukhalamo, tsamba likuwonetsa:

  1. Dzina la app
  2. Mtundu Wanu (ntchito ya ogwiritsa)
  3. Mkhalidwe

img

Tsamba la Mapulogalamu ndi poyambira ntchito zambiri za admin:

  1. Dinani batani la APP kuti mulankhule, lembani zolemba, macheza amakanema (ndi zina) pazokhudza nkhani.
  2. Dinani batani la ADMIN kuti mutsegule gulu la admin. Apa ndi pomwe ma Admins amawongolera ogwiritsa ntchito, milandu, kuphatikiza, ndi zina zambiri.
  3. Dinani ellipsis kuti mukonze zolipirira akaunti ndi zidziwitso.
  • Note Case Active imawonetsa batani la ADMIN kwa ogwiritsa ntchito. Ogwiritsa ntchito omwe alibe mwayi wa admin samawona batani la ADMIN (monga momwe zikuwonekera pazenera pansipa).
