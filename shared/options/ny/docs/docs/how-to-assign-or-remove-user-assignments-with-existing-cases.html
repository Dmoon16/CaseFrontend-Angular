Momwe Mungagawire Kapena Kuchotsa Zochita Zogwiritsa Ntchito Zomwe Zilipo

Nkhaniyi ikuwonetsa ma admins momwe angagawire ndikuchotsa ogwiritsa ntchito. Tikuwonetsanso momwe mungasinthire deta ya ogwiritsa ntchito.

Kutsegula mlandu womwe ulipo

  1. Lowani ku akaunti ya admin.
  2. Pezani pulogalamu patsamba la Mapulogalamu ndikudina batani la ADMIN.
  3. Dinani Cases menyu tabu.
  4. Onetsetsani kuti ACTIVE yasankhidwa mu submenu.
  5. Pitani pansi mpaka pagawo la ACTION.

Perekani kapena chotsani

Dinani Patsani Ogwiritsa Ntchito kuti mugawire wogwiritsa ntchitoyo kapena muchotse wogwiritsa ntchitoyo (koma osati nsanja). Patsamba lotsatira: Dinani ADD ndi kumaliza ntchitoyo. Dinani Chotsani kuti mufufute wogwiritsa ntchito.

Sinthani

Sizingatheke kusintha kapena kusintha zambiri za ogwiritsa ntchito patsamba la Milandu. M'malo mwake, tsatirani izi:

  1. Dinani Ogwiritsa menyu tabu.
  2. Pezani dzina la wogwiritsa ntchito ndikudina Edit .
  3. Pangani zosintha ndikudina SAVE.
