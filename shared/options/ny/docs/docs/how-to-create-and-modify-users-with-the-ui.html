Momwe Mungapangire ndi Kusintha Ogwiritsa ndi UI

Nkhaniyi ikuwonetsa ma admins momwe angapangire ogwiritsa ntchito ndikusintha zolemba ndi mawonekedwe a nsanja. Ma Admins amathanso kuwonjezera ogwiritsa ntchito kunkhani yatsopano ndikusintha milandu yomwe ilipo.

Kupanga wosuta

  1. Lowani ku akaunti ya admin.
  2. Pezani pulogalamu patsamba la Mapulogalamu ndikudina batani la ADMIN.
  3. Pa gulu la Admin, dinani tabu la Ogwiritsa ntchito.
  4. Dinani batani la CHATSOPANO. Zenera latsopano likuwonekera.

Lembani minda ndikudina sungani. Case Active imawonjezera dzina pamndandanda wa ogwiritsa ntchito pulogalamuyi.

Kusintha wogwiritsa ntchito

Kuti musinthe zolemba zochepa za ogwiritsa ntchito, gwiritsani ntchito chida cha Edit ndikusintha kwa aliyense wogwiritsa ntchito.

  • Chidziwitso Kuti musinthe zolemba zambiri za ogwiritsa ntchito nthawi imodzi, lingalirani kugwiritsa ntchito chida cha Import .
  1. Pa gulu la Admin, dinani tabu la Ogwiritsa ntchito.
  2. Pezani wosuta ndikudina Edit.
  3. Pangani zosintha ndikudina SAVE.

Masitepe otsatira

Case Active situmiza uthenga wa imelo. Ma Admins akuyenera kulumikizana ndi wogwiritsa ntchito ndikuwatumizira ulalo wakutsamba loyambira la Case Active. Wogwiritsa ntchitoyo amadina batani la LOGIN ndikumaliza mayendedwe olembetsa atsopano.

  • Nkhaniyi ikufotokoza za kulembetsa kwa ogwiritsa ntchito atsopano.
