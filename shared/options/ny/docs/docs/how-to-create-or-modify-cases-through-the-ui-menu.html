Momwe Mungapangire Kapena Kusintha Milandu Kudzera pa UI Menu

Nkhaniyi ikuwonetsa ma admins momwe angapangire mlandu ndi mawonekedwe a nsanja. Ma Admins amathanso kuwonjezera ogwiritsa ntchito pamilandu yatsopano ndikusintha milandu yomwe ilipo.

Kupanga mlandu

  1. Lowani ku akaunti ya admin.
  2. Pezani pulogalamu patsamba la Mapulogalamu ndikudina batani la ADMIN.
  3. Pa gulu la Admin, dinani tabu ya Cases menyu.
  4. Dinani batani la CHATSOPANO. Zenera latsopano likuwonekera.

img

Lembani minda yofunikira ndikudina SAVE.

Kuwonjezera ogwiritsa

Mwachidziwitso, ma admins amatha kuwonjezera ogwiritsa ntchito patsamba lino. Pitani pansi ndikudina batani la ASSIGN USERS .

img

Mugawo la Perekani Wogwiritsa, lembani minda. Chongani m'bokosi kuti mutumize wosuta imelo chidziwitso. (posankha)

img

Kusintha mlandu

Kuti musinthe zochitika zingapo, gwiritsani ntchito chida cha Edit ndikusintha pazochitika zilizonse.

  • Chidziwitso Kuti musinthe zochitika zambiri nthawi imodzi, ganizirani kugwiritsa ntchito chida cha Import
  1. Lowani ku akaunti ya admin.
  2. Pezani pulogalamu patsamba la Mapulogalamu ndikudina batani la ADMIN.
  3. Pa gulu la Admin, dinani tabu ya Cases menyu.
  4. Pezani mlandu ndikudina Edit.
  5. Pangani zosintha ndikudina SAVE.
