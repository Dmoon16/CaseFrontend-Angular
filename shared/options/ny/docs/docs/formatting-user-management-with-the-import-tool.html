zoneinfo

  • Izi zimakhazikitsa nthawi yogwiritsa ntchito. Lowani kontinenti ndi mzinda wopanda malo (gwiritsani ntchito underscore).
  • Muyenera kugwiritsa ntchito zina zomwe zili mu Mbiri > LANGUAGE & REGION > menyu yotsikira ya Timezone.
  • Gwiritsani ntchito nthawi yoyenera kwambiri. Mwachitsanzo, Boston sali potsikira pansi, kotero titha kugwiritsa ntchito New York (kapena mzinda uliwonse wanthawi ya EST/EDT).
