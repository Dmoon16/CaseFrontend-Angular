Hvernig á að uppfæra notandasnið

Þessi grein sýnir notendum hvernig á að uppfæra upplýsingar á prófílsíðunni sinni.

  • Notendur geta uppfært eigin prófíla.
  • Stjórnendur geta uppfært prófílsíðu hvers notanda sem er.

Fyrir notendur

Fyrsta skrefið er að opna prófílsíðuna. Notendur og stjórnendur fylgja þessum skrefum til að uppfæra sína eigin prófílsíðu.

  1. Skráðu þig inn á Case Active reikninginn þinn.
  2. Smelltu á Profile flipann efst á síðunni.
  3. Gerðu breytingar.
  4. Smelltu á Vista.

Flestir reitir eru valfrjálsir og sumir reitir eru nauðsynlegir. Nauðsynlegir reitir eru:

  • Fyrsta nafn
  • Eftirnafn
  • Netfang
  • Tungumál
  • Tímabelti

Fyrir stjórnendur

Stjórnendur geta einnig uppfært prófílupplýsingarnar fyrir aðra notendur. Þetta felur í sér þrjá reiti undir fyrirsögninni HOST USER UPPLÝSINGAR.

  1. Skráðu þig inn á Case Active admin reikninginn þinn.
  2. Finndu forrit og smelltu á ADMIN hnappinn.
  3. Smelltu á Notendur í efstu valmyndinni.
  4. Farðu í undirvalmyndina og veldu notendalistann sem þú vilt fá aðgang að (VIRKUR eða Óvirkur). Sjálfgefin stilling er Virk.
  5. Finndu notanda á listanum og smelltu á Breyta.
  6. Gerðu breytingar eftir þörfum. (sjá fyrir neðan)
  7. Smelltu á Vista.

Breyta notendaupplýsingum

Stjórnendur geta breytt notendaupplýsingum.

  1. Stilltu notandagerð (notandi eða stjórnandi).
  2. Stilltu notandastöðu (virkur á óvirkan).
  3. Sláðu inn einstakt auðkenni merkis. (valfrjálst)
  4. Smelltu á Vista.
