Hvernig á að skrá þig inn

Lærðu hvernig á að skrá þig inn á núverandi Case Active reikning.

Innskráning

  1. Farðu á innskráningarsíðuna .
  2. Smelltu á LOGIN hnappinn.
  3. Sláðu inn netfangið þitt.
  4. Smelltu á NEXT.

Staðfestir reikning

  1. Opnaðu nýjan vafraflipa og skráðu þig inn á tölvupóstreikninginn þinn.
  2. Afritaðu sex stafa kóðann.
  3. Fara aftur í Case Active flipann.
  4. Límdu kóðann inn í reitina. Nýr gluggi birtist.

Case Active birtir forritasíðuna.

  • Athugið Pallurinn sýnir ADMIN hnapp við hlið APP hnappsins fyrir notendur með stjórnandaréttindi.
