Forsníða notendatilviksgögn með innflutningsverkfærinu

Hér að neðan er listi yfir gagnareiti í CSV skrám. Hver innflutningsvalkostur hefur mismunandi samsetningu af sviðum.

case_ID

  • Sláðu inn málið User eða Tag ID.
  • Farðu á Málasíðuna > Breyta > Merkjaauðkenni EÐA Farðu á Málasíðuna > Breyta > Málaauðkenni í vefslóð.

notandanafn

  • Sláðu inn auðkenni notanda eða auðkenni merkis.
  • Farðu á notendasíðuna > Breyta > Merkjaauðkenni EÐA Farðu á notendasíðuna > Breyta > Notandaauðkenni í vefslóð.

málshlutverksskipan

  • Skildu reitinn eftir auðan.

auðkenni_hlutverks

  • Sláðu inn: viðskiptavinir, starfsfólk eða sjálfgefið.

mál_tilkynna

  • 0 = nei
  • 1 = Já, sendu notanda tölvupósttilkynningu.
