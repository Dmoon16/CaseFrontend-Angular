Импорт куралы менен колдонуучунун дайындоолорун кантип ыйгаруу, өзгөртүү же алып салуу керек

Бул макалада администраторлорго Импорт куралы менен ишти колдонуучуларды кантип башкаруу керектиги көрсөтүлөт.

Админдер:

  • колдонуучуларды ишке дайындоо
  • мурунтан эле ишке дайындалган колдонуучуларды өзгөртүү
  • ишке дайындалган колдонуучуларды алып салуу

Импорт куралын ачуу

  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Интеграция өтмөгүн, андан кийин субменюдагы ИМПОРТ баскычын чыкылдатыңыз.
  4. ЖАҢЫ баскычын басыңыз. Импортту кошуу диалог кутусу көрсөтүлөт.

Жазууларды түзүү

Бир же бир нече CSV файлдарын жүктөп алыңыз. Колдонуучуларды кошуу, өзгөртүү же алып салуу үчүн бул файлдарды колдонуңуз. Импортту кошуу диалог кутусу ачылгандан кийин:

  1. Тип талаасын басыңыз.
  2. Ашылма менюну сыдырып, бир параметрди тандаңыз (төмөндө караңыз). Шилтеме көрсөтүлөт.
  3. Үлгү жүктөп алуу шилтемесин басыңыз. CSV файлы түзмөгүңүзгө жүктөлөт.

Файлды ачып, жазууларды кошуңуз. CSV файлында формат берилиштерин кантип колдонууну үйрөнүү үчүн бул жерди басыңыз.

Файл параметрлери

Колдонуучуларды кейстерге дайындоо * Колдонуучуларды ишке кошуу үчүн бул параметрди тандаңыз. * Case_role_order уячасын бош калтырыңыз. * Колдонуучулар барагына өтүп, бул адам үчүн тэг ID көчүрүңүз. Тег ID талаасы бош болсо, уникалдуу идентификаторду киргизип, Сактоо баскычын чыкылдатып, IDти CSV файлына көчүрүңүз.

Кейстерге дайындалган колдонуучуларды өзгөртүү * Ишке дайындалган учурдагы колдонуучулардын дайындарын жаңыртуу үчүн бул параметрди тандаңыз. * Case_ID тилкеси үчүн Case Tag ID колдонуңуз.

Файл жүктөлүүдө

Сиз файлды дароо жүктөй аласыз же кийинчерээк Импортту кошуу диалог кутусуна кайтып келип, бул кадамдарды аткара аласыз.

  1. Импортту кошуу диалог кутусун ачыңыз.
  2. CSV файлы үчүн уникалдуу атын киргизиңиз.
  3. Файлды түзмөгүңүздөн жүктөө үчүн плюс (+) белгисин басыңыз.
  4. Тип талаасын чыкылдатып, параметрди тандаңыз. (жогоруда кара)
  5. SAVE басыңыз.

Иштетүү

Файлдагы жана тутумдук жүктөгү жазуулардын санына жараша, платформа импортту иштетүү үчүн бир нече мүнөт талап кылынышы мүмкүн.

Импорттоо аяктагандан кийин, платформа Колдонуучунун жазууларын тууралайт (б.а. Кошуулар, өзгөртүүлөр же алып салуулар).

Ийгиликсиздик

Импорттоо операциясы ишке ашпай калса, платформа Failed билдирүүсүн кайтарат. Көпчүлүк учурларда, ката себеби маалыматтарды форматтоо болуп саналат.

Мүчүлүштүктүн себебин көрүү үчүн:

  1. Аракет рубрикасынын астындагы Көрүү дегенди басыңыз.
  2. Кийинки бетте чычканды Failed сөлөкөтүнө алып барыңыз. Курал кеңеши катаны көрсөтөт.
  3. CVS файлына өтүп, катаны тууралап, оңдоңуз.
  4. Файлды кайра жүктөңүз.
