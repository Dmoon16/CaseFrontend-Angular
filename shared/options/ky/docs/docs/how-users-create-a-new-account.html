Колдонуучулар жаңы аккаунтту кантип түзүшөт

Администратор колдонуучуну кейске кошкондон кийин: * эгерде колдонуучу caseactive үчүн жаңы болсо, колдонуучу каттоо эсебин түзүү үчүн иш процессин бүтүрүп, ишке кирүүгө чакырууну кабыл алышы керек. * эгерде колдонуучуда учурда активдүү аккаунт бар болсо, колдонуучу ишке кирүү үчүн чакырууну кабыл алышы керек.

Бул макалада колдонуучу өз каттоо эсебин түзүү үчүн аткарышы керек болгон кадамдарды сүрөттөйт.

Баштоо

Колдонуучуну caseactive колдонмосуна кошуунун эки жолу бар

  • Админ тарабынан: Эгерде администратор UI платформасы же Импорт куралы аркылуу колдонуучуну түзсө, админ кирүү үчүн колдонуучуга account.caseactive.com шилтемесин жөнөтүшү керек.
  • Колдонуучу тарабынан: Эгер колдонуучу катталуу шилтемеси аркылуу аккаунт түзсө, колдонуучу кирүү үчүн автоматтык түрдө account.caseactive.com дарегине багытталат.

Кирүү

Колдонуучу CaseActive каттоо эсебине биринчи жолу киргенде, платформа колдонуучуну төмөнкүнү сунуштайт:

  • алардын электрондук почта эсебин ачып, андан кийин алты орундуу коопсуздук кодун көчүрүп/чаптаңыз
  • эсеп профилин бүтүрүү

Колдонуучу UPDATE чыкылдаткандан кийин, Колдонмолор барагы көрсөтүлөт. Баракта колдонмонун аталышы (компаниянын аталышы), колдонуучунун түрү (рол), статусу жана ЧАКЫРУУ баскычы көрсөтүлөт.

img

Колдонмого кирүү үчүн колдонуучу чакырууну кабыл алышы керек. Колдонуучу керек:

  1. ЧАКЫРУУ баскычын басыңыз.
  2. Уруксаттарды карап чыгыңыз.
  3. Шарттарды карап чыгуу жана түзөтүү (милдеттүү эмес) үчүн Эскертмелер өтмөгүн басыңыз.
  4. КАТТАЛУУну басыңыз.

img

Колдонуучу жактырганда, платформа колдонуучунун бетинде APP баскычын көрсөтөт. Бул баскыч колдонуучуга дайындалган иштерге мүмкүнчүлүк берет.

Колдонуучунун жөндөөлөрүн өзгөртүү үчүн колдонуучу эллипсти басат.

Колдонуучу төмөнкүлөрдү жасай алат: * колдонмону алып салуу * эскертме жөндөөлөрүн өзгөртүү
