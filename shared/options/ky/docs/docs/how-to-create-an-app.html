Колдонмону кантип түзүү керек

Бул макалада жаңы колдонуучуларга кантип акысыз сыноо эсебин түзүү керектиги көрсөтүлөт. Case Active бул каттоо эсеби үчүн жаңы колдонуучуга администратор артыкчылыктарын автоматтык түрдө берет.

Каттоо эсебин түзүү

  1. Кирүү барагына өтүңүз.
  2. Баштоо баскычын басыңыз.
  3. Электрондук почта дарегин киргизиңиз.
  4. Next Click.

img

Каттоо эсеби текшерилүүдө

  1. Жаңы браузер өтмөгүн ачып, электрондук почтаңызга кириңиз.
  2. Алты орундуу кодду көчүрүңүз.
  3. Иштин активдүү өтмөгүнө кайтуу.
  4. Кодду талааларга чаптаңыз. Жаңы терезе пайда болот.

Жаңы терезеде:

  1. Керектүү талааларды толтуруңуз.
  2. Тейлөө шарттарын окуп, макул болуу үчүн кутучаны белгилеңиз.
  3. АКЫСЫЗ КАТ ТҮЗҮҮ дегенди басыңыз.

img

Колдонмону орнотуу аякталууда

Кийинки бетте иш процесси көрсөтүлөт. Колдонмоңузду орнотууну аяктоо үчүн кадамдарды аткарыңыз. Маалыматты же каттоо эсебиңиздин статусун түзөтүү же жаңыртуу үчүн бул беттерге каалаган убакта кайрыла аласыз.

Колдонмолор жөнүндө бетинде компаниянын атын жана веб-сайт талааларын (милдеттүү) толтуруп, андан кийин Кийинки баскычын чыкылдатыңыз.

Эсептөө маалыматы бетинде Төлөм планы талаасын чыкылдатып, параметрлерди карап чыгыңыз. Демейки жөндөө акысыз. ТАПСЫРУУ дегенди басыңыз.

img

Сиз форманы тапшыргандан кийин, Case Active Колдонмолор барагын көрсөтөт. Сиз түзгөн же таандык болгон ар бир колдонмо үчүн барак төмөнкүнү көрсөтөт:

  1. Колдонмонун аты
  2. Сиздин түрүңүз (колдонуучунун ролу)
  3. Статус

img

Колдонмолор барагы көптөгөн администратор функциялары үчүн ишке киргизүү чекити болуп саналат:

  1. Ишке байланыштуу маселелер боюнча баарлашуу, документтерди толтуруу, видео чат (ж.б.) үчүн APP баскычын чыкылдатыңыз.
  2. Админ панелин ачуу үчүн ADMIN баскычын басыңыз. Бул жерде Администраторлор колдонуучуларды, иштерди, интеграцияларды жана башкаларды башкарат.
  3. Каттоо эсебин жана эскертме жөндөөлөрүн башкаруу үчүн эллипсти басыңыз.
  • Note Case Active администратор колдонуучулар үчүн ADMIN баскычын көрсөтөт. Администратор артыкчылыктары жок колдонуучулар ADMIN баскычын көрүшпөйт (төмөнкү экранда көрсөтүлгөндөй).
