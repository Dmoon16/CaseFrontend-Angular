Импорт куралы менен Колдонуучуну башкаруу маалыматтарын форматтоо

Төмөндө CSV файлдарындагы маалымат талааларынын тизмеси. Ар бир импорттоо опциясында талаалардын ар кандай айкалышы бар.

колдонуучунун аты

  • Колдонуучунун электрондук почта дарегин киргизиңиз.

берилген аты

  • Колдонуучунун атын киргизиңиз.

фамилия_аты

  • Колдонуучунун фамилиясын киргизиңиз.

жергиликтүү тил

  • Бул тилке колдонуучунун каалаган тилин текшерет.
  • Англисче үчүн en, испанча үчүн sp.

zoneinfo

  • Бул колдонуучунун убакыт алкагын орнотот. Континентти жана шаарды боштуксуз киргизиңиз (астын сызыкты колдонуңуз).
  • Профиль > ТИЛ жана РЕГИОН > Убакыт алкагы ачылуучу менюсунда параметрди колдонушуңуз керек.
  • Эң ылайыктуу убакыт алкагын колдонуңуз. Мисалы, Бостон ачылуучу тизмеде эмес, ошондуктан биз Нью-Йоркту (же EST/EDT убакыт алкагындагы каалаган шаарды) колдоно алабыз.

host_tag_id

  • Туруктуу энбелги киргизиңиз.
  • аз, 0-9, -, / уруксат берилет.

case_status

  • 0 = жабык
  • 1 = активдүү
