Колдонуучунун профилин кантип жаңыртса болот

Бул макалада колдонуучуларга профиль барагындагы маалыматты кантип жаңыртуу керектиги көрсөтүлгөн.

  • Колдонуучулар өз профилдерин жаңырта алышат.
  • Админдер каалаган колдонуучунун Профиль барагын жаңырта алышат.

Колдонуучулар үчүн

Биринчи кадам - Профиль барагын ачуу. Колдонуучулар жана администраторлор өздөрүнүн Профиль барагын жаңыртуу үчүн бул кадамдарды аткарышат.

  1. Case Active аккаунтуңузга кириңиз.
  2. Барактын жогору жагындагы Профиль өтмөгүн чыкылдатыңыз.
  3. Өзгөртүүлөрдү киргизиңиз.
  4. Сактоо баскычын басыңыз.

Көпчүлүк талаалар милдеттүү эмес жана кээ бир талаалар талап кылынат. Талап кылынган талаалар:

  • Биринчи аты
  • Акыркы аты
  • Электрондук почта дареги
  • Тил
  • Убакыт зонасы

Админдер үчүн

Администраторлор башка колдонуучулар үчүн профиль маалыматын жаңырта алышат. Бул HOST USER INFO аталышындагы үч талааны камтыйт.

  1. Case Active администратор эсебиңизге кириңиз.
  2. Колдонмону таап, ADMIN баскычын чыкылдатыңыз.
  3. Жогорку менюдан Колдонуучуларды басыңыз.
  4. Суб-менюга өтүп, сиз киргиңиз келген колдонуучу тизмесин тандаңыз (АКТИВДҮҮ же ӨЧҮКТҮҮ). Демейки жөндөө Активдүү болуп саналат.
  5. Тизмеден колдонуучуну таап, Түзөтүүнү чыкылдатыңыз.
  6. Талапка жараша өзгөртүүлөрдү киргизиңиз. (төмөндө кара)
  7. Сактоо баскычын басыңыз.

Колдонуучунун маалыматын өзгөртүү

Администраторлор колдонуучу маалыматын өзгөртө алат.

  1. Колдонуучунун түрүн коюңуз (колдонуучу же менеджер).
  2. Колдонуучунун статусун коюңуз (Активден өчүрүлгөн).
  3. Уникалдуу Тег ID киргизиңиз. (милдеттүү эмес)
  4. Сактоо баскычын басыңыз.
