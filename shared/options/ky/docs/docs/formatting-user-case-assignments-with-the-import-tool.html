Импорт куралы менен Колдонуучунун Кейс дайындоолорун форматтоо

Төмөндө CSV файлдарындагы маалымат талааларынын тизмеси. Ар бир импорттоо опциясында талаалардын ар кандай айкалышы бар.

case_ID

  • Иштин Колдонуучусун же Тег ID киргизиңиз.
  • Иштер барагына өтүңүз > Түзөтүү > Тег ID ЖЕ url дарегинде Иштер барагына > Түзөтүү > Иштин идентификаторуна өтүңүз.

user_ID

  • Колдонуучунун ID же Тег ID киргизиңиз.
  • Колдонуучулар барагына өтүңүз > Түзөтүү > Тег ID ЖЕ Колдонуучулар барагына өтүңүз > Түзөтүү > URL'деги Колдонуучунун ID'си.

case_role_order

  • Клетканы бош калтырыңыз.

case_role_ID

  • Киргизүү: кардарлар, кызматкерлер же демейки.

case_notify

  • 0 = жок
  • 1 = Ооба, колдонуучуга электрондук почта билдирүүсүн жөнөтүү.
