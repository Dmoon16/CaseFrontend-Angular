Коомдук катталуу шилтемеси менен колдонуучуларды кантип автоматтык түрдө дайындоого болот

Администраторлор тутум орнотууларын кантип конфигурациялоону үйрөнүшөт, ошондо жаңы колдонуучулар автоматтык түрдө жаңы автоматтык түрдө түзүлгөн ишке дайындалышат.

Орнотуу барагын ачуу

Биринчи кадам - Системаларды орнотуу барагын ачуу.

  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Системалар өтмөгүн, андан кийин субменюдагы СИСТЕМАНЫ ОРНАТУ дегенди басыңыз.
  4. Колдонуучунун КАТТАЛУУ рубрикасына ылдый сыдырыңыз.

Системанын жөндөөлөрү конфигурацияланууда

  1. Катталууга уруксат берүү жана Ишти автоматтык түрдө түзүү ылдый түшүүчү менюларды күйгүзүңүз.
  2. Рол менюсун чыкылдатып, параметрди тандаңыз (демейки сунушталат).
  3. Коомдук колдонуучуга катталуу шилтемесин көчүрүп, бөлүшүңүз.

Шилтемени жөнөтүү

Шилтемени чыкылдатып, катталууну аяктаган ар бир колдонуучу, Cases бетинде жаңы ишти түзүү үчүн платформаны иштетет жана бул учурда колдонуучуну автоматтык түрдө дайындайт.
