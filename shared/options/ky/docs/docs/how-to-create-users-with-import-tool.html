Импорт куралы менен колдонуучуларды кантип түзүү керек

Бул макалада админдерге Импорт куралы менен колдонуучуларды кантип түзүү керектиги көрсөтүлөт.

Импорт куралын ачуу

Администраторлор Импорт куралын жаңы колдонуучуларды жапырт жүктөө жана учурдагы колдонуучу жазууларын өзгөртүү үчүн колдоно алышат.

  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Администратор панелинен Интеграция өтмөгүн, андан кийин субменюдагы ИМПОРТ баскычын чыкылдатыңыз.
  4. ЖАҢЫ баскычын басыңыз. Импортту кошуу диалог кутусу жаңы серепчинин өтмөгүндө көрсөтүлөт.

Жазууларды түзүү

Импортту кошуу диалог кутусу ачылгандан кийин:

  1. Тип талаасын чыкылдатып, параметрди тандаңыз (төмөндө караңыз). Шилтеме көрсөтүлөт.
  2. Үлгү жүктөө шилтемесин басыңыз. CSV файлы түзмөгүңүзгө жүктөлөт.

Андан кийин, файлды ачып, жазууларды кошуңуз. CSV файлында формат берилиштерин кантип колдонууну үйрөнүү үчүн бул жерди басыңыз.

Type

Тип талаасын басканда, ылдый түшүүчү меню пайда болот. Жаппай импорттоо үчүн эки варианттын бирин тандаңыз.

: Exclamation: ЭСКЕРТҮҮ Ашылма менюда көптөгөн параметрлер бар, бирок экөө гана Импорт куралына байланыштуу. Ар бир параметрде башка үлгү CSV файлы бар.

Колдонуучуларды кошуу

  • Колдонуучунун жазууларын жөнөкөй жапырт жүктөө үчүн бул параметрди тандаңыз.
  • Колдонуучуларды кошуу Импорт түрү үчүн тилке аталыштары менен CSV файлын ачуу үчүн Жүктөп алуу үлгүсүн басыңыз.

Колдонуучуларды өзгөртүү

  • Учурдагы колдонуучулардын дайындарын жаңыртуу үчүн бул параметрди тандаңыз.
  • Колдонуучулардын Импорт түрүн өзгөртүү үчүн тилке аталыштары менен CSV файлын ачуу үчүн Жүктөп алуу үлгүсүн басыңыз.

Файл жүктөлүүдө

Сиз файлды дароо жүктөй аласыз же кийинчерээк Импортту кошуу диалог кутусуна кайтып келип, бул кадамдарды аткара аласыз.

  1. Импортту кошуу диалог кутусун ачыңыз.
  2. CSV файлы үчүн уникалдуу атын киргизиңиз.
  3. Файлды түзмөгүңүздөн жүктөө үчүн плюс (+) белгисин басыңыз.
  4. Тип талаасын чыкылдатып, параметрди тандаңыз. (жогоруда кара)
  5. SAVE басыңыз.

Иштетүү

Файлдагы жана тутумдук жүктөгү жазуулардын санына жараша, платформа импортту иштетүү үчүн бир нече мүнөт талап кылынышы мүмкүн.

Импорттоо аяктагандан кийин, платформа администратордун Колдонуучулар тактасында колдонуучу жазууларын тууралайт (б.а. кошот, өзгөртөт же өчүрөт).

Ийгиликсиздик

Импорттоо операциясы ишке ашпай калса, платформа Failed билдирүүсүн кайтарат. Көпчүлүк учурларда, ката себеби маалыматтарды форматтоо болуп саналат.

Мүчүлүштүктүн себебин көрүү үчүн:

  1. Аракет рубрикасынын астындагы Көрүү дегенди басыңыз.
  2. Кийинки бетте чычканды Failed сөлөкөтүнө алып барыңыз. Курал кеңеши катаны көрсөтөт.
  3. CVS файлына өтүп, катаны тууралап, оңдоңуз.
  4. Файлды кайра жүктөңүз.
