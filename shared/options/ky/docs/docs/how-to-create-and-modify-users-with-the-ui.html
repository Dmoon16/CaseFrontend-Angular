UI менен колдонуучуларды кантип түзүү жана өзгөртүү керек

Бул макалада админдерге колдонуучуларды кантип түзүү жана платформанын колдонуучу интерфейси менен жазууларды жаңыртуу жолдору көрсөтүлөт. Администраторлор колдонуучуларды жаңы ишке кошуп, учурдагы иштерди түзөтө алышат.

Колдонуучу түзүү

  1. Админ каттоо эсебине кириңиз.
  2. Колдонмолор бетинен колдонмону табыңыз жана ADMIN баскычын басыңыз.
  3. Администратор панелинен Колдонуучулар менюсу өтмөгүн чыкылдатыңыз.
  4. ЖАҢЫ баскычын басыңыз. Жаңы терезе пайда болот.

img

Талааларды толтуруп, САКТОО баскычын басыңыз. Case Active атын бул колдонмо менен байланышкан колдонуучулардын тизмесине кошот.

Колдонуучуну өзгөртүү

Колдонуучунун аз сандагы жазууларын жаңыртуу үчүн, Түзөтүү куралын колдонуп, ар бир колдонуучуга өзгөртүү киргизиңиз.

  • Эскертүү Колдонуучунун көп сандагы жазууларын бир отурганда өзгөртүү үчүн Импорт куралын колдонуңуз .
  1. Администратор панелинен Колдонуучулар менюсу өтмөгүн чыкылдатыңыз.
  2. Колдонуучуну таап, Түзөтүүнү басыңыз.
  3. Өзгөртүүлөрдү киргизип, SAVE баскычын басыңыз.

Кийинки кадамдар

Case Active колдонуучуга электрондук почта билдирүүсүн жөнөтпөйт. Админдер колдонуучу менен байланышып, аларга Case Active башкы бетине шилтеме жөнөтүшү керек. Андан кийин колдонуучу LOGIN баскычын чыкылдатып, жаңы катталуулар үчүн иш процессин аяктайт.

  • Бул макалада жаңы колдонуучулар үчүн катталуу процесси сүрөттөлөт.
