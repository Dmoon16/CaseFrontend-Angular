Як автоматично призначати користувачів за допомогою загальнодоступного посилання для реєстрації

Адміністратори дізнаються, як налаштувати системні параметри, щоб нові користувачі автоматично призначалися щойно створеній справі.

Відкриття сторінки налаштування

Першим кроком є відкриття сторінки налаштування системи.

  1. Увійдіть в обліковий запис адміністратора.
  2. Знайдіть програму на сторінці програм і натисніть кнопку АДМІНІСТРАТОР.
  3. Клацніть вкладку «Системи», а потім «НАЛАШТУВАННЯ СИСТЕМИ» у підменю.
  4. Прокрутіть униз до заголовка РЕЄСТРАЦІЯ КОРИСТУВАЧА.

Налаштування параметрів системи

  1. Установіть для спадних меню «Дозволити реєстрацію» та «Автоматичне створення запиту» значення «Увімкнути».
  2. Клацніть меню ролі та виберіть параметр (рекомендовано за умовчанням).
  3. Скопіюйте та поділіться посиланням на реєстрацію публічного користувача.

Відправка посилання

Кожен користувач, який клацає посилання та завершує реєстрацію, запускає платформу для створення нового запиту на сторінці запитів і автоматично призначає користувача в цьому випадку.
