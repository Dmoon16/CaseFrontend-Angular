Форматування даних керування справами за допомогою засобу імпорту

Нижче наведено список обов’язкових полів у файлах CSV для керування справами за допомогою інструмента імпорту.

ім'я користувача

  • Введіть електронну адресу користувача.

tag_id (ідентифікатор, на який можна посилатися пізніше)

  • унікальний ідентифікатор (використовуйте буквено-цифрові символи, допускаються дефіси)

case_status

  • 0 = закрито
  • 1 = активний

дата_відкриття

  • місяць/день/рік
  • дд/мм/рррр

приблизно

  • унікальний опис справи (використовуйте буквено-цифрові символи, допускаються дефіси)

малюнок
