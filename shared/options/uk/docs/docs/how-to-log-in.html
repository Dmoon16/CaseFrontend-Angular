Як увійти

Дізнайтеся, як увійти до наявного облікового запису Case Active.

Вхід в систему

  1. Перейдіть на сторінку входу .
  2. Натисніть кнопку ВХІД.
  3. Введіть адресу електронної пошти.
  4. Натисніть ДАЛІ.

Підтвердження облікового запису

  1. Відкрийте нову вкладку браузера та увійдіть до свого облікового запису електронної пошти.
  2. Скопіюйте шестизначний код.
  3. Поверніться до вкладки Активний випадок.
  4. Вставте код у поля. Відобразиться нове вікно.

Case Active відображає сторінку програм.

  • Примітка На платформі відображається кнопка АДМІНІСТРАТОР поряд із кнопкою APP для користувачів із правами адміністратора.
