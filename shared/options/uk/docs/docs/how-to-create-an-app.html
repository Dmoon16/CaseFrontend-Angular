Як створити додаток

У цій статті показано, як новим користувачам створити безкоштовний пробний обліковий запис. Case Active автоматично надає новому користувачеві права адміністратора для цього облікового запису.

Створення облікового запису

  1. Перейдіть на сторінку входу .
  2. Натисніть кнопку Почати.
  3. Введіть адресу електронної пошти.
  4. Натисніть Далі.

Підтвердження облікового запису

  1. Відкрийте нову вкладку браузера та увійдіть до свого облікового запису електронної пошти.
  2. Скопіюйте шестизначний код.
  3. Поверніться до вкладки Активний випадок.
  4. Вставте код у поля. Відобразиться нове вікно.

У новому вікні:

  1. Заповніть необхідні поля.
  2. Прочитайте умови використання та поставте прапорець, щоб погодитися.
  3. Натисніть СТВОРИТИ БЕЗКОШТОВНИЙ ОБЛІКОВИЙ ЗАПИС.

Завершення налаштування програми

Робочий процес відображається на наступній сторінці. Виконайте кроки, щоб завершити налаштування програми. Ви завжди можете повернутися на ці сторінки, щоб відредагувати або оновити інформацію чи статус свого облікового запису.

На сторінці «Про програму» заповніть поля назви компанії та веб-сайту (обов’язково), а потім натисніть «Далі».

На сторінці «Платіжна інформація» клацніть поле «План платежу» та перегляньте параметри. За замовчуванням встановлено Безкоштовно. Натисніть НАДІСЛАТИ.

Після надсилання форми Case Active відобразить сторінку програм. Для кожної програми, яку ви створюєте або до якої належите, на сторінці показано:

  1. Назва програми
  2. Ваш тип (роль користувача)
  3. Статус

Сторінка програм є початковою точкою для багатьох функцій адміністратора:

  1. Натисніть кнопку APP, щоб спілкуватися, заповнювати документи, спілкуватися у відеочаті (і багато іншого) щодо питань, пов’язаних із справою.
  2. Натисніть кнопку АДМІНІСТРАТОР, щоб відкрити панель адміністратора. Тут адміністратори керують користувачами, справами, інтеграцією тощо.
  3. Клацніть три крапки, щоб керувати налаштуваннями виставлення рахунків і сповіщень облікового запису.
  • Примітка . Case Active відображає кнопку АДМІНІСТРАТОР для користувачів із правами адміністратора. Користувачі, які не мають прав адміністратора, не бачать кнопку АДМІНІСТРАТОР (як показано на екрані нижче).
