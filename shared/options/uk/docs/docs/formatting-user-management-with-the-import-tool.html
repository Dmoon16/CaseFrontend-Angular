Форматування даних керування користувачами за допомогою засобу імпорту

Нижче наведено список полів даних у файлах CSV. Кожен параметр імпорту має різну комбінацію полів.

ім'я користувача

  • Введіть електронну адресу користувача.

дане_ім'я

  • Введіть ім'я користувача.

прізвище

  • Введіть прізвище користувача.

локаль

  • У цьому стовпці перевіряється обрана користувачем мова.
  • Введіть en для англійської, sp для іспанської.

зонаінфо

  • Це встановлює часовий пояс користувача. Введіть континент і місто без пробілів (використовуйте підкреслення).
  • Необхідно використовувати параметр у спадному меню «Профіль» > «МОВА ТА РЕГІОН» > «Часовий пояс».
  • Використовуйте найкращий часовий пояс. Наприклад, у розкривному меню немає Бостона, тому ми можемо використовувати Нью-Йорк (або будь-яке місто в часовому поясі EST/EDT).

host_tag_id

  • Введіть постійну мітку.
  • az, 0-9, -, / допускаються.

case_status

  • 0 = закрито
  • 1 = активний
