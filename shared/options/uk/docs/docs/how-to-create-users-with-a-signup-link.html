Як створити користувачів за допомогою посилання для реєстрації

У цій статті адміністраторам показано, як створювати користувачів, надсилаючи посилання для реєстрації.

Налаштування посилання для реєстрації

  1. Натисніть посилання « Надіслати їм заяву про реєстрацію » внизу діалогового вікна «Додати користувача». Відобразиться нова сторінка.
  2. Налаштуйте параметри реєстрації. (Дивіться нижче)

малюнок

Переконайтеся, що для поля « Дозволити реєстрацію» встановлено значення «Увімк.». (обов’язково) Якщо ввімкнено, Case Active генерує унікальне посилання.

Перегляньте наведені нижче параметри, а потім натисніть піктограму, щоб скопіювати посилання. Ви можете розмістити посилання на своєму веб-сайті або надіслати посилання одній чи кільком людям електронною поштою чи будь-яким каналом зв’язку.

малюнок

Створення кейса

За бажанням ви можете автоматично створити справу для кожного нового користувача та призначити роль користувача.

  1. Клацніть поле Auto Create Case і виберіть On. Відображаються поля «Роль користувача справи».
  2. Клацніть «Роль користувача справи» та виберіть потрібний параметр. (Дивіться нижче)

Ролі користувачів Case

Є три ролі користувача case.

  • клієнтів
  • за замовчуванням
  • персонал

Надсилання посилання

Поділіться посиланням з одним або кількома людьми. Кожна особа натискає посилання та завершує робочий процес, щоб активувати свій обліковий запис Case Active. Коли користувач завершить процес реєстрації, Case Active додасть користувача до списку користувачів.

малюнок

Наступні кроки

Case Active надсилає користувачеві сповіщення електронною поштою з посиланням для реєстрації. Користувач натискає посилання та завершує робочий процес для нових реєстрацій.

  • У цій статті описано процес реєстрації для нових користувачів.
