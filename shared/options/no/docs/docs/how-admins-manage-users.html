Hvordan administratorer administrerer brukere

Administratorer oppretter, endrer eller deaktiverer hvem som har tilgang til apper og saker. Når en bruker logger på plattformen for første gang, må de fullføre en profilarbeidsflyt.

  • Klikk her for å lære mer om den nye arbeidsflyten for brukerprofilen.

Administrere en bruker

Tabellen nedenfor viser fem måter å administrere en bruker på. Klikk på en lenke for å finne ut hvordan.

Metode Type Handlinger Detaljer
Bruker UI UI legge til og endre Admin-knapp > Brukere
Mens du oppretter en sak og tilordner bruker til sak UI legge til Admin-knapp > Saker > Ny knapp > Opprett sak-side
Opprett bruker automatisk ved hjelp av offentlig registreringslenke Brukerhandling legge til Admin-knapp > Innstillinger > Systemoppsett-siden
Importverktøy UI legge til og endre Admin-knapp > Integrasjoner > Importer
Bruker API (kommer snart) API legge til og endre N/A

Varsler

Plattformen sender ikke brukere et varsel når de legges til i en app. Administratorer kan imidlertid aktivere varsler for sakstildelinger.
