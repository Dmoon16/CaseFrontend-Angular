Hvordan oppdatere en brukerprofil

Denne artikkelen viser brukere hvordan de oppdaterer informasjon på profilsiden deres.

  • Brukere kan oppdatere sine egne profiler.
  • Administratorer kan oppdatere profilsiden til enhver bruker.

For brukere

Det første trinnet er å åpne profilsiden. Brukere og administratorer følger disse trinnene for å oppdatere sin egen profilside.

  1. Logg inn på din Case Active-konto.
  2. Klikk på Profil-fanen øverst på siden.
  3. Gjøre endringer.
  4. Klikk Lagre.

De fleste feltene er valgfrie, og noen felt er obligatoriske. De obligatoriske feltene er:

  • Fornavn
  • Etternavn
  • Epostadresse
  • Språk
  • Tidssone

For administratorer

Administratorer kan også oppdatere profilinformasjonen for andre brukere. Dette inkluderer tre felt under overskriften VERTBRUKERINFO.

  1. Logg på din Case Active-administratorkonto.
  2. Finn en app og klikk på ADMIN-knappen.
  3. Klikk på Brukere på toppmenyen.
  4. Gå til undermenyen og velg brukerlisten du vil ha tilgang til (AKTIV eller DEAKTIVERT). Standardinnstillingen er Aktiv.
  5. Finn en bruker på listen og klikk på Rediger.
  6. Gjør endringer etter behov. (se nedenfor)
  7. Klikk Lagre.

Endre brukerinformasjon

Administratorer kan endre brukerinformasjon.

  1. Angi brukertype (bruker eller leder).
  2. Angi brukerstatus (aktiv til deaktivert).
  3. Skriv inn en unik Tag-ID. (valgfri)
  4. Klikk Lagre.
