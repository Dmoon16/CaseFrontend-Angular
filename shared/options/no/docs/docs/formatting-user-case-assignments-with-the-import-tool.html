Formatering av brukertildelingsdata med importverktøyet

Nedenfor er en liste over datafelt i CSV-filer. Hvert importalternativ har en annen kombinasjon av felt.

case_ID

  • Skriv inn case User eller Tag ID.
  • Gå til Sakssiden > Rediger > Tag ID ELLER Gå til Sakssiden > Rediger > Saks-ID i url.

bruker-ID

  • Skriv inn brukerens ID eller Tag ID.
  • Gå til brukersiden > Rediger > Tag-ID ELLER Gå til brukersiden > Rediger > Bruker-ID i url.

case_rolle_order

  • La cellen stå tom.

saksrolle-ID

  • Angi: klienter, ansatte eller standard.

case_notify

  • 0 = nei
  • 1 = Ja, send brukeren et e-postvarsel.
