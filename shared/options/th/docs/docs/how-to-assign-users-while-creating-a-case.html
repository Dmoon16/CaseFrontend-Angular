วิธีกำหนดผู้ใช้ในขณะที่สร้างเคส

บทความนี้แสดงให้ผู้ดูแลระบบทราบวิธีกำหนดผู้ใช้ในขณะที่สร้างกรณีและปัญหาใหม่ หากผู้ใช้ยังไม่ได้ลงทะเบียนบนแพลตฟอร์ม คุณสามารถสร้างเพิ่มผู้ใช้ใหม่ในขณะที่สร้างกรณีใหม่ได้

การสร้างกรณี

  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่มผู้ดูแลระบบ
  3. คลิกแท็บเมนูกรณีและปัญหา
  4. คลิกปุ่มใหม่ หน้าต่างใหม่จะปรากฏขึ้น
  • กรอกข้อมูลในฟิลด์เพื่อสร้างกรณีและจากนั้นเลื่อนลงไปที่หัวข้อผู้ใช้ที่ได้รับมอบหมาย

การกำหนดผู้ใช้

ผู้ใช้ปัจจุบัน

ทำตามขั้นตอนเหล่านี้เพื่อกำหนดผู้ใช้ปัจจุบันให้กับเคสใหม่

คลิกปุ่มกำหนดผู้ใช้และกรอกข้อมูลในฟิลด์

  1. คลิกรายการดรอปดาวน์ผู้ใช้ แล้วเลือกตัวระบุ (เช่น อีเมล)
  2. ป้อนค่าในช่องว่าง (เช่น ที่อยู่อีเมล)
  3. หากมีเรกคอร์ดผู้ใช้อยู่ในแพลตฟอร์ม ฟิลด์จะแนะนำชื่อโดยอัตโนมัติ
  4. เลือกผู้ใช้ที่ถูกต้อง
  5. เลือกบทบาทของผู้ใช้ (ลูกค้า ค่าเริ่มต้น หรือพนักงาน)
  6. หรือเลือกช่องทำเครื่องหมายเพื่อส่งการแจ้งเตือนทางอีเมลถึงผู้ใช้
  7. คลิกบันทึก

Case Active เพิ่มผู้ใช้ในเคส ตอนนี้ ผู้ดูแลระบบสามารถคลิก บันทึก เพื่อปิดหน้าต่าง เพิ่มผู้ใช้ใหม่ หรือลบผู้ใช้

img

การเพิ่มผู้ใช้ใหม่

หากแพลตฟอร์มไม่พบบันทึกปัจจุบัน ให้คลิกปุ่มเพื่อเพิ่มผู้ใช้ใหม่

กรอกข้อมูลในฟิลด์ จากนั้นคลิก บันทึก เพื่อเพิ่มผู้ใช้ไปยังแพลตฟอร์ม ถัดไป ตั้งค่าบทบาทของผู้ใช้และคลิกบันทึกอีกครั้งเพื่อกำหนดผู้ใช้ให้กับกรณีและปัญหา สุดท้าย คลิก บันทึก อีกครั้งเพื่อบันทึกกรณีและปัญหา
