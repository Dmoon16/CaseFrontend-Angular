วิธีที่ผู้ใช้สร้างบัญชีใหม่

หลังจากผู้ดูแลระบบเพิ่มผู้ใช้ในเคส: * หากผู้ใช้ยังใหม่กับเคสแอ็กทีฟ ผู้ใช้ต้องทำเวิร์กโฟลว์ให้เสร็จสมบูรณ์เพื่อสร้างบัญชีและตอบรับคำเชิญเพื่อเข้าถึงเคส * หากผู้ใช้มีบัญชี caseactive ผู้ใช้จะต้องตอบรับคำเชิญเพื่อเข้าถึง case

บทความนี้อธิบายขั้นตอนที่ผู้ใช้ต้องทำเพื่อสร้างบัญชี

เริ่มต้นใช้งาน

มีสองวิธีในการเพิ่มผู้ใช้ไปยังแอปตัวพิมพ์เล็กและตัวพิมพ์ใหญ่

  • โดยผู้ดูแลระบบ: หากผู้ดูแลระบบสร้างผู้ใช้ผ่าน UI ของแพลตฟอร์ม หรือ เครื่องมือนำเข้า ผู้ดูแลระบบจะต้องส่งลิงก์ไปที่ account.caseactive.com ให้กับผู้ใช้เพื่อเข้าสู่ระบบต่อไป
  • โดยผู้ใช้: หากผู้ใช้สร้างบัญชีโดยใช้ ลิงก์ลงทะเบียน ผู้ใช้จะถูกเปลี่ยนเส้นทางโดยอัตโนมัติไปที่ account.caseactive.com เพื่อเข้าสู่ระบบต่อไป

เข้าสู่ระบบ

เมื่อผู้ใช้เข้าสู่ระบบบัญชี CaseActive เป็นครั้งแรก แพลตฟอร์มจะแจ้งให้ผู้ใช้:

  • เปิดบัญชีอีเมลของพวกเขา จากนั้นคัดลอก/วางรหัสความปลอดภัยหกหลัก
  • กรอกโปรไฟล์บัญชี

หลังจากที่ผู้ใช้คลิก UPDATE หน้าแอพจะปรากฏขึ้น หน้านี้จะแสดงชื่อแอป (ชื่อบริษัท) ประเภทผู้ใช้ (บทบาท) สถานะ และปุ่มคำเชิญ

img

ในการเข้าถึงแอป ผู้ใช้ต้องยอมรับคำเชิญ ผู้ใช้ควร:

  1. คลิกปุ่มคำเชิญ
  2. ตรวจสอบสิทธิ์
  3. คลิกแท็บการแจ้งเตือนเพื่อตรวจสอบและแก้ไข (ไม่บังคับ) ข้อกำหนด
  4. คลิกอนุมัติการเข้าถึง

img

เมื่อผู้ใช้อนุมัติ แพลตฟอร์มจะแสดงปุ่ม APP บนหน้าผู้ใช้ ปุ่มนี้ช่วยให้ผู้ใช้เข้าถึงกรณีและปัญหาที่กำหนด

หากต้องการแก้ไขการตั้งค่าผู้ใช้ ผู้ใช้คลิกที่จุดไข่ปลา

img

ผู้ใช้สามารถ: * ลบการเข้าถึงแอพ * แก้ไขการตั้งค่าการแจ้งเตือน

img

หลังจากที่ผู้ใช้ดำเนินการเวิร์กโฟลว์เสร็จสิ้น ชื่อผู้ใช้จะปรากฏในรายชื่อผู้ใช้ของผู้ดูแลระบบ รายการแสดงผู้ใช้ทั้งหมด เข้าสู่ระบบหรือไม่
