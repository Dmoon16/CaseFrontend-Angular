ผู้ดูแลระบบจัดการการกำหนดผู้ใช้ให้กับกรณีและปัญหาอย่างไร

มีห้าวิธีในการจัดการผู้ใช้และการกำหนดกรณีและปัญหา คลิกลิงก์เพื่อเรียนรู้วิธีการ

วิธี พิมพ์ การกระทำ รายละเอียด
ขณะที่ก่อคดี UI เพิ่ม ปุ่มผู้ดูแลระบบ > กรณีและปัญหา > ปุ่มใหม่ > ปุ่มกำหนดผู้ใช้
กำหนดผู้ใช้อัตโนมัติโดยใช้ลิงก์ลงทะเบียนสาธารณะ การกระทำของผู้ใช้ เพิ่ม ปุ่มผู้ดูแลระบบ > การตั้งค่า > หน้าการตั้งค่าระบบ
การใช้ UI กับกรณีที่มีอยู่ UI เพิ่มและลบ ปุ่มผู้ดูแลระบบ > กรณีและปัญหา > กำหนดผู้ใช้
เครื่องมือนำเข้า UI เพิ่ม แก้ไข และลบ ปุ่มผู้ดูแลระบบ > การผสานรวม > นำเข้า
การใช้ API (เร็วๆ นี้) เอพีไอ เพิ่ม แก้ไข และลบ ไม่มีข้อมูล
