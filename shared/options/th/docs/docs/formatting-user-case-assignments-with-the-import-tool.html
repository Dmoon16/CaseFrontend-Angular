การจัดรูปแบบข้อมูลการกำหนดกรณีผู้ใช้ด้วยเครื่องมือนำเข้า

ด้านล่างนี้คือรายการฟิลด์ข้อมูลในไฟล์ CSV ตัวเลือกการนำเข้าแต่ละรายการมีชุดข้อมูลที่แตกต่างกัน

case_ID

  • ป้อนกรณีผู้ใช้หรือ ID แท็ก
  • ไปที่หน้าเคส > แก้ไข > รหัสแท็ก หรือ ไปที่หน้าเคส > แก้ไข > รหัสเคสใน url

user_ID

  • ป้อนรหัสผู้ใช้หรือรหัสแท็ก
  • ไปที่หน้าผู้ใช้ > แก้ไข > รหัสแท็ก หรือไปที่หน้าผู้ใช้ > แก้ไข > รหัสผู้ใช้ใน url

case_role_order

  • ปล่อยเซลล์ว่างไว้

case_role_ID

  • ป้อน: ลูกค้า พนักงาน หรือค่าเริ่มต้น

case_notify

  • 0 = ไม่
  • 1 = ใช่ ส่งการแจ้งเตือนทางอีเมลให้ผู้ใช้
