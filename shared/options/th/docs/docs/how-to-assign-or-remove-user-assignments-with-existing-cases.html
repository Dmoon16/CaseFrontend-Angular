วิธีกำหนดหรือลบการกำหนดผู้ใช้ด้วยกรณีและปัญหาที่มีอยู่

บทความนี้แสดงให้ผู้ดูแลระบบทราบวิธีการกำหนดและลบผู้ใช้กรณีและปัญหา เรายังแสดงวิธีแก้ไขข้อมูลผู้ใช้

การเปิดเคสที่มีอยู่

  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่ม ADMIN
  3. คลิกแท็บเมนู กรณี และปัญหา
  4. ตรวจสอบให้แน่ใจว่าได้เลือก ACTIVE ในเมนูย่อย
  5. เลื่อนลงไปที่คอลัมน์ ACTION

กำหนดหรือลบ

คลิก กำหนดผู้ใช้ เพื่อกำหนดผู้ใช้ให้กับเคสหรือลบผู้ใช้ออกจากเคส (แต่ไม่ใช่แพลตฟอร์ม) ในหน้าถัดไป: คลิก เพิ่ม และดำเนินการเวิร์กโฟลว์ให้เสร็จสมบูรณ์ คลิก ลบ เพื่อลบผู้ใช้

img

แก้ไข

ไม่สามารถแก้ไขหรืออัปเดตข้อมูลผู้ใช้ในหน้ากรณีและปัญหาได้ ให้ทำตามขั้นตอนเหล่านี้แทน:

  1. คลิกแท็บเมนู ผู้ใช้
  2. ค้นหาชื่อผู้ใช้และคลิก แก้ไข
  3. ทำการเปลี่ยนแปลงและคลิกบันทึก
