วิธีสร้างและแก้ไขผู้ใช้ด้วย UI

บทความนี้แสดงวิธีสร้างผู้ใช้และอัปเดตเรกคอร์ดด้วยส่วนต่อประสานผู้ใช้ของแพลตฟอร์มแก่ผู้ดูแลระบบ ผู้ดูแลระบบยังสามารถเพิ่มผู้ใช้ในกรณีใหม่และแก้ไขกรณีที่มีอยู่

การสร้างผู้ใช้

  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่มผู้ดูแลระบบ
  3. ในแผงการดูแลระบบ คลิกแท็บเมนูผู้ใช้
  4. คลิกปุ่มใหม่ หน้าต่างใหม่จะปรากฏขึ้น

img

กรอกข้อมูลในช่องและคลิกบันทึก Case Active เพิ่มชื่อในรายชื่อผู้ใช้ที่เกี่ยวข้องกับแอพนี้

img

การปรับเปลี่ยนผู้ใช้

หากต้องการอัปเดตเรกคอร์ดผู้ใช้จำนวนเล็กน้อย ให้ใช้เครื่องมือแก้ไขและทำการเปลี่ยนแปลงกับผู้ใช้แต่ละคน

  • หมายเหตุ หากต้องการแก้ไขบันทึกผู้ใช้จำนวนมากในคราวเดียว ให้พิจารณาใช้ เครื่องมือนำเข้า
  1. ในแผงการดูแลระบบ คลิกแท็บเมนูผู้ใช้
  2. ค้นหาผู้ใช้และคลิกแก้ไข
  3. ทำการเปลี่ยนแปลงและคลิกบันทึก

img

ขั้นตอนถัดไป

Case Active ไม่ส่งการแจ้งเตือนทางอีเมลถึงผู้ใช้ ผู้ดูแลระบบควรติดต่อผู้ใช้และส่งลิงก์ไปยังหน้าแรกของ Case Active จากนั้นผู้ใช้คลิกปุ่ม เข้าสู่ระบบ และดำเนินการขั้นตอนการทำงานให้เสร็จสมบูรณ์สำหรับการลงชื่อสมัครใช้ใหม่

  • บทความนี้จะ อธิบายขั้นตอนการลงชื่อสมัครใช้สำหรับผู้ใช้ใหม่
