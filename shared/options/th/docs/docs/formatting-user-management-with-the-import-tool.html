การจัดรูปแบบข้อมูลการจัดการผู้ใช้ด้วยเครื่องมือนำเข้า

ด้านล่างนี้คือรายการฟิลด์ข้อมูลในไฟล์ CSV ตัวเลือกการนำเข้าแต่ละรายการมีชุดข้อมูลที่แตกต่างกัน

ชื่อผู้ใช้

  • ป้อนที่อยู่อีเมลของผู้ใช้

given_name

  • ป้อนชื่อผู้ใช้

นามสกุล

  • ป้อนนามสกุลของผู้ใช้

สถานที่

  • คอลัมน์นี้จะตรวจสอบภาษาที่ผู้ใช้ต้องการ
  • ป้อน en สำหรับภาษาอังกฤษ sp สำหรับภาษาสเปน

โซนอินโฟ

  • นี่เป็นการตั้งค่าเขตเวลาของผู้ใช้ ป้อนทวีปและเมืองโดยไม่ต้องเว้นวรรค (ใช้เครื่องหมายขีดล่าง)
  • ต้องใช้ตัวเลือกในเมนูแบบเลื่อนลง โปรไฟล์ > ภาษาและภูมิภาค > เขตเวลา
  • ใช้เขตเวลาที่เหมาะสมที่สุด ตัวอย่างเช่น บอสตันไม่ได้อยู่ในเมนูแบบเลื่อนลง เราจึงสามารถใช้นิวยอร์ก (หรือเมืองใดก็ได้ในเขตเวลา EST/EDT)

host_tag_id

  • ป้อนป้ายกำกับถาวร
  • อนุญาตให้ใช้ az, 0-9, -, /

กรณี_สถานะ

  • 0 = ปิด
  • 1 = ใช้งานอยู่
