วิธีสร้างหรือแก้ไขเคสด้วยเครื่องมือนำเข้า

บทความนี้แสดงวิธีสร้างหรือแก้ไขกรณีและปัญหาที่มีอยู่ด้วยเครื่องมือนำเข้าสำหรับผู้ดูแลระบบ

การเปิดเครื่องมือนำเข้า

  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่มผู้ดูแลระบบ
  3. ในแผงการดูแลระบบ ให้คลิกแท็บการรวม จากนั้นคลิกนำเข้าในเมนูย่อย
  4. คลิกปุ่มใหม่ กล่องโต้ตอบเพิ่มการนำเข้าจะแสดงในแท็บเบราว์เซอร์ใหม่

img

การสร้างกรณี

หลังจากกล่องโต้ตอบเพิ่มการนำเข้าเปิดขึ้น:

  1. คลิกช่อง Type แล้วเลือกตัวเลือก (ดูด้านล่าง) ลิงค์จะปรากฏขึ้น
  2. คลิกลิงค์ดาวน์โหลดตัวอย่าง ไฟล์ CSV จะดาวน์โหลดไปยังอุปกรณ์ของคุณ

จากนั้นเปิดไฟล์และเพิ่มกรณี

พิมพ์

เมื่อคุณคลิกฟิลด์ Type เมนูแบบเลื่อนลงจะปรากฏขึ้น เลือกหนึ่งในสองตัวเลือกสำหรับการนำเข้ากรณีจำนวนมาก

  • หมายเหตุ เมนูดร็อปดาวน์มีหลายตัวเลือก แต่มีเพียงสองตัวเลือกเท่านั้นที่เกี่ยวข้องกับการจัดการกรณีและปัญหา แต่ละตัวเลือกมีไฟล์ CSV ตัวอย่างที่แตกต่างกัน

เพิ่มกรณี

  • เลือกตัวเลือกนี้สำหรับการอัปโหลดกรณีและปัญหาจำนวนมากแบบง่ายๆ
  • คลิกลิงก์ดาวน์โหลดตัวอย่างเพื่อเปิดไฟล์ CSV ที่มีส่วนหัวของคอลัมน์สำหรับประเภทการนำเข้าสร้างกรณีและปัญหา

แก้ไขกรณี

  • เลือกตัวเลือกนี้เพื่ออัปเดตกรณีปัจจุบัน
  • คลิกลิงก์ดาวน์โหลดตัวอย่างเพื่อเปิดไฟล์ CSV ที่มีส่วนหัวของคอลัมน์สำหรับประเภทการนำเข้าแก้ไขกรณีและปัญหา

กำลังอัปโหลดไฟล์

คุณสามารถอัปโหลดไฟล์ได้ทันทีหรือกลับไปที่กล่องโต้ตอบเพิ่มการนำเข้าในภายหลังและดำเนินการตามขั้นตอนเหล่านี้

  1. เปิดกล่องโต้ตอบเพิ่มการนำเข้า
  2. ป้อนชื่อเฉพาะสำหรับไฟล์ CSV
  3. คลิกไอคอนเครื่องหมายบวก (+) เพื่ออัปโหลดไฟล์จากอุปกรณ์ของคุณ
  4. คลิกช่อง Type แล้วเลือกตัวเลือก (ดูด้านบน)
  5. คลิกบันทึก

การจัดรูปแบบ

นี่คือฟิลด์บังคับและข้อกำหนดแรก คลิก ที่นี่ เพื่อดูรายละเอียดการจัดรูปแบบ

tag_id (หรือ case_ID) ตัวระบุเฉพาะ (ใช้อักขระที่เป็นตัวอักษรและตัวเลขคละกัน ขีดกลางได้รับการยอมรับ)

case_status 0 = ปิด 1 = ใช้งานอยู่

date_opened เดือน/วัน/ปี mm/dd/yyyy

เกี่ยวกับ คำอธิบายที่ไม่ซ้ำกันของกรณี (ใช้อักขระที่เป็นตัวอักษรและตัวเลขคละกัน ยัติภังค์ได้)

img

กำลังประมวลผล

ขึ้นอยู่กับจำนวนของบันทึกในไฟล์และโหลดของระบบ แพลตฟอร์มอาจใช้เวลาหลายนาทีในการประมวลผลการนำเข้า

img

เมื่อการนำเข้าเสร็จสิ้น แพลตฟอร์มจะปรับเรกคอร์ดเคส (เช่น เพิ่มหรือแก้ไข) บนแดชบอร์ดเคสของผู้ดูแลระบบ

ความล้มเหลว

หากการดำเนินการนำเข้าล้มเหลว แพลตฟอร์มจะส่งคืนข้อความล้มเหลว ในกรณีส่วนใหญ่ สาเหตุของความล้มเหลวคือการจัดรูปแบบข้อมูล

img

หากต้องการดูสาเหตุของความล้มเหลว:

  1. คลิก ดู ใต้หัวข้อ ACTION
  2. ในหน้าถัดไป ให้เลื่อนเมาส์ไปที่ไอคอนล้มเหลว คำแนะนำเครื่องมือแสดงข้อผิดพลาด
  3. ไปที่ไฟล์ CVS แล้วปรับและแก้ไขข้อผิดพลาด
  4. อัปโหลดไฟล์อีกครั้ง
