วิธีสร้างแอป

บทความนี้แสดงวิธีสร้างบัญชีทดลองใช้ฟรีแก่ผู้ใช้ใหม่ Case Active ให้สิทธิ์ผู้ดูแลระบบแก่ผู้ใช้ใหม่สำหรับบัญชีนี้โดยอัตโนมัติ

การสร้างบัญชี

  1. ไปที่ หน้าเข้าสู่ระบบ
  2. คลิกปุ่มเริ่มต้น
  3. ป้อนที่อยู่อีเมล
  4. คลิกถัดไป

img

การยืนยันบัญชี

  1. เปิดแท็บเบราว์เซอร์ใหม่และลงชื่อเข้าใช้บัญชีอีเมลของคุณ
  2. คัดลอกรหัสหกหลัก
  3. กลับไปที่แท็บ Case Active
  4. วางรหัสลงในช่อง หน้าต่างใหม่จะปรากฏขึ้น

ในหน้าต่างใหม่:

  1. กรอกข้อมูลในฟิลด์บังคับ
  2. อ่านข้อกำหนดในการให้บริการและทำเครื่องหมายที่ช่องเพื่อยอมรับ
  3. คลิกสร้างบัญชีฟรี

เสร็จสิ้นการตั้งค่าแอป

ขั้นตอนการทำงานจะแสดงในหน้าถัดไป ทำตามขั้นตอนเพื่อสิ้นสุดการตั้งค่าแอปของคุณ คุณสามารถกลับมาที่หน้าเหล่านี้ได้ตลอดเวลาเพื่อแก้ไขหรืออัปเดตข้อมูลหรือสถานะบัญชีของคุณ

ในหน้า About App ให้กรอกชื่อบริษัทและช่องเว็บไซต์ (จำเป็น) จากนั้นคลิก Next

ในหน้าข้อมูลการเรียกเก็บเงิน คลิกฟิลด์แผนการชำระเงินและตรวจทานตัวเลือก การตั้งค่าเริ่มต้นคือ ฟรี คลิกส่ง

img

หลังจากที่คุณส่งแบบฟอร์ม Case Active จะแสดงหน้า Apps สำหรับแต่ละแอปที่คุณสร้างหรือเป็นเจ้าของ หน้านี้จะแสดง:

  1. ชื่อแอป
  2. ประเภทของคุณ (บทบาทของผู้ใช้)
  3. สถานะ

img

หน้าแอพเป็นจุดเริ่มต้นสำหรับฟังก์ชั่นการดูแลระบบมากมาย:

  1. คลิกปุ่มแอปเพื่อสื่อสาร กรอกเอกสาร วิดีโอแชท (และอื่นๆ) ในประเด็นที่เกี่ยวข้องกับกรณีและปัญหา
  2. คลิกปุ่มผู้ดูแลระบบเพื่อเปิดแผงผู้ดูแลระบบ นี่คือที่ที่ผู้ดูแลระบบจัดการผู้ใช้ กรณีและปัญหา การผสานรวม และอื่นๆ
  3. คลิกจุดไข่ปลาเพื่อจัดการการตั้งค่าการเรียกเก็บเงินและการแจ้งเตือนของบัญชี
  • หมายเหตุ Case Active แสดงปุ่ม ADMIN สำหรับผู้ใช้ที่เป็นผู้ดูแลระบบ ผู้ใช้ที่ไม่มีสิทธิ์ของผู้ดูแลระบบจะไม่เห็นปุ่ม ADMIN (ตามที่แสดงในหน้าจอด้านล่าง)
