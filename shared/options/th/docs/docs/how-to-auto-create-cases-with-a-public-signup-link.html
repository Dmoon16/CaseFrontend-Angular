วิธีสร้างกรณีอัตโนมัติด้วยลิงก์ลงทะเบียนสาธารณะ

บทความนี้แสดงวิธีสร้างกรณีและปัญหา (และเพิ่มผู้ใช้ในเวลาเดียวกัน) โดยส่งลิงก์สมัครเพียงลิงก์เดียวให้ผู้ดูแลระบบทราบ

ค้นหาลิงค์

  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่ม ADMIN
  3. ในแผงการดูแลระบบ คลิกแท็บเมนูการตั้งค่า
  4. ตรวจสอบให้แน่ใจว่าได้เลือกแท็บเมนูย่อยการตั้งค่าระบบแล้ว

img

การกำหนดการตั้งค่า

เลื่อนลงไปที่หัวข้อ USER SIGNUP

ตรวจสอบให้แน่ใจว่าตั้งค่าฟิลด์อนุญาตการลงทะเบียนเป็นเปิด (จำเป็น) เมื่อเปิดใช้งานคุณลักษณะนี้ Case Active จะสร้างลิงก์เฉพาะ

  1. คลิกช่องสร้างกรณีและปัญหาอัตโนมัติแล้วเลือกเปิด ฟิลด์ Case User Role จะปรากฏขึ้น
  2. เลือกบทบาท (ดูด้านล่าง)

img

บทบาทของผู้ใช้กรณี

มีบทบาทผู้ใช้สามกรณี

  • ลูกค้า (แนะนำ)
  • ค่าเริ่มต้น
  • พนักงาน

กำลังส่งลิงค์

แบ่งปันลิงก์กับบุคคลหนึ่งคนขึ้นไป ผู้ใช้แต่ละรายที่คลิกลิงก์และลงทะเบียนเสร็จสมบูรณ์จะทริกเกอร์แพลตฟอร์มเพื่อสร้างเคสใหม่ในหน้าเคสและกำหนดผู้ใช้ในเคสโดยอัตโนมัติ
