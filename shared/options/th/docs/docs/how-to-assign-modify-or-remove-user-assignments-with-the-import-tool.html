วิธีกำหนด แก้ไข หรือลบการกำหนดผู้ใช้ด้วยเครื่องมือนำเข้า

บทความนี้แสดงวิธีจัดการผู้ใช้กรณีและปัญหาด้วยเครื่องมือนำเข้าสำหรับผู้ดูแลระบบ

ผู้ดูแลระบบสามารถ:

  • กำหนดผู้ใช้ให้กับกรณีและปัญหา
  • แก้ไขผู้ใช้ที่กำหนดให้กับเคสแล้ว
  • ลบผู้ใช้ที่กำหนดให้กับกรณีและปัญหา

การเปิดเครื่องมือนำเข้า

  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่มผู้ดูแลระบบ
  3. คลิกแท็บการรวมแล้วคลิกนำเข้าในเมนูย่อย
  4. คลิกปุ่มใหม่ กล่องโต้ตอบเพิ่มการนำเข้าจะปรากฏขึ้น

การสร้างบันทึก

ดาวน์โหลดไฟล์ CSV อย่างน้อยหนึ่งไฟล์ ใช้ไฟล์เหล่านี้เพื่อเพิ่ม แก้ไข หรือลบผู้ใช้เคส หลังจากกล่องโต้ตอบเพิ่มการนำเข้าเปิดขึ้น:

  1. คลิกช่องประเภท
  2. เลื่อนดูเมนูแบบเลื่อนลงและเลือกตัวเลือก (ดูด้านล่าง) ลิงค์จะปรากฏขึ้น
  3. คลิกลิงค์ดาวน์โหลดตัวอย่าง ไฟล์ CSV จะดาวน์โหลดไปยังอุปกรณ์ของคุณ

img

เปิดไฟล์และเพิ่มบันทึก คลิกที่นี่เพื่อเรียนรู้วิธีใช้รูปแบบข้อมูลในไฟล์ CSV

ตัวเลือกไฟล์

กำหนดผู้ใช้ให้กับกรณีและปัญหา * เลือกตัวเลือกนี้เพื่อเพิ่มผู้ใช้ในกรณี * ปล่อยเซลล์ case_role_order ว่างไว้ * ไปที่หน้าผู้ใช้และคัดลอก ID แท็กสำหรับบุคคลนี้ หากช่อง ID แท็กว่างเปล่า ให้ป้อน ID เฉพาะ คลิกบันทึก จากนั้นคัดลอก ID ไปยังไฟล์ CSV

แก้ไขผู้ใช้ที่กำหนดให้กับกรณีและปัญหา * เลือกตัวเลือกนี้เพื่ออัปเดตข้อมูลสำหรับผู้ใช้ปัจจุบันที่กำหนดให้กับกรณีและปัญหา * ใช้รหัสแท็กกรณีสำหรับคอลัมน์ case_ID

img

ลบผู้ใช้ที่กำหนดให้กับกรณีและปัญหา * เลือกตัวเลือกนี้เพื่อลบผู้ใช้ที่กำหนดให้กับกรณีและปัญหา

กำลังอัปโหลดไฟล์

คุณสามารถอัปโหลดไฟล์ได้ทันทีหรือกลับไปที่กล่องโต้ตอบเพิ่มการนำเข้าในภายหลังและดำเนินการตามขั้นตอนเหล่านี้

  1. เปิดกล่องโต้ตอบเพิ่มการนำเข้า
  2. ป้อนชื่อเฉพาะสำหรับไฟล์ CSV
  3. คลิกไอคอนเครื่องหมายบวก (+) เพื่ออัปโหลดไฟล์จากอุปกรณ์ของคุณ
  4. คลิกช่อง Type แล้วเลือกตัวเลือก (ดูด้านบน)
  5. คลิกบันทึก

กำลังประมวลผล

ขึ้นอยู่กับจำนวนของบันทึกในไฟล์และโหลดของระบบ แพลตฟอร์มอาจใช้เวลาหลายนาทีในการประมวลผลการนำเข้า

เมื่อการนำเข้าเสร็จสิ้น แพลตฟอร์มจะปรับเรกคอร์ดผู้ใช้ (เช่น เพิ่ม แก้ไข หรือลบ) ในหน้ากรณีและปัญหา

ความล้มเหลว

หากการดำเนินการนำเข้าล้มเหลว แพลตฟอร์มจะส่งคืนข้อความล้มเหลว ในกรณีส่วนใหญ่ สาเหตุของความล้มเหลวคือการจัดรูปแบบข้อมูล

img

หากต้องการดูสาเหตุของความล้มเหลว:

  1. คลิกดูใต้หัวข้อการดำเนินการ
  2. ในหน้าถัดไป ให้เลื่อนเมาส์ไปที่ไอคอนล้มเหลว คำแนะนำเครื่องมือแสดงข้อผิดพลาด
  3. ไปที่ไฟล์ CVS แล้วปรับและแก้ไขข้อผิดพลาด
  4. อัปโหลดไฟล์อีกครั้ง
