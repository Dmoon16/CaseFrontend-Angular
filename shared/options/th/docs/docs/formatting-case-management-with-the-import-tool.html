การจัดรูปแบบข้อมูลการจัดการเคสด้วยเครื่องมือนำเข้า

ด้านล่างนี้คือรายการช่องที่ต้องกรอกในไฟล์ CSV เพื่อจัดการกรณีต่างๆ ด้วยเครื่องมือนำเข้า

ชื่อผู้ใช้

  • ป้อนที่อยู่อีเมลของผู้ใช้

tag_id (รหัสที่คุณสามารถอ้างอิงได้ในภายหลัง)

  • ตัวระบุที่ไม่ซ้ำกัน (ใช้อักขระที่เป็นตัวอักษรและตัวเลขคละกัน ขีดกลางได้รับการยอมรับ)

กรณี_สถานะ

  • 0 = ปิด
  • 1 = ใช้งานอยู่

date_opened

  • เดือนวันปี
  • มม/วว/ปปปป

เกี่ยวกับ

  • คำอธิบายที่ไม่ซ้ำกันของกรณี (ใช้อักขระที่เป็นตัวอักษรและตัวเลขคละกัน ขีดกลางได้รับการยอมรับ)
