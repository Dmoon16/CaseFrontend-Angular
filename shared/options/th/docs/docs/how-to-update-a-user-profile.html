วิธีอัปเดตโปรไฟล์ผู้ใช้

บทความนี้แสดงให้ผู้ใช้ทราบวิธีอัปเดตข้อมูลในหน้าโปรไฟล์

  • ผู้ใช้สามารถอัปเดตโปรไฟล์ของตนเองได้
  • ผู้ดูแลระบบสามารถอัปเดตหน้าโปรไฟล์ของผู้ใช้ใดก็ได้

สำหรับผู้ใช้

ขั้นตอนแรกคือการเปิดหน้าโปรไฟล์ ผู้ใช้และผู้ดูแลระบบทำตามขั้นตอนเหล่านี้เพื่ออัปเดตหน้าโปรไฟล์ของตนเอง

  1. เข้าสู่บัญชี Case Active ของคุณ
  2. คลิกแท็บโปรไฟล์ที่ด้านบนของหน้า
  3. ทำการเปลี่ยนแปลง
  4. คลิกบันทึก

ฟิลด์ส่วนใหญ่เป็นฟิลด์ที่ไม่บังคับ และบางฟิลด์จำเป็นต้องกรอก ช่องที่ต้องกรอกคือ:

  • ชื่อจริง
  • นามสกุล
  • ที่อยู่อีเมล
  • ภาษา
  • เขตเวลา

สำหรับผู้ดูแลระบบ

ผู้ดูแลระบบยังสามารถอัปเดตข้อมูลโปรไฟล์สำหรับผู้ใช้รายอื่น ซึ่งรวมถึงสามฟิลด์ภายใต้หัวข้อ HOST USER INFO

  1. เข้าสู่บัญชีผู้ดูแลระบบ Case Active ของคุณ
  2. ค้นหาแอปและคลิกปุ่มผู้ดูแลระบบ
  3. คลิกผู้ใช้ที่เมนูด้านบน
  4. ไปที่เมนูย่อยและเลือกรายชื่อผู้ใช้ที่คุณต้องการเข้าถึง (ใช้งานหรือปิดใช้งาน) การตั้งค่าเริ่มต้นคือใช้งานอยู่
  5. ค้นหาผู้ใช้ในรายการและคลิกแก้ไข
  6. ทำการเปลี่ยนแปลงตามต้องการ (ดูด้านล่าง)
  7. คลิกบันทึก

เปลี่ยนข้อมูลผู้ใช้

ผู้ดูแลระบบสามารถเปลี่ยนแปลงข้อมูลผู้ใช้

  1. กำหนดประเภทผู้ใช้ (ผู้ใช้หรือผู้จัดการ)
  2. ตั้งค่าสถานะผู้ใช้ (ใช้งานเป็นปิดใช้งาน)
  3. ป้อน ID แท็กที่ไม่ซ้ำกัน (ไม่จำเป็น)
  4. คลิกบันทึก
