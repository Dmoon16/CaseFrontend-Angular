วิธีสร้างหรือแก้ไขเคสผ่านเมนู UI

บทความนี้แสดงวิธีสร้างกรณีและปัญหาด้วยส่วนต่อประสานผู้ใช้ของแพลตฟอร์มแก่ผู้ดูแลระบบ ผู้ดูแลระบบยังสามารถเพิ่มผู้ใช้ในกรณีใหม่และอัปเดตกรณีที่มีอยู่

การสร้างกรณี

  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่มผู้ดูแลระบบ
  3. ในแผงการดูแลระบบ คลิกแท็บเมนูกรณีและปัญหา
  4. คลิกปุ่มใหม่ หน้าต่างใหม่จะปรากฏขึ้น

การเพิ่มผู้ใช้

ผู้ดูแลระบบสามารถเพิ่มผู้ใช้ในกรณีและปัญหาในหน้านี้ เลื่อนลงและคลิกปุ่มกำหนด ผู้ใช้

img

ในโมดอล กำหนดผู้ใช้ ให้กรอกข้อมูลในฟิลด์ ทำเครื่องหมายในช่องเพื่อส่งการแจ้งเตือนทางอีเมลถึงผู้ใช้ (ไม่จำเป็น)

การแก้ไขกรณี

หากต้องการอัปเดตกรณีและปัญหาจำนวนเล็กน้อย ให้ใช้เครื่องมือแก้ไขและทำการเปลี่ยนแปลงในแต่ละกรณี

  • หมายเหตุ เมื่อต้องการอัปเดตกรณีและปัญหาจำนวนมากในคราวเดียว ให้พิจารณาใช้ เครื่องมือนำเข้า
  1. เข้าสู่บัญชีผู้ดูแลระบบ
  2. ค้นหาแอพในหน้าแอพแล้วคลิกปุ่มผู้ดูแลระบบ
  3. ในแผงการดูแลระบบ คลิกแท็บเมนูกรณีและปัญหา
  4. ค้นหากรณีและคลิกแก้ไข
  5. ทำการเปลี่ยนแปลงและคลิกบันทึก
