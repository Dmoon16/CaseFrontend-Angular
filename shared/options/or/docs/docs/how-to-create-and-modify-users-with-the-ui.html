ଅଳ୍ପ ସଂଖ୍ୟକ ଉପଭୋକ୍ତା ରେକର୍ଡ ଅପଡେଟ୍ କରିବାକୁ, ସଂପାଦନ ଉପକରଣ ବ୍ୟବହାର କରନ୍ତୁ ଏବଂ ପ୍ରତ୍ୟେକ ବ୍ୟକ୍ତିଗତ ଉପଭୋକ୍ତାଙ୍କୁ ପରିବର୍ତ୍ତନ କରନ୍ତୁ |

  • ଟିପ୍ପଣୀ ଗୋଟିଏ ସିଟିଂରେ ବହୁ ସଂଖ୍ୟକ ଉପଭୋକ୍ତା ରେକର୍ଡକୁ ରୂପାନ୍ତର କରିବାକୁ, ଆମଦାନୀ ଉପକରଣ ବ୍ୟବହାର କରିବାକୁ ଚିନ୍ତା କରନ୍ତୁ |
  1. ଆଡମିନି ପ୍ୟାନେଲରେ, ୟୁଜର୍ସ ମେନୁ ଟ୍ୟାବ୍ କ୍ଲିକ୍ କରନ୍ତୁ |
  2. ଏକ ୟୁଜର୍ ସନ୍ଧାନ କରନ୍ତୁ ଏବଂ ଏଡିଟ୍ କ୍ଲିକ୍ କରନ୍ତୁ |
  3. ପରିବର୍ତ୍ତନ କରନ୍ତୁ ଏବଂ ସଞ୍ଚୟ କ୍ଲିକ୍ କରନ୍ତୁ |
