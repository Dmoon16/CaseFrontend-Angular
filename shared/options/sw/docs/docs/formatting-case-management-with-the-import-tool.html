Kuumbiza Data ya Usimamizi wa Kesi kwa Zana ya Kuingiza

Ifuatayo ni orodha ya sehemu zinazohitajika katika faili za CSV ili kudhibiti kesi kwa zana ya Leta.

jina la mtumiaji

  • Ingiza anwani ya barua pepe ya mtumiaji.

tag_id (kitambulisho unachoweza kurejelea baadaye)

  • kitambulisho cha kipekee (tumia herufi za alphanumeric, viambatisho vinakubaliwa)

hali_ya_kesi

  • 0 = imefungwa
  • 1 = amilifu

tarehe_iliyofunguliwa

  • mwezi/siku/mwaka
  • mm/dd/yyyy

kuhusu

  • maelezo ya kipekee ya kesi (tumia herufi za alphanumeric, hyphens zinakubaliwa)
