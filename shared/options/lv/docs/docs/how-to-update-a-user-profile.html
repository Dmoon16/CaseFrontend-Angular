Kā atjaunināt lietotāja profilu

Šajā rakstā ir parādīts lietotājiem, kā atjaunināt informāciju savā profila lapā.

  • Lietotāji var atjaunināt savus profilus.
  • Administratori var atjaunināt jebkura lietotāja profila lapu.

Lietotājiem

Pirmais solis ir atvērt profila lapu. Lietotāji un administratori veic šīs darbības, lai atjauninātu savu profila lapu.

  1. Piesakieties savā Case Active kontā.
  2. Lapas augšdaļā noklikšķiniet uz cilnes Profils.
  3. Veiciet izmaiņas.
  4. Noklikšķiniet uz Saglabāt.

Lielākā daļa lauku ir neobligāti, un daži lauki ir obligāti. Obligātie lauki ir:

  • Vārds
  • Uzvārds
  • Epasta adrese
  • Valoda
  • Laika zona

Adminiem

Administratori var arī atjaunināt profila informāciju citiem lietotājiem. Tas ietver trīs laukus zem virsraksta HOST USER INFO.

  1. Piesakieties savā Case Active administratora kontā.
  2. Atrodiet lietotni un noklikšķiniet uz pogas ADMINISTORS.
  3. Augšējā izvēlnē noklikšķiniet uz Lietotāji.
  4. Dodieties uz apakšizvēlni un atlasiet lietotāju sarakstu, kuram vēlaties piekļūt (AKTĪVS vai ATspējots). Noklusējuma iestatījums ir Aktīvs.
  5. Sarakstā atrodiet lietotāju un noklikšķiniet uz Rediģēt.
  6. Veiciet izmaiņas pēc vajadzības. (Skatīt zemāk)
  7. Noklikšķiniet uz Saglabāt.

Mainiet lietotāja informāciju

Administratori var mainīt lietotāja informāciju.

  1. Iestatiet lietotāja veidu (lietotājs vai pārvaldnieks).
  2. Iestatiet lietotāja statusu (Aktīvs uz Atspējots).
  3. Ievadiet unikālu taga ID. (neobligāti)
  4. Noklikšķiniet uz Saglabāt.
