Lietu pārvaldības datu formatēšana, izmantojot importēšanas rīku

Tālāk ir sniegts CSV failu obligāto lauku saraksts, lai pārvaldītu gadījumus, izmantojot importēšanas rīku.

lietotājvārds

  • Ievadiet lietotāja e-pasta adresi.

tag_id (ID, uz kuru varat atsaukties vēlāk)

  • unikāls identifikators (izmantojiet burtciparu rakstzīmes, tiek pieņemtas defises)

case_status

  • 0 = slēgts
  • 1 = aktīvs

datums_atvērts

  • mēnesis/diena/gads
  • mm/dd/gggg

par

  • unikāls reģistra apraksts (izmantojiet burtciparu rakstzīmes, tiek pieņemtas defises)
