Lietotāju lietu piešķiršanas datu formatēšana, izmantojot importēšanas rīku

Tālāk ir sniegts CSV failu datu lauku saraksts. Katrai importēšanas opcijai ir atšķirīga lauku kombinācija.

case_ID

  • Ievadiet reģistru User vai Tag ID.
  • Atveriet lapu Lietas > Rediģēt > Tag ID VAI dodieties uz lapu Lietas > Rediģēt > Lietas ID URL.

Lietotāja ID

  • Ievadiet lietotāja ID vai Tag ID.
  • Atveriet lapu Lietotāji > Rediģēt > Tag ID VAI dodieties uz lapu Lietotāji > Rediģēt > Lietotāja ID URL.

case_role_order

  • Atstājiet šūnu tukšu.

case_role_ID

  • Ievadiet: klienti, darbinieki vai noklusējuma.

case_notify

  • 0 = nē
  • 1 = Jā, nosūtīt lietotājam e-pasta paziņojumu.
