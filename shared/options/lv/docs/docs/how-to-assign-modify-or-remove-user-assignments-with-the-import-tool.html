Kā piešķirt, modificēt vai noņemt lietotāju piešķīrumus, izmantojot importēšanas rīku

Šajā rakstā administratori parāda, kā pārvaldīt gadījumu lietotājus, izmantojot importēšanas rīku.

Administratori var:

  • piešķirt lietotājus lietai
  • mainīt lietotājus, kas jau ir piešķirti lietai
  • noņemt lietai piešķirtos lietotājus

Importēšanas rīka atvēršana

  1. Piesakieties administratora kontā.
  2. Lietojumprogrammu lapā atrodiet lietotni un noklikšķiniet uz pogas ADMINISTORS.
  3. Noklikšķiniet uz cilnes Integrācija un pēc tam apakšizvēlnē noklikšķiniet uz IMPORTS.
  4. Noklikšķiniet uz pogas JAUNS. Tiek parādīts dialoglodziņš Add Imports.

Ierakstu veidošana

Lejupielādējiet vienu vai vairākus CSV failus. Izmantojiet šos failus, lai pievienotu, modificētu vai noņemtu gadījumu lietotājus. Kad tiek atvērts dialoglodziņš Importa pievienošana:

  1. Noklikšķiniet uz lauka Tips.
  2. Ritiniet nolaižamo izvēlni un atlasiet opciju (skatiet tālāk). Tiek parādīta saite.
  3. Noklikšķiniet uz saites Lejupielādēt paraugu. CSV fails tiek lejupielādēts jūsu ierīcē.

Atveriet failu un pievienojiet ierakstus. Noklikšķiniet šeit, lai uzzinātu, kā izmantot formāta datus CSV failā.

Failu opcijas

Piešķirt lietotājus gadījumiem * Izvēlieties šo opciju, lai lietai pievienotu lietotājus. * Šūnu case_role_order atstājiet tukšu. * Dodieties uz lapu Lietotāji un kopējiet šīs personas taga ID. Ja lauks Tag ID ir tukšs, ievadiet unikālu ID, noklikšķiniet uz Saglabāt un pēc tam kopējiet ID CSV failā.

Modificēt lietotājus, kas piešķirti gadījumiem * Izvēlieties šo opciju, lai atjauninātu datus par pašreizējiem gadījumiem piešķirtajiem lietotājiem. * Izmantojiet Case Tag ID kolonnai case_ID.

Noņemt lietai piešķirtos lietotājus * Izvēlieties šo opciju, lai noņemtu lietai piešķirtos lietotājus.

Notiek faila augšupielāde

Varat augšupielādēt failu uzreiz vai vēlāk atgriezties dialoglodziņā Importa pievienošana un izpildīt šīs darbības.

  1. Atveriet dialoglodziņu Importa pievienošana.
  2. Ievadiet unikālu CSV faila nosaukumu.
  3. Noklikšķiniet uz plusa (+) ikonas, lai augšupielādētu failu no savas ierīces.
  4. Noklikšķiniet uz lauka Tips un atlasiet opciju. (Skatīt iepriekš)
  5. Noklikšķiniet uz SAGLABĀT.

Apstrāde

Atkarībā no ierakstu skaita failā un sistēmas ielādes, platformas importēšanas apstrāde var aizņemt vairākas minūtes.

Kad importēšana ir pabeigta, platforma pielāgo lietotāju ierakstus (ti, pievieno, maina vai noņem) lapā Gadījumi.

Neveiksme

Ja importēšanas darbība neizdodas, platforma atgriež ziņojumu Failed. Vairumā gadījumu kļūmes cēlonis ir datu formatēšana.

Lai noskaidrotu neveiksmes cēloni:

  1. Noklikšķiniet uz Skatīt zem virsraksta Darbība.
  2. Nākamajā lapā virziet peles kursoru uz ikonas Neizdevās. Rīka padoms parāda kļūdu.
  3. Dodieties uz CVS failu un pielāgojiet un izlabojiet kļūdu.
  4. Augšupielādējiet failu vēlreiz.
