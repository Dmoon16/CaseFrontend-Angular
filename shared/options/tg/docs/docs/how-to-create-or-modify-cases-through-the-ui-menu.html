Чӣ гуна парвандаҳоро тавассути менюи UI эҷод кардан ё тағир додан мумкин аст

Ин мақола ба маъмурон нишон медиҳад, ки чӣ гуна парвандаро бо интерфейси корбарии платформа эҷод кунанд. Администраторҳо инчунин метавонанд корбаронро ба парвандаи нав илова кунанд ва парвандаҳои мавҷударо навсозӣ кунанд.

Эҷоди парванда

  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Дар панели идоракунӣ, ҷадвали менюи парвандаҳоро клик кунед.
  4. Тугмаи НАВ-ро пахш кунед. Равзанаи нав намоиш дода мешавад.

Майдонҳои заруриро пур кунед ва ЗАХИРА-ро пахш кунед.

Илова кардани корбарон

Ихтиёрӣ, маъмурон метавонанд корбаронро ба парванда дар ин саҳифа илова кунанд. Ба поён ҳаракат кунед ва тугмаи ТАЪИНИ ИСТИФОДАБАРОН -ро пахш кунед.

img

Дар модули Таъин кардани корбар, майдонҳоро пур кунед. Қуттиро барои фиристодани огоҳиномаи почтаи электронӣ ба корбар санҷед. (ихтиёрӣ)

Тағйир додани парванда

Барои навсозии шумораи ками парвандаҳо, асбоби Таҳрирро истифода баред ва ба ҳар як ҳолати алоҳида тағирот ворид кунед.

  • Эзоҳ Барои навсозии шумораи зиёди парвандаҳо дар як нишаст, истифодаи абзори Воридотро баррасӣ кунед
  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Дар панели идоракунӣ, ҷадвали менюи парвандаҳоро клик кунед.
  4. Як парвандаро ҷойгир кунед ва Таҳрирро клик кунед.
  5. Тағирот ворид кунед ва ЗАХИРА-ро пахш кунед.
