Чӣ тавр таъин, тағир додан ё хориҷ кардани супоришҳои корбар бо воситаи Воридот

Ин мақола ба маъмурон нишон медиҳад, ки чӣ тавр корбарони парвандаро бо воситаи Воридот идора кунанд.

Админҳо метавонанд:

  • истифодабарандагонро ба парванда таъин кунед
  • истифодабарандагони аллакай ба парванда таъиншударо тағир диҳед
  • истифодабарандагони ба парванда таъиншударо хориҷ кунед

Кушодани абзори воридот

  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Ҷадвали Интегратсияро клик кунед ва пас дар зерменю IMPORTS -ро клик кунед.
  4. Тугмаи НАВ-ро пахш кунед. Қуттии муколамаи Иловаи воридот намоиш дода мешавад.

Эҷоди сабтҳо

Як ё якчанд файлҳои CSV-ро зеркашӣ кунед. Ин файлҳоро барои илова, тағир додан ё нест кардани корбарони парванда истифода баред. Пас аз равзанаи муколамаи Иловаи воридот кушода мешавад:

  1. Майдони Навъро клик кунед.
  2. Дар менюи афтанда ҳаракат кунед ва интихоберо интихоб кунед (ба поён нигаред). Истинод нишон дода мешавад.
  3. Истиноди зеркашии намунаро клик кунед. Файли CSV ба дастгоҳи шумо зеркашӣ карда мешавад.

Файлро кушоед ва сабтҳоро илова кунед. Барои омӯхтани тарзи истифода бурдани маълумоти формат дар файли CSV ин ҷо клик кунед.

Имконоти файл

Корбаронро ба парвандаҳо таъин кунед * Барои илова кардани корбарон ба парванда ин интихобро интихоб кунед. * Ячеки case_role_order-ро холӣ гузоред. * Ба саҳифаи корбарон равед ва ID-и барчаспро барои ин шахс нусхабардорӣ кунед. Агар майдони ID-и барчасп холӣ бошад, ID-и беназирро ворид кунед, Захира-ро пахш кунед ва сипас ID-ро ба файли CSV нусхабардорӣ кунед.

Корбаронеро, ки ба парвандаҳо таъин шудаанд, тағир диҳед * Барои навсозии маълумот барои корбарони ҷории ба парванда таъиншуда ин интихобро интихоб кунед. * Барои сутуни case_ID ID Tag-ро истифода баред.

Хориҷ кардани корбароне, ки ба парвандаҳо таъин шудаанд * Барои нест кардани корбароне, ки ба парванда таъин шудаанд, ин интихобро интихоб кунед.

Боркунии файл

Шумо метавонед файлро фавран бор кунед ё дертар ба қуттии муколамаи Иловаи воридот баргардед ва ин қадамҳоро анҷом диҳед.

  1. Қуттии муколамаи Иловаи воридотро кушоед.
  2. Барои файли CSV номи беназирро ворид кунед.
  3. Барои бор кардани файл аз дастгоҳи худ тасвири плюс (+) -ро клик кунед.
  4. Майдони Навъро клик кунед ва интихоберо интихоб кунед. (ба боло нигаред)
  5. ЗАХИРА пахш кунед.

Коркард

Вобаста аз шумораи сабтҳо дар файл ва бори система, платформа метавонад барои коркарди воридот чанд дақиқа вақт гирад.

Вақте ки воридот ба итмом мерасад, платформа сабтҳои корбарро танзим мекунад (яъне илова мекунад, тағир медиҳад ё нест мекунад) дар саҳифаи парвандаҳо.

Нокомӣ

Агар амалиёти воридот ноком шавад, платформа паёми нокомро бармегардонад. Дар аксари ҳолатҳо, сабаби нокомӣ форматкунии маълумот аст.

img

Барои дидани сабаби нокомӣ:

  1. Намоишро дар зери сарлавҳаи амал клик кунед.
  2. Дар саҳифаи навбатӣ, мушро ба тасвири Ноком гузоред. Маслиҳати асбоб хатогиро нишон медиҳад.
  3. Ба файли CVS равед ва хатогиро ислоҳ кунед.
  4. Файлро дубора бор кунед.
