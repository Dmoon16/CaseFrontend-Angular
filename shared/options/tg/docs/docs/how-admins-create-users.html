Чӣ тавр маъмурон корбаронро эҷод мекунанд

Администраторҳо, ки метавонанд ба барномаҳо ва парвандаҳо дастрасӣ пайдо кунанд, эҷод мекунанд, тағир медиҳанд ё ғайрифаъол мекунанд. Вақте ки корбар бори аввал ба платформа ворид мешавад, онҳо бояд ҷараёни кории профилро анҷом диҳанд.

  • Барои гирифтани маълумоти бештар дар бораи ҷараёни нави профили корбар ин ҷо клик кунед.

Эҷоди корбар

Дар ҷадвали зер панҷ роҳи эҷоди корбар нишон дода шудааст. Истинодро клик кунед, то бифаҳмед, ки чӣ тавр.

Огоҳиномаҳо

Платформа ба корбарон ҳангоми илова кардани барнома огоҳинома намефиристад. Аммо, маъмурон метавонанд огоҳиномаҳоро барои таъиноти парванда фаъол созанд.
