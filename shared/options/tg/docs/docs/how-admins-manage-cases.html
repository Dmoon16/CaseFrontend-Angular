Чӣ тавр маъмурон парвандаҳоро идора мекунанд

Чор роҳи идоракунии парванда вуҷуд дорад. Истинодро клик кунед, то бифаҳмед, ки чӣ тавр.

Усул Навъи Амалҳо Тафсилот
Истифодаи UI UI илова кардан ва тағир додан Тугмаи Admin > Ҳолатҳо
Бо истифода аз истиноди бақайдгирии ҷамъиятӣ худкор эҷод кардани парванда Амали корбар илова кунед Тугмаи Admin > Танзимот > Саҳифаи танзимоти система
Воситаи воридотӣ UI илова кардан ва тағир додан Тугмаи Admin > Интегратсияҳо > Воридот
Истифодаи API (ба зудӣ меояд) API илова ва тағир диҳед Н/А
