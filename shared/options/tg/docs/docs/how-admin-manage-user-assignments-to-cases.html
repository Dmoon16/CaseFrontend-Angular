Чӣ тавр маъмурон супоришҳои корбарро ба парвандаҳо идора мекунанд

Панҷ роҳи идоракунии корбарон ва супоришҳои парванда вуҷуд дорад. Истинодро клик кунед, то бифаҳмед, ки чӣ тавр.

Усул Навъи Амалҳо Тафсилот
Ҳангоми эҷод кардани парванда UI илова кунед Тугмаи администратор > Ҳолатҳо > Тугмаи нав > Тугмаи корбаронро таъин кунед
Бо истифода аз истиноди бақайдгирии оммавӣ корбарро худкор таъин кунед Амали корбар илова кунед Тугмаи Admin > Танзимот > Саҳифаи танзимоти система
Истифодаи UI дар парвандаи мавҷуда UI илова ва хориҷ кунед Тугмаи администратор > Ҳолатҳо > Таъин кардани корбарон
Воситаи воридотӣ UI илова кардан, тағир додан ва хориҷ кардан Тугмаи Admin > Интегратсияҳо > Воридот
Истифодаи API (Ба наздикӣ) API илова кардан, тағир додан ва хориҷ кардан Н/А
