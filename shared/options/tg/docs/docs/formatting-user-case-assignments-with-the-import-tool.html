Форматкунии маълумот оид ба таъиноти парвандаи корбар бо воситаи воридот

Дар зер рӯйхати майдонҳои маълумот дар файлҳои CSV оварда шудааст. Ҳар як варианти воридот дорои маҷмӯи гуногуни майдонҳо мебошад.

парванда_ID

  • Корбар ё ID-и парвандаро ворид кунед.
  • Ба саҳифаи парвандаҳо гузаред > Таҳрир > ID-и барчасп ё ба саҳифаи парвандаҳо > Таҳрир > ID парванда дар url гузаред.

Шаҳодатномаи Корбар

  • ID-и корбар ё ID-и барчаспро ворид кунед.
  • Ба саҳифаи корбарон гузаред > Таҳрир > ID-и барчасп ё ба саҳифаи корбарон > Таҳрир > ID корбар дар url гузаред.

парванда_нақши_тартиби

  • Ҳуҷайраро холӣ гузоред.

case_rol_ID

  • Ворид кунед: мизоҷон, кормандон ё пешфарз.

парванда_хабардор кунед

  • 0 = нест
  • 1 = Бале, ба корбар огоҳиномаи почтаи электронӣ фиристед.
