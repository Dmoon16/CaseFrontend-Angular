Чӣ тавр ворид шудан

Бифаҳмед, ки чӣ тавр ба ҳисоби мавҷудаи Case Active худ ворид шавед.

Воридшавӣ

  1. Ба саҳифаи воридшавӣ гузаред .
  2. Тугмаи LOGIN -ро пахш кунед.
  3. Почтаи электронии худро ворид кунед.
  4. ОЯНДА-ро пахш кунед.

Тасдиқи ҳисоб

  1. Ҷадвали браузери нав кушоед ва ба ҳисоби почтаи электронии худ ворид шавед.
  2. Рамзи шаш рақамро нусхабардорӣ кунед.
  3. Бозгашт ба ҷадвали Парвандаи фаъол.
  4. Рамзро ба майдонҳо гузоред. Равзанаи нав намоиш дода мешавад.

Case Active саҳифаи Барномаҳоро нишон медиҳад.
