Чӣ тавр эҷод кардани корбарон бо воситаи воридот

Ин мақола ба маъмурон нишон медиҳад, ки чӣ гуна корбаронро бо воситаи Воридот эҷод кунанд.

Кушодани абзори воридот

Администраторҳо метавонанд асбоби Воридотро барои ба таври оммавӣ бор кардани корбарони нав ва тағир додани сабтҳои мавҷудаи корбар истифода баранд.

  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Дар панели идоракунӣ, ҷадвали Интегратсияро пахш кунед ва пас дар зерменю IMPORTS -ро пахш кунед.
  4. Тугмаи НАВ-ро пахш кунед. Қуттии муколамаи Иловаи воридот дар ҷадвали нави браузер намоиш дода мешавад.

Эҷоди сабтҳо

Пас аз равзанаи муколамаи Иловаи воридот кушода мешавад:

  1. Майдони Навъро клик кунед ва интихоберо интихоб кунед (ба поён нигаред). Истинод нишон дода мешавад.
  2. Истиноди зеркашии намунаро клик кунед. Файли CSV ба дастгоҳи шумо зеркашӣ карда мешавад.

Баъдан, файлро кушоед ва сабтҳоро илова кунед. Барои омӯхтани тарзи истифода бурдани маълумоти формат дар файли CSV ин ҷо клик кунед.

Навъи

Вақте ки шумо майдони Навъро пахш мекунед, менюи афтанда пайдо мешавад. Яке аз ду вариантро барои воридоти оммавӣ интихоб кунед.

:нидо: ЭЗОҲ Менюи афтанда дорои имконоти зиёд аст, аммо танҳо дуто ба абзори воридот алоқаманданд. Ҳар як интихоб файли CSV-и намунаи гуногун дорад.

Илова кардани корбарон

  • Ин хосиятро барои боргузории оддии умумии сабтҳои корбар интихоб кунед.
  • Барои кушодани файли CSV бо сарлавҳаҳои сутун барои намуди воридоти корбарон, истиноди Зеркашии Намунаро клик кунед.

Истифодабарандагонро тағир диҳед

  • Барои навсозии маълумот барои корбарони ҷорӣ ин хосиятро интихоб кунед.
  • Барои кушодани файли CSV бо сарлавҳаҳои сутун барои тағир додани намуди воридоти корбарон истиноди зеркашии намунаро клик кунед.

Боркунии файл

Шумо метавонед файлро фавран бор кунед ё дертар ба қуттии муколамаи Иловаи воридот баргардед ва ин қадамҳоро анҷом диҳед.

  1. Қуттии муколамаи Иловаи воридотро кушоед.
  2. Барои файли CSV номи беназирро ворид кунед.
  3. Барои бор кардани файл аз дастгоҳи худ тасвири плюс (+) -ро клик кунед.
  4. Майдони Навъро клик кунед ва интихоберо интихоб кунед. (ба боло нигаред)
  5. ЗАХИРА пахш кунед.

Коркард

Вобаста аз шумораи сабтҳо дар файл ва бори система, платформа метавонад барои коркарди воридот чанд дақиқа вақт гирад.

Вақте ки воридот ба итмом мерасад, платформа сабтҳои корбарро танзим мекунад (яъне илова мекунад, тағир медиҳад ё ғайрифаъол мекунад) дар панели корбарони администратор.

Нокомӣ

Агар амалиёти воридот ноком шавад, платформа паёми нокомро бармегардонад. Дар аксари ҳолатҳо, сабаби нокомӣ форматкунии маълумот аст.

Барои дидани сабаби нокомӣ:

  1. Намоишро дар зери сарлавҳаи амал клик кунед.
  2. Дар саҳифаи навбатӣ, мушро ба тасвири Ноком гузоред. Маслиҳати асбоб хатогиро нишон медиҳад.
  3. Ба файли CVS равед ва хатогиро ислоҳ кунед.
  4. Файлро дубора бор кунед.
