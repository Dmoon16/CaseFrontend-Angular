Чӣ тавр эҷод ё тағир додани парвандаҳо бо воситаи воридот

Ин мақола ба маъмурон нишон медиҳад, ки чӣ гуна парвандаҳои мавҷударо бо воситаи Воридот эҷод кардан ё тағир додан мумкин аст.

Кушодани абзори воридот

  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Дар панели идоракунӣ, ҷадвали Интегратсияро пахш кунед ва пас дар зерменю IMPORTS -ро пахш кунед.
  4. Тугмаи НАВ-ро пахш кунед. Қуттии муколамаи Иловаи воридот дар ҷадвали нави браузер намоиш дода мешавад.

img

Эҷоди парвандаҳо

Пас аз равзанаи муколамаи Иловаи воридот кушода мешавад:

  1. Майдони Навъро клик кунед ва интихоберо интихоб кунед (ба поён нигаред). Истинод нишон дода мешавад.
  2. Истиноди зеркашии намунаро клик кунед. Файли CSV ба дастгоҳи шумо зеркашӣ карда мешавад.

Баъд, файлро кушоед ва парвандаҳоро илова кунед.

Навъи

Вақте ки шумо майдони Навъро пахш мекунед, менюи афтанда пайдо мешавад. Яке аз ду интихобро барои воридоти парвандаҳои оммавӣ интихоб кунед.

  • Эзоҳ Менюи афтанда имконоти зиёде дорад, аммо танҳо дуто ба идоракунии парванда алоқаманданд. Ҳар як вариант дорои файли CSV-и намунаи гуногун аст.

Илова кардани парвандаҳо

  • Ин хосиятро барои боргузории оддии оммавии парвандаҳо интихоб кунед.
  • Барои кушодани файли CSV бо сарлавҳаҳои сутун барои навъи воридоти парвандаҳои Эҷоди парванда истиноди Зеркашии Намунаро клик кунед.

Тағйир додани парвандаҳо

  • Барои навсозии парвандаҳои ҷорӣ ин хосиятро интихоб кунед.
  • Барои кушодани файли CSV бо сарлавҳаҳои сутун барои навъи воридоти парвандаҳои тағир додани парванда истиноди Зеркашии Намунаро клик кунед.

Боркунии файл

Шумо метавонед файлро фавран бор кунед ё дертар ба қуттии муколамаи Иловаи воридот баргардед ва ин қадамҳоро анҷом диҳед.

  1. Қуттии муколамаи Иловаи воридотро кушоед.
  2. Барои файли CSV номи беназирро ворид кунед.
  3. Барои бор кардани файл аз дастгоҳи худ тасвири плюс (+) -ро клик кунед.
  4. Майдони Навъро клик кунед ва интихоберо интихоб кунед. (ба боло нигаред)
  5. ЗАХИРА пахш кунед.

Форматкунӣ

Инҳо майдонҳои ҳатмӣ ва талаботи аввал мебошанд. Барои баррасии тафсилоти форматкунӣ ин ҷо клик кунед.

tag_id (ё case_ID) идентификатори беназир (истифодаи аломатҳои алифбои рақамӣ, дефис қабул карда мешавад)

case_status 0 = баста 1 = фаъол

санаи_кушодашуда моҳ/рӯз/сол мм/дд/yyyy

дар бораи тавсифи нодири парванда (истифодаи аломатҳои алифбои рақамӣ, дефис қабул карда мешавад)

Коркард

Вобаста аз шумораи сабтҳо дар файл ва бори система, платформа метавонад барои коркарди воридот якчанд дақиқа вақт гирад.

img

Вақте ки воридот ба итмом мерасад, платформа сабтҳои парвандаро дар панели идоракунии парвандаҳо танзим мекунад (яъне илова мекунад ё тағир медиҳад).

Нокомӣ

Агар амалиёти воридот ноком шавад, платформа паёми нокомро бармегардонад. Дар аксари ҳолатҳо, сабаби нокомӣ форматкунии маълумот аст.

img

Барои дидани сабаби нокомӣ:

  1. Намоишро дар зери сарлавҳаи АКЦИЯ пахш кунед.
  2. Дар саҳифаи навбатӣ, мушро ба тасвири Ноком гузоред. Маслиҳати асбоб хатогиро нишон медиҳад.
  3. Ба файли CVS равед ва хатогиро ислоҳ кунед.
  4. Файлро дубора бор кунед.
