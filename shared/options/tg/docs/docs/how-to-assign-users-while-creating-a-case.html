Чӣ тавр таъин кардани корбарон ҳангоми сохтани парванда

Ин мақола ба маъмурон нишон медиҳад, ки чӣ гуна корбарро ҳангоми сохтани парвандаи нав таъин кунанд. Агар корбар аллакай дар платформа сабти ном нашуда бошад, шумо метавонед ҳангоми сохтани парвандаи нав корбари нав илова кунед.

Эҷоди парванда

  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Ҷадвали менюи Ҳолатҳоро клик кунед.
  4. Тугмаи НАВ-ро пахш кунед. Равзанаи нав намоиш дода мешавад.
  • Барои сохтани парванда майдонҳоро пур кунед ва сипас ба сарлавҳаи корбарони таъиншуда ба поён ҳаракат кунед.

Таъин кардани корбар

Истифодабарандаи ҷорӣ

Барои таъин кардани корбари ҷорӣ ба парвандаи нав ин қадамҳоро иҷро кунед.

Тугмаи ТАЪИНИ ИСТИФОДАБАРОН-ро пахш кунед ва майдонҳоро пур кунед.

  1. афтанда-и корбарро клик кунед ва идентификаторро интихоб кунед (масалан, почтаи электронӣ).
  2. Дар майдони холӣ арзиш ворид кунед (масалан, суроғаи почтаи электронӣ).
  3. Агар сабти корбар дар платформа мавҷуд бошад, майдон худкор номро пешниҳод мекунад.
  4. Корбари дурустро интихоб кунед.
  5. Нақши корбарро интихоб кунед (мизоҷон, пешфарз ё кормандон).
  6. Ихтиёрӣ, қуттиро барои фиристодани огоҳиномаи почтаи электронӣ ба корбар қайд кунед.
  7. ЗАХИРА пахш кунед.

Case Active корбарро ба парванда илова мекунад. Акнун маъмурон метавонанд барои пӯшидани тиреза ё илова кардани корбари нав ё хориҷ кардани корбар "Захира" -ро клик кунанд.

Илова кардани корбари нав

Агар платформа сабти ҷориро пайдо карда натавонад, тугмаро пахш кунед барои илова кардани корбари нав.

Майдонҳоро пур кунед ва сипас ЗАХИРА-ро пахш кунед, то корбарро ба платформа илова кунед. Баъдан, нақши корбарро таъин кунед ва бори дигар ЗАХИРА-ро пахш кунед, то корбарро ба парванда таъин кунед. Ниҳоят, барои захира кардани парванда як бори дигар ЗАХИРА-ро пахш кунед.
