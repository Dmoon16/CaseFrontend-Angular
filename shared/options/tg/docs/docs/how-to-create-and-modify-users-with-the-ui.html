Чӣ гуна корбаронро бо UI эҷод кардан ва тағир додан мумкин аст

Ин мақола ба маъмурон нишон медиҳад, ки чӣ гуна корбарон эҷод кунанд ва сабтҳоро бо интерфейси корбарии платформа нав кунанд. Администраторҳо инчунин метавонанд корбаронро ба парвандаи нав илова кунанд ва парвандаҳои мавҷударо таҳрир кунанд.

Эҷоди корбар

  1. Ба ҳисоби администратор ворид шавед.
  2. Дар саҳифаи Барномаҳо барномаеро ҷойгир кунед ва тугмаи ADMIN -ро клик кунед.
  3. Дар панели идоракунӣ, ҷадвали менюи Истифодабарандагонро клик кунед.
  4. Тугмаи НАВ-ро пахш кунед. Равзанаи нав намоиш дода мешавад.

Майдонҳоро пур кунед ва ЗАХИРА-ро пахш кунед. Case Active номро ба рӯйхати корбарони бо ин барнома алоқаманд илова мекунад.

img

Тағир додани корбар

Барои навсозии шумораи ками сабтҳои корбар, асбоби Таҳрирро истифода баред ва ба ҳар як корбари инфиродӣ тағирот ворид кунед.

  • Эзоҳ Барои тағир додани шумораи зиёди сабтҳои корбар дар як нишаст, истифода аз абзори Воридотро баррасӣ кунед.
  1. Дар панели идоракунӣ, ҷадвали менюи Истифодабарандагонро клик кунед.
  2. Истифодабарандаро ҷойгир кунед ва Таҳрирро клик кунед.
  3. Тағирот ворид кунед ва ЗАХИРА-ро пахш кунед.

Қадамҳои оянда

Case Active ба корбар огоҳиномаи почтаи электронӣ намефиристад. Администраторҳо бояд бо корбар тамос гиранд ва ба онҳо истиноди саҳифаи асосии Case Active фиристанд. Пас корбар тугмаи LOGIN -ро клик мекунад ва ҷараёни кории сабти номҳои навро анҷом медиҳад.

  • Ин мақола ҷараёни сабти ном барои корбарони навро тавсиф мекунад.
