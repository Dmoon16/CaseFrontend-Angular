Чӣ тавр маъмурон корбаронро идора мекунанд

Администраторҳо, ки метавонанд ба барномаҳо ва парвандаҳо дастрасӣ пайдо кунанд, эҷод мекунанд, тағир медиҳанд ё ғайрифаъол мекунанд. Вақте ки корбар бори аввал ба платформа ворид мешавад, онҳо бояд ҷараёни кории профилро анҷом диҳанд.

  • Барои гирифтани маълумоти бештар дар бораи ҷараёни нави профили корбар ин ҷо клик кунед.

Идоракунии корбар

Дар ҷадвали зер панҷ роҳи идоракунии корбар нишон дода шудааст. Истинодро клик кунед, то бифаҳмед, ки чӣ тавр.

Усул Навъи Амалҳо Тафсилот
Истифодаи UI UI илова кардан ва тағир додан Тугмаи Admin > Истифодабарандагон
Ҳангоми сохтани парванда ва корбарро ба парванда таъин кунед UI илова кунед Тугмаи администратор > Ҳолатҳо > Тугмаи нав > Саҳифаи парванда эҷод кунед
Бо истифода аз истиноди бақайдгирии ҷамъиятӣ корбарро худкор эҷод кунед Амали корбар илова кунед Тугмаи Admin > Танзимот > Саҳифаи танзимоти система
Воситаи воридотӣ UI илова ва тағир диҳед Тугмаи Admin > Интегратсияҳо > Воридот
Истифодаи API (ба зудӣ меояд) API илова ва тағир диҳед Н/А

Огоҳиномаҳо

Платформа ба корбарон ҳангоми илова кардани барнома огоҳинома намефиристад. Аммо, маъмурон метавонанд огоҳиҳоро барои таъиноти парванда фаъол созанд.
