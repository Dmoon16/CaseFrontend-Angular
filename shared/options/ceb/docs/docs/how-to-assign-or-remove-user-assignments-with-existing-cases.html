Unsaon Pag-assign o Pagtangtang sa Mga Assignment sa Gumagamit nga adunay Mga Kaso

Kini nga artikulo nagpakita sa mga admin kung unsaon pag-assign ug pagtangtang sa mga tiggamit sa kaso. Gipakita usab namo kanimo kung unsaon pag-usab ang datos sa user.

Pag-abli sa kasamtangan nga kaso

  1. Log in sa admin account.
  2. Pangitaa ang app sa Apps page ug i-klik ang ADMIN button.
  3. I-klik ang tab nga menu sa Cases .
  4. Siguroha nga ang ACTIVE gipili sa submenu.
  5. Pag-scroll paubos sa ACTION column.

img

I-assign o tangtangon

I-klik ang I-assign ang mga Gumagamit aron ma-assign ang usa ka tiggamit sa kaso o tangtangon ang tiggamit gikan sa kaso (apan dili ang plataporma). Sa sunod nga panid: I-klik ang ADD ug kompletoha ang workflow. I-klik ang tangtang aron mapapas ang usa ka user.

Usba

Dili mahimo ang pag-edit o pag-update sa datos sa tiggamit sa panid sa Mga Kaso. Hinuon, sunda kini nga mga lakang:

  1. I-klik ang tab sa Users menu.
  2. Pangitaa ang user name ug i-klik ang Edit .
  3. Paghimo og mga kausaban ug i-klik ang SAVE.
