Giunsa paghimo sa mga Admin ang mga Gumagamit

Ang mga admin naghimo, nag-usab o nag-disable kung kinsa ang maka-access sa mga app ug mga kaso. Kung ang usa ka tiggamit mag-log in sa plataporma sa unang higayon, kinahanglan nila nga makompleto ang usa ka workflow sa profile.

  • I-klik dinhi aron makat-on og dugang mahitungod sa bag-ong user profile workflow.

Paghimo og user

Ang lamesa sa ubos nagpakita sa lima ka mga paagi sa paghimo og usa ka user. Pag-klik sa usa ka link aron mahibal-an kung giunsa.

| Pamaagi | Type | Mga aksyon | Mga Detalye | Bag-ong user nakadawat og pahibalo? |
|------------------------|----------------|------- ---------------------|---------------------|---------------------- -------------------------|

Gamit ang UI | UI | idugang ug usba | Butang sa Admin > Mga Gumagamit | Dili |

Samtang nagmugna og usa ka kaso ug nag-assign sa user sa kaso | UI | idugang | Button sa Admin > Mga Kaso > Bag-ong buton > Paghimo og panid sa Kaso | Oo, kung mahimo |

Awtomatikong paghimo og user gamit ang public signup link | Aksyon sa tiggamit | idugang | Button sa Admin > Mga Setting > Panid sa System Setup | Oo, kung ang auto create nga kaso gi-enable |

Import nga himan | UI | idugang ug usba | Butang sa Admin > Mga Integrasyon > Mga Import | Dili |

Gamit ang API (sa dili madugay) | API | idugang ug usba | na | na |

Mga pahibalo

Ang plataporma wala magpadala sa mga tiggamit og pahibalo kung idugang sila sa usa ka app. Bisan pa, ang mga admin makahimo sa mga pahibalo alang sa mga buluhaton sa kaso.
