Giunsa pagdumala sa mga Admin ang mga Gumagamit

Ang mga admin naghimo, nag-usab o nag-disable kung kinsa ang maka-access sa mga app ug mga kaso. Kung ang usa ka tiggamit mag-log in sa plataporma sa unang higayon, kinahanglan nila nga makompleto ang usa ka workflow sa profile.

  • I-klik dinhi aron makat-on og dugang mahitungod sa bag-ong user profile workflow.

Pagdumala sa usa ka tiggamit

Ang lamesa sa ubos nagpakita sa lima ka mga paagi sa pagdumala sa usa ka user. Pag-klik sa usa ka link aron mahibal-an kung giunsa.

Pamaagi Matang Mga aksyon Mga Detalye
Paggamit sa UI UI idugang ug usbon Butang sa Admin > Mga Gumagamit
Samtang nagmugna og kaso ug i-assign ang user sa case UI idugang Butang sa Admin > Mga Kaso > Bag-ong buton > Paghimo og panid sa Kaso
Awtomatikong paghimo og user gamit ang public signup link aksyon sa user idugang Butang sa Admin > Mga Setting > Panid sa System Setup
Import nga himan UI idugang ug usbon Butang sa Admin > Mga Integrasyon > Mga Import
Paggamit sa API (sa dili madugay) API idugang ug usbon N/A

Mga pahibalo

Ang plataporma wala magpadala sa mga tiggamit og pahibalo kung idugang sila sa usa ka app. Bisan pa, ang mga admin makahimo sa mga pahibalo alang sa mga buluhaton sa kaso.
