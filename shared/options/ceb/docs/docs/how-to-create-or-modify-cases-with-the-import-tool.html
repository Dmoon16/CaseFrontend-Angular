Unsaon Paghimo o Pag-usab sa mga Kaso gamit ang Import Tool

Gipakita sa kini nga artikulo ang mga admin kung giunsa paghimo o pagbag-o ang mga naa na nga kaso gamit ang tool sa Import.

Pag-abli sa Import nga himan

  1. Log in sa admin account.
  2. Pangita ug app sa Apps page ug i-klik ang ADMIN button.
  3. Sa Admin panel, i-klik ang Integration tab ug dayon IMPORTS sa submenu.
  4. I-klik ang BAG-O nga buton. Ang Add Imports dialog box nagpakita sa bag-ong tab sa browser.

img

Paghimo og mga kaso

Human maablihan ang Add Imports dialog box:

  1. I-klik ang Type field ug pilia ang opsyon (tan-awa sa ubos). Nagpakita ang usa ka link.
  2. I-klik ang Download Sample nga link. Usa ka CSV file ang ma-download sa imong device.

Sunod, ablihi ang file ug idugang ang mga kaso.

Matang

Kung imong i-klik ang Type field, usa ka drop-down menu ang makita. Pagpili og usa sa duha ka mga opsyon alang sa mga import sa bulk case.

  • Mubo nga sulat Ang drop-down menu adunay daghang mga kapilian, apan duha ra ang may kalabutan sa pagdumala sa kaso. Ang matag kapilian adunay lahi nga sample nga CSV file.

Idugang ang mga kaso

  • Pilia kini nga kapilian alang sa usa ka yano nga daghang pag-upload sa mga kaso.
  • I-klik ang link sa Pag-download sa Sample aron maablihan ang usa ka CSV file nga adunay mga ulohan sa kolum para sa tipo sa pag-import sa Paghimo og mga kaso.

Usba ang mga kaso

  • Pilia kini nga kapilian aron ma-update ang mga karon nga kaso.
  • I-klik ang link sa Pag-download sa Sample aron maablihan ang CSV file nga adunay mga ulohan sa kolum para sa tipo sa pag-import sa Modify cases.

Pag-upload sa file

Mahimo kang mag-upload dayon og file o mobalik sa Add Imports dialog box sa ulahi nga petsa ug kompletoha kini nga mga lakang.

  1. Ablihi ang Add Imports dialog box.
  2. Pagsulod ug talagsaong ngalan alang sa CSV file.
  3. I-klik ang plus (+) icon aron ma-upload ang file gikan sa imong device.
  4. I-klik ang Type field ug pagpili og opsyon. (tan-awa sa ibabaw)
  5. I-klik ang SAVE.

img

Pag-format

Kini ang gikinahanglan nga mga natad ug unang mga kinahanglanon. Pag-klik dinhi aron masusi ang mga detalye sa pag-format.

tag_id (o case_ID) usa ka talagsaon nga identifier (gamit ang alphanumeric nga mga karakter, gidawat ang mga hyphen)

case_status 0 = sirado 1 = aktibo

date_open bulan/adlaw/tuig mm/dd/yyyy

bahin sa usa ka talagsaon nga paghulagway sa kaso (gamit ang alphanumeric nga mga karakter, gidawat ang mga hyphen)

img

Pagproseso

Depende sa gidaghanon sa mga rekord sa file ug system load, ang plataporma mahimong molungtad ug pipila ka minuto aron maproseso ang import.

img

Kung makompleto ang import, ang plataporma mag-adjust sa mga rekord sa kaso (pananglitan, pagdugang o pag-usab) sa dashboard sa Mga Kaso sa admin.

Kapakyasan

Kung mapakyas ang operasyon sa pag-import, ang plataporma magbalik sa usa ka Napakyas nga mensahe. Sa kadaghanan nga mga kaso, ang hinungdan sa pagkapakyas mao ang pag-format sa datos.

Aron makita ang hinungdan sa kapakyasan:

  1. I-klik ang View ubos sa ACTION nga ulohan.
  2. Sa sunod nga panid, i-hover ang mouse sa Failed icon. Gipakita sa tooltip ang sayup.
  3. Adto sa CVS file ug i-adjust ug ayoha ang sayop.
  4. I-upload pag-usab ang file.
