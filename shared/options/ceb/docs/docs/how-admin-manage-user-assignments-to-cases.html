Giunsa Pagdumala sa mga Admin ang Mga Assignment sa Gumagamit Sa Mga Kaso

Adunay lima ka mga paagi sa pagdumala sa mga tiggamit ug mga buluhaton sa kaso. Pag-klik sa usa ka link aron mahibal-an kung giunsa.

Pamaagi Matang Mga aksyon Mga Detalye
Samtang nagmugna og kaso UI idugang Button sa Admin > Mga Kaso > Bag-ong buton > Button sa Pag-assign sa mga Gumagamit
Auto assign user gamit ang public signup link Aksyon sa Gumagamit idugang Butang sa Admin > Mga Setting > Panid sa System Setup
Paggamit sa UI sa kasamtangan nga kaso UI idugang ug tangtangon Button sa Admin > Mga Kaso > I-assign ang mga Gumagamit
Import nga himan UI idugang, usba, ug tangtangon Butang sa Admin > Mga Integrasyon > Mga Import
Paggamit sa API (Moabot Na) API idugang, usba, ug tangtangon N/A
