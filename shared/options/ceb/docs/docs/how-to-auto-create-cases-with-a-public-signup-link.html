Giunsa ang Awtomatikong Paghimo og mga Kaso nga adunay usa ka Public Signup Link

Gipakita sa kini nga artikulo ang mga admin kung giunsa paghimo ang usa ka kaso (ug pagdugang usa ka tiggamit sa parehas nga oras) pinaagi sa pagpadala usa ka link sa pag-signup.

Pangitaa ang link

  1. Log in sa admin account.
  2. Pangita ug app sa Apps page ug i-klik ang ADMIN button.
  3. Sa Admin panel, i-klik ang Settings menu tab.
  4. Siguroha nga ang SYSTEM SETUP submenu tab gipili.

img

Pag-configure sa mga setting

Pag-scroll paubos sa USER SIGNUP nga ulohan.

Siguroha nga ang Allow Signup field gibutang sa On. (gikinahanglan) Kung kini nga bahin gipagana, ang Case Active makamugna og usa ka talagsaon nga link.

  1. I-klik ang Auto Create Case field ug pilia ang On. Ang Kaso User Role field gipakita.
  2. Pagpili ug papel. (Tan-awa sa ubos)

img

Mga papel sa tiggamit sa kaso

Adunay tulo ka mga papel sa tiggamit sa kaso.

  • mga kliyente (girekomenda)
  • default
  • kawani

Nagpadala sa link

Ipakigbahin ang link sa usa o daghang mga tawo. Ang matag user nga mag-klik sa link ug mokompleto sa pag-signup mag-trigger sa plataporma aron makamugna og bag-ong kaso sa pahina sa mga kaso ug awtomatik nga i-assign ang user sa kaso.
