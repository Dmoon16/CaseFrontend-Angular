Giunsa pagdumala sa mga Admin ang mga kaso

Adunay upat ka paagi sa pagdumala sa usa ka kaso. Pag-klik sa usa ka link aron mahibal-an kung giunsa.

Pamaagi Matang Mga aksyon Mga Detalye
Paggamit sa UI UI idugang ug usbon Butang sa Admin > Mga Kaso
Awtomatikong paghimo og kaso gamit ang public signup link Aksyon sa Gumagamit idugang Butang sa Admin > Mga Setting > Panid sa System Setup
Import nga himan UI idugang ug usbon Butang sa Admin > Mga Integrasyon > Mga Import
Paggamit sa API (sa dili madugay) API idugang ug usbon N/A
