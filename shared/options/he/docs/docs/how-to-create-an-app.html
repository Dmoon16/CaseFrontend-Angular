כיצד ליצור אפליקציה

מאמר זה מראה למשתמשים חדשים כיצד ליצור חשבון ניסיון בחינם. Case Active מעניק אוטומטית למשתמש החדש הרשאות ניהול עבור חשבון זה.

יצירת חשבון

  1. עבור לדף הכניסה .
  2. לחץ על כפתור התחל.
  3. הכנס כתובת אימייל.
  4. הקש "הבא.

img

אימות חשבון

  1. פתח לשונית דפדפן חדשה והיכנס לחשבון האימייל שלך.
  2. העתק את הקוד בן שש הספרות.
  3. חזור לכרטיסייה פעיל מקרה.
  4. הדבק את הקוד בשדות. חלון חדש מוצג.

בחלון החדש:

  1. מלא את השדות הנדרשים.
  2. קרא את תנאי השירות וסמן את התיבה כדי להסכים.
  3. לחץ על צור חשבון בחינם.

img

השלמת הגדרת האפליקציה

זרימת עבודה מוצגת בעמוד הבא. השלם את השלבים כדי לסיים את הגדרת האפליקציה שלך. אתה תמיד יכול לחזור לדפים אלה כדי לערוך או לעדכן מידע או את מצב החשבון שלך.

בדף אודות האפליקציה, מלא את שם החברה ואת שדות האתר (חובה) ולאחר מכן לחץ על הבא.

img

בדף פרטי חיוב, לחץ על השדה תוכנית תשלום וסקור את האפשרויות. הגדרת ברירת המחדל היא חינם. לחץ על שלח.

img

לאחר שליחת הטופס, Case Active מציג את דף האפליקציות. עבור כל אפליקציה שאתה יוצר או שייך אליה, הדף מציג:

  1. שם האפליקציה
  2. הסוג שלך (תפקיד משתמש)
  3. סטָטוּס

img

דף האפליקציות הוא נקודת השקה לפונקציות רבות של ניהול:

  1. לחץ על כפתור ה-APP כדי לתקשר, למלא מסמכים, וידאו צ'אט (ועוד) בנושאים הקשורים למקרה.
  2. לחץ על הלחצן ADMIN כדי לפתוח את לוח הניהול. זה המקום שבו מנהלי מערכת מנהלים משתמשים, מקרים, אינטגרציות ועוד.
  3. לחץ על הסינגל כדי לנהל את הגדרות החיוב וההתראות בחשבון.
  • הערה Case Active מציג את הלחצן ADMIN עבור משתמשי אדמין. משתמשים שאין להם הרשאות אדמין לא רואים את כפתור ADMIN (כפי שמוצג במסך למטה).
