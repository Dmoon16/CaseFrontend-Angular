עיצוב נתוני ניהול תיקים עם כלי הייבוא

להלן רשימה של שדות נדרשים בקובצי CSV לניהול מקרים עם כלי הייבוא.

שם משתמש

  • הזן את כתובת האימייל של המשתמש.

tag_id (מזהה שתוכל להתייחס אליו מאוחר יותר)

  • מזהה ייחודי (השתמש בתווים אלפאנומריים, מקפים מתקבלים)

case_status

  • 0 = סגור
  • 1 = פעיל

date_opened

  • חודש יום שנה
  • mm/dd/yyyy

על אודות

  • תיאור ייחודי של המקרה (השתמש בתווים אלפאנומריים, מקפים מתקבלים)
