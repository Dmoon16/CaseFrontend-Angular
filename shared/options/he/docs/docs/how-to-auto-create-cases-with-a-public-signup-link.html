כיצד ליצור תיקים אוטומטיים עם קישור הרשמה ציבורי

מאמר זה מראה למנהלי מערכת כיצד ליצור מקרה (ולהוסיף משתמש בו-זמנית) על ידי שליחת קישור הרשמה יחיד.

איתור הקישור

  1. היכנס לחשבון הניהול.
  2. אתר אפליקציה בדף האפליקציות ולחץ על הלחצן ADMIN.
  3. בחלונית הניהול, לחץ על הכרטיסייה תפריט הגדרות.
  4. ודא שלשונית המשנה SETUP SYSTEM נבחרה.

img

הגדרת הגדרות

גלול מטה אל הכותרת USER SIGNUP.

ודא שהשדה אפשר הרשמה מוגדר לפועל. (חובה) כאשר תכונה זו מופעלת, Case Active יוצר קישור ייחודי.

  1. לחץ על השדה צור מקרה אוטומטי ובחר מופעל. השדה 'תפקיד משתמש מקרה' מוצג.
  2. בחר תפקיד. (ראה למטה)

תפקידי משתמש במקרה

ישנם שלושה תפקידי משתמש מקרה.

  • לקוחות (מומלץ)
  • בְּרִירַת מֶחדָל
  • צוות

שולחת את הקישור

שתף את הקישור עם אדם אחד או יותר. כל משתמש שלוחץ על הקישור ומשלים את ההרשמה מפעיל את הפלטפורמה ליצור מקרה חדש בדף התיקים ולהקצות אוטומטית את המשתמש בתיק.
