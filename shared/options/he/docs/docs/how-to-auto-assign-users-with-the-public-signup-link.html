כיצד להקצות אוטומטית משתמשים באמצעות קישור ההרשמה הציבורי

מנהלי מערכת לומדים כיצד להגדיר את הגדרות המערכת כך שמשתמשים חדשים יוקצו אוטומטית למקרה חדש שנוצר אוטומטית.

פתיחת דף ההגדרות

השלב הראשון הוא פתיחת דף הגדרות המערכות.

  1. היכנס לחשבון הניהול.
  2. אתר אפליקציה בדף האפליקציות ולחץ על הלחצן ADMIN.
  3. לחץ על הכרטיסייה מערכות ולאחר מכן SETUP SYSTEM בתפריט המשנה.
  4. גלול מטה אל הכותרת USER SIGNUP.

הגדרת הגדרות מערכת

  1. הגדר את התפריטים הנפתחים אפשר הרשמה ויצירה אוטומטית של מקרה לפועל.
  2. לחץ על תפריט התפקידים ובחר אפשרות (ברירת מחדל מומלצת).
  3. העתק ושתף את הקישור הרשמה למשתמש ציבורי.

שולחת את הקישור

כל משתמש שלוחץ על הקישור ומשלים את ההרשמה מפעיל את הפלטפורמה ליצור מקרה חדש בדף המקרים ומקצה אוטומטית את המשתמש במקרה זה.
