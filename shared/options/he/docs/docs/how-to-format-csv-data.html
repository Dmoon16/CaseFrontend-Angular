כיצד לעצב נתוני CSV

להלן רשימה של שדות נתונים בקובצי CSV. לכל אפשרות ייבוא יש שילוב שונה של שדות.

שם משתמש

  • הזן את כתובת האימייל של המשתמש.

שם פרטי

  • הזן את השם הפרטי של המשתמש.

שם משפחה

  • הזן את שם המשפחה של המשתמש.

מקומי

  • עמודה זו בודקת את השפה המועדפת על המשתמש.
  • הזן en עבור אנגלית, sp עבור ספרדית.

zoneinfo

  • זה מגדיר את אזור הזמן של המשתמש. היכנס ליבשת ולעיר ללא רווחים (השתמש בקו תחתון).
  • יש להשתמש באפשרות בתפריט הנפתח פרופיל > שפה ואזור > אזור זמן.
  • השתמש באזור הזמן המתאים ביותר. לדוגמה, בוסטון לא נמצאת בתפריט הנפתח, אז אנחנו יכולים להשתמש בניו יורק (או בכל עיר באזור הזמן EST/EDT).

host_tag_id

  • הזן תווית קבועה.
  • az, 0-9, -, / מותרים.

case_status

  • 0 = סגור
  • 1 = פעיל
