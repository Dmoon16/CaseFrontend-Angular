כיצד להתחבר

למד כיצד להיכנס לחשבון Case Active הקיים שלך.

מתחבר

  1. עבור לדף הכניסה .
  2. לחץ על כפתור התחבר.
  3. הכנס את האימייל שלך.
  4. הקש "הבא.

אימות חשבון

  1. פתח לשונית דפדפן חדשה והיכנס לחשבון האימייל שלך.
  2. העתק את הקוד בן שש הספרות.
  3. חזור לכרטיסייה פעיל מקרה.
  4. הדבק את הקוד בשדות. חלון חדש מוצג.
