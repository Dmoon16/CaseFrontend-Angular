So aktualisieren Sie ein Benutzerprofil

Dieser Artikel zeigt Benutzern, wie sie Informationen auf ihrer Profilseite aktualisieren.

  • Benutzer können ihre eigenen Profile aktualisieren.
  • Administratoren können die Profilseite jedes Benutzers aktualisieren.

Für Benutzer

Der erste Schritt besteht darin, die Profilseite zu öffnen. Benutzer und Administratoren befolgen diese Schritte, um ihre eigene Profilseite zu aktualisieren.

  1. Melden Sie sich bei Ihrem Case Active-Konto an.
  2. Klicken Sie oben auf der Seite auf die Registerkarte Profil.
  3. Nehmen Sie Änderungen vor.
  4. Klicken Sie auf Speichern.

Die meisten Felder sind optional, und einige Felder sind erforderlich. Die erforderlichen Felder sind:

  • Vorname
  • Nachname
  • E-Mail-Addresse
  • Sprache
  • Zeitzone

Für Administratoren

Administratoren können auch die Profilinformationen für andere Benutzer aktualisieren. Dazu gehören drei Felder unter der Überschrift HOST USER INFO.

  1. Melden Sie sich bei Ihrem Case Active-Administratorkonto an.
  2. Suchen Sie eine App und klicken Sie auf die Schaltfläche ADMIN.
  3. Klicken Sie im oberen Menü auf Benutzer.
  4. Gehen Sie zum Untermenü und wählen Sie die Benutzerliste aus, auf die Sie zugreifen möchten (AKTIV oder DEAKTIVIERT). Die Standardeinstellung ist Aktiv.
  5. Suchen Sie einen Benutzer in der Liste und klicken Sie auf Bearbeiten.
  6. Nehmen Sie nach Bedarf Änderungen vor. (siehe unten)
  7. Klicken Sie auf Speichern.

Benutzerinformationen ändern

Administratoren können Benutzerinformationen ändern.

  1. Stellen Sie den Benutzertyp ein (Benutzer oder Manager).
  2. Setzen Sie den Benutzerstatus (Aktiv auf Deaktiviert).
  3. Geben Sie eine eindeutige Tag-ID ein. (Optional)
  4. Klicken Sie auf Speichern.
