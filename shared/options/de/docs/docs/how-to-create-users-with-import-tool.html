So erstellen Sie Benutzer mit dem Import-Tool

Dieser Artikel zeigt Administratoren, wie sie Benutzer mit dem Import-Tool erstellen.

Öffnen des Import-Tools

Administratoren können das Import-Tool verwenden, um neue Benutzer massenweise hochzuladen und vorhandene Benutzerdatensätze zu ändern.

  1. Melden Sie sich beim Administratorkonto an.
  2. Suchen Sie eine App auf der Seite Apps und klicken Sie auf die Schaltfläche ADMIN.
  3. Klicken Sie im Admin-Bereich auf die Registerkarte Integration und dann im Untermenü auf IMPORTS.
  4. Klicken Sie auf die Schaltfläche NEU. Das Dialogfeld „Importe hinzufügen“ wird in einer neuen Browser-Registerkarte angezeigt.

Bild

Aufzeichnungen erstellen

Nachdem das Dialogfeld Importe hinzufügen geöffnet wurde:

  1. Klicken Sie auf das Feld Typ und wählen Sie eine Option aus (siehe unten). Ein Link wird angezeigt.
  2. Klicken Sie auf den Link Beispiel herunterladen . Eine CSV-Datei wird auf Ihr Gerät heruntergeladen.

Als nächstes öffnen Sie die Datei und fügen Datensätze hinzu. Klicken Sie hier , um zu erfahren, wie Sie Formatdaten in der CSV-Datei verwenden.

Bild

Typ

Wenn Sie auf das Feld Typ klicken, wird ein Dropdown-Menü angezeigt. Wählen Sie eine von zwei Optionen für Massenimporte aus.

:exclamation: HINWEIS Das Dropdown-Menü hat viele Optionen, aber nur zwei beziehen sich auf das Import-Tool. Jede Option hat eine andere Beispiel-CSV-Datei.

Benutzer hinzufügen

  • Wählen Sie diese Option für einen einfachen Bulk-Upload von Benutzerdatensätzen.
  • Klicken Sie auf den Link Beispiel herunterladen, um eine CSV-Datei mit Spaltenüberschriften für den Importtyp Benutzer hinzufügen zu öffnen.

Benutzer ändern

  • Wählen Sie diese Option, um Daten für aktuelle Benutzer zu aktualisieren.
  • Klicken Sie auf den Link Beispiel herunterladen, um eine CSV-Datei mit Spaltenüberschriften für den Importtyp Benutzer ändern zu öffnen.

Hochladen der Datei

Sie können eine Datei sofort hochladen oder zu einem späteren Zeitpunkt zum Dialogfeld „Importe hinzufügen“ zurückkehren und diese Schritte ausführen.

  1. Öffnen Sie das Dialogfeld „Importe hinzufügen“.
  2. Geben Sie einen eindeutigen Namen für die CSV-Datei ein.
  3. Klicken Sie auf das Pluszeichen (+), um die Datei von Ihrem Gerät hochzuladen.
  4. Klicken Sie auf das Feld Typ und wählen Sie eine Option aus. (siehe oben)
  5. Klicken Sie auf SPEICHERN.

wird bearbeitet

Abhängig von der Anzahl der Datensätze in der Datei und der Systemlast kann die Plattform mehrere Minuten benötigen, um den Import zu verarbeiten.

Bild

Wenn der Import abgeschlossen ist, passt die Plattform Benutzerdatensätze im Benutzer -Dashboard des Administrators an (d. h. fügt hinzu, ändert oder deaktiviert sie).

Versagen

Wenn der Importvorgang fehlschlägt, gibt die Plattform eine Fehlernachricht zurück. In den meisten Fällen ist die Fehlerursache die Datenformatierung.

Bild

So sehen Sie die Fehlerursache:

  1. Klicken Sie unter der Überschrift Aktion auf Ansicht .
  2. Bewegen Sie auf der nächsten Seite die Maus über das Symbol Fehlgeschlagen. Der Tooltip zeigt den Fehler an.
  3. Gehen Sie zur CVS-Datei und passen Sie den Fehler an und beheben Sie ihn.
  4. Laden Sie die Datei erneut hoch.
