So erstellen und ändern Sie Benutzer mit der Benutzeroberfläche

Dieser Artikel zeigt Administratoren, wie sie Benutzer erstellen und Datensätze mit der Benutzeroberfläche der Plattform aktualisieren. Administratoren können auch Benutzer zu einem neuen Fall hinzufügen und vorhandene Fälle bearbeiten.

Erstellen eines Benutzers

  1. Melden Sie sich beim Administratorkonto an.
  2. Suchen Sie eine App auf der Seite Apps und klicken Sie auf die Schaltfläche ADMIN.
  3. Klicken Sie im Admin-Bereich auf die Menüregisterkarte Benutzer.
  4. Klicken Sie auf die Schaltfläche NEU. Ein neues Fenster wird angezeigt.

Bild

Füllen Sie die Felder aus und klicken Sie auf SPEICHERN. Case Active fügt den Namen der Liste der mit dieser App verknüpften Benutzer hinzu.

Bild

Ändern eines Benutzers

Um eine kleine Anzahl von Benutzerdatensätzen zu aktualisieren, verwenden Sie das Bearbeitungstool und nehmen Sie Änderungen an jedem einzelnen Benutzer vor.

  • Hinweis Um eine große Anzahl von Benutzerdatensätzen in einer Sitzung zu ändern, sollten Sie das Import-Tool verwenden .
  1. Klicken Sie im Admin-Bereich auf die Menüregisterkarte Benutzer.
  2. Suchen Sie einen Benutzer und klicken Sie auf Bearbeiten.
  3. Nehmen Sie Änderungen vor und klicken Sie auf SPEICHERN.

Bild

Nächste Schritte

Case Active sendet dem Benutzer keine E-Mail-Benachrichtigung. Administratoren sollten den Benutzer kontaktieren und ihm einen Link zur Case Active-Startseite senden. Der Benutzer klickt dann auf die Schaltfläche LOGIN und schließt den Workflow für neue Anmeldungen ab.
