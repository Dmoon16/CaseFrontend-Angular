So erstellen Sie Benutzer mit einem Anmeldelink

Dieser Artikel zeigt Administratoren, wie sie Benutzer erstellen, indem sie einen Registrierungslink senden.

Konfigurieren des Anmeldelinks

  1. Klicken Sie unten im Dialogfeld Benutzer hinzufügen auf den Link Registrierung senden . Eine neue Seite wird angezeigt.
  2. Konfigurieren Sie die Anmeldeoptionen. (siehe unten)

Bild

Stellen Sie sicher, dass das Feld Anmeldung zulassen auf Ein eingestellt ist. (erforderlich) Wenn aktiviert, generiert Case Active einen eindeutigen Link.

Überprüfen Sie die folgenden Optionen und klicken Sie dann auf das Symbol, um den Link zu kopieren. Sie können den Link auf Ihrer Website platzieren oder den Link per E-Mail oder einem anderen Kommunikationskanal an eine oder mehrere Personen senden.

Bild

Fall erstellen

Optional können Sie für jeden neuen Benutzer automatisch einen Fall erstellen und eine Benutzerrolle zuweisen.

  1. Klicken Sie auf das Feld Fall automatisch erstellen und wählen Sie Ein. Die Felder für die Kundenvorgangsbenutzerrolle werden angezeigt.
  2. Klicken Sie auf die Kundenvorgangsbenutzerrolle und wählen Sie eine Option aus. (siehe unten)

Bild

Fallbenutzerrollen

Es gibt drei Fallbenutzerrollen.

  • Kunden
  • Ursprünglich
  • Mitarbeiter

Senden des Links

Teilen Sie den Link mit einer oder mehreren Personen. Jede Person klickt auf den Link und schließt den Workflow ab, um ihr Case Active-Konto zu aktivieren. Sobald der Benutzer den Registrierungsworkflow abgeschlossen hat, fügt Case Active den Benutzer der Benutzerliste hinzu.

Bild

Nächste Schritte

Case Active sendet dem Benutzer eine E-Mail-Benachrichtigung mit dem Anmeldelink. Der Benutzer klickt auf den Link und schließt den Workflow für neue Anmeldungen ab.
