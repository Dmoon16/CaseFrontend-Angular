Formatieren von Benutzerfall-Zuweisungsdaten mit dem Import-Tool

Nachfolgend finden Sie eine Liste von Datenfeldern in CSV-Dateien. Jede Importoption hat eine andere Kombination von Feldern.

Fall-ID

  • Geben Sie den Fallbenutzer oder die Tag-ID ein.
  • Gehen Sie zur Seite "Fälle" > Bearbeiten > Tag-ID ODER Gehen Sie zur Seite "Fälle" > Bearbeiten > Fall-ID in URL.

Benutzeridentifikation

  • Geben Sie die ID oder Tag-ID des Benutzers ein.
  • Gehen Sie zur Seite Benutzer > Bearbeiten > Tag-ID ODER Gehen Sie zur Seite Benutzer > Bearbeiten > Benutzer-ID in URL.

case_role_order

  • Zelle leer lassen.

case_role_ID

  • Geben Sie ein: Kunden, Mitarbeiter oder Standard.

case_notify

  • 0 = nein
  • 1 = Ja, dem Benutzer eine E-Mail-Benachrichtigung senden.
