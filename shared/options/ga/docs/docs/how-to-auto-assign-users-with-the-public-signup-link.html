Conas Úsáideoirí a Sannadh go Uathoibríoch leis an Nasc Cláraithe Poiblí

Foghlaimíonn riarthóirí conas socruithe córais a chumrú ionas go sanntar úsáideoirí nua go huathoibríoch do chás nuachruthaithe.

Ag oscailt an leathanach socraithe

Is é an chéad chéim ná leathanach Socrú na gCóras a oscailt.

  1. Logáil isteach sa chuntas riaracháin.
  2. Aimsigh aip ar an leathanach Apps agus cliceáil ar an gcnaipe RIARACHÁIN.
  3. Cliceáil ar an táb Systems agus ansin SETUP SYSTEM ar an bhfo-roghchlár.
  4. Scrollaigh síos go dtí an ceannteideal USER SIGNUP.

Socruithe córais á chumrú

  1. Socraigh na roghchláir anuas Ceadaigh Cláraigh agus Cruthaigh Cás Uathoibríoch go On.
  2. Cliceáil ar an roghchlár ról agus roghnaigh rogha (moltar réamhshocraithe).
  3. Cóipeáil agus roinn an nasc Cláraigh Úsáideoir Poiblí.

Ag seoladh an nasc

Spreagann gach úsáideoir a chliceálann ar an nasc agus a chríochnaíonn an síniú an t-ardán chun cás nua a ghiniúint ar an leathanach Cásanna agus sannann sé an t-úsáideoir go huathoibríoch sa chás sin.
