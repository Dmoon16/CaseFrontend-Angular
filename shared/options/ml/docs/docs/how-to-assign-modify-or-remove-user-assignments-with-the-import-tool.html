ഇറക്കുമതി ടൂൾ ഉപയോഗിച്ച് ഉപയോക്തൃ അസൈൻമെന്റുകൾ എങ്ങനെ അസൈൻ ചെയ്യാം, പരിഷ്ക്കരിക്കാം അല്ലെങ്കിൽ നീക്കം ചെയ്യാം

ഇംപോർട്ട് ടൂൾ ഉപയോഗിച്ച് കേസ് ഉപയോക്താക്കളെ എങ്ങനെ കൈകാര്യം ചെയ്യാമെന്ന് ഈ ലേഖനം അഡ്‌മിനുകളെ കാണിക്കുന്നു.

അഡ്മിൻമാർക്ക് കഴിയും:

  • ഒരു കേസിൽ ഉപയോക്താക്കളെ നിയോഗിക്കുക
  • ഒരു കേസിൽ ഇതിനകം നിയോഗിച്ചിട്ടുള്ള ഉപയോക്താക്കളെ പരിഷ്ക്കരിക്കുക
  • ഒരു കേസിൽ നിയോഗിക്കപ്പെട്ട ഉപയോക്താക്കളെ നീക്കം ചെയ്യുക

ഇറക്കുമതി ഉപകരണം തുറക്കുന്നു

  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. ഉപമെനുവിലെ ഇന്റഗ്രേഷൻ ടാബിൽ ക്ലിക്ക് ചെയ്യുക, തുടർന്ന് ഇറക്കുമതി ചെയ്യുക.
  4. പുതിയ ബട്ടൺ ക്ലിക്ക് ചെയ്യുക. ഇറക്കുമതി ചേർക്കുക ഡയലോഗ് ബോക്സ് പ്രദർശിപ്പിക്കുന്നു.

റെക്കോർഡുകൾ സൃഷ്ടിക്കുന്നു

ഒന്നോ അതിലധികമോ CSV ഫയലുകൾ ഡൗൺലോഡ് ചെയ്യുക. കേസ് ഉപയോക്താക്കളെ ചേർക്കാനോ പരിഷ്‌ക്കരിക്കാനോ നീക്കം ചെയ്യാനോ ഈ ഫയലുകൾ ഉപയോഗിക്കുക. ഇറക്കുമതി ചേർക്കുക ഡയലോഗ് ബോക്സ് തുറന്ന ശേഷം:

  1. ടൈപ്പ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്യുക.
  2. ഡ്രോപ്പ്-ഡൗൺ മെനുവിലൂടെ സ്ക്രോൾ ചെയ്ത് ഒരു ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക (ചുവടെ കാണുക). ഒരു ലിങ്ക് പ്രദർശിപ്പിക്കുന്നു.
  3. ഡൗൺലോഡ് സാമ്പിൾ ലിങ്ക് ക്ലിക്ക് ചെയ്യുക. ഒരു CSV ഫയൽ നിങ്ങളുടെ ഉപകരണത്തിലേക്ക് ഡൗൺലോഡ് ചെയ്യുന്നു.

img

ഫയൽ തുറന്ന് റെക്കോർഡുകൾ ചേർക്കുക. CSV ഫയലിൽ ഫോർമാറ്റ് ഡാറ്റ എങ്ങനെ ഉപയോഗിക്കണമെന്ന് അറിയാൻ ഇവിടെ ക്ലിക്ക് ചെയ്യുക.

ഫയൽ ഓപ്ഷനുകൾ

കേസുകളിലേക്ക് ഉപയോക്താക്കളെ അസൈൻ ചെയ്യുക * ഒരു കേസിൽ ഉപയോക്താക്കളെ ചേർക്കുന്നതിന് ഈ ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക. * case_role_order സെൽ ശൂന്യമായി വിടുക. * ഉപയോക്താക്കളുടെ പേജിലേക്ക് പോയി ഈ വ്യക്തിയുടെ ടാഗ് ഐഡി പകർത്തുക. ടാഗ് ഐഡി ഫീൽഡ് ശൂന്യമാണെങ്കിൽ, ഒരു അദ്വിതീയ ഐഡി നൽകുക, സംരക്ഷിക്കുക ക്ലിക്കുചെയ്യുക, തുടർന്ന് ഐഡി CSV ഫയലിലേക്ക് പകർത്തുക.

img

കേസുകളിലേക്ക് അസൈൻ ചെയ്‌തിരിക്കുന്ന ഉപയോക്താക്കളെ പരിഷ്‌ക്കരിക്കുക * ഒരു കേസിൽ അസൈൻ ചെയ്‌തിരിക്കുന്ന നിലവിലെ ഉപയോക്താക്കൾക്കായി ഡാറ്റ അപ്‌ഡേറ്റ് ചെയ്യുന്നതിന് ഈ ഓപ്‌ഷൻ തിരഞ്ഞെടുക്കുക. * case_ID കോളത്തിനായി കേസ് ടാഗ് ഐഡി ഉപയോഗിക്കുക.

കേസുകളിൽ നിയുക്തരായ ഉപയോക്താക്കളെ നീക്കം ചെയ്യുക * ഒരു കേസിൽ നിയുക്തരായ ഉപയോക്താക്കളെ നീക്കം ചെയ്യാൻ ഈ ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക.

img

ഫയൽ അപ്‌ലോഡ് ചെയ്യുന്നു

നിങ്ങൾക്ക് ഉടൻ തന്നെ ഒരു ഫയൽ അപ്‌ലോഡ് ചെയ്യാം അല്ലെങ്കിൽ പിന്നീടുള്ള തീയതിയിൽ ചേർക്കുക ഇമ്പോർട്ടുകൾ ഡയലോഗ് ബോക്സിലേക്ക് മടങ്ങുകയും ഈ ഘട്ടങ്ങൾ പൂർത്തിയാക്കുകയും ചെയ്യാം.

  1. ഇറക്കുമതി ചേർക്കുക ഡയലോഗ് ബോക്സ് തുറക്കുക.
  2. CSV ഫയലിനായി ഒരു അദ്വിതീയ നാമം നൽകുക.
  3. നിങ്ങളുടെ ഉപകരണത്തിൽ നിന്ന് ഫയൽ അപ്‌ലോഡ് ചെയ്യാൻ പ്ലസ് (+) ഐക്കണിൽ ക്ലിക്ക് ചെയ്യുക.
  4. ടൈപ്പ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്ത് ഒരു ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക. (മുകളിൽ കാണുന്ന)
  5. സേവ് ക്ലിക്ക് ചെയ്യുക.

പ്രോസസ്സിംഗ്

ഫയലിലെയും സിസ്റ്റം ലോഡിലെയും റെക്കോർഡുകളുടെ എണ്ണത്തെ ആശ്രയിച്ച്, ഇറക്കുമതി പ്രോസസ്സ് ചെയ്യുന്നതിന് പ്ലാറ്റ്ഫോമിന് കുറച്ച് മിനിറ്റുകൾ എടുത്തേക്കാം.

ഇറക്കുമതി പൂർത്തിയാകുമ്പോൾ, കേസുകൾ പേജിലെ ഉപയോക്തൃ റെക്കോർഡുകൾ പ്ലാറ്റ്ഫോം ക്രമീകരിക്കുന്നു (അതായത് ചേർക്കുന്നു, പരിഷ്ക്കരിക്കുന്നു അല്ലെങ്കിൽ നീക്കംചെയ്യുന്നു).

പരാജയം

ഇറക്കുമതി പ്രവർത്തനം പരാജയപ്പെടുകയാണെങ്കിൽ, പ്ലാറ്റ്ഫോം ഒരു പരാജയപ്പെട്ട സന്ദേശം നൽകുന്നു. മിക്ക കേസുകളിലും, പരാജയത്തിന്റെ കാരണം ഡാറ്റ ഫോർമാറ്റിംഗ് ആണ്.

img

പരാജയത്തിന്റെ കാരണം കാണാൻ:

  1. പ്രവർത്തന തലക്കെട്ടിന് കീഴിലുള്ള കാണുക ക്ലിക്ക് ചെയ്യുക.
  2. അടുത്ത പേജിൽ, പരാജയപ്പെട്ട ഐക്കണിൽ മൗസ് ഹോവർ ചെയ്യുക. ടൂൾടിപ്പ് പിശക് കാണിക്കുന്നു.
  3. CVS ഫയലിലേക്ക് പോയി ക്രമീകരിച്ച് പിശക് പരിഹരിക്കുക.
  4. ഫയൽ വീണ്ടും അപ്‌ലോഡ് ചെയ്യുക.
