ഒരു ഉപയോക്തൃ പ്രൊഫൈൽ എങ്ങനെ അപ്ഡേറ്റ് ചെയ്യാം

ഉപയോക്താക്കൾക്ക് അവരുടെ പ്രൊഫൈൽ പേജിലെ വിവരങ്ങൾ എങ്ങനെ അപ്‌ഡേറ്റ് ചെയ്യാമെന്ന് ഈ ലേഖനം കാണിക്കുന്നു.

  • ഉപയോക്താക്കൾക്ക് അവരുടെ സ്വന്തം പ്രൊഫൈലുകൾ അപ്ഡേറ്റ് ചെയ്യാം.
  • അഡ്‌മിനുകൾക്ക് ഏത് ഉപയോക്താവിന്റെയും പ്രൊഫൈൽ പേജ് അപ്‌ഡേറ്റ് ചെയ്യാൻ കഴിയും.

ഉപയോക്താക്കൾക്കായി

പ്രൊഫൈൽ പേജ് തുറക്കുക എന്നതാണ് ആദ്യപടി. ഉപയോക്താക്കളും അഡ്‌മിനുകളും അവരുടെ സ്വന്തം പ്രൊഫൈൽ പേജ് അപ്‌ഡേറ്റ് ചെയ്യുന്നതിന് ഈ ഘട്ടങ്ങൾ പാലിക്കുന്നു.

  1. നിങ്ങളുടെ കേസ് ആക്റ്റീവ് അക്കൗണ്ടിലേക്ക് ലോഗിൻ ചെയ്യുക.
  2. പേജിന്റെ മുകളിലുള്ള പ്രൊഫൈൽ ടാബിൽ ക്ലിക്ക് ചെയ്യുക.
  3. മാറ്റങ്ങൾ വരുത്തുക.
  4. സേവ് ക്ലിക്ക് ചെയ്യുക.

മിക്ക ഫീൽഡുകളും ഓപ്ഷണലാണ്, ചില ഫീൽഡുകൾ ആവശ്യമാണ്. ആവശ്യമായ ഫീൽഡുകൾ ഇവയാണ്:

  • പേരിന്റെ ആദ്യഭാഗം
  • പേരിന്റെ അവസാന ഭാഗം
  • ഈ - മെയില് വിലാസം
  • ഭാഷ
  • സമയ മേഖല

അഡ്മിൻമാർക്ക്

മറ്റ് ഉപയോക്താക്കൾക്ക് പ്രൊഫൈൽ വിവരങ്ങൾ അപ്‌ഡേറ്റ് ചെയ്യാനും അഡ്മിൻമാർക്ക് കഴിയും. ഇതിൽ HOST USER INFO എന്ന തലക്കെട്ടിന് കീഴിലുള്ള മൂന്ന് ഫീൽഡുകൾ ഉൾപ്പെടുന്നു.

  1. നിങ്ങളുടെ കേസ് ആക്റ്റീവ് അഡ്മിൻ അക്കൗണ്ടിലേക്ക് ലോഗിൻ ചെയ്യുക.
  2. ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. മുകളിലെ മെനുവിലെ ഉപയോക്താക്കളെ ക്ലിക്ക് ചെയ്യുക.
  4. ഉപമെനുവിലേക്ക് പോയി നിങ്ങൾ ആക്സസ് ചെയ്യാൻ ആഗ്രഹിക്കുന്ന ഉപയോക്തൃ ലിസ്റ്റ് തിരഞ്ഞെടുക്കുക (സജീവമോ പ്രവർത്തനരഹിതമോ). സ്ഥിരസ്ഥിതി ക്രമീകരണം സജീവമാണ്.
  5. ലിസ്റ്റിൽ ഒരു ഉപയോക്താവിനെ കണ്ടെത്തി എഡിറ്റ് ക്ലിക്ക് ചെയ്യുക.
  6. ആവശ്യാനുസരണം മാറ്റങ്ങൾ വരുത്തുക. (താഴെ നോക്കുക)
  7. സേവ് ക്ലിക്ക് ചെയ്യുക.

അഡ്‌മിന് ഉപയോക്തൃ വിവരങ്ങൾ മാറ്റാനാകും.

  3. ഒരു അദ്വിതീയ ടാഗ് ഐഡി നൽകുക. (ഓപ്ഷണൽ)
  4. സേവ് ക്ലിക്ക് ചെയ്യുക.
