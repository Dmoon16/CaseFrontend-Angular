എങ്ങനെ ലോഗിൻ ചെയ്യാം

നിങ്ങളുടെ നിലവിലുള്ള കേസ് ആക്റ്റീവ് അക്കൗണ്ടിലേക്ക് എങ്ങനെ ലോഗിൻ ചെയ്യാമെന്ന് അറിയുക.

ലോഗിൻ ചെയ്യുന്നു

  1. ലോഗിൻ പേജിലേക്ക് പോകുക.
  2. LOGIN ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. നിങ്ങളുടെ ഇമെയിൽ നൽകുക.
  4. അടുത്തത് ക്ലിക്ക് ചെയ്യുക.

ഒരു അക്കൗണ്ട് പരിശോധിക്കുന്നു

  1. ഒരു പുതിയ ബ്രൗസർ ടാബ് തുറന്ന് നിങ്ങളുടെ ഇമെയിൽ അക്കൗണ്ടിലേക്ക് ലോഗിൻ ചെയ്യുക.
  2. ആറ് അക്ക കോഡ് പകർത്തുക.
  3. കേസ് ആക്റ്റീവ് ടാബിലേക്ക് മടങ്ങുക.
  4. ഫീൽഡുകളിൽ കോഡ് ഒട്ടിക്കുക. ഒരു പുതിയ വിൻഡോ ദൃശ്യമാകുന്നു.
