അഡ്മിൻ എങ്ങനെയാണ് ഉപയോക്താക്കളെ സൃഷ്ടിക്കുന്നത്

ആപ്പുകളും കേസുകളും ആർക്കൊക്കെ ആക്‌സസ് ചെയ്യാനാകുമെന്നത് അഡ്മിൻ സൃഷ്‌ടിക്കുകയോ പരിഷ്‌ക്കരിക്കുകയോ പ്രവർത്തനരഹിതമാക്കുകയോ ചെയ്യുന്നു. ഒരു ഉപയോക്താവ് ആദ്യമായി പ്ലാറ്റ്‌ഫോമിൽ ലോഗിൻ ചെയ്യുമ്പോൾ, അവർ ഒരു പ്രൊഫൈൽ വർക്ക്ഫ്ലോ പൂർത്തിയാക്കേണ്ടതുണ്ട്.

  • പുതിയ ഉപയോക്തൃ പ്രൊഫൈൽ വർക്ക്ഫ്ലോയെക്കുറിച്ച് കൂടുതലറിയാൻ ഇവിടെ ക്ലിക്ക് ചെയ്യുക.

ഒരു ഉപയോക്താവിനെ സൃഷ്ടിക്കുന്നു

താഴെയുള്ള പട്ടിക ഒരു ഉപയോക്താവിനെ സൃഷ്ടിക്കുന്നതിനുള്ള അഞ്ച് വഴികൾ കാണിക്കുന്നു. എങ്ങനെയെന്നറിയാൻ ഒരു ലിങ്ക് ക്ലിക്ക് ചെയ്യുക.

ഒരു ആപ്പിൽ ചേർക്കുമ്പോൾ പ്ലാറ്റ്‌ഫോം ഉപയോക്താക്കൾക്ക് അറിയിപ്പ് അയയ്‌ക്കില്ല. എന്നിരുന്നാലും, അഡ്‌മിനുകൾക്ക് കേസ് അസൈൻമെന്റുകൾക്കായി അറിയിപ്പുകൾ പ്രവർത്തനക്ഷമമാക്കാനാകും.
