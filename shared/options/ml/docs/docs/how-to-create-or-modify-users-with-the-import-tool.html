ഇറക്കുമതി ടൂൾ ഉപയോഗിച്ച് ഉപയോക്താക്കളെ എങ്ങനെ സൃഷ്ടിക്കാം അല്ലെങ്കിൽ പരിഷ്ക്കരിക്കാം

ഇറക്കുമതി ടൂൾ ഉപയോഗിച്ച് ഉപയോക്താക്കളെ എങ്ങനെ സൃഷ്ടിക്കാമെന്ന് ഈ ലേഖനം അഡ്‌മിനുകളെ കാണിക്കുന്നു.

ഇറക്കുമതി ഉപകരണം തുറക്കുന്നു

പുതിയ ഉപയോക്താക്കളെ ബൾക്ക് അപ്‌ലോഡ് ചെയ്യുന്നതിനും നിലവിലുള്ള ഉപയോക്തൃ റെക്കോർഡുകൾ പരിഷ്‌ക്കരിക്കുന്നതിനും അഡ്മിൻമാർക്ക് ഇംപോർട്ട് ടൂൾ ഉപയോഗിക്കാം.

  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. അഡ്മിൻ പാനലിൽ, ഇന്റഗ്രേഷൻ ടാബിൽ ക്ലിക്ക് ചെയ്യുക, തുടർന്ന് ഉപമെനുവിലെ ഇറക്കുമതി ചെയ്യുക.
  4. പുതിയ ബട്ടൺ ക്ലിക്ക് ചെയ്യുക. ഇമ്പോർട്ടുകൾ ചേർക്കുക ഡയലോഗ് ബോക്സ് ഒരു പുതിയ ബ്രൗസർ ടാബിൽ പ്രദർശിപ്പിക്കുന്നു.

റെക്കോർഡുകൾ സൃഷ്ടിക്കുന്നു

ഇറക്കുമതി ചേർക്കുക ഡയലോഗ് ബോക്സ് തുറന്ന ശേഷം:

  1. ടൈപ്പ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്ത് ഒരു ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക (ചുവടെ കാണുക). ഒരു ലിങ്ക് പ്രദർശിപ്പിക്കുന്നു.
  2. ഡൗൺലോഡ് സാമ്പിൾ ലിങ്ക് ക്ലിക്ക് ചെയ്യുക. ഒരു CSV ഫയൽ നിങ്ങളുടെ ഉപകരണത്തിലേക്ക് ഡൗൺലോഡ് ചെയ്യുന്നു.

അടുത്തതായി, ഫയൽ തുറന്ന് റെക്കോർഡുകൾ ചേർക്കുക. CSV ഫയലിൽ ഫോർമാറ്റ് ഡാറ്റ എങ്ങനെ ഉപയോഗിക്കണമെന്ന് അറിയാൻ ഇവിടെ ക്ലിക്ക് ചെയ്യുക.

ടൈപ്പ് ചെയ്യുക

നിങ്ങൾ ടൈപ്പ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഒരു ഡ്രോപ്പ്-ഡൗൺ മെനു ദൃശ്യമാകും. ബൾക്ക് ഇമ്പോർട്ടുകൾക്കായി രണ്ട് ഓപ്ഷനുകളിൽ ഒന്ന് തിരഞ്ഞെടുക്കുക.

:ആശ്ചര്യപ്പെടുത്തൽ: ശ്രദ്ധിക്കുക ഡ്രോപ്പ്-ഡൗൺ മെനുവിൽ നിരവധി ഓപ്ഷനുകൾ ഉണ്ട്, എന്നാൽ രണ്ടെണ്ണം മാത്രമേ ഇറക്കുമതി ടൂളുമായി ബന്ധപ്പെട്ടിട്ടുള്ളൂ. ഓരോ ഓപ്‌ഷനും വ്യത്യസ്‌തമായ സാമ്പിൾ CSV ഫയലുണ്ട്.

ഉപയോക്താക്കളെ ചേർക്കുക

  • ഉപയോക്തൃ റെക്കോർഡുകളുടെ ലളിതമായ ബൾക്ക് അപ്‌ലോഡിനായി ഈ ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക.
  • ഉപയോക്താക്കളെ ചേർക്കുക ഇമ്പോർട്ട് തരത്തിനായുള്ള കോളം തലക്കെട്ടുകളുള്ള ഒരു CSV ഫയൽ തുറക്കാൻ ഡൗൺലോഡ് സാമ്പിൾ ലിങ്കിൽ ക്ലിക്ക് ചെയ്യുക.

ഉപയോക്താക്കളെ പരിഷ്ക്കരിക്കുക

  • നിലവിലെ ഉപയോക്താക്കൾക്കായി ഡാറ്റ അപ്ഡേറ്റ് ചെയ്യാൻ ഈ ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക.
  • ഉപയോക്താക്കളെ മോഡിഫൈ ചെയ്യുക ഇമ്പോർട്ട് തരത്തിനായുള്ള കോളം തലക്കെട്ടുകളുള്ള ഒരു CSV ഫയൽ തുറക്കാൻ ഡൗൺലോഡ് സാമ്പിൾ ലിങ്കിൽ ക്ലിക്ക് ചെയ്യുക.

ഫയൽ അപ്‌ലോഡ് ചെയ്യുന്നു

നിങ്ങൾക്ക് ഉടൻ തന്നെ ഒരു ഫയൽ അപ്‌ലോഡ് ചെയ്യാം അല്ലെങ്കിൽ പിന്നീടുള്ള തീയതിയിൽ ചേർക്കുക ഇമ്പോർട്ടുകൾ ഡയലോഗ് ബോക്സിലേക്ക് മടങ്ങുകയും ഈ ഘട്ടങ്ങൾ പൂർത്തിയാക്കുകയും ചെയ്യാം.

  1. ഇറക്കുമതി ചേർക്കുക ഡയലോഗ് ബോക്സ് തുറക്കുക.
  2. CSV ഫയലിനായി ഒരു അദ്വിതീയ നാമം നൽകുക.
  3. നിങ്ങളുടെ ഉപകരണത്തിൽ നിന്ന് ഫയൽ അപ്‌ലോഡ് ചെയ്യാൻ പ്ലസ് (+) ഐക്കണിൽ ക്ലിക്ക് ചെയ്യുക.
  4. ടൈപ്പ് ഫീൽഡിൽ ക്ലിക്ക് ചെയ്ത് ഒരു ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക. (മുകളിൽ കാണുന്ന)
  5. സേവ് ക്ലിക്ക് ചെയ്യുക.

പ്രോസസ്സിംഗ്

ഫയലിലെയും സിസ്റ്റം ലോഡിലെയും റെക്കോർഡുകളുടെ എണ്ണത്തെ ആശ്രയിച്ച്, ഇറക്കുമതി പ്രോസസ്സ് ചെയ്യുന്നതിന് പ്ലാറ്റ്ഫോമിന് കുറച്ച് മിനിറ്റ് എടുക്കാം.

img

ഇറക്കുമതി പൂർത്തിയാകുമ്പോൾ, അഡ്‌മിന്റെ ഉപയോക്താക്കളുടെ ഡാഷ്‌ബോർഡിൽ പ്ലാറ്റ്‌ഫോം ഉപയോക്തൃ റെക്കോർഡുകൾ ക്രമീകരിക്കുന്നു (അതായത് ചേർക്കുന്നു, പരിഷ്‌ക്കരിക്കുന്നു അല്ലെങ്കിൽ പ്രവർത്തനരഹിതമാക്കുന്നു).

പരാജയം

ഇറക്കുമതി പ്രവർത്തനം പരാജയപ്പെടുകയാണെങ്കിൽ, പ്ലാറ്റ്ഫോം ഒരു പരാജയപ്പെട്ട സന്ദേശം നൽകുന്നു. മിക്ക കേസുകളിലും, പരാജയത്തിന്റെ കാരണം ഡാറ്റ ഫോർമാറ്റിംഗ് ആണ്.

പരാജയത്തിന്റെ കാരണം കാണാൻ:

  1. പ്രവർത്തന തലക്കെട്ടിന് കീഴിലുള്ള കാണുക ക്ലിക്ക് ചെയ്യുക.
  2. അടുത്ത പേജിൽ, പരാജയപ്പെട്ട ഐക്കണിൽ മൗസ് ഹോവർ ചെയ്യുക. ടൂൾടിപ്പ് പിശക് കാണിക്കുന്നു.
  3. CVS ഫയലിലേക്ക് പോയി ക്രമീകരിച്ച് പിശക് പരിഹരിക്കുക.
  4. ഫയൽ വീണ്ടും അപ്‌ലോഡ് ചെയ്യുക.
