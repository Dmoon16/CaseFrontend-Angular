UI മെനുവിലൂടെ കേസുകൾ എങ്ങനെ സൃഷ്ടിക്കാം അല്ലെങ്കിൽ പരിഷ്ക്കരിക്കാം

പ്ലാറ്റ്‌ഫോമിന്റെ ഉപയോക്തൃ ഇന്റർഫേസ് ഉപയോഗിച്ച് എങ്ങനെ ഒരു കേസ് സൃഷ്‌ടിക്കാമെന്ന് ഈ ലേഖനം അഡ്‌മിനുകളെ കാണിക്കുന്നു. അഡ്‌മിനുകൾക്ക് ഒരു പുതിയ കേസിലേക്ക് ഉപയോക്താക്കളെ ചേർക്കാനും നിലവിലുള്ള കേസുകൾ അപ്‌ഡേറ്റ് ചെയ്യാനും കഴിയും.

ഒരു കേസ് സൃഷ്ടിക്കുന്നു

  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. അഡ്മിൻ പാനലിൽ, കേസുകൾ മെനു ടാബിൽ ക്ലിക്ക് ചെയ്യുക.
  4. പുതിയ ബട്ടൺ ക്ലിക്ക് ചെയ്യുക. ഒരു പുതിയ വിൻഡോ ദൃശ്യമാകുന്നു.

ആവശ്യമായ ഫീൽഡുകൾ പൂരിപ്പിച്ച് സേവ് ക്ലിക്ക് ചെയ്യുക.

ഉപയോക്താക്കളെ ചേർക്കുന്നു

ഓപ്ഷണലായി, അഡ്‌മിന് ഈ പേജിലെ കേസിൽ ഉപയോക്താക്കളെ ചേർക്കാൻ കഴിയും. താഴേക്ക് സ്ക്രോൾ ചെയ്ത് ഉപയോക്താക്കളെ നിയോഗിക്കുക ബട്ടൺ ക്ലിക്ക് ചെയ്യുക.

അസൈൻ യൂസർ മോഡൽ എന്നതിൽ, ഫീൽഡുകൾ പൂരിപ്പിക്കുക. ഉപയോക്താവിന് ഇമെയിൽ അറിയിപ്പ് അയയ്‌ക്കാൻ ബോക്‌സ് ചെക്കുചെയ്യുക. (ഓപ്ഷണൽ)

ഒരു കേസ് പരിഷ്കരിക്കുന്നു

കുറച്ച് കേസുകൾ അപ്ഡേറ്റ് ചെയ്യുന്നതിന്, എഡിറ്റ് ടൂൾ ഉപയോഗിച്ച് ഓരോ വ്യക്തിഗത കേസിലും മാറ്റങ്ങൾ വരുത്തുക.

  • കുറിപ്പ് ഒറ്റ സിറ്റിങ്ങിൽ ധാരാളം കേസുകൾ അപ്ഡേറ്റ് ചെയ്യുന്നതിന്, ഇറക്കുമതി ടൂൾ ഉപയോഗിക്കുന്നത് പരിഗണിക്കുക
  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. അഡ്മിൻ പാനലിൽ, കേസുകൾ മെനു ടാബിൽ ക്ലിക്ക് ചെയ്യുക.
  4. ഒരു കേസ് കണ്ടെത്തി എഡിറ്റ് ക്ലിക്ക് ചെയ്യുക.
  5. മാറ്റങ്ങൾ വരുത്തി സേവ് ക്ലിക്ക് ചെയ്യുക.
