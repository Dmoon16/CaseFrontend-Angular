ഒരു ആപ്പ് എങ്ങനെ സൃഷ്ടിക്കാം

ഒരു സൗജന്യ ട്രയൽ അക്കൗണ്ട് എങ്ങനെ സൃഷ്ടിക്കാമെന്ന് ഈ ലേഖനം പുതിയ ഉപയോക്താക്കളെ കാണിക്കുന്നു. Case Active ഈ അക്കൗണ്ടിനായി പുതിയ ഉപയോക്തൃ അഡ്മിൻ പ്രത്യേകാവകാശങ്ങൾ സ്വയമേവ നൽകുന്നു.

ഒരു അക്കൗണ്ട് സൃഷ്ടിക്കുന്നു

  1. ലോഗിൻ പേജിലേക്ക് പോകുക.
  2. ആരംഭിക്കുക ബട്ടൺ ക്ലിക്ക് ചെയ്യുക.
  3. ഒരു ഇമെയിൽ വിലാസം നൽകുക.
  4. അടുത്തത് ക്ലിക്ക് ചെയ്യുക.

ഒരു അക്കൗണ്ട് പരിശോധിക്കുന്നു

  1. ഒരു പുതിയ ബ്രൗസർ ടാബ് തുറന്ന് നിങ്ങളുടെ ഇമെയിൽ അക്കൗണ്ടിലേക്ക് ലോഗിൻ ചെയ്യുക.
  2. ആറ് അക്ക കോഡ് പകർത്തുക.
  3. കേസ് ആക്റ്റീവ് ടാബിലേക്ക് മടങ്ങുക.
  4. ഫീൽഡുകളിൽ കോഡ് ഒട്ടിക്കുക. ഒരു പുതിയ വിൻഡോ ദൃശ്യമാകുന്നു.

പുതിയ വിൻഡോയിൽ:

  1. ആവശ്യമായ ഫീൽഡുകൾ പൂരിപ്പിക്കുക.
  2. സേവന നിബന്ധനകൾ വായിച്ച് അംഗീകരിക്കാൻ ബോക്സ് ചെക്കുചെയ്യുക.
  3. ഒരു സൗജന്യ അക്കൗണ്ട് സൃഷ്ടിക്കുക ക്ലിക്ക് ചെയ്യുക.

ആപ്പ് സജ്ജീകരണം പൂർത്തിയാക്കുന്നു

അടുത്ത പേജിൽ ഒരു വർക്ക്ഫ്ലോ പ്രദർശിപ്പിക്കുന്നു. നിങ്ങളുടെ ആപ്പ് സജ്ജീകരിക്കുന്നത് പൂർത്തിയാക്കാനുള്ള ഘട്ടങ്ങൾ പൂർത്തിയാക്കുക. വിവരങ്ങൾ അല്ലെങ്കിൽ നിങ്ങളുടെ അക്കൗണ്ട് നില എഡിറ്റ് ചെയ്യാനോ അപ്ഡേറ്റ് ചെയ്യാനോ നിങ്ങൾക്ക് എല്ലായ്പ്പോഴും ഈ പേജുകളിലേക്ക് മടങ്ങാം.

ആപ്പിനെക്കുറിച്ച് പേജിൽ, കമ്പനിയുടെ പേരും വെബ്‌സൈറ്റ് ഫീൽഡുകളും പൂരിപ്പിക്കുക (ആവശ്യമാണ്) തുടർന്ന് അടുത്തത് ക്ലിക്കുചെയ്യുക.

ബില്ലിംഗ് വിവര പേജിൽ, പേയ്‌മെന്റ് പ്ലാൻ ഫീൽഡിൽ ക്ലിക്ക് ചെയ്ത് ഓപ്ഷനുകൾ അവലോകനം ചെയ്യുക. സ്ഥിരസ്ഥിതി ക്രമീകരണം സൗജന്യമാണ്. സമർപ്പിക്കുക ക്ലിക്ക് ചെയ്യുക.

img

നിങ്ങൾ ഫോം സമർപ്പിച്ച ശേഷം, Case Active ആപ്പ് പേജ് പ്രദർശിപ്പിക്കുന്നു. നിങ്ങൾ സൃഷ്‌ടിക്കുന്ന അല്ലെങ്കിൽ ഉൾപ്പെടുന്ന ഓരോ ആപ്പിനും, പേജ് കാണിക്കുന്നത്:

  1. ആപ്പിന്റെ പേര്
  2. നിങ്ങളുടെ തരം (ഉപയോക്തൃ റോൾ)
  3. പദവി

നിരവധി അഡ്‌മിൻ ഫംഗ്‌ഷനുകൾക്കായുള്ള ഒരു ലോഞ്ച് പോയിന്റാണ് ആപ്‌സ് പേജ്:

  1. കേസുമായി ബന്ധപ്പെട്ട വിഷയങ്ങളിൽ ആശയവിനിമയം നടത്താനും ഡോക്‌സ് പൂരിപ്പിക്കാനും വീഡിയോ ചാറ്റുചെയ്യാനും (കൂടുതൽ കൂടുതൽ) APP ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  2. അഡ്മിൻ പാനൽ തുറക്കാൻ അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക. ഇവിടെയാണ് അഡ്‌മിനുകൾ ഉപയോക്താക്കൾ, കേസുകൾ, സംയോജനങ്ങൾ എന്നിവയും മറ്റും നിയന്ത്രിക്കുന്നത്.
  3. അക്കൗണ്ട് ബില്ലിംഗും അറിയിപ്പ് ക്രമീകരണങ്ങളും നിയന്ത്രിക്കാൻ എലിപ്‌സിസ് ക്ലിക്ക് ചെയ്യുക.
  • നോട്ട് കേസ് ആക്റ്റീവ് അഡ്‌മിൻ ഉപയോക്താക്കൾക്കായി അഡ്‌മിൻ ബട്ടൺ പ്രദർശിപ്പിക്കുന്നു. അഡ്‌മിൻ പ്രിവിലേജുകൾ ഇല്ലാത്ത ഉപയോക്താക്കൾ അഡ്‌മിൻ ബട്ടൺ കാണില്ല (ചുവടെയുള്ള സ്‌ക്രീനിൽ കാണിച്ചിരിക്കുന്നത് പോലെ).
