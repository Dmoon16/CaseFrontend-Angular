അഡ്‌മിനുകൾ എങ്ങനെയാണ് ഉപയോക്തൃ അസൈൻമെന്റുകൾ കൈകാര്യം ചെയ്യുന്നത്

ഉപയോക്താക്കളെയും കേസ് അസൈൻമെന്റുകളെയും നിയന്ത്രിക്കുന്നതിന് അഞ്ച് വഴികളുണ്ട്. എങ്ങനെയെന്നറിയാൻ ഒരു ലിങ്ക് ക്ലിക്ക് ചെയ്യുക.

രീതി ടൈപ്പ് ചെയ്യുക പ്രവർത്തനങ്ങൾ വിശദാംശങ്ങൾ
ഒരു കേസ് സൃഷ്ടിക്കുമ്പോൾ UI ചേർക്കുക അഡ്മിൻ ബട്ടൺ > കേസുകൾ > പുതിയ ബട്ടൺ > ഉപയോക്താക്കളെ അസൈൻ ചെയ്യുക ബട്ടൺ
പൊതു സൈൻഅപ്പ് ലിങ്ക് ഉപയോഗിച്ച് ഉപയോക്താവിനെ സ്വയമേവ നിയോഗിക്കുക ഉപയോക്തൃ പ്രവർത്തനം ചേർക്കുക അഡ്മിൻ ബട്ടൺ > ക്രമീകരണങ്ങൾ > സിസ്റ്റം സജ്ജീകരണ പേജ്
നിലവിലുള്ള കേസിൽ UI ഉപയോഗിക്കുന്നു UI ചേർക്കുക & നീക്കം ചെയ്യുക അഡ്മിൻ ബട്ടൺ > കേസുകൾ > ഉപയോക്താക്കളെ അസൈൻ ചെയ്യുക
ഇറക്കുമതി ഉപകരണം UI ചേർക്കുക, പരിഷ്ക്കരിക്കുക, നീക്കം ചെയ്യുക അഡ്മിൻ ബട്ടൺ > സംയോജനങ്ങൾ > ഇറക്കുമതികൾ
