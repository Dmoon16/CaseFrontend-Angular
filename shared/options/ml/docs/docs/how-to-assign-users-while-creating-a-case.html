ഒരു കേസ് സൃഷ്ടിക്കുമ്പോൾ ഉപയോക്താക്കളെ എങ്ങനെ അസൈൻ ചെയ്യാം

ഒരു പുതിയ കേസ് സൃഷ്ടിക്കുമ്പോൾ ഒരു ഉപയോക്താവിനെ എങ്ങനെ അസൈൻ ചെയ്യാമെന്ന് ഈ ലേഖനം അഡ്‌മിനുകളെ കാണിക്കുന്നു. ഉപയോക്താവ് ഇതിനകം പ്ലാറ്റ്‌ഫോമിൽ രജിസ്റ്റർ ചെയ്തിട്ടില്ലെങ്കിൽ, ഒരു പുതിയ കേസ് സൃഷ്‌ടിക്കുമ്പോൾ നിങ്ങൾക്ക് ഒരു പുതിയ ഉപയോക്താവിനെ സൃഷ്‌ടിക്കാം.

ഒരു കേസ് സൃഷ്ടിക്കുന്നു

  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. കേസുകൾ മെനു ടാബിൽ ക്ലിക്ക് ചെയ്യുക.
  4. പുതിയ ബട്ടൺ ക്ലിക്ക് ചെയ്യുക. ഒരു പുതിയ വിൻഡോ ദൃശ്യമാകുന്നു.
  • ഒരു കേസ് സൃഷ്‌ടിക്കാൻ ഫീൽഡുകൾ പൂരിപ്പിക്കുക, തുടർന്ന് അസൈൻ ചെയ്‌ത ഉപയോക്താക്കളുടെ തലക്കെട്ടിലേക്ക് സ്‌ക്രോൾ ചെയ്യുക.

ഒരു ഉപയോക്താവിനെ നിയമിക്കുന്നു

നിലവിലുള്ള ഉപയോക്താവ്

നിലവിലെ ഉപയോക്താവിനെ പുതിയ കേസിലേക്ക് അസൈൻ ചെയ്യാൻ ഈ ഘട്ടങ്ങൾ പാലിക്കുക.

ഉപയോക്താക്കളെ നിയോഗിക്കുക ബട്ടൺ ക്ലിക്ക് ചെയ്ത് ഫീൽഡുകൾ പൂരിപ്പിക്കുക.

  1. ഉപയോക്തൃ ഡ്രോപ്പ്-ഡൌണിൽ ക്ലിക്ക് ചെയ്ത് ഒരു ഐഡന്റിഫയർ തിരഞ്ഞെടുക്കുക (ഉദാ. ഇമെയിൽ).
  2. ശൂന്യമായ ഫീൽഡിൽ ഒരു മൂല്യം നൽകുക (ഉദാ. ഇമെയിൽ വിലാസം).
  3. പ്ലാറ്റ്‌ഫോമിൽ ഉപയോക്തൃ റെക്കോർഡ് നിലവിലുണ്ടെങ്കിൽ, ഫീൽഡ് ഒരു പേര് സ്വയമേവ നിർദ്ദേശിക്കുന്നു.
  4. ശരിയായ ഉപയോക്താവിനെ തിരഞ്ഞെടുക്കുക.
  5. ഒരു ഉപയോക്തൃ റോൾ തിരഞ്ഞെടുക്കുക (ക്ലയന്റുകൾ, ഡിഫോൾട്ട് അല്ലെങ്കിൽ സ്റ്റാഫ്).
  6. ഓപ്ഷണലായി, ഉപയോക്താവിന് ഇമെയിൽ അറിയിപ്പ് അയയ്‌ക്കാൻ ബോക്‌സ് ചെക്ക് ചെയ്യുക.
  7. സേവ് ക്ലിക്ക് ചെയ്യുക.

കേസ് ആക്റ്റീവ് ഉപയോക്താവിനെ കേസിൽ ചേർക്കുന്നു. വിൻഡോ അടയ്‌ക്കാനോ ഒരു പുതിയ ഉപയോക്താവിനെ ചേർക്കാനോ ഒരു ഉപയോക്താവിനെ നീക്കം ചെയ്യാനോ ഇപ്പോൾ അഡ്മിൻമാർക്ക് സംരക്ഷിക്കുക എന്നതിൽ ക്ലിക്ക് ചെയ്യാം.

ഒരു പുതിയ ഉപയോക്താവിനെ ചേർക്കുന്നു

പ്ലാറ്റ്‌ഫോമിന് നിലവിലെ റെക്കോർഡ് കണ്ടെത്താൻ കഴിയുന്നില്ലെങ്കിൽ, ഒരു പുതിയ ഉപയോക്താവിനെ ചേർക്കാൻ ബട്ടൺ ക്ലിക്കുചെയ്യുക.
