യുഐ ഉപയോഗിച്ച് ഉപയോക്താക്കളെ എങ്ങനെ സൃഷ്‌ടിക്കുകയും പരിഷ്‌ക്കരിക്കുകയും ചെയ്യാം

പ്ലാറ്റ്‌ഫോമിന്റെ ഉപയോക്തൃ ഇന്റർഫേസ് ഉപയോഗിച്ച് ഉപയോക്താക്കളെ എങ്ങനെ സൃഷ്‌ടിക്കാമെന്നും റെക്കോർഡുകൾ അപ്‌ഡേറ്റ് ചെയ്യാമെന്നും ഈ ലേഖനം അഡ്മിൻമാരെ കാണിക്കുന്നു. അഡ്‌മിനുകൾക്ക് ഒരു പുതിയ കേസിലേക്ക് ഉപയോക്താക്കളെ ചേർക്കാനും നിലവിലുള്ള കേസുകൾ എഡിറ്റ് ചെയ്യാനും കഴിയും.

ഒരു ഉപയോക്താവിനെ സൃഷ്ടിക്കുന്നു

  1. അഡ്മിൻ അക്കൗണ്ടിൽ ലോഗിൻ ചെയ്യുക.
  2. ആപ്പ് പേജിൽ ഒരു ആപ്പ് കണ്ടെത്തി അഡ്‌മിൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.
  3. അഡ്മിൻ പാനലിൽ, ഉപയോക്താക്കളുടെ മെനു ടാബിൽ ക്ലിക്ക് ചെയ്യുക.
  4. പുതിയ ബട്ടൺ ക്ലിക്ക് ചെയ്യുക. ഒരു പുതിയ വിൻഡോ ദൃശ്യമാകുന്നു.

ഫീൽഡുകൾ പൂരിപ്പിച്ച് സേവ് ക്ലിക്ക് ചെയ്യുക. ഈ ആപ്പുമായി ബന്ധപ്പെട്ട ഉപയോക്താക്കളുടെ ലിസ്റ്റിലേക്ക് കേസ് ആക്റ്റീവ് പേര് ചേർക്കുന്നു.

ഒരു ഉപയോക്താവിനെ പരിഷ്ക്കരിക്കുന്നു

കുറച്ച് ഉപയോക്തൃ റെക്കോർഡുകൾ അപ്‌ഡേറ്റ് ചെയ്യുന്നതിന്, എഡിറ്റ് ടൂൾ ഉപയോഗിച്ച് ഓരോ ഉപയോക്താവിനും മാറ്റങ്ങൾ വരുത്തുക.

  • കുറിപ്പ് ഒരു സിറ്റിങ്ങിൽ ധാരാളം ഉപയോക്തൃ റെക്കോർഡുകൾ പരിഷ്കരിക്കുന്നതിന്, ഇറക്കുമതി ഉപകരണം ഉപയോഗിക്കുന്നത് പരിഗണിക്കുക.
  1. അഡ്മിൻ പാനലിൽ, ഉപയോക്താക്കളുടെ മെനു ടാബിൽ ക്ലിക്ക് ചെയ്യുക.
  2. ഒരു ഉപയോക്താവിനെ കണ്ടെത്തി എഡിറ്റ് ക്ലിക്ക് ചെയ്യുക.
  3. മാറ്റങ്ങൾ വരുത്തി സേവ് ക്ലിക്ക് ചെയ്യുക.

Case Active ഉപയോക്താവിന് ഇമെയിൽ അറിയിപ്പ് അയയ്‌ക്കുന്നില്ല. അഡ്മിനുകൾ ഉപയോക്താവിനെ ബന്ധപ്പെടുകയും അവർക്ക് കേസ് ആക്റ്റീവ് ഹോം പേജിലേക്ക് ഒരു ലിങ്ക് അയയ്ക്കുകയും വേണം. തുടർന്ന് ഉപയോക്താവ് ലോഗിൻ ബട്ടൺ ക്ലിക്കുചെയ്‌ത് പുതിയ സൈനപ്പുകൾക്കായുള്ള വർക്ക്ഫ്ലോ പൂർത്തിയാക്കുന്നു.

  • ഈ ലേഖനം പുതിയ ഉപയോക്താക്കൾക്കുള്ള സൈൻഅപ്പ് വർക്ക്ഫ്ലോ വിവരിക്കുന്നു.
