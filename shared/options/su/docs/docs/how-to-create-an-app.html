Kumaha Jieun hiji App

Tulisan ieu nunjukkeun pangguna anyar kumaha cara ngadamel akun percobaan gratis. Case Active otomatis masihan hak husus ka admin pamaké anyar pikeun akun ieu.

Nyieun rekening

  1. Pindah ka kaca login .
  2. Klik tombol Mimitian.
  3. Asupkeun alamat surélék.
  4. Klik Next.

img

Verifikasi hiji akun

  1. Buka tab pangotektak anyar sareng log in kana akun email anjeun.
  2. Salin kode genep digit.
  3. Balik deui ka tab Case Active.
  4. Témpélkeun kodeu kana widang. Jandéla anyar némbongan.

Dina jandela anyar:

  1. Eusian widang nu diperlukeun.
  2. Baca sarat ladenan sareng centang kotak pikeun satuju.
  3. Klik BUAT AKUN GRATIS.

img

Ngarengsekeun setelan aplikasi

A workflow mintonkeun dina kaca salajengna. Lengkepan léngkah-léngkah pikeun ngabéréskeun nyetél aplikasi anjeun. Anjeun salawasna tiasa balik deui ka kaca ieu pikeun ngédit atanapi ngapdet inpormasi atanapi status akun anjeun.

Dina halaman Ngeunaan App, eusian nami perusahaan sareng halaman wéb (diperyogikeun) teras klik Teras.

img

Dina halaman Inpormasi Tagihan, klik widang Rencana Pembayaran sareng marios pilihan. Setélan standar nyaéta Gratis. Pencét KIRIM.

Saatos Anjeun ngalebetkeun formulir, Case Active mintonkeun kaca Aplikasi. Pikeun unggal aplikasi anu anjeun jieun atanapi milik, halamanna nunjukkeun:

  1. Ngaran aplikasi
  2. Jenis anjeun (peran pamaké)
  3. Status

img

Halaman Aplikasi mangrupikeun titik peluncuran pikeun seueur fungsi admin:

  1. Pencét tombol APP pikeun komunikasi, eusian dokumén, obrolan video (sareng seueur deui) ngeunaan masalah anu aya hubunganana.
  2. Pencét tombol ADMIN pikeun muka panel admin. Ieu dimana Admin ngatur pangguna, kasus, integrasi, sareng seueur deui.
  3. Klik elipsis pikeun ngatur tagihan akun sareng setelan bewara.
  • Catetan Case Active mintonkeun tombol ADMIN pikeun pamaké admin. Pamaké anu henteu ngagaduhan hak istimewa admin henteu ningali tombol ADMIN (sapertos anu dipidangkeun dina layar di handap).
