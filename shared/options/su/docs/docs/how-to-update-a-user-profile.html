Kumaha Ngamutahirkeun Propil Pamaké

Tulisan ieu nunjukkeun pangguna kumaha ngapdet inpormasi dina halaman Propilna.

  • Pamaké bisa ngamutahirkeun propil sorangan.
  • Admins tiasa ngapdet halaman Profil pangguna mana waé.

Pikeun pamaké

Léngkah munggaran nyaéta muka halaman Propil. Pamaké sareng admin nuturkeun léngkah-léngkah ieu pikeun ngapdet halaman Propil sorangan.

  1. Asup ka akun Case Active Anjeun.
  2. Klik tab Propil di luhureun kaca.
  3. Jieun parobahan.
  4. Pencét Simpen.

Kaseueuran widang opsional, sareng sababaraha widang diperyogikeun. Widang anu diperyogikeun nyaéta:

  • Nami payun
  • Nami pengker
  • Alamat email
  • Basa
  • Zona waktu

Pikeun admin

Admins ogé bisa ngamutahirkeun informasi profil pikeun pamaké séjén. Ieu ngawengku tilu widang dina judul HOST USER INFO.

  1. Asup ka akun admin Case Active Anjeun.
  2. Milarian aplikasi sareng klik tombol ADMIN.
  3. Pencét Pamaké dina ménu luhur.
  4. Pindah ka sub-menu sareng pilih daptar pangguna anu anjeun hoyong aksés (AKTIF atanapi TURUN). Setélan standar nyaéta Aktif.
  5. Milarian pangguna dina daptar teras klik Édit.
  6. Jieun parobahan sakumaha diperlukeun. (tingali kahandap)
  7. Pencét Simpen.

Robah inpo pamaké

Admin tiasa ngarobih inpormasi pangguna.

  1. Setel Jenis Pamaké (Pamaké atanapi Manajer).
  2. Setel Status Pamaké (Aktip ka Disabled).
  3. Asupkeun ID Tag unik. (opsional)
  4. Pencét Simpen.
