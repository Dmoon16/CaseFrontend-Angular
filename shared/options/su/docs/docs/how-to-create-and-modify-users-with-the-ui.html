Kumaha Jieun sareng Ngarobih Pamaké nganggo UI

Tulisan ieu nunjukkeun ka admin kumaha cara nyiptakeun pangguna sareng ngapdet rékaman nganggo antarmuka pangguna platform. Admin ogé tiasa nambihan pangguna kana kasus énggal sareng ngédit kasus anu tos aya.

Nyiptakeun pangguna

  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Dina panel Admin, klik tab menu Pamaké.
  4. Klik tombol NEW. Jandéla anyar némbongan.

img

Eusian widang teras klik SAVE. Case Active nambihan nami kana daptar pangguna anu aya hubunganana sareng aplikasi ieu.

img

Ngaropéa pamaké

Pikeun ngapdet sajumlah leutik rékaman pangguna, paké alat Édit sareng jieun perobahan ka unggal pangguna.

  • Catetan Pikeun ngarobih sajumlah ageung rékaman pangguna dina hiji waktos, pertimbangkeun nganggo alat Impor .
  1. Dina panel Admin, klik tab menu Pamaké.
  2. Milarian pangguna teras klik Édit.
  3. Jieun parobahan teras klik SAVE.

img

Léngkah saterusna

Case Active henteu ngirimkeun bewara email ka pangguna. Admins kedah ngahubungan pamaké sarta ngirim aranjeunna link ka kaca imah Case Active. Pamaké teras ngaklik tombol LOGIN sareng ngalengkepan alur kerja pikeun pendaptaran énggal.

  • Tulisan ieu ngajelaskeun alur kerja pendaptaran pikeun pangguna énggal.
