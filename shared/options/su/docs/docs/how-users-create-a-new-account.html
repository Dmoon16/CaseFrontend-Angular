Kumaha Pamaké Nyiptakeun Akun Anyar

Saatos admin nambihan pangguna kana pasualan: * upami pangguna anyar kana caseactive, pangguna kedah ngalengkepan alur kerja pikeun nyiptakeun akun sareng nampi uleman pikeun ngaksés kasus éta. * upami pangguna gaduh akun caseactive anu tos aya, pangguna kedah nampi uleman pikeun ngaksés kasus éta.

Artikel ieu ngajelaskeun léngkah-léngkah anu kedah dilakukeun ku pangguna pikeun nyiptakeun akunna.

Ngamimitian

Aya dua cara pangguna tiasa nambihan kana aplikasi caseactive

  • Ku admin: Upami admin nyiptakeun pangguna ngalangkungan UI platform atanapi alat Impor , admin kedah ngirimkeun tautan ka akun ka account.caseactive.com pikeun ngaléngkah login.
  • Ku pamaké: Lamun pamaké nyieun hiji akun maké link signup , pamaké bakal otomatis dialihkeun ka account.caseactive.com pikeun neruskeun login.

Asup

Nalika pangguna asup kana akun CaseActive na pikeun kahiji kalina, platform na nyarankeun pangguna pikeun:

  • Buka akun surelekna teras salin/témpél kode kaamanan genep digit
  • ngalengkepan hiji profil akun

img

Saatos pangguna ngaklik UPDATE, halaman Aplikasi nampilkeun. Halamanna nunjukkeun nami aplikasi (ngaran perusahaan), jinis pangguna (peran), status, sareng tombol UNDANGAN.

img

Pikeun ngaksés aplikasi, pangguna kedah nampi uleman. Pamaké kedah:

  1. Pencét tombol UNDANGAN.
  2. Pariksa idin.
  3. Klik tab Bewara pikeun marios sareng ngédit (opsional) istilah.
  4. Klik APPROVE ACCESS.

Nalika pangguna nyatujuan, platform nampilkeun tombol APP dina halaman pangguna. Tombol ieu masihan aksés pangguna kana kasus anu ditugaskeun.

Pikeun ngédit setélan pangguna, pangguna ngaklik élipsis.

img

Pamaké tiasa: * ngahapus aksés aplikasi * ngarobih setélan béwara

Saatos pangguna ngalengkepan alur kerja, nami pangguna muncul dina daptar Pamaké admin. Daptar nunjukkeun sadaya pangguna, asup atanapi henteu.
