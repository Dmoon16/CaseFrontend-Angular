Kumaha Admins Nyiptakeun Pamaké

Admins nyiptakeun, ngarobih atanapi nganonaktipkeun anu tiasa ngaksés aplikasi sareng kasus. Nalika pangguna pertama kali asup kana platform, aranjeunna kedah ngalengkepan alur kerja profil.

  • Klik di dieu pikeun leuwih jéntré ngeunaan workflow profil pamaké anyar.

Nyiptakeun pangguna

Tabel di handap nembongkeun lima cara pikeun nyieun pamaké. Pencét tautan pikeun diajar kumaha.

| Métode | Tipe | Tindakan | Rincian | Pamaké anyar nampi béwara? |
|------------------------|----------------|------- ---------------------|--------------------|---------------------- -------------------------|

Ngagunakeun UI | UI | tambahkeun & ngaropéa | Tombol Admin > Pamaké | Henteu |

Nalika nyiptakeun kasus sareng napelkeun pangguna kana kasus | UI | tambahkeun | Tombol Admin > Kasus > Tombol anyar > Jieun kaca Kasus | Leres, upami diaktipkeun |

Otomatis nyieun pamaké maké link signup publik | Peta pamaké | tambahkeun | Tombol Admin > Setélan > Kaca Setup Sistem | Leres, upami kasus jieun otomatis diaktipkeun |

Alat impor | UI | tambahkeun & ngaropéa | Tombol Admin > Integrasi > Impor | Henteu |

Ngagunakeun API (hadir) | API | tambahkeun & ngaropéa | na | na |

Bewara

Platformna henteu ngirimkeun bewara ka pangguna nalika aranjeunna ditambahkeun kana aplikasi. Nanging, admin tiasa ngaktipkeun bewara pikeun tugas kasus.
