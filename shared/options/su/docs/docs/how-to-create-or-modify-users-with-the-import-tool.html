Kumaha Nyiptakeun atanapi Ngarobih Pamaké nganggo Alat Impor

Tulisan ieu nunjukkeun ka admin kumaha cara nyiptakeun pangguna nganggo alat Impor.

Buka alat Impor

Admins tiasa nganggo alat Impor pikeun unggah seueur pangguna énggal sareng ngarobih rékaman pangguna anu tos aya.

  1. Asup ka akun admin.
  2. Milarian aplikasi dina halaman Aplikasi teras klik tombol ADMIN.
  3. Dina panel Admin, klik tab Integrasi teras IMPOR dina submenu.
  4. Klik tombol NEW. Kotak dialog Tambah Impor dipintonkeun dina tab browser anyar.

img

Nyieun rékaman

Saatos kotak dialog Add Imports dibuka:

  1. Klik widang Tipe tur pilih hiji pilihan (tingali di handap). Hiji link mintonkeun.
  2. Pencét tautan Unduh Sampel . A file CSV diundeur ka alat Anjeun.

Teras, buka file sareng tambahkeun rékaman. Klik di dieu pikeun neuleuman kumaha carana make data format dina file CSV.

img

Tipe

Nalika anjeun ngaklik médan Tipe, ménu turun-handap dipintonkeun. Pilih salah sahiji tina dua pilihan pikeun impor bulk.

: exclamation: CATETAN Menu turun-handap boga loba pilihan, tapi ngan dua nu patali jeung alat Impor. Unggal pilihan gaduh sampel file CSV anu béda.

Tambahkeun pamaké

  • Milih pilihan ieu pikeun unggah bulk basajan tina rékaman pamaké.
  • Klik tautan Unduh Sampel pikeun muka file CSV kalayan judul kolom pikeun jinis Impor Tambah pangguna.

Ngaropéa pamaké

  • Pilih pilihan ieu pikeun ngapdet data pikeun pangguna ayeuna.
  • Klik tautan Unduh Sampel pikeun muka file CSV sareng judul kolom pikeun jinis Impor pangguna Modifikasi.

Ngunggah file

Anjeun tiasa unggah file langsung atanapi uih deui ka kotak dialog Tambah Impor dina waktos engké sareng ngalengkepan léngkah-léngkah ieu.

  1. Buka kotak dialog Tambah Impor.
  2. Lebetkeun nami unik pikeun file CSV.
  3. Pencét ikon tambah (+) pikeun unggah file tina alat anjeun.
  4. Klik widang Tipe tur pilih hiji pilihan. (tingali luhureun)
  5. Pencét SAVE.

img

Ngolah

Gumantung kana jumlah rékaman dina file sareng beban sistem, platform tiasa nyandak sababaraha menit pikeun ngolah impor.

img

Nalika impor parantos réngsé, platform nyaluyukeun rékaman pangguna (nyaéta nambihan, ngarobih, atanapi nganonaktipkeun) dina dasbor Pamaké admin.

Kagagalan

Lamun operasi impor gagal, platform nu mulih pesen Gagal. Dina kalolobaan kasus, anu ngabalukarkeun gagalna nyaéta pormat data.

img

Pikeun ningali sabab gagalna:

  1. Klik Témbongkeun dina judul Aksi.
  2. Dina kaca salajengna, hover mouse dina ikon Gagal. tooltip mintonkeun kasalahan.
  3. Pindah ka file CVS sareng saluyukeun sareng ngalereskeun kasalahan.
  4. Unggah deui filena.
