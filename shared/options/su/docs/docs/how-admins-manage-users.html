Kumaha Admins Ngatur Pamaké

Admins nyiptakeun, ngarobih atanapi nganonaktipkeun anu tiasa ngaksés aplikasi sareng kasus. Nalika pangguna asup kana platform pikeun kahiji kalina, aranjeunna kedah ngalengkepan alur kerja profil.

  • Klik di dieu pikeun leuwih jéntré ngeunaan workflow profil pamaké anyar.

Ngatur hiji pamaké

Tabel di handap nembongkeun lima cara pikeun ngatur pamaké. Pencét tautan pikeun diajar kumaha.

Métode Tipe lampah Rincian
Ngagunakeun UI UI nambahkeun & ngaropéa Tombol Admin > Pamaké
Nalika nyiptakeun pasualan sareng pasihan pangguna kana kasus UI nambahan Tombol Admin> Kasus> Tombol Anyar> Jieun halaman Kasus
Otomatis nyiptakeun pangguna nganggo tautan pendaptaran umum Peta pamaké nambahan Tombol Admin > Setélan > Halaman Setup Sistem
Alat impor UI nambahkeun & ngaropéa Tombol Admin > Integrasi > Impor
Ngagunakeun API (geus datang) API nambahkeun & ngaropéa N/A

Bewara

Platformna henteu ngirimkeun bewara ka pangguna nalika aranjeunna ditambahkeun kana aplikasi. Nanging, admin tiasa ngaktipkeun bewara pikeun tugas kasus.
