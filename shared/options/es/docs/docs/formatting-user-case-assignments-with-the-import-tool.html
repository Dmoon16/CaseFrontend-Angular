Formateo de datos de asignaciones de casos de usuario con la herramienta de importación

A continuación se muestra una lista de campos de datos en archivos CSV. Cada opción de importación tiene una combinación diferente de campos.

Identificación del caso

  • Ingrese el ID del usuario o de la etiqueta del caso.
  • Vaya a la página Casos > Editar > ID de etiqueta O Vaya a la página Casos > Editar > ID de caso en url.

ID_usuario

  • Ingrese la identificación del usuario o la identificación de la etiqueta.
  • Vaya a la página Usuarios > Editar > ID de etiqueta O Vaya a la página Usuarios > Editar > ID de usuario en url.

case_role_order

  • Deje la celda en blanco.

case_role_ID

  • Introduzca: clientes, personal o predeterminado.

caso_notificar

  • 0 = no
  • 1 = Sí, enviar al usuario una notificación por correo electrónico.
