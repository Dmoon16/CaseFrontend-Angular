Cómo asignar automáticamente usuarios con el enlace de registro público

Los administradores aprenden a configurar los ajustes del sistema para que los nuevos usuarios se asignen automáticamente a un caso creado automáticamente.

Apertura de la página de configuración

El primer paso es abrir la página Configuración de sistemas.

  1. Inicie sesión en la cuenta de administrador.
  2. Busque una aplicación en la página Aplicaciones y haga clic en el botón ADMINISTRAR.
  3. Haga clic en la pestaña Sistemas y luego en CONFIGURACIÓN DEL SISTEMA en el submenú.
  4. Desplácese hacia abajo hasta el encabezado REGISTRO DE USUARIO.

Configuración de los ajustes del sistema

  1. Establezca los menús desplegables Permitir registro y Creación automática de casos en Activado.
  2. Haga clic en el menú de funciones y elija una opción (se recomienda la opción predeterminada).
  3. Copie y comparta el enlace de registro de usuario público.

Enviando el enlace

Cada usuario que hace clic en el enlace y completa el registro activa la plataforma para generar un nuevo caso en la página Casos y asigna automáticamente al usuario en ese caso.
