Cómo los usuarios crean una nueva cuenta

Después de que un administrador agregue un usuario a un caso: * si el usuario es nuevo en caseactive, el usuario debe completar un flujo de trabajo para crear una cuenta y aceptar la invitación para acceder al caso. * si el usuario tiene una cuenta caseactive existente, el usuario debe aceptar la invitación para acceder al caso.

Este artículo describe los pasos que un usuario debe completar para crear su cuenta.

Empezando

Hay dos formas en que se puede agregar un usuario a una aplicación caseactive

  • Por administrador: si el administrador crea un usuario a través de la interfaz de usuario de la plataforma o la herramienta de importación , el administrador debe enviar al usuario un enlace a account.caseactive.com para continuar con el inicio de sesión.
  • Por usuario: si el usuario creó una cuenta usando un enlace de registro , el usuario será redirigido automáticamente a account.caseactive.com para iniciar sesión.

Iniciando sesión

Cuando el usuario inicia sesión en su cuenta CaseActive por primera vez, la plataforma le pide al usuario que:

  • abra su cuenta de correo electrónico y luego copie/pegue el código de seguridad de seis dígitos
  • completar un perfil de cuenta

Después de que el usuario haga clic en ACTUALIZAR, se muestra la página Aplicaciones. La página muestra el nombre de la aplicación (nombre de la empresa), el tipo de usuario (función), el estado y un botón de INVITACIÓN.

imagen

Para acceder a la aplicación, el usuario debe aceptar la invitación. El usuario debe:

  1. Haga clic en el botón INVITACIÓN.
  2. Revisa los permisos.
  3. Haga clic en la pestaña Notificaciones para revisar y editar (opcional) los términos.
  4. Haga clic en APROBAR ACCESO.

imagen

Cuando el usuario aprueba, la plataforma muestra un botón de APLICACIÓN en la página del usuario. Este botón da acceso al usuario a los casos asignados.

Para editar la configuración del usuario, el usuario hace clic en los puntos suspensivos.

imagen

El usuario puede: * eliminar el acceso a la aplicación * modificar la configuración de las notificaciones

imagen

Una vez que el usuario completa el flujo de trabajo, el nombre de usuario aparece en la lista de usuarios del administrador. La lista muestra todos los usuarios, hayan iniciado sesión o no.
