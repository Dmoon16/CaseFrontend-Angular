ئىشلەتكۈچى ئارخىپىنى قانداق يېڭىلاش كېرەك

بۇ ماقالە ئىشلەتكۈچىلەرنىڭ ئارخىپ بېتىدىكى ئۇچۇرلارنى قانداق يېڭىلاشنى كۆرسىتىپ بېرىدۇ.

  • ئابونتلار ئۆزلىرىنىڭ ئارخىپىنى يېڭىلىيالايدۇ.
  • باشقۇرغۇچى ھەر قانداق ئىشلەتكۈچىنىڭ ئارخىپ بېتىنى يېڭىلىيالايدۇ.

ئىشلەتكۈچىلەر ئۈچۈن

بىرىنچى قەدەم: ئارخىپ بېتىنى ئېچىش. ئابونتلار ۋە باشقۇرغۇچىلار ئۆزلىرىنىڭ ئارخىپ بېتىنى يېڭىلاش ئۈچۈن بۇ باسقۇچلارنى قوللىنىدۇ.

  1. دېلو ئاكتىپ ھېساباتىڭىزغا كىرىڭ.
  2. بەتنىڭ ئۈستىدىكى ئارخىپ بەتكۈچىنى چېكىڭ.
  3. ئۆزگەرتىش.
  4. ساقلاش نى چېكىڭ.

كۆپىنچە بۆلەكلەر ئىختىيارى ، بەزى ساھەلەر تەلەپ قىلىنىدۇ. لازىملىق ساھە:

  • ئىسمى
  • فامىلىسى
  • ئېلخەت ئادرېسى
  • تىل
  • ۋاقىت رايونى

باشقۇرغۇچىلار ئۈچۈن

باشقۇرغۇچىلار يەنە باشقا ئىشلەتكۈچىلەرنىڭ ئارخىپ ئۇچۇرلىرىنى يېڭىلىيالايدۇ. بۇ HOST USER INFO ماۋزۇسىدىكى ئۈچ ساھەنى ئۆز ئىچىگە ئالىدۇ.

  1. دېلو ئاكتىپ باشقۇرغۇچى ھېساباتىڭىزغا كىرىڭ.
  2. بىر ئەپنى تېپىپ ADMIN كۇنۇپكىسىنى بېسىڭ.
  3. ئۈستىدىكى تىزىملىكتىكى ئىشلەتكۈچىنى چېكىڭ.
  4. تارماق تىزىملىككە كىرىپ ، زىيارەت قىلماقچى بولغان ئىشلەتكۈچى تىزىملىكىنى تاللاڭ (ACTIVE ياكى DISABLED). سۈكۈتتىكى تەڭشىكى ئاكتىپ.
  5. تىزىملىكتىكى ئىشلەتكۈچىنى تېپىپ «تەھرىرلەش» نى چېكىڭ.
  6. تەلەپكە ئاساسەن ئۆزگەرتىش كىرگۈزۈڭ. (تۆۋەنگە قاراڭ)
  7. ساقلاش نى چېكىڭ.

ئىشلەتكۈچى ئۇچۇرىنى ئۆزگەرتىڭ

باشقۇرغۇچىلار ئىشلەتكۈچى ئۇچۇرلىرىنى ئۆزگەرتەلەيدۇ.

  1. ئىشلەتكۈچى تۈرىنى (ئىشلەتكۈچى ياكى باشقۇرغۇچى) بەلگىلەڭ.
  2. ئىشلەتكۈچى ھالىتىنى تەڭشەڭ (چەكلەنگەنلەرگە ئاكتىپ).
  3. ئۆزگىچە بەلگە كىملىكىنى كىرگۈزۈڭ. (ئىختىيارىي)
  4. ساقلاش نى چېكىڭ.
