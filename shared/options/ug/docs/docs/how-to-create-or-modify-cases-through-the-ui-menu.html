UI تىزىملىكى ئارقىلىق دېلولارنى قانداق قۇرۇش ياكى ئۆزگەرتىش

بۇ ماقالە سۇپىنىڭ ئىشلەتكۈچى كۆرۈنمە يۈزى بىلەن قانداق قىلىپ دېلو تۇرغۇزۇشنى كۆرسىتىدۇ. باشقۇرغۇچى يەنە يېڭى ئەھۋالغا ئىشلەتكۈچى قوشالايدۇ ۋە مەۋجۇت ئەھۋاللارنى يېڭىلىيالايدۇ.

دېلو تۇرغۇزۇش

  1. باشقۇرغۇچى ھېساباتىغا كىرىڭ.
  2. ئەپلەر بېتىدىن بىر ئەپنى تېپىپ ، ADMIN كۇنۇپكىسىنى بېسىڭ.
  3. باشقۇرۇش تاختىسىدا «دېلو تىزىملىكى» بەتكۈچىنى چېكىڭ.
  4. يېڭى كۇنۇپكىنى بېسىڭ. يېڭى كۆزنەك كۆرسىتىلىدۇ.

ئىشلەتكۈچى قوشۇش

ئىختىيارىي ھالدا ، باشقۇرغۇچىلار بۇ بەتتىكى ئەھۋالغا ئىشلەتكۈچى قوشالايدۇ. ئاستىغا بېسىپ ASSIGN USERS كۇنۇپكىسىنى بېسىڭ.

img

ئىشلەتكۈچى تەقسىملەش مودېلىدا ، بۇ يەرنى تولدۇرۇڭ. ئىشلەتكۈچىگە ئېلېكترونلۇق خەت ئۇقتۇرۇشى ئەۋەتىش ئۈچۈن رامكىنى تەكشۈرۈڭ. (ئىختىيارىي)

img

دېلونى ئۆزگەرتىش

ئاز ساندىكى ئەھۋاللارنى يېڭىلاش ئۈچۈن «تەھرىرلەش قورالى» نى ئىشلىتىپ ، ھەر بىر يەككە ئەھۋالغا ئۆزگەرتىش كىرگۈزۈڭ.

  • ئەسكەرتىش بىر ئولتۇرۇشتا نۇرغۇن ئەھۋاللارنى يېڭىلاش ئۈچۈن ، ئىمپورت قورالنى ئىشلىتىشنى ئويلاڭ
  1. باشقۇرغۇچى ھېساباتىغا كىرىڭ.
  2. ئەپلەر بېتىدىن بىر ئەپنى تېپىپ ، ADMIN كۇنۇپكىسىنى بېسىڭ.
  3. باشقۇرۇش تاختىسىدا «دېلو تىزىملىكى» بەتكۈچىنى چېكىڭ.
  4. بىر ئەھۋالنى تېپىپ «تەھرىرلەش» نى چېكىڭ.
  5. ئۆزگەرتىش ئېلىپ «ساقلاش» نى چېكىڭ.
