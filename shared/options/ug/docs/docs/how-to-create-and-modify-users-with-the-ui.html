UI ئارقىلىق ئابونتلارنى قانداق قۇرۇش ۋە ئۆزگەرتىش

بۇ ماقالىدە سۇپىنىڭ ئىشلەتكۈچى كۆرۈنمە يۈزى ئارقىلىق ئابونتلارنى قانداق قۇرۇش ۋە خاتىرىلەرنى يېڭىلاش ئۇسۇلى كۆرسىتىلدى. باشقۇرغۇچىلار يەنە يېڭى ئەھۋالغا ئىشلەتكۈچى قوشالايدۇ ۋە مەۋجۇت ئەھۋاللارنى تەھرىرلىيەلەيدۇ.

ئىشلەتكۈچى قۇرۇش

  1. باشقۇرغۇچى ھېساباتىغا كىرىڭ.
  2. ئەپلەر بېتىدىن بىر ئەپنى تېپىپ ، ADMIN كۇنۇپكىسىنى بېسىڭ.
  3. باشقۇرۇش تاختىسىدا «ئىشلەتكۈچى تىزىملىكى» بەتكۈچىنى چېكىڭ.
  4. يېڭى كۇنۇپكىنى بېسىڭ. يېڭى كۆزنەك كۆرسىتىلىدۇ.

ئېتىزلارنى تولدۇرۇپ «ساقلاش» نى چېكىڭ. Case Active بۇ دېتال بىلەن مۇناسىۋەتلىك ئابونتلار تىزىملىكىگە ئىسىم قوشىدۇ.

img

ئىشلەتكۈچىنى ئۆزگەرتىش

ئاز ساندىكى ئىشلەتكۈچى خاتىرىسىنى يېڭىلاش ئۈچۈن «تەھرىرلەش قورالى» نى ئىشلىتىپ ، ھەر بىر ئىشلەتكۈچىگە ئۆزگەرتىش كىرگۈزۈڭ.

  • ئەسكەرتىش بىر ئولتۇرۇشتا نۇرغۇن ئىشلەتكۈچى خاتىرىسىنى ئۆزگەرتىش ئۈچۈن ، ئىمپورت قورالنى ئىشلىتىشنى ئويلاڭ.
  1. باشقۇرۇش تاختىسىدا «ئىشلەتكۈچى تىزىملىكى» بەتكۈچىنى چېكىڭ.
  2. ئىشلەتكۈچىنى ئىزدەپ «تەھرىرلەش» نى چېكىڭ.
  3. ئۆزگەرتىش ئېلىپ «ساقلاش» نى چېكىڭ.

img

كېيىنكى باسقۇچلار

Case Active ئىشلەتكۈچىگە ئېلېكترونلۇق خەت ئۇقتۇرۇشى ئەۋەتمەيدۇ. باشقۇرغۇچىلار ئىشلەتكۈچى بىلەن ئالاقىلىشىپ ، ئۇلارغا «ئاكتىپ ئاكتىپ» باش بېتىگە ئۇلىنىش ئەۋەتىشى كېرەك. ئىشلەتكۈچى ئاندىن LOGIN كۇنۇپكىسىنى چېكىپ يېڭى تىزىملىتىپ كىرىش جەريانىنى تاماملايدۇ.

  • بۇ ماقالە يېڭى ئىشلەتكۈچىلەرنىڭ تىزىمغا ئالدۇرۇش خىزمەت جەريانىنى تەسۋىرلەيدۇ.
