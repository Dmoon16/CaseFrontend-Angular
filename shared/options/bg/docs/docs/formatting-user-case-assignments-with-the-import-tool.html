Форматиране на данни за присвояване на потребителски случаи с инструмента за импортиране

По-долу е даден списък с полета с данни в CSV файлове. Всяка опция за импортиране има различна комбинация от полета.

case_ID

  • Въведете полето User или Tag ID.
  • Отидете на страницата Cases > Edit > Tag ID ИЛИ Отидете на страница Cases > Edit > Case ID в url.

user_ID

  • Въведете ID на потребителя или ID на етикет.
  • Отидете на страница Потребители > Редактиране > Идентификационен номер на маркер ИЛИ Отидете на страница Потребители > Редактиране > Идентификатор на потребител в url.

case_role_order

  • Оставете клетката празна.

case_role_ID

  • Въведете: клиенти, персонал или по подразбиране.

case_notify

  • 0 = не
  • 1 = Да, изпратете на потребителя имейл известие.
