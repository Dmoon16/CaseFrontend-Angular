አስተዳዳሪዎች የተጠቃሚ ምደባዎችን ለጉዳይ እንዴት እንደሚያስተዳድሩ

ተጠቃሚዎችን እና ጉዳዮችን ለማስተዳደር አምስት መንገዶች አሉ። እንዴት እንደሆነ ለማወቅ አገናኙን ጠቅ ያድርጉ።

ዘዴ ዓይነት ድርጊቶች ዝርዝሮች
ጉዳይ በሚፈጥሩበት ጊዜ ዩአይ ጨምር የአስተዳዳሪ አዝራር > ጉዳዮች > አዲስ አዝራር > የተጠቃሚዎችን መድብ አዝራር
ይፋዊ የመመዝገቢያ አገናኝን በመጠቀም ተጠቃሚን በራስ-ሰር መድቡ የተጠቃሚ እርምጃ ጨምር የአስተዳዳሪ ቁልፍ > መቼቶች > የስርዓት ማዋቀር ገጽ
አሁን ባለው ጉዳይ ላይ UI መጠቀም ዩአይ ያክሉ እና ያስወግዱ የአስተዳዳሪ አዝራር > ጉዳዮች > ተጠቃሚዎችን መድብ
መሳሪያ አስመጣ ዩአይ ይጨምሩ ፣ ያሻሽሉ እና ያስወግዱ የአስተዳዳሪ አዝራር > ውህደት > አስመጪ
ኤፒአይን መጠቀም (በቅርብ ጊዜ) ኤፒአይ ይጨምሩ ፣ ያሻሽሉ እና ያስወግዱ ኤን/ኤ
