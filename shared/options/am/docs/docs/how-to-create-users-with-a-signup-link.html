ይህ ጽሑፍ አስተዳዳሪዎች የምዝገባ አገናኝ በመላክ ተጠቃሚዎችን እንዴት መፍጠር እንደሚችሉ ያሳያል።

  2. የምዝገባ አማራጮችን ያዋቅሩ። (ከታች ይመልከቱ)

መመዝገቢያ ፍቀድ መስኩ እንዲበራ ያረጋግጡ። (የሚያስፈልግ) ሲነቃ ኬዝ አክቲቭ ልዩ አገናኝ ያመነጫል።

ከታች ያሉትን አማራጮች ይገምግሙ እና ከዚያ አገናኙን ለመቅዳት አዶውን ጠቅ ያድርጉ። አገናኙን በድር ጣቢያዎ ላይ ማስቀመጥ ወይም ሊንኩን ለአንድ ወይም ለብዙ ሰዎች በኢሜል ወይም በማንኛውም የመገናኛ ጣቢያ መላክ ይችላሉ.

ሊንኩን ለአንድ ወይም ከዚያ በላይ ሰዎች ያካፍሉ። እያንዳንዱ ሰው አገናኙን ጠቅ ያደርጋል እና የጉዳይ ገቢር መለያቸውን ለማግበር የስራ ሂደቱን ያጠናቅቃል። አንዴ ተጠቃሚው የምዝገባ የስራ ሂደትን ካጠናቀቀ በኋላ ኬዝ አክቲቭ ተጠቃሚውን ወደ የተጠቃሚዎች ዝርዝር ያክላል።
