ጉዳዮችን መፍጠር

የማስመጣት አክል የንግግር ሳጥን ከተከፈተ በኋላ፡-

  1. ዓይነት መስኩን ጠቅ ያድርጉ እና አንድ አማራጭ ይምረጡ (ከዚህ በታች ይመልከቱ)። አንድ አገናኝ ያሳያል.
  2. የማውረጃ ናሙና አገናኝን ጠቅ ያድርጉ። የCSV ፋይል ወደ መሳሪያዎ ይወርዳል።

በመቀጠል ፋይሉን ይክፈቱ እና ጉዳዮችን ያክሉ።

ዓይነት

የአይነት መስኩን ሲጫኑ ተቆልቋይ ሜኑ ይታያል። ለጅምላ ጉዳይ ማስመጣት ከሁለት አማራጮች አንዱን ይምረጡ።

  • ማስታወሻ ተቆልቋይ ሜኑ ብዙ አማራጮች አሉት፣ ግን ሁለቱ ብቻ ከጉዳይ አስተዳደር ጋር የተያያዙ ናቸው። እያንዳንዱ አማራጭ የተለየ የCSV ፋይል ናሙና አለው።

ጉዳዮችን ያክሉ

  • ለቀላል የጅምላ ጭነት ጉዳዮች ይህንን አማራጭ ይምረጡ።
  • ለጉዳይ ፍጠር አይነት የማስመጣት አይነት የCSV ፋይል ለመክፈት የአውርድ ናሙና አገናኙን ጠቅ ያድርጉ።

ጉዳዮችን አስተካክል።

  • ወቅታዊ ጉዳዮችን ለማዘመን ይህንን አማራጭ ይምረጡ።
  • የCSV ፋይል ለመክፈት የማውረጃ ናሙና አገናኙን ጠቅ ያድርጉ ከአምድ አርእስቶች ጋር ለጉዳይ አስመጪ አይነት።

ፋይሉን በመስቀል ላይ

ፋይሉን ወዲያውኑ መስቀል ወይም ወደ ተጨማሪ ቀን ወደ አስመጪ ማስታወቂያ ሳጥን መመለስ እና እነዚህን ደረጃዎች ማጠናቀቅ ትችላለህ።

  1. ከውጭ የሚመጡትን አክል የንግግር ሳጥን ይክፈቱ።
  2. ለCSV ፋይል ልዩ ስም ያስገቡ።
  3. ፋይሉን ከመሳሪያዎ ለመስቀል የመደመር (+) አዶን ጠቅ ያድርጉ።
  4. ዓይነት መስኩን ጠቅ ያድርጉ እና አንድ አማራጭ ይምረጡ። (ከላይ ይመልከቱ)
