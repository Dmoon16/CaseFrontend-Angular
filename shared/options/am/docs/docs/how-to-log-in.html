እንዴት እንደሚገቡ

ወደ ነባር ኬዝ ገቢር መለያዎ እንዴት እንደሚገቡ ይወቁ።

መግባት

  1. ወደ የመግቢያ ገጹ ይሂዱ.
  2. የመግቢያ አዝራሩን ጠቅ ያድርጉ።
  3. የእርስዎን ኢሜይል ያስገቡ.
  4. ቀጣይ የሚለውን ጠቅ ያድርጉ።

መለያ በማረጋገጥ ላይ

  1. አዲስ የአሳሽ ትር ይክፈቱ እና ወደ ኢሜል መለያዎ ይግቡ።
  2. ባለ ስድስት አሃዝ ኮድ ይቅዱ።
  3. ወደ ኬዝ ገቢር ትር ተመለስ።
  4. ኮዱን ወደ መስኮቹ ይለጥፉ። አዲስ መስኮት ይታያል.
