በዩአይ ሜኑ በኩል ጉዳዮችን እንዴት መፍጠር ወይም ማስተካከል እንደሚቻል

ይህ መጣጥፍ አስተዳዳሪዎች ከመድረክ የተጠቃሚ በይነገጽ ጋር እንዴት ጉዳይ መፍጠር እንደሚችሉ ያሳያል። አስተዳዳሪዎች ተጠቃሚዎችን ወደ አዲስ ጉዳይ ማከል እና ነባር ጉዳዮችን ማዘመን ይችላሉ።

ጉዳይ መፍጠር

  1. ወደ የአስተዳዳሪ መለያ ይግቡ።
  2. በመተግበሪያዎች ገጽ ላይ አንድ መተግበሪያ ያግኙ እና የ ADMIN ቁልፍን ጠቅ ያድርጉ።
  3. በአስተዳዳሪው ፓነል ላይ የጉዳይ ሜኑ ትርን ጠቅ ያድርጉ።
  4. አዲስ ቁልፍን ጠቅ ያድርጉ። አዲስ መስኮት ይታያል.

ተጠቃሚዎችን መጨመር

እንደ አማራጭ፣ አስተዳዳሪዎች በዚህ ገጽ ላይ ተጠቃሚዎችን ወደ ጉዳዩ ማከል ይችላሉ። ወደ ታች ይሸብልሉ እና ASSIGN USERS አዝራሩን ጠቅ ያድርጉ።

ጉዳይን ማስተካከል

አነስተኛ ቁጥር ያላቸውን ጉዳዮች ለማዘመን የአርትዕ መሳሪያውን ይጠቀሙ እና በእያንዳንዱ ጉዳይ ላይ ለውጦችን ያድርጉ።

  • ማስታወሻ በአንድ መቀመጫ ውስጥ ብዙ ቁጥር ያላቸውን ጉዳዮች ለማዘመን፣ የማስመጣት መሳሪያውን ለመጠቀም ያስቡበት
  1. ወደ የአስተዳዳሪ መለያ ይግቡ።
  2. በመተግበሪያዎች ገጽ ላይ አንድ መተግበሪያ ያግኙ እና የ ADMIN ቁልፍን ጠቅ ያድርጉ።
  3. በአስተዳዳሪው ፓነል ላይ የጉዳይ ሜኑ ትርን ጠቅ ያድርጉ።
  4. መያዣ ይፈልጉ እና አርትዕን ጠቅ ያድርጉ።
  5. ለውጦችን ያድርጉ እና አስቀምጥን ጠቅ ያድርጉ።
