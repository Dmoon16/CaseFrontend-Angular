በአስመጪ መሳሪያው የተጠቃሚ ምደባዎችን እንዴት መመደብ፣ ማሻሻል ወይም ማስወገድ እንደሚቻል

ይህ መጣጥፍ አስተዳዳሪዎች የጉዳይ ተጠቃሚዎችን በአስመጪ መሳሪያው እንዴት ማስተዳደር እንደሚችሉ ያሳያል።

አስተዳዳሪዎች የሚከተሉትን ማድረግ ይችላሉ:

  • ተጠቃሚዎችን ወደ ጉዳይ መድብ
  • አስቀድሞ በአንድ ጉዳይ ላይ የተመደቡ ተጠቃሚዎችን ቀይር
  • በአንድ ጉዳይ ላይ የተመደቡ ተጠቃሚዎችን ያስወግዱ

የማስመጣት መሣሪያውን በመክፈት ላይ

  1. ወደ የአስተዳዳሪ መለያ ይግቡ።
  2. በመተግበሪያዎች ገጽ ላይ አንድ መተግበሪያ ያግኙ እና የ ADMIN ቁልፍን ጠቅ ያድርጉ።
  3. በንዑስ ምናሌው ላይ የውህደት ትሩን እና ከዚያ IMPORTS ን ጠቅ ያድርጉ።
  4. አዲስ ቁልፍን ጠቅ ያድርጉ። የማስመጣት አክል የንግግር ሳጥን ያሳያል።

መዝገቦችን መፍጠር

አንድ ወይም ከዚያ በላይ የCSV ፋይሎችን ያውርዱ። የጉዳይ ተጠቃሚዎችን ለመጨመር፣ ለማሻሻል ወይም ለማስወገድ እነዚህን ፋይሎች ይጠቀሙ። የማስመጣት አክል የንግግር ሳጥን ከተከፈተ በኋላ፡-

  1. ዓይነት መስክ ላይ ጠቅ ያድርጉ።
  2. በተቆልቋይ ምናሌው ውስጥ ይሸብልሉ እና አንድ አማራጭ ይምረጡ (ከዚህ በታች ይመልከቱ)። አንድ አገናኝ ያሳያል.
  3. የማውረጃ ናሙና አገናኝን ጠቅ ያድርጉ። የCSV ፋይል ወደ መሳሪያዎ ይወርዳል።

ፋይሉን ይክፈቱ እና መዝገቦችን ያክሉ። በCSV ፋይል ውስጥ የቅርጸት ውሂብን እንዴት መጠቀም እንደሚቻል ለማወቅ እዚህ ጠቅ ያድርጉ።

የፋይል አማራጮች

ተጠቃሚዎችን ወደ ጉዳዮች መድቡ * ተጠቃሚዎችን ወደ መያዣ ለማከል ይህን አማራጭ ይምረጡ። * የጉዳይ_ሚና_ትዕዛዝ ሕዋስ ባዶ ይተዉት። * ወደ የተጠቃሚዎች ገጽ ይሂዱ እና ለዚህ ሰው የመለያ መታወቂያውን ይቅዱ። የመለያ መታወቂያ መስኩ ባዶ ከሆነ ልዩ መታወቂያ ያስገቡ፣ አስቀምጥን ጠቅ ያድርጉ እና መታወቂያውን ወደ CSV ፋይል ይቅዱ።

ለጉዳይ የተመደቡ ተጠቃሚዎችን ያሻሽሉ * ለጉዳይ የተመደቡ የአሁን ተጠቃሚዎች ውሂብ ለማዘመን ይህንን አማራጭ ይምረጡ። * ለcase_ID አምድ የጉዳይ መለያ መታወቂያ ይጠቀሙ።

ለጉዳዮች የተመደቡ ተጠቃሚዎችን ያስወግዱ * በአንድ ጉዳይ ላይ የተመደቡ ተጠቃሚዎችን ለማስወገድ ይህንን አማራጭ ይምረጡ።

ፋይሉን በመስቀል ላይ

ፋይሉን ወዲያውኑ መስቀል ወይም ወደ ተጨማሪ ቀን ወደ አስመጪ ማስታወቂያ ሳጥን መመለስ እና እነዚህን ደረጃዎች ማጠናቀቅ ትችላለህ።

  1. ከውጭ የሚመጡትን አክል የንግግር ሳጥን ይክፈቱ።
  2. ለCSV ፋይል ልዩ ስም ያስገቡ።
  3. ፋይሉን ከመሳሪያዎ ለመስቀል የመደመር (+) አዶን ጠቅ ያድርጉ።
  4. ዓይነት መስኩን ጠቅ ያድርጉ እና አንድ አማራጭ ይምረጡ። (ከላይ ይመልከቱ)
  5. አስቀምጥን ጠቅ ያድርጉ።

በማቀነባበር ላይ

በፋይሉ እና በስርዓት ጭነት ውስጥ ባሉ መዝገቦች ብዛት ላይ በመመስረት መድረኩ ማስመጣቱን ለማስኬድ ብዙ ደቂቃዎችን ሊወስድ ይችላል።

ማስመጣቱ ሲጠናቀቅ መድረኩ የተጠቃሚ መዝገቦችን (ማለትም ይጨምራል፣ ያስተካክላል ወይም ያስወግዳል) በጉዳይ ገጹ ላይ ያስተካክላል።

ውድቀት

የማስመጣት ክዋኔው ካልተሳካ መድረኩ ያልተሳካ መልእክት ይመልሳል። በአብዛኛዎቹ ሁኔታዎች, የውድቀት መንስኤ የውሂብ ቅርጸት ነው.

የውድቀቱን ምክንያት ለማየት፡-

  1. በድርጊት ርዕስ ስር ይመልከቱ የሚለውን ጠቅ ያድርጉ።
  2. በሚቀጥለው ገጽ ላይ መዳፊቱን ባልተሳካው አዶ ላይ አንዣብበው። የመሳሪያው ጫፍ ስህተቱን ያሳያል.
  3. ወደ ሲቪኤስ ፋይል ይሂዱ እና ያስተካክሉ እና ስህተቱን ያስተካክሉ።
  4. ፋይሉን እንደገና ስቀል።
