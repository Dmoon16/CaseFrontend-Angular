በሕዝብ መመዝገቢያ ሊንክ ተጠቃሚዎችን እንዴት በራስ ሰር መመደብ እንደሚቻል

አስተዳዳሪዎች አዲስ ተጠቃሚዎች በራስ-ሰር ለተፈጠረ አዲስ መያዣ እንዲመደቡ የስርዓት ቅንብሮችን እንዴት ማዋቀር እንደሚችሉ ይማራሉ ።

የማዋቀር ገጹን በመክፈት ላይ

የመጀመሪያው እርምጃ የሲስተም ማዘጋጃ ገጽን መክፈት ነው.

  1. ወደ የአስተዳዳሪ መለያ ይግቡ።
  2. በመተግበሪያዎች ገጽ ላይ አንድ መተግበሪያ ያግኙ እና የ ADMIN ቁልፍን ጠቅ ያድርጉ።
  3. በንዑስ ሜኑ ላይ የስርዓቶች ትርን እና ከዚያ SYSTEM SETUPን ጠቅ ያድርጉ።
  4. ወደ USER SIGNUP ርዕስ ወደታች ይሸብልሉ።

የስርዓት ቅንብሮችን በማዋቀር ላይ

  1. የፍቀድ ምዝገባ እና በራስ ሰር ፍጠር ተቆልቋይ ምናሌዎችን ለማብራት ያቀናብሩ።
  2. የሚና ምናሌውን ጠቅ ያድርጉ እና አንድ አማራጭ ይምረጡ (ነባሪው ይመከራል)።
  3. የህዝብ ተጠቃሚ ምዝገባ አገናኝን ይቅዱ እና ያጋሩ።

ሊንኩን በመላክ ላይ

አገናኙን ጠቅ የሚያደርግ እና ምዝገባውን ያጠናቀቀ እያንዳንዱ ተጠቃሚ መድረኩን ቀስቅሶ በኬዝ ገጽ ላይ አዲስ ጉዳይ ያመነጫል እና በዚህ አጋጣሚ ተጠቃሚውን በራስ-ሰር ይመድባል።
