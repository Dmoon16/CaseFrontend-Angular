Sida loo cusboonaysiiyo Profile-ka isticmaalaha

Maqaalkani waxa uu tusayaa isticmaalayaasha sida loo cusboonaysiiyo macluumaadka boggooda Profile.

  • Isticmaalayaashu waxay cusbooneysiin karaan muuqaalkooda.
  • Maamulayaashu waxay cusboonaysiin karaan bogga Profile ee isticmaale kasta.

Isticmaalayaasha

Tallaabada ugu horreysa waa in la furo bogga Profile. Isticmaalayaasha iyo maamulayaashu waxay raacaan tillaabooyinkan si ay u cusboonaysiiyaan boggooda Profile.

  1. Gal koontadaada Case Active
  2. Guji bogga Profile ee ku yaal xagga sare ee bogga.
  3. Isbeddel samee.
  4. Guji Save.

Beeraha intooda badan waa ikhtiyaari, beeraha qaarna waa loo baahan yahay. Goobaha loo baahan yahay waa:

  • Magaca koowaad
  • Magaca dambe
  • Cinwaanka emailka
  • Luuqad
  • Aagga waqtiga

Loogu talagalay admins

Maamulayaasha sidoo kale waxay u cusbooneysiin karaan macluumaadka astaanta guud ee isticmaalayaasha kale. Tan waxa ku jira saddex goobood oo hoos yimaada HOST USER INFO ciwaanka.

  1. Gal koontadaada maamulka Active Case
  2. Hel app oo dhagsii badhanka ADMIN.
  3. Guji Users liiska sare.
  4. Tag liiska hoosaadka oo dooro liiska isticmaalayaasha aad rabto inaad gasho (Active ama NASIB). Dejinta caadiga ah waa Firfircoon
  5. Hel isticmaale liiska oo guji Edit.
  6. Isbeddel ku samee sida loo baahdo. (hoos eeg)
  7. Guji Save.

Beddel macluumaadka isticmaalaha

Maamulka ayaa bedeli kara macluumaadka isticmaalaha.

  1. Deji Nooca Isticmaalaha (Isticmaal ama Maamule).
  2. Deji Xaaladda Isticmaalaha ( Firfircoon ilaa Naafada).
  3. Geli aqoonsi Tag gaar ah. (ikhtiyaar)
  4. Guji Save.
