Sida Isticmaalayaashu u Sameeyaan Xisaab Cusub

Ka dib markii maamuluhu ku daro isticmaale kiis: * haddii isticmaaluhu uu ku cusub yahay kiis, isticmaaluhu waa inuu dhamaystiraa socodka shaqada si uu u sameeyo akoon oo uu aqbalo martiqaadka gelitaanka kiiska. * Haddii isticmaaluhu leeyahay acount kiis-ka-shaqayneed oo jira, isticmaaluhu waa inuu aqbalaa martiqaadka inuu galo kiiska.

Maqaalkani waxa uu sharaxayaa tillaabooyinka ay tahay in isticmaaluhu buuxiyo si uu u sameeyo akoonkiisa.

Bilaabayaa

Waxaa jira laba siyaabood oo isticmaalaha loogu dari karo app caseactive

  • Maamule: Haddii maamulku ku abuuro isticmaale iyada oo loo marayo UI platform ama qalab soo dejinta , maamulku waa inuu u soo diraa isticmaalaha isku xirka account.caseactive.com si uu u sii wato gelitaanka.
  • Isticmaale: Haddii isticmaaluhu uu samaystay akoon isaga oo isticmaalaya isku xidhka is -diiwaangelinta , isticmaalaha waxa si toos ah loogu wareejin doonaa account.caseactive.com si uu u sii galo gelitaanka.

Galitaanka

Marka isticmaaluhu uu galo akoonkiisa CaseActive markii ugu horeysay, madalku waxay ku dhiirigelisaa isticmaalaha:

  • fur emailkooda koontada ka dibna koobbi/ku dheji lambarka amniga ee lixda lambar ah
  • buuxi profile akoonka

Kadib isticmaaluhu riixo UPDATE, bogga Apps ayaa soo bandhigaya. Boggu wuxuu tusinayaa magaca abka (magaca shirkadda), nooca adeegsadaha (doorka), heerka, iyo badhanka Casuumada.

Si loo galo abka, isticmaaluhu waa inuu aqbalaa martiqaadka. Isticmaaluhu waa inuu:

  1. Guji badhanka Casuumada
  2. Dib u eeg oggolaanshaha
  3. Guji tab Ogeysiiska si aad dib ugu eegto oo aad wax uga beddesho (ikhtiyaar) shuruudaha.
  4. Guji Oggolaanshaha Helitaanka

Marka isticmaaluhu ogolaado, madalku waxa uu soo bandhigayaa badhanka APP ee bogga isticmaalaha. Badhankan waxa uu siinayaa isticmaalaha marin u helka kiisaska loo qoondeeyay.

Si loo tafatiro goobaha isticmaalaha, isticmaaluhu waxa uu gujiyaa ellipsis-ka.
