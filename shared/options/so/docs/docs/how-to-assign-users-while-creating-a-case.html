Sida Loo Meeleeyo Isticmaalayaasha Marka La Sameynayo Kiis

Maqaalkani waxa uu tusayaa maamulka sida loo meeleeyo isticmaalaha marka la abuurayo kiis cusub. Haddii isticmaaluhu aanu hore uga diiwaan gashanayn goobta, waxaad abuuri kartaa isticmaale cusub marka la abuurayo kiis cusub.

Abuuritaanka kiis

  1. Gal koontada maamulka
  2. Ka hel app bogga Apps oo dhagsii badhanka ADMIN.
  3. Guji tab menu-ka kiisaska.
  4. Guji badhanka CUSUB Daaqad cusub ayaa soo bandhigaysa.

img

  • Buuxi meelaha si aad u abuurto kiis ka dibna hoos ugu dhaadhac ciwaanka Isticmaalayaasha La cayimay.

Meelaynta isticmaale

Isticmaalaha hadda

Raac tillaabooyinkan si aad ugu meelayso isticmaale hadda kiiska cusub.

Guji badhanka ASSIGN Users oo buuxi meelaha.

  1. Guji Hoos-u-dhaca Isticmaalaha oo dooro aqoonsi (tusaale iimaylka).
  2. Geli qiime goobta banaan (tusaale ciwaanka iimaylka).
  3. Haddii diiwaanka isticmaaluhu uu ka jiro goobta, goobta ayaa si toos ah u soo jeedinaysa magac.
  4. Dooro isticmaale sax ah
  5. Dooro door isticmaale (macmiil, macaamiis, ama shaqaale).
  6. Ikhtiyaar ahaan, calaamadee sanduuqa si aad u soo dirto isticmaalaha ogeysiiska iimaylka.
  7. Guji SAVE

Case Active waxa uu ku darayaa isticmaalaha kiiska. Hadda maamulayaashu waxay riixi karaan Save si ay u xiraan daaqada, ama ku daraan isticmaale cusub, ama ka saaraan isticmaale.

Ku darida isticmaale cusub

Haddii madalku aysan heli karin rikoor hadda, dhagsii badhanka si aad ugu darto isticmaale cusub.

Buuxi meelaha ka dibna dhagsii SAVE si aad ugu darto isticmaalaha goobta. Marka xigta, deji door isticmaale oo guji SAVE markale si aad ugu meelayso isticmaalaha kiiska. Ugu dambeyntii, dhagsii hal mar oo kale si aad u badbaadiso kiiska.
