Maitiro ekugovera, Kugadzirisa kana Kubvisa Mushandisi Migove neKupinza Tool

Ichi chinyorwa chinoratidza admins maitiro ekugadzirisa vashandisi vekesi neiyo Import turusi.

Admins vanogona:

  • kupa vashandisi kune imwe nyaya
  • shandura vashandisi vakatopihwa kune imwe kesi
  • bvisa vashandisi vakapihwa kesi

Kuvhura iyo Import tool

  1. Pinda kune admin account.
  2. Tsvaga app papeji yeMapurogiramu wobva wadzvanya ADMIN bhatani.
  3. Dzvanya iyo Integration tab uyezve IMPORTS pane submenu.
  4. Baya bhatani ITSVA. Iyo Wedzera Imports dialog box inoratidza.

Kugadzira zvinyorwa

Dhaunirodha faira rimwe chete kana anopfuura eCSV. Shandisa mafaira aya kuwedzera, kugadzirisa, kana kubvisa vashandisi vekesi. Mushure mekunge Add Imports dialog box yavhura:

  1. Dzvanya pakanzi Type.
  2. Dzvanya kuburikidza nekudonha-pasi menyu uye sarudza sarudzo (ona pazasi). A link inoratidza.
  3. Dzvanya iyo Download Sample link. Iyo CSV faira yekurodha kune yako kifaa.

img

Vhura faira uye wedzera zvinyorwa. Dzvanya apa kuti udzidze mashandisiro efomati data mu CSV faira.

Faira sarudzo

Ipa vashandisi kumakesi * Sarudza iyi sarudzo yekuwedzera vashandisi kune kesi. *Siya nyaya_role_order cell isina chinhu. * Enda kune Vashandisi peji uye tevedzera iyo Tag ID yemunhu uyu. Kana iyo Tag ID munda isina chinhu, isa yakasarudzika ID, tinya Sevha, wobva wakopa ID kuCSV faira.

Shandura vashandisi vakapihwa kumakesi * Sarudza iyi sarudzo yekuvandudza data yevashandisi varipo vakapihwa nyaya. * Shandisa Case Tag ID yecase_ID column.

img

Bvisa vashandisi vakapihwa kumakesi * Sarudza iyi sarudzo yekubvisa vashandisi vakapihwa kesi.

Kuisa faira

Unogona kurodha faira ipapo kana kudzokera kuAdd Imports dialog box pane imwe nguva inotevera uye zadzisa matanho aya.

  1. Vhura iyo Add Imports dialog box.
  2. Isa zita rakasiyana refaira reCSV.
  3. Dzvanya iyo yekuwedzera (+) icon kuti uise faira kubva pane chako kifaa.
  4. Dzvanya pakanzi Type uye sarudza sarudzo. (ona pamusoro)
  5. Dzvanya SAVE.

Processing

Zvichienderana nehuwandu hwemarekodhi mufaira uye system load, chikuva chinogona kutora maminetsi akati wandei kugadzirisa kupinza.

img

Kana kupinza kwapera, chikuva chinogadzirisa marekodhi emushandisi (kureva anowedzera, anoshandura, kana kubvisa) pane Cases peji.

Kukundikana

Kana iyo yekupinza basa ikatadza, puratifomu inodzorera Meseji Yakakundikana. Muzviitiko zvakawanda, chikonzero chekukundikana ndechekugadzirisa data.

Kuti uone chikonzero chekukundikana:

  1. Dzvanya Tarisa pasi peAction heading.
  2. Papeji rinotevera, isa mbeva pane Yakundikana icon. The tooltip inoratidza kukanganisa.
  3. Enda kuCVS faira uye gadzirisa uye gadzirisa kukanganisa.
  4. Isa faira zvakare.
