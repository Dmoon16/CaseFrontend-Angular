Maitiro ekugadziridza A User Profile

Ichi chinyorwa chinoratidza vashandisi maitiro ekugadzirisa ruzivo pane yavo Profile peji.

  • Vashandisi vanogona kugadzirisa maprofiles avo.
  • Admins anogona kugadzirisa iyo Profile peji yechero mushandisi.

Kune vashandisi

Nhanho yekutanga ndeyekuvhura iyo Profile peji. Vashandisi uye maadmins vanotevera matanho aya kuti vagadzirise yavo yavo Profile peji.

  1. Pinda kuCase Active account yako.
  2. Dzvanya iyo Profile tab kumusoro kwepeji.
  3. Ita shanduko.
  4. Dzvanya Save.

Minda mizhinji ndeyekusarudza, uye mimwe minda inodiwa. Minda inodiwa ndeiyi:

  • Zita rokutanga
  • Zita rokupedzisa
  • Chikero chetsamba yemumhepo
  • Mutauro
  • Timezone

Zvema admin

MaAdmins anogona zvakare kugadzirisa ruzivo rwevamwe vashandisi. Izvi zvinosanganisira minda mitatu pasi pe HOST USER INFO musoro.

  1. Pinda kuCase Active admin account yako.
  2. Tsvaga app wobva wadzvanya ADMIN bhatani.
  3. Dzvanya Vashandisi pane yepamusoro menyu.
  4. Enda kune sub-menu uye sarudza mushandisi runyorwa rwaunoda kuwana (ACTIVE kana DISABLED). Iyo default setting ndeye Active.
  5. Tsvaga mushandisi pane iyo rondedzero uye tinya Edit.
  6. Ita shanduko sezvinodiwa. (ona pazasi)
  7. Dzvanya Save.

Shandura ruzivo rwemushandisi

Admins anogona kushandura ruzivo rwemushandisi.

  1. Seta Rudzi rweMushandisi (Mushandisi kana Maneja).
  2. Seta iyo Mushandisi Status (Inoshanda kune Yakavharwa).
  3. Pinda yakasarudzika Tag ID. (sarudzo)
  4. Dzvanya Save.
