Maitiro ekugovera Vashandisi Uchiri Kugadzira Nyaya

Ichi chinyorwa chinoratidza admins nzira yekugovera mushandisi uchigadzira nyaya nyowani. Kana mushandisi asati atonyoreswa pachikuva, unogona kugadzira wedzera mushandisi mutsva uchigadzira nyaya nyowani.

Kugadzira nyaya

  1. Pinda kune admin account.
  2. Tsvaga app papeji yeMapurogiramu wobva wadzvanya ADMIN bhatani.
  3. Dzvanya iyo Cases menyu tab.
  4. Baya bhatani ITSVA. Hwindo idzva rinoratidza.

img

  • Zadza minda kuti ugadzire kesi uye wobva wakanda pasi kuenda kune Vakapihwa Vashandisi musoro.

Kugovera mushandisi

Mushandisi wazvino

Tevedza nhanho idzi kuti upe mushandisi aripo kune iyo kesi nyowani.

Dzvanya bhatani reASSIGN USERS uye zadza minda.

  1. Dzvanya iyo Mushandisi kudonhedza pasi uye sarudza chiziviso (semuenzaniso email).
  2. Isa kukosha mundima isina chinhu (semuenzaniso email kero).
  3. Kana iyo rekodhi yemushandisi iripo papuratifomu, iyo munda auto-inopa zita.
  4. Sarudza mushandisi akakodzera.
  5. Sarudza basa remushandisi (vatengi, default, kana vashandi).
  6. Sarudzo, tarisa bhokisi kuti utumire mushandisi email chiziviso.
  7. Dzvanya SAVE.

Case Active inowedzera mushandisi kune kesi. Iye zvino maadmins anogona kudzvanya Sevha kuvhara hwindo, kana kuwedzera mushandisi mutsva, kana kubvisa mushandisi.

Kuwedzera mushandisi mutsva

Kana chikuva chisingakwanisi kuwana rekodhi yazvino, tinya bhatani kuti uwedzere mushandisi mutsva.

Zadza minda wobva wadzvanya SAVE kuti uwedzere mushandisi pachikuva. Tevere, isa basa remushandisi uye tinya SAVE zvakare kuti upe mushandisi kune iyo kesi. Chekupedzisira, tinya SAVE imwe nguva zvakare kuchengetedza kesi.
