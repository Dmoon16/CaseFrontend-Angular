Maitiro Ekugadzira Vashandisi NeKupinza Turusi

Ichi chinyorwa chinoratidza admins maitiro ekugadzira vashandisi neiyo Import Turusi.

Kuvhura iyo Import tool

MaAdmins anogona kushandisa Yekupinza chishandiso kurodha vashandisi vatsva uye kugadzirisa marekodhi emushandisi aripo.

  1. Pinda kune admin account.
  2. Tsvaga app papeji yeMapurogiramu wobva wadzvanya ADMIN bhatani.
  3. Pane iyo Admin panhizha, tinya iyo yekubatanidza tebhu uye wobva waIMPORTS pane submenu.
  4. Baya bhatani ITSVA. Iyo Wedzera Imports dialog box inoratidzira mune itsva browser tab.

Kugadzira zvinyorwa

Mushure mekunge Add Imports dialog box yavhura:

  1. Dzvanya iyo Type ndima uye sarudza sarudzo (ona pazasi). A link inoratidza.
  2. Dzvanya iyo Download Sample link. Iyo CSV faira yekurodha kune yako kifaa.

Zvadaro, vhura faira uye uwedzere zvinyorwa. Dzvanya apa kuti udzidze mashandisiro efomati data mu CSV faira.

Type

Paunodzvanya Type ndima, inodonha-pasi menyu inoratidza. Sarudza imwe yezviviri zvingasarudzwa zvekupinza zvinhu zvakawanda kunze kwenyika.

:exclamation: CHERECHEDZA Iyo yekudonhedza-pasi menyu ine zvakawanda zvingasarudzwa, asi zviviri chete zvine chekuita neKupinza turusi. Imwe neimwe sarudzo ine yakasarudzika faira CSV faira.

Wedzera vashandisi

  • Sarudza iyi sarudzo yekureruka kwakawanda kurodha kwemarekodhi emushandisi.
  • Dzvanya iyo Dhawunirodha Sample link kuti uvhure CSV faira ine makoramu misoro yeiyo Wedzera vashandisi Kupinza mhando.

Shandura vashandisi

  • Sarudza iyi sarudzo yekuvandudza data yevashandisi varipo.
  • Dzvanya iyo Yekudhawunirodha Sample link kuti uvhure CSV faira ine makoramu misoro yeiyo Modify vashandisi Kupinza mhando.

Kuisa faira

Unogona kurodha faira ipapo kana kudzokera kuAdd Imports dialog box pane imwe nguva inotevera uye zadzisa matanho aya.

  1. Vhura iyo Add Imports dialog box.
  2. Isa zita rakasiyana nere CSV faira.
  3. Dzvanya iyo yekuwedzera (+) icon kuti uise faira kubva pane chako kifaa.
  4. Dzvanya pakanzi Type uye sarudza sarudzo. (ona pamusoro)
  5. Dzvanya SAVE.

Processing

Zvichienderana nehuwandu hwemarekodhi mufaira uye system load, chikuva chinogona kutora maminetsi akati wandei kugadzirisa kupinza.

Kana kupinza kwapera, chikuva chinogadzirisa marekodhi emushandisi (kureva anowedzera, anogadzirisa, kana kudzima) pane admin's Users dashboard.

Kukundikana

Kana iyo yekupinza basa ikatadza, chikuva chinodzosera Meseji Yakakundikana. Muzviitiko zvakawanda, chikonzero chekukundikana ndechekugadzirisa data.

Kuti uone chikonzero chekukundikana:

  1. Dzvanya Tarisa pasi peAction heading.
  2. Papeji rinotevera, isa mbeva pane Yakundikana icon. The tooltip inoratidza kukanganisa.
  3. Enda kuCVS faira uye gadzirisa uye gadzirisa kukanganisa.
  4. Isa faira zvakare.
