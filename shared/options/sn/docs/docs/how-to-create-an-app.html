Maitiro ekugadzira App

Ichi chinyorwa chinoratidza vashandisi vatsva maitiro ekugadzira yemahara account account. Case Active inongopa iyo itsva mushandisi admin kodzero dzeakaundi iyi.

Kugadzira account

  1. Enda kune peji rekupinda .
  2. Dzvanya bhatani rekuti Tanga.
  3. Isa email address.
  4. Click Next.

img

Kuongorora account

  1. Vhura tebhu nyowani uye pinda kune yako email account.
  2. Kopa kodhi ine manhamba matanhatu.
  3. Dzokera kuCase Active tab.
  4. Namira kodhi muminda. Hwindo idzva rinoratidza.

Muhwindo idzva:

  1. Zadza minda inodiwa.
  2. Verenga mitemo yebasa uye tarisa bhokisi kuti ubvumirane.
  3. Dzvanya CREATE A ACCOUNT YEMAHARA.

Kupedzisa kuseta app

A workflow inoratidzwa pane inotevera peji. Pedzisa matanho ekupedzisa kuseta app yako. Unogona kugara uchidzokera kumapeji aya kuti ugadzirise kana kugadzirisa ruzivo kana mamiriro eakaundi yako.

Pa peji rePamusoro peApp, zadza zita rekambani neminda yewebhusaiti (inodiwa) wobva wadzvanya Next.

Pa peji reRuzivo rwekubhadharisa, tinya chikamu cheKubhadhara Chirongwa uye ongorora sarudzo. Iyo default setting ndeye Mahara. Dzvanya SUBMIT.

img

Mushure mekutumira fomu, Nyaya Inoshanda inoratidza peji reMapurogiramu. Kune yega yega app yaunogadzira kana yeyako, peji rinoratidza:

  1. Zita reapp
  2. Yako Type (mushandisi basa)
  3. Status

Peji yeMapurogiramu inzvimbo yekutanga kune akawanda admin mabasa:

  1. Dzvanya bhatani reAPP kutaurirana, zadza zvinyorwa, kutaura kwevhidhiyo (uye nezvimwe) pane zvine chekuita nenyaya.
  2. Dzvanya bhatani reADMIN kuti uvhure admin panel. Apa ndipo apo maAdmins anotonga vashandisi, makesi, kubatanidzwa, nezvimwe.
  3. Dzvanya iyo ellipsis kubata account kubhadharisa uye zviziviso zvigadziriso.
  • Note Case Active inoratidza ADMIN bhatani revashandisi ve admin. Vashandisi vasina mvumo yekutonga havaone bhatani reADMIN (sezvinoratidzwa pahwindo pazasi).
