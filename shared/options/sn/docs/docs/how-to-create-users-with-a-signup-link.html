Ichi chinyorwa chinoratidza admins maitiro ekugadzira vashandisi nekutumira chinongedzo chekusaina.

Kugadzirisa chinongedzo chekunyoresa

  1. Dzvanya iyo Vatumire chinongedzo chekunyoresa pazasi pe Wedzera mushandisi dialog box. Peji itsva inoratidzwa.
  2. Gadzirisa sarudzo dzekunyoresa. (ona pazasi)

img

Ita shuwa kuti Bvumira Signup ndima yekumisikidza kuOn. (inodiwa) Kana yagoneswa, Case Active inogadzira chinongedzo chakasiyana.

Wongorora sarudzo dziri pazasi wobva wadzvanya icon kuti ukope chinongedzo. Iwe unogona kuisa chinongedzo pawebhusaiti yako kana kutumira chinongedzo kune mumwe kana vanhu vakawanda kuburikidza neemail kana chero nzira yekutaurirana.

Kugadzira nyaya

Sarudzo, iwe unogona otomatiki kugadzira kesi kune yega yega mushandisi uye kupa basa remushandisi.

  1. Dzvanya iyo Auto Gadzira Nyaya uye sarudza On. Iyo Case User Role minda inoratidza.
  2. Dzvanya iyo Nyaya Yemushandisi Basa uye sarudza sarudzo. (ona pazasi)

Kutumira chinongedzo

Govera chinongedzo nemunhu mumwe kana vakawanda. Wese munhu anodzvanya chinongedzo uye anopedzisa kufambiswa kwebasa kuti amise yavo Case Active account. Kana mushandisi apedza kusaina kufambiswa kwemabasa, Nyaya Inoshanda inowedzera mushandisi kune Runyorwa rweVashandisi.

Matanho anotevera

Case Active inotumira mushandisi email ziviso ine signup link. Mushandisi anodzvanya chinongedzo uye anopedzisa mafambiro ebasa ekusaina kutsva.

  • Ichi chinyorwa chinotsanangura kusaina mafambiro evashandisi vatsva.
