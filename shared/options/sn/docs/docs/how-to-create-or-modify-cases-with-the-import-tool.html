Maitiro ekugadzira kana kugadzirisa nyaya neiyo Import Turusi

Ichi chinyorwa chinoratidza admins kugadzira kana kugadzirisa nyaya dziripo neiyo Import tool.

Kuvhura iyo Import tool

  1. Pinda kune admin account.
  2. Tsvaga app papeji yeMapurogiramu wobva wadzvanya ADMIN bhatani.
  3. Pane iyo Admin panhizha, tinya iyo yekubatanidza tebhu uye wobva waIMPORTS pane submenu.
  4. Baya bhatani ITSVA. Iyo Wedzera Imports dialog box inoratidzira mune itsva browser tab.

img

Kugadzira nyaya

Mushure mekunge Add Imports dialog box yavhura:

  1. Dzvanya iyo Type ndima uye sarudza sarudzo (ona pazasi). A link inoratidza.
  2. Dzvanya iyo Download Sample link. Iyo CSV faira yekurodha kune yako kifaa.

Tevere, vhura faira uye wedzera makesi.

Type

Paunodzvanya Type ndima, inodonha-pasi menyu inoratidza. Sarudza imwe yesarudzo mbiri dzemakesi akawanda kubva kunze kwenyika.

  • Ziva Iyo yekudonha-pasi menyu ine zvakawanda zvingasarudzwa, asi zviviri chete zvine chekuita nekutonga kesi. Imwe neimwe sarudzo ine akasiyana emhando CSV faira.

Wedzera makesi

  • Sarudza iyi sarudzo yekunyorerwa kwakawanda kwemakesi.
  • Dzvanya iyo Dhawunirodha Sample link kuti uvhure CSV faira ine column misoro yeGadzira makesi ekunze mhando.

Shandura makesi

  • Sarudza iyi sarudzo yekuvandudza nyaya dzazvino.
  • Dzvanya iyo yekudhawunirodha Sample link kuti uvhure CSV faira ine column misoro yeiyo Modify makesi ekupinza mhando.

Kuisa faira

Unogona kurodha faira ipapo kana kudzokera kuAdd Imports dialog box pane imwe nguva inotevera uye zadzisa matanho aya.

  1. Vhura iyo Add Imports dialog box.
  2. Isa zita rakasiyana refaira reCSV.
  3. Dzvanya iyo yekuwedzera (+) icon kuti uise faira kubva pane chako kifaa.
  4. Dzvanya pakanzi Type uye sarudza sarudzo. (ona pamusoro)
  5. Dzvanya SAVE.

Kugadzira

Aya ndiwo minda inodiwa uye yekutanga zvinodiwa. Dzvanya pano kuti uongorore ruzivo rwemafomati.

tag_id (kana case_ID) chiziviso chakasiyana (shandisa alphanumeric characters, hyphens inogamuchirwa)

case_status 0 = yakavharwa 1 = inoshanda

date_yakavhurwa mwedzi/zuva/gore mm/dd/yyyy

nezve tsananguro yakasarudzika yenyaya (shandisa alphanumeric characters, hyphens inogamuchirwa)

Processing

Zvichienderana nehuwandu hwemarekodhi mufaira uye system load, chikuva chinogona kutora maminetsi akati wandei kugadzirisa kupinza.

img

Kana kupinza kwapera, chikuva chinogadzirisa marekodhi emakesi (kureva anowedzera kana kugadzirisa) pane admin's Cases dashboard.

Kukundikana

Kana iyo yekupinza basa ikatadza, chikuva chinodzosera Meseji Yakakundikana. Muzviitiko zvakawanda, chikonzero chekukundikana ndechekugadzirisa data.

Kuti uone chikonzero chekukundikana:

  1. Dzvanya Ona pasi pemusoro weACTION.
  2. Papeji rinotevera, isa mbeva pane Yakundikana icon. The tooltip inoratidza kukanganisa.
  3. Enda kuCVS faira uye gadzirisa uye gadzirisa kukanganisa.
  4. Isa faira zvakare.
