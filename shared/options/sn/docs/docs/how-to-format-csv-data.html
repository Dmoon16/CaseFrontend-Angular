Maitiro ekugadzirisa CSV Data

Pazasi pane rondedzero yedata data mu CSV mafaera. Imwe neimwe yekupinza sarudzo ine musanganiswa wakasiyana weminda.

username

  • Isa email kero yemushandisi.

zita rekupihwa

  • Isa zita remushandisi.

mhuri_zita

  • Isa zita rekupedzisira remushandisi.

locale

  • Koramu iyi inotarisa mutauro unofarirwa nemushandisi.
  • Pinda en yeChirungu, sp yeSpanish.

zoneinfo

  • Izvi zvinoseta nguva yemushandisi. Pinda mukondinendi neguta pasina nzvimbo (shandisa underscore).
  • Unofanira kushandisa sarudzo muProfile> LANGUAGE & REGION> Timezone yekudonhedza menyu.
  • Shandisa yakanakisa-yakakodzera nguva zoni. Semuenzaniso, Boston haisi pakudonhedza pasi, saka tinogona kushandisa New York (kana chero guta riri muEST/EDT time zone).

host_tag_id

  • Pinda chinyakare label.
  • az, 0-9, -, / vanotenderwa.

case_status

  • 0 = yakavharwa
  • 1 = inoshanda
