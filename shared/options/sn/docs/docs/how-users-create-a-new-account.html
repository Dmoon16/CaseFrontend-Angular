Mushure mekunge admin awedzera mushandisi kune imwe nyaya: * kana mushandisi ari mutsva kune kesi, mushandisi anofanira kupedzisa kufambiswa kwebasa kugadzira account uye kubvuma kukokwa kuti uwane iyo kesi. * kana mushandisi aine caseactive acounnt iripo, mushandisi anofanira kugamuchira kukokwa kuti awane nyaya yacho.

Ichi chinyorwa chinotsanangura matanho anofanirwa kuitwa nemushandisi kugadzira account yavo.

Kutanga

Pane nzira mbiri dzemushandisi dzinogona kuwedzerwa kune caseactive app

  • Na admin: Kana admin akagadzira mushandisi kuburikidza nepuratifomu UI kana Import chishandiso , admin anofanira kutumira mushandisi chinongedzo kuaccount.caseactive.com kuti aenderere mberi nekupinda.
  • Nemushandisi: Kana mushandisi akagadzira account achishandisa chinongedzo chekusaina , mushandisi anozoendeswa otomatiki kuaccount.caseactive.com kuti uenderere mberi nekupinda.

Logging in

Kana mushandisi apinda muCaseActive account yavo kekutanga, chikuva chinosimudzira mushandisi ku:

  • vhura email account yavo wobva wakopa/namira iyo nhanhatu-manhamba yekuchengetedza kodhi
  • zadzisa nhoroondo yeakaundi

Mushure mekunge mushandisi adzvanya UPDATE, peji reMapurogiramu rinoratidza. Iyo peji inoratidza zita reapp (zita rekambani), rudzi rwemushandisi (basa), chimiro, uye bhatani reKUKONZA.

Kuti uwane iyo app, mushandisi anofanira kugamuchira kukokwa. Mushandisi anofanira:

  1. Baya bhatani reKUKOKERA.
  2. Wongorora mvumo.
  3. Dzvanya iyo Zviziviso tebhu kuti uongorore uye kugadzirisa (sarudzo) mazwi.
  4. Dzvanya BVUMA VANHU.

Kana mushandisi akabvumidza, chikuva chinoratidza bhatani reAPP pane peji remushandisi. Iri bhatani rinopa mushandisi mukana kune akapihwa makesi.
