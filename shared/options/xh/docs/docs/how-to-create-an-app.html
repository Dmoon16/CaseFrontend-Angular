Uyenza njani iApp

Eli nqaku libonisa abasebenzisi abatsha indlela yokwenza i-akhawunti yesilingo sasimahla. I-Case Active inika umsebenzisi omtsha amalungelo olawulo kule akhawunti.

Ukwenza iakhawunti

  1. Yiya kwiphepha lokungena .
  2. Cofa iqhosha elithi Qalisa.
  3. Faka idilesi ye-imeyile.
  4. Cofa Okulandelayo.

Ukuqinisekisa i-akhawunti

  1. Vula ithebhu entsha yebrawuza kwaye ungene kwiakhawunti yakho ye-imeyile.
  2. Khuphela ikhowudi enamanani amathandathu.
  3. Buyela kwiCase Active thebhu.
  4. Ncamathelisa ikhowudi kwiindawo. Ifestile entsha ibonisa.

Kwifestile entsha:

  1. Gcwalisa iindawo ezifunekayo.
  2. Funda imigaqo yenkonzo kwaye ukhangele ibhokisi ukuze uvume.
  3. Cofa YENZA I-AKHAWUNTI YASIMAHLA.

Ukugqiba ukuseta usetyenziso

Ukuhanjiswa komsebenzi kwiphepha elilandelayo. Gqibezela amanyathelo ukugqiba ukuseta i-app yakho. Ungasoloko ubuyela kula maphepha ukuze uhlele okanye uhlaziye ulwazi okanye ubume beakhawunti yakho.

Kwiphepha elimalunga ne-App, gcwalisa igama lenkampani kunye neendawo zewebhusayithi (ezifunekayo) kwaye emva koko ucofe Okulandelayo.

Kwiphepha leNgcaciso yeBhili, cofa indawo yeSicwangciso seNtlawulo kwaye ujonge iinketho. Useto oluhlala lukhona lusimahla. Cofa THUMELA.

img

Emva kokuba ungenise ifomu, i-Case Active ibonisa iphepha le-Apps. Kusetyenziso ngalunye olwenzayo okanye olungulo, iphepha libonisa:

  1. Igama le-app
  2. Uhlobo lwakho (indima yomsebenzisi)
  3. Ubume

Iphepha leApps yindawo yokuqalisa imisebenzi emininzi yolawulo:

  1. Cofa iqhosha le-APP ukunxibelelana, gcwalisa amaxwebhu, ingxoxo yevidiyo (kunye nokunye) kwimiba enxulumene nemeko.
  2. Cofa iqhosha elithi ADMIN ukuvula indawo yolawulo. Apha kulapho ii-Admins zilawula abasebenzisi, iimeko, ukudityaniswa, kunye nokunye.
  3. Cofa i-ellipsis ukulawula ukuhlawulwa kweakhawunti kunye nesetingi yesaziso.
  • Qaphela Imeko Esebenzayo ibonisa iqhosha le-ADMIN kubasebenzisi bolawulo. Abasebenzisi abangenawo amalungelo olawulo abaliboni iqhosha elithi ADMIN (njengoko kubonisiwe kwikhusi elingezantsi).
