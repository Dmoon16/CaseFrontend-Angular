Uyihlaziya njani iProfayile yomsebenzisi

Eli nqaku libonisa abasebenzisi indlela yokuhlaziya ulwazi kwiphepha labo leProfayili.

  • Abasebenzisi banokuhlaziya iiprofayili zabo.
  • Abalawuli banokuhlaziya iphepha leProfayili yakhe nawuphi na umsebenzisi.

Kubasebenzisi

Inyathelo lokuqala kukuvula iProfayile yephepha. Abasebenzisi kunye nabalawuli balandela la manyathelo ukuhlaziya iphepha labo leProfayili.

  1. Ngena kwi-akhawunti yakho ye-Case Active.
  2. Cofa ithebhu yeProfayile phezulu kwephepha.
  3. Yenza utshintsho.
  4. Cofa Gcina.

Uninzi lwamasimi alukhethi, kwaye amanye amasimi ayafuneka. Iindawo ezifunekayo zezi:

  • Igama lokuqala
  • Ifani
  • Idilesi yemeyile
  • Ulwimi
  • Ixesha lendawo

Kuba admin

Abalawuli banokuhlaziya ulwazi lweprofayili kwabanye abasebenzisi. Oku kuquka iindawo ezintathu phantsi kwesihloko esithi HOST USER INFO.

  1. Ngena kwi-akhawunti yakho ye-Case Active admin.
  2. Khangela usetyenziso kwaye ucofe iqhosha elithi ADMIN.
  3. Cofa Abasebenzisi kwimenyu ephezulu.
  4. Yiya kwi-sub-menu kwaye ukhethe uluhlu lwabasebenzisi ofuna ukufikelela kulo (ISEBENZA okanye LUKHUBAZEKILE). Useto oluhlala lukhona luyasebenza.
  5. Khangela umsebenzisi kuluhlu kwaye ucofe Hlela.
  6. Yenza utshintsho njengoko kufuneka. (bona ngezantsi)
  7. Cofa Gcina.

Guqula ulwazi lomsebenzisi

Abalawuli banokutshintsha ulwazi lomsebenzisi.

  1. Cwangcisa uhlobo lomsebenzisi (uMsebenzisi okanye uMphathi).
  2. Cwangcisa iSimo soMsebenzisi (Siyasebenza ukuya kuValwe).
  3. Ngenisa i-ID yeTag eyodwa. (ukhetho)
  4. Cofa Gcina.
