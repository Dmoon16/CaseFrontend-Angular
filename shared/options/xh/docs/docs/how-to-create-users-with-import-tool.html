Ubenza njani abasebenzisi ngeSixhobo sokungeniswa ngaphandle

Eli nqaku libonisa ii-admins indlela yokwenza abasebenzisi ngesixhobo sokungeniswa.

Ukuvula isixhobo sokungeniswa ngaphandle

Abalawuli banokusebenzisa isixhobo sokuNgenisa ukulayisha ngobuninzi abasebenzisi abatsha kunye nokuguqula iirekhodi ezikhoyo zabasebenzisi.

  1. Ngena kwiakhawunti admin.
  2. Khangela usetyenziso kwiphepha leApps kwaye ucofe iqhosha elithi ADMIN.
  3. Kwiphaneli yoLawulo, cofa ithebhu yoManyano kwaye emva koko U-IMPORTS kwimenyu engaphantsi.
  4. Cofa iqhosha ENTSHA. Ibhokisi yencoko yababini ethi Yongeza Urhweba ngaphandle ibonisa kumkhangeli zincwadi omtsha thebhu.

Ukwenza iirekhodi

Emva kokuba Ibhokisi yencoko yababini Yongeza ngaphandle ivula:

  1. Cofa indawo yodidi kwaye ukhethe ukhetho (bona ngezantsi). Ikhonkco liyabonisa.
  2. Cofa ikhonkco lokukhuphela iSample . Ifayile ye-CSV ikhutshelwa kwisixhobo sakho.

Emva koko, vula ifayile kwaye wongeze iirekhodi. Cofa apha ukuze ufunde indlela yokusebenzisa idatha yefomathi kwifayile ye-CSV.

Uhlobo

Xa ucofa indawo yoHlobo, imenyu eyehlayo ibonisa. Khetha enye yezimbini iinketho zokurhweba ngaphandle kwesambuku.

:isikhuzo: QAPHELA Imenyu yotsalo-phantsi inokhetho oluninzi, kodwa zimbini kuphela eziyelelene nesixhobo sokungeniswa ngaphandle. Inketho nganye inesampulu eyahlukileyo yefayile ye-CSV.

Yongeza abasebenzisi

  • Khetha olu khetho kuluhlu olulula lokulayisha iirekhodi zabasebenzisi.
  • Cofa ikhonkco lokukhuphela iSample ukuze uvule ifayile ye-CSV enezihloko zekholamu zohlobo Longeza abasebenzisi.

Guqula abasebenzisi

  • Khetha olu khetho lokuhlaziya idatha yabasebenzisi bangoku.
  • Cofa ikhonkco lokukhuphela iSample ukuze uvule ifayile ye-CSV enezihloko zekholamu zohlobo lokuNgenisa abasebenzisi.

Kunyuswa ifayile

Ungalayisha ifayile kwangoko okanye ubuyele kwibhokisi yencoko yababini yoNgenisa Izinto ngaphandle kwamva kwaye ugqibezele la manyathelo.

  1. Vula i Dibanisa ibhokisi yencoko yababini.
  2. Ngenisa igama elilodwa lefayile ye-CSV.
  3. Cofa i-plus (+) icon ukulayisha ifayile kwisixhobo sakho.
  4. Cofa indawo yodidi kwaye ukhethe ukhetho. (jonga ngasentla)
  5. Cofa Gcina.

Iyaqhuba

Ngokuxhomekeke kwinani leerekhodi kwifayile kunye nomthwalo wenkqubo, iqonga lingathatha imizuzu emininzi ukucubungula ukungenisa.

Xa ukungenisa kugqityiwe, iqonga lihlengahlengisa iirekhodi zomsebenzisi (oko kukuthi, liyongeza, lilungise, okanye likhubaze) kwideshibhodi yabasebenzisi bolawulo.

Ukusilela

Ukuba umsebenzi wokungenisa elizweni uyasilela, iqonga libuyisela umyalezo ongaphumeleliyo. Kwiimeko ezininzi, unobangela wokusilela kukufomatha idatha.

Ukubona unobangela wokusilela:

  1. Cofa Jonga phantsi kwesihloko esithi Isenzo.
  2. Kwiphepha elilandelayo, hambisa imouse kwi icon Yoyisakele. Inqaku lesixhobo libonisa impazamo.
  3. Yiya kwifayile yeCVS kwaye ulungise kwaye ulungise imposiso.
  4. Layisha ifayile kwakhona.
