Ungena njani

Funda indlela yokungena kwi-akhawunti yakho ekhoyo yeCase Active.

Ngena

  1. Yiya kwiphepha lokungena .
  2. Cofa iqhosha elithi NGENA.
  3. Faka i-imeyile yakho.
  4. Cofa OKULANDELAYO.

Ukuqinisekisa i-akhawunti

  1. Vula ithebhu entsha yebrawuza kwaye ungene kwiakhawunti yakho ye-imeyile.
  2. Khuphela ikhowudi enamanani amathandathu.
  3. Buyela kwiCase Active thebhu.
  4. Ncamathelisa ikhowudi kwiindawo. Ifestile entsha ibonisa.

I-Case Active ibonisa iphepha le-Apps.

  • Qaphela Iqonga libonisa iqhosha le-ADMIN ecaleni kweqhosha le-APP kubasebenzisi abanamalungelo olawulo.
