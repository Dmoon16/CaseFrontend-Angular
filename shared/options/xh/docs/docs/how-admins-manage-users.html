Indlela Abalawuli Abalawula Ngayo Abasebenzisi

Abalawuli benza, balungise okanye bavale ukuba ngubani onokufikelela kwii-apps kunye namatyala. Xa umsebenzisi engena eqongeni okokuqala, kufuneka agqibezele iprofayile.

  • Cofa apha ukuze ufunde ngakumbi malunga nomsebenzi omtsha weprofayile yomsebenzisi.

Ukulawula umsebenzisi

Itheyibhile engezantsi ibonisa iindlela ezintlanu zokulawula umsebenzisi. Cofa ikhonkco ukufunda ukuba njani.

Izaziso

Iqonga alithumeli abasebenzisi isaziso xa bongezwa kwi-app. Nangona kunjalo, ii-admins zingenza ukuba izaziso zikwazi ukwenza izabelo zamatyala.
