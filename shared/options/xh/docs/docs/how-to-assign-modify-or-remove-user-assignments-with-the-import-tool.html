Eli nqaku libonisa indlela yokulawula abasebenzisi betyala kunye nesixhobo sokungeniswa.

Abalawuli banoku:

  • yabela abasebenzisi kwityala
  • guqula kancinane abasebenzisi abasele benikezelwe kwityala
  • susa abasebenzisi ababelwe ityala

Ukuvula isixhobo sokungeniswa ngaphandle

  1. Ngena kwiakhawunti admin.
  2. Khangela usetyenziso kwiphepha leApps kwaye ucofe iqhosha elithi ADMIN.
  3. Cofa i Uhlanganiso isithuba kwaye emva koko IMPORTS kwimenyu engaphantsi.
  4. Cofa iqhosha ENTSHA. Yongeza ngaphandle kwebhokisi yencoko yababini ibonisa.

Ukwenza iirekhodi

Khuphela ifayile enye okanye ngaphezulu kwe CSV. Sebenzisa ezi fayile ukongeza, ukulungisa, okanye ukususa abasebenzisi betyala. Emva kwebhokisi yencoko yababini ethi Yongeza ngaphandle ivula:

  1. Cofa indawo yoHlobo.
  2. Skrolela kwimenyu eyehlayo kwaye ukhethe ukhetho (jonga ngezantsi). Ikhonkco liyabonisa.
  3. Cofa ikhonkco lokukhuphela iSample. Ifayile ye-CSV ikhutshelwa kwisixhobo sakho.

img

Vula ifayile kwaye wongeze iirekhodi. Cofa apha ukuze ufunde indlela yokusebenzisa idatha yefomathi kwifayile ye-CSV.

Iinketho zefayile

Yabela abasebenzisi kwiimeko * Khetha olu khetho ukongeza abasebenzisi kwityala. * Shiya case_role_order cell ingenanto. * Yiya kwiphepha labasebenzisi kwaye ukope i-ID yeTag yalo mntu. Ukuba indawo ye-ID yeTag ayinanto, ngenisa i-ID eyodwa, cofa Gcina, uze ukope i-ID kwifayile ye-CSV.

Guqula abasebenzisi ababelwe iimeko * Sebenzisa i-Case Tag ID kwikholamu ye-case_ID.

Susa abasebenzisi ababelwe iimeko * Khetha olu khetho ukususa abasebenzisi ababelwe ityala.

img

Kunyuswa ifayile

Ungalayisha ifayile kwangoko okanye ubuyele kwibhokisi yencoko yababini yoNgenisa Izinto ngaphandle kwamva kwaye ugqibezele la manyathelo.

  1. Vula ibhokisi yencoko yababini ethi Yongeza Izinto ngaphandle.
  2. Ngenisa igama elilodwa lefayile ye-CSV.
  3. Cofa i-plus (+) icon ukulayisha ifayile kwisixhobo sakho.
  4. Cofa i-Thayipha indawo kwaye ukhethe ukhetho. (jonga ngasentla)
  5. Cofa Gcina.

Iyaqhuba

Ngokuxhomekeke kwinani leerekhodi kwifayile kunye nomthwalo wenkqubo, iqonga lingathatha imizuzu emininzi ukucubungula ukungenisa.

Xa ukungenisa elizweni kugqityiwe, iqonga lihlengahlengisa iirekhodi zomsebenzisi (oko kukuthi, liyongeza, lilungisa, okanye lisusa) kwiphepha le-Cases.

Ukusilela

Ukuba umsebenzi wokungenisa elizweni uyasilela, iqonga libuyisela umyalezo ongaphumeleliyo. Kwiimeko ezininzi, unobangela wokusilela kukufomatha idatha.

Ukubona unobangela wokusilela:

  1. Cofa Jonga phantsi kwesihloko esithi Isenzo.
  2. Kwiphepha elilandelayo, hambisa imouse kwi icon Yoyisakele. Inqaku lesixhobo libonisa impazamo.
  3. Yiya kwifayile yeCVS kwaye ulungise kwaye ulungise imposiso.
  4. Layisha ifayile kwakhona.
