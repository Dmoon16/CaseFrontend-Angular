Uzenza njani ngokuzenzekelayo amatyala ngekhonkco lokubhalisa loluntu

Eli nqaku libonisa i-admins indlela yokwenza ityala (kwaye ungeze umsebenzisi ngexesha elifanayo) ngokuthumela ikhonkco elilodwa lokubhalisa.

Ukufumana ikhonkco

  1. Ngena kwiakhawunti admin.
  2. Khangela usetyenziso kwiphepha leApps kwaye ucofe iqhosha elithi ADMIN.
  3. Kwiphaneli yoLawulo, cofa i Izicwangciso menu tab.
  4. Qinisekisa ukuba ithebhu yemenyu engaphantsi ye-SYSTEM SETUP ikhethiwe.

img

Ukuqwalasela useto

Skrolela ezantsi kwi-USER SIGNUP isihloko.

Qinisekisa ukuba indawo yoVumela uBhalo isetelwe ku Vula. (kuyafuneka) Xa olu phawu luvuliwe, i-Case Active yenza ikhonkco elilodwa.

  1. Cofa i-Auto Create Case field kwaye ukhethe Vula. I-Case User Role field ibonisa.
  2. Khetha indima. (bona ngezantsi)

Iindima zabasebenzisi betyala

Kukho iindima ezintathu zabasebenzisi.

  • abathengi (kucetyiswa)
  • ukungagqibeki
  • abasebenzi

Ukuthumela ikhonkco

Yabelana ngekhonkco nomntu omnye okanye ngaphezulu. Umsebenzisi ngamnye ocofa ikhonkco kwaye agqibezele ukubhalisa ubangela iqonga ukuvelisa imeko entsha kwiphepha lamatyala kwaye abele ngokuzenzakalelayo umsebenzisi kwimeko.
