कसरी प्रयोगकर्ता प्रोफाइल अपडेट गर्ने

यस लेखले प्रयोगकर्ताहरूलाई उनीहरूको प्रोफाइल पृष्ठमा जानकारी कसरी अद्यावधिक गर्ने भनेर देखाउँछ।

  • प्रयोगकर्ताहरूले आफ्नै प्रोफाइलहरू अद्यावधिक गर्न सक्छन्।
  • प्रशासकहरूले कुनै पनि प्रयोगकर्ताको प्रोफाइल पृष्ठ अपडेट गर्न सक्छन्।

प्रयोगकर्ताहरूको लागि

पहिलो चरण प्रोफाइल पृष्ठ खोल्न छ। प्रयोगकर्ताहरू र प्रशासकहरूले तिनीहरूको आफ्नै प्रोफाइल पृष्ठ अद्यावधिक गर्न यी चरणहरू पालना गर्नुहोस्।

  1. तपाईंको केस सक्रिय खातामा लग इन गर्नुहोस्।
  2. पृष्ठको शीर्षमा प्रोफाइल ट्याबमा क्लिक गर्नुहोस्।
  3. परिवर्तनहरू गर्नुहोस्।
  4. बचत क्लिक गर्नुहोस्।

प्रशासकहरूको लागि

प्रशासकहरूले अन्य प्रयोगकर्ताहरूको लागि प्रोफाइल जानकारी पनि अद्यावधिक गर्न सक्छन्। यसमा HOST USER INFO शीर्षक अन्तर्गत तीनवटा क्षेत्रहरू समावेश छन्।

  1. तपाईंको केस सक्रिय प्रशासक खातामा लग इन गर्नुहोस्।
  2. एउटा एप पत्ता लगाउनुहोस् र एडमिन बटनमा क्लिक गर्नुहोस्।
  3. शीर्ष मेनुमा प्रयोगकर्ताहरू क्लिक गर्नुहोस्।
  4. उप-मेनुमा जानुहोस् र तपाईंले पहुँच गर्न चाहनुभएको प्रयोगकर्ता सूची चयन गर्नुहोस् (सक्रिय वा असक्षम)। पूर्वनिर्धारित सेटिङ सक्रिय छ।
  5. सूचीमा एक प्रयोगकर्ता पत्ता लगाउनुहोस् र सम्पादन क्लिक गर्नुहोस्।
  6. आवश्यकता अनुसार परिवर्तन गर्नुहोस्। (तल हेर)
  7. बचत क्लिक गर्नुहोस्।

प्रशासकहरूले प्रयोगकर्ता जानकारी परिवर्तन गर्न सक्छन्।
