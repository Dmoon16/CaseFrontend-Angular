UI मेनु मार्फत केसहरू कसरी सिर्जना वा परिमार्जन गर्ने

यस लेखले व्यवस्थापकहरूलाई कसरी प्लेटफर्मको प्रयोगकर्ता इन्टरफेससँग केस सिर्जना गर्ने भनेर देखाउँछ। प्रशासकहरूले प्रयोगकर्ताहरूलाई नयाँ केसमा थप्न र अवस्थित केसहरू अद्यावधिक गर्न पनि सक्छन्।

मुद्दा सिर्जना गर्दै

  1. प्रशासक खातामा लग इन गर्नुहोस्।
  2. एप्स पृष्ठमा एउटा एप पत्ता लगाउनुहोस् र एडमिन बटनमा क्लिक गर्नुहोस्।
  3. व्यवस्थापक प्यानलमा, केस मेनु ट्याबमा क्लिक गर्नुहोस्।
  4. नयाँ बटन क्लिक गर्नुहोस्। एउटा नयाँ सञ्झ्याल प्रदर्शित हुन्छ।

आवश्यक क्षेत्रहरू भर्नुहोस् र बचत गर्नुहोस् क्लिक गर्नुहोस्।

प्रयोगकर्ताहरू थप्दै

वैकल्पिक रूपमा, प्रशासकहरूले यस पृष्ठमा केसहरूमा प्रयोगकर्ताहरू थप्न सक्छन्। तल स्क्रोल गर्नुहोस् र ASSIN USERS बटनमा क्लिक गर्नुहोस्।

असाइन प्रयोगकर्ता मोडलमा, क्षेत्रहरू भर्नुहोस्। प्रयोगकर्तालाई इमेल सूचना पठाउन बाकस जाँच गर्नुहोस्। (वैकल्पिक)

मुद्दा परिमार्जन गर्दै

थोरै संख्यामा केसहरू अद्यावधिक गर्न, सम्पादन उपकरण प्रयोग गर्नुहोस् र प्रत्येक व्यक्तिगत केसमा परिवर्तन गर्नुहोस्।

  • नोट एउटै बैठकमा केसहरूको ठूलो संख्या अद्यावधिक गर्न, आयात उपकरण प्रयोग गर्ने विचार गर्नुहोस्
  1. प्रशासक खातामा लग इन गर्नुहोस्।
  2. एप्स पृष्ठमा एउटा एप पत्ता लगाउनुहोस् र एडमिन बटनमा क्लिक गर्नुहोस्।
  3. व्यवस्थापक प्यानलमा, केस मेनु ट्याबमा क्लिक गर्नुहोस्।
  4. केस पत्ता लगाउनुहोस् र सम्पादन क्लिक गर्नुहोस्।
  5. परिवर्तनहरू गर्नुहोस् र बचत गर्नुहोस् क्लिक गर्नुहोस्।
