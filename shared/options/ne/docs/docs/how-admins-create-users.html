कसरी प्रशासकहरूले प्रयोगकर्ताहरू सिर्जना गर्छन्

प्रशासकहरूले एपहरू र केसहरू पहुँच गर्न सक्ने व्यक्तिहरू सिर्जना, परिमार्जन वा असक्षम पार्छन्। जब प्रयोगकर्ता पहिलो पटक प्लेटफर्ममा लग इन हुन्छ, तिनीहरूले प्रोफाइल कार्यप्रवाह पूरा गर्न आवश्यक छ।

  • नयाँ प्रयोगकर्ता प्रोफाइल कार्यप्रवाह बारे थप जान्न यहाँ क्लिक गर्नुहोस्।
