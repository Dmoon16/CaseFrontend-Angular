कसरी प्रशासकहरूले केसहरूमा प्रयोगकर्ता असाइनमेन्टहरू प्रबन्ध गर्छन्

प्रयोगकर्ताहरू र केस असाइनमेन्टहरू व्यवस्थापन गर्ने पाँच तरिकाहरू छन्। कसरी जान्नको लागि लिङ्क क्लिक गर्नुहोस्।

विधि टाइप गर्नुहोस् कार्यहरू विवरणहरू
मुद्दा सिर्जना गर्दा UI थप्नुहोस् प्रशासक बटन > केसहरू > नयाँ बटन > प्रयोगकर्ताहरूलाई तोक्नुहोस् बटन
सार्वजनिक साइनअप लिङ्क प्रयोग गरेर प्रयोगकर्तालाई स्वत: असाइन गर्नुहोस् प्रयोगकर्ता कार्य थप्नुहोस् प्रशासक बटन > सेटिङहरू > प्रणाली सेटअप पृष्ठ
अवस्थित केसमा UI प्रयोग गर्दै UI थप्नुहोस् र हटाउनुहोस् प्रशासक बटन > केसहरू > प्रयोगकर्ताहरू तोक्नुहोस्
उपकरण आयात गर्नुहोस् UI थप्नुहोस्, परिमार्जन गर्नुहोस्, र हटाउनुहोस् प्रशासक बटन > एकीकरण > आयातहरू
