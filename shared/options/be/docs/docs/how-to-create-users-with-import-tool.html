Як стварыць карыстальнікаў з дапамогай інструмента імпарту

Гэты артыкул паказвае адміністратарам, як ствараць карыстальнікаў з дапамогай інструмента Імпарт.

Адкрыццё інструмента імпарту

Адміністратары могуць выкарыстоўваць інструмент імпарту для масавай загрузкі новых карыстальнікаў і змены існуючых запісаў карыстальнікаў.

  1. Увайдзіце ва ўліковы запіс адміністратара.
  2. Знайдзіце праграму на старонцы праграм і націсніце кнопку АДМІНІСТРАЦЫЯ.
  3. На панэлі адміністратара націсніце ўкладку «Інтэграцыя», а затым «ІМПАРТ» у падменю.
  4. Націсніце кнопку НОВЫ. Дыялогавае акно "Дадаць імпарт" адлюстроўваецца ў новай укладцы браўзера.

Стварэнне запісаў

Пасля адкрыцця дыялогавага акна "Дадаць імпарт":

  1. Пстрыкніце поле " Тып " і абярыце опцыю (гл. ніжэй). Адлюструецца спасылка.
  2. Націсніце спасылку « Спампаваць узор ». Файл CSV спампоўваецца на вашу прыладу.

Далей адкрыйце файл і дадайце запісы. Націсніце тут , каб даведацца, як выкарыстоўваць дадзеныя фармату ў файле CSV.

малюнак

Тып

Калі вы націснеце поле "Тып", з'явіцца выпадальнае меню. Выберыце адзін з двух варыянтаў масавага імпарту.

:exclamation: ЗАЎВАГА . У выпадальным меню шмат параметраў, але толькі два звязаны з інструментам "Імпарт". Кожны варыянт мае свой узор файла CSV.

Дадаць карыстальнікаў

  • Выберыце гэты параметр для простай масавай загрузкі запісаў карыстальнікаў.
  • Пстрыкніце спасылку «Спампаваць узор», каб адкрыць файл CSV з загалоўкамі слупкоў для тыпу «Імпарт дадання карыстальнікаў».

Змяніць карыстальнікаў

  • Выберыце гэты параметр, каб абнавіць даныя для бягучых карыстальнікаў.
  • Пстрыкніце спасылку «Спампаваць узор», каб адкрыць файл CSV з загалоўкамі слупкоў для тыпу «Імпарт змяніць карыстальнікаў».

Загрузка файла

Вы можаце загрузіць файл адразу ці вярнуцца ў дыялогавае акно "Дадаць імпарт" пазней і выканаць гэтыя дзеянні.

  1. Адкрыйце дыялогавае акно «Дадаць імпарт».
  2. Увядзіце унікальнае імя для файла CSV.
  3. Націсніце на значок плюса (+), каб загрузіць файл з прылады.
  4. Націсніце поле " Тып " і абярыце варыянт. (гл. вышэй)
  5. Націсніце ЗАХАВАЦЬ.

малюнак

Апрацоўка

У залежнасці ад колькасці запісаў у файле і загрузкі сістэмы платформе можа спатрэбіцца некалькі хвілін, каб апрацаваць імпарт.

малюнак

Пасля завяршэння імпарту платформа карэктуе запісы карыстальнікаў (г.зн. дадае, змяняе або адключае) на прыборнай панэлі карыстальнікаў адміністратара.

Няўдача

Калі аперацыя імпарту не атрымоўваецца, платформа вяртае паведамленне "Збой". У большасці выпадкаў прычынай збою з'яўляецца фарматаванне дадзеных.

малюнак

Каб убачыць прычыну збою:

  1. Націсніце " Прагляд " пад загалоўкам "Дзеянне".
  2. На наступнай старонцы навядзіце курсор мышы на значок "Збой". Падказка адлюстроўвае памылку.
  3. Перайдзіце да файла CVS і выправіце памылку.
  4. Загрузіце файл яшчэ раз.
