Фарматаванне даных прысваення рэгістра карыстальніка з дапамогай інструмента імпарту

Ніжэй прыведзены спіс палёў даных у файлах CSV. Кожны варыянт імпарту мае розную камбінацыю палёў.

case_ID

  • Увядзіце рэгістр User або Tag ID.
  • Перайдзіце на старонку Справы > Рэдагаваць > Ідэнтыфікатар тэга АБО Перайдзіце на старонку Справы > Рэдагаваць > Ідэнтыфікатар справы ў URL.

Ідэнтыфікатар карыстальніка

  • Увядзіце ідэнтыфікатар карыстальніка або ідэнтыфікатар тэга.
  • Перайдзіце на старонку Карыстальнікі > Рэдагаваць > Ідэнтыфікатар тэга АБО Перайдзіце на старонку Карыстальнікі > Рэдагаваць > Ідэнтыфікатар карыстальніка ў URL.

case_role_order

  • Пакінуць ячэйку пустой.

case_role_ID

  • Увядзіце: кліенты, супрацоўнікі або па змаўчанні.

case_notify

  • 0 = не
  • 1 = Так, адправіць карыстальніку паведамленне па электроннай пошце.
