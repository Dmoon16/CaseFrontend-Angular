Як аўтаматычна ствараць кейсы па публічнай спасылцы для рэгістрацыі

Гэты артыкул паказвае адміністратарам, як стварыць заяўку (і дадаць карыстальніка адначасова), даслаўшы адну спасылку для рэгістрацыі.

Пошук спасылкі

  1. Увайдзіце ва ўліковы запіс адміністратара.
  2. Знайдзіце праграму на старонцы праграм і націсніце кнопку АДМІНІСТРАЦЫЯ.
  3. На панэлі адміністратара націсніце ўкладку меню Налады.
  4. Пераканайцеся, што выбрана ўкладка падменю SYSTEM SETUP.

малюнак

Налада параметраў

Пракруціце ўніз да загалоўка РЭГІСТРАЦЫЯ КАРЫСТАЛЬНІКА.

Пераканайцеся, што для поля «Дазволіць рэгістрацыю» ўстаноўлена значэнне «Уключана». (абавязкова) Калі гэтая функцыя ўключана, Case Active стварае унікальную спасылку.

  1. Націсніце поле "Аўтаматычнае стварэнне справы" і выберыце "Уключана". Адлюструецца поле Case User Role.
  2. Выберыце ролю. (Глядзі ніжэй)

малюнак

Ролі карыстальнікаў Case

Ёсць тры ролі карыстальніка case.

  • кліенты (рэкамендуецца)
  • па змаўчанні
  • персанал

Адпраўка спасылкі

Падзяліцеся спасылкай з адным або некалькімі людзьмі. Кожны карыстальнік, які націскае спасылку і завяршае рэгістрацыю, запускае платформу для стварэння новай справы на старонцы спраў і аўтаматычнага прызначэння карыстальніка ў справе.
