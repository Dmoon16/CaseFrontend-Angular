Фарматаванне дадзеных кіравання карыстальнікамі з дапамогай інструмента імпарту

Ніжэй прыведзены спіс палёў даных у файлах CSV. Кожны варыянт імпарту мае розную камбінацыю палёў.

імя карыстальніка

  • Увядзіце адрас электроннай пошты карыстальніка.

названае імя

  • Увядзіце імя карыстальніка.

прозвішча

  • Увядзіце прозвішча карыстальніка.

лакаль

  • У гэтым слупку правяраецца выбраная карыстальнікам мова.
  • Увядзіце en для англійскай, sp для іспанскай.

інфармацыя аб зоне

  • Гэта задае часавы пояс карыстальніка. Увядзіце кантынент і горад без прабелаў (выкарыстоўвайце падкрэсліванне).
  • Неабходна выкарыстоўваць опцыю ў выпадаючым меню Профіль > МОВА І РЭГІЁН > Часавы пояс.
  • Выкарыстоўвайце найбольш прыдатны гадзінны пояс. Напрыклад, Бостана няма ў выпадальным спісе, таму мы можам выкарыстоўваць Нью-Ёрк (або любы горад у часавым поясе EST/EDT).

ідэнтыфікатар_тэга_хаста

  • Увядзіце пастаянную метку.
  • az, 0-9, -, / дапускаюцца.

статус_справы

  • 0 = закрыта
  • 1 = актыўны
