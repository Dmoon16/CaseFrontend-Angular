Uburyo bwo Kurema no Guhindura Abakoresha hamwe na UI

Iyi ngingo irerekana abadmin uburyo bwo gukora abakoresha no kuvugurura inyandiko hamwe nu rubuga rwabakoresha. Abadamu barashobora kandi kongeramo abakoresha murubanza rushya no guhindura imanza zisanzwe.

Kurema umukoresha

  1. Injira kuri konte ya admin.
  2. Shakisha porogaramu kurupapuro rwa porogaramu hanyuma ukande buto ya ADMIN.
  3. Kuruhande rwa Admin, kanda ahanditse Abakoresha.
  4. Kanda buto NSHYA. Idirishya rishya ryerekana.

Uzuza imirima hanyuma ukande SAVE. Case Active yongeraho izina kurutonde rwabakoresha bahujwe niyi porogaramu.

Guhindura umukoresha

Kuvugurura umubare muto wabakoresha inyandiko, koresha igikoresho cyo guhindura hanyuma uhindure buri mukoresha kugiti cye.

  • Icyitonderwa Guhindura umubare munini wabakoresha inyandiko mwicaye rimwe, tekereza gukoresha igikoresho cyo Kuzana .
  1. Kuruhande rwa Admin, kanda ahanditse Abakoresha.
  2. Shakisha umukoresha hanyuma ukande ahanditse.
  3. Kora impinduka hanyuma ukande SAVE.

Intambwe ikurikira

Case Active ntabwo yohereza umukoresha imenyesha rya imeri. Abadamu bagomba kuvugana numukoresha no kuboherereza umurongo kurupapuro rwurugo. Umukoresha noneho akanda buto ya LOGIN arangiza akazi ko kwiyandikisha gushya.

  • Iyi ngingo isobanura ibikorwa byo kwiyandikisha kubakoresha bashya.
