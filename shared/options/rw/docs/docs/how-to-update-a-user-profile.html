Nigute ushobora kuvugurura umwirondoro wumukoresha

Iyi ngingo yerekana abakoresha uburyo bwo kuvugurura amakuru kurupapuro rwabo.

  • Abakoresha barashobora kuvugurura imyirondoro yabo.
  • Abayobozi barashobora kuvugurura urupapuro rwumwirondoro wumukoresha uwo ari we wese.

Kubakoresha

Intambwe yambere ni ugukingura urupapuro rwumwirondoro. Abakoresha na ba admin bakurikiza izi ntambwe kugirango bavugurure urupapuro rwabo.

  1. Injira kuri konte yawe yibikorwa.
  2. Kanda ahanditse Umwirondoro hejuru yurupapuro.
  3. Hindura.
  4. Kanda Kubika.

Imirima myinshi irahinduka, kandi imirima imwe irakenewe. Imirima isabwa ni:

  • Izina ryambere
  • Izina ryanyuma
  • Aderesi imeri
  • Ururimi
  • Timezone

Kubayobozi

Abadamu barashobora kandi kuvugurura amakuru yumwirondoro kubandi bakoresha. Ibi birimo imirima itatu munsi ya HOST USER INFO.

  1. Injira muri konte yawe ya Admin ikora.
  2. Shakisha porogaramu hanyuma ukande buto ya ADMIN.
  3. Kanda Abakoresha kuri menu yo hejuru.
  4. Jya kuri sub-menu hanyuma uhitemo urutonde rwabakoresha ushaka kugeraho (BIKORWA cyangwa BISANZWE). Igenamiterere risanzwe rirakora.
  5. Shakisha umukoresha kurutonde hanyuma ukande Guhindura.
  6. Kora impinduka nkuko bisabwa. (reba hano hepfo)
  7. Kanda Kubika.

Hindura amakuru yumukoresha

Abayobozi barashobora guhindura amakuru yumukoresha.

  1. Shiraho Ubwoko bw'Umukoresha (Umukoresha cyangwa Umuyobozi).
  2. Shiraho Umukoresha Imiterere (Ikora kuri Disable).
  3. Injira ID idasanzwe. (bidashoboka)
  4. Kanda Kubika.
