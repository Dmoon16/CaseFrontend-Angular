Uburyo ba Admin barema abakoresha

Abadamu barema, bahindura cyangwa bahagarike ushobora kubona porogaramu n'imanza. Iyo umukoresha yinjiye kumurongo wambere, bakeneye kurangiza umwirondoro wakazi.

  • Kanda hano kugirango umenye byinshi kubyerekeye umukoresha mushya umwirondoro wakazi.

Kumenyesha

Ihuriro ntabwo ryohereza abakoresha imenyesha iyo ryongewe kuri porogaramu. Ariko, ba admin barashobora gukora imenyesha kubikorwa bashinzwe.
