Uburyo Abakoresha Kurema Konti Nshya

Nyuma yuko admin yongeyeho umukoresha murubanza: * niba umukoresha ari mushya kuri caseactive, uyikoresha agomba kurangiza akazi kugirango akore konti kandi yemere ubutumire bwo kwinjira murubanza. * niba umukoresha afite acoount iriho, uyikoresha agomba kwemera ubutumire bwo kugera murubanza.

Iyi ngingo isobanura intambwe umukoresha agomba kurangiza kugirango akore konti yabo.

Gutangira

Hariho uburyo bubiri umukoresha ashobora kongerwaho muri progaramu ya caseactive

  • Na admin: Niba admin yaremye umukoresha akoresheje urubuga UI cyangwa Igikoresho cyo Kuzana , admin agomba kohereza umukoresha umurongo kuri konte.caseactive.com kugirango akomeze kwinjira.
  • Numukoresha: Niba umukoresha yarashizeho konte akoresheje ihuza ryinjira, uyikoresha azahita yoherezwa kuri konte.caseactive.com kugirango akomeze kwinjira.

Kwinjira

Iyo umukoresha yinjiye kuri konte yabo ya CaseActive kunshuro yambere, urubuga rusaba umukoresha kuri:

  • fungura konti yabo imeri hanyuma wandukure / wandike kode yumutekano itandatu
  • kuzuza umwirondoro wa konti

Nyuma yuko umukoresha akanze UPDATE, urupapuro rwa porogaramu rwerekana. Urupapuro rwerekana izina rya porogaramu (izina ryisosiyete), ubwoko bwabakoresha (uruhare), imiterere, na buto y'Ubutumire.

img

Kugera kuri porogaramu, uyikoresha agomba kwemera ubutumire. Umukoresha agomba:

  1. Kanda buto y'Ubutumire.
  2. Ongera usuzume uruhushya.
  3. Kanda ahanditse Amatangazo kugirango usubiremo kandi uhindure (bidashoboka) amagambo.
  4. Kanda ICYEMEZO.

img

Iyo umukoresha yemeye, urubuga rwerekana buto ya APP kurupapuro rwabakoresha. Akabuto gaha umukoresha uburyo bwimanza.

Guhindura igenamiterere ryabakoresha, uyikoresha akanda ellipsis.

img

Umukoresha arashobora: * kuvanaho porogaramu * guhindura igenamiterere ryo kumenyesha

img

Nyuma yuko umukoresha arangije akazi, izina ryumukoresha rigaragara kurutonde rwabakoresha ba admin. Urutonde rwerekana abakoresha bose, binjiye cyangwa batinjiye.
