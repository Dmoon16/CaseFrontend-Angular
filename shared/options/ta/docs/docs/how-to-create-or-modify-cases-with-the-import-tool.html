இறக்குமதி கருவி மூலம் கேஸ்களை உருவாக்குவது அல்லது மாற்றுவது எப்படி

இறக்குமதி கருவி மூலம் ஏற்கனவே உள்ள கேஸ்களை எப்படி உருவாக்குவது அல்லது மாற்றுவது என்பதை நிர்வாகிகளுக்கு இந்தக் கட்டுரை காட்டுகிறது.

இறக்குமதி கருவியைத் திறக்கிறது

  1. நிர்வாகி கணக்கில் உள்நுழைக.
  2. ஆப்ஸ் பக்கத்தில் பயன்பாட்டைக் கண்டறிந்து, நிர்வாகி பொத்தானைக் கிளிக் செய்யவும்.
  3. நிர்வாக குழுவில், ஒருங்கிணைப்பு தாவலைக் கிளிக் செய்து, துணைமெனுவில் இறக்குமதி செய்யவும்.
  4. புதிய பொத்தானைக் கிளிக் செய்யவும். இறக்குமதிகளைச் சேர் உரையாடல் பெட்டி புதிய உலாவி தாவலில் காண்பிக்கப்படும்.

வழக்குகளை உருவாக்குதல்

இறக்குமதிகளைச் சேர் உரையாடல் பெட்டி திறந்த பிறகு:

  1. வகை புலத்தைக் கிளிக் செய்து ஒரு விருப்பத்தைத் தேர்ந்தெடுக்கவும் (கீழே காண்க). ஒரு இணைப்பு காட்டுகிறது.
  2. பதிவிறக்க மாதிரி இணைப்பைக் கிளிக் செய்யவும். ஒரு CSV கோப்பு உங்கள் சாதனத்தில் பதிவிறக்குகிறது.

அடுத்து, கோப்பைத் திறந்து வழக்குகளைச் சேர்க்கவும்.

வகை

நீங்கள் வகை புலத்தைக் கிளிக் செய்யும் போது, கீழ்தோன்றும் மெனு தோன்றும். மொத்த கேஸ் இறக்குமதிக்கான இரண்டு விருப்பங்களில் ஒன்றைத் தேர்ந்தெடுக்கவும்.

  • குறிப்பு கீழ்தோன்றும் மெனுவில் பல விருப்பங்கள் உள்ளன, ஆனால் இரண்டு மட்டுமே கேஸை நிர்வகித்தல் தொடர்பானவை. ஒவ்வொரு விருப்பத்திற்கும் வெவ்வேறு மாதிரி CSV கோப்பு உள்ளது.

வழக்குகளைச் சேர்க்கவும்

  • கேஸ்களின் எளிய மொத்தப் பதிவேற்றத்திற்கு இந்த விருப்பத்தைத் தேர்வு செய்யவும்.
  • கேஸ்களை உருவாக்கு இறக்குமதி வகைக்கான நெடுவரிசை தலைப்புகளுடன் CSV கோப்பைத் திறக்க பதிவிறக்க மாதிரி இணைப்பைக் கிளிக் செய்யவும்.

வழக்குகளை மாற்றவும்

  • தற்போதைய வழக்குகளைப் புதுப்பிக்க இந்த விருப்பத்தைத் தேர்ந்தெடுக்கவும்.
  • மாற்றியமைக்கப்பட்ட வழக்குகள் இறக்குமதி வகைக்கான நெடுவரிசை தலைப்புகளுடன் கூடிய CSV கோப்பைத் திறக்க, பதிவிறக்க மாதிரி இணைப்பைக் கிளிக் செய்யவும்.

கோப்பை பதிவேற்றுகிறது

நீங்கள் இப்போதே ஒரு கோப்பைப் பதிவேற்றலாம் அல்லது பிற்காலத்தில் இறக்குமதிகளைச் சேர் உரையாடல் பெட்டிக்குத் திரும்பி இந்தப் படிகளை முடிக்கலாம்.

  1. இறக்குமதிகளைச் சேர் உரையாடல் பெட்டியைத் திறக்கவும்.
  2. CSV கோப்பிற்கான தனிப்பட்ட பெயரை உள்ளிடவும்.
  3. உங்கள் சாதனத்திலிருந்து கோப்பைப் பதிவேற்ற, பிளஸ் (+) ஐகானைக் கிளிக் செய்யவும்.
  4. வகை புலத்தைக் கிளிக் செய்து ஒரு விருப்பத்தைத் தேர்ந்தெடுக்கவும். (மேலே பார்க்க)
  5. சேமி என்பதைக் கிளிக் செய்யவும்.

இவை தேவையான புலங்கள் மற்றும் முதல் தேவைகள். வடிவமைப்பு விவரங்களை மதிப்பாய்வு செய்ய இங்கே கிளிக் செய்யவும் .

செயலாக்கம்

கோப்பு மற்றும் கணினி ஏற்றத்தில் உள்ள பதிவுகளின் எண்ணிக்கையைப் பொறுத்து, தளம் இறக்குமதியைச் செயல்படுத்த பல நிமிடங்கள் ஆகலாம்.

img

இறக்குமதி முடிந்ததும், நிர்வாகியின் கேஸ் டாஷ்போர்டில் வழக்கு பதிவுகளை (அதாவது சேர்க்கிறது அல்லது மாற்றுகிறது) இயங்குதளம் சரிசெய்கிறது.

தோல்வி

இறக்குமதி செயல்பாடு தோல்வியுற்றால், தளம் தோல்வியுற்ற செய்தியை வழங்குகிறது. பெரும்பாலான சந்தர்ப்பங்களில், தோல்விக்கான காரணம் தரவு வடிவமைப்பு ஆகும்.

img

தோல்விக்கான காரணத்தைப் பார்க்க:

  1. செயல் தலைப்பின் கீழ் காண்க என்பதைக் கிளிக் செய்யவும்.
  2. அடுத்த பக்கத்தில், தோல்வி ஐகானில் சுட்டியை நகர்த்தவும். உதவிக்குறிப்பு பிழையைக் காட்டுகிறது.
  3. CVS கோப்பிற்குச் சென்று பிழையைச் சரிசெய்து சரிசெய்யவும்.
  4. கோப்பை மீண்டும் பதிவேற்றவும்.
