ஏற்கனவே உள்ள வழக்குகளுடன் பயனர் பணிகளை ஒதுக்குவது அல்லது அகற்றுவது எப்படி

கேஸ் பயனர்களை எவ்வாறு ஒதுக்குவது மற்றும் அகற்றுவது என்பதை நிர்வாகிகளுக்கு இந்தக் கட்டுரை காட்டுகிறது. பயனர் தரவை எவ்வாறு மாற்றுவது என்பதையும் நாங்கள் உங்களுக்குக் காட்டுகிறோம்.

ஏற்கனவே உள்ள வழக்கைத் திறக்கிறது

  1. நிர்வாகி கணக்கில் உள்நுழைக.
  2. ஆப்ஸ் பக்கத்தில் பயன்பாட்டைக் கண்டறிந்து, நிர்வாகி பொத்தானைக் கிளிக் செய்யவும்.
  3. வழக்குகள் மெனு தாவலைக் கிளிக் செய்யவும்.
  4. துணைமெனுவில் ACTIVE தேர்ந்தெடுக்கப்பட்டுள்ளதை உறுதிசெய்யவும்.
  5. ACTION நெடுவரிசைக்கு கீழே உருட்டவும்.

ஒதுக்கவும் அல்லது அகற்றவும்

வழக்கிற்கு ஒரு பயனரை ஒதுக்க பயனர்களை ஒதுக்கு என்பதைக் கிளிக் செய்யவும் அல்லது வழக்கிலிருந்து பயனரை அகற்றவும் (ஆனால் இயங்குதளம் அல்ல). அடுத்த பக்கத்தில்: சேர் என்பதைக் கிளிக் செய்து பணிப்பாய்வுகளை முடிக்கவும். பயனரை நீக்க, அகற்று என்பதைக் கிளிக் செய்யவும்.

மாற்றியமைக்கவும்

வழக்குகள் பக்கத்தில் பயனர் தரவைத் திருத்தவோ புதுப்பிக்கவோ முடியாது. அதற்கு பதிலாக, இந்த வழிமுறைகளைப் பின்பற்றவும்:

  1. பயனர்கள் மெனு தாவலைக் கிளிக் செய்யவும்.
  2. பயனர் பெயரைக் கண்டுபிடித்து, திருத்து என்பதைக் கிளிக் செய்யவும்.
  3. மாற்றங்களைச் செய்து சேமி என்பதைக் கிளிக் செய்யவும்.
