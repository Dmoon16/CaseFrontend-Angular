UI மூலம் பயனர்களை எவ்வாறு உருவாக்குவது மற்றும் மாற்றுவது

தளத்தின் பயனர் இடைமுகத்துடன் பயனர்களை எவ்வாறு உருவாக்குவது மற்றும் பதிவுகளை எவ்வாறு புதுப்பிப்பது என்பதை இந்தக் கட்டுரை நிர்வாகிகளுக்குக் காட்டுகிறது. நிர்வாகிகள் புதிய வழக்கில் பயனர்களைச் சேர்க்கலாம் மற்றும் ஏற்கனவே உள்ள வழக்குகளைத் திருத்தலாம்.

ஒரு பயனரை உருவாக்குதல்

  1. நிர்வாகி கணக்கில் உள்நுழைக.
  2. ஆப்ஸ் பக்கத்தில் பயன்பாட்டைக் கண்டறிந்து, நிர்வாகி பொத்தானைக் கிளிக் செய்யவும்.
  3. நிர்வாக குழுவில், பயனர்கள் மெனு தாவலைக் கிளிக் செய்யவும்.
  4. புதிய பொத்தானைக் கிளிக் செய்யவும். ஒரு புதிய சாளரம் தோன்றும்.

புலங்களை நிரப்பி, சேமி என்பதைக் கிளிக் செய்யவும். இந்த ஆப்ஸுடன் தொடர்புடைய பயனர்களின் பட்டியலில் கேஸ் ஆக்டிவ் பெயரைச் சேர்க்கிறது.

ஒரு பயனரை மாற்றுதல்

குறைந்த எண்ணிக்கையிலான பயனர் பதிவுகளைப் புதுப்பிக்க, திருத்து கருவியைப் பயன்படுத்தி ஒவ்வொரு பயனருக்கும் மாற்றங்களைச் செய்யுங்கள்.

  • குறிப்பு அதிக எண்ணிக்கையிலான பயனர் பதிவுகளை ஒரே அமர்வில் மாற்ற, இறக்குமதி கருவியைப் பயன்படுத்தவும்.
  1. நிர்வாக குழுவில், பயனர்கள் மெனு தாவலைக் கிளிக் செய்யவும்.
  2. ஒரு பயனரைக் கண்டுபிடித்து, திருத்து என்பதைக் கிளிக் செய்யவும்.
  3. மாற்றங்களைச் செய்து சேமி என்பதைக் கிளிக் செய்யவும்.

அடுத்த படிகள்

கேஸ் ஆக்டிவ் பயனருக்கு மின்னஞ்சல் அறிவிப்பை அனுப்பாது. நிர்வாகிகள் பயனரைத் தொடர்புகொண்டு, கேஸ் ஆக்டிவ் முகப்புப் பக்கத்திற்கு இணைப்பை அனுப்ப வேண்டும். பயனர் பின்னர் LOGIN பொத்தானைக் கிளிக் செய்து புதிய பதிவுகளுக்கான பணிப்பாய்வுகளை நிறைவு செய்கிறார்.

  • இந்த கட்டுரை புதிய பயனர்களுக்கான பதிவு பணிப்பாய்வுகளை விவரிக்கிறது.
