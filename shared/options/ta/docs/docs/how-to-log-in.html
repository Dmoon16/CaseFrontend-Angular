உள்நுழைவது எப்படி

உங்கள் தற்போதைய கேஸ் ஆக்டிவ் கணக்கில் உள்நுழைவது எப்படி என்பதை அறிக.

உள்நுழைதல்

  1. உள்நுழைவு பக்கத்திற்குச் செல்லவும்.
  2. LOGIN பொத்தானைக் கிளிக் செய்யவும்.
  3. மின்னஞ்சலை பதிவுசெய்.
  4. அடுத்து என்பதைக் கிளிக் செய்யவும்.

கணக்கைச் சரிபார்க்கிறது

  1. புதிய உலாவி தாவலைத் திறந்து உங்கள் மின்னஞ்சல் கணக்கில் உள்நுழையவும்.
  2. ஆறு இலக்க குறியீட்டை நகலெடுக்கவும்.
  3. கேஸ் ஆக்டிவ் தாவலுக்குத் திரும்பு.
  4. புலங்களில் குறியீட்டை ஒட்டவும். ஒரு புதிய சாளரம் தோன்றும்.

கேஸ் ஆக்டிவ் ஆப்ஸ் பக்கத்தைக் காட்டுகிறது.

  • குறிப்பு நிர்வாக சலுகைகள் உள்ள பயனர்களுக்கு APP பொத்தானுக்கு அடுத்ததாக ஒரு நிர்வாகி பட்டனை இயங்குதளம் காட்டுகிறது.
