இறக்குமதி கருவி மூலம் பயனர் பணிகளை ஒதுக்குவது, மாற்றுவது அல்லது அகற்றுவது எப்படி

இம்போர்ட் டூல் மூலம் கேஸ் பயனர்களை எப்படி நிர்வகிப்பது என்பதை நிர்வாகிகளுக்கு இந்தக் கட்டுரை காட்டுகிறது.

நிர்வாகிகள் செய்யலாம்:

  • ஒரு வழக்குக்கு பயனர்களை நியமிக்கவும்
  • ஒரு வழக்கில் ஏற்கனவே ஒதுக்கப்பட்ட பயனர்களை மாற்றவும்
  • ஒரு வழக்கில் ஒதுக்கப்பட்ட பயனர்களை நீக்கவும்

இறக்குமதி கருவியைத் திறக்கிறது

  1. நிர்வாகி கணக்கில் உள்நுழைக.
  2. ஆப்ஸ் பக்கத்தில் பயன்பாட்டைக் கண்டறிந்து, நிர்வாகி பொத்தானைக் கிளிக் செய்யவும்.
  3. ஒருங்கிணைப்பு தாவலைக் கிளிக் செய்து, துணைமெனுவில் இறக்குமதி செய்யவும்.
  4. புதிய பொத்தானைக் கிளிக் செய்யவும். இறக்குமதிகளைச் சேர் உரையாடல் பெட்டி காண்பிக்கப்படும்.

பதிவுகளை உருவாக்குதல்

ஒன்று அல்லது அதற்கு மேற்பட்ட CSV கோப்புகளைப் பதிவிறக்கவும். கேஸ் பயனர்களைச் சேர்க்க, மாற்ற அல்லது அகற்ற இந்தக் கோப்புகளைப் பயன்படுத்தவும். இறக்குமதிகளைச் சேர் உரையாடல் பெட்டி திறந்த பிறகு:

  1. வகை புலத்தில் கிளிக் செய்யவும்.
  2. கீழ்தோன்றும் மெனுவை உருட்டி ஒரு விருப்பத்தைத் தேர்ந்தெடுக்கவும் (கீழே காண்க). ஒரு இணைப்பு காட்டுகிறது.
  3. பதிவிறக்க மாதிரி இணைப்பைக் கிளிக் செய்யவும். ஒரு CSV கோப்பு உங்கள் சாதனத்தில் பதிவிறக்குகிறது.

கோப்பைத் திறந்து பதிவுகளைச் சேர்க்கவும். CSV கோப்பில் வடிவமைப்புத் தரவை எவ்வாறு பயன்படுத்துவது என்பதை அறிய இங்கே கிளிக் செய்யவும்.

கோப்பு விருப்பங்கள்

வழக்குகளுக்கு பயனர்களை ஒதுக்கவும் * ஒரு வழக்கில் பயனர்களைச் சேர்க்க இந்த விருப்பத்தைத் தேர்வு செய்யவும். * கேஸ்_ரோல்_ஆர்டர் கலத்தை காலியாக விடவும். * பயனர்கள் பக்கத்திற்குச் சென்று இந்த நபருக்கான டேக் ஐடியை நகலெடுக்கவும். டேக் ஐடி புலம் காலியாக இருந்தால், தனிப்பட்ட ஐடியை உள்ளிட்டு, சேமி என்பதைக் கிளிக் செய்து, ஐடியை CSV கோப்பில் நகலெடுக்கவும்.

வழக்குகளுக்கு ஒதுக்கப்பட்ட பயனர்களை மாற்றவும் * ஒரு வழக்கில் ஒதுக்கப்பட்ட தற்போதைய பயனர்களுக்கான தரவைப் புதுப்பிக்க இந்த விருப்பத்தைத் தேர்வு செய்யவும். * case_ID நெடுவரிசைக்கு கேஸ் டேக் ஐடியைப் பயன்படுத்தவும்.

வழக்குகளுக்கு ஒதுக்கப்பட்ட பயனர்களை அகற்று * ஒரு வழக்கில் ஒதுக்கப்பட்ட பயனர்களை அகற்ற இந்த விருப்பத்தைத் தேர்வு செய்யவும்.

கோப்பை பதிவேற்றுகிறது

நீங்கள் இப்போதே ஒரு கோப்பைப் பதிவேற்றலாம் அல்லது பிற்காலத்தில் இறக்குமதிகளைச் சேர் உரையாடல் பெட்டிக்குத் திரும்பி இந்தப் படிகளை முடிக்கலாம்.

  1. இறக்குமதிகளைச் சேர் உரையாடல் பெட்டியைத் திறக்கவும்.
  2. CSV கோப்பிற்கான தனிப்பட்ட பெயரை உள்ளிடவும்.
  3. உங்கள் சாதனத்திலிருந்து கோப்பைப் பதிவேற்ற, பிளஸ் (+) ஐகானைக் கிளிக் செய்யவும்.
  4. வகை புலத்தைக் கிளிக் செய்து ஒரு விருப்பத்தைத் தேர்ந்தெடுக்கவும். (மேலே பார்க்க)
  5. சேமி என்பதைக் கிளிக் செய்யவும்.

செயலாக்கம்

கோப்பு மற்றும் கணினி ஏற்றத்தில் உள்ள பதிவுகளின் எண்ணிக்கையைப் பொறுத்து, தளம் இறக்குமதியைச் செயல்படுத்த பல நிமிடங்கள் ஆகலாம்.

இறக்குமதி முடிந்ததும், வழக்குகள் பக்கத்தில் பயனர் பதிவுகளை (அதாவது சேர்க்கிறது, மாற்றுகிறது அல்லது நீக்குகிறது) இயங்குதளம் சரிசெய்கிறது.

தோல்வி

இறக்குமதி செயல்பாடு தோல்வியுற்றால், தளம் தோல்வியுற்ற செய்தியை வழங்குகிறது. பெரும்பாலான சந்தர்ப்பங்களில், தோல்விக்கான காரணம் தரவு வடிவமைப்பு ஆகும்.

தோல்விக்கான காரணத்தைப் பார்க்க:

  1. செயல் தலைப்பின் கீழ் காண்க என்பதைக் கிளிக் செய்யவும்.
  2. அடுத்த பக்கத்தில், தோல்வி ஐகானில் சுட்டியை நகர்த்தவும். உதவிக்குறிப்பு பிழையைக் காட்டுகிறது.
  3. CVS கோப்பிற்குச் சென்று பிழையைச் சரிசெய்து சரிசெய்யவும்.
  4. கோப்பை மீண்டும் பதிவேற்றவும்.
