உள்நுழைதல்

பயனர் முதல் முறையாக தங்கள் கேஸ்ஆக்டிவ் கணக்கில் உள்நுழையும்போது, இயங்குதளம் பயனரைத் தூண்டுகிறது:

  • அவர்களின் மின்னஞ்சல் கணக்கைத் திறந்து ஆறு இலக்க பாதுகாப்புக் குறியீட்டை நகலெடுக்கவும்/ஒட்டவும்
  • கணக்கு சுயவிவரத்தை முடிக்கவும்

img

பயனர் புதுப்பிப்பைக் கிளிக் செய்த பிறகு, ஆப்ஸ் பக்கம் காண்பிக்கப்படும். பக்கம் பயன்பாட்டின் பெயர் (நிறுவனத்தின் பெயர்), பயனர் வகை (பங்கு), நிலை மற்றும் அழைப்பு பொத்தான் ஆகியவற்றைக் காட்டுகிறது.

பயன்பாட்டை அணுக, பயனர் அழைப்பை ஏற்க வேண்டும். பயனர் கண்டிப்பாக:

  1. அழைப்பிதழ் பொத்தானைக் கிளிக் செய்யவும்.
  2. அனுமதிகளை மதிப்பாய்வு செய்யவும்.
  3. விதிமுறைகளை மதிப்பாய்வு செய்து திருத்த (விரும்பினால்) அறிவிப்புகள் தாவலைக் கிளிக் செய்யவும்.
  4. அணுகலை அனுமதி என்பதைக் கிளிக் செய்யவும்.

பயனர் அங்கீகரிக்கும் போது, இயங்குதளம் பயனர் பக்கத்தில் APP பட்டனைக் காண்பிக்கும். இந்த பொத்தான் பயனர்களுக்கு ஒதுக்கப்பட்ட வழக்குகளுக்கான அணுகலை வழங்குகிறது.

பயனர் அமைப்புகளைத் திருத்த, பயனர் நீள்வட்டத்தைக் கிளிக் செய்கிறார்.
