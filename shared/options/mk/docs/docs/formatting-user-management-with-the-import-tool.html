Форматирање на податоци за управување со корисници со алатката за увоз

Подолу е листа на полиња со податоци во CSV-датотеките. Секоја опција за увоз има различна комбинација на полиња.

корисничко име

  • Внесете ја адресата на е-пошта на корисникот.

дадено име

  • Внесете го името на корисникот.

фамилијарно име

  • Внесете го презимето на корисникот.

локација

  • Оваа колона го проверува претпочитаниот јазик на корисникот.
  • Внесете en за англиски, sp за шпански.

zoneinfo

  • Ова ја поставува временската зона на корисникот. Внесете го континентот и градот без празни места (користете долна црта).
  • Мора да користи опција во паѓачкото мени Профил > ЈАЗИК и РЕГИОН > Временска зона.
  • Користете ја најдобро одговара временската зона. На пример, Бостон не е на паѓачкото мени, па можеме да го користиме Њујорк (или кој било град во временската зона EST/EDT).

host_tag_id

  • Внесете постојана етикета.
  • az, 0-9, -, / се дозволени.

случај_статус

  • 0 = затворено
  • 1 = активен
