Како администраторите управуваат со корисниците

Администраторите создаваат, менуваат или оневозможуваат кој може да пристапува до апликациите и случаите. Кога корисникот ќе се логира на платформата за прв пат, треба да го заврши работниот тек на профилот.

  • Кликнете овде за да дознаете повеќе за работниот тек на новиот кориснички профил.

Управување со корисник

Табелата подолу покажува пет начини за управување со корисник. Кликнете на врската за да дознаете како.

Метод Тип Акции Детали
Користење на интерфејс UI додадете и измени Администраторско копче > Корисници
Додека креирате случај и доделете корисник на случај UI додадете Администраторско копче > Случаи > Ново копче > Креирај страница на случај
Автоматски креирајте корисник користејќи јавна врска за регистрација Корисничка акција додадете Администраторско копче > Поставки > Страница за поставување систем
Алатка за увоз UI додадете и измени Администраторско копче > Интеграции > Увоз
Користење на API (наскоро) API додадете и измени N/A

Известувања

Платформата не им испраќа на корисниците известување кога ќе се додадат на некоја апликација. Сепак, администраторите можат да овозможат известувања за задачи на случаи.
