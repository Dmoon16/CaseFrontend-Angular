Formatiranje podataka za upravljanje predmetima pomoću alata za uvoz

Ispod je lista potrebnih polja u CSV datotekama za upravljanje slučajevima pomoću alata za uvoz.

korisničko ime

  • Unesite adresu e-pošte korisnika.

tag_id (id koji možete referencirati kasnije)

  • jedinstveni identifikator (koristite alfanumeričke znakove, crtice su prihvaćene)

case_status

  • 0 = zatvoreno
  • 1 = aktivno

date_opened

  • mjesec/dan/godina
  • mm/dd/gggg

o

  • jedinstven opis slučaja (koristite alfanumeričke znakove, crtice su prihvaćene)
