Formatiranje podataka o dodjeli slučajeva korisnika pomoću alata za uvoz

Ispod je lista polja podataka u CSV datotekama. Svaka opcija uvoza ima različitu kombinaciju polja.

case_ID

  • Unesite ID korisnika ili oznake za slučaj.
  • Idite na stranicu Predmeti > Uredi > ID oznake ILI Idite na stranicu Predmeti > Uredi > ID slučaja u URL-u.

Korisnički broj

  • Unesite ID korisnika ili ID oznake.
  • Idite na stranicu korisnika > Uredi > ID oznake ILI Idite na stranicu korisnika > Uredi > ID korisnika u URL-u.

case_role_order

  • Ostavite ćeliju praznom.

case_role_ID

  • Unesite: klijenti, osoblje ili zadano.

case_notify

  • 0 = ne
  • 1 = Da, pošaljite korisniku obavještenje putem e-pošte.
