Kako se prijaviti

Naučite kako se prijaviti na svoj postojeći Case Active račun.

Prijava

  1. Idite na stranicu za prijavu .
  2. Kliknite na dugme LOGIN.
  3. Unesite svoju e-poštu.
  4. Kliknite DALJE.

Provjera računa

  1. Otvorite novu karticu pretraživača i prijavite se na svoj nalog e-pošte.
  2. Kopirajte šestocifreni kod.
  3. Vratite se na karticu Case Active.
  4. Zalijepite kod u polja. Prikazuje se novi prozor.

Case Active prikazuje stranicu aplikacija.

  • Napomena Platforma prikazuje dugme ADMIN pored dugmeta APP za korisnike sa administratorskim privilegijama.
