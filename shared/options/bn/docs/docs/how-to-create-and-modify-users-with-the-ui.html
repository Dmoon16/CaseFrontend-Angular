কিভাবে UI এর মাধ্যমে ব্যবহারকারীদের তৈরি এবং পরিবর্তন করবেন

এই নিবন্ধটি প্রশাসকদের দেখায় কিভাবে ব্যবহারকারী তৈরি করতে হয় এবং প্ল্যাটফর্মের ব্যবহারকারী ইন্টারফেসের সাথে রেকর্ড আপডেট করতে হয়। প্রশাসকরা একটি নতুন ক্ষেত্রে ব্যবহারকারীদের যোগ করতে এবং বিদ্যমান ক্ষেত্রে সম্পাদনা করতে পারেন।

একজন ব্যবহারকারী তৈরি করা হচ্ছে

  1. অ্যাডমিন অ্যাকাউন্টে লগ ইন করুন।
  2. অ্যাপস পৃষ্ঠায় একটি অ্যাপ খুঁজুন এবং অ্যাডমিন বোতামে ক্লিক করুন।
  3. অ্যাডমিন প্যানেলে, ব্যবহারকারী মেনু ট্যাবে ক্লিক করুন।
  4. নতুন বোতামে ক্লিক করুন। একটি নতুন উইন্ডো প্রদর্শিত হবে।

ক্ষেত্রগুলি পূরণ করুন এবং সংরক্ষণ করুন ক্লিক করুন। কেস অ্যাক্টিভ এই অ্যাপের সাথে যুক্ত ব্যবহারকারীদের তালিকায় নাম যোগ করে।

img

একটি ব্যবহারকারী পরিবর্তন

অল্প সংখ্যক ব্যবহারকারীর রেকর্ড আপডেট করতে, সম্পাদনা টুল ব্যবহার করুন এবং প্রতিটি ব্যবহারকারীর পরিবর্তন করুন।

  • দ্রষ্টব্য এক বৈঠকে বিপুল সংখ্যক ব্যবহারকারীর রেকর্ড পরিবর্তন করতে, আমদানি টুল ব্যবহার করার কথা বিবেচনা করুন।
  1. অ্যাডমিন প্যানেলে, ব্যবহারকারী মেনু ট্যাবে ক্লিক করুন।
  2. একটি ব্যবহারকারী সনাক্ত করুন এবং সম্পাদনা ক্লিক করুন.
  3. পরিবর্তন করুন এবং সংরক্ষণ করুন ক্লিক করুন.

পরবর্তী পদক্ষেপ

কেস অ্যাক্টিভ ব্যবহারকারীকে একটি ইমেল বিজ্ঞপ্তি পাঠায় না। অ্যাডমিনদের উচিত ব্যবহারকারীর সাথে যোগাযোগ করা এবং তাদের কেস অ্যাক্টিভ হোম পেজে একটি লিঙ্ক পাঠানো। ব্যবহারকারী তারপর লগইন বোতামে ক্লিক করে এবং নতুন সাইনআপের জন্য ওয়ার্কফ্লো সম্পূর্ণ করে।

  • এই নিবন্ধটি নতুন ব্যবহারকারীদের জন্য সাইনআপ ওয়ার্কফ্লো বর্ণনা করে।
