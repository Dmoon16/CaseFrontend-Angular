কিভাবে ইম্পোর্ট টুল দিয়ে ব্যবহারকারীদের তৈরি বা পরিবর্তন করবেন

এই নিবন্ধটি প্রশাসকদের দেখায় কিভাবে আমদানি টুল দিয়ে ব্যবহারকারী তৈরি করতে হয়।

আমদানি টুল খোলা হচ্ছে

প্রশাসকরা নতুন ব্যবহারকারীদের বাল্ক আপলোড করতে এবং বিদ্যমান ব্যবহারকারীর রেকর্ড সংশোধন করতে আমদানি সরঞ্জাম ব্যবহার করতে পারেন।

  1. অ্যাডমিন অ্যাকাউন্টে লগ ইন করুন।
  2. অ্যাপস পৃষ্ঠায় একটি অ্যাপ খুঁজুন এবং অ্যাডমিন বোতামে ক্লিক করুন।
  3. অ্যাডমিন প্যানেলে, ইন্টিগ্রেশন ট্যাবে ক্লিক করুন এবং তারপর সাবমেনুতে আমদানি করুন।
  4. নতুন বোতামে ক্লিক করুন। একটি নতুন ব্রাউজার ট্যাবে আমদানি যোগ করুন ডায়ালগ বক্স প্রদর্শিত হয়৷

img

রেকর্ড তৈরি করা

আমদানি যোগ করুন ডায়ালগ বক্স খোলার পরে:

  1. টাইপ ক্ষেত্রে ক্লিক করুন এবং একটি বিকল্প নির্বাচন করুন (নীচে দেখুন)। একটি লিঙ্ক প্রদর্শন করে।
  2. ডাউনলোড নমুনা লিঙ্কে ক্লিক করুন. একটি CSV ফাইল আপনার ডিভাইসে ডাউনলোড হয়।

এরপরে, ফাইলটি খুলুন এবং রেকর্ড যোগ করুন। CSV ফাইলে ফর্ম্যাট ডেটা কীভাবে ব্যবহার করবেন তা শিখতে এখানে ক্লিক করুন।

img

টাইপ

আপনি টাইপ ক্ষেত্রে ক্লিক করলে, একটি ড্রপ-ডাউন মেনু প্রদর্শিত হবে। বাল্ক আমদানির জন্য দুটি বিকল্পের মধ্যে একটি বেছে নিন।

: বিস্ময়কর: দ্রষ্টব্য ড্রপ-ডাউন মেনুতে অনেকগুলি বিকল্প রয়েছে, কিন্তু শুধুমাত্র দুটি আমদানি সরঞ্জামের সাথে সম্পর্কিত। প্রতিটি বিকল্পের একটি আলাদা নমুনা CSV ফাইল রয়েছে৷

ব্যবহারকারীদের যোগ করুন

  • ব্যবহারকারীর রেকর্ডের একটি সাধারণ বাল্ক আপলোডের জন্য এই বিকল্পটি বেছে নিন।
  • ব্যবহারকারী যোগ করুন আমদানি প্রকারের জন্য কলাম শিরোনাম সহ একটি CSV ফাইল খুলতে ডাউনলোড নমুনা লিঙ্কে ক্লিক করুন৷

ব্যবহারকারীদের পরিবর্তন করুন

  • বর্তমান ব্যবহারকারীদের জন্য ডেটা আপডেট করতে এই বিকল্পটি বেছে নিন।
  • পরিবর্তন ব্যবহারকারীদের আমদানি প্রকারের জন্য কলাম শিরোনাম সহ একটি CSV ফাইল খুলতে ডাউনলোড নমুনা লিঙ্কে ক্লিক করুন৷

ফাইল আপলোড করা হচ্ছে

আপনি এখনই একটি ফাইল আপলোড করতে পারেন বা পরবর্তী তারিখে আমদানি যোগ করুন ডায়ালগ বক্সে ফিরে আসতে পারেন এবং এই পদক্ষেপগুলি সম্পূর্ণ করুন৷

  1. আমদানি যোগ করুন ডায়ালগ বক্স খুলুন।
  2. CSV ফাইলের জন্য একটি অনন্য নাম লিখুন।
  3. আপনার ডিভাইস থেকে ফাইল আপলোড করতে প্লাস (+) আইকনে ক্লিক করুন।
  4. টাইপ ক্ষেত্রে ক্লিক করুন এবং একটি বিকল্প নির্বাচন করুন। (উপরে দেখুন)
  5. SAVE এ ক্লিক করুন।

প্রক্রিয়াকরণ

ফাইল এবং সিস্টেম লোডের রেকর্ডের সংখ্যার উপর নির্ভর করে, প্ল্যাটফর্মটি আমদানি প্রক্রিয়া করতে কয়েক মিনিট সময় নিতে পারে।

img

যখন আমদানি সম্পূর্ণ হয়, তখন প্ল্যাটফর্ম অ্যাডমিনের ব্যবহারকারী ড্যাশবোর্ডে ব্যবহারকারীর রেকর্ডগুলিকে সামঞ্জস্য করে (যেমন যোগ করে, সংশোধন করে বা অক্ষম করে)।

ব্যর্থতা

যদি আমদানি ক্রিয়াকলাপ ব্যর্থ হয়, প্ল্যাটফর্মটি একটি ব্যর্থ বার্তা প্রদান করে। বেশিরভাগ ক্ষেত্রে, ব্যর্থতার কারণ ডেটা ফরম্যাটিং।

img

ব্যর্থতার কারণ দেখতে:

  1. অ্যাকশন শিরোনামের অধীনে দেখুন ক্লিক করুন।
  2. পরের পৃষ্ঠায়, ব্যর্থ আইকনে মাউস নিয়ে যান। টুলটিপ ত্রুটি প্রদর্শন করে।
  3. CVS ফাইলে যান এবং সামঞ্জস্য করুন এবং ত্রুটিটি ঠিক করুন।
  4. ফাইলটি আবার আপলোড করুন।
