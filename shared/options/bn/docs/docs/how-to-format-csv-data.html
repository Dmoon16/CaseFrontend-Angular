  • ইংরেজির জন্য en, স্প্যানিশের জন্য sp লিখুন।

অঞ্চলের তথ্য

  • এটি ব্যবহারকারীর সময় অঞ্চল সেট করে। মহাদেশ এবং শহরটি শূন্যস্থান ছাড়াই প্রবেশ করুন (একটি আন্ডারস্কোর ব্যবহার করুন)।
  • প্রোফাইল > ভাষা এবং অঞ্চল > টাইমজোন ড্রপ-ডাউন মেনুতে একটি বিকল্প ব্যবহার করতে হবে।
  • সবচেয়ে উপযুক্ত সময় অঞ্চল ব্যবহার করুন. উদাহরণস্বরূপ, বোস্টন ড্রপ-ডাউনে নেই, তাই আমরা নিউ ইয়র্ক ব্যবহার করতে পারি (বা EST/EDT সময় অঞ্চলের যেকোনো শহর)।
