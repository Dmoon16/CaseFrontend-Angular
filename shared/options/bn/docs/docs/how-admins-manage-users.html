অ্যাডমিনরা কীভাবে ব্যবহারকারীদের পরিচালনা করে

অ্যাডমিনরা তৈরি, সংশোধন বা অক্ষম করে যারা অ্যাপ এবং কেস অ্যাক্সেস করতে পারে। যখন একজন ব্যবহারকারী প্রথমবার প্ল্যাটফর্মে লগ ইন করেন, তখন তাদের একটি প্রোফাইল ওয়ার্কফ্লো সম্পূর্ণ করতে হবে।

  • নতুন ব্যবহারকারী প্রোফাইল ওয়ার্কফ্লো সম্পর্কে আরও জানতে এখানে ক্লিক করুন।

একজন ব্যবহারকারী পরিচালনা করা

নীচের টেবিলটি একজন ব্যবহারকারীকে পরিচালনা করার পাঁচটি উপায় দেখায়। কিভাবে শিখতে একটি লিঙ্ক ক্লিক করুন.

পদ্ধতি টাইপ কর্ম বিস্তারিত
UI ব্যবহার করে UI যোগ করুন এবং সংশোধন করুন অ্যাডমিন বোতাম > ব্যবহারকারী
একটি কেস তৈরি করার সময় এবং কেসটিতে ব্যবহারকারীকে বরাদ্দ করুন UI যোগ করুন অ্যাডমিন বোতাম > কেস > নতুন বোতাম > কেস পৃষ্ঠা তৈরি করুন
পাবলিক সাইনআপ লিঙ্ক ব্যবহার করে স্বয়ংক্রিয়ভাবে ব্যবহারকারী তৈরি করুন ব্যবহারকারীর কর্ম যোগ করুন অ্যাডমিন বোতাম > সেটিংস > সিস্টেম সেটআপ পৃষ্ঠা
ইম্পোর্ট টুল UI যোগ করুন এবং সংশোধন করুন অ্যাডমিন বোতাম > ইন্টিগ্রেশন > আমদানি

বিজ্ঞপ্তি

প্ল্যাটফর্ম ব্যবহারকারীদের কোনো অ্যাপে যোগ করার সময় কোনো বিজ্ঞপ্তি পাঠায় না। যাইহোক, অ্যাডমিনরা কেস অ্যাসাইনমেন্টের জন্য বিজ্ঞপ্তিগুলি সক্ষম করতে পারেন।
