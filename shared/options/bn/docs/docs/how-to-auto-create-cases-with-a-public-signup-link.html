একটি সর্বজনীন সাইনআপ লিঙ্কের মাধ্যমে কীভাবে স্বয়ংক্রিয়ভাবে কেস তৈরি করবেন

এই নিবন্ধটি প্রশাসকদের দেখায় কিভাবে একটি একক সাইনআপ লিঙ্ক পাঠিয়ে একটি কেস তৈরি করতে হয় (এবং একই সময়ে একজন ব্যবহারকারীকে যোগ করতে হয়)।

লিঙ্কটি সনাক্ত করা হচ্ছে

  1. অ্যাডমিন অ্যাকাউন্টে লগ ইন করুন।
  2. অ্যাপস পৃষ্ঠায় একটি অ্যাপ খুঁজুন এবং অ্যাডমিন বোতামে ক্লিক করুন।
  3. অ্যাডমিন প্যানেলে, সেটিংস মেনু ট্যাবে ক্লিক করুন।
  4. নিশ্চিত করুন যে SYSTEM SETUP সাবমেনু ট্যাবটি নির্বাচন করা হয়েছে।

সেটিংস কনফিগার করা হচ্ছে

ব্যবহারকারী সাইনআপ শিরোনামে নিচে স্ক্রোল করুন।

নিশ্চিত করুন যে সাইনআপের অনুমতি দিন চালু আছে। (প্রয়োজনীয়) যখন এই বৈশিষ্ট্যটি সক্ষম করা হয়, কেস অ্যাক্টিভ একটি অনন্য লিঙ্ক তৈরি করে।

  1. অটো ক্রিয়েট কেস ফিল্ডে ক্লিক করুন এবং চালু নির্বাচন করুন। কেস ব্যবহারকারীর ভূমিকা ক্ষেত্র প্রদর্শন করে।
  2. একটি ভূমিকা নির্বাচন করুন. (নিচে দেখ)

  • ডিফল্ট
  • কর্মী

লিঙ্ক পাঠাচ্ছি

লিঙ্কটি এক বা একাধিক ব্যক্তির সাথে শেয়ার করুন। প্রতিটি ব্যবহারকারী যে লিঙ্কটিতে ক্লিক করে এবং সাইনআপ সম্পূর্ণ করে তারা কেস পৃষ্ঠায় একটি নতুন কেস তৈরি করতে প্ল্যাটফর্মটিকে ট্রিগার করে এবং কেসটিতে ব্যবহারকারীকে স্বয়ংক্রিয়ভাবে বরাদ্দ করে।
