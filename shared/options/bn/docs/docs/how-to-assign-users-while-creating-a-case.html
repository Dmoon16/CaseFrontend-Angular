একটি কেস তৈরি করার সময় ব্যবহারকারীদের কীভাবে বরাদ্দ করবেন

এই নিবন্ধটি অ্যাডমিনদের দেখায় কিভাবে একটি নতুন কেস তৈরি করার সময় একজন ব্যবহারকারীকে বরাদ্দ করতে হয়। যদি ব্যবহারকারী ইতিমধ্যে প্ল্যাটফর্মে নিবন্ধিত না থাকে, আপনি একটি নতুন কেস তৈরি করার সময় একটি নতুন ব্যবহারকারী যোগ করতে পারেন।

মামলা তৈরি করছে

  1. অ্যাডমিন অ্যাকাউন্টে লগ ইন করুন।
  2. অ্যাপস পৃষ্ঠায় একটি অ্যাপ খুঁজুন এবং অ্যাডমিন বোতামে ক্লিক করুন।
  3. কেস মেনু ট্যাবে ক্লিক করুন।
  4. নতুন বোতামে ক্লিক করুন। একটি নতুন উইন্ডো প্রদর্শিত হবে।
  • একটি কেস তৈরি করতে ক্ষেত্রগুলি পূরণ করুন এবং তারপরে নির্ধারিত ব্যবহারকারী শিরোনামে নীচে স্ক্রোল করুন৷

একজন ব্যবহারকারীকে বরাদ্দ করা

বর্তমান ব্যবহারকারী

নতুন ক্ষেত্রে বর্তমান ব্যবহারকারীকে বরাদ্দ করতে এই পদক্ষেপগুলি অনুসরণ করুন৷

ব্যবহারকারীদের নিয়োগ বোতামে ক্লিক করুন এবং ক্ষেত্রগুলি পূরণ করুন৷

  1. ব্যবহারকারীর ড্রপ-ডাউনে ক্লিক করুন এবং একটি শনাক্তকারী নির্বাচন করুন (যেমন ইমেল)।
  2. ফাঁকা ক্ষেত্রে একটি মান লিখুন (যেমন ইমেল ঠিকানা)।
  3. প্ল্যাটফর্মে ব্যবহারকারীর রেকর্ড বিদ্যমান থাকলে, ক্ষেত্রটি স্বয়ংক্রিয়ভাবে একটি নাম প্রস্তাব করে।
  4. সঠিক ব্যবহারকারী নির্বাচন করুন.
  5. একটি ব্যবহারকারীর ভূমিকা নির্বাচন করুন (ক্লায়েন্ট, ডিফল্ট, বা স্টাফ)।
  6. ঐচ্ছিকভাবে, ব্যবহারকারীকে একটি ইমেল বিজ্ঞপ্তি পাঠাতে বাক্সটি চেক করুন৷
  7. SAVE এ ক্লিক করুন।

কেস অ্যাক্টিভ ব্যবহারকারীকে কেসে যুক্ত করে। এখন প্রশাসকরা উইন্ডোটি বন্ধ করতে সংরক্ষণ করুন ক্লিক করতে পারেন, বা একজন নতুন ব্যবহারকারী যোগ করতে পারেন, বা একজন ব্যবহারকারীকে সরাতে পারেন।

একটি নতুন ব্যবহারকারী যোগ করা হচ্ছে

যদি প্ল্যাটফর্ম একটি বর্তমান রেকর্ড খুঁজে না পায়, একটি নতুন ব্যবহারকারী যোগ করতে বোতামে ক্লিক করুন.

ক্ষেত্রগুলি পূরণ করুন এবং তারপরে প্ল্যাটফর্মে ব্যবহারকারীকে যুক্ত করতে সংরক্ষণ করুন ক্লিক করুন৷ এরপরে, একটি ব্যবহারকারীর ভূমিকা সেট করুন এবং ব্যবহারকারীকে মামলায় বরাদ্দ করতে আবার সংরক্ষণ করুন ক্লিক করুন। অবশেষে, কেসটি সংরক্ষণ করতে আরও একবার সংরক্ষণ করুন ক্লিক করুন।
