কিভাবে একটি ব্যবহারকারীর প্রোফাইল আপডেট করবেন

এই নিবন্ধটি ব্যবহারকারীদের তাদের প্রোফাইল পৃষ্ঠায় তথ্য আপডেট করতে দেখায়।

  • ব্যবহারকারীরা তাদের নিজস্ব প্রোফাইল আপডেট করতে পারেন।
  • অ্যাডমিনরা যেকোনো ব্যবহারকারীর প্রোফাইল পৃষ্ঠা আপডেট করতে পারেন।

ব্যবহারকারীদের জন্য

প্রথম ধাপ হল প্রোফাইল পেজ খুলতে হবে। ব্যবহারকারী এবং প্রশাসকরা তাদের নিজস্ব প্রোফাইল পৃষ্ঠা আপডেট করতে এই পদক্ষেপগুলি অনুসরণ করুন৷

  1. আপনার কেস অ্যাক্টিভ অ্যাকাউন্টে লগ ইন করুন।
  2. পৃষ্ঠার শীর্ষে প্রোফাইল ট্যাবে ক্লিক করুন।
  3. পরিবর্তন করা.
  4. Save এ ক্লিক করুন।

অধিকাংশ ক্ষেত্র ঐচ্ছিক, এবং কিছু ক্ষেত্র আবশ্যক। প্রয়োজনীয় ক্ষেত্রগুলি হল:

  • নামের প্রথম অংশ
  • নামের শেষাংশ
  • ইমেইল ঠিকানা
  • ভাষা
  • সময় অঞ্চল

অ্যাডমিনদের জন্য

প্রশাসকরা অন্যান্য ব্যবহারকারীদের জন্য প্রোফাইল তথ্য আপডেট করতে পারেন। এটি হোস্ট ব্যবহারকারী তথ্য শিরোনামের অধীনে তিনটি ক্ষেত্র অন্তর্ভুক্ত করে।

  1. আপনার কেস অ্যাক্টিভ অ্যাডমিন অ্যাকাউন্টে লগ ইন করুন।
  2. একটি অ্যাপ খুঁজুন এবং অ্যাডমিন বোতামে ক্লিক করুন।
  3. উপরের মেনুতে ব্যবহারকারী ক্লিক করুন।
  4. সাব-মেনুতে যান এবং আপনি অ্যাক্সেস করতে চান এমন ব্যবহারকারী তালিকা নির্বাচন করুন (সক্রিয় বা অক্ষম)। ডিফল্ট সেটিং সক্রিয়।
  5. তালিকায় একজন ব্যবহারকারীকে সনাক্ত করুন এবং সম্পাদনা ক্লিক করুন।
  6. প্রয়োজন অনুযায়ী পরিবর্তন করুন। (নিচে দেখ)
  7. Save এ ক্লিক করুন।

ব্যবহারকারীর তথ্য পরিবর্তন করুন

অ্যাডমিনরা ব্যবহারকারীর তথ্য পরিবর্তন করতে পারেন।

  1. ব্যবহারকারীর ধরন (ব্যবহারকারী বা ম্যানেজার) সেট করুন।
  2. ব্যবহারকারীর স্থিতি সেট করুন (অক্ষম থেকে সক্রিয়)।
  3. একটি অনন্য ট্যাগ আইডি লিখুন। (ঐচ্ছিক)
  4. Save এ ক্লিক করুন।
