প্রশাসকরা কিভাবে কেসে ইউজার অ্যাসাইনমেন্ট পরিচালনা করে

ব্যবহারকারী এবং কেস অ্যাসাইনমেন্ট পরিচালনা করার পাঁচটি উপায় রয়েছে। কিভাবে শিখতে একটি লিঙ্ক ক্লিক করুন.

পদ্ধতি টাইপ কর্ম বিস্তারিত
মামলা করার সময় UI যোগ করুন অ্যাডমিন বোতাম > কেস > নতুন বোতাম > অ্যাসাইন ইউজার বোতাম
পাবলিক সাইনআপ লিঙ্ক ব্যবহার করে স্বয়ংক্রিয়ভাবে ব্যবহারকারীকে বরাদ্দ করুন ব্যবহারকারীর কর্ম যোগ করুন অ্যাডমিন বোতাম > সেটিংস > সিস্টেম সেটআপ পৃষ্ঠা
বিদ্যমান ক্ষেত্রে UI ব্যবহার করা UI যোগ করুন এবং অপসারণ করুন অ্যাডমিন বোতাম > কেস > ব্যবহারকারীদের বরাদ্দ করুন
ইম্পোর্ট টুল UI যোগ করুন, সংশোধন করুন এবং অপসারণ করুন অ্যাডমিন বোতাম > ইন্টিগ্রেশন > আমদানি
