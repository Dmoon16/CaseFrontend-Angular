UI bilen ulanyjylary nädip döretmeli we üýtgetmeli

Bu makala, ulanyjylary nädip döretmelidigini we platformanyň ulanyjy interfeýsi bilen ýazgylary täzelemegiň usullaryny görkezýär. Adminler ulanyjylary täze işe goşup we bar bolan ýagdaýlary redaktirläp bilerler.

Ulanyjy döretmek

  1. Admin hasabyna giriň.
  2. Programmalar sahypasynda bir programma tapyň we ADMIN düwmesine basyň.
  3. Dolandyryş panelinde Ulanyjylar menýusyna basyň.
  4. Täze düwmä basyň. Täze penjire görkezilýär.

Meýdanlary dolduryň we Saklamak düwmesine basyň. “Case Active” bu programma bilen baglanyşykly ulanyjylaryň sanawyna ady goşýar.

Ulanyjyny üýtgetmek

Ulanyjy ýazgylarynyň az mukdaryny täzelemek üçin “Edit” guralyny ulanyň we her aýratyn ulanyja üýtgeşmeler giriziň.

  • Bellik Bir oturgyçda köp sanly ulanyjy ýazgysyny üýtgetmek üçin, import guralyny ulanmagy göz öňünde tutuň.
  1. Dolandyryş panelinde Ulanyjylar menýusyna basyň.
  2. Ulanyjyny tapyň we Üýtget düwmesine basyň.
  3. Üýtgeşmeler giriziň we Saklamak düwmesine basyň.
