आयात साधनासह वापरकर्ते कसे तयार किंवा सुधारित करावे

हा लेख आयात साधनासह वापरकर्ते कसे तयार करावे हे प्रशासकांना दाखवतो.

आयात साधन उघडत आहे

नवीन वापरकर्ते मोठ्या प्रमाणात अपलोड करण्यासाठी आणि विद्यमान वापरकर्ता रेकॉर्ड सुधारित करण्यासाठी प्रशासक आयात साधन वापरू शकतात.

  1. प्रशासक खात्यात लॉग इन करा.
  2. अॅप्स पृष्ठावर अॅप शोधा आणि प्रशासक बटणावर क्लिक करा.
  3. प्रशासन पॅनेलवर, एकत्रीकरण टॅबवर क्लिक करा आणि नंतर सबमेनूवर आयात करा.
  4. नवीन बटणावर क्लिक करा. जोडा आयात संवाद बॉक्स नवीन ब्राउझर टॅबमध्ये प्रदर्शित होतो.

img

रेकॉर्ड तयार करणे

Add Imports डायलॉग बॉक्स उघडल्यानंतर:

  1. Type फील्डवर क्लिक करा आणि एक पर्याय निवडा (खाली पहा). एक लिंक दिसते.
  2. डाउनलोड नमुना लिंक वर क्लिक करा. तुमच्या डिव्हाइसवर CSV फाइल डाउनलोड होते.

पुढे, फाइल उघडा आणि रेकॉर्ड जोडा. CSV फाइलमध्ये फॉरमॅट डेटा कसा वापरायचा हे जाणून घेण्यासाठी येथे क्लिक करा .

img

प्रकार

जेव्हा तुम्ही टाइप फील्डवर क्लिक करता, तेव्हा एक ड्रॉप-डाउन मेनू प्रदर्शित होतो. मोठ्या प्रमाणात आयात करण्यासाठी दोन पर्यायांपैकी एक निवडा.

: उद्गार: टीप ड्रॉप-डाउन मेनूमध्ये अनेक पर्याय आहेत, परंतु फक्त दोन आयात साधनाशी संबंधित आहेत. प्रत्येक पर्यायामध्ये वेगळी नमुना CSV फाइल असते.

वापरकर्ते जोडा

  • वापरकर्ता रेकॉर्डच्या साध्या मोठ्या प्रमाणात अपलोड करण्यासाठी हा पर्याय निवडा.
  • वापरकर्ते जोडा आयात प्रकारासाठी स्तंभ शीर्षकांसह CSV फाइल उघडण्यासाठी नमुना डाउनलोड करा दुव्यावर क्लिक करा.

वापरकर्ते सुधारित करा

  • सध्याच्या वापरकर्त्यांसाठी डेटा अपडेट करण्यासाठी हा पर्याय निवडा.
  • वापरकर्ते आयात प्रकार सुधारित करण्यासाठी स्तंभ शीर्षकांसह CSV फाइल उघडण्यासाठी नमुना डाउनलोड करा दुव्यावर क्लिक करा.

फाइल अपलोड करत आहे

तुम्ही लगेच फाइल अपलोड करू शकता किंवा नंतरच्या तारखेला जोडा आयात डायलॉग बॉक्समध्ये परत येऊ शकता आणि या पायऱ्या पूर्ण करू शकता.

  1. जोडा आयात संवाद बॉक्स उघडा.
  2. CSV फाइलसाठी एक अद्वितीय नाव प्रविष्ट करा.
  3. तुमच्या डिव्हाइसवरून फाइल अपलोड करण्यासाठी प्लस (+) चिन्हावर क्लिक करा.
  4. Type फील्डवर क्लिक करा आणि पर्याय निवडा. (वर पहा)
  5. SAVE वर क्लिक करा.

प्रक्रिया करत आहे

फाइल आणि सिस्टम लोडमधील रेकॉर्डच्या संख्येवर अवलंबून, प्लॅटफॉर्मला आयात प्रक्रिया करण्यासाठी काही मिनिटे लागू शकतात.

img

जेव्हा आयात पूर्ण होते, तेव्हा प्लॅटफॉर्म प्रशासकाच्या वापरकर्त्यांच्या डॅशबोर्डवर वापरकर्ता रेकॉर्ड (म्हणजे जोडते, सुधारित करते किंवा अक्षम करते) समायोजित करते.

अपयश

आयात ऑपरेशन अयशस्वी झाल्यास, प्लॅटफॉर्म एक अयशस्वी संदेश परत करतो. बर्याच बाबतीत, अपयशाचे कारण डेटा स्वरूपन आहे.

अपयशाचे कारण पाहण्यासाठी:

  1. क्रिया शीर्षकाखाली पहा वर क्लिक करा.
  2. पुढील पृष्ठावर, अयशस्वी चिन्हावर माउस फिरवा. टूलटिप त्रुटी दाखवते.
  3. CVS फाईलवर जा आणि समायोजित करा आणि त्रुटी दुरुस्त करा.
  4. फाइल पुन्हा अपलोड करा.
