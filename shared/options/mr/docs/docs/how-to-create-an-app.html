अॅप कसे तयार करावे

हा लेख नवीन वापरकर्त्यांना विनामूल्य चाचणी खाते कसे तयार करावे हे दर्शवितो. केस अ‍ॅक्टिव्ह या खात्यासाठी नवीन वापरकर्ता प्रशासक विशेषाधिकार स्वयंचलितपणे मंजूर करते.

खाते तयार करत आहे

  1. लॉगिन पृष्ठावर जा.
  2. प्रारंभ करा बटणावर क्लिक करा.
  3. ईमेल पत्ता प्रविष्ट करा.
  4. पुढील क्लिक करा.

खाते सत्यापित करत आहे

  1. एक नवीन ब्राउझर टॅब उघडा आणि तुमच्या ईमेल खात्यात लॉग इन करा.
  2. सहा-अंकी कोड कॉपी करा.
  3. केस सक्रिय टॅबवर परत या.
  4. फील्डमध्ये कोड पेस्ट करा. एक नवीन विंडो दिसेल.

नवीन विंडोमध्ये:

  1. आवश्यक फील्ड भरा.
  2. सेवा अटी वाचा आणि सहमत होण्यासाठी बॉक्स चेक करा.
  3. एक विनामूल्य खाते तयार करा क्लिक करा.

अॅप सेटअप पूर्ण करत आहे

पुढील पृष्ठावर कार्यप्रवाह प्रदर्शित होतो. तुमचा अॅप सेट करणे पूर्ण करण्यासाठी पायऱ्या पूर्ण करा. माहिती किंवा तुमची खाते स्थिती संपादित किंवा अद्यतनित करण्यासाठी तुम्ही नेहमी या पृष्ठांवर परत येऊ शकता.

अॅपबद्दल पृष्ठावर, कंपनीचे नाव आणि वेबसाइट फील्ड भरा (आवश्यक) आणि नंतर पुढील क्लिक करा.

बिलिंग माहिती पृष्ठावर, पेमेंट प्लॅन फील्डवर क्लिक करा आणि पर्यायांचे पुनरावलोकन करा. डीफॉल्ट सेटिंग विनामूल्य आहे. सबमिट करा क्लिक करा.

img

तुम्ही फॉर्म सबमिट केल्यानंतर, केस सक्रिय अॅप्स पृष्ठ प्रदर्शित करते. तुम्ही तयार करता किंवा संबंधित असलेल्या प्रत्येक अॅपसाठी, पृष्ठ दर्शवते:

  1. अॅपचे नाव
  2. तुमचा प्रकार (वापरकर्ता भूमिका)
  3. स्थिती

img

अॅप्स पृष्ठ हे अनेक प्रशासकीय कार्यांसाठी लॉन्चिंग पॉइंट आहे:

  1. केस-संबंधित समस्यांवर संवाद साधण्यासाठी, डॉक्स भरण्यासाठी, व्हिडिओ चॅट (आणि बरेच काही) करण्यासाठी APP बटणावर क्लिक करा.
  2. अॅडमिन पॅनल उघडण्यासाठी ADMIN बटणावर क्लिक करा. येथे प्रशासक वापरकर्ते, प्रकरणे, एकत्रीकरण आणि बरेच काही व्यवस्थापित करतात.
  3. खाते बिलिंग आणि सूचना सेटिंग्ज व्यवस्थापित करण्यासाठी दीर्घवृत्तावर क्लिक करा.
  • टीप केस सक्रिय प्रशासक वापरकर्त्यांसाठी ADMIN बटण प्रदर्शित करते. ज्या वापरकर्त्यांना प्रशासकीय विशेषाधिकार नाहीत त्यांना ADMIN बटण दिसत नाही (खालील स्क्रीनमध्ये दर्शविल्याप्रमाणे).
