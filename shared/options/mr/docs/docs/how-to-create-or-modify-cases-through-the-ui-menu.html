UI मेनूद्वारे प्रकरणे कशी तयार करावी किंवा सुधारित करावी

हा लेख प्लॅटफॉर्मच्या वापरकर्ता इंटरफेससह एक केस कसा तयार करायचा हे प्रशासकांना दाखवतो. प्रशासक नवीन केसमध्ये वापरकर्ते जोडू शकतात आणि विद्यमान प्रकरणे अपडेट करू शकतात.

केस तयार करणे

  1. प्रशासक खात्यात लॉग इन करा.
  2. अॅप्स पृष्ठावर अॅप शोधा आणि प्रशासक बटणावर क्लिक करा.
  3. प्रशासन पॅनेलवर, केसेस मेनू टॅबवर क्लिक करा.
  4. नवीन बटणावर क्लिक करा. एक नवीन विंडो दिसेल.

आवश्यक फील्ड भरा आणि सेव्ह करा वर क्लिक करा.

वापरकर्ते जोडत आहे

वैकल्पिकरित्या, प्रशासक या पृष्ठावरील केसमध्ये वापरकर्त्यांना जोडू शकतात. खाली स्क्रोल करा आणि वापरकर्ते नियुक्त करा बटणावर क्लिक करा.

असाइन युजर मॉडेलमध्ये, फील्ड भरा. वापरकर्त्याला ईमेल सूचना पाठवण्यासाठी बॉक्स चेक करा. (पर्यायी)

केस सुधारित करणे

काही प्रकरणे अद्यतनित करण्यासाठी, संपादन साधन वापरा आणि प्रत्येक वैयक्तिक प्रकरणात बदल करा.

  • टीप एका बैठकीमध्ये मोठ्या संख्येने प्रकरणे अद्यतनित करण्यासाठी, आयात साधन वापरण्याचा विचार करा
  1. प्रशासक खात्यात लॉग इन करा.
  2. अॅप्स पृष्ठावर अॅप शोधा आणि प्रशासक बटणावर क्लिक करा.
  3. प्रशासन पॅनेलवर, केसेस मेनू टॅबवर क्लिक करा.
  4. केस शोधा आणि संपादित करा वर क्लिक करा.
  5. बदल करा आणि सेव्ह करा वर क्लिक करा.
