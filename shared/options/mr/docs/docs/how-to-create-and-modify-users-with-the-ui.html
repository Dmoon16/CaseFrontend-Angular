UI सह वापरकर्ते कसे तयार आणि सुधारित करावे

हा लेख प्लॅटफॉर्मच्या वापरकर्ता इंटरफेससह वापरकर्ते कसे तयार करावे आणि रेकॉर्ड अद्यतनित कसे करावे हे प्रशासकांना दाखवतो. प्रशासक नवीन केसमध्ये वापरकर्ते जोडू शकतात आणि विद्यमान प्रकरणे संपादित करू शकतात.

वापरकर्ता तयार करणे

  1. प्रशासक खात्यात लॉग इन करा.
  2. अॅप्स पृष्ठावर अॅप शोधा आणि प्रशासक बटणावर क्लिक करा.
  3. प्रशासन पॅनेलवर, वापरकर्ते मेनू टॅबवर क्लिक करा.
  4. नवीन बटणावर क्लिक करा. एक नवीन विंडो दिसेल.

फील्ड भरा आणि सेव्ह वर क्लिक करा. केस अॅक्टिव्ह या अॅपशी संबंधित वापरकर्त्यांच्या सूचीमध्ये नाव जोडते.

img

वापरकर्ता सुधारित करणे

थोड्या संख्येने वापरकर्ता रेकॉर्ड अद्यतनित करण्यासाठी, संपादन साधन वापरा आणि प्रत्येक वापरकर्त्यामध्ये बदल करा.

  • टीप एका बैठकीत मोठ्या संख्येने वापरकर्ता रेकॉर्ड सुधारित करण्यासाठी, आयात साधन वापरण्याचा विचार करा.
  1. प्रशासन पॅनेलवर, वापरकर्ते मेनू टॅबवर क्लिक करा.
  2. वापरकर्ता शोधा आणि संपादित करा क्लिक करा.
  3. बदल करा आणि जतन करा वर क्लिक करा.

पुढील पायऱ्या

केस सक्रिय वापरकर्त्याला ईमेल सूचना पाठवत नाही. प्रशासकांनी वापरकर्त्याशी संपर्क साधावा आणि त्यांना केस ऍक्टिव्ह होम पेजवर लिंक पाठवावी. त्यानंतर वापरकर्ता लॉगिन बटणावर क्लिक करतो आणि नवीन साइनअपसाठी कार्यप्रवाह पूर्ण करतो.

  • हा लेख नवीन वापरकर्त्यांसाठी साइनअप वर्कफ्लोचे वर्णन करतो.
