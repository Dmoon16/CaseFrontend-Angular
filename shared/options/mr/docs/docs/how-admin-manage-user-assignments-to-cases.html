प्रशासन प्रकरणांमध्ये वापरकर्ता असाइनमेंट कसे व्यवस्थापित करतात

वापरकर्ते आणि केस असाइनमेंट व्यवस्थापित करण्याचे पाच मार्ग आहेत. कसे ते जाणून घेण्यासाठी लिंकवर क्लिक करा.

पद्धत प्रकार क्रिया तपशील
केस तयार करताना UI जोडा प्रशासन बटण > प्रकरणे > नवीन बटण > वापरकर्ते नियुक्त करा बटण
सार्वजनिक साइनअप लिंक वापरून वापरकर्त्यास स्वयं नियुक्त करा वापरकर्ता क्रिया जोडा प्रशासन बटण > सेटिंग्ज > सिस्टम सेटअप पृष्ठ
विद्यमान केसवर UI वापरणे UI जोडा आणि काढा प्रशासन बटण > प्रकरणे > वापरकर्ते नियुक्त करा
