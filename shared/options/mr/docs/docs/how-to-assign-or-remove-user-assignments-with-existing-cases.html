विद्यमान प्रकरणांसह वापरकर्ता असाइनमेंट कसे नियुक्त करायचे किंवा काढायचे

हा लेख प्रशासकांना केस वापरकर्त्यांना कसे नियुक्त करायचे आणि कसे काढायचे ते दाखवतो. वापरकर्ता डेटा कसा बदलायचा हे देखील आम्ही तुम्हाला दाखवतो.

विद्यमान केस उघडत आहे

  1. प्रशासक खात्यात लॉग इन करा.
  2. अॅप्स पृष्ठावर अॅप शोधा आणि प्रशासक बटणावर क्लिक करा.
  3. केसेस मेनू टॅबवर क्लिक करा.
  4. सबमेनूमध्ये ACTIVE निवडल्याचे सुनिश्चित करा.
  5. ACTION स्तंभापर्यंत खाली स्क्रोल करा.

नियुक्त करा किंवा काढा

केसमध्ये वापरकर्ता नियुक्त करण्यासाठी किंवा वापरकर्त्याला केसमधून काढून टाकण्यासाठी वापरकर्त्यांना नियुक्त करा क्लिक करा (परंतु प्लॅटफॉर्मवर नाही). पुढील पृष्ठावर: ADD वर क्लिक करा आणि कार्यप्रवाह पूर्ण करा. वापरकर्ता हटवण्यासाठी काढा क्लिक करा.

सुधारित करा

केसेस पेजवर वापरकर्ता डेटा संपादित करणे किंवा अपडेट करणे शक्य नाही. त्याऐवजी, या चरणांचे अनुसरण करा:

  1. वापरकर्ते मेनू टॅबवर क्लिक करा.
  2. वापरकर्ता नाव शोधा आणि संपादित करा क्लिक करा.
  3. बदल करा आणि सेव्ह करा वर क्लिक करा.
