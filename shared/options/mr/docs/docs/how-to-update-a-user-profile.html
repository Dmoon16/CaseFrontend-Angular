वापरकर्ता प्रोफाइल कसे अपडेट करावे

हा लेख वापरकर्त्यांना त्यांच्या प्रोफाइल पेजवर माहिती कशी अपडेट करायची ते दाखवतो.

  • वापरकर्ते त्यांचे स्वतःचे प्रोफाइल अपडेट करू शकतात.
  • प्रशासक कोणत्याही वापरकर्त्याचे प्रोफाइल पृष्ठ अद्यतनित करू शकतात.

वापरकर्त्यांसाठी

पहिली पायरी म्हणजे प्रोफाइल पेज उघडणे. वापरकर्ते आणि प्रशासक त्यांचे स्वतःचे प्रोफाइल पृष्ठ अद्यतनित करण्यासाठी या चरणांचे अनुसरण करतात.

  1. तुमच्या केस सक्रिय खात्यात लॉग इन करा.
  2. पृष्ठाच्या शीर्षस्थानी असलेल्या प्रोफाइल टॅबवर क्लिक करा.
  3. बदल करा.
  4. Save वर क्लिक करा.

बहुतेक फील्ड ऐच्छिक आहेत आणि काही फील्ड आवश्यक आहेत. आवश्यक फील्ड आहेत:

  • पहिले नाव
  • आडनाव
  • ईमेल पत्ता
  • इंग्रजी
  • वेळ क्षेत्र

प्रशासकांसाठी

प्रशासक इतर वापरकर्त्यांसाठी प्रोफाइल माहिती देखील अद्यतनित करू शकतात. यामध्ये HOST USER INFO हेडिंग अंतर्गत तीन फील्ड समाविष्ट आहेत.

  1. तुमच्या केस ऍक्टिव्ह ऍडमिन खात्यामध्ये लॉग इन करा.
  2. अॅप शोधा आणि प्रशासक बटणावर क्लिक करा.
  3. शीर्ष मेनूवरील वापरकर्ते क्लिक करा.
  4. उप-मेनूवर जा आणि तुम्हाला प्रवेश करायचा असलेली वापरकर्ता सूची निवडा (सक्रिय किंवा अक्षम). डीफॉल्ट सेटिंग सक्रिय आहे.
  5. सूचीमध्ये वापरकर्ता शोधा आणि संपादित करा क्लिक करा.
  6. आवश्यकतेनुसार बदल करा. (खाली पहा)
  7. Save वर क्लिक करा.

वापरकर्ता माहिती बदला

प्रशासक वापरकर्त्याची माहिती बदलू शकतात.

  1. वापरकर्ता प्रकार (वापरकर्ता किंवा व्यवस्थापक) सेट करा.
  2. वापरकर्ता स्थिती सेट करा (अक्षम करण्यासाठी सक्रिय).
  3. एक अद्वितीय टॅग आयडी प्रविष्ट करा. (पर्यायी)
  4. Save वर क्लिक करा.
