प्रशासक कसे वापरकर्ते तयार करतात

अ‍ॅप्स आणि केसेस कोण अ‍ॅक्सेस करू शकतात ते प्रशासक तयार करतात, सुधारतात किंवा अक्षम करतात. जेव्हा वापरकर्ता प्रथमच प्लॅटफॉर्मवर लॉग इन करतो, तेव्हा त्यांना प्रोफाइल वर्कफ्लो पूर्ण करणे आवश्यक आहे.

  • नवीन वापरकर्ता प्रोफाइल वर्कफ्लोबद्दल अधिक जाणून घेण्यासाठी येथे क्लिक करा .

अधिसूचना

प्लॅटफॉर्म वापरकर्त्यांना अॅपमध्ये जोडल्यावर त्यांना सूचना पाठवत नाही. तथापि, प्रशासक केस असाइनमेंटसाठी सूचना सक्षम करू शकतात.
