Cara Membuat atau Mengubah Suai Kes dengan Alat Import

Artikel ini menunjukkan pentadbir cara membuat atau mengubah suai kes sedia ada dengan alat Import.

Membuka alat Import

  1. Log masuk ke akaun admin.
  2. Cari apl pada halaman Apl dan klik butang ADMIN.
  3. Pada panel Pentadbir, klik tab Integrasi dan kemudian IMPORT pada submenu.
  4. Klik butang BARU. Kotak dialog Tambah Import dipaparkan dalam tab penyemak imbas baharu.

img

Mencipta kes

Selepas kotak dialog Tambah Import dibuka:

  1. Klik medan Jenis dan pilih pilihan (lihat di bawah). Satu pautan dipaparkan.
  2. Klik pautan Muat Turun Contoh. Fail CSV dimuat turun ke peranti anda.

Seterusnya, buka fail dan tambah kes.

taip

Apabila anda mengklik medan Jenis , menu lungsur turun dipaparkan. Pilih satu daripada dua pilihan untuk import kotak pukal.

  • Nota Menu lungsur turun mempunyai banyak pilihan, tetapi hanya dua yang berkaitan dengan menguruskan kes. Setiap pilihan mempunyai sampel fail CSV yang berbeza.

Tambah kes

  • Pilih pilihan ini untuk muat naik pukal ringkas kes.
  • Klik pautan Muat Turun Contoh untuk membuka fail CSV dengan tajuk lajur untuk jenis import Cipta kes.

Ubah suai kes

  • Pilih pilihan ini untuk mengemas kini kes semasa.
  • Klik pautan Muat Turun Contoh untuk membuka fail CSV dengan tajuk lajur untuk jenis import Ubah suai kes.

Memuat naik fail

Anda boleh memuat naik fail dengan segera atau kembali ke kotak dialog Tambah Import pada tarikh kemudian dan lengkapkan langkah ini.

  1. Buka kotak dialog Tambah Import.
  2. Masukkan nama unik untuk fail CSV.
  3. Klik ikon tambah (+) untuk memuat naik fail daripada peranti anda.
  4. Klik medan Jenis dan pilih pilihan. (lihat di atas)
  5. Klik SIMPAN.

img

Memformat

Ini adalah medan yang diperlukan dan keperluan pertama. Klik di sini untuk menyemak butiran pemformatan.

tag_id (atau case_ID) pengecam unik (gunakan aksara alfanumerik, tanda sempang diterima)

status_kes 0 = ditutup 1 = aktif

tarikh_dibuka bulan/hari/tahun mm/hb/tttt

tentang perihalan unik kes (gunakan aksara abjad angka, tanda sempang diterima)

img

Memproses

Bergantung pada bilangan rekod dalam fail dan beban sistem, platform boleh mengambil masa beberapa minit untuk memproses import.

img

Apabila import selesai, platform melaraskan rekod kes (iaitu menambah atau mengubah suai) pada papan pemuka Cases pentadbir.

Kegagalan

Jika operasi import gagal, platform mengembalikan mesej Gagal. Dalam kebanyakan kes, punca kegagalan ialah pemformatan data.

img

Untuk melihat punca kegagalan:

  1. Klik Lihat di bawah tajuk TINDAKAN.
  2. Pada halaman seterusnya, tuding tetikus pada ikon Gagal. Petua alat memaparkan ralat.
  3. Pergi ke fail CVS dan laraskan dan betulkan ralat.
  4. Muat naik fail sekali lagi.
