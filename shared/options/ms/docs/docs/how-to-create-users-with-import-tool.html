Cara Membuat Pengguna dengan Alat Import

Artikel ini menunjukkan pentadbir cara membuat pengguna dengan alat Import.

Membuka alat Import

Pentadbir boleh menggunakan alat Import untuk memuat naik secara pukal pengguna baharu dan mengubah suai rekod pengguna sedia ada.

  1. Log masuk ke akaun admin.
  2. Cari apl pada halaman Apl dan klik butang ADMIN.
  3. Pada panel Pentadbir, klik tab Integrasi dan kemudian IMPORT pada submenu.
  4. Klik butang BARU. Kotak dialog Tambah Import dipaparkan dalam tab penyemak imbas baharu.

Mencipta rekod

Selepas kotak dialog Tambah Import dibuka:

  1. Klik medan Jenis dan pilih pilihan (lihat di bawah). Satu pautan dipaparkan.
  2. Klik pautan Muat Turun Contoh . Fail CSV dimuat turun ke peranti anda.

Seterusnya, buka fail dan tambah rekod. Klik di sini untuk mengetahui cara menggunakan data format dalam fail CSV.

taip

Apabila anda mengklik medan Jenis, menu lungsur turun dipaparkan. Pilih satu daripada dua pilihan untuk import pukal.

:seru: NOTA Menu lungsur turun mempunyai banyak pilihan, tetapi hanya dua yang berkaitan dengan alat Import. Setiap pilihan mempunyai sampel fail CSV yang berbeza.

Tambah pengguna

  • Pilih pilihan ini untuk muat naik pukal ringkas rekod pengguna.
  • Klik pautan Muat Turun Contoh untuk membuka fail CSV dengan tajuk lajur untuk jenis Import Tambah pengguna.

Ubah suai pengguna

  • Pilih pilihan ini untuk mengemas kini data bagi pengguna semasa.
  • Klik pautan Muat Turun Contoh untuk membuka fail CSV dengan tajuk lajur untuk jenis Import pengguna Ubah suai.

Memuat naik fail

Anda boleh memuat naik fail dengan segera atau kembali ke kotak dialog Tambah Import pada tarikh kemudian dan lengkapkan langkah ini.

  1. Buka kotak dialog Tambah Import.
  2. Masukkan nama unik untuk fail CSV.
  3. Klik ikon tambah (+) untuk memuat naik fail daripada peranti anda.
  4. Klik medan Jenis dan pilih pilihan. (lihat di atas)
  5. Klik SIMPAN.

Memproses

Bergantung pada bilangan rekod dalam fail dan beban sistem, platform boleh mengambil masa beberapa minit untuk memproses import.

Apabila import selesai, platform melaraskan rekod pengguna (iaitu menambah, mengubah suai atau melumpuhkan) pada papan pemuka Pengguna pentadbir.

Kegagalan

Jika operasi import gagal, platform mengembalikan mesej Gagal. Dalam kebanyakan kes, punca kegagalan ialah pemformatan data.

Untuk melihat punca kegagalan:

  1. Klik Lihat di bawah tajuk Tindakan.
  2. Pada halaman seterusnya, tuding tetikus pada ikon Gagal. Petua alat memaparkan ralat.
  3. Pergi ke fail CVS dan laraskan dan betulkan ralat.
  4. Muat naik fail sekali lagi.
