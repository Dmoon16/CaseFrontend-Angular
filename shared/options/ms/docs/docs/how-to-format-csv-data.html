Cara Memformat Data CSV

Di bawah ialah senarai medan data dalam fail CSV. Setiap pilihan import mempunyai gabungan medan yang berbeza.

nama pengguna

  • Masukkan alamat e-mel pengguna.

nama diberi

  • Masukkan nama pertama pengguna.

nama keluarga

  • Masukkan nama akhir pengguna.

setempat

  • Lajur ini menyemak bahasa pilihan pengguna.
  • Masukkan en untuk bahasa Inggeris, sp untuk bahasa Sepanyol.

info zon

  • Ini menetapkan zon waktu pengguna. Masuk ke benua dan bandar tanpa ruang (gunakan garis bawah).
  • Mesti menggunakan pilihan dalam menu lungsur turun Profil > BAHASA & WILAYAH > Zon waktu.
  • Gunakan zon waktu yang paling sesuai. Contohnya, Boston tiada pada menu lungsur, jadi kami boleh menggunakan New York (atau mana-mana bandar dalam zon waktu EST/EDT).

hos_tag_id

  • Masukkan label kekal.
  • az, 0-9, -, / dibenarkan.

status_kes

  • 0 = ditutup
  • 1 = aktif
