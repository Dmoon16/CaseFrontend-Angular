Cara Pentadbir Mengurus Tugasan Pengguna Kepada Kes

Terdapat lima cara untuk mengurus pengguna dan tugasan kes. Klik pautan untuk mengetahui caranya.

Kaedah taip Tindakan Butiran
Semasa membuat kes UI Tambah Butang pentadbir > Kes > Butang baharu > Butang Tetapkan Pengguna
Tetapkan pengguna secara automatik menggunakan pautan pendaftaran awam Tindakan Pengguna Tambah Butang pentadbir > Tetapan > Halaman Persediaan Sistem
Menggunakan UI pada kes sedia ada UI tambah & buang Butang pentadbir > Kes > Tetapkan Pengguna
Alat import UI tambah, ubah suai & alih keluar Butang pentadbir > Integrasi > Import
Menggunakan API (Akan Datang) API tambah, ubah suai & alih keluar T/A
