Memformat Data Pengurusan Kes dengan Alat Import

Di bawah ialah senarai medan yang diperlukan dalam fail CSV untuk mengurus kes dengan alat Import.

nama pengguna

  • Masukkan alamat e-mel pengguna.

tag_id (id yang boleh anda rujuk kemudian)

  • pengecam unik (gunakan aksara abjad angka, tanda sempang diterima)

status_kes

  • 0 = ditutup
  • 1 = aktif

tarikh_dibuka

  • bulan hari tahun
  • mm/hh/tttt

kira-kira

  • penerangan unik kes (gunakan aksara abjad angka, tanda sempang diterima)
