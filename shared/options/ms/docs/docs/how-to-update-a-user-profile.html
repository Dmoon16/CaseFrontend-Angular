Cara Mengemas kini Profil Pengguna

Artikel ini menunjukkan kepada pengguna cara mengemas kini maklumat pada halaman Profil mereka.

  • Pengguna boleh mengemas kini profil mereka sendiri.
  • Pentadbir boleh mengemas kini halaman Profil mana-mana pengguna.

Untuk pengguna

Langkah pertama ialah membuka halaman Profil. Pengguna dan pentadbir mengikut langkah ini untuk mengemas kini halaman Profil mereka sendiri.

  1. Log masuk ke akaun Case Active anda.
  2. Klik tab Profil di bahagian atas halaman.
  3. Buat perubahan.
  4. Klik Simpan.

Kebanyakan medan adalah pilihan, dan beberapa medan diperlukan. Medan yang diperlukan ialah:

  • Nama pertama
  • Nama terakhir
  • Alamat emel
  • Bahasa
  • Zon masa

Untuk admin

Pentadbir juga boleh mengemas kini maklumat profil untuk pengguna lain. Ini termasuk tiga medan di bawah tajuk HOST USER INFO.

  1. Log masuk ke akaun pentadbir Case Active anda.
  2. Cari apl dan klik butang ADMIN.
  3. Klik Pengguna pada menu atas.
  4. Pergi ke sub-menu dan pilih senarai pengguna yang ingin anda akses (AKTIF atau LUHUR). Tetapan lalai ialah Aktif.
  5. Cari pengguna dalam senarai dan klik Edit.
  6. Buat perubahan mengikut keperluan. (lihat di bawah)
  7. Klik Simpan.

Tukar maklumat pengguna

Pentadbir boleh menukar maklumat pengguna.

  1. Tetapkan Jenis Pengguna (Pengguna atau Pengurus).
  2. Tetapkan Status Pengguna (Aktif kepada Dilumpuhkan).
  3. Masukkan ID Teg unik. (pilihan)
  4. Klik Simpan.
