Cara Membuat dan Mengubah Suai Pengguna dengan UI

Artikel ini menunjukkan pentadbir cara membuat pengguna dan mengemas kini rekod dengan antara muka pengguna platform. Pentadbir juga boleh menambah pengguna pada kes baharu dan mengedit kes sedia ada.

Mencipta pengguna

  1. Log masuk ke akaun admin.
  2. Cari apl pada halaman Apl dan klik butang ADMIN.
  3. Pada panel Pentadbir, klik tab menu Pengguna.
  4. Klik butang BARU. Tetingkap baharu dipaparkan.

img

Isi ruangan dan klik SIMPAN. Case Active menambah nama pada senarai pengguna yang dikaitkan dengan apl ini.

img

Mengubah suai pengguna

Untuk mengemas kini sebilangan kecil rekod pengguna, gunakan alat Edit dan buat perubahan kepada setiap pengguna individu.

  • Nota Untuk mengubah suai sejumlah besar rekod pengguna dalam satu masa, pertimbangkan untuk menggunakan alat Import .
  1. Pada panel Pentadbir, klik tab menu Pengguna.
  2. Cari pengguna dan klik Edit.
  3. Buat perubahan dan klik SIMPAN.

img

Langkah seterusnya

Case Active tidak menghantar pemberitahuan e-mel kepada pengguna. Pentadbir hendaklah menghubungi pengguna dan menghantar pautan kepada mereka ke halaman utama Case Active. Pengguna kemudian mengklik butang LOGIN dan melengkapkan aliran kerja untuk pendaftaran baharu.

  • Artikel ini menerangkan aliran kerja pendaftaran untuk pengguna baharu.
