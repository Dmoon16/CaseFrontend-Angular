Cara Menugaskan Pengguna Semasa Mencipta Kes

Artikel ini menunjukkan pentadbir cara menetapkan pengguna semasa membuat kes baharu. Jika pengguna belum berdaftar di platform, anda boleh membuat tambah pengguna baharu sambil mencipta kes baharu.

Mencipta kes

  1. Log masuk ke akaun admin.
  2. Cari apl pada halaman Apl dan klik butang ADMIN.
  3. Klik tab menu Kes.
  4. Klik butang BARU. Tetingkap baharu dipaparkan.
  • Isi medan untuk mencipta kes dan kemudian tatal ke bawah ke tajuk Pengguna Ditugaskan.

Menugaskan pengguna

Pengguna semasa

Ikut langkah ini untuk menetapkan pengguna semasa kepada kes baharu.

Klik butang TUGASKAN PENGGUNA dan isikan medan.

  1. Klik menu lungsur Pengguna dan pilih pengecam (cth. e-mel).
  2. Masukkan nilai dalam medan kosong (cth alamat e-mel).
  3. Jika rekod pengguna wujud dalam platform, medan auto-cadangkan nama.
  4. Pilih pengguna yang betul.
  5. Pilih peranan pengguna (pelanggan, lalai atau kakitangan).
  6. Sebagai pilihan, tandai kotak untuk menghantar pemberitahuan e-mel kepada pengguna.
  7. Klik SIMPAN.

Case Active menambah pengguna pada kes itu. Kini pentadbir boleh mengklik Simpan untuk menutup tetingkap, atau menambah pengguna baharu, atau mengalih keluar pengguna.

Menambah pengguna baharu

Jika platform tidak dapat mencari rekod semasa, klik butang untuk menambah pengguna baharu.

img

Isikan medan dan kemudian klik SIMPAN untuk menambah pengguna ke platform. Seterusnya, tetapkan peranan pengguna dan klik SIMPAN sekali lagi untuk menetapkan pengguna kepada kes itu. Akhir sekali, klik SIMPAN sekali lagi untuk menyimpan kes itu.
