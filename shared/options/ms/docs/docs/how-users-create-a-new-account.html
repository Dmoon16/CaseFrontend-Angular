Cara Pengguna Membuat Akaun Baharu

Selepas pentadbir menambahkan pengguna pada kes: * jika pengguna baharu kepada caseactive, pengguna mesti melengkapkan aliran kerja untuk membuat akaun dan menerima jemputan untuk mengakses kes tersebut. * jika pengguna mempunyai akaun caseactive sedia ada, pengguna mesti menerima jemputan untuk mengakses kes tersebut.

Artikel ini menerangkan langkah-langkah yang perlu dilakukan oleh pengguna untuk membuat akaun mereka.

Bermula

Terdapat dua cara pengguna boleh ditambahkan pada aplikasi caseactive

  • Oleh pentadbir: Jika pentadbir mencipta pengguna melalui UI platform atau Alat Import , pentadbir mesti menghantar pautan kepada pengguna ke account.caseactive.com untuk meneruskan log masuk.
  • Mengikut pengguna: Jika pengguna mencipta akaun menggunakan pautan pendaftaran , pengguna akan diubah hala secara automatik ke account.caseactive.com untuk meneruskan log masuk.

Log masuk

Apabila pengguna log masuk ke akaun CaseActive mereka buat kali pertama, platform menggesa pengguna untuk:

  • buka akaun e-mel mereka dan kemudian salin/tampal kod keselamatan enam digit
  • melengkapkan profil akaun

img

Selepas pengguna mengklik KEMASKINI, halaman Apl dipaparkan. Halaman menunjukkan nama apl (nama syarikat), jenis pengguna (peranan), status dan butang JEMPUTAN.

img

Untuk mengakses apl, pengguna mesti menerima jemputan. Pengguna hendaklah:

  1. Klik butang JEMPUTAN.
  2. Semak kebenaran.
  3. Klik tab Pemberitahuan untuk menyemak dan mengedit (pilihan) syarat.
  4. Klik LULUSKAN AKSES.

Apabila pengguna meluluskan, platform memaparkan butang APP pada halaman pengguna. Butang ini memberi pengguna akses kepada kes yang ditetapkan.

Untuk mengedit tetapan pengguna, pengguna mengklik elipsis.

img

Pengguna boleh: * mengalih keluar akses aplikasi * mengubah suai tetapan pemberitahuan

img

Selepas pengguna melengkapkan aliran kerja, nama pengguna muncul pada senarai Pengguna pentadbir. Senarai menunjukkan semua pengguna, log masuk atau tidak.
