Memformat Data Tugasan Kes Pengguna dengan Alat Import

Di bawah ialah senarai medan data dalam fail CSV. Setiap pilihan import mempunyai gabungan medan yang berbeza.

case_ID

  • Masukkan ID Pengguna atau Teg kes.
  • Pergi ke halaman Kes > Edit > ID Teg ATAU Pergi ke halaman Kes > Edit > ID Kes dalam url.

ID Pengguna

  • Masukkan ID pengguna atau ID Teg.
  • Pergi ke halaman Pengguna > Edit > ID Teg ATAU Pergi ke halaman Pengguna > Edit > ID Pengguna dalam url.

pesanan_peranan_kes

  • Biarkan sel kosong.

case_role_ID

  • Masukkan: pelanggan, kakitangan atau lalai.

case_notify

  • 0 = tidak
  • 1 = Ya, hantar pemberitahuan e-mel kepada pengguna.
