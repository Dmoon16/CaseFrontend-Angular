Cara Membuat atau Mengubah Suai Kes Melalui Menu UI

Artikel ini menunjukkan kepada pentadbir cara membuat kes dengan antara muka pengguna platform. Pentadbir juga boleh menambah pengguna pada kes baharu dan mengemas kini kes sedia ada.

Mencipta kes

  1. Log masuk ke akaun admin.
  2. Cari apl pada halaman Apl dan klik butang ADMIN.
  3. Pada panel Pentadbir, klik tab menu Kes.
  4. Klik butang BARU. Tetingkap baharu dipaparkan.

Isikan medan yang diperlukan dan klik SIMPAN.

img

Menambah pengguna

Secara pilihan, pentadbir boleh menambah pengguna pada kes di halaman ini. Tatal ke bawah dan klik butang TUGASKAN PENGGUNA .

img

Dalam mod Assign User, isikan medan. Tandai kotak untuk menghantar pemberitahuan e-mel kepada pengguna. (pilihan)

Mengubah suai kes

Untuk mengemas kini sebilangan kecil kes, gunakan alat Edit dan buat perubahan pada setiap kes individu.

  • Nota Untuk mengemas kini sejumlah besar kes dalam satu sidang, pertimbangkan untuk menggunakan alat Import
  1. Log masuk ke akaun admin.
  2. Cari apl pada halaman Apl dan klik butang ADMIN.
  3. Pada panel Pentadbir, klik tab menu Kes.
  4. Cari kes dan klik Edit.
  5. Buat perubahan dan klik SIMPAN.
