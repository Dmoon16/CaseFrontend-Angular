Cara Pentadbir Mengurus Pengguna

Pentadbir membuat, mengubah suai atau melumpuhkan orang yang boleh mengakses apl dan kes. Apabila pengguna log masuk ke platform buat kali pertama, mereka perlu melengkapkan aliran kerja profil.

  • Klik di sini untuk mengetahui lebih lanjut tentang aliran kerja profil pengguna baharu.

Mengurus pengguna

Jadual di bawah menunjukkan lima cara untuk mengurus pengguna. Klik pautan untuk mengetahui caranya.

Kaedah taip Tindakan Butiran
Menggunakan UI UI tambah & ubah suai Butang pentadbir > Pengguna
Semasa mencipta kes dan menetapkan pengguna kepada kes UI Tambah Butang pentadbir > Kes > Butang baharu > Cipta halaman Kes

Menggunakan API (akan datang) API tambah & ubah suai T/A

Pemberitahuan

Platform ini tidak menghantar pemberitahuan kepada pengguna apabila mereka ditambahkan pada aplikasi. Walau bagaimanapun, pentadbir boleh mendayakan pemberitahuan untuk tugasan kes.
