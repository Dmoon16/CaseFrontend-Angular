Испод је листа обавезних поља у ЦСВ датотекама за управљање случајевима помоћу алатке за увоз.

корисничко име

  • Унесите адресу е-поште корисника.

таг_ид (ид који можете касније да позовете)

  • јединствени идентификатор (користите алфанумеричке знакове, цртице су прихваћене)

цасе_статус

  • 0 = затворено
  • 1 = активан

дате_опенед

  • месец дан година
  • мм/дд/гггг
