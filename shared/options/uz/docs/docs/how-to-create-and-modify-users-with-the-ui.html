UI yordamida foydalanuvchilarni qanday yaratish va o'zgartirish mumkin

Ushbu maqola administratorlarga foydalanuvchilar yaratish va platformaning foydalanuvchi interfeysi bilan yozuvlarni yangilash usullarini ko'rsatadi. Shuningdek, administratorlar foydalanuvchilarni yangi ishga qo‘shishlari va mavjud ishlarni tahrirlashlari mumkin.

Foydalanuvchi yaratish

  1. Administrator hisobiga kiring.
  2. Ilovalar sahifasida ilovani toping va ADMIN tugmasini bosing.
  3. Administrator panelida Foydalanuvchilar menyusi yorlig'ini bosing.
  4. YANGI tugmasini bosing. Yangi oyna ochiladi.

img

Maydonlarni to'ldiring va SAQLASH tugmasini bosing. Case Active ismni ushbu ilova bilan bog'langan foydalanuvchilar ro'yxatiga qo'shadi.

img

Foydalanuvchini o'zgartirish

Kichik miqdordagi foydalanuvchi yozuvlarini yangilash uchun Tahrirlash vositasidan foydalaning va har bir foydalanuvchiga o'zgartirish kiriting.

  • Eslatma Ko'p sonli foydalanuvchi yozuvlarini bir o'tirishda o'zgartirish uchun Import vositasidan foydalanishni o'ylab ko'ring.
  1. Administrator panelida Foydalanuvchilar menyusi yorlig'ini bosing.
  2. Foydalanuvchini toping va Tahrirlash tugmasini bosing.
  3. O'zgartirishlar kiriting va SAQLASH tugmasini bosing.

Keyingi qadamlar

Case Active foydalanuvchiga elektron pochta xabarnomasini yubormaydi. Adminlar foydalanuvchi bilan bog‘lanib, ularga Case Active bosh sahifasiga havola yuborishlari kerak. Keyin foydalanuvchi LOGIN tugmasini bosadi va yangi ro'yxatdan o'tish uchun ish jarayonini yakunlaydi.

  • Ushbu maqolada yangi foydalanuvchilar uchun ro'yxatdan o'tish jarayoni tasvirlangan.
