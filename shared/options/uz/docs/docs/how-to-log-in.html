Qanday qilib tizimga kirish mumkin

Mavjud Case Active hisobingizga qanday kirishni bilib oling.

Tizimga kirish

  1. Kirish sahifasiga o'ting .
  2. LOGIN tugmasini bosing.
  3. Elektron pochtangizni kiriting.
  4. KEYINGI tugmasini bosing.

Hisobni tekshirish

  1. Yangi brauzer yorlig'ini oching va elektron pochta qayd yozuvingizga kiring.
  2. Olti xonali koddan nusxa oling.
  3. Case Active yorlig'iga qayting.
  4. Kodni maydonlarga joylashtiring. Yangi oyna ochiladi.

Case Active ilovalar sahifasini ko'rsatadi.

  • Eslatma Platformada administrator imtiyozlariga ega foydalanuvchilar uchun APP tugmasi yonida ADMIN tugmasi ko'rsatiladi.
