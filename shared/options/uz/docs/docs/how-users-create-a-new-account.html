Foydalanuvchilar qanday qilib yangi hisob yaratish

Administrator foydalanuvchini ishga qo‘shgandan so‘ng: * agar foydalanuvchi caseactive uchun yangi bo‘lsa, foydalanuvchi hisob yaratish uchun ish jarayonini yakunlashi va ishga kirish taklifini qabul qilishi kerak. * agar foydalanuvchida mavjud kassa faol hisob qaydnomasi bo'lsa, foydalanuvchi ishga kirish taklifini qabul qilishi kerak.

Ushbu maqola foydalanuvchi o'z hisobini yaratish uchun bajarishi kerak bo'lgan qadamlarni tavsiflaydi.

Ishni boshlash

Foydalanuvchini caseactiv ilovasiga qoʻshishning ikki yoʻli mavjud

  • Administrator tomonidan: Agar administrator UI platformasi yoki Import vositasi orqali foydalanuvchi yaratsa, tizimga kirish uchun administrator foydalanuvchiga account.caseactive.com saytiga havola yuborishi kerak.
  • Foydalanuvchi tomonidan: Agar foydalanuvchi roʻyxatdan oʻtish havolasi yordamida hisob yaratgan boʻlsa, foydalanuvchi tizimga kirishni davom ettirish uchun avtomatik ravishda account.caseactive.com saytiga yoʻnaltiriladi.

Tizimga kirish

Foydalanuvchi CaseActive hisobiga birinchi marta kirganida, platforma foydalanuvchidan quyidagilarni taklif qiladi:

  • ularning elektron pochta hisobini oching va olti xonali xavfsizlik kodini nusxalash/joylashtirish
  • hisob profilini to'ldiring

Foydalanuvchi UPDATE tugmasini bosgandan so'ng, Ilovalar sahifasi ko'rsatiladi. Sahifada ilova nomi (kompaniya nomi), foydalanuvchi turi (rol), holati va TAKLIF tugmasi ko‘rsatilgan.

img

Ilovaga kirish uchun foydalanuvchi taklifni qabul qilishi kerak. Foydalanuvchi quyidagilarni bajarishi kerak:

  1. TAKLIF tugmasini bosing.
  2. Ruxsatlarni ko'rib chiqing.
  3. Shartlarni ko'rib chiqish va tahrirlash (ixtiyoriy) uchun Bildirishnomalar yorlig'ini bosing.
  4. KIRISHNI TASDIQLASH tugmasini bosing.

img

Foydalanuvchi ma'qullaganida, platforma foydalanuvchi sahifasida APP tugmasini ko'rsatadi. Bu tugma foydalanuvchiga tayinlangan holatlarga kirish imkonini beradi.

Foydalanuvchi sozlamalarini tahrirlash uchun foydalanuvchi ellipsni bosadi.
