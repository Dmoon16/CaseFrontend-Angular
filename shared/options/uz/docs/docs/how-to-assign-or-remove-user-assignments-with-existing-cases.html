Mavjud holatlar bilan foydalanuvchi topshiriqlarini qanday belgilash yoki olib tashlash

Ushbu maqola administratorlarga ishi foydalanuvchilarini qanday tayinlash va olib tashlashni ko'rsatadi. Shuningdek, biz sizga foydalanuvchi ma'lumotlarini qanday o'zgartirishni ko'rsatamiz.

Mavjud ishni ochish

  1. Administrator hisobiga kiring.
  2. Ilovalar sahifasida ilovani toping va ADMIN tugmasini bosing.
  3. Ishlar menyusi yorlig'ini bosing.
  4. Pastki menyuda ACTIVE tanlanganligiga ishonch hosil qiling.
  5. ACTION ustuniga pastga aylantiring.

img

Tayinlash yoki olib tashlash

Ishga foydalanuvchi tayinlash yoki foydalanuvchini ishdan olib tashlash uchun Foydalanuvchilarni tayinlash tugmasini bosing (lekin platformadan emas). Keyingi sahifada: QO'SHISH tugmasini bosing va ish jarayonini yakunlang. Foydalanuvchini oʻchirish uchun “Oʻchirish” tugmasini bosing.

img

O'zgartirish

Ishlar sahifasida foydalanuvchi ma'lumotlarini tahrirlash yoki yangilash mumkin emas. Buning o'rniga quyidagi amallarni bajaring:

  1. Foydalanuvchilar menyusi yorlig'ini bosing.
  2. Foydalanuvchi nomini toping va Tahrirlash -ni bosing.
  3. O'zgartirishlar kiriting va SAQLASH tugmasini bosing.
