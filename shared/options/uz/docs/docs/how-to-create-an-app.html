Ilovani qanday yaratish kerak

Ushbu maqola yangi foydalanuvchilarga bepul sinov hisobini qanday yaratishni ko'rsatadi. Case Active yangi foydalanuvchiga ushbu hisob uchun avtomatik ravishda administrator imtiyozlarini beradi.

Hisob yaratish

  1. Kirish sahifasiga o'ting .
  2. Ishga kirishish tugmasini bosing.
  3. Elektron pochta manzilini kiriting.
  4. Keyingiga bosing.

img

Hisobni tekshirish

  1. Yangi brauzer yorlig'ini oching va elektron pochta qayd yozuvingizga kiring.
  2. Olti xonali koddan nusxa oling.
  3. Case Active yorlig'iga qayting.
  4. Kodni maydonlarga joylashtiring. Yangi oyna ochiladi.

Yangi oynada:

  1. Kerakli maydonlarni to'ldiring.
  2. Xizmat shartlarini o'qing va rozilik bildirish uchun katakchani belgilang.
  3. BEPUL HISOB YARATISH tugmasini bosing.

Ilovani sozlash tugallanmoqda

Keyingi sahifada ish jarayoni ko'rsatiladi. Ilovangizni sozlashni tugatish uchun amallarni bajaring. Maʼlumotni yoki hisobingiz holatini tahrirlash yoki yangilash uchun har doim ushbu sahifalarga qaytishingiz mumkin.

"Ilova haqida" sahifasida kompaniya nomi va veb-sayt maydonlarini (majburiy) to'ldiring va "Keyingi" tugmasini bosing.

img

Hisob-kitob ma'lumotlari sahifasida "To'lov rejasi" maydonini bosing va variantlarni ko'rib chiqing. Standart sozlama Bepul. SUBMIT tugmasini bosing.

img

Shaklni yuborganingizdan so'ng, Case Active Ilovalar sahifasini ko'rsatadi. Siz yaratgan yoki tegishli bo'lgan har bir ilova uchun sahifa ko'rsatiladi:

  1. Ilova nomi
  2. Sizning turingiz (foydalanuvchi roli)
  3. Holat

img

Ilovalar sahifasi ko'plab administrator funktsiyalari uchun ishga tushirish nuqtasidir:

  1. Muloqot qilish, hujjatlarni to‘ldirish, ish bilan bog‘liq masalalar bo‘yicha video chat (va boshqalar) uchun APP tugmasini bosing.
  2. Administrator panelini ochish uchun ADMIN tugmasini bosing. Bu yerda administratorlar foydalanuvchilarni, holatlarni, integratsiyalarni va boshqalarni boshqaradi.
  3. Hisobni toʻlash va bildirishnoma sozlamalarini boshqarish uchun koʻchmas nuqtani bosing.
  • Eslatma Case Active administrator foydalanuvchilari uchun ADMIN tugmasini ko'rsatadi. Administrator huquqlariga ega bo'lmagan foydalanuvchilar ADMIN tugmasini ko'rmaydilar (quyidagi ekranda ko'rsatilganidek).
