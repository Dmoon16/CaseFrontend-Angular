Hoe administrateurs gebruikerstoewysings aan sake bestuur

Daar is vyf maniere om gebruikers en saakopdragte te bestuur. Klik op 'n skakel om te leer hoe.

Metode Tik Aksies Besonderhede
Terwyl die skep van 'n saak UI byvoeg Admin-knoppie > Sake > Nuwe knoppie > Ken gebruikers toe-knoppie
Wys gebruiker outomaties toe met behulp van publieke aanmeldskakel Gebruikersaksie byvoeg Admin-knoppie > Instellings > Stelselopstellingbladsy
Gebruik UI op bestaande saak UI voeg by en verwyder Admin-knoppie > Sake > Wys gebruikers toe
Invoer instrument UI voeg, wysig en verwyder Admin-knoppie > Integrasies > Invoere
Gebruik API (binnekort beskikbaar) API voeg, wysig en verwyder NVT
