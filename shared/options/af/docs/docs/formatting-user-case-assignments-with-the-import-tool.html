Formatering van gebruikersgevalle-toewysingsdata met die invoernutsding

Hieronder is 'n lys van datavelde in CSV-lêers. Elke invoeropsie het 'n ander kombinasie van velde.

saak_ID

  • Voer die saak Gebruiker of Tag ID in.
  • Gaan na Gevalle-bladsy > Wysig > Merk-ID OF Gaan na Gevalle-bladsy > Wysig > Geval-ID in url.

gebruiker_ID

  • Voer die gebruiker se ID of Tag ID in.
  • Gaan na Gebruikersbladsy > Wysig > Merk-ID OF Gaan na Gebruikersbladsy > Wysig > Gebruikers-ID in url.

case_role_order

  • Laat sel leeg.

saak_rol-ID

  • Voer in: kliënte, personeel of verstek.

saak_kennisgewing

  • 0 = nee
  • 1 = Ja, stuur vir gebruiker 'n e-poskennisgewing.
