Hoe administrateurs gebruikers bestuur

Administrateurs skep, wysig of deaktiveer wie toegang het tot toepassings en gevalle. Wanneer 'n gebruiker vir die eerste keer by die platform aanmeld, moet hulle 'n profielwerkvloei voltooi.

  • Klik hier om meer te wete te kom oor die nuwe gebruikersprofiel-werkvloei.

Bestuur van 'n gebruiker

Die tabel hieronder toon vyf maniere om 'n gebruiker te bestuur. Klik op 'n skakel om te leer hoe.

Metode Tik Aksies Besonderhede
Gebruik UI UI voeg by en wysig Admin-knoppie > Gebruikers
Terwyl jy 'n saak skep en gebruiker aan saak toewys UI byvoeg Admin-knoppie > Sake > Nuwe knoppie > Skep saak-bladsy
Skep gebruiker outomaties met behulp van publieke aanmeldskakel Gebruikersaksie byvoeg Admin-knoppie > Instellings > Stelselopstellingbladsy
Invoer instrument UI voeg by en wysig Admin-knoppie > Integrasies > Invoere
Gebruik API (kom binnekort) API voeg by en wysig NVT

Kennisgewings

Die platform stuur nie aan gebruikers 'n kennisgewing wanneer hulle by 'n toepassing gevoeg word nie. Administrateurs kan egter kennisgewings vir saakopdragte aktiveer.
