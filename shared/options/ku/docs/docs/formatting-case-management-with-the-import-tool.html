Formatkirina Daneyên Rêveberiya Doza bi Amûra Import

Li jêr navnîşek qadên pêwîst ên di pelên CSV-ê de hene ku bi amûra Importê dozan birêve bibin.

navê bikarhêner

  • Navnîşana e-nameya bikarhênerê binivîse.

tag_id (nasnameyek ku hûn dikarin paşê referans bikin)

  • nasnameyek yekta (karekterên alphanumerîk bikar bînin, daçek têne pejirandin)

case_status

  • 0 = girtî
  • 1 = çalak

date_vekirî

  • meh / roj / sal
  • mm/dd/yyyy

ji dor

  • danasînek yekta ya dozê (karekterên alfan-hejmarî bikar bînin, daçek têne pejirandin)
