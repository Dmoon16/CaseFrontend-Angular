Meriv çawa Bi UI-yê Bikarhêneran Diafirîne û Biguherîne

Vê gotarê rêvebiran nîşan dide ka meriv çawa bikarhêneran biafirîne û tomaran bi navgîniya bikarhênerê ya platformê nûve bike. Rêvebir jî dikarin bikarhêneran li dozek nû zêde bikin û dozên heyî biguherînin.

Afirandina bikarhênerek

  1. Têkeve hesabê admin.
  2. Serlêdanek li ser rûpela Serlêdan bibînin û bişkoka ADMIN bikirtînin.
  3. Li ser panela rêveberiyê, tabloya menuya Bikarhêner bikirtînin.
  4. Bişkojka NÛ bikirtînin. Pencereyek nû tê xuyang kirin.

Qadan dagirin û bikirtînin SAVE. Case Active navê li navnîşa bikarhênerên ku bi vê sepanê ve girêdayî ne zêde dike.

img

Guhertina bikarhênerek

Ji bo nûvekirina hejmareke piçûk a tomarên bikarhêner, amûra Guhertinê bikar bînin û li her bikarhênerek kesane guhertinan bikin.

  • Nîşe Ji bo guheztina hejmareke mezin a tomarên bikarhêner di yek rûniştinê de, bi karanîna amûra Import bifikirin.
  1. Li ser panela rêveberiyê, tabloya menuya Bikarhêner bikirtînin.
  2. Bikarhênerek bibînin û bikirtînin Guherandin.
  3. Guhertinan bikin û bikirtînin SAVE.
