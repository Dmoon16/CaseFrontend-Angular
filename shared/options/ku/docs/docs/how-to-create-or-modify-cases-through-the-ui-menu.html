Meriv çawa bi navgîniya pêşeka UI-yê Dozayan Afirîne an Biguherîne

Vê gotarê rêvebiran destnîşan dike ka meriv çawa bi navgîniya bikarhêner a platformê re dozek diafirîne. Rêvebir jî dikarin bikarhêneran li dozek nû zêde bikin û dozên heyî nûve bikin.

Afirandina dozê

  1. Têkeve hesabê admin.
  2. Serlêdanek li ser rûpela Serlêdan bibînin û bişkoka ADMIN bikirtînin.
  3. Li ser panela rêveberiyê, tabloya menuya Cases bikirtînin.
  4. Bişkojka NÛ bikirtînin. Pencereyek nû tê xuyang kirin.

Zeviyên pêwîst dagirin û bikirtînin SAVE.

Zêdekirina bikarhêneran

Vebijarkî, rêvebir dikarin bikarhêneran li ser vê rûpelê zêde bikin. Li jêr bigerin û bişkoka ASSIGN USERS bikirtînin .

img

Di modala Assign User de, qadan dagirin. Qutikê bişopînin da ku ji bikarhêner agahdariyek e-nameyê bişînin. (bixwe)

img

Guhertina dozek

Ji bo nûvekirina hejmareke piçûk a bûyeran, amûra Biguherîne bikar bînin û di her dozek kesane de guhertinan bikin.

  • Nîşe Ji bo nûvekirina hejmareke mezin a bûyeran di yek rûniştinê de, bi karanîna amûra Import bifikirin
  1. Têkeve hesabê admin.
  2. Serlêdanek li ser rûpela Serlêdan bibînin û bişkoka ADMIN bikirtînin.
  3. Li ser panela rêveberiyê, tabloya menuya Cases bikirtînin.
  4. Dozek bibînin û bikirtînin Biguherîne.
  5. Guhertinan bikin û bikirtînin SAVE.
