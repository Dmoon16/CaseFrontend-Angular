व्यवस्थापक मामलों के लिए उपयोगकर्ता असाइनमेंट कैसे प्रबंधित करते हैं

उपयोगकर्ताओं और केस असाइनमेंट को प्रबंधित करने के पांच तरीके हैं। कैसे जानने के लिए एक लिंक पर क्लिक करें।

तरीका टाइप कार्रवाई विवरण
केस बनाते समय यूआई जोड़ना व्यवस्थापक बटन > मामले > नया बटन > उपयोगकर्ता असाइन करें बटन
सार्वजनिक साइनअप लिंक का उपयोग करके उपयोगकर्ता को स्वतः असाइन करें उपयोगकर्ता क्रिया जोड़ना व्यवस्थापक बटन> सेटिंग्स> सिस्टम सेटअप पृष्ठ
मौजूदा मामले पर यूआई का उपयोग करना यूआई जोड़ना हटाना एडमिन बटन > मामले > उपयोगकर्ता असाइन करें
आयात उपकरण यूआई जोड़ें, संशोधित करें और हटाएं व्यवस्थापक बटन > एकीकरण > आयात
