यूजर प्रोफाइल कैसे अपडेट करें

यह आलेख उपयोगकर्ताओं को उनके प्रोफ़ाइल पृष्ठ पर जानकारी अपडेट करने का तरीका दिखाता है.

  • उपयोगकर्ता अपने स्वयं के प्रोफाइल को अपडेट कर सकते हैं।
  • एडमिन किसी भी यूजर के प्रोफाइल पेज को अपडेट कर सकते हैं।

उपयोगकर्ताओं के लिए

पहला कदम प्रोफाइल पेज खोलना है। उपयोगकर्ता और व्यवस्थापक अपने स्वयं के प्रोफ़ाइल पृष्ठ को अपडेट करने के लिए इन चरणों का पालन करते हैं।

  1. अपने केस एक्टिव खाते में लॉग इन करें।
  2. पृष्ठ के शीर्ष पर स्थित प्रोफ़ाइल टैब पर क्लिक करें।
  3. परिवर्तन करें।
  4. सहेजें पर क्लिक करें।

अधिकांश फ़ील्ड वैकल्पिक हैं, और कुछ फ़ील्ड आवश्यक हैं। आवश्यक फ़ील्ड हैं:

  • संतोष
  • उपनाम
  • ईमेल पता
  • भाषा
  • समय क्षेत्र

व्यवस्थापकों के लिए

व्यवस्थापक अन्य उपयोगकर्ताओं के लिए प्रोफ़ाइल जानकारी भी अपडेट कर सकते हैं। इसमें HOST USER INFO शीर्षक के तहत तीन क्षेत्र शामिल हैं।

  1. अपने केस एक्टिव एडमिन अकाउंट में लॉग इन करें।
  2. एक ऐप ढूंढें और एडमिन बटन पर क्लिक करें।
  3. शीर्ष मेनू पर उपयोगकर्ता क्लिक करें।
  4. उप-मेनू पर जाएं और उस उपयोगकर्ता सूची का चयन करें जिसे आप एक्सेस करना चाहते हैं (सक्रिय या अक्षम)। डिफ़ॉल्ट सेटिंग सक्रिय है।
  5. सूची में एक उपयोगकर्ता का पता लगाएँ और संपादित करें पर क्लिक करें।
  6. आवश्यकतानुसार परिवर्तन करें। (नीचे देखें)
  7. सहेजें पर क्लिक करें।

उपयोगकर्ता जानकारी बदलें

व्यवस्थापक उपयोगकर्ता जानकारी बदल सकते हैं।

  1. उपयोगकर्ता प्रकार (उपयोगकर्ता या प्रबंधक) सेट करें।
  2. उपयोगकर्ता स्थिति (अक्षम करने के लिए सक्रिय) सेट करें।
  3. एक अद्वितीय टैग आईडी दर्ज करें। (वैकल्पिक)
  4. सहेजें पर क्लिक करें।
