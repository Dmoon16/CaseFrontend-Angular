नीचे CSV फ़ाइलों में डेटा फ़ील्ड की सूची दी गई है। प्रत्येक आयात विकल्प में फ़ील्ड्स का भिन्न संयोजन होता है।

मामले आईडी

  • मामला उपयोगकर्ता या टैग आईडी दर्ज करें।
  • केस पेज > एडिट > टैग आईडी पर जाएं या केस पेज पर जाएं > एडिट > url में केस आईडी।

उपयोगकर्ता पहचान

  • उपयोगकर्ता की आईडी या टैग आईडी दर्ज करें।
  • उपयोगकर्ता पृष्ठ पर जाएं > संपादित करें > टैग आईडी या उपयोगकर्ता पृष्ठ पर जाएं > संपादित करें > url में उपयोगकर्ता आईडी।
