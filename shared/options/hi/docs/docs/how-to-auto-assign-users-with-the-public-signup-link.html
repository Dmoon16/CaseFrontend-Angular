सार्वजनिक पंजीकरण लिंक के साथ उपयोगकर्ताओं को स्वत: असाइन कैसे करें

व्यवस्थापक सिस्टम सेटिंग्स को कॉन्फ़िगर करना सीखते हैं ताकि नए उपयोगकर्ताओं को स्वचालित रूप से एक नए ऑटो निर्मित मामले में असाइन किया जा सके।

सेटअप पेज खोल रहा हूँ

पहला कदम सिस्टम सेटअप पेज को खोलना है।

  1. व्यवस्थापक खाते में लॉग इन करें।
  2. ऐप पेज पर ऐप ढूंढें और एडमिन बटन पर क्लिक करें।
  3. सिस्टम टैब पर क्लिक करें और फिर सबमेनू पर सिस्टम सेटअप पर क्लिक करें.
  4. USER SIGNUP शीर्षक तक नीचे स्क्रॉल करें।

सिस्टम सेटिंग्स को कॉन्फ़िगर करना

  1. साइनअप की अनुमति दें और मामला स्वत: बनाएं ड्रॉप-डाउन मेनू को चालू पर सेट करें.
  2. भूमिका मेनू पर क्लिक करें और एक विकल्प चुनें (डिफ़ॉल्ट रूप से अनुशंसित है)।
  3. पब्लिक यूजर साइनअप लिंक को कॉपी और शेयर करें।

लिंक भेज रहा हूँ

प्रत्येक उपयोगकर्ता जो लिंक पर क्लिक करता है और साइनअप पूरा करता है, प्लेटफ़ॉर्म को मामले पृष्ठ पर एक नया मामला उत्पन्न करने के लिए ट्रिगर करता है और उस मामले में उपयोगकर्ता को स्वतः असाइन करता है।
