د موجوده قضیو سره د کارونکي دندې ګمارلو یا لرې کولو څرنګوالی

دا مقاله مدیران ښیې چې څنګه د قضیې کاروونکي ګمارل او لرې کړي. موږ تاسو ته دا هم ښیو چې څنګه د کارونکي ډیټا بدل کړئ.

د موجوده قضیه پرانستل

  1. د اډمین حساب ته ننوځئ.
  2. د ایپس پاڼې کې یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. د قضیې مینو ټب باندې کلیک وکړئ.
  4. ډاډ ترلاسه کړئ چې فعال په فرعي مینو کې غوره شوی.
  5. د عمل کالم ته ښکته شئ.

ګمارل یا لرې کول

د قضیې لپاره یو کاروونکي وټاکئ یا د قضیې څخه کاروونکي لرې کولو لپاره کاروونکي وټاکئ کلیک وکړئ (مګر پلیټ فارم نه). په بله پاڼه کې: ADD کلیک وکړئ او د کار جریان بشپړ کړئ. د کاروونکي حذف کولو لپاره لیرې کلیک وکړئ.

img

تعدیل

دا ممکنه نده چې د قضیې په پاڼه کې د کاروونکي ډاټا ایډیټ یا تازه کړئ. پرځای یې، دا ګامونه تعقیب کړئ:

  1. د کاروونکو مینو ټب باندې کلیک وکړئ.
  2. د کارونکي نوم ومومئ او په ایډیټ کلیک وکړئ.
  3. بدلونونه وکړئ او خوندي کړئ کلیک وکړئ.
