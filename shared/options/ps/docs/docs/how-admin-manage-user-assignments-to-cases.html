اډمین څنګه قضیو ته د کارونکي دندې اداره کوي

د کاروونکو او د قضیې دندې اداره کولو لپاره پنځه لارې شتون لري. د څرنګوالي زده کولو لپاره لینک کلیک وکړئ.

طریقه ډول کړنې جزیات
د قضیې رامینځته کولو پرمهال UI اضافه کول د اډمین تڼۍ> قضیې> نوې تڼۍ> د کاروونکو تڼۍ وټاکئ
د عامه لاسلیک کولو لینک په کارولو سره کارونکي په اتوماتيک ډول وټاکئ د کارن عمل اضافه کول د اډمین تڼۍ> ترتیبات> د سیسټم تنظیم پاڼه
په موجوده قضیه کې د UI کارول UI اضافه کول او لرې کول د اډمین تڼۍ> قضیې> کاروونکي وټاکئ
د واردولو وسیله UI اضافه کول، تعدیل کول، او لیرې کول د اډمین تڼۍ> ادغام> واردات
