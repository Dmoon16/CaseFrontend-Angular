څنګه ننوتل

خپل موجود Case Active حساب ته د ننوتلو څرنګوالی زده کړئ.

ننوځي

  1. د ننوتلو پاڼې ته لاړ شئ.
  2. د LOGIN تڼۍ کلیک وکړئ.
  3. خپل بریښنالیک دننه کړئ.
  4. په بل کلیک وکړئ.

د حساب تصدیق کول

  1. یو نوی براوزر ټب خلاص کړئ او خپل بریښنالیک حساب ته ننوځئ.
  2. شپږ عددي کوډ کاپي کړئ.
  3. د قضیې فعاله ټب ته بیرته راشئ.
  4. کوډ په ساحو کې پیسټ کړئ. یوه نوې کړکۍ به ښکاره شي.

Case Active د ایپس پاڼه ښکاره کوي.

  • یادښت پلیټ فارم د اډمین امتیازاتو کاروونکو لپاره د APP ت buttonۍ څنګ ته د اډمین تڼۍ ښیې.
