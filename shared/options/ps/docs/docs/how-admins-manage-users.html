اډمین څنګه کاروونکي اداره کوي

اډمینونه رامینځته کوي، بدلوي یا غیر فعالوي څوک چې ایپسونو او قضیو ته لاسرسی ولري. کله چې یو کارن د لومړي ځل لپاره پلیټ فارم ته ننوځي، دوی اړتیا لري د پروفایل کاري فلو بشپړ کړي.

  • د نوي کارونکي پروفایل کاري فلو په اړه د نورو معلوماتو لپاره دلته کلیک وکړئ.

د کاروونکي اداره کول

لاندې جدول د کاروونکي اداره کولو لپاره پنځه لارې ښیې. د څرنګوالي زده کولو لپاره لینک کلیک وکړئ.
