د وارداتي وسیلې سره د قضیې مدیریت ډیټا فارمیټ کول

لاندې د CSV فایلونو کې د اړینو ساحو لیست دی ترڅو د وارداتو وسیلې سره قضیې اداره کړي.

کارن نوم

  • د کارونکي بریښنالیک پته دننه کړئ.

tag_id (هغه ID چې تاسو یې وروسته حواله کولی شئ)

  • یو ځانګړی پیژندونکی (د الفانومیریک حروف وکاروئ، هایفنز منل شوي)

قضیه_حالت

  • 0 = تړل
  • 1 = فعال

نیټه_پرانستل شوه

  • میاشت ورځ کال
  • mm/dd/yyyy

په اړه

  • د قضیې یو ځانګړی توضیحات (د الفانومیریک حروف وکاروئ، هایفنز منل شوي)
