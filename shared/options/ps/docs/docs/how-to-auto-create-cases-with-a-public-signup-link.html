د عامه لاسلیک لینک سره د قضیو اتومات کولو څرنګوالی

دا مقاله مدیرانو ته ښیې چې څنګه قضیه رامینځته کړي (او په ورته وخت کې یو کارن اضافه کړئ) د یو واحد لاسلیک لینک لیږلو سره.

د لینک موندل

  1. د اډمین حساب ته ننوځئ.
  2. د ایپس پاڼې کې یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. په اډمین پینل کې، د ترتیباتو مینو ټب باندې کلیک وکړئ.
  4. ډاډ ترلاسه کړئ چې د سیسټم سیټ اپ فرعي مینو ټب غوره شوی.

د ترتیباتو ترتیب کول

د USER SIGNUP سرلیک ته لاندې سکرول کړئ.

ډاډ ترلاسه کړئ چې د لاسلیک کولو اجازه ورکولو ساحه په فعاله شوې ده. (اړین) کله چې دا فیچر فعاله شي، Case Active یو ځانګړی لینک پیدا کوي.

  1. په اتوماتيک ډول د قضیې په ساحه کې کلیک وکړئ او آن غوره کړئ. د قضیې کارونکي رول ساحه ښکاره کوي.
  2. یو رول وټاکئ. (لاندې وګورئ)

په قضیه کې د کاروونکي رول

د کاروونکي درې قضیې شتون لري.

  • پیرودونکي (سپارښتنه)
  • ډیفالټ
  • کارکوونکي

لینک لیږل

لینک د یو یا ډیرو خلکو سره شریک کړئ. هر کارن چې لینک کلیک کوي او لاسلیک بشپړوي د قضیې په پا onه کې نوې قضیه رامینځته کولو لپاره پلیټ فارم رامینځته کوي او په قضیه کې کارونکي په اوتومات ډول ګماري.
