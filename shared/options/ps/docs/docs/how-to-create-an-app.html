څنګه یو اپلیکیشن جوړ کړئ

دا مقاله نوي کاروونکو ته ښیې چې څنګه د وړیا آزموینې حساب رامینځته کړي. Case Active په اوتومات ډول د دې حساب لپاره نوي کارونکي اداري امتیازات ورکوي.

د حساب جوړول

  1. د ننوتلو پاڼې ته لاړ شئ.
  2. د پیل کولو تڼۍ کلیک وکړئ.
  3. د بریښنالیک پته دننه کړئ.
  4. بل کلیک وکړئ.

img

د حساب تصدیق کول

  1. یو نوی براوزر ټب خلاص کړئ او خپل بریښنالیک حساب ته ننوځئ.
  2. شپږ عددي کوډ کاپي کړئ.
  3. د قضیې فعاله ټب ته بیرته راشئ.
  4. کوډ په ساحو کې پیسټ کړئ. یوه نوې کړکۍ به ښکاره شي.

په نوې کړکۍ کې:

  1. اړین ځایونه ډک کړئ.
  2. د خدمت شرایط ولولئ او د موافقې لپاره بکس چیک کړئ.
  3. یو وړیا حساب جوړ کړئ کلیک وکړئ.

img

د اپلیکیشن تنظیم بشپړول

د کار فلو په راتلونکی پاڼه کې ښکاره کیږي. د خپل اپلیکیشن تنظیم کولو بشپړولو لپاره مرحلې بشپړ کړئ. تاسو کولی شئ تل د معلوماتو یا ستاسو د حساب حالت سمولو یا تازه کولو لپاره دې پاڼو ته ورشئ.

د اپلیکیشن پاڼې په اړه، د شرکت نوم او ویب پاڼې ساحې ډک کړئ (اړین) او بیا وروسته کلیک وکړئ.

img

د بلینګ معلوماتو پاڼې کې، د تادیې پلان ساحه کلیک وکړئ او اختیارونه بیاکتنه وکړئ. ډیفالټ ترتیب وړیا دی. په سپارلو کلیک وکړئ.

img

وروسته له دې چې تاسو فورمه وسپارئ، Case Active د ایپس پاڼه ښکاره کوي. د هر هغه اپلیکیشن لپاره چې تاسو یې جوړ یا تړاو لرئ، پاڼه ښیې:

  1. د اپلیکیشن نوم
  2. ستاسو ډول (د کارونکي رول)
  3. حالت

img

د ایپس پاڼه د ډیری اداري دندو لپاره د پیل ځای دی:

  1. د قضیې اړوند مسلو په اړه د خبرو اترو، اسنادو ډکولو، ویډیو چیٹ (او نور) لپاره د APP تڼۍ کلیک وکړئ.
  2. د اډمین پینل خلاصولو لپاره د ADMIN تڼۍ کلیک وکړئ. دا هغه ځای دی چې مدیران کاروونکي، قضیې، ادغام، او نور اداره کوي.
  3. د حساب بلینګ او د خبرتیا تنظیماتو اداره کولو لپاره په ellipsis کلیک وکړئ.
  • د یادښت قضیه فعال د مدیر کاروونکو لپاره د ADMIN تڼۍ ښیې. هغه کارونکي چې اداري امتیازات نلري د ADMIN تڼۍ نه ګوري (لکه څنګه چې لاندې سکرین کې ښودل شوي).
