د وارداتو وسیلې سره کاروونکي څنګه رامینځته کول یا بدلول

دا مقاله مدیران ښیې چې څنګه د وارداتو وسیلې سره کاروونکي رامینځته کړي.

د وارداتو وسیله پرانیستل

اډمین کولی شي د وارداتو وسیله وکاروي ترڅو نوي کاروونکي په پراخه کچه اپلوډ کړي او د اوسني کارونکي ریکارډونه بدل کړي.

  1. د اډمین حساب ته ننوځئ.
  2. د ایپس پاڼې کې یو اپلیکیشن ومومئ او د ADMIN تڼۍ کلیک وکړئ.
  3. په اډمین پینل کې، د ادغام ټب کلیک وکړئ او بیا په فرعي مینو کې واردات.
  4. په نوي تڼۍ کلیک وکړئ. د وارداتو اضافه کولو ډیالوګ بکس په نوي براوزر ټب کې څرګندیږي.

img

د ریکارډونو جوړول

وروسته له دې چې د وارداتو اضافه کولو ډیالوګ بکس خلاصیږي:

  1. د ډول په ساحه کې کلیک وکړئ او یو اختیار غوره کړئ (لاندې وګورئ). یو لینک ښکاره کیږي.
  2. د ډاونلوډ نمونې لینک باندې کلیک وکړئ. ستاسو وسیله ته د CSV فایل ډاونلوډ کیږي.

بیا، فایل خلاص کړئ او ریکارډونه اضافه کړئ. د CSV فایل کې د فارمیټ ډیټا کارولو څرنګوالي زده کولو لپاره دلته کلیک وکړئ.

img

ډول

کله چې تاسو د ډول په ساحه کې کلیک وکړئ، یو ډراپ-ډاون مینو ښکاره کیږي. د لویو وارداتو لپاره له دوو انتخابونو څخه یو غوره کړئ.

: exclamation: یادښت د ډراپ-ډاون مینو ډیری اختیارونه لري، مګر یوازې دوه د وارداتو وسیلې پورې تړاو لري. هر اختیار د مختلف نمونې CSV فایل لري.

کاروونکي اضافه کړئ

  • دا اختیار د کاروونکو ریکارډونو ساده لوی اپلوډ لپاره غوره کړئ.
  • د CSV فایل خلاصولو لپاره د نمونې ډاونلوډ کلیک وکړئ د کاروونکو اضافه کولو ډول لپاره د کالم سرلیکونو سره.

د کاروونکو بدلون

  • د اوسني کاروونکو لپاره د معلوماتو تازه کولو لپاره دا اختیار غوره کړئ.
  • د CSV فایل خلاصولو لپاره د نمونې ډاونلوډ لینک باندې کلیک وکړئ د بدلون کاروونکو وارداتو ډول لپاره د کالم سرلیکونو سره.

د فایل پورته کول

تاسو کولی شئ سمدلاسه فایل اپلوډ کړئ یا په وروستي نیټه کې د وارداتو اضافه کولو ډیالوګ بکس ته راستون شئ او دا مرحلې بشپړ کړئ.

  1. د وارداتو اضافه کولو ډیالوګ بکس خلاص کړئ.
  2. د CSV فایل لپاره یو ځانګړی نوم دننه کړئ.
  3. د خپل وسیله څخه فایل اپلوډ کولو لپاره په پلس (+) آیکون کلیک وکړئ.
  4. د ډول په ساحه کې کلیک وکړئ او یو اختیار غوره کړئ. (پورته وګورئ)
  5. خوندي کړئ کلیک وکړئ.

پروسس کول

په فایل او سیسټم بار کې د ریکارډونو شمیر پورې اړه لري، پلیټ فارم کولی شي د وارداتو پروسس کولو لپاره څو دقیقې وخت ونیسي.

img

کله چې واردات بشپړ شي، پلیټ فارم د مدیر د کاروونکو ډشبورډ کې د کاروونکي ریکارډونه تنظیموي (د بیلګې په توګه اضافه کول، تعدیل کول، یا غیر فعال کول).

ناکامي

که د وارداتو عملیات ناکام شي، پلیټ فارم یو ناکام پیغام بیرته راولي. په ډیری مواردو کې، د ناکامۍ لامل د معلوماتو بڼه کول دي.

img

د ناکامۍ علت معلومولو لپاره:

  1. د عمل سرلیک لاندې لید کلیک وکړئ.
  2. په بل مخ کې، موږک په ناکامه نښه کې واچوئ. وسیلې ټایپ خطا ښیې.
  3. د CVS فایل ته لاړ شئ او غلطي تنظیم او حل کړئ.
  4. فایل بیا پورته کړئ.
