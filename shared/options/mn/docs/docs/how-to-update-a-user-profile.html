Хэрэглэгчийн профайлыг хэрхэн шинэчлэх вэ

Энэ нийтлэл нь хэрэглэгчдэд өөрсдийн профайл хуудасны мэдээллийг хэрхэн шинэчлэхийг харуулж байна.

  • Хэрэглэгчид өөрсдийн профайлаа шинэчлэх боломжтой.
  • Админууд ямар ч хэрэглэгчийн профайл хуудсыг шинэчлэх боломжтой.

Хэрэглэгчдэд зориулсан

Эхний алхам бол Профайл хуудсыг нээх явдал юм. Хэрэглэгчид болон админууд өөрсдийн профайл хуудсыг шинэчлэхийн тулд эдгээр алхмуудыг дагана уу.

  1. Case Active данс руугаа нэвтэрнэ үү.
  2. Хуудасны дээд хэсэгт байрлах Профайл таб дээр дарна уу.
  3. Өөрчлөлт хийх.
  4. Хадгалах товчийг дарна уу.

Ихэнх талбарууд нь сонголттой бөгөөд зарим талбарууд заавал байх ёстой. Шаардлагатай талбарууд нь:

  • Нэр
  • Овог нэр
  • Имэйл хаяг
  • Хэл
  • Цагийн бүс

Админуудын хувьд

Админууд бусад хэрэглэгчдийн профайлын мэдээллийг шинэчлэх боломжтой. Үүнд HOST USER INFO гарчигтай гурван талбар орно.

  1. Case Active админ бүртгэлдээ нэвтэрнэ үү.
  2. Програмаа олоод ADMIN товчийг дарна уу.
  3. Дээд цэсний Хэрэглэгчид дээр дарна уу.
  4. Дэд цэс рүү орж, нэвтрэхийг хүсч буй хэрэглэгчийн жагсаалтаа сонгоно уу (ИДЭВХТЭЙ эсвэл ИДЭВХГҮЙ). Өгөгдмөл тохиргоо нь Идэвхтэй.
  5. Жагсаалтаас хэрэглэгчийг олоод "Засварлах" дээр дарна уу.
  6. Шаардлагатай өөрчлөлтүүдийг хийнэ үү. (доороос үзнэ үү)
  7. Хадгалах товчийг дарна уу.

Хэрэглэгчийн мэдээллийг өөрчлөх

Админууд хэрэглэгчийн мэдээллийг өөрчлөх боломжтой.

  1. Хэрэглэгчийн төрлийг (Хэрэглэгч эсвэл Менежер) тохируулна уу.
  2. Хэрэглэгчийн статусыг тохируулна уу (Идэвхтэй хүртэл идэвхгүй).
  3. Өвөрмөц Tag ID оруулна уу. (заавал биш)
  4. Хадгалах товчийг дарна уу.
