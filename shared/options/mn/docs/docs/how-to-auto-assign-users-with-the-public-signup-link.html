Нийтийн бүртгэлийн холбоосоор хэрэглэгчдийг хэрхэн автоматаар томилох вэ

Админууд системийн тохиргоог хэрхэн тохируулах талаар суралцдаг бөгөөд ингэснээр шинэ хэрэглэгчдийг автоматаар шинээр үүсгэсэн тохиолдолд автоматаар хуваарилдаг.

Тохиргооны хуудсыг нээж байна

Эхний алхам бол Системийн тохиргооны хуудсыг нээх явдал юм.

  1. Админ данс руу нэвтэрнэ үү.
  2. Програмын хуудаснаас програмаа олоод АДМИН товчийг дарна уу.
  3. Системийн таб дээр дараад дэд цэсэн дэх СИСТЕМ ТОХИРУУЛАЛТ дээр дарна уу.
  4. ХЭРЭГЛЭГЧИЙН БҮРТГЭЛИЙН гарчиг руу доош гүйлгэнэ үү.

Системийн тохиргоог хийж байна

  1. Бүртгүүлэхийг зөвшөөрөх, Тохиолдолд автоматаар үүсгэх унждаг цэсийг Асаалтай болгож тохируулна уу.
  2. Үүргийн цэсийг товшоод сонголтоо сонгоно уу (анхдагчаар сонгохыг зөвлөж байна).
  3. Нийтийн хэрэглэгчийн бүртгүүлэх холбоосыг хуулж, хуваалцаарай.

Холбоосыг илгээж байна

Холбоос дээр дарж, бүртгэлээ дуусгасан хэрэглэгч бүр Тохиолдлын хуудсан дээр шинэ тохиолдол үүсгэх платформыг идэвхжүүлж, тухайн тохиолдолд хэрэглэгчийг автоматаар оноодог.
