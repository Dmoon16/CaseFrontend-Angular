Кейс үүсгэх үед хэрэглэгчдийг хэрхэн томилох вэ

Энэ нийтлэл нь админуудад шинэ кейс үүсгэх үед хэрэглэгчийг хэрхэн томилохыг харуулж байна. Хэрэв хэрэглэгч платформ дээр бүртгүүлээгүй байгаа бол шинэ тохиолдол үүсгэхийн зэрэгцээ шинэ хэрэглэгч нэмэх боломжтой.

Кейс үүсгэх

  1. Админ данс руу нэвтэрнэ үү.
  2. Програмын хуудаснаас програмаа олоод ADMIN товчийг дарна уу.
  3. Cases цэсийн таб дээр дарна уу.
  4. ШИНЭ товчийг дарна уу. Шинэ цонх гарч ирнэ.

Хэрэглэгчийг томилж байна

Одоогийн хэрэглэгч

Одоогийн хэрэглэгчийг шинэ тохиолдолд оноохын тулд эдгээр алхмуудыг дагана уу.

ХЭРЭГЛЭГЧДИЙГ ОЛГОХ товчийг дараад талбаруудыг бөглөнө үү.

  1. Хэрэглэгчийн унждаг цэсийг товшоод танигч (жишээ нь имэйл) сонгоно уу.
  2. Хоосон талбарт утгыг оруулна уу (жишээ нь, имэйл хаяг).
  3. Хэрэв хэрэглэгчийн бүртгэл платформд байгаа бол талбар нь нэрийг автоматаар санал болгоно.
  4. Зөв хэрэглэгчийг сонго.
  5. Хэрэглэгчийн үүргийг сонгоно уу (үйлчлүүлэгч, үндсэн эсвэл ажилтнууд).
  6. Сонголтоор хэрэглэгчдэд имэйл мэдэгдэл илгээхийн тулд хайрцгийг чагтална уу.
  7. SAVE дээр дарна уу.

Case Active нь хэрэглэгчийг кейст нэмнэ. Одоо админууд Хадгалах товчийг дарж цонхыг хаах, шинэ хэрэглэгч нэмэх эсвэл хэрэглэгчийг устгах боломжтой.

Шинэ хэрэглэгч нэмж байна

Хэрэв платформ одоогийн бичлэгийг олж чадахгүй бол товчлуур дээр дарж шинэ хэрэглэгч нэмнэ үү.

Талбаруудыг бөглөж, дараа нь SAVE дээр дарж хэрэглэгчийг платформд нэмнэ. Дараа нь хэрэглэгчийн дүрийг тохируулаад SAVE товчийг дахин дарж хэрэглэгчийг кейст онооно. Эцэст нь, хэргийг хадгалахын тулд SAVE дээр дахин нэг удаа дарна уу.
