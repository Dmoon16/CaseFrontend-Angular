Импортын хэрэгслээр хэрэглэгчийн даалгаврыг хэрхэн оноох, өөрчлөх, устгах

Энэ нийтлэл нь админуудад Импортын хэрэгслээр кейс хэрэглэгчдийг хэрхэн удирдахыг харуулж байна.

Админууд:

  • хэрэглэгчдийг кейст хуваарилах
  • кейст аль хэдийн томилогдсон хэрэглэгчдийг өөрчлөх
  • хэрэгт томилогдсон хэрэглэгчдийг устгах

Импортын хэрэгслийг нээж байна

  1. Админ данс руу нэвтэрнэ үү.
  2. Програмын хуудаснаас програмаа олоод ADMIN товчийг дарна уу.
  3. Интеграцийн таб дээр дарж, дэд цэсэн дэх IMPORTS дээр дарна уу.
  4. ШИНЭ товчийг дарна уу. Импорт нэмэх харилцах цонх гарч ирнэ.

Бичлэг үүсгэх

Нэг буюу хэд хэдэн CSV файлыг татаж авах. Кейс хэрэглэгчдийг нэмэх, өөрчлөх, хасахын тулд эдгээр файлыг ашиглана уу. Импорт нэмэх харилцах цонх нээгдсэний дараа:

  1. Төрөл талбар дээр дарна уу.
  2. Унждаг цэсээр гүйлгээд сонголтыг сонгоно уу (доороос үзнэ үү). Холбоос гарч ирнэ.
  3. Жишээ татаж авах холбоос дээр дарна уу. CSV файлыг таны төхөөрөмжид татаж авдаг.

Файлыг нээж, бичлэг нэмнэ үү. CSV файл дахь форматын өгөгдлийг хэрхэн ашиглах талаар энд дарж үзнэ үү.

Файлын сонголтууд

Хэрэглэгчдийг кейсүүдэд оноох * Кейс рүү хэрэглэгч нэмэхийн тулд энэ сонголтыг сонгоно уу. * Case_role_order нүдийг хоосон орхи. * Хэрэглэгчийн хуудас руу орж, энэ хүний Тагийн ID-г хуулна уу. Хэрэв Tag ID талбар хоосон байвал өвөрмөц ID-г оруулаад Хадгалах дээр дарж ID-г CSV файл руу хуулна.

Кейсүүдэд томилогдсон хэрэглэгчдийг өөрчлөх * Энэ сонголтыг кейст томилогдсон одоогийн хэрэглэгчдийн өгөгдлийг шинэчлэх. * Case_ID баганад Case Tag ID-г ашиглана уу.

Кейсүүдэд томилогдсон хэрэглэгчдийг устгах * Кейст томилогдсон хэрэглэгчдийг хасахын тулд энэ сонголтыг сонгоно уу.

Файлыг байршуулж байна

Та файлаа шууд байршуулах эсвэл дараа нь Импорт нэмэх харилцах цонх руу буцаж очоод эдгээр алхмуудыг хийж болно.

  1. Импорт нэмэх харилцах цонхыг нээнэ үү.
  2. CSV файлд өвөрмөц нэр оруулна уу.
  3. Файлыг төхөөрөмжөөсөө байршуулахын тулд нэмэх (+) дүрс дээр дарна уу.
  4. Төрөл талбар дээр товшоод сонголтоо сонгоно уу. (дээрээс үзнэ үү)
  5. SAVE дээр дарна уу.

Боловсруулж байна

Файл дахь бичлэгийн тоо болон системийн ачааллаас хамааран платформ импортыг боловсруулахад хэдэн минут зарцуулагдана.

Импорт хийж дуусахад платформ нь Тохиолдлын хуудсан дээрх хэрэглэгчийн бүртгэлийг (өөрөөр хэлбэл нэмэх, өөрчлөх, устгах) тохируулдаг.

Бүтэлгүйтэл

Хэрэв импорт хийх ажиллагаа амжилтгүй болвол платформ нь Failed мессежийг буцаана. Ихэнх тохиолдолд бүтэлгүйтлийн шалтгаан нь өгөгдлийг форматлах явдал юм.

img

Алдаа дутагдлын шалтгааныг олж мэдэхийн тулд:

  1. Үйлдлийн гарчиг дор Харах дээр дарна уу.
  2. Дараагийн хуудсанд хулганаа Failed дүрс дээр аваачина. Зөвлөмж нь алдааг харуулна.
  3. CVS файл руу очоод алдаагаа засч залруулна уу.
  4. Файлыг дахин байршуулна уу.
