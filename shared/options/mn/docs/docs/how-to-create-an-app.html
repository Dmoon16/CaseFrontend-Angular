Хэрхэн програм үүсгэх вэ

Энэ нийтлэл нь шинэ хэрэглэгчдэд хэрхэн үнэгүй туршилтын данс үүсгэхийг харуулж байна. Case Active нь шинэ хэрэглэгчдэд энэ акаунтын админ эрхийг автоматаар олгодог.

Бүртгэл үүсгэж байна

  1. Нэвтрэх хуудас руу очно уу.
  2. Эхлэх товчийг дарна уу.
  3. Имэйл хаяг оруулна уу.
  4. Дараах товчийг дарна уу.

Бүртгэлийг баталгаажуулж байна

  1. Хөтчийн шинэ таб нээгээд имэйл хаяг руугаа нэвтэрнэ үү.
  2. Зургаан оронтой кодыг хуулна уу.
  3. Case Active таб руу буцах.
  4. Талбаруудад кодыг буулгана уу. Шинэ цонх гарч ирнэ.

Шинэ цонхонд:

  1. Шаардлагатай талбаруудыг бөглөнө үү.
  2. Үйлчилгээний нөхцөлийг уншаад зөвшөөрч байгаа нүдийг чагтална уу.
  3. ҮНЭГҮЙ БҮРТГЭЛ үүсгэх дээр дарна уу.

Аппын тохиргоог хийж байна

Дараагийн хуудсан дээр ажлын урсгал гарч ирнэ. Програмаа тохируулж дуусгахын тулд алхмуудыг гүйцээнэ үү. Та эдгээр хуудсууд руу буцаж орж мэдээлэл эсвэл бүртгэлийн статусаа засах, шинэчлэх боломжтой.

Аппликэйшний тухай хуудсан дээр компанийн нэр, вэбсайтын талбаруудыг (шаардлагатай) бөглөж, Дараа нь дарна уу.

Төлбөрийн мэдээллийн хуудасны Төлбөрийн төлөвлөгөө талбарыг товшоод сонголтуудыг шалгана уу. Үндсэн тохиргоо нь Үнэгүй. SUBMIT дээр дарна уу.

Маягтыг илгээсний дараа Case Active програмын хуудсыг харуулна. Таны үүсгэсэн эсвэл харьяалагддаг програм бүрийн хувьд хуудас нь:

  1. Програмын нэр
  2. Таны төрөл (хэрэглэгчийн үүрэг)
  3. Статус

img

Апп хуудас нь олон админ функцийг эхлүүлэх цэг юм:

  1. Хэрэгтэй холбоотой асуудлаар харилцах, бичиг баримт бөглөх, видео чат (болон бусад) хийх бол APP товчийг дарна уу.
  2. ADMIN товчийг дарж админ самбарыг нээнэ үү. Энд админууд хэрэглэгчид, кейс, интеграцчлал болон бусад зүйлийг удирддаг.
  3. Дансны тооцоо болон мэдэгдлийн тохиргоог удирдахын тулд зуйван дээр дарна уу.
  • Note Case Active нь админ хэрэглэгчдэд зориулсан ADMIN товчийг харуулдаг. Админы эрхгүй хэрэглэгчид ADMIN товчийг харахгүй (доорх дэлгэцэн дээр харуулсны дагуу).
