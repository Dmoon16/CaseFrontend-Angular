Импортын хэрэгслээр хэрэглэгчийн удирдлагын өгөгдлийг форматлах

CSV файл дахь өгөгдлийн талбаруудын жагсаалтыг доор харуулав. Импортын сонголт бүр талбаруудын өөр хослолтой.

хэрэглэгчийн нэр

  • Хэрэглэгчийн имэйл хаягийг оруулна уу.

өөрийн нэр

  • Хэрэглэгчийн нэрийг оруулна уу.

овог

  • Хэрэглэгчийн овог нэрийг оруулна уу.

нутаг дэвсгэр

  • Энэ багана нь хэрэглэгчийн сонгосон хэлийг шалгадаг.
  • Англиар бол en, Испанид sp гэж оруулна уу.

бүсийн мэдээлэл

  • Энэ нь хэрэглэгчийн цагийн бүсийг тохируулдаг. Тив, хотыг хоосон зайгүй оруулна уу (доогуур зураас ашиглана уу).
  • Профайл > ХЭЛ, БҮС > Цагийн бүс унадаг цэсийн сонголтыг ашиглах ёстой.
  • Хамгийн тохиромжтой цагийн бүсийг ашигла. Жишээлбэл, Бостон унадаг цонхонд байхгүй тул бид Нью-Йоркийг (эсвэл EST/EDT цагийн бүсийн аль ч хотыг) ашиглаж болно.

host_tag_id

  • Байнгын шошго оруулна уу.
  • az, 0-9, -, / зөвшөөрнө.

case_status

  • 0 = хаалттай
  • 1 = идэвхтэй
