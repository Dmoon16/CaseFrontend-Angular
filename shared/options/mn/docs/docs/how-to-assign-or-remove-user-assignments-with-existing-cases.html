Хэрэглэгчийн даалгаврыг одоо байгаа тохиолдлуудад хэрхэн оноох, устгах вэ

Энэ нийтлэл нь админуудад тохиолдлын хэрэглэгчдийг хэрхэн томилж, устгахыг харуулж байна. Мөн бид танд хэрэглэгчийн өгөгдлийг хэрхэн өөрчлөхийг харуулж байна.

Одоо байгаа хэргийг нээх

  1. Админ данс руу нэвтэрнэ үү.
  2. Програмын хуудаснаас програмаа олоод АДМИН товчийг дарна уу.
  3. Cases цэсийн таб дээр дарна уу.
  4. Дэд цэсэнд ACTIVE-г сонгосон эсэхийг шалгана уу.
  5. ҮЙЛ АЖИЛЛАГАА багана руу доош гүйлгэнэ үү.

Даалгах эсвэл хасах

Хэрэглэгчийг оноох дээр дарж хэрэглэгчийг кейст оноох эсвэл тухайн хэрэглэгчийг кейсээс хасах (гэхдээ платформоос биш). Дараагийн хуудсанд: ADD дээр товшоод ажлын явцыг дуусгана уу. Устгах дээр дарж хэрэглэгчийг устгана уу.

img

Өөрчлөх

Тохиолдлын хуудсан дээрх хэрэглэгчийн өгөгдлийг засах, шинэчлэх боломжгүй. Үүний оронд дараах алхмуудыг дагана уу.

  1. Хэрэглэгчийн цэсийн таб дээр дарна уу.
  2. Хэрэглэгчийн нэрийг олоод Засварлах дээр дарна уу.
  3. Өөрчлөлтүүдийг хийгээд SAVE дээр дарна уу.
