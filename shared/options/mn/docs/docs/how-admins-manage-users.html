Админууд хэрэглэгчдийг хэрхэн удирддаг вэ

Админууд апп болон хэрэглүүрт хандах боломжтой хүмүүсийг үүсгэх, өөрчлөх эсвэл идэвхгүй болгох. Хэрэглэгч анх удаа платформд нэвтрэх үед профайлын ажлын урсгалыг бөглөх шаардлагатай.

  • Хэрэглэгчийн шинэ профайлын ажлын явцын талаар илүү ихийг мэдэхийг хүсвэл энд дарна уу.

Хэрэглэгчийг удирдах

Доорх хүснэгтэд хэрэглэгчийг удирдах таван аргыг харуулав. Хэрхэн гэдгийг мэдэхийн тулд холбоос дээр дарна уу.

Арга Төрөл Үйлдлүүд Дэлгэрэнгүй мэдээлэл
UI ашиглаж байна UI нэмэх, өөрчлөх Админ товч > Хэрэглэгчид
Кейс үүсгэж, хэрэглэгчийг кейс рүү оноож байхдаа UI нэмэх Админ товч > Кейс > Шинэ товч > Кейс үүсгэх хуудас
Нийтийн бүртгүүлэх холбоосыг ашиглан хэрэглэгчийг автоматаар үүсгэнэ Хэрэглэгчийн үйлдэл нэмэх Админ товч > Тохиргоо > Системийн тохиргооны хуудас
Импортын хэрэгсэл UI нэмэх, өөрчлөх Админ товч > Интеграци > Импорт
API ашиглах (удалгүй гарах) API нэмэх, өөрчлөх Үгүй

Мэдэгдэл

Уг платформ нь хэрэглэгчдэд апп-д нэмэгдсэн үед мэдэгдэл илгээдэггүй. Гэсэн хэдий ч админууд хэргийн даалгаврын мэдэгдлийг идэвхжүүлж болно.
