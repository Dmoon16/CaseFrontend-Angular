Импортын хэрэглүүр ашиглан хэрэглэгчийн кейсийн даалгаврын өгөгдлийг форматлах

CSV файл дахь өгөгдлийн талбаруудын жагсаалтыг доор харуулав. Импортын сонголт бүр талбаруудын өөр хослолтой.

тохиолдол_ID

  • Кейс Хэрэглэгч эсвэл Тагийн ID-г оруулна уу.
  • Тохиолдолын хуудас > Засварлах > Тагийн ID руу очих ЭСВЭЛ Тохиолдлын хуудас > Засварлах > Тохиолдлын ID руу очно уу.

хэрэглэгчийн_ID

  • Хэрэглэгчийн ID эсвэл Tag ID-г оруулна уу.
  • Хэрэглэгчийн хуудас > Засварлах > Шошгоны ID руу очих ЭСВЭЛ Хэрэглэгчийн хуудас > Засварлах > Хэрэглэгчийн ID руу очно уу.

хэргийн_дүрийн_захиалга

  • Нүдийг хоосон орхи.

тохиолдлын_үүргийн_ID

  • Оруулна уу: үйлчлүүлэгч, ажилтнууд эсвэл анхдагч.

тохиолдолд_мэдэх

  • 0 = үгүй
  • 1 = Тийм ээ, хэрэглэгчдэд имэйл мэдэгдэл илгээнэ үү.
