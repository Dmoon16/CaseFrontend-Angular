Oblikovanje podatkov za upravljanje primerov z orodjem za uvoz

Spodaj je seznam zahtevanih polj v datotekah CSV za upravljanje primerov z orodjem za uvoz.

uporabniško ime

  • Vnesite e-poštni naslov uporabnika.

tag_id (ID, na katerega se lahko sklicujete pozneje)

  • edinstven identifikator (uporabite alfanumerične znake, dovoljeni so vezaji)

status_primera

  • 0 = zaprto
  • 1 = aktivno

datum_odprtja

  • mesec/dan/leto
  • mm/dd/llll

približno

  • edinstven opis primera (uporabite alfanumerične znake, dovoljeni so vezaji)
