Como os administradores gerenciam atribuições de usuários a casos

Há cinco maneiras de gerenciar usuários e designações de caso. Clique em um link para saber como.

Método Modelo Ações Detalhes
Ao criar um caso IU adicionar Botão Admin > Casos > botão Novo > botão Atribuir usuários
Atribuir automaticamente o usuário usando o link de inscrição público Ação do usuário adicionar Botão Admin > Configurações > página Configuração do sistema
Usando a interface do usuário no caso existente IU adicionar e remover Botão Admin > Casos > Atribuir usuários
ferramenta de importação IU adicionar, modificar e remover Botão Admin > Integrações > Importações
Usando API (em breve) API adicionar, modificar e remover N / D
