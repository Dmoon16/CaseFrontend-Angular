Como atribuir usuários ao criar um caso

Este artigo mostra aos administradores como atribuir um usuário ao criar um novo caso. Caso o usuário ainda não esteja cadastrado na plataforma, você pode criar adicionar um novo usuário ao criar um novo caso.

Criando um caso

  1. Faça login na conta de administrador.
  2. Localize um aplicativo na página Aplicativos e clique no botão ADMIN.
  3. Clique na guia do menu Casos.
  4. Clique no botão NOVO. Uma nova janela é exibida.
  • Preencha os campos para criar um caso e role para baixo até o título Usuários atribuídos.

Atribuindo um usuário

Usuário atual

Siga estas etapas para atribuir um usuário atual ao novo caso.

Clique no botão ATRIBUIR USUÁRIOS e preencha os campos.

  1. Clique no menu suspenso Usuário e selecione um identificador (por exemplo, e-mail).
  2. Digite um valor no campo em branco (por exemplo, endereço de e-mail).
  3. Se o registro do usuário existir na plataforma, o campo sugere automaticamente um nome.
  4. Selecione o usuário correto.
  5. Selecione uma função de usuário (clientes, padrão ou equipe).
  6. Opcionalmente, marque a caixa para enviar ao usuário uma notificação por e-mail.
  7. Clique em SALVAR.

Case Active adiciona o usuário ao caso. Agora os administradores podem clicar em Salvar para fechar a janela, adicionar um novo usuário ou remover um usuário.

Adicionando um novo usuário

Se a plataforma não encontrar um registro atual, clique no botão para adicionar um novo usuário.

Preencha os campos e clique em SALVAR para adicionar o usuário à plataforma. Em seguida, defina uma função de usuário e clique em SALVAR novamente para atribuir o usuário ao caso. Por fim, clique em SAVE mais uma vez para salvar o caso.
