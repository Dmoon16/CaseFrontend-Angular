Como criar usuários com ferramenta de importação

Este artigo mostra aos administradores como criar usuários com a ferramenta Importar.

Abrindo a ferramenta Importar

Os administradores podem usar a ferramenta de importação para fazer upload em massa de novos usuários e modificar registros de usuários existentes.

  1. Faça login na conta de administrador.
  2. Localize um aplicativo na página Aplicativos e clique no botão ADMIN.
  3. No painel Admin, clique na guia Integração e depois em IMPORTAÇÕES no submenu.
  4. Clique no botão NOVO. A caixa de diálogo Adicionar importações é exibida em uma nova guia do navegador.

Criando registros

Depois que a caixa de diálogo Adicionar importações for aberta:

  1. Clique no campo Tipo e selecione uma opção (veja abaixo). Um link é exibido.
  2. Clique no link Baixar amostra . Um arquivo CSV é baixado para o seu dispositivo.

Em seguida, abra o arquivo e adicione os registros. Clique aqui para saber como usar dados de formato no arquivo CSV.

Modelo

Quando você clica no campo Tipo, um menu suspenso é exibido. Escolha uma das duas opções para importações em massa.

:exclamation: NOTA O menu suspenso tem muitas opções, mas apenas duas estão relacionadas à ferramenta de importação. Cada opção tem um arquivo CSV de amostra diferente.

Adicionar usuários

  • Escolha esta opção para um simples upload em massa dos registros do usuário.
  • Clique no link Download Sample para abrir um arquivo CSV com cabeçalhos de coluna para o tipo Add users Import.

Modificar usuários

  • Escolha esta opção para atualizar os dados dos usuários atuais.
  • Clique no link Baixar amostra para abrir um arquivo CSV com cabeçalhos de coluna para o tipo Modificar usuários Importar.

Carregando o arquivo

Você pode carregar um arquivo imediatamente ou retornar à caixa de diálogo Adicionar importações posteriormente e concluir essas etapas.

  1. Abra a caixa de diálogo Adicionar importações.
  2. Insira um nome exclusivo para o arquivo CSV.
  3. Clique no ícone de mais (+) para carregar o arquivo do seu dispositivo.
  4. Clique no campo Tipo e selecione uma opção. (Veja acima)
  5. Clique em SALVAR.

Em processamento

Dependendo da quantidade de registros do arquivo e da carga do sistema, a plataforma pode demorar alguns minutos para processar a importação.

Quando a importação é concluída, a plataforma ajusta os registros do usuário (ou seja, adiciona, modifica ou desativa) no painel de usuários do administrador.

Falha

Se a operação de importação falhar, a plataforma retornará uma mensagem de falha. Na maioria dos casos, a causa da falha é a formatação dos dados.

Para ver a causa da falha:

  1. Clique em Exibir sob o cabeçalho Ação.
  2. Na próxima página, passe o mouse sobre o ícone Failed. A dica de ferramenta exibe o erro.
  3. Vá para o arquivo CVS e ajuste e corrija o erro.
  4. Carregue o arquivo novamente.
