Como atribuir usuários automaticamente com o link de inscrição público

Os administradores aprendem como definir as configurações do sistema para que novos usuários sejam atribuídos automaticamente a um caso recém-criado automaticamente.

Abrindo a página de configuração

A primeira etapa é abrir a página Configuração do sistema.

  1. Faça login na conta de administrador.
  2. Localize um aplicativo na página Aplicativos e clique no botão ADMIN.
  3. Clique na guia Sistemas e depois em CONFIGURAÇÃO DO SISTEMA no submenu.
  4. Role para baixo até o título USER SIGNUP.

Configurando as configurações do sistema

  1. Defina os menus suspensos Permitir inscrição e Criar caso automaticamente como Ativado.
  2. Clique no menu de função e escolha uma opção (o padrão é recomendado).
  3. Copie e compartilhe o link Inscrição de usuário público.

Enviando o link

Cada usuário que clica no link e conclui a inscrição aciona a plataforma para gerar um novo caso na página Casos e atribui automaticamente o usuário nesse caso.
