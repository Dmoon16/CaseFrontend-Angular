Como criar casos automaticamente com um link de inscrição público

Este artigo mostra aos administradores como criar um caso (e adicionar um usuário ao mesmo tempo) enviando um único link de inscrição.

Localizando o link

  1. Faça login na conta de administrador.
  2. Localize um aplicativo na página Aplicativos e clique no botão ADMIN.
  3. No painel do administrador, clique na guia do menu Configurações.
  4. Certifique-se de que a guia do submenu SYSTEM SETUP esteja selecionada.

img

Definindo configurações

Role para baixo até o título USER SIGNUP.

Certifique-se de que o campo Permitir inscrição esteja definido como Ativado. (obrigatório) Quando esse recurso está ativado, Case Active gera um link exclusivo.

  1. Clique no campo Auto Create Case e selecione On. O campo Função do usuário do caso é exibido.
  2. Selecione uma função. (Veja abaixo)

img

Funções do usuário do caso

Há três funções de usuário de caso.

  • clientes (recomendado)
  • predefinição
  • funcionários

Enviando o link

Compartilhe o link com uma ou mais pessoas. Cada usuário que clica no link e conclui a inscrição aciona a plataforma para gerar um novo caso na página de casos e atribuir automaticamente o usuário ao caso.
