Como atribuir ou remover atribuições de usuário com casos existentes

Este artigo mostra aos administradores como atribuir e remover usuários do caso. Também mostramos como modificar os dados do usuário.

Abrindo um caso existente

  1. Faça login na conta de administrador.
  2. Localize um aplicativo na página Aplicativos e clique no botão ADMIN.
  3. Clique na guia do menu Casos .
  4. Certifique-se de que ACTIVE esteja selecionado no submenu.
  5. Role para baixo até a coluna AÇÃO.

Atribuir ou remover

Clique em Atribuir usuários para atribuir um usuário ao caso ou remover o usuário do caso (mas não da plataforma). Na página seguinte: Clique em ADICIONAR e conclua o fluxo de trabalho. Clique em remover para excluir um usuário.

Modificar

Não é possível editar ou atualizar os dados do usuário na página Casos. Em vez disso, siga estas etapas:

  1. Clique na guia do menu Usuários .
  2. Localize o nome de usuário e clique em Editar .
  3. Faça as alterações e clique em SALVAR.
