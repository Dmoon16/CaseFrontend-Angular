Como criar ou modificar casos com a ferramenta de importação

Este artigo mostra aos administradores como criar ou modificar casos existentes com a ferramenta Importar.

Abrindo a ferramenta Importar

  1. Faça login na conta de administrador.
  2. Localize um aplicativo na página Aplicativos e clique no botão ADMIN.
  3. No painel Admin, clique na guia Integração e depois em IMPORTAÇÕES no submenu.
  4. Clique no botão NOVO. A caixa de diálogo Adicionar importações é exibida em uma nova guia do navegador.

img

Criando casos

Depois que a caixa de diálogo Adicionar importações for aberta:

  1. Clique no campo Tipo e selecione uma opção (veja abaixo). Um link é exibido.
  2. Clique no link Baixar amostra. Um arquivo CSV é baixado para o seu dispositivo.

Em seguida, abra o arquivo e adicione casos.

Modelo

Quando você clica no campo Tipo , um menu suspenso é exibido. Escolha uma das duas opções para importar caixas em massa.

  • Observação O menu suspenso tem muitas opções, mas apenas duas estão relacionadas ao gerenciamento de casos. Cada opção tem um arquivo CSV de amostra diferente.

Adicionar casos

  • Escolha esta opção para um simples upload em massa de casos.
  • Clique no link Baixar amostra para abrir um arquivo CSV com cabeçalhos de coluna para o tipo de importação Criar casos.

Modificar casos

  • Escolha esta opção para atualizar os casos atuais.
  • Clique no link Baixar amostra para abrir um arquivo CSV com cabeçalhos de coluna para o tipo de importação Modificar casos.

Carregando o arquivo

Você pode carregar um arquivo imediatamente ou retornar à caixa de diálogo Adicionar importações posteriormente e concluir essas etapas.

  1. Abra a caixa de diálogo Adicionar importações.
  2. Insira um nome exclusivo para o arquivo CSV.
  3. Clique no ícone de mais (+) para carregar o arquivo do seu dispositivo.
  4. Clique no campo Tipo e selecione uma opção. (Veja acima)
  5. Clique em SALVAR.

img

Formatação

Estes são os campos obrigatórios e os primeiros requisitos. Clique aqui para revisar os detalhes de formatação.

tag_id (ou case_ID) um identificador único (use caracteres alfanuméricos, hífens são aceitos)

case_status 0 = fechado 1 = ativo

data_abertura mês/dia/ano mm/dd/aaaa

sobre uma descrição única do caso (use caracteres alfanuméricos, hífens são aceitos)

img

Em processamento

Dependendo da quantidade de registros do arquivo e da carga do sistema, a plataforma pode demorar alguns minutos para processar a importação.

img

Quando a importação for concluída, a plataforma ajustará os registros de caso (ou seja, adicionará ou modificará) no painel de casos do administrador.

Falha

Se a operação de importação falhar, a plataforma retornará uma mensagem de falha. Na maioria dos casos, a causa da falha é a formatação dos dados.

Para ver a causa da falha:

  1. Clique em Exibir sob o cabeçalho AÇÃO.
  2. Na próxima página, passe o mouse sobre o ícone de falha. A dica de ferramenta exibe o erro.
  3. Vá para o arquivo CVS e ajuste e corrija o erro.
  4. Carregue o arquivo novamente.
