Como criar um aplicativo

Este artigo mostra aos novos usuários como criar uma conta de avaliação gratuita. O caso ativo concede automaticamente ao novo usuário privilégios de administrador para esta conta.

Criando uma conta

  1. Vá para a página de login .
  2. Clique no botão Começar.
  3. Digite um endereço de e-mail.
  4. Clique em Avançar.

Verificando uma conta

  1. Abra uma nova guia do navegador e faça login na sua conta de e-mail.
  2. Copie o código de seis dígitos.
  3. Retorne à guia Caso Ativo.
  4. Cole o código nos campos. Uma nova janela é exibida.

Na nova janela:

  1. Preencha os campos obrigatórios.
  2. Leia os termos de serviço e marque a caixa para concordar.
  3. Clique em CRIAR UMA CONTA GRATUITA.

img

Concluindo a configuração do aplicativo

Um fluxo de trabalho é exibido na próxima página. Conclua as etapas para concluir a configuração do seu aplicativo. Você sempre pode retornar a essas páginas para editar ou atualizar informações ou o status de sua conta.

Na página Sobre o aplicativo, preencha os campos nome da empresa e site (obrigatório) e clique em Avançar.

img

Na página Informações de cobrança, clique no campo Plano de pagamento e revise as opções. A configuração padrão é Grátis. Clique em ENVIAR.

img

Depois de enviar o formulário, o Case Active exibe a página Aplicativos. Para cada aplicativo que você cria ou ao qual pertence, a página mostra:

  1. o nome do aplicativo
  2. Seu tipo (função do usuário)
  3. Status

A página Aplicativos é um ponto de partida para muitas funções administrativas:

  1. Clique no botão APP para se comunicar, preencher documentos, conversar por vídeo (e muito mais) sobre questões relacionadas ao caso.
  2. Clique no botão ADMIN para abrir o painel de administração. É aqui que os administradores gerenciam usuários, casos, integrações e muito mais.
  3. Clique nas reticências para gerenciar as configurações de cobrança e notificação da conta.
  • Observação Case Active exibe o botão ADMIN para usuários admin. Os usuários que não têm privilégios de administrador não veem o botão ADMIN (conforme mostrado na tela abaixo).
