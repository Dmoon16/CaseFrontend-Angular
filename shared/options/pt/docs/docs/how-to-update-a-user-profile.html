Como atualizar um perfil de usuário

Este artigo mostra aos usuários como atualizar informações em sua página de perfil.

  • Os usuários podem atualizar seus próprios perfis.
  • Os administradores podem atualizar a página de perfil de qualquer usuário.

Para usuários

O primeiro passo é abrir a página de perfil. Usuários e administradores seguem estas etapas para atualizar sua própria página de perfil.

  1. Faça login na sua conta Case Active.
  2. Clique na guia Perfil na parte superior da página.
  3. Faça mudanças.
  4. Clique em Salvar.

A maioria dos campos é opcional e alguns campos são obrigatórios. Os campos obrigatórios são:

  • Primeiro nome
  • Sobrenome
  • Endereço de email
  • Linguagem
  • Fuso horário

Para administradores

Os administradores também podem atualizar as informações de perfil de outros usuários. Isso inclui três campos sob o cabeçalho HOST USER INFO.

  1. Faça login na sua conta de administrador do Case Active.
  2. Localize um aplicativo e clique no botão ADMIN.
  3. Clique em Usuários no menu superior.
  4. Acesse o submenu e selecione a lista de usuários que deseja acessar (ATIVA ou DESATIVADA). A configuração padrão é Ativo.
  5. Localize um usuário na lista e clique em Editar.
  6. Faça as alterações necessárias. (Veja abaixo)
  7. Clique em Salvar.

Alterar informações do usuário

Os administradores podem alterar as informações do usuário.

  1. Defina o tipo de usuário (usuário ou gerente).
  2. Defina o Status do usuário (Ativo para Desativado).
  3. Insira um ID de tag exclusivo. (opcional)
  4. Clique em Salvar.
