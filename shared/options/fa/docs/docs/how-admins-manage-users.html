چگونه ادمین ها کاربران را مدیریت می کنند

سرپرست‌ها افرادی را که می‌توانند به برنامه‌ها و موارد دسترسی داشته باشند، ایجاد، تغییر یا غیرفعال می‌کنند. هنگامی که یک کاربر برای اولین بار وارد پلتفرم می شود، باید یک گردش کار نمایه را تکمیل کند.

  • برای اطلاعات بیشتر در مورد گردش کار نمایه کاربر جدید اینجا را کلیک کنید.

مدیریت یک کاربر

جدول زیر پنج راه برای مدیریت یک کاربر را نشان می دهد. برای یادگیری نحوه کار روی پیوند کلیک کنید.

روش تایپ کنید اقدامات جزئیات
با استفاده از UI UI افزودن و اصلاح دکمه مدیریت > کاربران
در حین ایجاد کیس و اختصاص کاربر به کیس UI اضافه کردن دکمه Admin > Cases > New button > Create Case page
ایجاد خودکار کاربر با استفاده از پیوند ثبت نام عمومی اقدام کاربر اضافه کردن دکمه مدیریت > تنظیمات > صفحه راه اندازی سیستم
ابزار واردات UI افزودن و اصلاح دکمه Admin > Integrations > Imports
استفاده از API (به زودی) API افزودن و اصلاح N/A

اطلاعیه

این پلتفرم زمانی که کاربران به برنامه اضافه می شوند، اعلان ارسال نمی کند. با این حال، مدیران می‌توانند اعلان‌ها را برای تکالیف پرونده فعال کنند.
