نحوه تخصیص کاربران هنگام ایجاد کیس

این مقاله به مدیران نشان می‌دهد که چگونه یک کاربر را هنگام ایجاد یک کیس جدید اختصاص دهند. اگر کاربر قبلاً در پلتفرم ثبت نام نکرده است، می توانید در حین ایجاد یک مورد جدید، یک کاربر جدید اضافه کنید.

ایجاد پرونده

  1. وارد حساب کاربری مدیریت شوید.
  2. یک برنامه را در صفحه برنامه ها پیدا کنید و روی دکمه ADMIN کلیک کنید.
  3. روی تب منوی Cases کلیک کنید.
  4. روی دکمه NEW کلیک کنید. یک پنجره جدید نمایش داده می شود.

img

  • برای ایجاد یک مورد، فیلدها را پر کنید و سپس به عنوان Assigned Users بروید.

اختصاص یک کاربر

کاربر فعلی

برای اختصاص دادن کاربر فعلی به کیس جدید، این مراحل را دنبال کنید.

روی دکمه ASSIGN USERS کلیک کنید و فیلدها را پر کنید.

  1. روی منوی کشویی User کلیک کنید و یک شناسه (مثلا ایمیل) را انتخاب کنید.
  2. مقداری را در فیلد خالی وارد کنید (به عنوان مثال آدرس ایمیل).
  3. اگر رکورد کاربر در پلتفرم وجود داشته باشد، فیلد به صورت خودکار نامی را پیشنهاد می کند.
  4. کاربر صحیح را انتخاب کنید
  5. یک نقش کاربر (مشتریان، پیش فرض یا کارکنان) را انتخاب کنید.
  6. در صورت تمایل، کادر را علامت بزنید تا یک اعلان ایمیل به کاربر ارسال شود.
  7. روی ذخیره کلیک کنید.

img

Case Active کاربر را به کیس اضافه می کند. اکنون مدیران می توانند روی ذخیره کلیک کنند تا پنجره بسته شود، یا یک کاربر جدید اضافه کنند، یا یک کاربر را حذف کنند.

img

افزودن کاربر جدید

اگر پلتفرم نتوانست رکورد فعلی را پیدا کند، برای افزودن کاربر جدید روی دکمه کلیک کنید.

img

فیلدها را پر کنید و سپس روی SAVE کلیک کنید تا کاربر به پلتفرم اضافه شود. در مرحله بعد، یک نقش کاربری تنظیم کنید و دوباره روی SAVE کلیک کنید تا کاربر به کیس اختصاص یابد. در نهایت برای ذخیره کیس، یک بار دیگر روی SAVE کلیک کنید.
