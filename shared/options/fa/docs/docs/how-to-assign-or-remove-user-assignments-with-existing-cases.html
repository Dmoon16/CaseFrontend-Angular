نحوه اختصاص یا حذف تکالیف کاربر با موارد موجود

این مقاله به مدیران نحوه اختصاص و حذف کاربران کیس را نشان می دهد. ما همچنین به شما نشان می دهیم که چگونه داده های کاربر را تغییر دهید.

باز کردن پرونده موجود

  1. وارد حساب کاربری مدیریت شوید.
  2. یک برنامه را در صفحه برنامه ها پیدا کنید و روی دکمه ADMIN کلیک کنید.
  3. روی تب منوی Cases کلیک کنید.
  4. مطمئن شوید که ACTIVE در زیر منو انتخاب شده است.
  5. به سمت پایین به ستون ACTION بروید.

img

اختصاص دهید یا حذف کنید

برای اختصاص دادن کاربر به کیس یا حذف کاربر از کیس (اما نه پلتفرم) روی Assign Users کلیک کنید. در صفحه بعد: روی ADD کلیک کنید و گردش کار را کامل کنید. برای حذف یک کاربر روی حذف کلیک کنید.

img

تغییر

امکان ویرایش یا به روز رسانی داده های کاربر در صفحه Cases وجود ندارد. در عوض، این مراحل را دنبال کنید:

  1. روی تب منوی کاربران کلیک کنید.
  2. نام کاربری را پیدا کنید و روی Edit کلیک کنید.
  3. تغییرات ایجاد کنید و روی ذخیره کلیک کنید.
