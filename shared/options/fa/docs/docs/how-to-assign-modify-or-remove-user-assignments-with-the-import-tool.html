نحوه اختصاص، تغییر یا حذف تکالیف کاربر با ابزار Import

این مقاله به مدیران نشان می‌دهد که چگونه کاربران کیس را با ابزار Import مدیریت کنند.

مدیران می توانند:

  • کاربران را به یک مورد اختصاص دهید
  • کاربرانی که قبلاً به یک پرونده اختصاص داده شده اند را تغییر دهید
  • حذف کاربران اختصاص داده شده به یک پرونده

باز کردن ابزار Import

  1. وارد حساب کاربری مدیریت شوید.
  2. یک برنامه را در صفحه برنامه ها پیدا کنید و روی دکمه ADMIN کلیک کنید.
  3. روی تب Integration و سپس IMPORTS در زیر منو کلیک کنید.
  4. روی دکمه NEW کلیک کنید. کادر محاوره ای افزودن واردات نمایش داده می شود.

ایجاد رکورد

یک یا چند فایل CSV را دانلود کنید. از این فایل ها برای افزودن، اصلاح یا حذف کاربران مورد استفاده کنید. پس از باز شدن کادر محاوره ای Add Imports:

  1. روی فیلد Type کلیک کنید.
  2. در منوی کشویی پیمایش کنید و یک گزینه را انتخاب کنید (به زیر مراجعه کنید). یک لینک نمایش داده می شود.
  3. روی لینک دانلود نمونه کلیک کنید. یک فایل CSV در دستگاه شما دانلود می شود.

img

فایل را باز کنید و رکوردها را اضافه کنید. برای یادگیری نحوه استفاده از داده های فرمت در فایل CSV اینجا را کلیک کنید.

گزینه های فایل

اختصاص کاربران به موارد * این گزینه را برای افزودن کاربران به پرونده انتخاب کنید. * سلول case_role_order را خالی بگذارید. * به صفحه کاربران بروید و شناسه برچسب این شخص را کپی کنید. اگر قسمت Tag ID خالی است، یک شناسه منحصر به فرد وارد کنید، روی Save کلیک کنید و سپس شناسه را در فایل CSV کپی کنید.

img

اصلاح کاربران اختصاص داده شده به موارد * این گزینه را برای به روز رسانی داده ها برای کاربران فعلی اختصاص داده شده به پرونده انتخاب کنید. * از Case Tag ID برای ستون case_ID استفاده کنید.

img

حذف کاربران اختصاص داده شده به پرونده * این گزینه را برای حذف کاربران اختصاص داده شده به پرونده انتخاب کنید.

img

در حال آپلود فایل

می‌توانید فوراً یک فایل را آپلود کنید یا بعداً به کادر محاوره‌ای افزودن واردات بازگردید و این مراحل را کامل کنید.

  1. کادر محاوره ای Add Imports را باز کنید.
  2. یک نام منحصر به فرد برای فایل CSV وارد کنید.
  3. برای آپلود فایل از دستگاه خود، روی نماد مثبت (+) کلیک کنید.
  4. روی فیلد Type کلیک کنید و یک گزینه را انتخاب کنید. (بالا را ببین)
  5. روی ذخیره کلیک کنید.

در حال پردازش

بسته به تعداد رکوردهای موجود در فایل و بارگذاری سیستم، پلتفرم ممکن است چندین دقیقه طول بکشد تا وارد کردن را پردازش کند.

img

وقتی وارد کردن کامل شد، پلتفرم سوابق کاربر را در صفحه Cases تنظیم می‌کند (یعنی اضافه می‌کند، اصلاح می‌کند، یا حذف می‌کند).

شکست

اگر عملیات واردات ناموفق باشد، پلتفرم یک پیام Failed را برمی‌گرداند. در بیشتر موارد، علت خرابی قالب بندی داده ها است.

img

برای دیدن علت شکست:

  1. در زیر عنوان Action روی View کلیک کنید.
  2. در صفحه بعد، ماوس را روی نماد Failed قرار دهید. راهنمای ابزار خطا را نمایش می دهد.
  3. به فایل CVS بروید و خطا را تنظیم و رفع کنید.
  4. فایل را دوباره آپلود کنید.
