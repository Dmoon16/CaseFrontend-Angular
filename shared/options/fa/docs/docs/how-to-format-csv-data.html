نحوه قالب بندی داده های CSV

در زیر لیستی از فیلدهای داده در فایل های CSV آمده است. هر گزینه واردات دارای ترکیب متفاوتی از فیلدها است.

نام کاربری

  • آدرس ایمیل کاربر را وارد کنید.

داده_نام

  • نام کاربر را وارد کنید.

نام خانوادگی

  • نام خانوادگی کاربر را وارد کنید.

محل

  • این ستون زبان مورد نظر کاربر را بررسی می کند.
  • برای انگلیسی en، برای اسپانیایی sp را وارد کنید.

zoneinfo

  • این منطقه زمانی کاربر را تنظیم می کند. قاره و شهر را بدون فاصله وارد کنید (از زیرخط استفاده کنید).
  • باید از گزینه ای در منوی کشویی نمایه > LANGUAGE & REGION > منطقه زمانی استفاده کنید.
  • از بهترین منطقه زمانی استفاده کنید. برای مثال، بوستون در کشویی نیست، بنابراین می‌توانیم از نیویورک (یا هر شهری در منطقه زمانی EST/EDT) استفاده کنیم.

host_tag_id

  • یک برچسب دائمی وارد کنید.
  • az، 0-9، -، / مجاز هستند.

مورد_وضعیت

  • 0 = بسته
  • 1 = فعال
