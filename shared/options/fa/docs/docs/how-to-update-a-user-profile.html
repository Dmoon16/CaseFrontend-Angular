نحوه به روز رسانی پروفایل کاربری

این مقاله به کاربران نشان می دهد که چگونه اطلاعات صفحه نمایه خود را به روز کنند.

  • کاربران می توانند پروفایل های خود را به روز کنند.
  • ادمین ها می توانند صفحه پروفایل هر کاربری را به روز کنند.

برای کاربران

اولین قدم باز کردن صفحه پروفایل است. کاربران و مدیران این مراحل را برای به روز رسانی صفحه نمایه خود دنبال می کنند.

  1. به حساب Case Active خود وارد شوید.
  2. روی تب Profile در بالای صفحه کلیک کنید.
  3. تغییرات ایجاد کنید.
  4. روی ذخیره کلیک کنید.

اکثر فیلدها اختیاری هستند و برخی از فیلدها الزامی هستند. فیلدهای مورد نیاز عبارتند از:

  • نام کوچک
  • نام خانوادگی
  • آدرس ایمیل
  • زبان
  • منطقه زمانی

برای ادمین ها

مدیران همچنین می توانند اطلاعات نمایه را برای سایر کاربران به روز کنند. این شامل سه فیلد تحت عنوان HOST USER INFO است.

  1. به حساب مدیریت Case Active خود وارد شوید.
  2. یک برنامه را پیدا کنید و روی دکمه ADMIN کلیک کنید.
  3. در منوی بالا روی Users کلیک کنید.
  4. به منوی فرعی بروید و لیست کاربری را که می خواهید به آن دسترسی داشته باشید (فعال یا غیرفعال) انتخاب کنید. تنظیم پیش فرض Active است.
  5. یک کاربر را در لیست پیدا کنید و روی ویرایش کلیک کنید.
  6. در صورت نیاز تغییرات ایجاد کنید. (پایین را ببینید)
  7. روی ذخیره کلیک کنید.

تغییر اطلاعات کاربر

مدیران می توانند اطلاعات کاربر را تغییر دهند.

  1. نوع کاربر (کاربر یا مدیر) را تنظیم کنید.
  2. وضعیت کاربر (فعال به غیرفعال) را تنظیم کنید.
  3. یک شناسه برچسب منحصر به فرد وارد کنید. (اختیاری)
  4. روی ذخیره کلیک کنید.
