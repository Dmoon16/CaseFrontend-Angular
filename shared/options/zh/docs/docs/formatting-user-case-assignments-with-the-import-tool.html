使用导入工具格式化用户案例分配数据

以下是 CSV 文件中的数据字段列表。每个导入选项都有不同的字段组合。

案例ID

  • 输入案例用户或标签 ID。
  • 转到案例页面 > 编辑 > 标签 ID 或转到案例页面 > 编辑 > 网址中的案例 ID。

用户身份

  • 输入用户 ID 或标记 ID。
  • 转到用户页面 > 编辑 > 标记 ID 或转到用户页面 > 编辑 > url 中的用户 ID。

case_role_order

  • 将单元格留空。

case_role_ID

  • 输入：客户、员工或默认值。

案例通知

  • 0 = 否
  • 1 = 是，向用户发送电子邮件通知。
