如何使用公共注册链接自动分配用户

管理员学习如何配置系统设置，以便将新用户自动分配给新自动创建的案例。

打开设置页面

第一步是打开“系统设置”页面。

  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 单击系统选项卡，然后单击子菜单上的系统设置。
  4. 向下滚动到用户注册标题。

配置系统设置

  1. 将允许注册和自动创建案例下拉菜单设置为开。
  2. 单击角色菜单并选择一个选项（建议使用默认值）。
  3. 复制并共享公共用户注册链接。

发送链接

单击链接并完成注册的每个用户都会触发平台在案例页面上生成一个新案例，并在该案例中自动分配用户。
