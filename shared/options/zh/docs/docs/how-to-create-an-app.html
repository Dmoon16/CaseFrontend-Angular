如何创建应用程序

本文向新用户展示如何创建免费试用帐户。 Case Active 会自动授予新用户对此帐户的管理员权限。

创建一个帐户

  1. 转到登录页面。
  2. 单击开始按钮。
  3. 输入电子邮件地址。
  4. 点击下一步。

图片

验证帐户

  1. 打开一个新的浏览器选项卡并登录到您的电子邮件帐户。
  2. 复制六位数代码。
  3. 返回案例活动选项卡。
  4. 将代码粘贴到字段中。将显示一个新窗口。

在新窗口中：

  1. 填写必填字段。
  2. 阅读服务条款并勾选方框表示同意。
  3. 单击创建一个免费帐户。

图片

完成应用设置

工作流显示在下一页上。完成这些步骤以完成设置您的应用程序。您可以随时返回这些页面来编辑或更新信息或您的帐户状态。

在“关于应用程序”页面上，填写公司名称和网站字段（必填），然后单击“下一步”。

图片

在“账单信息”页面上，单击“付款计划”字段并查看选项。默认设置为免费。单击提交。

图片

提交表单后，Case Active 会显示 Apps 页面。对于您创建或属于的每个应用程序，该页面显示：

  1. 应用名称
  2. 您的类型（用户角色）
  3. 地位

图片

应用程序页面是许多管理功能的启动点：

  1. 单击 APP 按钮可就案例相关问题进行交流、填写文档、视频聊天（以及更多）。
  2. 单击管理按钮打开管理面板。这是管理员管理用户、案例、集成等的地方。
  3. 单击省略号以管理帐户账单和通知设置。
  • 注意Case Active 为管理员用户显示 ADMIN 按钮。没有管理员权限的用户看不到管理员按钮（如下图所示）。
