如何使用 UI 创建和修改用户

本文向管理员展示了如何使用平台的用户界面创建用户和更新记录。管理员还可以将用户添加到新案例并编辑现有案例。

创建用户

  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 在管理面板上，单击用户菜单选项卡。
  4. 单击新建按钮。将显示一个新窗口。

图片

填写字段并单击保存。 Case Active 将名称添加到与此应用关联的用户列表中。

图片

修改用户

要更新少量用户记录，请使用“编辑”工具并对每个用户进行更改。

  • 注意要一次性修改大量用户记录，请考虑使用导入工具。
  1. 在管理面板上，单击用户菜单选项卡。
  2. 找到一个用户并单击编辑。
  3. 进行更改并单击“保存”。

下一步
