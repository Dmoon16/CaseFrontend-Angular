如何通过 UI 菜单创建或修改案例

本文向管理员展示了如何使用平台的用户界面创建案例。管理员还可以将用户添加到新案例并更新现有案例。

创建案例

  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 在管理面板上，单击案例菜单选项卡。
  4. 单击新建按钮。将显示一个新窗口。

填写必填字段并单击保存。

添加用户

或者，管理员可以在此页面上将用户添加到个案。向下滚动并单击分配用户按钮。

图片

在分配用户模式中，填写字段。选中该框向用户发送电子邮件通知。 （可选的）

图片

修改案例

要更新少量个案，请使用“编辑”工具并对每个个案进行更改。

  • 注意要一次性更新大量案例，请考虑使用导入工具
  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 在管理面板上，单击案例菜单选项卡。
  4. 找到案例并单击编辑。
  5. 进行更改并单击“保存”。
