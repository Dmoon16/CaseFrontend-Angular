如何使用公共注册链接自动创建案例

本文向管理员展示了如何通过发送单个注册链接来创建案例（并同时添加用户）。

定位链接

  1. 登录到管理员帐户。
  2. 在“应用程序”页面上找到一个应用程序，然后单击“管理”按钮。
  3. 在管理面板上，单击设置菜单选项卡。
  4. 确保选择了系统设置子菜单选项卡。

图片

配置设置

向下滚动到用户注册标题。

确保允许注册字段设置为开。 （必需）启用此功能后，Case Active 会生成一个唯一链接。

  1. 单击“自动创建案例”字段并选择“打开”。个案用户角色字段显示。
  2. 选择角色。 （见下文）

图片

个案用户角色

存在三种案例用户角色。

  • 客户端（推荐）
  • 默认
  • 职员

发送链接

与一个或多个人共享链接。单击链接并完成注册的每个用户都会触发平台在案例页面上生成一个新案例并自动分配案例中的用户。
