Sådan opdaterer du en brugerprofil

Denne artikel viser brugere, hvordan de opdaterer oplysninger på deres profilside.

  • Brugere kan opdatere deres egne profiler.
  • Administratorer kan opdatere profilsiden for enhver bruger.

For brugere

Det første trin er at åbne profilsiden. Brugere og administratorer følger disse trin for at opdatere deres egen profilside.

  1. Log ind på din Case Active-konto.
  2. Klik på fanen Profil øverst på siden.
  3. Lave ændringer.
  4. Klik på Gem.

De fleste felter er valgfrie, og nogle felter er obligatoriske. De obligatoriske felter er:

  • Fornavn
  • Efternavn
  • Email adresse
  • Sprog
  • Tidszone

Til administratorer

Administratorer kan også opdatere profiloplysningerne for andre brugere. Dette inkluderer tre felter under overskriften HOST USER INFO.

  1. Log ind på din Case Active-administratorkonto.
  2. Find en app, og klik på ADMIN-knappen.
  3. Klik på Brugere i topmenuen.
  4. Gå til undermenuen og vælg den brugerliste, du vil have adgang til (AKTIV eller DEAKTIVERET). Standardindstillingen er Aktiv.
  5. Find en bruger på listen, og klik på Rediger.
  6. Foretag ændringer efter behov. (se nedenunder)
  7. Klik på Gem.

Skift brugeroplysninger

Administratorer kan ændre brugeroplysninger.

  1. Indstil brugertypen (bruger eller leder).
  2. Indstil brugerstatus (aktiv til deaktiveret).
  3. Indtast et unikt tag-id. (valgfri)
  4. Klik på Gem.
