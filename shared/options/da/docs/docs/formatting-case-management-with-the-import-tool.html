Formatering af sagshåndteringsdata med importværktøjet

Nedenfor er en liste over obligatoriske felter i CSV-filer for at håndtere sager med importværktøjet.

brugernavn

  • Indtast brugerens e-mailadresse.

tag_id (et id, du kan henvise til senere)

  • en unik identifikator (brug alfanumeriske tegn, bindestreger accepteres)

sagsstatus

  • 0 = lukket
  • 1 = aktiv

dato_åbnet

  • måned dag år
  • mm/dd/åååå

om

  • en unik beskrivelse af sagen (brug alfanumeriske tegn, bindestreger accepteres)
