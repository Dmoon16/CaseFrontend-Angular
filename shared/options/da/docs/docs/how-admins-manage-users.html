Sådan administrerer administratorer brugere

Administratorer opretter, ændrer eller deaktiverer, hvem der kan få adgang til apps og sager. Når en bruger logger ind på platformen for første gang, skal de fuldføre en profilarbejdsgang.

  • Klik her for at lære mere om den nye brugerprofil workflow.

Håndtering af en bruger

Tabellen nedenfor viser fem måder at administrere en bruger på. Klik på et link for at lære hvordan.

Metode Type Handlinger detaljer
Bruger UI UI tilføje og ændre Admin-knap > Brugere
Mens du opretter en sag og tildeler bruger til sag UI tilføje Admin-knap > Sager > Ny knap > Opret sagsside
Opret bruger automatisk ved hjælp af offentligt tilmeldingslink Brugerhandling tilføje Admin-knap > Indstillinger > Systemopsætningsside
Importværktøj UI tilføje og ændre Admin-knap > Integrationer > Importer
Bruger API (kommer snart) API tilføje og ændre N/A

Meddelelser

Platformen sender ikke brugere en notifikation, når de føjes til en app. Administratorer kan dog aktivere notifikationer for sagstildelinger.
