Carane Nganyari Profil Panganggo

Artikel iki nuduhake pangguna carane nganyari informasi ing kaca Profile.

  • Pangguna bisa nganyari profil dhewe.
  • Admin bisa nganyari kaca Profil pangguna apa wae.

Kanggo pangguna

Langkah pisanan yaiku mbukak kaca Profil. Pangguna lan admin tindakake langkah iki kanggo nganyari kaca Profil dhewe.

  1. Mlebu menyang akun Case Active.
  2. Klik tab Profil ing sisih ndhuwur kaca.
  3. Nggawe owah-owahan.
  4. Klik Simpen.

Umume lapangan opsional, lan sawetara kolom dibutuhake. Kolom sing dibutuhake yaiku:

  • Jeneng ngarep
  • Jeneng mburi
  • Alamat email
  • Basa
  • Zona wektu

Kanggo admin

Admin uga bisa nganyari informasi profil kanggo pangguna liyane. Iki kalebu telung kolom ing judhul HOST USER INFO.

  1. Mlebu menyang akun admin Case Active.
  2. Goleki app banjur klik tombol ADMIN.
  3. Klik Pangguna ing menu ndhuwur.
  4. Pindhah menyang sub-menu banjur pilih dhaptar pangguna sing pengin diakses (AKTIF utawa disable). Setelan gawan yaiku Aktif.
  5. Goleki pangguna ing dhaptar banjur klik Sunting.
  6. Gawe owah-owahan kaya sing dibutuhake. (ndeleng ngisor)
  7. Klik Simpen.

Ngganti informasi pangguna

Admin bisa ngganti informasi pangguna.

  1. Setel Tipe Panganggo (Panganggo utawa Manajer).
  2. Setel Status Panganggo (Aktif kanggo Disabled).
  3. Ketik ID Tag unik. (opsional)
  4. Klik Simpen.
