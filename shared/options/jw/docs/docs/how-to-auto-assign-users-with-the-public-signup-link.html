Cara Nggawe Otomatis Pangguna nganggo Link Ndaftar Umum

Admin sinau carane ngatur setelan sistem supaya pangguna anyar ditugasake kanthi otomatis menyang kasus sing mentas digawe kanthi otomatis.

Mbukak kaca persiyapan

Langkah pisanan yaiku mbukak kaca Persiyapan Sistem.

  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Klik tab Sistem banjur SYSTEM SETUP ing submenu.
  4. Gulung mudhun menyang judhul USER SIGNUP.

Konfigurasi setelan sistem

  1. Setel menu tarik-mudhun Allow Signup lan Auto Create Case menyang On.
  2. Klik menu peran lan pilih opsi (standar dianjurake).
  3. Nyalin lan enggo bareng link Ndaftar Pangguna Umum.

Ngirim link

Saben pangguna sing ngeklik link lan ngrampungake ndhaptar bakal micu platform kanggo ngasilake kasus anyar ing kaca Kasus lan menehi pangguna kanthi otomatis ing kasus kasebut.
