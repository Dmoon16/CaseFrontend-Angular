Carane Admin Nggawe Pangguna

Admin nggawe, ngowahi utawa mateni sapa sing bisa ngakses aplikasi lan kasus. Nalika pangguna mlebu ing platform kanggo pisanan, dheweke kudu ngrampungake alur kerja profil.

  • Klik kene kanggo sinau luwih lengkap babagan alur kerja profil pangguna anyar.

Nggawe pangguna

Tabel ing ngisor iki nuduhake limang cara kanggo nggawe pangguna. Klik link kanggo mangerteni carane.

| Cara | Jinis | Tumindak | Rincian | Pangguna anyar nampa kabar? |
|------------------------|----------------|------- --------------------------------------------- ------------------------|

Nggunakake UI | UI | nambah & ngowahi | Tombol Admin > Pangguna | ora |

Nalika nggawe kasus lan nemtokake pangguna kanggo kasus | UI | nambahi | Tombol Admin > Kasus > Tombol anyar > Gawe kaca Kasus | Ya, yen diaktifake |

Nggawe pangguna kanthi otomatis nggunakake link pendaptaran umum | Tumindak panganggo | nambahi | Tombol Admin > Setelan > Kaca Persiyapan Sistem | Ya, yen kasus nggawe otomatis diaktifake |

Alat ngimpor | UI | nambah & ngowahi | Tombol Admin > Integrasi > Impor | ora |

Nggunakake API (teka rauh) | API | nambah & ngowahi | na | na |

Kabar

Platform kasebut ora ngirim kabar marang pangguna nalika ditambahake menyang aplikasi. Nanging, admin bisa ngaktifake kabar kanggo tugas cilik.
