Cara Nggawe utawa Ngowahi Kasus nganggo Alat Impor

Artikel iki nuduhake admin carane nggawe utawa ngowahi kasus sing wis ana nganggo alat Impor.

Mbukak alat Impor

  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Ing panel Admin, klik tab Integrasi banjur IMPOR ing submenu.
  4. Klik tombol NEW. Kothak dialog Tambah Impor ditampilake ing tab browser anyar.

img

Nggawe kasus

Sawise kothak dialog Tambah Impor mbukak:

  1. Klik kolom Tipe banjur pilih opsi (ndeleng ngisor). A link ditampilake.
  2. Klik link Download Sample. File CSV diundhuh menyang piranti sampeyan.

Sabanjure, bukak file lan tambahake kasus.

Jinis

Nalika sampeyan ngeklik kolom Tipe , menu gulung mudhun ditampilake. Pilih salah siji saka rong opsi kanggo ngimpor kasus massal.

  • Cathetan Menu gulung mudhun nduweni akeh pilihan, nanging mung loro sing ana gandhengane karo ngatur kasus. Saben opsi duwe file CSV sampel sing beda.

Tambah kasus

  • Pilih opsi iki kanggo upload akeh kasus prasaja.
  • Klik link Download Sample kanggo mbukak file CSV kanthi judhul kolom kanggo jinis impor Gawe kasus.

Ngowahi kasus

  • Pilih opsi iki kanggo nganyari kasus saiki.
  • Klik link Download Sample kanggo mbukak file CSV kanthi judhul kolom kanggo jinis impor Modify case.

Ngunggah file

Sampeyan bisa langsung ngunggah file utawa bali menyang kothak dialog Tambah Impor mengko lan ngrampungake langkah-langkah iki.

  1. Bukak kothak dialog Tambah Impor.
  2. Ketik jeneng unik kanggo file CSV.
  3. Klik lambang plus (+) kanggo ngunggah file saka piranti sampeyan.
  4. Klik kolom Tipe banjur pilih opsi. (ndeleng ndhuwur)
  5. Klik SAVE.

img

Formatting

Iki minangka lapangan sing dibutuhake lan syarat pisanan. Klik kene kanggo mriksa rincian format.

tag_id (utawa case_ID) pengenal unik (nggunakake karakter alfanumerik, hyphens ditampa)

case_status 0 = ditutup 1 = aktif

tanggal_dibukak sasi/dina/taun mm/dd/tttt

babagan katrangan unik babagan kasus kasebut (nggunakake karakter alfanumerik, hyphens ditampa)

img

Ngolah

Gumantung saka jumlah cathetan ing file lan beban sistem, platform bisa njupuk sawetara menit kanggo proses impor.

Nalika impor rampung, platform nyetel cathetan cilik (yaiku nambah utawa ngowahi) ing dashboard Kasus admin.

Gagal

Yen operasi impor gagal, platform ngasilake pesen Gagal. Umume kasus, panyebab kegagalan yaiku format data.

Kanggo ndeleng sabab saka kegagalan:

  1. Klik View ing judhul ACTION.
  2. Ing kaca sabanjure, kisaran mouse ing lambang Gagal. Tooltip nampilake kesalahan.
  3. Pindhah menyang file CVS lan atur lan ndandani kesalahan kasebut.
  4. Upload file maneh.
