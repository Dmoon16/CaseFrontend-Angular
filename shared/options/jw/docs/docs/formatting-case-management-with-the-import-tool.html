Format Data Manajemen Kasus nganggo Alat Impor

Ing ngisor iki dhaptar kolom sing dibutuhake ing file CSV kanggo ngatur kasus nganggo alat Impor.

jeneng panganggo

  • Ketik alamat email pangguna.

tag_id (id sing bisa dirujuk mengko)

  • pengenal unik (nggunakake karakter alfanumerik, hyphens ditampa)

status_kasus

  • 0 = ditutup
  • 1 = aktif

tanggal_dibukak

  • wulan/dina/taun
  • mm/dd/yyyy

babagan

  • deskripsi unik saka kasus kasebut (nggunakake karakter alfanumerik, hyphens ditampa)
