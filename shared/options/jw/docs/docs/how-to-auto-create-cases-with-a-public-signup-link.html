Cara Nggawe Kasus Otomatis kanthi Link Ndaftar Umum

Artikel iki nuduhake admin carane nggawe kasus (lan nambah pangguna ing wektu sing padha) kanthi ngirim link signup siji.

Nemokake link

  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Ing panel Admin, klik tab menu Setelan.
  4. Priksa manawa tab submenu SYSTEM SETUP dipilih.

Konfigurasi setelan

Gulung mudhun menyang judhul USER SIGNUP.

Priksa manawa kolom Allow Signup disetel menyang On. (dibutuhake) Nalika fitur iki diaktifake, Case Active ngasilake link unik.

  1. Klik lapangan Nggawe Kasus Otomatis banjur pilih On. Kolom Peran Panganggo Kasus ditampilake.
  2. Pilih peran. (ndeleng ngisor)

Peran pangguna kasus

Ana telung peran pangguna cilik.

  • klien (dianjurake)
  • gawan
  • staf

Ngirim link

Nuduhake link karo siji utawa luwih wong. Saben pangguna sing ngeklik link lan ngrampungake ndhaptar bakal micu platform kanggo ngasilake kasus anyar ing kaca kasus lan kanthi otomatis nemtokake pangguna ing kasus kasebut.
