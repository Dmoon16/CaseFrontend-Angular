Carane Pangguna Nggawe Akun Anyar

Sawise admin nambahake pangguna menyang kasus: * yen pangguna anyar kanggo caseactive, pangguna kudu ngrampungake alur kerja kanggo nggawe akun lan nampa undhangan kanggo ngakses kasus kasebut. * yen pangguna duwe akun caseactive sing wis ana, pangguna kudu nampa undhangan kanggo ngakses kasus kasebut.

Artikel iki nerangake langkah-langkah sing kudu ditindakake pangguna kanggo nggawe akun.

Miwiti

Ana rong cara pangguna bisa ditambahake menyang aplikasi caseactive

  • Miturut admin: Yen admin nggawe pangguna liwat UI platform utawa alat Impor , admin kudu ngirim pangguna link menyang account.caseactive.com kanggo nerusake login.
  • Miturut pangguna: Yen pangguna nggawe akun nggunakake link signup , pangguna bakal kanthi otomatis dialihake menyang account.caseactive.com kanggo nerusake login.

Mlebet

Nalika pangguna mlebu menyang akun CaseActive sapisanan, platform kasebut njaluk pangguna supaya:

  • bukak akun email banjur salin/tempel kode keamanan enem digit
  • ngrampungake profil akun

Sawise pangguna ngeklik UPDATE, kaca Aplikasi ditampilake. Kaca kasebut nuduhake jeneng app (jeneng perusahaan), jinis pangguna (peran), status, lan tombol UNDANGAN.

Kanggo ngakses app, pangguna kudu nampa undhangan. Pangguna kudu:

  1. Klik tombol INVITATION.
  2. Priksa ijin.
  3. Klik tab Notifikasi kanggo mriksa lan nyunting (opsional) syarat kasebut.
  4. Klik APPROVE ACCESS.

Nalika pangguna nyetujoni, platform nampilake tombol APP ing kaca pangguna. Tombol iki menehi akses pangguna menyang kasus sing ditugasake.

Kanggo ngowahi setelan pangguna, pangguna ngeklik elipsis.

Pangguna bisa: * mbusak akses app * ngowahi setelan kabar
