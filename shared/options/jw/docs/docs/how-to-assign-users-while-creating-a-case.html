Cara Nemtokake Pangguna Nalika Nggawe Kasus

Artikel iki nuduhake admin carane nemtokake pangguna nalika nggawe kasus anyar. Yen pangguna durung ndhaptar ing platform kasebut, sampeyan bisa nambah pangguna anyar nalika nggawe kasus anyar.

Nggawe kasus

  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Klik tab menu Cases.
  4. Klik tombol NEW. Jendhela anyar ditampilake.
  • Isi kolom kanggo nggawe cilik banjur gulung mudhun menyang judhul Panganggo sing Ditugasake.

Nemtokake pangguna

pangguna saiki

Tindakake langkah iki kanggo nemtokake pangguna saiki menyang kasus anyar.

Klik tombol ASSIGN USERS lan isi kolom kasebut.

  1. Klik gulung mudhun pangguna banjur pilih pengenal (contone, email).
  2. Ketik nilai ing kolom kosong (contone, alamat email).
  3. Yen rekaman pangguna ana ing platform kasebut, kolom kasebut kanthi otomatis menehi jeneng.
  4. Pilih pangguna sing bener.
  5. Pilih peran pangguna (klien, standar, utawa staf).
  6. Opsional, centhang kothak kanggo ngirim kabar email marang pangguna.
  7. Klik SAVE.

Case Active nambahake pangguna menyang kasus kasebut. Saiki admin bisa ngeklik Simpen kanggo nutup jendhela, utawa nambah pangguna anyar, utawa mbusak pangguna.

Nambahake pangguna anyar

Yen platform ora bisa nemokake rekaman saiki, klik tombol kanggo nambah pangguna anyar.

Isi kolom banjur klik SAVE kanggo nambah pangguna menyang platform. Sabanjure, atur peran pangguna lan klik SAVE maneh kanggo nemtokake pangguna menyang kasus kasebut. Pungkasan, klik SAVE sepisan maneh kanggo nyimpen kasus kasebut.
