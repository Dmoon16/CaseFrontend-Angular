Cara Nggawe Pangguna nganggo Alat Impor

Artikel iki nuduhake admin carane nggawe pangguna nganggo alat Impor.

Mbukak alat Impor

Admin bisa nggunakake alat Impor kanggo ngunggah akeh pangguna anyar lan ngowahi cathetan pangguna sing wis ana.

  1. Mlebu menyang akun admin.
  2. Goleki app ing kaca Aplikasi banjur klik tombol ADMIN.
  3. Ing panel Admin, klik tab Integrasi banjur IMPOR ing submenu.
  4. Klik tombol NEW. Kothak dialog Tambah Impor ditampilake ing tab browser anyar.

Nggawe cathetan

Sawise kothak dialog Tambah Impor mbukak:

  1. Klik kolom Tipe banjur pilih opsi (ndeleng ngisor). A link ditampilake.
  2. Klik link Download Sample . File CSV diundhuh menyang piranti sampeyan.

Sabanjure, mbukak file lan nambah cathetan. Klik kene kanggo mangerteni carane nggunakake data format ing file CSV.

Jinis

Nalika sampeyan ngeklik kolom Tipe, menu gulung mudhun ditampilake. Pilih salah siji saka rong opsi kanggo impor akeh.

: exclamation: CATETAN Menu gulung-mudhun wis akeh opsi, nanging mung loro related kanggo alat Impor. Saben opsi duwe file CSV sampel sing beda.

Tambah pangguna

  • Pilih opsi iki kanggo upload akeh prasaja saka cathetan pangguna.
  • Klik link Download Sample kanggo mbukak file CSV kanthi judhul kolom kanggo jinis Impor Tambah pangguna.

Ngowahi pangguna

  • Pilih opsi iki kanggo nganyari data kanggo pangguna saiki.
  • Klik link Download Sample kanggo mbukak file CSV kanthi judhul kolom kanggo jinis Impor pangguna Modify.

Ngunggah file

Sampeyan bisa langsung ngunggah file utawa bali menyang kothak dialog Tambah Impor mengko lan ngrampungake langkah iki.

  1. Bukak kothak dialog Tambah Impor.
  2. Ketik jeneng unik kanggo file CSV.
  3. Klik lambang plus (+) kanggo ngunggah file saka piranti sampeyan.
  4. Klik kolom Tipe banjur pilih opsi. (ndeleng ndhuwur)
  5. Klik SAVE.

Ngolah

Gumantung saka jumlah cathetan ing file lan beban sistem, platform bisa njupuk sawetara menit kanggo proses impor.

Nalika impor rampung, platform nyetel cathetan pangguna (yaiku nambah, ngowahi, utawa mateni) ing dashboard Pangguna admin.

Gagal

Yen operasi impor gagal, platform ngasilake pesen Gagal. Umume kasus, panyebab kegagalan yaiku format data.

Kanggo ndeleng sabab saka kegagalan:

  1. Klik Ndeleng ing judhul Tindakan.
  2. Ing kaca sabanjure, kisaran mouse ing lambang Gagal. Tooltip nampilake kesalahan.
  3. Pindhah menyang file CVS lan atur lan ndandani kesalahan kasebut.
  4. Upload file maneh.
