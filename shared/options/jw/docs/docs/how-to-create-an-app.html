Carane Nggawe App

Artikel iki nuduhake pangguna anyar carane nggawe akun nyoba gratis. Case Active kanthi otomatis menehi hak istimewa admin pangguna anyar kanggo akun iki.

Nggawe akun

  1. Pindhah menyang kaca mlebu .
  2. Klik tombol Miwiti.
  3. Ketik alamat email.
  4. Klik Sabanjure.

img

Verifikasi akun

  1. Bukak tab browser anyar lan mlebu menyang akun email sampeyan.
  2. Nyalin kode enem digit.
  3. Bali menyang tab Case Active.
  4. Tempel kode menyang kolom. Jendhela anyar ditampilake.

Ing jendela anyar:

  1. Isi kolom sing dibutuhake.
  2. Waca syarat layanan lan centhang kothak kanggo setuju.
  3. Klik BUAT AKUN GRATIS.

img

Rampung persiyapan app

A workflow ditampilake ing kaca sabanjure. Rampungake langkah-langkah kanggo ngrampungake nyetel app sampeyan. Sampeyan bisa tansah bali menyang kaca iki kanggo ngowahi utawa nganyari informasi utawa status akun.

Ing kaca Babagan App, isi jeneng perusahaan lan kolom situs web (dibutuhake) banjur klik Sabanjure.

img

Ing kaca Informasi Tagihan, klik kolom Rencana Pembayaran lan deleng opsi kasebut. Setelan gawan punika Free. Klik SUBMIT.

img

Sawise sampeyan ngirim formulir, Case Active nampilake kaca Aplikasi. Kanggo saben aplikasi sing sampeyan gawe utawa dadi kagungane, kaca kasebut nuduhake:

  1. Jeneng app
  2. Tipe Panjenengan (peran pangguna)
  3. Status

img

Kaca Aplikasi minangka titik peluncuran kanggo akeh fungsi admin:

  1. Klik tombol APP kanggo komunikasi, ngisi dokumen, obrolan video (lan liyane) babagan masalah sing gegandhengan karo kasus.
  2. Klik tombol ADMIN kanggo mbukak panel admin. Iki ngendi Admin ngatur pangguna, kasus, integrasi, lan liya-liyane.
  3. Klik elipsis kanggo ngatur tagihan akun lan setelan kabar.
  • Cathetan Case Active nampilake tombol ADMIN kanggo pangguna admin. Pangguna sing ora duwe hak istimewa admin ora ndeleng tombol ADMIN (kaya sing ditampilake ing layar ing ngisor iki).
