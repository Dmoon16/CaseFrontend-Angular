Formating Data Tugas Kasus Panganggo nganggo Alat Impor

Ing ngisor iki dhaptar kolom data ing file CSV. Saben opsi impor duwe kombinasi kolom sing beda.

kasus_ID

  • Ketik ID pangguna utawa Tag.
  • Pindhah menyang kaca Kasus > Sunting > ID Tag UTAWA Pindhah menyang kaca Kasus > Sunting > ID Kasus ing url.

user_ID

  • Ketik ID pangguna utawa ID Tag.
  • Pindhah menyang kaca Pangguna > Sunting > ID Tag UTAWA Pindhah menyang kaca Pangguna > Sunting > ID pangguna ing url.

case_role_order

  • Ninggalake sel kosong.

case_role_ID

  • Ketik: klien, staf, utawa standar.

case_notify

  • 0 = ora
  • 1 = Ya, ngirim kabar email menyang pangguna.
