كيفية التعيين التلقائي للمستخدمين باستخدام رابط الاشتراك العام

يتعلم المسؤولون كيفية تكوين إعدادات النظام بحيث يتم تعيين المستخدمين الجدد تلقائيًا إلى حالة تم إنشاؤها تلقائيًا.

فتح صفحة الإعداد

الخطوة الأولى هي فتح صفحة إعداد الأنظمة.

  1. قم بتسجيل الدخول إلى حساب المسؤول.
  2. حدد موقع تطبيق على صفحة التطبيقات وانقر فوق الزر ADMIN.
  3. انقر فوق علامة تبويب الأنظمة ثم SYSTEM SETUP في القائمة الفرعية.
  4. قم بالتمرير لأسفل إلى عنوان تسجيل المستخدم.

تكوين إعدادات النظام

  1. قم بتعيين القائمتين المنسدلتين السماح بالتسجيل وإنشاء الحالة تلقائيًا على تشغيل.
  2. انقر فوق قائمة الدور وحدد خيارًا (يوصى باستخدام الإعداد الافتراضي).
  3. انسخ ارتباط اشتراك المستخدم العام وشاركه.

إرسال الارتباط

يقوم كل مستخدم ينقر على الرابط ويكمل التسجيل بتشغيل النظام الأساسي لإنشاء حالة جديدة على صفحة الحالات ويقوم بتعيين المستخدم تلقائيًا في هذه الحالة.
