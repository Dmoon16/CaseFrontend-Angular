كيفية إنشاء التطبيق

توضح هذه المقالة للمستخدمين الجدد كيفية إنشاء حساب تجريبي مجاني. تمنح Case Active تلقائيًا امتيازات مسؤول المستخدم الجديد لهذا الحساب.

إنشاء حساب

  1. اذهب إلى صفحة تسجيل الدخول .
  2. انقر فوق زر البدء.
  3. ادخل عنوان البريد الإلكتروني.
  4. انقر فوق {التالي.

IMG

التحقق من الحساب

  1. افتح علامة تبويب متصفح جديدة وقم بتسجيل الدخول إلى حساب بريدك الإلكتروني.
  2. انسخ الرمز المكون من ستة أرقام.
  3. ارجع إلى علامة التبويب Case Active.
  4. الصق الكود في الحقول. تظهر نافذة جديدة.

في النافذة الجديدة:

  1. املأ الحقول المطلوبة.
  2. اقرأ شروط الخدمة وحدد المربع للموافقة.
  3. انقر فوق إنشاء حساب مجاني.

IMG

استكمال إعداد التطبيق

يتم عرض سير العمل في الصفحة التالية. أكمل الخطوات لإنهاء إعداد التطبيق الخاص بك. يمكنك دائمًا العودة إلى هذه الصفحات لتعديل أو تحديث المعلومات أو حالة حسابك.

في صفحة حول التطبيق ، قم بملء اسم الشركة وحقول موقع الويب (مطلوب) ثم انقر فوق التالي.

IMG

في صفحة معلومات الفواتير ، انقر فوق حقل خطة الدفع وراجع الخيارات. الإعداد الافتراضي هو مجاني. انقر فوق 'إرسال.

IMG

بعد إرسال النموذج ، يعرض Case Active صفحة التطبيقات. تظهر الصفحة لكل تطبيق تنشئه أو تنتمي إليه:

  1. اسم التطبيق
  2. النوع الخاص بك (دور المستخدم)
  3. حالة

IMG

تعد صفحة التطبيقات نقطة انطلاق للعديد من الوظائف الإدارية:

  1. انقر فوق الزر APP للتواصل وملء المستندات ودردشة الفيديو (والمزيد) بشأن المشكلات المتعلقة بالحالة.
  2. انقر فوق الزر ADMIN لفتح لوحة الإدارة. هذا هو المكان الذي يدير فيه المسؤولون المستخدمين والحالات وعمليات الدمج والمزيد.
  3. انقر فوق علامة القطع لإدارة فوترة الحساب وإعدادات الإشعارات.
  • ملاحظة حالة الحالة النشطة تعرض الزر ADMIN للمستخدمين المسؤولين. المستخدمون الذين ليس لديهم امتيازات المسؤول لا يرون زر ADMIN (كما هو موضح في الشاشة أدناه).
