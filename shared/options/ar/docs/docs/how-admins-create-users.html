كيف يقوم المسؤولون بإنشاء المستخدمين

يقوم المسؤولون بإنشاء أو تعديل أو تعطيل من يمكنه الوصول إلى التطبيقات والحالات. عندما يقوم المستخدم بتسجيل الدخول إلى النظام الأساسي لأول مرة ، فإنه يحتاج إلى إكمال سير عمل ملف التعريف.

  • انقر هنا لمعرفة المزيد حول سير عمل ملف تعريف المستخدم الجديد.

إشعارات

لا يرسل النظام الأساسي للمستخدمين إشعارًا عند إضافتهم إلى أحد التطبيقات. ومع ذلك ، يمكن للمسؤولين تمكين الإعلامات لتعيينات الحالة.
