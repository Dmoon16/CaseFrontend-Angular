가져오기 도구를 사용하여 사용자 할당을 할당, 수정 또는 제거하는 방법

이 문서는 관리자에게 가져오기 도구를 사용하여 케이스 사용자를 관리하는 방법을 보여줍니다.

관리자는 다음을 수행할 수 있습니다.

  • 사례에 사용자 할당
  • 사례에 이미 할당된 사용자 수정
  • 사례에 할당된 사용자 제거

가져오기 도구 열기

  1. 관리자 계정에 로그인합니다.
  2. 앱 페이지에서 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 통합 탭을 클릭한 다음 하위 메뉴에서 가져오기를 클릭합니다.
  4. 신규 버튼을 클릭합니다. 가져오기 추가 대화 상자가 표시됩니다.

레코드 생성

하나 이상의 CSV 파일을 다운로드합니다. 이 파일을 사용하여 케이스 사용자를 추가, 수정 또는 제거하십시오. 가져오기 추가 대화 상자가 열린 후:

  1. 유형 필드를 클릭합니다.
  2. 드롭다운 메뉴를 스크롤하고 옵션을 선택합니다(아래 참조). 링크가 표시됩니다.
  3. 샘플 다운로드 링크를 클릭합니다. CSV 파일이 장치에 다운로드됩니다.

이미지

파일을 열고 레코드를 추가합니다. CSV 파일에서 형식 데이터를 사용하는 방법을 알아보려면 여기를 클릭하십시오.

파일 옵션

사례에 사용자 할당 * 사례 에 사용자를 추가하려면 이 옵션을 선택합니다. * case_role_order 셀을 비워 둡니다. * 사용자 페이지로 이동하여 이 사람의 태그 ID를 복사합니다. 태그 ID 필드가 비어 있는 경우 고유한 ID를 입력하고 저장을 클릭한 다음 ID를 CSV 파일에 복사합니다.

이미지

사례에 할당된 사용자 수정 * 사례에 할당된 현재 사용자의 데이터를 업데이트하려면 이 옵션을 선택합니다. * case_ID 열에 케이스 태그 ID를 사용합니다.

이미지

사례에 할당된 사용자 제거 * 사례에 할당된 사용자를 제거하려면 이 옵션을 선택합니다.

이미지

파일 업로드

파일을 바로 업로드하거나 나중에 가져오기 추가 대화 상자로 돌아가 다음 단계를 완료할 수 있습니다.

  1. 가져오기 추가 대화 상자를 엽니다.
  2. CSV 파일의 고유한 이름을 입력합니다.
  3. 더하기(+) 아이콘을 클릭하여 장치에서 파일을 업로드합니다.
  4. 유형 필드를 클릭하고 옵션을 선택합니다. (위 참조)
  5. 저장을 클릭합니다.

처리

파일 및 시스템 로드의 레코드 수에 따라 플랫폼에서 가져오기를 처리하는 데 몇 분 정도 걸릴 수 있습니다.

이미지

가져오기가 완료되면 플랫폼은 사례 페이지에서 사용자 레코드를 조정(예: 추가, 수정 또는 제거)합니다.

실패

가져오기 작업이 실패하면 플랫폼이 실패 메시지를 반환합니다. 대부분의 경우 실패 원인은 데이터 형식입니다.

이미지

실패의 원인을 보려면 다음을 수행하십시오.

  1. 작업 제목 아래에서 보기를 클릭합니다.
  2. 다음 페이지에서 실패 아이콘을 마우스로 가리킵니다. 툴팁에 오류가 표시됩니다.
  3. CVS 파일로 이동하여 오류를 조정하고 수정하십시오.
  4. 파일을 다시 업로드하십시오.
