가져오기 도구로 사용자를 생성하는 방법

이 문서는 관리자에게 가져오기 도구를 사용하여 사용자를 생성하는 방법을 보여줍니다.

가져오기 도구 열기

관리자는 가져오기 도구를 사용하여 새 사용자를 일괄 업로드하고 기존 사용자 기록을 수정할 수 있습니다.

  1. 관리자 계정에 로그인합니다.
  2. 앱 페이지에서 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 관리자 패널에서 통합 탭을 클릭한 다음 하위 메뉴에서 가져오기를 클릭합니다.
  4. 신규 버튼을 클릭합니다. 가져오기 추가 대화 상자가 새 브라우저 탭에 표시됩니다.

이미지

레코드 생성

가져오기 추가 대화 상자가 열린 후:

  1. 유형 필드를 클릭하고 옵션을 선택합니다(아래 참조). 링크가 표시됩니다.
  2. 샘플 다운로드 링크를 클릭합니다. CSV 파일이 장치에 다운로드됩니다.

그런 다음 파일을 열고 레코드를 추가합니다. CSV 파일에서 형식 데이터를 사용하는 방법을 알아보려면 여기 를 클릭하십시오.

이미지

유형

유형 필드를 클릭하면 드롭다운 메뉴가 표시됩니다. 대량 가져오기를 위한 두 가지 옵션 중 하나를 선택합니다.

:exclamation: 참고 드롭다운 메뉴에는 많은 옵션이 있지만 가져오기 도구와 관련된 옵션은 두 개뿐입니다. 각 옵션에는 다른 샘플 CSV 파일이 있습니다.

사용자 추가

  • 사용자 레코드의 간단한 대량 업로드를 위해 이 옵션을 선택하십시오.
  • 샘플 다운로드 링크를 클릭하여 사용자 추가 가져오기 유형에 대한 열 제목이 있는 CSV 파일을 엽니다.

사용자 수정

  • 현재 사용자에 대한 데이터를 업데이트하려면 이 옵션을 선택하십시오.
  • 샘플 다운로드 링크를 클릭하여 사용자 가져오기 유형 수정에 대한 열 제목이 있는 CSV 파일을 엽니다.

파일 업로드

파일을 바로 업로드하거나 나중에 가져오기 추가 대화 상자로 돌아가 다음 단계를 완료할 수 있습니다.

  1. 가져오기 추가 대화 상자를 엽니다.
  2. CSV 파일의 고유한 이름을 입력합니다.
  3. 더하기(+) 아이콘을 클릭하여 장치에서 파일을 업로드합니다.
  4. 유형 필드를 클릭하고 옵션을 선택합니다. (위 참조)
  5. 저장을 클릭합니다.

이미지

처리

파일 및 시스템 로드의 레코드 수에 따라 플랫폼에서 가져오기를 처리하는 데 몇 분 정도 걸릴 수 있습니다.

이미지

가져오기가 완료되면 플랫폼은 관리자의 사용자 대시보드에서 사용자 레코드를 조정(예: 추가, 수정 또는 비활성화)합니다.

실패

가져오기 작업이 실패하면 플랫폼이 실패 메시지를 반환합니다. 대부분의 경우 실패 원인은 데이터 형식입니다.

이미지

실패의 원인을 보려면 다음을 수행하십시오.

  1. 작업 제목 아래에서 보기 를 클릭합니다.
  2. 다음 페이지에서 실패 아이콘을 마우스로 가리킵니다. 툴팁에 오류가 표시됩니다.
  3. CVS 파일로 이동하여 오류를 조정하고 수정하십시오.
  4. 파일을 다시 업로드하십시오.
