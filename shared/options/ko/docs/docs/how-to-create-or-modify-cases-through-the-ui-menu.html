UI 메뉴를 통해 사례를 생성하거나 수정하는 방법

이 문서는 관리자에게 플랫폼의 사용자 인터페이스로 케이스를 생성하는 방법을 보여줍니다. 관리자는 새 사례에 사용자를 추가하고 기존 사례를 업데이트할 수도 있습니다.

사례 만들기

  1. 관리자 계정에 로그인합니다.
  2. 앱 페이지에서 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 관리자 패널에서 사례 메뉴 탭을 클릭합니다.
  4. 신규 버튼을 클릭합니다. 새 창이 표시됩니다.

사례 수정

적은 수의 사례를 업데이트하려면 편집 도구를 사용하고 각 개별 사례를 변경합니다.

  • 참고 한 번에 많은 사례를 업데이트하려면 가져오기 도구 사용을 고려하십시오.
  1. 관리자 계정에 로그인합니다.
  2. 앱 페이지에서 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 관리자 패널에서 사례 메뉴 탭을 클릭합니다.
  4. 사례를 찾아 편집을 클릭합니다.
  5. 변경하고 저장을 클릭합니다.
