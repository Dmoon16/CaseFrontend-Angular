UI로 사용자를 만들고 수정하는 방법

이 문서는 관리자에게 플랫폼의 사용자 인터페이스로 사용자를 생성하고 레코드를 업데이트하는 방법을 보여줍니다. 관리자는 새 사례에 사용자를 추가하고 기존 사례를 편집할 수도 있습니다.

사용자 생성

  1. 관리자 계정에 로그인합니다.
  2. 앱 페이지에서 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 관리자 패널에서 사용자 메뉴 탭을 클릭합니다.
  4. 신규 버튼을 클릭합니다. 새 창이 표시됩니다.

이미지

필드를 채우고 SAVE를 클릭합니다. Case Active는 이 앱과 연결된 사용자 목록에 이름을 추가합니다.

이미지

사용자 수정

적은 수의 사용자 레코드를 업데이트하려면 편집 도구를 사용하고 각 개별 사용자를 변경합니다.

  • 참고 한 번에 많은 수의 사용자 레코드를 수정하려면 가져오기 도구 사용을 고려하십시오.
  1. 관리자 패널에서 사용자 메뉴 탭을 클릭합니다.
  2. 사용자를 찾아 편집을 클릭합니다.
  3. 변경하고 저장을 클릭합니다.

이미지

다음 단계

Case Active는 사용자에게 이메일 알림을 보내지 않습니다. 관리자는 사용자에게 연락하여 Case Active 홈 페이지에 대한 링크를 보내야 합니다. 그런 다음 사용자는 로그인 버튼을 클릭하고 새 가입을 위한 워크플로를 완료합니다.

  • 이 도움말에서는 새 사용자를 위한 가입 절차에 대해 설명합니다.
