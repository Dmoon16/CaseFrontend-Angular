가입 링크로 사용자를 생성하는 방법

이 문서는 관리자에게 등록 링크를 전송하여 사용자를 생성하는 방법을 보여줍니다.

가입 링크 구성

  1. 사용자 추가 대화 상자 하단에 있는 가입 보내기 링크 를 클릭합니다. 새 페이지가 표시됩니다.
  2. 가입 옵션을 구성합니다. (아래 참조)

이미지

Allow Signup 필드가 On으로 설정되어 있는지 확인하십시오. (필수) 사용하도록 설정하면 Case Active에서 고유한 링크를 생성합니다.

아래 옵션을 검토한 다음 아이콘을 클릭하여 링크를 복사합니다. 웹사이트에 링크를 배치하거나 이메일이나 통신 채널을 통해 한 명 이상의 사람에게 링크를 보낼 수 있습니다.

이미지

사례 만들기

선택적으로 각각의 새 사용자에 대한 사례를 자동으로 생성하고 사용자 역할을 할당할 수 있습니다.

  1. Auto Create Case 필드를 클릭하고 On을 선택합니다. 사례 사용자 역할 필드가 표시됩니다.
  2. 사례 사용자 역할을 클릭하고 옵션을 선택합니다. (아래 참조)

이미지

사례 사용자 역할

세 가지 케이스 사용자 역할이 있습니다.

  • 클라이언트
  • 기본
  • 직원

링크 보내기

한 명 이상의 사용자와 링크를 공유합니다. 각 사람은 링크를 클릭하고 워크플로를 완료하여 Case Active 계정을 활성화합니다. 사용자가 가입 워크플로를 완료하면 Case Active에서 사용자를 사용자 목록에 추가합니다.

이미지

다음 단계

Case Active는 가입 링크가 포함된 이메일 알림을 사용자에게 보냅니다. 사용자는 링크를 클릭하고 새 가입을 위한 워크플로를 완료합니다.

  • 이 도움말에서는 새 사용자를 위한 가입 절차에 대해 설명합니다.
