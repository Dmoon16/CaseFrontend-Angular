로그인 방법

기존 Case Active 계정에 로그인하는 방법을 알아보세요.

로그인

  1. 로그인 페이지 로 이동합니다.
  2. 로그인 버튼을 클릭합니다.
  3. 이메일을 입력하세요.
  4. 다음을 클릭합니다.

계정 확인

  1. 새 브라우저 탭을 열고 이메일 계정에 로그인합니다.
  2. 6자리 코드를 복사합니다.
  3. Case Active 탭으로 돌아갑니다.
  4. 필드에 코드를 붙여넣습니다. 새 창이 표시됩니다.

Case Active는 앱 페이지를 표시합니다.

  • 참고 플랫폼은 관리자 권한이 있는 사용자를 위해 APP 버튼 옆에 ADMIN 버튼을 표시합니다.
