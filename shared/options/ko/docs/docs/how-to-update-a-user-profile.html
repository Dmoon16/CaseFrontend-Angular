사용자 프로필을 업데이트하는 방법

이 도움말에서는 프로필 페이지에서 정보를 업데이트하는 방법을 사용자에게 보여줍니다.

  • 사용자는 자신의 프로필을 업데이트할 수 있습니다.
  • 관리자는 모든 사용자의 프로필 페이지를 업데이트할 수 있습니다.

사용자를 위해

첫 번째 단계는 프로필 페이지를 여는 것입니다. 사용자와 관리자는 다음 단계에 따라 자신의 프로필 페이지를 업데이트합니다.

  1. Case Active 계정에 로그인합니다.
  2. 페이지 상단의 프로필 탭을 클릭합니다.
  3. 변화시키다.
  4. 저장을 클릭합니다.

대부분의 필드는 선택 사항이며 일부 필드는 필수입니다. 필수 필드는 다음과 같습니다.

  • 이름
  • 성
  • 이메일 주소
  • 언어
  • 시간대

관리자용

관리자는 다른 사용자의 프로필 정보를 업데이트할 수도 있습니다. 여기에는 HOST USER INFO 제목 아래 세 개의 필드가 포함됩니다.

  1. Case Active 관리자 계정에 로그인합니다.
  2. 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 상단 메뉴에서 사용자를 클릭합니다.
  4. 하위 메뉴로 이동하여 액세스하려는 사용자 목록(ACTIVE 또는 DISABLED)을 선택합니다. 기본 설정은 활성입니다.
  5. 목록에서 사용자를 찾고 편집을 클릭합니다.
  6. 필요에 따라 변경합니다. (아래 참조)
  7. 저장을 클릭합니다.

사용자 정보 변경

관리자는 사용자 정보를 변경할 수 있습니다.

  1. 사용자 유형(사용자 또는 관리자)을 설정합니다.
  2. 사용자 상태를 설정합니다(활성에서 비활성화됨).
  3. 고유한 태그 ID를 입력합니다. (선택 과목)
  4. 저장을 클릭합니다.
