관리자가 사례에 대한 사용자 할당을 관리하는 방법

사용자 및 사례 할당을 관리하는 방법에는 5가지가 있습니다. 방법을 알아보려면 링크를 클릭하십시오.

방법 유형 행위 세부
케이스를 만들면서 UI 추가하다 관리자 버튼 > 사례 > 신규 버튼 > 사용자 할당 버튼
공개 가입 링크를 사용하여 사용자 자동 할당 사용자 작업 추가하다 관리자 버튼 > 설정 > 시스템 설정 페이지
기존 사례에서 UI 사용 UI 추가 및 제거 관리 버튼 > 사례 > 사용자 할당
가져오기 도구 UI 추가, 수정 및 제거 관리 버튼 > 통합 > 가져오기
API 사용(출시 예정) API 추가, 수정 및 제거 해당 없음
