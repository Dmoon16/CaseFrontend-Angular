공개 가입 링크로 사례를 자동 생성하는 방법

이 문서는 관리자에게 단일 가입 링크를 전송하여 케이스를 생성하는 방법(동시에 사용자를 추가하는 방법)을 보여줍니다.

링크 찾기

  1. 관리자 계정에 로그인합니다.
  2. 앱 페이지에서 앱을 찾아 관리자 버튼을 클릭합니다.
  3. 관리자 패널에서 설정 메뉴 탭을 클릭합니다.
  4. SYSTEM SETUP 하위 메뉴 탭이 선택되었는지 확인하십시오.

설정 구성

USER SIGNUP 제목까지 아래로 스크롤합니다.

가입 허용 필드가 켜짐으로 설정되어 있는지 확인합니다. (필수) 이 기능을 활성화하면 Case Active에서 고유한 링크를 생성합니다.

사례 사용자 역할

세 가지 케이스 사용자 역할이 있습니다.

  • 클라이언트(권장)
  • 기본

링크 보내기

한 명 이상의 사용자와 링크를 공유합니다. 링크를 클릭하고 가입을 완료하는 각 사용자는 플랫폼을 트리거하여 사례 페이지에서 새 사례를 생성하고 해당 사례에 사용자를 자동으로 할당합니다.
