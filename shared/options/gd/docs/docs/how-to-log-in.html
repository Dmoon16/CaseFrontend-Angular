Ciamar a Log a-steach

Ionnsaich mar a logas tu a-steach don chunntas Case Active agad mu thràth.

Log a-steach

  1. Rach gu duilleag logadh a-steach.
  2. Cliog air a’ phutan LOGIN.
  3. Cuir a-steach am post-d agad.
  4. Cliog air Next.

A 'dearbhadh cunntas

  1. Fosgail taba brobhsair ùr agus log a-steach don chunntas post-d agad.
  2. Dèan lethbhreac den chòd sia-dhigitach.
  3. Till gu tab Case Active.
  4. Cuir a-steach an còd a-steach do na raointean. Tha uinneag ùr a’ nochdadh.

Bidh Case Active a’ taisbeanadh duilleag nan Apps.

  • Nota Bidh an àrd-ùrlar a’ taisbeanadh putan ADMIN ri taobh putan APP airson luchd-cleachdaidh le sochairean rianachd.
