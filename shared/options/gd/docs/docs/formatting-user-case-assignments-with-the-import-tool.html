Cruthachadh Dàta Sònrachaidhean Cùis Cleachdaiche leis an Inneal Import

Gu h-ìosal tha liosta de raointean dàta ann am faidhlichean CSV. Tha measgachadh eadar-dhealaichte de raointean aig gach roghainn in-mhalairt.

cùis_ID

  • Cuir a-steach a’ chùis Cleachdaiche no ID Tag.
  • Rach gu duilleag Cùisean> Deasaich> Tag ID NO Rach gu duilleag Cùisean> Deasaich> ID cùise san url.

ID cleachdaiche_

  • Cuir a-steach ID an neach-cleachdaidh no ID Tag.
  • Rach gu duilleag luchd-cleachdaidh> Deasaich> Tag ID NO Rach gu duilleag luchd-cleachdaidh> Deasaich> ID cleachdaiche san url.

cùis_ròl_òrdugh

  • Fàg an cealla falamh.

cùis_ròl_ID

  • Cuir a-steach: teachdaichean, luchd-obrach, no bunaiteach.

cùis_fios

  • 0 = chan eil
  • 1 = Seadh, cuir fios post-d chun neach-cleachdaidh.
