Nola sortu kasuak automatikoki erregistratzeko esteka publiko batekin

Artikulu honek administratzaileei kasu bat nola sortu (eta erabiltzaile bat aldi berean gehitu) erakusten die erregistratzeko esteka bakarra bidaliz.

Esteka kokatzea

  1. Hasi saioa administratzaile-kontuan.
  2. Bilatu aplikazio bat Aplikazioen orrian eta egin klik ADMIN. botoian.
  3. Admin panelean, egin klik Ezarpenak menuko fitxan.
  4. Ziurtatu SISTEMA KONFIGURAZIOA azpimenuaren fitxa hautatuta dagoela.

Ezarpenak konfiguratzea

Joan behera ERABILTZAILEAREN ERREGISTROA goibururaino.

Ziurtatu Baimendu erregistroa eremua Aktibatuta dagoela. (beharrezkoa) Eginbide hau gaituta dagoenean, Case Activek esteka esklusibo bat sortzen du.

  1. Egin klik Sortu kasua automatikoki eremuan eta hautatu Aktibatu. Kasuaren erabiltzaile-rola eremua bistaratzen da.
  2. Hautatu rol bat. (ikus behean)

Kasuaren erabiltzaile-rolak

Hiru kasuko erabiltzaile-rol daude.

  • bezeroak (gomendatua)
  • lehenetsia
  • langileak

Esteka bidaltzen

Partekatu esteka pertsona batekin edo gehiagorekin. Estekan klik egiten duen eta erregistratzea amaitzen duen erabiltzaile bakoitzak plataforma abiarazten du kasu berri bat sortzeko kasuen orrian eta erabiltzailea kasuan kasuko automatikoki esleitzeko.
