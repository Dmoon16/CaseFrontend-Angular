Nola automatikoki esleitu erabiltzaileak Erregistratzeko esteka publikoarekin

Administratzaileek sistemaren ezarpenak konfiguratzen ikasten dute, erabiltzaile berriak automatikoki automatikoki sortu berri den kasu bati esleitzeko.

Konfigurazio orria irekitzen

Lehen urratsa Sistemaren konfigurazio orria irekitzea da.

  1. Hasi saioa administratzaile-kontuan.
  2. Bilatu aplikazio bat Aplikazioen orrian eta egin klik ADMIN. botoian.
  3. Sakatu Sistemak fitxan eta, ondoren, SISTEMA KONFIGURAZIOA azpimenuan.
  4. Joan behera ERABILTZAILEAREN ERREGISTROA goibururaino.

Sistemaren ezarpenak konfiguratzea

  1. Ezarri Baimendu erregistroa eta Sortu kasua automatikoki goitibeherako menuak Aktibatu gisa.
  2. Egin klik rolaren menuan eta aukeratu aukera bat (lehenetsia gomendatzen da).
  3. Kopiatu eta partekatu Erabiltzaile Publikoaren Erregistratzeko esteka.

Esteka bidaltzen

Estekan sakatu eta erregistratzea osatzen duen erabiltzaile bakoitzak kasu berri bat sortzeko plataforma abiarazten du Kasuak orrian eta kasu horretan erabiltzailea automatikoki esleitzen du.
