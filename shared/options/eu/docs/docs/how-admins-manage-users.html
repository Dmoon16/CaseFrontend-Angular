Administratzaileek erabiltzaileak nola kudeatzen dituzten

Administratzaileek sortu, aldatu edo desgaitu nor atzi dezaketen aplikazioak eta kasuak. Erabiltzaile bat plataforman lehen aldiz saioa hasten denean, profilaren lan-fluxu bat osatu behar du.

  • Egin klik hemen erabiltzailearen profil berriaren lan-fluxuari buruz gehiago jakiteko.

Erabiltzaile bat kudeatzea

Beheko taulak erabiltzaile bat kudeatzeko bost modu erakusten ditu. Egin klik estekan nola ikasteko.

Metodoa Mota Ekintzak Xehetasunak
UI erabiliz UI gehitu eta aldatu Admin botoia > Erabiltzaileak
Kasu bat sortzean eta erabiltzailea kasuari esleitu UI gehitu Admin botoia > Kasuak > Berria botoia > Sortu kasua orria
Sortu automatikoki erabiltzailea erregistratzeko esteka publikoa erabiliz Erabiltzailearen ekintza gehitu Admin botoia > Ezarpenak > Sistemaren konfigurazioa orria
Inportatzeko tresna UI gehitu eta aldatu Admin botoia > Integrazioak > Inportazioak
APIa erabiltzen (laster egongo da) APIa gehitu eta aldatu N/A

Jakinarazpenak

Plataformak ez die erabiltzaileei jakinarazpenik bidaltzen aplikazio batera gehitzen direnean. Hala ere, administratzaileek kasuen esleipenetarako jakinarazpenak gaitu ditzakete.
