Inportazio tresnarekin kasuak kudeatzeko datuak formateatzea

Jarraian, inportazio tresnarekin kasuak kudeatzeko CSV fitxategietan beharrezkoak diren eremuen zerrenda dago.

erabiltzaile-izena

  • Sartu erabiltzailearen helbide elektronikoa.

tag_id (geroago erreferentzia dezakezun ID bat)

  • identifikatzaile bakarra (erabili karaktere alfanumerikoek, marratxoak onartzen dira)

kasu_egoera

  • 0 = itxia
  • 1 = aktiboa

ireki_data

  • hilabete/egun/urtea
  • mm/dd/aaaa

buruz

  • kasuaren deskribapen bakarra (erabili karaktere alfanumerikoek, marratxoak onartzen dira)

irudia
