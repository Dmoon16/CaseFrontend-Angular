Nola eguneratu erabiltzaile-profila

Artikulu honek erabiltzaileei beren Profil orrialdeko informazioa nola eguneratu erakusten die.

  • Erabiltzaileek beren profilak egunera ditzakete.
  • Administratzaileek edozein erabiltzaileren profil orria egunera dezakete.

Erabiltzaileentzat

Lehen urratsa Profil orria irekitzea da. Erabiltzaileek eta administratzaileek urrats hauek jarraitzen dituzte beren Profil orria eguneratzeko.

  1. Hasi saioa zure Case Active kontuan.
  2. Egin klik orriaren goialdean dagoen Profila fitxan.
  3. Egin aldaketak.
  4. Sakatu Gorde.

Eremu gehienak aukerakoak dira, eta eremu batzuk beharrezkoak dira. Beharrezko eremuak hauek dira:

  • Izena
  • Abizena
  • Posta elektroniko helbidea
  • Hizkuntza
  • Ordu eremu

Administratzaileentzat

Administratzaileek beste erabiltzaile batzuen profileko informazioa ere egunera dezakete. Honek HOST ERABILTZAILEAREN INFORMAZIOA goiburuko hiru eremu biltzen ditu.

  1. Hasi saioa zure Case Active administratzaile-kontuan.
  2. Bilatu aplikazio bat eta egin klik ADMIN. botoian.
  3. Sakatu Erabiltzaileak goiko menuan.
  4. Joan azpimenura eta hautatu sartu nahi duzun erabiltzaile-zerrenda (AKTIBOA edo DESGAITU). Ezarpen lehenetsia Aktiboa da.
  5. Bilatu erabiltzaile bat zerrendan eta egin klik Editatu.
  6. Egin aldaketak behar bezala. (ikus behean)
  7. Sakatu Gorde.

Aldatu erabiltzailearen informazioa

Administratzaileek erabiltzailearen informazioa alda dezakete.

  1. Ezarri Erabiltzaile Mota (Erabiltzailea edo Kudeatzailea).
  2. Ezarri erabiltzailearen egoera (aktiboa desgaituta).
  3. Sartu etiketa ID esklusibo bat. (aukerakoa)
  4. Sakatu Gorde.
