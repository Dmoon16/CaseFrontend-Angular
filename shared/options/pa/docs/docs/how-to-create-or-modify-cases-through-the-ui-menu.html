UI ਮੀਨੂ ਰਾਹੀਂ ਕੇਸਾਂ ਨੂੰ ਕਿਵੇਂ ਬਣਾਉਣਾ ਜਾਂ ਸੋਧਣਾ ਹੈ

ਇਹ ਲੇਖ ਪ੍ਰਸ਼ਾਸਕਾਂ ਨੂੰ ਦਿਖਾਉਂਦਾ ਹੈ ਕਿ ਪਲੇਟਫਾਰਮ ਦੇ ਉਪਭੋਗਤਾ ਇੰਟਰਫੇਸ ਨਾਲ ਕੇਸ ਕਿਵੇਂ ਬਣਾਇਆ ਜਾਵੇ। ਐਡਮਿਨ ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ ਨਵੇਂ ਕੇਸ ਵਿੱਚ ਸ਼ਾਮਲ ਕਰ ਸਕਦੇ ਹਨ ਅਤੇ ਮੌਜੂਦਾ ਕੇਸਾਂ ਨੂੰ ਅਪਡੇਟ ਕਰ ਸਕਦੇ ਹਨ।

ਕੇਸ ਬਣਾ ਰਿਹਾ ਹੈ

  1. ਐਡਮਿਨ ਖਾਤੇ ਵਿੱਚ ਲੌਗ ਇਨ ਕਰੋ।

  4. ਨਵਾਂ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਇੱਕ ਨਵੀਂ ਵਿੰਡੋ ਦਿਖਾਈ ਦਿੰਦੀ ਹੈ।
