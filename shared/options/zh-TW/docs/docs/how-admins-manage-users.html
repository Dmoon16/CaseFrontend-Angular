管理員如何管理用戶

管理員創建、修改或禁用誰可以訪問應用程序和案例。當用戶首次登錄平台時，他們需要完成一個配置文件工作流程。

  • 單擊此處了解有關新用戶配置文件工作流程的更多信息。

管理用戶

下表顯示了五種管理用戶的方法。單擊鏈接了解操作方法。

方法類型動作細節
使用界面用戶界面添加和修改管理按鈕 > 用戶
創建案例並將用戶分配給案例時用戶界面添加管理按鈕 > 案例 > 新建按鈕 > 創建案例頁面
使用公共註冊鏈接自動創建用戶用戶操作添加管理按鈕 > 設置 > 系統設置頁面
導入工具用戶界面添加和修改管理按鈕 > 集成 > 導入
使用 API（即將推出）應用程序接口添加和修改不適用

通知

當用戶被添加到應用程序時，該平台不會向用戶發送通知。但是，管理員可以為個案分配啟用通知。
