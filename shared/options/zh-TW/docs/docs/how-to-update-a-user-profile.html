如何更新用戶資料

本文向用戶展示如何更新其個人資料頁面上的信息。

  • 用戶可以更新自己的個人資料。
  • 管理員可以更新任何用戶的個人資料頁面。

對於用戶

第一步是打開個人資料頁面。用戶和管理員按照以下步驟更新他們自己的個人資料頁面。

  1. 登錄到您的案例活動帳戶。
  2. 單擊頁面頂部的配置文件選項卡。
  3. 做出改變。
  4. 單擊保存。

大多數字段是可選的，有些字段是必需的。必填字段是：

  • 名
  • 姓
  • 電子郵件地址
  • 語
  • 時區

對於管理員

管理員還可以更新其他用戶的個人資料信息。這包括主機用戶信息標題下的三個字段。

  1. 登錄到您的 Case Active 管理員帳戶。
  2. 找到一個應用程序並單擊“管理”按鈕。
  3. 單擊頂部菜單上的用戶。
  4. 轉到子菜單並選擇您要訪問的用戶列表（活動或禁用）。默認設置為活動。
  5. 在列表中找到一個用戶並單擊編輯。
  6. 根據需要進行更改。 （見下文）
  7. 單擊保存。

更改用戶信息

管理員可以更改用戶信息。

  1. 設置用戶類型（用戶或管理員）。
  2. 設置用戶狀態（活動到禁用）。
  3. 輸入唯一的標記 ID。 （可選的）
  4. 單擊保存。
