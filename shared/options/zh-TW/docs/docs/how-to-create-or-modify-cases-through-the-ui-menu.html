如何通過 UI 菜單創建或修改案例

本文向管理員展示瞭如何使用平台的用戶界面創建案例。管理員還可以將用戶添加到新案例並更新現有案例。

創建案例

  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 在管理面板上，單擊案例菜單選項卡。
  4. 單擊新建按鈕。將顯示一個新窗口。

填寫必填字段並單擊保存。

添加用戶

或者，管理員可以在此頁面上將用戶添加到個案。向下滾動並單擊分配用戶按鈕。

圖片

在分配用戶模式中，填寫字段。選中該框向用戶發送電子郵件通知。 （可選的）

修改案例

要更新少量個案，請使用“編輯”工具並對每個個案進行更改。

  • 注意要一次性更新大量案例，請考慮使用導入工具
  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 在管理面板上，單擊案例菜單選項卡。
  4. 找到案例並單擊編輯。
  5. 進行更改並單擊“保存”。
