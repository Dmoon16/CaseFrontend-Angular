如何創建應用程序

本文向新用戶展示如何創建免費試用帳戶。 Case Active 會自動授予新用戶對此帳戶的管理員權限。

創建一個帳戶

  1. 轉到登錄頁面。
  2. 單擊開始按鈕。
  3. 輸入電子郵件地址。
  4. 點擊下一步。

圖片

驗證帳戶

  1. 打開一個新的瀏覽器選項卡並登錄到您的電子郵件帳戶。
  2. 複製六位數代碼。
  3. 返回案例活動選項卡。
  4. 將代碼粘貼到字段中。將顯示一個新窗口。

在新窗口中：

  1. 填寫必填字段。
  2. 閱讀服務條款並勾選方框表示同意。
  3. 單擊創建一個免費帳戶。

圖片

完成應用設置

工作流顯示在下一頁上。完成這些步驟以完成設置您的應用程序。您可以隨時返回這些頁面來編輯或更新信息或您的帳戶狀態。

在“關於應用程序”頁面上，填寫公司名稱和網站字段（必填），然後單擊“下一步”。

圖片

在“賬單信息”頁面上，單擊“付款計劃”字段並查看選項。默認設置為免費。單擊提交。

圖片

提交表單後，Case Active 會顯示 Apps 頁面。對於您創建或屬於的每個應用程序，該頁面顯示：

  1. 應用名稱
  2. 您的類型（用戶角色）
  3. 地位

圖片

應用程序頁面是許多管理功能的啟動點：

  1. 單擊 APP 按鈕可就案例相關問題進行交流、填寫文檔、視頻聊天（以及更多）。
  2. 單擊管理按鈕打開管理面板。這是管理員管理用戶、案例、集成等的地方。
  3. 單擊省略號以管理帳戶賬單和通知設置。
  • 注意Case Active 為管理員用戶顯示 ADMIN 按鈕。沒有管理員權限的用戶看不到管理員按鈕（如下圖所示）。
