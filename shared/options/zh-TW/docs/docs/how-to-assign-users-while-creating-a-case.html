如何在創建案例時分配用戶

本文向管理員展示瞭如何在創建新案例時分配用戶。如果用戶尚未在平台上註冊，您可以在創建新案例時創建添加新用戶。

創建案例

  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 單擊個案菜單選項卡。
  4. 單擊新建按鈕。將顯示一個新窗口。

圖片

  • 填寫字段以創建案例，然後向下滾動到分配的用戶標題。

分配用戶

當前用戶

按照以下步驟將當前用戶分配給新案例。

單擊分配用戶按鈕並填寫字段。

  1. 單擊用戶下拉菜單並選擇一個標識符（例如電子郵件）。
  2. 在空白字段中輸入一個值（例如電子郵件地址）。
  3. 如果用戶記錄存在於平台中，該字段會自動建議一個名稱。
  4. 選擇正確的用戶。
  5. 選擇用戶角色（客戶、默認或員工）。
  6. 或者，選中該框以向用戶發送電子郵件通知。
  7. 單擊保存。

圖片

Case Active 將用戶添加到案例中。現在管理員可以單擊“保存”關閉窗口，或添加新用戶或刪除用戶。

圖片

添加新用戶

如果平台找不到當前記錄，點擊按鈕添加新用戶。

圖片

填寫字段，然後單擊保存以將用戶添加到平台。接下來，設置用戶角色並再次單擊“保存”以將用戶分配給案例。最後，再次單擊 SAVE 以保存案例。
