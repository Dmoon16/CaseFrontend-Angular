使用導入工具格式化用戶案例分配數據

以下是 CSV 文件中的數據字段列表。每個導入選項都有不同的字段組合。

案例ID

  • 輸入案例用戶或標籤 ID。
  • 轉到案例頁面 > 編輯 > 標籤 ID 或轉到案例頁面 > 編輯 > 網址中的案例 ID。

用戶身份

  • 輸入用戶 ID 或標記 ID。
  • 轉到用戶頁面 > 編輯 > 標記 ID 或轉到用戶頁面 > 編輯 > url 中的用戶 ID。

case_role_order

  • 將單元格留空。

case_role_ID

  • 輸入：客戶、員工或默認值。

案例通知

  • 0 = 否
  • 1 = 是，向用戶發送電子郵件通知。
