如何使用導入工具分配、修改或刪除用戶分配

本文向管理員展示如何使用導入工具管理案例用戶。

管理員可以：

  • 將用戶分配給案例
  • 修改已分配給案例的用戶
  • 刪除分配給案例的用戶

打開導入工具

  1. 登錄到管理員帳戶。
  2. 在“應用程序”頁面上找到一個應用程序，然後單擊“管理”按鈕。
  3. 單擊 Integration 選項卡，然後單擊子菜單上的 IMPORTS。
  4. 單擊新建按鈕。顯示添加導入對話框。

創建記錄

下載一個或多個 CSV 文件。使用這些文件來添加、修改或刪除案例用戶。添加導入對話框打開後：

  1. 單擊類型字段。
  2. 滾動下拉菜單並選擇一個選項（見下文）。顯示一個鏈接。
  3. 單擊下載示例鏈接。 CSV 文件將下載到您的設備。

打開文件並添加記錄。單擊此處了解如何在 CSV 文件中使用格式數據。

文件選項

將用戶分配給個案* 選擇此選項可將用戶添加到個案。 * 將 case_role_order 單元格留空。 * 轉到“用戶”頁面並複制此人的標籤 ID。如果標記 ID 字段為空，請輸入唯一 ID，單擊保存，然後將 ID 複製到 CSV 文件。

修改分配給個案的用戶* 選擇此選項可更新分配給個案的當前用戶的數據。 * 對 case_ID 列使用案例標籤 ID。

圖片

刪除分配給個案的用戶* 選擇此選項可刪除分配給個案的用戶。

上傳文件

您可以立即上傳文件或稍後返回添加導入對話框並完成這些步驟。

  1. 打開添加導入對話框。
  2. 輸入 CSV 文件的唯一名稱。
  3. 單擊加號 (+) 圖標從您的設備上傳文件。
  4. 單擊類型字段並選擇一個選項。 （看上面）
  5. 單擊保存。

加工

根據文件中記錄的數量和系統負載，平台可能需要幾分鐘時間來處理導入。

圖片

導入完成後，平台會在案例頁面調整用戶記錄（即添加、修改或刪除）。

失敗

如果導入操作失敗，平台將返回失敗消息。在大多數情況下，失敗的原因是數據格式化。

圖片

查看失敗原因：

  1. 單擊“操作”標題下的“查看”。
  2. 在下一頁上，將鼠標懸停在“失敗”圖標上。工具提示顯示錯誤。
  3. 轉到 CVS 文件並調整併修復錯誤。
  4. 再次上傳文件。
