Ինչպես նշանակել, փոփոխել կամ հեռացնել օգտատերերի հանձնարարությունները ներմուծման գործիքի միջոցով

Այս հոդվածը ցույց է տալիս ադմինիստրատորներին, թե ինչպես կառավարել գործի օգտվողներին Ներմուծման գործիքի միջոցով:

Ադմինները կարող են.

  • օգտվողներին հանձնարարել գործին
  • փոփոխել օգտատերերին, որոնք արդեն հանձնարարված են գործին
  • հեռացնել գործին հանձնարարված օգտվողներին
