Ինչպես են ադմինները ստեղծում օգտատերեր

Ադմինիստրատորները ստեղծում, փոփոխում կամ անջատում են, թե ովքեր կարող են մուտք գործել հավելվածներ և պատյաններ: Երբ օգտվողն առաջին անգամ մուտք է գործում հարթակ, նա պետք է ավարտի պրոֆիլի աշխատանքային հոսքը:

  • Սեղմեք այստեղ ՝ օգտվողի նոր պրոֆիլի աշխատանքային հոսքի մասին ավելին իմանալու համար:

Օգտագործողի ստեղծում

Ստորև բերված աղյուսակը ցույց է տալիս օգտվող ստեղծելու հինգ եղանակ: Սեղմեք հղման վրա՝ իմանալու համար, թե ինչպես:
