Ինչպես են ադմինները կառավարում օգտատերերին

Ադմինիստրատորները ստեղծում, փոփոխում կամ անջատում են, թե ովքեր կարող են մուտք գործել հավելվածներ և պատյաններ: Երբ օգտվողն առաջին անգամ մուտք է գործում հարթակ, նա պետք է ավարտի պրոֆիլի աշխատանքային հոսքը:

  • Սեղմեք այստեղ ՝ օգտվողի նոր պրոֆիլի աշխատանքային հոսքի մասին ավելին իմանալու համար:

Օգտագործողի կառավարում

Ստորև բերված աղյուսակը ցույց է տալիս օգտվողին կառավարելու հինգ եղանակ: Սեղմեք հղման վրա՝ իմանալու համար, թե ինչպես:

Մեթոդ Տիպ Գործողություններ Մանրամասներ
Օգտագործելով UI UI ավելացնել և փոփոխել Ադմինիստրատորի կոճակ > Օգտագործողներ
Գործ ստեղծելիս և օգտատիրոջը գործին հանձնարարել UI ավելացնել Ադմինիստրատորի կոճակ > Գործեր > Նոր կոճակ > Ստեղծել գործի էջ
Ավտոմատ ստեղծել օգտվող՝ օգտագործելով հանրային գրանցման հղումը Օգտագործողի գործողություն ավելացնել Ադմինիստրատորի կոճակ > Կարգավորումներ > Համակարգի կարգավորումների էջ
Ներմուծման գործիք UI ավելացնել և փոփոխել Ադմինիստրատորի կոճակ > Ինտեգրումներ > Ներմուծումներ
API-ի օգտագործում (շուտով) API ավելացնել և փոփոխել N/A
