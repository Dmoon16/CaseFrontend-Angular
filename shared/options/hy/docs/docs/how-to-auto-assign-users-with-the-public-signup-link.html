Ինչպես ավտոմատ կերպով նշանակել օգտվողներին հանրային գրանցման հղումով

Ադմինիստրատորները սովորում են, թե ինչպես կարգավորել համակարգի կարգավորումներն այնպես, որ նոր օգտվողները ավտոմատ կերպով նշանակվեն նոր ավտոմատ ստեղծված պատյան:

Կարգավորման էջը բացելը

Առաջին քայլը Systems Setup էջը բացելն է:

  1. Մուտք գործեք ադմինիստրատորի հաշիվ:
  2. Տեղադրեք հավելվածը Apps էջում և սեղմեք ADMIN կոճակը:
  3. Սեղմեք «Համակարգեր» ներդիրը, ապա «ՀԱՄԱԿԱՐԳԻ ԿԱՐԳԱՎՈՐՈՒՄ» ենթամենյուում:
  4. Ոլորեք ներքև դեպի ՕԳՏԱԳՈՐԾՈՂԻ ԳՐԱՆՑՈՒՄ վերնագիրը:

Համակարգի կարգավորումների կարգավորում

  1. Սահմանեք «Թույլատրել գրանցումը» և «Auto Create Case» բացվող ընտրացանկերը «Միացված»:
  2. Սեղմեք դերերի ընտրացանկը և ընտրեք որևէ տարբերակ (խորհուրդ է տրվում լռելյայն):
  3. Պատճենեք և տարածեք Հանրային օգտվողի գրանցման հղումը:

Ուղարկում է հղումը
