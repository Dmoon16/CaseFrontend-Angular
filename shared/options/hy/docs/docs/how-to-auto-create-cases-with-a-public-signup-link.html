Ինչպես ավտոմատ ստեղծել պատյաններ հանրային գրանցման հղումով

Այս հոդվածը ցույց է տալիս ադմինիստրատորներին, թե ինչպես ստեղծել գործ (և միաժամանակ ավելացնել օգտատեր)՝ ուղարկելով գրանցման մեկ հղում:

Հղման տեղորոշում

  1. Մուտք գործեք ադմինիստրատորի հաշիվ:
  2. Տեղադրեք հավելվածը Apps էջում և սեղմեք ADMIN կոճակը:
  3. Ադմինիստրատորի վահանակում սեղմեք «Կարգավորումներ» ցանկի ներդիրը:
  4. Համոզվեք, որ SYSTEM SETUP ենթամենյուի ներդիրն ընտրված է:

Կարգավորումների կարգավորում

Ոլորեք ներքև դեպի ՕԳՏԱԳՈՐԾՈՂԻ ԳՐԱՆՑՈՒՄ վերնագիրը:

Համոզվեք, որ «Թույլատրել գրանցումը» դաշտը միացված է: (պարտադիր է) Երբ այս հատկությունը միացված է, Case Active-ը ստեղծում է եզակի հղում:

  1. Սեղմեք Auto Create Case դաշտը և ընտրեք Միացնել: Ցուցադրվում է Case User Role դաշտը:
  2. Ընտրեք դեր: (տես ներքեւում)

Գործի օգտագործողի երեք դեր կա:

  • հաճախորդներ (խորհուրդ է տրվում)
  • լռելյայն
  • անձնակազմը

Ուղարկում է հղումը

Կիսվեք հղումով մեկ կամ մի քանի մարդկանց հետ: Յուրաքանչյուր օգտատեր, ով կտտացնում է հղումը և ավարտում գրանցումը, գործարկում է հարթակը գործերի էջում նոր գործ ստեղծելու և գործի մեջ օգտագործողին ավտոմատ նշանակելու համար:
