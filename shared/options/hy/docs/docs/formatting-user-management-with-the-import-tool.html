Օգտագործողի կառավարման տվյալների ձևաչափում ներմուծման գործիքի միջոցով

Ստորև բերված է CSV ֆայլերի տվյալների դաշտերի ցանկը: Ներմուծման յուրաքանչյուր տարբերակ ունի դաշտերի տարբեր համակցություն:

օգտագործողի անունը

  • Մուտքագրեք օգտվողի էլփոստի հասցեն:

տրված անուն

  • Մուտքագրեք օգտվողի անունը:

Ազգանուն

  • Մուտքագրեք օգտվողի ազգանունը:

տեղայնությունը

  • Այս սյունակը ստուգում է օգտագործողի նախընտրած լեզուն:
  • Մուտքագրեք en անգլերենի համար, sp՝ իսպաներենի համար:

zoneinfo

  • Սա սահմանում է օգտագործողի ժամային գոտին: Մուտքագրեք մայրցամաքը և քաղաքը առանց բացատների (օգտագործեք ընդգծում):
  • Պետք է օգտագործի մի տարբերակ Պրոֆիլ > LANGUAGE & REGION > Ժամային գոտի բացվող ընտրացանկում:
  • Օգտագործեք ամենահարմար ժամային գոտին: Օրինակ, Բոստոնը բացվող ցանկում չէ, այնպես որ մենք կարող ենք օգտագործել Նյու Յորքը (կամ EST/EDT ժամային գոտու ցանկացած քաղաք):

host_tag_id

  • Մուտքագրեք մշտական պիտակ:
  • az, 0-9, -, / թույլատրվում է.

դեպք_կարգավիճակ

  • 0 = փակ
  • 1 = ակտիվ
