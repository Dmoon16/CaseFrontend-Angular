Ինչպես ստեղծել կամ փոփոխել օգտվողներին ներմուծման գործիքի միջոցով

Այս հոդվածը ցույց է տալիս ադմինիստրատորներին, թե ինչպես ստեղծել օգտվողներ Ներմուծման գործիքի միջոցով:

Ներմուծման գործիքի բացում

Ադմինիստրատորները կարող են օգտագործել Ներմուծման գործիքը՝ նոր օգտվողներ զանգվածաբար վերբեռնելու և առկա օգտատերերի գրառումները փոփոխելու համար:

  1. Մուտք գործեք ադմինիստրատորի հաշիվ:
  2. Տեղադրեք հավելվածը Apps էջում և սեղմեք ADMIN կոճակը:
  3. Ադմինիստրատորի վահանակում սեղմեք Ինտեգրման ներդիրը և այնուհետև ենթամենյուում՝ ՆԵՐՄՈՒԾՈՒՄ:
  4. Սեղմեք ՆՈՐ կոճակը: Ավելացնել ներմուծումներ երկխոսության տուփը ցուցադրվում է դիտարկիչի նոր ներդիրում:

Գրառումների ստեղծում

Ավելացնել ներմուծման երկխոսության տուփը բացվելուց հետո.

  1. Սեղմեք Type դաշտը և ընտրեք որևէ տարբերակ (տես ստորև): Ցուցադրվում է հղում:
  2. Սեղմեք Ներբեռնման նմուշի հղումը: Ձեր սարքում ներբեռնվում է CSV ֆայլ:

:exclamation. ԾԱՆՈԹԱԳՐՈՒԹՅՈՒՆ Բացվող ընտրացանկը շատ տարբերակներ ունի, բայց միայն երկուսն են կապված Ներմուծման գործիքի հետ։ Յուրաքանչյուր տարբերակ ունի տարբեր նմուշի CSV ֆայլ:

  • Կտտացրեք Ներբեռնման նմուշի հղմանը, որպեսզի բացեք CSV ֆայլ՝ սյունակների վերնագրերով «Ավելացնել օգտվողների ներմուծում» տեսակի համար:

Փոփոխել օգտվողներին

  • Կտտացրեք Ներբեռնման նմուշի հղմանը, որպեսզի բացեք CSV ֆայլ՝ սյունակների վերնագրերով Փոփոխել օգտվողների ներմուծման տեսակը:
