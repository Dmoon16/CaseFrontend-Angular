Ինչպես ստեղծել կամ փոփոխել գործերը ներմուծման գործիքի միջոցով

Այս հոդվածը ցույց է տալիս ադմինիստրատորներին, թե ինչպես ստեղծել կամ փոփոխել առկա դեպքերը Ներմուծման գործիքի միջոցով:

Ներմուծման գործիքի բացում

  1. Մուտք գործեք ադմինիստրատորի հաշիվ:
  2. Տեղադրեք հավելվածը Apps էջում և սեղմեք ADMIN կոճակը:
  3. Ադմինիստրատորի վահանակում սեղմեք Ինտեգրման ներդիրը և այնուհետև ենթամենյուում՝ ՆԵՐՄՈՒԾՈՒՄ:
  4. Սեղմեք ՆՈՐ կոճակը: Ավելացնել ներմուծումներ երկխոսության տուփը ցուցադրվում է դիտարկիչի նոր ներդիրում:

Պատյանների ստեղծում

Ավելացնել ներմուծման երկխոսության տուփը բացվելուց հետո.

  1. Սեղմեք Type դաշտը և ընտրեք որևէ տարբերակ (տես ստորև): Ցուցադրվում է հղում:
  2. Սեղմեք Ներբեռնման նմուշի հղումը: Ձեր սարքում ներբեռնվում է CSV ֆայլ:

Հաջորդը, բացեք ֆայլը և ավելացրեք դեպքեր:

Տիպ

Երբ սեղմում եք « Տիպ » դաշտը, բացվող ընտրացանկը ցուցադրվում է: Ընտրեք երկու տարբերակներից մեկը մեծաքանակ գործի ներմուծման համար:

  • Նշում Բացվող ընտրացանկը շատ տարբերակներ ունի, բայց միայն երկուսն են կապված գործի կառավարման հետ: Յուրաքանչյուր տարբերակ ունի տարբեր նմուշի CSV ֆայլ:

Ավելացնել դեպքեր

  • Ընտրեք այս տարբերակը դեպքերի հասարակ զանգվածային վերբեռնման համար:
  • Կտտացրեք Ներբեռնման նմուշի հղումը, որպեսզի բացեք CSV ֆայլ՝ սյունակների վերնագրերով Ստեղծել գործերի ներմուծման տեսակի համար:

Փոփոխել դեպքերը

  • Ընտրեք այս տարբերակը՝ ընթացիկ դեպքերը թարմացնելու համար:
  • Կտտացրեք Ներբեռնման նմուշի հղումը՝ բացելու CSV ֆայլ՝ սյունակների վերնագրերով Փոփոխել դեպքերի ներմուծման տեսակի համար:
