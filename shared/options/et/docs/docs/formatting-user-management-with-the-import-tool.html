Kasutajahaldusandmete vormindamine imporditööriistaga

Allpool on CSV-failide andmeväljade loend. Igal impordivalikul on erinev väljade kombinatsioon.

kasutajanimi

  • Sisestage kasutaja e-posti aadress.

antud nimi

  • Sisestage kasutaja eesnimi.

perekonnanimi

  • Sisestage kasutaja perekonnanimi.

lokaat

  • See veerg kontrollib kasutaja eelistatud keelt.
  • Sisestage en inglise keeles, sp hispaania keeles.

tsooniinfo

  • See määrab kasutaja ajavööndi. Sisestage kontinent ja linn ilma tühikuteta (kasutage alljoont).
  • Peab kasutama valikut Profiil > KEEL JA PIIRKOND > Ajavöönd.
  • Kasutage kõige sobivamat ajavööndit. Näiteks Bostonit pole rippmenüüs, seega saame kasutada New Yorki (või mis tahes linna EST/EDT ajavööndis).

host_tag_id

  • Sisestage püsiv silt.
  • az, 0-9, -, / on lubatud.

case_status

  • 0 = suletud
  • 1 = aktiivne
