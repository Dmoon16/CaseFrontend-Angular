Kuidas luua kasutajaid registreerumislingiga

See artikkel näitab administraatoritele, kuidas luua kasutajaid, saates registreerumislingi.

Registreerumislingi konfigureerimine

  1. Klõpsake dialoogiboksi Lisa kasutaja allosas linki Saada neile registreerumine . Kuvatakse uus leht.
  2. Konfigureerige registreerumisvalikud. (vt allpool)

Veenduge, et välja Luba registreerumine oleks seatud väärtusele Sees. (nõutav) Kui see on lubatud, loob Case Active ainulaadse lingi.

Vaadake allolevad valikud üle ja seejärel klõpsake lingi kopeerimiseks ikooni. Saate lingi oma veebisaidile paigutada või saata lingi ühele või mitmele inimesele e-posti või mis tahes suhtluskanali kaudu.

img

Juhtumi koostamine

Soovi korral saate iga uue kasutaja jaoks automaatselt juhtumi luua ja määrata kasutajarolli.

  1. Klõpsake väljal Auto Create Case ja valige Sees. Kuvatakse väljad Case User Role.
  2. Klõpsake juhtumikasutaja rolli ja tehke valik. (vt allpool)

img

Juhtumi kasutaja rollid

Juhtumikasutaja rolle on kolm.

  • kliendid
  • vaikimisi
  • töötajad

Lingi saatmine

Jagage linki ühe või mitme inimesega. Iga inimene klõpsab lingil ja lõpetab oma Case Active'i konto aktiveerimiseks töövoo. Kui kasutaja on registreerimistöövoo lõpetanud, lisab Case Active kasutaja kasutajate loendisse.

img

Järgmised sammud

Case Active saadab kasutajale registreerumislingiga meiliteate. Kasutaja klõpsab lingil ja lõpetab uute registreerumiste töövoo.

  • Selles artiklis kirjeldatakse uute kasutajate registreerimise töövoogu.
