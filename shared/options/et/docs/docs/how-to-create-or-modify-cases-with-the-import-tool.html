Juhtumite loomine või muutmine imporditööriistaga

See artikkel näitab administraatoritele, kuidas luua või muuta olemasolevaid juhtumeid imporditööriistaga.

Importimise tööriista avamine

  1. Logige sisse administraatori kontole.
  2. Leidke rakenduste lehelt rakendus ja klõpsake nuppu ADMIN.
  3. Klõpsake administraatoripaneelil vahekaarti Integratsioon ja seejärel alammenüüs IMPORTID.
  4. Klõpsake nuppu UUS. Dialoogiboks Import Import kuvatakse uuel brauseri vahekaardil.

img

Juhtumite loomine

Pärast dialoogiboksi Import Imports avanemist:

  1. Klõpsake väljal Tüüp ja tehke valik (vt allpool). Kuvatakse link.
  2. Klõpsake linki Laadi näidis alla. CSV-fail laaditakse teie seadmesse alla.

Järgmisena avage fail ja lisage juhtumid.

Tüüp

Kui klõpsate väljal Tüüp , kuvatakse rippmenüü. Valige hulgikohvrite importimiseks üks kahest võimalusest.

  • Märkus . Rippmenüüs on palju valikuid, kuid ainult kaks on seotud juhtumi haldamisega. Igal valikul on erinev CSV-faili näidis.

Lisa juhtumid

  • Valige see suvand juhtumite lihtsaks hulgiüleslaadimiseks.
  • Klõpsake lingil Laadi näidis alla, et avada CSV-fail veerupäistega imporditüübi Loo juhtumite jaoks.

Muuda juhtumeid

  • Valige see suvand praeguste juhtumite värskendamiseks.
  • Klikkige lingil Näidis allalaadimine, et avada CSV-fail koos veerupäistega juhtumite imporditüübi jaoks.

Faili üleslaadimine

Saate faili kohe üles laadida või hiljem naasta dialoogiboksi Importimise lisamine ja need sammud lõpule viia.

  1. Avage dialoogiboks Impordi lisamine.
  2. Sisestage CSV-faili kordumatu nimi.
  3. Faili seadmest üleslaadimiseks klõpsake plussmärgi (+) ikoonil.
  4. Klõpsake väljal Tüüp ja tehke valik. (vt eespool)
  5. Klõpsake SAVE.

img

Vormindamine

Need on kohustuslikud väljad ja esimesed nõuded. Vorminduse üksikasjade vaatamiseks klõpsake siin .

tag_id (või case_ID) kordumatu identifikaator (kasutage tähtnumbrilisi märke, sidekriipsud on lubatud)

case_status 0 = suletud 1 = aktiivne

kuupäev_avatud kuu/päev/aasta kk/pp/aaaa

väiketähtede kordumatu kirjelduse kohta (kasutage tähtnumbrilisi märke, sidekriipsud on lubatud)

img

Töötlemine

Sõltuvalt failis olevate kirjete arvust ja süsteemi laadimisest võib platvormil impordi töötlemiseks kuluda mitu minutit.

img

Kui importimine on lõppenud, kohandab platvorm juhtumikirjeid (st lisab või muudab) administraatori juhtumite armatuurlaual.

Ebaõnnestumine

Kui importimine ebaõnnestub, tagastab platvorm Failed teate. Enamasti on rikke põhjuseks andmete vormindamine.

Rikke põhjuse nägemiseks tehke järgmist.

  1. Klõpsake pealkirja TEGEVUS all nuppu Kuva.
  2. Järgmisel lehel hõljutage kursorit ebaõnnestunud ikoonil. Tööriistaspikker kuvab vea.
  3. Minge CVS-faili ning korrigeerige ja parandage viga.
  4. Laadige fail uuesti üles.
