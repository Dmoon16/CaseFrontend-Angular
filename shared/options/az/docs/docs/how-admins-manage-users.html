Adminlər istifadəçiləri necə idarə edir

Adminlər proqramlara və işlərə kimin daxil ola biləcəyini yaradır, dəyişdirir və ya söndürür. İstifadəçi platformaya ilk dəfə daxil olduqda, profil işini tamamlamalıdır.

  • Yeni istifadəçi profili iş prosesi haqqında ətraflı öyrənmək üçün buraya klikləyin.

İstifadəçini idarə etmək

Aşağıdakı cədvəl istifadəçini idarə etməyin beş yolunu göstərir. Necə olduğunu öyrənmək üçün linkə klikləyin.

Metod Növ Tədbirlər Təfərrüatlar
UI istifadə UI əlavə et və dəyişdir Admin düyməsi > İstifadəçilər
İş yaratarkən və istifadəçini işə təyin edin UI əlavə edin Admin düyməsi > İşlər > Yeni düymə > İş səhifəsi yaradın
İctimai qeydiyyat linkindən istifadə edərək avtomatik istifadəçi yaradın İstifadəçi hərəkəti əlavə edin Admin düyməsi > Parametrlər > Sistem Quraşdırma səhifəsi
İdxal aləti UI əlavə et və dəyişdir Admin düyməsi > İnteqrasiya > İdxal
API-dən istifadə (tezliklə) API əlavə et və dəyişdir Yoxdur

Bildirişlər

Platforma istifadəçilərə tətbiqə əlavə olunduqda bildiriş göndərmir. Bununla belə, adminlər iş tapşırıqları üçün bildirişləri aktivləşdirə bilər.
