İdxal Aləti ilə İstifadəçi Təyinatlarını Necə Təyin etmək, Dəyişdirmək və ya Silmək olar

Bu məqalə adminlərə İdxal aləti ilə iş istifadəçilərini necə idarə edəcəklərini göstərir.

Adminlər edə bilərlər:

  • istifadəçiləri işə təyin edin
  • artıq işə təyin edilmiş istifadəçiləri dəyişdirin
  • işə təyin edilmiş istifadəçiləri silin

İdxal alətinin açılması

  1. Admin hesabına daxil olun.
  2. Proqramlar səhifəsində proqram tapın və ADMIN düyməsini klikləyin.
  3. Alt menyuda İnteqrasiya sekmesini və sonra İDXALLAR üzərinə klikləyin.
  4. YENİ düyməsini basın. İdxal əlavə et dialoq qutusu göstərilir.

Qeydlərin yaradılması

Bir və ya daha çox CSV faylını endirin. İş istifadəçilərini əlavə etmək, dəyişdirmək və ya silmək üçün bu faylları istifadə edin. İdxal əlavə et dialoq qutusu açıldıqdan sonra:

  1. Növ sahəsinə klikləyin.
  2. Açılan menyuda sürüşdürün və bir seçim seçin (aşağıya baxın). Bir keçid göstərilir.
  3. Nümunə Yüklə linkinə klikləyin. CSV faylı cihazınıza endirilir.

Faylı açın və qeydlər əlavə edin. CSV faylında format məlumatlarından necə istifadə edəcəyinizi öyrənmək üçün bura klikləyin.

Fayl seçimləri

İstifadəçiləri işlərə təyin edin * İstifadəçiləri işə əlavə etmək üçün bu seçimi seçin. * Case_role_order xanasını boş qoyun. * İstifadəçilər səhifəsinə keçin və bu şəxs üçün Tag ID-ni kopyalayın. Tag ID sahəsi boşdursa, unikal ID daxil edin, Saxla düyməsini basın və sonra ID-ni CSV faylına köçürün.

İşlərə təyin edilmiş istifadəçiləri dəyişdirin * İşə təyin edilmiş cari istifadəçilər üçün məlumatları yeniləmək üçün bu seçimi seçin. * Case_ID sütunu üçün Case Tag ID-dən istifadə edin.

İşlərə təyin edilmiş istifadəçiləri silin * İşə təyin edilmiş istifadəçiləri silmək üçün bu seçimi seçin.

Fayl yüklənir

Siz faylı dərhal yükləyə və ya daha sonra İdxal Əlavə et dialoq qutusuna qayıda və bu addımları tamamlaya bilərsiniz.

  1. İdxal əlavə et dialoq qutusunu açın.
  2. CSV faylı üçün unikal ad daxil edin.
  3. Faylı cihazınızdan yükləmək üçün artı (+) işarəsinə klikləyin.
  4. Növ sahəsinə klikləyin və bir seçim seçin. (yuxarıya bax)
  5. YAXŞI Klikləyin.

Emal edilir

Fayldakı qeydlərin sayından və sistem yükündən asılı olaraq, platforma idxalı emal etmək üçün bir neçə dəqiqə çəkə bilər.

img

İdxal tamamlandıqda, platforma Cases səhifəsində istifadəçi qeydlərini düzəldir (yəni əlavə edir, dəyişdirir və ya silir).

Uğursuzluq

İdxal əməliyyatı uğursuz olarsa, platforma Uğursuz mesajı qaytarır. Əksər hallarda uğursuzluğun səbəbi məlumatların formatlaşdırılmasıdır.

img

Uğursuzluğun səbəbini görmək üçün:

  1. Fəaliyyət başlığı altında Baxın üzərinə klikləyin.
  2. Növbəti səhifədə siçanı Uğursuz simgesinin üzərinə gətirin. Alət ipucu xətanı göstərir.
  3. CVS faylına gedin və səhvi düzəldin və düzəldin.
  4. Faylı yenidən yükləyin.
