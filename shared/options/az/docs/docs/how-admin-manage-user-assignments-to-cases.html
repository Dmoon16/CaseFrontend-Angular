Adminlər İstifadəçi Təyinatlarını Keyslərə Necə İdarə Edirlər

İstifadəçiləri və iş tapşırıqlarını idarə etməyin beş yolu var. Necə olduğunu öyrənmək üçün linkə klikləyin.

Metod Növ Tədbirlər Təfərrüatlar
Dava yaratarkən UI əlavə edin Admin düyməsi > İşlər > Yeni düymə > İstifadəçiləri Təyin et düyməsi
İctimai qeydiyyat linkindən istifadə edərək istifadəçini avtomatik təyin edin İstifadəçi Fəaliyyəti əlavə edin Admin düyməsi > Parametrlər > Sistem Quraşdırma səhifəsi
Mövcud halda UI istifadə UI əlavə et və sil Admin düyməsi > İşlər > İstifadəçiləri təyin edin
İdxal aləti UI əlavə edin, dəyişdirin və silin Admin düyməsi > İnteqrasiya > İdxal
API-dən istifadə (Tezliklə) API əlavə edin, dəyişdirin və silin Yoxdur
