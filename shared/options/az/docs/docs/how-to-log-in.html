Necə Daxil olmaq olar

Mövcud Case Active hesabınıza necə daxil olmağı öyrənin.

Daxil olur

  1. Giriş səhifəsinə keçin .
  2. LOGIN düyməsini klikləyin.
  3. E-poçtunuzu daxil edin.
  4. NEXT klikləyin.

Hesabın doğrulanması

  1. Yeni bir brauzer nişanı açın və e-poçt hesabınıza daxil olun.
  2. Altı rəqəmli kodu kopyalayın.
  3. Case Active tabına qayıdın.
  4. Kodu sahələrə yapışdırın. Yeni bir pəncərə göstərilir.

Case Active Proqramlar səhifəsini göstərir.

  • Qeyd Platformada admin imtiyazları olan istifadəçilər üçün APP düyməsinin yanında ADMIN düyməsi göstərilir.
