İstifadəçi profilini necə yeniləmək olar

Bu məqalə istifadəçilərə Profil səhifəsində məlumatları necə yeniləməyi göstərir.

  • İstifadəçilər öz profillərini yeniləyə bilərlər.
  • Adminlər istənilən istifadəçinin Profil səhifəsini yeniləyə bilər.

İstifadəçilər üçün

İlk addım Profil səhifəsini açmaqdır. İstifadəçilər və adminlər öz Profil səhifələrini yeniləmək üçün bu addımları yerinə yetirirlər.

  1. Case Active hesabınıza daxil olun.
  2. Səhifənin yuxarısındakı Profil sekmesine klikləyin.
  3. Dəyişikliklər edin.
  4. Saxla klikləyin.

Əksər sahələr isteğe bağlıdır və bəzi sahələr tələb olunur. Tələb olunan sahələr bunlardır:

  • Ad
  • Soyad
  • E-poçt ünvanı
  • Dil
  • Saat qurşağı

Adminlər üçün

Adminlər həmçinin digər istifadəçilər üçün profil məlumatlarını yeniləyə bilərlər. Buraya HOST USER INFO başlığı altında üç sahə daxildir.

  1. Case Active admin hesabınıza daxil olun.
  2. Proqramı tapın və ADMIN düyməsini basın.
  3. Üst menyuda İstifadəçilər düyməsini basın.
  4. Alt menyuya gedin və daxil olmaq istədiyiniz istifadəçi siyahısını seçin (AKTİV və ya QEYDİ). Defolt parametr Aktivdir.
  5. Siyahıda istifadəçi tapın və Redaktə et düyməsini basın.
  6. Lazım olduğu kimi dəyişikliklər edin. (aşağıya bax)
  7. Saxla klikləyin.

İstifadəçi məlumatını dəyişdirin

Adminlər istifadəçi məlumatlarını dəyişə bilər.

  1. İstifadəçi Tipini təyin edin (İstifadəçi və ya Menecer).
  2. İstifadəçi statusunu təyin edin (Aktivdən Qeyri-aktiv).
  3. Unikal Tag ID daxil edin. (isteğe bağlı)
  4. Saxla klikləyin.
