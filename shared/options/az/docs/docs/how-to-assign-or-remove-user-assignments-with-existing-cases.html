Mövcud hallarla istifadəçi tapşırıqlarını necə təyin etmək və ya silmək olar

Bu məqalə adminlərə iş istifadəçilərini necə təyin etmək və silmək yollarını göstərir. Biz sizə istifadəçi məlumatlarını necə dəyişdirəcəyinizi də göstəririk.

Mövcud işin açılması

  1. Admin hesabına daxil olun.
  2. Proqramlar səhifəsində proqram tapın və ADMIN düyməsini klikləyin.
  3. Cases menyu nişanına klikləyin.
  4. Alt menyuda AKTİV seçildiyinə əmin olun.
  5. AKSİYA sütununa sürüşdürün.

img

Təyin edin və ya silin

İstifadəçini işə təyin etmək və ya istifadəçini işdən çıxarmaq üçün İstifadəçiləri Təyin et üzərinə klikləyin (lakin platformadan yox). Növbəti səhifədə: ƏLAVƏ düyməsini klikləyin və iş prosesini tamamlayın. İstifadəçini silmək üçün sil klikləyin.

img

Dəyişdirin

Cases səhifəsində istifadəçi məlumatlarını redaktə etmək və ya yeniləmək mümkün deyil. Bunun əvəzinə bu addımları yerinə yetirin:

  1. İstifadəçilər menyu nişanını vurun.
  2. İstifadəçi adını tapın və Edit düyməsini basın.
  3. Dəyişikliklər edin və SAVE düyməsini basın.
