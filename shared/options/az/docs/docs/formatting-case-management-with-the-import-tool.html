İdxal Aləti ilə Case İdarəetmə Məlumatının Formatlanması

Aşağıda İdxal aləti ilə işləri idarə etmək üçün CSV fayllarında tələb olunan sahələrin siyahısı verilmişdir.

istifadəçi adı

  • İstifadəçinin e-poçt ünvanını daxil edin.

tag_id (daha sonra istinad edə biləcəyiniz id)

  • unikal identifikator (əlifba-rəqəm simvollarından istifadə edin, tire qəbul edilir)

case_status

  • 0 = bağlıdır
  • 1 = aktiv

tarix_açılır

  • ay/gün/il
  • aa/gg/yyyy

haqqında

  • işin unikal təsviri (əlifba-rəqəm simvollarından istifadə edin, tire qəbul edilir)
