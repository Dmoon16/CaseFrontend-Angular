İstifadəçilər Yeni Hesab Necə Yaradırlar

Admin istifadəçini işə əlavə etdikdən sonra: * əgər istifadəçi caseactive yenidirsə, istifadəçi hesab yaratmaq üçün iş prosesini tamamlamalı və işə daxil olmaq üçün dəvəti qəbul etməlidir. * əgər istifadəçinin mövcud caseactive hesabı varsa, istifadəçi işə daxil olmaq üçün dəvəti qəbul etməlidir.

Bu məqalə istifadəçinin öz hesabını yaratmaq üçün yerinə yetirməli olduğu addımları təsvir edir.

Başlanır

Bir istifadəçinin Caseactiv tətbiqə əlavə edilməsinin iki yolu var

  • Admin tərəfindən: Əgər admin platforma UI və ya İdxal aləti vasitəsilə istifadəçi yaradırsa, admin daxil olmağa davam etmək üçün istifadəçiyə account.caseactive.com saytına keçid göndərməlidir.
  • İstifadəçi tərəfindən: Əgər istifadəçi qeydiyyat linkindən istifadə edərək hesab yaratmışsa, istifadəçi daxil olmağa davam etmək üçün avtomatik olaraq account.caseactive.com saytına yönləndiriləcək.

Daxil olur

İstifadəçi CaseActive hesabına ilk dəfə daxil olduqda, platforma istifadəçidən aşağıdakıları etməyi təklif edir:

  • onların e-poçt hesabını açın və sonra altı rəqəmli təhlükəsizlik kodunu kopyalayın/yapışdırın
  • hesab profilini tamamlayın

İstifadəçi YENİLƏNİB düyməsini kliklədikdən sonra Proqramlar səhifəsi göstərilir. Səhifədə proqramın adı (şirkət adı), istifadəçi növü (rol), status və DƏVƏT düyməsini göstərilir.

Tətbiqə daxil olmaq üçün istifadəçi dəvəti qəbul etməlidir. İstifadəçi etməlidir:

  1. DƏVƏT düyməsini klikləyin.
  2. İcazələri nəzərdən keçirin.
  3. Şərtləri nəzərdən keçirmək və redaktə etmək (istəyə görə) üçün Bildirişlər sekmesine klikləyin.
  4. GİRİŞİ TƏSDİQ EDİN üzərinə klikləyin.
