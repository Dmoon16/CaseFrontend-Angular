İctimai Qeydiyyat Linki ilə İşləri Avtomatik Necə Yaratmaq olar

Bu məqalə adminlərə tək qeydiyyat linki göndərməklə işin necə yaradılmasını (və eyni zamanda istifadəçi əlavə etməyi) göstərir.

Linkin yerləşdirilməsi

  1. Admin hesabına daxil olun.
  2. Proqramlar səhifəsində proqram tapın və ADMIN düyməsini klikləyin.
  3. İdarəetmə panelində Parametrlər menyu nişanına klikləyin.
  4. SİSTEM SETUP alt menyu nişanının seçildiyinə əmin olun.

Parametrlərin konfiqurasiyası

İSTİFADƏÇİ QEYDİYYƏSİ başlığına aşağı diyirləyin.

Qeydiyyata icazə ver sahəsinin Aktiv olaraq təyin olunduğundan əmin olun. (tələb olunur) Bu funksiya işə salındıqda, Case Active unikal keçid yaradır.

  1. Avtomatik Case Yarat sahəsinə klikləyin və Aktiv seçin. Case User Role sahəsi göstərilir.
  2. Rol seçin. (aşağıya bax)

Case istifadəçi rolları

Üç halda istifadəçi rolu var.

  • müştərilər (tövsiyə olunur)
  • default
  • heyət

Link göndərilir

Linki bir və ya bir neçə şəxslə paylaşın. Linki klikləyən və qeydiyyatı tamamlayan hər bir istifadəçi, iş səhifəsində yeni iş yaratmaq və istifadəçini avtomatik təyin etmək üçün platformanı işə salır.
