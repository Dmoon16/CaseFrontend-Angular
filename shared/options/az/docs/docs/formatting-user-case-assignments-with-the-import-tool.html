İdxal Aləti ilə İstifadəçi Case Təyinatları Məlumatının Formatlanması

Aşağıda CSV fayllarında məlumat sahələrinin siyahısı verilmişdir. Hər bir idxal seçimində sahələrin fərqli kombinasiyası var.

case_ID

  • Case İstifadəçi və ya Tag ID-ni daxil edin.
  • Davalar səhifəsinə keçin > Redaktə > Teq ID-si və ya İşlər səhifəsinə keçin > Redaktə edin > URL-də Case ID-si.

İstifadəçi adı

  • İstifadəçinin ID və ya Tag ID-ni daxil edin.
  • İstifadəçilər səhifəsinə keçin > Redaktə > Teq ID-si və ya İstifadəçilər səhifəsinə keçin > Redaktə et > URL-də İstifadəçi ID-si.

hal_rol_sifarişi

  • Hüceyrəni boş buraxın.

hal_rol_ID

  • Daxil edin: müştərilər, işçilər və ya defolt.

hal_bildir

  • 0 = yox
  • 1 = Bəli, istifadəçiyə e-poçt bildirişi göndərin.
