ווי צו אַוטאָ באַשטימען יוזערז מיט די ציבור סיגנופּ לינק

אַדמיניסטראַטאָרס לערנען ווי צו קאַנפיגיער סיסטעם סעטטינגס אַזוי אַז נייַע ניצערס זענען אויטאָמאַטיש אַסיינד צו אַ נייַ אַוטאָ באשאפן פאַל.

עפן די סעטאַפּ בלאַט

דער ערשטער שריט איז צו עפֿענען די סיסטעם סעטאַפּ בלאַט.

  1. קלאָץ אין צו די אַדמיניסטראַטאָר חשבון.
  2. געפֿינען אַן אַפּ אויף די אַפּפּס בלאַט און גיט די ADMIN קנעפּל.
  3. דריקט דעם סיסטעם קוויטל און דעמאָלט SYSTEM SETUP אין די סובמעניו.
  4. מעגילע אַראָפּ צו די USER SIGNUP כעדינג.

קאַנפיגיער סיסטעם סעטטינגס

  1. שטעלן די לאָזן סיגנופּ און אַוטאָ שאַפֿן קאַסע פאַל-אַראָפּ מעניוז צו אויף.
  2. דריקט דעם ראָלע מעניו און קלייַבן אַן אָפּציע (פעליקייַט איז רעקאַמענדיד).
  3. קאָפּי און ייַנטיילן די פּובליק באַניצער סיגנופּ לינק.
