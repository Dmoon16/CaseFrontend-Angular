אראפקאפיע איינער אָדער מער קסוו טעקעס. ניצן די טעקעס צו לייגן, מאָדיפיצירן, אָדער באַזייַטיקן פאַל ניצערס. נאָך די לייג ימפּאָרץ דיאַלאָג קעסטל אָפּענס:

  1. דריקט דעם טיפּ פעלד.
  2. מעגילע דורך די פאַל-אַראָפּ מעניו און סעלעקטירן אַן אָפּציע (זען ווייטער). א לינק דיספּלייז.
  3. דריקט דעם אראפקאפיע מוסטער לינק. א CSV טעקע דאַונלאָודז צו דיין מיטל.

ופּלאָאַדינג די טעקע

  5. דריקט SAVE.
