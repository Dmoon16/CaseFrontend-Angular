Πώς να δημιουργήσετε και να τροποποιήσετε χρήστες με τη διεπαφή χρήστη

Αυτό το άρθρο δείχνει στους διαχειριστές πώς να δημιουργούν χρήστες και να ενημερώνουν εγγραφές με τη διεπαφή χρήστη της πλατφόρμας. Οι διαχειριστές μπορούν επίσης να προσθέσουν χρήστες σε μια νέα υπόθεση και να επεξεργαστούν υπάρχουσες περιπτώσεις.

Δημιουργία χρήστη

  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Στον πίνακα διαχειριστή, κάντε κλικ στην καρτέλα μενού Χρήστες.
  4. Κάντε κλικ στο κουμπί ΝΕΟ. Εμφανίζεται ένα νέο παράθυρο.

img

Συμπληρώστε τα πεδία και κάντε κλικ στο ΑΠΟΘΗΚΕΥΣΗ. Το Case Active προσθέτει το όνομα στη λίστα των χρηστών που σχετίζονται με αυτήν την εφαρμογή.

Τροποποίηση χρήστη

Για να ενημερώσετε έναν μικρό αριθμό εγγραφών χρηστών, χρησιμοποιήστε το εργαλείο Επεξεργασία και κάντε αλλαγές σε κάθε μεμονωμένο χρήστη.

  • Σημείωση Για να τροποποιήσετε έναν μεγάλο αριθμό εγγραφών χρηστών σε μία συνεδρίαση, εξετάστε το ενδεχόμενο να χρησιμοποιήσετε το εργαλείο Εισαγωγή .
  1. Στον πίνακα διαχειριστή, κάντε κλικ στην καρτέλα μενού Χρήστες.
  2. Εντοπίστε έναν χρήστη και κάντε κλικ στο Επεξεργασία.
  3. Κάντε αλλαγές και κάντε κλικ στο ΑΠΟΘΗΚΕΥΣΗ.

img

Επόμενα βήματα

Το Case Active δεν στέλνει στον χρήστη ειδοποίηση μέσω email. Οι διαχειριστές θα πρέπει να επικοινωνήσουν με τον χρήστη και να του στείλουν έναν σύνδεσμο προς την αρχική σελίδα του Case Active. Στη συνέχεια, ο χρήστης κάνει κλικ στο κουμπί LOGIN και ολοκληρώνει τη ροή εργασιών για νέες εγγραφές.

  • Αυτό το άρθρο περιγράφει τη ροή εργασιών εγγραφής για νέους χρήστες.
