Πώς οι χρήστες δημιουργούν νέο λογαριασμό

Αφού ένας διαχειριστής προσθέσει έναν χρήστη σε μια υπόθεση: * εάν ο χρήστης είναι νέος στην υπόθεση, ο χρήστης πρέπει να ολοκληρώσει μια ροή εργασίας για να δημιουργήσει έναν λογαριασμό και να αποδεχτεί την πρόσκληση για πρόσβαση στην υπόθεση. * εάν ο χρήστης έχει έναν υπάρχοντα λογαριασμό caseactive, ο χρήστης πρέπει να αποδεχτεί την πρόσκληση για πρόσβαση στην υπόθεση.

Αυτό το άρθρο περιγράφει τα βήματα που πρέπει να ολοκληρώσει ένας χρήστης για να δημιουργήσει τον λογαριασμό του.

Ξεκινώντας

Υπάρχουν δύο τρόποι με τους οποίους ένας χρήστης μπορεί να προστεθεί σε μια εφαρμογή που λειτουργεί κατά περίπτωση

  • Από διαχειριστή: Εάν ο διαχειριστής δημιουργήσει έναν χρήστη μέσω της διεπαφής χρήστη της πλατφόρμας ή του εργαλείου εισαγωγής , ο διαχειριστής πρέπει να στείλει στον χρήστη έναν σύνδεσμο προς το account.caseactive.com για να προχωρήσει στη σύνδεση.
  • Κατά χρήστη: Εάν ο χρήστης δημιούργησε έναν λογαριασμό χρησιμοποιώντας έναν σύνδεσμο εγγραφής , ο χρήστης θα ανακατευθυνθεί αυτόματα στο account.caseactive.com για να προχωρήσει στη σύνδεση.

Σύνδεση

Όταν ο χρήστης συνδέεται στον λογαριασμό του CaseActive για πρώτη φορά, η πλατφόρμα προτρέπει τον χρήστη να:

  • ανοίξτε τον λογαριασμό email τους και, στη συνέχεια, αντιγράψτε/επικολλήστε τον εξαψήφιο κωδικό ασφαλείας
  • συμπληρώστε ένα προφίλ λογαριασμού

img

Αφού ο χρήστης κάνει κλικ στο UPDATE, εμφανίζεται η σελίδα Εφαρμογές. Η σελίδα εμφανίζει το όνομα της εφαρμογής (όνομα εταιρείας), τον τύπο χρήστη (ρόλος), την κατάσταση και ένα κουμπί ΠΡΟΣΚΛΗΣΗ.

img

Για να αποκτήσει πρόσβαση στην εφαρμογή, ο χρήστης πρέπει να αποδεχτεί την πρόσκληση. Ο χρήστης θα πρέπει:

  1. Κάντε κλικ στο κουμπί ΠΡΟΣΚΛΗΣΗ.
  2. Ελέγξτε τις άδειες.
  3. Κάντε κλικ στην καρτέλα Ειδοποιήσεις για να ελέγξετε και να επεξεργαστείτε (προαιρετικά) τους όρους.
  4. Κάντε κλικ στην ΕΓΚΡΙΣΗ ΠΡΟΣΒΑΣΗΣ.

img

Όταν ο χρήστης εγκρίνει, η πλατφόρμα εμφανίζει ένα κουμπί APP στη σελίδα χρήστη. Αυτό το κουμπί δίνει στον χρήστη πρόσβαση σε εκχωρημένες περιπτώσεις.

Για να επεξεργαστείτε τις ρυθμίσεις χρήστη, ο χρήστης κάνει κλικ στην έλλειψη.

img

Ο χρήστης μπορεί: * να αφαιρέσει την πρόσβαση στην εφαρμογή * να τροποποιήσει τις ρυθμίσεις ειδοποιήσεων

img

Αφού ο χρήστης ολοκληρώσει τη ροή εργασίας, το όνομα χρήστη εμφανίζεται στη λίστα χρηστών του διαχειριστή. Η λίστα εμφανίζει όλους τους χρήστες, συνδεδεμένους ή μη.
