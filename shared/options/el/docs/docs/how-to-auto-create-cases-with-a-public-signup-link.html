Πώς να δημιουργήσετε αυτόματα θήκες με δημόσιο σύνδεσμο εγγραφής

Αυτό το άρθρο δείχνει στους διαχειριστές πώς να δημιουργήσουν μια υπόθεση (και να προσθέσουν έναν χρήστη ταυτόχρονα) στέλνοντας έναν μόνο σύνδεσμο εγγραφής.

Εντοπισμός του συνδέσμου

  1. Συνδεθείτε στον λογαριασμό διαχειριστή.
  2. Εντοπίστε μια εφαρμογή στη σελίδα Εφαρμογές και κάντε κλικ στο κουμπί ΔΙΑΧΕΙΡΙΣΤΗΣ.
  3. Στον πίνακα διαχειριστή, κάντε κλικ στην καρτέλα μενού Ρυθμίσεις.
  4. Βεβαιωθείτε ότι είναι επιλεγμένη η καρτέλα του υπομενού SYSTEM SETUP.

img

Διαμόρφωση ρυθμίσεων

Κάντε κύλιση προς τα κάτω στην επικεφαλίδα ΕΓΓΡΑΦΗ ΧΡΗΣΤΗ.

Βεβαιωθείτε ότι το πεδίο Allow Signup έχει οριστεί σε On. (απαιτείται) Όταν αυτή η δυνατότητα είναι ενεργοποιημένη, το Case Active δημιουργεί έναν μοναδικό σύνδεσμο.

  1. Κάντε κλικ στο πεδίο Auto Create Case και επιλέξτε On. Εμφανίζεται το πεδίο Case User Role.
  2. Επιλέξτε έναν ρόλο. (Δες παρακάτω)

img

Ρόλοι χρηστών υπόθεσης

Υπάρχουν τρεις ρόλοι χρήστη περιπτώσεων.

  • πελάτες (συνιστάται)
  • Προκαθορισμένο
  • προσωπικό

Αποστολή του συνδέσμου

Μοιραστείτε τον σύνδεσμο με ένα ή περισσότερα άτομα. Κάθε χρήστης που κάνει κλικ στον σύνδεσμο και ολοκληρώνει την εγγραφή ενεργοποιεί την πλατφόρμα για να δημιουργήσει μια νέα υπόθεση στη σελίδα υποθέσεων και να εκχωρήσει αυτόματα τον χρήστη στη θήκη.
