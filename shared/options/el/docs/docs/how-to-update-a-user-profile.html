Πώς να ενημερώσετε ένα προφίλ χρήστη

Αυτό το άρθρο δείχνει στους χρήστες πώς να ενημερώνουν τις πληροφορίες στη σελίδα προφίλ τους.

  • Οι χρήστες μπορούν να ενημερώσουν τα δικά τους προφίλ.
  • Οι διαχειριστές μπορούν να ενημερώσουν τη σελίδα προφίλ οποιουδήποτε χρήστη.

Για χρήστες

Το πρώτο βήμα είναι να ανοίξετε τη σελίδα Προφίλ. Οι χρήστες και οι διαχειριστές ακολουθούν αυτά τα βήματα για να ενημερώσουν τη δική τους σελίδα Προφίλ.

  1. Συνδεθείτε στον λογαριασμό σας Case Active.
  2. Κάντε κλικ στην καρτέλα Προφίλ στο επάνω μέρος της σελίδας.
  3. Κάνε αλλαγές.
  4. Κάντε κλικ στην Αποθήκευση.

Τα περισσότερα πεδία είναι προαιρετικά και ορισμένα πεδία είναι υποχρεωτικά. Τα απαιτούμενα πεδία είναι:

  • Ονομα
  • Επίθετο
  • Διεύθυνση ηλεκτρονικού ταχυδρομείου
  • Γλώσσα
  • Ζώνη ώρας

Για διαχειριστές

Οι διαχειριστές μπορούν επίσης να ενημερώσουν τις πληροφορίες προφίλ για άλλους χρήστες. Αυτό περιλαμβάνει τρία πεδία κάτω από την επικεφαλίδα ΠΛΗΡΟΦΟΡΙΕΣ ΧΡΗΣΤΗ HOST.

  1. Συνδεθείτε στον λογαριασμό διαχειριστή Case Active.
  2. Εντοπίστε μια εφαρμογή και κάντε κλικ στο κουμπί ADMIN.
  3. Κάντε κλικ στην επιλογή Χρήστες στο επάνω μενού.
  4. Μεταβείτε στο υπομενού και επιλέξτε τη λίστα χρηστών που θέλετε να αποκτήσετε πρόσβαση (ΕΝΕΡΓΗ ή ΑΠΕΝΕΡΓΟΠΟΙΗΜΕΝΗ). Η προεπιλεγμένη ρύθμιση είναι Ενεργή.
  5. Εντοπίστε έναν χρήστη στη λίστα και κάντε κλικ στην Επεξεργασία.
  6. Κάντε αλλαγές όπως απαιτείται. (Δες παρακάτω)
  7. Κάντε κλικ στην Αποθήκευση.

Αλλαγή στοιχείων χρήστη

Οι διαχειριστές μπορούν να αλλάξουν τις πληροφορίες χρήστη.

  1. Ορίστε τον Τύπο χρήστη (Χρήστης ή Διαχειριστής).
  2. Ορίστε την κατάσταση χρήστη (Ενεργή σε Απενεργοποιημένη).
  3. Εισαγάγετε ένα μοναδικό αναγνωριστικό ετικέτας. (προαιρετικός)
  4. Κάντε κλικ στην Αποθήκευση.
