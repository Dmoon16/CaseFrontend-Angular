Πώς να δημιουργήσετε χρήστες με σύνδεσμο εγγραφής

Αυτό το άρθρο δείχνει στους διαχειριστές πώς να δημιουργούν χρήστες στέλνοντας έναν σύνδεσμο εγγραφής.

Διαμόρφωση του συνδέσμου εγγραφής

  1. Κάντε κλικ στο σύνδεσμο Αποστολή εγγραφής στο κάτω μέρος του πλαισίου διαλόγου Προσθήκη χρήστη. Εμφανίζεται μια νέα σελίδα.
  2. Διαμορφώστε τις επιλογές εγγραφής. (Δες παρακάτω)

img

Βεβαιωθείτε ότι το πεδίο Να επιτρέπεται η εγγραφή έχει οριστεί σε On. (απαιτείται) Όταν είναι ενεργοποιημένο, το Case Active δημιουργεί έναν μοναδικό σύνδεσμο.

Ελέγξτε τις παρακάτω επιλογές και, στη συνέχεια, κάντε κλικ στο εικονίδιο για να αντιγράψετε τον σύνδεσμο. Μπορείτε να τοποθετήσετε τον σύνδεσμο στον ιστότοπό σας ή να στείλετε τον σύνδεσμο σε ένα ή περισσότερα άτομα μέσω email ή οποιουδήποτε καναλιού επικοινωνίας.

img

Δημιουργία θήκης

Προαιρετικά, μπορείτε να δημιουργήσετε αυτόματα μια υπόθεση για κάθε νέο χρήστη και να εκχωρήσετε έναν ρόλο χρήστη.

  1. Κάντε κλικ στο πεδίο Auto Create Case και επιλέξτε On. Εμφανίζονται τα πεδία Case User Role.
  2. Κάντε κλικ στο Role Case User και ορίστε μια επιλογή. (Δες παρακάτω)

img

Ρόλοι χρηστών υπόθεσης

Υπάρχουν τρεις ρόλοι χρήστη περιπτώσεων.

  • πελάτες
  • Προκαθορισμένο
  • προσωπικό

Αποστολή του συνδέσμου

Μοιραστείτε τον σύνδεσμο με ένα ή περισσότερα άτομα. Κάθε άτομο κάνει κλικ στον σύνδεσμο και ολοκληρώνει τη ροή εργασιών για να ενεργοποιήσει τον λογαριασμό του Case Active. Μόλις ο χρήστης ολοκληρώσει τη ροή εργασιών εγγραφής, το Case Active προσθέτει τον χρήστη στη λίστα χρηστών.

Επόμενα βήματα

Το Case Active στέλνει στον χρήστη μια ειδοποίηση μέσω email με τον σύνδεσμο εγγραφής. Ο χρήστης κάνει κλικ στον σύνδεσμο και ολοκληρώνει τη ροή εργασιών για νέες εγγραφές.

  • Αυτό το άρθρο περιγράφει τη ροή εργασιών εγγραφής για νέους χρήστες.
