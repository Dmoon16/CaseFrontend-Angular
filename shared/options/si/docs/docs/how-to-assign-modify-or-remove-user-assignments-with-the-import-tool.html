  • නඩුවකට පවරා ඇති පරිශීලකයින් ඉවත් කරන්න

  3. Integration ටැබ් එක ක්ලික් කර උපමෙනුව මත IMPORTS කරන්න.

වාර්තා නිර්මාණය කිරීම

ගොනු විකල්ප

නඩු වලට පරිශීලකයින් පවරන්න * නඩුවකට පරිශීලකයින් එක් කිරීමට මෙම විකල්පය තෝරන්න. * case_role_order කොටුව හිස්ව තබන්න. * පරිශීලකයන් පිටුව වෙත ගොස් මෙම පුද්ගලයා සඳහා ටැග් හැඳුනුම්පත පිටපත් කරන්න. ටැග් ID ක්ෂේත්‍රය හිස් නම්, අනන්‍ය හැඳුනුම්පතක් ඇතුළත් කරන්න, සුරකින්න ක්ලික් කරන්න, ඉන්පසු හැඳුනුම්පත CSV ගොනුවට පිටපත් කරන්න.
