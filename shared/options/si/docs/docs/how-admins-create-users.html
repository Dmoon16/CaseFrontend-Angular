පරිපාලකයින් පරිශීලකයින් නිර්මාණය කරන ආකාරය

පරිපාලකයින් විසින් යෙදුම් සහ අවස්ථා වෙත ප්‍රවේශ විය හැකි අය නිර්මාණය කිරීම, වෙනස් කිරීම හෝ අබල කිරීම. පරිශීලකයෙකු පළමු වරට වේදිකාවට ලොග් වූ විට, ඔවුන්ට පැතිකඩ කාර්ය ප්‍රවාහයක් සම්පූර්ණ කිරීමට අවශ්‍ය වේ.

  • නව පරිශීලක පැතිකඩ කාර්ය ප්‍රවාහය පිළිබඳ වැඩිදුර දැන ගැනීමට මෙතැන ක්ලික් කරන්න.

පරිශීලකයෙකු නිර්මාණය කිරීම
