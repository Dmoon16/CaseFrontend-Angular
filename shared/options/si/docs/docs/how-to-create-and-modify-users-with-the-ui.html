UI සමඟින් පරිශීලකයන් නිර්මාණය කර වෙනස් කරන්නේ කෙසේද

මෙම ලිපිය පරිපාලකයින්ට වේදිකාවේ පරිශීලක අතුරුමුහුණත සමඟ පරිශීලකයින් නිර්මාණය කරන ආකාරය සහ වාර්තා යාවත්කාලීන කරන ආකාරය පෙන්වයි. පරිපාලකයින්ට නව නඩුවකට පරිශීලකයින් එක් කිරීමට සහ පවතින අවස්ථා සංස්කරණය කිරීමට ද හැකිය.

පරිශීලකයෙකු නිර්මාණය කිරීම

  1. පරිපාලක ගිණුමට ලොග් වන්න.
  2. යෙදුම් පිටුවේ යෙදුමක් සොයාගෙන ADMIN බොත්තම ක්ලික් කරන්න.
  3. පරිපාලක පැනලයේ, පරිශීලක මෙනු ටැබය ක්ලික් කරන්න.
  4. නව බොත්තම ක්ලික් කරන්න. නව කවුළුවක් දිස්වේ.

ක්ෂේත්‍ර පුරවා සුරකින්න ක්ලික් කරන්න. Case Active මෙම යෙදුම හා සම්බන්ධ පරිශීලකයින්ගේ ලැයිස්තුවට නම එක් කරයි.

පරිශීලකයෙකු වෙනස් කිරීම

පරිශීලක වාර්තා කුඩා සංඛ්‍යාවක් යාවත්කාලීන කිරීමට, සංස්කරණ මෙවලම භාවිතා කර එක් එක් පරිශීලකයාට වෙනස්කම් කරන්න.

  • සටහන එක් වරකදී පරිශීලක වාර්තා විශාල ප්‍රමාණයක් වෙනස් කිරීමට, ආනයන මෙවලම භාවිතා කිරීම සලකා බලන්න.
  1. පරිපාලක පැනලයේ, පරිශීලක මෙනු ටැබය ක්ලික් කරන්න.
  2. පරිශීලකයෙකු සොයාගෙන සංස්කරණය කරන්න ක්ලික් කරන්න.
  3. වෙනස්කම් සිදු කර සුරකින්න ක්ලික් කරන්න.

  • මෙම ලිපිය නව පරිශීලකයින් සඳහා ලියාපදිංචි වීමේ කාර්ය ප්‍රවාහය විස්තර කරයි.
