නඩුවක් නිර්මාණය කිරීමේදී පරිශීලකයින් පවරන ආකාරය

මෙම ලිපිය පරිපාලකයින්ට නව නඩුවක් නිර්මාණය කිරීමේදී පරිශීලකයෙකු පවරන ආකාරය පෙන්වයි. පරිශීලකයා දැනටමත් වේදිකාවේ ලියාපදිංචි වී නොමැති නම්, ඔබට නව නඩුවක් නිර්මාණය කරන අතරතුර නව පරිශීලකයෙකු එක් කරන්න.

  3. නඩු මෙනු ටැබය ක්ලික් කරන්න.
  4. නව බොත්තම ක්ලික් කරන්න. නව කවුළුවක් දිස්වේ.
  • නඩුවක් සෑදීමට ක්ෂේත්‍ර පුරවා පසුව පවරා ඇති පරිශීලකයන් ශීර්ෂයට පහළට අනුචලනය කරන්න.

  2. හිස් ක්ෂේත්රයේ අගයක් ඇතුළත් කරන්න (උදා. ඊමේල් ලිපිනය).
  3. පරිශීලක වාර්තාව වේදිකාවේ තිබේ නම්, ක්ෂේත්‍රය ස්වයංක්‍රීයව නමක් යෝජනා කරයි.
  4. නිවැරදි පරිශීලකයා තෝරන්න.
  5. පරිශීලක භූමිකාවක් තෝරන්න (සේවාදායකයින්, පෙරනිමිය, හෝ කාර්ය මණ්ඩලය).
  6. විකල්පයක් ලෙස, පරිශීලකයාට ඊමේල් දැනුම්දීමක් යැවීමට කොටුව සලකුණු කරන්න.
  7. සුරකින්න ක්ලික් කරන්න.
