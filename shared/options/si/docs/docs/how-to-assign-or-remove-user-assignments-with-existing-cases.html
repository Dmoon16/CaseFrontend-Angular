පවතින අවස්ථා සමඟ පරිශීලක පැවරුම් පැවරීම හෝ ඉවත් කරන්නේ කෙසේද

මෙම ලිපිය පරිපාලකයින්ට නඩු භාවිතා කරන්නන් පවරන සහ ඉවත් කරන ආකාරය පෙන්වයි. පරිශීලක දත්ත වෙනස් කරන්නේ කෙසේදැයි අපි ඔබට පෙන්වන්නෙමු.

පවතින නඩුවක් විවෘත කිරීම

  1. පරිපාලක ගිණුමට ලොග් වන්න.
  2. යෙදුම් පිටුවේ යෙදුමක් සොයාගෙන ADMIN බොත්තම ක්ලික් කරන්න.
  3. නඩු මෙනු ටැබය ක්ලික් කරන්න.
  4. උප මෙනුවේ ACTIVE තෝරාගෙන ඇති බවට වග බලා ගන්න.
  5. ACTION තීරුව වෙත පහළට අනුචලනය කරන්න.

පවරන්න හෝ ඉවත් කරන්න

නඩුවට පරිශීලකයෙකු පැවරීමට හෝ නඩුවෙන් පරිශීලකයා ඉවත් කිරීමට (නමුත් වේදිකාවෙන් නොවේ) පරිශීලකයන් පවරන්න ක්ලික් කරන්න. ඊළඟ පිටුවේ: ADD ක්ලික් කර කාර්ය ප්‍රවාහය සම්පූර්ණ කරන්න. පරිශීලකයෙකු මකා දැමීමට ඉවත් කරන්න ක්ලික් කරන්න.

වෙනස් කරන්න

සිද්ධි පිටුවෙහි පරිශීලක දත්ත සංස්කරණය කිරීමට හෝ යාවත්කාලීන කිරීමට නොහැක. ඒ වෙනුවට, මෙම පියවර අනුගමනය කරන්න:

  1. පරිශීලක මෙනු ටැබය ක්ලික් කරන්න.
  2. පරිශීලක නාමය සොයාගෙන සංස්කරණය කරන්න ක්ලික් කරන්න .
  3. වෙනස්කම් සිදු කර සුරකින්න ක්ලික් කරන්න.
