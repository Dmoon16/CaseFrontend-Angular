යෙදුමක් නිර්මාණය කරන්නේ කෙසේද

මෙම ලිපිය නව පරිශීලකයින්ට නොමිලේ අත්හදා බැලීමේ ගිණුමක් සාදන ආකාරය පෙන්වයි. Case Active මෙම ගිණුම සඳහා නව පරිශීලක පරිපාලක වරප්‍රසාද ස්වයංක්‍රීයව ලබා දෙයි.

ගිණුමක් නිර්මාණය කිරීම

  1. පිවිසුම් පිටුවට යන්න.
  2. Get Started බොත්තම ක්ලික් කරන්න.
  3. ඊමේල් ලිපිනයක් ඇතුලත් කරන්න.
  4. Next ක්ලික් කරන්න.

ගිණුමක් තහවුරු කිරීම

  1. නව බ්‍රව්සර් ටැබ් එකක් විවෘත කර ඔබගේ විද්‍යුත් තැපැල් ගිණුමට ලොග් වන්න.
  2. ඉලක්කම් හයක කේතය පිටපත් කරන්න.
  3. Case Active පටිත්ත වෙත ආපසු යන්න.
  4. ක්ෂේත්‍ර තුළට කේතය අලවන්න. නව කවුළුවක් දිස්වේ.

නව කවුළුව තුළ:

  1. අවශ්ය ක්ෂේත්ර පුරවන්න.
  2. සේවා කොන්දේසි කියවා එකඟ වීමට කොටුව සලකුණු කරන්න.
  3. නොමිලේ ගිණුමක් සාදන්න ක්ලික් කරන්න.

යෙදුම් සැකසුම සම්පූර්ණ කරමින්

ඊළඟ පිටුවේ වැඩ ප්‍රවාහයක් පෙන්වයි. ඔබගේ යෙදුම පිහිටුවීම අවසන් කිරීමට පියවර සම්පූර්ණ කරන්න. තොරතුරු සංස්කරණය කිරීමට හෝ යාවත්කාලීන කිරීමට හෝ ඔබගේ ගිණුම් තත්ත්වය ඔබට සැම විටම මෙම පිටු වෙත ආපසු යා හැක.

යෙදුම ගැන පිටුවේ, සමාගමේ නම සහ වෙබ් අඩවි ක්ෂේත්‍ර (අවශ්‍ය) පුරවා ඊළඟ ක්ලික් කරන්න.

  3. තත්ත්වය

  2. පරිපාලක පැනලය විවෘත කිරීමට ADMIN බොත්තම ක්ලික් කරන්න. පරිපාලකයින් පරිශීලකයින්, සිද්ධීන්, ඒකාබද්ධ කිරීම් සහ තවත් බොහෝ දේ කළමනාකරණය කරන්නේ මෙහිදීය.
