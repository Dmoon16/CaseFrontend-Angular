Ako správcovia spravujú priradenia používateľov k prípadom

Existuje päť spôsobov, ako spravovať používateľov a priradenia prípadov. Kliknite na odkaz a zistite ako.

Metóda Typ Akcie Podrobnosti
Pri vytváraní prípadu UI pridať tlačidlo Správca > Prípady > tlačidlo Nový > tlačidlo Priradiť používateľov
Automaticky priradiť používateľa pomocou verejného odkazu na registráciu Akcia používateľa pridať Tlačidlo Správca > Nastavenia > Stránka Nastavenie systému
Použitie používateľského rozhrania na existujúcom prípade UI pridať a odstrániť tlačidlo Správca > Prípady > Priradiť používateľov
Nástroj na import UI pridať, upraviť a odstrániť tlačidlo Správca > Integrácie > Importy
Používanie API (čoskoro) API pridať, upraviť a odstrániť N/A
