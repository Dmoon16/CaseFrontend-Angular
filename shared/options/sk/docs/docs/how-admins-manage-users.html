Ako správcovia spravujú používateľov

Správcovia vytvárajú, upravujú alebo zakazujú, kto môže pristupovať k aplikáciám a prípadom. Keď sa používateľ prihlási na platformu prvýkrát, musí dokončiť pracovný postup profilu.

  • Kliknutím sem získate ďalšie informácie o novom pracovnom postupe používateľského profilu.

Správa používateľa

Nižšie uvedená tabuľka zobrazuje päť spôsobov, ako spravovať používateľa. Kliknite na odkaz a zistite ako.

Metóda Typ Akcie Podrobnosti
Používanie používateľského rozhrania UI pridať a upraviť tlačidlo Správca > Používatelia
Pri vytváraní prípadu a priraďovanie používateľa k prípadu UI pridať tlačidlo Správca > Prípady > tlačidlo Nový > stránka Vytvoriť prípad
Automaticky vytvoriť používateľa pomocou verejného odkazu na registráciu Akcia používateľa pridať Tlačidlo Správca > Nastavenia > Stránka Nastavenie systému
Nástroj na import UI pridať a upraviť tlačidlo Správca > Integrácie > Importy
Používanie API (už čoskoro) API pridať a upraviť N/A

Upozornenia

Platforma neposiela používateľom upozornenie, keď sú pridaní do aplikácie. Správcovia však môžu povoliť upozornenia na priradenia prípadov.
