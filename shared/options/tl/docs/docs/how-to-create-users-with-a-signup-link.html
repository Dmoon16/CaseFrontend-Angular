Paano Gumawa ng Mga User na may Link sa Pag-signup

Ipinapakita ng artikulong ito ang mga admin kung paano lumikha ng mga user sa pamamagitan ng pagpapadala ng link sa pag-signup.

Pag-configure ng link sa pag-signup

  1. I-click ang link na Magpadala sa kanila ng signup sa ibaba ng dialog box na Magdagdag ng user. Ang isang bagong pahina ay ipinapakita.
  2. I-configure ang mga opsyon sa pag-signup. (tingnan sa ibaba)

Tiyaking ang field na Payagan ang Pag-signup ay nakatakda sa Naka-on. (kinakailangan) Kapag pinagana, ang Case Active ay bumubuo ng isang natatanging link.

Suriin ang mga opsyon sa ibaba at pagkatapos ay i-click ang icon upang kopyahin ang link. Maaari mong ilagay ang link sa iyong website o ipadala ang link sa isa o higit pang mga tao sa pamamagitan ng email o anumang channel ng komunikasyon.

Paggawa ng kaso

Opsyonal, maaari kang awtomatikong gumawa ng case para sa bawat bagong user at magtalaga ng tungkulin ng user.

  1. I-click ang field na Auto Create Case at piliin ang On. Ang mga field ng Tungkulin ng Gumagamit ng Kaso ay nagpapakita.
  2. I-click ang Case User Role at pumili ng opsyon. (tingnan sa ibaba)

img

Mga tungkulin ng gumagamit ng kaso

Mayroong tatlong kaso ng mga tungkulin ng gumagamit.

  • mga kliyente
  • default
  • mga tauhan

Ipinapadala ang link

Ibahagi ang link sa isa o higit pang tao. Ang bawat tao ay nagki-click sa link at kinukumpleto ang daloy ng trabaho upang i-activate ang kanilang Case Active na account. Kapag nakumpleto na ng user ang workflow sa pag-signup, idaragdag ng Case Active ang user sa listahan ng Mga User.

img

Mga susunod na hakbang

Ang Case Active ay nagpapadala sa user ng isang abiso sa email na may link sa pag-signup. Ang user ay nag-click sa link at kinukumpleto ang daloy ng trabaho para sa mga bagong pag-signup.

  • Inilalarawan ng artikulong ito ang daloy ng trabaho sa pag-signup para sa mga bagong user.
