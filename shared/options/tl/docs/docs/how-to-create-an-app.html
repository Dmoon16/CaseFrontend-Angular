Paano Gumawa ng App

Ipinapakita ng artikulong ito ang mga bagong user kung paano gumawa ng libreng trial na account. Awtomatikong binibigyan ng Case Active ang bagong mga pribilehiyo ng admin ng user para sa account na ito.

Paggawa ng account

  1. Pumunta sa login page .
  2. I-click ang button na Magsimula.
  3. Ilagay ang iyong email na address.
  4. I-click ang Susunod.

img

Pag-verify ng account

  1. Magbukas ng bagong tab ng browser at mag-log in sa iyong email account.
  2. Kopyahin ang anim na digit na code.
  3. Bumalik sa tab na Case Active.
  4. I-paste ang code sa mga field. May lalabas na bagong window.

Sa bagong window:

  1. Punan ang mga kinakailangang field.
  2. Basahin ang mga tuntunin ng serbisyo at lagyan ng check ang kahon upang sumang-ayon.
  3. I-click ang GUMAWA NG LIBRENG ACCOUNT.

Kinukumpleto ang pag-set up ng app

Ang isang daloy ng trabaho ay ipinapakita sa susunod na pahina. Kumpletuhin ang mga hakbang upang tapusin ang pag-set up ng iyong app. Maaari kang bumalik anumang oras sa mga pahinang ito upang i-edit o i-update ang impormasyon o katayuan ng iyong account.

Sa pahina ng Tungkol sa App, punan ang pangalan ng kumpanya at mga field ng website (kinakailangan) at pagkatapos ay i-click ang Susunod.

Sa pahina ng Impormasyon sa Pagsingil, i-click ang field ng Payment Plan at suriin ang mga opsyon. Ang default na setting ay Libre. I-click ang SUBMIT.

img

Pagkatapos mong isumite ang form, ipinapakita ng Case Active ang pahina ng Apps. Para sa bawat app na iyong ginagawa o nabibilang, ipinapakita ng page ang:

  1. Ang pangalan ng app
  2. Iyong Uri (gampanan ng gumagamit)
  3. Katayuan

img

Ang pahina ng Apps ay isang lugar ng paglulunsad para sa maraming mga function ng admin:

  1. I-click ang APP button para makipag-usap, punan ang mga doc, video chat (at higit pa) sa mga isyung nauugnay sa kaso.
  2. I-click ang ADMIN button para buksan ang admin panel. Dito pinamamahalaan ng mga Admin ang mga user, kaso, pagsasama, at higit pa.
  3. I-click ang ellipsis para pamahalaan ang mga setting ng pagsingil at notification ng account.
  • Tandaan Case Active ay ipinapakita ang pindutan ng ADMIN para sa mga gumagamit ng admin. Ang mga user na walang mga pribilehiyo ng admin ay hindi nakikita ang pindutan ng ADMIN (tulad ng ipinapakita sa screen sa ibaba).
