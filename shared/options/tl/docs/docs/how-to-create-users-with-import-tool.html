Paano Gumawa ng Mga User gamit ang Import Tool

Ipinapakita ng artikulong ito sa mga admin kung paano gumawa ng mga user gamit ang tool na Import.

Binubuksan ang tool sa Pag-import

Maaaring gamitin ng mga admin ang tool sa Pag-import upang maramihang mag-upload ng mga bagong user at baguhin ang mga kasalukuyang tala ng user.

  1. Mag-log in sa admin account.
  2. Maghanap ng app sa page ng Apps at i-click ang ADMIN button.
  3. Sa Admin panel, i-click ang tab na Integration at pagkatapos ay IMPORTS sa submenu.
  4. I-click ang BAGONG button. Ang dialog box na Magdagdag ng Mga Pag-import ay nagpapakita sa isang bagong tab ng browser.

Paglikha ng mga talaan

Pagkatapos magbukas ang dialog box ng Add Imports:

  1. I-click ang field na Uri at pumili ng opsyon (tingnan sa ibaba). Nagpapakita ang isang link.
  2. I-click ang link na I- download ang Sample . Nagda-download ang isang CSV file sa iyong device.

Susunod, buksan ang file at magdagdag ng mga tala. Mag-click dito upang matutunan kung paano gamitin ang data ng format sa CSV file.

Uri

Kapag na-click mo ang field ng Uri, magpapakita ang isang drop-down na menu. Pumili ng isa sa dalawang opsyon para sa maramihang pag-import.

:exclamation: TANDAAN Ang drop-down na menu ay may maraming mga opsyon, ngunit dalawa lamang ang nauugnay sa tool na Import. Ang bawat opsyon ay may ibang sample na CSV file.

Magdagdag ng mga user

  • Piliin ang opsyong ito para sa isang simpleng maramihang pag-upload ng mga tala ng user.
  • I-click ang link na I-download ang Sample upang magbukas ng CSV file na may mga heading ng column para sa Add users Import na uri.

Baguhin ang mga user

  • Piliin ang opsyong ito upang i-update ang data para sa mga kasalukuyang user.
  • I-click ang link na I-download ang Sample upang magbukas ng CSV file na may mga heading ng column para sa uri ng Pag-import ng Baguhin ang mga user.

Pag-upload ng file

Maaari kang mag-upload kaagad ng file o bumalik sa dialog box na Magdagdag ng Mga Pag-import sa ibang araw at kumpletuhin ang mga hakbang na ito.

  1. Buksan ang dialog box na Magdagdag ng Mga Import.
  2. Maglagay ng natatanging pangalan para sa CSV file.
  3. I-click ang icon na plus (+) upang i-upload ang file mula sa iyong device.
  4. I-click ang field na Uri at pumili ng opsyon. (tingnan sa itaas)
  5. I-click ang I-SAVE.

Pinoproseso

Depende sa bilang ng mga tala sa pag-load ng file at system, maaaring tumagal ng ilang minuto ang platform upang maproseso ang pag-import.

Kapag nakumpleto ang pag-import, inaayos ng platform ang mga talaan ng user (ibig sabihin, nagdaragdag, nagbabago, o nag-disable) sa dashboard ng Mga User ng admin.

Kabiguan

Kung nabigo ang pagpapatakbo ng pag-import, magbabalik ang platform ng isang Nabigong mensahe. Sa karamihan ng mga kaso, ang sanhi ng pagkabigo ay ang pag-format ng data.

Upang makita ang sanhi ng pagkabigo:

  1. I-click ang View sa ilalim ng Action heading.
  2. Sa susunod na pahina, i-hover ang mouse sa icon na Nabigo. Ipinapakita ng tooltip ang error.
  3. Pumunta sa CVS file at ayusin at ayusin ang error.
  4. I-upload muli ang file.
