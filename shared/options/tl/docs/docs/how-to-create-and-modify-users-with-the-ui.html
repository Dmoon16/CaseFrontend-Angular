Paano Gumawa at Magbago ng mga User gamit ang UI

Ipinapakita ng artikulong ito sa mga admin kung paano gumawa ng mga user at mag-update ng mga tala gamit ang user interface ng platform. Ang mga admin ay maaari ding magdagdag ng mga user sa isang bagong kaso at mag-edit ng mga kasalukuyang kaso.

Paglikha ng isang gumagamit

  1. Mag-log in sa admin account.
  2. Maghanap ng app sa page ng Apps at i-click ang ADMIN button.
  3. Sa Admin panel, i-click ang tab na menu ng Mga User.
  4. I-click ang BAGONG button. May lalabas na bagong window.

img

Punan ang mga patlang at i-click ang SAVE. Idinaragdag ng Case Active ang pangalan sa listahan ng mga user na nauugnay sa app na ito.

img

Pagbabago ng isang user

Upang i-update ang isang maliit na bilang ng mga tala ng user, gamitin ang tool na I-edit at gumawa ng mga pagbabago sa bawat indibidwal na user.

  • Tandaan Upang baguhin ang isang malaking bilang ng mga tala ng user sa isang upuan, isaalang-alang ang paggamit ng tool sa Pag-import .
  1. Sa Admin panel, i-click ang tab na menu ng Mga User.
  2. Maghanap ng user at i-click ang I-edit.
  3. Gumawa ng mga pagbabago at i-click ang I-SAVE.

img

Mga susunod na hakbang

Ang Case Active ay hindi nagpapadala sa user ng notification sa email. Dapat makipag-ugnayan ang mga admin sa user at magpadala sa kanila ng link sa home page ng Case Active. Pagkatapos ay i-click ng user ang LOGIN button at kinukumpleto ang workflow para sa mga bagong signup.

  • Inilalarawan ng artikulong ito ang daloy ng trabaho sa pag-signup para sa mga bagong user.
