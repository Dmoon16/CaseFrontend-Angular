Paano Pamamahala ng Mga Admin ang Mga User

Ang mga admin ay gumagawa, nagbabago o nagdi-disable kung sino ang makaka-access ng mga app at case. Kapag nag-log in ang isang user sa platform sa unang pagkakataon, kailangan nilang kumpletuhin ang workflow ng profile.

  • Mag-click dito para matuto pa tungkol sa bagong workflow ng profile ng user.

Pamamahala ng isang gumagamit

Ang talahanayan sa ibaba ay nagpapakita ng limang paraan upang pamahalaan ang isang user. Mag-click sa isang link upang matutunan kung paano.

Pamamaraan Uri Mga aksyon Mga Detalye
Gamit ang UI UI magdagdag at magbago Button ng admin > Mga User
Habang gumagawa ng case at italaga ang user sa case UI idagdag Button ng Admin > Mga Case > Button na Bagong > Lumikha ng pahina ng Case
Awtomatikong lumikha ng user gamit ang pampublikong link sa pag-signup Aksyon ng user idagdag Admin button > Mga Setting > System Setup page
Tool sa pag-import UI magdagdag at magbago Button ng Admin > Mga Pagsasama > Mga Pag-import
Paggamit ng API (paparating na) API magdagdag at magbago N/A

Mga abiso

Ang platform ay hindi nagpapadala ng notification sa mga user kapag idinagdag sila sa isang app. Gayunpaman, maaaring i-enable ng mga admin ang mga notification para sa mga pagtatalaga ng kaso.
