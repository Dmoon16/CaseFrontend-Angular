Paano Gumawa ng Bagong Account ang Mga User

Pagkatapos magdagdag ng isang admin ng user sa isang case: * kung bago ang user sa caseactive, dapat kumpletuhin ng user ang isang workflow para gumawa ng account at tanggapin ang imbitasyon para ma-access ang case. * kung ang user ay may umiiral na caseactive account, ang user ay dapat tumanggap ng imbitasyon upang ma-access ang kaso.

Inilalarawan ng artikulong ito ang mga hakbang na dapat kumpletuhin ng isang user para gawin ang kanilang account.

Nagsisimula

Mayroong dalawang paraan upang maidagdag ang isang user sa isang caseactive na app

  • Ng admin: Kung lumikha ang admin ng user sa pamamagitan ng platform UI o Import tool , dapat magpadala ang admin sa user ng link sa account.caseactive.com upang magpatuloy sa pag-login.
  • Ayon sa user: Kung ang user ay gumawa ng account gamit ang isang signup link , ang user ay awtomatikong ire-redirect sa account.caseactive.com upang magpatuloy sa pag-login.

Nagla-log in

Kapag nag-log in ang user sa kanilang CaseActive account sa unang pagkakataon, ipo-prompt ng platform ang user na:

  • buksan ang kanilang email account at pagkatapos ay kopyahin/idikit ang anim na digit na code ng seguridad
  • kumpletuhin ang isang account profile

img

Pagkatapos i-click ng user ang UPDATE, ipapakita ang pahina ng Apps. Ipinapakita ng page ang pangalan ng app (pangalan ng kumpanya), ang uri ng user (role), status, at isang INVITATION button.

Upang ma-access ang app, dapat tanggapin ng user ang imbitasyon. Ang gumagamit ay dapat:

  1. I-click ang button na INVITATION.
  2. Suriin ang mga pahintulot.
  3. I-click ang tab na Mga Notification upang suriin at i-edit (opsyonal) ang mga tuntunin.
  4. I-click ang APPROVE ACCESS.

img

Kapag naaprubahan ng user, magpapakita ang platform ng button ng APP sa page ng user. Ang button na ito ay nagbibigay sa user ng access sa mga nakatalagang kaso.

Upang i-edit ang mga setting ng user, iki-click ng user ang ellipsis.

img

Ang user ay maaaring: * alisin ang app access * baguhin ang mga setting ng notification

img

Pagkatapos makumpleto ng user ang workflow, lalabas ang user name sa listahan ng Mga User ng admin. Ipinapakita ng listahan ang lahat ng user, naka-log in man o hindi.
