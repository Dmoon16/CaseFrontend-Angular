Paano Mag-update ng Profile ng User

Ipinapakita ng artikulong ito sa mga user kung paano mag-update ng impormasyon sa kanilang page ng Profile.

  • Maaaring i-update ng mga gumagamit ang kanilang sariling mga profile.
  • Maaaring i-update ng mga admin ang pahina ng Profile ng sinumang user.

Para sa mga gumagamit

Ang unang hakbang ay buksan ang pahina ng Profile. Sinusunod ng mga user at admin ang mga hakbang na ito upang i-update ang kanilang sariling page ng Profile.

  1. Mag-log in sa iyong Case Active na account.
  2. I-click ang tab na Profile sa tuktok ng page.
  3. Gumawa ng mga pagbabago.
  4. I-click ang I-save.

Karamihan sa mga field ay opsyonal, at ang ilang mga field ay kinakailangan. Ang mga kinakailangang field ay:

  • Pangalan
  • Huling pangalan
  • Email address
  • Wika
  • Timezone

Para sa mga admin

Maaari ding i-update ng mga admin ang impormasyon ng profile para sa ibang mga user. Kabilang dito ang tatlong field sa ilalim ng heading ng HOST USER INFO.

  1. Mag-log in sa iyong Case Active admin account.
  2. Maghanap ng app at i-click ang ADMIN button.
  3. I-click ang Mga User sa tuktok na menu.
  4. Pumunta sa sub-menu at piliin ang listahan ng user na gusto mong i-access (ACTIVE o DISABLED). Ang default na setting ay Aktibo.
  5. Maghanap ng user sa listahan at i-click ang I-edit.
  6. Gumawa ng mga pagbabago kung kinakailangan. (tingnan sa ibaba)
  7. I-click ang I-save.

Baguhin ang impormasyon ng user

Maaaring baguhin ng mga admin ang impormasyon ng user.

  1. Itakda ang Uri ng User (User o Manager).
  2. Itakda ang Katayuan ng User (Aktibo sa Disabled).
  3. Maglagay ng natatanging Tag ID. (opsyonal)
  4. I-click ang I-save.
