Paano Awtomatikong Lumikha ng Mga Kaso gamit ang Pampublikong Link ng Pag-signup

Ipinapakita ng artikulong ito ang mga admin kung paano gumawa ng case (at magdagdag ng user nang sabay-sabay) sa pamamagitan ng pagpapadala ng isang link sa pag-signup.

Hinahanap ang link

  1. Mag-log in sa admin account.
  2. Maghanap ng app sa page ng Apps at i-click ang ADMIN button.
  3. Sa panel ng Admin, i-click ang tab na menu ng Mga Setting.
  4. Tiyaking napili ang tab na submenu ng SYSTEM SETUP.

Pag-configure ng mga setting

Mag-scroll pababa sa USER SIGNUP heading.

Tiyaking nakatakda sa Naka-on ang field na Allow Signup. (kinakailangan) Kapag pinagana ang feature na ito, bubuo ng natatanging link ang Case Active.

  1. I-click ang field na Auto Create Case at piliin ang On. Ipinapakita ang field na Tungkulin ng Gumagamit ng Kaso.
  2. Pumili ng isang tungkulin. (tingnan sa ibaba)

img

Mga tungkulin ng gumagamit ng kaso

Mayroong tatlong kaso ng mga tungkulin ng gumagamit.

  • mga kliyente (inirerekomenda)
  • default
  • mga tauhan

Ipinapadala ang link

Ibahagi ang link sa isa o higit pang tao. Ang bawat user na nag-click sa link at nakumpleto ang pag-signup ay nagti-trigger sa platform na bumuo ng isang bagong kaso sa pahina ng mga kaso at awtomatikong italaga ang user sa kaso.
