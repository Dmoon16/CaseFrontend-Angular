Paano Awtomatikong Magtalaga ng Mga User gamit ang Public Signup Link

Natututo ang mga admin kung paano i-configure ang mga setting ng system para awtomatikong maitalaga ang mga bagong user sa isang bagong awtomatikong ginawang case.

Binubuksan ang pahina ng pag-setup

Ang unang hakbang ay buksan ang pahina ng System Setup.

  1. Mag-log in sa admin account.
  2. Maghanap ng app sa page ng Apps at i-click ang ADMIN button.
  3. I-click ang tab na Mga System at pagkatapos ay SYSTEM SETUP sa submenu.
  4. Mag-scroll pababa sa USER SIGNUP heading.

Pag-configure ng mga setting ng system

  1. Itakda ang Allow Signup at Auto Create Case na drop-down na menu sa On.
  2. I-click ang menu ng tungkulin at pumili ng opsyon (inirerekomenda ang default).
  3. Kopyahin at ibahagi ang link ng Public User Signup.

Ipinapadala ang link

Ang bawat user na magki-click sa link at kumpletuhin ang pag-signup ay magti-trigger sa platform na bumuo ng isang bagong kaso sa pahina ng Mga Kaso at awtomatikong magtatalaga sa user sa kasong iyon.
