Paano Magtalaga ng Mga User Habang Gumagawa ng Case

Ipinapakita ng artikulong ito sa mga admin kung paano magtalaga ng user habang gumagawa ng bagong case. Kung hindi pa nakarehistro ang user sa platform, maaari kang lumikha ng magdagdag ng bagong user habang gumagawa ng bagong case.

Paggawa ng kaso

  1. Mag-log in sa admin account.
  2. Maghanap ng app sa page ng Apps at i-click ang ADMIN button.
  3. I-click ang tab na menu ng Cases.
  4. I-click ang BAGONG button. May lalabas na bagong window.
  • Punan ang mga patlang upang lumikha ng isang case at pagkatapos ay mag-scroll pababa sa heading ng Mga Nakatalagang User.

Pagtatalaga ng isang user

Gumagamit ngayon

Sundin ang mga hakbang na ito para magtalaga ng kasalukuyang user sa bagong case.

I-click ang ASSIGN USERS button at punan ang mga field.

  1. I-click ang drop-down na User at pumili ng identifier (hal. email).
  2. Maglagay ng halaga sa blangkong field (hal. email address).
  3. Kung ang rekord ng user ay umiiral sa platform, ang field ay awtomatikong nagmumungkahi ng isang pangalan.
  4. Piliin ang tamang user.
  5. Pumili ng tungkulin ng user (mga kliyente, default, o kawani).
  6. Opsyonal, lagyan ng check ang kahon upang magpadala sa user ng notification sa email.
  7. I-click ang I-SAVE.

Idinaragdag ng Case Active ang user sa case. Ngayon ay maaaring i-click ng mga admin ang I-save upang isara ang window, o magdagdag ng bagong user, o mag-alis ng user.

Pagdaragdag ng bagong user

Kung ang platform ay hindi makahanap ng kasalukuyang record, i-click ang button para magdagdag ng bagong user.

img

Punan ang mga patlang at pagkatapos ay i-click ang I-SAVE upang idagdag ang user sa platform. Susunod, magtakda ng tungkulin ng user at i-click muli ang SAVE para italaga ang user sa case. Panghuli, i-click ang I-SAVE nang isa pang beses upang i-save ang case.
