Pag-format ng Data ng Pamamahala ng User gamit ang Tool sa Pag-import

Nasa ibaba ang isang listahan ng mga field ng data sa mga CSV file. Ang bawat opsyon sa pag-import ay may iba't ibang kumbinasyon ng mga field.

username

  • Ilagay ang email address ng user.

ibinigay na pangalan

  • Ilagay ang pangalan ng user.

apelyido

  • Ilagay ang apelyido ng user.

lokal

  • Sinusuri ng column na ito ang gustong wika ng user.
  • Ipasok ang en para sa Ingles, sp para sa Espanyol.

zoneinfo

  • Itinatakda nito ang time zone ng user. Pumasok sa kontinente at lungsod na walang puwang (gumamit ng underscore).
  • Dapat gumamit ng opsyon sa Profile > LANGUAGE at REGION > Timezone na drop-down na menu.
  • Gamitin ang pinakaangkop na time zone. Halimbawa, ang Boston ay wala sa drop-down, kaya maaari naming gamitin ang New York (o anumang lungsod sa EST/EDT time zone).

host_tag_id

  • Maglagay ng permanenteng label.
  • az, 0-9, -, / ay pinapayagan.

case_status

  • 0 = sarado
  • 1 = aktibo
