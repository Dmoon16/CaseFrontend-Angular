Pag-format ng Data ng Pamamahala ng Kaso gamit ang Tool sa Pag-import

Nasa ibaba ang isang listahan ng mga kinakailangang field sa mga CSV file upang pamahalaan ang mga kaso gamit ang tool sa Pag-import.

username

  • Ilagay ang email address ng user.

tag_id (isang id na maaari mong sanggunian sa ibang pagkakataon)

  • isang natatanging identifier (gumamit ng mga alphanumeric na character, tinatanggap ang mga gitling)

case_status

  • 0 = sarado
  • 1 = aktibo

date_open

  • buwan/araw/taon
  • mm/dd/yyyy

tungkol sa

  • isang natatanging paglalarawan ng kaso (gumamit ng mga alphanumeric na character, tinatanggap ang mga gitling)
