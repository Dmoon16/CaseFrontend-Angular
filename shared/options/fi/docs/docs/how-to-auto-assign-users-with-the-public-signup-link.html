Käyttäjien automaattinen määrittäminen julkisen kirjautumislinkin avulla

Järjestelmänvalvojat oppivat määrittämään järjestelmäasetukset niin, että uudet käyttäjät määritetään automaattisesti äskettäin automaattisesti luotuun tapaukseen.

Asetussivun avaaminen

Ensimmäinen vaihe on avata Systems Setup -sivu.

  1. Kirjaudu sisään järjestelmänvalvojan tilille.
  2. Etsi sovellus Sovellukset-sivulta ja napsauta JÄRJESTELMÄNVALVOJA-painiketta.
  3. Napsauta Systems-välilehteä ja sitten alivalikosta JÄRJESTELMÄN ASETUS.
  4. Vieritä alas USER SIGNUP -otsikon kohdalle.

Järjestelmäasetusten määrittäminen

  1. Aseta avattavat Salli kirjautuminen ja Automaattinen tapaus -valikot päälle.
  2. Napsauta roolivalikkoa ja valitse vaihtoehto (oletus on suositeltavaa).
  3. Kopioi ja jaa julkisen käyttäjän kirjautumislinkki.

img

Linkin lähettäminen

Jokainen käyttäjä, joka napsauttaa linkkiä ja suorittaa kirjautumisen loppuun, käynnistää alustan luomaan uuden tapauksen Tapaukset-sivulle ja määrittää käyttäjän siinä tapauksessa automaattisesti.
