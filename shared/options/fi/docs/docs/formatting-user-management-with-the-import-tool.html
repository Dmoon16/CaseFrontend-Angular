Käyttäjien hallintatietojen muotoilu tuontityökalulla

Alla on luettelo CSV-tiedostojen tietokentistä. Jokaisella tuontivaihtoehdolla on erilainen kenttien yhdistelmä.

käyttäjätunnus

  • Anna käyttäjän sähköpostiosoite.

etunimi

  • Syötä käyttäjän etunimi.

sukunimi

  • Anna käyttäjän sukunimi.

alue

  • Tämä sarake tarkistaa käyttäjän valitseman kielen.
  • Syötä en englanniksi, sp espanjaksi.

vyöhyketiedot

  • Tämä määrittää käyttäjän aikavyöhykkeen. Syötä maanosa ja kaupunki ilman välilyöntejä (käytä alaviivaa).
  • On käytettävä vaihtoehtoa Profiili > KIELI JA ALUE > Aikavyöhyke -valikosta.
  • Käytä parhaiten sopivaa aikavyöhykettä. Esimerkiksi Boston ei ole avattavassa valikossa, joten voimme käyttää New Yorkia (tai mitä tahansa kaupunkia EST/EDT-aikavyöhykkeellä).

host_tag_id

  • Anna pysyvä tunniste.
  • az, 0-9, -, / ovat sallittuja.

case_status

  • 0 = suljettu
  • 1 = aktiivinen
