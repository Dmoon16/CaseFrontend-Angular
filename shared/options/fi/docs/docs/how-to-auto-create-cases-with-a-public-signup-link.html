Tapausten automaattinen luominen julkisen kirjautumislinkin avulla

Tässä artikkelissa järjestelmänvalvojat voivat luoda tapauksen (ja lisätä samalla käyttäjän) lähettämällä yhden kirjautumislinkin.

Linkin paikantaminen

  1. Kirjaudu sisään järjestelmänvalvojan tilille.
  2. Etsi sovellus Sovellukset-sivulta ja napsauta JÄRJESTELMÄNVALVOJA-painiketta.
  3. Napsauta hallintapaneelissa Asetukset-valikkovälilehteä.
  4. Varmista, että JÄRJESTELMÄN ASETUS-alivalikko on valittuna.

img

Asetusten määrittäminen

Vieritä alas USER SIGNUP -otsikon kohdalle.

Varmista, että Salli kirjautuminen -kentän arvo on Käytössä. (pakollinen) Kun tämä ominaisuus on käytössä, Case Active luo yksilöllisen linkin.

  1. Napsauta Automaattinen tapaus -kenttää ja valitse Päällä. Tapauskäyttäjän rooli -kenttä tulee näkyviin.
  2. Valitse rooli. (Katso alempaa)

img

Tapauskäyttäjien roolit

Tapauskäyttäjärooleja on kolme.

  • asiakkaat (suositus)
  • oletuksena
  • henkilöstöä

Linkin lähettäminen

Jaa linkki yhdelle tai useammalle henkilölle. Jokainen käyttäjä, joka napsauttaa linkkiä ja suorittaa kirjautumisen loppuun, käynnistää alustan luomaan uuden tapauksen tapaussivulle ja määrittämään käyttäjän tapaukseen automaattisesti.
