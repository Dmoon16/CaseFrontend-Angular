Käyttäjien määrittäminen tapausta luotaessa

Tämä artikkeli näyttää järjestelmänvalvojille, kuinka käyttäjä määritetään uuden tapauksen luomisen yhteydessä. Jos käyttäjä ei ole vielä rekisteröitynyt alustalle, voit luoda uuden käyttäjän lisäämisen samalla kun luot uutta tapausta.

Tapauksen luominen

  1. Kirjaudu sisään järjestelmänvalvojan tilille.
  2. Etsi sovellus Sovellukset-sivulta ja napsauta JÄRJESTELMÄNVALVOJA-painiketta.
  3. Napsauta Tapaukset-valikkovälilehteä.
  4. Napsauta UUSI-painiketta. Uusi ikkuna tulee näkyviin.
  • Luo tapaus täyttämällä kentät ja vieritä sitten Assigned Users -otsikon kohdalle.

Käyttäjän määrittäminen

Nykyinen käyttäjä

Määritä nykyinen käyttäjä uuteen tapaukseen noudattamalla näitä ohjeita.

Napsauta ANNA KÄYTTÄJÄT -painiketta ja täytä kentät.

  1. Napsauta avattavaa Käyttäjä-valikkoa ja valitse tunniste (esim. sähköposti).
  2. Syötä arvo tyhjään kenttään (esim. sähköpostiosoite).
  3. Jos käyttäjätietue on olemassa alustassa, kenttä ehdottaa automaattisesti nimeä.
  4. Valitse oikea käyttäjä.
  5. Valitse käyttäjärooli (asiakkaat, oletus tai henkilökunta).
  6. Valitse vaihtoehtoisesti valintaruutu lähettääksesi käyttäjälle sähköposti-ilmoituksen.
  7. Napsauta TALLENNA.

Case Active lisää käyttäjän koteloon. Nyt järjestelmänvalvojat voivat sulkea ikkunan tai lisätä uuden käyttäjän tai poistaa käyttäjän napsauttamalla Tallenna.

Uuden käyttäjän lisääminen

Jos alusta ei löydä nykyistä tietuetta, napsauta painiketta lisätäksesi uuden käyttäjän.

Täytä kentät ja napsauta sitten TALLENNA lisätäksesi käyttäjän alustaan. Määritä seuraavaksi käyttäjän rooli ja napsauta uudelleen TALLENNA määrittääksesi käyttäjän tapaukseen. Napsauta lopuksi TALLENNA vielä kerran tallentaaksesi tapauksen.
