Kuinka luoda käyttäjiä kirjautumislinkin avulla

Tämä artikkeli näyttää järjestelmänvalvojille, kuinka käyttäjät luodaan lähettämällä kirjautumislinkki.

Ilmoittautumislinkin määrittäminen

  1. Napsauta Lisää käyttäjä -valintaikkunan alareunassa olevaa Lähetä heille kirjautumislinkkiä . Uusi sivu tulee näkyviin.
  2. Määritä kirjautumisasetukset. (Katso alempaa)

Varmista, että Salli kirjautuminen -kentän arvo on Käytössä. (pakollinen) Kun tämä on käytössä, Case Active luo yksilöllisen linkin.

Tarkista alla olevat vaihtoehdot ja kopioi linkki napsauttamalla kuvaketta. Voit sijoittaa linkin verkkosivustollesi tai lähettää linkin yhdelle tai useammalle henkilölle sähköpostitse tai millä tahansa viestintäkanavalla.

Tapauksen luominen

Valinnaisesti voit luoda automaattisesti tapauksen jokaiselle uudelle käyttäjälle ja määrittää käyttäjäroolin.

  1. Napsauta Automaattinen tapaus -kenttää ja valitse Päällä. Tapauskäyttäjän rooli -kentät tulevat näkyviin.
  2. Napsauta Case User Role -kohtaa ja valitse vaihtoehto. (Katso alempaa)

img

Tapauskäyttäjien roolit

Tapauskäyttäjärooleja on kolme.

  • asiakkaita
  • oletuksena
  • henkilöstöä

Linkin lähettäminen

Jaa linkki yhdelle tai useammalle henkilölle. Jokainen napsauttaa linkkiä ja suorittaa työnkulun loppuun Case Active -tilinsä aktivoimiseksi. Kun käyttäjä on suorittanut kirjautumisen työnkulun, Case Active lisää käyttäjän Käyttäjät-luetteloon.

img

Seuraavat vaiheet

Case Active lähettää käyttäjälle sähköposti-ilmoituksen, jossa on kirjautumislinkki. Käyttäjä napsauttaa linkkiä ja suorittaa uusien kirjautumisten työnkulun.

  • Tässä artikkelissa kuvataan uusien käyttäjien kirjautumisen työnkulku.
