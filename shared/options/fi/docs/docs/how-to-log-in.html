Kuinka kirjautua sisään

Opi kirjautumaan nykyiselle Case Active -tilillesi.

Kirjautua sisään

  1. Mene kirjautumissivulle .
  2. Napsauta LOGIN-painiketta.
  3. Syötä sähköpostiosoitteesi.
  4. Napsauta SEURAAVA.

Tilin vahvistaminen

  1. Avaa uusi selaimen välilehti ja kirjaudu sisään sähköpostitilillesi.
  2. Kopioi kuusinumeroinen koodi.
  3. Palaa Case Active -välilehteen.
  4. Liitä koodi kenttiin. Uusi ikkuna tulee näkyviin.

Case Active näyttää Sovellukset-sivun.

  • Huomautus Alusta näyttää PÄÄKÄYTTÄJÄ-painikkeen APP-painikkeen vieressä käyttäjille, joilla on järjestelmänvalvojan oikeudet.
