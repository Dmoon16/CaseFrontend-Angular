Felhasználói profil frissítése

Ez a cikk bemutatja a felhasználóknak, hogyan frissíthetik az információkat a profiloldalukon.

  • A felhasználók frissíthetik saját profiljukat.
  • A rendszergazdák bármely felhasználó profiloldalát frissíthetik.

A felhasználók számára

Az első lépés a Profil oldal megnyitása. A felhasználók és az adminisztrátorok követik ezeket a lépéseket saját profiloldaluk frissítéséhez.

  1. Jelentkezzen be Case Active fiókjába.
  2. Kattintson a Profil fülre az oldal tetején.
  3. Megváltoztatni.
  4. Kattintson a Mentés gombra.

A legtöbb mező nem kötelező, és néhány mező kitöltése kötelező. A kötelező mezők a következők:

  • Keresztnév
  • Vezetéknév
  • Email cím
  • Nyelv
  • Időzóna

Adminoknak

Az adminisztrátorok a többi felhasználó profiladatait is frissíthetik. Ez három mezőt tartalmaz a HOST USER INFO címsor alatt.

  1. Jelentkezzen be Case Active rendszergazdai fiókjába.
  2. Keressen egy alkalmazást, és kattintson az ADMIN gombra.
  3. Kattintson a Felhasználók elemre a felső menüben.
  4. Lépjen az almenübe, és válassza ki az elérni kívánt felhasználói listát (ACTIVE vagy DISABLED). Az alapértelmezett beállítás az Aktív.
  5. Keressen egy felhasználót a listában, és kattintson a Szerkesztés gombra.
  6. Szükség szerint végezze el a módosításokat. (lásd lejjebb)
  7. Kattintson a Mentés gombra.

Felhasználói adatok módosítása

A rendszergazdák módosíthatják a felhasználói adatokat.

  1. Állítsa be a Felhasználó típusát (Felhasználó vagy Menedzser).
  2. Állítsa be a felhasználói állapotot (Aktív értékre Letiltva).
  3. Adjon meg egy egyedi címkeazonosítót. (választható)
  4. Kattintson a Mentés gombra.
