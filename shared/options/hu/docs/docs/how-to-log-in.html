Hogyan lehet bejelentkezni

Ismerje meg, hogyan jelentkezhet be meglévő Case Active fiókjába.

Bejelentkezés

  1. Menjen a bejelentkezési oldalra .
  2. Kattintson a LOGIN gombra.
  3. Írd be az email címed.
  4. Kattintson a NEXT gombra.

Fiók ellenőrzése

  1. Nyisson meg egy új böngészőlapot, és jelentkezzen be e-mail fiókjába.
  2. Másolja ki a hatjegyű kódot.
  3. Térjen vissza a Case Active fülre.
  4. Illessze be a kódot a mezőkbe. Megjelenik egy új ablak.

A Case Active megjeleníti az Alkalmazások oldalt.

  • Megjegyzés A platform egy ADMIN gombot jelenít meg az APP gomb mellett a rendszergazdai jogosultságokkal rendelkező felhasználók számára.
