Ügykezelési adatok formázása az importáló eszközzel

Az alábbiakban a CSV-fájlok kötelező mezőinek listája található az esetek importálási eszközzel történő kezeléséhez.

felhasználónév

  • Írja be a felhasználó e-mail címét.

tag_id (az azonosító, amelyre később hivatkozhat)

  • egyedi azonosító (alfanumerikus karaktereket használjon, kötőjeleket elfogadunk)

case_status

  • 0 = zárva
  • 1 = aktív

dátum_megnyitás

  • hónap nap év
  • éééé.hh/nn

ról ről

  • a kis- és nagybetűk egyedi leírása (alfanumerikus karaktereket használjon, kötőjelek megengedettek)
