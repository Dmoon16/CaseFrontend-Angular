Kako ažurirati korisnički profil

Ovaj članak pokazuje korisnicima kako ažurirati podatke na svojoj stranici profila.

  • Korisnici mogu ažurirati vlastite profile.
  • Administratori mogu ažurirati stranicu profila bilo kojeg korisnika.

Za korisnike

Prvi korak je otvaranje stranice Profil. Korisnici i administratori slijede ove korake za ažuriranje vlastite stranice profila.

  1. Prijavite se na svoj Case Active račun.
  2. Pritisnite karticu Profil na vrhu stranice.
  3. Napravi promjene.
  4. Pritisnite Spremi.

Većina polja nije obavezna, a neka su obavezna. Obavezna polja su:

  • Ime
  • Prezime
  • Email adresa
  • Jezik
  • Vremenska zona

Za admine

Administratori također mogu ažurirati informacije o profilu za druge korisnike. Ovo uključuje tri polja pod naslovom HOST USER INFO.

  1. Prijavite se na svoj Case Active administratorski račun.
  2. Pronađite aplikaciju i kliknite gumb ADMIN.
  3. Kliknite Korisnici na gornjem izborniku.
  4. Idite na podizbornik i odaberite popis korisnika kojem želite pristupiti (AKTIVNO ili ONEMOGUĆENO). Zadana postavka je Aktivno.
  5. Pronađite korisnika na popisu i kliknite Uredi.
  6. Napravite promjene prema potrebi. (Pogledaj ispod)
  7. Pritisnite Spremi.

Promjena podataka o korisniku

Administratori mogu promijeniti korisničke podatke.

  1. Postavite vrstu korisnika (korisnik ili upravitelj).
  2. Postavite status korisnika (od aktivnog do onemogućenog).
  3. Unesite jedinstveni ID oznake. (neobavezno)
  4. Pritisnite Spremi.
