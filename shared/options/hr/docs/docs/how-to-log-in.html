Kako se prijaviti

Saznajte kako se prijaviti na svoj postojeći Case Active račun.

Prijavljivanje

  1. Idite na stranicu za prijavu .
  2. Pritisnite gumb PRIJAVA.
  3. Upišite svoju e-mail adresu.
  4. Kliknite DALJE.

Provjera računa

  1. Otvorite novu karticu preglednika i prijavite se na svoj račun e-pošte.
  2. Kopirajte šesteroznamenkasti kod.
  3. Povratak na karticu Case Active.
  4. Zalijepite kod u polja. Prikazuje se novi prozor.

Case Active prikazuje stranicu s aplikacijama.

  • Napomena Platforma prikazuje gumb ADMIN pokraj gumba APP za korisnike s ovlastima administratora.
