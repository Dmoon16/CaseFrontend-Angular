Formatiranje podataka o dodjeli korisničkih slučajeva pomoću alata za uvoz

Ispod je popis podatkovnih polja u CSV datotekama. Svaka opcija uvoza ima različitu kombinaciju polja.

case_ID

  • Unesite ID korisnika ili oznake slučaja.
  • Idite na stranicu Slučajevi > Uredi > ID oznake ILI Idite na stranicu Slučajevi > Uredi > ID slučaja u url-u.

korisnički_ID

  • Unesite ID korisnika ili ID oznake.
  • Idite na stranicu Korisnici > Uredi > ID oznake ILI Idi na stranicu Korisnici > Uredi > ID korisnika u url-u.

slučaj_uloga_redoslijed

  • Ostavite ćeliju praznu.

case_role_ID

  • Unesite: klijenti, osoblje ili zadano.

case_notify

  • 0 = ne
  • 1 = Da, pošalji korisniku obavijest e-poštom.
