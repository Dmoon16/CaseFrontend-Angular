Kako stvoriti korisnike s vezom za prijavu

Ovaj članak pokazuje administratorima kako stvoriti korisnike slanjem veze za prijavu.

Konfiguriranje veze za prijavu

  1. Kliknite vezu Pošalji im prijavu na dnu dijaloškog okvira Dodaj korisnika. Prikazuje se nova stranica.
  2. Konfigurirajte opcije prijave. (Pogledaj ispod)

img

Provjerite je li polje Dopusti prijavu postavljeno na Uključeno. (obavezno) Kada je omogućeno, Case Active generira jedinstvenu vezu.

Pregledajte opcije u nastavku, a zatim kliknite ikonu za kopiranje veze. Poveznicu možete postaviti na svoju web stranicu ili je poslati jednoj ili više osoba putem e-pošte ili bilo kojeg komunikacijskog kanala.

img

Izrada slučaja

Po želji, možete automatski stvoriti slučaj za svakog novog korisnika i dodijeliti korisničku ulogu.

  1. Pritisnite polje Auto Create Case i odaberite On. Prikazuju se polja Case User Role.
  2. Pritisnite Case User Role i odaberite opciju. (Pogledaj ispod)

Uloge korisnika slučaja

Postoje tri uloge korisnika case.

  • klijentima
  • zadano
  • osoblje

Slanje linka

Podijelite vezu s jednom ili više osoba. Svaka osoba klikne poveznicu i dovrši tijek rada kako bi aktivirala svoj Case Active račun. Nakon što korisnik dovrši tijek prijave, Case Active dodaje korisnika na popis korisnika.

img

Sljedeći koraci

Case Active korisniku šalje obavijest e-poštom s vezom za prijavu. Korisnik klikne vezu i dovrši radni tijek za nove prijave.

  • Ovaj članak opisuje radni tijek prijave za nove korisnike.
