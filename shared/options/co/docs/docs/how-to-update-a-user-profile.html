Cumu aghjurnà un Profil d'Usuariu

Questu articulu mostra à l'utilizatori cumu aghjurnà l'infurmazioni nantu à a so pagina di Profilu.

  • L'utilizatori ponu aghjurnà i so propri profili.
  • L'amministratori ponu aghjurnà a pagina di Profilu di qualsiasi utilizatore.

Per l'utilizatori

U primu passu hè di apre a pagina Profile. L'utilizatori è l'amministratori seguitanu sti passi per aghjurnà a so propria pagina di Profilu.

  1. Accedi à u vostru contu Case Active.
  2. Cliccate nantu à a tabulazione Profile in cima di a pagina.
  3. Fate cambiamenti.
  4. Cliccate Salvà.

A maiò parte di i campi sò opzionali, è certi campi sò obligatori. I campi richiesti sò:

  • Nomu
  • Casata
  • Indirizzu mail
  • Lingua
  • Timezone

Per l'amministratori

L'amministratori ponu ancu aghjurnà l'infurmazioni di u prufilu per altri utilizatori. Questu include trè campi sottu l'intestazione HOST USER INFO.

  1. Accedi à u vostru contu amministratore Case Active.
  2. Locate una app è cliccate u buttone ADMIN.
  3. Cliccate Users nantu à u menù superiore.
  4. Andà à u sottu-menu è selezziunate a lista di l'utilizatori chì vulete accede (ACTIVE o DISABLED). L'impostazione predefinita hè Active.
  5. Locate un utilizatore nantu à a lista è cliccate Edit.
  6. Fate cambiamenti cum'è necessariu. (vede sottu)
  7. Cliccate Salvà.

Cambia l'infurmazione d'utilizatore

L'amministratori ponu cambià l'infurmazioni d'utilizatori.

  1. Definisce u Tipu d'Usuariu (Usuariu o Manager).
  2. Stabbilisce u Status di l'Usuariu (Attivo à Disabilitatu).
  3. Inserite un ID tag unicu. (opcional)
  4. Cliccate Salvà.
