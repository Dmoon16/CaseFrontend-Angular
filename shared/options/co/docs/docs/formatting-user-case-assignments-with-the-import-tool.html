Formatting User Case Assignments Dati cù u Strumenta Import

Quì sottu hè una lista di campi di dati in i schedari CSV. Ogni opzione d'importazione hà una cumminazione differente di campi.

case_ID

  • Inserite u casu User o Tag ID.
  • Vai à a pagina Case> Edit> Tag ID O Vai à a pagina Cases> Edit> Case ID in url.

user_ID

  • Inserite l'ID di l'utilizatore o l'ID Tag.
  • Vai a pagina Users> Edit> Tag ID O Vai à Users page> Edit> User ID in url.

case_role_order

  • Lasciate a cellula in biancu.

case_role_ID

  • Inserite: clienti, persunale, o predeterminatu.

case_notify

  • 0 = no
  • 1 = Iè, mandate à l'utilizatori una notificazione email.
